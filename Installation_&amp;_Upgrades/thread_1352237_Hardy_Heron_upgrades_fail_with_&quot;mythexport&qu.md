### Post by lemming465 on 2009-12-12
I'm not sure how many exciting repositories you have enabled, but the [legacy mythexport]("https://help.ubuntu.com/community/MythExport_old") page suggests it is an addon to MythTV.  If you don't have a TV tuner card in the first place, or aren't using that in the second place, I'd go into synaptic, search on *myth*, and uninstall anything you aren't using.

Merely having error messages related to this when doing package maintenance is liable to be repeatedly annoying, but unlikely to progress to fatal.
