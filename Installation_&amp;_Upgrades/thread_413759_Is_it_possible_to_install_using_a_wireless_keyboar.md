### Post by viciouslime on 2007-04-19
Is it a usb keyboard? If so you might need to enable legacy support in your bios. You will also find that such a keyboard wouldn't work for the XP install either. If after enabling legacy support you still have problems, then maybe try installing with a normal wired keyboard and then plug your wireless one in  after ubuntu is installed.
