### Post by nickleboyblue on 2011-03-03
The only thing I can think if is that the ubuntu 10.10 cd might be checked in your software sources list.  To fix it, just open the software center, click edit->software sources, and uncheck it under the "Ubuntu Software" tab.

If that doesn't fix it for you, you might have found a bug.
