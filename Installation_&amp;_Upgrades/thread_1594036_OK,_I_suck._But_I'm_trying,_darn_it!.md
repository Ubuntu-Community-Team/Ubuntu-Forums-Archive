### Post by lechien73 on 2010-10-13
If you want to reduce your dependence on Windows, then find Linux alternatives to the software you're currently using. If there are no Linux alternatives, then see if the Windows software will run under Wine - or if not, then install VirtualBox and have a Windows virtual machine.

I found that the best way for me to stop booting Windows was to get rid of it completely. I was using video editing software, but found a suitable open source alternative. I was also using Photoshop, but now use the GIMP. Some proprietary telephone system management software that I needed for my job now runs in VirtualBox, and other Windows software runs quite happily under Wine.

So, just do a bit of research, make full backups, and then take the plunge :)
