### Post by trivialpackets on 2005-02-27
Typically no, I get the name resolution error as well.  All that it means is that you do not have an active internet connection at the time of booting.  During your boot sequence, gnome isn't running.  I don't recall ever seeing this, but it makes sense as it's not running.  If you're able to log into the GUI for Ubuntu I don't think there are any issues.
