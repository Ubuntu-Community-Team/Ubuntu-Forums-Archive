### Post by Mark Phelps on 2010-10-17
If you have an external drive, or another partition large enough, install Clonezilla and do a full backup of your current 10.04 install.

Only THEN, upgrade to 10.10.

That way, if you have serious problems with 10.10, you can restore 10.04 using Clonezilla.
