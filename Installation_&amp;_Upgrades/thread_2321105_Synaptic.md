If you were to install Ubuntu Mate you might find that you do not need to install Unity because the Ubuntu Mate developers have worked very hard to provide a large opportunity for customizing the Mate desktop. You will find that it is possible to run Ubuntu using the Compiz compositor just like in Ubuntu and that there are customisation options that will give you a Unity like user interface.
