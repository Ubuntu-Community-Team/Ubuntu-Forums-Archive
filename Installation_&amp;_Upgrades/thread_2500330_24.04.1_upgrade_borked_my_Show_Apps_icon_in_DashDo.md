### Post by piagrosser on 2024-09-10
I had the same problem after upgrading. It turned out, that in my gnome settings the animation of the show apps button was turned of for unknown reason. I fixed the problem using the dconf editor (maybe you have to install this). In the Dconf editor you can enter the path "/org/gnome/shell/extensions/dash-to-dock/animate-show-apps" and then activate "use default value".

I hope this will help you!
