### Post by crane on 2007-09-30
Yes you can. Just leave the FF partition alone when you install GG. When you get to the partitioner during installation just resize the FF install if it it using the entire disk and make a new partition, You can use the same swap partition with no problems. When you done you will be dual booting FF and GG or triple pooting it you have Windows on there as well.
