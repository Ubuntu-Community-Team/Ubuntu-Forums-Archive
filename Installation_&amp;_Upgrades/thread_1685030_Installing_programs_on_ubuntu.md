### Post by TBABill on 2011-02-10
And if you are pretty new to Ubuntu, you can also try the Ubuntu Software Center under Applications. Point and click functionality there with a search capability. As stated by oldfred, the preferred method is to install from Synaptic or the software center, but you can also install from .deb or from source. 
 
I'd suggest some time on [https://help.ubuntu.com](https://help.ubuntu.com) and reading about how the OS works, how software is installed, etc., to get an overview of the various methods and some solutions to problems if encountered.
