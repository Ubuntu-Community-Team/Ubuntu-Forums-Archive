### Post by grahammechanical on 2016-07-26
Something means anything. Please quote exactly what the message says.

Non free means that the software may or may not contain proprietary (not open source) code. Non free means that we are not free to modify the code or to re-distribute the modified code. Non free does not mean that we must pay for the software.

Some proprietary software is distributed as free to download, install and use in accordance with certain licenses and without charge. An example of this are video drivers that will install through the Additional Drivers utility.

Some open source software is distributed as Free and Open Source Software (FOSS) but there is a charge for using it and we must also comply with certain licenses.
