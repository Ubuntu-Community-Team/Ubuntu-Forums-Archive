### Post by GrumpySmurf on 2005-04-18
Hello,

I have a similar system to yours and didn't experience any difficulty or driver issues.  My MSI board is nforce3/250 and I have an external sound card (Envy 24 chipset).  

You might want to download the Ubuntu LiveCD to check it out before you install.  All your hardware should autodetect and operate as expected.  If you want 3D support for your graphics card, you'll need to install nvidia's binary drivers.  There is ample documentation for this on the main Ubuntu Linux site ([http://www.ubuntulinux.org/)](http://www.ubuntulinux.org/)).    I find the Unofficial Ubuntu Guide to be quite helpful as well: [http://www.ubuntuguide.org/](http://www.ubuntuguide.org/).  Good luck!
