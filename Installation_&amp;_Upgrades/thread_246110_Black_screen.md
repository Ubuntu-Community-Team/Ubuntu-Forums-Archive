### Post by alejandroD on 2006-08-29
I had a very similar problem to this, minus the noises. My problem was the nvidia drivers (I have a GeforceFX 5200). I solved this by installing the nvidia drivers at the command prompt in the recovery mode. Instructions can be found at [http://ubuntuguide.org/wiki/Dapper#How_to_install_Graphics_Driver_.28NVIDIA.29](http://ubuntuguide.org/wiki/Dapper#How_to_install_Graphics_Driver_.28NVIDIA.29)

Hope that helps.
