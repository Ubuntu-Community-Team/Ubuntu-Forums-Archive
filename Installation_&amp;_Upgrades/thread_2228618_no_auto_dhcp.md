### Post by ajgreeny on 2014-06-08
If this is on a laptop with only a wireless connection it may simply be that the wireless chip is not working in the live system.

You can still go ahead and install without a connection but you won't be able to add the restricted packages or updates until you have finished installing and can enable the wireless.  If you can attach by cable you should be able to get a connection and install with updates, etc etc.
