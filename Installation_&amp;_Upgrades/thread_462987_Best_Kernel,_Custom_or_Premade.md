### Post by Topsiho on 2007-06-03
From Synaptic information I get that the k7 linux-images are now obsolete and should be replaced by linux-image-generic.

From the descriptions of the linux-image-generic options I read that these should not be installed directly, but that the meta package for Linux-image-generic should be installed.

I used to use the k7 linux images as I have an AMD Athlon processor but did not find any advantage compared to an ordinary x86 kernel image.

I suggest you read the information in Synaptic yourself.

And of course you could build a kernel yourself, which could be a lean and mean kernel specially made for your hardware. Not sure about security if you do this.
