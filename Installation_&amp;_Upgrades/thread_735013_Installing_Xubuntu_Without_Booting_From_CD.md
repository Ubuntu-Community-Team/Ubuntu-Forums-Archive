### Post by wpshooter on 2008-03-25
You need to use the ALTERNATE (text based) CD version which does not run the O/S directly from the CD but indeed installs the O/S directly from the CD to the computers hard drive.

If there is not anything else on these computers that is needed to be saved, I would first WIPE the hard drives on them with [www.killdisk.com](www.killdisk.com) to get everything off of them before installing Xubuntu.

Good luck.
