### Post by Idefix82 on 2011-08-13
Just download an ISO image, burn it on CD, boot from CD, choose the "install Ubuntu, use entire hard drive" option (or something like that). You don't even need to have XP installed for that, the HD can be completely blank. Basically, it doesn't matter, what's on the hd, it is all going to be wiped out during the Ubuntu install.

Before you do that, I recommend choosing the "Try Ubuntu from CD" option to make sure that all the hardware is recognised. Ideally, you should also have internet access during the install, then Ubuntu will immediately download various updates.
