### Post by erginemr on 2008-04-03
I believe you should download **compizconfig-settings-manager**, **python-compizconfig** and probably **libcompizconfig0** from:
[http://packages.ubuntu.com/hardy/compizconfig-settings-manager](http://packages.ubuntu.com/hardy/compizconfig-settings-manager)
[http://packages.ubuntu.com/hardy/python-compizconfig](http://packages.ubuntu.com/hardy/python-compizconfig)
[http://packages.ubuntu.com/hardy/libcompizconfig0](http://packages.ubuntu.com/hardy/libcompizconfig0)

put them in the same directory in a pen-drive and copy all to your desktop. Then, from a terminal:
```
cd ~/Desktop/*your_directory_here*
sudo dpkg -i *.deb
```
should install the necessary packages to run **ccsm**.

However, I have found this thread for you:
[http://ubuntuforums.org/showthread.php?t=678814](http://ubuntuforums.org/showthread.php?t=678814)

where the last post suggests adding:
```
Section "Screen"
...
[B][COLOR="Red"]	Option		"AddARGBGLXVisuals"	"True"
[/COLOR][/B]...
EndSection
```
into section "Device" in your /etc/X11/xorg.conf file. You can edit this file with:
```
gksudo gedit /etc/X11/xorg.conf
```
