### Post by mrady on 2011-10-28
Upgrade went well but found that after desktop came up after logging in with my only account on the machine that the desktop was frozen. Could move mouse but couldn't click on another thing. 

Found that logging in under a Guest account on the same machine and then switching back to my main account there was a dialog window for Synaptic Package Manager flashed up for a second with the Admin account password prompt dialogue window. 

It disappeared and the desktop was frozen as before. I tried typing my main account password and it worked and Synaptic Package Manager came up. 

Found that Synaptic Package Mgr was set up in my Start Up list. I removed it and problem hasn't reoccurred since.

Bit weird but problem resolved.
