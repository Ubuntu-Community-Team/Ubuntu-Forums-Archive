### Post by tubasoldier on 2006-11-20
once you have it installed open a terminal and type in ```
aptitude install kubuntu-desktop
```

or alternatively for a more lightweight solution
```
aptitude install kde-base
```

### Post by aysiu on 2006-11-21
You cannot *add* Kubuntu from the live CD (otherwise known as the Desktop CD).

You can:

1. Install over (i.e., erase) your Ubuntu installation using the live CD.

2. Add Kubuntu with the Alternate CD (not the live one)

3. Add Kubuntu over the internet using the online repositories
