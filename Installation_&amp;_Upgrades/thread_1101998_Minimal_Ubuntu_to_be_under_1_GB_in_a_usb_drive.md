### Post by snowpine on 2009-03-23
> **Gokul Nagarajan said:**
> Hi Snowpine,

8.04 does not have the built in USB creator like 8.10 does. However, you can use a utility called Unetbootin to achieve the same goal, or read the tutorial at [www.pendrivelinux.com](www.pendrivelinux.com). Either method will fit the full Ubuntu ISO onto a 1gb pen drive.

I recommend a pen drive type of install over a full install (like it sounds like you're doing) because a full install will wear out the pen drive. 

If you'd rather do a minimal install, here's a good tutorial: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)
