### Post by snowpine on 2010-09-28
I would not install Xubuntu with less than 256mb of RAM, personally.

Also I have not used Wubi before, but my understanding is it installs Xubuntu "inside of" Windows; this is probably not the best choice for performance on old hardware. Better to install it on its own separate partition as a dual boot.
