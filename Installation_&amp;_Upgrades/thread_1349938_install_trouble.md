Hey, shansen,

The login screen on a live CD seems to occur when there is something wrong with the CD or CD drive. I would recommend installing from a USB drive using [USB-creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") (in Ubuntu) or [Unetbootin]("http://unetbootin.sourceforge.net/") (in Windows).

Good luck!
