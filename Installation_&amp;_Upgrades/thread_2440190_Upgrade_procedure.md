### Post by Impavidus on 2020-04-07
Lubuntu 18.04 can't be upgraded at all. The changes from 18.04 (with LXDE desktop) to anything later (with LXQt desktop) are so substantial that a failure would be too likely, so that upgrade isn't offered.

32 bit can only be upgraded to 32 bit, 64 bit only to 64 bit. And 18.04 is the last 32 bit release supported, so 32 bit 18.04 cannot be upgraded regardless the flavour.
