### Post by TheFu on 2020-04-14
There are thousands of youtube videos explaining how to install Linux onto other systems.  Linux doesn't use .exe files.  

Step 1: get the iso file
Step 2: Place the iso file onto media to be installed from.  That can be an optical disc or flash storage or SDCard or ... any other media from which your computer has the ability to boot.  This must be a bit-for-bit placement, not a simple copy.  The common term used is "burning it".

Step 2 has different methods, depending on the OS you have available already.
[https://ubuntuforums.org/showthread.php?t=2422843&p=13872720#post13872720](https://ubuntuforums.org/showthread.php?t=2422843&p=13872720#post13872720) explains and has a link for Windows instructions.
