### Post by louieb on 2007-08-22
Hello, It going to be tough to keep all your settings.  For your personal stuff such as your bookmarks you will need to backup your home folder. For your wireless settings  and network setting probably  going to start over.  I started using Ubuntu w/ v5.10 also, IMHO 7.04 is so much slicker that you will be glad  you did. You can use the manual partitioning and reuse the the current 5.10 partitions no problem.

7.04 uses more memory that 5.10 did. 5.10 runs  great in 256MB. to get the same in 7.04 probably going to need 512MB.
