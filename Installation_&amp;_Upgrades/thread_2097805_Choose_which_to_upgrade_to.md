### Post by Frogs Hair on 2012-12-24
Upgrades go in order except for LTS releases. An upgrade from 10.04 to 12.04 is possible. If you were running 11.04 you would have to go to 11.10 -12.04 and so on.   [https://help.ubuntu.com/community/PreciseUpgrades](https://help.ubuntu.com/community/PreciseUpgrades)
