### Post by lemming465 on 2008-05-05
It depends on what you mean by "manually install".  If you mean "add it from the standard repositories" go to System -> Administration -> Synaptic package manager, search for it, click the check box, and press the Apply button.

If you mean "download source", I have an entire 45 minute openoffice impress presentation on [building tools from source]("https://mywebspace.wisc.edu/jeleinwe/web/BTS/") which you could try.
