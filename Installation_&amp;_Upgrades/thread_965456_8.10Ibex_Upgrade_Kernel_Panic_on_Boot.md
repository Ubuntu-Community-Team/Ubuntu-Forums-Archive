### Post by bribera on 2008-11-03
My hunch is that this is a 64-bit specific bug; there's never enough QA love for the 64-bit versions.

Since I don't even have 4gb of ram (and since this was a work machine, with all of my important files safely tucked away in version control), I wiped and set up Xubuntu 32-bit. Things are dandy now.

Henceforth, my rule of thumb is this: unless I actually need the extended memory addressing (ie, a server), there's not a compelling reason to go 64-bit.
