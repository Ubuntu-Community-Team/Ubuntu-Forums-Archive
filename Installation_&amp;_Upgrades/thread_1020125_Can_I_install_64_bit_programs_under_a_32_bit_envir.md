### Post by ushimitsudoki on 2008-12-23
The latest version of VirtualBox actually supports 64-bit guests on a 32-bit host. It's experimental, but if you just need to check out a 64-bit only application or something like that, you might want to give it a look.

Look at the 4th or so bullet point in the [Changelog]("http://www.virtualbox.org/wiki/Changelog") for more.
