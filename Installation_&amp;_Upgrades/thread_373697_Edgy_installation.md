### Post by pixeldotz on 2007-03-02
Nero Express should have a button that allows you to use Nero Class mode. I only use Nero Classic as i find Express crippling as to wha the program can really do.

once you get classic running, you should have an option to burn an 'image' somewhere in your menus.

select it.
in the dialog menu, make the drop down say  *.iso instead of Nero Image File (*.nrg).
find your ubuntu iso and click ok.
select your writing speed.
make sure the options to close the disk and DISK AT ONCE are checked.
once it starts burning there shoudl be a box at the bottom left that says VERIFY WRITTEN DATA, make sure it's checked.

that should work nicely and you should now be able to boot your ubuntu cd.
