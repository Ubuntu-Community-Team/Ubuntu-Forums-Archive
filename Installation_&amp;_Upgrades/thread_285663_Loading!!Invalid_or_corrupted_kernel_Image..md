HI

I had all sorts of message about corrupt files whilst trying to install a burnt image.  I ended up burning the image on the same burner I was using for the install, luckily it was USB.  This cured all the problems and the disk now checks out ok.  If you can, I would advise using the same burner for writing the image and installing Ubuntu.
