### Post by jhawk on 2011-10-27
Windows Developer Preview and Wubi work in amd64.

I had to install WUBI got and error.
However, on reboot Ubuntu completed install.

Getting Ubuntu to work from Menu was another story,
I downloaded a BCD edit tool that supports Win Dev. Preview at
[http://boyans.my3gb.com](http://boyans.my3gb.com)   0.9.3.1 Version.

This allowed me to set default to Ubuntu and then I had the option of Windows Developer Preview or Ubuntu.

System is stable and working fine.

You might get an error free install by running WUBI as administrator by right clicking and picking that option.

enjoy
