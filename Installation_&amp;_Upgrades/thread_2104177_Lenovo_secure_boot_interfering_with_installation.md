### Post by dorruk on 2013-01-12
SecureBoot can be easily disabled from BIOS. It always comes loaded with these new laptops that are Windows 8 preinstalled.

See this: [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)  Scroll to the very bottom of the page.
