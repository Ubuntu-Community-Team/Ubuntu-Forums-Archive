### Post by dstew on 2008-03-11
HP's are fairly notorious for using graphics adapters that Ubuntu has trouble with. Usually you can get around this with a little work.

To get the Live CD to work, try booting again in "Safe Grapics Mode." I think you can select this from the very opening menu. If not, press F6 to get a list of further Live CD boot options, and it should be there.

If all else fails, you can install Ubuntu using the Alternate Install CD, which uses a text-based installer.

EDIT: By the way, how much memory does it have? The live CD needs at least 256 Mb free (not given to video), but more is better.
