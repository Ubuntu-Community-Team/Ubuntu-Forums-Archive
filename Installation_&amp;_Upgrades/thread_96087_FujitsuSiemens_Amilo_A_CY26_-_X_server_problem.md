### Post by daller on 2005-12-01
Use this as an install argument:

linux vga=771

Or if you already have a lot on your pc: (doubt that)

Add this at the end of the kernel line in /boot/grub/menu.lst:

vga=771

example: (my own laptop)

title           Ubuntu, kernel 2.6.12-10-386
root            (hd0,0)
kernel          /boot/vmlinuz-2.6.12-10-386 root=/dev/hda1 ro quiet splash **vga=771**
initrd          /boot/initrd.img-2.6.12-10-386
savedefault
boot
