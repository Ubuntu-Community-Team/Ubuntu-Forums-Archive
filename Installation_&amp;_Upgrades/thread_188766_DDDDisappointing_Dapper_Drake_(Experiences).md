### Post by xerman on 2006-06-04
Hi all,

Excuse me, for these days. I couldn't write the rest of the thread as I got some issues that killed firefox while writing and I did not know that the thread was saved on the server. The issue was due to who knows what. All I can say is that I was running xgl-compiz.

The experiences are the following:
Configuration:
Gigabyte GA-K8NF-9+AMD64 3200+
1GB ram
2x120GB hd
LG DVD Reader
Pioneer DVD/RW
Nvidia 6600 256MB
Philips 170S LCD.
Ubuntu Dapper 6.06 64
Installed: Gnome+KDE+Abiw+Gnumeric+Koffice+Inkscape+Qcad+Xgl-compiz(right now not activated due to issues)+Splashy+GnomeBaker+QDVDauthor+MonoDevelop+DevHelp+Screem+Reader32+Wine+Glabels
+Planner+Thunderbird+Sylpheed+GnomeChat.

1- Installing the system produces a whole bunch of not needed foreign fonts that populate the font menu and don't add any improvement. When the user checks the selected locale during the install process just that locale and its fonts should be installed. I had to uninstall manually through Synaptic those fonts for Hangul, Lao, and many more. This uninstall requires removal of "ubuntu-desktop". Say what???

2- I'm having issues as stated in another thread with Ubuntu keeping network preferences. Everytime I boot and get to gnome I have to choose the Network Location as Ubuntu does not keep the network preferences. This happens since I have Gnome and KDE both installed.

3- There is still no 64bit/32bit easy mix. Right now I'm using Dapper64 but I'll have to "downgrade" to Dapper32 due to application needs.

4- OpenOffice takes 25 seconds to launch while on Breezy it took just 4. This is a Drawback.

5- I cannot use Chinese whithin OpenOffice. I can though in Abiword and other gnome-apps.

6- Evolution does not allow me to select font face. This may not be important with latin characters, but it really is when using non latin. Printing preview of certain chinese emails produces dissapearing of half, two thirds the characters of the email. I've tried Thunderbird with same results.

7- Everybody says font rendering has improved. I see the same rendering as in Breezy. On my LCD monitor fonts have shadows withouth using xgl-compiz. And certain ones are chunky. (Nvidia 6600 256MB).

8- While DesktopCD has a graphic boot, once installed, the system shows up the same grub as those old times.

9- BloqNum on startup is not still default, and Gnome does not have a check box to mark BloqNum by default on startup. By the way, to install "numlockx" the only thing needed to do is go to Synaptic and mark "numlockx" for installing. That's it. No need to edit no file, no X config file, no nothing.

10- To configure my scanner (multifunction ethernet with its own IP address) to be used by Xsane I had to surf the web to get the HPLIP info needed, and I just could set it to be used by Xsane adding a line to the Command in the applications menu option of Xsane with Alacarte.

11- There is no KDE theme that matches Human Gnome. This would be interesting for users that need certain KDE apps to keep the Desktop experience as friendly as possible.

12- Once one sets certain bookmars whithin Nautilus the Places menu expands a lot. Why there isn't a Bookmars sub menu, and a Recent apps submenu?

13- Applications menu structure is perfect for default install. Once one starts to install other apps, distribution of these is a bit chaotic.

14- Deskbar has issues with xgl-compiz not allowing to type text in the textfield on the top panel. This is ok as xgl-compiz is an alpha package.

15- Xgl-compiz theme makes dissappear the close,maximize,minimize buttons on certain windows. Close button on Dialogs never exist, and the only way to move those windows is by pressing "Alt" or the assinged key plus mouse click&move.

16- There is still no easy way to difference Gnome apps from KDE when selecting them to install.

17- "aptitude" should be default.

18- HPLIP for printing and scanning on hp products is 3 version numbers behind. I still cannot acces the card reader of the multifunction.

19- Digital cameras could mount on Desktop just like another storage device in nautilus. Same for mobile phones and pda's.

20- Xgl-Compiz produces variety of panel issues. I once boot the PC and found 3 Trashes on the bottom panel. Desktop pager looses desktop separator. This is ok as I know is still alpha soft.

21- I get many times the "Force quit Dialog" in any app, from Firefox to Inkscape.

This is what I remember now I had interest in showing. My apologies for the missunderstanding. I did not mean to open a thread with this subject just to say Hi.

Regards and apologies to all.
Take care
