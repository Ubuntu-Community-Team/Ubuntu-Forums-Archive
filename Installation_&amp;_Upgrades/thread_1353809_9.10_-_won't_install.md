### Post by mickie.kext on 2009-12-13
Are you using NVIDIA card with old **AGP** interface? If yes, that is known bug which was discovered during Karmic's Beta but it did not get fixed.

[https://bugs.launchpad.net/ubuntu/+source/xserver-xorg-video-nv/+bug/385658](https://bugs.launchpad.net/ubuntu/+source/xserver-xorg-video-nv/+bug/385658)

[https://bugs.launchpad.net/ubuntu/+source/xserver-xorg-video-nv/+bug/385703](https://bugs.launchpad.net/ubuntu/+source/xserver-xorg-video-nv/+bug/385703)

It happens only if you have NVIDIA AGP card which uses PCI-e to AGP bridge chip. (ie. GF7 and GF6 series)
