### Post by tommcd on 2011-01-05
> **SpeedXLight said:**
> Hello, I'm not able to do a direct wire connection and I want to setup a web server using Ubuntu server but I don't feel like messing with drivers and config files. ...
The reality here is that if you want to install a command line only server system (no GUI) then you will need to be able to use the terminal to install drivers, install packages, and to configure things.
If the wireless works on the desktop version of Ubuntu, then it should work on the server edition as well. 
You could of course just use the desktop version of Ubutnu on your server if you are afraid of "*messing with drivers and config files*".
