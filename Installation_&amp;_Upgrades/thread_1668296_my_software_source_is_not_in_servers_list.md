### Post by yeats on 2011-01-16
You can go to System -> Administration -> Software Sources and click on the Download From drop down and select Other...  From there click Select Best Server and the system will automatically select the fastest server for you.

For what it's worth, I don't see Iran sources on my list either.  Hope that helps!
