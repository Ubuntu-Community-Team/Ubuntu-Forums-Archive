If you set the Update Manager to alert you of new releases, you'll be notified when 12.10 is available to upgrade.  It won't be long now.  Which version are you using now?  It will upgrade each version until it gets to 12.10.  If you're using 12.04 it is only one step.

### Post by darkod on 2012-10-05
Make sure you have a full backup of all your important data before trying to upgrade wubi.
Also, if you use this as a "main" system, why would you want to upgrade to Beta? If this is some kind of test system on the other hand, you can try but don't complain if it fails. :) It's still in development after all.

I believe there is an option in Update Manager where you can say to inform you of development releases too, if you select that it should prompt you there is an upgrade to 12.10 available.

If there is no such option in Update Manager, there is surely a way, you only have to google a bit more.

PS. In fact, why would you want to upgrade from LTS to standard release? Any important feature that is included in it? If it's only because it's the latest, better don't bother and happily use your LTS for years to come. :)

### Post by Mark Phelps on 2012-10-05
> **DaComboMan said:**
> Is it possible to upgrade to latest beta (12.10)?

WUBI user.

Possible -- but, really a BAD idea!  The beta is work in progress.  Why you would change a stable install for one that is not, is your business.  But you need to understand that if anything goes wrong, you can NOT roll back to your current install; instead, you would have to reinstall from scratch!  One reason why "upgrading" to a beta version is generally a BAD idea.
