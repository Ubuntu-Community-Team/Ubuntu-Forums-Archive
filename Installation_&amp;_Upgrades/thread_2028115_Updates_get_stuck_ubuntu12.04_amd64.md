### Post by drs305 on 2012-07-20
Try disabling the security and partners repositories to see if the rest will install. Since you are already at a terminal, open Software Sources with:
```
gksu software-properties-gtk &
``` 
Untick the security repository in "Updates" and the partners repository in "Other Software" and try the apt-get commands again.

You could also try switching servers in the "download from" on the main tab.
