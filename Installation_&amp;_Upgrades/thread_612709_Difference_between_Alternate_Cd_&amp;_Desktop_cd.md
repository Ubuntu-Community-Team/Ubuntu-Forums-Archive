### Post by Sef on 2007-11-14
> What is the difference b/w tha alternate cd & the desktop cd ???

The alternate cd is just an install cd; the Desktop cd is a live cd.

> Can a desktop cd be converted to alternate cd ???

No, it cannot.

> Or how to upgrade via a desktop cd ???

You cannot upgrade with the Desktop cd.   If you installed the previous version, then Ubuntu should inform you there is a newer version available.  If not, then System > Administration > Update Manger.
