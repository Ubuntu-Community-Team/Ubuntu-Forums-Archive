### Post by oldos2er on 2009-07-27
Beryl is deprecated; compiz-fusion is installed by default, to enable it go to menus System, Preferences, Appearance, Visual Effects, and check 'Normal' or 'Extra'. You might want to install compizconfig-settings-manager to fine-tune the settings. Here's more info: [https://help.ubuntu.com/community/CompositeManager/CompizFusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion)
