### Post by howefield on 2013-02-03
If you enable the Partner repositories in your software sources, you will be able to install skype.

Open the Ubuntu Software Centre and from the Edit menu, select Software Sources, go into the "Other Software" tab, and check the Partner repository. After reloading the sources, you will be able to install skype.

Thunderbird is already included in Ubuntu, would this substitute for outlook express for you ?

IDM, what's that ? A download manager, plenty of addons available for firefox, eg, [https://addons.mozilla.org/en-US/firefox/addon/downthemall/](https://addons.mozilla.org/en-US/firefox/addon/downthemall/)
