### Post by bapoumba on 2007-01-13
Hi,
if you've installed alternate cd, your user login is oem, and the password the one you gave during the install. Is that the problem ?

There is a way to make a new user with a new login and a new password one logged in ubuntu.
Open a terminal (> Accessories > Terminal) and type **sudo oem-config-prepare**
Next boot up and loggin, you will be prompted to create a new user.
