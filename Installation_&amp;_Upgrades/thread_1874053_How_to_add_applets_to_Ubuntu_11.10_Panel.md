I recently upgraded from Ubuntu 11.04 to 11.10, and actually like it! It displays the Unity Desktop. However, that gets a little crowded, even though I can shrink the icon size using Compiz. Holding down Alt while right clicking the Panel does not work in 11.10.

Is there a way to add an applet to the top Panel in Ubuntu 11.10? Thanks.

### Post by Frogs Hair on 2011-11-02
Unity in  11.10 uses a different panel and the add to panel options that were  available in classic Ubuntu on 11.04 are gone . Drag and drop to the panel is no longer supported in Gnome 3 . 

The panel in 11.10 is white listed so , panel icons from an application that supports a panel icon will appearer . Someone may find an unofficial work around for the drag and drop issue , but I am not aware of any at the moment .

### Post by Mahelita on 2011-11-08
I solved this problem by installing gnome ( sudo apt-get install gnome ), rebooting and choosing "GNOME Classic" from the drop down menu in the user login window.
But you will loose the Unity desktop you seem to like (I hate it).

### Post by mtetwa on 2011-11-15
To add an app to the (left) panel: open the app/program (for example via 'dash'): you'll see it in the panel. Then right-click its icon in the panel and choose 'keep in launcher'. Done.
