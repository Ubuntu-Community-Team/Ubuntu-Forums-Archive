### Post by grahammechanical on 2012-09-30
What did you install from? It maybe that you are unfamiliar with how things are done in 12.04 and onwards. You can try 2 things.

1) Press the Super key (windows key) and type help and read the Ubuntu Desktop User Guide to familiarize yourself with 12.04, the Dash and Launcher.

2) Press Super + a and the Dash will open in the Applications lens. Give it time to search for all your applications. Look for the Installed heading. Click on See xx more results and search for the icons of your applications.

I did not find any problems with my Wine and Windows type program appearing in the Dash with I upgraded from 11.10 to 12.04.
