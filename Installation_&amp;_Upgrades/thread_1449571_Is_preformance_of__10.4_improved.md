### Post by Mark Phelps on 2010-04-08
Any performance benchmarks run at this time on 10.04 will be invalid because it's still in beta testing.  It's typical to have lots of testing code remain resident in an OS until the production release is prepared.

So, you would do best checking back after 10.04 is released.  That will contain the "production" code and will yield more accurate benchmark results.
