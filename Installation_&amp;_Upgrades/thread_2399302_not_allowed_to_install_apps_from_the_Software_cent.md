I do not run the desktop version of Ubuntu but just from reading what you typed, I am a bit confused.

You said you upgraded from 16.04.5 to 18.04.1 which I assume was an in-place upgrade on top of your existing OS.

The next line down, you say "clean install of 18.04.1" as if you re-formatted the drive and installed 18.04.1 (and not on top of an existing OS).

Regarding the inability to install apps, is your account able to sudo?  I do not know about the desktop GUI but to install packages on a server, it requires elevated rights via sudo.  It is possible the upgrade wiped out who was able to sudo or the process changed.
