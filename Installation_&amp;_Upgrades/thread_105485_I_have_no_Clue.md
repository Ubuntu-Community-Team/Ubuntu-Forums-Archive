### Post by doitashimashite on 2005-12-18
You downloaded the Ubuntu ISO file? If so, that's a good start.
Then, you burned the CD? How?

(In your CD burning software program, you should find an option that says "burn CD from image" then select this Ubuntu ISO file (which is the image) and burn the CD.)

Make sure your computer's BIOS has the CD drive set as bootable (before harddisk) and boot from CD. The rest is easy.

Good luck!
