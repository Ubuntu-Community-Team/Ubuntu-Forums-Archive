Its difficult to understand what you are saying... but lets say you have XP on the first partition (hda1) and you want to install Ubuntu to the second partition then during the install process you select the second partition (freespace) and then let the installer auto partition that freespace (it will partition that second partition into swap and root).

When it comes to installing GRUB choose the MBR.
