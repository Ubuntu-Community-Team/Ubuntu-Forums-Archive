Go here: [http://www.avast.com/eng/download-avast-for-linux-edition.html](http://www.avast.com/eng/download-avast-for-linux-edition.html) and download the "avast4workstation_1.3.0-2_i386.deb" file.

Once downloaded go to the folder where it was down loaded and right click on the file name. Select to open with the Gdebi Package Installer. It will take care of everything for you and leave the shortcut under Applications/Accessories. If you have the 64 bit Ubuntu installed this will not work since Avast is only available for the 32 bit version.

It is a lot easier than using the file you downloaded.

You likely will not need an AV but, I feel, won't hurt. It will scan a Windows drive or partition if you dual boot. And you can use it to scan attachments you receive through Ubuntu email programs. At least you will not pass along something when you forward anything with attachments.
