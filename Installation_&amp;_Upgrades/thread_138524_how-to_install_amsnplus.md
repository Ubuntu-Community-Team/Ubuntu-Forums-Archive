### Post by florizs on 2006-03-02
there are multiple ways to this I will outline 2 of them.
it comes with ubuntu and can be obtained by this method.

you can use the graphical installer.
open the system menu, go to synaptic.
find the package amsn and click install.

or you can open a terminal and type: > sudo apt-get install amsn

also you can download the installer from the amsn website and do it manually but since you are a newbie as you proclaim. I suggest the above methods.

ps. you must have the universe/multiverse repositories enabled: 
to see how to do that look here
[http://ubuntuguide.org/#repositories](http://ubuntuguide.org/#repositories)

hope this helps
