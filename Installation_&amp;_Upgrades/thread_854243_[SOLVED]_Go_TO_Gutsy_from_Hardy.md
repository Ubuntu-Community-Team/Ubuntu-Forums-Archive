### Post by meindian523 on 2008-07-09
From within Hardy,I very much doubt that.What you could do now,is copy your home folder to some external storage,including all hidden files(see them by pressing Ctrl+H),then reinstall with the Gutsy CD you are downloading.
While installing Gutsy,give the same username and password you used in Hardy,and then replace the home folder which is created by default with the one you backed up.All your personal settings should remain the same after doing this.
