### Post by thed0ctor on 2012-03-29
Hello,

I compiled a program (specifically xournal 0.4.5 with some patches) and I'm trying to set it as the default program to open xoj files. I went to the properties menu for an xoj file and can't seem to find it in the list of applications. I also have to use either the terminal, or browse to the folder to open the application. Not sure how to add this to the list of applications.

I was going to ask this on the [sourceforge page]("http://sourceforge.net/projects/xournal/forums") but I thought it was more of a compiling in general question so I posted it here.

Thanks in advance!

---

### Post by Paddy Landau on 2012-03-30
Right-click the xoj file > Properties > Open With > Add > Use a custom command > Browse > (find your program) > Add > Close
