Just use the *User Agent Switcher* extension. As you can see [here]("http://packages.ubuntu.com/hardy/useragentswitcher") it is in the repos, so you can install it in *Firefox* by selecting *Tools -> Add-ons -> Get Add-ons -> Get Ubuntu Extensions* or from the [extension's page at the *Mozilla* site]("https://addons.mozilla.org/en-US/firefox/addon/59").
