### Post by craig10x on 2015-02-17
You can do that AFTER your installation is finished....when you boot into your new ubuntu, just open software center and install ubuntu restricted extras...by the way, when you do that, often the agreement for microsoft fonts (if you want them...i always get them) may not come up and get installed (everything else will though)...if that happens, you can always install the ms-core fonts separately, using either ubuntu software center or install synaptic package manager and do it with that...It may show it as being installed, but unless the agreement was checked, it isn't, so just uninstall and re-install that...

When i install ubuntu, i always install that stuff afterwards, rather then check the boxes...The latest updates will come up after you boot into ubuntu and you can install them at that time, also...
So, no need to have either boxes checked during install...
