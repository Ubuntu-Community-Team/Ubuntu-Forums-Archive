### Post by kaldor on 2010-09-16
Be sure to get a basic grasp on the terminal. It's not truly needed, but it will make life easier once you get the hang of it. I hated using the terminal at first, but now I tend to use it even with basic computer usage to copy/move/delete files instead of manually clicking and dragging/etc all the time.

It's pretty easy to learn :)

Depends on what you want to do, though. Maybe CentOS would be a better choice for you. The install media lets you choose between a server and a desktop on installation and it's pretty well the standard when it comes to Linux servers.
