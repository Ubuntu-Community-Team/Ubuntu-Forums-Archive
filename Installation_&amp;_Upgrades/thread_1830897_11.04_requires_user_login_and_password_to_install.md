### Post by coffeecat on 2011-08-22
The username in the live session is "ubuntu" and there is no password - that is, just press enter.

However, if you are being prompted to login when you first boot up the CD then there is something wrong with the CD. Since you have checked the md5sum of the downloaded ISO you need to see if you have a bad burn. 

When the CD first starts booting, you will see a purple screen and two small icons of a stick figure and keyboard at the base of the screen. Tap the spacebar. At the text menu that appears you can do a disc integrity check. If it passes, there is still something wrong, perhaps dust on the CD, perhaps a dirty CDROM lens, perhaps a failing optical drive. Whatever, do not try to install from a live CD that is prompting you to login. You will most likely get a corrupted install.

The only time a live session should prompt you for username and password is if you log out and in again. Since there is no logout/in option in the GUI in 11.04, the only way of doing this is with a magic key combination, so unless you know this key combination you will not see the login screen in a live CD which is not faulty.
