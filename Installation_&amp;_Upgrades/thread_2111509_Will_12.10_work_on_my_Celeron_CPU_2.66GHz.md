### Post by presence1960 on 2013-02-02
You can run 12.10 on your machine. If Unity is too much for your RAM you can install LXDE desktop from Software Center. Once installed you can reboot. When you get to the Ubuntu log in screen click the ubuntu logo just top right of your password box. Choose LXDE to use as desktop. Log in and you will be using the LXDE Desktop which is a nice lightweight desktop environment. LXDE stands for **L**ightweight **X**11 **D**esktop **E**nvironment.
