### Post by nikopol on 2005-02-24
That's strange - I didn't think you would get offered different versions on install - where did you get your CD from?

As things stand the current status is as follows:
warty - current stable release (released in October 2004 IIRC)
hoary - future stable release (2 months away) - works ok but bugs being thrashed out on it as we speak. Only use it if you're ready to file bug reports and put up with some unstability....
grumpy, perky - futures releases that are not yet available to installl (I think?!)

All will have GUI by default - but you can set it to not start if you want to use Ubuntu for something else. What do you have in mind? A server? A "dumb" terminal?
