this may be a bad burn if it is using this much time...try too use ISOBURN instead. download it here: [http://www.free-codecs.com/ISOBurn_download.htm](http://www.free-codecs.com/ISOBurn_download.htm)

you may also have low system specs:
-You need at least 384MB of RAM to use the Ubuntu live CD
-You need at least 128MB RAM to run the Xubuntu live CD, and 192MB RAM to install Xubuntu form the live CD

If you wonder what Xubuntu is it is only Ubuntu using Xfce instead of GNOME.
Xfce is a lightweight windows manager and works great on older computers. Xubuntu looks, and runs about the same as Ubuntu the biggest difference is the menu, and you can't customize it as much as Ubuntu or Kubuntu
