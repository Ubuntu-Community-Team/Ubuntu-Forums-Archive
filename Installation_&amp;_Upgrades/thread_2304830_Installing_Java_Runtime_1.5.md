### Post by slickymaster on 2015-11-30
There are applications that required a specific Java Runtime Environment version. You can get the one you need from the Oracle Java Archive: [http://www.oracle.com/technetwork/java/javasebusiness/downloads/java-archive-downloads-javase5-419410.html](http://www.oracle.com/technetwork/java/javasebusiness/downloads/java-archive-downloads-javase5-419410.html)

After downloading and installing it, and since you'll end up with multiple versions of Java installed, all you have to do is to choose which one you want to use by running the following command```
sudo update-alternatives --config java
```
