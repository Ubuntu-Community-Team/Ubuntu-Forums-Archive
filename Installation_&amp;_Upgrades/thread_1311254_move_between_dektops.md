### Post by paulisdead on 2009-11-02
Ya this got unset for me too.  Go to System>>Preferences>>CompizConfig Settings Manager
Click On Desktop>> Viewport Switcher
Go to the last tab, Desktop Based Viewport Switching, and set the mousewheel buttons you want to use.  I used button 4 for left, and 5 for right.
