### Post by sikander3786 on 2010-12-08
Wubi is a simple method of installing Ubuntu or some other Linux distro inside Windows just as you'd have installed a program. It creates a virtual HDD which is shown as a folder on one your Windows partition. It obviously takes up space from the Windows partition and that space can vary between 3GB to 30GB. I would suggest to assign at least 10GB space though.

32-bit and 64-bit are computer architectures. There are separate ISOs for both of them and you don't need to/can't assign a parameter for that.

Some informative links.

[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

[http://en.wikipedia.org/wiki/32-bit](http://en.wikipedia.org/wiki/32-bit)

[http://en.wikipedia.org/wiki/64-bit](http://en.wikipedia.org/wiki/64-bit)

And remember, wubi was just intended for testing. It is a bit slower and a bit less stable. If you intend to use Ubuntu as a production OS, you better go for a proper dual boot. We'll definitely try to help you set it up if you decide to go this way.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

If you still want to install Ubuntu using the Wubi installer, I'd suggest to apply *Permanent Fix* as suggested here. It would definitely save you lots of troubles later.
