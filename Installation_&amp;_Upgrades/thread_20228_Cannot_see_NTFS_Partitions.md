### Post by mwalt2 on 2005-03-15
I'm a noob too, but you have to mount the Windows partitions before you can see them.  NTFS is read-only.  If you want to change files on a windows partition, you'll need it formated in FAT32.  Look here on how to mount partitions/drives:  [http://ubuntuguide.org/#mountunmountntfs](http://ubuntuguide.org/#mountunmountntfs)
