### Post by dandnsmith on 2009-07-09
I'd try booting from a liveCD in an external CD device, and then copying the data from one to the other using the dd command.
You'd need to be careful about the parameters for this command, or face a bit of sorting out.

Then you have to sort out the booting, as you want to use the files in the other partition - OK if you know what to do, but you may need to do some reading.
