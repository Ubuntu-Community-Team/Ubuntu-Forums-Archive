### Post by mörgæs on 2010-11-21
The best is to create the USB stick with the latest release of Unetbootin (not the one found in 10.04) and use it for installation. 

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

While installing, remove all unneeded USB connections but keep the cable for web access.
