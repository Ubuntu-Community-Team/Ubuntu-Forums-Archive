### Post by Therion on 2009-04-22
You don't really "uninstall" an OS.  What you could do would be to boot from the install CD and then simply choose to install again.  
When you get to the partitioning section of the install choose the option "**Guided - use entire disk**".  Accept the defaults from there and you'll have a fresh, clean install.
