### Post by stephenmac7 on 2010-02-27
To remove wine first delete /home/<username>/.wine

Then remove the contents of: /home/<username>/.local/share/applications/wine/Programs

After that use the Synaptic Package Manager and search for "wine" with no quotes. Remove all packages that are installed in the list.

Hope you have success removing wine:p
