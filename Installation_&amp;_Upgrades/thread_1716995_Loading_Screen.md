Check on things like "usplash builder" and "usplash" and things like that.  It's fairly easy once you have a nice set of images in 1024x768 plus a few other resolutions (which you can magnify or reduce with GIMP), and then run the make command for a custom splash screen.  Around 15 minutes work if you already have the half-a-dozen images required and know what you're doing.
