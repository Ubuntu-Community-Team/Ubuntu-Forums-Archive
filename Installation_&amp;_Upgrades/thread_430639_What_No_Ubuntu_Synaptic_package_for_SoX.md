### Post by xp_newbie on 2007-05-02
OK - found the answer myself.

It turns out that I needed to check the "**Community maintained (Universe)**" checkbox in the **Ubuntu 6.06 LTS (Binary)** channel. This is in the Repositories dialog box, of course.
