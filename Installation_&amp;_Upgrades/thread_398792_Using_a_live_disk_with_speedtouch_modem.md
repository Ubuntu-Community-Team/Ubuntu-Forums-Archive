### Post by louieb on 2007-04-01
I believe the kUbuntu Live CD is more of a demo CD. It doesn't have the slick stuff to save session settings. If you like the KDE desktop try the Knoppix CD, a couple of others are Puppy and Slax. All three are made to be used as a live CD and will save your settings to a USB stick.
