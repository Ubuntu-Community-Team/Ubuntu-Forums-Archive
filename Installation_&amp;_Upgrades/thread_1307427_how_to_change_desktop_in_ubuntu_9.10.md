### Post by GamesMasta on 2009-11-02
This problem exists in the old version and the new one. I'm stuck in the other desktop theme and have no panels to minimize/maximize etc. I used to previously solve the problem by running /usr/bin/gnome-panel.

Try opening up terminal and type in: /usr/bin/gnome-wm. If that fixs the problem, then create a shortcut on your desktop to that place and run it everytime you need to reboot.
