### Post by megahertz on 2007-10-21
I wasn't able to boot with 7.10 live CD. I execute wubi-cdboot.exe under Win XP and a folder was created on my C: drive. Then when booting an option window appear asking from witch OS I would like to boot. Windows or Ubuntu. I choose Ubuntu and it boot from 7.10 live CD with no problem.
On next boot I choose Windows and when it finished booting a window asked me if I would like to uninstall wubi-cdboot. I answered yes and all folder created on my drive was deleted and the windows master boot record was restored.
