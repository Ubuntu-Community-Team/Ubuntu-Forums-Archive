### Post by eldon.t on 2008-06-20
okay i finaly found the nasty bug

it was the activation of USB keyboad support in the BIOS that produces the kernel panic on that kernel version.

I'm using a Gigabyte GA-P31-DS3L Motherboard (BIOS Revision F8)

For some reasons turning the CPU Multithreading off would fix the kernel panic and let the BIOS USB Keyboard support work fine.

Of course without keyboard support i am unable to control grub boot menu.

But well besides that everything seems to work fine..
