### Post by Sumstah on 2011-05-28
On the purple screen, press enter or any key.
Then it may display a language screen, choose your language.
This should display the menu to try install etc, you can try installing now but you may still get like a black screen or a error message.
If you do, then at the menu press F6, then go to 'nomodeset' and press enter, this should put a cross next to nomodeset, then press esc. Then you could either try or install it as you wish.

The above worked for me.

Hope it works for you.
