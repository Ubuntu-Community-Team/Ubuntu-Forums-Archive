I currently have Ubuntu and Windows dual boot and it is using Grub bootloader which was installed with Ubuntu. If I were to install Xubuntu over the Ubuntu partition (I know it is possible to change the interface on Ubuntu to Xfce, but I'd rather do a fresh install) would that overwrite/mess up my bootloader? ### Post by ajgreeny on 2012-06-09
The xubuntu OS will overwrite the grub bootloader as well as the OS itself, so you should end up exactly where you are now with the same situation; just a new grub menu, but still with Xubuntu and Windows on it, as before.
