### Post by mivo on 2007-10-18
It does look like flame bait, and the topic title is a little unfortunate (and unfair). I said this last night already: Ubuntu is a work-in-progress and there is no need up upgrade to a new release right when it is released, if your current system works well. I have never picked up a new Windows version when it came out (usually waited a year), and I see no reason to update to Gutsy right on the first day. It's not a race. Everyone should also spend some quality time with the Live CD before upgrading their system.

Personally, I'll wait a couple weeks, maybe a month, and then see about updating this machine here. I expect a virgin machine in two or three weeks, and I'll try to set it up with 7.10 if the Live CD looks promising. But my running and working Feisty system moves to 7.10 only when I'm sufficiently sure that I won't run into trouble.

Like I said, there is no rush. Gutsy is not an iPhone or a LotR movie.
