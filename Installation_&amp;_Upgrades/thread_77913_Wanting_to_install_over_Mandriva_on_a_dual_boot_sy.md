### Post by pablo13 on 2005-10-17
Hi! Here you have my opinion about both questions:

1. Ubuntu installer will ask you about installing the grub boot loader. It wil also ask you about other operating systems (like WinXP) and should configure it automatically

2. I recommend you to install in a clean partition ... You can tell the installer to format the partition but you will loose the data you had, so first you'll need to do a backup of the data you want to save
