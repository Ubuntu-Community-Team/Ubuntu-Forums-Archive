### Post by taurus on 2006-07-16
Three things.  Before you burn it to a media, run the md5sum to check the integrity of the ISO file.  After Ubuntu boots from the CD, run a test on the media before you actually install it.  Last, make sure you burn it at a slower speed like 4x!!!
