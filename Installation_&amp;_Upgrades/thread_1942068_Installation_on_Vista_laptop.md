### Post by Mark Phelps on 2012-03-17
> **livewire94 said:**
> I am finding that Ubuntu uses a lot more cpu than Vista.  ...

My laptop locked up several times in Ubuntu and never in Vista.

Maybe the new Ubuntu will work better.

As to the first, there is a known kernel bug with recent and current Ubuntu versions that prevents the CPU from cutting back on power demands when the PC is idle.  This tends to make the PC run hot, and on laptops, uses up the battery faster.  So, it's not really using MORE CPU; instead, it's not letting the CPU throttle-back.  But, the user experience is basically the same.

As to the second, unless you're running a Sandy Bridge processor, and you're probably not, the new Ubuntu will NOT work better as the new kernel is only fixed for those processors.
