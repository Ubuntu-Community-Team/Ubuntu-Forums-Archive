It is situations like this that Wubi (the official Ubuntu-within-Windows installer) was *designed* for.

For the uninitiated, *yes*, it works with 64-bit Windows *and* 64-bit versions of *buntu (I have a triple going with 64-bit Vista Ultimate, 64-bit Kubuntu 8.10 (which I will upgrade to 9.04 in a week or less) and 64-bit Windows 7 7077 (I installed Kubuntu from within Vista, and all three appear on the boot menu), and it's no harder than installing a Windows application (in fact, it's easier than installing some applications).
