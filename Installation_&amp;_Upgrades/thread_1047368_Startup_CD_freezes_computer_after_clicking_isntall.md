You should try the alternate cd iso, as the computer only has 256 mb RAM. From the official instructions, to run the Desktop CD, a minimum of 384 mb RAM is needed. To download the alternate iso, go to the same place you downloaded the one you have, and scroll down the page; IIRC, there is a box on the right side of the page where you may select the alternate iso.
