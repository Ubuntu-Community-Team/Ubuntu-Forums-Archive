### Post by johnn1949 on 2007-11-06
Try reading this first.  
[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

It depends on how you want to install.  If you have windows and you want to add ubuntu as a dual boot just go back one page from where you are and use the guided (automatically make space on your main hard drive).  When you choose the size of the new partition the size you choose will be the new size of the of the windows partition.  just subtract the the amount you want for ubuntu (10 GBt is enough).  if it shows 50GBts in windows partition make new size 40GBts.  Then installer will put ubuntu into the 10 GBts of free space you created and automatically make a swap partition for you.

If you want to use the whole disk and have ubuntu only(which I doubt) choose use whole disk.

good luck and keep using these forums, that's how I got it working.
