### Post by dannyboy79 on 2008-03-21
when you put the cd in your cd-rom, does it show up on your desktop? If so, you should be able to browse it using Nautilus. Then you'll find out how the program is packaged, whether it's a .deb or if it needs to be compiled from source or what. You could copy it off the cd by dragging and dropping if need be. Good luck
