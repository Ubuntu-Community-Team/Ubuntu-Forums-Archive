### Post by darkod on 2012-07-22
The option to upgrade directly from the last LTS (10.04) will not show up until 12.04.1 is released in August. If you don't want to wait, you can do it now by starting update manager with:
sudo update-manager -d

It might be better to wait one month more because some issues noticed in 12.04 might be sorted in 12.04.1.

In any case, it's good practice to try a live cd of 12.04 first, and to have a backup of all your important data before you start the upgrade.
