### Post by ajgreeny on 2008-02-08
Go to System>Preferences>Appearance in the menu and then on the Visual Effects tab set to None.
Alternatively get a package called compiz-switch from [http://forlong.blogage.de/article/pages/Compiz-Switch](http://forlong.blogage.de/article/pages/Compiz-Switch) which works brilliantly.  You will have to download the .deb package and then double click on it to install, but it works simply; one click on the item in the menu turns compiz on, the next click off, on, off etc etc.  It is very useful if compiz upsets play of videos in the firefox plugins, which does happen sometimes.
