so far i have the scanner working as a simple usb scanner, i have not scripted any of the buttons or anything that complex yet.  when i do ill repost to this.

so far what works WELL for me is 

alt + f2

run
xscanimage

then save as a pnm file and open the file in gimp and convert the pnm to jpeg.

works well i can zoom in to extreme detail easily.

load preview window. then acquire preview. then hit cancel preview as soon as your image is loaded and ready to crop

then crop the image to the borders. adjust scan resolution, and file size and name the file and hit scan.
