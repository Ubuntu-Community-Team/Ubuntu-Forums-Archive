### Post by Delta_Farce on 2006-12-05
The alternate install CD uses a different guided install program. Instead of a pretty X desktop, you get terminal type windows and menus.

It's very similar to the 6.06 install CD, or the FreeBSD isntaller. It's intuitive and works well, and you'll end up with the same OS.

I used it on my system when Edgy showed me a black screen, so I'd recommend giving it a try.
