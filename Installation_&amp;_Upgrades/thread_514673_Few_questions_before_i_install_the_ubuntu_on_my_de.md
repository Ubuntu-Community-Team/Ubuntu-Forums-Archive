will i be able to connect my external hardrives ? Yes.

> webcam ? is there a msn video for cam ? Webcam support could be if-fy. Read more here:
[https://wiki.ubuntu.com/HardwareSupportComponentsMultimediaWebCameras](https://wiki.ubuntu.com/HardwareSupportComponentsMultimediaWebCameras)

> will i be able to use skype on it ? Yes, but you may not get all the features available for Windows' Skype. Don't like it, complain to Skype.

> will i be able to use the firefox? Yes, in fact, Firefox is the default web browser in Ubuntu.

> I just got a new creative sound card will i be able to use it with ubuntu? I don't know. Check this out:
[https://wiki.ubuntu.com/HardwareSupportComponentsSoundCardsCreativeLabs](https://wiki.ubuntu.com/HardwareSupportComponentsSoundCardsCreativeLabs)

The best way to check hardware compatibility is to boot the Desktop CD. It is a "live CD" that allows you to run a fully functional Ubuntu off the CD itself and your computer's RAM. It does not affect your hard drive.

### Post by aysiu on 2007-08-02
> **cooldudeny said:**
> I am having touble installing unbuntu on my pc everything goes fine till i want ubuntu to format the drive and install ubuntu 

it is getting stuck at 59% can someone please help  me ?

i tried 2 times and still the same thing need some advice
Well, it could be one or more of the following:
1. You have a corrupted CD (corrupted during download and/or during burn)
2. You don't have enough RAM to install from the live CD
3. You're having some kind of hardware failure

To make sure it's not #1, read this:
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

If you have 256 MB of RAM or less, it may be #2.

And if it's not #1 or #2, it's probably #3.
