### Post by LegendofTom on 2010-07-28
For simplicity, you should install it under whichever partition (Xp or 7) houses the boot manager you use.  That way you will see all three options when you boot.  Besides that, Wubi installs inside your Windows partition as to not create partitioning problems, so you shouldn't hesitate to try it.
