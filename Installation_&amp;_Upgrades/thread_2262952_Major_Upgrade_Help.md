### Post by grahammechanical on 2015-01-28
> [COLOR=#000000] I am guessing this is because I am too far behind in updates.[/COLOR]

More likely because the version of Ubuntu that is installed has reached End of Life and is no longer supported. When a version of Ubuntu has reached end of life the software repositories are closed and moved to an archive. We can no longer update that version of Ubuntu and it is difficult to upgrade to a newer version.

You do not say what version of Ubuntu is on that machine but the versions of Ubuntu that are presently supported are 12.04 LTS, 14.04 LTS and 14.10. I recommend doing a fresh install of either 12.04 LTS or 14.04 LTS but not 14.10 because non-LTS Ubuntu releases only have 9 months support. Try for Ubuntu 14.04 has that has 5 years support from April 2014.
