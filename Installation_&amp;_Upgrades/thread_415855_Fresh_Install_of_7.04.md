### Post by tommcd on 2007-04-20
If you have multiple partitions (e.g., a separate /home), just install fiesty to the /root partition where edgy now lives. Don't format the partitions you want to keep. 
I have a windows partition, a root partition for ubuntu, a swap partition, a root partiton for zenwalk, and a /data partiton for my stuff. I just installed fiesty to the root partition where edgy was, and all other partitions remain as they were.
