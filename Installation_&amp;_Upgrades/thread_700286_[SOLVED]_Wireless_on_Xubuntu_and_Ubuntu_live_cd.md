### Post by miriad on 2008-02-18
On live discs, wireless only works with native, non-proprietary drivers, which untill Hardy, only exist for the Intel 2100 and 2200 cards as far as I know. For wireless once you get a boot loader installed/working, You'll probably need to install a proprietary driver (Intel cards), or ndiswrapper and a windows driver.
