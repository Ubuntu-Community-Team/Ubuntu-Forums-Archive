### Post by jaygo on 2008-08-18
clearly, replacing the optical drive would be a good idea but let's assume you can't do that for now. The xubuntu installer ISO is under 600mb so it can fit on a thumb drive, though it will be **slow**. If your laptop can boot from a USB drive, then it will work, AFAIK.

You'll need to make the thumb drive bootable first, then copy the ISO over to it. I don't have any experience doing that so just search for "bootable usb drive" and you should figure it out quickly.

GLHF
