### Post by aysiu on 2008-04-04
It really shouldn't ask for the CD. If it does, you can disable that software source by going to System > Administration > Software Sources and unchecking the box next to the CD-ROM source. Then click **Reload** when prompted to.

After that, follow these instructions:
[http://www.psychocats.net/ubuntu/java](http://www.psychocats.net/ubuntu/java)
