### Post by Elfy on 2011-09-14
To upgrade from 10.04 to 11.04 you would first need to upgrade to 10.10. 10.04 is an LTS release and you CAN upgrade them to the next LTS without the version in between - the next LTS is 12.04.

To upgrade 10.04 to 10.10 - open upgrade manager, open settings at the bottom - give password as required.  Change releaase upgrade to Normal Release - reload and close that window. 

10.10 should be available as an upgrade.
