### Post by Topsiho on 2009-08-31
A (Ubuntu) live CD is just a regular Ubuntu CD, which you can download here:

[http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download)

Be sure you don't download the alternate version, as that one is not a live CD. Be also sure that you download the right version for your computer, usually the .386 one for a PC, maybe the 64 bits version.

With a LiveCD you can try using Ubuntu, and you can use it for partitioning the hard disk(s), and for fixing troubles as root.
And of course you can use it for installing Ubuntu ...

Hope this helps.

Topsiho
