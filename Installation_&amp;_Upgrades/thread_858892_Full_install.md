### Post by perlluver on 2008-07-14
You will have a /home, it just won't be seperate from root.  So it will all be together.

Edit: The only disadvantage is, if you ever re format the drive you will lose home.  Unless you back it up to another drive.  With a seperate /home, you would be able to not format home and keep it intact.
