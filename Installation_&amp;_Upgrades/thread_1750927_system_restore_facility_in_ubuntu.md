### Post by Rubi1200 on 2011-05-06
Hi karthick,
the answer to your question is basically no, there is no Windows-like system restore function.

But, what you should do is always keep a backup of important data.

You might want to also consider using Clonezilla as a backup and restore option:
[http://clonezilla.org/](http://clonezilla.org/)

Another option is to keep a record of your installed software which you can then use if you need to reinstall:
