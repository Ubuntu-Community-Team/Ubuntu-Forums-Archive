### Post by howefield on 2016-08-31
Midori is already in the Ubuntu repositories so that would be the version that you have installed. The output of

```
apt-cache policy midori
```

would likely confirm it.

Given that it looks like it is the current version, I'd suggest disabling the ppa that you added.

```
sudo rm /etc/apt/sources.list.d/midori-ubuntu-ppa-xenial.list
```

or commenting it out..

```
sudo nano /etc/apt/sources.list.d/midori-ubuntu-ppa-xenial.list
```

and place a # sign at the start of any lines that don't already have it. Save out and close with Ctrl + O. enter and Ctrl + X.
