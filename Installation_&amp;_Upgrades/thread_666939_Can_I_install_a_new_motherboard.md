### Post by rechick on 2008-01-14
On more than one occasion, for testing, I've taken a Ubunutu installed hard disk and installed into a different computer. Each time, so far, when I boot the drive in the new computer it functions just fine. As mentioned, if you have disparate video cards that may prove a stick wicket  but other than that Ubuntu is excellent in portability  between mainboards.
