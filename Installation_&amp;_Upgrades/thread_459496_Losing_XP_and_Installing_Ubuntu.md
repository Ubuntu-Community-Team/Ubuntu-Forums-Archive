### Post by merlinus on 2007-05-30
When the live cd (and ubuntu) is up-and-running, look for an icon on the desktop named Install.  Click on it and it will start the installation process.

After answering a few questions about language and keyboard, you will get to a partitioning screen.  Select Use Entire Disk, or something like that.  I think it is the first option.

### Post by ep2011 on 2007-05-30
This is very simple. The Ubuntu 7.04 CD is a LiveCD. When booted, it does nothing to your hard drive, unless specified to install. This is great for users who want to use public computers, but have their favorite operating system - Or if they want to try it out before hand. What you do is when you boot into the live Cd, just double click Install on the desktop, and it is very easy from there on. When you go to the partioning section (Formating XP) Just choose to Erase the disk and reformat with only ubuntu. If you need help at that step just post. It is very easy.

For a little more information on the installation of Ubuntu, you can go here: [http://psychocats.net/ubuntu/installing](http://psychocats.net/ubuntu/installing)
