### Post by Patb on 2008-05-04
Creekchub, I had similar problems some time back and can only partly answer your question.  

I do not let the installer automatically select my keyboard by reading selected keystrokes.  Instead, I select it manually from the lists given during the install process.  A couple of screens later there is an opportunity to specify the domain name or something (not sure of my terminology here).  At that point, check that your keys work the way they should, especially the non-alphanumeric ones.  If they are incorrect, go back and try another keyboard layout.  This saves going through a full install before finding out your system keyboard layout is not right.  

Hope this helps. Cheers, Pat.
