### Post by limey_rick on 2010-04-26
See this post: [http://ubuntuforums.org/showthread.php?t=1460847&highlight=unrecoverable+error+10.04](http://ubuntuforums.org/showthread.php?t=1460847&highlight=unrecoverable+error+10.04)

I had the same problem and used the Alternative install as suggested and this solved the problem. 

One note: on the alternative CD install, GRUB2 did not install in the MBR of the disk I had set as first in my PC BIOS. So to get GRUB2 to boot correctly, I had to change the order of boot disks to reflect which MBR GRUB2 installed on. 

Normally, if this happens, you can run a Live CD version and re-install grub where you want it. However since you can't run 10.04 RC1 LiveCD, you have to use 9.10 or something else and this version of GRUB2 did not work for my setup. Just something to watch out for.
