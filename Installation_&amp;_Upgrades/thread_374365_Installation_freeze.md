pedrozavala,
I had the same problem when I was installing on my AMD 64bit 3200+.
I ended up installing the 32bit alternative CD text version of Ubuntu and everything worked fine.  
Not sure why but the 32 bit version worked when the 64bit didn't but another user did the same thing and he said the 32bit version of Ubuntu works faster on his 64bit AMD than the 64bit version.
So try installing the 32bit Alternative CD version through the Text menu and let me know how it turns out or if you run into more problems.
