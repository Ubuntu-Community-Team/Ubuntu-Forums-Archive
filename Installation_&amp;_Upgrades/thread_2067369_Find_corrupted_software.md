### Post by deadflowr on 2012-10-06
Install synaptic package manager.

Select custom filters.
Select Broken
Mark the broken packages, and then run fix broken packages.

If packages can't, or won't get fixed, in the terminal type ubuntu-bug packagename to report it as a bug.
