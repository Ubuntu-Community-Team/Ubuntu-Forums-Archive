### Post by aysiu on 2006-06-16
If an upgrade works, it works better, of course, because you keep all your installed programs and don't have to redo anything.

If a clean install works, at least you know it's clean.

A clean install is more likely to work, but it could create more work for you in the end than a working upgrade would.
