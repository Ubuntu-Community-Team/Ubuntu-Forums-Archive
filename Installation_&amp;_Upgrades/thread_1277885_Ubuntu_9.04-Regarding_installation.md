to burn the iso as an image on CD see [here]("https://help.ubuntu.com/9.04/switching/installing-burning.html").

Prior to burning you want to [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM") the downloaded iso. The hashes must match exactly, if even one character is off the iso is corrupted. If you have a corrupted iso try using a torent to get the iso.
