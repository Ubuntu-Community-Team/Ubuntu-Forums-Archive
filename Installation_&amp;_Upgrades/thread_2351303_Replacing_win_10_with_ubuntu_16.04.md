### Post by yancek on 2017-02-01
> Can ubuntu 16.04 erase win 10 during installation or must the disc be wiped first? 		

Yes.  There is an option to "Erase Disk and Install Ubuntu" which will do exactly that.  If you have any personal data on the computer drive you are installing Ubuntu to, that will be erased also.  If you have windows 10 installed UEFI (very likely) best do some reading such as the Ubuntu documentation link below.  UEFI installs are different than older MBR installs.  You might post the manufacturer of the computer as some make it difficult to install anything but windows.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
