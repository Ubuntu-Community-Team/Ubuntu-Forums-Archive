### Post by oldfred on 2012-10-01
It looks like you installed grub2's boot loader to the MBR of the flash drive and to the sda1 partition. It needs to be in the MBR of the sda drive not the partition.

You may be able to boot if you boot from flash drive. Have you tried that? It looks like the grub in the MBR of the flash refers to the sda1 partition.

If you can boot, this will reset grub. You need to select sda.

#to get grub2 to remember where to reinstall on updates:
sudo dpkg-reconfigure grub-pc
#Enter thru first pages,spacebar to choose/unchoose drive, enter to accept, do not choose partitions
