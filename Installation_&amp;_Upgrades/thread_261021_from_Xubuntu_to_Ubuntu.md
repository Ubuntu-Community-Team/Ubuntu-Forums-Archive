### Post by ClarkePeters on 2006-09-19
I see a lot of posts about installing Ubuntu first and then installing Xubuntu later, but just for anyone who needs to know or is curious, if for whatever reason you start with Xubuntu and want to go give Ubuntu a try then you can certainly install Ubuntu from within Xubuntu either by apt-get install ubuntu-desktop, or better yet start the synaptic manager and choose ubuntu-desktop. (assuming your not using Xubuntu initially because you have extremely limited resources, I'm not sure how Ubuntu would run on old or very small systems)

gotta' love ubuntu, it goes both ways!  :D
