Yes you will have final version if you updated.

System->administration->system monitor->should say 9.10 (Karmic).

You say you upgraded to beta first. Was that from 9.04 release?
You could be missing some new features if you did. Most likely ext4, and possibly GRUB2 (1.97beta4). Although most other stuff will be the same.

Just make sure to download and burn 9.10 final cd in case anything happens and you need to install again (or install to another computer).
