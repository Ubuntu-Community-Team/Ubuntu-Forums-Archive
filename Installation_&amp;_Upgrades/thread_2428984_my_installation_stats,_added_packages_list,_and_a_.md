first, i did the full fresh formatted install from the ISO image on a USB memory stick i had made back in April.  there were **1579** packages installed.

then i logged in, opened a terminal, switched to root, and did "[COLOR=#0000cd][FONT=courier new]**apt-get -y dist-upgrade**[/FONT][/COLOR]".  afterwards there were **1586** packages, so it looks like **7** packages were added due to upgraded dependencies.

then i added my own add-on packages.  the list i used probably had many package names that are already installed because i was in a hurry and just went through a big list and selected what i wanted to be sure i had.  there were **1903** packages so **317** new ones were installed.
