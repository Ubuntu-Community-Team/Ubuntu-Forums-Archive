### Post by snowpine on 2010-10-28
You can install using the [Minimal CD]("https://help.ubuntu.com/community/Installation/MinimalCD").

[Here]("http://www.psychocats.net/ubuntu/minimal") is a good tutorial, with screen shots!

Keep in mind that [Ubuntu System Requirements]("https://help.ubuntu.com/community/Installation/SystemRequirements") are still in effect regardless of how you install. If the system is very old, you might have better luck with Puppy, SliTaz, AntiX, etc.
