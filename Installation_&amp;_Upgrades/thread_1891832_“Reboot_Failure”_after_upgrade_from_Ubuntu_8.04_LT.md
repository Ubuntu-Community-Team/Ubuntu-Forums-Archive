### Post by ThePooh on 2011-12-06
[SIZE=2]I upgraded from Ubuntu 8.04 LTS to 10.04 LTS on my 6 year old desktop PC. Following the upgrade the PC would not reboot: it displayed &#8220;Starting Up&#8221; twice and then the screen went blank. Attempting, via GRUB, to boot the older of the two available versions of Linux (2.6.24-29) resulted in a host of error messages, the foremost being the hard drive could not be found.[/SIZE]
  

  [SIZE=2]After **many** days experimenting with GRUB parameters (acpi=off, etc) with no success, I changed the monitor configuration from VGA to DVI. Presto, there was the Ubuntu login screen. It was there all along, I just couldn't see it!  I'm guessing Ubuntu 8 supports VGA and Ubuntu 10 does not. Nevertheless, I love Ubuntu 10![/SIZE]
  

  [SIZE=2]I'm posting this in case someone else has the same issue.[/SIZE]
