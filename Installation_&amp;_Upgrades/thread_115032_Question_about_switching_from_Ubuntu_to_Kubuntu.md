So I'm currently running Ubuntu on my desktop, and am considering a switch to Kubuntu. What I'm wondering is, is a full reinstall recommended, or is there some simpler or better method for switching?

Also, I keep my /home in its own partition, so I can pretty easily save everything in ~ (it's a single user machine), but I welcome any advice about which dotfiles I should keep and which I should toss.

### Post by timfrost on 2006-01-09
If you run 
```

 sudo apt-get install kubuntu-desktop

```
that will add the kubuntu packages.  You will then be able to log in to either gnome or KDE

No need to reinstall.
