### Post by Rex Bouwense on 2013-03-02
The automatic check is only done once a day (or what ever you set it for).  It is entirely possible that there are no updates to install when it does it's check.  When you manually check the automatic check may have done many hours before and bingo there are now updates to install.  You can leave the automatic update feature do it's job.  Try it and see.
