### Post by stchman on 2007-07-09
> **lukanikos said:**
> Please can somebody tell me how to install rarlinux-3.7.0.tar.gz?

If you just want to open a .rar archive it is far easier to install the unrar package.

sudo apt-get -y install unrar

This will allow File Roller or other compression programs to decompress a .rar archive.
