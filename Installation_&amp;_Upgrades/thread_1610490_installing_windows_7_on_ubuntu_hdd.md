### Post by zvacet on 2010-11-01
Boot your Ubuntu live Cd and under system find** gparted**.With it shrink ubuntu partition and on that unallocated place make NTFS partition.Put Windows 7 install CD and install it on that NTFS partition.You will have to reinstall grub after that.Follow [this]("https://help.ubuntu.com/community/Grub2#Reinstalling from LiveCD") guide.
