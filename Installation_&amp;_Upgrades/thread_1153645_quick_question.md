### Post by lovinglinux on 2009-05-09
The latest Spore version is considered [Platinum on Wine database](http://appdb.winehq.org/objectManager.php?sClass=version&iId=13839), which means it should work without problems.

Installing in Wine won't affect the install DVD. Nevertheless, Spore has a limited number of allowed installs. I don't know how this things work, but the DRM protection included in Spore should consider a Wine installation as a regular one. So use it wisely.

You can read more about Spore DRM controversy [here](http://en.wikipedia.org/wiki/Spore_(game)#SecuROM_controversy).
