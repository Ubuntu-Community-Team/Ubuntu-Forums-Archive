Yes. I used to do that before I ditched Windows.

I would suggest you install Windows on a different hard-drive as linuxes.

But if thats not possible, make one 100MB partition in the beginning of the for boot (call it boot). Then make one at least 10 GB partition for Windows. Then make two partitions for the linuxes.

You will need to install windows first, as noted above. It will need to be located in the beginning of the hard drive, too. After you have installed windows, install your linuxes.

You should be able to find hundreds of guides online and on these forums. * Do your research beforehand. * Figure out what your hardware is, where your hard-drive is and how to partition it.

There is another alternative, too - you could run windows as an virtual operating system from linuxes with Qemu, VirtualBox or VMWare, but this is a degree more difficult.
