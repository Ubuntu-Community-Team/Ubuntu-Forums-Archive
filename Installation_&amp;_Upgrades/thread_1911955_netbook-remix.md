You don't!

There is no UNR any longer now that unity is the DE for Ubuntu.

The original UNR versions for 9.10 and 10.04 were a bit like unity with the launcher bar on the left, etc.  Now you just use the normal Ubuntu .iso download and then customise the desktop as far as you want, or more likely, not as far as you want, because there are few changes possible at the moment.
