### Post by Pawprint on 2009-12-04
I'm currently (i.e., at this very moment, not all the time) running Kubuntu 9.10 from a USB stick. I would like to install Ubuntu 9.10 to a different USB stick (which is also, of course, connected to this computer), and I mean a full install, not a Live install, for which I could have used UNetBootIn. (That *is* allowed on a USB stick, right?)

I have an .ISO for Ubuntu (but not a burned disc, or this whole question would be moot), which is currently mounted. Is there any command I can run to begin the Ubuntu installation process while I'm running Kubuntu?

---

### Post by aysiu on 2009-12-05
No, there isn't a command.

You need to either burn the .iso to CD or "burn" it to USB using UNetBootIn.

Once you've done that, load up the CD or USB live session, and you can install Ubuntu to another USB stick (a real install, not live session).
