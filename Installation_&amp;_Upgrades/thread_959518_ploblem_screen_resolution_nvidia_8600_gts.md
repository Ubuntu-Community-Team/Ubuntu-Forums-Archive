### Post by Partyboi2 on 2008-10-28
Have you tried changing the screen resolution using Nvidia X Server Settings (System>Admin>Nvidia X Server Setting) if you cannot find it under System>Admin then you probably need to install it by searching for it in Synaptic or from the terminal (Applications>Accessories>Terminal)
```
sudo apt-get install nvidia-settings
```
