### Post by namely on 2007-09-22
A search of the forums didn't show anyone else running Soulseek under Wine, so I'll post this here.  I just now got the Windows Soulseek client 156c running beautifully under Wine 0.9.33 in Ubuntu 7.04.  A couple of small things don't work, like sorting in search windows and connection status icons in the userlist, but all the major functionality is there, and it's fast, faster than the same version of the Soulseek client under Windows 2000 on the same computer, which was reborn as a Feisty machine about a week ago.

I've been using Nicotine+ 1.2.6 for most of that week, and love the app and its features, but am disappointed that the Soulseek server won't allow non-official clients like Nicotine to participate in its distributed search, so general file searches don't return matching results from a Nicotine share (info from Nicotine+ tracking page and slsk forums).  Browsing from the userlist and room searches do work, so I would see the occasional upload, but the mostly-empty queue was giving me feelings of desolation.  

It would be nice if the Soulseek administrators would consider allowing Nicotine+ to participate in the distributed search, if that's technically feasible.

Anyway, it is possible, and it's good, so keep trying if you haven't succeeded already.
