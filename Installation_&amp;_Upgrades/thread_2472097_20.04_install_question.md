### Post by ubfan1 on 2022-02-18
I think the 2048 sector start is for SSD alignment (see [https://en.wikipedia.org/wiki/Solid-state_drive#Partition_alignment](https://en.wikipedia.org/wiki/Solid-state_drive#Partition_alignment) ). Even the old start of 34 had room (LBA 1) for a protective MBR (see   [https://en.wikipedia.org/wiki/GUID_Partition_Table](https://en.wikipedia.org/wiki/GUID_Partition_Table) ).
