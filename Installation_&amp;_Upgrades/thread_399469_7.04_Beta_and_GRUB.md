### Post by taurus on 2007-04-02
For ntfs filesystem:

```
sudo mkdir /media/windows
sudo mount -t ntfs /dev/hda2 /media/windows -o nls=utf8,umask=0222
```

For fat32/vfat filesystem:
```
sudo mkdir /media/windows
sudo mount -t vfat /dev/hda2 /media/windows -o iocharset=utf8,umask=000
```
Otherwise, just copy the file on the CD to your harddrive and then burn it again with k3b.
