### Post by presence1960 on 2009-07-15
what is your current OS? if Ubuntu right click the iso and choose write to disk or open brasero and burn as an image to CD. If you are in windows see this : [burn image]("https://help.ubuntu.com/9.04/switching/installing-burning.html")

For a usb flash drive use [unetbootin ]("http://unetbootin.sourceforge.net/") or if in ubuntu install usb creator by opening a terminal and running this command > sudo apt-get install usb-creator  When installed go System > Administration > USB Startup Disk Creator.

Make sure you set BIOS to boot from USB or Removable drive when trying to boot from the usb drive!
