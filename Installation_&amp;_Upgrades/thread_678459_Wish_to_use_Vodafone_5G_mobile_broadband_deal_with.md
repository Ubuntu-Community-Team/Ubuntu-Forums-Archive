### Post by John Read on 2008-02-02
Problem solved.

Searching the Ubuntu Forums either for Vodafone or Huawei E220 gives you the URL where you can download a very nice GUI utility that does everything.

The main URLs you will need are:
    [https://forge.vodafonebetavine.net/frs/?group_id=12](https://forge.vodafonebetavine.net/frs/?group_id=12)
    [https://forge.vodafonebetavine.net/projects/vodafonemobilec/](https://forge.vodafonebetavine.net/projects/vodafonemobilec/)

In my profile I used:
    Profile name:  Vodafone mobile broadband
    Device model: Huawei E220

Connection preferences:
    Username: vodafone  {lowercase}
    Password: vodafone   {lowercase}
    Preferred connection: 3G preferred
    Authentication mode: Default
    APN Host: vfinternet.au

DNS settings;
   Leave blank


It works very nicely.

Some of the other posts in the forums include screenshots too.

Thank you to all contributors who make it possible for people like me to get connected.

John R.
