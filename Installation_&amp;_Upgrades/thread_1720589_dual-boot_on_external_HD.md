### Post by oldfred on 2011-04-03
+1 on Dutch70's suggestion of unplugging drive.

If you cannot do that or do not want to, you have to use manual install so you get the combo box and which drive to install the grub boot loader into. Otherwise it will install to the internal. That is fixable if done wrong but not necessary.
