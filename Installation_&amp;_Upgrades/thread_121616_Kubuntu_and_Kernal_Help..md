### Post by ajgreeny on 2006-01-25
Install kubuntu-desktop using synaptic and you will have the equivalent of kubuntu on your desktop.  You can add any separate KDE apps you want bit by bit as you want them if they are not in the kubuntu package.  Good luck!

For the kernel of choice I think you need the linux-image-2.6.12.-10-686 kernel also on synaptic.  The description says it's for all the pentium chips but doesn't specifically mention Pentium M.  Give it a try and see what happens, if it's no good you can revert to the 386 image when grub comes up.
