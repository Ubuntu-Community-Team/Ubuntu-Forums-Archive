### Post by ajgreeny on 2010-05-03
So, did you upgrade or install?  If you did an online install, the system should have upgraded everything that is in the normal ubuntu repositories, but not anything you have installed from other sources, such as any PPAs, or other third party repos (medibuntu?) or indirectly using dpkg.

If you did a reinstallation of the system over the old install, there is no way that the system can update everything.
