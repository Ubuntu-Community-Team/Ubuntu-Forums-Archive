### Post by Partyboi2 on 2009-03-12
You need to replace "download_directory" with the actual name of the directory where you have downloaded the intrepid.iso. So if the intrepid iso is on your Desktop you would type
```
cd ~/Desktop
```

If you plan to upgrade to intrepid without doing a clean install you will need to use the alternate cd.
