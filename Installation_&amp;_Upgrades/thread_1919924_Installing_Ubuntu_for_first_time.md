### Post by Bucky Ball on 2012-02-03
Welcome. 

You need to burn the ISO to a disk (make disk image) and boot from the disk. You need to change the boot device when you switch your computer on. From there you can either try Ubuntu from the disk or install. Sounds like you are going the wrong way about it. There is no clickable file to boot from. You need to create a bootable CD (in other words you can safely delete those files from the hard drive and just boot from the CD). ;)

### Post by Bucky Ball on 2012-02-03
Yea, because you will find nothing to boot from and you need to change the boot device, either by getting into the BIOS and changing your boot device to CD, or by pressing a key (sometimes F12) at boot to bring up a screen that lets you choose the device. Choose CD (making sure the CD is in the machine that is). There is nothing to look for manually on the CD to click and boot. It doesn't work that way. Like installing Windows or any other OS. You boot from the bootable install CD.

I'm also wondering if you have just created a data disk with all the files on it and not made a disk image of the Ubuntu ISO. You need to create a disk image for this to work, not the former, a data disk.

I could be totally wrong but just to clarify ... are you in Windows or something else and trying to find a file to boot from the Ubuntu CD? In other words, you're browsing the CD from within another OS?
