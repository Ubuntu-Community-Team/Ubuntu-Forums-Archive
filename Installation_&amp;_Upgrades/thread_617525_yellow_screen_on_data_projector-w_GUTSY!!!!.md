### Post by dimeotane on 2007-11-19
I was doing quite well under feisty making my presentations at work using the data projector.  The monitors control panel allowed me to easily switch to the correct resolution for the external data projector and back to my resolution for my laptop. Restarting xwindows with a ctrl-alt-backspace did the trick.  Worked great! :)

After upgrading to gusty... the projection screen has a strong yellow color cast.  ( I now think this particular problem may be due to a faulty cable, not gutsy..)  The control panel only provides 800x600 for the data projector, and will only switch back my laptop resolution to 1024x768 (instead of full resolution at 1280x800).  Plus I need to 'log out' first to make the change.  And for some reason when I do it takes forever to log back in.  :(

I'm in the process of tinkering with my xorg.conf file to see if I can fix the problem.  But I must say this 'upgrade' to gutsy has not been going very smooth.  :(
It's finicky.. works ok one day and then not the next.  I'm thinking a fresh install of the OS could be good to try.
