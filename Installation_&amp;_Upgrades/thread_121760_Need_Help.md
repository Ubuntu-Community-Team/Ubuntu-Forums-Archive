### Post by rttm on 2006-01-26
I had the same problem with my ISO's, it turn out that I had 2 intermittent bad bits in one of my memory module. Normal memory tests did not find these errors, I used the memory jump test. This test found it after the first run. I replaced my memory module and retested. This time all memory passed. I reinstalled and this time the same ISO that appeard to bad installed properly. Not all memory test will find all errors. I had tried different cd, cd readers, hard drive, original factory 5.10 cd all seem to point to bad media. I hope my $.02 can worth helps.
