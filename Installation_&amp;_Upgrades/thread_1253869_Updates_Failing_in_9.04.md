### Post by drs305 on 2009-08-30
I checked four of the 'not found' references and in each case there is a newer version in the repository. Try updating your sources and then upgrade again. If that doesn't work, try another server (Synaptic, Settings, Repositories, Download from > Other)

```

sudo apt-get update
sudo apt-get upgrade

```
