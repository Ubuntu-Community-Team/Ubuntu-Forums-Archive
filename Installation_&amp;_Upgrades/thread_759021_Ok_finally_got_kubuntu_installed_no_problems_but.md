Despite what others have said, Kubuntu does not have Firefox installed by default. Installing it is very easy.

K Menu > System > Adept Manager, type Firefox in the search box, mark it for install and then click 'apply changes'.


Also, when you post it is helpful to use a subject header that reflects the nature of your post. You want to know how to install Firefox so a good subject header would have been 'How do I install Firefox?'
