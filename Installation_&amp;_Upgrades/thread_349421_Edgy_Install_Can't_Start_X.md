I'm sorry for the long post, but this one has me baffled.  I currently run dapper (server and desktop) and I have a very non-technical friend that I had setup on Hoary way back when. The problem is that his machine is a little older, so the live cd was way to slow.  I went to the alternatives cd and had problems because his machine has a flaky cd drive. I tried three times and even burned an Edgy disk at 4x, but no dice.  It then just did the install of a command line system and it made it through that. I've re-run dpkg-reconfigure xserver-xorg and that didn't help.  If anyone could just help me find what's causing this error, I'd be very grateful.  Maybe I'm blind, but I'm missing some kind of typical error like it can't load a driver, etc that usually causes X not to start.  Is there a package other than ubuntu-desktop that I need to install?  I assumed that would install all the dependencies.

Also, when the X start hangs, is there a way to go to a command line?  I forgot to try Alt-F2 to get to another console, but after X refuses to start I just get a window where I can type anything, but it's not a shell so I can't do anything.
