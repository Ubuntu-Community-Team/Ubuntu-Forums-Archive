### Post by Mighty_Joe on 2009-08-05
What you want is a "persistent" Live USB.  Unetbootin will not create one.  Ubuntu ships with [usb-creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") which can create a persistent live USB drive.
[See this post]("http://ubuntuforums.org/showthread.php?t=1193680") for all the ways to install Ubuntu to a USB drive, including my preference, a full install.
