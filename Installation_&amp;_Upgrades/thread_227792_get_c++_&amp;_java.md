### Post by reacocard on 2006-08-02
open the terminal (Applications->accessories->Terminal) and enter this:
```
sudo apt-get install build-essential
```
will install gcc and the other things for C/C++ compilation.

Take a look at Synaptic (System->Administration->Synaptic Package Manager) for getting software you want. just search it.
