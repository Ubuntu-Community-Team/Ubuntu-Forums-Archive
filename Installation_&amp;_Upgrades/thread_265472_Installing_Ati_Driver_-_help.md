### Post by {spitFire} on 2006-09-26
Open your xorg.conf file and check out the following:

nano -w /etc/X11/xorg.conf

In the 'device' section, make sure that the value for 'driver' is not "vesa" (which should be the default in your case), but is "fglrx".

And, in the 'screen' section, make sure the value for 'identifier' is 'aticonfig-Screen[0]'.

Now, try fglrxinfo on a console.
