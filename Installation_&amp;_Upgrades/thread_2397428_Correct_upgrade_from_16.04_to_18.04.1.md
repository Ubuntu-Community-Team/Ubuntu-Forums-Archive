### Post by Impavidus on 2018-07-29
Apparently the upgrade to the next LTS hasn't been enabled yet. That shouldn't take more than a few days. But it doesn't really matter, as the upgrade to any new release also takes you to 18.04. There's no difference between 18.04 and 18.04.1 as far as upgrades are concerned. The only difference is that the installer for 18.04.1 already includes all patches released in the first three months after the release of 18.04, which you have to download and install separately if you install 18.04. But with an upgrade, you get the latest patches anyway.
