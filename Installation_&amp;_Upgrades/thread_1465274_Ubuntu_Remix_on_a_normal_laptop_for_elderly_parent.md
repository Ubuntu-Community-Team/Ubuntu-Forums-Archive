### Post by snowpine on 2010-04-29
Welcome to the forums! I think that sounds like a great idea, and it should work fine, so long as the computer in question [meets the minimum hardware recommendations]("https://help.ubuntu.com/community/Installation/SystemRequirements"). :) 

Also keep in mind there are a few applications that are left out of the default Netbook install, for example Brasero disk burning software is excluded because most netbooks don't have optical drives. You can easily install any "missing" applications from the Software Center if you think your parents will need them.
