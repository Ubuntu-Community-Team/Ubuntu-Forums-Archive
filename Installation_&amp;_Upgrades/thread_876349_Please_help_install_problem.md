### Post by linux_tech on 2008-07-31
Verify the old pc meets the minimum installation requirements;
requirements for 8.04 are:     
    * 300 MHz x86 processor
    * 64 MB of system memory (RAM)
    * At least 4 GB of disk space (for full installation and swap space)
    * VGA graphics card capable of 640x480 resolution
    * CD-ROM drive or network card 
note: The minimum memory requirement... is 384MB for the LiveCD installation.

refer to this link for the install-
[http://www.futuredesktop.org/](http://www.futuredesktop.org/)

Is the install just stopping or freezing before it reaches the partitioning stage?
