Here's one option:
[http://ubuntuforums.org/showthread.php?t=179902](http://ubuntuforums.org/showthread.php?t=179902)

You could also set your 2nd hard drive in bios to boot before your Window's drive, before you install Ubuntu...or as suggested in the link I gave, physically disconnect your Window's drive, connect the Ubuntu drive to the first controller or as master(make sure jumper settings are correct), install Ubuntu, then reconnect your Window's drive.  If you have "hot key" that allows you to select which drive to boot, you could use it to switch OS.
