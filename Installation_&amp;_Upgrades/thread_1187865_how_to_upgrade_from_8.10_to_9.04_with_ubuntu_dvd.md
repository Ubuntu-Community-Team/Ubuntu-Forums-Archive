### Post by _Purple_ on 2009-06-15
If you ordered from shipit, I think they only send CDs. I am not sure though. Did you order the Desktop version? You need Alternate version to upgrade, else you can only do a clean installation in which case you have to backup all your files. Even when upgrading using Alternate iso, it is better to backup all your files just to be on the safe side. :)

Please check this link for more informations:
[https://help.ubuntu.com/community/JauntyUpgrades](https://help.ubuntu.com/community/JauntyUpgrades)
