### Post by oldfred on 2012-08-24
It still is your choice. Usually you can fix things but sometimes that takes longer than just a clean install.

But a clean install means you have to have good backups which you should have anyway. I cheat with a large drive and keep my old install and do a clean install into another 25GB / (root) partition. All data is on other partitions which I link back so I can boot either system.
