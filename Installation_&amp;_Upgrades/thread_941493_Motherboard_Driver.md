### Post by Pumalite on 2008-10-08
Just get your computer ready. If only Ubuntu; make sure your drive is formated. If dual boot it matters if it's Vista. In that case you have to allocate space for Ubuntu with the Vista partitioner. Then install Ubuntu. With 256 MB of memory or less; go for Xubuntu Alternate CD. You cannot boot a Ubuntu Live CD with less than 512 MB of RAM; unless you make a 500 MB /swap partition ahead of time. As for the driver; you mkight need i or 2 AFTER the installation. If you are installing in Raid; that's another matter.
