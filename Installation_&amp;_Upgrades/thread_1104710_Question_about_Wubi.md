> If I tell Wubi to install Ubuntu somewhere, does the partition get completely erased, or is Ubuntu installed together with the Windows files already there?
No, Wubi is like a "virtual" drive or a regular folder found in c:\ubuntu. If you unintall it from control panel it goes away like any other windows program.
Ubuntu will work just like a full intallation it will be slighty slower but if you have enough RAM, 512MB or better it will be faster that XP/Vista. Defrag windows before you run the installation.
to access windows files from Ubuntu look for the \host folder, also avoid hard reboot when using Wubi
