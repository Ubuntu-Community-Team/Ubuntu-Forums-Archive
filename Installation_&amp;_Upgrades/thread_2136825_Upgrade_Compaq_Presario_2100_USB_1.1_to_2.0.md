Also, since I want to drive a hard disk, the pcmcia card must provide enough power for that purpose.

### Post by Ralph L on 2013-05-20
Just to close out this thread, I bought an AKE USB2.0 CARDBUS BC168 32 bit PC CARD.  It fits flush into the PCMCIA slot, but has an external cord that you plug into one of the USB 1.0 ports to supply more power to the PC Card.  With the cord connected I had enough power to run my USB 2.0 disk.  (The USB 2.0 port provides all the power to the disk).  The disk ran fine.  

I did not try a firewire to usb adapter as this seemed to solve my problem.
