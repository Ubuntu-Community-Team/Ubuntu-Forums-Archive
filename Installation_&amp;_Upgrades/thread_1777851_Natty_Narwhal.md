### Post by howefield on 2011-06-08
I'd say yes, but then your mileage may vary.

You'd want to back up all your data that you wouldn't want to lose if you haven't already, and I'd suggest taking a Clonezilla disk image if you can. This will mean you can get back to your starting point relatively easily should the process go wrong.

If you upgrade you may log into the new Unity interface, should you not like it you can log into the previous and familiar gnome environment by selecting Ubuntu Classic from the Session menu at the login screen.

### Post by ghostborg on 2011-06-08
Try out the Unity interface for a while and if it doesn't work for you, Logout and select your user name and at that time optional settings will appear at the bottom to select different installed  Desktops. You can select Gnome Classic then type in your password. My understanding is that the Unity interface sits on top of the Gnome Classic at this time.

I also noticed with 11.04 and Compiz that when you mess around with the settings in Compiz Config Settings Manager it seems to clear some important check-boxes. Like Effects->Window Decoration and things like Window Management -> Move,Resize windows. Many people have been thrown off by this so just re-check them and all should be good.
