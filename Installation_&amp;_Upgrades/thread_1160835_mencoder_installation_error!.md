### Post by tommcd on 2009-05-16
Ubuntu 7.10 reached end of life on April 18, 2009.
[http://www.ubuntu.com/news/ubuntu-7.10-eol](http://www.ubuntu.com/news/ubuntu-7.10-eol)
So 7.10 is no longer supported. You options:
1. Do a dist-upgrade to 8.04.
2. Do a clean install of 9.04. 
I would choose option 2. If you want to do stuff like video encoding with mencoder, then you want an up to date system. Ubuntu 9.04 is very good imo.
