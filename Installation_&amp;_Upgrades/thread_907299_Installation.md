### Post by smurphy_it on 2008-09-01
Download the .iso file to your hard drive, then use a CD-Burning program (like nero for example) to burn it to disc.

Whatever program you use, you don't drag and drop the .iso file, you create the CD (disk image) from the ISO file.

For ease of use, you could always download easyiso(no install) then just run the program and choose "create cd from ISO", point it to the .iso file and if you have more than one optical drive, choose the right one... A few minutes later you'll have the Ubuntu burned off on a cd.
