### Post by solar george on 2007-04-24
The desktop install cd boots to a live CD session (an ubuntu system running from the CD) and allows you to install.

The alternate CD uses a text-menu-based install system (its not as scary as it sounds) no live session.

The alternate cd is better than the desktop unless 
A. You are new to ubuntu and want to try before you install and do so from a 'nice' environment.
B. You have a fast computer and want to browse the internet at the same time as installing.

> Will the desktop cd let me choose where to install grub boot loader, or I will need the alternate desktop cd?
They should both let you install the grub loader where you want (I'm not sure because I only use the alternate cd when doing installs, not the live/desktop)
