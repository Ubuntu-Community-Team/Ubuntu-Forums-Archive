### Post by linhtinh321 on 2007-06-25
Install "GNOME partition editor", then run it from System -> Administration -> GNOME partition editor. You can see what you want.

### Post by mikewhatever on 2007-06-25
You can also use 'sudo fdisk -l' in the terminal. That would give you all but the sizes.
