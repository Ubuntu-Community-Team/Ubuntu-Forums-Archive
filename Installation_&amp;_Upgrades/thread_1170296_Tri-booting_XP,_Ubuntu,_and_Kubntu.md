### Post by ushimitsudoki on 2009-05-26
There's not much point in dual-booting Ubuntu/Kubuntu, because you can just install both DE (Gnome/KDE), and then select which one you like at the login manager. Done this way, you get the same basic results.

That being said, it is true that this won't be *exactly* the same as "true" triple booting. 

Most guides are going to talk about a triple boot with 2 (or more) different Linux distros + Windows, rather than 2 "flavors" of the same distro + Windows. [Here's one]("http://www.go2linux.org/triple-boot-windows-two-linux"), and there are a ton more on Google.
