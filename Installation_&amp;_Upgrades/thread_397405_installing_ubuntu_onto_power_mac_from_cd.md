### Post by ajgreeny on 2007-03-30
I suspect you just burned the image  file to CD as a single file rather than as a disk image.  I'm not sure how things work in Macs, but you will probably need to burn another CD, as a disk i9mage this time.
To check what you did last time open up the CD using your file manager.  If there is just one file, the same as you downloaded, then it will not boot.  There should be a number of different files on the CD in several folders, not just one.
