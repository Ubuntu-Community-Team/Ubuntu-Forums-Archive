### Post by IsawSp4rks on 2008-03-16
Follow the directions in this guide:-

[http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide)

specifically - Step 2.2 **Method 2: Install the Catalyst 8.3 Driver Manually**

That should ensure you have the latest driver properly installed, including removal of any older drivers.

That's what I've always used (even though I did try Envy once, I found it to be slightly buggy when removing/uninstalling the driver) to get Compiz Fusion and Games working properly.  I personally do not find the Restricted Driver Manager installation to be 100% successful when working with ATI  hardware on Gutsy.
