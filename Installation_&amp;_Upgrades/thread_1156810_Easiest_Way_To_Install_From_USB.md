### Post by ashmew2 on 2009-05-12
Grab the Ubuntu .ISO from [www.ubuntu.com](www.ubuntu.com) (Must be a Live CD)

Get UnetBootin from here : [http://lubi.sourceforge.net/unetbootin.html](http://lubi.sourceforge.net/unetbootin.html)

Start unetbootin.
Select Iso image.
Click Start.

You have a usb in less than 5 minutes :D

Just plug it in the laptop.
Configure it too boot from USB.

BAM! You've nailed it :P
