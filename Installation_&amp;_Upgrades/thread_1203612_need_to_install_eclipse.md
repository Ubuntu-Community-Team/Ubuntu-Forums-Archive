First, for example using Synaptic Package Manager, make sure you have a Java Development Kit installed, for example sun-java6-jdk.

Then make sure a C++ compiler is installed, for example g++-4.3.

Download the current version of Eclipse with the CDT from Eclipse.org, not the version that shows up in the Synaptic Package Manager.  Eclipse is just a Java application program so you can just unpack it in a directory under your home directory.  You can run Eclipse just by double-clicking on the Eclipse executeable.

For convenience you will want to drag the Eclipse executeable icon to the Applications/Programming list.
