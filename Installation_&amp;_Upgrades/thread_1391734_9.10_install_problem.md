I would suggest reinstallling Ubuntu 9.10 once again. Don't import anything.

If your problem persists you can first have a look at your grub config file to see what's in it. Assuming it still won't boot, use the live DVD/CD. Mount the drive containing the new Ubuntu (and supposedely GRUB); open console and go to the drive and then go to something like /etc/grub/grub.cfg (can't remember the exact file name and path by heart). Open it with vi or gedit and check if you see something particular.
