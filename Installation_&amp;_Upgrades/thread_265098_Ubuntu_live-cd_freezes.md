### Post by wpshooter on 2006-09-26
I'm sort of scratching my head on that one.  6 years old but state of the art, seems to be a contradiction in terms.  Sorry, I did not see the word **WAS**.

But in any case, here is what I would do.

Download the KILLDISK program from this site and WIPE the entire hard drive clean.
[http://www.killdisk.com/](http://www.killdisk.com/)

Once you have done that, put in your DESKTOP Ubuntu CD, making sure of course that the CD drive is your first boot device in BIOS.

When the initial Ubuntu boot screen menu comes up, run the verification test on the CD.

If it, passes the verification test, i.e. no errors, then reboot and this time choose the load/install choice on the menu.

If and when the system gets up to the desktop, the double click on the install icon on the desktop.

Make sure your let your CD spin down to a stop between each parameter that you are prompted with in the installation process.

When you get the prompt regarding the keyboard language, make sure that you click on the test box at the bottom of the screen (you don't have to type anything in it, just click on it) and then click back on the language choice at the top of the screen and then click on forward button.  Make sure you give the CD's install process plenty of time between answering each prompt.

If this doesn't work, you may have to download, burn and try the ALTERNATE Ubuntu CD.

Good luck.
