### Post by acetheace on 2010-02-04
hmm ... Optiplex computers are rather low quality and so it could be an issue with the hardware itself, I've had quite a few problems with GRUB and boot-up on them myself, unless it's been working well for 6 months + in which case it's probably something else.

Did you change the BIOS to boot up on CD by default, and if you did, did you change it back to hard drive?

After checking that, if it's set to boot by hard drive and it doesn't work, I would try booting the live CD again just to make sure it's not a problem with the hardware... you could also make sure all the files are on the hard drive on a live CD; I've had issues with the hard drives on Optiplex as well.

In addition to answering whether there are any other operating systems on the hard drive - failure to boot could mean a number of things
