### Post by steve457 on 2007-06-30
My first ever attempt using Ubuntu has been a success!  My goal is to setup a file sharing server using Ubuntu and software raid-5.  I just purchased the following components this week for my server:

Asus P5K-WS (Intel P35 chipset) supports 1033 MHz FSB
Intel 4300 Core 2 Duo
2GB Crucial Ballastic memory
Gigabyte NVidia GeForce 7300GS
Maxtor 500GB SATA II Harddrive w/16mb buffer
Lite-On 20x DVD RW drive SATA

All this week I was debating whether I should run Ubuntu or Windows XP Pro for my server, and kept going back and forth between the two.  Just two days ago I again decided to try Ubuntu and decided that if I had problems I would just wipe it out and install Windows.  Well... the installation went smoother than I could have imagined.  Even smoother than a Windows install.  I  inserted the Ubunto 7.0.4 disk into my drive, and the computer booted right off of it.  I then used the graphical installer and just sat back.  When the computer rebooted Ubuntu was working fine, the network connection was up and running, and my sound even worked!

Now, I'm in the process of installing a bunch of other software that I plan on using.  I'll work on getting the raid-5 functionality once I purchase my other hard disks (am planning 3x750GB drives).  In the meantime I am trying to get Samba up and running, among other things.  So far my experience with Ubuntu has gotten off to a great start.
