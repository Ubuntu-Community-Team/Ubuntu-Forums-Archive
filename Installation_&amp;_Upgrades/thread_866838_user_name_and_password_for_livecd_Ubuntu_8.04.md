### Post by Gallienus on 2008-07-22
Is the screen that asks you for your password identical to the one that you see when you boot *without* the LiveCD? If the screens *are* identical, try your normal password for that computer. If that works, then you probably aren't actually booting from the LiveCD, even though you think you are. Make sure that your computer is set up to boot from CD in your BIOS settings.
