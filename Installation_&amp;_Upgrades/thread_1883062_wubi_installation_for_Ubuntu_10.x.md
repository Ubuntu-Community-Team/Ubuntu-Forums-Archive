### Post by Karlchen on 2011-11-18
Hello, bofred.

Have you still got the ISO-file for Ubuntu 10? Which version 10 exactly by the way? v10.04 (Lucid Lynx) or v10.10 (Maverick Meercat)?
In case you still have got the ISO which you used to perform the WUBI installation of Ubuntu 10, the appropriate wubi.exe is located inside this ISO-file. You can extract it from the ISO-file e.g. with the help of 7-Zip.
In case you do not have the ISO-file any longer you may get the ISO-file here e.g.:
Lucid: [http://swtsrv.informatik.uni-mannheim.de/pub/linux/distributions/ubuntu-release/10.04.3/](http://swtsrv.informatik.uni-mannheim.de/pub/linux/distributions/ubuntu-release/10.04.3/)
Maverick: [http://swtsrv.informatik.uni-mannheim.de/pub/linux/distributions/ubuntu-release/10.10/](http://swtsrv.informatik.uni-mannheim.de/pub/linux/distributions/ubuntu-release/10.10/)
Select the appropriate ISO-file which you need and the wubi.exe which is located in the same folder.
Store both on your local disk in the same folder, launch Wubi and tell it use the ISO file located in the same folder.

Good luck,
Karl
