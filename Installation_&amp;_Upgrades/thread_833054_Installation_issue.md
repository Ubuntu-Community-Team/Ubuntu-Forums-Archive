### Post by avtolle on 2008-06-18
At 256 mb Ram, you really should try the "alternate install" iso. According to the official documentation, to use the Desktop CD you need to have 384 mb Ram to handle the graphical installer. Note: according to the same official documentation, Ubuntu with the Gnome DE will run on 256 mb Ram, and this has been confusing to many. To get the "alternate install" cd (text based, but after installation you will have the same packages as to the DE and other apps as on the Live CD), go to the d/l page, scroll down, and check the "alternate install" box. Good luck.
