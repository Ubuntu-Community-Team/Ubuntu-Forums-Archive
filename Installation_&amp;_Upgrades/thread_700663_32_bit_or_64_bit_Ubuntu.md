### Post by Pumalite on 2008-02-19
> **GSZX1337 said:**
> Why would the Alternate CD be better? I've never heard of the Alternate CDs being any better.

The Alternate CD avoids possible hardware and/or graphics problems. It's text based and in fact easier to use for installation. Among other things, it gives you more control. I use it all the time. The Live CD was made more for new people interested in Ubuntu; wanting to try the OS without altering their computer configuration.
