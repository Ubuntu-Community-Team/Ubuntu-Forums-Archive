### Post by presence1960 on 2009-07-24
> **iamsolo said:**
> can anyone tell me how to fully uninstall it

boot from the Ubuntu live cd. choose "try ubuntu without any changes to your computer". when the desktop loads go System > Administration > Partition Editor. Either format your disk as NTFS or delete Ubuntu's partition(s) and leave them as unallocated space. Then quit the Live CD and boot from your Windows disk.
