### Post by Sef on 2008-05-16
> -check cd for defects,

What happens when you run that one?  If it doesn't run then something is wrong.  If it runs, it will tell you if there is a problem or not.

Possible causes of problems:

1) Have you md5summed the download?

2) Have you burned the iso as an iso?

3) Have you burned the iso at 4x or less?

4) The disk is bad.
