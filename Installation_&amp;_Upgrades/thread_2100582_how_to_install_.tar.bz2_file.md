
for the latest beta, install that ppa & update before installing:

[https://launchpad.net/~mozillateam/+archive/firefox-next](https://launchpad.net/~mozillateam/+archive/firefox-next)

note: tar* is a compressed format, google around "ubuntu tar" for more info
