### Post by anystupidname on 2009-06-20
Assuming your primary IM account is yahoo, try using an IP in this range for the server:
66.163.181.166-66.163.181.189
instead of scs.msg.yahoo.com

More info:
[http://developer.pidgin.im/ticket/8853](http://developer.pidgin.im/ticket/8853)
[http://www.celticwolf.com/useful-information/faqs/26-pidgin-yahoo](http://www.celticwolf.com/useful-information/faqs/26-pidgin-yahoo)
[http://ubuntuforums.org/showthread.php?t=1006172](http://ubuntuforums.org/showthread.php?t=1006172)

UPDATE: Pidgin 2.5.7 appears to have at least a partial fix for this...
