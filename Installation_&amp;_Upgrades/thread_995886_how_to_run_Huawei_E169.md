### Post by brianbaru on 2008-11-28
using hardy
 i am using a huawei 160g which i recieved from 3 uk this week
the tutorial i followed to install was origanaly posted by fingers and thumbs 

basically

sudo wvdialconf

creates wvdial.conf

then edit /etc/wvdial.conf

change phone number, username, password as below

my wvdial.conf

[Dialer Defaults]
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0
Modem Type = Analog Modem
Baud = 9600
New PPPD = yes
Modem = /dev/ttyUSB0
ISDN = 0
Phone = [COLOR="Red"]*99#[/COLOR]
Password = [COLOR="red"]three[/COLOR]
Username = [COLOR="red"]three[/COLOR]

i then manual edited network settings

select point to point

General

tick enable this connection
connection type gprs/umts
access point name 3internet
phone number *99#
dial prfix BLANK

account data

username three
password three

modem tab
modem port /dev/ttyS0
dial type tones 
volume off

open a terminal type [COLOR="red"]wvdial Defaults [/COLOR](the capital D is important


with ibex i only had to plug it in and it was recognised
asked to be configured
selected united kingdom
selected 3
wait awhile
click firefox
all ok
