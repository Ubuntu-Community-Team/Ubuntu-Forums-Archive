### Post by hno3 on 2012-04-27
I think my answer won't correctly answer your question because you probably are looking for something like - installing minimum possible stuff in Unity and remove stuff which is not required based on your requirements (as in for what purpose you use the system) and then install it. Something like customizing. But I have an alternative answer which is this: Xubuntu and Lubuntu. These are not exactly a minimal versions of Ubuntu but they use different desktop environment that Ubuntu which are lighter on your resources. Ubuntu uses Unity, whereas Xubuntu uses Xfce and Lubuntu uses X11 Desktop Environment. I too once wanted to use minimum combination of Gnome 2 not Unity (I use 10.04) but I tried Xubuntu three months back and am very happy about it.
