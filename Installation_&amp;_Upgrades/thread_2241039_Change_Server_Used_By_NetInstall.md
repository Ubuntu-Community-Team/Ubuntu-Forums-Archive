### Post by buzzingrobot on 2014-08-24
OK, it's far from obvious, but, in "expert" mode, at least, the displayed server name is one entry on a list that can be navigated with the cursor keys. At the top of the list is an option to manually specify a server.

I suspect this works in the normal default install mode, but I haven't done another install to check.
