### Post by warp99 on 2010-02-23
Boot to a LiveCD on the machine you downloaded the ISO. Create a Live USB pendrive using the USB disk creator tool located in System > Preferences. Make sure you have access to the original ISO you downloaded. One you do this you can install from the Live USB instead.

Ever since I started installing from a USB instead of a CD there's been zero problems with decompressing files.

### Post by warp99 on 2010-02-25
If you have another hard drive with Ubuntu already installed you can drop that into this machine just to see if it boots. If it does you could install the OS on a separate machine and just move the hard drive to the new machine. Just make sure you don't install any proprietary video drivers until the hard drive is in the new machine. I had to do this once with an older machine that just would not boot with a CD.
