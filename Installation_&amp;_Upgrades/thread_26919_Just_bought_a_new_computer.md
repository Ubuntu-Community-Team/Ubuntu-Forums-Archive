### Post by shakin on 2005-04-14
Yes, you must install Windows first and then you can choose to install Ubuntu on either unpartitioned space on your Windows drive or on a second drive. You don't need to worry about how that will work because the Ubuntu installer will automatically detect Windows and when you boot your computer it will give you the option of selecting either one.

I would recommend the 32-bit version of Ubuntu. You won't be losing any functionality or speed, but the 32-bit release is better tested than the 64-bit one and everything should work easier. The 64-bit version is nice if you have more than 4GB of RAM or have a specific 64-bit app that you want to run.
