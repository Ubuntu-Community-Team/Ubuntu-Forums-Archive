### Post by ubnoobie on 2006-06-17
a "server" installation off any of the "alternate" install cd's (the 'old school' console installer) is just a "bare" or console-only install (of 'ubuntu-standard') with no graphical desktop -- it does not include the server optimised kernel or "lamp" packages. 

[COLOR="DarkRed"]*do not confuse the above with...*[/COLOR]

the server edition installation cd -- which is a bare or console install with a server-optimised kernel if supported by your hardware; a basic "lamp" install is an install option. this cd does not install a graphical desktop in either installation option (must be installed later, after system in installed).
