### Post by Partyboi2 on 2008-10-28
The username and password is case sensitive so check to see if that might be the problem. Try booting into recovery mode and changing the password in recovey mode
[https://help.ubuntu.com/community/LostPassword](https://help.ubuntu.com/community/LostPassword)
If that does not work then since it is a new install maybe consider reinstalling but check the [[COLOR=Blue]md5sum[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM")  of the iso you downloaded and also check the cd for defects (one of the options at the main menu of the ubuntu cd) as there should have been a stage where you would create a user and password through the installation process.
