### Post by tommcd on 2009-11-25
For ADSL and DSL internet connections that use PPPoE and require a login with a username and password, you can use **pppoeconf** to configure your internet connection. See this article:
[https://help.ubuntu.com/8.10/internet/C/modems-adsl-pppoe.html](https://help.ubuntu.com/8.10/internet/C/modems-adsl-pppoe.html)
The pppoeconf package is installed by default on Ubuntu 9.10.
