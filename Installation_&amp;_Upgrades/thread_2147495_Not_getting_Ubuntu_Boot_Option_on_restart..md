[I]I also had problems with install. 
Got notification of error: unknown file system to error: invalid arch independent ELF magic. Read lots of forums but none offered me the solution. 
I had a SATA drive connected to an older computer and my IDE drive removed. On the new SATA drive I had installed Xubuntu. It turned out that the motherboard the CD-ROM drive on the IDE connection saw as the first hard drive. After I had removed all IDE connections and only used the SATA drive the install ran well and I had no more problems. 
[/I]
