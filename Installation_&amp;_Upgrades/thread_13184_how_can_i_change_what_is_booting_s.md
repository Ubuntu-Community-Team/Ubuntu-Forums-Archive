### Post by rudi on 2005-01-30
[QUOTE=roxxe]how do i change that modules.conf since its autogenerated? ???[/QUOTE]
   Read the header of modules.conf:
    ```
### This file is automatically generated by update-modules"
  #
  # Please do not edit this file directly. If you want to change or add
  # anything **please take a look at the files in /etc/modutils** and read
  # the manpage for update-modules.
  #
``` 
   it seems that the man pages for update-modules is depricated, but after a quick google i cam up with this link:
   [http://www.zevils.com/cgi-bin/man/man2html?update-modules+8]("http://www.zevils.com/cgi-bin/man/man2html?update-modules+8")
   
 If you look in /etc/modutils open the file aliases to see if your modules il listed.
 
   Hope you can disable it now
