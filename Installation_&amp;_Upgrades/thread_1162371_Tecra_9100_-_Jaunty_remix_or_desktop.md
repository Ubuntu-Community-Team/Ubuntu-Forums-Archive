### Post by Turambar25 on 2009-06-05
I have tried the Remix in the 8.04 and 9.04 flavors and I have used 8.10 and 9.04 on a laptop and desktop. 

The main differences, as I have seen them, are the UME user interface and a trimmed down package set on the Remix. Both of which seem to be designed to account for the 2 Gig drive capacity and 9 inch screen of some netbooks.

I think for a unit of that size the desktop version is going to be the most workable. You have plenty of room to see and install packages, so there seems to be no benefit to running the smaller OS.
