### Post by meng on 2007-02-17
Make sure the BIOS is configured to boot from the CD before the hard drive.

### Post by meng on 2007-02-17
You press some key (maybe it's del, esc, f1 thru f12) to edit the BIOS. If you're lucky, the key will be shown on screen at boot time.
