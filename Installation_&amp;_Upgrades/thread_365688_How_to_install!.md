### Post by Qrk on 2007-02-19
There is a guide on the wiki

[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)

Installing things on linux is either very very easy or somewhat difficult. Using add and remove applications, or a more advanced version, called Synaptic (under System->Administration) makes installing open source programs as easy as clicking.

Getting closed sourced apps working can be a pain in the butt, each one has a different method of installing itself.
