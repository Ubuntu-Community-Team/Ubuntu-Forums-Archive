### Post by diablozx9 on 2007-10-20
Just finished 7.04 to 7.10 upgrade.
here is what I found.

1. I couldn't perform the upgrade at fist because of slow server response (Ubuntu is getting popular eh?)

2. Next issue, I couldn't upgrade because I had to little room in my boot directory. I have run Ubuntu without fresh install for years now.. A couple of serches showed the I had to many Linux upgrades in my boot directory. There is an easy command to remove each old one,, Perform a few searches if you have same issue.

3. Download of new load is VERY slow. It took about 12 hours. I just left it on....

4. Flawless upgrade after that.. This was the best one yet. I have worked in IT for 20 years and can say that this was the smoothest upgrade I have ever seen. NO microsoft upgrade ever was or ever will be that good.

I have a 3Gig AMD 64bit CPU  (but run 32 bit version). Nvidia 256 Meg Vid card (I only ever saw one Nvidia card had issues with Linux- an old 440 MX), 1.5 Gig ddr. 80 Gig drive.

Cheers and Thanks.
