### Post by benanzo on 2007-05-05
just boot to the Feisty disk and start the install normally.  when it comes time to partition be sure to select manual partitioning.  then find your (current) root partition and select it as the new root partition (it will be formatted automatically.)  Then for /home select your (existing) /home partition and make sure that the Format? checkbox is NOT CHECKED.   Then it should install normally.  For good measure though, I would strongly recommend you make a copy of the things you can't replace just in case.

Good luck.
