### Post by Bucky Ball on 2015-06-06
Welcome. Open a terminal and type in 'lifeograph'. It will launch. For some reason it is not in my Apps menu either (I just installed it). You will need to make a launcher for it if you don't want to be launching it from a terminal all the time. 

In Xubuntu, you can right click on the desktop, 'Create launcher' and simply use 'lifeograph' as the command. I have no idea what the procedure is in Unity/Ubuntu. Good luck.
