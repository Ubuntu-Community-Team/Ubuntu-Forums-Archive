First you need to backup your existing Ubuntu. The easiest way I know how is to use QuickStart (in my sig).
You can backup your Windows drive with it as well, just to be safe.

Now comes the tricky part.

With the Live CD: [Installing a Dual-Boot with Windows and Ubuntu]("http://www.psychocats.net/ubuntu/installing")
With the Alt CD: [Install Alternate CD Ubuntu]("http://users.bigpond.net.au/hermanzone/")

Once you have that done, use the Restore function in QuickStart to restore your /home folder and you're done.
Any further questions feel free to ask.

Bruce
