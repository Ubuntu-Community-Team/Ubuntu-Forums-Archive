### Post by cszikszoy on 2008-09-01
Are you using the Live CD when this happens?  Did you ever make it to the installer?

If you're having problems booting with the Live CD, you might want to try using the Alternate CD.  It's specifically made for installing on older computers with less RAM.

Also, if RAM and processing power are an issue, I would try XFCE instead.  It's a bit less memory intensive as gnome is and runs very well on older hardware.  I have an XFCE desktop set up on an old Pentium 2 with 128 mb of ram and it's great!  (Oh, and a whole 2 gigs of hard drive space :) )
