### Post by aysiu on 2005-12-13
Completely blank... meaning it doesn't have a hard drive? Doesn't have a CD-ROM drive?

By the way, you don't install the ISO directly. The ISO is a disk image. You need to burn it to a disk (as a "disk image," not as data). You boot from the disk, and the installer appears.

### Post by JoshHendo on 2005-12-13
No offence, though from what you have said (not knowing what to do with the iso file), I think windows would be a better choise for you. Anyway....

You need to burn the iso file onto a CD. Just use the software that comes with your CD burner (if you don't have a CD burner, stick with windows). Configure your BIOS to boot from a CD. (When turning the computer on You should see something that says press "Del" or "Delete". Do that and you will be in your BIOS. Set it so that it will boot from a CD). Put the CD in your drive and it will boot off it (hence setting up the bios to boot from a cd). It will do a few things. If it asks you to do something, do what it says :). When you get to the partitioning part, if you want only ubuntu on your system, select the first option, if you want it dual booted, set it up manually. Continue the installation. This is simple step by step. Then it will restart into your new ubuntu system :). Though I must say that your question is very general. About the tutorials, there are pleanty of them on the net. Search. About been on the ubuntu website, I don't see any instructions on how to install windows on the MS website. If you can't mangae to install it, stick with windows.
