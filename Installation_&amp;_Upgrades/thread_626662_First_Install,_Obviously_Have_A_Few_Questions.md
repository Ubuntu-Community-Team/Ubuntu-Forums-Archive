### Post by jpl80 on 2007-11-29
First off, If you could get a dedicated machine to run ubuntu, that would be best. 

But if you must dual-boot: 

I would defragment your hard drive, twice. Then just install Ubuntu. Before anything is actually installed, the partitioner should let you select which part of the hardrive to install Ubuntu onto.

That's the general idea, but you may want to read a little more about the Ubuntu partitioner before you proceed.

Here's a fairly nice explanation:
[http://www.psychocats.net/ubuntu/partitioning](http://www.psychocats.net/ubuntu/partitioning)
