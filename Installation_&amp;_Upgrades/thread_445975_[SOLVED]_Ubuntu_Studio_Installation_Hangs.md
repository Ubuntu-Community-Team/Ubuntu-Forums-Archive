The "Detecting Hardware" progress bar sits at 0% for a few minutes, then jumps to 6% and says "Loading module 'trm290' for 'IDE Chipset Support'..." Nothing after that happens. I left it like that for about 4 hours yesterday, and it just sits there. The same thing happens if I select "Text Mode Install for Manufacturers," or "Check CD For Defects" from the installation menu.

Could it be that I'm just not waiting long enough? If so, why is everything going so slow? Any ideas?
