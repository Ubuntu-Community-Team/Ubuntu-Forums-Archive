### Post by zanox on 2009-08-22
Hi to all! It's known on the web that version 3.26 blue-utilz (the bluetooth manager) on Hardy permits to send files to nokia mobile phones as my 6680, but not to receive from phone. 
The solution is a downgrade to 3.24 version, that's I done.
So, now I have the update icon enabled on the top-right corner, and it shows obviously that there are updates for blue-utilz. My question is: how can I remove this files from update list to avoid updating them when I update other components? Thank you for help ;-)

---

### Post by Partyboi2 on 2009-08-22
Hi, you can open Synaptic (System>Admim>Synaptic) and do a search for 'blue-utilz' then highlight it and go to Package>Lock Version, this should prevent it from upgrading.
