### Post by sudodus on 2013-09-17
Welcome to the Ubuntu Forums :-)

I think you have problems with the graphics driver. Try the boot option nomodeset or some of the other [boot options]("https://help.ubuntu.com/community/BootOptions").

You can also try another version of Ubuntu. If the computer is new, try next the version (available as a beta version for testing). It might have better drivers for very new hardware.

[http://iso.qa.ubuntu.com/qatracker/milestones/302/builds](http://iso.qa.ubuntu.com/qatracker/milestones/302/builds)

If it is older, try the version ***12.04 LTS***. Consider trying also some of the other flavours of Ubuntu: Kubuntu, Lubuntu and Xubuntu.
