### Post by urukrama on 2008-06-18
You can use docks that require compositing (like AWN or Kibadock) in Openbox if you use xcompmgr. For help on how to do so, see my [Openbox guide]("http://urukrama.wordpress.com/openbox-guide/#xcompmgr").

You could also try wbar, which doesn't require compositing.
