### Post by Therion on 2009-04-20
Download a Jaunty Daily Build CD, run it "Live" and see how it works out for you.  You can then do a full-blown install to your hard drive from the same CD (when you're done being blown away by Jaunty generally speaking) and the regular updates, via the Update Manager, will keep you current even after Jaunty goes final.  No muss, no fuss, no bother.  No rubbing, no buffing!!

Plus you'll beat the "server crush" on release day.

[http://cdimage.ubuntu.com/daily-live/current/](http://cdimage.ubuntu.com/daily-live/current/)
