Your meerkat questions can be answered (and probably already have been) here...

[http://ubuntuforums.org/forumdisplay.php?f=385](http://ubuntuforums.org/forumdisplay.php?f=385)

Keep updating with the update manager and you will have 10.10 automatically. You won't require to install separately.

10.04 to 10.10 will require either an upgrade (update manager will inform you that a new version is available - unless you have set it not to) or a clean install, it's your choice as to which you prefer.
