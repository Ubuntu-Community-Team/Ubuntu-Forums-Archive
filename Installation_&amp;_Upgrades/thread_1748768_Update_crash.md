### Post by tommcd on 2011-05-04
Cut your losses and learn from this experience.
I always do clean installs of Ubuntu and I never have problems. I have used every version of Ubuntu going back to the inaugural 4.10 on many different computers. I have never had a problem with clean installs. 
Doing dist-upgrades of Ubuntu are often problematic. This is especially true since so many Ubuntu users are using unsupported third party repositories on their systems. This includes those all too often problematic PPA repos that so many people use.
