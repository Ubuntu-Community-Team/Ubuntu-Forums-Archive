### Post by zigga15 on 2009-04-05
Ubuntu 8 is great out of the box. Most of your stuff should work without getting any drivers like you would with windows. You will have all your stuff there ready to go.

If you want to download programs go to synaptic package manager, this is located in System>Administrator> this is basically an interface for a huge repository that contains lots and lots of programs for ubuntu 8. Basically you choose what you want and its automatically installed for you... you dont even have to press next 1000 times.

all the programs have different names but you can usually google search for "irc client" or "linux word processor" to find what you need. You can even search in synaptic because there is a description for every program.
