### Post by ad_267 on 2008-07-04
You can just add a launcher to the menu yourself. Check that it's installed by pressing Alt-F2 and running "cxoffice" or whatever command you use to run this application. Then right click on your menu and select Edit Menus and add a launcher for this command.
