### Post by bubwitmaingay on 2009-03-06
A *.iso is like *.rar or *.zip because it is an archive file of an entire CD. That means all the contents of the CD was copied to one big file with a *.iso format.

All you have to do is burn the *.iso file into another CD or DVD. If you are using Nero, choose the "Burn Image" option (most CD/DVD Burning tool have "Burn Image" options). Then load the *.iso file for Nero to burn. By the way formats like *.iso are called IMAGE files.

After you burn the image file, you now have the CD installer (or LiveCD). 8)
