### Post by darkod on 2009-11-21
When booted with the LiveCD in terminal run:
sudo fdisk -l

or
sudo blkid

Make sure the drive is NOT marked as raid array member. I think there was similar situation and that was the problem. Also make sure you don't have any raid settings ON in BIOS.
