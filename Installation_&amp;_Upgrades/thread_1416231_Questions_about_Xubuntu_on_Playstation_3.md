### Post by yokohama1970 on 2010-03-03
I have been using Xubuntu 9.04 on my 40gb PS3 for about a month. For basic web-browsing & e-mail, it is great. I tried Xubuntu 9.10, but it was much slower. So i switched back. Please note that you will not be able to "upgrade" to Karmic 9.10 through the synaptic package manger, that is not the power pc/ibm/ps3 version. All in all, Xubuntu 9.04 works fantastic on my ps3. Previously I was using Ubuntu 8.10 intrepid on my ps3.

Also, I picked up the HP elite wireless keyboard & a Logitech v220 cordless mouse, solid as a rock! Finally found a wireless mouse that works on the arm of my couch or chair. I have been unable to install my Logitech media pro ps3 keyboard/trackpad combo. I use it with my MacBook Pro & Mac Mini (OS X 10.6.2).
