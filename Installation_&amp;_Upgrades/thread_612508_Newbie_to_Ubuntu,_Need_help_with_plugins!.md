### Post by jfinkels on 2007-11-14
To install the java plugin for firefox, type the following in the terminal (Applications > Accessories > Terminal):```
sudo aptitude install sun-java6-plugin
``` To install the Flash plugin for firefox, enable the multiverse repository under System > Administration > Software Sources, and then type the following in the terminal:```
sudo aptitude install flashplugin-nonfree
```

For more information, read all the links in my signature. Welcome to free culture.
