### Post by catherine on 2006-06-11
I'm attempting to install Xubuntu from the text installer. The installation itself seemed to go without a hitch, but when I rebooted, I was told that the system couldn't find an operating system.

I then tried 'Rebooting from first hard drive', from the boot cd's menu. I received the following error message:

"isolinux: Disk error 01, AX = 0201, drive 80"

Does anyone know what this means? Or where I might find out what the problem is? It seems like something rather crucial! ;)

---

### Post by catherine on 2006-06-11
I worked out what the problem was - the hard-drive was disabled in the BIOS set-up. No wonder it couldn't find the operating system!

Still - it makes me wonder if that was the problem all along... As in, if that was what made Windows 98 'mysteriously' disappear - we'll never know... ;-)
