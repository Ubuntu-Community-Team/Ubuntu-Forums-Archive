### Post by mansourk on 2010-07-26
you could go to Application>Ubuntu Software Centre and search for your required softwares and install it from there. or you can go to System>Administration>Synaptic Package Manager and search for your software, mark it for installation and click the apply button. also it can be installed with typing " sudo apt-get install programname" in witch the programname is the name of your required program.
