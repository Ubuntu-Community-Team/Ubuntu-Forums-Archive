### Post by strixtux on 2017-12-19
Get yourself a used 2.5" HDD with less than 20 GB space on it. Connect it via USB and install a Linux system DVD ISO as boot medium on it with start media creator. Set BIOS so that USB Boot is first option. Then reboot, follow instructions and enjoy the way it works. I went almost nuts with DVD due to scratches and RW hiccups. I am using old HDD now instead and well, it works flawlessly and even better - fast. It's a bit bricolage but well, life is like that, at times.
The 2.5" are my favorites as they run on USB power only and need no external power source. USB adaptors cost a few bucks but they are worth every single penny.
