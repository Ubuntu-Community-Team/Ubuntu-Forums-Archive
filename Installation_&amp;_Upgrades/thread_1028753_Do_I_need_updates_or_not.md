### Post by donkyhotay on 2009-01-02
Updates don't *necessarilly* mean it will be slower, some updates do increase performance but it is more likely to add features which slows things down. Personally I did updates on my webserver until it died (hardware issue, I know how to fix it I just can't afford to) for the various security/stability updates. If it's only for your local LAN and not going to be connected to the internet I probably wouldn't worry but if the page is meant to be viewable from the internet (even if there is no admin access from the internet) then I would do so. Ultimately though, it's up to you.

edit:Oh, and I only did security updates which helped make certain I only got the ones I really needed
