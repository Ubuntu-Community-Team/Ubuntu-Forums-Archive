What kind of graphics card is it?

xserver-xorg is a package that is part of X Windows.  X windows (Ususally juts referred-to as "X") is the layer of programming that uses your video card to display stuff other than console text.

You can run your computer without X.  Unlike Microsoft windows, everything is seperate in Unix (Linux).  You can burn cds, listen to music, connect to the net and surf the web in text mode if you want to.  Everyting can be done from the comman-line.  Usually, graphical applications are just display programs that call the comman-line program to do the real work behind the scenes.

Anyway, your card should not be crashing the system like that.  How long do you let it run?
