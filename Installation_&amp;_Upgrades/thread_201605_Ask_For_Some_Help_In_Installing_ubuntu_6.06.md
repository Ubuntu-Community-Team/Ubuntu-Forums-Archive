### Post by Herman on 2006-06-22
The default username would be: ubuntu

The 'alternate install' CDs should produce an install similar to those illustrated in my signature links under this post. You should have been prompted for a username.

If you have a genuine, uncorrupted Ubuntu alternate install CD. It should pass an an MD5sum test. An md5sum test is easy to do on a newly downloaded .iso , [*click here to see how.*]("http://www.users.bigpond.net.au/hermanzone/p17.htm#Why_integrity_check_your_downloaded_.iso")
You can also run an integrity test by selecting 'check CD for defects' from the first screen you get before you begin your install. I have never heard of anyone tampering with any Ubuntu install .iso downloads, but if someone lives in a non-democratic country or even some democracies it might theoretically be possible. More likely, you were given the opportunity to type a password but were distracted somehow and missed it.
A scratch or a small lump of dirt on your install CD can cause unpredictable behaviour during your install. I had one CD that I had MD5 tested and used many times and was a good CD, but suddenly it started listing my CD-ROM drive as a device to be partitioned. Then I noticed the scratch.
Regards, Herman.
