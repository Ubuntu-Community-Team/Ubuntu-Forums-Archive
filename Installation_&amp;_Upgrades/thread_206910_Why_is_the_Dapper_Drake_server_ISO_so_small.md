### Post by az on 2006-06-30
Installing from the live cd is the fastest way to install the Ubuntu desktop and you end up with *exactly* the same system as if you installed with the classic installer.

The alternate cd is the ubuntu desktop cd with the classic installer.

The server cd is the server packages (no desktop) and the classical installer.
