### Post by computerfreak97 on 2011-06-17
Fixed this, just wanted to post what I did:

1. Hard reboot the machine, theres nothing else to do
2. Go into synaptic package manager and just select compiz and all associated packages for upgrade
3. Clear all downloaded files (somewhere in the pack. man.)
4. Upgrade what is selected in pack. man
5. If that succeed, go ahead an use the update manager and it will take care of the rest
