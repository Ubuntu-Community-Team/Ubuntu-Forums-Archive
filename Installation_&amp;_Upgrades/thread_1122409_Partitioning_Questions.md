Okay, just a question. I have Ubuntu 8.10 which takes up my whole 250gb hard drive, but I was wondering if I could, in any way, partition this without having to do a clean install. ---

### Post by Partyboi2 on 2009-04-11
You can boot a Ubuntu live cd and use gparted (System>Admin>Partition Editor) and resize your current partition then create a new partition with the newly unallocated space. Remember to back up your important stuff before working on partitions.
