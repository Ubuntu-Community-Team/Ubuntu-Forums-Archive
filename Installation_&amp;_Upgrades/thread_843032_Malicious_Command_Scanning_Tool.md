I used to post a lot in Visual BASIC news groups because I did a lot of programming in that language (version 6).  There were some pretty cool code repositories where you could download and try out varous VB code, and it was such a widely used language that you could find just about anything.  

Unfortunately, some bozos would do the same type of thing with VB code -- and these were often full programs, so it was not too hard for someone to hide some malware code inside some benign code.

I do need to caution that such a tool isn't full proof.  Nothing could be.  However, it was a pretty cool tool for a VB newbie who isn't up to speed with the language yet.
