### Post by nerdy_kid on 2010-04-01
i might end up upgrading to the beta2, you _can_ actually restore your entire current system if you backup the whole drive first (which is what i will do) with

```

sudo dd if=/dev/sda of=externaldrivemountpoint/file.img bs=4M

```


as long as sda is your drive, that will create a file with the entire drive backed up.  Then if something major fries, then you can restore it.  you can also pick files off the image.
