### Post by Rubi1200 on 2011-02-05
Hi and welcome to the forums :-)

Did you install Windows over the Ubuntu install? If yes, then it is gone.

If you just installed Windows back to where it was and left the Ubuntu partition alone, then you can reinstall GRUB to have the dual-boot option.

See here:
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)

If you need more help, please ask.
