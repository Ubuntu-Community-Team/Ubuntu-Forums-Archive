### Post by doas777 on 2008-10-24
ahh! well heres what i do:

first open a terminal and enter:

```
sudo apt-get install ubuntu-restricted-extras
```

then go to medibuntu at [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu) , and follow the repository how to. it's easy, just copy and paste into terminal.

that should get you all the codecs you need, but let us know if you have any trouble, and there are numerous tutorials available.
