### Post by Xian on 2006-03-04
Since you'll need to make sure both are not mounted when you do the resizing I would suggest you use the [Gparted LiveCD](http://gparted.sourceforge.net/livecd.php). There is a lot of documentation of how to use Gparted on the net if you are not familiar with the application. And backup anything you need before making any partition changes.
