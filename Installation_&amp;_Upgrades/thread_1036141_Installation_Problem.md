### Post by taurus on 2009-01-10
There are a few things you have to do first before you can install it on your laptop.

1.  Run the checksum on the ISO image right after you downloaded to check the integrity of the file.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

2.  Burn the ISO file as an image file and at a slow speed like 4x.

3.  At the initial menu, run the scan cd for defects to make sure the CD is good.

What's the spec of your laptop anyway?
