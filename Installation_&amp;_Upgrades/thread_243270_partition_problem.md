### Post by ciscosurfer on 2006-08-24
A good rule of thumb says to install XP first, then other Linux distro after that (i.e., Ubuntu).  There are ways to install XP on top of Ubuntu, but it involves modifying your current GRUB install, and some other modifications.

I would check these two links for more information with mounting windows partitons and recovering Ubuntu after a Windows install:
[https://help.ubuntu.com/community/MountingWindowsPartitions](https://help.ubuntu.com/community/MountingWindowsPartitions)
[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)
