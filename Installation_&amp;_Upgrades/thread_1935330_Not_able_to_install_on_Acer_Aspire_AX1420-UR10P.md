### Post by pmains on 2012-03-04
I just bought a new Acer desktop, model AX1420-UR10P. It comes loaded with Windows 7. My plan was to back up the hard drive (if possible) and then install Ubuntu Studio.

Well, instead of a normal BIOS, I was able to access the "Windows Boot Manager" by hitting f2. When I contacted support, they informed me that they did not recommend doing what I wanted to do, but offered to sell me the support necessary to do this for $100+.

Since that's not going to happen, I was hoping that someone here on Ubuntu Forums had some idea what needs to be done (beside returning it). Is this related to the [Acer "hidden partition"]("http://techtips.salon.com/boot-acer-hidden-partition-2763.html")? Is this UEFI? Is there a way to blow away the firmware and install a normal BIOS?

---

### Post by pmains on 2012-03-04
Well, the answer is embarrassingly simple. Hit F12 instead of F2. That allows you to change the boot order. Ubuntu Studio is being installed as I type.
