### Post by 1roxtar on 2009-04-29
I have an Acer Aspire One netbook too. I did a dual-install of Ubuntu 9.04 using Wubi and it works great. My webcam works...audio, wireless and wired internet...EVERYTHING is working great.  Using the LiveCD and creating partitions....all that stuff almost ruined another laptop I had (Gateway).  I used Wubi to install Ubuntu on that one too and it also works perfectly now.  My suggestion would be to install it using Wubi.  If you already have an Ubuntu ISO on your hard drive, place it in the same folder that you have Wubi downloaded to and it will find it automatically. Otherwise it will try to download another copy of Ubuntu.
