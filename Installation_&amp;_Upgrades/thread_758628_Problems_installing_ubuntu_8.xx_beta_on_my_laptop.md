### Post by dstew on 2008-04-18
Do a CD integrity check (opening menu item). If the CD is bad, and the .iso you downloaded is good (Md5 sum is correct) then you have a bad burn. Burning an image CD is very hard on the CD burner, every bit has to be correct. Use high quality CD-R media, and a burn speed no higher than 4x, and try again.
