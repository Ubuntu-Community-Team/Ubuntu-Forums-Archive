### Post by Tempelton on 2011-04-13
So I've finally had enough of constant problems with Realtek wireless card in Windows XP installed on my MSI Wind U100. I've decided to switch the OS to Ubuntu 10.10 Netbook edition and it was a good move as the wireless now works perfectly even in my bathroom. :) BUT... There seems to be something wrong with the language. During the installation I've set Polish as a default language. When I launched the system for the first time, I realized that only a few phrases of the GUI were in Polish. My Ubuntu installation is almost 99% english. Even Firefox is in English! What was the point of language selection then? Or did I overlook something? How to change it to Polish?

EDIT:
Ah, I've managed to fix that. I had to install some language packages via System->Languages and sort them by priority. I must have missed that information window when the system loaded for the very first time. :)
