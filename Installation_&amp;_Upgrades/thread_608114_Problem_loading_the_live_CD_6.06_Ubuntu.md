### Post by atomkarinca on 2007-11-09
first of all you should be sure that you burnt the cd properly. the best way to do this:

- do a checksum of the .iso file
- burn the cd at the minimum speed possible (4x mostly, meaning 600 kb/s)
- if you can't boot the live cd you can always try the alternate cd (while downloading you should check the box at the bottom)
- finally you can try a more leightweight version of ubuntu: [xubuntu]("http://www.xubuntu.com/") or [fluxbuntu]("http://www.fluxbuntu.com/js.html").
