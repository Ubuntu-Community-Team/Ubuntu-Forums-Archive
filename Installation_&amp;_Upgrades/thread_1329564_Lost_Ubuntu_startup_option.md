Hello! I recently used the traditional partition installation method to have Ubuntu 8.03 installed on my pc. I there after installed windows xp again because of corrupt files and realised i don't have Ubuntu as a start up operating system option! What can i do to get it back without reinstalling Ubuntu afresh? I want to avoid that whole process of updating it again. Please help me.

### Post by darkod on 2009-11-17
No need to reinstall, Ubuntu is still there. Just grub has been overwritten by windows. This should help:
[http://ubuntuforums.org/showthread.php?t=224351](http://ubuntuforums.org/showthread.php?t=224351)

Tip: If you have Ubuntu 8.04 that means you have grub, not grub2. The instructions are different but grub2 is used only in the new Ubuntu 9.10.
