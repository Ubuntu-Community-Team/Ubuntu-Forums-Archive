### Post by kidders on 2007-04-29
Hi there,

If you're good with Linux, you could probably fiddle the CD's working environment, so installation files can be copied to a hard disk (assuming you have more than one of them). Unless your CD drive is *reallllly* slow, you'd probably have installed Ubuntu normally by the time you'd finished though!

What you might be wondering is whether Ubuntu offers a feature (similar to some Windows installers) that automatically copies installer archives during the installation process. Unfortunately, the answer is no. :-( You see, doing so would interfere with some aspects of the installation itself ... much as it does in the MS world.

Having said all this, I would be happy to walk you through the technique you could use to hack the installer. If you have more than one hard disk drive, you should be able to make it work properly without *too* much hassle.
