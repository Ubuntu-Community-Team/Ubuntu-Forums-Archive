### Post by quattroman on 2007-07-12
Yes. You can.

Just make sure the computers you plug into are able to boot from USB device, most new mainboards are capable of.

I believe you will run into some hardware problems, when changing pcs due to different hardware configurations.

-------------------- TO DO SO----------------

Just run the installation CD, I use alternative, do not like the Live CD install, so during partions make select the USB drive for destination of the OS. 

*you can look in your bios for the drive before booting the install cd to make sure it is recognized*
---------------------

Hope it helped.
