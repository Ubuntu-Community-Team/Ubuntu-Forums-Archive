### Post by kellemes on 2007-10-19
Instead of excepting defaults you should have visited the Ubuntu Wiki (at least) to get prepared..
You can't expect defaults to work on every system.

One thing you may try is down/burn/boot [Super Grub Disk]("http://supergrub.forjamari.linex.org/"). From there you can boot from any partition on disk.. assuming you know where you installed Ubuntu you won't have any trouble booting it. Also it provides all kind of ways of restoring Grub if needed.
