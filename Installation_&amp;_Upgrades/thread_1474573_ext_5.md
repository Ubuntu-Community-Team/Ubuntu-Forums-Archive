### Post by srs5694 on 2010-05-06
The last I'd heard, all eyes are on Btrfs to be the next "big" filesystem after ext4fs. Btrfs incorporates features like those found in Sun's ZFS.

For most people, though, there's no need to "keep up with the Joneses" in filesystems. Most desktop users won't see much difference between ext3fs and anything else (ReiserFS, XFS, JFS, ext4fs, or even Btrfs). To be sure, there *are* differences between them, but they're fairly minor for typical desktop users, which I gather most of this forum's readers are. Some Btrfs features might be important on the desktop, but Btrfs is still new enough that you should only be playing with those features on non-critical systems and if you already know a fair amount about filesystems and their text-mode tools.
