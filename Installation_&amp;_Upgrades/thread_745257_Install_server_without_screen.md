### Post by MrFSL on 2008-04-04
Interesting... 

Well you could do it if the Ubuntu disk booted with proper ssh support.

I suppose you could build this in by making a custom Ubuntu install CD.

Check out these two links:
[https://help.ubuntu.com/community/InstallCDCustomization](https://help.ubuntu.com/community/InstallCDCustomization)
[https://help.ubuntu.com/community/LiveCDCustomization](https://help.ubuntu.com/community/LiveCDCustomization)

I would try to build a install disk that was automated and also installed SSH and configured it for me by default. I would test this using VMWare (which is freely available). 

Good luck!

***EDIT***
Eventually you will want a screen. Either now or later you will need one.
