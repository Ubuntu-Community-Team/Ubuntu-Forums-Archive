### Post by Bucky Ball on 2009-07-22
What kind of design? Try the Gimp for photo shop and Blender is good.

Applications->Graphics->Gimp

and Blender is not installed by default with Ubuntu (part of Ubuntu Studio which might be better suited for you) and can be obtained from the repos:

System->Administration->Synaptic Package Manager

Then click 'Search' and search for 'blender'. Click the little box to the left 'Mark for Installation' then 'Apply' in the left of the upper toolbar.

You will then find the application in Applications->Graphics

There is plenty more. Try 'Applications->Add/Remove' and have a look. :)

When you first boot ubuntu it will ask which language you use, then you will get a list of options to install ubuntu to your hard drive, or to boot into the livecd, or to check the disk for defects.

If you're asking what we do after booting into an already installed ubuntu, I go to: System > Administration > Software Sources > Download From: and choose my local ISP (internode), then reload, and then go to update manager, so all my updates come down without counting towards my usage.
