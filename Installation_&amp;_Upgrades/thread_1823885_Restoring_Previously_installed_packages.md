### Post by manzdagratiano on 2011-08-13
Have you checked that the target USB drive is still functional? USB drives are not very resilient under the stress of hosting an entire OS (I installed Arch Linux on one of them before - worked fine for quite some time, and then eventually gave out). Aside from that, I don't really understand the 'previously installed packages' part since there should not have been any packages before. If the target USB drive is still working, I would suggest starting the install from scratch.
