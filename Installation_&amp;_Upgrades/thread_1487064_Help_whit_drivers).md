### Post by sgosnell on 2010-05-18
Download the .iso for Ubuntu, burn it to a CD or USB drive, then boot to it.  Select the option to try Ubuntu without making any changes to your system.  Play around with it and see if everything works.  If it does, you can then run the installation program.  You can either replace everything, or install Ubuntu side beside with the existing OS and boot to either as you choose.  Ubuntu is very flexible that way.

### Post by sgosnell on 2010-05-18
Your choice.  You can burn it to CD, or to a DVD, or to a USB drive.  You'll need unetbootin to burn to a USB drive.  For burning to a CD, I like [ImgBurn](http://www.imgburn.com/index.php?act=download).  You cannot just copy the .iso to a CD and have it work, you have to 'burn' the image.  Using a DVD-R is a waste of media, because the image will fit on a standard CD.
