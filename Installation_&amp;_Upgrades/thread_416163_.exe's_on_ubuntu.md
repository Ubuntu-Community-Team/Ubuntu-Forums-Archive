### Post by Ziox on 2007-04-20
> **v3mm said:**
> any way to get executable files on ubuntu??:lolflag:

Ubuntu (and all Linux distributions) cannot natively run windows executable files (.exe, .msi, etc...).  Instead, most distributions offers other package (software) management application that's more centralized and more (generally speaking) stable than MS Windows' separate executable files.  

There are many free, open source programs that are the equivalents of proprietary Windows software, just do a search in Synaptic.

However, if you must need the Windows Software, you can experiment with WINE (sudo aptitude install wine), which can run most Windows programs fairly well.  Or another option is to experiment with virtual machines (either Virtualbox or VMware).
