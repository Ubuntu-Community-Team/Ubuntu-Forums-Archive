### Post by warp99 on 2008-04-02
> **damd73 said:**
> I installed Ubuntu 5.04: Hoary Hedgehog Release on an old Dell Inspiron 3000 laptop that was running Win95. Additional info:

200MHZ Pentium MMX
128MB RAM

This machine will be used for lightweight web surfing.  

Ubuntu is running sloooooowly. Is there anything I can do to streamline Ubuntu so it will run faster on this old laptop?

Thanks!

First of all that version of Ubuntu (5.04) is deprecated. Support has been discontinued long ago:

[https://lists.ubuntu.com/archives/ubuntu-security-announce/2006-October/000418.html](https://lists.ubuntu.com/archives/ubuntu-security-announce/2006-October/000418.html)

So basically you're running an **unpatched system.** You need to install the current version 7.10 Gutsy or wait for the new version 8.04 Hardy, which comes out of beta in a few weeks. 

Edit:

On second thought with that machine being 200 mhz/128 MB of ram I would not suggest **any** version of Ubuntu, but instead go with either Puppy Linux or Damn Small Linux (DSL):

[http://www.puppylinux.org](http://www.puppylinux.org)

[http://damnsmalllinux.org/](http://damnsmalllinux.org/)

but don't surf the internet with Hoary 5.04, please. It's a jungle out there.
