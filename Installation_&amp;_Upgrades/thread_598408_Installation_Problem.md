### Post by taurus on 2007-10-31
You can have XP and Ubuntu on the same machine with the same harddrive.  You just need to create two partitions from that same harddrive, one for XP and one the Ubuntu.  Then, install GRUB when you install Ubuntu so you can boot both OSes.
