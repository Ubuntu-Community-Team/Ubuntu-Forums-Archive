### Post by Sef on 2008-05-16
Easy way to install Sun java is this:

Applications > Add/Remove > Change "Show" to "All Available Applications" > search Sun Java 6 > Click on Sun Java 6 Runtime > Apply Changes > Apply > Close

Ubuntu does not use .rpm.  It uses .deb  To read about changing .rpm to .deb, read [How to Install Anything in Ubuntu]("http://monkeyblog.org/ubuntu/installing/").
