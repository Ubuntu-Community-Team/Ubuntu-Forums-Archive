Hi all
Just got a shiny new PC, and attempted to put 7.04 x64 on it in a dual boot configuration and the installer wouldn't have it! It failed when trying to resize the existing NTFS partition, telling me it couldn't continue as the partition would be too small.

Now, I've tried to boot a few tools I've got here to edit the partition directly but because I've got such new hardware none of them recognise it and hence I can't install Ubuntu without wiping the whole disk. Which I'm not prepared to do (yet...!)

Any ideas?

EDIT: sorted, just needed to run chkdsk /f on the windoze install before the ubuntu installer could get a hold of it. No messages told me that though, I just guessed.
