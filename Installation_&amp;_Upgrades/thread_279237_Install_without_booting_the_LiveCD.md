### Post by meng on 2006-10-17
Not the Server CD, the Alternate CD. The Server CD only installs server packages. The Alternate CD installs desktop packages, but using a text-based installer and without the live try-before-you-buy environment.

AFAIK there is no way to bypass the desktop using the Desktop CD. I have tried many times without success, and I have posted this answer many times without refutation.
