### Post by oldfred on 2010-04-18
You use the manual install procedure and you have to select which partitions to use for root (/), swap and optionally /home. 

You need to know your partitions by number. I had two spare partitions for future use labeled beta and data and of course selected the wrong one so my labels were wrong (easy to correct).
