### Post by tgalati4 on 2007-09-19
The purpose of a Live CD is to try Ubuntu without touching the harddisk.  You have to manually activate the "Install" icon to start the installation procedure.  It's a good idea to make a backup of data.  Ubuntu normally does a fine job squishing windows to make room for itself.  But sometimes users do strange things--like hitting the reboot button when things aren't going fast enough.
