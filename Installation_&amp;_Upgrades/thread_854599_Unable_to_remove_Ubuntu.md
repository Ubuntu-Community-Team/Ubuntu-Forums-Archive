### Post by jpeddicord on 2008-07-09
If you did, there will be a folder called "ubuntu" in C:.

See [https://wiki.ubuntu.com/WubiGuide#head-7cd5a1eda23f1e9960c28ef3a2f4e8645c5ea87d](https://wiki.ubuntu.com/WubiGuide#head-7cd5a1eda23f1e9960c28ef3a2f4e8645c5ea87d) for information on uninstalling, particularly the bit about C:\ubuntu\uninstall.exe.
