### Post by horseatingweeds2 on 2017-09-01
I found the issue. This Lenovo system has secure boot. I ran boot-repair and it told be to disable secure boot. After, boot-repair ran successfully and now my Windows garbage comes up on grub now. Sorry for the bother.

### Post by oldfred on 2017-09-01
Did you leave Windows fast start up on? Which is hibernation.
Grub only boots working Windows, or Windows that is not hibernated, nor needs chkdsk.

You should be able to directly boot Windows from UEFI boot menu, often f10 or f12 check your manual. Make sure fast start up is off. Windows updates, that you may not even see may turn fast start up back on.

       Fast Start up off (always on hibernation)
