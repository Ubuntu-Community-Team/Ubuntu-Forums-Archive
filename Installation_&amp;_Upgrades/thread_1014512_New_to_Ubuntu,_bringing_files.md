### Post by aheckler on 2008-12-17
If you know how to partition a disk you could theoretically do this:

[LIST=1]
[*]Shrink your Windows partition
[*]Create a new partition
[*]Move all your files to the new partition
[*]Install Ubuntu over top of the Windows partition only
[*]Move your files back over to the Ubuntu partition
[*]Remove the partition so Ubuntu uses the whole drive
[/LIST]
