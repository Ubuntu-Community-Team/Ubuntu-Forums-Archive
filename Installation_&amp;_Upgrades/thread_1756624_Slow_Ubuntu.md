### Post by irv on 2011-05-12
If you are using Unity or Gnome for a Desktop, try installing Xfce with is a lighter DE. This is a good way to speed up old machines. It is not as memory hungry. It is a little lighter on the graphics. It is worth giving it a try. Another good choice is Mint which uses Debian like Ubuntu but it's default DE is Xfce. Another light DE is LXDE. Go to distro watch and check out some of the distros for older machines and give them a try, you might find one that you really like. It all about choice.
