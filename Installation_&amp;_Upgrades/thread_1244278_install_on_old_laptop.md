### Post by Mighty_Joe on 2009-08-19
Locate a [download mirror]("http://www.ubuntu.com/getubuntu/downloadmirrors") close to you. They will usually have old versions.  
I would not expect an old version to run well on your hardware if the current version does not.  Ubuntu has always been a "kitchen sink" distro, trying to provide a stable and full-featured desktop. There are lighter versions of Ubuntu, like Xubuntu or CrunchBang, but they are still going to have a heavy footprint compared to Puppy.  You could also try to fudge a [ubuntu minimal install]("http://ubuntuforums.org/showthread.php?t=1155961"), but again, it's still going to be heavy.
Puppy Linux or [DSL]("http://www.damnsmalllinux.org/") are probably your best bet for running on old hardware as they are tailored for machines with minimal resources, as opposed to Ubuntu, which tries to be a feature-complete desktop for current hardware.

### Post by snowpine on 2009-08-19
Hi there, the oldest Ubuntu version that is still supported is 8.04... don't go any older than that. Some users actually feel that the current version (9.04) is the fastest yet.

You haven't given system specs for your laptop, so it's impossible to give a recommendation... but here are the minimum requirements for Ubuntu so you can make your own decision:

> Recommended minimum requirements

Ubuntu should run reasonably well on a computer with the following minimum hardware specification. However, features such as visual effects may not run smoothly.

    * 700 MHz x86 processor
    * 384 MB of system memory (RAM)
    * 8 GB of disk space
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection
