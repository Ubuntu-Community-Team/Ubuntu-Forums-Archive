After upgrading to Kubuntu 9.10, the laptop of a friend of mine takes a really long time to boot. On the boot screen, it displays the messages:

```

/: waiting for /dev/disk/by-uuid/<whatever>
```

After a while it works, but it takes a really long time. This didn't happen under 9.04. I can't tell you about any warnings or complications in the upgrade process, because I was not present for the upgrade and if any appeared my friend did not note them.
