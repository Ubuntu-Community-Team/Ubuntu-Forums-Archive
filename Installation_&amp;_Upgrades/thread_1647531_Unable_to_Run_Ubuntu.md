### Post by sikander3786 on 2010-12-17
Try the 'nomodeset' boot parameter.

Hit any key as soon as the computer starts booting from the disk and gain access to the boot options page. Press F6, enable nomodeset and continue boot.

[https://help.ubuntu.com/community/LiveCDBootOptions#Main%20Page%20Options](https://help.ubuntu.com/community/LiveCDBootOptions#Main%20Page%20Options)

If that doesn't help, from the same page, check your disc for defects.
