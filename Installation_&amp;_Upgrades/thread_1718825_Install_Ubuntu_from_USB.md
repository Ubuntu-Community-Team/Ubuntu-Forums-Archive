If you cannot boot from the CD and cannot boot from USB, your laptop OSs cannot be maintained for either Windows or Ubuntu.  Ideally a repair of the CD is preferred.

The only option other than replacing the CD is to remove the Hard Drive, install Ubuntu onto it from a separate PC, and returning the hard drive to the laptop.  This option carries some risks...
