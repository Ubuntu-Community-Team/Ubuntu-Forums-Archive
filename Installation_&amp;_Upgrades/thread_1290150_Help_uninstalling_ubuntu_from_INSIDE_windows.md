### Post by Mark Phelps on 2009-10-13
It doesn't come up in the partition manager app because it's NOT a partition -- it's a file, installed as an MS Windows application, that behaves like a partition to Ubuntu.  You have to remove it by uninstalling the application through Control Panel.
