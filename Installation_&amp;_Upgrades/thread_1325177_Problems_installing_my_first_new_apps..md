### Post by Kelvin Gardiner on 2009-11-13
Hi,

I hope your enjoying using Linux.

Normally in Ubuntu applications are installed from the Ubuntu repositories. You can think of this as a large storage place for apps that have been packaged for Ubuntu.

To solve your two problems:

I assume "Picas" was a typo and you want Picasa. Ubuntu comes with F-Spot photo manager. F-Spot is similar to Picasa and I believe it supports Picasa web albums.

To install a flash player. Open Ubuntu Software Center from the Ubuntu Menu and type flash in the search box. You should see a flash plugin listed, selected it and click install.
