### Post by rokclimb15 on 2009-10-30
I was able to resolve the issue of the install cd hanging at boot on my Dell Optiplex 360 with nvidia 9300.  I always typically have to boot the installers in safe graphics mode on this hardware, which I did have to do in this case.  But for 9.10 I also had to disconnect one of the monitors connected to my video card.

So, if you're having trouble getting past the black and white throbber graphic, try safe graphics mode with one monitor only.
