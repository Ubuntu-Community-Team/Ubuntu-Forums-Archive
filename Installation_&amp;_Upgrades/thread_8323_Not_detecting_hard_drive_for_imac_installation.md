### Post by Rob on 2004-12-16
A friend has bought an imac to put Ubuntu on. It is about four years old, 400MHz processor, about 13GB disk, and came with Mac OS 9.* installation disks. Before starting we verified it worked fine by starting up Mac OS.

We had a go at installing Ubuntu last night but failed. It went OK until the disk partitioning bit where it only gave us the option of partitioning manually. If we tried to do this it never gave the option of specifying how we wanted the disk to be partitioned.

It seems the hard drive is not being recognised. I've had a good search of the forums and wiki but I can't find anyone else having this as a problem. My friend is going to try putting a different drive in, but we'd welcome any other suggestions.

Thanks a lot,

Rob.

### Post by Rob on 2004-12-17
Putting another ide drive in solved the problem. Not sure why, as the original drive was definitely working before we started or we wouldn't have been able to boot up Mac OS.

My friend is going to try putting the original hard drive back in to see if it was as simple as a loose connection, although that seems fairly unlikely.
