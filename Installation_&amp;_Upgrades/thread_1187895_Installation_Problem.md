### Post by presence1960 on 2009-06-15
Did you try Check CD for defects when you boot it? Did you burn the CD at the slowest possible speed? If you did this then look here:

[https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

scroll down to the part describing F4 when you boot the Live CD. You may need to use safe graphics mode to install. 

If nothing on that link will work try the alternate text based installer. You can get it here : [http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)
