### Post by FooAtari on 2008-02-06
I just moved from Kubuntu to Ubuntu :)

I was having issues running Compiz in Kubuntu, but it worked fine in Ubuntu.  on both 6.10 and 7.10 I have persoanlly found Kubuntu to be slightly less stable than Ubuntu.

For wireless support I would recommend PCLinuxOS or Linux Mint.  I had wireless working on both with minimum fuss.  PCLinuxOS was the easiest to setup, I had an few issues with Mint but it was still relatively easy.  I needed a copy of the windows driver though.

I really like Mint.  It's a nice straight forward distribution, it's not bloated with multiple apps that do same thing, has a nice simple GUI and looks good.  While I'm not sure I would use it as my primary OS on the desktop it was perfect for my laptop which is mainly used by girlfriend for browsing the Internet.
