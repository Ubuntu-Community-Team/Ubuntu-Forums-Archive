### Post by Ensnared on 2006-04-23
The filename of the file you've downloaded makes me think you've not downloaded the LiveCD but a regular installation-CD. The LiveCD image should be named "ubuntu-5.10-live-i386.iso".

I also don't know why you've downloaded a rar-file, because as far as I can tell, all the official download-sources provide iso-files...

To download the proper image, go [here](http://www.ubuntu.com/download). Click on your region or your country (like Europe), and click the "PC (Intel x86) live CD" link under the "Live CD" heading.

Once downloaded, burn it to a CD with a program that supports iso-images (I guess most programs do nowadays). Do not burn the file as a file on the CD, but choose "Burn Image" or something like that from somewhere in the menus in your CD-brurner application.

It's possible that the file you have downloaded is actually an iso, but WinRAR is set up to handle files of that type, if WinRAR supports iso-files that is. But still, if you want a LiveCD you have to download a different image as the one you have is an installation-CD image and the two are not at all the same in 5.10 (e.g. Breezy - they will be in 6.06 Dapper, or so I'm told).

EDIT
Just found a link to [this guide](https://wiki.ubuntu.com/BurningIsoHowto) which deals with how to burn the image to a CD. It is linked from the page where you download the images.
