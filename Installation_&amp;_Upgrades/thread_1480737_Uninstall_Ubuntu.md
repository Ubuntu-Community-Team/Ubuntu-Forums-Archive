### Post by ki4anr on 2010-05-11
I assume you are talking about uninstalling Ubuntu from the hard disk. You cannot uninstall Ubuntu from inside Ubuntu. You can reformat the disk or install another distro from a CD-ROM and install it in the same partition you made for Ubuntu. Why might I ask are you wanting to uninstall? Do you want to install a Windows product instead? If that is the case just insert the Windows CD in the CD-ROM and follow the instructions to put Windows on the entire drive. That will overwrite Ubuntu completely.

If you were talking about uninstalling something like a program inside of Ubuntu, that is another topic altogether.

Let us know what you meant.
