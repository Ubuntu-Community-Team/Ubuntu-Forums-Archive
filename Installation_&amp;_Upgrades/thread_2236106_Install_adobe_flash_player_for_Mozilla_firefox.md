### Post by deadflowr on 2014-07-24
You should disregard that file and instead instead flash via ubuntu software center.

Ubuntu comes with a program called Software Center, open it and do a search for flash.
Adobe Flashplugin should appear, or something called flashplugin-installer.
Install that program, the one called flashplugin-installer.
What it will do is install a package which then get the flash plugin directly from Adobe and install it.
From then on, anytime a new version(albeit not really a new version, but anytime a fix comes out) you will get it with all of your other updates automatically.

Hope this helps.
