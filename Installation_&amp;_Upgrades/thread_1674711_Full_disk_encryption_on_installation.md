### Post by Herman on 2011-01-24
:D It works for me.

I recommend the fully encrypted file system type of installation for those who are interested in security, especially if the Ubuntu installation will be in a laptop, netbook or in a removable drive such as an external USB drive. Those are more likely to be lost or stolen than a desktop type of machine and without encryption, your security and privacy could be at risk.

You need to download the Ubuntu 'Alternate Installation' CD to install Ubuntu with a fully encrypted file system. :)
