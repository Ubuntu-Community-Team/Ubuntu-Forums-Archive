### Post by Herman on 2007-10-20
Hello xitdedragon,
I always do it that way, most of the time I install the new Ubuntu beside the older one and when I'm happy that the new system is ready, I transfer everything across from one partition to the other. I have big enough hard disks to do that. If you are using the new Gutsy Gibbon live CD to install with, it has a new migration assistant, that will take care of a lot of those things for you if you tell it to, if you are not deleting the other partition first.
We still need to have things backed up on some other media though, so even if you have a smaller hard drive, you should be able to copy your stuff to CDs or DVD disks or an external USB hard drive, or another computer, or something like that.

You shouldn't have any problem restoring into a newer version. It always works for me. 

Whatever else you'll need would be listed in my                                                       [Back Up and Restore]("http://users.bigpond.net.au/hermanzone/p13.htm") web page, that should help to remind you of most of the things you should need.

If you have extra software, be sure to try to think of any other files you might want to keep that I didn't mention in that page. For example, I use Password Manager, (from the Add/Remove Programs menu), so I looked for the file for that in my old Ubuntu, and just pasted it into my new installation, installed Password Manager, started it, and there were all my passwords! :)

Ubuntu is a great operating system because everything is so easy to do.

Regards, Herman :)
