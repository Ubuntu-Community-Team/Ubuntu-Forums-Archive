### Post by Lars Noodén on 2013-06-24
You can try both, though I would recommend Lubuntu.  To install both, install one from CD or DVD and then add the other one.  You'll find the packages *ubuntu-desktop* and *lubuntu-desktop* in the repository.  Add the one you need.   Then when you've decided on one, you can remove the other:

[http://www.psychocats.net/ubuntu/purelubuntu](http://www.psychocats.net/ubuntu/purelubuntu)
[http://www.psychocats.net/ubuntu/pureubuntu](http://www.psychocats.net/ubuntu/pureubuntu)
