### Post by gregoryigelmund on 2010-04-12
I've had a similar problem, so I started firefox in safe-mode:
```
firefox -safe-mode
```
I deactivated all addons and could start firefox without problems.

When I tried to enable step-by-step I recognized that firefox freezes again when I enabled the "Shockwave Flash Plugin" (10.0 r45).

So the freeze seems somehow related to that plugin and since I deactivated this everything is back to normal behaviour.


P.S. Maybe not too bad I cannot use the flash player anymore. +1 for productivity since you dont depend on using flash for you work! :)
