### Post by papibe on 2013-06-11
> **stephenb92 said:**
> What about the fact that my drivers are set to win 7 64bit only?
Driver management works a little different in Ubuntu/Linux.

Both proprietary drivers from Nvidia and AMD are packaged specially so they can be installed within Ubuntu itself. As a rule of thumb, installing a driver from the manufacturer's website would be left as a last resort.

During the installation you'll probably used the open source driver (don't worry about it), and then when you boot into your new install, look into the application 'Additional Drivers' to install the proprietary AMD drivers.

Let us know how it goes.
