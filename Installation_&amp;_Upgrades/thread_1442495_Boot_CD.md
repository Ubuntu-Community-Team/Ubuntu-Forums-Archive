You didn't choose the "Try Ubuntu ..." option FIRST to see if Ubuntu has any problems detecting your hardware and installing the correct drivers.

You should ALWAYS use that option first on a machine on which you have not previously installed Ubuntu.

Once you do that, you can report back with any problems you encounter and see what is involved in fixing them.
