### Post by Quackers on 2010-12-20
It seems that you have fallen victim to the "faults" in the new ubiquity installer. I'm afraid your Windows installation has been over-written by Ubuntu.
If you want to get it back you should stop using Ubuntu now! Boot up from the Ubuntu Live usb and select "try ubuntu" then when the desktop loads use testdisk to try and recover your system.
I'm not sure whether testdisk is installed by default on the Live usb. If it isn't go to System > Admin > synaptic package manager and type testdisk into the search box.When it comes up in the main window right-click it and select mark for installation, then click on the green tick to apply the change.
When it is installed you should go to the site below to get details on how to use it to recover data.
[http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step](http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step)
