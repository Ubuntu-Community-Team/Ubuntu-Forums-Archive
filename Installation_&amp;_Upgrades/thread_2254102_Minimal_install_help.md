### Post by sudodus on 2014-11-25
With mini.iso you need to download a large amount of data anyway. It is also called netboot, not because it boots via the network, but because it fetches almost the whole system to be installed via the network. There is one advantage: it fetches the current version of each package, so you need not download it twice (an old version in the iso file and a new version during updating/upgrading).

The trusty mini.iso works from USB as well as from CD/DVD.

See this link

[https://help.ubuntu.com/community/Lubuntu/Documentation/MinimalInstall](https://help.ubuntu.com/community/Lubuntu/Documentation/MinimalInstall)
