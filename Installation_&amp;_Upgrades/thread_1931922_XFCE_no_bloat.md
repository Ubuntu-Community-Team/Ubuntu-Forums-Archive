### Post by ajgreeny on 2012-02-26
If you installed **xubuntu-desktop** metapackage you would get everything that is included in xubuntu, including all those packages, games etc etc that you don't want.

However, I think that if you install the metapackage **xfce4** it will not bring in any games with it, just thunar file-manager, xfce panel, and all the libraries and utilities needed for the DE.
