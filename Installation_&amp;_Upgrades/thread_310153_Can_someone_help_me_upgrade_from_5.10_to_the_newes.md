### Post by engla on 2006-11-30
You first have to upgrade to the next release, and then the next after that. It should work fine with a new clean install. 

So first you have to update to 6.06 ("Dapper"):

[https://help.ubuntu.com/community/DapperUpgrades](https://help.ubuntu.com/community/DapperUpgrades)

Then you can (you don't have to!) upgrade to the latest release 6.10 ("Edgy"). It's not as stable, so some people would tell you to stay with 6.06.

But then, once upgraded to Dapper and restarted, follow:

[https://help.ubuntu.com/community/EdgyUpgrades](https://help.ubuntu.com/community/EdgyUpgrades)

### Post by Sef on 2006-11-30
Do you want to upgrade to Dapper, the latest Stable or Edgy, the latest unstable?  

Either way, the best way is to do a clean install: i.e. download and burn the one you want and install it over Breezy (5.10.)  

**Back up** your data first, in case there is a problem.
