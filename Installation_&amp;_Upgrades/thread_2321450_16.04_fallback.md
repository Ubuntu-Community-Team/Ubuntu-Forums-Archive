### Post by deadflowr on 2016-04-22
You mean like gnome-panel?
Yes, but only if you install it.

Like 14.04 there is no fallback by default.

14.04, by default, uses a different kind of fallback mode.
If a system cannot handle unity's full features, then it runs in low-graphics mode.
16.04 does the same thing.

If you want a different environment, then you need to install that environment.
