### Post by Rocket2DMn on 2008-10-17
If you want to install Gnome (the desktop environment that plain old Ubuntu uses), you don't have to make a new install, you just install the DE
```
sudo aptitude install ubuntu-desktop
```
Then log out, under Sessions choose Gnome, and log back in.
