### Post by lechien73 on 2010-07-20
Glade is available through the Ubuntu Software Center under the Applications menu. Just open it, type "glade" in the search box, and install the GTK+2 User Interface Builder. Or in a terminal window, type:

```
sudo apt-get install glade

```
