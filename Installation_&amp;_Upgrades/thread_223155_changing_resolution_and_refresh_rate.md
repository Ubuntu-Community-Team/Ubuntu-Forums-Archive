### Post by confused57 on 2006-08-10
I suppose he means opening and editing the file:

```
sudo gedit /etc/X11/xorg.conf
```

You might want to back it up before editing:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```

You can also edit this file in a more interactive way:
```
sudo dpkg-reconfigure xserver-xorg
```

Here's an excellent tutorial for configuring the xserver:
[http://users.bigpond.net.au/hermanzone/p7.html](http://users.bigpond.net.au/hermanzone/p7.html)
