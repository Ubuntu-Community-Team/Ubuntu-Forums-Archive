### Post by DFlame on 2008-10-21
I'd recommend the regular version of Ubuntu, unless you have a particularly low spec laptop. Would need more information on your hardware to make a definite recommendation.

That said, here's some system requirements: [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

OpenOffice can cover your needs for (obviously) office works, while GIMP can be used for graphic design. It is even possible to run Adobe Photoshop when used with WINE.

Ubuntu 8.04 would be best to use as it is a long term support release. Intrepid (which will be released towards the end of the month) will only be better if you'd like to try the latest and greatest.
