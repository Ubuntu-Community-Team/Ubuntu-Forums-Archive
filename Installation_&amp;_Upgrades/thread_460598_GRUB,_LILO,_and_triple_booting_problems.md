Have you read the link I gave you, using symlinking?...I used it to boot Zenwalk(which uses lilo) from my primary grub menu.lst.  You'll need to use the grub CLI to investigate where the vmlinuz is that you need to symlink to.

My Zenwalk entry was:
title   Zenwalk
root   (hd0,6)
kernel  /boot/vmlinuz-2.6.20 ro
initrd   /initrd.img
savedefault
boot

You should be able to investigate from Sabayon or using the Ubuntu live cd:
```
sudo grub
```
will get you to a grub prompt
```
find /vmlinuz
```
if you know which is your Ubuntu root partition:
```
null (hd0,6)/
```
then click on "Tab", if your vmlinuz is listed...you should be able to set up your symlink entry...if it isn't and there's a boot directory:
```
null (hd0,6)/boot/
```
click "Tab", if your vmlinuz image is listed, then you'll need the kernel line to be "/boot/vmlinuz..."

Herman's website(the link I listed) explains it very well.
