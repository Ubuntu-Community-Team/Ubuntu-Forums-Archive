### Post by paxsonsa on 2012-08-22
Hey man! Hows it going??
So for Ubuntu 12.04 LTS you need to grab the Alternate CD and use that to install.

Couple things.

When you boot and get to the install screen (e.g. it should say UBUNTU and then underneath Install Ubuntu) You will see down at the bottom are some options.
-Hit F4 and and choose the Command line System. 
**[COLOR="Red"]NOTE: THIS WILL ONLY INSTALL A COMMAND LINE SYSTEM NOT GRAPHICAL DE! IT WILL BOOT INTO A FULLSCREEN TERMINAL. [/COLOR]**
-Next if you want control the packages installed hit F6 and choose 'EXPERT MODE' then proceed with the installation. 

I had to install Ubuntu 12 this way for Graphics Card Reasons but it was nice having that control and plus you can build your own DE  and Gui which is nice to have choices versus the crappy Unity layout.
