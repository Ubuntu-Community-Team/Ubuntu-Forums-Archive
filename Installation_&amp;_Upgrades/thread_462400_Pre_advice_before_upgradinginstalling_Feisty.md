### Post by aysiu on 2007-06-02
It's almost always better to do a clean install (after [creating a separate /home partition](http://www.psychocats.net/ubuntu/separatehome) first).

Upgrades can be tricky. If you have a working installation right now [back it up](http://www.psychocats.net/ubuntu/backup) before attempting an upgrade... or even a reinstall.
