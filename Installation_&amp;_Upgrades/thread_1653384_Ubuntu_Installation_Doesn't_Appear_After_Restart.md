### Post by Quackers on 2010-12-26
BIOS (which stands for Basic Input Output System ) is the first thing to wake up when you start your computer. It picks up the time and sets up all your hardware so that the operating system can speak to it. It also recognises all available hard drives and other boot devices (like cd/dvd drive and usb flash drives, for example).
The bios needs to be told where to look first for a boot device. Normally this would be set to the hard drive, but to boot from a cd before the hard drive you would need to set the cd drive as the first boot device.
To do this you need to enter the bios settings in your computer. This is done on a one time only basis by tapping a key during boot up. The key can vary with different manufacturers. It is often F12 or the "del" key. You would need to check your manual for info.
To permanently change the bios boot device you would press a different key during boot and enter the bios that way.
Your manual will tell you which key does which job.
