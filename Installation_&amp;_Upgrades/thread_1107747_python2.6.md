### Post by snova on 2009-03-27
> **jerzyo said:**
> Why the latest python version of python in ubuntu is 2.5 where the re are also 2.6, 3.0 and 3.1

Ubuntu is not a rolling release distro; versions of packages remain the same for the most part throughout a release. Especially for something as important as Python, breaking things would be a major problem.

As cariboo907 said, Python 2.6 will be the default in Jaunty. 3.0, however, will probably not be the default in any distro for a considerable time, though it is available.

> The same problem I have with eclipse (in packages 6.0 whereas in reality 8.0)
