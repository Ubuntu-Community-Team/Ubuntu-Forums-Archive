### Post by cotcot on 2006-04-02
There is nothing to install. Just download and unpack.
I got blender as follows :

 - Go to the blender website : [http://blender.org/cms/Blender.31.0.html](http://blender.org/cms/Blender.31.0.html)
 - Download blender for i386 python 2.4 static
 - Unpack the tar
 - in terminal : "blender" and it should start (with "blender -w" you avoid that blender takes the full screen; so keep the task bar)
