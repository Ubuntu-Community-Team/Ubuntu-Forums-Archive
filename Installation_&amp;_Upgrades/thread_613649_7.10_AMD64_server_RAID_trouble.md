### Post by siouzi on 2007-11-28
nVidia RAID = just software in the BIOS (or something like that) meaning it's a FAKERAID, not a real fully independent hardware raid controller. Use the installer to set up software raid. For RAID1 (mirror), see for example this guide [http://users.piuha.net/martti/comp/ubuntu/en/raid.html]("http://users.piuha.net/martti/comp/ubuntu/en/raid.html").

If it was a real raid and the raid hardware failed, the raid array would most likely be unrecoverable without the same exact controller hardware!
