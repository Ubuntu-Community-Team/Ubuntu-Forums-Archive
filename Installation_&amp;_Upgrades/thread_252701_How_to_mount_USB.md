### Post by cotcot on 2006-09-07
USB divices are normally automatically detected. 
Gnome-volume-manager is the program doing this. Check (synaptic) if you have it running.
To get USB devices recognised first boot the computer and than plug in the USB tool. If you want to unplug the device first unmount (right click on the item and select umount) it before taking it out.
