### Post by bren on 2007-08-15
boot from the LIVE CD 

open a terminal window (if you are using the standard CD this is via the menu APPLICATIONS-->ACCESORIES-->TERMINAL and type the commnd 

lshw -short

paste the results here

the most likely thing is that your hardware has a prob or the CD has a problem

did you get to the main menu when you boot from LIVE CD where there is an option to test the CD?

did you do it?

did you burn the CD at slow speed?

did you check the CD MD5Sum after burning?
