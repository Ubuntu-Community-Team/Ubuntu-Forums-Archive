### Post by Tylerjd on 2010-04-05
Directly copying a system is easy. All you need is an Ubuntu Live CD, and and external HDD/ big flashdrive. Start up from live CD, use GParted (**System>Administration>Gparted**). Copy hda or sda (whatever is your internal HDD) to USB HDD. Do reverse to put on new machine. I'm not sure about copying from Toshiba to dell or whatever, if it will detect the right drivers, esp if you have made changes tot he kernal. I wouldn't recommend it.
