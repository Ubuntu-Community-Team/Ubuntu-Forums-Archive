### Post by sikander3786 on 2010-12-09
Welcome to the forums :-)

We need some more info. If we try to fix your existing install, we definitely need you to boot an Ubuntu Live CD/USB to perform some commands.

As the USB stick is not either booting, we need to know the errors encountered there. How was the USB created?

First, check the MD5SUM of image used.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If ok, you can use unetbootin to burn it to USB disc.

[http://unetbootin.sourceforge.net](http://unetbootin.sourceforge.net)

Once completed, make sure you are booting from the USB drive and that need to be configured in Bios menu ;-)
