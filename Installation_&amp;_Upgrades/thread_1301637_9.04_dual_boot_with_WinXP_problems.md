### Post by dstew on 2009-10-26
I don't have any advice for you about getting XP to connect to the internet. 

As for the Ubuntu resolution problem, after you get a useable display, you can use the graphical interface tool **displayconfig-gtk** to improve the picture. To access it, hit Alt-F2 to get a command line, and enter```
gksudo displayconfig-gtk
```You can enter you monitor type, and then try some of the resolutions.
