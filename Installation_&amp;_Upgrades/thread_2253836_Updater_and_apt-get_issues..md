### Post by ian-weisser on 2014-11-22
Open a terminal.

Try the following commands, in order.
Your system will prompt you for a password. Enter your password - it WON'T show on the screen, but it's okay: The system is still receiving the keystrokes.
Please copy-and-paste the entire output here.
```
sudo apt-get update
sudo apt-get upgrade
```
