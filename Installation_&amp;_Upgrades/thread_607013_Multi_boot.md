**If** you want to manage Grub from Mandriva that is..
You can let Ubuntu install the bootloader and restore your Grub-setup from Madriva with [Super Grub Disk]("http://supergrub.forjamari.linex.org/").
It's better to let Ubuntu install Grub.. because when you restored your "Mandrive-Grub" you can simply point to the /boot/grub/menu.lst generated by Ubuntu to create a nested menu.
How to.. [http://users.bigpond.net.au/hermanzone/p15.htm#1._Configfile]("http://users.bigpond.net.au/hermanzone/p15.htm#1._Configfile")
