### Post by la3875 on 2008-04-13
Ubuntu ROCKS!!! I can't believe the flexibility of the OS! Here's what I did - I've been using Ubuntu since I first read about 5.10 in of all places PC World. Regardless I was intrigued and set out to install as a dual-boot so as not to disrupt the family computing harmony. Recently the 'ol PIII-933 has been feeling a bit old and I acquired a P4-2GHz machine and decided to move the family to the new-old box. I took the Ubuntu drive out and installed it in the new box which in addition to having twice the processor, it also had twice the RAM and an Nvidia video card vs. the ATI in the 933. Well, after boot up the system noted the new video card and asked if I wanted to install the restricted driver. I did and about 20 minutes later, I'm on the new machine like nothing happened! Try that with Window$!

Bottom line... This OS and community ROCKS!!! Keep up the great work all.:guitar:
