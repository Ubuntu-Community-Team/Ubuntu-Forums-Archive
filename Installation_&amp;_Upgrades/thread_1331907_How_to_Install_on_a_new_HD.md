### Post by phillw on 2009-11-19
Hi,

Boot from the CD, choose install, choose use entire disk and let it get on with it.

If it is the only HD then just slice a bit of swap space for it - if it is dual HD system - make sure you tell ubuntu which disk to use !!

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)
