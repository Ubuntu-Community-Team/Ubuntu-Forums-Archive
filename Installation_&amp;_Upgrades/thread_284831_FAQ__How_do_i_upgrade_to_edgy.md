**Current Edgy users:**
If you've been running edgy since the RC or knot 3 etc. then as long as you have dist-upgraded, you are now running Final and have no reason to do anything more.

"I am scared to crash my dapper box if i upgrade to edgy"
Just create a ghost image of your ubuntu partition and restore it if you're not fine with edgy. A common tool for that is partimage :
[http://www.partimage.org/Main_Page](http://www.partimage.org/Main_Page)
