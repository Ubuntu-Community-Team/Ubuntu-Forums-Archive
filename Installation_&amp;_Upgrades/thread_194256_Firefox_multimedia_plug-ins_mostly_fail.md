### Post by kornelix on 2006-06-11
In my new  installation of Ubuntu 6.06, I cannot get many web video formats to play. 

I tried the Ubuntu desktop guide first, then Easyubuntu, then Automatix. Results seem to be about the same: many video formats in web pages do not work: the plug-in is missing, refuses to start, starts and disappears, or displays distorted output. I also tried the RealPlayer install procedure from Adobe (unpack tar file and run install program) but this made no difference. 

For a variety of video formats (and a few laughs) try [www.boreme.com](www.boreme.com).

For my collection of various video files, my experience is that Xine works best and that gxine, totem, mplayer, and VLC have problems with one format or another. One shockwave file will play in firefox but not with any other engine.

So far Ubuntu 6.06 is not working as smoothly as my prior Fedora 5 system. I changed horses because of Ubuntu's reputation and the availability of more complete and professional documentation. 

I hope these problems will get sorted out over the next weeks or months or years. In the mean time, any advice would be appreciated.
