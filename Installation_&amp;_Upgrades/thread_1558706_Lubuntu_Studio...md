### Post by -kg- on 2010-08-22
Yes, probably, as long as you have sufficient system resources (as in, why are you using Lubuntu?).  Ubuntu Studio is pretty much compiled for the Gnome Desktop Environment (GDE), and undoubtedly you will have to install some additional libraries and such to be able to run some of the software.

For instance, I am running GDE but use K3B for burning, which is based on KDE (Kubuntu).  In order to use it, I also have to download certain libraries and software used in KDE.

I have no doubt that you will have to download (and run) some GDE libraries and software in order to use certain aspects of Studio.  The link you provided was based on a normal Ubuntu (GDE) installation.  As long as you have the resources it will work, but if you're running Lubuntu because of limited resources, then you might want to check it out and make sure it will run well enough.

Of course, I don't know...if similar software packages are available for LXDE, then you'll be good to go.
