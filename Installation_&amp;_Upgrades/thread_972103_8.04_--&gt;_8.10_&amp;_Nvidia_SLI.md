### Post by ChaOConnor on 2008-11-06
Well I tested it out last night and it worked!  Here's what I did:
- Install Ubuntu 8.04
- Install EnvyNG
- Have EnvyNG Install the latest Nvidia Drivers & reboot
- Run all the applicable updates for 8.04
- Insert the 8.10 Alternate Install CD, run the upgrade having it download the latest updates from the internet.

And my video card settings took!  Rebooted into 8.10 and everything was working like it was in 8.04!  I realize this is only helpful for those wishing to do a clean install of 8.10, but in my case it was more academic and it worked.
