### Post by presence1960 on 2009-07-21
Boot the live CD and hit F4 and choose safe graphics mode. Then continue with the install. If that doesn't work then try the alternate text based installer from here: [http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)

see here: [https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions) for instructions for safe graphic mode and other boot options for the Live CD
