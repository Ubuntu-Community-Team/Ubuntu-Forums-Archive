### Post by foresthill on 2011-05-21
The system is an Asus A7V8X-X motherboard, with an Athlon XP 2400 processor, 512 Mb's of RAM and an Nvidia GeForce Ti 200 graphics card.

It's definitely an "older" system, built around the year 2000 or so. 

Install went flawlessly, though Unity did not work (fine with me). I updated the system with 138 Mb's of updates, and when I restarted, I was offered a non-proprietary driver for my legacy Nvidia card, which I installed, and it seems to be working fine.

Please note though, that this the 3rd system I have attempted to install 11.04 on, and it's the only one that's worked. The other two were my Gateway laptop and my Pentium D desktop with an Nvidia 7950 graphics card. 

So I just wanted to let people know that it is indeed possible to install 11.04. I had almost given up hope.
