For this reason of better compatibility, I'm thinking of installing the Netbook Edition. The problem for me is my current data stored on Ubuntu partition that I have no way to backup. Here my questions comes? Is there any way to upgrade to the remix edition without (at least theoretically)removing this data? If I do it from PPA, which is "untrusted" should it install safely? Then what should I do to get rid of gnome? remove ubuntu-desktop?

### Post by sikander3786 on 2010-09-08
Install the package on your existing install. That will save you all the hard work of backing up and restoring your data. Ubuntu Netbook package is in the repositories of 10.04. It is not the untrusted ppa I believe.

```

sudo apt-get install ubuntu-netbook

```

That should be absolutely fine to install ubuntu-netbook on Ubuntu desktop. No worries. Install it, try it and if it works remove ubuntu-desktop by using the following command.
