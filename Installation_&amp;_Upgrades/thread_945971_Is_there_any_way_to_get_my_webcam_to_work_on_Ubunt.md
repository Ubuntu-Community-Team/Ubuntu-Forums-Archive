### Post by Sef on 2008-10-14
If you use only msn, then download amsn from Add/Remove.  It supports webcams.   Applications > Add/Remove > SHOW: change to All Available Applications > SEARCH: type in amsn > check on box next to amsn > apply changes > apply > close

If you use another im, then download Kopete from Add/Remove.
