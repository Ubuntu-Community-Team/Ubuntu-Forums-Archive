Hi nmlferreira.

My guess it is a problem with Firefox settings (as described in this [thread]("http://ubuntuforums.org/showthread.php?t=1948114&highlight=magnet")).

You can manually load a magnet link on Transmission this way:
[LIST]
[*]Right click on link containing the magnet link.
[*]Select 'Copy Link Location'.
[*]Open transmission and go to File -> Add URL.
[*]Paste the link.
[/LIST]
Hope that helps,
