### Post by Larkspur on 2011-06-20
> **rccharles said:**
> Could not find Windows hd.  How do I list all storage media. Where is the Mac OS Finder/Windows explorer?

Either click the top icon on the launcher bar or hold Super+1.  The file browser will open; on the left hand side will be a list which will have the windows hd in it.  Click to mount it.

> **rccharles said:**
> Could not find Terminal.   Ran search, but seems limited to just your user data.  How do I do a search for the terminal applications.

Press the Super key and start typing "terminal".  Press enter or click on the icon when it appears.


> **rccharles said:**
> Is there a way of switching the old User Interface from the running from CD version?


Robert

Press Alt+F2.  Type "gnome-panel".  A new panel with a conventional menu appears.
