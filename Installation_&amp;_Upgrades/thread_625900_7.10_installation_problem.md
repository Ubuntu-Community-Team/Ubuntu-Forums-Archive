### Post by tinycamp on 2007-11-28
is your hdd ide or sata?

when booting grub, go to the menu and modify your boot command, remove the "quiet" parameter, like this:

```

#kernel         /boot/vmlinuz-2.6.22-14-generic root=/dev/sda1 ro quiet splash
kernel         /boot/vmlinuz-2.6.22-14-generic root=/dev/sda1 ro splash
```

when you boot this, you'll see the status of the boot process, check if its not staying on "waiting for filesystem"

this happens sometimes with sata drives
