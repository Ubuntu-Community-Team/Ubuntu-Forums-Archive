### Post by oldfred on 2010-02-14
I had my home inside my 32 bit install for 3 years. I followed psychocats instructions on creating a separate /home while 32 bit and installed the 64 bit version with new home I had just created. I have not seen any issues. 
I also created a separate /data partition and moved all my data from my old install there and now home is very small ~1GB. Firefox and Thunderbird profiles normally in home are also in another shared NTFS partition for the occasional boot into windows.
