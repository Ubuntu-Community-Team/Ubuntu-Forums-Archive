### Post by mikewhatever on 2014-08-08
What security features do you need?  In other words, "make secure" from what? 
Note that the stuff you've listed - DNScrypt, VPN, Mac changer and ccleaner - have nothing to do with security. They are related to privacy, which isn't the same.
Ubuntu doesn't come preconfigured with any of that, and, IMHO, there is no reason to just pile them up.

### Post by ajgreeny on 2014-08-08
[https://help.ubuntu.com/community/Security](https://help.ubuntu.com/community/Security)
should tell you all you need to know (and quite a lot that probably need not bother you).

You need to forget the old windows security model and requirements; they simply are not needed on a standard desktop system running Ubuntu, and as with any OS the biggest risk is the person using the keyboard and mouse.
