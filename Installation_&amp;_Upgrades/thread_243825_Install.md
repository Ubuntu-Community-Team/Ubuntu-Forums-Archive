### Post by cleentrax on 2006-08-25
I've used both the live CD and the alternate install several times. The Live CD actually seems faster, and you can use it for browsing, etc., while the install happens in the background.

However, the alternate install CD does give you additional options, like installing server mode, or setting up a RAID.

### Post by az on 2006-08-25
> **sexualpotatoes said:**
> How do I install without booting into the live? what are the boot options?

The boot options are listed on the bottom of the screen.  Press the F keys.

Unless you have a specific reason to not use the Desktop cd, you should use it.  It is much faster to boot into the live session and then install using Ubuquity rather than the traditional installer.  

The live cd installs in less than fifteen minutes on a modern machine.  It takes just under an hour to do the same thing with the alternate cd.
