### Post by presence1960 on 2011-02-14
Quite a few BIOS "see" a bootable USB as hard disk and lists them in BIOS under hard disk. With the USB plugged in, boot and go into BIOS and check under hard disk. Set the USB to boot before the hard disk, that is if it is indeed listed under hard disk. I have a Biostar mobo and that is what it does for USBs.
