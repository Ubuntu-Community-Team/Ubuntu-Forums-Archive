### Post by Rocket2DMn on 2008-12-10
I merged your posts, please don't start multiple threads on the same topic.
If you are having problems with an install cd, you should go back and check the md5sum on the .iso file
[uhelp]community/HowToMD5SUM[/uhelp] - how to check the md5sum
[uhelp]community/UbuntuHashes[/uhelp] - hashes to compare against
If the md5sums don't match, then you need to try redownloading the .iso (perhaps use a torrent, these tend to be more fail proof since sums are checked as the file downloads).

Here is a guide for burning if you need help with that at all - it is recommended you burn at a slow speed like 4x to help prevent write errors
[uhelp]community/BurningIsoHowto[/uhelp]
