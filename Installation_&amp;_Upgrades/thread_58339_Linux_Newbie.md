### Post by blastus on 2005-08-19
How old is the computer? Really really old computers were not able to boot from the CD but anything less than 6 years old should be able to boot from the CD. I even have a Pentium 200 and I'm quite sure that it can boot from a CD because I've installed Fedora Core on it and don't remember using floppy disks. I have another machine from 1999 that boots from the CD just fine also. 

You need to go into the BIOS when you boot up and change the boot order. It may be different on your machine but with every machine I've owned, the DEL key goes into the BIOS. On newer computers you have to be quick on the draw because they boot so fast but on older computers it's not a problem.

One thing you may want to check is if your CD-ROM drive can read CD-Rs and CR-RWs. If it can't or has difficulty reading them, you may not be able to boot from the CD drive. This was the case with my PII 350, the stupid thing wouldn't boot from a CD-R but it was able to read them most of the time without errors. Once I put in a brand new CD-ROM drive, it could magically boot from a CD.

Compatibility across CD drives and CD media has been really crappy except for the last couple of years. Burners *usually* have no trouble reading CD-Rs and CD-RWs they themselves burn. Like compatibility across CD-ROM drives, so too compatibility across CD burners has been really crappy except for the last couple of years. For example, I can burn a CD-RW on my old burner from 1999 and my new burner from 2003 can read it but it can't overwrite it!!! :mad:
