### Post by kc1di on 2013-02-17
Hello Threaljoker, 

if your machine is capable of 64 bit then the windows ubuntu installer should automatically install the 64 bit version However if this is a newer machine that has UEFI secure boot it is recommend that you do not use the WUBI installer but install ubuntu to it's own partition.

here are a couple pages that will be of help to you.
[https://wiki.ubuntu.com/WubiGuide]("https://wiki.ubuntu.com/WubiGuide")

### Post by uaebuntu on 2013-02-18
Minecraft works perfectly well on Linux, probably better than on equivalent machine with Windows 7.

It's fairly easy to install under linux, you just need to change a couple of graphics libraries. I run it with OPENJDK 7 java, though it works just as well with Oracle Java 7.

Have proper icons in the Unity menu bar. 

Most of the instructions for MC are for the windows version, and there are differences for adding mods and texture packs.

The instructions are all on uTube.
