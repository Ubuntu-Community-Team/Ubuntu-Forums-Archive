### Post by Brandon Williams on 2009-05-20
To install netbook remix, simply run 'sudo aptitude install ubuntu-netbook-remix'. Before switching to the UNR interface, [Read here](http://www.ubuntumini.com/2009/04/desktop-swticher-is-kind-of-broken.html) about the buggy desktop-switcher, and to find a link to use for downloading an updated package that you should install before using the desktop-switcher.
