### Post by everettattebury on 2009-05-15
I'm not certain this is what you're asking, but try this:

Run gconf-editor and look under Desktop->Gnome->Lockdown and uncheck the box next to "disable_user_switching".  Then replace the logout applet in your panel with the applet called "User Switcher".  

If you don't see that applet in the list, you will need to install the package "fast-user-switch-applet".
