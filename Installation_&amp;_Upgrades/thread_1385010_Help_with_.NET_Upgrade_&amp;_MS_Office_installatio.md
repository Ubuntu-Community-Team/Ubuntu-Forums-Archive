### Post by .nedberg on 2010-01-19
I don't think MS Office will run in Wine.

I see three solutions:

1. Convert your resume into something OpenOffice can read (either manually or automatic)
2. Install Windows in a virtual machine (or dual boot) and use MS Office there
3. Use a friends computer with MS Office on it

I would go for 1. It is a bit of work, but should not take to much time. You should be able to get the text from the old file, just add the formating and save as .odt. Or if you know someone with MS Office, use their computer to convert the file into .doc or preferably .odt (MS Office will write .odt with a plug in from Sun).
