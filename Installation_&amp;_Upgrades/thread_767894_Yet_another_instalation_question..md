### Post by Lord Landis on 2008-04-26
Are you running Windows, or is this in Ubuntu?

If you're in Windows...  Did you download the ISO?  If so, you either don't have CD burning software that will support burning an ISO to disc or you haven't configured it in the default program list.  Anyway, you'll need to run some burning software that can burn an ISO.  The latest versions of most software will do this.  I recommend using Nero since even the free version will work for what you need.

If you're already in Ubunutu, you should be able to burn it successfully from either K3B or GnomeBaker.
