### Post by ajgreeny on 2014-05-05
It's a long time since I've heard of anyone still using a Win98 machine or pentium 3 CPU.  However, if you are going to get anything from the ubuntu family of OSs to boot on that it will probably be no more resource hogging than Lubuntu, or if you want to use something based on but not part of the ubuntu family, try Bodhi (very different DE, but still uses ubuntu repos), or maybe Crunchbang, based on Debian not ubuntu, but still therefore doing a lot of things, eg package management, the same way as ubuntu.

I have no real idea of how fast, or maybe I should say slow, the system will be when using a P3 CPU, so it will be interesting to hear how you get on.
