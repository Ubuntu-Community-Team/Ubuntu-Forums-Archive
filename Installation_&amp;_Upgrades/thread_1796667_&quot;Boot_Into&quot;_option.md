### Post by Mark Phelps on 2011-07-04
Not sure what you're asking, but booting into EITHER OS is going to require shutting down the other OS first.

Either Linux or Windows will provide you the feature of adding OS entries to their boot menu -- although Linux is generally easier to use.

Next time you're in Ubuntu, open a terminal and enter "sudo update grub".  That will regenerate the GRUB menu for you.

When you reboot, you should then see a selection menu.
