### Post by cariboo on 2010-10-20
Both the restricted drivers and the open source drivers work well in Maverick. If you haven't installed any additional drivers, you are already using the open source nouveau drivers, to get 3D just install libgl1-mesa-dri-experimental, it's in the repositories.

If you want to install the restricted drivers, go to System->Administration->Additional Drivers.
