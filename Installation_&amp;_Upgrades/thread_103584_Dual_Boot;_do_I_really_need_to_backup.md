### Post by localzuk on 2005-12-14
Yes. it does happen. I have seen a lot of customers lose data through lack of backups and then trying something like installing linux. Most of the time it is due to selecting the wrong option in the partitioning stages - and everyone occasionally does that.

So the advice to back up stands. You don't have to back up everything, just documents and music etc... Unless you use something like 'Ghost' to create a disc image.

### Post by psusi on 2005-12-14
You should ask yourself if you can live with the data loss.  Would it only be annoying if you lost the data, or can you not live without it?

If you could live with the data loss, and the risk of that happening is small, then you may choose to skip the backup.  If you can not live without that data, then absolutely you should back it up.
