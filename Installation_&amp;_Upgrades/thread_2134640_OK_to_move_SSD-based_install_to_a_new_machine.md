### Post by darkod on 2013-04-12
In general it should be fine. I recently replaced everything on my desktop except the ssd and it worked right away.

Depending on the graphics you might have issues with video driver, but apart from that there should be no real problems. Investigate in advance whether you need some driver or special procedure about the graphics in the new machine.
