### Post by Mark Phelps on 2010-03-31
If you really need to use MS Office 2007 every day, Wine is not going to be a good solution for you.  If you look through the WineHQ links for the individual Office components, you will see them rate Silver -- meaning, that only some of the features work.

If you're lucky, those features that work are the ones you need; if not, you'll find the utility limited.
