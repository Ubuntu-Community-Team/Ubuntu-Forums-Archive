### Post by qamelian on 2006-10-30
Dapper is meant to be the current "workhorse" version of Ubuntu. If you want the tried and true stable version, then Dapper is what you should use. Dapper is meant for getting day to day work done.   Edgy is the beginning of a new development cycle that will result in a new long-term support release (sort of Dapper: The Next Generation). While Edgy might be perfectly stable and usable for you, it potentially contains some new bits that might make you life interesting in some very unpleasant ways.   Personally, I have my production desktop machine running Dapper because I don't need bright, shiny, new toys on it. I just want to get work done. On my laptop. which exist solely for insane experimentation, I'm running Edgy. For me, Edgy has been just as stable as Dapper and has resolved a couple of issues I had with Dapper. This hasn't been the case with everyone.  Just be sure to do some homework on both versions before making a decision.
