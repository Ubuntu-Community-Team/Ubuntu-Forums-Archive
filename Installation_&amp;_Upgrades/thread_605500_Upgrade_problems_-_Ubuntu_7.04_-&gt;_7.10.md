### Post by fish2ways on 2007-11-07
My advice would be to download/burn a Gutsy install CD. Boot into that, copy and save your personal files from Fiesty. Then do a fresh install of Gutsy over Fiesty. That's how I did it and it went without a hitch.
The cause of your upgrade problem will probably take longer to find and resolve.
My own personal experience of both Linux and Windows is never upgrade, backup then do a fresh clean install of the new operating system. It saves a lot of grief in the long run.
