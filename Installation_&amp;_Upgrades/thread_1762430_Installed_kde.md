### Post by wandalalakers on 2011-05-19
Right click on an empty space on the task bar,
You will see:

Task Manager Settings
Unlock Widgets
or
Task Manager Settings
Panel Options
Remove this Task Manager

If you see Task Manager Settings and Unlock Widgets,
select Unlock Widgets.
Right click the task bar again.
Click panel options, panel settings
Put your mouse cursor on the word "Height" and drag the bar up to increase the icon size or drag it down to decrease them.
Click the red x to the right of the panel bar when you are done.
I also use the panel settings option to move application icons when I want them.

If you see Task Manager Settings, Panel Options and Remove this Task Manager
Click panel options, panel settings
Put your mouse cursor on the word "Height" and drag the bar up to increase the icon size or drag it down to decrease them.
