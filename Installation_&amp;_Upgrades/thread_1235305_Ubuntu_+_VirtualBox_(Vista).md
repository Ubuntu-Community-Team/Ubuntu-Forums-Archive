### Post by RedSingularity on 2009-08-09
If you are dual booting you can keep a portion of it NTFS.  If you are running vista inside ubuntu via virtualbox you can format to a EXT3 system.  Of course you need the Vista install CD to install it in virtualbox.  If you dont have the Windows CD you will be better off dual booting.

### Post by presence1960 on 2009-08-09
If you have a recovery partition I would back that up prior to doing anything to your hard disk. This way if need be you can restore back to factory settings. Most manufacturers give you a one time option of burning to CD/DVD the recovery partition. This will give you bootable media to restore your machine to it's factory image.
