### Post by Partyboi2 on 2009-03-10
To change the login name you could do
```
sudo usermod -l new old
```Replace "new" with the actually new login name you want to use and replace "old" with your current login name.
For example if I was going to change PartyBoi2 login name  to Ubuntu it would look like
```
sudo usermod -l Ubuntu PartyBoi2
```
