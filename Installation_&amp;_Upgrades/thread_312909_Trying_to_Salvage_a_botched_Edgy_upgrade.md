### Post by bapoumba on 2006-12-05
Hi,
if you dl [alternate edgy]("http://www.gtlib.gatech.edu/pub/ubuntu-releases/edgy/"), you can install it on your root (/) partition. When you get to partitioning, choose the manual one, and set up you root on whichever hd? it currently is.

Do not forget to set up your /home partition, ie select on which /hd? it is but NOT FORMAT it.
After you're done with the installation process, you'll have the same /home, with all your goodies in there.
