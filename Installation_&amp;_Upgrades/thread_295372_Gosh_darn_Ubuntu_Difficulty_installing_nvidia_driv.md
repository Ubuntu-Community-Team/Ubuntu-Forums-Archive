### Post by tvst on 2006-11-08
The drivers on the NVidia website need to be compiled.  You may use Automatix for a quick and easy way to install the drivers (and future-proof, since otherwise you'd have to repeat the procedure below each time you update your kernel), but if you want to compile them instead, here's what to do:

Have you ever compiled anything on your computer? That is, do you have the package build-essential? You'll need it. You'll also need you kernel headers. To install these, just copy and paste this line into a terminal:

sudo apt-get install build-essential linux-headers-`uname -r`

(the "uname -r" part (in back-quotes) is automatically substituted for the correct linux version)

I think that's all you need.

You can now run the NVidia install program, which should automatically compile and install the drivers.

### Post by K.Mandla on 2006-11-08
I've touched up the title of your thread so it better reflects the problem you're having, and also to better conform to our forum guidelines. ;)

The easiest way to install the nvidia drivers is to make sure your [universe and multiverse repositories are enabled]("https://help.ubuntu.com/community/Repositories/Ubuntu"), and install nvidia-glx through Synaptic, or with *sudo aptitude install nvidia-glx*.

If you use an older video card (around GeForce4 440 MX or earlier), you might need the *nvidia-glx-legacy *instead. I forget where the dividing line is; the best way to find out is to see which one works best. There's more information [here]("https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia").
