### Post by Mark Phelps on 2011-04-26
Not to be discouraging, but if this is an older PC, you just may not be able to upgrade anymore ...

Case in point ... I have a 5-year old Fujitsu Tablet PC.  Installed 8.04 on it, customized it -- worked great!  It came with XP on it, and Hardy worked so well that I switched over to using Ubuntu full-time.

Every single Ubuntu version since then has FAILED to work properly on this Tablet.  Every single one!

In stark contrast, both Vista, and later, Win7, installed OK on it, found all the hardware, installed all the drivers, and they worked -- great!

And now, since 8.04 has reached end-of-life, so has Ubuntu on my Tablet PC.

So, you may just NOT be able to run anything newer on it. Sorry...
