### Post by ajgreeny on 2011-02-12
Unfortunately I need a translation, but I think you may be able to reinstall gksu etc etc using recovery mode, so boot into that using the grub menu, holding shift at boot time if you don't normally see grub menu at bootup.  Choose the command line with networking, then run command ```
apt-get install gksu
``` no need for **sudo** as you're already root in recovery mode.  You can install anything else at the same time this way if needed, eg sudo, dpkg etc etc.

Good luck.  Come back again if that is not successful.
