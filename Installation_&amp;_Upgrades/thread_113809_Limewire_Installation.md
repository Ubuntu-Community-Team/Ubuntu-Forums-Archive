### Post by cstudent on 2006-01-07
If it starts to load and then stalls during the splash screen it's probably your version of Java. Do you have the Sun jre1.5 version installed? That seems to work best. If you get absolutely nothing try to launch it in a konsole window. I think it should be  /usr/bin.runLime.sh that will start the program. See if you get any error messages when you launch it this way and if so, post them here.
