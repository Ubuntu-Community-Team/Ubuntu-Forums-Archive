### Post by 2hot6ft2 on 2010-02-25
First you can't install "everything" that is in synaptic for the simple reason that there would be conflicts. For example you can't have both wicd and network-manager installed at the same time. Graphics drivers would be another example.

As for the size of the repos. I think it's only something like 50 or 60 gigs. I'm only making that assumption because others have talked about putting the repos. on dvd's. Keeping in mind that the repos. when installed would expand to who knows what.

Ultimate edition is what I use and it has a whole lot of things installed by default. Some of which goes unused but it does offer things that are not available in the standard ubuntu like TheeMahns nautilus scripts. And a gamers edition. You can find more it it here:
[http://ultimateedition.info/](http://ultimateedition.info/)
and their forum here:
[http://forumubuntusoftware.info/](http://forumubuntusoftware.info/)
