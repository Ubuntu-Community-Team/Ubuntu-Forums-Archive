### Post by Fingers &amp; Thumbs on 2008-07-15
Try not using gnome-ppp, instead type wvdial into terminal. If you have no luck with that try editing /etc/wvdial.conf

```
gksudo wvdial.conf
```

and adding the line:

> Stupid Mode = 1

No, this really is not a joke.

now try wvdial again.
