Hi I'm in Australia and had some initial problems with ubuntu 9.10 when upgraded.

At first firefox would not display the correct web pages, as opposed to ubuntu 9.04 where it works straight out of the box.

I had to install java runtime environment, java plug-ins through firefox, flash player and manually configure the DNS servers provided by my ISP.

All working good, just letting those out there know.
