### Post by sudodus on 2014-08-03
I think you have problems with a new version of the kernel (that was upgraded for Ubuntu Server). Ubuntu desktop 10.04 passed end of life more than one year ago, and nobody checked how the new kernel would cooperate with the desktop program packages. See this link
[URL="http://ubuntuforums.org/showthread.php?t=2229730"]
Forums Staff recommendations on EOL Ubuntu releases, in particular 10.04 desktop.[/URL]


0. Please ***backup*** your personal files (documents, pictures,...) to an external drive.

1. [Try Ubuntu (Kubuntu, Lubuntu, Xubuntu,  ...) before installing it]("http://ubuntuforums.org/showthread.php?t=2230389")

2. Based on the results from trying some flavours of Ubuntu and maybe some other linux distro, ***decide*** what to install.

3. Install (make a fresh installation)

4. Restore (copy back) your personal files (documents, pictures,...) to the new system.
