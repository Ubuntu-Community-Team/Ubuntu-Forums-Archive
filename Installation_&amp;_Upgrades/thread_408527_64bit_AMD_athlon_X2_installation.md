### Post by nienaber on 2007-04-13
Well, I have looked over the forums and seen that a large number of people have had little to no luck getting their AMD installation to work. I personally had this same issue. In my office where I work we bought 2 new machines:
AMD 64 Athlon X2 and Cure 2 Duo both nice 64bit machines!
I downloaded the ubuntu 64bit installation CD and installed ubuntu on the Intel machine inside of 20min (I love how fast ubuntu is). However, when I went to install on the AMD machine it wasn´t as loving as the intel machine had been. The boot CD would freeze every time I booted. I tried the regular i386 and it wouldn´t boot either. I was at a loss for a bit then decided to download the alternative-i386 and presto it worked. I took the OEM installation route and 20min later ubuntu came to life on my AMD machine. 
The kernel used is the 32bit one and I am contemplation upgrading to the 64bit kernel but the lack of software for the 64bit is still a little concerning so I might just stick to the 32bit for a bit longer. 
You can find the alternatives to download all right here:
[http://ftp.gui.uva.es/sites/ubuntu.com/releases/edgy/](http://ftp.gui.uva.es/sites/ubuntu.com/releases/edgy/)
Good luck
