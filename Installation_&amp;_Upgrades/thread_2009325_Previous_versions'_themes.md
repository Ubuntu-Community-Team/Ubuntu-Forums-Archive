### Post by Enigmapond on 2012-06-24
To my knowledge there are no packaged themes for previous distros. The only way to do that is to find out what was the default theme used in 9.10 and download the individual themes...i.e. icons, gtk, window decorations and apply but doubtful it will look exactly the same.

### Post by Frogs Hair on 2012-06-24
If you are using 11.10 or 12.04 themes from 11.04 or older versions won't work because they are GTK 2 themes. I was unable to locate the official 9.10 themes package, but some of the themes carried over from 9.04 to 9.10 can be found at softpedia.

[http://linux.softpedia.com/get/Desktop-Environment/Themes/New-Wave-40415.shtml](http://linux.softpedia.com/get/Desktop-Environment/Themes/New-Wave-40415.shtml)
