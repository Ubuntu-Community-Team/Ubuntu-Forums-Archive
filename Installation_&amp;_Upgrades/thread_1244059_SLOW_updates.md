### Post by misfitpierce on 2009-08-19
Usually around this time in middle of night the server's are updating or other various things and it is hard to get updates. Wait till later or mid-day to get them or so and everything should run fine. There is no way to speed up what speed you get from the server's though. If you live closer to wherever the server's are you may get a slight speed boost but there are a lot of people getting updates etc. but im sure they are not that overloaded or Canonical would get some more servers up for Ubuntu updates etc.

### Post by rdionicio on 2009-08-20
I just made my changes and the speeds were SO much better. On the first tab you'll see "Download From" click on the drop down and select "other"

Then click on the button "Select Best Server" and it will do a test to see which server is best from your location.
