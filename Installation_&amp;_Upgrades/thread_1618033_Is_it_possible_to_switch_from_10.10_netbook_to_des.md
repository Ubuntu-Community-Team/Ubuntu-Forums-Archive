### Post by ajgreeny on 2010-11-10
You probably have all you need already; just choose the gnome desktop if it's 10.04, or the ubuntu desktop if it is 10.10, from the login screen session menu, bottom right.  If you really feel it is needed you can use ```
sudo apt-get install ubuntu-desktop
```but it may not really be needed, as it will pull in things that are not much use on a netbook, eg brasero and compiz, though they could be useful on a proper laptop.
