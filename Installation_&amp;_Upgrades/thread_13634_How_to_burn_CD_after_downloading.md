### Post by crane on 2005-02-01
The file you downloaded should be an .iso file.
If you using windows open whatever software you use to burn CD's (something like Nero) you should find an option to write cd from ISO file.
If your on a linux system running gnome just rightclick on file an select write to CD.

You don't have to do anything special to the iso file itself.
