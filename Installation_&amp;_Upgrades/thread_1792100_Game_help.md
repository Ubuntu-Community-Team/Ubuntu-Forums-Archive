### Post by Toz on 2011-06-27
A good resource is the wine appdb. Search for the game and see the ratings, comments and suggestions for getting the game to work.

_Link:_ [http://www.winehq.org/search?cx=partner-pub-0971840239976722%3Aw9sqbcsxtyf&cof=FORID%3A10&ie=UTF-8&q=world+of+goo&siteurl=appdb.winehq.org%2F#849]("http://www.winehq.org/search?cx=partner-pub-0971840239976722%3Aw9sqbcsxtyf&cof=FORID%3A10&ie=UTF-8&q=world+of+goo&siteurl=appdb.winehq.org%2F#849")

Another option is to install PlayOnLinux (from the multiverse repository) and see if its supported. There appears to be a script for it: [http://www.playonlinux.com/repository/?script=349]("http://www.playonlinux.com/repository/?script=349")
