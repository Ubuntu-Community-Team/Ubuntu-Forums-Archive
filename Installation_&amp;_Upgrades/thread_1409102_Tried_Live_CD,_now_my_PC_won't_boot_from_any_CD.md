I'm not quite sure what you mean by this comment in bold above, but it should not make any difference to the live CD that there is no hard disk with a formatted partition, nor would it stop you installing if you wanted to; you do the formatting during the installation.  Did the live CD run OK when you booted to it the first time, or do you mean that you could not get the live CD running?  If the latter, then it really does sound like a hardware failure.

Incidentally there is not much point comparing live CD speed with an install; the CD access time, even on a CD drive that works fast, means that it will run very slowly in comparison to a hard drive install.
