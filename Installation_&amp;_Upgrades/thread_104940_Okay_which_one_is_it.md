### Post by jimcooncat on 2005-12-17
If you find the default Gnome desktop usable (and many people do) then stick with it. It's been well tweaked for Ubuntu.

You can easily install kubuntu-desktop from Synaptic or from the command line with apt-get. This gives you KDE which gives you a more complex environment; more power, more options. Some people love it. I find it just too much to deal with. Supposedly kubuntu will be even better optimized for Ubuntu in the April 2006 relase of Dapper Drake.

Also there's xubuntu-desktop, with which I've become recently enamoured with. Very light and quick, but you need to do some tweaking to get to an environment that's highly usable. It has some nice toys available for it. I'm hoping that this package gets more attention with future Ubuntu releases; it's very nice for low resources -- whether you have a small machine or are just stingy with your desktop.

The downside of playing with new desktop environments is that when you go to uninstall them they leave a bit of cruft on your machine, which shows up in the environment you finally choose. For example, I'm still left with some KDE tools on the menus of my Gnome desktop after playing with kubuntu.

Once you install the packages, you simply hit the Sessions button on the login screen before you enter your username/password. You can then pick which environment you want to use, then a popup will appear asking if you want to make it the default, or just use it for the current session. Easy!

I've put a lot of my opinion in the descriptions above, other folks in these forums won't agree with all I've said. If you do plan to check out the other desktop environments, be sure to check out the Ubuntu wiki and these forums before you start so you get the best experience with them.
