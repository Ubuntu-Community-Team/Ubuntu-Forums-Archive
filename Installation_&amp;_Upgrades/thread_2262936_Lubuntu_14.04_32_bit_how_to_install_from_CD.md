### Post by slickymaster on 2015-01-28
Did you tried to burn the iso?

CD-ROM capacities are normally expressed with binary prefixes, subtracting the space used for error correction data. A standard 120 mm, 700 MB CD-ROM can actually hold about 737 MB (703 MiB) of data with error correction (or 847 MB total).

[Lubuntu (Intel x86) desktop CD]("http://cdimage.ubuntu.com/lubuntu/releases/14.10/release/lubuntu-14.10-desktop-i386.iso") iso size is presently 705 MB.
