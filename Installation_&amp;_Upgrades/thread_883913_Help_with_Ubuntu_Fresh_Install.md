### Post by snowpine on 2008-08-08
Hi there, sometimes the installer can be a little finicky, and if you just keep trying, it might work the 2nd or 3rd time :)
But I would recommend the Alternate Install CD, which seems to work better on older computers than the Live CD. They both install the same system, but the Alternate CD uses less system resources as it is text-based and doesn't have the live environment.
