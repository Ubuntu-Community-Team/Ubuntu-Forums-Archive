### Post by sikander3786 on 2010-10-10
> I have tried other versions of ubuntu 7.04-10.04

Do you mean all of them failed to load?

First of all check the MD5SUM of the downloaded image.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If the sums match, burn a new disc, this time at the lowest possible speed and try to boot it.

If you are not sure that your CD/DVD drive is in a good working condition, you can create a USB startup disc and try to install from it.

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
