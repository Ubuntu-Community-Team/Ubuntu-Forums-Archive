### Post by K.Mandla on 2006-09-14
The only other common element could be the CD itself. If your download wasn't intact, or if the burn process was corrupted somehow, you might be installing a persistent error from a faulty CD. 

But of course, that would only be the case if you're using the same CD over and over again. And if you've done Ubuntu, Xubuntu and Kubuntu installs, you probably aren't. ... :-k 

You could check that easy enough by using the test-this-CD option at the boot screen. Chances are you've already done that, though. ...

Have you tried installing a server (from any alternate/install CD) and upgrading to ubuntu-desktop (or one of the other -desktops) as a download?
