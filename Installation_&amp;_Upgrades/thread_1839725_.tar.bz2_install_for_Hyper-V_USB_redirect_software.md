### Post by ajgreeny on 2011-09-06
You need to run the .sh file from a terminal, so navigate to where it is with ```
cd  /path/to/installation.sh
``` and run it with the command ```
./installation.sh
``` Or you can navigate to it in nautilus, right click, go to Properties, and ensure it is executable.  You can then double click on it to run it.
