Also, would Ubuntu 6.06 run on a laptop even though is says desktop version? All the 'new' computers in our house are Acers, with 'recovery disks' for Windows XP Home Edition. 
My laptop is an Acer Aspire 3000, and Dapper Drake installs in it perfectly!
I'm dual booting Ubuntu 'Breezy' and 'Dapper' in that one.
I use the 'Alternate Install' CD, but the 'Desktop CD' is easier for most people to install with. 'Desktop' is just a name, you can use it for laptops or desktop computers.

You should read this excellent web-site:[http://www.psychocats.net/ubuntu/windowstoubuntu.html]("http://www.psychocats.net/ubuntu/windowstoubuntu.html")

With the Windows 'Recovery Disk', there isn't any option to install Windows XP to a partition, it will wipe the whole disk and claim it all for Windows XP. It restores the computer to its factory shipped state.
But we don't need to worry about that as long as Windows is installed already before we want to partition the disk and install Ubuntu. The Ubuntu installer can do the partitioning for you and install Ubuntu on your new partitions. So re-install your XP Home Edition first, then install Ubuntu.

It won't be until you need to re-install Windows some day that the 'Recovery disk' might be inconvenient, as it will wipe Ubuntu even if you don't want it to. I used to always be scanning and turning up loads of spyware and malware when I only had Windows XP. I used to like to re-install Windows XP fairly regulary. As soon as I found out I can use Ubuntu for all my internet purposes and keep Windows strictly off-line, I found no more spyware or malware in Windows. All scans came up 100% clean! You will probably never need to use your 'Recovery Disk' again if you do that, because Windows will stay clean forever. Eventually you will probably find you can do everything better with Ubuntu and eventually you won't care about Windows anymore.

Good luck with your installing, Regards, Herman :D
