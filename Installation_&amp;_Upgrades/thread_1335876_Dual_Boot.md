### Post by lindsay7 on 2009-11-24
What Nitstorm is suggesting is call a Wubi installation.  It will install Ubuntu under windows and from this you can see if all of your hardware will work with Ubuntu.  Note however that this is not a true dual boot installation.  If your hardware works well with the Wubi setup you can delete it under windows using the add/remove function under the control panel.  You can the set up a true dual boot installation if you want.  It will run faster and better with this setup.
