### Post by Bluejacket on 2008-08-16
It's mostly new software. 8.04 happens to be LTS version, so it's supported for 3 years on desktops instead of the normal 18 month support for other Ubuntu releases.

You can download and compile the new software versions yourself, but upgrading the OS might be easier. The version upgrade is supposed to work flawlessly, but be sure to make backups before you try it.

I just upgraded from Kubuntu 7.10 to 8.04, my first upgrade ever. Previously I've always done a fresh install when I've upgraded to newer version. Everything went perfectly, I didn't have to reconfigure anything myself. All my settings were kept, and Firefox 3 asked if it should import my settings from FF 2.x. I clicked yes, and my passwords, bookmarks, addons etc. were imported.

If you're happy with your computer, the upgrade might not be worth the risk. If you're the kind of user who wants to run the latest (stable) software, go ahead and upgrade. If you plan to upgrade to 8.10 later on, you either need to do a fresh install or upgrade to 8.04 first anyway.

Once again, don't forget to back up any important data.
