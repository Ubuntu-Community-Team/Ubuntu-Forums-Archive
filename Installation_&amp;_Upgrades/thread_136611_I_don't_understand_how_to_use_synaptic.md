### Post by thoffland on 2006-02-26
From one newbie to another, Synaptic is a simple way to install software. It really makes things easy, and you can find pretty much everything you need using it. 

The first thing you'd probably need to do is add more repositories. Follow the instructions here: 
[http://ubuntuguide.org/#repositories](http://ubuntuguide.org/#repositories)

Now just search for what your looking for, and click on the box when you find it. Choose install then hit apply at the top. I've found it's useful to click on the down arrow to show the terminal while it's running just in case it's waiting for input from you. 

Take a look at Automatix too... this is a really helpful program:
[http://ubuntuforums.org/showthread.php?t=66563](http://ubuntuforums.org/showthread.php?t=66563)

Good luck!
