### Post by erginemr on 2008-02-14
If I understood you correctly, Firefox is trying to install them to itself. What you should do is to right-click on the green button that says "install plugin", and save it to your desktop. 

You then need to open Thunderbird, find Add-ons from the menu, press install button to select those two packages one after another. Restart Thunderbird and you should be all set.
