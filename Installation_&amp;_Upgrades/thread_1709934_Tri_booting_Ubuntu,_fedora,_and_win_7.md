Welcome to UF Leetgaming

 Yes you can, I boot Vista, Ubuntu & Fedora, amongst others. If I remember correctly Fedora gives you the option of choosing to install grub or not. If so, you want to choose not to install it. If it doesn't though, you just need to reinstall grub back to Ubuntu if that's what you want controlling your grub menu. It's not very hard at all. There's like 2 commands to run. 
[[COLOR="Blue"]https://help.ubuntu.com/community/Grub2#Reinstalling%20GRUB%202[/COLOR]]("https://help.ubuntu.com/community/Grub2#Reinstalling%20GRUB%202")

As always, back up, back up, back up!!! 

You said you're using windows boot manager on start up. How did you get to that after installing Ubuntu? Just do the same thing with Fedora.
