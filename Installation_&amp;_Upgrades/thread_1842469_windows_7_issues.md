### Post by Mark Phelps on 2011-09-11
When you say "it's fropzen", is that your whole PC? Or is that just Ubuntu?

If it's just Ubuntu, how far does it get into the boot process before freezing?

As for Win7 downloading updates, you should have it set to only NOTIFY you of updates, NOT to automatically download and install them.  That way, you can pick a time to install them when you're not busy on the PC.
