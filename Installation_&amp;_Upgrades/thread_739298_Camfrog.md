### Post by Taino on 2008-03-29
It depends on what your trying to do... :KS

If your looking for the camfrog chat program for linux just like the windows one there doesnt seem to be a camfrog chat program for linux.

If you run VirtualBox "maybe" you can use the windows camfrog chat program, VirtualBox allows you to install Windows OS in a "VirtualBox" inside Linux, then you would install the Windows camfrog chat onto the Windows OS that is installed "Virtually" on your Ubuntu PC.

-----
Now then, If your looking to run the server program (this is not a chat program its the software to hosts the room) there is one for linux but this is not the chat program its the "rooms" part of the system, you cant chat with this, and its not in Ubuntu/Debian format .deb its in .rpm (RedHat/Fedora format).

### Post by Taino on 2008-03-29
Try using "wine"... [http://www.winehq.org/]("http://www.winehq.org/")

[Screenshot]("http://appdb.winehq.org/screenshots.php?iAppId=3710&iVersionId=5493") of Camfrog running in "wine"

To install "wine".. from a terminal window...
```

sudo apt-get install wine

```

Install "wine" in Ubuntu, then install the windows version of the camfrog chat program into "wine" on Ubuntu.
