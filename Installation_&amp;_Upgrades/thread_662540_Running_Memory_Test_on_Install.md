### Post by Gina on 2008-01-09
The memory test will run continuously.  The screen tells you which test it is doing and it's progress and also the progress of the complete test - called a **pass** - the whole thing will be repeated over and over again with a **pass count** shown.

On my older desktop the test took about 20mins - your machine is much faster but with more memory to test.  But look at the pass count - I'm betting it will show several dozen at least.

More likely problem installing is a faulty CD - have you run the CD data check?

---

### Post by Sef on 2008-01-09
You should run it at least about 8 hours.   If it runs for that long and there are no errors likely there are no problems that it can check.  Read more at the [Mem86 web site]("http://www.memtest86.com/tech.html").
