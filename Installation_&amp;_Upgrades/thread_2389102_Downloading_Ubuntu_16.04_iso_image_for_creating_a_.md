### Post by C.S.Cameron on 2018-04-11
Do you want to make the bootable USB using Windows or Linux, (Ubuntu)?
If Windows see **UNetbootin**: [https://unetbootin.github.io/](https://unetbootin.github.io/)  or  **YUMI**: [https://www.pendrivelinux.com/yumi-multiboot-usb-creator/](https://www.pendrivelinux.com/yumi-multiboot-usb-creator/)
For Linux see **mkusb**: [https://help.ubuntu.com/community/mkusb](https://help.ubuntu.com/community/mkusb)
