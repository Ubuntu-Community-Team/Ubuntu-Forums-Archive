### Post by askreet on 2007-04-19
The only option you have is to get the Ubuntu-Server 7.04 media.  It does not contain a GUI, and is somewhere around 200 MB.  You can install it and then:
```

apt-get install ubuntu-desktop

```

After that you will pretty much have a full version.  Right now, it will take some time with everyone upgrading.
