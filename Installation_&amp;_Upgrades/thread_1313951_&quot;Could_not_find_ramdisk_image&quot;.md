Hi,

I downloaded the Ubuntu 9.10 32bit ISO, and I used the utility cd2usb on winXP to create a live USB booting disk.  The process worked correctly.  I am now trying to install ubuntu on my acer aspire revo.  The USB disk is detected properly as a boot option, and when I select it I am taken to the ubuntu Installer boot menu.

I then select "Install Ubuntu".  I am now taken to a black screen with the following error:

Could not find ramdisk image: /casper/initrd.gz
boot:

What can I do to make this work?

Thanks.

EDIT: I found a solution for this.  I simply had to use unetbootin to make the boot disk, and then everything worked fine.
