### Post by An Sanct on 2011-07-18
Welcome to the forums!

If you wanted to install in VirtualBox, then your windows partition will be untouched - except for the virtual disk image file (VDI). So it is normal, that Ubuntu in a Virtual Machine (VM) will not see the host's Disk.

[http://www.psychocats.net/ubuntu/virtualbox](http://www.psychocats.net/ubuntu/virtualbox)
