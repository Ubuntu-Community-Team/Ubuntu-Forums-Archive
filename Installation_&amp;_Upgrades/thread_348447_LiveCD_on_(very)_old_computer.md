### Post by PurplePenguin on 2007-01-28
I find that [Puppy Linux]("http://www.puppylinux.org/user/index.php") is a bit easier to use and more polished than DSL, in my opinion.  Both are great little distros, though. :)  

Puppy has a bunch of wizards that can be used to do pretty much anything.

The coolest thing about these small distros is that the whole thing can be loaded into memory.  This frees up the CD drive so you can go in and burn files.  It picks up any existing drives and makes it really easy to mount them.  I use it all the time for rescuing broken Windows systems. :D
