### Post by arubislander on 2012-01-23
Welcome to the forums.

What you are looking for is an OEM install. You'll need to download the alternate cd image first (For Oneiric it would be ubuntu-11.10-alternate-*.iso). On boot you will be presented with an installation language selection menu. After you have selected a language you should press F4 and choose "OEM install (for manufacturers)" or it's equivalent in the language of your choice, then select Install Ubuntu.

You will be presented with a text based installer, but if you follow the prompts attentively you should be OK.
