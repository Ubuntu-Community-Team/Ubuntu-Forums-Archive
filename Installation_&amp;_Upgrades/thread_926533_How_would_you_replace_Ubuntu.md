### Post by lswest on 2008-09-22
You can choose to install Ubuntu, and at the partition screen choose manual, set up all your old partitions with mountpoints, don't format.  It should re-write system files (it may also thoroughly mess up your system).  I would suggest just using a seperate /home partition and backing up tweaked system files.
