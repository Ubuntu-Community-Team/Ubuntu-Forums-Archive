### Post by darkod on 2010-02-08
I don't know anything about this model, but there is no difference in installing from usb stick as opposed to cd. The only difference is that you actually boot from usb stick.

I recommend you first select the Try Ubuntu option from the main menu and that will run it as live desktop. If there is anything it doesn't like about your hardware, it should show it. Try internet, sound, video, etc.
