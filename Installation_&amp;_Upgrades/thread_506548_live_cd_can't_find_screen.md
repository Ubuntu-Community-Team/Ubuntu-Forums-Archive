### Post by lot3k on 2007-07-21
I had run into this same issue on my T61 widescreen laptop.

It seems that the live cd becomes confused when confronted with too many modes.  What I would recommend doing is using F4 to set a mode, then after it fails type:

sudo nano /etc/X11/xorg.conf

look for the final set of lines that define your color depth and mode:

quickest way to do this is to hit ctrl+w which will open a search bar, then type "modes" and press enter

You will notice multiple lines defining Depth and Modes, what you should do is delete them all and leave only:

Subsection "Display"
                   Depth       24
                   Modes        "1280x1024"   (this can be whatever resolution you want)
EndSubSection

Now press ctrl+x and save your work

now from the command line type:

startx -- -depth 24

It should boot accordingly, if another error occurs check the log it generates in /var
