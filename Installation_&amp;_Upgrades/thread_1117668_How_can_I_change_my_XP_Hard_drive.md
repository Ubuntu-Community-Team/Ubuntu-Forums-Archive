### Post by Mark Phelps on 2009-04-06
You don't want to backup your XP installation to DVDs if it uses any significant part of the 40GB -- because the DVD swapping when you go to reload it will drive you crazy!!

Better option with commercial software would be to use something like Acronis True Image, hook up both the old and new drives, boot from the Acronis CD, and use the utility to migrate the existing XP installation to the new drive.  Then, remove the old drive, hook up the new drive in its place, reboot, and you're ready to go.
