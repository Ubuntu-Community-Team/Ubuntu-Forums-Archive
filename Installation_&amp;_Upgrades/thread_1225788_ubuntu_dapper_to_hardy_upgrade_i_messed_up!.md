### Post by Sidewinder1 on 2009-07-30
At this point I believe that you're dealing with damage control, as I do not think the old version can be repaired. I would do the following:
Boot to the Hardy Live CD; copy all data files (music, video, documents, bookmarks, spreadsheets, etc. and anything else that you do not wish to loose forever) to an external hard disk, pen-drive, CD/DVD or whatever else you have that has the capacity for all of that data.
Then install Hardy from the live CD into the same partition where Dapper was.
For future reference, I think to *upgrade (as opposed to a clean install)* you need to do it version by version. IE: you can't skip a version...

HTH,
