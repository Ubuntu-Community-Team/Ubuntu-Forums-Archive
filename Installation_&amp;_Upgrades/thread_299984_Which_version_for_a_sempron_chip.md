### Post by zgornel on 2006-11-16
Use the i386 to be sure. The 3000+ Sempron should have AMD64 (64 bit instr support) if it has the  E6 stepping. You can check this by watching the serial number of the processor - shoud end with a BX. (BO and BA endings = E3, D0 steppings do not have AMD64).
