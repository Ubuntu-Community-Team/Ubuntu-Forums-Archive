### Post by VeeDubb on 2010-04-18
Unless I'm really missing something here, it sounds like you're taking a live CD, sticking it in the drive while windows is running, and expecting it to "do something."


That's not how it works.

You put the disc in the drive, reboot the computer, and make sure it boot from the disc instead of your windows hard drive.  (Most computers are set up to do this by default)


If you want to play with linux while windows is running, you need WUBI, or a virtual machine.
