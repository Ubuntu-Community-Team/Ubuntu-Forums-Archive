Hi all,


I work for a company that is very interested in replacing it's Windows desktops/laptops with linux desktops. The M$ licensing as grown in the last couple of years to the 10k mark and it's becoming unbearable to support.

Of all the distros I've looked at IMHO, Ubuntu is certainly the best. 
The process itself revolves around proving to the administration board that all the Windows software the users currently use have an open source alternative.
The obvious candidates are evolution for Outlook and Open Office for Office 2003.
We also have an internal CRM but we can work around that by using RDP to a terminal server.

### Post by Scunizi on 2007-03-13
Here's a couple of links that might help you out.  If programs are in an RPM format they can be converted (usually) to a deb file for installation using Alien.  Alien is available in Synaptic and referanceable on the forums.  

[http://www.tech-edv.co.at/lunix/UTILlinks.html](http://www.tech-edv.co.at/lunix/UTILlinks.html) = viewers of different formats

[http://www.tech-edv.co.at/lunix/CADlinks.html](http://www.tech-edv.co.at/lunix/CADlinks.html) = list of different CAD programs, their licensing and associated file formats they work with.

Hopefully something here will work for you.  I'd sure like to see another company migrate.
