### Post by snowpine on 2009-11-07
Impossible to help you troubleshoot if you don't share the steps you took. "I tried something but it didn't work" isn't enough information.

The following terminal command worked well for me in 8.04:

```
sudo apt-get install ubuntu-restricted-extras
```

Why don't you give that a try, then if you get any error messages, copy and paste them here.
