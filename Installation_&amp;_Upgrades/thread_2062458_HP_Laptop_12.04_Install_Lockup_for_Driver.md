### Post by mikewhatever on 2012-09-25
You need to press any key right after the BIOS screen to see the boot menu of the installation CD. That's the time the CD starts booting, and you might see some text at the top of the screen or just a purple background. Hitting any key should take you to the language selection, and then the boot menu.
