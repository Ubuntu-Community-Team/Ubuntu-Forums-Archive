### Post by 2F4U on 2013-02-19
On my installation, the recovery mode enables the nomodeset grub kernel parameter. On some graphics cards, this parameter is needed to prevent black screens at boot time. The following thread explains what this parameter means and how to set it:

[http://ubuntuforums.org/showthread.php?t=1613132](http://ubuntuforums.org/showthread.php?t=1613132)

Another way to get around the problem may be to install the proprietary ATI graphics drivers.
