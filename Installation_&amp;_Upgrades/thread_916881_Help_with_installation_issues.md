### Post by Bucky Ball on 2008-09-11
Nope. You don't extract anything. You should end up with an iso image after your download (one single file). You make a bootable cd from that and boot from it. Then install. Just remember, this bears no relation to windoze.

Use whatever cd burning software you like (in Windoze) and make a disk image using that iso. Then boot. It is not as complicated as what you are doing. The extracted files you have will mean nothing whatsoever to Windoze.
