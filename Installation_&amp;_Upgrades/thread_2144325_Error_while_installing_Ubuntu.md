### Post by fantab on 2013-05-11
You should first check the integrity of your downloaded Ubuntu .iso file with [MD5Sum Check]("https://help.ubuntu.com/community/HowToMD5SUM"). If there is problem then re-download. Then Burn it properly using instructions on [This Page]("http://www.ubuntu.com/download/desktop").

If you still have problems then post with complete info about your machine's hardware, like Graphics, Cpu, RAM... Is your machine booting with UEFI?

Is it a single Ubuntu OS you want or are you trying to Dual-Boot?
