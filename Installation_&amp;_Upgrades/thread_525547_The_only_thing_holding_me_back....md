### Post by mikewhatever on 2007-08-14
> **Shoadow56 said:**
> And then if something doesn't work?

A live cd session lets you try Ubuntu without installing it, running entirely from RAM and the CD. If something does not work, you can boot back to Windows, or whatever you use, and search for solutions or ask for help. Most of the problems are not at all hard to fix, but if you find no solution, try another distro.
