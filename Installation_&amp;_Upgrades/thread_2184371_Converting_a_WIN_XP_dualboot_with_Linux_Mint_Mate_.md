### Post by sudodus on 2013-10-29
It it possible to use VirtualBox to set up a virtual machine and run Windows in that machine. I have a computer with similar specs, the CPU is an AMD Athlon 64 X2 Dual Core Processor 4400+ and there is 2GB RAM. It works well (of course slower because the virtualization). You can easily find at lot of information if you browse for ***VirtualBox tutorial*** or something similar.

example: [https://help.ubuntu.com/community/VirtualBox](https://help.ubuntu.com/community/VirtualBox)
