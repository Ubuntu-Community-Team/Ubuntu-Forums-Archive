### Post by dabl on 2007-09-28
Even though he did it on a thumb drive, I think the principles and techniques are reasonably well described here:

[http://kubuntuforums.net/forums/index.php?topic=3081748.0](http://kubuntuforums.net/forums/index.php?topic=3081748.0)

You'll need to have a "Boot USB device" option set in your BIOS, for it to boot from your USB drive.  I'm not sure about the "switchable in BIOS" capability -- I guess if you can "turn off" the internal hard drive in BIOS, then that would leave only the USB drive to boot from.  But I don't know if that's possible -- these newer BIOSes are way too "helpful", in some cases ...
