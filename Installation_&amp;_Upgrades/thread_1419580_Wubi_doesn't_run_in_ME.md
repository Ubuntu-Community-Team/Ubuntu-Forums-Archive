### Post by snowpine on 2010-03-02
Swap the hard drive into your laptop with working CD, install Ubuntu, swap the hard drive back again.

That would be my recommendation, if possible. But first, check to make sure your hardware meets the recommended minimum requirements (if it's running ME it's probably quite old): [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)
