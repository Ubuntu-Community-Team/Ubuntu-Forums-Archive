### Post by benn333 on 2007-05-06
Try installing the Nvidia drivers available through the package manager. (Way easier than installing the ones from the Nvidia web site.) For instructions, follow this doc:
[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)

If you're prompted for a root/sudo password, try your user password. See here:
[https://help.ubuntu.com/7.04/administrative/C/](https://help.ubuntu.com/7.04/administrative/C/)
