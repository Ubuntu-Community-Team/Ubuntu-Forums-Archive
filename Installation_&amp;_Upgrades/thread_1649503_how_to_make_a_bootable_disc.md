### Post by sikander3786 on 2010-12-20
*Rubi1200* have mentioned nearly everything you need.

I would just want to add that there are simplified instructions on the Ubuntu download page as well ;-) See option # 2 there "Burn your CD or create a USB drive'. They are nice, easy and step-by-step instructions.

[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)

And you can also boot using a flash drive and install Ubuntu (if Bios supports). Things you need is an Ubuntu image, a flash drive and a flash drive burning utility. There are many out there but I would recommend Unetbootin. It is self explanatory once you download and run it.

[http://unetbootin.sourceforge.net](http://unetbootin.sourceforge.net)

Feel free to ask if you are confused about any thing :-)
