### Post by mcduck on 2005-08-26
[QUOTE=dink](its a 200 gig drive but always says 186 capacity)

Will the install find this partition and delete it for me?[/QUOTE]
There is no hidden partition. Hard disk makers count sizes different way than rest of computer people.

Normally, 1 Gigabyte is 1024 Megabytes, or 1024*1024 kilobytes jne. This is the way your programs use the disk. So 1GB is 1024*1024*1024=1073741824 bytes.

Now HD manufacturers count that 1 Gigabyte is 1000MB. So 1GB is 1000*1000*1000=1000000000 bytes.
