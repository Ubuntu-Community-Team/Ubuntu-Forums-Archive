### Post by lechien73 on 2011-05-05

You can set the system-wide proxy under System Settings -> Network Proxy in 11.04. If you're in 10.10 or before, then System -> Preferences -> Network Proxy. I used to use Ubuntu behind a squid proxy, and that's where I set it.
