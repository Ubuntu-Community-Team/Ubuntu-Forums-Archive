### Post by abhaysahai on 2006-02-20
Grub is a boot loader. In lay mans terms,  GRUB is a program which helps in loading the operating system ( actually kernel )  image into the processor. 
Grub also allows us to select which kernel ( Operating system) image we want to load. 
If you have Ubuntu installed, chances are that you already have Grub installed and see this as the screen after BIOD ( oops another tricky term !!) boot, where you get options like Ubuntu kernel 2.6.12 

Hope this helps.

Abhay
