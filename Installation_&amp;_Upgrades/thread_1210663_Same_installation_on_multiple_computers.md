### Post by Partyboi2 on 2009-07-11
Hi, if you have your machines networked you could try[[COLOR=Blue] AptProxy[/COLOR]]("https://help.ubuntu.com/community/AptProxy").
Or if they are not networked,you could create a cd/dvd using [[COLOR=Blue]aptoncd[/COLOR]]("http://aptoncd.sourceforge.net/").
