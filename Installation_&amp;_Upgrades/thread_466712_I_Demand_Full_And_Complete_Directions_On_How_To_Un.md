So, here's the first thing to try:

You said you have set your boot device in BIOS to boot from the CD.  In most BIOSes, and I am  seeing yours as no exception, they are in a certain order.  Set it so that the CD-ROM is the first device.  Then save and exit the BIOS configuration screen.  Usually this is done with [F10].  It'll try to start up, but this is the time to put the Windows CD in, then TURN THE COMPUTER COMPLETELY OFF AND BACK ON.  This makes sure the new BIOS settings apply to the boot.

When it boots from the Windows CD, don't expect it to immediately bring up a pretty splash and a liveCD environment.  (Ubuntu does that.  Ubuntu is bad.  Windows has a clunky text installer that you have to pay attention to or you can screw it up bad.  Windows is good.)  But you WILL notice at the bottom of the screen is a text label that says "Press Any Key to Boot from CD". Do not wait.  Press it.  And do it in time for the computer to catch it.  If you don't, it will assume you don't want it to boot from CD and continue on to the hard drive, which contains the evil (yet easy to install) Ubuntu (Which is not Windows).

Then you'll go through the joys of setting Windows up again.  Make sure to format the entire hard drive after deleting all the partitions.  You may have to track down the drivers to install during boot if you are running with a SATA or SCSI drive.  Treat it the same during the installation process as you would a hardware RAID array.  The drivers are NOT included in XP, so you will have to look for them.
