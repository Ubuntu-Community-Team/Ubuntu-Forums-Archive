Unetbootin is an application that let's you copy/burn a downloaded iso to a usb stick or cd --creating a live os for installing the OS to any partition of your choice-- it isn't installing the os inside windows partition--that's what Wubi does.  

Wubi is the only one I'm aware of to provide > ubuntu installation like windows applications.If you don't want to use Wubi, you'll need to burn a live os and install it on a separate partition.
