### Post by Kareeser on 2009-05-08
You would have to first resize your hard drive so that Windows can install onto the freed space on its own partition.

Once that is completed, you will notice that GRUB was erased, since Windows installs its own boot loader.

You'll have to then follow the instructions here:
[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)
