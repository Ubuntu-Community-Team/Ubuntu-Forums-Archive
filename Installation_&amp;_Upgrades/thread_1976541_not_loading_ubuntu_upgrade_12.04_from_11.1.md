### Post by mörgæs on 2012-05-09
Hi, welcome to the fora.

'Recovering' is a term used for getting access to a file which has been deleted, which is not the case here. Your photos are fine, but the surrounding system is not.

Before experimenting I recommend that you try a live boot, which will give read access to all files on the hard drive. In the live boot, select everything of value and copy it into a USB stick.
