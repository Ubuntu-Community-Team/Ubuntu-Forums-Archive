### Post by mprice on 2009-12-26
You really don't even have to pick any of those all you have to do is go to System>Administration>Synaptic Package Manager and do a search for "flashplugin-nonfree or you can enter this command in the terminal. 

```
sudo apt-get install flashplugin-nonfree
```

This will install flash player for you.
