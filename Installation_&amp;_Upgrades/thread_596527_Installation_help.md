### Post by ACrazyGerman on 2007-10-29
I am installing Ubuntu on my second hard drive which already has a bunch of files on it, can I install ubuntu on the second hard drive with out formatting and creating a partitioning the drive while not damaging my files?

---

### Post by taurus on 2007-10-29
You can use gparted from the LiveCD to resize your second harddrive, leaving some empty space to install Ubuntu.  You have to create an empty partition to install Ubuntu on or else Ubuntu will wipe the whole harddrive if you want to install it.

---

### Post by Pumalite on 2007-10-29
You can install to the second drive without any problems, but in this process, the drive will be formatted and anything in it will be lost, but you'll have a shinny new system.

### Post by Pumalite on 2007-10-29
As taurus suggested, you can partition your 2nd drive and keep one half for data and the other half for the system. To that 'data' partition you can restore your backups. You can leave your Windows files in the first hard drive because you'll be able to 'read and write' to that drive if you so choose.
