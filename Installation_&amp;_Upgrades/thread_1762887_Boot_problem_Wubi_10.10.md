### Post by briapro on 2011-07-02
I'm back to report that what has solved the problem (at least I can get back into the OS) was running chkdsk. I never tried this because my win xp pro system was functioning fine (in the dual boot). I tried copying over one of the files from the wubi megathread and it did not work. Before when I could not boot into my xp system and it crashed I would have to run chkdsk to get back into Ubuntu. So after all this time locked out of Ubuntu but able to boot xp I gave chkdsk a shot and it worked. Maybe this will be of help to someone else having boot problems with a wubi 10.10 install. \\:D/
