### Post by Pethegreat on 2007-12-08
I don't think this is related but I will share my CD problem and the fix for it. 

I have an AM2 +4800 processor on an ECS nForce4M -A motherboard. I was running a live CD(ubuntu 7.10) on a sony CD/DVD drive. It worked just fine untill I added a second CD drive. I added a 5 year old benq CD drive in the place of the sony. I moved the sony down on the cable. I stick in my CD boot up the computer and get a screen with a flashing "_" Try again with the CD in the sony drive, and I get the same result. For some reason or another the BIOS would not reconize both drives. 

The fix was simple. I disconnected the benq drive. I assume live CD's don't like multiple CD/DVD drives?
