### Post by ankspo71 on 2011-10-12
Hi,
By default Xubuntu does not have a keyboard layout switcher applet installed, but there is one available called "xfce4-xkb-plugin" and it is also a part of the "xfce4-goodies" package if you want more plugins. 

Here is the xkb plugin's homepage:
[http://goodies.xfce.org/projects/panel-plugins/xfce4-xkb-plugin](http://goodies.xfce.org/projects/panel-plugins/xfce4-xkb-plugin)
