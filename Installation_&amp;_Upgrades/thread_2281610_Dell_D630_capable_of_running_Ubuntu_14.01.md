The wireless issue can be solved easily and fairly inexpensively with a mini/nano USB 'known-to-work' wifi adapter if necessary. I had (recently sold) a Thinkpad core2 duo machine. It came with 1 GB RAM which worked, I added a second 1 GB. If I were AudryH, I'd research how to create a live DVD or USB  and how to choose the boot device for Dell D630. Create a live USB or DVD, try to boot it and see what works and what doesn't. This would answer the wifi question. The biggest cost is likely time. I've found booting from DVD slower but more reliable than USB on older machines. If USB booting works it's certainly faster.

Unetbootin is available for Windows to create bootable USBs from .iso. I often see ImgBurn for Windows recommended to create bootable DVDs from .iso if Windows doesn't have built-in burning software.
