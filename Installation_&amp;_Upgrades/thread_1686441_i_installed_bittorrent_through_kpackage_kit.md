### Post by gmargo on 2011-02-12
The **bittorrent** package contains the original bittorrent command-line client.  It won't show up in any menu.

If you are using KDE, I recommend the **ktorrent** package.  Which will show up in your menu.:P (There are many other bittorrent gui clients - check kpackagekit or synaptic or "apt-cache search bittorrent" on the command line.)
