### Post by fice on 2007-04-14
Im running kubuntu as well but I rarely use the adept package manager since I tend to find it a bit annoying as far as managing packages goes. I use the apt-get command directly on the konsole so Im not really sure how accurate my answer can be but I would think its the difference between updating your system to the most stable versions of kubuntu or the somewhat safe new versions. 

If you look at how the whole ubuntu/kubuntu/etc distributions are set up you can see that there are quite a few versions of the same distribution at once like 6.06 which is the most stable distribution out there but also the oldest one, this version has been out enough time for people to test it throughly and make sure is very stable on the other hand there is also the 6.10 version which is a newer yet also stable version except it hasnt had the same amount of testing as the 6.06 version. I would suppose the difference between the safe upgrade and the full upgrade would be to which of those two versions do you want to upgrade. There is also the experimental one which has had pretty much no testing on it called Feisty or 7.06 I think, this is the newest version and includes pretty much only experimental versions of most of the software but to update to this version you have to change your repositories so i dont think thats what one of those options. 

Hope this was helpful
