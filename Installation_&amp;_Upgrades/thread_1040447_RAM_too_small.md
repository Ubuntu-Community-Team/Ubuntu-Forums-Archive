### Post by stefangr1 on 2009-01-15
You can install it using the Alternate-CD, which supports installing on systems with at least 256mb RAM. You shouldn't expect sky-rocking performance, but it will run fine. You could also download the Xubuntu-8.04-alternate cd, which promises slightly better performance using a simpler gui.
