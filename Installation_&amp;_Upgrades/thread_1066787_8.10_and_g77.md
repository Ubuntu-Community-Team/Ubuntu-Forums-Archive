### Post by mwparis on 2009-02-17
Condensing previous poster's comments. Here's what I did to get g77 to work on my 8.10 installation:

1) Place the lines:

deb [http://archive.ubuntu.com/ubuntu/](http://archive.ubuntu.com/ubuntu/) hardy universe 
deb-src [http://archive.ubuntu.com/ubuntu/](http://archive.ubuntu.com/ubuntu/) hardy universe 
deb [http://archive.ubuntu.com/ubuntu/](http://archive.ubuntu.com/ubuntu/) hardy-updates universe 
deb-src [http://archive.ubuntu.com/ubuntu/](http://archive.ubuntu.com/ubuntu/) hardy-updates universe

in the file /etc/apt/sources.list *after* the lines for the "intrepid universe" repositories.

2) sudo apt-get install g77

3) Subsequent updates via synaptic will try to remove g77. You probably don't want this. So you'll have to issue:

sudo aptitude update
sudo aptitude safe-upgrade

And you'll have to do it each time there's an update.

Two questions remain:

1) Is g77 going back into the supported code base?

2) Can synaptic be adjusted to not remove g77?

Any help with these questions much appreciated.
