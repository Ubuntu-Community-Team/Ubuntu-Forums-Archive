Hi, I am just seeking advice before its time to upgrade, I have Ubuntu 12.04LTS installed on my ASUS P5N73-AM, 2 gig of RAM and the standard onboard graphics card, it runs 12.04LTS fine, but on two occassions in the past I tried to install 13.04 on it, both times it installed with no hassles, right up to the point where the system booted up, I would get as far as seeing the desktop but then it would just freeze, this happened both times, so my question is, is this Likely to happen if and when I upgrade to 14.04LTS or is 14.04LTS going to be more comaptible? I am pretty sure the issue is something to do with the graphics card drivers as I had the same issue when I tried to run a Proprietary driver in 12.04.

Grant

### Post by mörgæs on 2014-04-15
If you have a fine working system you should wait a couple of months before considering anything else. If there's no need for the new system it's best to let the dust settle. 

Regarding hardware compatibility please run ```
sudo lshw -sanitize > lshw.txt
``` and post lshw.txt in CODE tags.
