### Post by rictus007 on 2007-01-13
Hi everyone, I just install Ubuntu on one of my computers and I receive, when booting, this error:
Grub Error 21

Because I was not able to find any thread to resolve my issue I will post what I do in order to resolve the issue.

My computer has two HD. 

Primary Master - Windows
Primary Slave - Ubuntu

But the second HD was not "installed" on the Bios so when I was booting it was just not recognize my grub.  The default option of the Bios was to automatically detect the HD but for some reason did not find this second HD.  I just add it manually and...guess what it works!!!
