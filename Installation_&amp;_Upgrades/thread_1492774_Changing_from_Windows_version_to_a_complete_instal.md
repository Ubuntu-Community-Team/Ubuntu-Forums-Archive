### Post by lechien73 on 2010-05-25
My recommendation would be to:

[LIST=1]
[*]Back up your existing /home folder.
[*]During installation create the same username as your current one.
[*]Re-install the software.
[*]Restore the old /home folder.
[/LIST]
This is the procedure I used.
