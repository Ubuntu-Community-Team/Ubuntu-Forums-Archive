Error message:
 Configuring Hotway
 port 110 in use
   It looks like inetutils-inet (PID 5415) is already using port 110, the standard port for POP3 servers,Hotway will need to use something else, Unfortunately there is no simple way for this configuration script to set up your inetd to use a different port, so you will need to do that yourself.

Alternatively uninstall any other POP3 servers you may have installed and then reconfigure Hotway.
