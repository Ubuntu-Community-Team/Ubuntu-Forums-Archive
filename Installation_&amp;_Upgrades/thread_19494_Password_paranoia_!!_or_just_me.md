### Post by alastair on 2005-03-12
At install, is it not YOUR login password that it asks for i.e. it creates your user - you log in as "your username" with this password you entered.

So - logged in as YOU, a command like :

sudo ls /

should run with no password prompt.

Unless you added a root password afterwards.

See :

[http://www.ubuntulinux.org/support/documentation/faq/root](http://www.ubuntulinux.org/support/documentation/faq/root)
