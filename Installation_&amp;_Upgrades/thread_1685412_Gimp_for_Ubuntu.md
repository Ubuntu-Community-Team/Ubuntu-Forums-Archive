### Post by Rocket2DMn on 2011-02-10
Just a bit of background information - Ubuntu used to ship with GIMP.  They stopped in Ubuntu 10.04.

In the future, you should always check the Ubuntu repositories for software before looking elsewhere - the repos are very large and is the safest and most reliable source of compatible software.  There are multiple options for searching including Software Center (Applications->Ubuntu Software Center) and Synaptic (System->Administration->Synaptic Package Manager), as well as multiple command line options if you're interested.

Welcome to Ubuntu!
