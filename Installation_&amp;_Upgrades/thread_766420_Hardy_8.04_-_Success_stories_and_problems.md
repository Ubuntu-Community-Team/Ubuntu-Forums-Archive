### Post by tjajab on 2008-04-25
I upgraded to Hardy yesterday, making all possible mistakes, mostly because of lack of patience: I did not backup my data, nor did I put my home folder on a separate partition, and I used the http dist-upgrade feature.

Luckily, the upgrade went very well, there were two minor things which I could correct, and one thing which I haven't yet sorted out:
1) New Firefox beta seemed to run slow and was buggy - I purged it and reinstalled it, then it worked flawless.
2) Nvidia drivers seemed to be disabled, compiz advanced desktop settings were enabled but also a bit buggy. Under System->Administration->Hardware drivers the Nvidia driver was marked enabled but said "not in use". I disabled it, enabled it and restarted and things started to work fine again.
3) Startup takes somewhat longer now (I thought it was supposed to go faster), and I see a lot of text flashing by, for example that /etc/inittab was not found (wasn't it removed by default several distros ago?) - so I guess there is something I could do to make the startup smoother, I just have no idea what.

How did your upgrade work out?
