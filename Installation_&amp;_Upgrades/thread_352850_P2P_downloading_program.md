### Post by Herman on 2007-02-03
I don't know how to do this manually, step by step either, I just download automatix from here: [Home of **Automatix**]("http://www.getautomatix.com/")

I used it to install frostwire in my wife's edgy install and she baked me a chocolate cake with chocolate icing! :)

Kinda hard to get her away from the computer though, and my web pages are loading a mite slow today trying to squeeze down the same wire...
Chocolate cake with my coffee keeps me quiet though, I'm not complaining. :)

I downloaded the automatix installer, mine's called automatix_7.0-test5-6.10edgy1_i386.deb, which comes in as a compressed file, which I copied to my /home/username folder and my USB thumbdrive. With the thumbdrive I can transfer it to other computers like my wife's one for example.

Then, I just double-clicked on it.
It installs to 'Applications'-->'System Tools'-->'Automatix'. You click on it there to start it. It's reasonably obvious how to use it. It takes some time to download and install the software you choose. I find it best to only install a few things at a time with it, otherwise I find I have to sit there and wait too long for my computer to be ready to try things out.
