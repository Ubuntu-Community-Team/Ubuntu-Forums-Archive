### Post by TheFu on 2018-01-20
If ".ios" is really ".iso", the you can use **dd** on the Mac to copy bit-for-bit the ISO file onto an SDHC or Flash drive. That should boot from almost any Windows machine.  There are lots of how-tos for using 'dd' on a Mac.
