### Post by Imoen_tw on 2007-12-11
Compaq computers have a "special" partition on the hard drive so that the system bios recognizes the hard drive. I just went through this last week. Flash the bios with the hard drive installed, unformatted with no partitions. Then when you install the OS select to use all AVAILABLE space at most, not the whole hard drive.
