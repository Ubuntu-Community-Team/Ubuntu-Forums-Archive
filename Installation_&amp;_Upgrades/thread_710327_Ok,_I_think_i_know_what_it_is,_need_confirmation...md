### Post by taurus on 2008-02-28
To get it to work, there are a few steps you need to follow.

1.  Run a checksum on the ISO image right after you've downloaded it.

2.  Burn it at a slow speed like 4x.

3.  At the first menu after booting from the CD, run the Check Disc for Defects option to make sure the CD is good.
