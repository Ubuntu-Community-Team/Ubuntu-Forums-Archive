### Post by lisati on 2010-06-29
A wubi installation works by placing Ubuntu on the Windows partition, so if you remove Windows, you'll be removing Ubuntu as well.

If you want to remove Windows, you will have to either do a fresh install of Ubuntu into its own partition, or find some way of moving Ubuntu to its own partition first.

Either way, it's a good idea to make sure you have a current backup of important data first (including recovery partitions).
