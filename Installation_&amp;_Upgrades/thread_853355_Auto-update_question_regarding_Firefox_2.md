### Post by cfeagans on 2008-07-08
I'm using Hardy Heron w/Firefox 3 and I've noticed in the last few days auto-update suggests a security update for Firefox 2.

My problem is that I'm s=assuming that if I allow the update it will install FF2. I thik I did this once before by accident & had to re-install FF3, uninstalling FF2. 

Is there a way to keep the update manager from suggesting FF2? Is it a matter of removing or editing a repository?

---

### Post by cfeagans on 2008-07-08
I actually sorted this out on my own. In the event anyone else has a similar problem, I decided to poke around in the Synaptic Package Manager and noticed that I had actually installed both versions of FF (FF2 & FF3) without realizing it. 

I simply removed FF2 and, of course, the prompt to update is now gone.
