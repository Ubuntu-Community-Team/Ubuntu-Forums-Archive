### Post by az on 2006-12-01
New upstream versions of stuff will be seen in the developmental release.  A LTS release can get some new versions, but the new features would have to be very stable and very popular to be included as an update.

Otherwise the current Dapper kernel will just get security updates and bug fixes, but no new features.
