### Post by taurus on 2006-06-20
Did you happen to download and install the server version?  If you get an error message when you type "startx" (without the " " ) from the prompt, then you are installed the server version.  To install Gnome, run
```

sudo apt-get install ubuntu-desktop

```
However, if you have a slow laptop with less than 256MB of RAM, I recommand you use XFce4 instead...
```

sudo apt-get install xubuntu-desktop

```
