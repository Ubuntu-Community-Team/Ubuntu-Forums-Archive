### Post by Rubi1200 on 2010-12-03
Make sure the username starts with lowercase:

e.g.

Your Name: John Smith

User Name: must be john NOT John

Also, make sure the password conforms to the minimum requirements.

If possible, post a screenshot of what you see to the forums; it may be a bug.

Hope this helps.
