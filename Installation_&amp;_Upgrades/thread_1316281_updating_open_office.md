### Post by flipper9 on 2009-11-05
> **jerrydc said:**
> I don't understand what you mean by proposed and backports.  However, I have just about all buttons checked in System/Administration/Software Sources, and use Update Manager to do its thing.  Should I leave all that alone or would it help if I manually downloaded the latest version of Open Office?

Under System/Administration/Software Sources, goto the Updates Tab.  Check the proposed or pre-release updates and the unsupported or backport updates checkboxes (the last two below security and recommended).  Backports allow you to get updates for newer versions of packages introduced in newer versions of ubuntu, if someone created them.
