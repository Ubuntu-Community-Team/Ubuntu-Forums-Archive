### Post by erfahren on 2007-12-14
first let me explain this: there are actually two types of CDs, the LiveCD (which you can use to either run Ubuntu off the CD or install it), and the "Alternate" CD which is basically just a text based installer. (Sometimes using the text based installer is necessary, or works better, in rare cases for some that have problems installing with the LIveCD.)

the DVD contains both the text based installer and the ability to run Ubuntu off of the DVD (just like using the LiveCD). 

I think the DVD has more of the basic, commonly downloaded, libraries and programs as well, but they could just as well be downloaded. - so I guess if someone doesn't have a *fast* internet connection and ordered a CD or DVD to be sent to them, the DVD would probably be the way to go.

Using either the DVD or the CD the same updates would be necessary once Ubuntu is installed.
