### Post by Mark Phelps on 2009-03-31
As  long as you use the Vista disk management tool to shrink your Vista partition, you should be OK.  If you use the Ubuntu partitioner, you run the risk of corrupting the Vista OS partition.  You will need a Vista DVD to be able to boot from that to repair the corruption.  If you don't have a Vista DVD you can torrent an image of one from the Neosmart forums.  But you will have to burn the CD yourself.
