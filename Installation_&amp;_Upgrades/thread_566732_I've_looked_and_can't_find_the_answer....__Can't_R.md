### Post by Mr_JMM on 2007-10-04
When you download an iso file, there is always a chance that some of the information will be missing or corrupt or just wrong.

You need to check the downloaded iso file is 100% correct. To do this you compare the hash codes. A 'sum' that represents the information in that iso file. You then use a program to read the 'sum' calculated from the actual info in the file to what it should be.

[http://ubuntuforums.org/showthread.php?t=566495&highlight=hash%23%5D](http://ubuntuforums.org/showthread.php?t=566495&highlight=hash%23%5D)
