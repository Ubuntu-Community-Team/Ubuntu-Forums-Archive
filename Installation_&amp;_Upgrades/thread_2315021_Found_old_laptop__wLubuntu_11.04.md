### Post by sudodus on 2016-02-25
256 MB RAM is not enough for any of the current Ubuntu versions or flavours, not even ***Lubuntu***, if you want to use it for real tasks. There might also be problems with the graphics chip or card. But you can play with it :-) See this link

[Lubuntu/AdvancedMethods](https://wiki.ubuntu.com/Lubuntu/AdvancedMethods)

But there are other and even more light-weight linux distros, for example ***Wary Puppy***, ***TahrPup*** and ***Tiny Core***. Read more at this link

[Old hardware brought back to life](http://ubuntuforums.org/showthread.php?t=2130640)
