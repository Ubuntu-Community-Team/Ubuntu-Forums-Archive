### Post by d0nny2600 on 2007-12-13
The best way to install to the directory you want is to download the file as a tar ball and install manually. You can then use ./configure to set the install path. Sorry if it seems complex but this is the best way to have total control over your system. You will learn an awful lot by compiling your own sources and installing where you want to!
