### Post by Mark Phelps on 2010-03-25
IF you're running an Ubuntu version newer than 8.10 -- you're completely wasting your time!

ATI dropped support for Linux drivers for your card nearly a year ago.

If you try to force an install, you'll only end up trashing your display, and then you have to boot into a root console to remove the drivers.

The open source drivers are installed by default -- and that's all that is available now.
