### Post by FlyDoc on 2005-12-27
You don't have to move jumpers at all. Start Ubuntu installation, when you come to partitioning hard disks, just select manual partitioning and choose where to install Ubuntu. Whole installation goes pretty much the same way as in Windows ;)
