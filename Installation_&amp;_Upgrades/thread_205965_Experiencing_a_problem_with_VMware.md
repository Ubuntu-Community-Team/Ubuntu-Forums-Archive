### Post by rcarring on 2006-06-29
Make the dynamic hard file no bigger than 10GB

I don't know why your system is lagging. I suggest that when you are installing XP you opt to create a partition rather than just hit enter.

a) boot with xp disk
b) create partition
c) choose this as the target for xp
d) choose format as ntfs quick
e) proceed with the install

I might also suggest that you create an iso image of your installation media and use this rather than the physical cd as it will be quicker to install.
