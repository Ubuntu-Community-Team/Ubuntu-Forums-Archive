### Post by tgalati4 on 2007-05-10
I've done it with DSL on an old HP Omnibook.  Formatted and installed DSL using a notebook drive to IDE ribbon connector on an old desktop machine.  Put the drive in the notebook and it booted normally.

The CD reader on the Omnibook was shot and there was no USB boot in the bios, so I tried it and it worked.

I don't see why it wouldn't.  There will be some borked modules (like sound and wireless) but they are normally borked on most notebooks anyway, so you will have to experiment to get everything working properly.
