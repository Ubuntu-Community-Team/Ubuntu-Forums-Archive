### Post by alfonz on 2006-02-14
if you log into windows with administrator rights, go to Control Panel>Administrative Tools>Computer Management

and on the left hand side in the tree folder view you will see under storage>disk management

you will be able to see the partitions as unknown partition, you can reformat them from there to and NTFS or FAT32 file system



Hmmm I wonder how you killed it, did you accept default settings when it came to setting up partitions option. If you choose to do custom settings then you have the option of setting partitions to your liking and leaving the windows ones alone.
