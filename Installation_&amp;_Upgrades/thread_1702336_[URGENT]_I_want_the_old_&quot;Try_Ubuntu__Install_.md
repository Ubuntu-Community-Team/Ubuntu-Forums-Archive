Why don't you just use the Ubuntu alternate install CD? It is a text based, install only CD. It is very similar to a Debian install CD. It will install the very same Ubuntu system that the graphical live CD installs.
[http://www.ubuntu.com/desktop/get-ubuntu/alternative-download](http://www.ubuntu.com/desktop/get-ubuntu/alternative-download)
If you want to install Ubuntu without a graphical desktop, you can also do that from the alternate install CD. Just select "*command line system*" (or whatever it says exactly).

Write back if you need more help.
