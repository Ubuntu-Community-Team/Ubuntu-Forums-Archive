### Post by jimreynold2nd on 2009-04-05
Well, VMware is still a better user experience IMHO (I know... it's not open sourced, it endangers your freedom... whatever... but it still has better codes - for now...).

So to the original poster, you don't have to download the RPM package (and take the hassle of having to convert it to DEB). 
For the current state of your machine, uninstall that package you installed first.
Then just download the .bundle executable and run it. It installs perfectly fine in all of the distros I tried.
