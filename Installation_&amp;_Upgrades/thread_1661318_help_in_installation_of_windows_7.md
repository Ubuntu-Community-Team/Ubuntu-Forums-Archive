### Post by wookieshaver on 2011-01-06
Your best bet would be to run a clean install (wipe and format of the entire hard drive) of Windows 7 as the boot loader for Windows 7 is slightly different and will mess up your Ubuntu install anyway. Once Windows 7 is installed, you can shrink the hard disk volume using disk management in windows 7, this will make a certain amount of the free space on the drive unallocated. The shrink of the volume is necessary to allow for Ubuntu to format and use. Then reinstall Ubuntu on the unallocated space you freed up. 
 
Hope that helps!
