### Post by iiiears on 2011-01-11
Reviving an old machine can be rewarding but is always a challenge.
Though the kernel will happily work on most anything from a gumstick to the latest supercomputer the challenge is in application software. 

Some desktops to try are Lubuntu (LXDE) Xubuntu (XFCE) Flubuntu (Fluxbox) Blackbox, Enlightenment, There are many others on freedesktop.org. They will get you on the web though 600 mhz won't play videos or music more than 128kbs. You can use the "alternate" install iso image to install apache webserver or lighthttp (there are many others) but these will more easily allow you to develop dynamic pages. without a "No execute" bit i would serve static pages.

Whatever you decide most applications for debian rely on the same libraries that Ubuntu uses and a light desktop starts to slow down after installing just a few apps. 

So.. Minimal install alternate iso image add as few apps as you can manage to get the functionality you want and backup everything.
