### Post by merlinus on 2007-09-02
Unfortunately, there are often problems with updating to a newer version of ubuntu versus a fresh install, especially going up two levels.

So bugs from one version are then multiplied in the next, and the next, etc.

Also, if ubuntu 7.04 runs slowly, you may not have enough RAM, so xubuntu would be a better choice.
