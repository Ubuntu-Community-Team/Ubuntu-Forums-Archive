### Post by mörgæs on 2013-04-08
```
sudo apt-get install xubuntu-desktop
```

```
sudo apt-get install lubuntu-desktop
```

and so on. 

At the end you will have a number of desktops to choose from at the login screen.

### Post by fantab on 2013-04-08
Yes. By 'copy' you do mean install, right?

Why don't you consider installing Buntus on different partitions? Instead of choosing "Desktop" from Login Screen you will be doing so from Grub Menu. Moreover if you install all the desktops then there will be, for instance, three or four file-managers... and things could get messy. About 10GB-20GB of HDD partitions should be enough.
