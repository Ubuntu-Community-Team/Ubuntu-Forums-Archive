### Post by Specs4 on 2010-08-14
Well first, Ubuntu cannot run .exe files, but there is something called Wine that allows you to.
You can find out more about Wine at [http://www.winehq.org/](http://www.winehq.org/)
But even with Wine it may not work because Maplestory is incompatible with Ubuntu, but there are still some ways to install it on a Virtual OS, using programs like VMware or VirtualBox to run Windows inside of Ubuntu just like running any other application on Ubuntu. You can find out how to do this by simply Google searching "Maplestory on Ubuntu" and you will find plenty of ways.
