### Post by Mark Phelps on 2010-02-16
If you're asking for a DVD because you want install media that contains the bulk of the Ubuntu packages so you don't need the internet to do installations, last time I checked, that was NOT what the DVD was for.

Instead, it contained much the same packages as the CD, but it contained language packs in addition.

Just don't be disappointed if you insert your "DVD" and find that Ubuntu wants to connect to the internet whenever you go to install an application.
