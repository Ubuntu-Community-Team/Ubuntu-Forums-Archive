### Post by Dr. Nick on 2006-07-29
It would most likely require a total reinstall as all the files would be replaced with 64bit versions. A "upgrade" may be possible but I would think it would be a pretty "hacked up" method

You can still stick with the 32bit version like you have now, It may boot up on the new hardware, but I would backup anything before switching. If you leave the 32bit ver just get the k7 kernel
