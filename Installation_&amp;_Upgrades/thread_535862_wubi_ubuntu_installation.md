### Post by abhilash82 on 2007-08-27
I have also used Wubi for installing Ubuntu Feisty and it worked well for me. So you don't have to worry about the installation it does by downloading from the net. In order to use Wubi from a live CD you need to have the Alternate ISO for Feisty. The link below will explain you of the other options that are available.
[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

But I would recommend you using the Live CD that you have to install Ubuntu directly to your hdd. The Wubi option makes disk access very slow and makes you Linux installation dependent on Windows. This screen cast below should help you if you planned to do an install from the Live CD.
[http://www.youtube.com/watch?v=mduooMLIUl8](http://www.youtube.com/watch?v=mduooMLIUl8)
