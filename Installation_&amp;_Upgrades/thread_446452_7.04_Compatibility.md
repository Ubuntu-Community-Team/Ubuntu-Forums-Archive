### Post by Soybean on 2007-05-18
Works great with some RAID controllers (in my case, Ubuntu handles my RAID controller a lot better than Windows does, actually). Is RAID support part of the 945 chipset? I think it is, so... if the chipset is well supported, the RAID aspect should be fine too.

The easiest way to be sure, though, is, just to try booting from an Ubuntu install CD. It's a Live CD, so you can poke around with the OS and see how well it recognizes your hardware before you even begin the installation. :)
