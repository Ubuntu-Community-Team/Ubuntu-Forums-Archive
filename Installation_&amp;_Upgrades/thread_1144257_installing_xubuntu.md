### Post by taurus on 2009-04-30
How old is the laptop?  If you know the brand and model, go to their website and look it up.

Make sure you burn the ISO image to a CD as in ISO image, not a data file.  If you insert the disc into a machine running windows, you should see a bunch of files and directories, not one large file that you've downloaded.  Then, you need to go into the BIOS and set to boot from the CD first.
