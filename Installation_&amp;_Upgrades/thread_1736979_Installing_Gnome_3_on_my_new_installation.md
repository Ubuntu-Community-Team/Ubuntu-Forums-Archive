### Post by kansasnoob on 2011-04-23
> **marin123 said:**
> [http://www.omgubuntu.co.uk/2010/11/install-vanilla-gnome-3-in-maverick/](http://www.omgubuntu.co.uk/2010/11/install-vanilla-gnome-3-in-maverick/)

But check the link included there:

[https://launchpad.net/~ricotz/+archive/testing](https://launchpad.net/~ricotz/+archive/testing)

> WARNING - Make sure you know what you are doing. **[COLOR="Red"]Using this and the gnome3 ppa could break your system![/COLOR]**
So you need to know how to repair it using something like ppa-purge.

To use the included packages you also need to add the Gnome3 PPA
ppa:gnome3-team/gnome3 ([https://launchpad.net/~gnome3-team/+archive/gnome3](https://launchpad.net/~gnome3-team/+archive/gnome3))

And that link:

[https://launchpad.net/~gnome3-team/+archive/gnome3](https://launchpad.net/~gnome3-team/+archive/gnome3)

> This package contains packages from GNOME3 and their dependencies so they can be used in Ubuntu 11.04 (Natty). **[COLOR="Red"]This PPA is EXPERIMENTAL and MAY BREAK YOUR SYSTEM. There is no downgrade process.[/COLOR]**

Also, if you check the filters on both of those pages, it shows packages only for Natty!
