### Post by Therion on 2008-11-07
My *suggestions...*

Make sure your stuff is backed up on your external, then check again.

Do a clean install of Ubuntu.

If you find you're still freaked out by not having access to Windows, and you just can't stand it, install Virtual Box and run XP (or whatever) in a virtual machine until you calm down.  Once rationality returns you can just delete the VM with a couple clicks of your mouse.

Really, it's easier than you think to let go...

And being Microsoft-free has feel-good vibe all its own.
