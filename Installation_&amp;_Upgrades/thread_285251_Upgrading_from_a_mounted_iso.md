### Post by Belyel on 2006-10-26
I don't know if Xubuntu will be different for you than vanilla Ubuntu was for me, but trying to upgrade using a mounted iso resulted in a very broken install.  The upgrade got about 1/2 done (I think), then must have changed sources.list and gotten lost.  I got an error about dependencies, the upgrade crashed out, then I got a great error message that the system updgrade was not successful and my system may be unstable.  It did say that it tried to roll back the upgrade, but it left several packages broken.  When I restarted, X wouldn't start, but I at least had the command line.  I finally fixed some of the broken dependencies caused by the upgrade crashing out, and now I'm in Windows getting ready to burn a CD of the alternate.iso to try again.
