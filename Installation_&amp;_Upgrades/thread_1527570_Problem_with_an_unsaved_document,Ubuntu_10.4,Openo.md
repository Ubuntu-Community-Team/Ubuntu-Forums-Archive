### Post by mikewhatever on 2010-07-09
Try pressing alt and drag the window with the mouse, or alt+f7 and move the mouse. The second one is particularly good if you use the touchpad. Hit alt+f7 again to release the window.

### Post by mcduck on 2010-07-09
..or hold down the Alt key, that allows you to move a window from any point inside the window instead of just the titlebar.

Alt + left mouse button = move
Alt + middle mouse button = resize
