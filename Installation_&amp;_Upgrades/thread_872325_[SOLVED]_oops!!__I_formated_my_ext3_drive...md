### Post by tamoneya on 2008-07-27
if you can download RIP linux on another computer or without using the drive in question.  Once you download RIP linux burn the iso to a cd and boot from it.  There is a program on there called test disk which will search the disk for files (normally the computer doesnt search for the files instead it asks the partition MFT where the files are).  It should find your files for you.  You should have some other drive or somewhere that you can save the files to.

[http://www.tux.org/pub/people/kent-robotti/looplinux/rip/](http://www.tux.org/pub/people/kent-robotti/looplinux/rip/)
