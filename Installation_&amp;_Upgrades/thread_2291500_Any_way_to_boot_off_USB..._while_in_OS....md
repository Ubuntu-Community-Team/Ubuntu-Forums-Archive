### Post by yancek on 2015-08-20
Installing Ubuntu or any Linux system to a flash drive is not a complex process.  If you have a  64GB drive, you might as to a normal, full install.  You would need to check first to see if you can boot from a flash drive.  To do that, you would need to access the BIOS.  Can you do that?  If you want to boot Ubuntu from inside Chrome OS, you would need to install VirtualBox or some other virtualization software and from what you said, this computer belonging to the school, you won't have administrative access so you won't be allowed to install software.
