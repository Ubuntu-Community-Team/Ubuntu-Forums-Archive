### Post by seeklinux on 2006-10-27
Sorry if this has been communicated clearly already, but I downloaded the desktop ISO **4 times** before the md5sum matched what was indicated in the MD5SUMS file. Using bittorrent I think it checks for you, but I tried once and it failed (said it was corrupted).  So I switched to HTTP download, which was actually incredibly fast. *Fortunately,* I kept checking the md5sum value rather than assuming it was OK (I have rarely had such a problem).

So maybe I had strange luck but to save people a lot of unnecessary trouble, I would suggest the admins remind people in a prominent place to check the md5sum if they download via HTTP. If that has already been done, please excuse my post.

Thanks.
