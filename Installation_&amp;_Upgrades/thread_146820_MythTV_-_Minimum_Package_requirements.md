### Post by chuckyp on 2006-03-19
MythTv's wiki is pretty complete as far as dependencies go.  I'm pretty sure you will need atleast some sort of X environment as the front end and the backend configuration and viewing are done through an x window.  If you are looking for a minimalist way of doing it perhaps xubuntu-desktop after a server install.  Or possibly downloading the xubuntu install cd.  After you have your wm up and running then you would be able to install mythtv with out a hitch.

Also it would be possible to just install x without a window manager and run mythtv.  Just start the x server and launch applications via a terminal etc..
