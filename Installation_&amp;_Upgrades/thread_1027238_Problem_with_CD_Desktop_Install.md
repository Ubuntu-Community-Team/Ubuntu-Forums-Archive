### Post by gettinoriginal on 2009-01-01
Just insert a username and password, and it will progress, but it will not install without your permission.  After username password, it will ask you, "use it without install"  "install" etc.  Click on use it without install to try it out, this will assure you that all your hardware etc is compatible.  Just be aware that anything you do while using the live CD will not be saved when you eject it.
