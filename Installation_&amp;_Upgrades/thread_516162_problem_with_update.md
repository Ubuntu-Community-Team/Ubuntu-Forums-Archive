### Post by d_kammerer on 2007-09-13
**This post could be related to an Ubuntu bug filed at**: [https://bugs.launchpad.net/ubuntu/+source/fontconfig/+bug/104553](https://bugs.launchpad.net/ubuntu/+source/fontconfig/+bug/104553) [br].[/br] ---------------------------- [br].[/br] [br].[/br]
				This workaround worked for me:
```
apt-get install fontconfig 2>&1 | grep failed | cut -d : -f 1 | xargs touch
```

Thanks to wolfe99
[https://bugs.launchpad.net/ubuntu/+source/fontconfig/+bug/104553/comments/18](https://bugs.launchpad.net/ubuntu/+source/fontconfig/+bug/104553/comments/18)
