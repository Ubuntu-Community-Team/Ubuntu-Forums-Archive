Dear all,
I just Installed Ubuntu Jaunty Jackalope on Win XP Pro using WUBI . I noticed that during boot I was not able to select OS since the keyboard was no responding. Even function keys were not active. It is a serial port keyboard connected to USb via an adaptor. Before the installation of Ubuntu I could easily play with keyboard keys during booting windows.

Mouratos

### Post by mouratos1a on 2009-07-14
OK!!! Fellas, solved myself... Just needed to go to Bios and select USB keyboard Controlled by BIOS and not by OS as is the default choice...
I hope this helps someone...
