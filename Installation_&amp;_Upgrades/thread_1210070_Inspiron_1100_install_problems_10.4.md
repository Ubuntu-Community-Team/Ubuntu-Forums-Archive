### Post by Mark Phelps on 2009-07-13
From a quick Google, I saw that the Inspiron has only 256MB of memory.  With that little memory, you're going to have trouble installing using the desktop LiveCD, and even if you get it installed using the text-mode alternate installer, you'll find that you won't be able to use Gnome and the machine is likely to work very slowly.

You should consider more "minimal" installations than Ubuntu.
