### Post by rickbeton on 2010-05-02
You could try reinstalling Grub. If you have the 'alternate' CD, it has a useful feature for doing this.

Take care if you have more than one hard drive which one has Grub installed - it must of course be bootable. This same one must be the first choice for the BIOS boot menu.

You can investigate these issues using the "Disk Utility" (System > Administration menu on the Live CD).

Rick
