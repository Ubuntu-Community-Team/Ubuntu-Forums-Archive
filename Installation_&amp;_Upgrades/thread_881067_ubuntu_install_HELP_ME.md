### Post by neilneil2000 on 2008-08-05
It seems to me like you have the server version of the disk and not the desktop version this is why you are confronted with the text based interface.

You should be able to install using the text based method and I seem to remember the banner message saying something about hwo to start the process off.

If you don't know what certain options mean it is generally safe to accept the default.  I would advise however to unplug any drives you don't want wiped if you are really unsure what you are doing.

Once the server version is installed, the following commands should get you the graphical desktop installed:

```

sudo apt-get update
sudo apt-get install ubuntu-desktop

```
