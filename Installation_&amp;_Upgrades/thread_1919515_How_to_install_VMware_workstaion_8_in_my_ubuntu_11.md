### Post by drmrgd on 2012-02-03
You'd be just as well off reading the documentation from VMware in terms of how to install.  They have very detailed instructions for how to go about this, and if you're going to read posts here, you may as well just read the documentation they provide!  Here is a link to the documentation:

[http://www.vmware.com/support/pubs/ws_pubs.html]("http://www.vmware.com/support/pubs/ws_pubs.html")

See the "Getting Started with VMware Workstation" .pdf for installation instructions.  Page 19 is where the Linux Host installation starts (although you should read the beginning of the document to be sure your system is set up well for this).
