### Post by Mark Phelps on 2011-12-16
IF this is a laptop, there is most likely a switch or button on the case that enables/disables WiFi.  It's pretty easy to accidentally switch that OFF. Check to confirm that Wifi is switched ON.

Also, boot back into Windows, check Device Manager, and confirm that the wireless device is enabled.
