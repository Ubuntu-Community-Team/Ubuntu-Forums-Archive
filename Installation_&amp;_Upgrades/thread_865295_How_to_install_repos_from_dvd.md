### Post by ajgreeny on 2008-07-20
This is probably easiest using synaptic.  Go to Settings>>Repositories and then the second tab "Third Party Software" and the Add CD-ROM button.  You will also need to remove all the usual Ubuntu repos by using the other tabs on the synaptic settings dialog or it will try to use those in preference to your CD.
