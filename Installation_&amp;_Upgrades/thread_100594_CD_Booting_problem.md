### Post by 23meg on 2005-12-08
Make sure that your cd is not corrupted. Is it an official (printed) one, or one that you burned? If you burned it, did you do an MD5 sum check on the iso? If yes, try burning another CD at a lower speed.

If all fails, take a look at [Smart Boot Manager]("http://btmgr.webframe.org/"), maybe it will help you.
