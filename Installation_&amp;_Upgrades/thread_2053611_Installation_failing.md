### Post by coldraven on 2012-09-06
I do all my installs from a 1GB SD card, it's quicker and you don't waste a CD.
You can create one using "Startup Disc Creator" in Ubuntu or Unetbootin on a Windows/Mac/Linux box.
You need to alter the BIOS setup to make booting from USB the first option.
Change it back after a successful install.

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
