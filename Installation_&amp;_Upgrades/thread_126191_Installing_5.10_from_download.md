### Post by Qrk on 2006-02-06
People on slower connections are more likely to have a problem.There are several easy solutions, here are three.

```
wget *address_of_file*
```

which is less likely to be interupted. This, of course, will work in a Linux system but not windows.

The second, nicer option is to use bittorrent. Bittorrent is a better download manager and can resume interupted downloads.

The third is to install a download manager and use that.
