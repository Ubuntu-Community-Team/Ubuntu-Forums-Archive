### Post by LADmaticCA on 2009-06-06
If you already have an Ubuntu live cd, just reboot your machine and enter the setup (Usually pressing F1 or Del). Once in the setup, look for a boot menu and an option to change boot device priority. Make sure the CD/DVD ROM is the first device. Save, exit and the machine should boot from the live cd.
