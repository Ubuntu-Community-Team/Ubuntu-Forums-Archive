### Post by ajgreeny on 2013-12-24
The simple answer to that is, I'm afraid, no.

One possible way out is to use the mini install iso, and when you install that at a time with internet available, just choose to add the ubuntu-desktop (or kubuntu, xubuntu, lubuntu version if preferred) and you will end up with the system including all updates.  [Minimal CD]("https://help.ubuntu.com/community/Installation/MinimalCD")

It can be quite slow installing this way as it depends on an internet connection, and if that is slow, your installation will also be slow.
