### Post by lidex on 2010-01-20
Open a terminal and (Applications>Accessories>Terminal) and enter these commands one at a time. Copy and paste the output into a post. Enclose the text in code tags.

```
lspci -nn | grep VGA
gedit /etc/X11/xorg.conf
```
