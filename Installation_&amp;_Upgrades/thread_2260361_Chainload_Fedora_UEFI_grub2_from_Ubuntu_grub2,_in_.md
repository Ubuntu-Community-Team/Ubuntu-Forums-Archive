### Post by valmar-lp on 2015-01-15
[COLOR=#000000][FONT=verdana]Ok, I got this to work. I extracted the fedora-ca.cer certificate from the Fedora source package of shim[/FONT][/COLOR]

[COLOR=#000000][FONT=verdana]Then on Ubuntu I installed the mokutil program and I added the key to the allowed keys of the Ubuntu bootloader:[/FONT][/COLOR]

[COLOR=#000000][FONT=verdana]1) sudo mokutil --import fedora-ca.cer and set a password[/FONT][/COLOR]
[COLOR=#000000][FONT=verdana]2) reboot, ubuntu reboots in the MokManager[/FONT][/COLOR]
[COLOR=#000000][FONT=verdana]3) add key, by typing in the password set in point 1[/FONT][/COLOR]

[COLOR=#000000][FONT=verdana]The Ubuntu grub can now boot the Fedora kernel in Secure Boot.[/FONT][/COLOR]
