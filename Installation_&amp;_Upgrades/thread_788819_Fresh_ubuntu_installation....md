### Post by Partyboi2 on 2008-05-10
use unetbootin ([http://lubi.sourceforge.net/unetbootin.html](http://lubi.sourceforge.net/unetbootin.html)) windows version. Install ubuntu then use gparted to delete windows and resize ubuntu to use the unallocated space.


Edit: reading the unetbootin website you will not have to worry about using gparted to resize as unetbootin can replace the existing OS entirely.
