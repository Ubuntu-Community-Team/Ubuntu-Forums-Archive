### Post by dabl on 2007-09-06
Sometimes a drive can be "picky" about the media that is in it, or the burned image that it's reading.

I'll wager a tiny sum that if you will download the appropriate Feisty ISO image from the Ubuntu site, and burn the CD yourself on that same drive, at no more than 4X speed, it will be very happy to read a CD that it made.  :)
