### Post by kalikiana on 2006-10-27
As you can see [here](http://www.xubuntu.org/news/6_10_released), XUbuntu 6.1 is finally realeased officially although the download came a few hours before the announcement. :)

Now XfceB2 is included but it appears to be a pretty recent snapshot compared to my already installed XfceB2, eg. the system icons don't appear on the desktop any more. What I'd emphasize is we get a calculator again, a spreadsheet app, an additional software installer ('Add/Remove') which is sort of a simplified Synaptic (although no replacement for me). Further more there is gxine instead of xfmedia - I'll see if the newbie may beat mplayer unlike its predecessor. There are more plugins included (like thunar archive plugin or the dict applet).

By the way, I did an upgrade via the GUI update manager and it works just fine! :)

In case you're missing the trash can on the desktop this will help:
```
Thunar trash://
```
