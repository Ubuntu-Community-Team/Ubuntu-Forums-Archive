### Post by yancek on 2015-07-18
Operating system like Ubuntu Linux or windows are generally not "uninstalled".  You do that with applications or programs used by the operating system.  The general process is to format the partition you want to use before or during the install.

> Is there a way to "reload from backup" and use the original ubuntu iso as the "backup" so I can revert to normal?

I'm not sure what you mean by that?  You can use software such as clonezilla to clone an installed partition or drive including the entire system.  You can also boot the iso directly with a proper entry in the grub.cfg menu.  Explain more specifically what you want.
