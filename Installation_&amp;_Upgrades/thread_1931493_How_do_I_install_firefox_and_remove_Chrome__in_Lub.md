### Post by ajgreeny on 2012-02-25
In exactly the same way as you install and remove anything else, except it's not chrome but chromium in Lubuntu!

Open up synaptic, (the best package manager available for any linux distro, in my opinion), search for chromium and remove it, then search for firefox and install it.

Removing chromium may also mean that you have to remove lubuntu-desktop package, but that is just a metapackage, not an aplication, and is there simply to make sure all the dependencies of Lubuntu are installed.  Removing lubuntu-desktop does not stop anything working and will only be required if you try to carry out a distro upgrade, eg from 11.10 to 12.04 in April.  Alternatively, you could always leave chromium on the system but simply not use it, which is what I have done on my 11.04.
