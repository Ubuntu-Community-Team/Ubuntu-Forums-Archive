### Post by arashiko28 on 2007-11-05
**To the administrators, please make this sticky, this a very common problem upon Xubuntu 7.10 users.**

If when you try to run terminal after a fresh install of Xubuntu 7.10 and instead of running you get the same effect as if you have pressed **ctrl+alt+backspace**, is because terminal is not installed.

Go to **Applications, System, Add/Remove...** and in the All list, look for terminal, _select all the **terminal** unchecked boxes_ and install them, I also installed **Konsole**, in the case that accidentally delete gnome (did it once) terminal will go away too (trust me it does), so in Konsole you can redeem your mistakes.

This solved my problem, hope that can solve yours too.:)
