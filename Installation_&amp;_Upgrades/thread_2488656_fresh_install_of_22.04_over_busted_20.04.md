### Post by TheFu on 2023-07-11
> **pjstock said:**
>  
are there any tricks to reinstalling while still keeping files and applications that I had previously?


I wouldn't.  I'd plan on putting the data back post-install.  Best to remove all the cruft from the prior release AND the failed attempted upgrade.
There's one exception - I'd keep everything in the HOME directory.  This isn't hard, provided you split off a separate file system just for HOME in the prior release. This is not done by default.  If you don't remember doing it, then you didn't and should assume everything on the drive needs to be wiped as part of the fresh install.

As usual, installs sometimes go badly due to human error, so backup anything you don't want to lose and hope you don't need it.
