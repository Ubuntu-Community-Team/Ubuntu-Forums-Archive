### Post by doublewitt on 2010-03-14
Go to this webpage:
[https://launchpad.net/~dominik-stadler/+archive/dsta-karmic-ppa](https://launchpad.net/~dominik-stadler/+archive/dsta-karmic-ppa)

After that, click on "READ ABOUT INSTALLING" and follow the instructions there... everything should work fine afterwards. Activate your update manager for the Tellico updates.

TELLICO is really good! 
One "other" thing I have found it useful for...
I made a database or file using the "New custom collection" option and then named it as "**PROBLEMS AND SOLUTIONS**". I entered the "PROBLEM" field, the "SOLUTION" field (paragraph) along with all referral fields such as URL's, eMails, Names, etc. And so whenever I encounter a problem whether it be a software issue or an Ubuntu system issue, I record all problems and solutions in Tellico. Later on I can always refer to this information. I find this to be so useful! I carefully manage my collection of problems & solutions...
