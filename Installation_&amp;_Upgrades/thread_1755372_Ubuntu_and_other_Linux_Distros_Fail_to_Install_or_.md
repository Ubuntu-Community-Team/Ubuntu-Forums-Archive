### Post by RootsLINUX on 2011-05-11
Do you have a spare hard drive that you could try to install on? My Ubuntu installation was freezing up a couple weeks ago. I set aside a partition on a spare drive and successfully installed there. Turned out that the hard disk I was trying to install on (which was brand new) was defective and had to be returned.

If you can boot from the CD, try opening the Disk Utility program and looking at the status of your hard disk for any problems. That application confirmed for me that my drive was defective and on the verge of failure.
