### Post by aJayRoo on 2007-09-26
I'm not really sure what you mean, but there is a chance you do not have a problem at all. When most Linux/Unix systems ask for a password, you type it in but it is not echoed to the screen. This means it will not seem as if any characters are being typed however the system is taking the password. This is a security measure. Try typing the password and continuing as normal even though it appears no password is being typed. I hope that sorts your problem out.
