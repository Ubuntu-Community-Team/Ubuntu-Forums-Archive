 If you don't have constant net acces I would wait until hoary gets released then you have a lot less upgrades. The only upgrades you have to download are security updates and if you want to the mysterious hoary-updates (see [http://www.ubuntuforums.org/showthread.php?t=21492](http://www.ubuntuforums.org/showthread.php?t=21492))

you can download .deb files and put them on an external disk. But you have to check if you have the packages they depend on. And if you don't put them on the disk too. And check what packages they depend on ... etc. Most of the time there aren't too much packages that you need in order to install a package but you can be unlucky sometimes.

I would wait for the release of hoary. Connect it to the internet to download or install all packages you would want to use. And after that whenever you are connected do the security upgrades.
