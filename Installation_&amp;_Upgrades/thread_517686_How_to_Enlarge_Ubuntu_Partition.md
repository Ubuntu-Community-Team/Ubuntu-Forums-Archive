### Post by sfgeorge on 2007-08-04
Hi Woody,

I've heard others say good things about [GParted]("http://gparted.sourceforge.net/"), which can be used to create, delete, and resize partitions.  I believe that you should be able to delete the old Ubuntu installation, and use some or all of that space for your active Ubuntu partition.  Just go to the web site, download the LiveCD, and create a cd from the downloaded iso file.

Make sure to read up on the documentation.  And backup your /home files for just-in-case purposes.

Good luck,

Stephen
