### Post by scrooge_74 on 2007-10-03
If your harddrive is partitioned in a way your /home is separated from your system files things are easier

You could also make a backup of files in the /etc directory since config files for apps in the system are there.

Maybe make a note of what is installed in your system before using a hose on it. But to tell you the truth the only times I do a clean install is because I am either bored to dead on the weekend or I made some serious mistake on the system
