### Post by Mark Phelps on 2011-04-19
Unless someone else here has actually tried Ubuntu on this laptop, the only way you will know is to boot it using an Ubuntu LiveCD (or the USB equivalent).

There is so much variance in laptop hardware, and the drivers needed for the same, that only a live test will tell you for sure.

### Post by ghostborg on 2011-04-19
I had an older Hp laptop zv6031us that I tried it on and the only problem I had was finding sound and wifi drivers.

After I tried the live cd I used the WUBI installer which
allows you to install Ubuntu on the same partition as Windows which gives you some time to play around with it before wiping out Windows or dual booting. The live cd can give you a good idea of how Ubuntu will play nice with your hardware but is limited to a point of actually trying it out which is why WUBI is a good next step.
