### Post by Toz on 2011-08-05
See:  [https://bugs.launchpad.net/ubuntu/+source/linux/+bug/572521]("https://bugs.launchpad.net/ubuntu/+source/linux/+bug/572521").

Using the Live CD, at the boot screen where you can choose to install or try ubuntu, press F6 and choose the "nomodeset" option.
