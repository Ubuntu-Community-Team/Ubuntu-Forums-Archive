**Starting Off**
Ok, I made an 8Gb partition in Virtualbox that I am going to use as my work area.  I should use no where near this amount, but I'll be covered if I decide to keep it for a test environment later.  To make sure I don't start with any bloat I am going to install with the Ubuntu Minimal Install CD.  This will install just the base CLI system.  The minimal install CD is somewhere in the community docs, just google "Ubuntu Minimal CD" and it should be easy to find.

I'll start posting images after the install.  If you want to know what the install process looks like go find some install help docs, they are all over.  No need to make redundant images.  The real difference in this install is that it will download all it's install packages because the install CD (or ISO since I am doing this virtually) is only 9Mb!  So you will need a wired NIC to do the setup, or if like me you are going virtual you can use wireless because the virtual system is NATing through the already running wifi.  At the end of the install it will ask what other packages you want to install in addition of the base system, I choose nothing so I am running lite.
