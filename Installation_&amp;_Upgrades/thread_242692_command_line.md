### Post by Jagot on 2006-08-24
When you type your password in a terminal you will not see the cursor move - just type the password and press enter and it will work.  It's a security feature that does catch a few people out.

There are lots of things you can do with the terminal - have a look here to get you started:

[http://linuxcommand.org/](http://linuxcommand.org/)

### Post by aysiu on 2006-08-24
Bottom line: when you enter your password, you'll see no indication that it's being accepted. It **is** being accepted, though. Just press *Enter* when you're done.

This is not a security feature; otherwise, they wouldn't give you visual feedback when you type in your password for Synaptic Package Manager either.

It's an inconsistency that the developers are aware of, but they think it's too complicated to fix it.

Read more [here](http://ubuntuforums.org/showthread.php?t=214393).
