### Post by qamelian on 2006-10-26
Yes, it will retain your settings. It's still a good idea to back up the contents of you home folder (including hidden files and folders) just in case.

### Post by qamelian on 2006-10-26
> **KDayz said:**
> @qamelian

  How would i go about doing that?
Well, you could just copy the entire home folder to another drive or, depending on the amount of data, burn it to CD. There are also a number of backup utilities in the repositories. One that I found very easy to use is called simplebackup. Just remember to grab all the file and folder whose names start with dots (.) as these general contain your personalized settings etc. You can make them visible in Nautilus by pressing Ctrl-H while in your home folder in Nautilus.
