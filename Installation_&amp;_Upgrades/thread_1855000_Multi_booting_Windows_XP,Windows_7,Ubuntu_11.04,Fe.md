### Post by oldfred on 2011-10-05
What have you installed so far? Windows requires primary partition for first install and best if second windows is also a primary  parition.

Most Linux work just find from logical partitions. 

Which system will you use the most. That may determine how you allocate space. You also need a NTFS shared data partition so Windows XP & 7 and all the other systems can easily share data. Swap can be shared, but do not share /home.
