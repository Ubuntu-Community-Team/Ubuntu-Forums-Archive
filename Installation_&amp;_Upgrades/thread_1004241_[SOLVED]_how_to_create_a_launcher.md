### Post by taurus on 2008-12-07
You can try creating a launcher for it on the top panel.  Place a cursor up there and press the right button.  Pick the first option, Creating Custom Launcher.  For the Command option (the second line), browse to where you unpack songbird and click on that.  Once you're done, you just have to click the icon on the panel to run it.
