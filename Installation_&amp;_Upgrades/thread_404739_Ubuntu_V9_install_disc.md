### Post by TravisNewman on 2007-04-08
Wubi installs Ubuntu from inside Windows, which is ok if you kept Windows... you'll actually need an Ubuntu cd to install Ubuntu by booting off of the cd. Go to ubuntu.com to download the cd image, and burn that image to a cd. You can't just write the one file, you have to use the "burn image file to disk" option to record the iso.
