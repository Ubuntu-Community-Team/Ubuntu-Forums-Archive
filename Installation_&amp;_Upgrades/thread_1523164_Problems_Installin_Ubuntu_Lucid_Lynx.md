### Post by Rubi1200 on 2010-07-03
There might be a problem with your keyboard settings. Do you use a USB keyboard?

2 options:

a) change the keyboard settings in BIOS; look for  *"Legacy USB Support"* or *"Legacy Keyboard Device*" and set to ON then save and try again

b) plug in another keyboard and see if it works with that
