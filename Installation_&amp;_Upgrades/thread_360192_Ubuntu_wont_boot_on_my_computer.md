### Post by meng on 2007-02-12
Immediately the computer starts to boot, you have to press a key to edit your BIOS settings, that's what will boot from the CD before the hard drive. The key could be del, esc, f1, f2, just about any key on the keyboard. If you're lucky, there'll be a message on the screen at boot time telling you what key you need to press in order to edit your BIOS settings. You have to then navigate the menu system to find where to change the order of devices your computer will boot from, and make sure the CD drive is first.
