### Post by Mark Phelps on 2009-03-02
You're probably NOT going to be able to install Vista without DVD media, because although you can mount and load it using some vistualization app or as an ISO, at some point in the install, it will reboot and look for the Vista stuff in a DVD, and since it rebooted, the ISO will no longer be mounted.
