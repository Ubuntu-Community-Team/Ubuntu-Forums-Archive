### Post by mastablasta on 2019-06-13
you don't have to make new installation of the OS because you changed the CPU. drivers in linux are loaded on each boot. system checks the hardware and loads appropriate drivers from kernel.  so you can just switch the hardware and it will boot. 

the only specific is proprietary driver (like Nvidia) where the kernel is patched with closed source driver.
