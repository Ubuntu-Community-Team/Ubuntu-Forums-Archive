### Post by snowpine on 2009-08-10
You don't even need to reinstall--you can add the CrunchBang desktop to your existing Xubuntu install using the alternative install script:

[http://www.crunchbanglinux.org/wiki/downloads#alternative_installation](http://www.crunchbanglinux.org/wiki/downloads#alternative_installation)

Log out and select Sessions->Openbox from the login screen.
