### Post by jl2533 on 2023-10-15
I ran into the same problem and although my system still functioned after reboot with ethernet network, apt continued to fail on this package when trying to install network-manager.

I could purge network-manager to restore apt capability, but to work around this issue I had to move the network connection causing the problem which perhaps /etc/netplan migration was having problems with?
```
sudo mv /etc/NetworkManager/system-connections/NetworkManager-USRP ~/NetworkManager-USRP.bak
```
