### Post by confused57 on 2006-07-09
You won't be able to upgrade from 5.10 to 6.06 using the shipit CD, which is the Dapper Desktop(Live) CD.
You'd have to use the Dapper alternate CD for upgrading to Dapper...do you have someone with a fast internet connection, who would download and burn you a CD from the alternate iso?

The Breezy shipit CD's contained a Live CD and an install CD, from which you could upgrade to Breezy from an earlier version.

The Dapper shipit CD is only the LiveCD(Desktop), which can be used for a fresh install, but not upgrading.

If Breezy is working well for you, I would recommend not being in a hurry to upgrade...in fact, Dapper may run slower on an older machine and I've seen posts on the Forum that Dapper doesn't support some older hardware that Breezy does.  If you're having problems with Breezy, then an upgrade may help.
