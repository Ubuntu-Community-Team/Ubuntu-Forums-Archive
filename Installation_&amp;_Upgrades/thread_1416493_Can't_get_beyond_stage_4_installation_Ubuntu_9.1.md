### Post by dstew on 2010-02-26
On step 4, did you choose the option "Erase and use entire disk"? It should prepare the disk for you, and show you what partitions it wants to use on the step 5 screen.

If step 4 stops, it might be you have a hardware problem with the computer. Have you tested the memory (opening menu option)? Did you check the disk for defects (another opening menu option)? Could the hard disk have some problems? To check the hard disk, on a computer with no operating system, you can use a disk analysis program from the hard disk manufacturer. Seagate has a bootable hard disk analysis and repair program that works on its drives, and also on the drives of other manufacturer's disks. It is called SeaTools. I have used the [SeaTools for DOS]("http://www.seagate.com/ww/v/index.jsp?locale=en-US&name=SeaTools&vgnextoid=720bd20cacdec010VgnVCM100000dd04090aRCRD") disk, and it helped me rescue a bad disk.
