### Post by mörgæs on 2014-10-24
Better to run ```
sudo apt-get dist-upgrade
``` because this command also upgrades the kernel. Reboot afterwards.

After the reboot ```
sudo apt-get autoremove
``` deletes old kernels, saving space.
