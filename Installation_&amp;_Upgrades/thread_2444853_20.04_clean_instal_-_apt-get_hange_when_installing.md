I'm doing a clean install of 20.04 and was attempting to install restricted-extras with the command:

```
sudo apt install unbuntu-restricted-extras 
```

As part of this it tries to run the ttf-mscorefonts-installer, which displays its EULA. I scroll through the EULA from top to bottom. and then try to highlight/click on <OK>, but nothing happens - the <OK> is not active and the installation is hung. All I can do is terminate the terminal session and kill the apt process, then  when I try again, i tells me to run 

```
sudo dpkg --configure a
```

which I do, but then whenever I try to install anything else (eg synaptic), it also tries to update the ttf installer and I'm back where I started. If I try to remove the ttf package it says that it can't do it as it is in a badly broken state, and to try re-installing it -- whereupon the circle begins again!

Any suggestions on how to fix this and an alternative way of installing the resitriced-extras package (without the TTF fonts, which I can get elsewhere, perhaps).

Use TAB, directional keys and ENTER.
