### Post by wilee-nilee on 2012-05-26
> **linuxsam777 said:**
> Why do all of the persistent USB ubuntu sticks always end up with input output error. Basically every time I use that they deteriorate after a while.

Not really enough info but a ISO loaded stick should basicaly not be updated, especially kernels, you are always still using the ISO.

The persistent does not clean out like a regular install, and is virtually impossible to clean out, it will fill up sooner or later. If you just delete it though you will have the stick working without a persistent.

A full install on a stick is a better use really, a 8 gig stick will run as long as it is still good if not filled up with extra stuff.
