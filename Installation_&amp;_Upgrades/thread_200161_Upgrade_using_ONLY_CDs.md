### Post by SpacePony on 2006-06-20
Wow. For starters you need to use the *Alternate install CD* to upgrade from Breezy to Drapper. Assuming that all works fine, but you have no grub, you can use a LiveCD to fix that. 

So you would download a LiveCD, burn it to CD, then once you've upgraded and lost your grub, boot up using the LiveCD and reinstall it.

What trouble are you having with the paritions?
What does the installer say it's doing when it crashes?
When you lose grub, is that after a successful Ubuntu upgrade?
How many times have you tried to upgrade?
