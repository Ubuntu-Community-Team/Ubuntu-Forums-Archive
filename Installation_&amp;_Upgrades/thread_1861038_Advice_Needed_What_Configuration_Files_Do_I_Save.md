### Post by oldfred on 2011-10-15
I backup all of /home but only copy a few config settings to a new install. But I have all my data except .wine for Picasa in other data or shared partitions.

If you have customized any system setting, you may want to back that up also, but do not just reinstall as new defaults may be different.

You want to backup a list of installed applications. But again there have been a lot of changes & you may want to edit it. For example you may not want to reinstall OpenOffice since LibreOffice is now the standard.

My backup is presuming a new install.
