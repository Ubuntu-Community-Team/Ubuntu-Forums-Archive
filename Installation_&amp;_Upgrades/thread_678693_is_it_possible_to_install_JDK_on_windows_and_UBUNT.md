### Post by luisromangz on 2008-01-26
You should download the JDK for Linux, from Ubuntu repositories. Ubuntu is a separate operating system from Windows, so you can't install Windows software (as is the JDK for Windows). Some Windows programs can be run in Linux using Wine, but that isn't the adviced procedure when there is a Linux version of a program.
