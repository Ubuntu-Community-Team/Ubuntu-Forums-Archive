### Post by mörgæs on 2011-05-24
Hi, welcome to the fora.

If you want to repair your 11.04 installation, running

```
sudo rm /var/lib/apt/lists/* -vf
sudo apt-get clean
sudo apt-get update
sudo apt-get upgrade

```should do the trick.

If you want 10.04 back, the only option is a fresh install.
