### Post by drphilngood on 2006-11-27
I upgraded to Breezy using the CD so I can see no reason why it shouldn´t work going from Breezy to Dapper.  Therefore, use this [guide]("https://help.ubuntu.com/community/DapperUpgrades").  Be sure to use the "Alternate Install CD" if you decide to upgrade from the CD(this is mentioned in the guide).

In addition, be sure to backup everything in preparation for the worst(a new install of Dapper over Breezy) and post back if you have any problems.

Finally and most importantly, _I am no expert!_  Consequently, you may want to wait for verification from one of the forum elders before beginning.

Good luck!
