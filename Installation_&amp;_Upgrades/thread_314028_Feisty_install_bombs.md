I downloaded the beta Feisty i386 desktop install CD, checked its MD5sum and burn to CDR using my laptop running Ubuntu 6.06. 

During booting, there are some error messages: 

gconfd (ubuntu-3610) : GConf server is not in use, shutting down
gconfd (ubuntu-3610) : Error releasing lockfile: Failed to link '/tmp/gconfd-ubuntu/lock/.... to '/tmp/gconfd-ubuntu/lock/ior': No such file or directory. 

But the boot appears to be OK. I see the install icon and double click on it. After setting the keyboard, time zone, language, I get a blank screen. Ctrl-Alt-BackSpace restarted GDM, and after 10 seconds, the ubuntu user is log'ed in again. Try the install icon and got the same result. 

That marks the end of my test of Feisty beta :-).
