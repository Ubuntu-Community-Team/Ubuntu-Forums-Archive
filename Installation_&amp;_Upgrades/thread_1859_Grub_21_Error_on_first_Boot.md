### Post by jeffjj on 2004-10-23
Solved my own problem...I switched from using the raid ports to just using the IDE ports on the mobo and it is currently loading up. I was not running raid anyway so I just took out the extra hard drive and switched to an IDE port...hopefully smooth sailing now!
