### Post by taurus on 2007-10-27
Don't let WinRAR unpack it.  The file that you have downloaded should be in .iso extension.  Then, use any burning software to burn it to a CD media as an image file, NOT a data file.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29)
