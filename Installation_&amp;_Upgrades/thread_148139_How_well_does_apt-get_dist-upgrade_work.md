### Post by oakz on 2006-03-21
My experience on 2 machines is that a dist-upgrade worked without any problems. On the other hand, I have friends who have had major problems with this approach, so try it at your own risk.

My 2 cents: It obviously depends on what you plan to use this machine for, but if its just for home use I would install the latest dapper flight CD. It is pretty stable by this point, I have not had any problems with it. I think you are more likely to have problems installing breezy and then dist-upgrading to dapper than if you just installed the latest dapper.
