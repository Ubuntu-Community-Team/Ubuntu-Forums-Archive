### Post by coffeecat on 2011-09-16
With 11.04 you can run the classic gnome desktop, almost the same as you have at the moment in Maverick. You can select either Unity or classic gnome at the login screen. You have to enter your username first, after which you can select either Unity (I think it says just "Ubuntu") or classic gnome at the bottom of the screen. Then enter your password and you are taken to your selected desktop which becomes your default one until such time as you log into the other.

If the default video driver does not support 3d with your graphics card, the system will fall back to classic gnome anyway.

Things will change in 11.10 with the transition to gnome 3. The default desktops will be Unity 3d and Unity 2d as a fallback. It will be possible to install a package to get a classic gnome near-lookalike but there will be functional differences.

Do give Unity at least a passing try. There are many (including myself) who did not like it at first, but who now much prefer it. There are a couple of links in my sig which you might find useful.

By the way, when posting a url, please do not enclose it in code tags. If you post it as-is, the forum software makes it a clickable link, which it is not in code tags.

Good luck! :)
