I have the same system. I use my 4GB ssd internal drive as root and bought an extra 4GB usb card for home. I run Arch and DWM on it. 

I have tried Lubuntu and it should run better being it's more lightweight. I had to build it with the [MinimalCD]("https://help.ubuntu.com/community/Installation/MinimalCD") though.
