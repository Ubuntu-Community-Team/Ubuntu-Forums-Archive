### Post by mick55 on 2009-10-13
You're using a beta version of Ubuntu, that's what gives.
Alpha's, beta's , Release Candidates are not final products.
You're lucky that your system is still working at all.

The next upgrade you do could magically fix your issues
or render your machine un-bootable.

These pre-release distributions should only be installed on
a spare computer or in a Virtual Machine.
The developers use these bug reports to make the final product
stable.The "fix" is to remove all the 9.10 stuff and go back to 9.04.

I am sure you don't want to have to do that so, you could wait 
for the next update, or boot into a previous kernel and 
see if it boots up quicker.

It just doesn't make sense to spend a lot of time troubleshooting
a beta distribution that could change tomorrow.

Good luck
mick
