### Post by spiderbatdad on 2010-01-25
That looks like it is for running linux inside windows. If you want to run linux from a bootable usb check out unetbootin.
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

From windows you would install the windows version obviously. Use it to create your bootable linux live-usb.

### Post by leorolla on 2010-01-25
The way I do it:

1. Run Ubuntu somewhere (otherwise Install Wubi and boot into it)

2. Use USB-Creator and and a Ubuntu Desktop ISO file to create an USB.

3. Boot from the USB, insert another USB and choose it as destination.

It takes ages and I don't know how portable it is in tems of different hardwares...
but it works.
