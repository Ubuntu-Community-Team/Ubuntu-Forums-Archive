I'm a long time redhat/fedora user and have just jumped to Ubuntu. But I'm experiencing a problem that I hope someone can help me with.

I have a fedora PC here which is running the current 2.3 (rpm) version of OO and it works fine.

So I'm wondering if others which have installed the current deb version of OpenOffice have experienced this problem. It is serious to me since I can't reliably move Impress-generated documents from one OpenOffice PC to another.
