### Post by Mark Phelps on 2014-03-13
Upgrading to a prerelease version of Ubuntu is generally a BAD idea unless you are highly experienced in fixing problems and prepared for the inevitable crashes and bugs that come with prerelease versions.

If you seriously want to consider such an "upgrade", you should first visit the Ubuntu+1 section of these forums to get an idea of what problems folks have been encountering running the 14.04 version.  And, if you have questions, please post them there -- this section is for Released versions of the OS.
