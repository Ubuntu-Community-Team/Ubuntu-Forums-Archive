### Post by Juergen on 2005-08-17
But you can read it from Windows? 
Did you burn it yourself? There should be more than one file on it...

I didn't have to try this myself, but this seems for you:
[https://wiki.ubuntu.com/Installation/FromWindows](https://wiki.ubuntu.com/Installation/FromWindows)
Try the CD approach.

Even if you don't exactly know what you are doing ;-) - just do what you are told.
(Of course you can omit donwloading the CD, since you have it already)

You might have to press F8 or F10 or any other F*-key (don't remember) to get to the Windows-Bootmenu.
You have to press the key **before** Windows starts booting, so press it (repeatedly) while the BIOS is still showing its messages.
If one key doesn't work, just reboot and try another - one of them should work.

### Post by Juergen on 2005-08-17
For almost all people it is just 'put CD in, boot, let installer do its work, ready'

Your box doesn't boot from this CD (does it boot from other CDs? Maybe try another CD-drive), so you have to do it like the wiki-entry says.
It is not as easy as clicking 'install.exe', but it is also not the normal way to install Ubuntu.

Some distros provide images for boot-floppys for situations like yours, but AFAIK Ubuntu doesn't.

Do you want to create a modified Install-CD for your app? That would require quite a bit of knowledge...
Or just give away a 2nd CD with your app?
