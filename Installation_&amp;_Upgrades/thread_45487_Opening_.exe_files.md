### Post by Devi0s on 2005-06-30
Natively, Ubuntu, nor any other Linux distribution, can not execute nor run .exe files.

However, using emulation programs like Wine, or by running Windows inside of linux with software like QEMU, VMWare, etc., it is possible.

Also, many .exe files you download that are self-etracting archive files can be decompressed (unzipped) by renaming the file.exe to file.zip, then using unzip as if it were just a .zip file to begin with.
