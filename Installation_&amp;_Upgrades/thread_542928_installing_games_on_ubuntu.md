### Post by Vadi on 2007-09-04
I'm downloading it right now, so I can guide you through it.

To answer your topic title though: Tons of ways. The easiest is click on Applications - Add/remove. See the game category on the left? Just check off any of games that you think you want to give a try, click apply, and voila. It'll download the game and install for you.

The second easiest is getdeb.net - allows you to also download games that aren't in add/remove, and installation is pretty easy too.

### Post by Vadi on 2007-09-04
I downloaded the .tar.bz2, right clicked, selected 'Extract Here'. It made an Assault_Cube folder. Opening that, the first thing that catches my eye is... README.html. Those things are quite important to read, so I read it - hmm, this is interesting:
Documentation
Basics

    * get started - before you play, basics, instructions on running the game 

Click there, I get...

 Running the game

Windows: Start the AssaultCube.bat

Linux: cd to the main AssaultCube directory and execute AssaultCube.sh 

I'm too lazy to use the terminal, so I just double-clicked on AssaultCube.sh. Selected 'run'... and nothing happened.

Okay, fine, I opened the terminal, used the 'cd' commands to get to the Assault_Cube folder, typed in ./assaultcube.sh, and it worked then.

Hope that helps. So you don't even need to install it - just run the assaultcube.sh from the terminal.
