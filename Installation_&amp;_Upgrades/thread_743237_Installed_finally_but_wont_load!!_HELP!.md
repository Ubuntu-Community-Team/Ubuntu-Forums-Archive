### Post by blazercist on 2008-04-02
when you get the option to press ESC for grub do so,  try recovery mode.  If that doesn't work then go back to grub and edit the regular grub line by removing quiet splash and then boot with that command.  you should now see verbose output instead of the splash screen, pay attention to the last couple of lines before the screen goes black and post them here.  also, you might want to list the hardware thats inside your laptop for us.  Mainly the graphics card and try doing a search for that graphics card model on the forums.
