### Post by Kadrus on 2007-11-18
Open Terminal and type this:
```
 sudo apt-get install sun-java6-jdk

```
and for Java applets that can be run in Mozilla Firefox type this:
```
sudo apt-get install sun-java6-plugin

```

I hope that Helped :)
