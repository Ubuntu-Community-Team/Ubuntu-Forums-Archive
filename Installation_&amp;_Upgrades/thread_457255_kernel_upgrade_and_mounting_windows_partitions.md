### Post by taurus on 2007-05-28
So you have two CD/DVD drives.  You can modify your /etc/fstab 

```
gksudo gedit /etc/fstab
```
and comment out by placing a # in front of the entry for your CDROM in there.

```
/dev/scd0 /media/cdrom0 udf,iso9660 user,noauto 0 0
```

Then, add these two entries for your two CD/DVD drives.

```
/dev/hdc /media/cdrom0 udf,iso9660 user,noauto 0 0
/dev/hdd /media/cdrom1 udf,iso9660 user,noauto 0 0
```
Save it and reboot.
