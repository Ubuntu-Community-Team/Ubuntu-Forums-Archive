### Post by kidders on 2007-04-19
Hi there and welcome,

Ubuntu may be the wrong distro for a PC that is *that* low-spec. There's no reason not to try, but you *might* find that your hardware isn't terribly well supported.

If I wanted to get Linux going on an old box, I'd probably opt for a more customisable distro that I could tweak & pare down a bit.

> **ramkal said:**
> Should I consider alternate install or xubuntu?To be honest, I probably wouldn't attempt to run Xorg at all, personally. It would be a tremendous waste of RAM, disk space & CPU cycles. Having said that, a Ubuntu install is quite quick ... *waaay* faster than something more customisable, like Gentoo, etc. ... so there's no harm in trying.
