### Post by Josh1 on 2007-04-20
Well this morning I was having an error with Ubuntu not authenticating when trying to do updates using the update manager to 7.04.

So I simply typed:
```
sudo gpg
```

Then ran update manager again to update to 7.04 and it worked fine! :-)
