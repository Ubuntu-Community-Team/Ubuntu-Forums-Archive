### Post by mjitkop on 2008-12-02
Hi, mechanicalmetal.

Actually, I am surprised to read that you installed XP after Ubuntu was already installed. I have read that usually Windows wants to be installed first for a dual boot to work. You may have been lucky. ;)

Anyway, as far as I know, for most people including myself, dual boots are created ny installing Ubuntu after Windows has already been installed first. I have done it several times without compromising my Windows installation.

All you have to do is start the installation process with the Live CD and make sure you do not format the Windows partition that will be detected automatically by the partitioner.

Good luck. :)
