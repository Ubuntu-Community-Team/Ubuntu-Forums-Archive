### Post by kyknos12 on 2017-03-21
try extract again some place for example /home/initrd and repack
cd /home/initrd
find . | cpio --create --'format=newc' | gzip > /home/initrd16.lz
but to operate the new changes will need to modify and the new changes in scripts orders
