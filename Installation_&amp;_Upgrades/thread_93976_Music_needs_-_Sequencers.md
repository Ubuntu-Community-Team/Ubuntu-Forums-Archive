### Post by 23meg on 2005-11-23
Both are in the repositories. ```
sudo apt-get install zynaddsubfx rosegarden
``` will install both. 

Read the following to learn about different ways of installing software in Ubuntu: [http://www.psychocats.net/linux/installingsoftware.php](http://www.psychocats.net/linux/installingsoftware.php)

And take a look at this site for some amazing audio apps you'll never find in Windows land: [http://www.linux-sound.org/](http://www.linux-sound.org/)
