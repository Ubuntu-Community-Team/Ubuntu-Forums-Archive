### Post by arvevans on 2008-04-28
At first glance, it sounds like you might not have burned your downloaded Linux onto CD as an "ISO Image".  You have to do that in order to add the bootable track to the CD.

Your CD burner program should have an option to make an "ISO Image" type CD.  It may be called "ISO 9660" or something else with "ISO" in the name.
