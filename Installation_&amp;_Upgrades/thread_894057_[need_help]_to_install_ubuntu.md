### Post by dstew on 2008-08-19
With respect to the laptop, you will have to make room for Ubuntu by shrinking the Vista partition. Use the Disk Management tools in Vista to do that. You will need about 30 Gb of space for Ubuntu.

With respect to your PC, sometimes the Advanced BIOS settings for disks can make them invisible to Ubuntu. Check to see if there are some BIOS disk settings that can be changed.
