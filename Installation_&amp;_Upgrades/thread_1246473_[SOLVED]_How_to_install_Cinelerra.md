### Post by wildhostile on 2009-08-22
Hi JoshStrobl,

You are about to install the official cinelerra version ==> [http://www.heroinewarrior.com/cinelerra.php](http://www.heroinewarrior.com/cinelerra.php) . For this version (cinelerra HV 4.0) you only have to download the binary archive which is a tar.bz2 file. If you untar this file it will give you a folder. In this folder just run ./cinelerra in a terminal.

On [www.cinelerra.org](www.cinelerra.org) there is another version of cinelerra which seems to be more stable (the 2.1 community version). For this one I advice you to compile it unless you want a .deb file as oldos2er suggest you.

Have fun.
