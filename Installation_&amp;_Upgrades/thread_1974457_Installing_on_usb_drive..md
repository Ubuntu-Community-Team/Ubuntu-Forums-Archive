### Post by oldfred on 2012-05-06
If it is a smaller flash drive, zvacet's link to persistent install is a good choice. But if larger you also can do a full install.

Pros & cons of persistent install over direct install to flashdrives - C.S.Cameron
[http://ubuntuforums.org/showthread.php?t=1655412](http://ubuntuforums.org/showthread.php?t=1655412)

If USB drive is flash or standard rotating drive, the install is the same, just for flash you want to make some settings like you would for a SSD to minimize writes - Flash is slow with writing.

Installer for the newer versions is slightly different but it is the same process.
It does not have to be encrypted BIOS based system:
Standard full install with screenshots to flash or SSD:
Ubuntu Encrypted Flash Memory  Installation
[http://members.iinet.net/~herman546/p19.html](http://members.iinet.net/~herman546/p19.html)

[http://ubuntuforums.org/showthread.php?t=1404664](http://ubuntuforums.org/showthread.php?t=1404664)

I have a full install of 12.04 in an 8GB partition on my 16GB flash drive. I do not have swap as I have 4GB RAM and do not load many apps at once.
