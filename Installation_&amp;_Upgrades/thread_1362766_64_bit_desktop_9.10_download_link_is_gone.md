### Post by avtolle on 2010-04-25
You will need to download the iso file (which I understand you have done), and then (my recommendation) check the md5sum of the iso, as explained in the links provided previously. Then, the iso needs to be burned as an image to a CD (personally, I use good quality CD-Rs) at a slow speed (4x is my preference). Then, as you did with the CD you received for the 32 bit version, you will boot from it and install the 64 bit system.
