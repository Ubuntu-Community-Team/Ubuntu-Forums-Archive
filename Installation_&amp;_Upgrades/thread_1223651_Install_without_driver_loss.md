### Post by Mark Phelps on 2009-07-26
> **Psycho.mario said:**
> ... Is there any way i can get around this, and install all the drivers?

Basically, no.  What you're asking, unless I misunderstand you, it to carry around a "portable Ubuntu" such that whatever machine you happen to connect to the hard drive, the OS will automatically have the correct hardware drivers.

That's just not feasible.  Even if you install it to a USB drive/stick, it' still going to install the drivers for the hardware it detects.

There's no way to install drivers for all possible hardware.
