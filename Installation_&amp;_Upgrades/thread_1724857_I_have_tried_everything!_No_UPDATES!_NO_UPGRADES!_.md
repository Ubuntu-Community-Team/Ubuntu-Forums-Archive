### Post by squidjam on 2013-03-17
Should you need to keep your old PC as it is, remember that EOL distros go to that special place in the cloud called "old-releases".

On synaptic, go to Settings >> Repositories, and click on the Other Software tab.

Edit any entry for repositories provided by Ubuntu and change the URI to [http://old-releases.ubuntu.com/ubuntu/](http://old-releases.ubuntu.com/ubuntu/)
keeping everything else the same.  Close it and open a terminal.

Do a sudo apt-get install -f to fix broken packages on the terminal (just in case).

Reopen synaptic and click Reload.

It should be noted though that this is not a best practice, and although you might have your own reasons for not doing so, you should upgrade... Better yet, do a clean install of a newer version.

Peace!
