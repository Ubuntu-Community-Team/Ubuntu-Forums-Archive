### Post by sudodus on 2013-12-13
Welcome to the Ubuntu Forums :-)

[http://www.pcadvisor.co.uk/reviews/laptop/3464776/compaq-cq58-review/](http://www.pcadvisor.co.uk/reviews/laptop/3464776/compaq-cq58-review/)

The previous link states that the computer is not very powerful, so I suggest that you run a lighter desktop environment than standard Ubuntu. Try Xubuntu or Lubuntu, and you will get a fast and responsive system.

Try 12.04 LTS for the main 'production' system, unless the drivers for graphics and wifi won't work. Then try the newest version, 13.10. LTS means long time support, and is available for Ubuntu, Xubuntu and Kubuntu (12.04).

You can make a multi-boot system and try several flavours and versions of Ubuntu as well as other operating systems. These test systems need only 8-12 GB (per system). The new system under development 'Trusty Tahr' will also get LTS. It can be found at [http://iso.qa.ubuntu.com/](http://iso.qa.ubuntu.com/) and [the Ubuntu +1 Forum]("http://ubuntuforums.org/forumdisplay.php?f=427")

### Post by sudodus on 2013-12-15
Well, the CQ58 will be good with a light desktop environment, but I would recommend a more powerful computer if you want to run standard Ubuntu with Unity or Kubuntu.

HP Envy M6 with Intel core i5 is fast enough for any of the Ubuntu flavours. AMD graphics will probably work fairly well with the built-in driver, but there are issues with the proprietary drivers for some of the chips/cards if you want advanced features. I have better experience of nvidia. I suggest that you check/ask in a separate thread about the graphics, particularly if you want advanced graphics.
