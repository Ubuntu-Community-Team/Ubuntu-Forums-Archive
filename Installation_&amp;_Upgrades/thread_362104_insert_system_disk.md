### Post by logos34 on 2007-02-15
Not sure about system disk message...Maybe make sure drives are set for 'auto' in bios.  If you're trying to run RAID then you need to read up on fakeraid and dmraid.

Don't forget to flash your bios with latest update.

**[https://wiki.ubuntu.com/Asus_P5NSLI?highlight=%28P5N%29](https://wiki.ubuntu.com/Asus_P5NSLI?highlight=%28P5N%29)**

Once you get up and running in Ubuntu you can run Passmark burn-in test afterward (free trial period offer).

**[http://www.passmark.com/products/bitlinux.htm](http://www.passmark.com/products/bitlinux.htm)**

A related board (P5NSLI) has some power mngmt and audio jack issues--might want to check this out:

**[http://support.asus.com/download/download.aspx?SLanguage=en-us&model=P5N-E%20SLI](http://support.asus.com/download/download.aspx?SLanguage=en-us&model=P5N-E%20SLI)**
