### Post by snowpine on 2013-04-04
Ubuntu 9.10 is "end of life" and completely unsupported since April 2011. I recommend that you perform a fresh reinstall of the current Ubuntu release following these directions (***make sure you first back up all files/documents to an external drive!***): [http://www.ubuntu.com/download/help/install-desktop-latest](http://www.ubuntu.com/download/help/install-desktop-latest)

If you are an intermediate user looking for an exciting and time-consuming upgrade adventure that probably won't work, you can try this: [https://help.ubuntu.com/community/EOLUpgrades](https://help.ubuntu.com/community/EOLUpgrades)

### Post by snowpine on 2013-04-04
Toshiba does not have any Ubuntu-certified hardware: [http://www.ubuntu.com/certification/desktop/](http://www.ubuntu.com/certification/desktop/)
But your machine exceeds the minimum hardware requirements: [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)
If you use the forum Search feature (top right of the page), you can search to see if any other users have attempted to install on Toshiba Satellite A200-AH6.
And/or you can run the Live DVD or Live USB to verify to your satisfaction whether the current Ubuntu is a good match for your hardware.
