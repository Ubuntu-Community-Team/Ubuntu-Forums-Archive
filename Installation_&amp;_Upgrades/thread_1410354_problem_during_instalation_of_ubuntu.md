### Post by darkod on 2010-02-18
Do you have USB keyboard?

If you do, go into BIOS and check whether USB Legacy support is enabled. It should be for USB keyboard to work properly before the OS is booted (after that the OS drivers take over, but not when you are at the cd boot menu).
