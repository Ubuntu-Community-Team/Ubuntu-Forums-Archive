### Post by GreatDanton on 2012-12-08
[QUOTE=N64Cam;12389877]cam@Cam:~$ sudo leafpad /etc/apt/sources.list
sudo: leafpad: command not found

Leafpad is installed on Lubuntu. I assume that you are using Ubuntu, so instead of leafpad use gedit. Also for graphical apps you should use gksudo instead of sudo

```
gksudo gedit /etc/apt/sources.list
```
