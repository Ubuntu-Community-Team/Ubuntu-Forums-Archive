### Post by rillip on 2007-05-01
If your home directory is it's own partition, then you can simply install from the CD, format the root partition, but not format the /home partition.  That will install Ubuntu over Fedora and leave your home directory intact and useable.

Me, personally, I would make a backup just in case. ;)
