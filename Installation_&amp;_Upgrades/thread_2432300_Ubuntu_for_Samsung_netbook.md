### Post by CatKiller on 2019-11-30
...maybe?

According to Intel's Ark that Atom is 64-bit, which is a good start. Support for 32-bit is being phased out. The 1 GB RAM will be limiting for many websites these days, but should be able to run something like Lubuntu OK. People did report that they were able to run Linux on that machine back in the day. There was something called samsung-tools that people used to handle quirks with the keyboard and so on, which [seems to be](https://launchpad.net/~voria/+archive/ubuntu/ppa) just about still alive.

If you go into it with realistic expectations it might be a fun project.

### Post by ajgreeny on 2019-11-30
It might be worth having a look at plain old Debian, using the netinstall version; a small download and it lets you add whatever you want; I chose the LXDE desktop package and left it as slim as that for a start, then just added packages as I needed them.

I am running it on the equivalent of the old Acer ee series netbooks with an Atom 27N (32bit) and 1G Ram, and whilst not a fast runner, it is certainly quicker than Lubuntu 18.04 was, and very usable.
