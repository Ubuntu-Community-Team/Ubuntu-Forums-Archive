### Post by stu644 on 2009-12-21
to download flash player go to [http://get.adobe.com/flashplayer/](http://get.adobe.com/flashplayer/) and select **.deb for Ubuntu 8.04+** then click **Agree and install now** a window will appear and ask you what action you want to take, select **Open with GDedi Package installer (default)**.  when it has opened click **Install Package** at the top right.  this should install flash player you may need to close and reopen firefox.
