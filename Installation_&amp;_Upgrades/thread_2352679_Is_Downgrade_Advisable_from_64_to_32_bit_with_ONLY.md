### Post by ajgreeny on 2017-02-14
I am pretty certain that the 1GB ram is your limiting factor, and I also recommend that you try Lubuntu instead of Xubuntu.

DO NOT DOWNGRADE TO 15.04, if that is the version you really meant, as it is no longer supported, and if you meant to type 14.04, that is supported only until April this year, so there is little point in moving to that.
Lubuntu 16.04, which I am using on an old Atom N270 netbook with 1GB ram very successfully, and it would be my No. 1 choice for your wife's machine as well.
