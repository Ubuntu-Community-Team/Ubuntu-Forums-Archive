### Post by HermanAB on 2007-10-28
Ubuntu Server edition is meant for use in a data centre where machines are installed headless and thus need no GUI, since they are administered remotely over SSH, which uses X forwarding for the GUI display on the local machine.

So if you wish to run a machine with a screen and keyboard attached, but wish to have maximum performance for networked applications, then use a light weight desktop system, such as Xubuntu.

Cheers,

Herman
