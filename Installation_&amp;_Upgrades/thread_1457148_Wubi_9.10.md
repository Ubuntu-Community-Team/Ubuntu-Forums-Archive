### Post by cleverusername on 2010-04-18
I'm not too sure what you mean here - what I understand is that you're installing Ubuntu 9.10 via Wubi on Windows, but when it tries to download the .iso to install, you cancelled it.

Is the install you did with no network access via Wubi or through a CD/DVD media?
EDIT: If done from a CD, did you install by booting from the CD or inside Windows through Wubi?

Also, if you have the Ubuntu 9.10 .iso in the same folder as your Wubi 9.10, it will skip the download and just use the .iso you provided, given that you choose the right flavour of ubuntu (Ubuntu 9.10, Kubuntu 9.10, Xubuntu 9.10, etc) from the drop-down box. If you're running Wubi from a Ubuntu 9.10 CD, it should be the same.

If your internet's too slow to download a 700mb iso, you could always request a free Ubuntu install cd from shipit ([https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)). Although I've never done this myself, I know people who have.
