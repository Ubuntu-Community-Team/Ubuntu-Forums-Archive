### Post by BobvanderPoel on 2008-09-09
Just thought I should post some of my experiences/fixes here on my recent upgrade from 7.10 to 8.04. I've seen some similar queries, so it is nice to know I'm not alone. This is all using kde 3.5.9 installed over the gnome-ubuntu (and that is a problem!)

1. Firefox-3 didn't support the mailto tag. 2 solutions:
   - use the binary from firefox
   - install the firefox-gnome support package.

I did both and can confirm that both methods work. I recommend the 2nd.

2. I could not log out of gnome or kde sessions. Had to hit ctrl-alt-backspace. Lots of reading and fooling around pointed to the keytouch program. Fix is to edit the file /etc/X11/Xsession.d and replace the line:

  KEYTOUCHLAUNCH=/usr/bin/keytouchd-launch

with

  KEYTOUCHLAUNCH="/usr/bin/keytouchd-launch --exit-with-session"

3. My dpms screen saver (to turn off the monitor after a set time) needed to be reset after every login/out. I don't understand this one, but here's the fix:

 - use the package manager to remove the gnome-power-manager package. This also removes the gnome-session-manager package. I seldom use gnome, so I don't care.

If someone can point me to a method to fix the problem I'll reinstall. I did try the "obvious" fix which is to enable the power saver in gnome, but no rewards.

Hope this helps someone!
