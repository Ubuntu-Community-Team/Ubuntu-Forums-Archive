### Post by aysiu on 2007-01-04
First of all, I would recommend **not** getting rid of Windows just yet.

Take it slow. I've seen far too many new users dive right into deleting Windows and then later regretting it.

I would use the live CD (also known as the Desktop CD) for a week or two, enough time to get familiar with the interface and also see what hardware detection problems you might run into later with a hard drive installation.

Once you decide to install Ubuntu, a dual-boot may be a good choice. That dual-boot could last you a month or maybe even a year before you decide to delete Windows completely.

Here are a couple of good dual boot guides:

Desktop CD - [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)
Alternate CD - [http://users.bigpond.net.au/hermanzone](http://users.bigpond.net.au/hermanzone)
