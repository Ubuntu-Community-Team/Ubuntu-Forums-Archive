### Post by grahammechanical on 2015-08-19
Yes you can. But, and it is a big but. The installation will most likely over-write the Ubuntu boot loader (Grub) and you will not be able to load Ubuntu. So, you better (a) back up your data; (b) investigate Boor Repair. 

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)

Boot Repair can repair the Ubuntu boot loader but if the install of Windows 10 erases the Ubuntu partition, then Boot Repair cannot repair that. So, be careful with the options you choose. That applies also to installing Ubuntu on a machine with Windows pre-installed. Make the wrong choice and we loose the existing OS.

There are several threads on this forum about windows 10. Check them out as well. There are times when I double check that I am actually viewing a Ubuntu forum because over the last few days I have seen so many posts about Windows 10. :)

### Post by Mark Phelps on 2015-08-20
First of all, Win10 does not come on a "CD" -- it is far too big to fit in 700MB; instead, you must have a DVD.

Second, unless you purchased an OEM or Retail version of Win10, you will NOT have a product key with which to activate Win10, and since MS has changed their activation method with Win10, there is no way you can enter a product key to get it activated.  IT will pop up a screen asking for a product key, but unless you have the OEM or Retail version on the DVD, nothing you enter will work.
