### Post by ian-weisser on 2014-09-28
We're all volunteers here, so please don't disparage us of 'not bothering' to help you.
No answers means either 'Insufficient information' or 'I don't know' or 'I'm in a different time zone.'

'Beeps' during POST, before the splash screen usually indicate a hardware fault.
Is there a pattern to the beeps? How many beeps are there? Do they repeat? Do any lights blink?
When exactly does it beep? During the BIOS screen? Before the Ubuntu splash screen? During the Ubuntu splash screen? While typing Xs? Some other time?

When exactly do you need to hit shift? How did you discover that? How many times do yo need to hit shift? How long does the system take to respond?

Did your install have any error messages?
If you boot from your install media, does the same problem occur?
