### Post by sudodus on 2016-07-13
Welcome to the Ubuntu Forums :-)

I don't know which key it should be, but found F2 in links to similar Samsung computers. Sometimes the UEFI menu is reached via the Escape key. Try to read whatever documentation (paper, on-line etc), and ask the company, where you bought the computer, or test all keys you can think of.

Sometimes you must press that key at once or during boot/reboot, a long continuous key-press, sometimes you must tap the key repeatedly (maybe twice per second) in order to issue a signal in a narrow time interval. These methods are different between computers, and you must find the way for your particular computer.

-o-

If you still cannot get into your UEFI/BIOS menu, you can [try to] install Ubuntu into the computer without getting into that menu.

You can remove the internal drive (hard disk drive or SSD), connect it to another computer, and install a basic Ubuntu system into it without any proprietary drivers. In most cases (but not all) an installed Ubuntu system is portable, so it is likely to work, when you move the internal drive back to your problematic Samsung computer.
