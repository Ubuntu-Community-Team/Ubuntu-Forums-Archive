### Post by MinstrelBoy on 2009-11-07
Hi there
I've got a shuttle with Win XP and wubi 9.04 dual boot.
I updated my wubi 9.04 to 9.10 with all working apart from the above.
Solved the shutdown problems by following this
==========================================
sudo edit /etc/rc6.d/S40umountfs
replace
fstab-decode umount -f -r -d $WEAK_MTPTS
with
fstab-decode umount -r -d $WEAK_MTPTS
and
fstab-decode umount -f -v -r -d $WEAK_MTPTS
with
fstab-decode umount -v -r -d $WEAK_MTPTS
==========================================
Partially solved the black wallpaper by choosing no visual effects. Desktop wallpaper now back again. My display card is an ATI 9200, maybe an update will fix this properly sometime ?
One other small issue, I use Fuse, the ZX Spectrum emulator and that keeps crashing under 9.10, any suggestions to cure this would be greatly appreciated.
