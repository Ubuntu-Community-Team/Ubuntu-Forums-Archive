### Post by smssms on 2012-07-18
Has anyone managed to install Ubuntu (10.04 LTS) on a Flagstone/ViaSat eclypt hardware encrypted hard drive?

I've got the ecylpt drive totally blank, initialised and user set up, but when logged into it I can't get the Ubuntu installation CD to 'kick in' and start the install...

---

### Post by smssms on 2012-07-20
SOLVED

Turns out it was my laptop's fault! It was cutting power to the HDD even in a 'soft' reboot, so I was losing the authentication to the encrypted HDD as I switched over to the install CD.

Just put the HDD in a different laptop to install the OS :)
