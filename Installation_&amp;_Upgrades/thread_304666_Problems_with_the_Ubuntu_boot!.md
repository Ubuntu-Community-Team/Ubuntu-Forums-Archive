### Post by ahill on 2006-11-22
Thought Id try out Unbuntu this morning since Ive just got a cable modem and can finally download big files!

Was having similar problems to the OP. Took me three goes to get Ubuntu to boot from the CD. After the first two I read this thread and decided to try just one more time before consigning the iso to the recycle bin and cursing Ununtu forever. Lucky I did.

First go, I just downloaded the iso and burnt it with Nero Express (on my windows machine). Tried booting it on three different machines (2 old dells and an old laptop) it would get as far as the first splash screen where you get to choose from the menu, then having selected the option to boot, on one machine it would load a bit then freeze, the other 2 would freeze for a couple of minutes before bringing up a dialog indicating an IO error reading the disk. All three machines gave this result selecting the option to check the CD.

Second go, I verified the md5 of the iso I had downloaded and then went and installed ImgBurn and tried burning it with that (at max speed). After burning it tried verifying and indicated that the cd had an error. Not surpringly it gave same symptoms as the first one.

Third time worked better. I selected 8x (and re-reading this thread see that even slower was recommended) and that CD burned ok. Took a look at the cd in windows this time (oh cool Abiword. Have to give that a shot some time!) and then threw it in one of the aforementioned dell boxes, and was succesfully able to boot it. 

Moral of the story: Try burning it as slowly as you can.

ftr: I used an AOPEN DUW1616/ARR onto Sony CD-Rs
