### Post by Dutch70 on 2011-04-03
Just boot up the live cd/usb, select "*Try Ubuntu*" to make sure it's going run on your hardware. You can also browse the web while installing from there.

If you're not dual booting with windows, just select "use entire disk" when you get to that option.
