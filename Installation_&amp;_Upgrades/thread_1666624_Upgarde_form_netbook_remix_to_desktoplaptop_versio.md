### Post by cipherboy_loc on 2011-01-13
Yeah. Open up the Terminal (Applications -> Accessories -> Terminal) and copy+paste:

```

sudo apt-get install ubuntu-desktop gnome

```

The first command installs the default ubuntu-desktop packages, the second installs GNOME. The first is what "makes" Ubuntu desktop edition a desktop edition, the second is just extra "fluff," which might be a waste of space for you, depending on your setup.
