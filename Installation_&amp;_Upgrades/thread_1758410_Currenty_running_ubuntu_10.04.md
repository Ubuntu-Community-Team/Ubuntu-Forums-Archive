### Post by Dutch70 on 2011-05-14
It's possible to install windows after Ubuntu, but it's much more difficult. Windows should be installed first. 

Here is a link you may find useful.
[[COLOR="Red"]RecoveringUbuntuAfterInstallingWindows[/COLOR]]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows")

You can find a few more by googling for "installing windows after ubuntu"

Make sure you install windows to a primary partition, not a logical partition.
