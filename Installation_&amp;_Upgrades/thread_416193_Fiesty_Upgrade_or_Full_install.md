I believe the upgrade and the full install are about the same size (since almost all the packages will be updated.) Maybe Upgrade might be even larger than the Full Install because it'll need to download updates for software you've install (which aren't installed by default). 

I would suggest to do a clean full install because each ubuntu version offers something new (often dramatic), and there is always a chance of packages being broken.  Sometimes it is just not worth the hassle.  (Of  course, your mileage will vary).

As for the first point, I believe that you CANNOT burn the Feisty Upgrade to a disc and install else-ware because the Upgrade download separate .deb files, not full ISOs.
