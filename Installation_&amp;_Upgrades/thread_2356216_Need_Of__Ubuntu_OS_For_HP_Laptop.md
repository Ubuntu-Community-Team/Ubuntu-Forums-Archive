### Post by ajgreeny on 2017-03-21
As this appears to be a very new HP model it is very likely that it suffers in the same way as many HP models when using UEFI rather than legacy BIOS, that being that the UEFI allows only bootloaders with a name which includes "windows" in the /boot/efi partition to boot an OS.

This is a non standard UEFI setup, but it is not a complete show-stopper.
There is much info on workarounds for this and other UEFI problems in the thread at [http://ubuntuforums.org/showthread.php?t=2147295](http://ubuntuforums.org/showthread.php?t=2147295) so read through that, looking in particular at the HP sections and you may find a way around your problem.
