### Post by QTB on 2010-08-13
My laptop does not support booting from esata, so is it possible to boot ubuntu and or grub from a usb drive and load another OS installation on the esata drive?

---

### Post by lemming465 on 2010-08-15
You'll have to have a boot partition on another hard drive which the BIOS likes, or USB stick or CD or something to boot from, but as long as your linux kernel recognizes the e-sata drive, you should otherwise be able to run Linux from it, even if you can't boot it directly.
