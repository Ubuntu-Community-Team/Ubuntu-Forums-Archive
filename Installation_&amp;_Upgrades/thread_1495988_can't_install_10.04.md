It's not enough to download the software (ISO) from the internet. You also have to burn that ISO image onto a CD _with a specific type of program_ that's capable of burning ISO specific files (images). Make sure that your CD is compatible with your CD/DVD drive too. For ISO burning software just google **free burning ISO windows** (or whatever your current OS is) and you should get plenty of solutions. Also, make sure that your bios is set up to boot from CD first and if possible make it a regular CD as opposed to a DVD disk.
