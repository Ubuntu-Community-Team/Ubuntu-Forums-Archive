### Post by dino99 on 2013-02-20
the issue is due to that theme:

plymouth-theme-mib-ubuntu 

so remove it:

sudo apt-get purge plymouth-theme-*
sudo apt-get install plymouth-theme-ubuntu-logo
sudo dpkg-reconfigure plymouth
