### Post by finer recliner on 2009-06-04
you will need more than vmware player to do this. you need a vmware server to host the virtual OS, and then vmware player to view the virtual OS (make sense?).

vmware does offer a free version of vmware server with many of the "enterprise" features stripped out.

an easier solution might be to install ubuntu using WUBI. this will install ubuntu on top of windows similar to any other windows application.
