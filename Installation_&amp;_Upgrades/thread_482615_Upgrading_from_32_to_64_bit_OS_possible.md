### Post by arvevans on 2007-06-23
If your Acer has a 64-bit CPU it should be able to run the 64-bit version of Ubuntu 7.10.  You will need to download an re-install most applications in order to get the 64-bit compiled versions.

I just completed building a new AMD X2 64-bit 3200 based computer and installed the 7.10 64-bit OS on it.  "Almost" all the regular applications from Ubuntu repositories will run fine on it.  I did find a couple of non-Ubuntu supported applications (Google Picasa, for example) that didn't work because they have never been compiled for 64-but architecture.

I used a 1 GB USB pen-drive to copy all of my /home/user_ID files (including hidden files) from the 32-bit machine to my new 64-bit one.  Everything comes up working just like it did on the old computer.
