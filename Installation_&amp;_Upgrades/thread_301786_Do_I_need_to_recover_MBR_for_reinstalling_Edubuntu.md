### Post by Joe_CoT on 2006-11-17
As far as booting, all reinstalling Edubuntu will do is reinstall grub. Since grub then loads the windows bootloader, no, shouldn't cause any issue.

The only issue is reinstalling windows, which overrides the boot loader ;)
