### Post by inlimbo on 2011-09-08
I was getting this message while trying to install ubuntu from a external USB cdrom:
Unable to find a medium containing a live file system.

It turns out I was using the USB 3.0 ports. I moved the cdrom to the black USB ports under the PS/2 port, and the installation worked like a charm! :D
