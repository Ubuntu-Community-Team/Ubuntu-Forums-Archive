For the sake of clarity. Are you able to run the live session? How well does the session run? If I was installing on that machine I would not tick the box "Install third party software." That option installs some proprietary audio & visual codecs and also a proprietary video driver. And it could be the case that the proprietary video drivers that come with 14.04 no longer support the video adapter in that machine.
