### Post by Rasa1111 on 2010-09-07
Hiyah Sugoi, 

You can find the Ubuntu releases here~ [http://releases.ubuntu.com/](http://releases.ubuntu.com/)

Lucid Lynx is probably the one you want, So ~ [http://releases.ubuntu.com/lucid/](http://releases.ubuntu.com/lucid/)

 The most common is titled "ubuntu-10.04.1-desktop-i386.iso"  
That is for 32 bit systems. 

Or if you want the same thing, but want to download it as a torrent (usually faster, better) instead, it would be titled 
"ubuntu-10.04.1-desktop-i386.iso.torrent " 

You just download that .iso image, Once it is downloaded, 
go to the file location, and it will be an ".iso" file, 
that file goes into your burning program to put on disc. 
burn it to a disc, with a program that burns .iso images...(the file you just downloaded)
And you are ready to use the CD. :)
