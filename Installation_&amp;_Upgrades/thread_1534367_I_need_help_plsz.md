### Post by RavensKrag on 2010-07-19
Did you double check you BIOS to make sure you can boot the XP CD? I have dual-booted many times now and have never had this problem.  Yes, I know you said you do not currently have a dual boot setup.

Are you trying to achieve a dual boot system are you returning to Windows?  For future reference, it is better to install Windows first when performing a dual boot setup, and then install Linux next to it.  

But this is because the Windows bootloader doesn't play nice with Linux.  I have not heard of a case where the opposite problem occurs.

Of course if I made the wrong assumptions I suppose you could ignore all of that ^_^;
