### Post by Mark Phelps on 2009-11-03
If you're getting a desktop, there IS a driver installed -- the open source driver -- the only one that will work with 9.04 or newer with the legacy cards.

As to the x2300, the newer ATI drivers might work -- there have been mixed reports.
