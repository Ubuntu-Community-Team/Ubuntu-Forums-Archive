### Post by racingronnie on 2008-03-01
I found the solution:

In the BIOS I went to:

Integrated Peripherals --> VIA OnChip IDE Devices --> SATA Mode = IDE

I changed it to:

Integrated Peripherals --> VIA OnChip IDE Devices --> SATA Mode = RAID

Now I am able to install 6.06 server A64 and x86 without problems.
