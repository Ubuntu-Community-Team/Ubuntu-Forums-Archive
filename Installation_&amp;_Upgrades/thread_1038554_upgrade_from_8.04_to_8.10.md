> **srtraveler said:**
> Is there a compelling reason to upgrade from 8.04 to 8.10? Also, when I use the Update Manager in my system/administration drop down, will it ever offer me this upgrade option?

This article might help answer your first question:
[http://www.ubuntulinux.org/news/ubuntu-8.10-desktop](http://www.ubuntulinux.org/news/ubuntu-8.10-desktop)

As to your second question, there have been a lot of issues with the upgrade. It's very picky - for example, it wouldn't install on this system until I purchased and installed a brand new DVD - even though my old DVD burned the CD successfully, and it passed the checksum test on the install and ran the LiveCD version. I too never saw an upgrade notice like I have for prior editions.

Upgrading did fix some issues for me. For example, in 8.04, a website I use, Visual Thesaurus, refused to function in any browser on either my desktop or my laptop. Once I upgraded to 8.10, it worked.

Overall, I'm pleased with 8.10.
