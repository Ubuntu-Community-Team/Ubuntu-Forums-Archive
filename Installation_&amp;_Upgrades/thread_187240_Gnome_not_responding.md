### Post by pellgarlic on 2006-06-07
hi jadm55,

are you using a "release candidate" cd, or the "long term support" cd (if you are not sure, look at the name of the iso file you downloaded to burn the cd - if it has "rc" at the end of it, it is a "release candidate"). the final release of dapper is now available, so first make sure you are using that, as the release candidates contain bugs which have been fixed in the final release.

you should also check that the md5 sum is correct - you can download programs for linux and for windows which check this, and you can download the text file containing the correct md5 sum from the same place you downloaded the cd iso from. this is a check to make sure the iso has not become corrupted during the download, as can happen sometimes.

when you double-click on the "install to hard drive" icon, does it freeze straight away? or does it ask you to set the geographical location and time first? at what point does it freeze?

another option to consider is to download the "alternate" cd iso, which cannot be used as a "Live" cd, but allows you to install using a different method (text-based), which some people find works when the "live" cd install doesn't.
