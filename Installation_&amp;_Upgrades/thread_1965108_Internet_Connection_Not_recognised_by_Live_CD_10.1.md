### Post by darkod on 2012-04-25
If there is no DHCP you will have to set the IP, subnet mask, gateway and DNS servers manually.

But first of all, are you aware that 10.10 is no longer supported? If you can, get hold of a newer version.

To set the network information manually you can click the Network Manager icon on top right, and select Edit Connections. On the wired tab select the connection and click on Edit (I think).

That will allow you to enter the necessary information manually.
