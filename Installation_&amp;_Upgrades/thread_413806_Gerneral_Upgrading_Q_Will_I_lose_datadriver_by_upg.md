### Post by growler on 2007-10-20
No, you should keep all settings and setup.

I downloaded the 7.10 alt.iso and burned the CD with Gnomebaker.. Put it in the drive to check (in Fiesty) and after the CD icon appeared on the desktop another window appears with "Do you want to upgrade...or something similar.

OK... I thought ..lets go!

If you go ahead, another window will ask if you want to use the internet for the update. I selected NO because Gutsy was brand new... and 40min later booted into the same desktop setup, home setup and all user files intact including working wireless!!!

The only "Catch" was desktop resolution of 640x480 and VESA video driver. I tried for about 10min to change to 1280x1024 and nvidia but settings wouldn't stay set????

Update manager then announced .."Updates available".... including a new nvidia driver...OK update away and reboot produces shiny new Gutsy that looks exactly the same as my Fiesty setup.
