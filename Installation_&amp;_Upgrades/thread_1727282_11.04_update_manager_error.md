### Post by plucky on 2011-04-30
> **xic816 said:**
> How can you delete the system files? O.o

```
sudo rm /var/lib/apt/lists/*
```

It will complain about removing a "partial" directory,just ignore it.Then run ```
sudo apt-get update
``` to reload the index files.

Good Luck
