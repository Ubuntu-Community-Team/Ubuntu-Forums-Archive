### Post by jerrrys on 2012-09-09
If you want to know what these files (or any package) are, install **Synaptic Package Manager**.  This program will give you a description of all your files.

[In terminal:]("https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal")

sudo apt-get install synaptic
