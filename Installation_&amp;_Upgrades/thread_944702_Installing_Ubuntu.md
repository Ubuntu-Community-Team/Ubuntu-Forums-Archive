### Post by cariboo on 2008-10-11
I'd check to see if all the hardware is supported before even thinking about changing the OS. Novell is know for using proprietary drivers in the OS's. If you find that every thing is supported, just install Ubuntu overtop of SUSE. When  you get to the partitoning stage, if you're not sure about partitioning, use automatically use the whole hard drive. and let the installation do it's thing.
