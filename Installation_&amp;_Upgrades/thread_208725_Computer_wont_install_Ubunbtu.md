### Post by Dr. Nick on 2006-07-04
They are all "premanent" versions in the sense that any of the 3 (server,desktop,alternate install) can all be installed to your hard drive

the desktop cd just lets you try ubuntu first without harming your computer. This is helpfull incase your computer doesnt support ubuntu you will find out before erasing anything. If it is supported and you like it then you can install it using the  exact same cd.

the other 2 versions you can not test before installing, so if they do not work then you may end up reinstalling what you used to use.
