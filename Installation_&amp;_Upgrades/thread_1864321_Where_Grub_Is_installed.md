### Post by westie457 on 2011-10-18
Bootinfoscript available from here [http://bootinfoscript.sourceforge.net/](http://bootinfoscript.sourceforge.net/) will tell you what is where. There is instructions on how to use it.

One way to upgrade your installation from a LiveCD/USB.

Boot up using the LiveCD in Try Mode. 
At the Live Desktop click on the install icon.
The first couple of screens are like a normal install.
At the where would you like to install screen you should get an option to upgrade. This will leave your personal files alone and reset the system to the new OS leaving Grub exactly where it was put the first time.

If you do not get the option to upgrade choose the 'Something else' option. If you are not sure what to do ask here for more help and guidance.
