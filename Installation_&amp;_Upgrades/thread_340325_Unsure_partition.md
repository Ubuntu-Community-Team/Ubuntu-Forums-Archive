### Post by derby007 on 2007-01-17
>>>I saw a partition picture showing windows - FAT32 - ubuntu - swap (is 7 - 42 - 10 - 1 setup alright?)
Is this your own picture or someone elses? i think my win partition is something like 14GB & Ubuntu takes up around 3 to 5GB & 1GB should do for swap. Ubuntu will be put on an 'EXT3' partition & SWAP on a 'SWAP' partition. Ideally win should be your 1st installation & then resize whats left for Ubuntu & Swap. The installation should set up a Grup boot manager for u, allowing u to choose which op system to boot into. good luck...
