### Post by rindi on 2013-07-18
It looks like I have found a bug in the Ubuntu Installer. When you install Ubuntu (or any other ubuntu based distro that uses the ubuntu installer) on a laptop which has a built in camera, the installer exits before it finishes, at the point at which it turns the camera on to take a user picture to show as icon at the logon prompt.

As there is no option anywhere to turn either the camera off, or to disable the use of the camera during the installation, my only option currently is to start the installation on a PC without camera, and then when finished move the HD back to the laptop I need the distro on.
