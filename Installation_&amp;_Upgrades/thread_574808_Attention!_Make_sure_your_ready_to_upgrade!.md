The new Ubuntu is coming soon and everyone needs to be prepared. The upgrade will be available in the update manager, but we need to be prepared in case something unexpected happens. Follow these steps to make sure your prepared.
[LIST=1]
[*]Make sure you have an installation for any version of linus available
[*]Back up all important documents, files, pictures, etc. to a local hard drive, floppy, or cd
[*]Make sure everything important is backed up
[*]After you've backed up everything, restart the computer
[/LIST]
After you've done that you're ready to upgrade to 7.10 (when it comes out). Follow these steps.
[LIST=1]
[*]Go to update manager
[*]Click the ok box where it says,"Upgrade to Ubuntu 7.10"(or something like that)
[*]Let it install
[*]Make sure you restart your computer when your done
[/LIST]
After you've installed 7.10 you have a few things to do
[LIST=1]
[*]Upload all the files you backed up(if you don't see them)
[*]Make sure you have internet
[*]Go on the ubuntu forums and talk about how cool the new 7.10 is
[/LIST]
Notice: 7.10 may look a lot like 7.04, that's because if you use this method, all your settings should be saved. If your not sure if you are running 7.10 you can simply click on the web browser and that will tell you which system your running(if you still have that as your homepage).

I hope this info is helpful.
