### Post by neurostu on 2008-01-31
Linux doesn't quite work the same way as windows.  When you install a program in windows it places it in c:\Program Files\<Program Name>, however, when you install in linux each program places its components where they are needed some in /usr/lib/ which contains libraries, some in /usr/bin/ which contain binaries  so it isn't as straight forward as a pc.

Hope this helps.
