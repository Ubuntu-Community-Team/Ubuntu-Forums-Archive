### Post by nad on 2005-04-28
There are three ways to indicate a default boot line to grub.

One: Put it first in the list.

Two: Use the 'default' option to indicate which line number in your list is your chosen default.

Three: Use the savedefault option.

Instructions are contained in the file that you need to edit, /boot/grub/menu.lst .

### Post by tonym on 2005-04-28
This is controlled by /boot/grub/menu.lst

The default is set by the default command  - see [http://www.gnu.org/software/grub/manual/html_node/default.html#default](http://www.gnu.org/software/grub/manual/html_node/default.html#default)

The easiest thing is probably to move the entry you want to the top but be carefull of the group of Ubuntu entries that are autogenerated.
