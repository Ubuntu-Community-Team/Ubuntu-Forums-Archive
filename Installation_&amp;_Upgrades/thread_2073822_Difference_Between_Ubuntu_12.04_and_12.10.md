### Post by Frogs Hair on 2012-10-20
Hello and Welcome

I don't upgrade but do install the latest releases (my choice). If you are happy with your current  setup don't bother because 12.04 will be supported much longer than 12.10. Each new release comes with the latest Ubuntu modified kernel so that is a reason some upgrade or change releases. The newer kernel sometimes includes features people need.

Although  upgrades work fine for some users you don't have to look very hard on the forum to find numerous problems associated with them. Old familiar programs often disappear with each new version.    If you have 12.04 PPAS chances are many won't work in 12.10 and the same is true for themes because 12.10 uses Gnome 3.6 and 12.04 uses 3.4.


         
See the link for changes [http://www.omgubuntu.co.uk/2012/10/ubuntu-12-10-released](http://www.omgubuntu.co.uk/2012/10/ubuntu-12-10-released)
