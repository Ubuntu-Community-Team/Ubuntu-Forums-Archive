### Post by robert shearer on 2012-12-17
There are a great number of RAW processing applications available and depending on your own personal workflow one or more of them will suit.

I use ufraw from the ubuntu repo.
Rawtherapee from the ppa is preferred by many.

Open the Ubuntu software centre and type 'raw' to see others.

Sometimes you have to just use several and then choose the one that works for your needs.
Install some, try them, then uninstall the ones that you like least and work with the one that suits **you** best.

It is safe although you may hear people telling you to stick to the outdated software in the repo for "stability". I find the fear unfound as long as you are careful in case an update needs to remove something.

It is a good idea to install ppa-purge in the rare event that you need to remove the ppa and downgrade your packages (say when a conflict occurs)

To be honest the flexibility of ppa is a main reason I use Ubuntu as my main OS, otherwise I could use Fedora or Debian.
