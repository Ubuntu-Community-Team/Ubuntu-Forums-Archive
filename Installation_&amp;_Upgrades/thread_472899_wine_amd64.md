### Post by sams2100 on 2007-06-18
Well if you are able to run the internal wine apps then I would assume you have Wine installed.  What you may not know is that Wine does not run all windows apps, it only runs a few of the major ones such as IE, Photoshop, World of Warcraft.  Sometimes you get lucky and it will run what you need, other times it will just spit out a bunch of errors and fail.  If it fails and you need the windows app, then you'll have to install something like VMWare and just run native windows through virtualization.
