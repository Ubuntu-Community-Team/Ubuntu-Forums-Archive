### Post by jollysnowman on 2009-11-28
Use the Ubuntu-minimal CD. [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD) From here, you can install gnome, kde, or xfce.

If you have the Ubuntu desktop CD, just do ```
sudo apt-get install xubuntu-desktop
``` But now you'll be left with a lot of leftover stuff from regular Ubuntu.
