### Post by YesWeCan on 2011-09-09
You need to format sda5 as swap not as ext4.

Also edit the fstab file in sda1 to remove the encrypted swap entry:

sudo mount /dev/sda1 /mnt
gksudo gedit /mnt/etc/fstab

then add a # as I have shown in red to comment out that line. Or just delete the line. Then add a new line for sda5

```
# /etc/fstab: static file system information.
#
# Use 'blkid -o value -s UUID' to print the universally unique identifier
# for a device; this may be used with UUID= as a more robust way to name
# devices that works even if disks are added and removed. See fstab(5).
#
# <file system>                               <mount point>   <type>  <options>           <dump>  <pass>
proc                                              /proc        proc    nodev,noexec,nosuid    0       0
# / was on /dev/sda1 during installation
UUID=cf83a87f-cae0-452c-881e-5aa63a4b3b99         /            ext4    errors=remount-ro      0       1
# swap was on /dev/sda5 during installation
#UUID=d1802771-9a23-4b52-bdde-ad53f28458ab        none         swap    sw                     0       0
/dev/fd0                                          /media/floppy0  auto    rw,user,noauto,exec,utf8 0       0
[COLOR="Red"]#[/COLOR]/dev/mapper/cryptswap1 none swap sw 0 0

[COLOR="red"]/dev/sda5                                         none         swap    sw                     0    0[/COLOR]
```

Save the changes.
Reboot.
