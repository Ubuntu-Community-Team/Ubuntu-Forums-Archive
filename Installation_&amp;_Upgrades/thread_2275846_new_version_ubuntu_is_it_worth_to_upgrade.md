### Post by grahammechanical on 2015-04-28
When Ubuntu 16.04 is released software updater will give you an option to upgrade to 16.04. But we cannot upgrade from 14.04 to 15.04. First we have to upgrade to 14.10 and then we will get the option to upgrade to 15.04.

You need to change a setting in software & updates so that you get notified of the next Ubuntu release instead of the next LTS release.
