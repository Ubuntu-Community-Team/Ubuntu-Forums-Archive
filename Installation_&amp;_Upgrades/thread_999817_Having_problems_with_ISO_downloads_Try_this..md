### Post by forkandles on 2008-12-02
For those people who experience interrupted or bad downloads of Ubuntu ISOs, try using d4x and Flashgot.

d4x (Downloader for X) is a download manager and is available via Synaptic.

Flashgot is available via Firefox Add-ons (Firefox > Tools > Add-ons (Search for Flashgot).

I used to have problems with some ISO downloads in the past but using this combo means my downloads are perfect every time.
In fact, if you go to the Debian website, you will be strongly advised to use a download manager to avoid such problems.
