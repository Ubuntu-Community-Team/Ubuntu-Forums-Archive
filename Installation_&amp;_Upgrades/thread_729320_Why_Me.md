### Post by Ganymedes on 2008-03-19
If the graphics card is not supported at all, then it won't boot to the Live CD - I had that with 3dLabs Oxygene VX-1. Try with the alternate CD or better still, try with another graphics card.

Also if the memory chips are faulty, Windows might install well - the problem arise later. Try the memory checks with Ubuntu.
