### Post by K.Mandla on 2006-12-14
It'll install. Or it should, but as you have discovered, there can be flukes.

Next time you try it (if you decide to try it again :roll: ), press ALT+F4 or one of the other function keys. One of those should give you an installation log and tell you if Ubuntu froze up and why.

P.S.: If you're really having trouble with 6.10, try 6.06.1. It has a reputation of being less ... finicky ... than 6.10.
