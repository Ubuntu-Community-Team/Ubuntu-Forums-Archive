### Post by Yardarm on 2008-12-18
I think Ubuntu Ultimate is a fork from Ubuntu and hence there would be no migration path, but you could certainly do a fresh install.  However, since it is probably based off of Ubuntu with a few package changes and I doubt it has made any changes to the packages themselves, you'll probably have the same issues you have now.

If you want a true fresh perspective of compiz, how about trying a different branch of Linux like Fedora or SuSE?

### Post by Yardarm on 2008-12-18
You can't go to the update manager and tell it you want to upgrade to Ubuntu Ultimate.

But you could download the ISO and install it over top of (or along side) your existing Ubuntu installation.  It looks like the home for that is here: [http://ultimateedition.info/]("http://ultimateedition.info/")
