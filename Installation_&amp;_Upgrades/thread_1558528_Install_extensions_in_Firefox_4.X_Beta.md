### Post by lovinglinux on 2010-08-22
[http://firefox-tutorials.blogspot.com/2010/08/firefox-4-download-install-customize.html](http://firefox-tutorials.blogspot.com/2010/08/firefox-4-download-install-customize.html)

What you are looking for is in the Add-on Compatibility section.

Keep in mind [NoScript]("https://addons.mozilla.org/en-US/firefox/addon/722/") and [Adblock Plus]("https://addons.mozilla.org/en-US/firefox/addon/1865/") are both compatible with the latest Beta, so you don't need to override compatibility if you don't use other extensions.
