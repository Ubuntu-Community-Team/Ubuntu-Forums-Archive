### Post by luxerama on 2006-06-28
If you are trying to install Ubuntu 6.06 LTS (Dapper Drake) then it is normal that it boots the live cd first. Once booted up you should see the option "install to hard drive" (or something like that). It is positioned on the desktop and looks a bit like a hard drive symbol. Once you open that the installer should load.
Hope that solves it.
