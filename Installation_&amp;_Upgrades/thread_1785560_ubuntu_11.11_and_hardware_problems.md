### Post by TenPlus1 on 2011-06-18
Run System -> Administration -> Hardware Drivers and install your gfx card (nvidia/ati) and reboot, it should go straight into Unity, failing that select it from the session box during log-on...

---

### Post by Quackers on 2011-06-18
Run Additional Drivers to see if any graphics card drivers are available for your card. If so install them and reboot. Unity may be waiting for you when you get back :-)
