### Post by 67GTA on 2007-04-29
Enable the driver
Open a terminal and type "sudo gedit /etc/X11/xorg.conf"

Then look for the section that looks like this:
Section "Device"
	Identifier	"ATI RADEON X1600"
	Driver		"fglrx"
	BusID		"PCI:1:0:0"
	VideoRam	524288
	Option		"UseFBDev"		"true"

Make sure the driver is fglrx as mine is save, and reboot

### Post by applefreakpeeps on 2007-06-28
hey chrisN_uk

1. To find out which version of driver u have installed, open a terminal window and type fglrxinfo. That will tell you the version of fglrx u have (Note this will work only if your DRI is working else you will get information only about the MESA renderer). The other way is to open Catalyst Control Center/Old Ati Control Panel and check.

2. To install the latest ATI Driver manually follow these steps

a) Download ATI driver (Version 8.38.6) from the ATI site as a .run file

b) Install the needed tools for generating packages using apt: 

sudo apt-get install module-assistant build-essential debhelper debconf dh-make fakeroot libstdc++5 linux-headers-generic

c) Generate the packages after switching to the directory containing the run file

sudo bash ati-driver-installer-8.38.6*.run --buildpkg Ubuntu/feisty

d) Install the generated packages

sudo dpkg -i xorg-driver-fglrx_8.38.6-1*.deb \
fglrx-kernel-source_8.38.6-1*.deb \
fglrx-amdcccle_8.38.6-1*.deb \

e) Remove any old drivers from /usr/src

sudo rm /usr/src/fglrx-kernel*.deb

f) Compile the kernel module

sudo module-assistant prepare
sudo module-assistant update
sudo module-assistant build fglrx
sudo module-assistant install fglrx
sudo depmod -a

g) Create the following folder and the symbolic link to the fglrx kernel module

sudo mkdir /lib/modules/$(uname -r)/volatile
sudo ln -s /lib/modules/$(uname -r)/misc/fglrx.ko /lib/modules/$(uname -r)/volatile/fglrx.ko

h) Configure the xorg.conf file

sudo aticonfig --initial

i) Restart X-server by pressing Ctrl+Alt+Backspace

j) Open a new terminal and type fglrxinfo. You should get something like

$ fglrxinfo
display: :0.0  screen: 0
OpenGL vendor string: ATI Technologies Inc.
OpenGL renderer string: Radeon X1800 Series Generic
OpenGL version string: 2.0.6474 (8.38.6)
