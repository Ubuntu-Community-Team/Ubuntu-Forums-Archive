### Post by bulldog on 2006-12-15
No you won't :D 
First try booting in recovery mode and when X-server is crashing enter your way trough the errors and login.
You're now in a console.
```
sudo dpkg-reconfigure -phigh  xserver-xorg
``` and try starting X again
```
startx
```
If this doesn't do the trick you can try ```
sudo depmod -ae
```I read this on the forum,but I don't know what the command does,but it helped people with similar problems.

There's a topic about this update problem you should take a look at it,maybe you can get better help from there,because I'm not really familiar with ATI graphics.

[http://ubuntuforums.org/showthread.php?t=318206](http://ubuntuforums.org/showthread.php?t=318206)
