The Ubuntu install disk can do this for you.

If you want to manage your partitions, I suggest using GParted, see the link in my sig. It is very easy to use.

To do this from Windows, under Computer Management->Disk Management, you can alter Windows partitions.

Search for "Computer Management" if you don't know where it is. I don't use XP, and when I did, I had my own mmc for it.
