Where did you download the file from ?  If it is a zip file then I "think" there is a possibility that you have downloaded the wrong file.

I believe that if you download it from the Ubuntu website, the file that you wind up with will have an ISO extension.  Then all you have to do is BURN that ISO image file to a CD, which will in turn be a BOOTABLE Ubuntu CD installation CD.

Make sure that the FIRST thing that you do when you boot to the Ubuntu CD is to run the CD integrity check function that is found on the initial Ubuntu boot screen menu.
