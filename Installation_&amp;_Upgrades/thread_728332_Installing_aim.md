### Post by Existentialist on 2008-03-19
I think most people would recommend not using aim, especially with linux.  There is a free alternative pidgin which works with the aim protocol along with the msn, yahoo, and a few others.  Once you use it I'm sure you see why people don't use aim.  You can install it by searching in the programs list for pidgin, or:

sudo apt-get install pidgin
