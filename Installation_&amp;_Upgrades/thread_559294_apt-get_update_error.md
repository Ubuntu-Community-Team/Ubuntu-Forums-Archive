### Post by zvacet on 2007-09-25
```
gksudo gedit /etc/apt/sources.list
```

Change [http://in.archive.ubuntu.com](http://in.archive.ubuntu.com)  to ** [http://archive.ubuntu.com](http://archive.ubuntu.com)**

Do it with all lines.Save the file.

```
sudo aptitude update
```
