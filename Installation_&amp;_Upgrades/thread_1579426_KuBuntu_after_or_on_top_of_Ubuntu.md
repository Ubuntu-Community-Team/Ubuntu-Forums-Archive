### Post by wilee-nilee on 2010-09-21
Ubuntu and Kubuntu are just desktops you can have both in the same OS. You would just choose which one at login that you wanted. You will also see some of programs in each desktop in the other, that would not be there on a single install, but are generally cross compatible.

Here is a link for removing and installing full desktops.
Look to the bottom left corner in the panel for playing around. Make sure that if you use this that you are choosing the distro your running.
[http://psychocats.net/ubuntu/index.php](http://psychocats.net/ubuntu/index.php)

Not sure if this answers your question. To know if you can add more partitions post the output of this from the terminal.
Code: sudo fdisk -l
