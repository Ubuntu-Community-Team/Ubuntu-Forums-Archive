### Post by TenPlus1 on 2008-08-29
Are you using the 32-bit or 64-bit LiveCD ?  and did you download/burn the .iso file for it...

Personally I'd use the 32-bit LiveCD for installation as it works better with 64-bit systems and is more compatible with things like flash/java/certain software...

If you are downloading the .iso, use a torrent program like Deluge and Force Recheck after it is completed (to make sure it's all there), and burn it at low-speed (4x/8x) for a safe install cd...
