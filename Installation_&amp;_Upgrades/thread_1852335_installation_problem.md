### Post by Mark Phelps on 2011-10-01
> **mahi08it020 said:**
> i have installed ubuntu in windows but now windows is expired....,if i reinstall windows then ubuntu will be currept or not..

Ubuntu installed inside Windows is treated like any other application by Windows, meaning, if you reinstall Windows, you will have to reinstall ALL your apps, including Ubuntu.

Also, don't understand about Windows being "expired".  The only Windows licenses that have time expirations are pre-release, and they generally can't be extended.

If you're doing the 30-days-without-a-key thing, you're going to run up against a limit because the rearm command that is used to extended that can only be used a few times.
