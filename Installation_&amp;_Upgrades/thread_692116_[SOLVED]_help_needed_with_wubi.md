### Post by jan quark on 2008-02-09
I do not recommend wubi

sry for starting my answer with this :)

but wubi is "only" a virtualization not a correct "hard" install to the drive
so do not expect the stability of a hard drive installation

installing ubuntu as the second OS after Vista is not a problem
ubuntu will not make any harm to your vista system, the grub boot loader will recognize that there is another system installed and will implement vista into the boot process, so you can switch between the two operating systems

have a look at this guide, describing a dual boot system vista - ubuntu

[http://apcmag.com/5046/how_to_dual_boot_vista_with_linux_vista_installed_first](http://apcmag.com/5046/how_to_dual_boot_vista_with_linux_vista_installed_first)
