bcf748498cb6b07003b27b37c90ec019 *ubuntu-7.04-beta-desktop-i386.iso

and make sure that your burning it as an image and not as a file

### Post by wpshooter on 2007-04-06
BURN the ISO image at 4X or less.

As far as checking it is concerned, it is much more important, IMO, to check the integrity of the CD by running the verification function that is found on the initial Ubuntu boot screen menu, than checking the ISO sumcheck.

Good luck.
