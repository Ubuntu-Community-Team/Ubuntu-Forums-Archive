### Post by Kobalt on 2007-05-30
You should first make a working xorg.conf file to be able to have a graphical session. To do so, when you get that error message press Ctrl+Alt+F1 and log in. Enter ```
sudo dpkg-reconfigure xserver-xorg
```. Answer the questions (leave default choice when you don't know) and reboot. Now you should have a graphical session back. From there you can reinstall properly you nVidia drivers and then restore your old xorg.conf settings to get back your dual monitor working.
