### Post by por100pre1 on 2007-10-29
This will transform Gutsy to Ubuntu Studio Gutsy with all its software:

```
sudo aptitude install ubuntustudio-desktop ubuntustudio-audio ubuntustudio-audio-plugins ubuntustudio-graphics ubuntustudio-video
```

If you just want the looks:

```
sudo aptitude install ubuntustudio-look
```

If you prefer the Ubuntu Studio Feisty looks download this:

[http://ubuntustudio.org/files/ubuntustudio-feisty-art_0.1_all.deb](http://ubuntustudio.org/files/ubuntustudio-feisty-art_0.1_all.deb)

Let us know if this works or not.
