Hello folks,

I recently partitioned my drive in order to dual boot Vista Premium and Ubuntu 7.10 on the same hard drive. 

I have an Acer Aspire T690, with:
- Intel 2140 Dual Core Processor
- Nvidia Geforce 8400 GS Video Card

My moniter is a Samsung Syncmaster 226BW with a native resolution of 1680 x 1050

I booted the LiveCD and Ubuntu loaded and ran perfectly. So, I installed Ubuntu from the LiveCD. GRUB worked fine, and it let me choose to load Ubuntu.

I did, and instead of loading, the screen flickers a few times and I get this message repeated about a trillion times:

> "serial 8250: too much work for irq18"

I think it might have to do with my graphics drivers, but I'm new at this and I don't know how to check. Graphics-Safe mode resulted in a bunch of flickering. The mode tried to load in 1600 x 1200, and my monitor would not let it, and it eventually failed.

I'm not really sure what to do at this point, and its a shame because I was really pumped about getting into Ubuntu. 

Any suggestions would be appreciated!!! Thanks in advance!

### Post by kellemes on 2008-04-09
Boot into recovery-mode and type the following to try to get your graphics running..
```
dpkg-reconfigure xserver-xorg
```
