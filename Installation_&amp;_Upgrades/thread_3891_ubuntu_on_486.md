### Post by jdong on 2004-11-10
Ubuntu is compiled for the 486 architecture and optimized for Pentium 4's.

So yes, it'll work; however, the installer wants more than 32MB (not 32mb, more) of RAM, so you'd have better luck installing + setting up swap space on a newer computer, than placing the hard drive back.
