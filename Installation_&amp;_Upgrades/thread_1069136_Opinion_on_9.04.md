Ubuntu, Kubuntu, Xubuntu and Edubuntu releases are always released on the same day. However talking about Xubuntu and old machines...Xubuntu is not light at all! Xubuntu is all about choosing to use XFCE, which is lighter then GNOME because it has less applications installed by default.

Xfce uses GTK+, and that does actually make it a fork of GNOME. Xfce is modyfied GNOME with less applications by default. Mousepad in example is just a modyfied version of Gedit which contains less features.
