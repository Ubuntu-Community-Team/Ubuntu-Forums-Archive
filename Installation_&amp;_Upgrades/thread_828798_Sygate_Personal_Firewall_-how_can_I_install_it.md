### Post by colo on 2008-06-14
You can't, as it is a Windows-only app. But rest assured you do not need a desktop personal firewall when running GNU/Linux (which Ubuntu is a nice variant of). That's reserved for people running software they can't trust, and that does not really apply to an operating system where all the software you run is downloaded from reviewed, canonical repositories.

---

### Post by ajgreeny on 2008-06-14
Colo's right, you don't need to add anything to Ubuntu to get a firewall, it has one running by default (iptables) which keeps all ports invisible until you ask them to do something, such as use the internet or email, etc etc.
As I've said elsewhere, if you're not running a server system, forget your windows experiences and go with the safety of linux and Ubuntu.  You don't need virus checking software either; both superfluous.
