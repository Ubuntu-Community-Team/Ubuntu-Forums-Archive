### Post by merlinus on 2007-09-11
You should back up everything in /home, and make sure Show Hidden Files is activated.

Barring that, you might use gparted live cd to shrink your ubuntu partition and make another, and copy the data there.

Then on reinstall, do not format the new partition.

[http://gparted.sourceforge.net/](http://gparted.sourceforge.net/)
