### Post by vedek on 2009-08-13
with firefox there is another option. you can download a little script call ubuntuzilla and that will automatically keep you updated.

[http://sourceforge.net/projects/ubuntuzilla/files/](http://sourceforge.net/projects/ubuntuzilla/files/) ----- this is the file 

Or type

```
sudo apt-get install ubuntuzilla
```it will prompt you for your login password enter that then it will download then you need to type in the following

```
ubuntuzilla.py -a install -p firefox
```type it in as it appears and you will be sweet.
