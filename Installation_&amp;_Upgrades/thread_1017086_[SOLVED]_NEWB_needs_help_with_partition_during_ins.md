### Post by StevenHarper on 2008-12-20
Hi, if you are trying to setup the computer so that Ubuntu is the only operating system and you no longer want to keep any other operating systems then you want to select the Use Entire disk option on the 3rd stage of the installer.

This WILL kill all the data on the disk you choose.

It sounds like you have tried to re-size an old windows partition, this isn't what you want to do.

Let the install remove ALL the data and make a new file structure on the disk.
