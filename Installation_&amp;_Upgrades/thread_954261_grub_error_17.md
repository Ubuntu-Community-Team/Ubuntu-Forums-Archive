### Post by caljohnsmith on 2008-10-21
When you set up your partitions in the installer, you have to specify one of them for your main Ubuntu file system, which has mount point of "/". So for whichever partition you want for your main Ubuntu install, just set the mount point on that partition as "/" and you shouldn't get that error.
