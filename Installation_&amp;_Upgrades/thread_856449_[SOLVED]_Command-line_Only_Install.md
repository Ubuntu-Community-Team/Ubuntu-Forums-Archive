### Post by Bradl3yJ on 2008-07-11
I press F6 for other options, and it just brings up Boot Options field. Do i need to append cli to the end of that or what?

EDIT: I believe I have found it. Press F4 for "Modes" rather than F6 for "Other Options". Under modes there is a CLI only install options.

EDIT: CONFIRMED. F4 for "Modes", then the Command Line Only option is exactly what I was looking for. Thank you.
