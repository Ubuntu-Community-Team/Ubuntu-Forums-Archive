### Post by y0diggity on 2008-12-28
Go to System-->preferences-->Main Menu and select "add".
This will allow you to add a new launcher to your menu. type in the Name you want it to display as, as well as the command to launch it from the terminal - probably "squash" - and you can even select an icon for it.

Hope that helped.
