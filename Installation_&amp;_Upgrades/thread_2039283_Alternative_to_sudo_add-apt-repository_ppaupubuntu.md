### Post by drmrgd on 2012-08-08
I'm not sure I completely understood the question.  However, if you're just asking how to add repository sources a different way than with the command line, it's fairly straightforward.  Open the Ubuntu Software Center, and under the "Edit" menu, there is an option for Software Sources.  In the "Other Software" tab, you can enter the repository information, and update from there.  Here is a link to the Ubuntu Community Docs on Repositories to help you along:

[https://help.ubuntu.com/community/Repositories/Ubuntu/](https://help.ubuntu.com/community/Repositories/Ubuntu/)

I'm not sure how this will be different than doing it by the CLI in terms of your proxy, though.
