### Post by linear on 2006-03-09
You'll still need OS 9 around to use BootX, so don't be too quick to format. You can use Apple's Disk Utility to format/partition, but it does destructive partitioning only. You won't be able to boot directly from the Ubuntu CD, but if you can't read it at all, it may be bad...

### Post by linear on 2006-03-10
Disk Utility comes with the OS, and is usually in the Applications/Utilities folder. Should also be on the install CD - in fact you would need to boot from the install CD to re-partition the disk. (An earlier version was called Drive Setup. Don't recall which name they had in OS9).
