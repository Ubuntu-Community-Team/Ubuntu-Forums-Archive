### Post by Wim Sturkenboom on 2007-04-15
If it has the same specs as [this one](http://www.dealtime.co.uk/xPF-Compaq-Presario-1200-XL103-155726-073), I would not consider Ubuntu. It will be painfully slow (if it wants to run at all).

The normal CD (live and install) requires 256MB memory ( see at end of [http://www.ubuntu.com/products/whatisubuntu/desktopedition](http://www.ubuntu.com/products/whatisubuntu/desktopedition) ). That's probably the cause of your problem. If so, get the alternate CD and try again.
