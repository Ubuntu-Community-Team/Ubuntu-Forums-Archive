### Post by aysiu on 2006-09-08
The DVD contains all three major desktop environments (KDE, Gnome, XFCE) and a lot of other packages (what's known as the entire Restricted and Main repositories).

The CD has one desktop environment (either KDE, Gnome, or XFCE) and a few packages basic to that environment.
