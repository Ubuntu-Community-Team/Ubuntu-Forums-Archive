### Post by oldfred on 2014-01-29
USB2 ports make system a bit slower, but Ubuntu uses RAM as cache, so if you do not load too many apps at once and reuse apps it can still be very fast. I use 64 bit Ubuntu with 1.5GB of RAM, but not Unity as it needs better video card/chip. 
How much RAM and what video card/chip. If not current then Lubuntu may be better choice. Or Xubuntu.

Download the Desktop live system. Install to flash drive and see how well it works. Flash will be even a bit slower than a hard drive but you can see how well system functions and test features you are concerned about.
