### Post by namdung on 2008-12-24
I recommend Xubuntu for low hardware specifications. An Alternate Installation CD is recommended. 

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

If booting from a CD ROM is not possible, then you may try making a bootable USB.

UNETBOOTIN available for both Windows & Linux.
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
