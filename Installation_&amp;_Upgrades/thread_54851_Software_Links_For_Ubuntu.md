### Post by Juergen on 2005-08-07
Do you have Ubuntu running from hd now? (in another thread I read something of you using the live-cd...)

If so, you could just follow the instructions here [https://wiki.ubuntu.com/AddingRepositoriesHowto](https://wiki.ubuntu.com/AddingRepositoriesHowto)
After that you have access to 16000+ packages made for Ubuntu. Most of the stuff you want is probably in there.

For those apps that aren't packaged for Ubuntu, you can always get the sources and compile yourself.
But that's a step more advanced, so you should probably stick with the Ubuntu packages for a while and first become comfortable with Linux.
