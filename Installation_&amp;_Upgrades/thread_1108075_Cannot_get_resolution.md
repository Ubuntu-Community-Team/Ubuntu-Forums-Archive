### Post by Mark Phelps on 2009-03-28
You can fiddle with screen resolutions in the System --> Preferences --> Screen Resolution applet by trying different monitors and settings, but if you're trying to do this with a laptop with an external monitor connected, that could be a problem.

It's been a while since I did this -- I had a 1600x1200 resolution set up, and when I rebooted with two monitors connected, the second being 1024x768, both screens were reset to that resolution and no amount of fiddling would change the larger screen -- until I rebooted with only it connected.

I've read that Ubuntu is working on improving this, especially for 9.04.  But I haven't tried it in 8.10 and can't tell you if they've fixed that.
