### Post by Catsworth on 2007-01-19
I'm assuming two things:

1.  That you managed to log in ok.....
2.  That you are using the 'sudo' command to perform the admin tasks.....

The password that you need to use is the password that you used to log in to your account.  If your account was the first one set up then this should allow you to use the sudo command and your own account password to perform administrative tasks.

If you're being asked for a password via a dialogue box, then just type your password in that box.
