### Post by psifive on 2009-10-09
I'm new to this forum so forgive me if I'm incorrect in posting this here:

I rebuilt an old machine with some new memory and a HD and booted it from the 9.04 LiveCD. The LiveCD menu came up correctly and so I thought everything would be fine but every option I seleced from the menu just showed a pop-up message box with the boot option I'd selected (e.g. "/caspe", "live-install", "memtest", "rescue" etc.)

I couldn't find an explanation anywhere on-line so (after burning a new LiveCD with no luck) I started swapping all the new components for the old ones. On replacing the memory it worked fine and eventually just putting the new memory sticks back in worked too. 

So, I wanted to share that this odd behaviour can be a symptom of a memory problem. If you see it try re-seating all your sticks and try again!

Now everything is Jaunty! :)
