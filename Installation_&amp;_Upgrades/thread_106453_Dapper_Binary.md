### Post by az on 2005-12-21
Debian Potato came before Woody.  Sarge came after Woody.  Etch is in development.

Ubuntu Warty was the first, followed by Hoary, breezy and next, Dapper

Warty Warthog
Hoary Hedgehog
Breezy Badger (came close to being Bendy Badger, actually)
Dapper Drake.

Dapper will cap off the first set of releases.  The plan is for every set of releases to have a common feel - the colors and the naming scheme.  I can assume that some medium-term goals are set by these sets of four releases, too, with certain things being a real priority to get right by the last and best release of the set.
