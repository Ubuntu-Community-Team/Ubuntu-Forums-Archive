### Post by nightraven696 on 2008-01-14
Do you need an easy way to quickly reinstall Ubuntu the way you like?

Use and modify this script version 1.2.

I had the need to make a script to assist me in making my work easier.

Since I am writing a guide for ubuntu to be published on [http://www.ubuntu.osgids.org]("http://www.ubuntu.osgids.org") in due time, I reinstall Ubuntu once or twice a week to start with a clean slate to make sure that i encounter the same difficulties as my users/readers.

Installation steps.

1. install via Ubuntu cd
2. install Ati or Nvidia driver
3. get all updates
4. reboot (if new kernel)
5. run tunescript as sudo (mainly installing 'good' software)
6. run notsudo as normal user (gnome and look and feel)
7. use aptoncd (Syn Pack Manager) to burn all the packages on a dvd or iso
8. Next time you do not download the new packages but use aptoncd to put the packages back on your drive   NO MORE LONG DOWNLOADS
9. Read carefully and delete what you do not need - it works fine here on an i386 but there are no garantuees.
