I don't see anything wrong with your boot script output. I would suggest that you re-install grub by using the following commands from the Ubuntu Live cd/usb desktop. Please copy/paste them into a terminal window one at a time, pressing enter after each line.
```
sudo mount /dev/sda1 /mnt
sudo grub-install --root-directory=/mnt /dev/sda
```
If these commands run without any error please reboot and see if Ubuntu boots ok.
