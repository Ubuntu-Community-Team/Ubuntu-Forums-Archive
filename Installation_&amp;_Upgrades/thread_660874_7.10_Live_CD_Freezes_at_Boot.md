### Post by njparton on 2008-01-08
At the first boot screen, highlight the safe graphics install option, hit F6, remove "quiet" and "splash" from the command line option that pops up, hit F4, select the appropriate resolution, then hit return to start up.

This should at least get you past any graphics driver issues.
