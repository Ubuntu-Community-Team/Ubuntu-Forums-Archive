Hardy is an LTS and therefore, by default, will not offer to upgrade to the 6-month versions (intrepid would be next for you)

That said ... yes it is ODD that you don't have the release upgrade option.

Try this (after being updated in 8.04)

Press ALT +F2 and type

```
update-manager -d
```

See if update manager gives you the option.

### Post by staf0048 on 2009-11-27
Under the updates tab, there should be a drop down menu.  If you have 8.04, the default setting is LTS (Long Term Support), so you wont see anything in the update manager until 10.04.  You should be able to select the other option (I forget the actual title) from that drop down menu.  

That being said, if you're goal is to upgrade to 9.10 from 8.04, I'd highly suggest downloading the new ISO, burning to dics and upgrading that way.  Otherwise, your computer is going to to make you go through the 8.10 and 9.04 versions before you get to 9.10, which will take a very long time and would be a complete waste (in my opinion).
