Welcome. You will get better support here and people will take into account that you are new to it. :)

Firstly, how did you create the install media and from where did you download the ISO (and what release are you attempting to install)? Just in case you're doing otherwise, you need to use the Ubuntu ISO to burn a disk image to USB or disk. You can't drag and drop the ISO or create a data disk. It won't work. You need a bootable disk image.

Secondly, are you checking the md5sum of the ISO once it's downloaded, if you think you are creating the disk/USB correctly? You can also 'check disk for defects' from the options screen on the live installer, but doesn't sound like you're getting that far ...
