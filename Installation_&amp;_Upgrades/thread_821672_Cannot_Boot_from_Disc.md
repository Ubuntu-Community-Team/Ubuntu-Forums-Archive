### Post by danielph on 2008-06-07
Can you try it on another computer? 

It is often a problem with the CD that causes this. There is an MD5 checksum that you can check against the downloaded ISO file. If you have a good image file then try reburning the disc at a slower speed.

md5sum is the utility for checking from a command line: md5sum -c [FILE].iso
