### Post by darkod on 2012-05-02
Are you sure the cd and the usb stick are prepared correctly?

You need to burn the ISO image to the cd, not as a single file. You need to burn a cd with it, depending on the software you use the option name might differ.

After that it should boot correctly from cd. If the usb is prepared correctly, it should boot from it too.

When the Try Ubuntu or Install menu shows up, select Try and see if that loads the desktop without any problems.
