### Post by grahammechanical on 2014-06-17
I have just run a Ubuntu Gnome live session image from the 1st November 2013. It is from the Ubuntu Gnome 14.04 development period and it is the only Ubuntu Gnome Image I have. From the start of the loading process until the live session desktop is loaded the experience is completely Ubuntu Gnome and not Ubuntu. The live session desktop is the Gnome 3 shell user interface just as it would be if Ubuntu Gnome was installed to hard disk. I know this to be true because I did install this image to hard disk.

I cannot confirm your complaint. Perhaps you were expecting Ubuntu Gnome to have a different user interface than Gnome 3 Shell. That was never the intention of the Ubuntu Gnome developers. This is the official Gnome 3 shell look and it is what we get with Ubuntu Gnome.

[http://www.gnome.org/gnome-3/](http://www.gnome.org/gnome-3/)
