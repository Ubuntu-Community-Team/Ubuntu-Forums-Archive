### Post by mcduck on 2009-02-20
Do you have internet connection on the computer? In that case you should forget about downloading program packages from the net and installing them by hand. Just use Ubuntu's package management tools and everything will be done automatically.

Actually you should be able to install AWN simply by clicking this link:
[apt://avant-window-navigator]("apt://avant-window-navigator")
or run this in a terminal:
```
sudo apt-get install avant-window-navigator
```
..or find the package with Synaptic Package manager and use that to install it.

For tar.gz packages, they usually contain source code for the program, and should include instructions how to compile the code into a functional program and install it.
