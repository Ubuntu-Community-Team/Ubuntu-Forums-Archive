### Post by Liakath on 2008-07-19
Dear redneckclutch,

> **redneckclutch said:**
> I am currently running ubuntu,and want to switch to ubuntu studio. How would i do this without re-installing everything?

Ubuntustudio is in the repos. You need not reinstall; you can add ubuntustudio from the repos.

You can search for "ubuntustudio" in synaptic and select the package "ubuntustudio-desktop" and any other you require and install. If you install it will remove "ubuntu-desktop" and "ubuntu-sounds" and replace with "ubuntustudio-desktop" and "ubuntustudio-sounds".

Any problems faced you may post back?

Liakath
