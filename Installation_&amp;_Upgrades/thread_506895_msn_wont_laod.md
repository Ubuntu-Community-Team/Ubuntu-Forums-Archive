### Post by liquidfunk on 2007-07-22
First of all. You are not using Windows, you are using Ubuntu, which is a Distribution of Linux. 

 Linux is not Windows. Meaning that Windows programs, without much configuration, cannot work under Linux.

 MSN Messenger/ Windows Live Messenger, hence the name are designed to run on Windows. The only version of MSN messenger that works, is 4.5. 

 Try using another Instant Messenger client such as:

Gaim/Pidgin
Kopete
Amsn
Trillian

 You shouldn't assume that just because you can download it, and you have Wine that it will work. 

 If there is a program that you must have and there is no alternative, have a look on the Wine App database, which tells of programs that work and don't work under Linux using Wine.

 [http://appdb.winehq.org/](http://appdb.winehq.org/)

 That is all.
