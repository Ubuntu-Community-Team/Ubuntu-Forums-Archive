You can at least make a CD and upgrade:

[http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)

Check the section on that page called "Upgrading Using the Alternate CD/DVD"

I'd recommend making a CD and using that, as opposed to putting it on a USB drive. It seems like booting from a CD will be much simpler.

If you must boot the Alternate CD from a USB drive, please check: [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
