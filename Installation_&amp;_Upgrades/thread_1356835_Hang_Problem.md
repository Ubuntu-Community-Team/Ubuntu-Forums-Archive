### Post by quixote on 2009-12-17
It could be a bad download or a bad burn on the CD.  Will it boot normally on another computer?  If not, the solution for both of those is simple: re-download, and burn another CD.

It's important to figure out which version you're using.  If it's recent, say Karmic, 9.10, maybe it just won't run on the old hardware.  Try [Dapper 6.06]("http://releases.ubuntu.com/dapper/"), maybe, or [Gutsy Gibbon]("http://old-releases.ubuntu.com/releases/gutsy/").  I'm not sure Gutsy is still supported.  It was a very good, stable release with USB 2 support, and good wireless manager.  Dapper was a Long Term Support release, so I think it'll be supported through 2011.
