### Post by davidmohammed on 2010-05-18
Welcome aboard.  The easiest way to judge this is to run ubuntu directly from the CD - called the live CD.  

This way, it won't affect your existing windows system, but you get to use the full ubuntu desktop.  

Have a play, run some programs, audio, browse wirelessly etc.  

If you don't have any issues then go for it - install onto your hard-disk.

### Post by walkerchuckwalker on 2010-05-18
There should be no problems, I'm running it on Intel 1.66GHz with 2GB if RAM and 20GB free space. You can contact me if you want some dual boot help. I just got mine fixed to work pretty well. 

Use this if you have a flash drive:

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
