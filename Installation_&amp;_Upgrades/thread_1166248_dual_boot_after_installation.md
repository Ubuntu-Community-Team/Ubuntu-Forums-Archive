### Post by cablejimmy on 2009-05-21
If I've already installed ubuntu on my system, already installed programs, made files, customized my system, etc.

If after all this, I decide I want to dual boot Windows XP alongside ubuntu, how would I do that without uninstalling my operating system, deleting all my files, etc.?

---

### Post by maybeway36 on 2009-05-21
First, make a free space with the GParted partitioner running from th Live CD. Then install windows into that free space. Once that's done, you can boot the live CD again to reinstall GRUB ([https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)).
