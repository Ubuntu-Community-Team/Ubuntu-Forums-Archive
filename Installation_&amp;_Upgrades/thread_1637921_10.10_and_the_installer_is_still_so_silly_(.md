### Post by Quackers on 2010-12-05
To get Windows booting again you will need to set the first hard drive as first boot device then run the Startup Repair from the Windows repair disc. This may need to be run 3 times in total. 
At the moment you have grub2 installed there. This will replace grub2 with the Windows bootloader. You also have one Windows XP boot file in sda1. I have no idea how that got there with a Windows Vista (or 7) system.
Do you have Vista or Windows 7?
If this repairs Windows bootloader and Windows boots ok again we can then move on to the second hard drive, which has grub legacy installed at the moment. This will need to be purged and grub2 installed in its place. This should get Ubuntu booting again.
