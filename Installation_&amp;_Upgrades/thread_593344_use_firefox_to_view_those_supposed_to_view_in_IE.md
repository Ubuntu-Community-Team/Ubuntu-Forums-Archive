### Post by johnnyg on 2007-10-27
The IE tabs plugin is not an option on linux -- IE is a windows app + embedding it in a firefox tab relies on you being on the windows platform.

Your best bet if you need to run IE on linux is to run it under wine.

The ies4linux project makes it easy to get IE working, see here:

[http://www.tatanka.com.br/ies4linux/page/Main_Page](http://www.tatanka.com.br/ies4linux/page/Main_Page)

Note that you mileage will vary with this.
