### Post by mörgæs on 2011-06-25
A kernel panic is a serious failure. It compares to the blue screen in Windows.

It is not certain that we can save the installation, but you could try booting the computer and run

```
sudo apt-get clean
sudo apt-get update
sudo apt-get upgrade
```

Please post the error messages, if any.
