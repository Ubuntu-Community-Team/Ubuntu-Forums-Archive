### Post by Neutrosider on 2011-09-06
you can install distros and bootloader to usb sticks the same way you would install it on normal hard drives. if you speak german, this may help you:
[http://www.pspking.de/forum/showthread.php?tid=49250](http://www.pspking.de/forum/showthread.php?tid=49250)

it's a tutorial i wrote some time ago. It explains how you can install a real ubuntu (not only the live system) and the ubuntu live system (to install it on another PC) to the same usb stick.

you just use it as Harddrive. In your case i would just make 5 GB or more unpartitioned space and then start the mint installer/live CD. then you just chosse the usb stick as installation destination and it will be installed there. make sure to tell the installer to install grub on the usb stick.

when you've done everything right, and you tell the pc to boot from the usb stick, you will see your bootloader.
