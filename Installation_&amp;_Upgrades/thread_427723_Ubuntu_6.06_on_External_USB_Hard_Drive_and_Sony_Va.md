I am not certain if it will help, but smartboot manager can sometimes allow booting from a device that the computer will not normally boot from.  You will not have a floppy disk drive, I assume, so you will needto get the CD version it from [http://sourceforge.net/projects/btmgr/](http://sourceforge.net/projects/btmgr/)
and burn it to a CD.  I'm not sure exactly how to do this (I've used a floppy before) but the site has a lot of info which may help you.
