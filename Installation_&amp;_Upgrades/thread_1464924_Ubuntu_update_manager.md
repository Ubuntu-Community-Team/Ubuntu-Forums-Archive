### Post by revidnus on 2010-04-28
You may not want to enable the pre-release updates as well as the unsupported updates if you don't need them. Those packages may not be fully tested yet and can cause problems with some systems. I personally haven't had any  problems with them but some have. My advice is stick with stable releases if you want to run stable programs.
