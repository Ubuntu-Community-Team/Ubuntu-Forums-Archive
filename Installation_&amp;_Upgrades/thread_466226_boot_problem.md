### Post by Yoooder on 2007-06-06
When you installed Ubuntu, did you create a new partition for it?  Did you resize the Fedora partition, or did you overwrite the Fedora partition?

From the short description you gave, it sounds as though Ubuntu is occupying the partition in which Fedora originally used.  If this is the case, then that partition--if not your whole Fedora installation--will have been formatted and/or lost.
