### Post by mick8985 on 2008-04-01
Alright, here is how it is done
[http://ubuntuforums.org/showthread.php?t=740924]("http://ubuntuforums.org/showthread.php?t=740924")

Follow my tutorial up until the you install syslinux. (obviously don't install syslinux)

Now copy the folder /casper from the liveusb, over to /boot on your hard drive

edit you grub menu
```
sudo nano /boot/grub/menu.lst
```

paste in this entry

```
title           liveusb
root            (hd0,0)
kernel          /boot/casper/vmlinuz file=/dev/sdb1/preseed/ubuntu.seed boot=casper quiet splash --
initrd          /boot/casper/initrd.gz
```

where (hd0,0) should be amended to whatever it is for your other boot options, and sdb1 should and amended to the path of your usb drive.

Enjoy!

PS. There is also another boot configuration I would appreciate if you tried, purely out of interest

Follow my entire tutorial for installing extlinux

```
title         liveusb2
root          (hd1,0)
makeactive
chainloader   +1
```

If this works then either your bios is capable of picking up usb disks but the BIOS programmers are too lazy to add usb boot as an option, or the grub microkernel can recognise it.
