### Post by pytheas22 on 2008-08-04
Yes, you can always download individual packages from [http://packages.ubuntu.com](http://packages.ubuntu.com)--but keep in mind that you may need to download lots of dependencies in order to install something.  To make the dependency stuff simpler, in Synaptic, you can select a package to install and click File>Generate Download Script and it will make you a nice script with links to each package that you need to download.

[APTonCD]("http://aptoncd.sourceforge.net/") is another solution.
