### Post by darkmatter on 2005-10-11
A typical 'full' install is 2-3 GB, and would require regular maintainence (clearing packages downloaded by apt-get/Synaptic, etc.) to keep size at a premium. 

The best case scenario when space is an issue is to begin with a server install and only add on what you absolutely need.
