### Post by Impavidus on 2014-12-17
It doesn't matter too much. 64 bit can be a little faster, 32 bits can do a little more with the available memory. Keep in mind though that if you ever want to upgrade your hardware to 4GiB of ram or more, it would be best to install a 64 bit OS beforehand. It's not possible to upgrade your OS from 32 to 64 bit, other than doing a clean install.

> [COLOR=#000000]The amount of memory is a factor for the decision when it is > 4GB.[/COLOR]

I am running 64 bit Ubuntu because I have a 64 bit CPU even though I only have 1GB of RAM. If we are going to set rules, then

If CPU is 32 bit then install 32 bit OS. If CPU is 64 bit then install either 32 bit OS or 64 bit OS. If CPU is 64 bit and memory is more than 4GB then install 64 bit OS.
