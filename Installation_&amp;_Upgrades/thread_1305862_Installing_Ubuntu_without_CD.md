### Post by tommcd on 2009-10-30
If you have access to another computer with a CDROM drive running Windows or Ubuntu, you can use the "usb startup disc creator" from the live CD to make a usb install drive. This would be the easiest way. If you only have the netbook then you can use UNetbootin to create a usb install drive. See this tutorial:
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
Write back if you need more help.

And welcome to the Ubuntu forums!
