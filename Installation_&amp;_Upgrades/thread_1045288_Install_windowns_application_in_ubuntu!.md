### Post by Mark Phelps on 2009-01-20
As some folks say in their taglines -- Linux is NOT Windows!!

Put differently, you can't install Windows programs in Linux, at least, not directly.

CodeWeavers provides an emulator (of sorts) known as Wine allowing you to install SOME windows apps (inside Wine) -- but the results vary considerably by application and version.

Suggest you check the CodeWeavers site app database for the products and versions you want to run in Wine.  Unless those get a Platinum or Gold rating, you're wasting your time with Wine.

A more robust solution is to use VirtualBox (or other virtualization software) to create a virtual machine and install MS Windows into that.  But that requires Windows installation media to work.  Suggest you check the Virtualization forum at the bottom of the main Ubuntu forums page for more details.
