### Post by MilosDusan on 2007-05-01
Sure thing..

Click System -> Administration -> Language Support

You can change your default language there..

---

### Post by MilosDusan on 2007-05-01
I should also point out that you have to download the language packages for it.. Under 'Supported Languages,' select the language(s) you want, and click 'Apply'.. It will then download the new package files and install them.. You will *then* be able to select it from the default language list ;)
