### Post by DuckHook on 2018-10-27
Chrome is a Google app containing proprietary elements. Therefore, Ubuntu does not carry Chrome in its repos. You will not be able to install it. The FOSS equivalent is Chromium, which is available in the repos.

### Post by deadflowr on 2018-10-28

### Post by Adam_GUI on 2018-10-28
Google Chrome has gone 64-bit for its installer.
Even if you wanted to go to Google's Chrome page to add their package, it flat won't run on your system.

Chromium-browser is as close as you can get on a 32-bit install now.
