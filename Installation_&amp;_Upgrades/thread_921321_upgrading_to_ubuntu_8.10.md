### Post by WRDN on 2008-09-16
If you are using Ubuntu 8.04 currently, then when Intrepid Ibex is released, the update will appear in the Update Manager. If you are using a previous version, then you must upgrade through the versions- for example, if you were running Gutsy Gibbon (7.10) then you would have to upgrade to 8.04 then 8.10.

If you wish to upgrade now to the Intrepid Alpha release, then open the Terminal and type: 

```
sudo update-manager -d
```

Note that this is not recommended for production machines.
