### Post by lechien73 on 2010-07-09
The default font in OpenOffice Writer appears to be "Liberation Serif".

You can check - and change - the default font by going to the Tools menu and choosing Options. Then click on OpenOffice.org Writer > Basic Fonts to change the default.

If you want to see what fonts are installed, then I use gnome-specimen, which can be installed by typing:

```
sudo apt-get install gnome-specimen
```

It will then appear in the Applications > Graphics menu.
