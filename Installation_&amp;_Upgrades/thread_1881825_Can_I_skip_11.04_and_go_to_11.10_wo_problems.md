### Post by Mark Phelps on 2011-11-16
Ubuntu 11.04 brought a lot of changes, and 11.10 added to those.

You would do best burning an 11.10 Ubuntu Deskstop CD, booting from it, choosing Try Ubuntu -- and seeing how well it works.  That's really the only way you will know.

Also, if you have an external drive, you should consider installing Clonezilla and imaging off your current Ubuntu install to that drive.  That way, if you really don't like 11.10, you have an easy way to get your previous version back -- as Ubuntu has no roll-back capabilities.
