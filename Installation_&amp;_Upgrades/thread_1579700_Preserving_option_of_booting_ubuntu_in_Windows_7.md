### Post by Mark Phelps on 2010-09-22
Sorry to tell you this, but a wubi-based install (which is apparently what you did) is NOT intended for long-term use. It's intended to be used for trial usage.

If you really wanted an Ubuntu install that would survive multiple different MS Windows installs, you should have created a separate partition and installed it that way.  That would have made it completely independent of MS Windows.
