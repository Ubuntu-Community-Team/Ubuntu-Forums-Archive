### Post by julianb on 2010-03-01
> I've downloaded other ISO files, reburned etc and its the same each time.

This sounds like a really irritating problem, I'm sure glad it didn't happen to me the first time I tried to use Ubuntu! I might have given up on linux completely.

From your description, it's not clear if you tried the Alternate Install CD and the ordinary ("liveCD") version of Ubuntu 9.10. Or did you try a different ISO like Kubuntu/Xubuntu, or did you simple re-download the exact same ISO?

Some computers have passwords built in to the BIOS or the hard drive - if this is the problem, the offending part would have to be replaced if you can't find the password (and the manufacturer's default password / administrator password doesn't help.)

Another thing to consider is using Linux Mint, which is a very similar to Ubuntu but different enough (so that the problem you're encountering, if it's specific to Ubuntu, probably is not a problem with Mint.)
