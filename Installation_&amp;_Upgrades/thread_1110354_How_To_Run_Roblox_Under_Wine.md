### Post by Mark Phelps on 2009-03-29
Basically, you can't.  CodeWeavers, the suppliers of Wine, maintain a compatibility data base of Windows Apps, rated from "garbage" (the worst), to "platinum" (the best).

If you're going to run Wine, you need to familiarize yourself with their database.  The link below is to their page for Roblox:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=9392]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=9392")
