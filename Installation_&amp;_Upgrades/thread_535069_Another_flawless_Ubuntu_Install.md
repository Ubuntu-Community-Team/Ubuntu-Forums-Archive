### Post by Warren Watts on 2007-08-26
I have been using Ubuntu as my primary OS for three months now, and I couldn't be more pleased.
A friend of mine who runs a computer charity [(PC Santa)]("http://www.pc-santa.org/") gave me an old 700 Mhz Duron system today, so I dragged it home to add to the collection (oh, my wife LOVED that!).
Tossed in an old 10GB hard drive, a basic 8MB AGP card, 256 MB of RAM and a 10/100 network card I had lying around, booted a Ubuntu 7.04 Live CD, and boom, everything worked.  No unrecognized hardware, no xorg config promlems, nothing.  Even had internet access right from the Live CD.
I went ahead and installed Ubuntu from the CD, and now I have a new computer!

I now have three working Ubuntu systems, none of which have a processor faster than 1.0GHz or more than 384MB of RAM, and I am as pleased as punch.

Long Live Ubuntu!
