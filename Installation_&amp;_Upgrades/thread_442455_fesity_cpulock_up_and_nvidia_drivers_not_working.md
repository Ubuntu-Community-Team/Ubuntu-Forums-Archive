### Post by hal8000 on 2007-05-13
Goede Avond,

Regarding Nvidia. First you need to know if the Nvidia kernel module has been loaded,
so you can try

lsmod | grep nvidia

Example
anc@orac:~$ lsmod | grep nvidia
nvidia               4713780  32 
i2c_core               22784  1 nvidia
agpgart                35400  2 nvidia,intel_agp


If you see nvidia in the listing then the module has been loaded, and its just necessary to change the driver line in the graphic card section of xorg.conf from "nv" to "nvidia"Section "Device"
Identifier "NVIDIA Corporation NV34 [GeForce FX 5200]"
Driver "nvidia"
Option "RenderAccel" "false"
BusID "PCI:1:0:0"
Option "NvAGP" "0"
Option "IgnoreDisplayDevices" "DFP,TV"
Option "NoRenderExtension" "Off"
Option "AllowGLXWithComposite" "Off"

EndSection

In your case you already have it set to "nvidia" so either the kernel module has not loaded or not been installed, try:

sudo modprobe nvidia

or if that command fails, use

sudo apt-get install nvidia-glx nvidia-kernel-common

Hope that helps.
