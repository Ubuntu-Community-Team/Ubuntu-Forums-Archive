### Post by grahammechanical on 2016-04-29
Perhaps you need to click that box or move to it using tab. All that will happen if you do not confirm acceptance of the EULA is that you will not get mscorefonts downloaded & installed. If you need those Microsoft fonts then it will be possible to install them after the installation is finished by installing Ubuntu Restricted Extras.

Regards.

### Post by DuckHook on 2016-04-29
Have never tried, so don't know. Did you back up critical data before upgrade attempt? If so, then in worst case, aborting mid-stream is only an annoyance. If not, then possibly major issue (see my signature). Do not reboot before trying <Ctrl>+<C>. At least this gives you the chance to end upgrade process with some degree of grace. Before doing anything drastic, try a few more key presses: press <Tab> a few times before <Enter> to see if the problem is not misplaced focus.
