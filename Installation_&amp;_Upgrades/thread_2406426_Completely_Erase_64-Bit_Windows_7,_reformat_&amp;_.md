### Post by TheFu on 2018-11-20
Setup the Lubuntu installer on a bootable flash drive following any instructions you like.  Verify that it boots, then choose "Try Lubuntu" and see that everything works - networking, audio, mouse, video, ... 

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick) explains making a bootable flash drive, if you need more details.  Windows [https://help.ubuntu.com/community/Installation/FromUSBStickQuick](https://help.ubuntu.com/community/Installation/FromUSBStickQuick) or Linux can do it.

Then after you are satisfied that things are working, just install. During the install, tell LUbuntu to use the entire disk. That will wipe anything that was already there away.  

If you want any of the files there, be certain to back them up BEFORE doing the installation.
