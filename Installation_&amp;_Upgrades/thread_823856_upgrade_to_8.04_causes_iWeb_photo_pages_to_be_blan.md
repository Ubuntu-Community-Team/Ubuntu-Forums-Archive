I have upgraded from Ubuntu/Xubuntu 7.10 to Ubuntu/Xubuntu 8.04 to find that iWeb photo pages no longer work in FireFox.  Is this a giant step backward?  Apple's iWeb program is becoming more and more popular and its photo pages are the best part.  Threads on this forum are even asking for an iWeb type of web programming environment in linux.  Did the people in charge dummy down FireFox for a reason?  I reinstalled Xubuntu 7.10 on two systems to recover compatibility.  If you are running 8.04, go to this site and click on a photo to check compatibility:

[http://web.mac.com/mcneilm](http://web.mac.com/mcneilm)

Windows, Mac and earlier versions of Ubuntu work fine.

### Post by mjm8mjm8 on 2008-10-18
A recent upgrade to Xubuntu 8.10 Beta cleared up the iWeb photo page problem.  It looks as if recent changes to FireFox have removed the problem.  I have also noticed that the iWeb video pages can be now made to work easily.  All you have to do is add the video plugins that come up by default.  I added both the plugin for QuickTime and the plugin for WMV files.
