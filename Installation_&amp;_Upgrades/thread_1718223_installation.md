### Post by mastablasta on 2011-03-31
A nice manual also exists for install and also later on use. : [http://ubuntu-manual.org/](http://ubuntu-manual.org/)
 
As suggested try it first by booting form CD/USB. This will load the whole OS to RAM so any changes you make will be lost on reboot. And also it won't change you current setup/windows. The downside is it might be a bit slow to boot (especially from CD) as it needs to read all the data and transfer it to RAM.
