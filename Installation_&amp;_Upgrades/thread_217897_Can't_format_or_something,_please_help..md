### Post by T700 on 2006-07-17
Is there currently another OS on the disk, such as Windows?  If you simply tell the Ubuntu installer to take 20G and there is not 20G of free, *unformated*, space, it will fail like this.  What you need to do is to:

1.  **Backup your data!**
2.  Defragment the disk.
3.  Carefully follow the directions on the Ubuntu install to free up space and set up a dual boot.  Use care.  If you don't do this part right, you will not be able to access your prior OS.
4.  If you don't care about a dual boot, just tell it to format the entire disk.

Paul

ps.  Don't skip step one.
