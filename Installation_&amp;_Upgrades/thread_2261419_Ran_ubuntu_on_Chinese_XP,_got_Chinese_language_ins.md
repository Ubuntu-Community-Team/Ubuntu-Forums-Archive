### Post by grahammechanical on 2015-01-18
At some point the session should give us an option to select the language to use for installing from a list of languages. I installed Ubuntu a few days ago and I did in fact get the offer to select a language.

You can try this. At the first purple screen press Enter. That should reveal a screen where we can select a language. Select the language and press Enter. At the underlying screen select either Try Ubuntu or Install Ubuntu.

Are you sure that you are installing Ubuntu and not Ubuntu Kylin? It is an official version of Ubuntu but for the Chinese people.
