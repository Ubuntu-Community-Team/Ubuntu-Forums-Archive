### Post by darkod on 2010-06-30
Usually to use ubuntu from usb stick you would install onto the stick as destination, not use it regularly as liveUSB.
If you want to install onto the stick, you could still do this I guess: get another temp stick, create it with the studio ISO even though it's not liveISO. Boot with this temp stick and it will kick off the installer.
Have the target stick connected too and just use it as destination to install onto.
