### Post by Levander on 2007-07-01
It's going to be much easier to erase the old partition and install Ubuntu from scratch.

SuSE could have some stuff in there that will mess up the Ubuntu install process.  Things like installing over other distributions, except for the previous Ubuntu distribution (like upgrading to Feisty from Edgy), no one does any real testing of that.

You are on your own if you try to install over the top of SuSE.
