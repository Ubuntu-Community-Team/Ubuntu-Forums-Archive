### Post by paahtis75 on 2008-08-17
Following keyboard problem appeared after 7.10 -> 8.04.1.

Drop vdr-sfxe from full-screen mode to windowed. Start terminal, gedit or any other app where you can type. Now if I go back to vdr-sfxe full-screen by double-clicking window (not title bar) keyboard is still typing to last used program. So I have to select it by alt-tab before keyboard works normal.

VLC-player keyboard shortcuts works after double-click to full-screen

---

### Post by paahtis75 on 2008-08-18
Found out that when desktop visual effects (not sure what in english but straight translation from finnish) are disabled problem is active. When visual effects are activated keyboard works as in 7.10 with vdr-sxfe. So no need to alt-tab after switching to full-screen with double-click.
