### Post by scramasax on 2008-03-13
> **rookie4evr said:**
> if u have IPOD ????
would I TUNE work ...

No, it won't.  Michael Robertson has offered to port iTunes to Linux, but hasn't been taken up on the offer.

[http://www.michaelrobertson.com/archive.php?minute_id=231](http://www.michaelrobertson.com/archive.php?minute_id=231)

Rhythmbox, which is what Ubuntu comes with by default, is fine as a music player, and you could install GtkPod to move stuff on and off the iPod, but the bottom line is _whatever_ way you try do to this you will lose a helluva lot of the functionality of iTunes (not so mention the ease of use the that iTunes/iPod combination has).  If you're dual-booting you'd be better to use the iPod in the Windows partition with iTunes.
