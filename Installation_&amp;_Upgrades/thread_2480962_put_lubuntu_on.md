### Post by TheFu on 2022-11-16
> **oneleded said:**
> it takes practice. anybody know if ryzen 3, on a laptop is slow. i was told that, by a salesperson, an think i' getting snookered. also said web pages load slow.

Define "slow".  BTW, the CPU isn't the only consideration for what makes a computer fast or slow. There are other considerations like RAM, storage, GPU and workload.

In 2015, I bought a Core i3-5015U and found it to be an excellent system, though it was limited by just 4GB of RAM. Besides that, it was light, fast, great for battery life and performed everything I asked, within reason, due to the RAM limitation.  It was faster than a Core i5 from a few years prior.  

Core i3 systems today are as fast as Core i5 systems from a few years ago, unless there are other limitations.

So the only valid answer is "it depends".  But probably not.  I've seen poorly setup Core i7 systems perform slower than Celeron systems.
