### Post by qamelian on 2007-01-14

You don't need to uninstall Ubuntu. You can use the partitioning tool in the Mandrake installer to remove the Ubuntu partitions before you recreate the partitions for Mandrake.

### Post by meng on 2007-01-14
Or, if you don't feel the need to resize/reorganize your partition table, you could direct Mandrake to install onto the Ubuntu partition, thereby overwriting whatever was already on there.
