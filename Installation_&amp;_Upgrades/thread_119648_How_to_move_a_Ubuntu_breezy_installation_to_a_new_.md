I have a ubuntu breezy installation on a hard disk which has been upgraded from the net and additional packages installed. Recently I bought a higher capacity hard disk and want to move the ubuntu installation to the new hard disk along with all the upgradations done. Is there a way to do this without a fresh installation and further upgradations? Will a simple copy (or tar-untar) will do? Please help. Thanks in advance.

saurav

### Post by frodon on 2006-01-20
I use norton ghost to perform this kind of actions but you can also use [mondo]("http://www.mondorescue.org/about/about.html") .
Or backup your disk following this thread : [http://www.ubuntuforums.org/showthread.php?t=81311&highlight=restore](http://www.ubuntuforums.org/showthread.php?t=81311&highlight=restore) then perform a restore on the new disk  and then re-install grub if you need it : [http://www.ubuntuforums.org/showthread.php?t=76652&highlight=restore](http://www.ubuntuforums.org/showthread.php?t=76652&highlight=restore)
