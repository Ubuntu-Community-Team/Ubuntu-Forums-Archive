### Post by Ek0nomik on 2007-07-17
You can download the Live CD or the Alternate CD to install Ubuntu on your new laptop.

Both the Live and Alternate CD work great.  Most people use the Live CD, but if you run into errors upon booting, than I would try the Alternate CD (which is a text-based installer).

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

### Post by merlinus on 2007-07-17
Before trying to install, clean out temporary files and unused programs, defrag several times, and use vista's disk manager to resize the partition and make room for ubuntu.

Otherwise you are bound to run into lots of problems.
