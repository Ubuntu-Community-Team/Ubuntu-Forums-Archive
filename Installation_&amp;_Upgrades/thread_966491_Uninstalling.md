I have a couple of applications I want to uninstall from Ubuntu 8.04.  They are not listed in add/remove.  How do you remove them?

---

### Post by OrangeCrate on 2008-11-01
Try searching for them in Synaptic Package Manager. If you find them, you can mark them for complete removal there.

Edit:

BTW, if you're on Intrepid, use the regular search function if the quick search doesn't find them. Quick search doesn't seem to work that well (at least for me), it doesn't find everything.

### Post by OrangeCrate on 2008-11-02
Add/Remove is a subset of Synaptic. It provides a quick way to add or subtract new features, but it'll never be as complete as Synaptic.

Whether it's really needed, is a continue source of debate, and it periodically comes up in conversations around here. Here's one thread as an example, there are more if you want to search for them:
