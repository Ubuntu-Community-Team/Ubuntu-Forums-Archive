Well, my balding hair!  It's an OS, why are we worrying about nothing?!:lolflag:

Seriously though, the solution is pretty straight forward.  At least for my setup, since I have spare harddisk to play with:
1. clone my installation on a second harddisk
2. make sure the cloned harddisk can boot to an identical setup
3. upgrade and pray
4. if something broke fatally, then swap back to old harddisk; back to a working system
