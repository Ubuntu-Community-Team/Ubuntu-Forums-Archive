### Post by oldfred on 2012-11-14
Welcome to the forums.

Install of Ubuntu should not change Windows, unless when you installed you shrank Windows partition from the installer. Always best to use Windows tools to shrink the Windows partition and reboot a couple of times. Windows wants to update its new size and will run chkdsk and make other repairs. Then you can install in the unallocated space or use Something Else to create your own partitions.

If you just get the blank screen, was that before a grub menu or after. Often related to video issues and the need for proprietary drivers. What video card/chip do you have.

Natty or later Video issues. MAFoElffen
