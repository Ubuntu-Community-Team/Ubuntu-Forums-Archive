Please tell me if there are recommendations for upating Ubuntu, for example with 12.04.2 to 12.04.3?

---

### Post by fantab on 2013-10-23
If you are upgrading from 12.04.2 to 12.04.3 then there are no recommendations. Just let the update-manager take care of it.

or from Terminal:
```
sudo apt-get update && sudo apt-get upgrade
```

### Post by fantab on 2013-10-23
From 12.10 you'll first have to upgrade to 13.04 and then to 13.10.
It will be better and cleaner if you do a fresh install of 13.10 by replacing 12.10.
