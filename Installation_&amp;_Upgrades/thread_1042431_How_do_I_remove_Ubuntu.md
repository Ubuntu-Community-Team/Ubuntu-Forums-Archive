### Post by ibutho on 2009-01-17
Its probably best to get rid of grub first. You can do this from within Windows (use fixmbr in the terminal) or using a live cd like [Super Grub Disk]("http://www.supergrubdisk.org"). After that you can remove the partitions using the Ubuntu live cd or just format them from within Windows.
