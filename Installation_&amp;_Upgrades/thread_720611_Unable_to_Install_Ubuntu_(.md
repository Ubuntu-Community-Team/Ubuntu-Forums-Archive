### Post by Rocket2DMn on 2008-03-10
You can try loading the LiveCD into safe graphics mode, it's an option when you load it up I think.  You may have to hit F6 or something to see it, I'm not entirely sure.

Or, you can use the alternate install CD available at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom for alternate cd. It is an easy to use text based installer without a live session, so graphics are not a problem during installation. 

Once installed you can setup the nvidia proprietary drivers, preferably from System->Administration->Restricted Drivers Manager
