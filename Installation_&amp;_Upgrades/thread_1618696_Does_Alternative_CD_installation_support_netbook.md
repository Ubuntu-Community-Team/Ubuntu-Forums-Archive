### Post by tommcd on 2010-11-11
The alternate install CD supports everything that the desktop CD supports. It is in fact the same operating system. The only difference is that it is a text based installer, very similar to a Debian install CD.
If you are talking about the Ubuntu netbook version, I don't believe there is an alternate CD for that.
