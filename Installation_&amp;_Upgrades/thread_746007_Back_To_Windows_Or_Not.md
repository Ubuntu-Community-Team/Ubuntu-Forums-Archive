### Post by lotus49 on 2008-04-05
Much as I hate to say it, if gaming is important to you, you will not be able to say goodbye to XP.

Some games will run under Wine but the process is fraught with problems and can be desperately unreliable.  It is very unlikely that Wine will run all the games you want.

Running XP in a virtual machine may work for games that require few resources (but it may not) but don't expect to be able to run anything that requires a reasonably fast machine unless your computer is spectacularly (and I really mean spectacularly) fast.

The short answer is you have a choice between running XP native, or not being able to play the latest games.  One day perhaps games publishers will release Linux (and Mac for that matter) versions of their games promptly, but don't hold your breath waiting.

### Post by keithp on 2008-04-06
A suggestion!

I have two hard drives in my computer.

Primary master has only XP installed on it.
Primary slave has Linux installed on it. 

Both drives are completely separate as far as the installed OS is concerned. (No Linux GRUB on the XP drive, for example).

If I switch the computer on and leave it to boot, it boots into XP. However, if, when the BIOS is booting I press F8, a menu comes up for me to chose which hard drive to boot, I just select the Linux drive.

So I have two Operating systems installed, without the problems of dual booting two OSs on one drive.

Although it is F8 with my motherboard, it may be a different function key to press with other boards. 

Cheers

Keith
