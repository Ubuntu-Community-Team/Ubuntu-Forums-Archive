### Post by pierceTN on 2013-02-02
Hello openation1,

The first thing you would do is download the iso ([http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop))

After you download it, you need to make it bootable on your drive, I use unetbootin. ([http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/))  

after you have downloaded the image and installed unetbootin, plug in your usb drive in to your computer, then open unetbootin, select "disk image" then open your iso file of ubuntu, then press OK.

it will load the files on the USB and make it bootable. 

thats it.

if you want to install it on your computer, reboot and go to your boot menu and select usb, then select default and it will boot up in Ubuntu. you can install it after it boots up.



hope that helps,


-Pierce
