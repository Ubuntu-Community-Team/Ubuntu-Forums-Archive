### Post by outleradam on 2010-04-21
I had many difficulties with the Kubuntu installation from the Beta2  release and the Daily Live release.

Here's the big negatives
1. I wanted to install it on SDB1.  It would not let me change the boot loader installation folder.  There were no options for that.
2. It crashed numerous times, I was only able to submit one bug report.


Here's my oppinions.
+++The graphics were really really cool 
+++KDE is very customizable
+++Alot of details were added
+++Did I mention the graphics?!?!?  Holy cow!

---Seems like every other program I tried crashed
---Unfamiliar programs
---Desktop gives unfinished look, like the bar at the top should be worked on or filled up

Overall:
Kubuntu seems unrefined.  Kubuntu leaves me feeling like "who the hell bothered to make this and didn't debug it".   Kubuntu scares me because it is too different and needs refinement.  Between the crashing and the funny looking/acting desktop, it will  be a long time before I try it again.
