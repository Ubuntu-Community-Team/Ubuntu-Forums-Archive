### Post by matt_man22 on 2006-06-06
You are correct, the alternate installation is text-based and uses less memory.

They both install the the Desktop Environment (DE), which is Gnome.  To run your system on 192mb, look into alternative window managers like Xubuntu (Xfce) or icewm, etc.  A search should bring up instructions for installing these.
