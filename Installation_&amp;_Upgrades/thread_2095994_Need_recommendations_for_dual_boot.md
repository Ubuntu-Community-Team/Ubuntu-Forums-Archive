### Post by irv on 2012-12-18
You can install Ubuntu along side Windows. When you start the install one option is to run along side Windows. It will install Ubuntu and leave Windows alone. It will set up the grub boot loader to dual boot Ubuntu or Windows. Which ever one you choose.
Try running the live DVD or USB first to make sure everything works OK.
Here is a link to a step by step install with screen shots:
[https://help.ubuntu.com/community/GraphicalInstall]("https://help.ubuntu.com/community/GraphicalInstall")
