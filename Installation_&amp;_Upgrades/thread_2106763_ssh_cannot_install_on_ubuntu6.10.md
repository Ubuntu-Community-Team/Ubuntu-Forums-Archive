### Post by sudodus on 2013-01-20
Welcome to the Ubuntu Forums :-)

Ubuntu 6.10 is hopelessly past end of life. There are no updates since several years, so there will be many security holes, if you connect to the internet.

Please post the specs of the computer, at least

- brand and model name
- cpu
- ram
- graphics card or chip

and we can suggest what current system to try instead. There are several versions and flavours of Ubuntu, and there are several other linux distros, that work well with old hardware.

Without knowing anything about your computer I would suggest Lubuntu 12.04, which has the light-weight desktop environment LXDE, or Ubuntu Server 10.04 LTS if you don't want a graphical desktop interface.
