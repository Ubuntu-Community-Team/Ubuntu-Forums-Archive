### Post by littlebigman on 2010-07-15
For those having the same issue, try running the following commands:

```

dpkg --purge $(COLUMNS=200 dpkg -l | grep "^rc" | tr -s ' ' | cut -d ' ' -f 2)

dpkg --purge nginx

```

Generally speaking, if you want apt-get to remove everything, including configuration files, run "apt-get --purge remove mypackage".
