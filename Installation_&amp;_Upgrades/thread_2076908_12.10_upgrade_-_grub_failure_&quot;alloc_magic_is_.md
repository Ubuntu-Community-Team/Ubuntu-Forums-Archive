### Post by user0000 on 2012-10-27
dino99, thank You! It is fixed.

For anyone with similar problem and full disk encryption:
0) Get livecd/liveusb
1) Follow steps 2,3,4 from here [http://askubuntu.com/questions/59974/how-to-mount-with-encrypted-lvm-root-during-installation](http://askubuntu.com/questions/59974/how-to-mount-with-encrypted-lvm-root-during-installation)
2) [http://ubuntuforums.org/showthread.php?t=1581099](http://ubuntuforums.org/showthread.php?t=1581099)
You need to do section **Chroot**, only difference is at step A - instead of 
```
sudo mount /dev/sdXY /mnt/temp  # Mount the main Ubuntu partition. Example: sudo mount /dev/sda5 /mnt/temp
```
you need something like
```
sudo mount /dev/mapper/YOURROOTDIRNAME /mnt/temp
# for example I had
sudo mount /dev/mapper/vol1-man /mnt/temp

```

(Maybe before 2) read Section "What can I try first?" and try, didn't help me.)
