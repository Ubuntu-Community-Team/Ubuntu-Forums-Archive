### Post by richardjennings on 2008-05-07
Depending on whether or not you use windows for games & the spec of your pc, it might be worthwhile installing windows in a virtual machine. The windows c: drive is in a image file that can be easily backed up, moved to another pc etc. I have an amd 3000+ and 1gb of ram and can run apps in ubuntu and xp at the same time without issue. This approach does not work well for games.
