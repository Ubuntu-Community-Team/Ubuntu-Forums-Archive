### Post by Pumalite on 2009-01-01
If you are only using Ubuntu; use Gparted and reformat the hard drive. Also burn a new CD after doing md5sum on the iso. Burn at 4x or less. Do not use CD-RW
Clean the lens in your burner
Download and install ImgBurn:
[http://www.imgburn.com/index.php?act=download](http://www.imgburn.com/index.php?act=download)
Go to Mode>ISO>Burn
Select 1x as your speed of burn.
