I have so far posted 2 messages to this forum in an effort to find the cause of my problem, but had no responses to either message.

However, I have been able to ascertain that the cause of my Black Screen Of Death (BSOD) was caused by Ubuntu v6.06 LTS not being able to use the in-built video hardware in my Compaq ProLiant 1600.

There is a Cirrus Logic CL5446 built into the motherboard.  This device can be disabled (Refer Switch 1 on the label under the Cover for the PCI slots), then other PCI Video support installed.

I used an old Trio S3 PCI card, which worked fine (This machine does not support AGP video cards).

So, is this considered a Bug?  If it is, how do I report it?
