### Post by MarkBr12 on 2012-06-08
Application>Ubuntu Software Center>Type "Wine" into the search bar and choose to download and install the latest version

Insert the Photoshop CD into the disc drive. Choose "Browse Disc" when the dialog pops up and open the folder called "Adobe Photoshop CS2".


Locate "Setup.exe" and right-click on it. From the right-click menu, choose "Open With Wine Windows Emulator." This opens the Photoshop install dialog. Follow the system-specific install instructions.

Advance in [http://www.techyv.com/questions/how-use-adobe-photoshop-obuntu](http://www.techyv.com/questions/how-use-adobe-photoshop-obuntu)
