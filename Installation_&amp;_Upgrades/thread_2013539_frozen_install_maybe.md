### Post by msammels on 2012-07-04
Hey XeroFlare,

Welcome to UbuntuForums!. OK, if you are doing a true CD install, and not masking it with a USB install, like I do, then you will need to check for a few things.

First of all, is your CD clean? What speed did you burn it at? Sometimes burning at a lower speed can help.

Did you check updates to be installed while downloading? Depending on the speed of your connection, this could cause it to hang. Perhaps if you removed that option, and just tried installing what was on the CD, it would work better.

But this sounds like either a corrupt ISO, unsupported hardware, or a dodgy CD.
