### Post by souki on 2010-03-12
I had the same problem, the installer try to install grub on the first partition of the first hard drive
in my case, I was installing from USB key identified as /dev/sda on the raid array identified as /dev/sdb

the blinking cursor you see means that grub didn't install, you have to boot with a live cd and install grub : [http://ubuntuforums.org/showthread.php?t=224351](http://ubuntuforums.org/showthread.php?t=224351)
