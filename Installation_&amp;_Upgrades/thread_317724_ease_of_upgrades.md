### Post by A_608 on 2006-12-12
Agree, that is a large jump. Do you have any files that you need to transfer? You will see backup mentioned in many places. I just completed an upgrade to 6.10 from 6.06. I did not have any trouble. I burned the iso image of the alternate version of 6.10 on a disk. There's a command to use(have to find and post) that opens the disk and runs the upgrade file that is on the disk. I followed instructions exactly. I used the disk. Installation took a while, but I have a successful upgrade. 

Considering what version you have, a fresh install seems the better way to go.
