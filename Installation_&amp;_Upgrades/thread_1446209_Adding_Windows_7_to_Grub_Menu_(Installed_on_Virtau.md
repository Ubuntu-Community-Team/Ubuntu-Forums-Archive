### Post by Alan James on 2010-04-03
Because VirtualBox is a program running within Ubuntu you will not be able to boot directly to it. If you install Windows 7 nativity on your machine you will be able to add it to Grub. However, for future reference you should install Windows before you install Ubuntu, because then Grub will automatically detect and add it to the list.
