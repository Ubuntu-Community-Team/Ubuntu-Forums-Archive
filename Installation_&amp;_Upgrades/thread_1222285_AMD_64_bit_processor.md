### Post by presence1960 on 2009-07-24
MD5SUM the iso you downloaded and check against ubuntu hashes : [here]("https://help.ubuntu.com/community/HowToMD5SUM")

Then try what running_rabbitt said. Did you burn the iso as an image to disk? Did you burn it at slowest possible speed (4x or 2x would be good)?

If all else fails check this out: [boot options live cd]("https://help.ubuntu.com/community/BootOptions")

very last resort: download the alternate text based installer. get it [here]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate")

Post back if you still have problems.
