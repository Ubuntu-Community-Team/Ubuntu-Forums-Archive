### Post by techhelper1 on 2007-10-03
I found a way to install ubuntu onto a usb harddrive and it works prefectly on every computer I used.


Install Ubuntu to the USB Drive like a regular plain install.
Then Download Super Grub Disk iso here [http://sgd.howto-linux.de/download/binaries/sgd/cdrom/sgd_current.iso]("http://sgd.howto-linux.de/download/binaries/sgd/cdrom/sgd_current.iso")
Burn it.
Boot it.
Select Language.
Follow On Screen instuctions.
Go down to Linux and press enter.
Boot Automatically.
Find the linux partition.
Boot!
