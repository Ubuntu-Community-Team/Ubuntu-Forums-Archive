### Post by snowpine on 2009-01-29
Feisty repositories have moved here: [http://old-releases.ubuntu.com/releases/](http://old-releases.ubuntu.com/releases/)

Edit your /etc/apt/sources.list file and substite "old-releases" for "us-archive" and "security". Then you should be able to update. Keep in mind there will be no new security updates, so running Feisty is not recommended if high security is a concern. :)
