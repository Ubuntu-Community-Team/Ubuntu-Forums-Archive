### Post by dstew on 2010-03-01
Are you installing from a Live CD? I think there is a "Simple Graphics Mode" for installation that you can get to by pressing F6. The other thing to do is download and burn the [Alternate Install CD]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate"). This Ubuntu installer uses a text-based installation interface that works more reliably than the Live CD.
