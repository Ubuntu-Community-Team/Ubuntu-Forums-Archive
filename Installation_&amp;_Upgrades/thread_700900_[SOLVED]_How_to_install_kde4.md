### Post by Roanoke on 2008-02-18
I am currently runnung ubuntu with kde and gnome, but would like to install kde4. However, I can't find it in synaptic. Can anyone help? I only find developmental versions.

EDIT: Nevermind, I got it. I had to add 'deb [http://ppa.launchpad.net/kubuntu-members-kde4/ubuntu](http://ppa.launchpad.net/kubuntu-members-kde4/ubuntu) gutsy main' to '/etc/apt/sources.list' and refresh in synaptic.
