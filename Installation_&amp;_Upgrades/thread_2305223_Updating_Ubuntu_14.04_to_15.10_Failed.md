### Post by grahammechanical on 2015-12-03
For those who do not know.

With LTS release we can upgrade directly to the next LTS release. In fact Software & Updates will be set to notify us of the next LTS release. With interim releases we can only update to the next release. And Software & Updates is set accordingly. We can change these settings.

Interim releases only have 9 months support. So, if we install an interim release we should get used to the practice of upgrading to the next release within 3 months of its release. If we are on an LTS release and it is 6 months or less to the next LTS release, then it is best to wait.

Regards.
