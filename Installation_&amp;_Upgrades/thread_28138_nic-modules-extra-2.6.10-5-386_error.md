### Post by stoop on 2005-04-19
I have sucessfully installed hoary 5.04 on my main computer.  I was attempting to install this on my old computer.  Its a Micron with a celeron 300mhz and 96 megs of ram.

Ive chosen to do both a server install and a normal install and each time it gets to ~34% and hangs on unpackaging this module nic-modules-extra-2.6.10-5-386.  It will flash the screen, and go back to the blue screen + status bar and get to 1%.  It does this looping several times before hanging on a black screen with a small white cursor bar at the bottom left.

I have tryed with and without a nic card installed to see if it was my old ISA nic.  It gives me the same problem either way.

Ive tryed the 5.04 release CD and an ISO I downloaded about 2 weeks before the official release.  Neither have worked.  Any help would be great.

---

### Post by stoop on 2005-04-19
I figured out the problem.  One of the RAM sticks was not seated correctly.  I fixed this bumping the machine from 32 to 96 megs and the installation went fine (but very slow)
