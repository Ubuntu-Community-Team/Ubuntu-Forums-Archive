### Post by An Sanct on 2011-02-23
The speed of your internet connection is critical here...

To speed up your updates you can do this:

Run Update manager, in the left bottom corner, click [settings...], select the tab "Ubuntu Software", from the drop down "Download from:" select "Other..." then click "Select Best server".

This test will get you the best performing updates server from the world wide list.
