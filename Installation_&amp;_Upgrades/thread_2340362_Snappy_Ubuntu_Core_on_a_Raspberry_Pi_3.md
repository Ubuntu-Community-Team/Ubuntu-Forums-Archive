### Post by brian62 on 2016-10-17
If you've tried to install Snappy Ubuntu Core on a Raspberry Pi 3, you've probably run into troubles. That's because the packages (specifically the firmware) in regular Ubuntu Core are not compatible with the new Raspberry Pi 3. Canonical released a temporary workaround for this in the form of a new image file and they are working on a permanent fix. The modified packages can be found in this PPA - [https://launchpad.net/~ubuntu-raspi2/+archive/ubuntu/ppa-rpi3](https://launchpad.net/~ubuntu-raspi2/+archive/ubuntu/ppa-rpi3) .

Installing Snappy Ubuntu Core on the Pi 3 follows the same process as on the Pi 2. Remember that Ubuntu Core is a barebones OS meant for embedded solutions and the Internet Of Things. If you expect to run a desktop environment or run desktop applications, you're going to have a bad time.

On the flip side, Ubuntu Core comes pre-packaged with the awesome Snappy system which works great. I did a demo of the Snappy system which you can watch here - [https://www.youtube.com/watch?v=VbDAvTEBxew](https://www.youtube.com/watch?v=VbDAvTEBxew)
