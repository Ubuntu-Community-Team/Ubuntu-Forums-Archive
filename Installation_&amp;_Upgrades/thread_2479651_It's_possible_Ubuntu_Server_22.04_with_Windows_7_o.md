### Post by QIII on 2022-10-02
Windows 7 is no longer supported.  Because it is no longer updated with security patches, it is dangerous for you and for all of the rest of us if you connect to the web with it.

Windows 7 cannot use UEFI.  UEFI compatibility was introduced with Windows 8.  So no, there is nothing wrong with UEFI in as much as the machine is concerned.

Please choose a supported version of Windows which is compatible with UEFI and install both Windows and Linux using it.
