### Post by snowpine on 2009-12-23
You've got UNR backwards; it is the same kernel, different user interface (the netbook "launcher" instead of the standard Gnome desktop).

If you want the launcher, UNR 9.10 is a good bet. If you prefer a regular desktop, go with "regular" Ubuntu 9.10.

You can use the Search feature to research the 1000HE specifically. In my experience, 9.10 has excellent support for most of the older eee models.

Good luck!
