### Post by grahammechanical on 2021-12-29
If you install Ubuntu 21.10 then be prepared for it to reach the end of its support life at the end of July 2022. At that time you will be able to do an online upgrade to Ubuntu 22.04 Long Term support (LTS) that will be supported for a minimum of 5 years and an option to have an extra 5 years Extended Security Maintenance (ESM).

Ubuntu should run on that hardware but I have my doubts about the CPU. What is its speed? If Ubuntu 21.10 does run satisfactory then you might try Lubuntu 21.10. That will also need to be upgraded to Lubuntu 22.04 in July 2022. There are other alternatives such as Xubuntu 21.10.

### Post by ajgreeny on 2021-12-30
No, it is the user interface or GUI that differs between them.

[LIST=1]
[*]Ubuntu uses a gnome DE which requires probably the highest amount of RAM and I believe 2G will mean it runs slowly.
[*]Kubuntu uses a KDE GUI requiring lower RAM but still is probably going to run slowly with 2G, though I have had no experience of KDE for about 15 years.
[*]Xubuntu (my choice, and very configurable) will run reasonably on 2G but better with more if you can get it for that machine.
[*]Lubuntu used to use LXDE which is no longer developed, so now uses LXQt, and of the main *buntu versions will probably be fastest on that machine.
[*]
[/LIST]

Apart from the GUI all of the family of OSs from Ubuntu use the same underpinning systems.
Try as many as you can as live systems just to see which DE you prefer, remembering that the live system will run slower than an installed system.

There is a good **Sticky thread** at [https://ubuntuforums.org/showthread.php?t=2415676](https://ubuntuforums.org/showthread.php?t=2415676) which tells you much more about this subject.
