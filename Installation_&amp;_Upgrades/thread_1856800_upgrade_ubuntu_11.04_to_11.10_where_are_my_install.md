### Post by garvinrick4 on 2011-10-09
Windows key shows dash and can navigate to apps there.
To see all installed applications if not in dash you can take a peek in
```
nautilus /user/share/applications
```Open them and in make as favorite in launcher so as to be available there.
Hit windows key and type in first few letters of application and will pop up
in dash.
Also in main menu application (alacarte) can check which ones you want to show in right side of dash in your catagories. 
System settings has a list of some packages.
Sometimes I use a dock (docky) to add applications there when open (right click and pin to dock).
