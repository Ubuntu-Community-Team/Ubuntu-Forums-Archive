### Post by Mark Phelps on 2009-07-04
To use MS products in Ubuntu, you have to install Wine or Crossover.  Both are sponsored by CodeWeavers, who maintain a compatibility database of Windows apps and versions.  The link below is to their page for MS Access, and as you can see, if you want to use the latest version (2007), you're out of luck:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=12]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=12")
