### Post by Truefire on 2008-07-15
Windows programs are _not_ meant for Ubuntu. 

Aim has a linux version already installed:
Applications>Internet>Pidgin Instant Messenger.
I use AIM on that all the time.

If you want, go to Applications>Add/Remove to see all programs available.
Installing them is as easy as checking them off, and hitting Apply.

There, you'll find Ubuntu alternatives for all your Windows programs,
but note that zunes, and the latest generation iPods are NOT supported,
due to Microsoft and Apple playing keep-away with the necessary tools.

If you need more help, hop on the forums at [www.easygeek.org](www.easygeek.org)
