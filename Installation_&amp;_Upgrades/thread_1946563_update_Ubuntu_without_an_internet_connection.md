### Post by Mark Phelps on 2012-03-26
If you have no Internet connection at all, you can't do the updates.

The link showed how to do the update from one machine -- which DID have Internet connection -- and then migrate the updated packages to a second machine -- which did NOT have Internet connection.

But, to start the process, you must have an Internet connection to find and download the updated packages.
