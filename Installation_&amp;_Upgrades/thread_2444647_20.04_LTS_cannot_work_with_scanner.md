I have tried that, but that did not work. My HP Photosmart was connected on WiFi and it was found and configured for both printing and scanning in 18.04 with no problem. 

In any case, I solved it. 

- Ran hp-check and discovered that there were a few dependencies that were not met. Some packages were installed with :amd but the hp script did not account for that. 
- I installed as many of the dependencies that hp-check complains were missing, and was able to address most except python3-pyqt4. 
- Then I ran hp-scan, hp-info, hp-setup and my WiFi connected printer was found. 
- hp-levels and scanimage -L both now find the printer / scanner combo

The Simple Scan software now works.
