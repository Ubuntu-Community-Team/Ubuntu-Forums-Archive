### Post by mikewhatever on 2013-12-07
Run 'gksu software-properties-gtk', select the Updates tab, and set the 'Notify me of a new version of Ubuntu' to Never.
PS: Don't complain when support ends for 13.04 and you start getting error. You've asked for it.
