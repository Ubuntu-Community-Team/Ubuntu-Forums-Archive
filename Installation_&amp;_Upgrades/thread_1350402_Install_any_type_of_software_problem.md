There are various things that are considered "program centers" within Ubuntu. Synaptic and "apt-get" (the command line version of synaptic) are also considered "program centers". Make sure you close these as well. I am guessing a re-start will kill all "software centers" that you are running. Personally, I like Synaptic more than the new "Software Center" in 9.10.
