### Post by blaarev on 2007-03-15
I've also given up on installing the live-boot CD's (6.06) for this reason.  We have been using earlier versions of Ubuntu in my library to recycle old hardware (Dell GX1 with 192 MB), but the new install procedure is agonizingly slow, if it works at all.  I have used  the server version which uses a modified version of the Debian install, and it installs and runs just fine.  But then I have to use apt-get to install X and window managers.
