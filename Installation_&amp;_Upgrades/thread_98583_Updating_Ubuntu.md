Im not all that familiar with how Symantec package manager works and want to know if i can update to Breezy from Warty without reinstalling the whole system from a CD image

---

### Post by trevorv on 2005-12-03
Go to Settings>Repositories , and then wherever it says "warty" replace it with "breezy". Than just click OK, Reload, Mark All Upgrades and Apply.
