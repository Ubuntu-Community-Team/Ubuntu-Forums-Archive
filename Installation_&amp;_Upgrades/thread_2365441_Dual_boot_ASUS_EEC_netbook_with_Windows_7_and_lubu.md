### Post by yancek on 2017-07-06
Boot the Lubuntu installation DVD/flash drive and run this command from a terminal:  sudo gparted
That should show information on your drives/partitions so post it here.  If you have a standard windows 7 install it will be MBR which allows only 4 primary partitions.  Posting that output should give enough info for someone to make a suggestion.  Do you want to keep windows 7.
