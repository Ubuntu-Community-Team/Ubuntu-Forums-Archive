### Post by tamoneya on 2008-07-10
provided that during your first install you made a separate home partition you should be all set.  

As for programs that are 64 bit most linux programs provide 64 bit versions as it is a fairly simple matter of recompiling it for 64 bit most of the time.  The repositories typically have both 32 bit and 64 bit versions and detect what you are running on your computer and install the correct ones for you.
