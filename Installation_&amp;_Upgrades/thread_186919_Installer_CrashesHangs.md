### Post by sinfree on 2006-06-03
I finally got it working.  First, I had the memory in the DIMM 2 slot instead of the DIMM 1 slot.  When I switched that the instability pretty much went away.  Second, after that it was hanging on the apt part of the install, so I simply installed without the network cord, then worked on getting the network up afterward.  Anyhow, it is working great!
