### Post by Mark Phelps on 2010-10-09
You mentioned expanding your Windows partition.  Just make sure that you do that expansion using the Win7 Disk Management utility.  Don't use any Ubuntu utilities to do that.

If you want to create a separate NTFS data partition using Ubuntu utilities, that should be OK.
