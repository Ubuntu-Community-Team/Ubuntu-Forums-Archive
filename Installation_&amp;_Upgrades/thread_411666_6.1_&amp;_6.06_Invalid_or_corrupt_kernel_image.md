### Post by kidders on 2007-04-19
Hi there,

If you have Linux experience, you might like to try booting with a different Linux boot CD and running  the installer from there. You *should* be able to get it to cooperate. Alternatively, you could order a Ubuntu CD, as you suggested.

Sometimes, asking your software to burn a little slower can help, particularly if you're using cheap discs. Try burning one more copy (but at maybe 2x or 4x) with full data verification. You *may* have better luck.
