### Post by Mark Phelps on 2014-04-14
Boot using the Ubuntu install media.  When you get to the desktop, open a terminal and enter "sudo fdisk -lu" (Lowercase L, not a one).  That will list out the partitions and filesystems on the drive.  Post that information back here so we can see it.
