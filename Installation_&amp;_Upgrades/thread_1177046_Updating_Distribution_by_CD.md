### Post by raymondh on 2009-06-03
Hello Jimmy,

Upgrading by CD usually means doing a fresh, clean install. If you do, you might as well do a fresh install of 9.04 Jaunty.  Some questions to ponder:

1.  Do you have a separate home partition.  If so, a manual install on top of the old 8.04 root whilst maintaining/not formatting your home partition needs to be done.
2.  Have you tested the 9.04 LiveCD to see how your machine works with it?
3.  Save your files in a safe place ... any install has a risk in losing files.

Hope this helps.  Let us the forum know if you need assistance.  Good luck.
