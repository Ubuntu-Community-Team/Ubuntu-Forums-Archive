### Post by az on 2006-02-20
Ubuntu is an OS with a streamlined set of applications included for general desktop use.

Edubuntu is the Ubuntu OS with added software that would appeal to the classroom.  

If you install Ubuntu and then add the edubuntu-desktop package (you install software packages to install software) you will end up with a system like if you had just installed edubuntu in the first place.

Both can be installed beside Windows.  The installer will offer to shrink your existing windows partition to create free space for Ubuntu or Edubuntu.
