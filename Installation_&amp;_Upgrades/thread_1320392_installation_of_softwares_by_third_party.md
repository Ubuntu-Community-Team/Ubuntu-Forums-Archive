Hi, another way to install it instead of using the real player 11 bin is to download the deb package from [[COLOR=Blue]here[/COLOR]]("http://www.real.com/realcom/R?href=http://forms.real.com/real/player/download.html?f=unix/RealPlayer11GOLD.deb").

### Post by aquavitae on 2009-11-09

The instructions for 9.04 should work with 9.10.

### Post by mkvnmtr on 2009-11-25
I installed Realplayer and flash through the Synaptic Package Manager. As I recall I had to enable the Medibuntu repository to get Realplayer and the main repository to get the latest flash- It will be unusual if the programs you need are not in a repository that can be enabled in your software sources. In that manner updates will come when they are available.
