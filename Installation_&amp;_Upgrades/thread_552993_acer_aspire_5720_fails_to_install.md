### Post by Pumalite on 2007-09-17
No problem. After you have the system installed and you end up with a blank screen: Ctrl+Alt+F1 to get a command line, then:
sudo /etc/init.d/gdm stop
sudo dpkg-reconfigure xserver-xorg, choose most defaults, choose 'vesa as you driver, then: startx or
sudo /etc/init.d/gdm start
If you have problems, post back.
