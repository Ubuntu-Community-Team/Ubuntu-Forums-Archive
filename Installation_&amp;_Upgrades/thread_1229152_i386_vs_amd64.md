### Post by Mike'sHardLinux on 2009-08-01
Hello.

No worries. The 32 bit version is for Intel AND AMD processors. The "AMD64" version is for AMD AND Intel processors that are 64 bit. I do not think it is possible to install the 64 bit version on a 32 bit processor. 

I am running the AMD64 version on my Intel e6400 right now. runs great!
