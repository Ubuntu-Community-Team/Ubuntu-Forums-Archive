### Post by Xian on 2005-06-23
[QUOTE=WeeMan]
I've tried to install Clamav, Firestarter and a Videoplayer... all with the same result.[/QUOTE]
Clamav is available in the Universe [Utilities] repo.
Firestarter is available in the Universe [System Admin] repo.
I'm not familiar with Videoplayer (unless you mean Vidoelan).

Please make sure you have the Universe repos enabled in Synaptic.
Then do a 'reload' in Synaptic and try again.

[Synaptic How-To](https://wiki.ubuntu.com//SynapticHowto)
