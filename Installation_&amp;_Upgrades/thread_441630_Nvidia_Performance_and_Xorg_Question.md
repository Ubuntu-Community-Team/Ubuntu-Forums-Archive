### Post by psyke83 on 2007-05-18
Hi,

Try installing the latest beta nvidia driver using the current "unstable" Envy release, see: [http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

You don't need to allow it to reconfigure xorg.conf, it should already be set to use the nvidia driver. However, you need to do two things:

1. Edit /etc/modprobe.d/blacklist and add to the end of the file:
```
blacklist nvidia_new
```
2. Optionally, run in a terminal or add to your session:
```
nvidia-settings -a InitialPixmapPlacement=2
```

I've found the latest beta driver much more responsive for 2D operations, and compiz/beryl works with no black window bug or pixmap errors.
