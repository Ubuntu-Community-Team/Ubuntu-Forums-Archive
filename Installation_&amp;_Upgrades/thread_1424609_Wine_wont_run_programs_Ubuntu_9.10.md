### Post by Mark Phelps on 2010-03-08
Wine is real hit-and-miss for MS Windows apps.

Just to make sure you didn't install apps that do NOT work with Wine, you should go to the WineHQ Application Compatibility website below and do searches for the names and versions of your apps:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

Ratings of Garbage mean that your apps are NOT going to work, no matter what you do.**
