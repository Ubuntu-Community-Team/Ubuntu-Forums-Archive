### Post by Paul T. on 2009-06-16
Sounds like you need to install the relevant codecs for media player(s) to read the files. Go to add/remove programs, in the 'show' drop down list select 'all available applications', select 'all' in list on left hand pane and search for 'gstreamer', in resulting list select all relevant plugins and install. (I can't take the credit for this method, found it in a book!)
Worked for me :)
