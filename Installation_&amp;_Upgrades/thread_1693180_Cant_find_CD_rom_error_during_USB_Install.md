### Post by persyval on 2011-03-10
I had the same problem with my proliant server.
I eventually got it working by downloading unetbootin.
[http://unetbootin.sourceforge.net/]("http://unetbootin.sourceforge.net/")

And using the ubuntu 10.10_netinstall_x64 distibution.
You will need to have an active internet connection for this to work but because all files will be downloaded the error of looking for the cd rom drive is gone.
