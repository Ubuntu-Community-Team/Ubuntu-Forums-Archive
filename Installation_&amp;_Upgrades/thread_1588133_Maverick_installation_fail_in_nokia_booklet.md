### Post by mikewhatever on 2010-10-05
Nokia booklet seems to be the device powered by the ill supported gma500 graphics, aka Poulsbo. Intel decided to provide zero support for the device, and so it's been community supported for the past two and a half years with varying levels of success.
Check out the [Poulsbo Wiki page]("https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsPoulsbo/") for updates on the current status.
There is also a megathread. If interested, run a forum search for 'gma500'.
