### Post by skern03 on 2008-12-20
Are you booting from the CD? If not, do so, then try this. When you get the install screen with the select languages part, choose your language. Then you'll get a menu that has a list of selections, something like
LiveCD
Install
...
Move the cursor to Install with the arrow keys. Hit the F6 key. This permits you to enter install options. You'll see what appears to be a non-editable line of text. Type in the following:
noapci
then hit enter. See if that works.

BTW, even with that, 8.10 won't install for me at all. It always crashes with an Ubiquity error; however, I don't get a black screen. It almost completely loads, but fails before it installs things like keyboard drivers. That's really fun ;-) 

I switched back to 8.04. You might d/l that distribution and see if it works any better.

Good luck!
