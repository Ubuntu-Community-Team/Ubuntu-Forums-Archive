### Post by P4man on 2010-11-01
Test the USB key for errors; when you boot from it, right after the bios you see a keyboard logo at the bottom. Press any key. YOu will get a menu and one of the options is the test the isntallation disc for errors. Run that.

If the USB key is fine, the most likely explanation is that your harddrive is dead.
