### Post by kebes on 2007-05-17
Yes. During the installation you can control where it installs. You can tell it what disk to install on, and what partition to use. You can also use the installer program to change partitions if you need to.


I'm guessing that you intend to dual-boot, with Windows on one disk, Ubuntu on the other? This is a typical setup and is quite easy to do. Just make sure you pick the correct disk during installation.

Does that answer your question?
