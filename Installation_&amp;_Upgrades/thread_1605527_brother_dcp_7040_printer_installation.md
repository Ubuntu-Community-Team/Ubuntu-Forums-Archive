### Post by mr_luksom on 2010-11-01
I have a brother printer as well, it takes a bit of messing around to set it up. 

You need to download the Linux drivers from the brother site:

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html)

And follow the instructions there.

It is a bit complicated, but if you follow he instructions, and install both the cups and lpr drivers, you will get there.

Post back of you need any help.
