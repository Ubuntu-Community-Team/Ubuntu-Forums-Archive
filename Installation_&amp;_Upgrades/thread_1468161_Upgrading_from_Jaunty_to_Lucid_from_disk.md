### Post by bmuluu on 2010-05-01
Well for starters, you have to upgrade to Karmic Koala before you can upgrade to Lucid Lynx.

If you have an installation CD, you could do a clean install and then get your system back to what you are used to.

To upgrade a running system to a newer version you need the alternate-installation CD of the version you are upgrading to or you do a network install.
I read in [http://ubuntuforums.org/showthread.php?t=1466406&highlight=alternate+i386](http://ubuntuforums.org/showthread.php?t=1466406&highlight=alternate+i386) that you have to have the normal install CD as well if you are going to upgrade using an alternate CD.

You can't upgrade from a normal install disc but you can do a clean install.
Considering that your Internet connection is a slow one, you have to choose between upgrading over that or downloading the ISO's of Karmic and Lucid.

Tough one.
