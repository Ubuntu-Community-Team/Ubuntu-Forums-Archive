### Post by kiwisparkles on 2007-09-06
I have the same graphics card, and ran into exactly the same problem. I thought that it might have had something to do with the fact that I ran the update for the graphics driver at the same time as the kernel update, but now I'm not so sure.

I got around the problem by simply editing my Grub config so that I was booting from the older kernel again... Not exactly the best solution, but at least it's a work around that gets me into X again.

For the record, I have the following Hardware & Software:
Gigabyte P35 DS3P
Intel Core Duo ra ra 2.2GHz
2Gb Memory
Feisty Fawn 64Bit
