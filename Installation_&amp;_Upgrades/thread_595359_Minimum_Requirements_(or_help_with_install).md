### Post by ofb on 2007-10-29
> At least 256 MB of RAM is required to run the desktop install CD. Install requires at least 4 GB of disk space.

Maybe the first thing is run the CD integrity test, then the memory check.

Warning: the memory check takes a looong time. And it does indefinite multiple passes just in case the first pass missed something - unlikely, but they're being thorough. Just do the one pass, which was a little over 3 hours on a P3 450 528mb I checked.

If those are good, and you can't find others with your problem, Dell specific or not, then consider that your CD player might be getting flakey. Though if you've had the problem at the same point of several passes, that's unlikely.

This is what Ubuntu is using for memory check if you'd like to read a little before you run it.
[http://www.memtest86.com/](http://www.memtest86.com/)
