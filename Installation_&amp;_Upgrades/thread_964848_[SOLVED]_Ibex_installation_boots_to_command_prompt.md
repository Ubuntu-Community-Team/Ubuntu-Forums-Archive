### Post by Pumalite on 2008-10-31
You have to prepare your self for an upgrade:
1.-Have your OS fully updated
2.-Remove all third parties from /etc/apt/sources.list (including the Medibuntu folder if you have one.)
3. Backup your XP and all your data, so if you have a problem; it will be a p'iece of cake to start again.
4.-I'd TruImage XP and pack it in an external drive.
And lastly; you might have the same problems if there is an incompatibility.
I personally prefer a clean install.
