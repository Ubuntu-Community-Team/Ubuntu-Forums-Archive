### Post by Zensunni on 2005-05-03
I think Ubuntu is the single most conversion-ensuing linux that has ever hit. And with that, there are stupid windows people like me who haven't a clue about anything to do with linux. I've noticed these forums are flogged with people who are just trying to figure out the preliminary basics of linux. 

Most of the guides and help files are geared toward people who are already familiar with linux. It just assumes you already know basics and starts going into the Ubuntu steps without a whole lot of explanation (which is only reasonable because other users don't want to read every little baby step). However, I think there should be some conversion resources for us people who just need to figure out the very basics: something that addresses windows users and can make some comparisons between the two.

Maybe there are already linux guides out there which just need to be posted. I'm sure there's a lot of support, seeing as gnome and kde were around before ubuntu (i'm guessing they ran on red hat or debian). However, anything I've found is either just command line usage (like the one on linux.org) or it is really complex and irrelevant to complete newbies.

If anyone has any good resourses, articles and guides for post windows users that are just starting in linux for use on their home PC, please throw us a bone! The main help subjects I have currently in mind are: installing, file extentions, networking(especially ms compadable networks), windows equivalent programs, hardware management, and device drivers. There are many more things, i'm sure, but those are what immediately came to mind.

I'll start posting any resourses I may find along the way.
