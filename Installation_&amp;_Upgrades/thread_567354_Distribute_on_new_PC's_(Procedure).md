### Post by maybeway36 on 2007-10-04
If you put in the alternate install CD and use the "install for manufacturers", just log in as "oem" with your password, and run sudo oem-config-prepare when you're done.
(You may also have to distribute source code to comply with the GPL, depending on your situation.)
