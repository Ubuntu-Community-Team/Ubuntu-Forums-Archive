### Post by Pumalite on 2008-03-31
Get Gparted Live CD: [http://gparted.sourceforge.net/download.php](http://gparted.sourceforge.net/download.php)
Burn to disk and boot from it. Delete the Feisty partitions, once 'unallocated', make 'New' partitions of the same size and mount point. Then install Gutsy, go 'Manual' and use already prepared partitions.
