### Post by PmDematagoda on 2007-12-10
I think what you may have is a broken Windows XP which may be due to this:-

System files being on the wrong partitions when splitting was done which can be attributed to fragmentation.

I think the best option for you is to reinstall XP again, after you install XP simply reinstall GRUB to the MBR using [Super GRUB]("http://supergrub.forjamari.linex.org/?section=download").

But before this, let's just make sure it is XP, not GRUB itself. Post the output of:-
```

cat /boot/grub/menu.lst
```
