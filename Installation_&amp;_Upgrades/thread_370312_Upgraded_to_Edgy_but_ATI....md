### Post by Irony on 2007-02-25
I found the following at [https://help.ubuntu.com/community/BinaryDriverHowto/ATI?highlight=%28ati%29#head-d8c6fd05bce340dfc3ad483abf0e18997868540b](https://help.ubuntu.com/community/BinaryDriverHowto/ATI?highlight=%28ati%29#head-d8c6fd05bce340dfc3ad483abf0e18997868540b)

*The ATI binary drivers do not support Composite. Ensure that the following entry is not in your xorg.conf:*

[PHP]Section "Extensions" 
    Option "Composite" "true"
EndSection[/PHP]
[I]
If there is no Composite entry, add one specifying "false" rather than true. This was necessary on my system. [/I]

So I'll give this a go.

---

### Post by Irony on 2007-02-26
It worked! I simply added the entry;

[PHP]Section "Extensions" 
    Option "Composite" "false"
EndSection[/PHP]

To my xorg.conf file and I now have acceleration - additionally blender works in accelerated fashion whereas before when I fired up blender the screen would just go blank then go to log in.
