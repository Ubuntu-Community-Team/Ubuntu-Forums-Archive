### Post by aysiu on 2006-08-30
Alternate is good for special installations--OEM, barebones (server), minimal RAM systems.

Desktop is the normal installation for 512 MB of RAM or more on a desktop computer for home use.

See the differences here...

[http://www.psychocats.net/ubuntu/installing.html](http://www.psychocats.net/ubuntu/installing.html)

### Post by mlind on 2006-08-30
Alternate cd is the older ncurses-based installer which (in my experience) seems to work better. I'm not sure if LVM install support has been included to Ubuntu 6.06.1, but it didn't work with 6.06 Desktop cd.. If you plan to setup LVM, go with Alternate cd.
