### Post by oldfred on 2023-04-24
How is grub failing?
If gpt partitioned drive you must have bios_grub partition on drive you are installing grub into.
If installing in BIOS mode, you must specify a drive like sda, not a partition like sda1.
If UEFI install, you need an ESP - efi system partition on first drive, not necessarily on install drive?

Post this:
sudo parted -l
