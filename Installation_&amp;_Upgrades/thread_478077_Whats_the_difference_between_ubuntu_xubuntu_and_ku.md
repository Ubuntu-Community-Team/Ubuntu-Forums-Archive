### Post by zyzzer on 2007-06-19
Although I haven't had the opportunity to try them all out, Kubuntu and Ubuntu appear to be pretty similar except that Kubuntu runs the KDE environment and Ubuntu runs the GNOME environment.  (I remember reading somewhere that GNOME ran similar to older versions of linux and KDE ran more like Windows - I could be wrong though).

Xubuntu I'm a little more familiar with, since that's all my laptop can handle at the moment (hey be nice, it's like 4  years old).  Xubuntu is aimed more for older systems since the OS is a sort of "Lite" version of Ubuntu.  I couldn't even get my computer to install Ubuntu because it was soooo slow.  When I put in the Xubuntu disk it installed much easier.

I'm not sure how much that helps, but if you would like more information, Wikipedia's always a great source.
