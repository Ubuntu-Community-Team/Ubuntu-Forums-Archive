### Post by Sef on 2007-04-26
If you are downloading from the Desktop CD, try installing from the alternate cd.

Go down to' [PC (Intel x86) alternate install CD]("http://mirrors.csumb.edu/ubuntu/7.04/")' and click on it.

Sometimes the Desktop CD will have problems with the graphic cards, while the alternate cd will not.
