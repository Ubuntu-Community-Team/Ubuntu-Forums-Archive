### Post by Partyboi2 on 2008-02-05
the alternate cd is a text base installer 
and does not give you the option to preview ubuntu before installing like the live cd does. The alternate cd is recommended for older hardware, but in saying that, if people are having problems trying to install ubuntu via the live cd the alternate cd seems to work for some.
You can get it from [here]("http://releases.ubuntu.com/releases/7.10/")
