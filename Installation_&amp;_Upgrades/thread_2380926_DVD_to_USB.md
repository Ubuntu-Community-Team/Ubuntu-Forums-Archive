### Post by SeijiSensei on 2017-12-24
I use [http://cdimage.ubuntu.com/](http://cdimage.ubuntu.com/) as my source for images.  It contains all the various flavors of Ubuntu.

If you're running Windows, then use [Rufus]("https://rufus.akeo.ie/") to create the USB.  Ubuntu comes with its own utility called Startup Disk Creator.  It's in the System section of the application menu.

### Post by sudodus on 2017-12-24
You can use [mkusb](https://help.ubuntu.com/community/mkusb) to clone from an Ubuntu boot DVD to a USB pendrive. But as said in previous answers, 16.04.2 LTS is obsolete. If there are problems with 16.04.3 LTS (with the zesty kernel), you can try 16.04.1 LTS with the xenial kernel. See this link,

[How to select the version and flavour of Ubuntu](https://ubuntuforums.org/showthread.php?t=2230389&p=13540865#post13540865)
