### Post by 2F4U on 2012-08-25
Updates are important security updates and bug fixes:

[https://help.ubuntu.com/community/Repositories/Ubuntu#Ubuntu_Updates](https://help.ubuntu.com/community/Repositories/Ubuntu#Ubuntu_Updates)

while backports contain newer versions of programs:

[https://help.ubuntu.com/community/UbuntuBackports](https://help.ubuntu.com/community/UbuntuBackports)

You might notice the warning in the beginning of the article.
