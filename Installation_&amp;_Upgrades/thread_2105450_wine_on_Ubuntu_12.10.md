### Post by dino99 on 2013-01-16
install the latest wine1.5:

- add the ppa       [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)
- update, then : 
 sudo apt-get install wine1.5

then when you need to run an exe, right click on it and select "open with wine"

thats it
