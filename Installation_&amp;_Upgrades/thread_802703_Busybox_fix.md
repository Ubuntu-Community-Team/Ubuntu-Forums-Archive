### Post by avadyn on 2008-05-21
Like some of the people trying to install Ubuntu 8.04, I've been getting the Busybox error thing. Reading the other threads I came across a solution which worked for me, and may not work for anyone else but I would just like to post it.

Press F6 at the screen where you have the list of choices (Install, check CD for errors, etc) and type "all_generic_ide" without the quotes.

Oh and make sure you press F6 on the install selection

post if this worked for you!
