### Post by lemming465 on 2009-01-25
> ... is the newest version of Ubuntu just too "user-friendly"
In the sense that both the live desktop and text installers install a default set of packages (base system + ubuntu-desktop metapackage), yes it's too friendly - by design.  I gave up on Debian for a while back in the late '90s when it used to take me 4 hours (!) to run the package selection step; life was too short.

If you need more control over the packages than what you get from synaptic or apt-get after the basic install, you can get a much more minimal install using the server CD.
