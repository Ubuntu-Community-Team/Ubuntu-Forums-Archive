### Post by catman57 on 2007-12-31
Hi Ilia,

got the problem solved, at least for me! I just started again from the beginning, that is following the steps as listed in "Quickguide: Installing/upgrading...":

- download the torrent .iso
- check the CD/DVD drive (I used the newest one of mine)
- check the checksum
- burn the image at slowest possible speed and verify
- after inserting the CD first use the verify/check CD option, and let it check itself

After all checks were successful everything turned out fine! Network OK but Log In here doesn't work via Live-CD, graphics OK (though it's ATI...), Scanner is working as well as the printer. So tomorrow I'll go ahead and install that Gutsy Gibbon on his own HD.

Hope this helps you too - Philipp
