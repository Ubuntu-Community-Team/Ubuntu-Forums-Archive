### Post by Partyboi2 on 2008-12-27
Open a terminal (Applications>Accessories>Terminal) and change to the directory where the downloaded file is. If its on your Desktop then you would
```
cd ~/Desktop
```
then extract the gprsec.tar.bz2 file
```
tar xvjf gprsec.tar.bz2 
``` then change into the newly created directory
```
cd GPRS_Easy_Connect_301
``` 
then run the install script
```
sudo ./INSTALL.sh 
```
