### Post by howefield on 2010-02-07
There is no winzip for Linux, you might be able to get it working in wine, but the reports on the wine database aren't very encouraging.

The archive manager which comes with Ubuntu will handle zip files, and you can add the capability to handle other compression types by installing extra package, eg rar, unrar,..
