I found this code and it work, go to "older version" and then open up terminal and write in code and then it will configure itself so latest will work. And about the AMD Unsupported Hardware is just an ugly watermark that i have no idea how to get rid of (it is actually supported)

```

sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update
sudo apt-get upgrade

```
