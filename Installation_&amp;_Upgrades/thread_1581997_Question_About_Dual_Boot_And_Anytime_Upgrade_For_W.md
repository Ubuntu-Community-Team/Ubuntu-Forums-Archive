### Post by Mark Phelps on 2010-09-26
> **TheBritishEditor said:**
> Thanks for replying, but I managed to restore my hard drive to an earlier date while in safe mode, and it's all good now. But the question still stands - If I have data loss, what will happen to my Windows Anytime Upgrade, and will I be able to re-use it?

When you restore to the factory defaults, the machine will be in the state it was when you first unpacked it.  Given that you later upgraded it, then yes, the Anytime Upgrade will be gone.

So, you will have to apply the upgrade a second time.  It MIGHT give you activation problems, given that you've already used up the upgrade license.  IF that happens, it only takes a 5-minute call to MS to explain the situation and get the activation problem fixed.
