### Post by wpshooter on 2009-04-23
First, are you testing the integrity of your Ubuntu CD by running the verification function that is found on the initial Ubuntu boot screen menu ?

I would get [www.killdisk.com](www.killdisk.com) and WIPE the hard drive completely clean.  But let me say that I have never tried this on a solid state drive before, so I have no experience with doing this on a solid state but I know that it does work perfectly on mechanical hard drives.

Then, after WIPING the drive, try your installation to the solid state drive from the ALTERNATE (text based) CD version of Ubuntu.

Good luck.
