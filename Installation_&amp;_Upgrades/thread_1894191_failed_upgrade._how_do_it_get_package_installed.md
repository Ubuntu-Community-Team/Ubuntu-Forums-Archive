### Post by westie457 on 2011-12-12
Just reinstall and at the screen for where to install select 'something else'. In the window that opens select the partition you were using previously - there will be a box that says 'do not use' - click on this and select a file system the same as you used before, next go to the box 'use as' and select '/'. Repeat if you have a separate /home. Do not check the box to format. Continue with the installation.

Very nearly 100% of installs this way work keeping all of your previously installed progs and apps and upgrading them as necessary. Occasionally some will be removed if they are no longer in the repositories.

Hope this helps
