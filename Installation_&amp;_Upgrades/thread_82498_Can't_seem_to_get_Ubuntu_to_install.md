1. Corrupted ISO during download
2. Corrupted CD during burn.
3. Faulty hardware

### Post by Stephan Smith on 2005-10-31
Hi, Phil,
  If you have access to a windows box and 1 floppy disk, you can go to: [www.startdisk.com](www.startdisk.com) , tick Ultimate Boot Disk, and d/l the Windows 98 one. Install it on the floppy, then boot your comp from it. You should be able to do fdisk from that floppy. If that doesn't work, exit out of UBD, but leave the floppy in the drive. This will leave you at an A: prompt, and you can run fdisk from there. I hope this helps! I've had to do it a couple times, myself.
