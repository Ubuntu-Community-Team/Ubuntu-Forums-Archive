### Post by solidus126 on 2010-04-30
SOLVED!

The specific problem I was experiencing is detailed in this bug report:

[https://bugs.launchpad.net/ubuntu/+source/grub/+bug/571364](https://bugs.launchpad.net/ubuntu/+source/grub/+bug/571364)

The measure that fixed it for me was adding "nohz=off" to the end of my grub menu entries, as can be read about here in the Kernel Boot Fix section:

[https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks/ToshibaNB205](https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks/ToshibaNB205)

Hope this helps!

-solidus126
