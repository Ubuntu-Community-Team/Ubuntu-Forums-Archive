### Post by larytet on 2009-04-07
Usually F2 or Del after power up should bring BIOS setup program up. It can be also F12. 

Boot from CD normally presents in the modern BIOS. If there is nothing there and hard disk and floppy disk are the only available options you can consider to copy the ISO image to the laptop or use WUBI
