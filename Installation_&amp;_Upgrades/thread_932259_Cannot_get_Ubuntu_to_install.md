### Post by Elfy on 2008-09-28
Did you check the md5sum for the iso before you burnt it as an iso - [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

The md5sum can be found here [http://releases.ubuntu.com/8.04/MD5SUMS](http://releases.ubuntu.com/8.04/MD5SUMS)

Did you check that the burn was ok with the check cd for defects.

Does your laptop have sufficient memory to run the livecd - > At least 256 MB of RAM is required to run the alternate install CD (384MB of RAM is required to use the live CD based installer).
