### Post by tuxxy on 2009-10-14
You need to just download the .rar unarchiver and it will allow you to extract using the default tool.  Search for unrar using Synaptic or type this in terminal
```

sudo apt-get install unrar
```

EDIT: btw unrar is included in the package ubuntu-restricted-extras which is good to install on a fresh system.  It includes other plugins, codecs etc
