### Post by gordintoronto on 2009-09-29
Sadly, Ubuntu has quite a few "download pages," so I can't tell where you were.

Start at the home page and click "get Ubuntu."

You should have chosen which version you wanted, such as Desktop, Server, or Netbook.  Then which flavour: Ubuntu, Kubuntu, etc.  Then 32-bit or 64-bit.  And where to download from.

For example, the 32-bit desktop Ubuntu is called
  ubuntu-9.04-desktop-i386.iso
