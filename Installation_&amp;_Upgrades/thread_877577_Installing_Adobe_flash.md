### Post by kostkon on 2008-08-01
You can install it from *Synaptic* (*System -> Administration -> Synaptic*). Search for the *flashplugin-nonfree* package and then install it.

Alternatively, you can visit a page that contains *Flash* and you will be asked if you want to install the plugin. You will be given an option of three plugins. Make sure you select the *Adobe* one.
