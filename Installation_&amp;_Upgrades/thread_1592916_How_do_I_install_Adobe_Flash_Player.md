### Post by sikander3786 on 2010-10-11
You might also be interested in ubuntu-restricted-extras which installs a whole bunch of audio/video codecs, java, flash, fonts etc. Saves you some time...

```
sudo apt-get update && sudo apt-get install ubuntu-restricted-extras
```
