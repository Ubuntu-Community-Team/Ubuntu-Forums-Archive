### Post by fakier on 2005-11-28
No, (or yes) it means you don't have to download that big ISO with all the packages in it. It's a small package (10MB) with only a installer and some network drivers, all the needed stuff is downloaded from the internet during install. So, if you do not want something installed it's not downloaded.

This was always possible for Debian (only their package/iso was 100MB because the included the base system in it).
