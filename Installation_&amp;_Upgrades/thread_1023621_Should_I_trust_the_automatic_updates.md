### Post by Idefix82 on 2008-12-28
Definitely install the important security updates! Whether you install the recommended updates is up to you.
Your caution is not entirely unreasonable. Sometimes updates do break something. But it happens rarely and you are never the only one, so with the help of the community things can always be fixed. That's one of the main differences between Windows and Linux: things can almost never break beyond repair. The other difference is that Ubuntu won't install anything to spy on you, so if that's your worry, you can leave that behind :)

Welcome to the Ubuntu community!
