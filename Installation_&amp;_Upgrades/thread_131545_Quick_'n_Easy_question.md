### Post by byen on 2006-02-16
Hey Strev,
Your way would be correct but you can save a step there by doing:
1.Backup data
2.format partition while using windows installer
(if you have the original Windows CD, all you have to do is make a seperate partition  which appears as part of the Windows install process..its easy too)
3.Install Windows
4.Install ubuntu in the extra partition that you created using Windows

now..Im saying this assuming that you have the windows cd.  If not, yes, your way would be correct. But just to be sure I would defrag your drive after installing windows (before resizing partition for Ubuntu). Some may say its unecessary but I suggest it anways!

Goodluck with your installation.Hope it goes smooth!
