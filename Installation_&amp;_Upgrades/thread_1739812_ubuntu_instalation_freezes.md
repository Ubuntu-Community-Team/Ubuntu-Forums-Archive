### Post by Quackers on 2011-04-26
During booting, at the first purple screen (with the stickman icon at the bottom) press any key. Then choose your language and on the next screen you should see an option to check the cd for errors. See if that comes up with a problem. It could be a bad burn. If it finds an error, try burning the image again, but use the slowest burn speed.
