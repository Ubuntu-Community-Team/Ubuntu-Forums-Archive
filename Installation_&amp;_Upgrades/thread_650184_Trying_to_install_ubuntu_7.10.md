### Post by Partyboi2 on 2007-12-26
If you are using the manual partition option when installing you will need to make sure that your ubuntu partition has a  / (root) 
Hightlight the partition you are creating for ubuntu.
Click on "Edit Partition" then change the "Mount point" to /
then ok.
