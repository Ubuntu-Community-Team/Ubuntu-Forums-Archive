Since your Pentium 4 can handle 64 bit instructions, and you have 2 GB RAM, it should work with 64-bit Cubuntu. But I suggest that you try also some iso file (with a similar desktop environment) based on a 32-bit system to see which works better in your computer. A 64-bit system may be faster, but need more RAM for the same tasks, so it will depend on how you use the computer.

Another important thing is the cooperation between drivers and hardware. Some driver might have dropped some old hardware, so that you may have better luck with an older version (for example based on Ubuntu 12.04 LTS rather than 14.04 LTS).

Try the iso files live before installing!
