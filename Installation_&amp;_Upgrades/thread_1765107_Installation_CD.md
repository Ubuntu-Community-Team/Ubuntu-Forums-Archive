### Post by ajgreeny on 2011-05-22
If you see tye .iso file on the CD you just burned a data CD, not a disk image, which is what you need for booting and installation.  Most if not all windows CD burning software has an image burning option, so look again in your application, and try again.  When you burn the CD this time, do it at the slowest speed possible for greater accuracy.
