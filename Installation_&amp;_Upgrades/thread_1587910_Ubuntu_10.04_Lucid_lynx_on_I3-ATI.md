### Post by Mark Phelps on 2010-10-04
Since you have Win7, BEFORE you do anything else, you need to run the Backup feature to create and burn a Win7 Repair CD.  You will need that later to restore boot capability if you run into trouble during the dual-boot configuration.

Also, no one can tell you for sure how well Ubuntu will run on your machine.  Part of that is how well Ubuntu detects your hardware and installs the proper drivers.  That is best seen by booting from the Ubuntu CD and selecting the Try option (NOT the Install option) and then letting your machine startup and run in LiveCD mode.

If you DO decide to install in dual-boot mode, be sure to use the Win7 Disk Management utility to shrink the Win7 OS partition to leave room for Ubuntu.  Do NOT allow the Ubuntu installer to shrink the Win7 OS partition, since that runs the risk of corrupting your Win7 OS partition and rendering it unbootable.
