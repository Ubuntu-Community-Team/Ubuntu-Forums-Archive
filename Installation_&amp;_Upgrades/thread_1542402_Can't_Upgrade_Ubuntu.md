### Post by Revolutionary101 on 2010-07-30
Well I think the 10.04 disk is seeing the Ubuntu 8.04 install and is assuming you don't want a clean install, just the upgrade. Therefore, it is trying to do the upgrade but is unable to. This is because when you upgrade you have to go up by one version (ex. 8.04>8.10>9.04>9.10>10.04).
