### Post by oldfred on 2010-06-30
External is still just another drive. The important thing is you must install grub to the external and set the external to boot first in BIOS. If external is not available then your BIOS will boot the current internal.

To make sure GRUB is installed to the external you must use the advanced button on the last partition or ready to install screen.
