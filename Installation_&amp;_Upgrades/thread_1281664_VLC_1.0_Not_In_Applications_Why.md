### Post by chrisamiller on 2009-10-03
Because you installed from a non-official repo, and whoever created that package didn't take the time to have the package installer automatically create an icon. You'll have to take it up with that package's maintainers.

In the meantime, you can use the menu editor to add a shortcut yourself.
