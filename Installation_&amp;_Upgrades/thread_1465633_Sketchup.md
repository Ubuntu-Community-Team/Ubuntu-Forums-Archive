### Post by Fenris_rising on 2010-05-15
I have just installed sketchup 7 via wine. This is the first time I have been able to do so. It has a few minor 'glitches' within the workspace but no show stoppers. Exporting a structure as 2D results in a blank document so I use screenshot. I do have compiz effects on so this may be causing the minor issues.

The main reason _I think_ I can run it now is I have changed to an Nvidia based graphic card. These are the main spec's whether they help you I don't know -

GPU - ASUS EN8400GS PCIEx2 512mb Nvidia based chip
OS - Karmic
Wine - 1.1.44

EDIT:

I have upgraded to the beta nvidia driver 252 and only had to go into the regedit to change one setting. check out winehq its the setting regarding HW_OK change from 0 to 1

Previously I had an ATI card and although I could install GSKUP the workspace remained black.
