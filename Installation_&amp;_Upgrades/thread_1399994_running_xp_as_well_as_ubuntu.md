### Post by darkod on 2010-02-06
I don't see why not. However, for XP you have to take one important consideration: it doesn't have drivers for SATA by default. MS started including them from vista.
So, more or less, if you have a SATA hdd in your machine, XP will not see a hdd to install onto. To make the problem worse, SATA drivers have to be provided only on floppy, CD doesn't count because you'll have the XP install cd in the drive.
Another option is a process called slipstreaming, where you find the SATA drivers for your computer and put them into XP before hand. With a cd created from such an ISO image, you will be able to install on SATA hdd.
