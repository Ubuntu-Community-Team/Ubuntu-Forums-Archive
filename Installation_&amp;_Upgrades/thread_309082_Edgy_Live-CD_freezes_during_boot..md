### Post by Mr Green on 2006-11-29
I have been trying to install Edgy from scratch for some days now but with no luck.

When I tried to boot the installation cd it just froze up with the Ubuntu logo showing and that progressbar going back and forth. There would be no activity from neither the cd or the harddisk. I even tried the alternate install but no luck there either.

Last night I found out what was causing the problem. My cd-drives was not configured properly relating to which one is slave and which one is master. Both my drives were set up to be master and when I corrected that it installed perfectly.

Just thought I'd share how I solved this problem as I've seen that there are plenty others that keep struggling with the same thing.

I was very happy when I could conclude that the fault was my own, and that Ubuntu could not be blamed.
