### Post by grahammechanical on 2013-02-15
You can try this

1) Run the live CD/DVD
2) At the first purple screen press Enter
3) At the next screen press Enter to select English as the default language or select some other language and press Enter.
4) Press F6
5) From the menu select nomodeset and press Enter
6) Press F10 to boot.

See if that gets you to a live desktop from which you can run the Install  utility.
