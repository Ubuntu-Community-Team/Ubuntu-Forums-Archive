### Post by handy on 2007-04-24
Due to keyboard timing problems that came along with Edgy, & have remained with Feisty, I must use Dapper on my main Gigabyte GA-K8NS Ultra-939 motherboard. Which is obviously dissapointing.  I have posted bugs for both versions.

I have 2 more boxes, both with Asus A7N266-VM motherboards in them.  This board is a real problem with the FreeBSD OS's, but has run very well with Edgy, & Feisty, up until today's update where I lost reliability with my Pioneer DVD burner.

The above 2 experiences have me now running Dapper on 2 out of 3 machines, with the intention of leaving it that way untill Dapper is no longer supported, afterwhich I will revue the situation, leaving Ubuntu for another distro' if the problems have still not been fixed, or using another OS...

I require little from an OS these days; reliable CD & DVD burning is essential (NeroLinux), a simple safe web browsing experience, the ability to access word .doc's, exel spreadsheets & powerpoint files, to be able to view .pdf's, jpegs, png's & use raw file format from a camera & manipulate said images, to be able to rip DVD's simply & easily (DVD Shrink 3.2), to print to my LAN enabled LJ-5 & the ability to play Guild Wars.

Unless it is for exploration of a new OS on a machine set aside for the purpose, I don't feel like repairing / reinstalling my OS after updates / upgrades anymore.  It was fun once, it is boring & tedious now, therefore Dapper will be home until the end of the LTS agreement.  

Fortunately I like Dapper. :)
