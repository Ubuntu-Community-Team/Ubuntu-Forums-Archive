### Post by rbmorse on 2007-06-13
Tobi,

Evolution 2.10.2 is currently running on the Gutsy Gibbon pre-release (7.10) so, if nothing else, if you wait until October and upgrade to the next version of Ubuntu when it is released the problem will resolve itself. If the extra calendars are merely a cosmetic problem I would recommend that you simply wait a bit.

Evo 2.10.2 will not build on Ubuntu 7.04, nor will the .deb package from the Gutsy file repository install into Feisty.  Apparently, Evo 2.10.2 depends on newer versions of certain libraries than are included with Feisty, and trying to upgrade those libraries causes the rest of Feisty to break.
