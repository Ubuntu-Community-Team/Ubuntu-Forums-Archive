### Post by feige on 2010-07-01
[http://ubuntu9.com](http://ubuntu9.com) is a website which can provide top fast/near/stable mirror for your Ubuntu update & upgrade.

top fast test based on average download speed, top near test is on you geo location,and top stable test is on download speed variance.

you can select the best mirror and download its sourcelist to /etc/apt/source.list or put them to /etc/apt/source.list.d/.
