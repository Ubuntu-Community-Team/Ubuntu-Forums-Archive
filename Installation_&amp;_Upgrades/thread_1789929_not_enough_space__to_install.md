### Post by mörgæs on 2011-06-24
You could try this:

In a Windows boot, shrink the size of the Windows installation.

Reboot to verify that everything still works.

Back up your user files from the old Ubuntu.

Test the new release of Ubuntu in a live boot.

If it works well, open Gparted and delete the old Ubuntu partitions.

Install the new Ubuntu from the live boot.
