### Post by Effebee on 2008-01-18
The latest xserver-xorg-core 2:1.3.0.0.dfsg-12ubuntu8.1 update breaks BOINC Manager (boincmgr). Just downgrade to 1.3.0.0.dfsg-12ubuntu8 to solve the problem.


```
sudo apt-get install xserver-xorg-core=2:1.3.0.0.dfsg-12ubuntu8
```
