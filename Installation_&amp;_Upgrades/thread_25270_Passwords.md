### Post by edubarr on 2005-04-09
Ubuntu doesn't have a root password (the root account is 'locked'). It uses a tool called 'sudo' instead, which allows your user to run programs or commands as the root/admin. All you need to do is type your password whenever they as for one. Everything should work fine.

You can find more info at the wiki link bellow
[http://www.ubuntulinux.org/wiki/RootSudo/](http://www.ubuntulinux.org/wiki/RootSudo/)
