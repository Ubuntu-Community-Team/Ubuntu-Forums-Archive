### Post by Wermut on 2005-09-09
Chandros, first you need to realize that Ubuntu is not Windows. Therefore, to run Windows applications you will need a special program (emulator / compatibility layer). You can try wine, which I don't reccomend for a newbie, or you can try Cedega ([www.transgaming.com)](www.transgaming.com)), which is specifically designed to run games (but you will have to pay for Cedega).

But instead you should use programs developed for Linux, which are in most cases even better than their Windows counterparts:
- AntiVirus: You don't need it; with Ubuntu you are safe from Viruses, Worms, Spyware etc.
- Flash: Install the package "flashplayer-mozilla" with Synaptic
- Messenger & IRC: There is Gaim, which is installed by default, whic supports a lot of protocols. For IRC, I use XChat.
- To install printers: System > System administration > Printers

For games, there is no way around a program like Wine or Cedega, since very few commercial games are developed natively for Linux.

However, if the customers of your Internet Cafe demand playing the latest games, you are probably better off with Windows.

### Post by confused_user on 2005-09-24
Win32 nasties can hide in smb shares. I have 3 large fat32 storage arrays serving media to several windows boxes on a network.

The windows machines got infected with sbot.win32, my etrust AV cleaned the windows boxes - but they kept getting infected again - i directed etrust running on the windows box to scan one of the smb shares on the linux box and there it was,

So even though the many flavors of linux are inherently more secure than win2k or xp, they do have there fair share of problems, windows malware can lurk within a linux file system and use removalable media or a tcp/ip connection to propagate to other win32 file systems.

So now i have installed some AV app on my linux server, it has win32, win16 and win8 virus definitions as well as some linux lists.

It is true to say that there are not many known linux viruses, but they are out there and the threat is growing and will continue to do so.

You should examine the reasoning behind viruses in order to decide on how best to protect your systems. Many viruses are released simply to make Microsoft look bad, while an equal number are released by MS and its subsidiaries or sympathizers to counter act the effects of a previous virus. Rivalry among virus authors often prompts retaliatory releases ; remember MSBlaster?, shortly after blaster was in the news a new virus was identified = nachiworm, the original nachi worm was a self propagating worm that spread as fast as blaster did, its purpose was to remove blaster from an infected system and then download a patch from windowsupdate.com to close the vulnerability that allowed blaster to work.

it will not be long before some aggressive business stratagey, annoyed windows programmers etc... decide to target Linux.

So the best advice is to use AV and not just assume that you cant get a virus because your using linux.

As for using ubuntu to run an internet cafe?, better go do some linux courses and do a lot of reading, otherwise your heading into a world of hurt. Done with proper forethought and planning though this could be an incredible cost and time saver and will no doubt provide a far more stable and secure platform. Educating your users on the other hand will be a significant problem which i expect is why you wanted to use windows apps on your linux system.

I have never run an internet cafe before but i laugh as i imagine the security problems you must face, people trying to use your system and network to cause damage elsewhere, users breaking your operating systems all the time, constantly upgrading hardware to meet the needs of the time.

Linux can help with this, users that dont know linux dont know hoe to fiddle with it, they wont be able to use their usual exploits, file permissions is much better in linux.

blah blah blah, malware protection is one of my personal interests so sorry if i went on a bit.
