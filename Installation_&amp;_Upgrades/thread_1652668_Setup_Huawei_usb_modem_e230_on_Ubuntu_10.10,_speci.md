Hello guys.
Just want to say thank you for great software. 
Installed Ubuntu 10.10

Friends asked me to setup internet via Usb modem. 
I thought it will take a lot of time, as usually things, what does not work out of box!

BUT I WAS WRONG!
IT TOOK FOR ME 10 MINUTES.

1. 
apt-get install openssh-server
apt-get install wvdial

2. 
reboot

3. 
was asked for Pin code. I typed.

4. 
on top right corner is NETWORK MANAGNER, clicked something with mobile. 
choose country, then provider, finished.

5. 
again clicked with right click on Network Manager, connect newly made connection. connection established.


WORKS LIKE CHARM.!
THANKS TO UBUNTU 10.10 TEAM.
SPECIAL THANKS TO WVDIAL TEAM.
