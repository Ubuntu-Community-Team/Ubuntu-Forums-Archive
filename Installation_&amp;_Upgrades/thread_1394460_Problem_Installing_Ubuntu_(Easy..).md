What type of install do you want to do?  If you are trying to do an install within Windows I probably cannot help much.

Typically you would boot to the CD but from what you said it is not clear if you are trying to start the CD from within Windows.

One other item if you are trying to boot and are just going straight to Windows the problem might be that you have to "Hit any key to boot CD ...." type of message.  Depending on the computer and if you Windows installed sometimes you have to hit a key during the boot process for it to boot to CD.

And last item assuming you are trying to boot and it is just not working, you may need to go into BIOS (DEL, F10, ESC) depending on your computer make and make sure your CD ROM is set to be the first boot device in the priority.
