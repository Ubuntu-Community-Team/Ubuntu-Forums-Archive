The fastest Pentium 3 processor was 1.0Ghz, or maybe something like 1.2Ghz, if memory serves.  I have many old systems of that spec still in-use, however.  Check your CPU - most socket 370 processors can be upgraded to the 1.0Ghz P3 (if you're using a slower one, for example - I've done this many times).  Also check your bus speed - if you can add 512 megabytes of PC133 memory, this system should run Ubuntu well.  I have four of them around my house, all running Windows XP, haha!  :-p

Check ebay for 1.0Ghz P3's and PC133 ram...
