### Post by rtylershaw on 2005-02-28
Posting for anyone else who might encounter the same problem ... no replies necessary.

Installed yesterday to an amd64 system with an MSI k8n neo mainboard. I'd had problems with my front usb ports in Windows, but the rears worked fine so I put it down to a busted case and learned to live with it. Under linux, no USB ports worked.

After endless messing about I discovered the problem wasn't with Ubuntu, but with the connection to the front USB ports on my mainboard. It's a well-known and solvable issue with the k8n neo; a fix can be found below:

[http://forum.msi.com.tw/index.php?topic=57472.msg381397#msg381397](http://forum.msi.com.tw/index.php?topic=57472.msg381397#msg381397)

Hope this saves time for somebody.
