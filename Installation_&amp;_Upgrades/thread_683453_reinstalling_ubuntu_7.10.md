### Post by chewearn on 2008-01-31
Most people pick-up the habit of reinstalling OS when something is broken from their days in MS-Windows (I was in the same situation once, too).

The good news is, this nasty fix is often not needed for linux.  The difficult part is finding which software is broken, but reinstall that software is usually trivial.

To find out what you accidentally did in synaptic, open synaptic menu > file > history.  Here, you can retrace what you did.

### Post by chewearn on 2008-01-31
Once you enable CCSM, the changing of number of desktop has to be done via CCSM (yeah, I know, it a bit of a hack right now).

Go to: Advanced Desktop Effects Settings > Generak Options > Desktop Size tab; change Horizontal Virtual Size to 4.
