### Post by kc1di on 2019-03-25
I'm not sure what you mean by reset if you want to remove a server then you can do that by going to edit and Account settings and remove the account you no longer want 
if you want to reset the whole program and start new with no accounts you can go to you /home folder or partition and set your file manager to show hidden files and delete the file .thunderbird.  and when you restart thunderbird it will ask for you to create a new e-mail account (warning though if you delete the file you will loose all the e-mails you've already saved.) good Luck.
