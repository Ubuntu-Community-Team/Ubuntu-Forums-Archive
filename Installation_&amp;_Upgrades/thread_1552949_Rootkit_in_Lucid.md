### Post by Rubi1200 on 2010-08-14
What exactly are you concerned about?

Did you check the log?

Are you aware that tools like rkhunter produce false positives?

Did you Google for any anomalies?

File system warnings with rkhunter are quite frequent because file properties change.

It is not necessarily a rootkit or anything else, just the program telling you to check IF something might be wrong.

You know your system better than anyone else, and can tell if something has changed recently.
