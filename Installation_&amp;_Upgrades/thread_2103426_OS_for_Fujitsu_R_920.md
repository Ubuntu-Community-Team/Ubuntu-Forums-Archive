### Post by dino99 on 2013-01-10
E5 is well supported by the actual linux kernels since a while.
Then you need to test the other hardware found into that new workstation.
Download the 12.10 (quantal) or 13.04 (raring) dev iso, install it on an usb stick, then boot on it without installing, to know about possible issue.

I suppose you will install the server iso:

[http://www.ubuntu.com/download/server](http://www.ubuntu.com/download/server)
