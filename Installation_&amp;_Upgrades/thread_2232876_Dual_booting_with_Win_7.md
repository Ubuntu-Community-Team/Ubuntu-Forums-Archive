### Post by yancek on 2014-07-04
Are you saying that in the Installation Type windows, you do not see the option to "Install Alongside other operating system"?  If that is the case, you can select the "Something Else" option which is basically a manual install.  You will have to have an understanding of Linux naming conventions for drives/partitions/filesystems and know what partition windows is on.  If you get to the window where the partitions should be listed and they are not, you're on dangerous ground if you are not familiar with the above.
