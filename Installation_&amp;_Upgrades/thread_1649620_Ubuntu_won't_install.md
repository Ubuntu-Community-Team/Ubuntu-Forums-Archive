### Post by tommcd on 2010-12-20
> **kenn_R said:**
> When I try to install ubuntu from the bootable CD ... which I know works, because I used it on my other computer
It is an older computer that I upgraded to xp, is it possible it's age is affecting or causing this?

First, just to rule out a bad CD, it would be helpful if you could run the *Check CD for defects* option from the live CD:
[https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)
If te CD check reports no errors, then give us your hardware specs as others have said. 
In particular, if you have less than 256MB memory, then you should probably install Ubuntu via the alternate install 
(text based) CD. 
Or better yet, install Lubuntu on an older machine: [http://lubuntu.net/](http://lubuntu.net/)
