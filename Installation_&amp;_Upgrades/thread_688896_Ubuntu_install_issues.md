I recently decided to make Ubuntu my primary OS.  I just repartitioned my laptop to give Ubuntu the majority of the hard drive and Windows about 10 gigs.  

I've installed Ubuntu twice now from the Ubuntu 7.10 cd, and evertime Ubuntu loads, there is no loading screen, just a black screen, however it does load Ubuntu eventually.  It also seems to take a bit longer to boot than it did with the partition I had earlier.

Also, I tried to enable the Visual effects, and I get an error message that says "The Composite extension is not available".

### Post by charlescarroll1 on 2008-02-06
I solved my problem (probably the hard way).  

I wiped the partition I had originally made for Ubuntu and installed Ubuntu 7.04 (from CD) instead, which installed flawlessly with no problems afterwards.  

From there I updated to 7.10 via Update Manager and the problems have seemed to go away.  I don't know if it was the disc I was using or what, but this guy...

[http://ubuntuforums.org/showthread.php?t=686631](http://ubuntuforums.org/showthread.php?t=686631)

... had the exact same thing happen to him, while he was doing something very similar.
