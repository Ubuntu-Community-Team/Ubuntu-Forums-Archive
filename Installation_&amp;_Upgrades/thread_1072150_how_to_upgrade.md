### Post by Kevbert on 2009-02-17
Assuming that you're using Kubuntu. KDE4.2 is still work in progress but will appear as part of the Jaunty release in April. If you want to install Jaunty now (not recommended as still in Alpha) you can get it by going to Konsole and typing the following:
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade
```
Even though it seems very stable to me there's a good chance something may break prior to the full release. Be prepared to do a re-install at some stage if you wish to try it.
