The .ipk file extension is for packages designed for the Itsy Package Management System (iPKG)

These packages are for use with portable devices such as the Sharp Zaurus PDA the iPAQ. The package system is not intended for use with a standard Linux (desktop) system.

You won't be able to install such a package on a normal system. You will need to find a .deb of libtiff3g if it is not available for installation through Synaptic.
