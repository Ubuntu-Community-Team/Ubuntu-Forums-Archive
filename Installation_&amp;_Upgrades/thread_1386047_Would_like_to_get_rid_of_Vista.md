### Post by Cabs21 on 2010-01-20
Dont know if you can remove vista and keep ubuntu if your running it in a virtual machine.  However if you back up your data/settings from the Ubuntu virtual machine on a removable device (USB or CD) you could do a fresh install of ubuntu from a Live CD.  This will format and wipe your hard drive clean so that you can install ubuntu from a clean slate.  

Make sure you save any data you want from windows as it will be erased when formatting.

### Post by darkod on 2010-01-20
There might be some special procedure to do it, but the short answer is, NO. Wubi is not full ubuntu and is running inside windows, as it says. So getting rid of vista will get rid of wubi too.

As recommended, copy all the data you need from wubi and vista to external drive, then just use the ubuntu cd to install ubuntu on the whole drive as only OS (if you still want to get rid of vista).
