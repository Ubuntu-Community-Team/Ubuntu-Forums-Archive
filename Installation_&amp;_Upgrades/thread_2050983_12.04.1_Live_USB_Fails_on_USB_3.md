### Post by iaw4 on 2012-09-01
I have a Shuttle SH67H3 machine from 2011.  It's standard Intel Ingredients.  The Shuttle can boot the Live USB installer for 12.04.1 very nicely when the stick is inserted into the USB 2.0 port of the Shuttle, but it dies (at some mount point) when it is inserted into the USB 3.0 port of the Shuttle.  Go figure...

I just wanted to tell others about this problem.  Once the full distribution is installed on the hard disk, the USB 3 ports are usable without trouble.  since an install is a one-time issue, the fix is just to use the USB 2 ports.
