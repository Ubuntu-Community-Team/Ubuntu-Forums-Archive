### Post by MARP1961 on 2011-12-29
You obviously need the Windows install disk of the version you deleted. Did you burn recovery disks? If not you will need to 'obtain' exactly the right version to allow your Product Number to activate Windows once more. If you have your disk, it is bootable and can format and install Windows again without problems.

You can also use your Ubuntu disk to format: Boot from disk, choose 'Try Ubuntu' and access **GParted **in the applications menu. GParted can do the job of a Windows disk except reinstall Windows!
