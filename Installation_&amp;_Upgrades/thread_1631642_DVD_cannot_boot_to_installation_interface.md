### Post by tommcd on 2010-11-27
Don't bother with a DVD version of Ubuntu.
Get the 64bit Ubuntu live CD from here:
[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)
Then see this for how to burn the iso image to a CD:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Be sure to burn the CD at the *slowest possible speed*.
Then when you boot from the CD, first run the option to *check CD for Defects*: [https://help.ubuntu.com/community/Installation/CDIntegrityCheck](https://help.ubuntu.com/community/Installation/CDIntegrityCheck) If the CD check reports no errors then the CD is valid and can be used to install Ubuntu.
And welcome to the Ubuntu forums!
