### Post by neu5eeCh on 2010-10-01
Just wanted to add my two cents. I also don't like Ubuntu's new Font. 

It's spindly and hard to read - esp. on my 1080p laptop. Isaas, I changed the fonts by going to systems --> preferences -->  Appearance Preferences --> (Fonts Tab). Once there, I changed my Application font to Britannica Unicode Sans Roman. Don't know if you would like that but it certainly is far more readable than Ubuntu's font.

I've noticed though, that Firefox doesn't render web page fonts as well as it used to. I have no clue why this is the case. I **did** install the proprietary ATI 3D Graphics accelerator - which was buggy under 10.04.
