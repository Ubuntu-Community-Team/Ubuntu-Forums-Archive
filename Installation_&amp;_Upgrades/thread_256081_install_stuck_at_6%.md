### Post by Fully216 on 2006-09-12
Any system should be able to run the 'proper' live CD based on the correct system hardware, because the live CD does not use the hard drive at all.  It seems to be stuck at places where the install is trying to access the internet, although the install can be done without internet access as mine was.  I assume you do have the system hooked up to the net, no?

I would try running the live CD for a while.  You should be able to do everything there that you would be able to do on an installed machine, the only difference is it runs off CD instead of your HD.  This will let us know if there is a hard drive issue with your machine or if it was a download/burn issue with the CD.  See if you can browse the web with the live CD as well, which you should be able to.

Make sure you are downloading the correct CD version (ex. x86, amd, etc).  You will know if the CD doesn't run.
