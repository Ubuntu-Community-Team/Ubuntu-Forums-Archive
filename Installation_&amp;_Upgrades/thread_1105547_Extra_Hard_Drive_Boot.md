### Post by lindsay7 on 2009-03-25
One way, probably not the best, if there is one, would be to disable the windows hard drive and install Ubuntu on the other. You would have to enable or plug in the drive that you want to use when you start the computer. Not to easy unless you case is open all the time. Plus, I have found the hard way that drives do not like to be messed with if the power is on. So you have problems doing it this way.

I have not done this, but I guess you could also set your bios to boot from the windows drive or the Ubuntu drive everytime you start the computer. This would be in the bios under to boot priority. You would have to switch boot order to something like this.

1. cd drive 2.drive a. then drive b  3. other device

or

1.cd drive 2. drive b. then drive a  3. other device

You should consider installing Ubuntu under windows. You do have to choose which one you want ot use when you start the computer but it is easy and you can uninstall Ubuntu under windows (control panel) at any thime.
