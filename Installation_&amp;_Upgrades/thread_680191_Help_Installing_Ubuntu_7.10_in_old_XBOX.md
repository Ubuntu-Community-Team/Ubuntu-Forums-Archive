### Post by xyphur on 2008-01-27
I'm not sure you can run Ubuntu on the original Xbox, but I have XDSL (Damn Small Linux for Xbox) running on mine... I would assume a regular install of Ubuntu would be too much for the Xbox due to it's limited hardware specs (mainly the lack of sufficient RAM).

The following link should help familiarize you with running Linux on the 1st gen Xbox

[http://www.xbox-linux.org/wiki/Getting_Started](http://www.xbox-linux.org/wiki/Getting_Started)

Here is more info on XDSL and how to go about running it depending on your particular setup (modchip, softmod, etc): [http://www.xbox-linux.org/wiki/XDSL_HOWTO](http://www.xbox-linux.org/wiki/XDSL_HOWTO)
