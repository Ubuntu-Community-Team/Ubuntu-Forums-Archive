### Post by fmorriso on 2008-12-10
ubuntu 8.10 desktop began hanging during the reboot process after I shutdown my computer and then physically removed a USB thumb drive. I never did a logical "unmount" on that drive before shutting down, which seems to be the key to understanding both the problem and (at least for me) the cure.

Apparently, ubuntu 8.10 doesn't like it if you simply physically remove a USB thumb drive after shutting down.  You don't realize this until your next boot hangs.  The cure (again, at least for me) was to:
[LIST=1]
[*]shutdown
[*]put the thumb drive back in the same slot it was in previously
[*]reboot
[*]right-click the icon for the thumb drive and choose Unmount.
[/LIST]
Hope this helps others experiencing the same issue.
