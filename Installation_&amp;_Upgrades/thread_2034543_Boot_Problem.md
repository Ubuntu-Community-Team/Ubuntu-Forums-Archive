### Post by Ep1c0wl on 2012-07-28
The reason it is automatically booting to Windows is because your primary boot device is your HDD. If you would like to boot to the live disk, you need to either go into the bios options and change the boot order or just select the CD/DVD drive to boot from when your computer starts up.

256Mb of RAM is fairly minimal but it can be done, especially if you run lighter releases such as Lubuntu, but this is a different issue to the ones being discussed here.

You should probably try the dual boot install (backup your data first!) and then start a new thread if you want help.
