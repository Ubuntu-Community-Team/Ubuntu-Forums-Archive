### Post by JOHNNYG713 on 2013-03-21
The LTS version of Ubuntu 10.04 is supported for three years for desktop  releases and five years for server releases; from Ubuntu 12.04 onwards,  LTS support will be for [five years]("http://www.h-online.com/news/item/Canonical-extends-Ubuntu-12-04-LTS-desktop-support-1365034.html")  on both desktop and server. Ubuntu 10.04 LTS will continue to receive  maintenance updates and security fixes until April 2013 on desktops and  April 2015 on servers. Standard releases provide 18 months of updates  for both the desktop and server versions; Standard releases from 13.04 on will be supported for nine mouths.
