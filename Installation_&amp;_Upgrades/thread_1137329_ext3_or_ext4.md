### Post by autocrosser on 2009-04-25
There are pluses & negatives for both---I've been using ext4 from the first (Jaunty alpha 2) & really like the speed--On the negative side--ext4 is using delayed allocation that "can" have some problems with data loss--I have only had 2 problems (both very early on) & it has been very smooth for me--My system (2 installs ready for Karmic testing, 1 Jaunty install, 1 Jaunty64 install that will also be Karmic testing & 1 Fedora11 install) is totally ext4 & I have bid ext3 a fond farewell---

If I were you, I would look thru the closed Jaunty-testing forum (in Development & Programming) & search about ext4--There were several active threads during testing---I am a ext4 "fanboy", so read all about it before you make up your mind....:)
