```

...
PATH=/var/lib/gems/1.8/bin:$PATH

```

 
Actually, correct syntax for PAM config variable definition:
```

PATH DEFAULT=${PATH}:/var/lib/gems/1.8/bin

```
 
See [http://manpages.ubuntu.com/manpages/hardy/man5/pam_env.conf.5.html](http://manpages.ubuntu.com/manpages/hardy/man5/pam_env.conf.5.html)
