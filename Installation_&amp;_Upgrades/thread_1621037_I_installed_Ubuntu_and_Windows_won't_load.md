### Post by oldfred on 2010-11-14
Since Ubuntu is not nearly as large as windows, You should be able to get some data back with photorec which is part of test disk. It also recovers old partial files, many types of files, bits and pieces sometimes combined and you have to separate. It can be time consumming but it depends on how valuable your data is and how good your backups are.

Testdisk & photorec
[http://www.psychocats.net/ubuntucat/recoverdeletedfiles/](http://www.psychocats.net/ubuntucat/recoverdeletedfiles/)
