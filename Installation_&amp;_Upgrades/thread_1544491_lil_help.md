### Post by Maverick_Meerkat on 2010-08-02
Greetings,

If you are trying to install and run windows apps on Ubuntu you need to get WINE. You can find it here: [http://www.winehq.org/download/](http://www.winehq.org/download/) 

Alternately, you would need to use virtual machine software such as Sun Virtualbox. [http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads)

I suggest trying WINE first.
