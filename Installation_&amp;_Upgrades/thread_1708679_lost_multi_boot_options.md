### Post by Quackers on 2011-03-17
Yes, assuming that the Ubuntu-installed grub2 is in control. This can change if other installations of grub get updated. Watch for that. You will be able to tell by the top entry in the grub menu - it will change when a different grub is in control.
If that happens, you can just boot into Ubuntu 10.10 (if that is the one that you want to control booting) and run ```
sudo grub-install /dev/sdX
``` where sdX is the hard drive bootable in your bios (usually sda or sdb, for instance - NOT sda5 - no partition number required, as you want grub in the mbr, not a partition).
Glad you got things back :-)
