The memstick is a "live installation" which is intended for you to try the OS and discover whether there are compatibility issues with your hardware. It is a highly compressed version of the OS which is not intended for log term use, for extended tests you can make your memstick [persistent.]("https://help.ubuntu.com/community/LiveCD/Persistence") You can install the full OS to a memstick to achieve running a fully "installed" OS. Read [this thread]("http://ubuntuforums.org/showthread.php?t=1840379") which provide guidance.
