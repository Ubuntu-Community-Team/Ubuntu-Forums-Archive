### Post by Malcolm Waterman on 2010-03-09
I tried running Ubuntu Linux 9.10 on my computer directly from the CD, without installing, and when it had finished loading neither the keyboard nor the mouse would work. I got a message saying that restricted drivers had been blocked. I can't press what I think to be the unblock command because both the keyboard and mouse had been blocked.

---

### Post by Malcolm Waterman on 2010-04-19
Ubuntu does work properly when it is put into a different computer. I was wondering if vital drivers for the working of the mouse and keyboard could have been blocked by Ubuntu.

---

### Post by Malcolm Waterman on 2010-06-17
Problem's solved. Incorrectly set setting in BIOS. Needed to be set to Legacy USB support.
