### Post by Partyboi2 on 2007-12-17
Yes you can dual boot Vista with Ubuntu. what you will need to do is resize Vista to make some room for ubuntu. To do that you will need to use the tool that comes with Vista. this link should help you get started.
[http://apcmag.com/5046/how_to_dual_boot_vista_with_linux_vista_installed_first](http://apcmag.com/5046/how_to_dual_boot_vista_with_linux_vista_installed_first).
512 ram is enough to be running both o/s as only one o/s will be running at once.
If you run into any problem, ask and someone should be able to help or put you in the right direction. :)
