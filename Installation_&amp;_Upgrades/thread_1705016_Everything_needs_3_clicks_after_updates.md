Craig,

Based on the symptoms, I think a background process is eating up the CPU.  This makes mouse clicks and movements slow way down or get ignored.

Please menu to System >> Administration >> System Monitor.  When the monitor comes up, select the 'resources' tab.  That will show you how much CPU and Memory are getting used up. The 'Processes' tab will show you WHO is using up the CPU.
