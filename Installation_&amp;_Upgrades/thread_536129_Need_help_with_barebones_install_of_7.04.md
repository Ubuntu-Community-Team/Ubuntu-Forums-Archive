### Post by jnorthr on 2007-08-27
Can you advise if you have completed an installation of ubuntu from the 'live cd' onto your machine ? You do not need to be online to do the install. In fact, you will not be able to maintain a connection while doing so as there is so much copied to your machine during the install. 

Can you run ubuntu from the CD ? If so then we can safely say your machine is capable of supporting ubuntu. You will probably not be able to gain an internet connection while running from the Cd as it does not use your hard disk (or only a tiny amount) so it will not allow you to do that. You can use the Cd to explore the various tools and packages  to see if it fits your needs/you like it. When you are satisfied that it does, you can do an install using an icon on the desktop ( i think ).
