### Post by kodos on 2005-11-07
I tried reburning the install cd, and the newly burned cd had a different md5sum than the previously burned on.  Installing using the newly burned cd went without incident.  So if you have come accross this theard with a similar issue, try re-burning your install cd, perhaps at a lower speed, and try installing again.
