### Post by expelledboy on 2009-06-14
Start up the machine and when it gets to the grub timeout hit the excape button to get into the menu list. Instead of booting into ubuntu using the first entry use the second.. the *recovery mode* kernel.

It will take you to another menu list where you should select the *fix broken packages* and then used the *xfix* or *fix X* option.. I forget what it was exactly. :)

It should then work. You might have to then afterwards reinstall your graphics drivers.
