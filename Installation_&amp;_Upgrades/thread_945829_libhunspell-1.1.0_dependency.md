There are several references when I search the web for the libhunspell dependency error I am getting.

As a work around I will go back to 8.0.4 and install Firefox-2.

The reason I went to 8.1.0 was for the atheros drivers which worked perfectly for several USB wifi devices.

I will need to use a wireless to ethernet bridge to get wifi access on 8.0.4.

Just put this out here in case anyone had a similar issue.
