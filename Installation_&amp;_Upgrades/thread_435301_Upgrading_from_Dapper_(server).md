You can't upgrade directly from Dapper to Feisty.  

[https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)

But you can upgrade to Edgy and then to Feisty.  A reinstall might be easier if you haven't done much customization (users, chroot, etc).  Myself, I upgrade my server because it's just too much trouble redoing all the user info.

Here's the upgrade method
[http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_upgrade_from_Hoary_Hedgehog_-.3E_Breezy_Badger_-.3E_Dapper_Drake](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_upgrade_from_Hoary_Hedgehog_-.3E_Breezy_Badger_-.3E_Dapper_Drake)

I think that you want Method B.
