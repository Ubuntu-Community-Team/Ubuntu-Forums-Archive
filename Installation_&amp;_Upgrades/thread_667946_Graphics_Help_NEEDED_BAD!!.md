### Post by xeth_delta on 2008-01-14
> **bbrownderville said:**
> Ok. so i changed it and went to save it and it says i dont have permission to change it or so forth.  how do i go about bypassing that?  i am a total nubee.

No problem, you have to open the file as a user with administrative privileges. To do so, you can open a terminal/console and type the following:
```
gksudo gedit /etc/X11/xorg.conf
```
or if you want to use nano
```
sudo nano /etc/X11/xorg.conf
```
For nano, you save the file with Control-O and exit with Control-X.

In both cases you will be prompted for your password.

### Post by xeth_delta on 2008-01-14
> **bbrownderville said:**
> my max resolution is 1280 x 800.. and i need a bigger resolution..

Once again editing xorg.conf might have the solution. Have a look at that file and you will notice the following:
```
Section "Screen"
Identifier "Default Screen"
Device "Intel Corporation Mobile 945GM/GMS, 943/940GML Express Integrated Graphics Controller"
Monitor "Generic Monitor"
DefaultDepth 24
SubSection "Display"
Modes "1280x800"
EndSubSection
EndSection
```

In the line reading Modes, add the extra resolutions you would like, for instance:
```
Modes "1280x800" "1600x1000"
```
or if you want your screen configured directly for another resolution
```
Modes "1600x1000"
```
