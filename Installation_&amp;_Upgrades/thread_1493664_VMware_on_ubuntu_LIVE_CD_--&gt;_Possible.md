### Post by lechien73 on 2010-05-26
You can customise the Live CD, there's quite a few tutorials on how to do it, including:

[https://help.ubuntu.com/community/LiveCDCustomization](https://help.ubuntu.com/community/LiveCDCustomization)
and
[http://ubuntuforums.org/showthread.php?t=515574](http://ubuntuforums.org/showthread.php?t=515574)

The latter tries to automate everything using a bash script, but since it was written for Feisty and Gutsy, I don't know if it will work with Lucid. I'd probably stick to the first reference, if I were you.
