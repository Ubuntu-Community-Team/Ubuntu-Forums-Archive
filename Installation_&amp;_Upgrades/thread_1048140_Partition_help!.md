### Post by mdewet on 2009-01-23
if you want to install only Ubuntu on it then 1 partition could do..otherwise you can set up a 10 GB partition for the ubuntu install, 1 GB swap, and the rest for data etc..this can all be done from the Ubuntu installation GUI

this howto gave me all i needed to know when I started

[https://help.ubuntu.com/community/HowtoPartition](https://help.ubuntu.com/community/HowtoPartition)
