Hi,

I have a small recommendation for distribution upgrade process. 

When an upgrade process is started, the update manager should bring up a new window and show a progress bar (like the way it does now in update manager window), and show the new features and improvements in the new distribution with pictures and text. Kind of like how Windows shows during a new install highlighting the new features in the new OS being installed. 

The user will still be able to look at the shell output if he likes using the same UI as it is now.
