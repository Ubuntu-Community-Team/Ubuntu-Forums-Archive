I found the easiest way, re-enable root to be able to login under the gnome gui. uninstall Ubuntu's firefox. Download Mozilla's version of it and then install it in /usr/lib/. You can even install it without uninstalling Ubuntu's version just the same, just make sure shortcuts point to the right folders for each. You can always just reactivate the update features too as root and it'll find the Mozilla updates as well.
