### Post by sharperguy on 2006-09-28
When you install ubuntu, you should get a dialogue to split (partiton) your drive into two parts for ubuntu and windows.

I thinks baisically there is a bar you drag to set the proportion to each operating system, it seems to be easy, although I didn't use that method as I  have a second drive with ubuntu on.

As far as choosing, when you turn on your computer, you will be given a menu with Ubuntu, Ubuntu Recovery Mode, Memory Checker, and Windows.

Ubuntu boots automatically after 10 seconds, although you can change that if neccesary.
