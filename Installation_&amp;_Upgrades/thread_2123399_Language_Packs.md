### Post by grahammechanical on 2013-03-09
You will not need to install language packages because you chose a language during the installation. If you want to install another language or another language keyboard layout after installation you can do that through System Settings> Language Support and Keyboard. We only install Ubuntu with one language anyway. Other languages for the system or simply as keyboard layouts are done after installation using system settings. 

During installation Ubuntu copies files from the CD to RAM and towards the end of the installation process it removes some of those files (including language packs) from RAM. I am not sure that these are downloaded during installation. Unless we tick to update during installation and there are newer versions of those packages available.

Regards.
