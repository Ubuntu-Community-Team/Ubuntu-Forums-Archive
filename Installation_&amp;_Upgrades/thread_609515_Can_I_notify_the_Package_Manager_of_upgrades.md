### Post by ofb on 2007-11-11
How it works is Ubuntu has regular 6 month releases with current versions of various software tested to work together. The updates you get before the next release will be bug & security patches, but not version upgrades. Version upgrades come with the next Ubuntu.

This is the Ubuntu method of presenting a stable system & avoiding dependency heck.

Sometimes there are exceptions via the backports repository, or if you are using repositories set up for Ubuntu by your particular software. I think claws-mail, for example, has their own repository that you can add to the Third Party section.
