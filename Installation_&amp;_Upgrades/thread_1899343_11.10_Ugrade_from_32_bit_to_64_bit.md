---

### Post by BC59 on 2011-12-23
First you have to uninstall the kernel and install the 64bit version. Then the applications you have installed are of the 32bit architecture. So you have to uninstall them and install the 64bit version.
Finally you have to do so many changes, that a fresh 64bit install is easier and much more effective.
