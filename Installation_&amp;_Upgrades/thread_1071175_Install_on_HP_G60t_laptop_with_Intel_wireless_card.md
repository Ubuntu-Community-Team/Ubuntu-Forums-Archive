### Post by supportgeek on 2009-04-27
Hi,

I have an HP G60t with the N wireless upgrade.

I've been using Ubuntu version 8.10 on this laptop without issues except for the mouse pad button did not work.

Meaning the little button in the middle/top of the mouse pad does nothing when you try to disable it.

Last week I upgraded to version 9.04 and all seems to be working great.

I do not think Ubuntu or Linux has support for N wireless yet but the card seems to be working fine in G mode.

Over all this is a great laptop for the price and runs very smooth with Ubuntu 9.04.
