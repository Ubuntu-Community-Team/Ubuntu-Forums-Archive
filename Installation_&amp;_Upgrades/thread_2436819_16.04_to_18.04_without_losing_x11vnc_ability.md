### Post by TheFu on 2020-02-13
Upgrades between major versions will always, always, loose network connectivity.

I have done major upgrades over **ssh** a few times, but only when I knew I'd have access the following day.

Regardless, performing a full, know-you-can-restore backup before this operation is NOT optional. It is mandatory, unless you are fine loosing everything.

VNC should **never** be used remotely without either a full VPN or ssh tunnel used.
