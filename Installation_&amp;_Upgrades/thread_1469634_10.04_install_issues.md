```

acpi=off

```
or some noapic option. To do this you should press a random screen at boot startup see [http://www.ubuntu.com/getubuntu/releasenotes/1004#Boot%20options%20hidden%20by%20default%20on%20Desktop%20and%20Netbook%20CDs]("http://www.ubuntu.com/getubuntu/releasenotes/1004#Boot%20options%20hidden%20by%20default%20on%20Desktop%20and%20Netbook%20CDs")
