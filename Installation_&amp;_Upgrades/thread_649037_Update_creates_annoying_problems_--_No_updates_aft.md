### Post by kishorebudha on 2007-12-24
I have installed Gutsy on my Inspiron 6000 (dual boot with XP. XP being in first partition). Laptop specs are 1.73ghz Centrino, 1GB Memory, ATI Mobility Radeon x300, Intel Pro/Set 2200BG, 1200x800 wxga screen, Ricoh SD Card reader, I have installed Gutsy some 7 times now -- tweaking around till I messed it up and did a reinstall every time. In particular I noted the following problems after installing updates.

1. Computer freeze up, excluding mouse. This was solved by disabling/uninstalling Compiz.
2. SD Card slot is read as mmcblk0p1 and has problems in mounting/unmounting. Mounting/unmounting can only be done through root access.
3. Wireless is randomly not recognised. Rebooting does not help. Only rebooting with XP and then rebooting with Ubuntu brings wireless back.

Out of hunch, I did a clean reinstall and have not applied any of the software updates. And what do I see?

1. No computer freeze up when using firefox with compiz.
2. SD Card drive is recognized as SD Card and there is no issue with mount unmount using the GUI.
3. Wireless is stable.
