### Post by opm8 on 2007-10-22
It's obvious to me from the many threads here with Adept problems that this utility is broken and largely useless for upgrading.  

I would like to discuss the feasibility of moving away from a clunky gui-based upgrade and using a command-line method. It seems with every upgrade things get worse.  6.04 - 6.10 was a nightmare, 6.10 - 7.04 was only slightly better, but still with many issues. Now with the push to 7.10 we have a utility that in many cases refuses to do the very thing the "official" method is asking of it: to simply perform a version upgrade. 

I get the impression Adept is not used for anything besides upgrades.  Synaptic is much more user friendly if you're into guis but nothing beats apt-get or aptitude.  There should be no reason to force users to go through the hell of a broken front end to a perfectly good package management system when something as critical as version upgrades is concerned.  Adept is clearly not ready to be relied on.  In the future I would like to see a set of "official" command-line instructions to complement whatever gui-based method is going to be used.  How hard can it be?  We're not all pathetic noobs -- some of us actually understand linux.
