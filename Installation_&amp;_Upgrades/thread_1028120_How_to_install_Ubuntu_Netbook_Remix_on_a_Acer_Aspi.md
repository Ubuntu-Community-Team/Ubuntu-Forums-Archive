There *is* an available Ubuntu Netbook Remix 1.0.1 iso. I installed it in my Aspire One and it worked great :D!

Just beware it is **NOT** a live CD/USB. If you boot and hit enter it **will wipe the drive completely** and install Ubuntu Netbook Remix! It does not offer drive partitioning in the install...

Check out:
[https://wiki.ubuntu.com/UNR#Installation](https://wiki.ubuntu.com/UNR#Installation)

I've tried Acer's version of Linpus Lite they put on the Aspire One's with the SSD drives. It is way faster on boot up and shut down but if you don't like the restrictive desktop it's not as enjoyable to use.... That's why I switched to UNR (besides the fact all of our other systems are Kubuntu/Ubuntu based :biggrin:). UNR is rather slow booting when compared to Linpus Lite which is Xfce based but a lot more flexible.

Enjoy!
