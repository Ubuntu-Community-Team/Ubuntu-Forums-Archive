### Post by oldfred on 2011-02-18
Is it this?

boot.ini entry:
c:\wubildr.mbr="Ubuntu-Linux"


If editing windows files like boot.ini
(Using nano instead of gedit, it's better for dos-style linebreaks)
Linux, of course, uses only LF as newline and DOS is expecting CR/LF so text files look wrong in DOS.

How to edit the Boot.ini file in Windows XP
[http://support.microsoft.com/kb/289022/](http://support.microsoft.com/kb/289022/)

### Post by Daanii on 2011-02-18
Thanks very much, that trail led me straight to the solution. 

Turns out that Windows no longer has boot.ini, as of Vista. But there is a nice little third-party editor called EasyBCD that lets you edit boot.mgr (the replacement for boot.ini). That took care of it, and it was indeed easy. 

One little annoyance out of the way. You made my afternoon. Thanks again.
