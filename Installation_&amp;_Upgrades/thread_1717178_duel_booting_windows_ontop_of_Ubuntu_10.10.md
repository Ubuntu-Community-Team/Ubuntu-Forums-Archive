### Post by Rubi1200 on 2011-03-29
Hi and welcome to the forums :-)

First, backup all important data.

Second, partition the drive from the Ubuntu CD using GParted but do not format.

Third, install Windows to the unallocated space.

Fourth, reinstall GRUB to allow dual-booting.
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD]("https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD")
