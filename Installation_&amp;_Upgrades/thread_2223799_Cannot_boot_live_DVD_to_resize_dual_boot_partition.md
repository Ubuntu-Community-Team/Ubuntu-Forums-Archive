### Post by squakie on 2014-05-14
How did you create the media?   For example, you can't just copy/burn the files to it - you must use one of the tools.  To make an Ubuntu USB install/boot media use unetbootin.  To make an Ubuntu install/boot media you must burn the ISO image to the disk as an image - you can't just burn the ISO to disk.
