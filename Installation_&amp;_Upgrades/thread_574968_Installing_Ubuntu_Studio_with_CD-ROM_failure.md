### Post by aikiwaza on 2007-12-26
for anyone else who reads this... the problem turns out that the ubuntu studio disk I was using wasn't a live disk and therefore there was nothing to mount the usb cdrom to.  When I used the 7.04 Feisty disk (which is a live disk) it worked perfectly... hopefully this might save someone some headache.
