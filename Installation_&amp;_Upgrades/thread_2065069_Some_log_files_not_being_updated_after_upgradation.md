The last line in messages log sections (as on 26th Sept) reads: "kernel logging (proc) stopped". ### Post by 2F4U on 2012-10-01
The log file /var/log/messages has been deprecated in Ubuntu 11.04

[http://askubuntu.com/questions/51265/where-is-var-log-messages](http://askubuntu.com/questions/51265/where-is-var-log-messages)

See also comment #8 from this bug report

[https://bugs.launchpad.net/ubuntu/+source/rsyslog/+bug/762505](https://bugs.launchpad.net/ubuntu/+source/rsyslog/+bug/762505)

The log file you should be looking at is /var/log/syslog.
