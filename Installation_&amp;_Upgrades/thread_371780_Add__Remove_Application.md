### Post by Koybe on 2007-03-03
Sorry for the late answer.

I guess the problem is the gr repository.

Copye back your sources.list and change it by remove every gr. in the file :

```
sudo cp /etc/apt/sources.list /etc/apt/sources.list.old
sudo sed -e s/gr.// /etc/apt/sources.list.old > /etc/apt/sources.list
sudo apt-get update
```
