### Post by stimpsonjcat on 2007-04-24
> **xp_newbie said:**
> can they import Word documents in a way that OpenOffice weren't able to deal with so far?
Unfortunately not. the .doc format is a proprietary secret. there are, however, two ODF plugins available for Word. one is sponsored by microsoft and can be downloaded [from sourceforge]("http://sourceforge.net/projects/odf-converter"). the other one was/is developped by sun. [download here]("https://sdlc3d.sun.com/ECom/EComActionServlet;jsessionid=D2A0D10565429B3D5E836CB70A15100B")
I think that both programs need a version of word that supports the newer "Open"XML file format.
they're both free software.
