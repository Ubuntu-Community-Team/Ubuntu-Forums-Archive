### Post by logos34 on 2008-02-26
As far as the slider goes, whatever you set it to equals the new (shrunken) size of the vista partition, with the remaining to be used for ubuntu.

But you don't want to do that.  Use Vista's own disk utility to shrink it down.  Then go back and install ubuntu on the freed space.
