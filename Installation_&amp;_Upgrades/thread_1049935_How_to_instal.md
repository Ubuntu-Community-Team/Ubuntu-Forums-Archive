### Post by bgerlich on 2009-01-25
Aside from some custom configuration you don't need to use the terminal. Software is installed either using the Add/ Remove in Applications (set "Show" to all available applications for the full list) or by downloading software packages from sites like [http://www.getdeb.net/](http://www.getdeb.net/).

If you want to learn how to use the terminal (console) check out this tutorial:
[https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)
