### Post by lhtown on 2007-01-27
Don't sweat it. We all run tons of stuff from universe. I wouldn't call it untested although I have had one or two beta/alpha stage programs that were basically non-functional for me. Basically, that is like running it from the Debian equivalent. I wouldn't expect updates on universe stuff, but in 6 months, you can do a complete system upgrade.

If you were a system administrator of a number of computers and were using Ubuntu's professional support options, you would probably want to try to stay out of universe.

That said, if I am trying to decide between two packages and one is in universe and the other is in the supported repository, I will usually try the one in the supported repository first since I assume that if the Ubuntu people like it, it is probably the better of the two.

Welcome to Ubuntu. Debian is nice, but Ubuntu is almost like running Debian unstable with polish.
