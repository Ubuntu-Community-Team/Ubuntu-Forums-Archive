### Post by Thorndeux on 2010-05-12
I just wanted to point out that Wesnoth 1.8 has been released (but is not yet in the Ubuntu repositories). So you might want to check out whether the problem persists if you install the new version.

Edit: 1.8 IS in the repositories, it's just not the default as it was not ready at Lynx's release. In Synaptic search for a package called 'wesnoth-1.8'.

Edit 2: The problem does persist in 1.8, but as I play in fullscreen mode anyways the Ctrl+F solution is an acceptable workaround for me.
