### Post by ruibernardo on 2007-08-18
If your XP partition is the first on the disk, delete all the rest and create an extended partition, then install a fresh Ubuntu install.

Create an extended partition and install Ubuntu in there (with or without a separate /home partition in it). Inside the extended partition you can install all the partitions you want, although your problem about Internet/network is another issue (192.168.1.1 is a private network, not internet, so you must have changed something in the mean time).

Don't forget to install GRUB on the boot of the drive (it's the default option on install anyway).
