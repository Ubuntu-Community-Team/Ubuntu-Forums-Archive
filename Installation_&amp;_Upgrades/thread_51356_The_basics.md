### Post by angkor on 2005-07-26
The easiest way of installing software on Ubuntu is using a program called Synaptic, you'll find it in System-> Administration. Synaptic installs .deb files specially designed for Ubuntu.

If you downloaded the program source there's probably a Readme file included which tells you how to install the program.
