Hi,

Instead of logging in as root, please use ```
sudo
``` before the commands you wish to issue. If you want Webmin (or any other GUI application) to start with root priviledges then use ```
gksudu webmin
``` if you are running ubuntu, 

or ```
kdesu webmin
``` for kubuntu. 

Starting GUI applications with just sudo can seriously mess up permissions.

[http://ubuntuforums.org/showthread.php?t=716201](http://ubuntuforums.org/showthread.php?t=716201)

On the matter of samba, if you find your processor goes to 100% usage (i.e. the computer seems to hang) open up a fresh terminal session and issue ```
top
```, if you see gvfsd-smb-browse running at 100% (or close to), then it is a bug. There is a thread on that happening over here --> [http://ubuntuforums.org/showthread.php?t=1426517](http://ubuntuforums.org/showthread.php?t=1426517)   If you are affected, please add any information you can to the report.

Regards,

Phill.
