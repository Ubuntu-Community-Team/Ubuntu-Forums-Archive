### Post by wpshooter on 2007-01-24
When you say "I have set my BIOS to CD priority", if by that you mean that you have set the CD-Rom drive as the FIRST boot device and then you have to CD in the drive and it is NOT booting to the Ubuntu CD, then it is most likely because you **COPIED** the ISO image to a cd instead of **BURNING** it to the CD.  It has to be BURNED not copied.

Burning the ISO to the CD will make the CD a bootable device otherwise it is NOT.
