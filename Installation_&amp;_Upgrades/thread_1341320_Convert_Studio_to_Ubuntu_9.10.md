### Post by lisati on 2009-11-29
Short of a fresh install, replacing your existing installation, there are a number of options. If all you want is the "normal" Ubuntu desktop appearance, the easiest is probably to enter this on the command line: ```
sudo aptitude ubuntu-desktop
```
It will ask for your password, which won't show as you type.
