I recently upgraded to Ubuntu 21.10. The upgrade failed, but with a few simple command line tools I was able to fix it and get working again. 

Posting here in case this is helpful to someone else.

[IMG]http://www.penguintutor.com/images/blogimages/ubuntu-oh-no-fail.png[/IMG]

From there you can issue apt commands to complete the upgrade.
