So I'm selling a computer, but I don't want to just give the buyer a unusable computer. I'm planning on installing Ubuntu 12.04 64-bit on it. My question is- Can I install Ubuntu without having to configure the user settings or root password? I want the seller to be able to configure this the first time that he or she turns it on for the first time. Kind of like how Windows does it, where it lets you type in the name for the user and their password the first time you turn on the computer.

### Post by lisati on 2012-07-03
You might consider doing an OEM installation, which will do the installation, let you apply updates etc, and prompt the user to set up a new user the first time it's rebooted after installation. The last time I did one - a year or two back - I needed to use the alternate install disk.

### Post by DoctorLumpy on 2012-07-03
Thank you!!! I did some quick Google searching and found that you can press F4 at the splash screen and choose to create an OEM installation.
