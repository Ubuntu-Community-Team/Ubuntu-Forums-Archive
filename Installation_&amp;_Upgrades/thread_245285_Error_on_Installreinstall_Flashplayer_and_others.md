William:

My suggestion would be that you **NOT** use Automatix to install these.  I have found both Automatix and Easy Ubuntu not to be very reliable in doing installations.

Instead my suggestion for you to get Flashplayer installed properly is to do this.  This is the procedure that I follow and it works fine.  Go to the anandtech.com website and when you do, the site will tell you that you need flashplayer installed (see the green bar and little green icon - click on them).  It will first attempt to automatically install flashplayer but if yours works like mine, it will not be able to do it automatically and will prompt you to do it manually by going to the flashplayer website.  Once you get to the flashplayer website just follow the instructions that it gives you for downloading and installing the flashplayer.  Basically you just download a zip or tar file and then find your way to where you downoladed the file (**after you have closed your browser**) and just right hand click on it and expand it.  Then go to the terminal and change directory to the one the expand command created and the run the flashplayer installer from within that directory.

Good luck.
