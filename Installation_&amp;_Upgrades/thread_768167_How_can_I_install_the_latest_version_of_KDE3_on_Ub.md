### Post by gtdaqua on 2008-04-26
the latest version in kde3 is kde 3.5.9 which is very stable. KDE 4 is still buggy. 

you can do:

```

sudo apt-get install kubuntu-desktop

```

that will simply install all the necessary libraries and other modules for the latest KDE 3.5.9.

### Post by .nedberg on 2008-04-26
If you just installed Kubuntu 8.04, then you have the latest version. If you are still on 7.10 then you are still on KDE 3.5.8.

Have a look here if you want to install KDE 3.5.9 on Kubuntu 7.10:
[http://kubuntu.org/announcements/kde-359.php](http://kubuntu.org/announcements/kde-359.php)

If you are on Ubuntu then installing kubunt-desktop will give you 3.5.8, and you can update as stated above.
