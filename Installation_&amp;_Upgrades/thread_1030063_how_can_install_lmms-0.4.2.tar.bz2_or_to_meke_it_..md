### Post by MarblePanther on 2009-01-04
Most linux distros include a package manager with repos to download software.  In Ubuntu it is called Synaptic. Open it up and search for lmms and install.  That easy.  Sorry if you already knew that.  Maybe you are just trying to install a bleeding edge version of lmms?  If not then install from synaptic.  Also there is a nice alternative to lmms called Rosegarden.

[http://www.rosegardenmusic.com/](http://www.rosegardenmusic.com/)

### Post by mc4man on 2009-01-04
If your using Intrepid (8.10) you can get Imms in a .deb from this launchpad ppa 

[https://launchpad.net/~tobydox/+archive](https://launchpad.net/~tobydox/+archive)

Just change the drop down for source list entries to Intrepid and copy & paste the 2 lines, 1 at a time into System -> Admin.. -> Software Sources -> Third Party -> add,......,  then reload

You'll find the apps (2) in synaptic
