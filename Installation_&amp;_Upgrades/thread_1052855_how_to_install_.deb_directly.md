### Post by Partyboi2 on 2009-01-28
You can either double click on it to start the install process, or you can open a terminal (Applications>Accessories>Terminal) and change to where the file is eg if on the Desktop
```
cd ~/Desktop
```then you can install by
```
sudo dpkg -i package.deb
``` Change 'package' to actual name of package.
