### Post by YesWeCan on 2011-03-08
Install them separately. Connect on drive and install one OS. Then disconnect it. Connect the other drive etc.

Then connect both. Tell the bios to boot the Ubuntu drive first. Once booted, run 'sudo update-grub' in a console to add Windows to the Grub2 boot-loader menu.
