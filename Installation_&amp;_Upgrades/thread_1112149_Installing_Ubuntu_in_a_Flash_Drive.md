### Post by Operan on 2009-03-31
**Do not need to burn Ubuntu iso file to CD!**

- First, download iso file to your hard drives
- Create a virtual machine to boot this iso file , Ubuntu Live Cd will run 
- Insert your USB flash drive ( minium 1GB ) into your running Ubuntu Live machine
- Click System > Create a USB Startup disk then follows some steps to finish.
