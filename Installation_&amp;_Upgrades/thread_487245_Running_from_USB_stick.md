### Post by milton1 on 2007-06-28
Ubuntu can be installed on a usb stick, assuming it is large enough (at least 5 GB, if not more). However, in order to boot, you must make sure that your computer is set up to boot from usb. If you meet both requirements, you should be able to select the usb drive as the install point during installation.
