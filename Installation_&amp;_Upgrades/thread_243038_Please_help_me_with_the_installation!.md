Do NOT copy the ISO to the hard drive.

Go into BIOS and make sure that CD drive is first boot device.

Put CD with ISO in CD drive and boot machine.

This will boot to the Ubuntu ISO image on the CD.

Make sure the first thing you do is check the intergrity of the CD ISO by choosing the sum check utility function on the initial boot menu.

Good luck.

### Post by orb9220 on 2006-08-24
Yes you have to burn the iso to cd then reboot with cdrom drive as first  boot device in bios and the ubuntu liveCD in the drive.

When you reach the Live desktop try things out to see if they work sound,networking,wireless,etc..

If everything works then you can install to Hardrive by double-clicking install icon on desktop.

You will need at least a 5gb partition for installing.
