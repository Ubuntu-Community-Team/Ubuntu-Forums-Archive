### Post by Haxor on 2007-11-08
Hey ankur_gupta10,

It could be a problem related to the burner, or the brand of the CDs.
With some generic brands or even with branded CDs sometimes the CD-RW needs a firmware upgrade.
Have you tried to use these discs in another computer to see if they behave in the same way?
Try the firmware upgrade first and use CD-RWs to burn the ISO to see if the problem persists, if it does, try burning the image in another computer, you CD-RW might be getting to the limit of its useful lifetime.

Hope that helped.

Haxor
