### Post by zimitt on 2011-10-15
I was able to get mine back. I went into synaptic package manager and did a search for evolution. 
I cant remember which package it was, but one of the packages had a little exclamation point ! in a grayed out box, to the left. (It was the only gray box) 
I highlighted it and applied. 
After that was installed, I was able to install the evolution package, which I wasn't able to previously. 
After it installed, (or maybe it was when I started evolution) a warning popped up to explain the changes in the program, and advancing would transfer saved files to evolution. 
I hope this works for you.
