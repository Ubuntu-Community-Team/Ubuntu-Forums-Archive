### Post by Sef on 2007-12-21
>  I want to upgrade to Gutsy but is it recommended, I am doing fine with Feisty but I am curious.


If you are doing fine with Fiesty, then there is no real need to update to Gutsy.  There is no 100% guarantee that your system will work fine with Gutsy.  If you do not care about keeping upgraded, then I would advise to wait till Hardy Heron is out (April 2008) and update to that.  It is a Long Term Stable (LTS) verion, so the desktop will be supported for 3 years instead of the normal 18 months.  Fiesty's support will run out in October 2008.  Heron's support will run until April 2011.
