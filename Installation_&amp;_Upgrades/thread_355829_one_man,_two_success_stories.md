### Post by drumkitcat on 2007-02-07
Yes, today is a good day my friends!

I was finally able (after at least 4 attempts each) to get Xubuntu 6.1 working on one old pII computer, and ubuntu 6.06 working on an AMD K6-2 450 computer!

Here's how;  in case anyone cares.. but who knows, someone might!

- in both cases I found the Alternate CD version worked best!  by far...

- for the AMD processor it was (and is) using an ASUS motherboard - this had to have the bios flashed to the latest version to be able to correctly see the old Western Digital 80gb drive I had installed ubuntu onto.. (say "ubunto onto" three times fast!)

After I flashed the bios, it was able to boot up ubuntu and hence, recognize the drive, immediately!  

Now in both computers, I need to try and get the wireless  network cards working with my existing home network... 

but, it's still a good day!
