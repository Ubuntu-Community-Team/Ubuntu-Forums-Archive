### Post by salemboot on 2009-11-10
Kooka is a great scanning application.

It's also only supported in version 3.5 of QT.

There are some 'libqt3' entries in 'synaptic'.

Install those and go get a debian universe binary of the application.

You could also build it from source.

[http://kooka.kde.org/](http://kooka.kde.org/)
