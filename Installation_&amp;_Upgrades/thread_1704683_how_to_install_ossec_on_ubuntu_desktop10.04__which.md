I am not sure what the "***osses hids***" are, but I will take a guess:

To update the Ubuntu install in the Virtual Machine with the latest OS updates, there is an Update Manager for that.  You get to it using the menu:  System >> Administration >> Update Manager.

The only issue you might have is: If you are doing this at work and you have a proxy server you must go through to reach the internet, you will have more Network Setup to do.

If you are working at home or from a location that does not use a proxy server, you should be able to use the Update Manager without addition network setup.
