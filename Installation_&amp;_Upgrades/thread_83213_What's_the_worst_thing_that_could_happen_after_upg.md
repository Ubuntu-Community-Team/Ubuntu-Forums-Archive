### Post by Xian on 2005-10-28
[QUOTE=Rhawi Dantas]What should I do to prevent myself of errors, like grub errors because I need my windows working too, or breaking up things in my system that are already configured ?[/QUOTE]
It is unlikely you will encounter grub issues since there are no partition changes when doing a version upgrade with APT. The most likely problems you will find (depending on what you have installed) is that some package dependencies will need to be resolved, and you may notice some user configurations that need to be reset.
