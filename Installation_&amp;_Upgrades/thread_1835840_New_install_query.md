### Post by Iantos on 2011-08-30
hey, if you have windows 7 installed already, you won't have much of a problem...just restart the pc with the ubuntu installation cd, choose to boot from cd...thereafter while the installation runs you will be able to choose on which partition you want to install the ubuntu distro,and you will also be able to choose the file system appropriately..there's not much of a problem if you know how to format hard disk to install an operating system. once it is installed, when you boot the pc, a windows boot loader will ask you to choose which operating system you want to boot to.
