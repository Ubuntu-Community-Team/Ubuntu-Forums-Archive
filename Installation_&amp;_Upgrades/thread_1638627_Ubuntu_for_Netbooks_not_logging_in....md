I installed Ubuntu for Netbooks from Wubi off of an .iso that I downloaded from the ubuntu site, however, how I did it may have cuased some problems. This is what I did...

1. Download the Ubuntu 10.10 for Netbooks image from the Ubuntu site.

2. Using Virtual Clone Drive, mounted the image from inside Windows 7.

3. Installed it into a 30 GB install like normal.

4. Rebooted into Ubuntu and completed the install.

5. Attempted to log on to Ubuntu for the first time and was greeted with the error message, Unity is missing a required driver, please log in to a Desktop session.

6. I logged into the desktop session like it told me to, and installed the updates, and drivers.

The updates just finished, will reboot and see what happens.
