### Post by yogesh.girikumar on 2010-02-04
Hi,


       ubuntu-9.10-desktop-i386.iso.torrent is what you need.


       Ubuntu ISO files are named in this convension:
   ```
OS_NAME-x.yy-distro_type-arch.iso
```  Where      
[LIST=1]
[*]OS_NAME is either Ubuntu / Kubuntu / Edubuntu / Xubuntu etc..
[*]x.yy is the version number x corresponds to the year of release and yy corresponds to the month of release
[*]distro could be desktop / server / alternate
[*]arch could be i386 which is for 32 bit systems and amd64 for 64 bit systems
[/LIST]
        About version number schemes: [https://help.ubuntu.com/6.10/ubuntu/about-ubuntu/C/version-numbers.html](https://help.ubuntu.com/6.10/ubuntu/about-ubuntu/C/version-numbers.html)
