### Post by grahammechanical on 2014-01-20
It seems that there is only a 32 bit version of Ubuntu 12.04 LTS and it is out of stock.

[http://shop.canonical.com/product_info.php?products_id=976](http://shop.canonical.com/product_info.php?products_id=976)

Why only a 32 bit version for purchase? The 32 bit Ubuntu will run on both 32 bit and 64 bit CPUs made by either Intel or AMD. The 64 bit version of Ubuntu will only run on 64 bit CPUs made by either Intel and AMD. If a customer does not know whether their CPU is 32 bit or 64 bit then there is a possibility of them purchasing the wrong disk and not getting a Ubuntu Live disk that works on their system. By only selling 32 bit disks then the customer will always get a live disk of Ubuntu that will work on their machine.
