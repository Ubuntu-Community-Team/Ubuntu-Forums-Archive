### Post by shad0w_walker on 2007-05-24
If you dont have a good reason to use 64bit, i wouldnt bother switching for the time being (Im running 64bit simply because the 32bit kernel wouldnt recognise my 4Gb ram) I will admit there are some small speed boosts, but overall it is not worth it for the extra hassle to get certain things working (Mainly Flash, Java, other closed source things)

As for stability, my system has been rock solid so far. The main draw back is there are a lack of 64bit deb files for various programs so you end up compiling (If its open source and you can) or installing all sorts of 32bit libaries and running the 32bit version (Mostly for closed source stuff like flash)
