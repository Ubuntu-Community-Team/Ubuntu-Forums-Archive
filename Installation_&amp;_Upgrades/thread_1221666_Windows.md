### Post by presence1960 on 2009-07-24
Boot from the Ubuntu Live CD. Choose "try ubuntu without any changes to your computer". When the desktop loads go System > Administration > Partition Editor. Using this software delete all Ubuntu partitions. You can format the resulting space as NTFS or leave it as unallocated space. Quit the Ubuntu live CD and boot your windows install CD.

Sorry about the previous comment. I will try to help you as it is obvious that you gave Ubuntu a try, you are not a windows only user in the wrong forum.

Looking at your previous posts maybe you should consider a dual boot between Ubuntu & windows. That way you can run those programs that are windows programs in windows and have Ubuntu for the rest. This has another decided advantage. because you would have windows to fall back on if you get stuck, you can learn ubuntu at your own pace, there would be no rush. if you can't figure something out in ubuntu you can boot into windows and complete your task. Then you can come back here for help with Ubuntu. Just a suggestion, but if you choose to be windows only good luck. Don't let anyone make you feel bad about your choice of OS. There is no shame in using Windows. I say this even though I strongly dislike windows. I respect your freedom and right to choose.
