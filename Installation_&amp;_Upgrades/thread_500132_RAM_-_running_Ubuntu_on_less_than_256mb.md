### Post by az on 2007-07-13
The live cd won't boot with 128 megs of ram.  The installer will not work with less than 256 megs of ram.

If you install Ubuntu and then remove all but 128 megs of ram, your system will crawl - it will be pretty much unusable.  Once you pass the 256 megs of ram nark, Ubuntu's performace surpasses XPs with equivalent ram.
