### Post by Jussi Kukkonen on 2006-10-19
The server install does not have a GUI by default. You can either re-install from the other CD, or install a desktop environment on the server... The latter requires downloading stuff from the internet: install ubuntu-desktop, kubuntu-desktop or xubuntu-desktop package, depending on which DE you want (Gnome, KDE and XFCE respectively).
