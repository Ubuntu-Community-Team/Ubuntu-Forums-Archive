### Post by Rat2000 on 2013-04-26
I've upgraded to 13.04. Everything seem to have gone well but now when I boot, the Grub menu is show properly (I dualboot with Windows 8), Ubuntu seems to start and for a moment I get the TTY1 login prompt.
It goes away quickly and I just get a blinking "-", hard disk activity dies down and I'm stuck there. I can get into recovery mode from Grub. I've tried the few recovery options. Updating the packages works but flashes a message about an error in a fstab line.

Any help would be welcomed.

Edit: Ok I found out how to make my system work. In the root terminal from the recovery menu I did an "apt-get upgrade". All packages where up to date but it said it had skipped updating "nvidia-experimental-310".
So I run "apt-get remove nvidia-experimental-310". Reboot and here I am with a working 13.04.
