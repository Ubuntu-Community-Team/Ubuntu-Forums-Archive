My disk is partitioned thusly:

Part #1  NTFS (Windows)
Part #2  EXT3 (linux)

### Post by max.diems on 2006-12-09
Edit menu.lst-- add
```
title           Windows XP
rootnoverify           (hd1,0)
map (hd1) (hd0)
map (hd0) (hd1)
chainloader +1
makeactive
boot

```

### Post by confused57 on 2006-12-09
You need to open a terminal(Applications--Accessories---Terminal), enter(copy&paste):
```
sudo cp /boot/grub/menu.lst /boot/grub/menu.lst_backup
```
this creates a backup, if you need to restore...

```
gksudo gedit /boot/grub/menu.lst
```
ignore any error messages

then add the following to the very end of the file:
```
title        Microsoft Windows XP Home Edition
root        (hd0,0)
savedefault
makeactive
chainloader    +1
```
exit & save

Here's an excellent site, lots of info about bootloaders:
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)
