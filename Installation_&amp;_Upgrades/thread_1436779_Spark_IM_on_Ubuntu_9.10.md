### Post by tom4everitt on 2010-03-23
When you click on linux, you see four different linux versions right? 

Choose the one ending with .deb (.deb is like a .exe for windows). Download it. When its done you double click on it and click install package. It should now appear under Applications->Internet.

### Post by tom4everitt on 2010-03-23
Sorry, that was only for openfire. 

For Spark there is only a tar.gz available for linux. Download that. Extract it. Inside it you will find a file named Spark. Double click that and choose run. (No installation is required.)

Put the extracted folder somewhere. You can then add an entry in applications->internet by using the Main Menu in System->Preferences (use the full path of the Spark file as command).
