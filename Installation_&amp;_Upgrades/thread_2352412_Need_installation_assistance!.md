### Post by TheFu on 2017-02-12
No. Requests like this just aren't possible.
Google found this explanation: [https://askubuntu.com/questions/6263/how-can-i-install-ubuntu-on-an-ipad](https://askubuntu.com/questions/6263/how-can-i-install-ubuntu-on-an-ipad)

However, you can buy a VPS for $6/month and install an ssh client on the ipad, then access it remotely using the network.  Some people do this for their day to day work.  [http://yieldthought.com/post/31857050698/ipad-linode-1-year-later](http://yieldthought.com/post/31857050698/ipad-linode-1-year-later)  Also - so you know, Ubuntu Server doesn't have any GUI, so ssh is really the only way in remotely. If you load "server" and expect a GUI, forget it.

I use a remote desktop from a chromebook into my primary desktop running Ubuntu, but I use a more efficient tool that doesn't work on iOS (or Android).
