### Post by ubukool on 2007-10-22
Hi!

I think if you do a clean install of Gutsy, then all will be well.  Maybe you will have a few teething problems, but maybe not.  I'm currently using Feisty and I upgraded to Gutsy twice, once to the alpha release and once to the beta release, but I reverted to my Feisty install again both times because there seemed to be quite a few problems.  This is quite a common experience if upgrading from one release to another, although I experienced no problems upgrading from Edgy to Feisty.  

I'm pretty sure the best thing to do is to start with a clean install, but it's a lot of work to do that for me because my system is so different from the basic install, it would take a lot of work to get it back to that standard on a Gutsy baseline.  If you're starting from scratch, go for Gutsy because you're probably going to want to upgrade at some point in the future anyway, and it's better if you start with the latest release.

Just some thoughts - good luck! :)

### Post by GrammatonCleric on 2007-10-22
If you are comfortable working through minor issues as they come along  then upgrading is should be fine.  I upgraded my laptop  to one of the early releases and most things went without issue.    The only real problems I had upgrading from Fiesty to Gutsy was with video drivers and vpnc.   I had the official NVIDIA drivers installed I would recommend removing them, if you have them installed, before you upgrade.
