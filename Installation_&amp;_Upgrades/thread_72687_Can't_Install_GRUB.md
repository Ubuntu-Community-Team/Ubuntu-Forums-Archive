### Post by SilentCacophony on 2005-10-07
Hi. This is kind of a stab in the dark, but I believe some computers have a bios setting to protect the Master Boot Record (MBR) of the HardDisk. I believe that is a possible culprit.

If you can get to the bios settings (usually holding del or F1 while booting,) check for an option like that, and disable it if it's set.

Otherwise, if you could provide any more detail as to the error message, it would help.
