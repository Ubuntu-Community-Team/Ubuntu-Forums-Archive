### Post by Sef on 2010-06-10
> My friend put a windows 7 .iso file on the netbook.... 

A windows 7 iso violates Microsoft's policies on making more than one copy, so thread is locked.

From [Microsoft Store]("http://store.microsoft.com/Help/ISO-Tool"):

> Copies

To install the software, you may make one copy of the ISO file on a disc, USB flash drive, or other media.
After you&#8217;ve installed the software and accepted the license terms that accompany the software, those license terms apply to your use of the software. The license terms for Windows 7 permit you to make one copy of the software as a back-up copy for re-installation on the licensed computer. If you do not delete your copy of the ISO file after installing the Windows 7 software, the copy of the ISO file counts as your one back-up copy.
