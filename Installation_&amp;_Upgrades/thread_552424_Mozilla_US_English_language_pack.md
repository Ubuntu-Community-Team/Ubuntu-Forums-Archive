I was wondering the same thing, then found this post in the Feisty bug reports that has some useful information:

[https://bugs.launchpad.net/ubuntu/+source/firefox/+bug/66015](https://bugs.launchpad.net/ubuntu/+source/firefox/+bug/66015)

Basically, you need to right-click in an editable text area, then the drop menu will have an entry called "Languages" on which you should be able to select en_US or English / United States.  As far as I can find, there is no way to change this through Edit -> Preferences although I bet you can change it through about:config
