### Post by Syirrus on 2006-06-02
If X-server crashes when you boot of the 6.06 live CD and you happen to have a widescreen monitor,  then this may help you get X started.  Apparently VESA will only support max resoultion of 1400x1050.  As soon as I adjusted my resolution to that, Ubuntu Dapper 6.06 booted.

My hardware
Pentium D 3.6 
2GB DDR2 RAM
ATi X1900XTX
Apple 20" Cinematic Display LCD 16:10 1680x1050

I hope this help at least one lone ranger

Syirrus
