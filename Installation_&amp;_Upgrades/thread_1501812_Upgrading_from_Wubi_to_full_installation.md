### Post by wilee-nilee on 2010-06-04
You might consider dual booting the windows and Ubuntu setup, basically if you have a OEM it will be hard to replace and you might use it again in the future.

You might also confirm that you have nothing to save and  or your all backed up so that a fresh install of a booted live cd is all that is needed for a full install on the hard drive.

If you have nothing to save and you don't want windows at all boot the live cd and just do a fresh guided install on the whole hard drive. Some like to have a static home partition so that another fresh install or upgrade can be done without changing the separate home partition, so others can provide more help in that.
