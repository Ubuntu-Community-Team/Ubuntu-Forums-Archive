### Post by Dark Hornet on 2007-12-06
I have an AMD FX-57, and a BFG 8800 GTX..the trick with the Live CD is this:  When you boot up the computer, and it comes to the "install Ubuntu" menu, hit F6 for options, and delete word "open", and make the word "splash--" into "nosplash--".  This worked for me to get the live CD up and running, I hope it helps.  Good Luck!
