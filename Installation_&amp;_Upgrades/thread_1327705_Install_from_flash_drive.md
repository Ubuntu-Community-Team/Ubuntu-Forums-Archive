### Post by mac9416 on 2009-11-15
Hey there!

Yes, it is very possible. You need to use [USB-creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") or [Unetbootin]("http://unetbootin.sourceforge.net/") to create a USB live install. Using those tools, you can also set your live install to be persistant, so you can keep your apps and configuration between powerdowns.

Hope that helps!
