### Post by engla on 2006-02-05
[QUOTE=Alighera]ive been thinking of finally going through with installing ubuntu (i have three different versions sitting around) but im still kind of doubtful... if anyone could just clear some things up for me than thatd be great.


Q1: Does Ubuntu come preloaded with Media apps like say Winamp or some sort of media player?
[/QUOTE]
Ubuntu comes with a music player called Rhythmbox and a video player called Totem. There are lots of others that are available for installation, for example XMMS if you like something that mimics winamp. 

However, many common filetypes like mp3 and WMP don't work out-of-the box. Since most people want to use these kinds of files, there are guides on how to enable this.

[QUOTE=Alighera]
Q2: Can i run my favorite games like Diablo, Civ 3, and Half Life through Ubuntu?[/QUOTE]
This is something you shouldn't count on. Ubuntu is not windows, and not meant to be. That said, it seems that some games are easily made to run in Ubuntu [but with special software installed for this purpose]


[QUOTE=Alighera]
Q3: Can Ubuntu pick up my plug and play devices like sound cards, etc.?

Q3.5: If Ubuntu can't pick up my sound card is it possible to get Linux drivers?

and finally Q4: Can Ubuntu read .avi format media and devices like my Zip and Floppy Drives?

im thinking of installing Ubuntu onto a Windows 98 machine i have around and those are all the things i used it for previously so if i can continue using it for the same reasons but a little more optimized that thatd be great. Thanks[/QUOTE]
The questions about hardware are hard for us to answer specifically, but generally stuff like sound cards and zip drives often work. There is a way for you to find out, without installing Ubuntu, and without modifying your computer configuration at all: a Live CD!

Ubuntu is available in a Live CD version, this is a CD that you can download and burn to a disc, then boot from that. It will run Ubuntu from the CD without installing anything, and that lets you check if sound, video, zip drives etc etc can works without effort or not.
