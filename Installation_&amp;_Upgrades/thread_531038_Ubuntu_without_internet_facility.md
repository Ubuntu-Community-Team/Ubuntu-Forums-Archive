### Post by jnorthr on 2007-08-21
I have the same issues with my machine. It is not connected to the internet. I do have access to a windows xp machine. i can download packages that are .deb versions rather than .rpm versions. I download them to my usb external hard disk then take the disk to my ubuntu machine, insert it into the usb plug and ubuntu 'sees' the disk. Then i can double click on the .deb file and it gets installed. Sometimes there are problems because a package may depend on another package and you then need to download that package before you can install the first .deb file. It can be very frustrating to know all the dependencies before you try to load a package.

Go here: [http://packages.ubuntu.com/](http://packages.ubuntu.com/)  for packages. choose your version of ubuntu and drill down to that page of available packages.
