### Post by McLogic on 2006-02-22
It will take some work to get the system running if you want to run Linux on a Turkish laptop -- that is just the facts. It may take you a full day, or a full weekend, but I bet you can get it up and running if you have a little bit of persistence.

When I started with Ubuntu, I had to get 855resolution to change my screen resolution. I had to get the suspend working, by fooling with a couple of shell scripts and doing a little kernel something. You will then have to go to the keyboard shortcuts to get some of your function keys running. Some of your function keys will not work. It took me a full Saturday (about 9 hours) to get the system running.

Much of it was a process where I tried to figure out how people have described the problem that I was having in order to find the right keywords. Then use that set of keywords when searching the Ubuntu-specific sites first, and then the rest of the Linux web sites if I didn't find the answers. 

Then you just need to back up your files and try what has worked for other people. If it works, then you move to the next bit of trouble. If it dosen't work, then you look for a new set of keywords or add the version of Ubuntu (+"5.10") to you search keywords in order to get rid of answers that only worked on old versions.

Just don't give up, it will work!
