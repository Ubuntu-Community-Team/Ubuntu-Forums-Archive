### Post by efflandt on 2010-12-09
You have to burn the iso to a CD as an image, not as a data file.  Some WinXP computers came with a version of Roxio or Sonic that could burn images, and some didn't.  If you have WinXP or Vista, there is a free download [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm)

It sounds like Win7 may already have that capability natively [http://www.win7articles.com/burn-iso-images-in-windows-7.html](http://www.win7articles.com/burn-iso-images-in-windows-7.html)
