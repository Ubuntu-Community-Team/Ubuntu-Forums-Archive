### Post by muskratmx on 2008-03-04
> Wheres a good place to learn everything i need to know to keep ubuntu up and running..

Like learn stuff about the grub and terminal because i dont really know what those things are

That's a big order "everything i need to know", One thing to remember, Ubuntu is a distro of linux, based on another distro called debian. A few things you learn will be Ubuntu particuler, but most things are Debian or Linux in general.

Grub, it's not Linux, it's a bootloader, and in reality is an OS unto it's self.

The Terminal is universal for all OSs. DOS, Windows, BSD, Linux and a whole lot more. Linux and Unix commands are pretty much the same. So don't limit your search for knowledge just to Ubuntu arena.

As for Grub invest in yourself and read this manual, it's not that long and is a great read. [http://www.gnu.org/software/grub/manual/grub.html](http://www.gnu.org/software/grub/manual/grub.html) It will save you a lot of greif later.
