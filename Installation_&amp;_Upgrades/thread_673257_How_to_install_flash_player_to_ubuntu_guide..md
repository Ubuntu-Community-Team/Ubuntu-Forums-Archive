How to install flash player to ubuntu.

1.) Download flash player from the adobe website. Make sure it is the tar.gz format.Save the file to your desktop.

Heres the link 

[http://fpdownload.macromedia.com/get/flashplayer/current/install_flash_player_9_linux.tar.gz](http://fpdownload.macromedia.com/get/flashplayer/current/install_flash_player_9_linux.tar.gz)

2.) Right click the package named: install_flash_player_9_linux.tar.gz on desktop. Click on extract here. A folder should appear called just "install_flash_player_9_linux"

3.) Open this folder. Click on the file inside called flash player installer. terminal should open follow the steps shown and there you have it.

Hope this helps its not much but it's a start (im new)
