### Post by aysiu on 2006-12-19
You can download it here:
[http://us.releases.ubuntu.com/edgy/ubuntu-6.10-alternate-i386.iso.torrent](http://us.releases.ubuntu.com/edgy/ubuntu-6.10-alternate-i386.iso.torrent)

BitTorrent is probably the best way to download if you're sharing bandwidth with other downloads.

Unless you know exactly what files to copy over, having the LAN connection between computers won't help. If you want an **exact** copy of the Edgy computer, you can tar that up and then restore it on the Dapper computer, using [this tutorial](http://ubuntuforums.org/showthread.php?t=81311&highlight=howto+back+up+restore).
