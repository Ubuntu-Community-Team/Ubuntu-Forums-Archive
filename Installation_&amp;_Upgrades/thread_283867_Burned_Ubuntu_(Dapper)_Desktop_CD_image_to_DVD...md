### Post by louieb on 2006-10-24
After your mac boots up browse the DVD. 
If all you see is one file named ubuntu-something.iso then all you did was copy the image file to the DVD, this is not what you want. 

What you want to see when you browse the DVD is a bunch of folders and few other files. If you see that then you burned the DVD as it was intended. Just do the same thing using a CD-R disk.

I don't do MAC.  But I just googled "burn iso on mac" and it looks like you will find the answer if you do the same.
