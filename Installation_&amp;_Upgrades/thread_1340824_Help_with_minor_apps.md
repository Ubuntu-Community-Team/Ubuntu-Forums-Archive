Not sure why you have "lost" these, but try going to; System > Preferences > Startup Applications

You should be able to enable them if they are listed.

If they are not listed you may need to add them. For example, the settings for the Network Manager are:-
- Name: Network Manager
- Command: nm-applet --sm-disable
- Comment: Control your network connections

If this works for Network Manager, I'll give you the others that you are missing.
