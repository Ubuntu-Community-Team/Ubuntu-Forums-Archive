### Post by notebook_ftw on 2006-11-15
Just pop the Ubuntu CD in and boot from it.  Ubuntu contains a built-in partitioner which will partition the hard drive for the installation, and it will automatically install grub allowing them to boot into either Windows or Ubuntu at startup.

### Post by Infamous Flame on 2006-11-15
You can resize the partition without losing any data (as long as you don't shrink it too far, i'm not sure whether ubuntu will let you shrink it too much or not so be careful). To be on the safe side, you might want to boot into the expert installation instead of the default one. (Insert install CD, when the ubuntu logo and information for booting comes up, type expert and press return.
