### Post by grahammechanical on 2015-11-15
It is recommended that we install Ubuntu with an ethernet or wired connection to a router. Then the installation can be updated during the install process. The same installation process may detect the need for a WiFi driver which it can then download and install. This would making an after installation setup of WiFi much easier. So, having an ethernet connection during installation is considered as standard for Ubuntu.
