### Post by ajgreeny on 2017-01-23
It is most likely that your hardware is simply not up to running Ubuntu with the unity desktop as it has too low specs for that version which needs good 3D graphics for a start.

You could probably do a lot better with another DE version such as Lubuntu or Xubuntu, both of which run well on more lowly hardware.
Give them a try and come back again if there is still no luck.
