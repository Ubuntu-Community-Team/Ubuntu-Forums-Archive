### Post by jerome1232 on 2008-07-10
a:
download an 8.04.1 Ubuntu server edition cd, install the system, once you are into your base CLI only system, type```
sudo apt-get update && sudo apt-get install ubuntu-desktop
```
subsitute kubuntu-desktop if you want kde
and xubuntu-desktop if you want xfce

b:
just download the 8.04 Ubuntu Desktop edition cd, it's not much different really
