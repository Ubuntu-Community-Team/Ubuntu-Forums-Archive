### Post by onlyonce on 2008-05-07
Hardware configuration:
IBM Netfinity xSeries 220
* Vídeo S3 Savage4 (8MB)
* Monitor LG FLATRON L1530S

I will make a brief and simple summary

1. Install Ubuntu Server 8.04
2. After install and reboot run the commands bellow as root:
   > sudo apt-get update
   > sudo apt-get install ubuntu-desktop
3. Restart
4. In the command line run this bellow
   > sudo dpkg-reconfigure -phigh xserver-xorg
5. Edit the file /etc/X11/xorg.conf as bellow:
