### Post by Partyboi2 on 2009-01-19
If you go through gigs like they are going out of fashion you probably better off buy a 1 tb hard drive and use that. If you are wanting a cheaper option then you can hook up your 2nd hard drive (60gig) and use that for Ubuntu while keeping the other hard drive for windows.
Ubuntu can read/write to ntfs as well so you can always make another partition on your 250 gig for storage between the two.
