### Post by bulldog on 2006-11-09
If you have enough space on your hard disk you should first defragment Windows several times.
Then boot from the live cd and hit the install button.
Answer the questions till you get the partitioner.

Now you have three options,but let Ubuntu use the whole hard disk isn't a good idea,you should lose your windows.:D 

If you feel confident you can choose manual partitioning.

But I recommend to let Ubuntu take care of it.
