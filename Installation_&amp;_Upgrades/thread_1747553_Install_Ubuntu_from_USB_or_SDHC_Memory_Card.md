Hello! My laptop can boot from usb or SDHC memory card,
 and I have a 8GB SDHC card in my digital photo camera,
 Is there a way to boot Ubuntu from the SDHC memory card and  install it on hardisk (in a ext3/4 partition)? And so sparing a DVD/CD?:popcorn:

---

### Post by stargazer418 on 2011-05-02
You can try using UNetbootIn ([here]("http://unetbootin.sourceforge.net/")) to burn the ISO to the SD card. When it's done, remember to make sure the SD card is set as the highest boot priority and it should boot.
