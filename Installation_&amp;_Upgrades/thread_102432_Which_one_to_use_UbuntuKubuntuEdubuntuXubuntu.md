### Post by nrwilk on 2005-12-12
No one can tell you which to use because it's more a matter of preferrence than anything else.

Have you used Linux before?  If not, you probably want ubuntu (which uses gnome as it's window manager).  It's lighter and a small bit easier to pick up.

If you've used linux, did you like Gnome or KDE more?  If you liked KDE, than you want Kubuntu, which uses KDE as it's window manager.

If you want a lighter window manager than you might want Xubuntu, which uses XFCE.

EDUbuntu is geared towards schools and learning environments.

Whatever way you go, you can still install other window managers without having to reinstall the whole system.  Don't worry about it too much.
