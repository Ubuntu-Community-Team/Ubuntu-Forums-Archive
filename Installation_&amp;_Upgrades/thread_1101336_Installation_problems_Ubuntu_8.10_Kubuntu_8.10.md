### Post by taurus on 2009-03-20
1.  Did you run the checksum to check the integrity of the ISO image before you burnt it?

2.  Did you burn it at a slow speed like 4x?

3.  Did you run check cd for defects at the first screen after you boot from the CD?

p.s.  I wouldn't try to boot Ubuntu LiveCD (desktop CD) with only 256MB of RAM.  If you really want to use a window manager, try the Xubuntu alternate CD.

[http://www.xubuntu.org](http://www.xubuntu.org)[/code]
