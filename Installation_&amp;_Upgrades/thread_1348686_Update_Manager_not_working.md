### Post by phillw on 2009-12-07
Hi,

Go to System --> Administration --> Update Manager. Select settings, And make sure that there's a tick in the Important Security Updates, Recommended Updates and Under Automatic Updates - this should be set for "Daily".

That should get it to fetch your updates for you.

Regards,

Phill.
