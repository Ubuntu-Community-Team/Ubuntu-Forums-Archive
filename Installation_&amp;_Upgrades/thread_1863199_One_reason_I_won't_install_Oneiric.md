Unity is brand new, it is silly to expect everything to be in the same place and work the same way. Try it for a couple of weeks and, y'know, **read the instructions** before you pass judgment!

There are tons of "how to use Unity" tutorials on the web such as: [http://www.ubuntugeek.com/list-of-ubuntu-unity-keyboard-shortcuts.html](http://www.ubuntugeek.com/list-of-ubuntu-unity-keyboard-shortcuts.html)
