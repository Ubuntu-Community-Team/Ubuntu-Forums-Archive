### Post by mrrogers on 2008-03-08
I'm sure this question has been asked before so I apologize in advance if anyone has heard this.  I just put Ubuntu on my computer and so far everything is going great.  I really like it a lot, but I want to keep Vista as a backup.  I am wondering how I can shrink the Vista partition so that Vista only has space for itself and any programs I install and the rest is turned into a shared partition between the two drives for music, etc.  First, I have read in forums that Vista will need 40 GB of space to run so I am thinking that is what I'll shrink it to (if possible).  Secondly, what file system should I use for the shared space? I have heard of people using ext3 and a program so that Vista can read it.  I've also heard of people using NTFS since that supposedly is readable by default in Ubuntu.  This is just a little confusing for me and any help would be greatly appreciated.  Thanks!

---

### Post by Pumalite on 2008-03-08
Use Vista Partitioner to shrink Vista after you have backed up all important files. The new space can be formated anything you want because you can read and write to ntfs from Ubuntu. And you can see ext3 from Windows with a special driver.
