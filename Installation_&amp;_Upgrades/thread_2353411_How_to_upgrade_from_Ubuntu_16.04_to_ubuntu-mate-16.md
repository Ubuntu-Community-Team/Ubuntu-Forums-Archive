### Post by howefield on 2017-02-21
Ubuntu 16.04 with Unity and you want to upgrade to Mate 16.10 - is that correct ?

If so, I'd suggest doing a clean install with the Mate Live media and mounting your existing / partition but not marking it for formatting, this will keep your /home folder intact so you shouldn't lose any data. I'm assuming that you don't currently have a separate /home partition, would that be correct ?

It should go without saying that one must back up all important data before performing such a potentially risky manouvre such as upgrading your OS.
