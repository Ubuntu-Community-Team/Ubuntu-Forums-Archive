### Post by cgroza on 2010-12-15
I am assuming that you click the Install icon from the desktop after loading the Live environment.
Try the other way:
When the LiveCD loads, a small rectangle will appear at the bottom. When you see it press the spacebar.
A menu will appear with something like: Try Ubuntu, Install Ubuntu.
Select the install Ubuntu option and go through the installation procedure if it works.

If the above does not work, you may want to try the Alternate CD.
