### Post by Seisen on 2007-10-28
The only difference between the live cd and the alternate cd is the alternate cd has a text-based installer. So when you upgrade to Gutsy it will pull all of the upgrades for the packages you have and also some new ones. If you worried about not having enough ram you can always try Xubuntu which you can install using apt-get and remove Ubuntu.
