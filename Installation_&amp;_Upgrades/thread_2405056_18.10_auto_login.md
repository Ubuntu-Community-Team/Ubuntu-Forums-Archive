### Post by dino99 on 2018-10-31
What are the steps you have followed to set autologin ?

From System Settings ->  Details -> Users : there you need first to unlock that screen (click on top right 'unlock'), then you might be able to 'enable' auto login.
On next reboot, if autologin still does work, then glance at 'journalctl -b' to find a related error logged  and report on launchpad against gdm3.
