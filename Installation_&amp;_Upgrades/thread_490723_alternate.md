### Post by Herman on 2007-07-02
Yes, the Ubuntu 'Alternate' CD will install the same operating system as the 'Desktop' Live/Install CD will, except with the 'Alternate' CD you will have more options available if you want or need a special type of installation.
The 'Alternate' CD is based on the old text based debian installer and it was the only way to install Ubuntu when the first releases of Ubuntu came out.

For anyone who has used Linux already, it's quite simple to use the 'Alternate CD', but new users fresh from Windows found it 'confusing' because it offers a lot of options that they couldn't understand, and lacked a familiar GUI interface that they were accustomed to. 
There were also a lot of new users who wanted to try Ubuntu out first before they installed so they can see if it will work in their hardware okay or not, or even if they like Ubuntu before they decide to install. So the 'Live CD' was made.
Later the Live CD was modified and had its own partitioner and installer added it and so it was renamed the 'Desktop' CD.

The 'Alternate' CD is still popular with Xubuntu users if they are choosing Xubuntu because of a lack of RAM to run the bigger desktops.
The 'Alternate' CD does not require as much RAM and loads faster, but if there is a swap area the 'Desktop CD' can install faster on moderate RAM machines once it's booted up. If you have a really modest RAM machine though, only the 'Alternate' CD will probably work at all. Probably Xubuntu users will find that the 'Alternate' CD will be the only one that will work.

The 'Alternate' CD is still quite popular with a lot of the more experienced users who know what they are doing. Some people still use it just because they learned with it and they just like it. 

Click on my sig links to read all about the 'Alternate' CD and see illustrations to guide you through your installations. 
Regards, Herman :D

### Post by Herman on 2007-07-02
Yes, the 'Desktop' or Live CD is good as long as you have say 512 mb or so of memory. There are lots of time when the Live CD comes in handy to perform operations on the hard disk installed system with too, from time to time.

If you have less than 128 MB of RAM, see this web page, you might have to install this way: [URL="https://help.ubuntu.com/community/InstallingXubuntu"]https://help.ubuntu.com/community/InstallingXubuntu
[/URL]
If you have around 128 mb of RAM or a little more, you should probably use the 'Alternate' CD for installing Xubuntu.

If you have around 256 MB of RAM, you can use either the 'Alternate' or the 'Desktop' CD for installing with.
In some computers the 'Desktop' CD's installer will freeze or take a very long time to do anything unless you have a pre-made linux swap area on your hard disk. You can make a swap area first with a [GParted -- LiveCD]("http://gparted.sourceforge.net/livecd.php"). The 'Desktop' installer will automatically find that swap area and use it, and the 'Desktop' CD will actually perform the install faster than the 'Alternate' CD, (not counting boot-up time).

If you have more than 256 MB, depending on other details of your hardware, your computer should run the 'Desktop' CD alright, but a pre-made swap area might still help.

If you have around 512 mb RAM or more, you shouldn't have any problems. The 'Desktop' CD will run well and perform quite fast. Most computers these days probably have around 512 mb of memory or more. The 'Desktop' CD would be the best choice for most people.

If you have 1 GB of RAM any LiveCD will probably run as fast as a hard disk installed OS.
