### Post by Mark Phelps on 2011-09-28
Use the Debian package search link below:

[http://www.debian.org/distrib/packages]("http://www.debian.org/distrib/packages")

Enter wine in the keyword box.

Note, however, that installing using individual deb files is really HARD because you will have to solve dependencies yourself.  Which means, you may have to end up downloading dozens of deb files before you are done.
