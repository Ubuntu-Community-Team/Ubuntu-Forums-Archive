### Post by Mark Phelps on 2010-09-29
There's over 20 THOUSAND packages, and that's with only a typical set of Repos installed.  I'm not aware of any bulk list of packages with descriptions. 

Also, my advice is to leave the packages alone.  It's not unusual for apps to share packages, especially libraries.  If you start removing stuff, you run the risk of rendering one or more apps unusable.
