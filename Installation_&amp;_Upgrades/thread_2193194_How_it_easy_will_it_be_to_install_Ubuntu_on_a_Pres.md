### Post by kurt18947 on 2013-12-11
If you're fairly comfortable with working in your machine, you might be able to add RAM to make your machine functional with the 'lighter' distros.  The other question I'd have is the chipset.  According to a data sheet found via Google ([http://h10025.www1.hp.com/ewfrf/wc/document?cc=us&lc=en&docname=c00303953](http://h10025.www1.hp.com/ewfrf/wc/document?cc=us&lc=en&docname=c00303953)), that machine has an SiS  chipset like you said.  I don't have an experience with them but do recall reading about some problems with SiS video chips.  SiS did make motherboard chipsets as well as video chip sets.  It appears to use a common memory stick - PC3200 184 pin - that you can find used on Ebay for not much money.  Depending on how much time and money you wish to devote, there appear to be 1 AGP and two PCI slots open from the factory.  You might be able to plug a video card into one of those slots assuming they're unused.  It looks like you could bring the RAM up to 2 GB. which is plenty IME and get an AGP video card - about $25-$30 for both items.  I have bought from Xtreme RAM and no problems

[http://www.ebay.com/itm/NEW-2GB-DDR-PC3200-2X-1GB-PC-3200-400MHz-High-Density-DESKTOP-RAM-for-VIA-SIS-/331050488061?pt=US_Memory_RAM_&hash=item4d142590fd](http://www.ebay.com/itm/NEW-2GB-DDR-PC3200-2X-1GB-PC-3200-400MHz-High-Density-DESKTOP-RAM-for-VIA-SIS-/331050488061?pt=US_Memory_RAM_&hash=item4d142590fd)

I don't really have a recommendation for video card.  Generally Nvidia is a good choice but I don't know which older Nvidia open source drivers work well or which of the older Nvidia chipsets are supported by available Nvidia proprietary drivers.
