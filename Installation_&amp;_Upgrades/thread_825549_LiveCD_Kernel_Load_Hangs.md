### Post by mxg01 on 2008-06-11
When you down load the ISO the first thing you should do is check the MD5 sum to make sure the ISO has not been corrupted during download.

Here's how to do it:

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If it's ok then you should burn it to CD but it's best to burn it at a slower speed. x4 or less.

Try that and see if you get a better CD copy.
