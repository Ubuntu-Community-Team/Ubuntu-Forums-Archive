### Post by DJiNN on 2008-01-27
Although Xubuntu is great, and you'll enjoy playing around with it, it's still quite a "Loaded" distro. If you really want to make your machine fly, even with a low spec such as PIII with 128mb ram, then i'd agree with the above poster on Tinyflux (PCLOS based) or better still, [TinyMe]("http://www.mypclinuxos.com/doku.php/tinyme:home"), which is still PCLOS based but even faster (IMHO). I have it on several PIII's here & it's REALLY fast. Knocks Xubuntu into the corner, and i don't say that lightly because Xubuntu is great (I'm using the 64 bit version right now, on the machine that i'm typing this into. )

It's  fast, friendly, has an iconified desktop and also uses the PCLOS "Control Centre" which really helps when it comes to setting up loads of stuff that you may need (Networking, Screen, Audio etc etc)

Whatever you choose,  have fun! :)
