### Post by The Spy on 2008-11-02
This thread is meant to help out those who have had or are having troubles with their upgrades (from Hardy to Intrepid for instance).

I had problems with Network Manager after I upgraded, because I had the Hardy version of Network Manager 7 installed. So I had to manually install those packages, which I found here, [http://packages.ubuntu.com]("http://packages.ubuntu.com/") under the Intrepid packages list, in the Network subdirectory, along with it's dependencies, etc.

I recommend this site for if you are missing needed packages, and have another computer connected to the Internet.

Please post any problems you have had with upgrading to Intrepid Ibex and how you solved that problem.
