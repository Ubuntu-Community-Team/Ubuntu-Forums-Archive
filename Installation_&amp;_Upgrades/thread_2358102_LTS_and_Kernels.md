Hi.

Ubuntu 16.04 shipped with kernel 4.4 LTS which (I think) will be supported for the lifetime of the distro.
But if you install the current 16.04.2 it will install kernel 4.8 which is not LTS.

So what happens when a kernel reaches EOL on a LTS Ubuntu?

---

### Post by ajgreeny on 2017-04-09
Having installed the first of the many hardware enablement stacks, either manually, or by starting with the 16.04.2 point release, tyhe next HWE will automatically be uodated when it appears in the repos, so there is no need to worry about the intermediate kernel versions losing support as they will have been replaced.
See [https://wiki.ubuntu.com/Kernel/LTSEnablementStack](https://wiki.ubuntu.com/Kernel/LTSEnablementStack) which I think explains all, and also points you to [https://wiki.ubuntu.com/Kernel/RollingLTSEnablementStack](https://wiki.ubuntu.com/Kernel/RollingLTSEnablementStack) which explains the details of the auto-updating.
