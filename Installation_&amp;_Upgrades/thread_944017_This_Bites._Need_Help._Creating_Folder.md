### Post by iaculallad on 2008-10-10
To disable your home folder from displaying in your Desktop, Press Alt+F2 and input '**gconf-editor**' (w/o quote). Navigate to **apps->nautilus->preferences**, on the right-side window pane, look for **desktop_is_home_dir** and uncheck it. Close the configuration window editor.
