### Post by grahammechanical on 2013-06-27
Something needs confirmation that you accept the licencing terms. You need to tick a confirmation box. I get this when I install Ubuntu restricted extras. We need to accept Mircosofts license agreement to download and install Microsoft TrueType core fonts. Unfortunately the confirmation box is often hidden by the Software Centre page. Minimze the Window and you may find another dialog box underneath. Or look for a box to tick on the dialog thatb you are seeing. You may need to scroll down the page as if you are reading it.
