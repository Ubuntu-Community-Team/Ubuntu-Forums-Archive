### Post by Quackers on 2010-12-13
Go to System menu > Administration menu > Synaptic Package Manager
When synaptic opens enter in the search box (top right) ubuntu-restricted-extras and it will appear in the main panel. 
Right-click it and select Mark for installation.
When it asks for confirmation click on Mark.
Then click on the green arrow in the tool bar.
When it asks for confirmation click on Apply
