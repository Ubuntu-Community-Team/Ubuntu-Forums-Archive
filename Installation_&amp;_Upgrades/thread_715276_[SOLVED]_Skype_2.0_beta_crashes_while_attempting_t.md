### Post by diablo75 on 2008-03-04
I just downloaded the latest deb file from Skype's website.  An older version of Skype was once on the computer, but I used Synaptic to completely remove it before attempting to install the Skype 2 beta.

After the install, I attempted to run, and a new window appears for a split second then disappears.

So I though I might try a sudo apt-get --purge remove skype, then reinstalled the deb.  Same problem still occurs.

Edit:

I fixed the problem.  A long time ago, I had used Automatix to install Skype.  I don't know what the version number was on that Skype, but I do remember the Interface for it looking distincly more windows like, more graphical.  Anyway, Beta 2.0 is up and running on my computer, after I used Automatix to reinstall skype, and then uninstall it, and then run the 2.0 beta deb installer.  That's it.
