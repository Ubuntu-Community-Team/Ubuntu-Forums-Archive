### Post by Digitallysick on 2005-12-08
You would need to partition your hard drive (not unless you have 2 harddrives) if you only have one, like the most of us, you will need to partition it. 

For me this was the hardest part , because if you do it wrong, you can destroy windows =) i highly suggest if this is your first time that you backup anything on your windows machine that you might want.

 Your best bet is to use partition magic, and use it to create a partition (id say at least 20 gigs ) (you can probably use alot less, but its up to the individual) It should have the option to create a linux partition, then put in ur ubuntu disk, reboot, go through the linux setup, when it comes to the partition table, you should see the partition you created.
