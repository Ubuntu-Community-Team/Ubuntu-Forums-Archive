### Post by TheFu on 2022-08-29
Sure.  Restore from the full system backup that you made before the upgrade.  Everything should work just like it did before.  Easy-peasey. 
Of course, that old system needs to be supported, so ... 22.04 or 20.04 are really the only viable answers today.  Additionally, for most DEs (desktop environments) support for 20.04 will end in April, so you'll need to move to 22.04 LTS.

While in October, 22.10 will be released, I never recommend using non-LTS to anyone.  The support period is just too short. Who wants to reload a new OS every 6 months? Nobody I know.
