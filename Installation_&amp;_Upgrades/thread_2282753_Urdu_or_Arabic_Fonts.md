### Post by grahammechanical on 2015-06-16
To type in different languages we do 2 things. We install a keyboard layout for that language and we install a language pack for that language. You do not say what version of Ubuntu you have so I can only give you information relevant to the version I am using which may not be exacly the same as your system.

Go to System Settings and load Keyboard to add a keyboard layout. We will get an icon in the top panel that will let us switch keyboard layouts. Then load Language Support to install language packs that include dictionaries and spell checkers.

Ubuntu Help will give some information about doing this.

For the web browser, it again depends on the browser that you are using. Which we do not know. In Firefox we can go to the View menu and select Character Encoding and change the encoding. I see 2 sets of encoding for Arabic but nothing for Urdu. Although setting Auto-Detect to On might be all you need.

Regards.
