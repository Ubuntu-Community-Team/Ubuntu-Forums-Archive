### Post by alastair on 2005-03-04
I don't know if this will help - but this type of "support" is almost impossible anyway.

Download a CD image file - either the "live" CD (test - run from CD) or the full install CD image file.

Burn to a CD

Install

e.g.

You have an AMD processor - so :

If you want to just TEST (not install) - download the live CD :

hoary-live-amd64.iso
[http://www.mirrorservice.org/sites/cdimage.ubuntu.com/cdimage/releases/5.04/array-6/hoary-live-amd64.iso](http://www.mirrorservice.org/sites/cdimage.ubuntu.com/cdimage/releases/5.04/array-6/hoary-live-amd64.iso)

If you want to try an install :

hoary-install-amd64.iso
[http://www.mirrorservice.org/sites/cdimage.ubuntu.com/cdimage/releases/5.04/array-6/hoary-install-amd64.iso](http://www.mirrorservice.org/sites/cdimage.ubuntu.com/cdimage/releases/5.04/array-6/hoary-install-amd64.iso)

This is a CD Image (i.e. ISO) - burn it as an image. Then boot from it and see how far you get.
