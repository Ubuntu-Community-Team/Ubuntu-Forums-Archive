### Post by Rocket2DMn on 2008-08-01
Just FYI, an anti-virus program is not necessary in linux because there aren't currently any viruses "in the wild" for linux.  If one were to ever be released, a patch would be released so fast that the virus would not have time to spread.
The primary reason for installing an AV program in linux to prevent passing on viruses to others through email and whatnot.
