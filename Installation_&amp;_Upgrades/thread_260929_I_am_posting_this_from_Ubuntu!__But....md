### Post by orb9220 on 2006-09-20
First ubuntu comes with iptables which is the firewall by default. If you install Firestarter in synaptic that will give you a gui frontend to  iptables which will allow you to make changes to iptables.

Second Anvir software would only be neccessary if you were running server's to protect email and file servers for windows people. And there is a couple of viriuses for linux but very rare. By nature you have to have root permission to install anything. You have to have root to run anything that will modify the file system. 

Now linux can't protect itself from the dumb user who clicks on things they shouldn't. The good news is a windows virus would just sit there not being able to do anything even if it got into linux OS somehow.

Yes No registry,No Defrag is neccessary either.
