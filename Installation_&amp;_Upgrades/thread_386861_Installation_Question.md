### Post by byen on 2007-03-17
Hello sinclac,
Welcome to the forums, Let me see if I can be as clear as possible..
As you know ...Windows XP uses the NTFS file system for its Operating System. And like most things developed my Microsoft, it is not very compatable with other operating systems and is proprietary . On the other hand Ubuntu (Linux) Uses various other formats such as ext3 for its OS needs. During the installation process you will be given an option where you can create a new partition in your existing hard drive onto which you can install Ubuntu. By doing so you will not only create a new partition without ruining your current Xp installtion but also have a seperate drive which would have ubuntu installed. The end result being.... Having an option to choose either Windows or Ubuntu during boot.  

I hope that answers your question. If not.... Just ask! Goodluck!
