Just out of curiosity... why do you need to know this???
If you go to ...
[http://packages.ubuntu.com/](http://packages.ubuntu.com/)
You can look up whatever is the current version of Firefox for the versions of Ubuntu that are supported in the drop down box for the distributions that are listed there.
Just type **firefox** in the Keyword: line on that page; and select the *buntu version you want in the Distribution: drop down box.
Also, 10.10 is in the future. So, unless you have a time machine, you will have to wait until 10.10 is released to discover what is the default Firefox in 10.10.
