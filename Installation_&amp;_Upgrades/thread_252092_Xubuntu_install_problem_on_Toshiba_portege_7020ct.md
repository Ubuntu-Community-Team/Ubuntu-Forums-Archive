### Post by bodycoach2 on 2006-09-16
It's my understanding that the Xubuntu alternate install in necessary for machines with under 192 megs of RAM. I've installed it (6.06.1) on two iMac G3, 333 Mhz, 160 Meg.

I don't think your machine has enough memory to run the LiveCD. Burn the alternate, and try that.

Xubuntu is a great distro, by the way. Simple, efficient, and fast on lower end equipment.
