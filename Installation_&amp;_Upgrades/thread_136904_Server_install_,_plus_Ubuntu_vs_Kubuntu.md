### Post by aysiu on 2006-02-26
From [Kubuntu's FAQ](http://www.kubuntu.org/faq.php): > Kubuntu is the first Ubuntu derived distribution. Our Kubuntu CDs are made up of Ubuntu's base plus KDE. You can get exactly the same effect by installing Ubuntu and adding the KDE packages (and removing the Gnome packages) from the Ubuntu archives.

If you install the server, there's no difference because the difference between Ubuntu and Kubuntu is the difference between Gnome and KDE. The server install gives you no desktop environment.

Debian Sarge is stable, has about seven CDs' worth of software, and repositories all over the place. Ubuntu is updated every six months and is based on Debian unstable and has a set of only a few repositories with a vast selection of software packages.

The support on those Ubuntu Forums is unparalleled.
