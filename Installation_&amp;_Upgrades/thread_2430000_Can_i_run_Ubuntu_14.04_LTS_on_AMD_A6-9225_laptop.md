### Post by ajgreeny on 2019-10-26
I don't know specifically about that laptop, but Lenovo machines are usually very good with Linux and Ubuntu.

However, whether or not it would run 14.04 is a pointless question; 14.04 lost support back in April and the repos are no longer active so you will no longer get upgrades to any applications, not even security upgrades.

Try 16.04 or better still 18.04, and if you don't like the gnome desktop version in the new versions of Ubuntu try one of the other DE versions, eg Xubuntu, Kubuntu, Ubuntu-Mate, etc etc.

### Post by Impavidus on 2019-10-26
Ubuntu 14.04 is dead by now. I suggest you try either 18.04 LTS or 19.10. 18.04 LTS has longer support, 19.10 has newer features and more bugs. Both will allow for a direct upgrade to 20.04 LTS when that's released.

If the live disk works, it should work after install too. Ubuntu works out of the box on most laptops. Sometimes you need some tweaks, but I read that Lenovo is one of the easier brands.
