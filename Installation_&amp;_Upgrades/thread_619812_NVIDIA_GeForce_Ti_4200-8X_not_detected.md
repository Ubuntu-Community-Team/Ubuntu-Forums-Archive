### Post by angryfirelord on 2007-11-21
Whoa, something's not right there. :)

Anyway, you can do the following:
```
sudo apt-get install nvidia-glx
```
Next, do the following:
```
sudo gedit /etc/X11/xorg.conf
```
Scroll down until you hit the area called *Section "Device"*. Next to Driver, change it to say "nvidia" (with the quotes). It should look like this:
```
Driver          "nvidia"
```
Save and reboot. You should have 3D working. To confirm, type (after reboot):
```
glxinfo | grep rendering
```
If it says yes, then it's working.
