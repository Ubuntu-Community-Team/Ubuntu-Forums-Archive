### Post by ManamiVixen on 2013-03-22
Also, if you want a traditional Gnome Desktop. Install the Ubuntu Gnome Remix. It's a spanking new Ubuntu respin added to the Ubuntu Family and is backed by Canonical!
[https://wiki.ubuntu.com/UbuntuGNOME/ReleaseNotes/12.10](https://wiki.ubuntu.com/UbuntuGNOME/ReleaseNotes/12.10)

When installed, in the Details section of System Settings, throw the "Force Fallback" Switch, logout and log back in to get Gnome Fallback. alt+right click lets you edit the panels. install compiz to get effects and load it on login with "compiz --replace"
