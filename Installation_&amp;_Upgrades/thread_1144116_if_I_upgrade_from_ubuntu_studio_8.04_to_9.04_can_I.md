### Post by zvacet on 2009-04-30
If you upgrade you will do that on top of existing partition and then there will not be Hardy anymore.Second(or should I say it first),you can not upgrade in way to skip versions.If you want to upgrade from Hardy to Jaunty you will go like this: **8.04>8.10>9.04**.If you want to have both versions install Jaunty on new partition.Third option is to do fresh install of Jaunty on top of Hardy partition.If you decide to do that back up your files/data or make separate home partition if don´t have it already.To create separate home you can follow [this]("http://www.ivankuznetsov.com/2008/04/moving-home-to-its-own-partition.html") guide.
