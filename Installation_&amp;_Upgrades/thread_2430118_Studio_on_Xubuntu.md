### Post by uRock on 2019-10-28
> **moonguide said:**
> I'm using Xubuntu 18.04 on a handful of boxes. I'd like to explore things related to Ubuntu Studio. Can I just add applications to my existing installations, or do I need to install Ubuntu Studio from scratch?

Yes, you can install just the applications. The best way I've found is searching ubuntustudio in Synaptic Package Manager, then selecting the packages, such as ubuntustudio-audio, ubuntustudio-video, or ubuntustudio-photography. When you mark to install, you can go through the list to see what it is installing and watch for anything that might be removed.
