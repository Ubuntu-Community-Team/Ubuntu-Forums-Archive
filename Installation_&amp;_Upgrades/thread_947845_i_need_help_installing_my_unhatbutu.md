### Post by Partyboi2 on 2008-10-14
You can change the "default" setting in your grub menu.list so that it will boot into windows after the timer has reached zero. You can also increase or decrease the time before it boots the operating system by making changes to the "timeout" setting.
For changing the "default" setting see [[COLOR=Blue]this[/COLOR]]("https://help.ubuntu.com/community/GrubHowto/ChangeDefaultOS")
For changing the "timeout" setting see [[COLOR=Blue]this[/COLOR]]("http://users.bigpond.net.au/hermanzone/p15.htm#timer")
