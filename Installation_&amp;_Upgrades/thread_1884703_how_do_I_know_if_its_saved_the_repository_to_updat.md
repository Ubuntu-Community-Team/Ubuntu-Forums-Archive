deb-source repos are for source code and are only needed if you are compiling your software. You can leave those disabled (commented out with #) until you need to compile, as in you don't need them if you are installing from the Software Center, Synaptic or the command line using apt.

Other than that, everything looks OK to me (though I don't know what blue fish is). I'd disable all the deb-src lines, especially for KDE and Icecat.

More info...
[https://help.ubuntu.com/community/Repositories/CommandLine](https://help.ubuntu.com/community/Repositories/CommandLine)
