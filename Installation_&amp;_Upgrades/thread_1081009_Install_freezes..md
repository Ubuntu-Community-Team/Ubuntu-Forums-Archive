### Post by Svensk023 on 2009-02-26
Try running the install in "safe graphics mode." At the installation disc boot menu, hit "F4" and select "safe graphics mode" and then run the install.

I have had problems where the install menu couldnt detect my screen resolution or I have also had it where I would get alot of screen tearing. But when using the safe graphics install method always solves my problem.

try it, and let me know how it works out!
