### Post by Partyboi2 on 2008-08-27
Once you have downloaded the ubuntu iso, check the [[COLOR=Blue]md5sum [/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM")to see if it matches. If it does burn it to a cd at x4 or less speed as a iso file not as data. Then boot your machine with the ubuntu cd, making sure that your machine is set to boot from cd rom first in your bios. Once the ubuntu cd has booted and taken you to a main menu select the option to check the cd for defects. If this passes with no errors then select the 2nd option in the menu "Install ubuntu" and follow the prompts.
