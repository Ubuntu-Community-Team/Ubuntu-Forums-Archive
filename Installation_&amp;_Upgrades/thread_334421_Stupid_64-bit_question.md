### Post by djgrandmarquis on 2007-01-08
I have an Athlon64, and I have run both flavors, Dapper AMD64 and now I run Edgy i386. For a desktop, I would recommend the 32-bit version. There are a few shortcomings with 64-bit software, for instance Firefox, which is not officially released by Mozilla in a 64-bit version. Both 64- and 32-bit Firefox will show up in Synaptic, but the newest version wasn't available in 64-bit. In addition, add-ons like Flash don't come in 64-bit.

Perhaps most importantly, I experienced hardly any performance difference between the two. The 64-bit software supposedly increases performance in number-crunching (think databases, etc.) and not many desktop applications. So, all things considered, I installed i386 Edgy when I installed Kubuntu Edgy.

Just my opinion. Good question. And you're correct, the new 64-bit architectures are backwards-compatible with 32-bit.
