### Post by ubuntnewb on 2007-03-26
I'm not sure how many people this will work for, but I wish I'd tried this early on as it would have saved me a lot of headaches. When I was trying to install on my Dell 640m my system would consistently hang when booting from the live CD. I was able to get it to boot and install in "safe graphics mode", but when it rebooted it would go back to hanging up and giving an x window error. Eventually I tried installing with an external monitor and that did the trick. Once everything was installed I updated and installed the 915resolution fix and now I'm able to use my laptop LCD without issue. I'm sure there's a way to install/fix this issue using the console, but after going through pages of posts and nothing working this could be the simplest solution (especially for others like me who NEED a gui =)

Good luck!
