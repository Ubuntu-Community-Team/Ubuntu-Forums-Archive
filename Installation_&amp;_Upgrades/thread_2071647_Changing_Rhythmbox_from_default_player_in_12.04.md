### Post by zombifier25 on 2012-10-16
Go to System Settings > Details > Default Applications.

---

### Post by deadflowr on 2012-10-16
You can also set the default per file type.
Simply right click on the media file, go to properties and then go to the open with tab, choose the program you'd like to open that particular file type(.ogg, mp3, or what have you) then click set as default. And now any time you click on that file type, the program you set for it will open by default.
