### Post by kcreek33 on 2009-01-31
I have a 500 GB external drive with A LOT of data already on it and don't have enough room on my laptop to backup everything.  Is there anyway I can install Ubuntu on a small partition I've created without losing my current data?

Also, if I have to wipe out the whole disk, what is the easiest way to set aside a partition that I can use for both the Ubuntu I'm installing and Windows to store my current data on the external drive?

---

### Post by Pumalite on 2009-01-31
You don't need to install Windows to store data. You can use Ubuntu's /home or make a fat32 partition. Unless you have files larger than 4 GB in which case I advise you to stick to Ubuntu's /home.
