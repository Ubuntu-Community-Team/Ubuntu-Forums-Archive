### Post by scorpion-kde on 2008-02-24
If you end up in BusyBox, that usually means something has failed.  I suppose it could be mistaken, and everything is fine.  If that is the case, typing "exit" at the BusyBox prompt, which may allow booting to continue.  The more likely case is that you have solved one problem only to encounter another.
