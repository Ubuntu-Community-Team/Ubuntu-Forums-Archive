### Post by kcbnac on 2007-06-25
I also am interested in figuring out how to build/get a 'current' installation disc (that I can update before I burn) but in the meantime, if all your machines are at one location, I suggest a local mirror:

[https://help.ubuntu.com/community/Debmirror](https://help.ubuntu.com/community/Debmirror)

You can set this to be any single machine, the i386 arch with all 4 repos enabled comes to about 17GB currently, for Feisty (7.04).  I highly recommend this if they're all at one location, and you can set the machine to auto-update its mirror with a cronjob as well.
