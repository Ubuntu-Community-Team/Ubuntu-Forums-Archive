### Post by tkiesel on 2005-02-17
Hi wjacobusse,

For an AMD Duron, you'll want to grab the i386 version using whatever method you prefer. There's an option for direct download from a central server, a bittorrent link (which I used, and am seeding as I type :) ) and via the jidgo program.

I decided to go with the stable 4.10 release and use that until I feel comfortable upgrading to the newest version.  The file to look for is warty-release-install-i386.iso

Good luck and welcome aboard!
-T

### Post by matt on 2005-02-17
After installing per tkiesel's instructions, grab the k7 kernel - for AMD processors - to improve performance. (sudo apt-get install linux-k7)
