### Post by laloruelas on 2009-04-16
To upgrade from Ubuntu 8.10 beta to ubuntu 8.10 RC on a desktop system, press Alt+F2 and type in "update-manager -d" (without the quotes) into the command box. Update Manager should open up and tell you: New distribution release '9.04' is available. Click Upgrade and follow the on-screen instructions. no clean install needed
