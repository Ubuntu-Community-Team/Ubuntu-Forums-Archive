### Post by oldos2er on 2009-12-12
If you upgraded, the system keeps grub legacy. If you want to install grub2, run **sudo apt-get install grub-pc**

 But first you might want to familiarize yourself with grub2:

[https://help.ubuntu.com/community/Grub2](https://help.ubuntu.com/community/Grub2)
