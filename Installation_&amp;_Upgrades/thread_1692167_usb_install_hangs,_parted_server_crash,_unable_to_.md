I ran into the install hang trying to install via usb drive.
(Toshiba Satellite A205)

I tried multiple iso versions, of ubuntu and kabuntu, and the alternate installs would not even boot.
Tried all the boot options in the F6 menu, unplugged all usb devices except for the drive i was installing from.
Verified the iso integrity on all of them.  Nothing....

I made the most progress / got past the most common hang spots by removing the usb drive once the install began (at the "choose language" screen) and reconnecting it after the keyboard screen.

I finally burned the iso to dvd and the install completed without issue.

skip the usb for the full install.  use a dvd.
