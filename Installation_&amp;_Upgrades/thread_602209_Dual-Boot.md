### Post by Herman on 2007-11-04
Yes, I am typing to you right now from a Windows 98SE and Xubuntu dual boot machine but I also have the IceWM, Fluxbox, and Gnome Desktops installed as well. Right now I'm logged into IceWM. :)

You can use the Xubuntu 'Desktop' CD if you have more than around 256 mb or RAM. That's the best CD to use if you have the RAM, but it you only have 256 mb or around that, you will find it hard to install unless you use a [GParted LiveCD]("http://gparted.sourceforge.net/") to partition your hard disk first and make a swap area (about 1 GB), so the CD can use that for extra RAM during the install.
Aysiu's website is a good website to see about how to install with the  Live CDs,[COLOR=#3333ff]** [http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)**[/COLOR]

If you have less than 256 mb of RAM, you should use the Xubuntu 'Alternate CD'. 
My first sig link, [Illustrated Dual Boot Site]("http://users.bigpond.net.au/hermanzone/") has three different examples in it about how to install with the 'Alternate CD'.

Regards, Herman :)

### Post by Herman on 2007-11-04
Installing is the same regardless of which *buntu you are installing, Fluxbuntu, Xubuntu, Ubuntu of Kubuntu.

What matters is how much RAM you have. The Desktop CD will give you a better installation if you have the RAM to use it. I like it because of the migration assistant it has in it. 
Besides, even after the installation is over, you might need the Live CD to do certain jobs every once and a while, such as partitioning work, file ssytem checking, or solving various types of booting problems, if you ever get any of those. 

The 'Alternate' CD is good because you can use it in a computer with less than 256 mb of RAM, (as low as 32 mb some say), and there are some people around here who just like the 'Alternate' CD better. I guess it's a matter for personal choice.

How much RAM does your computer have?

Regards, Herman :)
