### Post by lidex on 2009-11-22
Check for other options in bios for boot order. Do you have usb cdrom or other device attached? In my bios I select the main boot options and then the order of the devices in that category in a separate entry. So when I select HDD I then have to specify which HDD as only one of the three will actually boot. Aside from that have a look here:
[http://www.boot-land.net/forums/index.php?showforum=79]("http://www.boot-land.net/forums/index.php?showforum=79")
