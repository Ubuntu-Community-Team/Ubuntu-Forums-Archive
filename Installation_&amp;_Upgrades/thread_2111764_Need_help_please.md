### Post by darkod on 2013-02-02
You can't install wubi (ubuntu inside windows) on a computer that is not already running windows.

You need to download the ISO file for the 32bit or 64bit desktop OS, burn a cd from the image (not just the ISO file on the cd), and boot the computer where you want to install with it.

If you want only ubuntu on that computer, you can use the Erase and use whole disk option, that will install it for you.

Alternatively you can use manual partitioning if you feel comfortable with it.
