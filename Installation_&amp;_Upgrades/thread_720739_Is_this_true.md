### Post by Whiffle on 2008-03-10
Yes and no.  The issue here is that some laptops BIOS's are too agressive in powersaving, causing the hard drive to sleep too often (and then be woken up).  Ubuntu doesn't change this behavior, and so running it could hurt something.  I'm pretty sure Ubuntu doesn't enable the extreme power saving behavior unless its put into laptop mode though.

I don't have any issue with it.  If you do some searching theres a few threads on it.
