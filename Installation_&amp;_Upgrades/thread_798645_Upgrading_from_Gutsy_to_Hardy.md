### Post by meanerelk on 2008-05-18
No need to reinstall. You can just update and reboot. To do so, use the update manager to check for updates, install them, then check for updates again. It will notify you that a new distro version is available, and you can begin the update process.

More detailed instructions here: [http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)
