### Post by howard-gilbrt on 2014-04-28
After many hours of struggle I solved this by deleting  .gstreamer-0.10 from my home directory and then copying in .gstreamer from my 13.10 installation.
Hope this helps someone.  (It is a 64bit system)

PS This work in Firefox for Flash but not HTML5. If anyone knows how to get that please tell

LATER: 

Solved other sound problems by deleting .config/pulse from my home directory and then copying in  .config/pulse from the a 14.04 usb stick " try xubuntu".
For some reason the installation of 14.04 did not get this .config/pulse directory correctly
