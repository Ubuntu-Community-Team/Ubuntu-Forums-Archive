### Post by Rocket2DMn on 2008-02-07
The LiveCD does not work for everybody, but you should really install the latest version anyway, Gutsy replaced Feisty last October.  You can download here - [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
Since you have graphics problems, I would suggest using the Alternate Install cd by checking the box at the bottom of that page before proceeding.  This gives you a text based installer without a live session, but it works very well.
If you still have problems with the graphical stuff (X server) after installing, that can be easily troubleshooted then.
