### Post by Casey on 2005-04-14
This is in the wrong place, as your edit has noted, but I didn't see your question in another forum so let me answer it here.  You have downloaded the Live CD which is simply a linux environment run from the CD (In this case Hoary).  You need to download the install CD to install.  The previous poster was mentioning that it may be possible to install from the live CD (and it very well might be possible - it is with some distributions), but it will be easiest for you to just burn the install CD image instead and go from that.

Edit: And while I'm thinking about it, you can upgrade straight form Warty -> Hoary without using the CD if you would like to.  Just follow the Warty Upgrade steps.
