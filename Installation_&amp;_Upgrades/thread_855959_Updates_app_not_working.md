### Post by iaculallad on 2008-07-11
Open up your terminal and do:

```
sudo apt-get update && sudo apt-get upgrade
```

If that doesn't work, try to select other download server on your Software Sources.

System->Administration->Software Sources, under "Ubuntu Software" tab -> "Download From", select "Main Server". Click on Close, and choose reload on the next window that pops.

And again, in your terminal:

```
sudo apt-get update && sudo apt-get upgrade
```
