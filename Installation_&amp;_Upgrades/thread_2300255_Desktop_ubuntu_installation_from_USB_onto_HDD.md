### Post by sudodus on 2015-10-24
Welcome to the Ubuntu Forums :-)

0. Decide what versions and flavours of Ubuntu you want to test

1. Copy/clone/flash an Ubuntu iso file(s) into a USB stick

2. Boot 'a live session' from the USB stick and 'try Ubuntu'

3. Start the installer from the live session and  install Ubuntu into the internal HDD

There are many help pages. You can start with these

[FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it](http://ubuntuforums.org/showthread.php?t=2230389)

The computer hardware makes a difference - if you tell us about it, we have a better chance to give good advice. Se please specify your computer

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

Do you want to install Ubuntu only, or alongside Windows?
