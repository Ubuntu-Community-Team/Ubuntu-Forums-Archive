Update:

well i was able to get the system to install, removing the quiet splash -- from the boot option when installing was the only way i could get into the instal enviroment, at which point the install hung up, after closing the window out the live desktop appeared. one thing i did notice is that the hardrive i had put in was brand new and no partition or filesystem was on the disk. i didnt think that this would matter and it may not still. but i added an ext4 fs to the disk and ran the installer from the desktop and all seems fine now.
