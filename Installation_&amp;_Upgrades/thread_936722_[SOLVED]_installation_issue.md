### Post by growingneeds on 2008-10-03
Please check the integrity of the CD that you are trying to install from. This can be done during boot time. Only proceed to install from the CD if it passes the integrity check.

If the install fails, do try the [Alternate Desktop CD]("http://www.ubuntu.com/getubuntu/download") for Ubuntu 8.04.1. It is less taxing on the system and generally gives a better success rate for installation on older systems.

### Post by iponeverything on 2008-10-03
Choose one of the other partitioning options.

If you nothing on that drive that you care about just tell it to auto partition and to use all the space. 

what might be going on is that there could be an existing partition and the install program is trying to shrink it for you. Is the HD light lit while it is stuck?  While its stuck do a ALT-F[1-12] -- see if you can find the VC where there is some info on what is going on.

good luck and don't give up.
