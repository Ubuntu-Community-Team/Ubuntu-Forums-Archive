### Post by The Cog on 2012-12-24
I think startup disk creator is badly named - it is for making bootable USB memory sticks. For burning to a CD or DVD from a .iso image, use the Burn Image option in your favourite CD/DVD burning program. I tend to use xfburn personally, but that's probably because it's the default program with Xubuntu (the desktop I use at the moment). I think Ubuntu uses gnomebaker by default. If you're on Kubuntu, then k3b is your default and is very good.
