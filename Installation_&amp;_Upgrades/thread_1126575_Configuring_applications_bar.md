### Post by MaindotC on 2009-04-15
I'm not sure why you'd install it with the RPM as you could just install it from the repositories but anyway...Click System -> Preferences -> Main Menu.  Then select the menu where you want it to appear - probably the System Tools menu because that's usually where it appears - and click "New Item".  Locate the binary which should be in /usr/bin and pick an icon.
