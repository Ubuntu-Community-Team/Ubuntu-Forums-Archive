### Post by Gina on 2008-01-10
Firstly, have you run the CD data test?  I've found CDs very iffy - you generally need to burn at a low speed (something like 4x) to get a relaible result.  Have you run the MD5Sum check on the CD in the machine it was burnt on?

If the CD (or DVD) isn't the problem it would be worth trying earlier versions.  Dapper (v6.06.1) is a Long Term Support release and generally more reliable than other releases.
