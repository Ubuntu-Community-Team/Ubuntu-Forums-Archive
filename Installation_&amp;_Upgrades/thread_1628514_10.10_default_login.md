### Post by sikander3786 on 2010-11-23
No you don't need a username or password to log in a Live Session.

Please check the integrity of downloaded image.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If it is correct, press any key while the computer starts booting from the disc. You'll be presented with a menu and that contains an option to check the disc for defects, check it.

[https://help.ubuntu.com/community/LiveCDBootOptions](https://help.ubuntu.com/community/LiveCDBootOptions)

I believe it is a bad burn or a corrupt ISO, if so, download a new one or burn a new disc (at the lowest possible speed), whichever applicable.

Instructions on burning a disc,

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

You can also use a USB drive to boot Ubuntu if supported by Bios.

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

[http://unetbootin.sourceforge.net](http://unetbootin.sourceforge.net)

Good Luck!
