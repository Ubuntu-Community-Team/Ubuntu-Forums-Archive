To all,

Saw this idea in an earlier post.

For laptops with a floppy drive, the Ubuntu 7.04 live cd stalls with a tty error.  

Solution:

..start boot with live CD
..after booting process starts, insert a non bootable floppy into the floppy drive (doesn't matter what is on the floppy disk)
..live CD will now fully load the Ubuntu 7.04 program on RAM

Enjoy
