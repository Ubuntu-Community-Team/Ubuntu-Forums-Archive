### Post by renzokuken on 2007-06-01
you need to burn "the image". not a data cd or copy the contents as you have done. an iso is an "image of a cd" not the contents of it

use this program for windows [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm) to burn the cd properly. then make sure your bios is set "to boot from cd"
