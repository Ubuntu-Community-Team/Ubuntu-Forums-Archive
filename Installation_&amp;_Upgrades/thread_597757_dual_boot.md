### Post by lisati on 2007-10-30

> thanks man...i'll try that...btw it won't f**k up my windows partition will it????

As long as you have enough room in your Windows partition for Windows to do it's stuff  & enough room on your HD for a Linux partition, there shouldn't be a problem. 

Note: If you go ahead and install Windows first, it's a good idea to (a) backup important files & (b) defragment your windows partition a few times. Even if turns out that you didn't need to, it wouldn't hurt.
