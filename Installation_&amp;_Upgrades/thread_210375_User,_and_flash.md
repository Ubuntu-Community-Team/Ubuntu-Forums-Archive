### Post by aysiu on 2006-07-06
1. [Make sure you have extra repositories](http://www.psychocats.net/ubuntu/sources)

2. Paste this command into the terminal ```
sudo aptitude update && sudo aptitude install flashplugin-nonfree
```

3. Read these two links:
[http://www.psychocats.net/ubuntu/permissions](http://www.psychocats.net/ubuntu/permissions)
