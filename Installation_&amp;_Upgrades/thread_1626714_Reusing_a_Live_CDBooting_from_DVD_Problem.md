### Post by ajgreeny on 2010-11-20
It should normally work burning a CD image onto a DVD with no problem, so I am baffled.

You can not burn anything else on to a previously used CD unless it is a CD-RW, which some people frown on for burning ISO images.  I have always used CD-RW for that and never had a problem, however, so I am not sure I believe that story.

Perhaps it was a bad burn, so maybe try again at the slowest speed you can, and don't use the computer for anything else while burning.
