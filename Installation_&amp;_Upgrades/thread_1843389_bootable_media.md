### Post by Hakunka-Matata on 2011-09-13
> **waterpipes said:**
> How do I get to root if the system will not boot

Usually one boots to a LiveCD when everything is broke, that's the starting point. 

If your LiveCD/USB isn't working, then you have to boot to Windows and try to make a working LiveCD/USB.

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/) works well to build the LiveCD/USB.  

A good working LiveCD/USB has a bunch of files on it, the thumbnail shows this.

Are you sure you've selected Boot from CD as first boot device in BIOS?
