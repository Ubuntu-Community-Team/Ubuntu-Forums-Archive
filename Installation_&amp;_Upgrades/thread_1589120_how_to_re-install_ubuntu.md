### Post by tommcd on 2010-10-06
> **nuey said:**
>  How would I go about removing the previous ubuntu and then reintalling it?

How did you install Ubuntu in the first place? If you want to reinstall, just do it the same way. Simply choose to reformat your current root partition and reinstall Ubuntu to your root partition.

Note: If you do not have a separate home partition, you will need to back up all of your data first.
If you are reinstalling Ubuntu this is a good opportunity to create a separate home partition if you do not currently have one:
[http://www.psychocats.net/ubuntu/installseparatehome](http://www.psychocats.net/ubuntu/installseparatehome)
