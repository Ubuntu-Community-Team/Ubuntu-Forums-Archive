### Post by dabl on 2011-03-05
1. If the "other" OS is Win 7, then you need to use the Win 7 disk partitioning tool to shrink the Win 7 partition by at least 6 or 8GB -- ideally more than that, to make room for a Ubuntu partition.

2. Use the Alternate Desktop installation CD, and use "manual" partitioning, for more control over the process.

### Post by dabl on 2011-03-05
The "Alternate Install" is a different CD.  It has more packages on it, and uses a character-mode installer.  There's no "desktop" on it, just the installation menu.
