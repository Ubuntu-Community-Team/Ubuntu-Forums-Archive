### Post by Dreamer Fithp Apprentice on 2014-06-21
Several possibilities come to mind. Might be as simple as installing the latest propietary driver for your graphics card. You could use Synaptic or whatever you like for browsing the repository and search the brand name of your graphics card. For me, for instance, the driver metapackage to install is nvidia-current. I'm not sure what the default settings on an out-of-the-box Ubuntu are anymore but you may have to enable propietary drivers as a class. That used to be in the menu. 

Once, I had this exact problem, well, same symptom I should say. For me, what worked was reinstalling xorg, thus:```
sudo apt-get purge xorg
sudo apt-get install xorg
```I was afraid it would uninstall all my graphical programs as dependencies and I'd have to reinstall them all but it didn't. It took less than a minute.

There is at least one, maybe more, similar threads around here somewhere, that might have some other suggestions.
