### Post by MAFoElffen on 2014-12-03
Like "Unity," XFCE, OpenBox, and LXDE--> Gnome Shell is a Desk-Top environment. It is a graphical Layer between you and the Linux kernel core system them provides a  grahica user interface to your applications. Depending which level of Gnome you boot into, You will have diiferent ways to get to your installed applications.

If you had installed Ubuntu from Ubuntu Gnome Desktop Edition... then that would have been a distribution with the Ubuntu Core, the Gnome shell desktop and a suite of applications... 

So it's seems to be working fine, I think you had an expectation that Gnome-Shell was something else. There are shell applets that extend it's fuctionality and how the menus look and feel in it, There are applications you can install, but Gnome-hsell is just a way to get to them... It is not an application or group of applications. Think of it as a menu'ing system.

If in the standard view of Gnome 3, if you take the mouse to the upper left corner, does the search app perspective appear? (sort of like Unity's) But if you don't have a graphics card capable of 3D graphics, it would never be able to work with that view, then would have to boot in a metacity, or Gnome Classic view, with the old styled menus.
