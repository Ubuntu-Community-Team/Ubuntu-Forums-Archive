### Post by TheFu on 2020-08-14
It depends on how the software was installed whether any special updating is needed or not.

If you always use the package manager to find, download, and install software, then the updates for that package manager will handle it.  There are different settings to control whether updates happen automatically or not.  I have them disabled on all my systems; reasons.

If you manually download some package and manually install it, then you'll need to manually maintain it.  That is the rule regardless of how any manual software is installed.

On Linux, answers are seldom simple, but almost always have very clear reasons.
