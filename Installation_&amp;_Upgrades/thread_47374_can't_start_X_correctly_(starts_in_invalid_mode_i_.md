### Post by sizzam on 2005-07-08
I've found a solution that works on my machine.

After I get up and running, I issue this command:

sudo dpkg-reconfigure xserver-xorg

On the first screen, select your own video driver and choose vesa instead of i810.

I also set my video memory to 65536.

Continue on accepting defaults until you get to:
Use kernel framebuffer device interface?  I chose 'no'.

Continue on choosing defaults.  When finished, CTRL+ALT+F1.  

Then:  sudo /etc/init.d/gdm restart

When x started, my display flipped out for a second, but it then corrected itself and my video now works correctly,   My res is 1280x1024x24 depth.
