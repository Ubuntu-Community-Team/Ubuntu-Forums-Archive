### Post by Partyboi2 on 2009-04-30
Hi, if you are wanting to upgrading to Jaunty from Intrepid without an internet connection, you can download the [[COLOR=Blue]jaunty alternate cd[/COLOR]]("http://noncdn.releases.ubuntu.com/jaunty/") iso from a machine that has internet connection and burn the iso to disk then boot up intrepid and when you have reached the desktop insert the jaunty alternate cd and an option to upgrade should pop up.
