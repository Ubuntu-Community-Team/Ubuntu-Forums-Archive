### Post by benanzo on 2007-05-04
just go to ubuntu.com and download the ISO and burn it to a disk.  Then make backup of all the stuff you want to keep on your windows disk and reboot with the ubuntu disk in the drive.  This will boot to a LiveCD where you can play around with ubuntu without installing it.  If you like it you can click the Install button on desktop and install it.
