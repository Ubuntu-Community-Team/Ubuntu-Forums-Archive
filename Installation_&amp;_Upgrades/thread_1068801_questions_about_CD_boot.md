### Post by meindian523 on 2009-02-14
It doesn't matter what ISO program you use,as long as you burn as a CD image.And use [http://www.nullriver.com/downloads/Install-winMd5Sum.exe](http://www.nullriver.com/downloads/Install-winMd5Sum.exe) to run a md5 check on your downloaded ISO.Read [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM) and compare with the appropriate md5sum given here: [http://releases.ubuntu.com/hardy/MD5SUMS](http://releases.ubuntu.com/hardy/MD5SUMS)
