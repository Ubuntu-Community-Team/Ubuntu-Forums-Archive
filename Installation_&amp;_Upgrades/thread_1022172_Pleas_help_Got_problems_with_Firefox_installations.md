### Post by taurus on 2008-12-26
The one from the Applications -> Internet is still pointing at the version from the repos while you installed the new version somewhere else in your $HOME.  So if you want to use the new one that you installed by hand, create a launcher on the top panel.

Place a cursor on the top panel and click the right button -> Add to Panel -> Custom Application Launcher -> Add.  In _N_ame;, type in Firefox and in Comm_a_nd:, browse to where you installed the new version of firefox.  Then click Add.  Now, if you want to run the new version, just click on that icon on the top panel.
