### Post by crazy bird on 2011-11-24
On your top panel go to: *System > Administartion > Synaptic Packagemanager*.
In synaptic click on *Search* and type in **blender**.
When this package shows up, rigth click  on that package and select *Mark for installation* and press *Apply*.
 
When a windows pops up asking to install several other packages, just click on *Mark for installation* since these packages are dependencies of Blender.
