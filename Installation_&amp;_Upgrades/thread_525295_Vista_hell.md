### Post by gokussj4nr on 2007-08-14
I had a very similar problem with my comp a few weeks ago, but I had installed Fedora Core 6 instead of Ubuntu.  I fixed the problem by downloading the Fedora Core 6 rescue CD and booting from it.  Then I deleted the partition FC6 made using "fdisk."  After that, I tried the Windows installation again with success.  It seems Windows has issues recognizing harddrives already partitioned with Linux.  Hope this helps :)
