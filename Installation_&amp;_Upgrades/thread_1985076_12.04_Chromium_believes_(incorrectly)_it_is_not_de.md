### Post by DrJohn999 on 2012-05-22
After upgrading 11.10 --> 12.04, Chromium insists on nagging at each start that it is not the default browser. It *is* the default browser: 
```
~/.local/share/applications$ cat mimeapps.list

[Default Applications]
text/html=chromium-browser.desktop
x-scheme-handler/http=chromium-browser.desktop
x-scheme-handler/https=chromium-browser.desktop
x-scheme-handler/about=chromium-browser.desktop
x-scheme-handler/unknown=chromium-browser.desktop

```

The default is functional: clicking on a link in email starts Chromium.

However, chrome://settings/browser shows Chromium as not being the default. Clicking on Make Default appears to set it as default, but restarting reverts back to not being the default.

Strange and annoying.
