### Post by garybrlow on 2007-02-11
Allo Junio!

Beryl is a fork of Compiz. Refer to this article [http://en.wikipedia.org/wiki/Beryl_(window_manager)]("http://en.wikipedia.org/wiki/Beryl_(window_manager)")

For easy installation of the official NVIDIA driver use the ENVY script found here  [http://albertomilone.com/nvidia_scripts1.html]("http://albertomilone.com/nvidia_scripts1.html")
including description and instruction details. 

You can also find Ubuntu NVIDIA and ATI drivers here  [http://albertomilone.com/driver.html]("http://albertomilone.com/driver.html") if you want to manually install them. Includes repositories and details.

For the Official NVIDIA Drivers you can get them here [http://www.nvidia.com/object/unix.html]("http://www.nvidia.com/object/unix.html") 
 if you want to compile it yourself. Note, you have recompile every time the kernel is updated.

Beryl runs fine with NVIDIA FX5200/128MB or higher on 256MB system RAM. Instructions for installation are located here [URL="http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu"]
http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu[/URL]. I suggest using the "Install Beryl on Ubuntu Edgy with nVidia (1.9xxx or higher)" section since it uses the official drivers and it works fine for me. Other methods are also found there.
 
Howtos are also available here in the forums. If you're using feisty, the recent X11 update broke Beryl, a recompile might be needed.

Cheers!!! :razz: 

GaryBrlow :D
