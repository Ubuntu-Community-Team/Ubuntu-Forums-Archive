### Post by wilee-nilee on 2009-11-09
you can remove it with gparted and reformat it from the main install, you may have to install gparted to the main install. You can't remove it if your booting from the usb. The boot-able usb can be used indefinitely. So if you remove and reformat you can install another OS to it until it wears out.
