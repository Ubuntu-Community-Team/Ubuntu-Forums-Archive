### Post by running_rabbit07 on 2009-08-22
Before starting the install process try the "boot without make changes" option to make sure that Ubuntu works on your system. The go to the next step.

At the menu screen press [F4] to start the install in text mode. Text mode just means it won't be loading all of the graphics to start the install. Instead of using the mouse, you will be using the arrow buttons on your keyboard.
