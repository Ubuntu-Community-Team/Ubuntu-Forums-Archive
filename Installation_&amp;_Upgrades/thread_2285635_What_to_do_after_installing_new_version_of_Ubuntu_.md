### Post by ubfan1 on 2015-07-07
Yes, the 64 bit CPU can run 32 bit OSes just fine.  As for the compatiblity of 32bit vs 64bit configurations, certainly the Ubuntu libraries will be in different paths, so some things might be affected -- safer to reinstall, but if you're backed up, might not hurt to try the old 32 bit configs.
