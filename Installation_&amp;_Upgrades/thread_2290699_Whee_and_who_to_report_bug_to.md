### Post by PaulW2U on 2015-08-14
> **wjbmd48 said:**
> Who do I report this bug to?

If you really think you have found a bug then from within Ubuntu type Alt-F2 and then type **ubuntu-bug firefox**. 

Or from within a terminal type **ubuntu-bug firefox**.

In each case follow the prompts. You'll need a Launchpad ID to complete the bug report though after being asked if a bug report has already been reported. You might see a list of bugs already reported that match what you are now reporting.
