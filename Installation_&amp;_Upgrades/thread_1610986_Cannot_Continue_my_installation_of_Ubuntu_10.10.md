### Post by Rubi1200 on 2010-11-02
Hi and welcome to the forums :-)

The user name cannot start with a uppercase letter.

Name: John Smith

User name: needs to be john not John

This is why the installation will not proceed.

Try again and I am sure you will be able to finish the installation.

Good luck!
