### Post by Qrk on 2007-02-15
Technically the system requirements are 32 MB RAM and a pentium class processor, both of which you satisfy. However, the graphical installer from the live cd requires 192 MB of RAM. You'd have to use the alternate install cd, which you can download from the ubuntu website. You also don't have enough power to run Gnome, KDE or XFCE. That means that the normal Ubuntu, Kubuntu or Xubuntu desktops won't run. Instead you'll have to use a lighter window manager, like Fluxbox, Blackbox, IceWM or Windowmaker. 

Here is a wiki page on the subject 
[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)
