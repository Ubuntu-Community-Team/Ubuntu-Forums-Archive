### Post by CaesarLike on 2007-10-28
Hi there,

I have just completed installing 7.10 onto a second pc.  My first was onto an old 1.7 gig Celeron, which went flawlessly (see my earlier post).

The install to a second machine, a much more modern pc, went not quite as smoothly as the first.  But this was more my fault than Gutsy's.  And whereas the previous install was an upgrade, this one was a fresh install to an empty drive.

Ok first the PC specs:

Gigabyte 965-DS3 Mother board.
Intel C2D 6420 processor
2 gigs of ram.
N-Videa Geforce 7600 GT GPU

OK, so first install attempt did not work.  Froze at around 85% point.  Caused from the drive having an existing windows primary partition on it.  When I removed this partition, so drive had 100% unallocated space, and tried to install again, all went as it should.  The second install attempt went without even the slightest problem.  PC booted up from the drive first time.  All the hardware either recognised, or simply ignored by Gutsy if it did not know it.  Either way, no problems there.

As for performance, its like the Linux version of "The WOW Starts Now!"  The PC literally goes like a rocket!  I have never seen this PC run so fast.  Even the internet runs better and faster under Gutsy, than it ever did in XP, which is what this PC has been running up to now.

Now its time to install Automatix, and get Ubuntu going fully loaded.

So the advice here is this:  When doing a fresh install, remove all existing partitions so the drive has 100% unallocated space.  Makes the install much easier and less problematic.
