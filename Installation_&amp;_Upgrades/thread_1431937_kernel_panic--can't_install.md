### Post by dstew on 2010-03-17
I assume you are trying to boot a Live CD. Maybe the CD has errors on it. Have you tried to boot that same CD in other computers?

If the CD is bad, but you think the .iso it came from is good, I would suggest burning another, but at a very slow speed, no more than 4x. This is because software image CD's need every single bit to be perfect, where music CDs or video DVD's are very tolerant of bit drop-out.
