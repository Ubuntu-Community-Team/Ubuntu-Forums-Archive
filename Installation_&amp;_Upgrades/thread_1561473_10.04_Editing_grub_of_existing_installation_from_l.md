### Post by oldfred on 2010-08-26
256MB is a little light for the full install.

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)
[https://help.ubuntu.com/community/Installation/SystemRequirements#Lightweight%20GUI%20alternative%20%28Xubuntu%29](https://help.ubuntu.com/community/Installation/SystemRequirements#Lightweight%20GUI%20alternative%20%28Xubuntu%29)


To get the menu with only Ubuntu and using grub2 you have to hold the shift key down from BIOS until menu appears. You can use e to edit kernel line and add parameters but that will be just for this one time boot.

You need to edit /etc/default/grub to get it to permanently add changes.

General info:
[http://www.gnu.org/software/grub/manual/](http://www.gnu.org/software/grub/manual/)
