### Post by Mark Phelps on 2009-12-14
If you read the links, you'll discover that every time you boot in and out of your Windows OS, you will be forced to reactivate.  The instructions in the posts worked because they were for Windows XP, and for earlier service packs.

This problem has been reported over and over again when folks tried to do this virtualization stuff with Vista and discovered that after a very small number of reactivations, it wouldn't reactivate anymore. Since Win7 is based off the Vista activation model, I would be very surprised if these hacks work with Win7.
