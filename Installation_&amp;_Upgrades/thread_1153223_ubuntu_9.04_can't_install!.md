### Post by nowhere@cox.net on 2009-05-08
I don't know what your particular issue is but here are some standard things to let the community know in order to help you.
[LIST=1]
[*]Which distribution? Ubuntu Kubuntu etc...
[*]Which install type? Server, desktop, alternative...
[*]Burned it to CD? What speed did you burn?
[*]Dual boot (side by side install with Windows) or fill entire HD with Ubuntu or Wubi perhaps?
[*]At what point does it tell you installation failed?
[*]Were you able to verify the md5sum of your download iso file to make sure it wasn't corrupt?
[/LIST]

I have several times had unsuccessful installs because of the CD Media. I now ALWAYS burn at the slowest speed my drive will let me. This used to give me an error in the middle of the installation of packages when it encountered a bad sector on the disk. You can also try a better brand of disk if you are using a cheapy...

Hope this helps, if not, post some more info.
