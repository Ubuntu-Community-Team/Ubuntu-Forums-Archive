### Post by Dennis N on 2018-07-03
> i have transmission, with 16.04lts., still dont have a clue how to use this.
From that web page you refer to, click the 'magnet', select 'transmission' then 'open link'. You get the transmission 'options' window where in this case there is nothing to do except click 'open'. Then main transmission window opens, it automatically starts the download (see screenshot). Wait till its done downloading the iso, then when download is complete, it will continue in upload mode, and your file is shared with other users. Proper thing to do is to let it run for a time. You can set the amount of upload (ratio to what you received) in the gear menu. Experts say that with transmisssion, there is no need to verify the downloaded iso with sha256 sum.
