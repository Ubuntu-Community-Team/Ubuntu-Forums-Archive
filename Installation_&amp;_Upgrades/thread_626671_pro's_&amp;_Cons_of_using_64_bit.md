### Post by SeanHodges on 2007-11-29
I run 64bit at home.

Bearing in mind that there are workarounds for pretty much everything, you will have immediate problems with the following things:

- Adobe Flash Player (it does have "Gnash" player, which is getting better but still has some glitches)

- Java (although I'm not sure on the status of this these days, I havent installed it since I upgraded to Gutsy)

- The win32codecs package, although there are very few videos not supported by free alternative codecs these days.

In general it functions just as well as 32bit Ubuntu, and it's getting better all the time. However, if you need the above software, and you're not prepared to put a bit of time into getting them working, you're probably better off staying with 32bit and checking again at the next release of Ubuntu in April.
