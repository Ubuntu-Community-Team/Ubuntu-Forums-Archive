### Post by Sef on 2008-12-24
Applications > Add/Remove > Show: All Available applications or All Open-Source Applications > Search: Gstreamer > click on box next to Gstreamer > click apply changes > click apply > close.

All Available applications = Multiverse = open source and proprietary applications

All Open Source Applications = Universe = only open source applications
