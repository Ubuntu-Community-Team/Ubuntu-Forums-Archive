### Post by foresthill on 2011-09-13
Does it start out black? Or can you see the boot screen when the computer first boots up from the CD? 

If you can see the first part of the install process, hit any key, choose your language, then when the screen comes up to let you choose what you want to do (Install, Live CD, etc.) hit F6 and select "ACPI=Off". Then proceed to install and see if that works.

I have a notebook with Intel HD 3000 integrated graphics and that's what I had to go to get the installer to run.

Also, you might try downloading the AMD 64 version of 11.04 and see if you have better luck with that version.

Just some ideas, HTH.
