### Post by IgnorantGuru on 2007-02-14
Not sure I understand what you're trying to do.

If you want to network the linux box with an XP laptop, I would suggest getting a switch ($30) or router ($50).  Wire them up, reboot, and they should talk (Linux should see shared folders on the laptop).

As for setting up a modem in Ubuntu - I'm not the person to ask.  I would google the modem model and "ubuntu" and see what you get - I'm sure there are How-Tos on setting up dial-up.  But not all dial-up ISPs support linux (like AOL requires Windows software last I knew).

Do them a favor- talk them into getting broadband if available.
