### Post by Mark Phelps on 2015-05-03
> will Ubuntu recover and work, or will it be hopelessly confused and not work? 

As long as you don't have any restricted drivers installed, when you reboot the drive on the new machine, it should be able to find and install all the drivers it needs.  In this way, Linux is a lot more forgiving than MS Windows when migrating to a new machine.
