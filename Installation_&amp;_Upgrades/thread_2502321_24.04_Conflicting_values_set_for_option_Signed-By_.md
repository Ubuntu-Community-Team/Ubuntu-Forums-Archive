### Post by John.St on 2024-11-22
Found the solution myself:

sudo rm /etc/apt/sources.list.d/boot-repair.list 
sudo rm /etc/apt/sources.list.d/yannubuntu-ubuntu-boot-repair-noble.sources
sudo rm /etc/apt/sources.list.d/danielrichter2007-ubuntu-grub-customizer-noble.sources.save 

Then run Software & Updates
Remember your server name
Change server
Close
Reload
