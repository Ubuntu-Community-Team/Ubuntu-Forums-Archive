### Post by Mr A Mouse on 2008-05-18
Okay, here are a couple of things to try:

1: Consider downloading the CD again. It sounds like you're using the same CD you downloaded a year ago--Ubunty has had a major release in the last year. (If you're not using the same CD, I apologize for the confusion on my part.)
2: Check the CD for physical damage--scratches or the like.
3: See if the command prompt will respond to the word "boot" (no quote marks, of course).

Beyond that, the only thing I can think of is to put the CD in, turn the computer completely off (a full shutdown), then bring the power back up. As the power comes up, try going into your BIOS menu and making sure you can boot from the CD.

Good luck! :)
