### Post by QIII on 2009-09-24
Medibuntu isn't a program.

It is a repository of programs that can't normally be included in Ubuntu for legal and copyright type reasons that might vary from locale to locale.

If you have added the Medibuntu repository, you can add applications from the repository via Synaptic, the terminal or Applications | Add/Remove

[http://www.medibuntu.org/](http://www.medibuntu.org/)
