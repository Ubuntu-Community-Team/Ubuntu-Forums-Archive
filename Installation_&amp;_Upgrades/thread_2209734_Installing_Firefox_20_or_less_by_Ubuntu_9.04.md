Hi Guido. Ubuntu 9.04 is far too old. Ubuntu is not like Windows - you can't use an old version of Ubuntu.

An old version of Ubuntu will not be able to find its *software repositories*, which means you will not be able to install any software.
An old version of Ubuntu will have security flaws and it is not safe to use an old version of Ubuntu to visit websites that store your confidential data. You must especially never use an old version of Ubuntu for online banking.

The only versions of Ubuntu you can use are Ubuntu 12.04 (it is supported for a long period of time) or Ubuntu 13.10 (the latest Ubuntu but only supported until July 2014). In April, a new version of Ubuntu called Ubuntu 14.04 comes out which will be supported for five years.

Please do not attempt to use an old Ubuntu. You must use Ubuntu 12.04, 13.10 or the new 14.04.

If your computer is not powerful enough for a recent version of Ubuntu, you can install Lubuntu 13.10 or Xubuntu 13.10 which are more suitable for an old computer.
