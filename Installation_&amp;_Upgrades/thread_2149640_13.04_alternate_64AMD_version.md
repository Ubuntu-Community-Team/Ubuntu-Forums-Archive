### Post by sudodus on 2013-05-30
> **dryder said:**
> 
- if I use the Desktop, will I still be able to choosed the hdd, manually partition it and decide where I want grub installed?


Yes, during the installation you select ***'Something else'*** at the partitioning screen (window) and at the very bottom of the window, there is an option to select where to install the boot-loader. The rest of grub is installed into the system partition (or boot partition if you select a separate one). And you can select your partitions as you like.

I often use this method, but I prefer to make the partitions with ***gparted*** before I start the installer, and then only select those recently created.

I use the alternate installer only for computers with low RAM. By the way, Lubuntu's alternate iso files have survived into 13.04, and they are included in the daily builds of Saucy (13.10).
