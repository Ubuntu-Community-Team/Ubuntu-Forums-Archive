### Post by dosware on 2008-05-02
Details: 

2004 HP laptop w/ 512MB RAM

My previous install included many 3rd party Gutsy repositories- but Heron upgrade process disabled them (good). I did have to re-install non-free ATI video and Broadcomm WIFI driver in order to re-enable wifi and compiz. Note that Heron notified me of this in the notification area. Yes, I had to have a wired connection to get WIFI working again.

Broadcomm and ATI non-free upgrades proceeded without issue. Everything works 100% without tweaking. Conclusion: Heron kicks Windows A$$ on a ZX5000.

Note that my original install with Gutsy required Xorg.conf editing (see below). Also, highly recommend that you install and run syndaemon to sanely control the zx5000 touchpad.

Below is my Xorg.conf for zx5000:
