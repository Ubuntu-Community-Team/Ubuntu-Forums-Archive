### Post by llamakc on 2006-06-05
You can simply continue by re-running Synaptic. Personally, I use the command line for this:

APPLICATIONS | ACCESSORIES | TERMINAL 

sudo apt-get dist-upgrade

and after you type in your password it will get back to finishing the task of installing/configuring your packages.
