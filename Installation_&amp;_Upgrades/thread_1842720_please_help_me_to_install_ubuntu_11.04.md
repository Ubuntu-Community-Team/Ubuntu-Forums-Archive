You don't have the correct file for this. The Alternate ISO isn't one you can install within Windows via Wubi.

You can boot the computer from that CD and do a proper dual-boot installation, but it's a text-based installer and not quite as user-friendly as the regular GUI installer.

Or, just download the "desktop" ISO, or let Wubi download the correct file for you.

But yeah, sorry you can't use Wubi with that file you've downloaded.
