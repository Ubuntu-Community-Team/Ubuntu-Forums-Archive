### Post by warmaster4z2 on 2013-03-13
Try this solution to your problem, it worked for me:

[COLOR=#000000]1. Download Grub Customizer from Synaptic Package Manager.[/COLOR]

[COLOR=#000000]2. Go to System Tools and open the program.[/COLOR]

[COLOR=#000000]3. Right below the "Save" button on the upper left hand side of the screen you should see three tabs (List Configuration, General Settings, Appearance Settings)[/COLOR]
[COLOR=#000000]click on Appearance Settings and use the drop down arrow to select one of the resolution sizes, I used 1024x768x16. Now maybe if you're lucky this will solve your problem.[/COLOR]

[COLOR=#000000]4. Now click "Save", exit the program, and restart your computer to see if the problem is fixed.[/COLOR]
