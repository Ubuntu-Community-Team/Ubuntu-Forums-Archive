### Post by rainwalker on 2007-09-08
System -> Preferences -> Main Menu
Over on the right side of that window there are buttons to create new launchers/menus. 
I'm guessing you want a launcher, so click the "New Item" button and specify the command or location of the file.
If you're not sure where it is, check the .deb file (you can look it up in System -> Administration -> Synaptic Package Manager). Select the package and look at it's properties. In the properties window there's a tab for all of the files that were installed.
