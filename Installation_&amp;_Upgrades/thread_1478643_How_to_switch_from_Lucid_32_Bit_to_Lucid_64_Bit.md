### Post by SolitaryMan1941 on 2010-05-09
I recently upgraded from Karmic to Lucid via the Update Manager. I would like to upgrade further by switching from 32 Bit to 64 Bit. I downloaded the Lucid 64 Bit ISO and wrote an install disk. When I reboot the computer with the install disk nothing happens. Any ideas?

---

### Post by wil08son on 2010-05-09
Your computer may look to boot from the hard drive before the CD/DVD drive. When your computer initially starts up, something like "F12: Boot Menu" should flash on the screen. Hit whichever function key you need to and then choose to boot from the computer's CD/DVD drive.
