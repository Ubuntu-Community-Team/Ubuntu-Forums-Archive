### Post by taurus on 2007-12-26
Shutdown the Add/Remove window.  Then go into System -> Administration -> Synaptic Package Manager -> Settings -> Repositories and put a check mark on all options _except_ the last one, Sources code, and CDROM.  Close it and click Refresh.  Now, close down synaptic and see if you can add/install anything with Add/Remove.
