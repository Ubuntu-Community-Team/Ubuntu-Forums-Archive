### Post by railmeat2 on 2014-04-04
This is resolved by setting nomodeset on the [COLOR=#000000][FONT=monospace]GRUB_CMDLINE_LINUX_DEFAULT line of /etc/default/grub, then running sudo update-grub.

All seems well now. [/FONT][/COLOR]
