### Post by sikander3786 on 2010-10-10
Edit the xorg.conf file by

```
sudo gedit /etc/X11/xorg.conf
```

Section "Device" should look something like this.

```
Section "Device"
    Identifier    "Default Device"
    Driver    "nvidia"
    Option    "NoLogo"    "True"
```
