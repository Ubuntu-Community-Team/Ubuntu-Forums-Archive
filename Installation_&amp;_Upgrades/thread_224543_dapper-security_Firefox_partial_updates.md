### Post by eXisor on 2006-07-28
You make very good point, sir, and it is frankly disappointing that this is not already done right accross the board for all updates in X/K/Ubuntu, especially to the massive core modules.

Programmatically this is surely not rocket science.

1) Make a local copy of the installed version
2) Download the relevent patch to take version from xx to xx+yy
3) Apply patch to copy
4) Check hashes/md5 checksums etc
4) Install updated copy

Especially since not everyone has broadband, and not everyone has unlimited capacity even if they are broadband.
