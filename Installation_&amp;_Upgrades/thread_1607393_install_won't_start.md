### Post by sikander3786 on 2010-10-27
First of all Check the disc for defects.

[https://help.ubuntu.com/community/LiveCDBootOptions](https://help.ubuntu.com/community/LiveCDBootOptions)

If the CD is ok, probably a hardware fault.

1. Try **nomodeset** option from F6 menu at startup.
2. Check your RAM.
3. Check the HDD for any problems associated to it. Bad sectors / bad cables etc.
