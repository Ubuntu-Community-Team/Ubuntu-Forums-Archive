### Post by OrangeCrate on 2009-03-17

Yes. Why not just upgrade rather than do a fresh install? Go to System > Administration > Software Sources > Updates, and change the Release upgrade from Long term support releases only, to normal releases. Then run Update Manager, and you'll see the option to upgrade to 8.10.

Edit:

I don't quite understand your concern about a download limit for an upgrade. If you're talking about downloading a new ISO, sure, but not an upgrade. Sorry, if my suggestion won't work for you. A new install might be your only option.
