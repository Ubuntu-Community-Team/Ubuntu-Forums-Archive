### Post by iaculallad on 2008-05-26
Actually, VNC (viewer or you can issue the command **sudo apt-get install vncviewer** if in case you can't locate it on Terminal Server Client) is already installed on your system. You just have to go to Applications->Internet->Terminal Server Client and from there, under the "General" tab, choose VNC as the protocol.

For the Server, goto System->Preferences->Remote Desktop
