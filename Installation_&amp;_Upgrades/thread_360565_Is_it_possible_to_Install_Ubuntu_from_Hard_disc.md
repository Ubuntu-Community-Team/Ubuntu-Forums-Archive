Portage may have certain advantages and disadvantage over apt-get, but can surely you don't consider Gentoo's system of download+compile to be faster than apt-get's system of download?

If you are looking to speed up the update process, you can either download programs from the DVD (not useful for updates, just for installing new programs), install your own repository (too much work and pain), use a leaner system (that would probably require fewer or at least smaller security updates), or get a faster internet connection. A faster computer would help somewhat also.

The other obvious "solution" is to update your computer less frequently or when the computer isn't being used.
