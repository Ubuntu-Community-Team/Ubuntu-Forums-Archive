---

### Post by darkod on 2012-04-21
Yes, but you will have to use the manual install method (Something Else) because ubuntu by default doesn't use existing partitions.

You will have to tell it which partitions to use for what, with selecting them and hitting the Change button.

The root partition to be used again as mount point /, and the swap for swap.
