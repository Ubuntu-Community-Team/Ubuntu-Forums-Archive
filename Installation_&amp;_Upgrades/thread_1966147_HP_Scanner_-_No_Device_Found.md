### Post by makitso on 2012-04-26
Clean install Precise.  Multi function HP Offjet 6500 wireless printer.  Printer work as expected.  However, can not find device when attempting to use scanner.  Solution was to install newer HPLIP driver 3.12.4.  Download from web site and execute.  Then sudo hp-setup.  After this scanner worked.
