### Post by Jordan777 on 2007-06-29
Back up your data first.  Just if you have another computer or hard drive back up that data.  Then I would do a disk cleanup and defragment the hard drive.  Then if you use the Ubuntu Live cd it has Gparted on it which allows you to edit and make new partitions.  I don't know if you really want to try this on a work laptop though.
