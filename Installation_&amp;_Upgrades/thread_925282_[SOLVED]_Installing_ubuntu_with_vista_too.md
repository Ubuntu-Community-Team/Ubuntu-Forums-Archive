### Post by Pumalite on 2008-09-20
Use the Vista partitioner and allocate space for Ubuntu. Get an iso from here:
[http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download)
(I'd advise the Desktop version)
Burn a CD following this guide:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Install Ubuntu in the allocated space. (Go Manual)
