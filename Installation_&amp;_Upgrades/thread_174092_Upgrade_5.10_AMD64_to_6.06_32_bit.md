### Post by Kobalt on 2006-05-11
I think a dist-upgrade will get you the 64bit version of Dapper...
If you want the 32bit version you should download/install a 32 bit version and make a fresh install. 
But it's sad, cause the 64bit version of dapper is blazing fast, you'll really lose in speed. I guess though you want to install a lot of 32bit stuff, but give the 64bit version a try, it's worth it. 
Plus if nobody uses the 64bit version then it will not get better any faster... Help us fix the bugs by reporting them on Launchapd ;)
