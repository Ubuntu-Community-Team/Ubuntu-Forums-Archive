### Post by SuperDuck on 2007-10-11
Hello-

1) Back up any data that you don't want to be lost.  Burn it to a CD/DVD or put it on an external hard drive.  

2) Download an .iso file, which is an image of a CD.  You'll need to burn it to a CD as an .iso using a special program.  There are several free utilities available for Windows that will burn .iso's.  If you burn the file to a CD just as a regular data file, it will not work.

To obtain an .iso, download it from the site of a Linux distribution that you wish to you.  I would suggest either:

[http://www.ubuntu.com/](http://www.ubuntu.com/)
or 
[http://linuxmint.com/](http://linuxmint.com/)

3) Insert the CD into your machine and reboot.  They will be what's called a "Live CD", so the Linux operating system will boot straight off the CD, as opposed to booting to your hard drive.  This is handy because it lets you make sure everything works before you install the OS.  This is very beneficial for you because it allows you to enjoy Linux without having to bugger up your system as it is now.  You can run it off the CD for a while, and install it if you think you want to make the switch!

4) On the desktop is an "Install" icon.  Double click on it, follow the prompts (it's similar to installing Windows), and you'll have Linux on your machine.

If you don't want to muck about with the .iso files (which isn't all that bad), you can order pre-made CD's from various places, often for a few bucks.  Some computer stores have them for free.

Good luck!

### Post by dabl on 2007-10-11
Yep, what SuperDuck said.

Also, for Vista users, I read that one should first use _Vista's_ partitioning utility to free up the space for Linux, before you boot the Linux installation CD.  So, you need to figure that out, and make yourself a respectable sized free space -- the basic *buntu OS will need a 6GB partition for the root filesystem, and then you should allow 0.5GB for swap (unless you want it to hibernate, in which case it is 1.5X RAM), and then add however much you want for your data to that.  :)

### Post by PmDematagoda on 2007-10-11
The different names have different desktop environments as default, Ubuntu has Gnome, Kubuntu has KDE and Xubuntu has Xfce. You can use either Ubuntu(Gnome) for a "unique and simple GUI" or Kubuntu(KDE) for a "Windows look and feel". If you prefer a light desktop you can use Xubuntu.

Edubuntu mainly emphasises on education and is good for students.
