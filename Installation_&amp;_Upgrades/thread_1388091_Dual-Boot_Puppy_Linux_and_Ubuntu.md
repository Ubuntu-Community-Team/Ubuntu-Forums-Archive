### Post by C.S.Cameron on 2010-01-22
You should be able to install Ubuntu to the same HDD as Puppy and end up with a choice to boot Ubuntu, Puppy or 98 at start up.
I think a menu giving thes options should come up automatically, if not tap the Esc key.
Just do NOT choose to use entire disk when you get to partitioning.
