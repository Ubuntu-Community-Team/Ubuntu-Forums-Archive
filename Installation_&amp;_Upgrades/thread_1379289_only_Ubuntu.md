### Post by Grenage on 2010-01-12
Yes, a fresh install is usually nice and easy.  Simply put the CD in your drive and boot from it.  The CD drive is usually the primary boot device, if not, you can change that in the BIOS.

By telling Ubuntu to use the whole disk it will overwrite anything you currently have, so be sure it's what you want and that you have backed up anything you want to keep.
