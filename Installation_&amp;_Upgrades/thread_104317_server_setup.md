### Post by wylfing on 2005-12-15
Although I do run a home server (a headless Debian box), I've never installed the "server" version of Ubuntu. But there is no reason why the "regular" version of Ubuntu wouldn't work just as well for a server -- they are the same thing except with slightly different default packages installed. So if you don't want to use the command line just install the regular version.

As for Windows programs, you can run *some* things with wine or Crossover Office, and you can run quite a lot of Windows games with cedega, but other than that you're probably out of luck.
