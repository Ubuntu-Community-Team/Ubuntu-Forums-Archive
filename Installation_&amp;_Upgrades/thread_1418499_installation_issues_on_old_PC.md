### Post by Mark Phelps on 2010-02-28
I don't know the basis of your particular problems, but I did try using Ubuntu and then, Xubuntu, on an older P3 800MHz laptop -- and the performance was so slow as to make it all but unusable.

So, even if you do get to install on your K6 500, don't be surprised if it's really slow.

If you're trying 9.10, you might try one of the older versions to see if they install/work better.

Barring that, you should consider a leaner distro meant to run on slower processors -- like Puppy Linux.
