### Post by sotiris2 on 2015-02-19
So you installed an extra controller in the form of ? A pci card? I would suggest that you boot from an original usb port and check if you can see the new ports then. It is possible that the bios won't be able to boot from addon usb ports. What is certainly not possible is to bypass bios, and have them boot if bios can't do it.
