### Post by dabl on 2007-08-13
> **A$h X said:**
> 

 it simply hangs with no message just a blank screen with a flashing cursor at the top left.



Probably Alt-F1 or Ctrl-Alt-F1 will bring up a console screen and CLI login prompt.  If so, the problem is that your video hardware is not correctly auto-detected.  You can either configure a VESA display, or make an Alternate Install CD and install in VGA text mode.  Here's how to do the VESA thing: [http://kubuntuforums.net/forums/index.php?topic=3085112.0](http://kubuntuforums.net/forums/index.php?topic=3085112.0)

HTH! :)
