### Post by markekeller on 2008-03-08
Hey Anders!

The Emesene [installation instructions]("http://emesene.org/trac/wiki/Installation") say that you've already done everything you need to do.  Now you can just run it!

To do so, you can just double-click on the file called 'emesene', and select run.  If you want, you can create a desktop icon for it, by right-clicking on the desktop, selecting 'Create Launcher...' and then click the 'Browse' button next to 'Command', and navigate through the folders until you find the emesene binary.
