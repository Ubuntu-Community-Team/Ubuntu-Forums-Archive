### Post by drpjkurian on 2014-04-23
Hi 
I upgraded my kubuntu 12.04 to 14.04 which was not pleasant surprise because the response and briskness of the OS vanished. So I did a clean install of the OS in the root partition. Still it was slow and i noticed that it was due to Dolphin file manager and Gwenview.
I changed my file manager to 'Konquerer' and 'imageviewer' (the best, light and fastest image viewer programme) which made my machine once again brisk and crisp... You can try this tip if require

The other problem was my dragon player was not playing videos.
I installed the following codec from package manager and they are as follows 
h.264 codec
DivX MPEG-4 decoder

I pasted the above mentioned codecs in package manager and it suggested few packages which I installed. I noticed that VLC was installed in due process, I uninstalled the programme through muon software centre and checked the video playing capability of dragon player and found out to be fine. I think it requires library of VLC.
My machine seems to be trustworthy for the next five years
