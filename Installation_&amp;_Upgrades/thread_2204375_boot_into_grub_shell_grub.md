### Post by oldfred on 2014-02-08
How large is hard drive?

Some older BIOS do not boot from partitions beyond 137GB. So either a /boot partition or all of / (root) need to be inside the first 137GB of drive. I often suggest 10 to 25GB for / and rest for /home or data partitions so entire drive is still used.
