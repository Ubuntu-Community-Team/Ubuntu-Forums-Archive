### Post by oldfred on 2018-04-09
It looks like that system has nVidia.
You probably need nomodeset boot parameter until you install a nVidia driver from Ubuntu repository.
Did you install in UEFI boot mode?

If UEFI, you press escape inbetween UEFI screen & grub loading. If BIOS it is hold shift key. But if UEFI fast boot on, you may not have time to press any keys.
 At grub menu you can use e for edit, scroll to linux line and replace quiet splash with nomodeset.
