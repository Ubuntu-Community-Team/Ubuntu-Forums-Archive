Whether I use "add/Remove" or "Synaptic Package Manager" it does not download or install the software. Comes up with following message. Being new to this game what does the following error message mean I have to do?

### Post by Pumalite on 2008-03-25
At the Terminal:
sudo dpkg --configure -a
sudo apt-get update
sudo apt-get upgrade
