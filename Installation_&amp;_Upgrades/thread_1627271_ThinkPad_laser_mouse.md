### Post by jawwad.hakeem on 2010-11-25
I was able to configure the ThinkPad Bluetooth Laser Mouse by Lenovo (JL 0510) using the default bluetooth device discovery wizard.

Step 1: Press the bluetooth visibility button at the back of the mouse.
Step 2: Left click the bluetooth icon
Step 3: Select preferences
Step 4: Click setup new device

Make sure you run the steps quick enough so not to loose the visibility of the mouse. The above should aid you in configuring the mouse and be up and running in a minute.
