### Post by grahammechanical on 2015-04-09
We set a password during installation. Is that not correct? Ubuntu, on our behalf, uses that password to login to Linux and then load on top of Linux. Without that password Ubuntu is broken.

We can set Ubuntu to load without a user login and we can set to deactivate the lock screen. A default installation will have Lock screen enabled and also a tick mark against "Require my password when waking from suspend." All this is independent of the time settings for Lock Screen or Turn screen off when Inactive.

System Settings>Brightness & Lock.

Regards.
