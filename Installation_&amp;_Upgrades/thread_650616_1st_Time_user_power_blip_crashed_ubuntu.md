ok next time you get a problem like that try starting in recovery mode from the grub menu 
this should take you to a command line where you can fix things
things that can really help are;learn to use apt-get package manager,nano for editing files and ls command for listing your directorys.
these are all command line apps and you can practice their use in a terminal,having this basic knowledge has got me out of bother a number of times
the add/remove program in the menu will give a good selection of what's on linux
