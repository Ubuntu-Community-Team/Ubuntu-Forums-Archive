### Post by snowpine on 2008-11-04
You can also install a command line only Ubuntu using the mini.iso:

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Boot up with the mini CD and type 'CLI' to install the minimal system. Then, you can install whatever else you want on top of it, for example 'sudo apt-get install ubuntu-desktop' if you want everything.

What are the computer's specs? That may be part of the problem if it's a really, really old PC.
