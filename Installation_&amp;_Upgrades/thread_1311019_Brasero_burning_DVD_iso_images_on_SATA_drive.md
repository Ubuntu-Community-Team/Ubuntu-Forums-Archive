### Post by Jim_in_Omaha on 2009-11-02
With 9.04 I was getting the long error log and failure message with Brasero when burning a DVD image on my new SATA drive but the DVD would actually play without a problem. It never did that error on the IDE drive. 

I did the upgrade to 9.10, fixed the grub kernel problem, the VMware player problem, the JAVA problem where it takes 100% cpu resources on all cores. Then I tried burning the same image that I've burned a dozen times with errors. It worked like it's supposed to. That was worth the upgrade and having to fix a few other small issues.

Thanks again to the Ubuntu Community and the vast array of knowledge. Just takes a little time and patience to make it work.:popcorn:
