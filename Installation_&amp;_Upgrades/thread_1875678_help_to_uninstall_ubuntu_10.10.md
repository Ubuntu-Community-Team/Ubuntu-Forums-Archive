### Post by darkod on 2011-11-05
Just enter win7, open disk management and delete the ubuntu partitions. Then create one ntfs partition from that space.
After that you will need to restore windows bootloader to the MBR which you can do with your win7 dvd. To be sure you know how to do it, you can investigate first "how to restore win7 boot process" in google.

PS. If you want to leave yourself the option to install ubuntu again easily, when you delete the partition don't create ntfs from the whole space. Leave about 8-10GB at the end unallocated (not belonging to any partition). Ubuntu needs less than 3-4GB to install. If you leave unallocated space in future you can install without any deleting or resizing of existing partitions.
