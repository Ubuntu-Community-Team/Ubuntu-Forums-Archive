### Post by kurt18947 on 2012-01-22
A couple things come to mind.  There are sometimes settings to select/deselect USB keyboard input.  There may also be something about 'legacy USB'  If you're using a USB keyboard, these should be selected on.  You could also try a PS/2 keyboard if you have one.  PS2 is the small round connector oftentimes colored purple for keyboard, green for mouse.  USB keyboards used to come with a PS2-USB adapter, I don't know if they still do.  I'd guess you need to check BIOS settings then do something like 'F10' to accept.  You can't do that without a functioning keyboard.  I've kept an old PS2 keyboard around just for such situations.  

If the above doesn't work and your machine is a desktop, there's probably a way to reset the BIOS to default.  Usually it involves shorting 2 pins on the motherboard for a few seconds or removing the round battery that maintains BIOS settings for a minute or two.    If you have or can download the motherboard manual there should be instructions in it. Be sure to unplug your machine before removing the cover.   I hope you can fix your problem without irreversible problems.
