### Post by mike1947 on 2012-03-31
The new cairo dock has both beautiful graphics and is self scaling.  Some of the applets like weblets, google, and web search don't appear to always work certainly not on my platforms with 3.0.0.0 but the rest is beautiful.  However there is a nasty problem for the slightly careless.  Global menu which gives direct control of an app to the dock i.e. minimize maximize, close in its focus steals the windows menu by default making some windows useless. (Lyx, rkward).  However when it simply steals the decoration, not the menu it works perfectly and is a nice additon:lolflag:
