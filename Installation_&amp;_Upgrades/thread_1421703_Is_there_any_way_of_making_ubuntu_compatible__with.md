### Post by Mark Phelps on 2010-03-04
Your question is somewhat different from the thread title, but if your asking can you run MS Windows programs "inside Ubuntu" without the aid of third-party stuff such as Wine (and other variants), the answer is NO.

There are alternatives involving using VirtualBox to set up a virtual machine inside Ubuntu, installing MS Windows inside of that -- but I don't know if that meets your definition of "inside Ubuntu".

VB allows you to run virtually any MS Windows program, but you have to install the MS Windows OS, you will need a license to activate that, and you will suffer some degree of performance degradation. Plus, as far as I know, you can't run 3D accelerated games that way.
