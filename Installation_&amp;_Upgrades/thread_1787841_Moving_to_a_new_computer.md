### Post by ellis rowell on 2011-06-21
I have recently moved to a new computer twice. The first move was from an Acer desktop to an Advent desktop. Instead of installing Ubuntu on the HDD of the new machine, I took the HDD (Ubuntu) from the old computer and installed it in the new computer, disconnecting the Windows HDD. This leaves the Windows HDD as new to be re-connected if you want to sell the computer later.

My latest move has been from an Acer 5101 laptop to an Acer 5742 laptop. After checking that both HDD's were SATA, I then swapped them over in the later laptop. It does save having to install from scratch, not only the system but the applications and data too.
