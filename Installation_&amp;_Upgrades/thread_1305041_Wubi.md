### Post by ajgreeny on 2009-10-29
There is a wubi equivalent for Mint called mint4win, which as far as I'm aware does exactly the same thing, but until you have windows running again, you can't use it, of course, because it is just the same as installing anything else in windows, so you need that first.

If I were you, and the drive is big enough, I would backup all my personal files to a usb drive using the live CD, then shrink the windows partition with gparted, and install Ubuntu or Mint to the free space just made.  When you are able you can then reinstall windows, or get Win7 if you want and dual boot, though you will need to reinstall grub if you do things in that order
