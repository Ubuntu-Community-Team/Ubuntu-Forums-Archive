### Post by taurus on 2006-10-07
It really depends on what you plan to install!  If you only want to run the server stuff, then not a whole lot but if you plan to have a desktop running on your machine, then probably at least 2.5GB.  So, if you have a 5GB or 10GB of free space, you should be all set.  And if you plan to download or store a bunch of personal stuff like photos, mp3s, or movies, then you need to have a lot of empty space...

### Post by K.Mandla on 2006-10-07
The live CD installer has to have 2Gb for a root partition. I think a straight Ubuntu installation uses about that much. 

On the other hand, the server installation only needs 300Mb, but even a very basic GUI is probably going to push 600Mb. I think 630Mb is the lowest I've gotten it, and that didn't include a boot partition or swap. :rolleyes:
