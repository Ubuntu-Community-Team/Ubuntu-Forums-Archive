### Post by sikander3786 on 2011-03-05
Windows should detect Ubuntu partitions as unknown partitions at least and should be able to delete them.

If not, you can boot from an Ubuntu Live CD. Fire up GParted from System > Administration sub-menu and delete the Ubuntu partition and create the new one's. Then install Windows.

For dual-booting Windows and Ubuntu, you might need to do manual partitioning and this page might help.

[http://ubuntu4beginners.blogspot.com/2011/02/manual-advanced-partitioning-in-ubuntu.html](http://ubuntu4beginners.blogspot.com/2011/02/manual-advanced-partitioning-in-ubuntu.html)
