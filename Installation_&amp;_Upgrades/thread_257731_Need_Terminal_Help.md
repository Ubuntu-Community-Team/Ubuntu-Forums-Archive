### Post by whizbaby on 2006-09-15
This one is good (don't be afraid of the *advanced*)
[http://www.tldp.org/LDP/abs/html/](http://www.tldp.org/LDP/abs/html/)

here a few commands
**man *command*** gives the manual of *command*
**ls *directory*** lists *directory*'s contents
**cd *directory*** changes the current position to *directory*
**pwd** print working directory (on screen)
**less *file*** show contents of *file*
**grep *pattern file/path*** search for *pattern* in *file/path*
**mkdir *directory*** make *directory*
**rm *file/directory*** remove *file/directory*
**chmod *file*** change *file* permissions
(for the exact use please read the respective manual)
