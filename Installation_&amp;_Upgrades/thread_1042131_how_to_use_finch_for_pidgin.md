Ok, first look here:

[http://developer.pidgin.im/wiki/Using Finch](http://developer.pidgin.im/wiki/Using Finch) 

or here:

[http://www.cognitivecombine.com/2009/12/how-to-use-finch-instant-messaging-from-the-command-line/](http://www.cognitivecombine.com/2009/12/how-to-use-finch-instant-messaging-from-the-command-line/)

But there are more commands. They are not so listed, but I found out one where you use the meta key plus 'm'. This allows you to move around the finch window so that it is not overlapping with other windows. Also the meta key plus 'r' allows resizing. I found I like to move and resize the windows so they dont overlap. There may be more commands that I dont know. I havent seen a full list anywhere. There are also quite a few in the manual page: man finch.
