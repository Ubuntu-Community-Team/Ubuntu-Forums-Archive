### Post by ian-weisser on 2015-11-30
The simpler, safer way to install XP is to NOT dual boot.
Instead, install a VM host like Virtualbox (in the Ubuntu repositories, just one click in Software Center), and install XP as a VM Guest.

+1 ^^^

This is the way I do it for updating my satnav device which is impossible to do with Linux of any sort.  I don't care about the possibility of XP being compromised due to it being internet connected as I have a good working snapshot of the XP VM; if it gets corrupted or virus infected I'll just delete the VM and use the good snapshot.

You will, however need a laptop with pretty good specs to run VBox at a good speed, though luckily XP runs quite well with pretty small ram resources.
