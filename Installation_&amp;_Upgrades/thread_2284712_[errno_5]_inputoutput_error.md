### Post by tgalati4 on 2015-07-01
If you are trying to install standard Ubuntu (with the Unity interface), then you might not be happy with the experience:  [http://julianstirling.co.uk/ubuntulubuntu-on-a-samsung-nc10/](http://julianstirling.co.uk/ubuntulubuntu-on-a-samsung-nc10/)

Try an older version (which will be unsupported) but at least working, so that you can get the feel of how Ubuntu works and get up to speed on Linux.

[https://help.ubuntu.com/community/NC10](https://help.ubuntu.com/community/NC10)

[https://nc10ubuntu.wordpress.com/](https://nc10ubuntu.wordpress.com/)

12.04 might be a candidate:  [http://askubuntu.com/questions/357112/which-ubuntu-derivative-for-my-netbook](http://askubuntu.com/questions/357112/which-ubuntu-derivative-for-my-netbook)

Because the NC10 is old, I would not expect 14.04 to install and run out-of-the-box.  Older versions have been installed and run OK, but a lot of tweaking is needed to get everything working.  It will test your patience.
