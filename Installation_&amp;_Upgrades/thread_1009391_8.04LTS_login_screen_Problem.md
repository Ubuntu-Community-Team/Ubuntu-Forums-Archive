### Post by Partyboi2 on 2008-12-12
Open a terminal (Applications>Accessories>Terminal) and type

```
sudo dpkg-reconfigure -phigh xserver-xorg
``` to update xorg.
then
Ctrl+Alt+Backspace to restart xserver.
