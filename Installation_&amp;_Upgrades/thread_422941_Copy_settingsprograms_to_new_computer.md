### Post by IgnorantGuru on 2007-04-25
You should always install Ubuntu fresh on the new pc (unless the hardware is exactly the same).  Then install any drivers and additional software for that pc.  Add the same users you had on the old pc.

Once you have that, copy /home from the old pc to the new pc.  This should transplant most of your software and desktop settings.
