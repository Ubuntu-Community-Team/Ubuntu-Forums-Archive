### Post by oldfred on 2010-06-02
I like having each operating system on separate hard drives and that operating systems boot loader in the MBR of that drive. Each drive can be booted if need be, but Ubuntu will find the other systems and also let you boot them.

One of the best dual boot sites. Herman now has an example of two drives.
[http://members.iinet.net.au/~herman546/index.html](http://members.iinet.net.au/~herman546/index.html)

I prefer dual boot but do not use my XP much anymore. I have thought about virtualization but I do not care about max speed for gaming of a virtual system.
