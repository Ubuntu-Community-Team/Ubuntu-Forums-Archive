### Post by kc1di on 2013-02-10
it should not be a problem. 

when you do the install you'll come to a section on Partitioning the hard drive, select do something else , see attachments.

partition that second drive with at least 2 gigs of swap space and 10 gigs for a root / partition.  I like to partition the rest of the disk for a separate /home partition also. 

use ext4.  

there are any number of install guides on the web. which you may find helpful to read.
