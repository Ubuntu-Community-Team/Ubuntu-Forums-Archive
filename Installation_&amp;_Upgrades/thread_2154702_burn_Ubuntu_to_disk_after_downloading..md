### Post by ajgreeny on 2013-06-15
Puppy runs in ram if you were running it as a live CD or USB, so if you tried to save the .iso file to puppy's home folder you may well have run out of space, as it will depend on the amount of ram you have and the size of any swap, though I can't remember how swap is dealt with in puppy.  Try downloading again but this time save the .iso file to a hard drive or large enough USB drive and you should be good to go.
