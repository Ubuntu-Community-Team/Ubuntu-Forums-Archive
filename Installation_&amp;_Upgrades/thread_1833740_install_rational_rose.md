### Post by Mark Phelps on 2011-08-27
You need to install Wine first -- and to do that, you need to go to the Wine subforum here and read the threads.

But, there's a strong possibility that Rational Rose will not work, anyway.  The WineHQ site lists the testing results for different versions of products, and no one has tried any Rose versions with any recent Ubuntu version.

So, while you're in the Wine subforum, you should do a search to see if anyone else has tried using Rose.  You could also post a support thread there if you don't find anything.
