### Post by ajgreeny on 2011-07-26
Sure you can.  There will be no green tick in the box where it says internet or network connection, if you have no connection, and just make sure not to tick the boxes for "install 3rd party software" or "install updates" or the install will lag as it looks to do the impossible..

I prefer to do those after installation, as I know exactly what I want to add, and it makes the installation process many times faster.  If you do have a connection, either cabled or wireless, it is up to you which you choose to do.
