I used the instructions at [http://www.pendrivelinux.com/2007/09/28/usb-ubuntu-710-gutsy-gibbon-install/]("http://www.pendrivelinux.com/2007/09/28/usb-ubuntu-710-gutsy-gibbon-install/") to install Gutsy onto an 8Gb thumb drive. I use this alternately on a machine at home with an nVidia video card and a machine at work also with an nVidia video card. The one problem that I have is that although I can get the nVidia restricted driver installed and working on either machine, the configuration doesn't persist through a reboot, including reboots on the same machine. I'm choosing the persistent boot option, and other configuration changes I make persist through reboots, but not the choice of video driver. On either machine, I can do the installation, press Ctrl+Alt+Backspace, and get the updated driver, but if I reboot I'm back to the nv driver.

I would like to use the nvidia driver as default on both machines, because a) the home machine has dual monitors and with the nv driver the second monitor is unused and flashes very annoyingly, and b) the work machine has a 1920x1200 LCD monitor and the only way I can get the native resolution is with the nvidia driver.
