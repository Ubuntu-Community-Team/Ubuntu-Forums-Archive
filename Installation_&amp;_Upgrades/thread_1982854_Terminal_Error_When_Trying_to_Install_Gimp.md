### Post by hakermania on 2012-05-19
In Ubuntu Linux, you cannot install applications without giving your password.
You should give:
```

sudo apt-get install gimp

```
then type in your password (nothing will be shown, not even asterisks (*))
and hit enter.

An easier way would be to use the Ubuntu Software Center :)
