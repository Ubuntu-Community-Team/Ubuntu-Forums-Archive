### Post by kane77 on 2009-04-30
If you have computer where cd/dvd rom works (or you have an iso image) you can boot into ubuntu (either livecd or real system) and run System -> Administration -> USB Startup Disk Creator which will create bootable disk for you.. [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
