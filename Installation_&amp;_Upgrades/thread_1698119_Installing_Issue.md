### Post by tommcd on 2011-03-01
If you are selecting a user name with capital (upper case) letters, try using a user name with only lower case letters. It should then allow you to proceed.
This is a known bug in the Ubutnu installer.

If that does not help, when you boot the Ubuntu CD, be sure to run the option to *Check CD for Defects*: [https://help.ubuntu.com/community/Installation/CDIntegrityCheck](https://help.ubuntu.com/community/Installation/CDIntegrityCheck)
You may need to hit the space bar after the Ubuntu live CD boots to get the check Cd for defects option. 
This will ensure that the CD is valid and nothing on it is corrupted.
Write back if you need more help.
