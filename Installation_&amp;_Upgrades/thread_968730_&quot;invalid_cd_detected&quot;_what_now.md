### Post by Partyboi2 on 2008-11-02
The md5sum hashes for intrepid can be found [[COLOR=Blue]here[/COLOR]]("http://ubuntu.mirrors.tds.net/releases/intrepid/MD5SUMS") make sure they match before burning. What speed did you burn the iso to disk at? I would recommend x4 or less if you have not already done so. Try another burn program like [[COLOR=Blue]imgburn[/COLOR]]("http://www.imgburn.com/") to burn the iso to disk and see how you go.
