### Post by billio on 2011-10-14
I have just upgraded from Ubuntu 10.10 to 11.04 on a Lenovo 3000 N100.

When the system starts it presents me with the logon screen. I can type the password, press return and open the desktop. However when the desktop is displayed there is no mouse pointer so I cannot click on anything. 

The keyboard works. I can use the arrow keys and return to select icons on the desktop, but the few isons I have icons are only for individual files.

The mouse pointer is visible on the login screen, but it cannot be moved by the trackpad or the USB mouse attached to the computer. This prevents me from logging to a different desktop or safe mode.

Please can someone suggest a way forward ?.

### Post by billio on 2011-10-18
Well one can't just sit around and wait for an answer. So ...

Obtain cross-head screwdriver.
Switch laptop off, remove power cable, turn laptop over, remove disk drive cover, remove disk drive, insert drive into spare USB interface, connect to the desktop computer, copy important files across to the desktop.

Replace disc drive, cover, turn over, plug in, switch on laptop, insert Ubuntu 11.10 disc in CD drive, boot and perform a clean install. Copy saved files across the network as required.

An alternative approach would be to buy a new disc drive.
