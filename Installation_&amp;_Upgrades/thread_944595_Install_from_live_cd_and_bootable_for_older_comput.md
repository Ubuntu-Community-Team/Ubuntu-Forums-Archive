### Post by cariboo on 2008-10-11
Ubuntu is to large to install on such a small hard drive, you may be better off installing Puppy or Damn Small Linux on it. the best way to do that would be to install the distribution on the hard drive when you have it in the enclosure, then install the hard drive back in the laptop.

To the previuos poster, one thing that really bugs me is people mixing up hard drive storage with memory. Think of it this way, your hard drive can be considered permanent storage, while ram can be considered temporary storage, memory is between your ears. :)
