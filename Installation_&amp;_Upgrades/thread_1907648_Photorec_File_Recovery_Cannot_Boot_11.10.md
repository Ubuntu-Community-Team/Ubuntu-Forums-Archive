### Post by mikewhatever on 2012-01-11
If you can still see the files when running from the live CD, just copy them over to some external storage. There is no need to use Photorec, boot from an hdd or reinstall anything.

I'd also advise looking at when the Disk Utility has to say about the state of the HDD. Your hard drive may be dead or dying.
