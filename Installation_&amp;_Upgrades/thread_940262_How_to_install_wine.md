### Post by Partyboi2 on 2008-10-06
Double click on the wine_1.1.5~winehq0~ubuntu~8.04-0ubuntu1_i386.deb and this will start the installer. You don't need to extract it.

You can also install wine by add/remove (Applications>Add/Remove) or by opening a terminal and typing
```
sudo apt-get install wine
```
