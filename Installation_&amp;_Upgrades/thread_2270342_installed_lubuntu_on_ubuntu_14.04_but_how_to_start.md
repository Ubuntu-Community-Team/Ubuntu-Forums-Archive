### Post by bapoumba on 2015-03-22
What package did you install ? lubuntu-desktop is the one you should have. Freeing space when you add a flavor upon another one is not trivial, some files are in common some are not. Best is to fresh install the flavor. Do you have separate /home and data partitions ? In any case, back up everything you'd be crying for if you'd loose, and backup the very important stuff on 2 different media/location.
