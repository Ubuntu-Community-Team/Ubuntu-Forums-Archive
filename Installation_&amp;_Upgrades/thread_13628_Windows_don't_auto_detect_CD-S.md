### Post by spartas on 2005-02-01
The only way to get ubuntu on to a machine initially is through the CD drive at boot.  You may want to try putting the CD in the drive while in ME to make sure the CD is even recognized by the drive.

Just make sure the CD is already in the drive when it's booted up (e.g. You're not putting the CD in while it's booting, after it's already searched the CD drive and found nothing).  The easiest way to ensure this is to test both cases at the same time by booting into ME, insert the CD and make sure it is at least recognized (doesn't have to autoplay) then leave it in the drive and reboot.

Other than that, there's not much more I can suggest, other than double-checking that the system is set to boot off the CD drive first.
