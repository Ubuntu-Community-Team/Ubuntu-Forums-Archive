### Post by audiomick on 2009-11-12
Hi.
You need to to get an .iso and make a CD out of it. The instructions are to be found at Ubuntu.

Any Ubuntu variation will offer you the choice during the installation between installing next to the existing windows ( or whatever other operating system may be on the computer) or removing everything and only installing Ubuntu.

You need to decide if you want to put a 32 bit or a 64 bit version on.
I use 64 bit, which I am happy with, but in all fairness it needs to be said that there are still occasional _minor_ annoyances, like the odd plugin for firefox that wont work and such
