### Post by lukeiamyourfather on 2010-12-09
Almost always the drivers needed are already part of the kernel or in kernel modules included with Ubuntu. Rare and specialized hardware can be difficult to get working but common hardware like your's is not an issue. Install the Ubuntu system first and then start looking for drivers if something isn't working. If everything works then don't worry about it and continue on. ;)
