### Post by bojzi on 2006-06-19
There's always Automatix ([http://ubuntuforums.org/showthread.php?t=190025)](http://ubuntuforums.org/showthread.php?t=190025)). It's a very nice click and go install script!

### Post by bukwirm on 2006-06-19
Go to the Synaptic Package Manager (System > Administration > Synaptic). It allows you to install packages fairly easily - just click on the checkbox.

To use xine as a backend for the Totem media player, search for the package 'totem-xine' and install it.

Macromedia Flash Player, I belive must be installed from macromedia's website.
