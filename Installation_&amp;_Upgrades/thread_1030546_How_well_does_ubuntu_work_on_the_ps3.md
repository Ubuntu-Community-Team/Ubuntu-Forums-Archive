### Post by jbullfrog76 on 2009-01-07
I just (finally) got a PS3 and have been looking into Linux options for it.  Since you're familiar with Ubuntu, you might want to check out psubuntu.com, which seems to have a lot of information.

Apparently, there are versions of Ubuntu, Kubuntu, and Xubuntu for the PS3.

Also, you can have linux on the PS3 and still have the ability to play games.  To set it up, you have to create a partition for linux, which wipes the hard drive (and any save game data).  So be sure to back up your save game data somewhere before you repartition the drive.  

I haven't installed linux yet (since I only got the PS3 yesterday evening), so I can't tell you much more.  Good luck!
