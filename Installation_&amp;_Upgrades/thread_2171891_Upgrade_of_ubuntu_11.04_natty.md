### Post by QIII on 2013-09-02
Hello and welcome to the Forums!

There is an "approved" way to upgrade from EOL versions, but I really recommend against it.  Too much can go wrong and problems can compound -- leaving you with a useless system.

For reference, the process is detailed [here]("https://help.ubuntu.com/community/EOLUpgrades").

What I would recommend instead, however, is that you back up all of your important data and do a fresh install of a currently supported version.  If you have a separate /home partition, this can be done without disturbing it.  But back up anyway.

If you reinstall at this point, I would go with 12.04 (Precise), which is a Long Term Support (LTS) version.  If you install the current release, you will have to upgrade at least once before the next LTS version is released.

Best wishes!
