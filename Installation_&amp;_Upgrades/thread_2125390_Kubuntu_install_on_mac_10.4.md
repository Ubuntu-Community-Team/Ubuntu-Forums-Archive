### Post by Lars Noodén on 2013-03-14
What model of hardware do you have, specifically?  If it is the older (and somewhat better) hardware then it is using the PPC processor and you will need to find the appropriate disk image.  If you have PPC hardware, then look for the disk images labeled [powerpc](https://wiki.ubuntu.com/PowerPCDownloads/).
