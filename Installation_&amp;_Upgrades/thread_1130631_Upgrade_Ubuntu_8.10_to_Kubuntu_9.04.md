### Post by tommcd on 2009-04-20
This is an interesting question. I'm not sure if you can tell update manager to go from Ubuntu to Kubuntu. Anyway, this page discusses various ways to upgrade to 9.04:
[https://help.ubuntu.com/community/JauntyUpgrades](https://help.ubuntu.com/community/JauntyUpgrades)
When the upgrade finishes you can just install the Kubuntu desktop:
[http://www.psychocats.net/ubuntu/kde](http://www.psychocats.net/ubuntu/kde)
I would first download an iso of Kubuntu 9.04. That way if things get screwed up, you can just do a clean install of Kubuntu 9.04.
Personally, I would favor a clean install over an upgrade. Since your /home is on a separate partition all your data will be safe.
