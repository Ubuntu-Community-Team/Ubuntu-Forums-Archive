### Post by AlexDudko on 2012-10-20
A new installation is required. You can upgrade Ubuntu to the next version only or to the next LTS version if you use LTS (for example, Ubuntu 10.04 which is LTS could be upgraded directly to Ubuntu 12.04 LTS, but Ubuntu 10.10 could be upgraded only to Ubuntu 11.04). Since the next version 9.10 has reached its EOL (End of Life) and your version is 9.04 there's no option to upgrade any more.
