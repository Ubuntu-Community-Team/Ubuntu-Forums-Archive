If your goal is to move data between the computers, then you are fine.  Just put your data files on the external drive and your in.

If you know that the other computer does not have the applications that you need, then you have a problem.  Windows XP has a database called the registry where most if not all programs put all of their important settings.  The only way to get the entries into that registry reliably is to install the applications on that computer.  You need to bring the install executables with you or the application CD if you have it to install the programs on the other computer.  If you install the apps on your computer and use the external drive as the place to put the programs, you only have 1/2 of what you need.  When those applications fire up and ask the registry for the stored values in the registry, the Mexican registry will have no information available.

If you try to install XP on the external drive, you will have the registry in the right place but then you have a foundation problem.  Unless the computer in Mexico is an exact hardware duplicate to your current computer, XP may or may not be able to start because it is trying to run the wrong drivers, dll files, etc.

Recommendation:
Confirm whether the Mexican computer has the apps you need first.  
Put all of your data that you will be working with on the external drive.
Once you know what apps are available in Mexico, gather those CD's or download the install apps and put those on the external hard drive.
When you get to Mexico, install the apps and go to work.
