### Post by albertran on 2006-10-30
After upgrading to Edgy, everything seems to work fine, EXCEPT that I don't seem to have access to the file system under a GUI, except for my own home directory.  Nothing else shows up on a GUI, no matter what file manager I use.  They all just show an inaccessible Root, with Home and Media subfolders.  No /etc, no /root, no /var, nothing.

I'm sure I'm not the only one with this problem, has someone already found a solution?

---

### Post by albertran on 2006-10-30
Anyway, figured it out.  For some reason passing understanding, Edgy puts every single folder on / except for Home and Media in .hidden.

Should have been the first place I looked.

Other than that, everything seems to be working fine.
