### Post by lisati on 2007-12-05
There a a couple of possibilities. To start the ball rolling here are some ideas:
[LIST=1]
[*]Sometimes a "Bad burn" will let you boot from the disk but will get stuck at some point. Use the "Check CD for defects" option on the first page to check this (if it shows)
[*]Some hardware combinations have trouble with the "Live CD" version of the installer and give the illusion of getting stuck when it's just taking a super-long time to get on with the job. It is possible to download an "alternate" text-based installer disk which might work better.
[/LIST]

EDIT: Some of the more technically-aware users might be able to quiz you on your particular hardware and offer other suggestions that will avoid having to get hold of replacement disks. Good luck!
