### Post by jonrkc on 2005-05-23
Sorry I don't have an Intellimouse to try out, but I would suggest if you could plug in a different mouse (borrowed, maybe) and see if it works better--buying a replacement mouse might be cheaper than spending time on the problem for you.  

Another thing I've tried before, when my (Logitech optical wireless) mouse wouldn't work, under Mandrake Linux, was to switch from the USB port to the PS2 port using the adapter plug that came with the mouse. Then it worked.  A later release of Mandrake allowed the mouse to work on USB.  And when I added a cheap USB hub, all the USB devices worked better.

Hope you get your problem fixed soon.
