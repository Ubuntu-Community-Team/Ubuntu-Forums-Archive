### Post by guiverc on 2024-09-25
I agree with u/nu29914.

First I'd want to know the Lubuntu ISO you're using, as release details let us know what *software stack* you're using, which is details we can compare with your provided hardware details. You've not provided that basic detail.

The ISO I mention is useful as some releases had updated installers put on them, which is why I mentioned that; but release detail is the starting point.  As Linux Mint booted; you can contrast the *software stack* on that system with the Lubuntu ISO and know why they differed in that regard, alas you didn't provide any specifics (*booting and using system*). As for installers, Linux Mint & Lubuntu differ.

I'll provide the Lubuntu manual link for Lubuntu 24.04 LTS and the Installation page - [https://manual.lubuntu.me/lts/1/1.3/installation.html](https://manual.lubuntu.me/lts/1/1.3/installation.html)

You'll note reference is made to the SAFE GRAPHICS option which can be required for hardware using NVIDIA for example.   For older hardware however, the important bit is the release & ISO details (*as Lubuntu ISOs for a release can install with different stacks; default selected by ISO used to install**; for older hardware I generally recommend using the GA kernel stack** if its an option for your unstated release*).
