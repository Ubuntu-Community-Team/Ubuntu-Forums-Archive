### Post by az on 2005-05-10
Never mix repositories.  Stick with ubuntu repositories.

No, debian does not use the same kernel configuration as ubuntu so you will not find the ubuntu kernel headers in debian!

The package name is linus-headers and it should be on your cd as well as in main.
