### Post by Mighty_Joe on 2009-10-26
Make sure you are using the Sun version of Java.  Ubuntu used to use [GCJ]("http://gcc.gnu.org/java/") which is a very slow and incomplete implementation.  Now the default is [OpenJDK]("http://openjdk.java.net/"), which is an open-sourced version of Sun Java.  I haven't tried it to compare it with Sun Java.  
If you have Sun Java, you may want to look at your video drivers.
