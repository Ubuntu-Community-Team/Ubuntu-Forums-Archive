### Post by wpshooter on 2008-11-14
Did you actually install the O/S to the hard drive or are you still running the O/S from the CD ?

You MUST click on "**INSTALL**" once you get up to the Ubuntu desktop (the O/S is **NOT** installed to the hard drive simply by choosing "**INSTALL**" from the initial Ubuntu boot screen).

Thanks.
