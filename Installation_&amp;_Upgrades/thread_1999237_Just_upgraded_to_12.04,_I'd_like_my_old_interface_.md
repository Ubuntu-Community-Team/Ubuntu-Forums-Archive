### Post by oldfred on 2012-06-07
In 12.04 that is gnome-panel. It is based on gnome3 not the old now obsolete gnome2. It is close, some things have moved around a bit, but not much more than some upgrades.

gnome3 clasic
sudo apt-get install gnome-panel
On login screen click on cog-wheel/star and choose 

or:
Open the Ubuntu Software Centre and search for &#8216;gnome-panel&#8217;, hit install and log out.
To move/add/edit panels or applets press Alt+Right Click
