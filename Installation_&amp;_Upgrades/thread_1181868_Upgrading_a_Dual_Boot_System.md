### Post by Idefix82 on 2009-06-08
When you install the newer version, it will automatically detect the OSs already installed and configure GRUB accordingly. The only question is, why do you want to install 8.10? If you want an LTS version, you should go for 8.04. If you want the newest, go for 9.04.
