### Post by gordintoronto on 2009-09-29
I did this a month ago.  Evolution has a "backup settings" action under the File tab.  That got me a file containing a couple of hundred megabytes.  After transfering the file to my new system, I did a "restore settings" and I was up and running: email worked and all my old messages and contacts were present.

I don't know what the versions were.  I went from Gutsy to Jaunty.

I actually copied the file to a shared folder on a Windows computer and retreived it from there.
