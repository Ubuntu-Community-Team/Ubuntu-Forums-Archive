### Post by mcduck on 2006-01-04
no problem, install LAME with synaptic. I think you may have to enable universe/multiverse repositories to do that if you haven't done it already. Have a look at Ubuntu 5.10 Starter Guide in gnome's help..

If you want to rip some CD's, I suggest installing Grip.

edit. sorry, you wanted a decoder, not encoder. mpg123 and gstreamer-plugins are what you want, and you'll get them with apt-get or synaptic. There are instructions how to install multimedia codecs in the same guide..
