> End state is to update to 10.10 but I don't know if I can skip the un-installed updates to 9.10

No, you have to update from 9.10 > 10.04 > 10.10.  If you skip 10.04, then your system will end up broken, and a clean install will be necessary. Make sure that all your current system is fully updated first before upgrading.
