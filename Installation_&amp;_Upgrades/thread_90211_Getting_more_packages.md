### Post by Kyral on 2005-11-14
And don't mix packages from Debian unless you have to.

Basically if you want something, run through this checklist

If its in the Ubuntu Repos, install from there (This should be about 95% of the stuff what with Main, Restricted, Universe, Multiverse, and Backports)

If not, go to the program's website and look for .deb files (Priority Ubuntu then Debian) and download that :D
