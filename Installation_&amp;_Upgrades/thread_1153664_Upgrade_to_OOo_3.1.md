### Post by mistrynitesh on 2009-05-09
Hello people!

As you all must have known, OpenOffice.org has released version 3.1 of its productivity suite. And going by the experts' reviews, it seems worth trying.

I want to continue using Ubuntu 8.04 on my computer and do not expect the Ubuntu guys to backport the latest OOo to 8.04. So I have downloaded the packages from [OpenOffice.org]("http://download.openoffice.org/").

The question is, whether is there any way to upgrade my current OOo installation with the latest one? The [wiki page]("http://wiki.services.openoffice.org/wiki/Documentation/FAQ/Installation/How_can_I_upgrade_an_older_version_of_OpenOffice.org_to_a_newer_one%3F") of OOo site says no. So the option remaining with me is to remove the existing version and install the latest one.

These are the steps that I am going to follow to acheive my objective...
1. Start my package manager, go to the line that says "OpenOffice.org-core"
2. Click on the icon besides it and click "Mark for Complete Removal"
3. Click 'Mark' , 'Apply' , 'Apply' and wait for all the packages to be removed.
4. Now follow the instructions for installation as suggested on this [wiki page]("http://wiki.services.openoffice.org/wiki/Documentation/Administration_Guide/Linux") of OOo site.

Please confirm the above or is there any other better way of doing this?

Have a nice day...
