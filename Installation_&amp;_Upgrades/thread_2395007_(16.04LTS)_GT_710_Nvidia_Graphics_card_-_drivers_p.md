### Post by #&amp;thj^% on 2018-06-25
Great thanks. All looks good there. ;)
the drivers from an added ppa seem to help in this case.
```
sudo add-apt-repository ppa:graphics-drivers/ppa
```
followed with:
```
sudo apt update && sudo apt full-upgrade
```
and 
```
sudo apt install nvidia-384
```
and reboot to load any new changes made. and fingers crossed. :)
**EDIT:I should mention** to revert any changes with any added PPA **_(If needed)_** use ppa-purge via:
install with:
```
sudo apt-get install ppa-purge
```
in our case here clean up with:
```
sudo apt-add-repository --remove ppa:graphics-drivers/ppa
```
And Substitute "ppa:graphics-drivers/ppa" with the correct named PPA
