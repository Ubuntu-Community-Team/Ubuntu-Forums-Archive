### Post by arkham on 2006-08-16
The slowness is probably the result of the low amount of RAM in the machine - in order to run a Live CD, there is usually the need for a RAM Disk (a chunk of RAM reserved and treated like a disk) and in order for apps to run quickly and not be swapped in/out all the time you need a large chunk of free RAM besides this disk.

192MB is pretty much the minimum required to run the Live CD, but as we all know, the minimum will not be enough to run something well.  Once you have actually installed Ubuntu on the HD you will find it far more responsive, but the Live CD will at least give you an idea of the functionality available.......
