### Post by tommcd on 2009-08-01
If you are burning the CD from Windows, you may want to try Infra Recorder:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Burn at the slowest possible speed, as has already been suggested. Then boot up the CD and choose the option "Check disc for defects". If it passes the test, then proceed with the installation.
