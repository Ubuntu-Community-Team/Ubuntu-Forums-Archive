drakedog - if you're using the Visual Basic modules in Access 2003, you're actually using VBA (Visual Basic for Applications), rather than Visual Basic.  The two are not the same, despite many similarities, so running a version of Visual Basic probably won't give you what you're after.

I agree with other comments that dual-booting is likely to be the easiest solution.
