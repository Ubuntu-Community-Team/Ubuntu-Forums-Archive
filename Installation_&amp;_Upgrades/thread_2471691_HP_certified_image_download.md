HP Support's Linux FAQ: [https://support.hp.com/us-en/document/c00269588](https://support.hp.com/us-en/document/c00269588)

Quote from there:
> 
 [h=2]How do I reinstall Ubuntu?[/h]  First, back up all your personal files and data to an  external storage device, if possible. Then choose one of the following  options to reinstall Ubuntu:
  
[LIST]
[*] Visit [www.ubuntu.com]("http://www.ubuntu.com/") (in English) to download a free image of the operating system. See the [Ubuntu Reinstallation]("https://help.ubuntu.com/community/UbuntuReinstallation") Web page (in English) for more information.

[*] If your computer came with Ubuntu and is still under warranty, [Contact HP Support]("http://www.hp.com/go/contacthp") for a reinstallation disc.
[/LIST]
