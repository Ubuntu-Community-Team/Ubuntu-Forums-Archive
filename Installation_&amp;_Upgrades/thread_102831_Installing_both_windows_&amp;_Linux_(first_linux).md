### Post by anarchist_hippy on 2005-12-12
I'm new here... I'm working ubuntu about 5-6 days.

I install Ubuntu under windows XP SP2. Worked succesfully. But I formatted (:rolleyes: I broked Ubuntu). After I install Ubuntu, I can't install windows long time. But now, I can... How? Here it is... 

First, boot from Ubuntu CD. On partition manager, create a partition named Fat32, primary, directory "/windows". Then create a partition named ex2, secondary, directory "/". Then install Ubuntu (DON'T FORGET TO INSTALL OS CHOOSING). When restart computer instert windows Cd and install it to FAT32 partition with convert to NTFS.

After, when restart, instert ubuntu CD. Install it normally to "ex2" part of HDD. 

Reboot. 

Here is the OS choosing window. Choose Ubuntu. And continue to install...

Reboot.

Here is the 2 "workable" OS in your HDD ;)

I hope this will be useful...
