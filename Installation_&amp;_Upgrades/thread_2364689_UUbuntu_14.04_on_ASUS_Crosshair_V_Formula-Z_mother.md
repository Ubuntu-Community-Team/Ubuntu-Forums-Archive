### Post by edwardsah3 on 2017-06-26
I have been fighting an intermittent problem on this motherbard for about a year. The system would crash with low-level errors about the sata drive. After all else failed, I read the instructions and found that one could specify an IDE, rather than AHCI configuration for the SATA driver. So far, that has been the fix. i just thought I would post this in case others have had similar difficulties.
