### Post by taurus on 2008-02-26
Did you burn the ISO image that you downloaded to a DVD as an image file?  In other words, you are not supposed to just copy that one large ISO file to a DVD.  You have to use a burning software on your machine and burn it as an image file so you can boot from it.

Also, make sure you go into the BIOS and have CDROM as the first bootable device.
