### Post by mc_bin on 2011-02-25
since you have a 60GB HDD and 1GB RAM, 

/boot - 500MB 
/SWAP - 1024MB (swap usually set the same as your RAM size)
/usr - 20GB (for generous install)
/home - rest (music video etc. usually the biggest)
hope this helps 

mc_bin
