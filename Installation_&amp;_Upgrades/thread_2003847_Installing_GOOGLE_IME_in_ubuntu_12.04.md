### Post by arpanchavdaeng on 2013-06-08
There is no support of Google IME for linux till now but there is other way to do it.

1. Install Chrome browser or Chromium in your ubuntu.

2.Use Google IME Chrome Extension .
Here is link : [https://chrome.google.com/webstore/detail/google-input-tools-by-goo/mclkkofklkfljcocdinagocijmpgbhab?hl=en](https://chrome.google.com/webstore/detail/google-input-tools-by-goo/mclkkofklkfljcocdinagocijmpgbhab?hl=en)

3.Its done!! Add your language and write in chrome. Use stackedit (Search in Web store) for Pleasant writing experience with markdown or you may use Google Docs offline.


Other method is to use Ibus in ubuntu.
