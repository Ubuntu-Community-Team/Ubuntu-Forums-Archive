### Post by aBaldrich on 2010-09-20
There two possible problems:

1) The .iso you downloaded could be corrupt. You can verify the file with [MD5sum]("https://help.ubuntu.com/community/HowToMD5SUM").
2) The burning process could have failed, resulting in a corrupt CD. You can check the CD integrity in the LiveCD Menu.
