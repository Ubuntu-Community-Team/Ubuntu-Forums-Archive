### Post by grahammechanical on 2022-04-14
Are you wanting to install Ubuntu? During the install process files will be download over the internet. How is your machine connected to the router/hub/modem? If the computer is connected by Ethernet cable then select the Realtex PCI express Ethernet adapter. If you are connected to the router/hub/modem by Wifi then select the Qualcomm Atheros wireless network adapter.

All this assumes that the router/modem/hub is itself connected to an Internet Service Provider.
