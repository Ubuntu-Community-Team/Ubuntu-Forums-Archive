### Post by sh4d0w808 on 2008-03-24
Hello brueneboy,

Yes, it is possible with [Ubuntu Alternate CD]("http://releases.ubuntu.com/7.10/ubuntu-7.10-alternate-i386.iso"). U can find a very similar installation about Xubuntu in the [Full Circle Magazine]("http://fullcirclemagazine.org/"), Issue 3. The point where U have to take more attention is the partitioning step. If U are not familiar in the Command Line Interface partitioning, I recommend to create the space for Linux under Windows or using [Gparted LiveCD]("http://gparted-livecd.tuxfamily.org/"). If U are working on your real system, then I also recommend to backup all of your important datas!

Another possibility to try the Alternate CD: [VirtualBox]("http://www.virtualbox.org/") or [VmWare Server]("http://www.vmware.com/products/server/") - all of these are free to use (at least for non-commercial purposes). Using any virtualization suite, U can get experience with CLI-based partitioning.
