### Post by Mark Phelps on 2013-05-02
IF it's a flashing little white line, then it is trying to load -- and that can take a LONG time.  When it gets further into the loading process, that flashing cursor will disappear.  So, you need to give it some time (several minutes, at least) to see if it gets beyond the flashing cursor.
