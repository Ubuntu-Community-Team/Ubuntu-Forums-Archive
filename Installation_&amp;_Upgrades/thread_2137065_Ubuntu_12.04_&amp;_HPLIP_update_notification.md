### Post by tgalati4 on 2013-04-19
HPLIP has a built-in update checker.  You would only need to update if you have a brand-new HP printer with features that are not available in your current *hp-toolbox*.  

I'm running 12.10 and it gives me a button that takes me to the HPLIP website for manual installation:  [http://hplipopensource.com/hplip-web/install/manual/index.html](http://hplipopensource.com/hplip-web/install/manual/index.html)

In the Configuration tab==> Preferences==> System Tray Icon, there is a checkbox to turn off notification with a message box showing the current version and the update available:

Currently HPLIP-3.12.6 version is installed.
Latest HPLIP-3.13.4 version is available for installation
