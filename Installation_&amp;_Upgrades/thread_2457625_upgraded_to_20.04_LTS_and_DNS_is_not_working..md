### Post by SeijiSensei on 2021-02-06
The easiest way to specify a DNS server in new versions of Ubuntu running systemd-resolved is to add the server's address to the "DNS" line in /etc/systemd/resolved.conf and restart systemd-resolved with
```
sudo systemctl restart systemd-resolved
```
As far as I know resolvconf has been deprecated and is not used by 20.04.

If you add servers to /etc/resolv.conf, they will be lost when the system is next booted. resolv.conf is rebuilt from scratch at every reboot. You need to edit resolved.conf to specify static DNS servers.
