### Post by Insane_Homer on 2008-04-25
Yes it's safe.

It will install to a directory within the NTFS partition you destignate, then it adds a single line to the boot.ini file to call GRUB.

No Vista data will be affected. Just free space required, it's IS that simple.

Good Luck!
