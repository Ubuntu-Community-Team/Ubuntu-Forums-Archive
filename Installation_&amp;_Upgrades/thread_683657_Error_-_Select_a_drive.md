### Post by PmDematagoda on 2008-01-31
When you boot the Live CD press the "Check CD for defects" so as to ensure that the Live CD has been burnt properly.

If you get any errors during the CD check then you will need to burn a new Live CD, also make sure that you burn the CD at a speed of 4x or less, a speed higher than 4x can corrupt the CD causing problems like the one you are currently facing.
