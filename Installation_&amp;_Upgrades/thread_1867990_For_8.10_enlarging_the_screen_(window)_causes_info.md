### Post by Old_Grey_Wolf on 2011-10-23
8.10 has not been supported for a long time. There are no security patches for it after April 2010, and you have to use the old-releases repository for it if you need to install some software. I would suggest downloading and installing 10.04 or newer. 10.04 is supported until April 2013.
