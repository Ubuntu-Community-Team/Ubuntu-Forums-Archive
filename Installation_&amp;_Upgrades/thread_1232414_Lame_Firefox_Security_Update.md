### Post by darkpixel on 2009-08-05
The firefox security update this morning was totally lame.

When I close firefox, it asks if I want to save my tabs.  I say 'yes', because I usually have at least 30-40 tabs open between several windows.

After installing the security update, firefox started behaving strangely, and the notification icon said I need to restart firefox.

So I hit File->Exit.

It didn't ask to save, it just closed.
When I re-opened it, I got the Ubuntu welcome page, rather than my expected 30-40 tabs.

Same thing happened at work.

On a third computer, I quit firefox before installing the update and it worked as expected.

If your update is going to screw up a running app, maybe it should recommend that the user close the app before causing them to lose data.
