### Post by ChompTheMan on 2008-06-13
This article should help you: [http://linuxfreedom-technoshaun.blogspot.com/2008/06/how-to-make-fully-customized-live-kde.html]("http://linuxfreedom-technoshaun.blogspot.com/2008/06/how-to-make-fully-customized-live-kde.html")
