### Post by aysiu on 2009-06-27
I assume you mean netbooks when you say "mini."

It's kind of a tricky question.

Some netbooks come with some kind of Ubuntu installation on them (the Dell Minis and the HP Minis), but they're usually slightly older versions (8.04 instead of 9.04) and tweaked to work out of the box with some of the proprietary components Dell or HP includes (Broadcom wireless, for example). So if you're okay with the stock installation, I'd recommend getting Ubuntu preinstalled on a netbook. And even if you do install 9.04 or 9.10 later, the fixes for any problems that come up are usually quick ones.

And it's the same deal for any of the other netbooks (even the ones that don't come with Linux preinstalled). Usually you can install Ubuntu or some variant (EeeBuntu for the Asus Eee PCs, for example) and have either everything work out of the box or almost everything (maybe a function key won't work or you'll have to edit some configuration file to get your microphone working).

Your best bet is probably to Google the name of the mini netbook you want and the word *Ubuntu*. For example, *samsung nc10 ubuntu* or *eee pc 1000he ubuntu*.
