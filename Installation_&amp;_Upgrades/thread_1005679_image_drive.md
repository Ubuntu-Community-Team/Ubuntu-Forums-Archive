### Post by Mark Phelps on 2008-12-09
Since the elonex webbook can be ordered with either XP or Linux, it will obviously "take" XP.  But -- and this is important -- you can NOT install XP with the harddrive attached to a different machine, load the harddrive into the elonex, and expect it to work.  You will have to install XP with the harddrive resident in the elonex.
