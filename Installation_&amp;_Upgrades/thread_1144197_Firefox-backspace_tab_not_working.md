### Post by gooligan on 2009-04-30
Try setting browser.backspace_action to 0
If you are not familiar with how to do this,
1. go to "about:config"
2. Click on "I'll be careful..."
3. Search for backspace
4. Set the value to 0

That should set your backspace to go back in history
