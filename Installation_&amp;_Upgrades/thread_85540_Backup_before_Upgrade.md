### Post by Xian on 2005-11-03
As far as user prefs, settings, and info, if you will just back up the contents of your /home/username directory you should be well protected. I don't use any of the mozilla products but AFAIK all user configs should be stored in your own home folder.

```
$ cd /home/username
$ ls -a
```
