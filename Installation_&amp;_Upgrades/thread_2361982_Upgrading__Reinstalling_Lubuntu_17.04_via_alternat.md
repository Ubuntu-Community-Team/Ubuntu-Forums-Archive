### Post by howefield on 2017-05-23
You can't upgrade from one version of (x)Ubuntu to another using the live media.

It would be a fresh install and yes, once you have created the Live media you would follow the same steps as per the standard installer except for the fact it is considerably more "cumbersome" using the alternate installer, particularly the partitioning stage. Not at all difficult but possibly easier to get wrong. Do you have a specific reason for using the Alternate installer ?
