### Post by aysiu on 2006-09-29
I'm not sure exactly what you're asking, but the "hidden" partition is usually quite small, and I would guess it's not hidden at all from the Ubuntu installer.

You can create new partitions, too, by resizing existing partitions, so nothing will be erased (you'd be an idiot not to back up your installation first, though, including important files).
