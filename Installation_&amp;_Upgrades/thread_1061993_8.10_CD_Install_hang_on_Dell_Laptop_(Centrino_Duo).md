### Post by taurus on 2009-02-06
1.  Run the checksum to check the integrity of the ISO image before you burn it.

2.  Burn it at a slow speed like 4x.

3.  At the initial screen, run the check cd for defects to make sure everything is okay.

3.a.  And try not to run the LiveCD (desktop CD) on a machine with less than 386MB of RAM.
