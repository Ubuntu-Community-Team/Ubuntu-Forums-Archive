Hi!
Xubuntu uses the XFCE window manager and Ubuntu uses Gnome. XFCE requires less memory, so it's better for low-end PCs. If you have an older system (like I do) or if you prefer speed, Xubuntu is the way to go in the "-buntu family".
Also, Xubuntu has some lighter apps in terms of memory usage. For example: Abiword for word processing and Gnumeric for spreadsheets instead of OpenOffice. Of course, you can add any application you like from the repositories, if you have good internet connection.
Greetings!
