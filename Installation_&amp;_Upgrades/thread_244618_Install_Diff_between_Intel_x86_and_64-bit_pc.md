### Post by nyinge on 2006-08-26
1) Intel and AMD have designed the chip so that they are backward compatible with 32-bit apps.  (This is essential for a smooth transition to 64-bit realm.)
2) Your PC is performing in 32-bit mode if you install the 32-bit OS.  Performance wise, it is minimal at this point in time, however, you avoid the issues of incompatibilities such as drivers.
