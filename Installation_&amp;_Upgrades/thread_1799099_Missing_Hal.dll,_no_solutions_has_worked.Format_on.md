To repair your XP install there 2 ways to do it.

1.  After selecting Windows from the Grub Menu immediately start tapping the F8-key. This will bring up an options screen that EG continue Windows boot Boot into Safe-Mode and Repair using the recovery console. It has been a few years since I had to use it so am not sure of the commands to use.

2.  Grab your XP-disc, stick it in the CD drive and reboot.
   At the welcome screen choose your language and keyboard layout       EG English  United Kingdom. Click on Continue. 
   In the next screen choose Repair Your Computer and follow the prompts to allow XP to fix everything it needs to.


After the repairs everything should be back to normal however you might need to update Grub from a LiveCD.

Hope this is helpful.
