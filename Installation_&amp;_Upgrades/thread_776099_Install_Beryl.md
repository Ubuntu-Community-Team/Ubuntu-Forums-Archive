### Post by lswest on 2008-04-30
first off, you installed beryl?  therefore there is no "script" you require.  Secondly, there is no more "beryl" that program merged with compiz to form compiz-fusion, which comes pre-installed with 7.10 and up, and the only driver/apps required would be the compizconfig-settings-manager and the drivers for your graphics card.  So, i'm not entirely sure what you're asking.

*EDIT*ah, you edited it, well the above still applies, all you require to run compiz-fusion is graphics drivers, and for more advanced effects you require ccsm which can be installed with ```
sudo apt-get install compizconfig-settings-manager
```
