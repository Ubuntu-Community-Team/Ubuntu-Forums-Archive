### Post by Tatty on 2008-04-27
Defrag your windows partition at least 3 times.  The defragger that comes with windows isnt very good so it needs some extra work.

When you boot with the ubuntu live CD go to System->Administration->Partition Editor This opens gparted - a graphical partition editor.  Its pretty straightforward from here.
