### Post by Karlchen on 2013-06-08
Hello, subhasis83.

Create a bootable pendrive holding the Ubuntu 13.04 32-bit live system. - [Making a bootable USB of Ubuntu]("http://www.psychocats.net/ubuntu/usb").
Boot your netbook from this pendrive and launch the Ubuntu 13.04 live system.
Thus you will be able to answer your own question.
