### Post by sudodus on 2013-11-21
You can try the [One Button Installer, 'OBI']("http://ubuntuforums.org/showthread.php?t=2172971")                 which uses tarballs with 'already installed systems'. Ubuntu based flavours re-spins and distros are portable between computers. It might be an easy way for you to make make a tarball of your system (using the OBI) and then install from that tarball into another computer (using the OBI). And yes, the OBI works in text mode.

*Edit*: another alternative is the Ubuntu ***mini.iso***, which is also a text mode installer.
