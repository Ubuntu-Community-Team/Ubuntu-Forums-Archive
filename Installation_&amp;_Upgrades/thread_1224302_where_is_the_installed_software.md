### Post by ChadMMc on 2009-07-27
go into a terminal and type
```
whereis avg
```(Put whatever the name of the app is in place of avg if needed)

Actually you could install the clam anti-virus system instead if you wanted.
  search for clam in the add/remove application or do a search for clam in synaptic package manager.
