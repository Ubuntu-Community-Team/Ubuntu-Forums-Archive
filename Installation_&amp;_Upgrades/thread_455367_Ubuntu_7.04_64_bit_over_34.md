### Post by vigleik on 2007-05-26
Most of the time everything will be the same, 32 bit or 64 bit, with the 64 bit version just a little bit faster. A few things are more difficult to do on the 64 bit version though. For example, installing flash for Firefox is a bit more difficult. So if you don't want the hassle you might as well just install the 32 bit version.
