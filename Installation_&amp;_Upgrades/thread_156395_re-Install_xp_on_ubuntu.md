### Post by amohanty on 2006-04-07
1. Download, verify and burn live CD: its a life saver
2. Boot up using that
3. Use **gparted** to make a small dinky (~5GB should be more than enough) for windows
4. Install Windows
5. [Restore Grub]("https://wiki.ubuntu.com/RecoveringUbuntuAfterInstallingWindows")

HTH
