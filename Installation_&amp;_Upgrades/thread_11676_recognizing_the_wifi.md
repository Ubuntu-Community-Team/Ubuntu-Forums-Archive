### Post by SubZero_AK on 2005-04-05
Currently there is no Linux support for your particular wireless card.  In order to get it to work you have to use NDISWRAPPER.  That program uses the windows driver to make your card work under Linux.  I don't have time to explain how to make it work, but if you search on ndiswrapper you will come up with the solution.
