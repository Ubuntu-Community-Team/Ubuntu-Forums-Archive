### Post by drs305 on 2011-05-16
Are both OSs on the same drive? If they are on different drives you may need to change the boot order in BIOS so the Ubuntu drive boots first.

You can also try holding down the SHIFT key to see if the Grub menu appears. 

Otherwise, please click on the "BIS" link in my signature line. Download and run the boot info script, then post the contents of RESULTS.txt.
