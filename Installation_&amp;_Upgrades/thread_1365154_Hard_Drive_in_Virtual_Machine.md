### Post by SecretCode on 2009-12-29
Do you want to run an OS installed on another physical partition as a virtual machine while you're running another OS?

VirtualBox can do it, but it takes careful setup - and watch out for windows activation problems.

Read this: [http://www.virtualbox.org/manual/UserManual.html#rawdisk](http://www.virtualbox.org/manual/UserManual.html#rawdisk)

And read a lot of threads at forums.virtualbox.org - you'll have to do some searching.
