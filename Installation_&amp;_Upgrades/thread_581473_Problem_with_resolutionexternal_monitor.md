### Post by jrasmussen0 on 2007-10-19
You may want to try to use the restricted (ie. ATI) driver for that video card, fglrx.  There is a gui program called 'restricted-manager' that should be able to help you configure everything.  If you do use fglrx, you will need to disable the composite setting in the configuration file.
