### Post by Quackers on 2011-01-05
Have you tried ```
sudo update-grub
``` in a terminal? This will configure grub.cfg and will hopefully pick up your Windows installation.
If this does not work, please go to synaptic package manager and make sure that os-prober is installed. If it is not you will need to install it and then run the above command again.
