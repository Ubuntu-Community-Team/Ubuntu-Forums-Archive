### Post by Partyboi2 on 2009-05-10
> Setting up mythtv-common (0.21.0+fixes19961-0ubuntu8) ...
mkdir: cannot create directory `/root/.mythtv': File existsYou could try moving your current directory out of the way, so that it creates a new directory
```
 sudo mv /root/.mythtv /root/.mythtv.old
```
```
sudo dpkg --configure -a
```
