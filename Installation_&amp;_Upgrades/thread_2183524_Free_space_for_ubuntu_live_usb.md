### Post by ajgreeny on 2013-10-25
Everything goes into the persistence file of your live system, but if you really need a lot of space (more than 4GB) you can use a separate partition.
Details at [http://ubuntuforums.org/showthread.php?t=2165496](http://ubuntuforums.org/showthread.php?t=2165496)

---

### Post by sudodus on 2013-10-25
You can also consider making a normal installed system (but installed in a USB drive). This has the advantage of being more stable and it is possible to update the kernel, which is not possible in a persistent live system).
