### Post by snowpine on 2010-01-11
Ubuntu Studio also has a real time kernel (linux-rt), new artwork & themes, and a few other default settings. "Regular" Ubuntu can be converted to Studio by installing the linux-rt and ubuntustudio-* packages from Synaptic Package Manager.

If you are installing fresh, maybe Ubuntu Studio is easier, because everything's preconfigured and on one DVD.

[http://ubuntustudio.org/](http://ubuntustudio.org/)
