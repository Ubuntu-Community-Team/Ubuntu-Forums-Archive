### Post by miscellanyous on 2015-05-23
I not 100% sure but I think it's solved. The problem was with cryptsetup blocking recognition of a secondary partition where Ubuntu 11.10 was installed.

This did the trick:
```
~$ sudo apt-get remove cryptsetup
~$ sudo dpkg --configure -a
```

and reboot.
