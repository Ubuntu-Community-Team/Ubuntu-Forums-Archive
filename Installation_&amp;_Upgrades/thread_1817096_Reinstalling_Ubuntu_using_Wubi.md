### Post by garvinrick4 on 2011-08-02
Download the .iso file from Ubuntu. Burn this .iso file to a CD Disk. While in windows stick
the CD in the tray and open it on your desktop and click on WUBI. It will ask you your user
name and password and how big you want Ubuntu install.(at least 10 gig) click install and
in 15 minutes or so it is done. Now at boot menu will have choice of Windows or Ubuntu.
All inside of Windows, will be a folder right next to Users in C: drive called Ubuntu.
[Download Ubuntu | Ubuntu]("http://www.ubuntu.com/getubuntu/download")
