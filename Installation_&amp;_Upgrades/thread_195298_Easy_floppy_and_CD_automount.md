### Post by wawax on 2006-06-12
Automagic mount of floppies and cds can be easily done on ubuntu. I hadn't found it in repository, but very good instructions about installing submount you can find on that page:
[English version]("http://gentoo-wiki.com/HOWTO_Submount")
[Polish version (automatyczne montowanie dyskietek i cd)]("http://wiki.gentoo.pl/wiki/Submount")

Notice (you have to do this before compiling submount):
You have to install first linux-source, linux-headers and all compiling stuff (the easiest way is to install kde-dev packet - it will install KDE devel, but also everything you need to compiling applications).

Regards, Slawek.

PS. I know, manual mount/umount is what Ubuntusers like, but no one else. ;)
