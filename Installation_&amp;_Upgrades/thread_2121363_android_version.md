### Post by grahammechanical on 2013-03-01
Please do not mind me saying so, but you misread that advert. Recently Canonical made available a few developer previews of images and source code. [http://developer.ubuntu.com/2013/02/releasing-the-ubuntu-touch-developer-preview-and-sdk-alpha/](http://developer.ubuntu.com/2013/02/releasing-the-ubuntu-touch-developer-preview-and-sdk-alpha/) These are highly experimental and are not complete. The hope is that some manufacturers will release phones and tablets running Ubuntu by October or later. We should not expect to get Ubuntu images for our phones and tablets in the same way we get Ubuntu desktop images today. Please read this [http://www.jonobacon.org/2013/02/21/five-ubuntu-touch-facts/](http://www.jonobacon.org/2013/02/21/five-ubuntu-touch-facts/) There is a section of this forum called Mobile Technology Discussions. That is the place to go.
