### Post by Mark Phelps on 2009-07-30
There's a distinct possibility that this is a problem with the 64-bit video driver.  Since you have only 2GB of RAM, you're gaining little if anything using 64 bit instead of 32 bit.  As a test, suggest downloading, burning, and booting from a 32 bit LiveCD to see if you get better graphics support.
