### Post by pipe2 on 2015-11-28
Hello,

I'm trying to install uefi version of 15.10, but it always freezes on some step of installation (somewhere from 'pick language dialog' to 'choose disk partition layout'). This laptop is MSI GE62 with two disks SSD and HDD. SSD contains win10 and an empty partition for linux. Also it has hybrid Intel and Nvidia graphics.

I tried the following:
making usb with ubuntu usb creator
making usb with universal usb installer
installing from a live cd
adding nomodeset kernel parameter
installing from live ubuntu system ('try without installing' option) live system works fine, but freezes when I try to install
disabling Intel RST (followed a guide from Intel)
disabling fast boot (also have win 10 installed)
in bios disabled quick boot and changed to ide from ahci mode

Thanks in advance ;)

---

### Post by pipe2 on 2015-11-28
The problem was with the Intel Speedstep. Had to disable it in the bios.
Then updated bios and now nothing freezes even with it turned on.
