### Post by mikewhatever on 2007-02-11
Try the Alternate install CD, which is a text mode installer. It can be downloaded from Ubuntu download page. Hopefully, you'd be able to install. You'd probably still have graphic issues on boot, but then, it is possible to deal with them. The guide below will give you some tips on using the Alternate installer. If you do not have Windows, simply disregard that info.
