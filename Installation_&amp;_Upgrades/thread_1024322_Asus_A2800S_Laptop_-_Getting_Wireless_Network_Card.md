### Post by bmach on 2008-12-28
This is a post to help out anyone else who is attempting to install Ubuntu on an Asus A2800S laptop. After a fresh Ubuntu install you'll find that the wireless network card doesn't work and isn't shown as a device under the network configuration.

Good news is that it's easy to get it working. All you have to do is following the steps outlined by Joe2Shoe in this thread:

[http://ubuntuforums.org/showthread.php?t=990216]("http://ubuntuforums.org/showthread.php?t=990216")

This will also apply for anyone who has the following Network Controller showing up from the command 

```
lspci

```

```
Network controller: Broadcom Corporation BCM4306 802.11b/g Wireless LAN Controller
```

I hope this helps.
