### Post by confused57 on 2008-01-19
You need at least 320 Mb ram to install using the Gutsy 7.10 Desktop cd, so you could either install more ram or try installing with the Xubuntu alternate cd:
[http://mirror.anl.gov/pub/ubuntu-iso/DVDs/xubuntu/7.10/release/](http://mirror.anl.gov/pub/ubuntu-iso/DVDs/xubuntu/7.10/release/)

You could probably install with the Gutsy alternate cd, but your system would be slow with the amount of ram you have.

Here's a link for installing with the alternate installer:
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)
