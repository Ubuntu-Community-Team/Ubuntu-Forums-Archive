### Post by ymo on 2008-09-26
According to [http://www.ubuntu.com/products/WhatIsUbuntu/desktopedition](http://www.ubuntu.com/products/WhatIsUbuntu/desktopedition) the 8.04 Alternate CD can be used to install on a system with only 256MB. The same should be true for 7.10 but 8.04 has the advantage of being LTS so should keep you going much longer.

Alternatively, Xubuntu might be more suitable for your system.

If you don't upgrade, AFAIK you will stop getting patches.
