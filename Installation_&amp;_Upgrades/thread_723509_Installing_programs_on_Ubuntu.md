### Post by Calash on 2008-03-13
I don't have an Ubuntu install infront of me, so my menu names may be off a bit...sorry about that.

To start, the best place to go is the Add/Remove programs area in your main menu.  It is a simplified version of the package manager that will download the applications and everything they need from Internet servers called Repositories.  You can search for what you want and click Install do download it.

Once you get your feet wet you can try Synaptic, under the Administation menu area.  This is a more advanced package manager.  As with before you can search and install what you want, but the install is a bit less user friendly, IMHO.

Then there are the command line utilities, but that is a bit more advanced.

If a program is not in the Repositories, you may be able to download it from a web site.  If the file has a .deb extension this is a package file and should install with little problem.  If not, you will need to do some work, Alien to convert it or compile it from source.  Both these are more advanced, so stick with the easy ways first until you feel comfortable.

Also, make sure you only download source/packages from trusted sources.  The Repositories are all good, at least the ones that are included by default.


Edit:  Or just use the link posted above....they will explain it much better than I did :)
