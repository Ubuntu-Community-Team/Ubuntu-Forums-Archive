$HOME/Downloads/unetbootin-linux-568

run

enter sudo password to give access to have it beat up ur usb stick.

locate your ubuntu iso i got mine @ [http://releases.ubuntu.com/precise/](http://releases.ubuntu.com/precise/)

and put it on ur cd key.  if u dl it with seamonkey or firefox its probably located @ $HOME/Downloads also....

reboot your computer and select in bios / boot order USB then cd then hd to boot from the usb rather than hard drive when usb stick is plugged in.
