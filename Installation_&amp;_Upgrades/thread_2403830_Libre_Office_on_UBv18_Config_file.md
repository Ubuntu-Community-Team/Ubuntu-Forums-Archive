### Post by ajgreeny on 2018-10-16
Where is this LibreConfig file you speak of?

In Ubuntu the configuration for Libreoffice is all in ~/.config/libreoffice/4/user; is that what you are asking about?

As for the icon missing top left, I think this is probably because of the move from unity in 16.04 to gnome in 18.04.
However, if you open a terminal with Ctrl+Alt+T an icon for it will appear in the favourites, ie the icons in the left hand launcher-bar; if you right click on that icon you will see an option to **Add to favourites** or maybe **Lock to favourites**; I don't use Ubuntu so I'm not sure about the wording of the context menu from the right click.
