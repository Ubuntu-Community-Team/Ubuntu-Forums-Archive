### Post by MattCarp on 2006-06-04
I have a Pentium III machine with 768MB, but at the moment on Ubuntu 6.06, I'm running 

Firefox
OpenOffice Calc
Thunderbird e-mail
the system monitor
XMMS playing an MP3
a terminal session
Nautilus file browser

 and am at 206 MB in use.

I can't say that I've used more than 256MB.

I'd say ubuntu does a pretty good job with memory, but could do better.
