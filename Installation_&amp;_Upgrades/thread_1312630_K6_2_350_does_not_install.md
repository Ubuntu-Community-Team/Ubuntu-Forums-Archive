I think it's because you only have 256 MB of Ram, the bare minimum that Ubuntu needs to run. Of this 256 MB a number of MB's may be taken by your graphics card and there you are: too little working space for Ubuntu.

You might do 2 things (probably more, but I can now think of these two :) ):

1. Get some more memory. Memory is cheap now, but the memory you need possibly hard to find and also not so cheap. Go to a good computer store.

2. You might try a lighter version, Xubuntu. Don't know how much memory it needs, might be only 128 MB's.
