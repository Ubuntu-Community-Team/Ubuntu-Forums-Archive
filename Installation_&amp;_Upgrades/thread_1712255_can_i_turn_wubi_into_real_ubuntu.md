### Post by Rubi1200 on 2011-03-22
Hi,
you have 2 options:

1. migrate the current Wubi install to a dedicated partition on the drive and dual-boot with Windows

2. make a fresh install, choosing the option to erase and use entire disk from the Ubuntu CD

Are you sure you want to get rid of Windows?

Personally, I recommend you dual-boot for some time before making the decision to use only Ubuntu. After all, you never know if you might need Windows for work, gaming, or the like.

If you need more information on either of the options I mentioned, feel free to ask.
