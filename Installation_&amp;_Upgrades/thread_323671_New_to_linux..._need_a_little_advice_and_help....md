### Post by aysiu on 2006-12-22
The lab computers would have to be able to boot from USB. I'm not sure they can do that.

Why not just use the Ubuntu live CD (Desktop CD)? It's probably easier to create a persistent home on a USB key than to boot from USB (since not all modern computers support that).

By the way, in this situation, Knoppix would much better suit your needs than Ubuntu.
