### Post by kerry_s on 2007-03-30
Partial updates usually mean not all the packages are in the repo yet. You can just install whats there and get the rest later. I would wait awhile and check the forums to see if there are any problems if your not use to fixing your system if something go's wrong.  Feisty is still in testing and things can still go wrong even though it is mostly stable. Updates do not need to be applied right away and since your asking, i recommend you wait till others have tested the updates first, if you don't see anything wrong with the updates on the forum then go ahead. You been warned.
