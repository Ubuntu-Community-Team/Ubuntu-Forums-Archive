### Post by mhgsys on 2009-03-29
If you don't remember your username or password., 
Reboot in recovery mode, you'll be root there in console.

in recovery mode 
go to your home dir to find your username

```

cd /home

```

once there typ
```

ls

```


then when you know your username

```


passwd yourusername

```

Enter new UNIX password: 
Retype new UNIX password: 

voila, username and password back.
