Hi,

I already have 2 empty partitions that used to have Ubuntu on them. There's one clean formatted as ext3, and there's one formatted as swap (haven't re-formatted this one yet).

How would I go about installing Ubuntu to the existing partition?

Thanks.

---

### Post by Partyboi2 on 2008-12-13
When you are at the partitioning stage of the install choose "Manual" then set the ext3 partition (the empty one) mount point to / and set the /swap partition to /swap then process with the install.
