### Post by dantrevino on 2007-11-03
I assume you're running feisty.

Gutsy has the 2.4 series by default.  For feisty, you'll need to wait for someone to package it.  When/if they do, it'll likely be in 'backports'.  If so, you would need to enable "unsupported updates" on the updates tab in synaptic.
