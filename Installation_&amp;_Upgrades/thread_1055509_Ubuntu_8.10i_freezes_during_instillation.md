### Post by avtolle on 2009-01-30
I presume you are using the Live CD. Try installing with the "Safe Graphics" mode (an option on the initial menu, as I recall; or, you may have to hit F4 (other options) to get this choice.

You might also consider using the alternate install CD for this; it uses a text-based installer, and can work where the Live CD doesn't as the result of graphics card issues.
