### Post by lol on 2006-02-10
If you really need to latest version and cannot find a .deb package for it, then yes, you will have to download an archive (src or binary) and install from it.

If you need to install from sources, I would recommend using checkinstall, which build a nice little package that is easy to uninstall later.
