Finally found the solution!!
You have to add a line to this file
>  /etc/modprobe.d/blacklist
you need to
 > sudo gedit
and the line is
> blacklist intel_agp
Just add it at the end of the file and save it, you'll be able to boot with your fx5200 enabled by bios, and your ubuntu will actually boot.
