### Post by magic8ball on 2011-08-06
Sounds like you need to change the boot device order on your computer so that the optical drive is first.  During startup on my laptop, I can either press F1 to get into BIOS (and ultimately change the boot order) or F12 to get directly to a boot order selection list.  Your computer will have some similar option.

If you've changed the boot order and it still doesn't work, maybe the CD your trying to use is bad.

Good Luck!
