### Post by paddy1978 on 2006-08-26
Perhaps there's some kind of hardware fault - in which case it would affect linux and windows?

I've had random errors before caused by faulty RAM. On the live CD, I think one of the options is "memory test". It takes a while to run, but should pick up any problem caused by this.

Also, are you 100% sure the CD has burned correctly? I've got a copy of Knoppix that I burned on my desktop - it works fine on that but crashes with a CD error on my laptop! (Instructions on how to check CD [here]("https://help.ubuntu.com/community/HowToMD5SUM?highlight=%28md5%29"))
