### Post by sebastianabate on 2010-03-20
Actually the original problem was solved... you just have a new one  ](*,)

It seems that you have a corrupted status file, try this

1) Backup the current file

sudo cp /var/lib/dpkg/status /var/lib/dpkg/status.orig

2) Restore the backup file (yes, Ubuntu keeps a backup of this file by its own, isn't it great?)

sudo cp /var/lib/dpkg/status-old /var/lib/dpkg/status

3) Reinstall the problematic dependency (just in case)

sudo apt-get install --reinstall libltdl7

4) Check if everything is ok

sudo apt-get -f install
