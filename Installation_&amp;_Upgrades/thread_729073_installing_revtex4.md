### Post by eltadcirad1 on 2008-06-11
The best way (in Hardy Heron) is to install the texlive-publishers package,
either with synaptic, or by typing

sudo apt-get install texlive-publishers

at the command prompt. This will install revtex 4 amongst similar packages from other publishers.
