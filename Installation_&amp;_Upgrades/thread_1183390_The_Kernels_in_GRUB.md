### Post by Shibblet on 2009-06-10
I just downloaded the Ubuntu Update for Kernel 2.6.28-13,
Now I have that kernel and 2.6.28-11 in the Grub menu.

How do I get rid of the old one?  Not just from the menu, but remove the old kernel as well.

---

### Post by cariboo on 2009-06-10
Go to System-->Administration-->Synaptic Package Manager, and search for the kernel you want to get rid of:

```
kernael-image-2.6.28-11
```

then mark it for complete removal. I normally keep the previous kernel version for a couple of days, just in case there are any problems.
