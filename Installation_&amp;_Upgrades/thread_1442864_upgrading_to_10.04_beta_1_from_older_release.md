### Post by Mark Phelps on 2010-03-31
10.04 is still in Beta and will be for a while yet.  You should NOT be updating a "production" system (i.e., one you rely on daily to do useful work) to any Beta version.

One major reason is that there is no "roll-back" capability to go back to the previous, stable version of Ubuntu.

If you insist on using 10.04 while it's in Beta, you should be posting your questions to the correct forum: Development & Programming, Lucid Lynx Testing.

I will be asking the Mods to move this thread there.  Look there for any future reponses.
