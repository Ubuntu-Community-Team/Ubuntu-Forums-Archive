### Post by dannyboy79 on 2007-02-08
any dual-boot thread or tutorial will suffice. when you talk about dual booting, the "other" os is really irrelevant. you should try out the live cd first to see what of your hardware works "out of the box" before you go an install it. so download the live cd as well as the alternate install cd. use the live cd to test it out, then if you want to install it, use the alternate cd to install it, this will give you options for installing grub in the correct place so that you can still boot into windows or linux. good luck.
