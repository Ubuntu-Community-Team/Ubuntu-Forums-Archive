### Post by linuxbz on 2007-10-24
Try just disconnecting the ethernet cable (or disabling wireless) before starting the install. Especially if you are installing from DVD, you can install everything you want and then let the individual programs be upgraded later.  You'll need to setup networking later, then, but that isn't too hard.
