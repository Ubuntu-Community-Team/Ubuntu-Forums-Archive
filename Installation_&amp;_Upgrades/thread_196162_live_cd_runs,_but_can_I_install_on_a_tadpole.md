### Post by DarthMandeep on 2006-06-13
I'm not familiar with OS X yet (I'll fix that soon as my sister will be getting a MacBook for college) but in Windows and Linux when you burn an iso file you often have to tell your burner program that it is an iso file and has to be burned in a specific way. CD/DVD burner programs often have an option like "Burn from ISO" that has to be selected.

I Googled and found this, hope it helps. [http://www.uab.edu/it/question-answer/os_osx/burn_iso/index.html](http://www.uab.edu/it/question-answer/os_osx/burn_iso/index.html)

If the liveCD works, then you should be able to install. If the laptop will not boot from the disc and it has a bios option to do so, then chances are the disc was burned improperly.
