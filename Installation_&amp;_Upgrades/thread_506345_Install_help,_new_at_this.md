Perhaps your laptop has very low specs, that's why it's so laggy, because Ubuntu will sit on your laptos RAM and then load everything from it, so if you have a 32 RAM laptop it'll be laggy.

I recommend you to use the alternate install method. When the boot image shows up, choose "install in text mode" and it'll start installing, although if your laptop have really low specs try the "install a server" option, then when it finnishes you'll endup in text mode, loging and write "sudo apt-get install xubuntu-desktop" and that'll install a light desktop so you don't have a bloated laptop.
