**Edit**: No offense, but I don't really like that Ubuntu book. I read it in a bookstore, and these are the three objections I have to it:

1. It claims to be for both the novice and the professional. I'm an advanced novice, and most of the stuff in that book is too basic for me. I can't imagine a professional would really get anything out of it.

2. It spreads FUD about Windows. I like Ubuntu better than Windows. I know Windows has its flaws, but lying that Windows is always unstable and Ubuntu is always stable is just... lying. One example the book uses is the mouse cursor freezing, which, funnily enough, has happened to me in Ubuntu but *not* in Windows XP.

3. It doesn't encourage best practice, as it instructs people to use *sudo* for graphical applications instead of *gksudo* or *kdesu*.
