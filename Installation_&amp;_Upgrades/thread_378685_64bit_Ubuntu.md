### Post by mixium on 2007-03-07
Hi,

I'm not certain, but I would recommend against it. There is just too many things that could go wrong. I would back up all files, hidden and otherwise in your home folder to discs and then do a clean install. Also, before you start using some applications on your new system, such as mail clients that have stored email locally, you need to read up on migrating that data before you start them up for the first time.

A well planned migration and data backup is smart. Take a few days and don't be in a hurry.

I can tell you that I am extremely happy that I am running in 64 bit mode. It is so much smoother and peppier, well worth any trouble.
