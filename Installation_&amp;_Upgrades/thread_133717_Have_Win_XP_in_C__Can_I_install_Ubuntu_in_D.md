### Post by linuxden on 2006-02-20
If D/: is a second partition, or a separate drive, then yes you can install Ubutu or any other linux distro on it... It wont remove or harm any data on the windows partition as long as you dont overwrite the windows partition...

Make sure you know the different sizes of c: and d: so that you recognise them when they are named hda and hdb... (or hda1 and hda2 if you only have 1 drive...)

### Post by xequence on 2006-02-20
Linux doesent use the letter names for drives, but yes, you can install to that drive/partition that windows calls D.

But if you have anything on that drive (such as music, documents, etc) you will need to delete them. Or make another partition for ubuntu... A partition is basically splitting one physical drive into more then one "virtual" drive.
