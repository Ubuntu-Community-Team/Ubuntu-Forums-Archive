### Post by Miguel on 2005-11-14
Hi wilford,

You can definitely install Ubuntu on your friend's PC. It will surely work (unless it has some very esotheric ultra bleeding edge hardware). 

If you have a fast connection and spare CD's, you can try downloading the Ubuntu 5.10 live CD. Put it on the CD-ROM drive and tell your BIOS to boot from the CD. Now, the CD will try to detect all the hardware and will load a fully working Linux desktop... without installing a single file. If your friend likes it and feels comfortable, you can download the install CD and install it (both CD's are practically identical, except for the booting process)

About the drivers: In Linux, the drivers are in the Kernel. Unless you have very esotheric hardware that needs propietary (non-free as in freedom) drivers, chances are the drivers you need are all included on the cd... Actually, I would bet that Ubuntu comes with more drivers than WinXP.

Note: ATi (9200 and later) and nVidia (all) Graphic cards need propietary drivers for good hardware 3d acceleration. However, fully working free drivers are provided with the CD. If you need 3d acceleration, there are some good HOWTO's in this forum.

Hope it helps,

Miguel

PS: With a live CD and a USB pen drive, you can also recover all your friend's files. Without installing any other OS.
