### Post by motapa on 2009-07-13
Well, you can run sudo apt-get install mplayer, or sudo apt-get install vlc. These media players can play many proprietary media formats such as mp3 or wma. you can continue to download mplayer codecs found at [http://www.mplayerhq.hu/MPlayer/releases/codecs/all-20071007.tar.bz2](http://www.mplayerhq.hu/MPlayer/releases/codecs/all-20071007.tar.bz2).

This tutorial provides a good tutorial.
[http://www.ubuntugeek.com/install-mplayer-and-multimedia-codecs-libdvdcss2w32codecsw64codecs-in-ubuntu-810-intrepid-ibex.html](http://www.ubuntugeek.com/install-mplayer-and-multimedia-codecs-libdvdcss2w32codecsw64codecs-in-ubuntu-810-intrepid-ibex.html)
