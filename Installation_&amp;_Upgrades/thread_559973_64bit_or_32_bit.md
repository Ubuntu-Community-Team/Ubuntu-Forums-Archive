Unless you are actually using some software that utilizes the 64-bit capability (and, to date, there are very few for desktop users), I would recommend the 32-bit version. The 64-bit version is still a bit of a "work in progress" and hopefully will mature by the time 64-bit software actually comes into play.

My core2duo E6600 just flies using the 32-bit, I doubt that at this point the 64-bit would provide any noticeable performance boost to compensate for the issues that it currently brings with it.
