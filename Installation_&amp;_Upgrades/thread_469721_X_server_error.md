### Post by thomasaaron on 2007-06-10
When you ran sudo dpkg-reconfigure, did you select nVidia as the graphics driver?

From a terminal, enter:

sudo gedit /etc/X11/xorg.conf

There is a line in this file that specifies your graphics driver.
Make sure it is set to "nVidia"
Then reboot.

If that doesn't work, try setting it to "nv"
This won't be as good as the actual nVidia driver, but it will at least allow you to get back into GDM.
Let me know how this works. I'll subscribe to this thread, and if you can't get it figured out, I'll send you a sample xorg.conf file on Monday.

Best,
Tom

### Post by thomasaaron on 2007-06-10
I see.
Here is how to install the nvidia driver.

From a terminal, enter:
sudo apt-get install nvidia-glx

Once it is finished installing, enter:
sudo nvidia-glx-config enable

Then run dpkg-reconfigure (and select the nVidia option).
sudo dpkg-reconfigure xserver-xorg

Reboot your computer and (with a little luck ;) gdm should start.

Best,
Tom
