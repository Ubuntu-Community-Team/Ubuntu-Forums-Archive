### Post by jeffus_il on 2008-01-29
I have seen some problems like yours on the forum, and a solution was to repartition the hard disk using the "Partition Editor" (gparted) from the LiveCd before running "Install". Initialize the disk by creating a new partition table(sometimes called a disk label) and set up your partitions, and make the filesystem.
