### Post by yogesh.girikumar on 2010-02-12
Hi,
       You can try a quick fix for this in VLC:

       When the video is playing,
       
1. Select Video menu

2. Navigate to "Deinterlace" 

3. Select "discard" or "mean"       Deinterlacing sometimes solves such issues. If it still doesn't work, you can check if it's a driver problem or a problem with the display settings. If it's a display settings problem, check this link out: [http://psubuntu.com/wiki/SetupMonitor](http://psubuntu.com/wiki/SetupMonitor)
