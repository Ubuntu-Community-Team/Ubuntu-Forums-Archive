### Post by Andrew.Z on 2009-11-01
> **kennethtb said:**
> Bleachbit froze half way through requiring a shutdown

Did you enable all the options?  If you enable 'wipe free disk space,' it can easily take 15 minutes or *much* more to finish!  That option really is very slow by its nature (not a bug).

If that does not help, install the [latest BleachBit](http://bleachbit.sourceforge.net) (currently version 0.7.0) from the .debs at the home page.  Ubuntu 9.04 has an *ancient* version 0.3.1, and Ubuntu 9.10's version is somewhat old (BleachBit 0.6.3).

If that does not help either, then try to isolate the problem to a single option.
