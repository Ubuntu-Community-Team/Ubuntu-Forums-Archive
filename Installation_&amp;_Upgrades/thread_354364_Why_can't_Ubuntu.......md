### Post by aysiu on 2007-02-05
It has to do with ideology.

And there are many arguments about this, actually, since one of the things Ubuntu is/was planning to do (I don't know what the outcome ended up being) had to do with installing Nvidia drivers by default.

Theoretically, Ubuntu is dedicated to open source (not proprietary) software and only one application per task. It does have a DVD version but doesn't popularize that version.

Bottom line: it's a *decision* (like it or not), not a deficiency. If Ubuntu *wanted* to be Sabayon, it could, but it has decided not to be, for better or worse.
