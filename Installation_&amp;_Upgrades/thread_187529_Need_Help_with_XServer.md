**I found this on another forum:**

My Via Unichrome driver is working under Dapper. I am running the 386 version on an AMD Athlon 64 3000+ with the k7 kernel and my graphics card is the Via K8M800 from an ASUS K8V-MX. All I had to do was install the Via driver through apt-get:

### Post by Magoo69 on 2006-06-05
OK I've sorted this problem.  The kernel wont recognize the onboard video or an agp card (can't see anything for them in lspci), unless you specify "pci=conf1" in the boot loader configuration.

So all I did was edit the file /boot/grub/menu.lst and added 'pci=conf1' on the same line as the boot line to start ubuntu.

**This is specific for an Asus P5VDC-MX motherboard, but it may apply to some others as well?**
