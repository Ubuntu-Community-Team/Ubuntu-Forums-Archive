### Post by aysiu on 2006-02-04
Go to Applications > Accessories > Terminal and ```
sudo apt-get update
sudo apt-get install kubuntu-desktop
``` Log out.
Session > KDE
Log in.

Of course, if you are ever thinking of *un*installing Kubuntu later, you may want to copy and paste the list of packages that are to be installed.
