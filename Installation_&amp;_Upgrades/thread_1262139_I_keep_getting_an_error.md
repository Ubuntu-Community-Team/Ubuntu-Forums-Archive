### Post by QIII on 2009-09-09
Ah!  Scratch all this.  I didn't see "Wubi".  Wubi is not (IMHO) the way to go if you really want to use Ubuntu.  It's a great way to test it out, however.

So keep my original post in mind for the future...



Be darn careful with Vista if you are trying to dual boot!

Defrag twice and use VISTA to resize your partitions.  Don't do the resize with the LiveCD.  Let Vista decide where it is going to move stuff around to.

When you get to setting up your partitions in the Ubuntu install process, select (I may not be exact here) "Largest contiguous space".

Please read this thoroughly before you move forward:

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
