### Post by nalmeth on 2006-12-11
Dapper and Edgy are short for Dapper Drake and Edgy Eft.

Understandably this is confusing, but these are just nicknames for releases of Ubuntu.

Ubuntu has a 6-month release cycle. That means a new version is officially released every 6 months. Dapper Drake is the long-term support version, which means it's more stable, and is professionally supported for 3 years. 
Edgy Eft is the newest release, which pushes on more cutting edge, and sometimes unstable software. It is still quite stable, and many users use edgy just so they can have newer versions of their favorite software, with a little bit less hassle.

As mentioned, you have the lowest requirements for the Desktop install CD. Those requirements are based on how well the graphical installer will run on top of the liveCD environment. (The performance you are seeing with the liveCD is not what it will be like when installed).

If you can't wait for your extra RAM (which will be more than enough), I would suggest you download the Alternate install CD instead. 

Go here: [http://www.ubuntu.com/products/GetUbuntu/download?action=show&redirect=download](http://www.ubuntu.com/products/GetUbuntu/download?action=show&redirect=download)

Pick either Ubuntu 6.10 (Edgy) or 6.06 (Dapper). Pick your country, pick a mirror, and look for the Alternate install CD, which is a text based installer.

EDIT: I forgot you ordered the disc's, if you don't have a net connection, or another OS installed, you may have to wait for more RAM, or try the install again, giving it a LOT of time.

Good luck
