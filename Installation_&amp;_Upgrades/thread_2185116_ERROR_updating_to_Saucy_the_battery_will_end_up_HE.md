### Post by Baldrick_NZ on 2013-11-02
Yep, you've somehow installed the development version. 

If you want the latest stable version, you'll need to download 13.10 from the main Ubuntu web page and do a fresh install.. Either that or download the latest LTS (Long Term Support) version which is 12.04, and do a fresh install of that.

The development versions are just that. The next version in testing phase, and as such weird and not-so-wonderful things - such as what you're currently experiencing - happen.

Hope that helps.
