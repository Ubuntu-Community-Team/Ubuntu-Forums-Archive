### Post by Seeker84 on 2013-04-28
It is possible to have your data saved from startup to startup, you just need to set Persistent memory.  USB-creator lets you create 4 Gb of space, though I haven't tried to use all of it.  I did a quick test on my USB key, added a file to the Desktop then restarted, the file stayed there.

Hope this helps.

### Post by C.S.Cameron on 2013-04-28
A second alternative is to do a Full install to USB.
Most computers see a flash drive as just another hard drive.

If you are doing this for the first time I would suggest disabling the internal drive first.

An 8GB drive is large enough for this.
