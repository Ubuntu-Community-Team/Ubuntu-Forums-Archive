### Post by matt_symes on 2010-12-08
Hi

> I  think it is not possible to access the data from Ubuntu on a seperate  partition so they have to be on the same one if I am right.

No. You *can* :D access  data on different partitions and even on different drives using Ubuntu.

You can install Ubuntu on a separate partition than XP or even a separate drive drive and it will be able to see all the other drives and partitions on your PC.

It might need a small amount of configuration to set it up but people on this forum can help you out.

In any case, if you cannot boot into windows XP, i would back up all your data from that partition before installing another OS. You can do this from a Ubuntu LiveCD or from Puppy.
