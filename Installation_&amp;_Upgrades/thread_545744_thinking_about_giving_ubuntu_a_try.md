### Post by merlinus on 2007-09-08
NTFS is for windows.  ext3, ext2, reiserfs and a few others are for linux.

But yes, you can install ubuntu on your second hdd.  The installation will format the partitions correctly.

There are drivers you can install so windows and ubuntu can read and write to each others' partitions.

-merlin
