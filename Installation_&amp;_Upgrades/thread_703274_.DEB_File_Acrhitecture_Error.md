### Post by taurus on 2008-02-21
Assuming you have downloaded this one, openraider-0.1.0.20030512.tar.gz, and it was saved to ~/Desktop.

```
cd ~/Desktop
tar xvzf OpenRaider-0.1.0
make
sudo make install
```
And if you want to install it through synaptic or apt-get, then you need to edit /etc/apt/sources.list

```
gksudo gedit /etc/apt/sources.list
```
and add these three lines to the end of it.

```

# OpenRaider
deb http://openraider.sourceforge.net/debian/ ./
deb-src http://openraider.sourceforge.net/debian/ ./ 

```
Save it and run

```

sudo apt-get update
sudo apt-get install -u openraider

```
Please look at the README for more info.

```
cat README
```
