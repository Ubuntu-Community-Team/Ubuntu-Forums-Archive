### Post by reiki on 2007-04-16
If you're getting a BIOS error then it really doesn't have anything to do with Ubuntu. When you put the DVDRW in, did you just pull out the CDRW and install the DVDRW in its place using the same connections? If so, did you make sure the DVDRW drive was umpered teh same as the CDRW you replaced? For example, if the jumpers on the CDRW were set to "Slave" ... did you make sure the DVDRW is also set as a slave?

I just did this yesterday myself. Removed a CDRW and put a DVDRW in its place. My machine (BIOS) detected the drive and Ubuntu had no problem at all. I didn't have to do anything in Ubuntu.
