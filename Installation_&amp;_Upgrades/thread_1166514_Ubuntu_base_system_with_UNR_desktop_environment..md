### Post by snowpine on 2009-05-21
There would be little to no benefit to what you describe. UNR is not a "lightweight desktop environment." It is a heavy-weight, buggy, and unstable DE in my opinion.

What you probably read about was an Ubuntu minimal install plus a lightweight desktop environment or windows manager like icewm, fluxbox, openbox, etc. Here is one of many good tutorials on the topic (with screenshots): [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

(note: instead of burning a CD, you can use Unetbootin to "burn" the installer to a USB thumb drive: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/))

If you are just looking for "a faster version of Ubuntu" save yourself a lot of trouble and try CrunchBang: [http://crunchbanglinux.org/](http://crunchbanglinux.org/) Its creator did all the work of a minimal install for you.
