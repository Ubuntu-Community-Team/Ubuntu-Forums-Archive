### Post by dfreer on 2007-07-10
You might want to look into getting the ubuntu repositories on DVD or creating a local repository cache. You will never get the latest and greatest packages this way.

The problem with downloading from another computer is that most programs require other prograns or dependencies to install correctly, and you end up scurrying back and forth between the computer with internet and the one without. I don't really know of any distros that package all of the required dependencies in one nice package, and I imagine it would be hard to implement.
