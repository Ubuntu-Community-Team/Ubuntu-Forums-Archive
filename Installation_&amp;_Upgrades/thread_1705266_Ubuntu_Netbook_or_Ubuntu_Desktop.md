### Post by JRV on 2011-03-13
The netbook CD has both desktops available after it is installed.  After you click on your name on the login screen you can choose which desktop to use in the bottom panel.

The netbook CD does not have desktop effects if you use the standard Gnome desktop. So if you don't want the eye candy you only need to download one CD.

I prefer the standard Gnome desktop on my netbook.
