### Post by yo5qcd on 2011-01-01
I'm posting this because I've noticed many discussions "around the tail" concerning in fact this simple problem. 
Many Netbooks have a PCIe card reader, and the BIOS does not recognize it as bootable. So, the SDHC prepared to be used as an istallation media does not work. 
The solutions are three: 1.wait (often indefinitely) for a BIOS upgrade from the manufacturer. 2.Use an external CD and finally 3.spend a couple of bucks on a decent card reader and use it as an USB HDD.
This solution has proven to be most effective on my Aspire One AOD-260 supplied with Win7 Starter. I Installed Ubuntu Maverick alongside Winzoz from a SDHC using a Canyon card reader.
