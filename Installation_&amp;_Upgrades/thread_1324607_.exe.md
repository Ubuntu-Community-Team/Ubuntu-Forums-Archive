### Post by dhavalbbhatt on 2009-11-12
> **Bone Crusher said:**
> Can I install .exe files on xubuntu? I don't know which application to use for it.

The short answer would be "NO". You can't install .exe files in Ubuntu. Having said that, to install a .exe file, you will have to install wine (a windows emulation program) and install the exe program there.

### Post by Dude-PWB- on 2009-11-12
In the general sense, no. Exe's are a windows file, and this is linux.

However, depending on what you are trying to run, you can install wine from the repositories and run an exe through wine.

Go to [http://winehq.org](http://winehq.org) and check to see if the application you are trying to run is supported in wine.

### Post by MartinEve on 2009-11-12
Just to complicate matters, in some cases the answer is "yes".

If the application is written in the .NET framework, then these .exe files will be executable by using the Mono runtime.

As has already been mentioned, non-.NET executables *may* work under Wine, but this is speculative.
