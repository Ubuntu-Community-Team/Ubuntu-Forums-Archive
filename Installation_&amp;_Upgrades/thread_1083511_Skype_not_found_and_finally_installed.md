### Post by UBUminJ on 2009-03-01
I am using Ubuntu 8.10 and tried to install Skype

I followed
[https://help.ubuntu.com/community/Skype](https://help.ubuntu.com/community/Skype)
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

and updated the sources list with the Skype and the Medibuntu repository.
When trying to locate Skype package in the Synaptic Package manager, I don't find it. Also searching via Add&Remove doesn't locate it.

Then I downloaded Skype via their homepage:
[http://www.skype.com/go/getskype-linux-ubuntu](http://www.skype.com/go/getskype-linux-ubuntu)
and opened it - I get a message that the same package is available via a software channel.

Two days ago, I had a similar problem when installing Skype on a laptop. Suddenly, however, I got it. I don't remember that I did something different.

Finally I followed
[http://technical-itch.co.uk/2007/09/18/how-to-install-skype-on-ubuntu/](http://technical-itch.co.uk/2007/09/18/how-to-install-skype-on-ubuntu/)
and used
sudo aptitude install skype

This worked.
