### Post by oracle2b on 2010-06-15
Well since you have 4 gigs of ram you'll want to address all of it so you should install ubuntu 64-bit. Install 64 bit applications if there available and I think virtualbox has a 64-bit version.

Install [Ubuntu tweak]("http://ubuntu-tweak.com/") to make things easier for yoruself.
Codecs and common stuff could be installed from the software center or by following this [guide.]("https://help.ubuntu.com/community/Medibuntu") I use wine for office 2003,2007 and picasa 3.6 but I have a Vbox install of xp for other things. Check the [Wine app database]("http://appdb.winehq.org/") for compatible apps.

If flash becomes a issue settle for using ubuntu 32-bit but you can only access 3 gigs of ram.
