### Post by Keith_Helms on 2016-08-03
I doubt you'll find any single app that does most of what you listed.  I use

Audacious to play music.  I like it because it's simple.  It doesn't want to import all my audio files into a database.  It kind of strikes me as a successor to the old winamp player.

abcde with a little frontend script I wrote to rip CDs from the command line.  I used to use a gui program named grip a few years back but the developers stopped supporting it.

K3b to burn discs.

I don't copy audio files to different formats often, but ffmpeg will do that.

Naturally there are half a dozen alternatives to each of these in the repository.  If you'll post the specific features you'd like to have somebody will probably nominate a program that comes close to meeting them.
