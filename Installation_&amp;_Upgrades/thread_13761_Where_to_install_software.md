No laughing. In fact instead, how about a welcome!

First of all.....do you know that most software in Ubuntu doesn't have to be installed be hand?

Open up the program called Synaptic (under system configuration) and  hit reload. Then after it reloads, "search" (using the button) for thunderbird. Click on the white box to the far left on the line that says "mozilla-thunderbird" and click install. then click on the "apply" button (looks like checkmark) and the program will be downloaded and installed like magic!
 
If you want the newest Thunderbird, do this first:

[http://ubuntuguide.org/#extrarepositories](http://ubuntuguide.org/#extrarepositories)

Then hit reload and install thunderbird.

Once you get used to it, installing programs in Ubuntu will feel so easy that the "next, next, next' I agree, next" process of installing programs for windows will seem quite dated!

For future reference (or if you still want to install thunderbird the old fashioned way), you C: in windows is equivelent to your "/home/yourname/" directory in Linux. Anything install or put in the folder can be easily accessed by yourself. 

Also, I would take I look at the rest of the guide I posted if I was you- it will help you get many other commonly needed programs. 

Good luck!
