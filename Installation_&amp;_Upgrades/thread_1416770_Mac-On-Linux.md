Neal, IIRC, mac4lin is basically a theme to make your Gnome Desktop look and act like OSX.  Mac-On-Linux is an app that'll let you run OS9 or OSX in a window or full screen on your Linux powered PPC.

IIRC it was installed as a package, much the same as Basilisk, and you had to start it from the command line.  You could, of course, make a shorcut (alias) to it, but nobody seems to have made that part of the package. Kinda odd, I think, I mean, after all this is the Mac OS we're talking about, a Graphic, not CommandLine user interface...

It emulates a 603e processor, IIRC, so theoretically it can run BeOS natively, too.  I never knew that back then, but I have BeOS on a UMAX and find it interesting.  I'd like to run it on my 'new' eMac.
