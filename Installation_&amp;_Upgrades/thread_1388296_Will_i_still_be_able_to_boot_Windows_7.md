### Post by raymondh on 2010-01-23
Installing through WUBI means that Ubuntu will be a "file system" within windows.  It also means that Ubuntu cannot run unless windows (in this example) cannot boot.

This is a 'dated' [wubi-guide]("https://wiki.ubuntu.com/WubiGuide") ... but it should give you ideas/reference about a wubi install.  

So to answer your question ... yes, you will still be able to boot windows7 as you will be given a option to boot either windows or ubuntu.

Do research more about wubi and windows7 as I seem to recall that there are some issues windows7 and 9.10 (wubi-installed).

Good luck and have a working/tested back-up of your files.

Raymond
