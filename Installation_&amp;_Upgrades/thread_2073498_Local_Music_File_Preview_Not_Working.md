### Post by stevedude on 2012-11-15
Found the answer:

Go to the Music Lens, Select "Filter Results", Select "Sources", Select "Rhythmbox".

The previews work again. For those that might have uninstalled Rhythmbox, try selecting Banshee. If neither are installed, my guess is you have to have Rhythmbox [default music player for Ubuntu 12.10] installed in order for previews to work.
