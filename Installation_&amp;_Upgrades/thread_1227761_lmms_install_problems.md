### Post by Partyboi2 on 2009-07-31
Hi, you can try starting it from the terminal (Applications>Accessories>Terminal) with
```
lmms
```

Edit: You can also right click on the Desktop and choose to create a launcher and add
Name: Lmms
Command: lmms
Comment: blank

Another thing to check is "Main Menu" (System>Pref>Main Menu) that it is listed and that it has a tick next to it under Applications>Sound & Video
