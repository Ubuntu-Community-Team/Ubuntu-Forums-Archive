### Post by Zelut on 2006-02-04
Well you could do that.  You could put that drive in a cd-bootable computer and install it that way.  Linux doesn't complain about hardware changes in that situation the way windows would (big surprise huh?)

I had a machine that wouldn't boot to CD so I actually found an old floppy drive, installed it, upgraded the BIOS from the manufacturer & then installed from CD.  The upgrade provided cd-bootable function.  Just a thought.
