### Post by diverge on 2006-01-10
Yes, you can install via FireWire.

And, no, you probably will not be able to boot from it.  This is less to do with FireWire and more to do with BIOS issues that are just as applicable to USB installs.  For more info, read my recent post on [Booting to USB/FireWire Devices]("http://www.ubuntuforums.org/showthread.php?t=115164").

In the resource list should be information specifically related to installing/booting over FireWire.  Doing a general search along the lines of "Install FireWire" would also be a good place to figure out the proper installation sequence.  Unless you have a Mac, however, I advise against actually installing until a Supreme Ubuntu Bootdisk is compiled.
