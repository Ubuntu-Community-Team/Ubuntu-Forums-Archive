### Post by Herman on 2006-04-24
> is it safe to install ubuntu onto a cdrwYes, I use CD-RWs and CD-R disks and I like the RD-RW disks better for software, because they seem to be better quality generally, and if a burn doesn't pass a checksum test I can always just erase it and try again. I have had better luck with CD-RWs than CD-Rs anyway though. 
Also, when the software goes out of date, (is superceded by a newer version), you can use the same CR-RW disk again.
> i tired before but it was slow copyingI have a variety of different brands of CD-Rs and CD-RWs here, and most have printed on them a burn speed they claim to be capable of. Some of my wife's CD-Rs she uses for music have up to 52x printed on them. My good CD-RWs I use for software only claim to be good for 1x to 4x or up to 10x at the max, but I only ever burn software at the slowest speed possible 1x for best quality. It simply is not worth it to rush the job of recording the software to CD by a few minutes and risk maybe having all kinds of trouble with the software later. It only takes a few extra minutes.
> also is there a way to copy ubuntu setup files to the harddisk and install Jazzmuzic is correct> There is a "live" version of Ubuntu that runs entirely from a CDThat's right, and there is a 'Live' version and an 'install' version that will copy the set-up files (into the RAM first I think), and then install Ubuntu on your hard disk for you.
It's a good idea to try out the 'Live' version that runs from your CD-ROM drive first, and see if it works okay in your computer and you like it. When you decide you do like it, and your computer likes it too, then use the install disk to install it.
If you want to find out all about that, click on my left-hand sig link and there's a fourteen page website I made there that will tell you three different ways to install Ubuntu, plus lots of things you can do with it to get you started off. You should also see the Official Ubuntu Wiki, (there's a button for that at the top of every forum page).
And finally, I'd like to say 'welcome' and I hope you have as much fun and enjoy Ubuntu as much as the rest of us! Regards, Herman :-D
