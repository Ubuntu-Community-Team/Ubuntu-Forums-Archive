### Post by howefield on 2010-04-04
To learn how to add a repository...

[https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)

You can add the VLC repository to your software sources, using the instructions here...

[http://www.videolan.org/vlc/download-ubuntu.html](http://www.videolan.org/vlc/download-ubuntu.html)

This will keep your VLC up to date, likewise you can find a repository for KDE. 

Firefox should be taken care of via the official repositories.
