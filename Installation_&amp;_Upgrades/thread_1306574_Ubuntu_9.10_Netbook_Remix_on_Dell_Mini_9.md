### Post by grover66 on 2009-10-30
Please use the following steps to install Ubuntu Netbook Remix on a Dell Mini 9;

1. Install UNR (Ubuntu Netbook Remix) 9.10 using a LiveCD or Flash Drive.

2. After reboot, type the following to get wireless working;

sudo apt-get install bcmwl-kernel-source

3. Reboot.

Wireless should now work!!
