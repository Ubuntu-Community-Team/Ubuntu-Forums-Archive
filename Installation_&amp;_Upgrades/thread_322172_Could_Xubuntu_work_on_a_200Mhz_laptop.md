### Post by Herman on 2006-12-20
Hello Portable_Jim,
I have tested Xubuntu and I think it is very nice, but I haven't tried it in a very modest computer. The smallest computer I have running right now has a 1300 MHz processor and 225 mB Ram.  
I do know for sure that both Ubuntu or Kubuntu used to work reasonably well in my old 'Book PC' (pictured in my avatar).  The only thing it had trouble with was running Gnome Partition Editor after the 'swapoff' command was entered, but it would do everything else I needed.  That old computer has 128 mB Ram and 400 Mhz processor. I presume Xubuntu would have worked much better than Ubuntu or Kubuntu, but that old computer is out of action at the moment due to lack of a power supply so I can't test Xubuntu in it now.

Here is a link to the System [Requirements]("https://help.ubuntu.com/community/InstallingXubuntu#head-dd4eea4a07adb87125d696a0da184397a5b4f588") for [Xubuntu.]("http://www.xubuntu.org/")

That says the processor speed doesn't really matter, and recommends 64 to 128 mB Ram, but like I said, I found that even Ubuntu used to run okay with 128 mB of RAM, so I think Xubuntu should still work okay with a lot less than what they recommend, but I'm not sure how it would work with as little as 32 mB. Maybe someone else will be able to help here with some relevant facts.
Do they still make the type of RAM you have, and have you any empty RAM slots on your motherboard where you could add a pair of modules? Probably they would be pretty cheap if you can find out where you can get some.

Here is a link for [Xubuntu-Lite]("http://www.ubuntulite.org/dokuwiki/doku.php") and its [Hardware Requirements,]("http://www.ubuntulite.org/dokuwiki/doku.php?id=hardware_requirements")
I have never tried that at all, but it looks like it would be worth considering. Looks like that is made for 32 mB of Ram.

Regards, Herman. :D
