### Post by GWBouge on 2011-03-05
1) Use the 'Task Manager' tab to add shortcuts.  Just make sure that the Task Manager applet is used in the Applets tab.

2) The 'Install' button is for installing a new applet.  To use one from the list, just drag it from the list to the 'Active Applets' section at the bottom.

3) In the 'Preferences' tab, about mid-way through, there is an 'Icon effects' field.  Click on 'Edit' and you have a bunch of options.

### Post by Frogs Hair on 2011-03-05
1. When you open an application and the icon appears on the dock, right click the icon and add as launcher. Drag and drop doesn't always work.

2. Open the applets tab select the applet you want to add and use the arrows below the applets list to add or remove applets .

3.Open dock preference and the menus on the right have a list of settings.
The themes tab offers different dock themes. As for the icons looking like mac you will have to find a set that makes that possible
