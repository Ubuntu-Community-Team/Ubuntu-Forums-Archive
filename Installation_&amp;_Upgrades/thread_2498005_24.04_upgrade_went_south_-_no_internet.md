Hi bk,

I think we've all been in this situation but it sounds like not all hope is lost if you can SSH to the box on your local network. That means the host is at least being assigned an IP address.

To start off, I wouldn't use ping to troubleshoot networking. It's not 1995 anymore and most networks and hosts (wrongly) block ICMP traffic.

Here's a few commands to help you diagnose the issue(s):

route
dig a ubuntuforums.org +short
dig @8.8.8.8 a ubuntuforums.org +short
curl -IL ubuntuforums.org
