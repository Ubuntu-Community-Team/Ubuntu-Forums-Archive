### Post by Mark Phelps on 2014-11-15
Option 2.b) indicates that it will erase ALL the files -- but unfortunately, most folks do not understand that ALL means ALL.  The Mint developers changed the warning message so that it more clearly indicates that the entire drive will be erased and reformatted, but Canonical stubbornly refuses to make such a change.

You might be able to recover your previous stuff using Testdisk: [http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step]("http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step")
