### Post by bhishan on 2009-05-19
For Steam/CS open a terminal and type in this.

Code:

sudo apt-get install wine

Then go here ([http://appdb.winehq.org/appview.php?iVersionId=1554](http://appdb.winehq.org/appview.php?iVersionId=1554))  and follow the instructions.
