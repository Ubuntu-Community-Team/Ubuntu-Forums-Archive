### Post by linuxphoenix on 2009-11-07
Dear Friends,
              I want to share with you my experience with Ubuntu 9.10. I performed a clean install. Opened Firefox, clicked on a Youtube video. I was prompted to download and install the Flash Plug in. I clicked Agree and Flash was installed. Ever since then my PC started freezing repeatedly requiring rough reboots. Decided to remove the Flash Plugin. Failed first with Synaptic and then with the Terminal. Error messages in both. 
Formatted the partition and again installed Ubuntu 9.10. This time I installed ubuntu-restricted-extras package using apt-get. Things have worked perfectly since then. I am able to view Flash dependent content and the PC has never frozen once.
I don't know if this is just my experience. Or is it that others would have the same problem? 
My advice to everyone is to install ubuntu-restricted-extras package and avoid installing the Flash plugin directly.
Hope this will help some people.
