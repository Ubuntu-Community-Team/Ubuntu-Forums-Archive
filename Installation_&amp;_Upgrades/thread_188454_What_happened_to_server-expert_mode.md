### Post by Werner_vd_Walt on 2006-06-04
With 5.10 you were able to type in "server-expert" as an installation option. This allowed for great control over the installation procedure and also allowed the setup of the root user.
 
I have tried any variation of this option with the new 6.06 server install by pressing F6 and then changing the boot line there, but I can't get to the old server-expert mode even though the F3 help option states that you can still type in the same parameter.
 
Any advice please?? What am I doing wrong?](*,)

### Post by Werner_vd_Walt on 2006-06-04
Okay, I solved the problem... for all those who prefer the old way installing and having more control over what you do.
You simply wait for the graphical installer to load up and then press ESC. Confirm you want to leave the graphical boot menu and voila you end up back at the old familiar boot: prompt. Type server-expert here (or whatever other you may have used in the past) and away you go as before.
 
Rather frustrating as this is not clearly documented or indicated in the graphical boot menu F1 help and it took me the better part of a day and a whole lot of reading to come to this solution, simple as it may be.
