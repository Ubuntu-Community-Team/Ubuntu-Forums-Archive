### Post by mohamedtoma73 on 2011-11-09
to fix black screen after installing or upgrading ubuntu:
ubuntu 11.04
1- boot into recovery mode
2-choose failsafe graphic mode
3-choose run failsafe graphic mode for this session once
4-in the desktop  press system tab from the top panel;choose administrative and then login screen ;unlock and then change the the type of login from ubuntu into ubuntu classic
5-reboot and boot into normal mode  as usual
6- install the catalyst driver  from  ati as in [cchtml.com]("http://http://wiki.cchtml.com/index.php/Ubuntu_Natty_Installation_Guide")
7-change login screen setting from ubuntu classic into ubuntu
8-reboot and it id done
ubuntu 11.10
1-during installation choose not to log in automatically
2-boot  repeatedly  and alternately into normal mode and recovery mode  until desktop screen appear normally
3-install the catalyst driver as in[ cchtml.com]("http://wiki.cchtml.com/index.php/Ubuntu_Oneiric_Installation_Guide")
4-reboot and it is done
