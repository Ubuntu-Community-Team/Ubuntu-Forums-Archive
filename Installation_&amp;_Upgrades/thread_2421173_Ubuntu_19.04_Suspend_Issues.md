Thought I should make a post about this just to let anyone who might be able to do anything know.
I installed Ubuntu 19.04 on my Acer Aspire E15 laptop and it had a couple of issues, the main one being it couldn't suspend properly. If I did suspend, the screen would be completely off when I tried to wake it back up (no backlight), but the indicator leds showed the computer thought it was awake. I've now downgraded to LTS 18.04.2, but I still wanted to bring light to the issue.
