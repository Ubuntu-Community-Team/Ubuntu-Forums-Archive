### Post by grahammechanical on 2013-03-14
No, it is not the same. Ubuntu Unity User Interface is sitting on the Gnome 3 Desktop Environment (DE). By installing Gnome 3 shell you get the User Interface that the Gnome organization is developing. When we install Ubuntu Gnome (that is the new name since it was given approval to be a Ubuntu flavour) we get a lot more than just a User Interface. I have installed Ubuntu Gnome by the way - for testing. See, these release notes.

[https://wiki.ubuntu.com/UbuntuGNOME/ReleaseNotes/12.10](https://wiki.ubuntu.com/UbuntuGNOME/ReleaseNotes/12.10)

You see we get different default applications as well. Regards.
