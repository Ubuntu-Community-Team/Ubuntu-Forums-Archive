### Post by PmDematagoda on 2007-09-18
The 64 bit version of Ubuntu 7.04 is very useful if you have over 4Gb in RAM, as the 32 bit version only recognises 3.2Gb maximum, the 64 bit version is also very useful for memory intense applications such as video encoding, etc.:)

### Post by PmDematagoda on 2007-09-18
