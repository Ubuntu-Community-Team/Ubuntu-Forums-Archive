### Post by fridayjones85 on 2006-01-30
I love (K)ubuntu. 

For years I've been an Operating System vagabond, drifting from Linux distribution to Linux distrobution, and each time found myself forced to return to Windows because of poor hardware recognition, awkward configurations, or what-have-you. I was (and to an extent, still am) too new to the world of Linux to understand the more complex yet necessary configurations, and so made a quest of finding one that was particularly user-friendly. 

After a while, I found Mepis Linux and loved it; it was remarkably simple... but lacked the huge support community that that Ubuntu enjoys, so I made the switch. 

This brings me to my point; for all that is wonderful and remarkable about (K)ubuntu, I find myself terribly disappointed in the installation; unless you are giving (K)ubuntu an entire harddrive to itself, you are required to know a great deal about the Linux hard drive partitioning process and the included text-based interface... and both are fields in which I am horribly inexperienced. 

Considering that the majority of computer newbies live in the Windows world and would be terrified of the possibility of destroying their precious XP installation, wouldn't it make sense to make the installation process as simplistic as possible? I've found that the automated Mepis installer is extraordinarily painless, especially when creating a dual-boot system. 

Is there some reason that the Mepis system couldn't feasibly be used in (K)ubuntu? It would be a huge help in converting my friends to the world of Linux; I'm the resident geek in my circle, and if I can't handle the install process, my friends will shy away from scary new technologies. 

Thanks for any light anyone can shed on the subject!

### Post by aysiu on 2006-01-30
> **fridayjones85]
After a while, I found Mepis Linux and loved it said:**
>  This has been my exact experience as well. In terms of ease of use Mepis beats Ubuntu hands-down. 

For example, we have a quite involved HowTo here on restoring or reinstalling Grub to the MBR. In Mepis, you click on Mepis OS Center and then Install Grub. 

We have long explanations about how to use sudo or create a launcher for *gksudo nautilus* or *kdesu konqueror*. Mepis just comes with a launcher for Konqueror in Super User Mode.

Mepis is a live CD that's also an installer CD. Ubuntu has two separate CDs.

Mepis was my first real Linux, and if it weren't for Mepis, I'd probably still be using Windows right now. That said, when I needed support and questions answered, it was like the blind leading the blind in the Mepis Lovers area.

Ubuntu is an excellent distro, but it's not an excellent *first* distro. It may, in fact, be the best *second* distro for users new to Linux.
