### Post by jdong on 2007-04-21
The DVD ISO will install the same default set of programs as the normal CD, but contains more software on the CD (all binaries, not source). It is semi-comprehensive of the Main repository.

It's for people who do not have regular or fast access to the Internet and wish to have more packages at hand.
