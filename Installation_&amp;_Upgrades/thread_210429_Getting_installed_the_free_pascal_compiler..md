### Post by FredB on 2006-07-07
Here is what I tried.

1 - get free pascal compiler from [here]("http://www.freepascal.org/down/i386/linux.html")
2 - get to terminal, unpack the tar file using tar xvf *.tar
3 - the install is automated, and install datas in /usr/local/share by default
4 - to get the IDE, type fp on your command line
5 - to get the compiler, try fpc

Hope it helps ;)
