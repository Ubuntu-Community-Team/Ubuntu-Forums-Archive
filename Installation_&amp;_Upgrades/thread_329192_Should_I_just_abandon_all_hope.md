### Post by fredj on 2007-01-01
Edgy (6.10) does have bluetooth support built in so there is a chance that you can
get your keyboard and mouse to work. Can't really tell you any more than that.
I use a Logitech wireless trackball but that uses a usb2 receiver.
You should be able to install a dual boot system just by installing 6.10 to free space on
your disk drive. It should install the Grub boot loader which will allow you to dual boot
without needing any other software. Back up any important data before you start!
