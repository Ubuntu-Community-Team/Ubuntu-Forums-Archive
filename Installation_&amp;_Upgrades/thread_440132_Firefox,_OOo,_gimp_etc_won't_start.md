### Post by hal8000 on 2007-05-13
At the command prompt type:

sudo apt-get install nvidia-glx

sudo apt-get install nvidia-kernel-common


When you see the message Nvidia module cannot load, its because the driver in /etc/X11/xorg.conf is set to "nvidia", as in your case. You can cure this by changing the
line that starts Driver to
Driver "nv"

You would look for this section is xorg.conf

Section "Device"
Identifier "Generic Video Card"
Driver "nvidia"
# BusID "PCI:1:0:0"
Option "NvAGP" "1"
Option "AddARGBVisuals" "True"
Option "AddARGBGLXVisuals" "True"
Option "NoLogo" "True"
EndSection


Save the file and restarting the xserver. This will give you a working graphics system but
without opengl support and 3D acceleration. It is quicker to type the first commands in a terminal.
