Step 1. Go to Applications Menu -> Accessories -> Application Finder

Scroll down to Update Manager

Step 2. Drag and Dtop Update Manager to the panel 0 top or panel 1 bottom

Step 3. Edit the item Properties and chose the icon to change

You will find the software updater icon.
