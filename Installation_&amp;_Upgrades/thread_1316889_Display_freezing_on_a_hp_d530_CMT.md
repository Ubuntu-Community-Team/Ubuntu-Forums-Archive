I am planning on added just one thing back at a time to the system to see if that one thing will cause my computer to freeze. It will take some time, but then I will know for sure if it was something I installed. Most of what I have to install will come from the repositories anyways. In any case, I am just happy to have a stable system again.

Ubuntu still rules and 9.10 is still awesome!!

I hope this helps others who are having the same problem that I was having.
