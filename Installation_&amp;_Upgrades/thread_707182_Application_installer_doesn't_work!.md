### Post by Kevbert on 2008-02-25
Hi.
It looks like your software sources have not been activated.
You need to be connected to the web.
Go to System - Administration - Software Sources.
Now enter your password.
Check that under Ubuntu Software, main, universe, multiverse and restricted are selected and that a server close to your location is selected from the pull down menu.
Under the updates tag now check that Important and Recommended updates are selected.
Close the Window.
Now go to System - Administration and select Update Manager.  Click on Check - you may find you need some updates.
