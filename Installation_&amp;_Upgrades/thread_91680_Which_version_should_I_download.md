### Post by taurus on 2005-11-17
PPC is for PowerPC; AMD64 is for AMD 64bit; and x86 is for all pentium chips and AMD XP (or 32bit).  You can upgrade your kernel from i386 to i686 after you install ubuntu if you wish.  If you want to run kubuntu, you can install it from ubuntu with

sudo apt-get install kubuntu-desktop

Then at the login screen, you have a choice to use either Gnome or KDE as your desktop...  If you have enough space, install ubuntu 5.10 and use the command above to install KDE!
