Based on how you've described the problem. I would probably want to eliminate the optical drive as a source of the problem. I know you said you can read from it but I have seen drives that will read but are not able to boot an iso. And the drives I've seen do that act as you've outlined-of course the disc could have been scratched or gotten a fingerprint between the time it worked and now-no? Well maybe it's a long shot-so shoot me but can you boot from the windows install disc?
