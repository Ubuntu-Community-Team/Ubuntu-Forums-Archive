### Post by rukuartic on 2007-06-01
Ubuntu isn't quite like a program that gets "uninstalled".

If you want to dual boot, drop in the Windows install CD and boot from that. (You might need to press a special button before GRUB comes up or modify BIOS settings to boot from a CD). Once Windows is installed, if you want to dual boot with Ubuntu there are several guides around on the internet you can use.
