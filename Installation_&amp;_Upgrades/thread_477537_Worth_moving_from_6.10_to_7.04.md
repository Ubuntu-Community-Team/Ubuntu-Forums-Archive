### Post by rax_m on 2007-06-18
Interesting you should ask, b'cos I just upgraded my 6.10 to 7.04 a couple of hours ago ;)

So far, some of the issues that I had with 6.10 haven't really been resolved with 7.10:
- Sound working by default (Toshiba ACPI issue)
- Desktop effects don't work for me .. tho if I look hard enough I could probably fix it


The good side:
- Seems to be a lot more responsive than edgy :)
- The new network manager applet is much better and I can use WPA without CLI
- Newer versions of all software especiall Openoffice :) 

Other than that I can't say there is much of a difference. I did find configuring everything was a breeze (but that's probably down to experience now). 

I would say if everything is working for you in 6.10 then there's no real reason to move.. unless you just like to be "cutting edge".

Depnds on what you want to do.  Feisty has some advantages over Edgy.

1.  Better wireless support
2.  Restricted drivers for ATI and Nvidia a click away
3.  Openoffice.org default install has much nicer fonts
4.  Feisty will help you find a proper codec for multimedia files
5.  Updated kernel supports more hardware


If your PC is running like a champ with Edgy and you see no need to upgrade then stay where you are.

If you do decide to upgrade then do a clean install.  I have had nothing but problems doing the upgrade manager route.
