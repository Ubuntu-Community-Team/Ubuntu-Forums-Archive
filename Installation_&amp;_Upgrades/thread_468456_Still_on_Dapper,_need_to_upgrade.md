### Post by newlinux on 2007-06-08
If you don't reinstall, you have to go dapper->Edgy->Feisty. You shouldn't lose any data and your web server should work, but there are no guarantees. For some people it is seemless, for others it isn't, and that's why some just backup their system, install the new distro, and then reinstall apps (also why some keep a separate /home partition).
