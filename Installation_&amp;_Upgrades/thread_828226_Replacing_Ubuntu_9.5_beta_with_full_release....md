### Post by ibutho on 2008-06-13
Are you talking about Opera? The reason I ask is because there isn't an Ubuntu 9.5. If you are talking about opera, you can run the ADD/Remove Software program or Synaptic, search for Opera and then uninstall it. You can then download the stable release from the Opera website and install it by doing
```
sudo dpkg -i opera_9.50.etch-qt3_i386.deb
```
