### Post by Jussi Kukkonen on 2006-09-22
> **textguru said:**
> Beginners question, please bear with me. I installed Dapper 6.06 Desktop CD. How can I check if Apache, PHP and MySQL is already running on my system? If not, how can I install them? Any reference that I can read on?
Unless you explicitly installed them, they aren't installed. You can check this by e.g. by doing a *ps ax*, and seeing if you find apache, php or mysql processes.

The wiki is a good place to look for HOWTOs. I did the searching for you this time: [https://help.ubuntu.com/community/ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP)
