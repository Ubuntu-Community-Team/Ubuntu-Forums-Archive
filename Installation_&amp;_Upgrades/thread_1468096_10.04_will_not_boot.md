### Post by wojox on 2010-05-01
Boot a LiveCD and run:

```
gksudo gedit /etc/default/grub
```

Look for and add nomodeset:

```
GRUB_CMDLINE_LINUX_DEFAULT="splash quiet nomodeset"
```

Save and reboot. That did it for me.
