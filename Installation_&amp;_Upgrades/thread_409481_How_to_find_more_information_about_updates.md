### Post by tictacman on 2007-04-14
Having run ubuntu for a few months now, has taught me to be a bit cautious about rushing to update to the latest packages.  Windows inspired glee for updates and the mentality of newer is better has faded.  I've noticed sometimes its possible to upgrade to a package that can be little buggy at times.  Is there anyway to find more info about the packages updates?  Such as why they are being updated for instance.  I was hoping for some kind of advisory web page or mailing list. That way I can make more informed decision about whether to update or not.

---

### Post by 23meg on 2007-04-14
Security update announcements are posted to the ubuntu-security-announce mailing list, as well as the News & Announcements forum section ([here]("http://ubuntuforums.org/forumdisplay.php?f=13")). You can also choose the update in the update manager and click " Changes and description of the update" at the bottom to get more information.
