### Post by yosibeck on 2008-04-25
After succesfully using Feisty & Gutsy for quite some time on my PC, I was waiting to install Hardy on my new Toshiba laptop the moment the official release came out.
I decided to make a new Hardy installation on my laptop. It detected my wireless connection and bluetooth mouse with no problems.
But then to my surprise I just cannot make Hardy connect to my Brother MFC 440CN network printer over my wireless connection.
Of course I downloaded the latest drivers from the brother website and followed all instructions as I did in the past with Gutsy but to no avail.
I also saw on the brother website that one has to make a manual sudo install because a directory that used to exist and is needed for installation is not there anymore in Hardy. Did that too.
When searching for a new printer Hardy does find the MFC-440CN and goes through the process as if setting it up, but when I try to print something nothing happens and I get the message: "Printer might not be connected". Of course the printer is connected and works just fine over the wireless network under Windows, so it is a Hardy problem.
If anyone has advise I would be most grateful. Without a printer I am at a loss.
Is it possible that all the procedures that I followed are just for a USB cable connection and not for use of the printer as a network printer?? Are there other settings that have to be set for connecting the printer over a wireless connection?
Thanks, Yossi

### Post by yosibeck on 2008-04-25
OK Thanks to help from others - the answers are on [http://solutions.brother.com/linux/sol/printer/linux/cups_network.html]("http://solutions.brother.com/linux/sol/printer/linux/cups_network.html")
Printer works just fine now
