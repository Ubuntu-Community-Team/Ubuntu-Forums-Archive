### Post by lH)4~mK0e on 2006-08-29
There is certain cases where, yes you could preserve your home directory.  You would have to specify a separate partition for /home when you did the install of 5.10.  Then all you have to do is run a custom partitioning when you install 6.06.  If you didn't do it this way, you would have to backup your home directory to a cd or dvd and then copy it back over when you do the new install.  I would suggest doing that anyway so you have a backup of your files.  Once a month is acceptable, more if you are paranoid.
