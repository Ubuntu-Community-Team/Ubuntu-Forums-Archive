### Post by Scunizi on 2006-06-11
I installed Dapper 6.06 fresh after running it while in flight mode for a couple of month.  I thought I'd give it a clean start after having messed around with it quite a bit.  After the install and numerous boots, I decided to see how 6.06 would handle multiple usb storage devices.  It will only recognize one at a time.  At least on my system.  So I left one plugged in for back up of some smaller files.  It is formatted vFat, 1 gig and newer supporting usb 2.0.  After attempting a morning wakup boot I'm greated with "Uncompressing Linux ...... Ok.  Loading Linux Kernal" ..... then nothing.... wait.... Nothing.... wait..... NOTHING  .... Aarrggg.... I thought I'd borked my system somehow.  Numerous boots, bios changes, knoppix live loaded to help troubleshoot,  nothing.  I turned the machine off and walked away to clear my head before the next step.. reinstall.  I must have flashed on something in my dreams that night.  I got up, tried to boot, same issue.  Put in the new 6.06 Live/Install cd and attempted to boot.  SAME problem at the SAME place!  I flashed on the flash drive, removed it and tried to reboot.  The system came up like normal!  Anyone else have issues like this?  I'm not an experienced Launchpad user so searching for a bug there is daunting but I'll give it a shot.
