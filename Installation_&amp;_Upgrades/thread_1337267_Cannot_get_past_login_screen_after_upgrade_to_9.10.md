### Post by jaylsi on 2009-11-25
Do you have a graphic card? If yes, the problem could be that  you need to install driver for it.

Try this:
1) On GNOME login, select "Failsafe GNOME" under session list, and log in
2) Go to System->Administration->Hardware Drivers to install the proprietary driver 
3) Reboot
