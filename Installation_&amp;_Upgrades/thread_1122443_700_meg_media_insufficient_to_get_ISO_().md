### Post by Mark Phelps on 2009-04-11
I know that 8.04LTS (32-bit) fits on a standard CD because I recently burnt one myself.

The 32-bit ISO image I recently downloaded has the following sizes:
1) desktop -- 699.1MB
2) alternate -- 697.1MB

Sorry to ask basic questions, but are you sure you're actually burning the ISO to the CD, not just copying the ISO file to it?

Also, are you sure it's a 700MB CD, not an older 650MB CD?
