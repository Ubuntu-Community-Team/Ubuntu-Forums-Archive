### Post by TheFu on 2017-03-24
Welcome to the forums.

Good backups are 1,000 more important than RAID-anything. Even with RAID, you still need solid backups. There are 1,000 things that versioned backups solve and only 3 things that RAID solves.  For many people, RAID is just a faster way to write corrupt data to multiple disks.

Yes, you can setup RAID for HOME after the fact.  I've never used SW-RAID any other way.
