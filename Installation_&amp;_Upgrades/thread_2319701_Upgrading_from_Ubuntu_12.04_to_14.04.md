### Post by grahammechanical on 2016-04-06
You need to create an new user account for that original user with the original username & password. You do that in System Settings>User Accounts. Then you should be able to login as the original user and access the original user's home folder.

I am beginning to wonder if this upgrade from a ISO image on a USB stick with a new username & password was actually a fresh install. In which case, all bets are off.
