It will not run a desktop without a working XServer. That is the binding graphics layer to the base linux system.

Boot. After the boot messages, hold down the <Shift> key. The Grub menu should appear. Press the <e> key. You will be in edit mode of the first Grub menu item. <Arrow Down> to the line that starts with "linux". <Arrow Right> to where is says " quiet splash " and replace that text with " --verbose text ". Press <Cntrl><x> to boot.

It will boot with verbose messaging turned on so you can see any errors. It will continue to a TTY Text console login. From there you can reinstall xserver-xorg-video-all and xserver-xorg-video-nouveau via
```

sudo apt-get install xserver-xorg-video-all xserver-xorg-video-nouveau

```
Good luck and waiting to hear how it goes for you.
