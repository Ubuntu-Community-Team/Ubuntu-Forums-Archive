### Post by dstew on 2009-02-17
It may be a graphics card incompatibility problem. On the first screen when the Live CD boots, you might have a choice to boot in "Safe Graphics Mode". You press F4 or F6 I think to get this choice. Maybe that will allow the live CD to give you a workable desktop.

If you can't get the Live CD to work, you can install the same Ubuntu system using the [Alternate Install CD]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate"). It uses a text-based installer that works more reliably with most graphics cards.
