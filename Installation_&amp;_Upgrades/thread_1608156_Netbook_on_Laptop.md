### Post by ajgreeny on 2010-10-28
Yes you can.  You can burn the iso to a disk or use a USB drive, if the laptop will boot from USB.

On a larger screen you may prefer to use the gnome desktop from the session menu on the login page.  Other than the lack of a few applications, the Netbook edition is the same as the desktop edition.  You can even add ubuntu-desktop package if you want to end up exactly the same as the desktop edition, but it is not really needed, just add brasero for CD burning and compiz if you want it and that should be enough.
