### Post by bubwitmaingay on 2009-03-08
You can find alternative softwares for linux which are similar to windows at [http://linuxappfinder.com]("http://linuxappfinder.com"). Just search for the alternative and it will list them for you.

In your machine you can go to : Applications>add/remove
Just navigate into this because it will list all supported applications made for Ubuntu or third-party applications which can be used for Ubuntu.
