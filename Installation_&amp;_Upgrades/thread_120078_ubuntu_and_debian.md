### Post by matthew on 2006-01-20
The really simple answer is "no." Without getting really technical, they don't share the same kernel and the various components of the operating system/distro are not generally the same (i.e. Debian still uses XFree86 and Ubuntu uses Xorg, they use different versions of gcc, lots more, but here's where it will get more and more technical). Anyway, they can use different partitions on the same hard drive and you can make a shared partition for them to share files from one to the other (I've done this when I was running Debian Sid and Ubuntu Dapper in parallel...decided I prefer Ubuntu, but they're both great).
