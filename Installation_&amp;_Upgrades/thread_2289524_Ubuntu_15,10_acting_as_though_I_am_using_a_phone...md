### Post by grahammechanical on 2015-08-04
Perhaps you downloaded and installed Ubuntu Desktop Next 15.10. And not Ubuntu 15.10. Compare these two locations.

[http://cdimage.ubuntu.com/ubuntu-desktop-next/daily-live/current/](http://cdimage.ubuntu.com/ubuntu-desktop-next/daily-live/current/)

[http://cdimage.ubuntu.com/ubuntu/daily-live/current/](http://cdimage.ubuntu.com/ubuntu/daily-live/current/)

You would not be the first to make this mistake with 15.10 under development.

The last ISO image for Ubuntu Desktop Next was built on 29th May. And it is indeed the phone OS. I think that Ubuntu Desktop Next has been replaced by Snappy Ubuntu Desktop Next which is built on Snappy Ubuntu core. It does not come as an ISO image but another kind of image that can be installed in KVM or image restored onto a USB memory stick. That is also the Ubuntu phone OS. But one day it might morph into a true Ubuntu desktop.

Perhaps it should be called Snappy Ubuntu Desktop NotYet.
