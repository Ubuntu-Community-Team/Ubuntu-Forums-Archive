### Post by david-blythin on 2020-04-15
I did the same thing with

sudo do-release-upgrade -m desktop -d

got the info here:-

[https://kifarunix.com/upgrade-ubuntu-18-04-to-ubuntu-20-04/](https://kifarunix.com/upgrade-ubuntu-18-04-to-ubuntu-20-04/)

and yes, it's obviously not the full release, it's a testing release that has to be viewed as use at your own risk.
I have a few minor issues, mostly due to the fact that community based stuff won't catch up for a while, but most things work OK - all the important stuff anyway.
