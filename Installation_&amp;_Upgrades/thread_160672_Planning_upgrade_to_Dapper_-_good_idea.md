### Post by aysiu on 2006-04-15
[QUOTE=Kapre]hiyall! To those of you that are planning to upgrade to Dapper, please take some time and read this [post]("http://www.ubuntuforums.org/showthread.php?t=93157") by Ubuntu_Demon(worth the time before upgrading to Dapper).[/QUOTE] That was posted in November. A lot has changed since then. **Edit**: Sorry. I see that it was edited three weeks ago. Still, I believe Dapper is now more or less stable. I'm using it as my primary OS for almost a week now, and no serious issues have come up. Sure, random errors have come up, but nothing to bring my system to a halt.

[quote=Melat0nin]
- how easy is it to reinstall Ubuntu, and should I just do the upgrade or should I install Dapper from an ISO?[/quote] It's pretty easy to reinstall, especially if you have a separate /home or data partition. I think you should do an upgrade, and if it borks your system, then you can do a reinstall. > - If i install Dapper now, before it is released, what will that mean come the release date? Will I have to reinstall it with the released version, or what? No. You'll just keep doing ```
sudo apt-get update && sudo apt-get dist-upgrade
``` every now and then to get updates. > - Is it even worth installing a pre-release version so close to the release date? Well, ask yourself what you think you're gaining by installing Dapper early.
> - briefly, what are the advantages of Dapper over Breezy? Right now? Random programs giving you error messages, like the Gnome panel crashing when you try to open certain programs.

The only real advantage I see in Dapper over Breezy right now is Firefox 1.5, and it's a lot easier to use [this script](http://www.psychocats.net/ubuntu/firefox) to install 1.5 in Breezy than it is to upgrade your entire system to Dapper.
