### Post by Montblanc_Kupo on 2009-01-19
Personally... I've had nothing but problems using 64bit versions. I tend to recommend 32bit regardless. The advantages of the 64bit are outweighed by the conflicts and instability, IMO.

i386 will run on a 64bit machine... x86_64 / i686 won't run on a 32bit machine.
