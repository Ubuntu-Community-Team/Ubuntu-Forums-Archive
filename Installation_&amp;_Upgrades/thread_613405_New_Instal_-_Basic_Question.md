### Post by zdude on 2007-11-14
The .iso is a disc "image"! You don't copy the image to the CD you need to load and burn the image. If using Nero look for a "burn image" menu item, select it and it will ask for a image, pick the *.iso file and it should create the CD.The finished CD should contain quite a few files, not one.
Good luck.
You can google "burning iso image" and get more info.
