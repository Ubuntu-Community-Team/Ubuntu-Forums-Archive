### Post by darkod on 2012-03-07
In my opinion the biggest tie decider is the amount of RAM (present and future upgrade plans).
32bit can address only 4GB, in reality about 3.5GB. If you have or plan to have more RAM then by using 32bit you are effectively not using it.

For the rest, depends if you have particular apps that you are interested in and whether 32/64 makes difference for them.
