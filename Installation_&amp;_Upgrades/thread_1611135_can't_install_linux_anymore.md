### Post by eblume on 2010-11-01
Try downloading and booting from Knoppix ([http://www.knoppix.net/](http://www.knoppix.net/)). The rule of thumb is that "If Knoppix can't boot it, it doesn't boot." If Knoppix DOES boot your machine without errors, the problem is with Ubuntu (or more probably, with the specific CD you burned Ubuntu on to - probably a scratch or a data error -- but also possibly a driver error in the Ubuntu system software). If Knoppix DOESN'T boot without error but Windows *does*, then something very, very fishy is going on.
