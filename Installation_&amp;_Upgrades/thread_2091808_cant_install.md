### Post by westie457 on 2012-12-06
Take a look at this page for advice on getting the Live Desktop running. [https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

Usually the 'nomodeset' option is enough to get the system running. 
After a few seconds the 'Additional Drivers' should notify you that there is other drivers for your system and give you the opportunity to install them. A working network connection is necessary for this so plug a cable in.

Any problems with this please post back here.

Thank you.
