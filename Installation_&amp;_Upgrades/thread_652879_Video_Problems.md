### Post by tgilber1 on 2007-12-29
Try using the "alternate desktop CD", which is a text-based installer to get by the graphics issue.  When installing, try installing the linux-restricted kernel with nvidia-glx-new module.  
To get this version, you will have to select the checkbox underneath "Start Download" at the following URL:  [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
