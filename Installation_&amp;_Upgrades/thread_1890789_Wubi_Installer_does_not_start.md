### Post by Whiplash! on 2011-12-04
Hello everybody,

I just downloaded Ubuntu 11.10 and the matching wubi installer to install on my netbook. However, wubi does not even start. When I double-click it, there is just the UAC message displayed and after I click to allow wubi access, it appears to start the program but nothing happens. I am using an Asus Eee PC 1016P with Win7 starter. Any ideas what could be wrong with this? I am clueless...

Thanks for your help!

---

### Post by Whiplash! on 2011-12-04
found the solution - wubi always tried to read from another disk image that was stored on D:. After I moved this iso file to another directory, it would start without any problem. -> This thread can be closed
