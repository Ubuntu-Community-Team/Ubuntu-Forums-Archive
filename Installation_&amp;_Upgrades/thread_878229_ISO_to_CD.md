### Post by alzie on 2008-08-02
To do the md5sum you need a program to check the hash encoded in the download.  I use DigestIT from : [http://www.kennethballard.com/modules/xproject/index.php?op=viewSummary&pid=2](http://www.kennethballard.com/modules/xproject/index.php?op=viewSummary&pid=2)

I'm sure there are others.  You compare the generated number with a number provided at the website.

I believe when you start the live cd there is an option to check the cd for defects.  I'd try that to see if the cd is ok.

If you computer has low memory available you may need to look at a alt install disc.  Its a command line installation, but not difficult.  I managed to figure it out :wink:

I hope this helps
