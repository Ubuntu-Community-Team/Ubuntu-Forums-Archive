### Post by bubwitmaingay on 2009-03-06
Yup, you should have made the persistence space larger. The 4gig is the minimal install, thus will not hold on for more to come in (such as updates). I have tried that in my machine once, when I still dual-boot with XP. It was very frustrating that it did not update. Try 6gig or 10.
