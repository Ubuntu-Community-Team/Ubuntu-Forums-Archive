### Post by louieb on 2006-12-19
I have a old P3 computer that refused to boot to CD without using [Smart Boot Manager]("http://louboldt.com/ilinuxsbm.htm"). I started to swap out the CD drive to see if that would fix it. What I found the CD was plugged in as Master but the jumper was set to Slave. Changed the jumper and that fixed that.  You said your PC is new, maybe someone was in a hurry when the CD drive was plugged in.
