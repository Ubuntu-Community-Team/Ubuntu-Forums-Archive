### Post by a_satari on 2007-11-09
Installing Ubuntu is quite easy. Download ubuntu and create a cd. put it in the cd drive, reboot and run the live cd. Once the ubuntu live cd is running, click install and follow instructions. 

As to uninstalling vista, perhaps you can do it while installing ubuntu using its partition manager, that is, if vista is in a partition. If this is the case, you could format the partition and install ubuntu in its place.

I'm new to ubuntu and don't know very much about it. However I hope this helps a little.
