### Post by raymondh on 2009-06-01
Hello Brian and FTU,

Yes, there is a bug regarding the switcher.  What happens is that you do lose the panels.  There is a deb file you will need to install to correct.  Until then, it is not advisable to use the switcher

[https://bugs.launchpad.net/desktop-switcher/+bug/349519](https://bugs.launchpad.net/desktop-switcher/+bug/349519)
[https://bugs.edge.launchpad.net/desktop-switcher/+bug/349519/comments/59](https://bugs.edge.launchpad.net/desktop-switcher/+bug/349519/comments/59)

Regards,
