### Post by seldenthuis on 2007-10-22
Unless you have more than 4 GB of RAM or you use some software that only runs on 64bit, there is no real reason to use it.  Even though there has been quite some progress for Linux on 64bit, some things still don't work correctly (e.g. Flash IIRC).

As for your question about NumLock, look at
[http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_turn_on_Num_Lock_on_GNOME_startup]("http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_turn_on_Num_Lock_on_GNOME_startup")
The instructions are for Feisty, but they should also work on Gutsy.
