### Post by kawabago on 2012-10-20
After installing xscreensaver on both 12.04 and 12.10 the GLslideshow screensaver was not opening files. To fix the problem install package libwww-perl. I found the problem by running /usr/lib/xscreensaver/glslideshow in a terminal to find out what was missing.
