### Post by zanglang on 2007-07-28
You'll need to extract its contents into a directory, run 'make', and then 'sudo make install'. There's a file called INSTALL in it that you can read for more detailed instructions.

Anyway, ndiswrapper should already be bundled in Ubuntu, so you can just install the package 'ndiswrapper-utils'. ;)
