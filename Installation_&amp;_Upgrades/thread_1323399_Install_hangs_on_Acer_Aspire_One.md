### Post by oylenshpeegul on 2009-11-29
I had a similar problem when I installed on the flashdrive from another Ubuntu machine using USB Startup Disk Creator. The Acer would boot from the resulting flashdrive, allow me to choose "English," and then just sit there.

I installed on the flashdrive again with "Discard on shutdown, unless you saved them elsewhere" selected. This time the Acer booted from the resulting flashdrive and continued on to the desktop, allowing me to install.

I'm not familiar with Windows, but if makeusb.exe has a similar option, it might be worth trying it.
