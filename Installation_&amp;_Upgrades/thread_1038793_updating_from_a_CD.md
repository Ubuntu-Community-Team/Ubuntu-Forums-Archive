### Post by cariboo on 2009-01-13
IF you need to download packages without a network connection you should go [here]("http://packages.ubuntu.com/"), and search for the packages you want to download. You will also have to download the dependencies that are listed with the package. 

For downloading updates, I would recommend aptoncd, just install it on your Ubuntu computer that is connected to the internet,

From Synaptic:
> APT removable repository creator and package backup tool for Debian based
systems.

This tool will allow you to create a media (CD or DVD) to use to install
software via APT in a non-connected machine, as well upgrade and install
the same set of softwares in several machines with no need to re-download
the packages again.
