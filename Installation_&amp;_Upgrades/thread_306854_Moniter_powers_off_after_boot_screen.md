### Post by livingtarget on 2006-11-26
Probably a monitor (or even video card) which can't do a certain resolution. You can try to remove the splash option. There is a custom option in the cd's boot menu where you can remove the word splash as far as I remember.

If you don't manage that just let it boot regardless, try letting it run for 2/3 minutes to see if it's actually doing something. If it reaches the desktop, then it will usually switch resolution.

That's just a guess, it might help.
