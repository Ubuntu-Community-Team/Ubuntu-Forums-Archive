### Post by darkod on 2010-05-15
As far as I know, because 10.04 is LTS version, the ISO is upgraded from time to time (not sure how often). In between, until the next LTS, the images of other releases are the same as when first released.

For example, for 9.10 the images were the same until 10.04 release. But for 10.04 they will change because it's LTS. Next to the files there is a date so you know when the image was changed/uploaded.

### Post by Catharsis on 2010-05-16
The normal releases remain exactly the same. LTS releases get updated periodically, i.e. 10.04, 10.04.1, 10.04.2, etc.

Hardy is currently 8.04.4; Lucid 10.04.1 is due out 7/29/2010.
