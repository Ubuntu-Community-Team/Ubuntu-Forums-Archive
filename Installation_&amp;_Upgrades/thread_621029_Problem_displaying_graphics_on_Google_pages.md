### Post by Ehtetur on 2007-11-23
If it's png & jpg images not showing up in firefox, this should fix it:
Go to Edit --> Preferences --> Content (TAB) and make sure that the boxes are checked next to block popups, load images, enable javascript, and enable java.
All those are/should of been enabled by default in firefox.

Now if you mean images that are shown by Shockwave Flash or javascript, you need to first install support for them, if you haven't already...
Fire up synaptic and install the following apps:
flashplugin-nonfree
j2re1.4-mozilla-plugin

The jre 1.4 works great on the x64_86 version of Gutsy.. If you're using Gutsy for x86, there's the newer jre 1.6 that will work.
