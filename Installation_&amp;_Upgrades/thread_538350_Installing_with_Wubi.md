### Post by be4truth on 2007-08-30
Are you online during the install process? My experience is that when installing Ubuntu around 75% Ubuntu tries to go online. If the link is there it goes fast afterwards. If no cable is connected and no wireless card around it goes fast as well as Ubuntu doesn't even look for a connection. If something like an Internet connection is around but not working properly Ubuntu takes easily 15 to 30 min to try to connect and wait for the timeout.

### Post by logos34 on 2007-08-30
Even assuming you manage to get through the install, your system specs are way too low to run it properly, if at all.  You have only half the min RAM, and your 731 MHz pentium chip isn't up to snuff.  

> [http://wubi-installer.org/faq.php#requirements](http://wubi-installer.org/faq.php#requirements)
Requirements

What are the system requirements?

256 MB RAM and an 1 GHz or faster Intel/AMD processor is recommended for optimal performance, though Xubuntu might work on less. As for disk space, the installation requires a minimum of 4GB. This space is mostly used by the virtual hard disk file. Most computers purchased within the last 3 years should be able to run Ubuntu fine, and Xubuntu is suitable for older computers.

I think you're only option is to shrink windows and install to hard disk.  Try Xubuntu.  It's lightweight.  Should run fine on your machine.
