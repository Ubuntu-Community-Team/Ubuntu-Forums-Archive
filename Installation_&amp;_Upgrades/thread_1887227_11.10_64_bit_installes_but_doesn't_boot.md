### Post by darkod on 2011-11-26
In the boot menu select the uubntu entry but hit 'e', not Enter. (if you have only ubuntu and don't get a boot menu press Shift to show)

That will show the boot lines. In the line starting with linux, add nomodeset before quiet splash. Press Ctrl + B to boot.

If that worked, you might need to make it permanent, but lets see if it worked first.
