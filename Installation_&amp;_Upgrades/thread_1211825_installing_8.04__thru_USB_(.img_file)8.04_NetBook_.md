### Post by quixote on 2009-07-15
I'm not sure Remix is available for 8.04 (Hardy heron), since NBR is pretty new.  The Jaunty (9.04) version is really nice, though.  Why not try that?  There's a link to download right at the top of [http://ubuntu.com](http://ubuntu.com).

You'll need an empty usb flash drive, or one where you don't mind losing what's on it.  It should be at least 2GB.  If you have a computer running Jaunty, making the bootable flash drive is really simple.  Install "usb-imagewriter" using Synaptic.  Then, under Accessories the Imagewriter package appears.  After that you just select your download, then your usb thumbdrive, and you're done.  The thumbdrive acts exactly like a LiveCD.
