### Post by ajgreeny on 2012-04-15
On a low powered notebook or netbook you might be better with Xubuntu or Lubuntu, rather than Ubuntu. However, have you installed any Additional Drivers that might be needed by clicking on the dash button top left and typing "Additional drivers".  If not give that a try and you may find that the lag in your system has gone.

If it is a new and powerful notebook Ubuntu with Unity should be fine as long as the hardware is compatible, and you have added the drivers as above.

### Post by darkod on 2012-04-15
When ubuntu is booted it doesn't matter what windows programs you have.

As already said, ubuntu might be too much if this is a netbook machine, although it runs fine on my Samsung NC10 netbook for example.

Try running an earlier version in live mode with the usb stick (you don't need to install to test).

In fact, first try to run this 11.10 version in live mode (using the Try Ubuntu option) and see how it runs in live mode first. If it still has many problems, try 10.04 LTS.
Then try Xubuntu or Lubuntu.

See how that goes.

After you decide what is running fine for you, you can consider installing over this 11.10.
