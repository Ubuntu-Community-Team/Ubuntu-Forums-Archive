### Post by merlinus on 2007-05-20
Ubuntu needs at least two partitions, one for / and one for /swap.  I believe you need to specify this during the installation process.  Select Edit for each one, after the partitions are made, and then pick from the dropdown list.

If you have enough room on your hard disk, then creating a /home partition is also a very good idea.

Good luck!
