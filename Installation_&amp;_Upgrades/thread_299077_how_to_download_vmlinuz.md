### Post by catlett on 2006-11-13
It's a common pain in the butt, but there is a way around it. Just right-click on the file and select "save link as". When the download window opens, just select the folder you want to save it in like any other download. Even though it says save 'link' you will actually be downloading the file. Don't ask me why it says link, I don't know. All I know is it works.
