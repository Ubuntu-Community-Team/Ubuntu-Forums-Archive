### Post by Grenage on 2012-02-20
If you encrypted your new home directory, you've probably lost your data; everything would have been overwritten.

If your last install was encrypted, you're probably not going to have much luck, even if you have the key.  If you don't have the key, I'm sorry but you have no chance at all.

If there was no encryption, you could try testdisk/photorec from a CD, but stop using that drive as it is.
