### Post by steveneddy on 2008-09-27
There are seven run levels available with your Ubuntu installation.

Just hit

Ctrl+Alt+F1

log in and have at it.

To get back to the GUI, hit 

Ctrl+Alt+F7

F1 through F6 are all text based interfaces, or terminal interfaces.

F7 is the GUI - F8 is the system monitor events page.

There is no need to install the Server Edition just for a text interface.
