### Post by ytsejam1138 on 2009-04-07
On the disc there is a file in the root called Wubi.exe run this and it will install Ubuntu within Windows.  After installing, you you will be given a choice of which OS to boot when you turn on your machine.  If you ever want to uninstall Ubuntu, you can do it from Windows control panel.
