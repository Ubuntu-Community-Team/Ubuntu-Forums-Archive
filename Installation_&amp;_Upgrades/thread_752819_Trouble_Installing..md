### Post by keratos on 2008-04-12
Sounds like an issue with the video.

Download the "alternate" CD image from the ubuntu site.

It is a text base install but provides options for you to specify video and monitor hardware as well as other options. It is not that difficult and the defaults are usually good enough. 
Just be sure to choose all your video (card, resolution, monitor etc) options.
