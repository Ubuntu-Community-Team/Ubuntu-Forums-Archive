Yes. If you install Beta 1, it'll notify you there are a pile of upgrades available which will bring you to beta 2. Then the only difference between you and everyone else is you're installing the updates to get to release on a daily basis where everyone else will install them all at once on the release date. Either way you'll end up at the same spot on release day.
