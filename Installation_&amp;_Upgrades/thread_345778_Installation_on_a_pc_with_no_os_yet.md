### Post by maxamillion on 2007-01-24
Your motherboard is not set to boot from cd-rom. When it boots it will say something like "hit DEL to enter setup" ... hit whatever key it asks for, make sure the cd-rom drive is in the boot order before the hard drive, save and exit .... reboot. You should be fine from there.
