### Post by diebels on 2005-06-15
[QUOTE=SadJoker]I don't really understand that ...  ](*,)[/QUOTE]
 No wonder. It does not concern your problem.

The clue is that your harddisk is filled up with XP and there is no room for ubuntu. So you need to shrink the space thats allocated for XP on the disk, so there is an empty space that can be allocated for ubuntu.

You should be able to do all this when you install ubuntu. But it's not failsafe, so a backup is needed before you try. A defrag in XP will enable you to shrink it more.
