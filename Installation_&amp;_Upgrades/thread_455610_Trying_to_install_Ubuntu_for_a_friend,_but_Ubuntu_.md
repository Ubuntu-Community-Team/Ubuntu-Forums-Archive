### Post by rillip on 2007-05-27
Sounds to me like you've either got very slow IO on the CD drive or the memory is way too small.  If you can't run the liveCD well, don't expect the full instlal to go much better.  Some system specs might help, but in general 256 is considered the minimum, 512 at least recommended.  You might try a distro with a lightweight window manager; Xubuntu has one, for example.  Typically it needs less RAM.
