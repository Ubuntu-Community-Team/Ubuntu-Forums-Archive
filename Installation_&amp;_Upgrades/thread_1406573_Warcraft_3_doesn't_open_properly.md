### Post by tom4everitt on 2010-03-24
Two tips:

1. In System->Administration->Hardware drivers there will be a button to activate proprietary drivers, if there are any available. Otherwise they're probably installed already. Do you get fancy effects when minimizing windows or switching workspaces?


2. Install the wine beta instead of wine. Remove wine and search and install wine1.2 instead. If still doesn't work you can try wine1.2 in lucid instead, the beta is out.
