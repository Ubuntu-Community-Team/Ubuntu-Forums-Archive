You'll have to do what's called an install from hard disk.  

Follow this howto:

[https://help.ubuntu.com/community/Installation/FromWindows?highlight=%28installation%2Ffromwindows%29](https://help.ubuntu.com/community/Installation/FromWindows?highlight=%28installation%2Ffromwindows%29)

Or better yet, **UNetbootin** (new):
[http://lubi.sourceforge.net/unetbootin.html](http://lubi.sourceforge.net/unetbootin.html)

Edit: As Pumalite said above, you'll need to use Xubuntu, but I'm not sure if the UNetbootin installer will give you a choice of desktops.  2nd EDIT: It does and xubuntu is among the choices, just like on the alternate install.  When you get to the "Select and Install Software" stage, choose "Xubuntu".

Good luck!
