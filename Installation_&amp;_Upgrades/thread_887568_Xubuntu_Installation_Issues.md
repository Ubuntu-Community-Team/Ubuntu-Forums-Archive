### Post by purplepaint on 2008-08-12
As far as I understand it, the Live CD doesn't work on computers with relatively low amounts of RAM, while the the Alternate CD uses a simple text interface (which is still easy to use) instead, so I would recommend it.  Good luck!

---

### Post by snowpine on 2008-08-12
According to the Xubuntu website, 192 mb is the minimum ram for the live cd and 128 is the minimum for the alternate cd. The alternate cd should work on your computer, however, you might want to consider picking up an extra stick of ram for better performance. Good luck!
