I had headaches with this error when getting application data...
... the problem was also manifesting in Add/Remove as saying the software was incompatible with my computer type (i386). 
I later realised all the 'gzip' errors werefor files on  australian servers, 
so went to System - Administration - Software Sources and changed "Download From" to "Main Server"

Viola. All is well.
