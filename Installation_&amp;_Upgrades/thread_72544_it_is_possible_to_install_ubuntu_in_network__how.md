### Post by orev on 2005-10-06
Very possible, and not very difficult.

goto:

System>Networking

and then input your network settings.

You should be able to conect to the network without restarting the computer.

If you are trying to connect the ubuntu machine directly to a router/modem, it may find the network automatically on boot.
