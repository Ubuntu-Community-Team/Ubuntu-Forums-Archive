### Post by 2F4U on 2013-05-24
It is the decision of the Kubuntu project which browser they include by default. This is usually not a big issue, since the browser they provide by default should be enough for testing, and once the system is installed, you can add any software you want.

### Post by ronnoc on 2013-05-28
Since Kubuntu tries to stay as close as possible to upstream KDE, it uses the default browser in KDE which is rekonq. Having said that, as noted above, you can easily install any browser you like after the live session is up and running. rekonq is based on WebKit, and is a very capable browser in it's own right.
