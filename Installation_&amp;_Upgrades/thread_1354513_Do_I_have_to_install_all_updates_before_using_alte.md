### Post by slakkie on 2009-12-14
It is standard advise to upgrade your system to the latest and greatest within your current release before upgrading, however.. 

If you only have jaunty repo's in your sources.list (so no jaunty-security, jaunty-updates, etc) then you can upgrade as well, without having to enable those repositories and upgrade your machine. The package versions on the alternate CD are higher and will be installed regardless.

You can use use the alternate CD to upgrade without having to update all your current packages before the upgrade.

And if you have a look in my sig (Upgrade Ubuntu) it will show you a method to install from the image of the alternate CD rather then burning it and then upgrading. Best of luck!
