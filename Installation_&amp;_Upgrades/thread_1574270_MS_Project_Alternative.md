### Post by lechien73 on 2010-09-14
OpenProj can open Microsoft Project .mpp files. It can be downloaded from [here]("http://openproj.org").

Visio uses a proprietary file format which, it appears, has not been reverse engineered yet. Your best bet is either to install Visio under Wine on Ubuntu, or use Dia - which can be installed through the Ubuntu Software Centre.

Dia will open and save .vxd files, and you can export to .vxd from Visio.
