### Post by ajgreeny on 2010-05-13
If you mean that nothing happens when you click on the **shutdown** or **restart** button, then yes it happens to me and others occasionally, though I've not been able to find any pattern to it.

There's no need to crash the system!  Just open a terminal or go to the cli with Alt+Ctrl+F1 and use the command ```
sudo poweroff
``` or ```
sudo reboot
``` or make a launcher to run one or both of those commands in terminal, on your desktop for those occasions.
