### Post by Pumalite on 2007-09-20
If you don't know, it's probably integrated. You need Alternate CD: [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Do md5sum on iso, burn at 4x
Install Grub to MBR (default)
That will give you menu to boot either OS
