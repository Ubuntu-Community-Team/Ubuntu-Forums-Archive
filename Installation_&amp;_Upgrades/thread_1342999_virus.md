### Post by audiomick on 2009-12-01
You should give us more information.
Which new ubuntu disc, where did it come from.
What is Comodo?
OK, I just looked. It is a windows virus scanner thing.

So presumably you put the cd in the drive whilst windows was running, and the virus scanner gave you a warning.

If the disc is one that you downloaded the iso for, and burnt yourself, it is very unlikely that there is a virus on it, unless you have a virus on your computer that smuggles itself into cd's that are burnt from your computer. I don't know if such a thing even exists.

More likely is that your Comodo is over cautious, can't recongnize what is on the cd, and just assumes it must be a virus.

You should rather boot into the cd and run the self check that is in the start menu. If that checks ok, the cd should be ok.

There are virtually no virus or malware problems with Linux systems.
