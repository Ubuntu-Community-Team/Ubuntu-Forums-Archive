### Post by mvs1207 on 2012-05-19
I believe that this happens when you have lost internet connection when upgrading.  This happened to me as well.  I think this window is letting you know that updating MS-FONTS has failed as it requires additional files to be downloaded.

If you think you upgrade is not complete try this link: [https://help.ubuntu.com/community/SynapticHowto#Broken_Upgrade_or_Installation](https://help.ubuntu.com/community/SynapticHowto#Broken_Upgrade_or_Installation)
