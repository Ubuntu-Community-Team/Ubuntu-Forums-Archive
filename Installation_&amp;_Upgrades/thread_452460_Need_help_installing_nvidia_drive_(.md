Although I have never had Dapper installed on my system, I do believe that was a new implementation in Fiesty. Dapper does not have the restricted drivers under system administration. Depending upon your grapics card (and I am assuming you are using a nvidia card) you would need to get the drivers through the synaptic package manger. You could also get them through automatix, that is what  had done in Breezy. Also in terminal you could use ```
sudo apt-get install nvidia-glx 
or 
sudo apt-get install nvidia-glx-new
```
Which one you use will depend on the graphics card you have installed. You may want to do a search for your graphics card and see if it is supported.
