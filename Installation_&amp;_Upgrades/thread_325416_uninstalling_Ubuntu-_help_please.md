### Post by cilynx on 2006-12-25
You'll still need to reclaim the space using a windowsy tool.  Usually Windows can't see Linux partitions.  If you boot the Ubuntu Live CD, you can nuke your Ubuntu partition.  Once you've done that, you can reclaim the space from inside of XP.
