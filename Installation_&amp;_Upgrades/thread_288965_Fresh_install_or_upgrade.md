### Post by angkor on 2006-10-30
Other. Download the edgy .iso of your choice burn it to a disc and back up all of your important stuff.

Try the upgrade (I'd use aptitude but maybe you should try the recommended update-manager) if it fails, try to fix it. Can't fix it? Do a clean install.

What I've found to be helpful when upgrading is commenting out all exotic non-official repositories (put there by yourself or apps like automatix etc) in your /etc/apt/sources.list. 

ps. My upgrade was a breeze, I've upgraded Debian in the past and Ubuntu twice since Hoary, this (third one on ubuntu) was the best one by far with an amazing 0 errors.
