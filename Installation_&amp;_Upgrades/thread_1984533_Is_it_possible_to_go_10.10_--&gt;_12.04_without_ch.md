### Post by darkod on 2012-05-22
At first moment, no.
But once the 12.04 is up and running, you can install Gnome Classic (package name is actually gnome-panel) that looks almost the same as the interface before Unity.
Also, depending what settings you have touched, not sure if you can get something broken during the upgrade if there is no equivalent in 12.04.

And since 10.10 is not an LTS, you can upgrade directly online. But with the 12.04 and internet connected, if you boot with the cd it should offer you upgrade directly to 12.04 using the cd + internet (not sure why it needs it but users have reported the upgrade option is there only if you are connected to the internet). So it's probably best to have wired connection to internet while doing that.

You don't want the wifi breaking and messing up your upgrade.
