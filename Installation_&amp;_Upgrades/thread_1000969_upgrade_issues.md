### Post by Partyboi2 on 2008-12-03
Try disabling any 3rd party repos from Software Sources (System>Admin>Software Sources>3rd party Software) also try disabling "pre-release" and "unsupported" updates from under the "update" tab if they are enabled then try doing a partial upgrade.
