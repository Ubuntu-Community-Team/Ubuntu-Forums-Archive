### Post by drs305 on 2009-05-07
Yes you can. Select manual partitioning and designate the ubuntu system and the other (swap?) partitions. Elect to have them formatted assuming you want a clean install. Just make sure to designate the correct partitions and it will leave Windows alone.

Of course, back up your important data before doing any partitioning work.
