### Post by penguin386 on 2009-09-05
My Mini 9 Netbook does not have a CD/DVD drive.  I wanted to upgrade from Ubuntu 8.04 to 9.04.  I looked on the web for ways to make a bootable USB drive.  I was unsuccessful creating one.  Then I ran across this:

[http://www.amazon.com/gp/product/B002JDVMJO/ref=ox_ya_oh_product](http://www.amazon.com/gp/product/B002JDVMJO/ref=ox_ya_oh_product)

It's a bootable USB flash drive that looks like a Penguin.  Way cool!  I ordered one of these for about $13 including shipping.  It's so easy to boot and install Ubuntu from this USB flash drive.  It's better than downloading an ISO file and burning it to a CD or DVD, then trying to boot from my flaky USB CD/DVD drive.

It boots to a Live version of Ubuntu.  You can run Ubuntu just fine from the USB flash drive, which makes it a great disaster recovery tool.  You can also install Ubuntu while booted from the USB drive.  This by far was the easiest Linux installation I've ever done.  

Also, some proceeds from these USB Penguin flash drives will be donated to the World Wildlife Federation to Save the Penguins!
