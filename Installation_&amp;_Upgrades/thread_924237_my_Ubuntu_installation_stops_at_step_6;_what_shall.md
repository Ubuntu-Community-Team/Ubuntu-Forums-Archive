### Post by nvance on 2008-09-19
Greetings,

Try to do the install with the text format (I think this is still an option) or do the install within XP (not booting from the livecd).  

Lastly, depending on where the livecd is from, it might have encountered an error when it was burned.  Boot from the cd and choose the option to check the cd for errors to ensure that you got a clean burn.  I have had the issue also that I just had to re-burn the cd/dvd at a lower rate of speed and then upon trying to re-install it worked properly.  Also, I think there is an alternate install ISO file that you can download that might work.

Good Luck, I hope you can figure out the issue.
