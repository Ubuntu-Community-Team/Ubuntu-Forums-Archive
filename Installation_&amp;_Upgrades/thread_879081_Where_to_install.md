Can Ubuntu be installed on a secondary drive other than "C" - one that has been repartitioned?

---

### Post by xen-uno on 2008-08-03
It can be installed on any unformatted partition or even formatted if the data is expendable (it will be re-formatted as ext3). You need an additional small partition (say 2 GB) for the Ubuntu swap file.
