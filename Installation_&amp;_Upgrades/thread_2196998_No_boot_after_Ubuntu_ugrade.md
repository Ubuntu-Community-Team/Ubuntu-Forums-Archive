I have a dual system, with Windows 7.
The loader is Ubuntu.

### Post by fantab on 2014-01-01
Boot Ubuntu 13.10 DVD/USB and 'Try Ubuntu without installing'.
Download and install '[**Boot-Repair**]("https://help.ubuntu.com/community/Boot-Repair")'.
Run the '*Recommended Repair*' and make note of the LINK it creates.
BR should be able to fix the boot for ya. If it doesn't, then post the LINK here.
