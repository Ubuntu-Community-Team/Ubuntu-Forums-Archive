### Post by coldraven on 2013-06-18
Were you trying to copy a file bigger than 4GB to a FAT32 ext. drive?
That won't work, you would need to format the ext. drive to NTFS or ext4.

Edit: max file size for FAT32 is 4GB minus 1 Byte.
