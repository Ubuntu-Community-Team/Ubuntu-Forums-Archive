### Post by chadeldridge on 2007-10-18
I figured this out, so if anyone else is having the issue I hope this helps:

sudo apt-get remove tzdata

answer its omg omg yes im sure message

sudo apt-get install tzdata 

rerun the update manager

### Post by daveysmith on 2007-10-19
A sorta quicker way, uninstalling far less packages:

As root:

dpkg -r tzdata
apt-get install tzdata

It then re-installed tzdata and carried on with life
