### Post by oldfred on 2014-10-21
What brand & model system?

Some systems have modified UEFI to only boot the Windows entry. You have used one of the work arounds to boot Ubuntu.
UEFI has a one time boot entry that does not seem to have the Windows only restriction, so from Windows the BCD is resetting the UEFI to have a one time boot of ubuntu allowed.

There are several other work arounds. Most find renaming bootx64.efi and copying grub as bootx64.efi and booting the hard drive also works. UEFI does not restrict hard drive from being in boot menu.
