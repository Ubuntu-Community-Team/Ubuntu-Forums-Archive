### Post by renzokuken on 2007-10-19
ahh the joys of the command line, the most powerful thing at your disposal.

to edit smb.conf use nano i.e.

```
sudo nano /etc/samba/smb.conf
```

to add users

```
sudo useradd *name* -m -G users
sudo smbpasswd -a *name*
sudo /etc/init.d/samba restart
```

should you get really stuck and want a gui for "fallback" reasons, you can install it via 

```
sudo apt-get install ubuntu-desktop
```

for gnome...or kubuntu-desktop for KDE, or xubuntu-desktop for XFCE

as for ntfs drives, ntfs-3g is great......just a case of installing it and editing fstab (although if you did have a gui you could run **sudo apt-get install ntfs-config** and a wizard will do it for you
