### Post by cfabbriumd on 2007-09-15
Heres a few things:

1) 6.04 (Dapper)/6.10 (edgy) are older versions. You may want to start off with the newest version 7.04 (feisty) or even wait until 7.10 (gusty) comes out offically in October. To obtain a copy, you can go to:

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

2) All installation CDs have a live portion and have an installer within the live version. There should be an icon on the desktop for that. Look in the applications menu; it should be there as well.

3) If all else fails, download the alternate CD by clicking the checkbox:

> Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer.

This will initiate a text-based install. It's designed for lower end computers, but I like to use it anyway myself.

Hope this helps and welcome to Linux :)
