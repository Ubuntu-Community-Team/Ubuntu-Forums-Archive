### Post by Partyboi2 on 2009-03-18
You can boot without the splash screen to get a better idea of what is going on. At the main menu press F6 and change 'splash' to 'nosplash' and press enter. 
If you choose the first option at the main menu to 'try ubuntu without making changes' you should be able to test how well it runs before proceeding with the install. (it should load up to the ubuntu desktop)
What are your system specs?
