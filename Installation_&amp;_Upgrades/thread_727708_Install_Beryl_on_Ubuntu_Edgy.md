### Post by PmDematagoda on 2008-03-18
You can install Beryl on your VGA card, but it would be a bit of a waste as Beryl is outdated and so is Edgy.

Download Ubuntu Gutsy(latest version of Ubuntu) from [here]("http://www.ubuntu.com/getubuntu/download") and install it. Ubuntu Gutsy also brings an added benefit by having Compiz-Fusion(the new Beryl) built-in so this means that using compositing effects is not much of a trouble.
