theoretically, you should be able to dual-boot ubuntu and winxp, without affecting your existing winxp installation and files - many people have such setups working on non-raid systems. i am not sure about the raid setup though, as i am not familiar with the workings of raid. i did find this page, which might help:

[http://www.ubuntu-in.org/wiki/SATA_RAID_Howto](http://www.ubuntu-in.org/wiki/SATA_RAID_Howto)
