### Post by Rocket2DMn on 2008-03-10
OK, let's try a few things to at least get you a clean slate.  First, start by removing Envy:
```
sudo envy --uninstall-all
```
Now purge the fglrx drivers that you have:
```
sudo apt-get remove --purge xorg-driver-fglrx
```
Then reconfigure your Xserver and use the "ati" open source driver (which I see is already in your xorg.conf, but do it fresh anyway) - [http://ubuntuforums.org/showthread.php?t=690760](http://ubuntuforums.org/showthread.php?t=690760) .  If that doesn't get you in, reconfigure to use the vesa driver.
Now you should be back in Ubuntu, though you may not have full graphics capacity.  Now you should be able to correctly enable the restricted drivers from System->Administration->Restricted Drivers Manager and choose the ati ones.  You may need to restart X after that with CTRL+ALT+BACKSPACE.
