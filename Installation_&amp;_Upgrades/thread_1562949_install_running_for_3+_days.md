### Post by sikander3786 on 2010-08-28
Might be either the HDD or the CD-ROM Drive or the CD-ROM Disc going bad. Or it might well be problem with the data cables running between them.

If it would've happened to me I would

1. Change the HDD and CD-ROM data cable.
2. Change the CD-ROM Drive if available.
3. Check disc for errors.
4. Memtest to ensure RAM is in good condition.
