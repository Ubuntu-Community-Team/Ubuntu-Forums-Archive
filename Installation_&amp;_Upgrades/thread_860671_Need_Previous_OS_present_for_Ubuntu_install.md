### Post by mikewhatever on 2008-07-15
No. All you need to load Ubuntu from the Live CD is to set the BIOS to boot from cd-rom first. In fact, you can load it with or without the hard drive.

---

### Post by Shazaam on 2008-07-15
Nope. Blank hard drives work fine. You will probably need to go into the Dell's bios and set the cd/dvd drive to boot first (Boot Order). I think F12 is the key to get into the bios (or F2).
