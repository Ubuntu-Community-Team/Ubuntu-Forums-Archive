### Post by fasho on 2006-08-14
I am trying to install the latest version of Ubuntu on my pc. I get to the main screen and hit enter to "Start or Install Ubuntu". A dialog box comes up, which says it is loading a kernal. It finishes and reboots the machine and I am back to the main Ubuntu screen.

In circles I go. I have an HP machine. I think the bios is blocking the install, but I am  not an expert. 

Can someone help?

Fasho

---

### Post by cvanp11 on 2006-08-14
Go to the boot menu (the one that first comes up when you start the CD), press F6, then type "acpi=off" and press enter.  This should make it work.

### Post by fasho on 2006-08-14
That worked. Thanks.
