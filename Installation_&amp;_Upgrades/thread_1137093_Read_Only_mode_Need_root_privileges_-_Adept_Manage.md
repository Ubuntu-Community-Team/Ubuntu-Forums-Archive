### Post by wicky_ts on 2009-04-25
I am using the same version. To install any package the easiast way is to use the synaptic package manager. in your desk top menu 

system>administration>synaptic package manager

you can search the package you want to install. and it will show all the packages with the searched word. 
I think what you need is the flash plugin to see online moves, videos. if that is the case search for 

adobe-flashplugin

then the package will search and give you the results. mark it and click the applly buttan.
