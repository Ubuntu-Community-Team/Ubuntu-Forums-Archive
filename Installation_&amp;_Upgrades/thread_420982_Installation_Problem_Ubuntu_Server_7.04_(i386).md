### Post by Big Ed on 2007-04-24
If you need a graphical user interface (gui), then Ubuntu Server is not for you.  Server is a stripped down version that supplies only what is needed on a server.  It does a very good job at that, but no gui is included.

If you are setting up a desktop machine, then use the desktop iso.  
If you are setting up a server and you need (or just want) a gui, then use the desktop iso.  
[http://www.ubuntu.com/products/whatisubuntu/desktopedition](http://www.ubuntu.com/products/whatisubuntu/desktopedition)

If you just want to set up a lean server, then the server iso is for you.
[http://www.ubuntu.com/products/whatisubuntu/serveredition](http://www.ubuntu.com/products/whatisubuntu/serveredition)

Ubuntu Server gives you LAMP (Linux, Apache, MySQL and PHP) as an installation option.  This is a real time saver since it will be mostly configured and working when you finish the install.
