### Post by deadflowr on 2021-11-12
Run
```
sudo add-apt-repository --remove ppa:vbernat/haproxy-1.6
```
to remove the dead ppa.
Luckily, that is all you should need to do as the ppa never installed anything.

It's a general problem with tutorials as the advice given can sometimes become outdated. And outdated quickly, in some cases.
(It's one feature I like about the omgubuntu website as it throws a banner warning on any page over a year old about how the information the article contains may already be out-of-date.
Not all tutorial websites or videos have such a feature, instead opt to simply set it and forget it. Which leads to problems like the one this thread has.)
