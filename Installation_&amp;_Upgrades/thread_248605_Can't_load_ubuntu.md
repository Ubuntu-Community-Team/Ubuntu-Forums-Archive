### Post by galileon on 2006-09-01
you need to set your bios to boot on cdrom first: its different for different computers, but generally, try pressing F1, F2, F8, F12, DEL, etc, until you either arrive at the bios screen or see a boot menu (be careful, if you see the windows boot menu, you pressed the f8 too late)

if you get to bios, look where it says boot device priority, and try to get CDROM first, Hard disk second for example...

on my desktop, f8 shows a bios boot menu, which asks if i would like to boot into hard disk or cdrom...
