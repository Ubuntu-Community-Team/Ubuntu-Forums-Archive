### Post by Marlonsm on 2009-04-08
If you're talking about the partition size the point is that you'll have more space for things you use on Ubuntu.
The install size could be as small as 1Gb (or even less), but then, you wouldn't have space to store documents and programs in the Ubuntu partition.
