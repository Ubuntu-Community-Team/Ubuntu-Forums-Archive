### Post by az on 2006-12-23
There is no server edition - there are just packages for everything.  The server install cd just contains the LAMP server packages as well as the base system.  So if your wireless card work on the desktop, it will work on a server.  It's just another network interface.

See here for a beginner howto on hosting a web server:
[https://help.ubuntu.com/community/WebServerHosting](https://help.ubuntu.com/community/WebServerHosting)

for other server software (like ftp or mail,) see:

[https://help.ubuntu.com/community/Servers](https://help.ubuntu.com/community/Servers)
