### Post by Madskillzz on 2011-04-10
Worked for me  :To  upgrade from Ubuntu 10.10 on a desktop system, press Alt+F2 and type  in  "update-manager -d" (without the quotes) into the command box.  Update  Manager should open up and tell you: New distribution release  '11.04' is  available. Click Upgrade and follow the on-screen  instructions.
