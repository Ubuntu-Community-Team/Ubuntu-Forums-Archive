### Post by Mark Phelps on 2010-05-15
It's not a "demo version"; it's a LiveCD.  The version on disk is the "full version", not a demo.

You should NEVER force an install on an untried machine -- bad idea.

Boot from the LiveCD and try it out first.  Anything that doesn't work will range from trivial to impossible to fix after installation.

Post back here with any problems you encounter using the LiveCD.
