### Post by frodon on 2009-03-10
This is an issue few users are having for ages.

Things to try to get it fixed :
- Update graphic card drivers, try with vesa drivers to see if you have the problem with it too.
- Clear your .gnome2 directory (after backing it up) and restart your session and see if you have still the issue. This will unfortunately clear some of your settings but you can put back your files one per one after this till you get again the issue.

This annoyance is most of the time the result of graphic driver bugs or mess in .gnome2 directory config files.

Test this and please post back what you get.
