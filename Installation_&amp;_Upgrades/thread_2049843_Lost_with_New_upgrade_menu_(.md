### Post by irv on 2012-08-29
A little more information. What did you upgrad from? and what did you upgrade too?
I am assuming you upgraded to Ubuntu 12.04 with Unity as a desktop.
If so just click on the Ubuntu logo in the top left hand corner of the screen. This is called the dash. You can also open it by pressing the "super Key" which is the windows key on your keyboard. Also holding down the super key will open a box with many other shortcut keys.
One the dash is open you can run programs from there. You can also just start typing the program you want and it will find it for you.
The launch on the left side of the screen is where you keep your most use programs. you can lock them to the launch by opening them and then when they appear in the launch just right click on the icon and select lock to launcher. I would suggest you go on line and do a search for help with using Unity. Also there are some good books listed in the Software center which should be on the launcher.
Hope this helps

### Post by irv on 2012-08-29
By the way I just wanted to say that those little grey icons in the dash are called lens. You can add lens to your dash. Take for example News Lens. you can it by opening a terminal and typing:
```
sudo add-apt-repository ppa:scopes-packagers/ppa
```
```
sudo apt-get update && sudo apt-get install unity-lens-news
```
Go to this link to learn more about the [News Lens]("http://www.omgubuntu.co.uk/2012/05/unity-news-lens-helps-you-find-news-fast").
Also if you want to install other Lens go [HERE]("http://www.omgubuntu.co.uk/?s=News+lens"):
See what you would be missing if you went back to a menu OS. This is one reason why I would miss Unity. And there are many other reasons I would miss it.
