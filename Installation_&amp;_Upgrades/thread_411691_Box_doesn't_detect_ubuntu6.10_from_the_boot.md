### Post by ajgreeny on 2007-04-17
Make sure the CD is set to boot before the hard disk, and also check that your CD is an image of the iso file not just a copy.  There should be several folders and files on the CD if you look at it in a file manager like windows explorer, assuming you are on windows at the moment.  If there is just one file (the iso) then it is burnt wrongly and needs doing again, making sure you chose "Burn a disk image" or similar when doing so.
