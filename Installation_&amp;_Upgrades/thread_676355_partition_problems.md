### Post by Pumalite on 2008-01-23
If you are using Vista, you should allocate space for Ubuntu from WITHIN Vista. If XP, then get Gparted Live CD: [http://gparted.sourceforge.net/download.php](http://gparted.sourceforge.net/download.php)
Burn to disk, boot from it and resize your XP partition. Then install Ubuntu.
