### Post by badrinarayan on 2005-07-14
There is no root password. You use [font=Courier New]sudo command[/font] whenever you need root privileges and supply **your** password. If you insist on a root password, you may want to see [http://www.ubuntuguide.org/#setchangeenablerootpassword](http://www.ubuntuguide.org/#setchangeenablerootpassword)

### Post by ubuntp on 2005-07-14
There is no root password. You can switch to root by using "sudo su", the password is  the password for the user you created.
