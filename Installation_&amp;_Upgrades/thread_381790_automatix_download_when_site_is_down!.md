Automatix constantly updates its packages with critical security and other updates. You should advise users to upgrade to the latest versions of automatix before attempting to install anything from it.
The easiest way to do it is run automatix once, and close it (which will add the automatix repository to sources.list)
then do:
```
sudo apt-get update
sudo apt-get upgrade
```
Now run the latest version of automatix and install stuff from it.
