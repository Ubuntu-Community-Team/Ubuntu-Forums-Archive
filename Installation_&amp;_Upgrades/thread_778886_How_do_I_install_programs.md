### Post by hok on 2008-05-02
There are several ways to install programs depending on what you want to install.

Usually you don't need to download the program first.  Ubuntu "knows" where to get it. You just say what you want to install and Ubuntu fill download and install it for you.

The easiest way is to use the "add/remove" option on the "applications" menu.

The above only has a limited set of applications.  For other applications use the synaptic installer.  Usually all you need to do is find the package you want to install in the lists (eg. do a search) then tick anything you want to install, click apply and follow the menu.  Again, you don't need to download the package first, Synaptic will do that for you.

If the package you want to install is not listed under synaptic then you may have to add a repository (a server that synaptic will include when searching for packages to download).

Finally, Windows programs will not install on Ubuntu.  There is a package called "wine" that can run some Windows programs on Linux if that is what you are trying to do.
