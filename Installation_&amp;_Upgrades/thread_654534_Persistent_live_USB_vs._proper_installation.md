### Post by Herman on 2007-12-31
I haven't been using mine for very long, only a two or three weeks so far and I make a lot of changes and re-install very often so I don't know how well mine will go in the long run. I read somewhere that they 'wear out' if we do too much copying in and out? I'm not sure if that's true or not yet.

I can tell you that yes, we can update the software, add software, add users, and they seem to behave much like a real hard disk installation. I have a 4 GB USB Flash memory stick and it cost me around $60 though, and about 699 MiB of it is occupied by the operating system. I have hard drives I paid $100 each for brand new that hold 160 GB, so flash memory is expensive real estate on a per GB basis, Only one of my four computers has a BIOS that can boot the USB flash drive, the others can only read it, so it's not as usable in almost any computer like a Live CD would be.

For speed though, it's like lightning! I don't know how to measure it, but it seems faster than a hard disk install, maybe it's my optimistic imagination though. I'm having a lot of fun with mine.
