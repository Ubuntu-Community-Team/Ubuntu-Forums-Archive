### Post by Elshadii on 2006-10-25
There are a few things that may be contributing to your situation.  Are the systems giving you problems using older cdroms? if so then you might have written your Ubuntu cd to fast example: some 32x cdroms have difficulty reading 52x burnt cds.  Another question: the Ubuntu cd's are live cds which means that the whole operating system is loaded into your memory (ram) not on the hard drive until you hit the install button.  So, if you are using lower than 256 megs of ram then you possibly run into trouble there.  Maybe some ohter reply's will render more info for you.  How old are your systems?
