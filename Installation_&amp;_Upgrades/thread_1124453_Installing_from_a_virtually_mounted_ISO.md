### Post by linuxuser21 on 2009-04-13
To check if there's any problem with your CD, select "check CD for defects" on the first screen.  If there is no problem with the CD, your computer might have some problems to why it's not installing.  In that case, it wouldn't do any good to install it from a virtual ISO.
