### Post by darkod on 2011-11-09
If you can, download 11.10 and burn a cd. Boot with the cd in live mode (Try Ubuntu option) and see how it runs. If there is any hardware incompatibility running the live mode should show it.
If live mode runs OK, most probably the new system will too.

Another option: Leave 11.04 since you are happy how it runs. In six months, in April 2012, the new LTS version 12.04 comes out. Then you can consider doing a clean install of the latest LTS version over your 11.04.
