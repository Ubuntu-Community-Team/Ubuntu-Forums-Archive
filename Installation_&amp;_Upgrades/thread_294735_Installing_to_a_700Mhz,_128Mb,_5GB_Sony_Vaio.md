### Post by sawjew on 2006-11-07
I have a Compaq Evo notebook with 850Mhz PIII and I initially had only 128 Mb of ram so I installed Xubuntu.  I was very impressed with how well this computer went with Xubuntu compared to Windows 2000 which I have dual booted.  Xubuntu was appreciably faster than win 2000 but I ended up changing over to Ubuntu because of the lack of a decent network browser in Xubuntu.  I really like nautilus's ability to browse a network and access files without the need for mounting/unmounting all the time.  

Apart from the network browsing I have found Xubuntu to be excellent and I am eagerly waiting the addition of a network browser to the Xubuntu system then I will probably return to it.  I have found Gnome to be a little slower but not unbearably so, but I upgraded my ram to 384Mb and the system now performs quite well.

I found the biggest hassle with only 128Mb ram was trying to use firefox and openoffice, there was no way I could use them both at the same time without the system slowing to the point of unusability.

My advice is use Xubuntu if you don't need a network browser and if you need to use openoffice get some more ram.
