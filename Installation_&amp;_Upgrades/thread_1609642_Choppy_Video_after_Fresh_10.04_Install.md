### Post by moonguide on 2010-10-30
I have an Nvidea GeForce2 MX/MX 400 video card in my machine. Yesterday I put a new disk drive in and installed 10.04 on it. The machine had been running 8.04. I followed the advice to run the Nvidia driver. Things look fine except when I run a program that expects the high-end video hardware. The only one I currently have installed is Foobilliard. It worked smoothly when running 8.04, but now the video is choppy. It acts as if the acceleration isn't there.

I recall turning off some eye candy thing with 8.04. It took some searching, but I finally found the "Visual Effects" tab in System>Preferences>Appearance. I found it set to "Normal." I changed it to "None" and now the billiard balls roll smoothly.

I'm assuming this has something to do with the speed of the machine and the version of the video hardware. I'm running 10.04 on an Athlon XP 2100+ with 1 Gb of RAM.
