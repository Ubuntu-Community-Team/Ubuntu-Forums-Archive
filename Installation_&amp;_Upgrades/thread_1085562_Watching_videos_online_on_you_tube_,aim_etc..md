### Post by ianto` on 2009-03-03
This problem  can be overcome by typing  the following command into terminal.

```
sudo apt-get install flashplugin-nonfree
```

That command installs the Flash plugin that is utilised by Firefox, Epiphany and other browsers so that you can watch flash videos online. Restart your web-browser after you install the plugin.


I hope that this is helpful for you,
Ianto.
