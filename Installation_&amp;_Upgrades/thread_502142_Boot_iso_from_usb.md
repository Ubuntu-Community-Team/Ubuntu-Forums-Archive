### Post by Daveth on 2007-07-16
you would have to have a motherboard that supports booting from a usb device (mine does not), then get into the bios when you start your pc up (usually the delete key or spacebar), then change the boot preference order so the usb gets checked first. This will find the iso file and off you go. If the mb does not allow this then you will have to get your files from usb to cd/dvd
