### Post by mastablasta on 2013-01-10
no problems. i have it installed on mashcine but it has only 1.3 GB ram so there is not much difference between 32bit and 64 bit. i also have another one wiht only 256MB ram thta is 64 bit and has 32 bit OS. and one with WinXp home (which is again 32bit OS).
 
the default kernel in 12.10 is PAE enabled kernel so ram over 4 GB will be addressed. all the way up to 64 GB RAM.
more info on PAE: [http://en.wikipedia.org/wiki/Physical_Address_Extension](http://en.wikipedia.org/wiki/Physical_Address_Extension) 
 
however why would you install 32 bit OS on 64 bit CPU (especially if you have more ram?). 64bit version should work much faster if you have plenty of ram.
