Here is the release notes for 9.04 Jaunty

[http://www.ubuntu.com/getubuntu/releasenotes/904](http://www.ubuntu.com/getubuntu/releasenotes/904)

Though it says 256mb minimum, remember that some of your memory is being used by your graphics card.

4 suggestions :

1.  Add more RAM or,
2.  Use the alternate install disc (which is text based, less resource intensive when installing) or
3.  Consider Xubuntu or,
4.  Consider a WUBI install.

Some reference links:

[http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)
[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

Good luck.  Back up your files.
