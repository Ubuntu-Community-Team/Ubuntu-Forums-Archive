### Post by frodon on 2006-06-01
The desktop CD don't use the same installer, it use a graphical installer which is more user friendly (it allow you to surf on internet during the install for example) than the traditional text based installer of the alternate CD.
However only the alternate CD allow LVM and RAID setting options.
More infos there : [https://lists.ubuntu.com/archives/ubuntu-devel-announce/2006-May/000140.html](https://lists.ubuntu.com/archives/ubuntu-devel-announce/2006-May/000140.html)

### Post by Hezekiah on 2006-06-01
Also, if it matters for you, you need to use the alternate install cd if you want to install grub anywhere other than the MBR.
