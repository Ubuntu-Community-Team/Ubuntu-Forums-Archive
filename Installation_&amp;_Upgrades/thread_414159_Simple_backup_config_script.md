### Post by gregh on 2007-04-19
This a the 3rd version of the backup script that will allow your personal settings (settings under your home directory in hidden folders) to be zipped and stored prior to wiping and reinstalling so you dont lose all your settings etc..

It currently looks for your personal setting for  ssh, evolution, thunderbird, firfox/mozilla, googleearth, gnucash, gftp, gaim and postal2(saved games etc)

It is a perl app that that uses either a GUI or if you pass a -s switch it will run silently or automated.
