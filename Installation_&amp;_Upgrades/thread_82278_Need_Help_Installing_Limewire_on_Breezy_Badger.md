### Post by renesr on 2005-10-26

>Install applications
>Internet
>More applications
>Java Web Start 1.4
 
Go to [http://ubuntuguide.org/](http://ubuntuguide.org/)  and find in Unofficial Ubuntu Guide 5.04 "Limewire"
Follow the instructions .

That will do.
