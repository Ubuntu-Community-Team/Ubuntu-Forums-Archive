### Post by fidamehran on 2010-05-10
From what I know, you cannot install in an Unallocated space if you want to install Ubuntu from within Windows. You need to boot into the Ubuntu drive (CD or USB [if your motherboard supports it]) and then install it from boot menu before entering Windows. Then in the options of where you want to install, you can specify that you want it installed in the unallocated space.
