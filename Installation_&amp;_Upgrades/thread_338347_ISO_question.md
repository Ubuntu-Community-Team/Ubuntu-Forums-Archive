### Post by kebes on 2007-01-14
For the time being, Intel 64-bit chips use the standard desktop install. (The AMD 64 install is only for AMD systems.)

As for running VMWare, more RAM is of course always an advantage, but you should be able to run a VWMare image on a modern computer such as you describe without too much trouble. I'm not sure how fast WoW will be inside the image... but my guess is that it will be acceptable.
