I had the same problem.  I installed Ubuntu from CD and selected the "within windows" option.  This was "big mistake".  The install went through the install and rebooted.  I got a "missing HAL.DLL" during boot.  I was able to boot from XP cd and copy a copy of the HAL.DLL from my system backup. Long story short, I had to rebuild PC's Windows OS.  Recovery took 6 days.
I would not recommend installing ubuntu ... at least by taking the from "within windows" option.
