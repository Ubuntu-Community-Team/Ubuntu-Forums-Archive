### Post by Jagot on 2006-07-26
Ubuntu most certainly sounds ready for you.  I've used SuSE 10.1 and I had constant problems with the broken package management system - the only fix for which seemed to involve having to use the broken package manager!  I'll try again at 10.2 though.  That made it intolerable for me, but there are some decent guides around which help with setting up multimedia, like this one:

[http://www.thejemreport.com/mambo/content/view/254/1/](http://www.thejemreport.com/mambo/content/view/254/1/)

To be honest, if the package manager was working then there wouldn't be a whole lot of difference between the two in ease of use, but I did notice some things that were made slightly more difficult, such as the installing of my NVidia drivers, which had to be done from source with SuSE.

If ease of use is your thing you may want to check out Automatix:

[http://getautomatix.com/](http://getautomatix.com/)

It's the easiest way to install a ton of useful packages including those required for multimedia support.
