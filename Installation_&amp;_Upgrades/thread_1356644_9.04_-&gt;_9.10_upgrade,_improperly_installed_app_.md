### Post by rslee on 2009-12-16
After running the Jaunty to Karmic install from the upgrade manager, I'm booting to a command prompt.  After searching the forum and trying a few things, I got to a message saying something had been gotten interrupted and to run

```
sudo dpkg --configure -a
```

That was running along fine until this error:

```
Errors were encountered while processing: toggl-desktop
```

I tried to install that application some time ago and failed (it's 32-bit, I'm 64).  How can I remove it?  sudo apt-get remove toggl-desktop can't find the package.

Thanks!

---

### Post by rslee on 2009-12-16
OK,

```
sudo dpkg -P toggl-desktop
```

got rid of it... and now I have a GUI!
