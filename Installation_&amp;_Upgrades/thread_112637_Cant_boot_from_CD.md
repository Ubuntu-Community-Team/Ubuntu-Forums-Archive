I have just treid to use the Live CD on a Dell PC running Windows XP home SP2.

### Post by ekarni on 2006-01-04
Double check the boot order in BIOS -- is it set to attempt to boot from the DVD drive before the main hard drive?  If it's set to hard drive first, it may never read the DVD during bootup.
