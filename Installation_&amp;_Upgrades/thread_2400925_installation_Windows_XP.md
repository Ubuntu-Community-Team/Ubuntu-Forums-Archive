### Post by LHammonds on 2018-09-13
The suggestion is to not use Windows XP if you can avoid it.  Not because we are anti-Windows but because many of the best anti-virus packages will no longer install and work on the outdated platform.  There are no more WindowsUpdate patches to fix vulnerabilities.  And there are a ton of known vulnerabilities.

So like oldos2er said, if you are going to run WinXP, do not browse the Internet with it.  Also, be sure to configure your firewall to block all incoming traffic.  Windows tends to let all traffic through if on the same IP subnet.

LHammonds
