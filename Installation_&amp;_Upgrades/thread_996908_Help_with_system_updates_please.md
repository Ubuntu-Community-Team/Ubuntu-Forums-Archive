### Post by taurus on 2008-11-29
You have checked and your network connection is still working right now, right?  You can always try another server.  Go into System -> Administration -> Synaptic Package Manager -> Settings -> Repositories and use the Main Server.  Press Reload and see what happens if you update your machine now.  Remember, if you want to run it from a terminal, you first need to close down synaptic since you cannot have synaptic open and run apt-get from a terminal at the same time.
