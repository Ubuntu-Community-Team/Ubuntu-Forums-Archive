### Post by spcwingo on 2009-04-16
For some programs you have to log out then back in to see it.  Some you have to add to the menu by hand (right click "applications" then select "edit menu").  If it's a cli app, it won't show in the menu at all unless you add it by hand.

### Post by inobe on 2009-04-16
if it doesn't already appear try launching the app from terminal, if the app doesn't startup there may have been trouble installing it or the application dated.

if it does work from terminal you can create a quick launch icon somewhere.

right click on your desktop and select create launcher

in the dialog you will see four options

type: leave it set to application
name: name of the application
command: browse to the executable
comment: optional

at the top left click the no icon and browse to the apps directory and choose the png icon.

that will create a desktop icon, you can then drag it where you want it.
