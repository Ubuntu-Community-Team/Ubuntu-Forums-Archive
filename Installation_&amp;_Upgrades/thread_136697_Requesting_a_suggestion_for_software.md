### Post by Jimmey on 2006-02-28
Usually, to run an executable:

> /nameOfParentDirectory/./nameOfExecutable

For example, I want to run Blender from /home/james/blexec:

> /home/james/blexec/./blender.

If you install a piece of software with dpkg, then you can usually run that software, through the terminal, by typing it's name.
