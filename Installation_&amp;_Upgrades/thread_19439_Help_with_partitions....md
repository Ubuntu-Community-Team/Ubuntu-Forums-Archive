You may try to download the latest debian installer.

[http://cdimage.debian.org/pub/cdimage-testing/sarge_d-i/i386/rc2/sarge-i386-businesscard.iso](http://cdimage.debian.org/pub/cdimage-testing/sarge_d-i/i386/rc2/sarge-i386-businesscard.iso)

Write it (30.7M) to a usb stick and boot from that.  It should be able to partition your drive safely.
