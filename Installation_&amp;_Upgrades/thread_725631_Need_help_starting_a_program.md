### Post by Lord Landis on 2008-03-16
It's possible that citrix isn't installed in your path, but if you used apt to install it, it should be.  Try coding 'locate citrix' and see what comes up.  You will most likely get a very long output, but it should at least point you in the right direction.  If you have **no** results with that command, then it's not installed.
