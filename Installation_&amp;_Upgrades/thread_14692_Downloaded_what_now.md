 So what did you download? If you want to install on a HD you got to download an ISO file (an image of the CD) and burn that to a CD (as an image, not as a file). No extraction needed befor burning. Just make sure that the burning program detects ISO file as an image.

See: [http://www.ubuntulinux.org/download/](http://www.ubuntulinux.org/download/)

The same applies if you would like to use the LiveCD: you download an ISO image, burn that on a CD and use the DC to run the computer. Nothing gets installed in the HD.
