### Post by Snowbat on 2007-02-09
While the machine will run without a monitor, and usually without a keyboard (set Errors,Halt-On: All Except Keyboard or similar in BIOS) and even without a graphics card, an OS installer expects you to answer some questions.

One possible way to do it is with an [automatic installation]("http://archive.ubuntulinux.org/ubuntu/dists/breezy/main/installer-i386/current/doc/manual/en/ch04s06.html") but it will probably be easier for you to borrow a keyboard and monitor from friend/family/colleague/neighbour.  You will *need* a keyboard to set Boot from HD in your BIOS after installation anyway.
