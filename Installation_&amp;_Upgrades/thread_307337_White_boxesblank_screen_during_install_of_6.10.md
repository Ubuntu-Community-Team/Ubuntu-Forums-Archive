### Post by jefferson101 on 2006-11-26
I tried searching, but couldn't find anything about my problem.  Basically, while trying to install the alternate cd of 6.10, the screen goes black with 2 white boxes the size of cursors, one in the middle of the screen near the top and one on the same line on the far left.  The computer is completely unresponsive at this point.  It has happened 3 times in the same place, while it says it is installing xserver-xorg, I think.  I tried using the livecd, and it says that the partitioning is failed, which is odd because the alternate cd partitioned it just fine.  Oh and the Livecd runs fine, no graphical problems like the white boxes.
A little background on my hardware might help...  It's a Dell Inspiron e1505 running windows xp media center, that I want to leave untouched.  I am trying to install ubuntu on an external 250g usb hard drive so I can dual boot (windows from the internal and ubuntu from the external).  I've never installed ubuntu on a computer before, but I'm still optimistic, I just need a little nudge in the right direction here.

---

### Post by jefferson101 on 2006-11-27
After some more searching I actually found a few people with this problem, and was able to fix it.  Basically you just need to change the resolution setting to your normal resolution on the very first screen that loads from the install disc, and then it doesn't have the problem with the white boxes.  I'm now happily enjoying Ubuntu for the first time!  Very impressed by the way.
