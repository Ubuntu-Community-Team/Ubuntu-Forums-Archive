### Post by wpshooter on 2006-09-23
Try this.

Go to this website and download Killdisk and use it to WIPE the hard drive on the computer you want to install Ubuntu on.

[http://www.killdisk.com/](http://www.killdisk.com/)

Then when you start the installation from the DESKTOP/live CD, make sure that you let the CD spin down between each of the parameters that you are prompted with during the installation process.  In particular when the one regarding keyboard language comes up make sure you click on the type box at the bottom (you don't have to type anything in it, just click on it) and then go back up to the top and highlight the language choice again and then click on forward button.

Good luck.
