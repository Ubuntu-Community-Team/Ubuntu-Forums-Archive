Ideally you need to state your connection type, wireless, ethernet, adsl and
whether you're using broadband or dialup etc.

Whats the output from:

ping -c4 google.com

If the command returns 4 data packets, your mirror is down, if it returns host unreachable then your connection is not setup.
You must provide all the information for someone to help you
