### Post by SecretCode on 2010-01-08
If you're still running under vmware, it probably won't have access to the network card - virtual machines run with emulated virtual hardware (except the CPU itself).

If you want to go to a native (perhaps dual boot) install - your best bet is definitely downloading the ISO. To upgrade you'd have to download far more than that, and I don't believe it would be trouble-free anyway.
