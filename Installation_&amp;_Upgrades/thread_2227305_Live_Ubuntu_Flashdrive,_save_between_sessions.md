### Post by sudodus on 2014-06-01
You can make a ***persistent*** USB boot drive with the 'Startup Disk Creator' alias usb-creator or with Unetbootin. These programs offer the option to create a storage space for persistence (that you can save tweaks, documents, pictures etc).

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

and [Paul Sutton's Unetbootin how to]("http://zleap.net/unetbootln-usb-how_to/")

It is possible to add persistence to your current USB system, but I think it is easier to start from the beginning with one of these tools.

If you have problems with erasing the drive in 'Startup Disk Creator', you can erase the files with some other tool, or use Unetbootin instead.
