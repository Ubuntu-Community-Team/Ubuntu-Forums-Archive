### Post by critin on 2012-09-05
When 12.04 was first released, installing was a problem.  It's been fixed but you may have a bad download.  You might need to get it again.

There are three places to install from, the boot option, the desktop and one in the dash.  Open the dash and you should see the install icon.  You've tried two, now try the third.  That's the one that worked for me, other users could only install from boot, etc.  Depends on the hardware I think.
