### Post by giantclam on 2009-05-10
I have been working on trying to install this for days and nothing worked.  I finally got it working today by using the easy instructions on [this]("http://randomcryptography.blogspot.com/2009/03/installing-truecrypt-61a-on-ubuntu-904.html") page:

[http://randomcryptography.blogspot.com/2009/03/installing-truecrypt-61a-on-ubuntu-904.html](http://randomcryptography.blogspot.com/2009/03/installing-truecrypt-61a-on-ubuntu-904.html)

In case the above link is broken, basically the author said that you need to install 
libgtk2.0-0 first, in order to install truecrypt.

The version of truecrypt that says x86 is actually the 32-bit package.  The term "x86" is a holdover from the old PC days of 16-bit machines, the first of which was the 8086.  The current 32-bit machines are relatives of that architecture.  
  
I am using 9.04, but I am imagining and hoping it will work for you on 8.04.

Cheers!
