### Post by taurus on 2006-02-18
Is the drive a DVD/CD or only a CD?  Make sure you burn the ISO to the DVD as an image file, not a data file.  In other words, when you mount it under Ubuntu, you should see a few files and a bunch of directories (folders) on the DVD.  If you only a one big file, then you have just copied it from your harddrive to the DVD, which is a wrong process...
