### Post by Jussi Kukkonen on 2006-06-02
It won't be straightforward, but you don't necessarily need the usb hd at all (if you have an internet connection ). Take a look at [https://wiki.ubuntu.com/Installation/FromWindows](https://wiki.ubuntu.com/Installation/FromWindows)

If you decide to try, notice that the instructions are still for Breezy -- you'll at least  need to change the netboot installer download path if you want to install Dapper (just change breezy to dapper)
