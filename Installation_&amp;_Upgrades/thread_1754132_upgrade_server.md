You said you have 10.10 Server <> 11.04 Server.
The packages and services  "you have" will be updated in the distribution upgrade.  It will not add services that you don't have. (Server base + installed services + installed additional packages)

There is changes in the tty Graphics, where you can customise how the Text and TTY consoles display (adjust the resloution and palette), but other that that, you should not run into any of the graphics challenges that the desktop users are runinning into.  

There was a few change with the APM settings... minor kind of thing.
