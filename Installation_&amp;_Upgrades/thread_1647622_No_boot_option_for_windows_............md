### Post by coffeecat on 2010-12-17
> **rc12 said:**
> i checked some post and i ran boot_info_script software and here is the outcome >>

[noparse]@rc12, According to the bootscript output you posted you have two hard drives: the first one (sda) is 1TB, the second (sdb) is 160GB. The first drive is entirely occupied by the Ubuntu partitions - there is no sign of any Windows partition on it. The second drive has one NTFS partition, but no boot files so I suspect this was originally your Windows D: (or E:) drive.

Did you have Windows 7 installed to the 1TB drive? Because if so, I have bad news - it has been replaced.

If, however, your Windows C; drive was on the 160GB drive, then the boot files are missing and it should be repairable.

You are concerned about your data. If your Windows C: drive was on the 1TB drive, did you have any backups?[/noparse]
