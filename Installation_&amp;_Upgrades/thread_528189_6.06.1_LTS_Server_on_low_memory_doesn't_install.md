### Post by wolfear on 2007-08-28
Dr.-
I have a 6.06 server running (well..it was running till I goofed up, I'm reloading it now) on a 133mhz with 64 megs.
Slow as heck, but it does work and I didn't need a desktop installed. I run from command line, SSH or using Webmin from my workstation.

I'm using a 6.06 Server Cd that I burded myself from ISO. I select the "Install LAMP Server" option from the menu.

One thing..make sure your CD drive will read the CD.
I ran across that with the original CD drive that was in this dinosaur.
I stole the newer CD-R/W from my work station and used the smart-boot floppy and everything went just fine.

After you install, if your sytem starts rebooting in a loop, you will need to boot back up using the install CD and enter rescue mode.
Execute a root shell and:
[1] apt-get install linux-386
it will take a while
once that is done:
[2] apt-get remove linux-server

The server is compiled for 686 and won't work with some older systems.
