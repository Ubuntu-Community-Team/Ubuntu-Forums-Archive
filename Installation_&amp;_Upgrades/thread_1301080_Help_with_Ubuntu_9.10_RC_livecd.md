### Post by Mark Phelps on 2009-10-26
> **lightningfox said:**
> ... Desktop effects worked fine on previous versions of Ubuntu on my computer, so why is this happening? ... ATI Radeon 9200 Pro graphics card ... Maybe this is a problem related to my graphics card?

Exactly -- AMD/ATI dropped driver support for all but the newer HD 3x/4x/5x cards a few months back, relegating everyone with the older legacy cards to using the Radeon Open Source driver.

So, what it boils down to is how well the open source driver supports your particular card.

And, before you ask, NO, there are no new ATI drivers that will work in Ubuntu 9.04 or 9.10 with the legacy cards.
