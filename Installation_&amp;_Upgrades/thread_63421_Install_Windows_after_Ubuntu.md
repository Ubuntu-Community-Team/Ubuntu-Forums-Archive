### Post by Nopposan on 2005-09-07
I installed Ubuntu Linux over a Windows XP operating system that had become buggy due to malware & subsequent removal.  (This is on an old PC that we're planning to give away to a friend.)  We have a Windows 98 retail install CD that I'd like to put on there and have a dual-boot machine.

However, when I try to boot from CD with the Win 98 CD loaded I still go directly into Grub and then Ubuntu, even though the CD drive spins for awhile.  If the Ubuntu install CD is in the CD drive then I get a boot from the CD drive and go straight into the Ubuntu install process.  So, how do I get my Win 98 install CD to behave more like the Ubuntu install CD?

Please forgive me if I've left out important details; 'reason is I probably don't know them myself.  'Would have posted on the "Absolute Beginners" forum, but I thought maybe the install forum would be more appropriate.

### Post by mlomker on 2005-09-07
Windows 98 CD's cannot be booted; it has to be installed using the setup floppy that comes with it.  Win98 came out in the good ol' days when laptops actually had floppy drives.   :-P
