Hi, I have had this problem for a while and suddenly found a fix! Well it isn't supposed to be a fix but it worked. 

Go to Synaptic Package Manager, Click Settings, then Repositories and then click the 3rd party tab. Click add and put this in "deb-src [http://wine.budgetdedicated.com/apt](http://wine.budgetdedicated.com/apt) feisty main" then click ok, close the settings window and click Reload on Synaptic. Then it should all work!
