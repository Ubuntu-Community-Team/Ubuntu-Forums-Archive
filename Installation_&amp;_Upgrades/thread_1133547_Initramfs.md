I haven't tried Xubuntu installation, but with other variants you get, on boot, a grub menu with choices.
If, when the menu appears, you hit the TAB key (select the menu option you wnat first), you drop into a position where you can alter the options  This gets the menu line entry which you have selected, and you can both add and delete stuff before running it.
