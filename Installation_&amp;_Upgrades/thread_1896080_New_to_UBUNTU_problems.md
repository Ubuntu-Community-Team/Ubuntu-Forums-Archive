### Post by nothingspecial on 2011-12-16
Hi, welcome to the forums :)


To kill a misbehaving app, press Alt-F2 then type

```
xkill
```

Then click the app's window.

Have you installed ubuntu-restricted-extras from the software centre? That will install all the necessary codecs to play your media files.

Also, if banshee is playing up try playing your songs or movies with movie player. See if that behaves better.
