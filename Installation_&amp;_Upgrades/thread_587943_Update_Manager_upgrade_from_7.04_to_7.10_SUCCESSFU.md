### Post by gerrymoth on 2007-10-23
Was a bit apprehensive in upgrade to 7.10 after reading various posts in here, but went for it last night and upgraded from 7.04 to 7.10 using the Update Manager.

It did take most of the night, probably cos I kept falling asleep, but got there in the end this morning with no errors, no conflicts and working Bluetooth, Internet Connection (WifI), Chat (Pidgin), etc..

I know some people say you should do a fresh install, but thought I'd try the Update Manager to see if it worked and help the Ubuntu guys know that it does work for some people.

I noticed after I upgraded, restarted and then checked for updates again using the Update manager that Firefox had a update from 2.0.0.6 to 2.0.0.8, but after installing that my internet was still working fine and didn't notice any slowing down that some have experienced.

Gaim contacts had transfered over to the new Pidgin, with no problems.

All my additional software I'd installed was still there and working and all my files where still in my HOME folder.

I'll have a good play around with the new distro this week, but so far so good.

Thanks Ubuntu, been a convert now for 10 months and loving it.
