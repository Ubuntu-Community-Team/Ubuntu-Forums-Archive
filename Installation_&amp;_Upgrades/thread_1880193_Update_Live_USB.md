### Post by dandnsmith on 2011-11-13
I haven't tried it with 11.xx, but you used to be able to do this if you set up the USB to have 'persistence' - which requires special provision of a filesystem on the USB, and (possibly) a modification to the boot parameters.

I did hear that the system was changing, but i came across details of how to do it on the [pendrivelinux site](http://www.pendrivelinux.com/), and perhaps [this page within the site](http://www.pendrivelinux.com/yumi-multiboot-usb-creator/)
