### Post by Lord Stig on 2010-03-11
It should come up in Applications, Sound & Video or Multimedia.
If you can't find it you can run it from the command line:
go to Applications, Accessories, Terminal. Type in "vlc" (all lowercase) and it should appear.

You can set up a shortcut on your menu bar for VLC by right clicking, choosing Add to Panel, double-clicking on the Launcher option, then browsing down the list of files till you find one called "vlc".

If you wanted to add it to the Applications, Sound & Video menu then there is an option in the System menu somewhere called Main Menu which from memory) allows you to add, remove and move around all the programs that appear in your menus.

Hope this helps.
