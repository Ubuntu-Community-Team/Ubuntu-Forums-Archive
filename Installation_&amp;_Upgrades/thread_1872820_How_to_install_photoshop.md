### Post by Mark Phelps on 2011-11-02
Not a Wine fan ... so keep that in mind when reading my comments ...

First, the ratings in WineHQ can be misleading because all you get is a single color -- and that says little about the actual experience of trying to install or use the App.

Second, if you look closely at the testing result for CS5, you'll see that NONE of them are for Ubuntu 11.10.  Given the scope of changes for each Ubuntu release, I would be very surprised if a version that worked for 11.04 still worked for 11.10.  So, with no testing results that apply, you're breaking new ground here.

Third, if you look at the bug reports on the same page, you'll see three recent bugs for CS5, one of them being the installer.

Given all three issues, unless some miracle occurs, it's unlikely that you're going to be able to use CS5 the way you want in Ubuntu.
