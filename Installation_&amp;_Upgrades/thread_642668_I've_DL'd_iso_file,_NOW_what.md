### Post by PmDematagoda on 2007-12-17
I suggest that you use the Alternate CD to install Ubuntu.

The Alternate CD is the text-based installer for Ubuntu, after the install is finished you will have to configure your GUI.

You can get the Alternate CD from [here]("http://www.ubuntu.com/getubuntu"), check the checkbox near the end of the page that will allow you to download the Alternate CD.
