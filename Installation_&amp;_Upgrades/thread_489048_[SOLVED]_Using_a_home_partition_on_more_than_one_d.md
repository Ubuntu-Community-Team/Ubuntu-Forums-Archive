I haven't attempted it myself, but I wouldn't think it would be advisable...the 2nd Ubuntu install to the shared /home partition would probably overwrite configurations files with the same name..."may" possibly work sharing a separate /home with 2 Ubuntu distros, but definitely not recommended for Ubuntu & another distro.

I would recommend setting up a separate data partition to share important files among different distros.
