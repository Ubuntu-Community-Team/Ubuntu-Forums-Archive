### Post by nick-f on 2006-02-08
i had the same problem on my epia m10000 nehemiah. it was impossible to install ubuntu, because there was absolutely no way to read the screen. the trick that i did was in the bios there was an option something like "display type" and it was set by default to "CRT+LCD" but i changed it to "CRT" and it worked like a charm.
