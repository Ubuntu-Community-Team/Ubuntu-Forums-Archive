### Post by Herman on 2010-01-06
You should be able to do that if you choose the Ubuntu 'Alternate Installation' .iso when you download Ubuntu. 
Burn that to disc and give it a spin.
That's the one to use if you want to do something different, hence the name 'Alternate'.

My first sig link contains some examples of how it works, just in case you haven't used one before. 
A long time ago those were the only way to install Ubuntu, but a few people thought they were confusing to use. 
They're quite simple actually.
