I am not sure I understand all parts of your post, but here is a hint about how to put the file on a CD.  iso files are special files and have to be burned in a very specific manner. Most CD burning software have options to burn iso images (Nero does). Look for "burning image files". Else you can also use isorecorder ([http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm))

Once you have a CD burnt you can just stick it in and follow the instructions. Use this as a guide [http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

### Post by chajuram on 2005-11-28
Hi,

I know it can be frustrating at times. Also I am sure you are burning coasters. But keep at it and you will burn the right disk. 

Also make sure your download is right. check the md5sum using md5summer (or any other program). Errors in burning can result at two stages.

1. File download...check using md5summer (or any other program that claims to check md5sums)
2. CD burning...use a good CD burning software. It is possible that you will have to try a couple of times to get it right. Some suggestions.
   * Burn at slower speeds
   * Use good quality CDs
   
Chajuram.
