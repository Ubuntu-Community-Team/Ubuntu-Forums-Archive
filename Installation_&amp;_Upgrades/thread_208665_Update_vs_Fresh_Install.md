### Post by ericesque on 2006-07-03
If you're at a university, you should have access to high speed network.  What I did while I was on campus networks was use uTorrent to get the dapper .iso  then, if the computers don't have a program that will burn the .iso as a useable disc, simply burn the file to the cd, take it back to your PC and burn it to a usable disc there.  A dapper torrent should only take a couple hours on a decent connection.

Oh, and I'll echo the sentiment about using a fresh install.  I installed the beta of dapper and updated as releases were made.  A week after the release I just felt like things should be faster.  So I did a fresh install and it payed off big time.
