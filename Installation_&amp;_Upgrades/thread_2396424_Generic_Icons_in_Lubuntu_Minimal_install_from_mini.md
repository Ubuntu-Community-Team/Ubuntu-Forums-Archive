### Post by Dennis N on 2018-07-15
I often find a different icon theme is more to my liking. An icon theme is set in "Customized Look and Feel" dialog and changes all icons to those of the theme. The default offerings there may be limited, so also look at icon themes at gnome-look.org which you might want to download.
