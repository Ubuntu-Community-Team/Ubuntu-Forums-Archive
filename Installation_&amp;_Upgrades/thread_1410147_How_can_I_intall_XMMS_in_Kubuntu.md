### Post by johnson.d on 2010-02-19
The xmms package can be found in kubuntu as gxmms2.

you can install gxmms2 package in Kubuntu by using the following steps:

1) Goto K menu-> Settings-> System settings
2) Click add and remove packages
3) A Window opens in which there is a search bar. Type gxmms2 in the search bar and press enter
4) It will list the package available as gxxms2. Click on the Down arrow present in the right hand side corner.
5) Click on the apply button present in the right hand side bottom corner of the window.
6) xmms will get installed in the system.

The following command installs the xmms core files and dependencies needed:

$ sudo apt-get install gxmms2.

You can run xmms from 
K menu-> Multimedia-> gxmms2
