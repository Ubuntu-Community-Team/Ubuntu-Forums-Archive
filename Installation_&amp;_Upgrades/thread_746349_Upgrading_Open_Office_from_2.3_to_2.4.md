### Post by darolu on 2008-04-08
You will need to uninstall the current openoffice.org on your computer via Synaptic (system - administration - synaptic) or console; then install 2.4, you can use the RPM installation with the instructions that are in the openoffice.org site, is the easiest way.

I personally love 2.4 Calc (spreadsheet), the new feature that moves you automatically to the cell where you started to input data is just wonferful (after pressing tab several times), saves a lot of time when inputing data, this feature alone worths the change in my opinion, plus all the other features makes it really worthy, go for it.
