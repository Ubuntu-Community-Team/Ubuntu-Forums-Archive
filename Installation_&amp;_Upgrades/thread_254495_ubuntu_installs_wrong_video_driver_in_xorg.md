### Post by BoHu on 2006-09-10
When I install Ubuntu on my computer it configures xorg to use the onboard video chip instead of the nVidia card. So when I boot up the first time, I get a blank screen.  This has happened when installing Ubuntu on other people's computers, too. The Ubuntu liveCD won't work on any of these machines, only the alternate-install. To fix this, I use a Beatrix (or sometimes Mepis 3.43) live CD to edit the xorg.conf file. Good thing I know how :-)

Y'all probably already know about this problem but I thought I should report it anyway. This kind of thing would be a deal-breaker for most newbies.

By the way, Ubuntu is not the only distribution that does this. I have the same problem with PClinuxOS.
