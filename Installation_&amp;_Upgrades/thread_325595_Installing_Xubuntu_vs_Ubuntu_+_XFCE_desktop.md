### Post by mojoman on 2006-12-26
Ubuntu comes with different programs and uses different libraries than Xubuntu ao the answer is that Ubntu with XFCE is not the same as Xubuntu. However, you could install the XFCE desktop, which is a lighter desktop than Gnome, and try it out but if you're looking for a lightweight installation or have rather modest specs on your computer I would recommend the full Xubuntu. Any program that runs comes with Ubuntu can be installed in Xubuntu but you might end up installing a whole lot of additional dependending packages so it's a good idea to simulate any installs, thus checking what they are depending on before installing them. That said, I really liked Xubuntu and XFCE and I really recommend it.
