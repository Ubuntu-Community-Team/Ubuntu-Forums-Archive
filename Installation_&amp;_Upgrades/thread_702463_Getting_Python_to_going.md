### Post by Nameless_one on 2008-02-20
If you installed it, just executing the command 'python' on the command line runs the interactive python shell. 

If you're more used to wirking with GUIs, you should install IDLE, it's a graphical IDE for Python (the package name is 'idle'). 

Also, you can run python scripts (.py) simply by running ```
$ python script.py
```, or by making them executable and simply running ```
$ ./script.py
```.
