When I eject the cd, it will go ahead and boot windows without restarting.

I don't know if for some reason the sony viao laptop doesn't like the ubunto  cd, which seems unlikely, or if I burned the cd wrong for the laptop.  The cd works on a desktop machine just not on the laptop.

### Post by meng on 2006-12-06
Does anything come up before the flashing cursor? There are boot options such as noapic and nolapic, you should probably try both of these (press one of the F-keys to see the other options available).
