### Post by Mark Phelps on 2009-11-09
In terms of Ubuntu, No.  There is no Ubuntu version that will install and run with 64MB of RAM.

You can go to the distrowatch.com website and look at the Linux versions that use less resources, like DSL, Puppy Linux, Tiny Me, Tiny Core,  but 64MB of RAM is really low.  It may be that these won't run on that as well.
