I do not know the answer to your question but it might help others to give a useful answer if you told us if these installs were to be Ubuntu only on the laptop or dual boot. Is the installation to be over a network?

Every installation of Ubuntu requires a username and password of the person who is to be the administrator on the system. Commercial suppliers of computers with Ubuntu pre-installed put in what is called an OEM install. When the purchaser receives the machine and first boots the machine he is required to set a user name and password as well as a location and other individual settings. This link explains an OEM install.

[https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview](https://help.ubuntu.com/community/Ubuntu_OEM_Installer_Overview)
