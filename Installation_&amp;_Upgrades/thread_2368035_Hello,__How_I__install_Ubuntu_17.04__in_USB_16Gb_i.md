### Post by yancek on 2017-08-06
I'm not sure if I'm reading your post correctly about installing Ubuntu "FROM" windows.  I'm not aware of any method to do that.  You can create a bootable usb within windows and use that to install Ubuntu to an internal or external drive.  On the other hand, using the Linux Grub bootloader, you can boot most any Linux or windows and install it without creating a bootable usb or DVD.  Since all you have is windows this is not relevant so if you are simply trying to create a bootable usb within windows the suggestions above should work.
