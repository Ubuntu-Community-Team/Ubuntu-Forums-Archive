### Post by T700 on 2006-07-12
I would *not* suggest the server install unless you really want to run a server--not a good way to learn a new OS.  Further, the server has no GUI; you will be dropped a command line.  Why not do the normal desktop install, enjoy the GUI, learn about the various programs, and in a few weeks, install a server, if you like?

The installation process is very simple.  Put the disk in.  Reboot.  Follow the prompts.

Paul
