### Post by Partyboi2 on 2009-07-28
You don't have to upgrade to Jaunty, but if you are wanting to upgrade to 9.10 when it comes out, you can not skip releases so you would need to upgrade to 9.04 before upgrading to 9.10. Or you can backup your important data and do a clean install without having to worry about upgrading to 9.04 first.
When you upgrade you should get a later kernel version.
