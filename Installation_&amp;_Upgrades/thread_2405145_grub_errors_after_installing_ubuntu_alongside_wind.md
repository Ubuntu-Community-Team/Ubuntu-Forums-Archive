### Post by oldfred on 2018-11-01
Your 17.04 reached EOF - end of life in Jan 2018.
Best to use LTS long term support versions, so you do not have to update every 9 months. LTS has 5 years of support.
Use 16.04 or 18.04. Only if very new hardware or you just want to experiment may you want to install 18.10. 

Is system UEFI or BIOS?
Is Windows in stalled in UEFI or BIOS boot mode?
Did you install Ubuntu in same boot mode as Windows?
What brand/model system.

Download 18.04 and start over.
