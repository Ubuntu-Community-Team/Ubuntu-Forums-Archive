### Post by Mark Phelps on 2012-01-26
IF your intention is to install from INSIDE Windows using the Wubi installer, you can still do that using a CD or USB, but you still have to boot into Windows first.  If you boot using the CD or USB, it will present options for installing OUTSIDE of Windows, and it will need to modify your partitions to do that.
