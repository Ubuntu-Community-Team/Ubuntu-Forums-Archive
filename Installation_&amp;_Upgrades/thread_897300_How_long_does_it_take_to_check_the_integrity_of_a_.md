### Post by Afkpuz on 2008-08-22
Sounds like a bad burn to me.  Also, it may be that you need to use the alternate install CD.  This is mostly to alleviate any video card problems you might have as it uses a text based installer.  Use the "check disc for errors" at boot up and then if that says your disc is fine, use an alternate install disc.
