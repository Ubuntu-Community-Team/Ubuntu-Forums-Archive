I currently have Windows 7 and Ubuntu 11.04 (The beta) running on my Laptop.

I want to know if it's possible to get 11.04 (Full) through the update manager? If so, how?

Secondly, if I wanted to just go back to 10.10, could i install it over the current 11.04? I don't want to copy files over or anything like that, just a fresh start.

### Post by lechien73 on 2011-05-05
Yes, you should be able to update to the full version of 11.04 from the beta, try opening a terminal window and running:

```
update-manager -d
```

There's no official downgrade from 11.04 back to 10.10, you would need to re-install. For that reason it's best if you separate /home to a different partition, then you can re-install 10.10 over your 11.04 installation.
