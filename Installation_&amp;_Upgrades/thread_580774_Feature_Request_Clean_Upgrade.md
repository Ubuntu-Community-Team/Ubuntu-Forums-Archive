### Post by daengbo on 2007-10-19
Another ubiquity feature request:

During an upgrade, the user would be offered the opportunity to do a "Clean Upgrade," meaning taking a list of currently-installed packages, wiping the root partition, installing the new, upgraded system, and installing the packages in the list.

This would all be easier if /home were installed on a separate partition (a safe and commonly requested default), but user could be prompted to back up the old home to an external device and restore when the upgrade is finished.
