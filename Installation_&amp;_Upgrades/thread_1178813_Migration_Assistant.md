### Post by Partyboi2 on 2009-06-05
Hi, I think you can only use the Migration Assistant during the install process. If you installed Ubuntu on the same partition as Windows using wubi then you can go to
'places > computer > file system > host' or if you installed Ubuntu to its own partition then you should be able to access all your files on Windows by going to
'places > computer > file system'.
