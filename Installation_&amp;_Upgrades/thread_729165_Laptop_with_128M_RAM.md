### Post by Herman on 2008-03-19
You can use the Ubuntu or Xubuntu 'Alternate' CD to install in a computer with as little as 32 MB of RAM. 
I've tried it myself with as little as 64MB of memory and had good results. Here are a couple of links to prove it, about how to perform a minimal system installation in a low memory machine.
[Windows98SE+Ubuntu Gutsy Gibbon]("http://users.bigpond.net.au/hermanzone/p2.htm")  This example is based on the excellent Ubuntu Community Doc's How-to about [Installation/LowMemorySystems]("https://help.ubuntu.com/community/Installation/LowMemorySystems") - Install Ubuntu on older computers with low memory.
The minimal installation is a little more work but it results in quite a snappy operating system even with very little RAM. My favorite desktop was the IceWm desktop, but I now have added another memory card in that computer and it has 375 MB or RAM now, and it's running Ubuntu Gutsy Gibbon just fine. I added more Desktops so can choose between IceWM, Fluxbox, XFCE, Gnome or KDE.
It's a good idea to add more RAM if you can.

 I know  you should be able to do a full installation of Ubuntu or Xubuntu if you have 128 MB of RAM. I have done so many times.
Your system will be a pretty slow, but it will do everything. (Well, almost everything).

The normal installation simpler and easier for new users, but you didn't say you were a new user so maybe you're an experienced Linux user looking for a fast system, so make up your own mind which way to go. I think I would prefer the regular install and slower speed myself.

I agree with Pumalite, Puppy Linux is another of my favorites too.

Regards, Herman :)
