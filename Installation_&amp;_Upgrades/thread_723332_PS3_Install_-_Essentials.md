### Post by Nadim on 2008-03-16
I'm rather surprised myself at the relative lack of PS3 Ubuntu folks out there.  I'm no expert, but I do happen to be writing this right now on my PS3, running Ubuntu (Gutsy Gibbon), so I can at least give a few pointers based on my experiences.  

Anyway, the first thing I would recommend is, as soon as you get your PS3 (and before you play any games, etc.), repartition the HD, setting aside the 10 GB partition for the "Other OS"...saves a lot of work backing up (and potentially screwing up) your PS3 game saves, etc.

For the most part, following one of the guides out there should work fine... I think I used this one:

[https://help.ubuntu.com/community/PlayStation_3]("https://help.ubuntu.com/community/PlayStation_3")

The one major issue I ran into was that the installer is pretty finicky...I'm hooked up to my 1080p projector (yes, It is cool running linux on a 100 inch "monitor"...).  When I ran the installer with the default boot line, it was useless...formatted 4:3 meant I couldn't see critical things like the "next" button.  More disturbingly, running with the "right" mode (1080p) just didn't work...I'm guessing it didn't have enough memory in live mode, but it just crawled for a few hours, never really made it through.  Starting in 720p mode (I believe the numeric code to use was "3", or "131" if you want full screen mode too) worked well, so that'd what I'd recommend if you've got an HD set hookup.  

After that, it basically worked fine.  I haven't yet had any success getting it to work with my bluetooth keyboard and mouse, so I've got annoying USB cables going on right now... If anyone has any ideas there, let me know.  I love linux, but I have to say, I just hooked up a mac mini to my other TV, and it was ludicrously easier and cleaner.
