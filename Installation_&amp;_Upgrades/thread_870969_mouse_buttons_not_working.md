### Post by upchucky on 2008-07-27
you may wanna copy paste this into your xorg. it is from mine but it may work just make sure you have the usb mouse connected when you boot the pc.

if the display does not load on a reboot then just remove the section from your xorg using the command line, beware that if you do not know how to get the editor working from the command line, then read how to boot and edit from the command line before playing with your xorg.

```
Section "InputDevice"
# generated from default
Identifier "Mouse0"
Driver "mouse"
Option "Protocol" "auto"
Option "Device" "/dev/psaux"
Option "Emulate3Buttons" "no"
Option "ZAxisMapping" "4 5"
EndSection
```
