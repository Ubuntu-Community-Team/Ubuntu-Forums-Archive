Ok, so the Iconia install took a few tries. For those of you still stuck on the keyboard, use Prefereces>Assistive Technology to setup the on screen keyboard and moues.  add 'onboard' to the prefered applications and select run at start. Enable assistive technologies when promted. Also set the mouse to right click when you long left click. For ease of use, I also added onboard to my menus and to the launcher panel.
