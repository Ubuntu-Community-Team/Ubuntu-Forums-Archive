### Post by logos34 on 2008-12-12
boot the live cd and this time choose 'manual' partitioning...Tell it to use the current hardy / (check 'format' option).  It will automatically use the existing /swap

good luck

add: you might also want to back up your /var/cache/apt/archives with AptOnCd--it'll save you the hassle of re-downloading all the updates
