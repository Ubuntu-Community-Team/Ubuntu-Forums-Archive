What is the laptop make?
There are some machines which make it hard to install and use anything except Windows, but as far as I'm aware, there is always a way around this.
Some of the difficulties I speak of are listed in some detail in the long thread at [https://ubuntuforums.org/showthread.php?t=2147295](https://ubuntuforums.org/showthread.php?t=2147295) which may help you sort out the problem in your case, but again, if that still does not help come back again with further questions.
