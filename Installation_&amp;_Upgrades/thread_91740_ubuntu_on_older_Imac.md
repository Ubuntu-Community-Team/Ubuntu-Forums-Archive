### Post by aysiu on 2005-11-17
Those are some pretty modest specs. You'd have to do a very minimal install and use XFCE instead of Gnome or KDE.

The best way to see if it works, though, is to use a live CD.

**Note**: the live CD will be *extremely* slow on your computer, as it runs entirely off RAM (all 64 MB) and the CD itself, and it runs a Gnome desktop by default (XFCE would be something you'd have to install later).

Still, a live CD will give you a good idea how well it works (not how fast it runs).
