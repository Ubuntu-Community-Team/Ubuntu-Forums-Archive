### Post by kevmaster on 2007-10-20
I usually also backup my /etc because there are a lot of settings in there. Some of them might be useful later. And it's not painful to backup because it's just some text files.

Another tip:
Since Feisty (I believe), you insert the CD and after a couple of steps in the installation process it will report that it has found an existing Ubuntu installation. You can then tell it to import all your current settings & files (I think it will just copy the /home, and maybe perform a couple of tricks for portability)
