### Post by cariboo on 2008-09-24
Those applications in Other were always installed, you must have changed a configuration file somewhere to get them to show. I had that problem during the hardy alpha's, but haven't seen them since. If you don't want to see the Other menu, right-click on Applications, then click Edit Menus. In the window that comes up make sure you remove the check mark beside other. The Other menu will now not show up.
