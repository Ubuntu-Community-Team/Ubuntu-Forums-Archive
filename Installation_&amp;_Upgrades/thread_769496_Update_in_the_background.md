### Post by deadowl on 2008-04-26
I know this is a questions forum, but I decided to post an answer... sorry.

For some reason this is a popular idea on Brainstorm, but it's already doable... but apparently it's not obvious enough.

Go to System->Administration->Software Sources->Updates

Then under Automatic Updates: change the radio button to download updates in background. You can also choose to install only security updates automatically.
