### Post by Belyel on 2006-11-03
The AMD64 is meant for the AMD brand processors.  After you install the i386, you can update it to the 686smp kernel to take advantage of the dual core processor.  Or, if you install edgy eft, it comes with the generic kernel that is already optimized for most processors, including dual core.
