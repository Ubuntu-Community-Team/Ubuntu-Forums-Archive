### Post by Partyboi2 on 2008-09-27
If you have already installed ubuntu using the wubi installer (installing inside windows) you can move your ubuntu installation to its own partition. See [[COLOR=Blue]here[/COLOR]]("https://wiki.ubuntu.com/WubiGuide#How%20do%20I%20migrate%20to%20a%20real%20partition,%20and/or%20get%20rid%20of%20Windows%20entirely?")

Edit:
If you decide to do a clean install from scratch instead of moving your installed ubuntu to its own partition, download the ubuntu desktop live cd from [[COLOR=Blue]here[/COLOR]]("http://releases.ubuntu.com/8.04.1/") if you have not already got it. Then check the md5sum that it matches. See [[COLOR=Blue]here[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM") for how to. If that passes then burn the iso as a iso not data  to a disk at x4 or less, see [[COLOR=Blue]here. [/COLOR]]("https://help.ubuntu.com/community/BurningIsoHowto")After you have burnt it to disk, boot the newly made cd, making sure that in your bios that your cdrom is set to boot first. Once you get to the ubuntu main menu screen select the option to check the cd for defects. If this passes then choose the option to install and follow the on screen instructions. If you get stuck anywhere post back where you got stuck and someone should be able to help.
