### Post by spelingchampeon on 2006-08-20
I know the title of this post is a little odd, but I wanted to pass this stuff on to everyone at this point. If it saves 1 person from the drudgery I went through, I'll be happy. 

My 1st PC for Ubuntu (Breezy) was extremely SLOW. It was a 1gig Celeron, 256meg pc100 ram and a 10gig 5400rpm HD. I built it as an MP3 server that I controlled remotely using VNC. Everything I did remotely was slow.. no actually sluggishly slow. If I copied 15 songs from a USB thumbdrive, it would take almost 10 minutes. Moving around in the files directory was like **pushing** a car through a drive through car wash. Needless to say, I knew I had to do something. 

For about $110.00USD, I bought a used motherboard-AMD 2000 cpu, upgraded to 512meg pc133 ram, and an 80gig 7200rpm HD. Installed all into the same box.. and installed Dapper. If you could see this machine scream!! This is what computing should be like. There are NO problems whatsoever with the speed of this machine. The slowest thing about this box is VNC, and thats to be expected.

I am only posting this because of posts that I had read with the same issues I was going through.
