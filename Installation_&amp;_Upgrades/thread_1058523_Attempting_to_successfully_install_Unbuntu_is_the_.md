### Post by RoseKnight on 2009-02-02
Instruction:

1) buy 4gig flash drive
2) download Latest Ubuntu Iso
3) download Latest version of uNetbootin
4) use unetbootin to put iso on flash drive
5) stick flash drive into target computer
5.5) make sure bios is set to boot from USB before harddrive
6) follow onscreen instructions
7) ubuntu installed.
