### Post by K.Mandla on 2007-01-02
> **xerio said:**
> Is there a way to go about installing Ubuntu without it automatically installing grub?
If you press escape at any time during the installation process, you'll get a list of installation steps. If you stop the process near the end of the install, you should be able to install LILO or possibly skip the bootloader process.
