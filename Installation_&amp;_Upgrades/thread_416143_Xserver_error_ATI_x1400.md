### Post by pertti on 2007-04-23
Please, be more specific with the error that you are getting (copy and paste here any error message) and what you did to get into this situation (installation? upgrade? what commands did you try?), so we can help you out. Also post your hardware configuration (especially the brand and model if it's a laptop).

There are [a lot of problems with ATI X1xxx cards]("https://bugs.launchpad.net/ubuntu/+source/xorg-server/+bug/89853") which prevent an easy installation, but people are mostly getting around these issues. I have a Dell 6400/E1505 laptop with a X1400, and didn't have any problem upgrading from Edgy to Feisty, but [this thread]("http://http://ubuntuforums.org/showthread.php?t=399913") (a howto for that laptop) may help you.

Anyhow, let's hope that there's a fix soon for this problem (keep an eye on that bug report).

### Post by Bealer on 2007-04-23
Do the dpkg-reconfigure xserver-xorg again.

This time select "vesa"

A colour depth of 16

And untick all screen resolutions apart from 800x600 and 1024x768

See if it works then.

Once you're in just install the ATI (fglrx) driver via the How To:
[http://wiki.cchtml.com/index.php/Ubuntu_Feisty_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Feisty_Installation_Guide)

I raised a similar thread here:
[http://ubuntuforums.org/showthread.php?t=418156&highlight=x1950+feisty](http://ubuntuforums.org/showthread.php?t=418156&highlight=x1950+feisty)
But eventually came to the above solution for my hardware.
