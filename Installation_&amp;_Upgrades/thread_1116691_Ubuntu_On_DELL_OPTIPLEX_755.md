### Post by resorb on 2009-04-17
I had the same problem. When the ubuntu screen came up - I selected "English" as my language. I then pressed "F4" and did "Safe Graphics" - then selected "Install". 

That solved my issue of having blank/black screens and I was able to install normally after that.

Though my install media was 8.10, Intrepid.
