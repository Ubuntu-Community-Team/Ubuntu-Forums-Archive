### Post by 505 on 2007-09-01
Version 2.19 is a test version, so it is not recommended to upgrade to that. Just stay with the version you have now. It is part of Gnome, which is also version 2.18. When the new Ubuntu comes out in two months, 2.20 will probably be included.

Something about version numbering. In many open source projects (Linux, Gnome) even version number (2.16, 2.18) are stable version, while odd numbers (2.17, 2.19) are development version. Thus 2.19 is the development version for 2.20. When all the Gnome applications are finish for the new release the version number will be bumped to 2.20
