### Post by taurus on 2007-11-04
If you have Ubuntu LiveCD, boot from it.  Then, open gparted (System -> Administration) and delete all the partitions on your harddrive.  Then, either convert that one partition to fat32 or ntfs.  Reboot and insert your windows CD in to boot from it.
