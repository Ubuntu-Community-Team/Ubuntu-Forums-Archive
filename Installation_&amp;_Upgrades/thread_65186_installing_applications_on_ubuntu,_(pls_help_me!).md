### Post by Zotova on 2005-09-13
To install programs you do not have to use commands.

Under the system tools menu click 'Add/Remove Programs'. This opens synaptic.

With synaptic you can then search for the program you want to install, tick it for installation and then click apply. All the work will then be done for you.
