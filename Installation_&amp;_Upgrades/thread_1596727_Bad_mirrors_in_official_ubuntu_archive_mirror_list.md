### Post by pixnaps on 2010-10-14
Wow.  I'd been wondering why I hadn't gotten any security updates since upgrading to Maverick RC a week or two ago.  It turns out that it's because I'd let Ubuntu's Software Sources application "select best server" for me, and the server it chose ([mirrors.ccs.neu.edu]("https://launchpad.net/ubuntu/+mirror/mirrors.ccs.neu.edu-archive")) **doesn't appear to include updates for any versions of Ubuntu except Intrepid Ibex!**

How crazy is that?  The [general list of Ubuntu mirrors]("https://launchpad.net/ubuntu/+archivemirrors") reports the mirror as 'up to date', despite the fact that it doesn't actually maintain updates for current versions of Ubuntu.

This strikes me as a serious problem.  Software Sources should not select as a "best server" a mirror that doesn't even contain updates for your version of Ubuntu.

Anyone know what's going on here?
