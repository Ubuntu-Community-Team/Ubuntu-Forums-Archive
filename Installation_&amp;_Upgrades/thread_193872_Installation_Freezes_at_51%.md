Hello,
Dunno if this is going to be a lot of help, but before you boot, you can type in special boot options, to help certain systems along abit, I think you have to press F3, and then type in the options you want, for example "noapci" solves problems with power-management.
The only other thing that springs to mind is to check that the cd is ok, especially if it is a "homemade" burned cd.
It would also help if you turned off the "boot-gui" to see exactly where it is hanging. I'm afraid someone else reading this is going to have to tell you how to do this, I forgot, I think the boot-option "splash" & "noquiet" have to be removed, but like I said, I'm not sure (you get to the boot-options by pressing F7, or expert-mode).
