### Post by ganewbie on 2010-10-19
Gotit, 
You can have Ubuntu installed with both English and French on the same computer. I got to install the French language (assuming you already have English installed).
Go to system--> Administration--> Language.
Voila, now you could have your choice of any other language.
The change takes effect after you re-login.
It is amazing as all the add ons software packages are in French now and I can go back and forth.
