### Post by Lars Noodén on 2012-08-31
When you tried Lubuntu did you also try it without LXDE and running just Openbox by itself?  That should be faster, same with running any other lighter Window Manager like Fluxbox, FVWM or Oroborus.

If you've tried those and Lubuntu is still slow, then go for another distro like [Puppy Linux](http://puppylinux.org/main/Overview%20and%20Getting%20Started.htm).
