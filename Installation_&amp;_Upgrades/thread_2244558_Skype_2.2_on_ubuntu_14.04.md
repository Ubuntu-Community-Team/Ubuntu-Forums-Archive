### Post by Bucky Ball on 2014-09-17
You can get the dialpad by clicking the little dialpad icon in the conversation window when you're in a call. Make a test call and check it out.

Took me an age to find it and a lot of frustration when I rang places and had to make a selection using numbers. The keyboard doesn't work for it, but the Skype pop-up dialpad does.

I wouldn't try going that far back. As mentioned, possibly won't run, or if it does, problematically, if there's such a word! ;)
