### Post by johnc4510 on 2007-04-23
When you boot to the livecd, there will be an icon on the desktop that says install. Use this to install the OS. Linux and Ubuntu require a  /  partition which is called the root partition and is where the system files are located. A  /swap  partion, which is used to augment your systems memory. I would also suggest a /home   partition. This is where all your personal items are kept. By having it, when you upgrade to a new version of Ubuntu, you will not lose any of your personal items. Such as Music files, etc.

I have a 60gig hard drive and have it partitioned as follows. This is only a guide though:

/                   8 gig

/swap         1gig    Note: swap should be twice your system memory

/home         The rest of the drive

Hope this helps
