### Post by JC Cheloven on 2010-11-09
Sure. If you don't need nothing but a command line interface (CLI) and some packages, use the alternate CD installation media.

Choose install a command line system. You will have apt-get installed by default (obviously among many other usual utilities from the GNU project). So you can install any package you need from the repos.
