This won't probably won't happen
considering the fact that Winetools can help you break the law(installing IE,Outlook,WMP...if you don't have a Win98 licence.)

Most Windows users who switch to GNU/Linux will have started with the Free Software Windows versions.(OpenOffice,Firefox,Gimp,Gaim,...). so no need to learn anything new.

Wine should only be used as a last resort. Which means search for a native GNU/Linux product. Only if you can't find one (or if the GNU/Linux alternative doesn't have certain features you want), you can try Wine.

Wine(eventhough it is a good product that I like very much) isn't perfect. A new user might incorrectly blame this on GNU/Linux Wine(Which means bad publicity).
