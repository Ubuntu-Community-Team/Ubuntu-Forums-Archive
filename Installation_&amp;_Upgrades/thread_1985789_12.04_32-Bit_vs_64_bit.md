### Post by Maro848 on 2012-05-23
I run 64 bit on my systems and have yet to have any hardware incompatibilities. The only difference between 32 and 64  bit is that 32 bit is meant more for older computers that have 32 bit processors and 64 bit is for newer machines with 64 bit processors. Other than that there is no difference in the operating system or the hardware it will work on.
