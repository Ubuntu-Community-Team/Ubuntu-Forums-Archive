### Post by mikewhatever on 2011-03-02
So, you are running W7 and Ubuntu at the same time, right? Is one of the OSs installed in a virtual machine? Which one? I think any computer running two OSs at once will be kind of slower, let alone a netbook. 
Anyway, to answer the question, there is no uninstall button for an OS, instead, you just delete the partition.
