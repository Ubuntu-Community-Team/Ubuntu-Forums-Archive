### Post by shivam.seth on 2008-05-17
No, your laptop need not be connected to internet while installing Ubuntu. Just use the Ubuntu installation cd. Put it in the cd drive and restart the computer. It will automatically ask you that do you want to boot from the cd....say yes!

The whole process after that is smooth enough, so that if you can understand english, you can install Ubuntu without any experts' help.

While you don't need to connect to internet while installing Ubuntu, you might need internet for getting some packages necessary for playing your mp3s or other non-free audio, video codecs..... Or you can download it from some other computer, transfer those packages to your comp and install the required codecs or softwares!
