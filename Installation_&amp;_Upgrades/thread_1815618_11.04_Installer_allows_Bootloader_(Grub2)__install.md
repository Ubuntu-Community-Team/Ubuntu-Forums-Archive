### Post by oldfred on 2011-07-31
There is no grub boot loader installed to sda2, you must have selected sda??

A grub in the MBR normally will not work with a new install. Although I did see one user reinstall to the same partition and the old grub still worked, but I think that was an exception.

And you have to reinstall the windows boot loader to the MBR to get windows to work. 

I think the wish list for Ubuntu includes a backup of the MBR, but that does not manage whether the MBR is grub, grub2, lilo, windows, syslinux or anything else.
