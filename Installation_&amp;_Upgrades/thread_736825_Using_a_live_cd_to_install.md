### Post by Antarctica on 2008-03-27
No.

A CD is usually a read-only media, meaning that once the data is written to the CD, no more data can be added.  Ubuntu and Windows are two completely different operating systems.  Ubuntu has no native support for Windows.  You can access the hard drive that Windows is installed onto, but Ubuntu and Linux have no native installer for Windows-based programs.  Your best option is to install the game on Windows.  Besides, why would you want to do such a thing?  The Live CD is usually meant for either permanently installing Ubuntu or repairing an operating system.
