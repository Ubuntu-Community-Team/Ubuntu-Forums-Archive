### Post by Buddhika Udaya Sri on 2012-04-26
yes, you can do it very easily. You can format your Ddrive as NTFS or Ext4 . Then you have  to create a bootable CD or penfrive using your   ubuntu12.04.iso . 
You can create a bootable pen drive using unetbootin . 
[http://unetbootin.sourceforge.net/#install](http://unetbootin.sourceforge.net/#install)

Then you can install Ubuntu in D Drive. 

It's better to create a small partition 500mb or 1 Gb etc, for swap area when you format your D drive.
