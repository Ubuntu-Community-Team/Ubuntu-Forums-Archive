### Post by ajgreeny on 2011-10-05
Difficult to know, but it is worth booting to the CD again, and as soon as you see the first screen hit Enter.  You will then see a language choice screen, then a menu with a "Check the install CD" or similar description.  Do that and you may see some error messages when it finishes, or it may simply be a glitch for that session you used last time.

If errors show I advise you to either burn a new CD or download again, but check the md5sum of the iso file first.  All the info on how to do that is here:-

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
