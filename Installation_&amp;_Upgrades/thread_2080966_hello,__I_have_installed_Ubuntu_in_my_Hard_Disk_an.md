### Post by pkadeel on 2012-11-05
1. Do you want to be able to use greek as text input in your documents?
 or
2. Do you want full greek localization in ubuntu?

For the first issue you need to enable greek keyboard from
 System Settings > Keyboard Layout

For second you need to install language support from 
System Settings > Language Support
