### Post by aysiu on 2008-02-27
Horsepower has nothing to do with the Ubuntu version numbers. Ubuntu 7.10 can fly on an old computer just as much as Ubuntu 5.10 could... as long as you don't use Gnome, KDE, or Xfce.

A lightweight window manager like IceWM or Fluxbox will give you a better performance on any Ubuntu version.

Breezy is no longer supported by security updates, so you really should update it to a later version.
