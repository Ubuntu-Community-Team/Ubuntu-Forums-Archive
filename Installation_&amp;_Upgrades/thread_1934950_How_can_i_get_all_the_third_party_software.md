### Post by howefield on 2012-03-03
Easiest way might be to install the package ubuntu-restricted-extras.

Either via the Software Centre or in a terminal type

```
sudo apt-get install ubuntu-restricted-extras
```

---

### Post by Bucky Ball on 2012-03-03
How's about in 'Software Sources'? Make sure there's a tick by 'Software restricted by copyright or legal issues (Multiverse)' and in the 'Other Software' tab make sure there is a tick by 'Independent - provided by third-party software developers.' 

And as howefield suggests; ubuntu-restricted-extras.
