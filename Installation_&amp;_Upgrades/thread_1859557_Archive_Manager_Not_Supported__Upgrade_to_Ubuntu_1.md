### Post by JayBofMA on 2011-10-14
BTW... I found using Ubuntu Tweak the best solution for correcting this.  It seems that the configuration file installed in the 11.10 upgrade associates the Microsoft file types with Archive Manager, and not Libre Office (Sorting on the Associated Application column was helpful.)
