### Post by Rocket2DMn on 2008-03-25
Did you check the md5sum of the disc?
[HowTo]("https://help.ubuntu.com/community/HowToMD5SUM")
[Hashes]("https://help.ubuntu.com/community/UbuntuHashes") to check against
Then burn the .iso at a slow speed like 4 to prevent write errors.
Otherwise you may need to use the alternate install cd from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom for alternate cd.  It is a text based installer, but it was really easy to use.
