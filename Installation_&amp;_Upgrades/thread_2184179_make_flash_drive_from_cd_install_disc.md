### Post by fantab on 2013-10-28
You can use Brasero Disk Burner (which installs with Ubuntu) to make an .iso of the CD/DVD. 
USe 'Disk Copy' option and save the file as an image... once the .iso is ready BURN it to a USB with 'Startup Disk Creator' or [Unetbootin]("http://unetbootin.sourceforge.net/").
