### Post by wpshooter on 2008-12-15
I would suggest that first you make/create a separate directory/folder in your /home partition (something like - linksys), then copy the tar.gz file to that directory, then right hand click on the file and choose EXTRACT HERE.

Once that is done you will be able to install the program that you need.

Probably install by running something like: ./INSTALL at the terminal prompt.  Make sure that you are at the proper location before issuing the install command.  You may want to see if there is a readme file to give you exact instructions for the installation.

Good luck.
