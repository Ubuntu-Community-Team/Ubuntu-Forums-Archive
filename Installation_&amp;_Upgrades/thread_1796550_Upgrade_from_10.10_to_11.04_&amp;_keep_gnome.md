I have can't see a difference between the upgrade to 11.04 and the fresh install to 11.04.

On 11.04 you can choose at the login screen, Ubuntu classic.  This will give you the regular gnome desktop.

It's written that this option might not be available by default in 11.10.  Even if the easily default selectivity is removed, I'm sure a user would still be able to still install gnome to 11.10.

By the way, I found the Unity to be very lacking until I discovered docky.  The biggest thing I missed was being able to individually select a specific task or window from running programs.  Unity will bring up the whole group and bury your work.  The docky addon resolves this annoyance.
