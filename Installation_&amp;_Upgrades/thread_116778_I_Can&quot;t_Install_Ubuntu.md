### Post by Xumbi on 2006-01-13
Did anything happen at all when you booted your system with the Ubuntu CD in the drive?  Did you see the Ubuntu logo?  If you did, post any error messages you saw after that.

If you didn't see the Ubuntu logo, then that means that either the CD you burned is bad, or your laptop's BIOS isn't set to boot from your CD drive before your hard drive.

I would recommend downloading and burning an Ubuntu live CD first, make sure you can get that to run, then try the installation again.

### Post by ajgreeny on 2006-01-13
Make sure you burn a disk image of the iso file you downloaded, not simply copy it to the cd.  Look in your file manager to see what is on the cd;  if it is just one iso file, then you got it wrong.  There should be several files showing in a number of folders.

### Post by gosh on 2006-01-13
Did you download it in windows?

The file has an .iso extension.
You can burn it to cd with Nero, Alcohol 120%, Easy CD Creator, etc.
Look on download.com for a program if you don't have a cd-burner.
