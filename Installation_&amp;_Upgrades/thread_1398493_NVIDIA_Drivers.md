### Post by audiomick on 2010-02-04
The easy way is to install and then wait for the system to offer you proprietary hardware drivers. That usually comes within a minute or so of the reboot after the install, assuming the computer has internet access. If nothing comes, you can do it by hand through
system> administration>  hardware drivers.

I have read here that it is possible to add a nVidia repository and get drivers direct from there using apt-get / the synaptic package manager. This apparently supplies a newer version than is included in the Ubuntu repositories. I am not sure, but I think there are instructions on the nVidia site somewhere.
