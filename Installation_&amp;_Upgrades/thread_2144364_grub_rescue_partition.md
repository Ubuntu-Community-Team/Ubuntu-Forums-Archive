### Post by fantab on 2013-05-11
If you have a Ubuntu install disk you can use it to boot, 'Try Ubuntu', run Gparted from the desktop and delete all 'unnecessary' partitions. Or if you don't have Ubuntu Live, you can use [Gparted LiveDVD/USB]("http://gparted.sourceforge.net/livecd.php").

Now use your Windows install disk to reinstall and repartition accordingly.
