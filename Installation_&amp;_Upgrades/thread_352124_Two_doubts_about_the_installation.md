### Post by aysiu on 2007-02-03
During installation, you can select what language you want for Ubuntu. I believe Italian is one of them, but you can check by clicking the **Install** icon off the live (Desktop) CD. You don't actually have to go through the entire installation.

Also, the default Kubuntu has an update notifier. When updates are available, there's a little pop-up, and you can just click it, enter your password, and install all the updates.

I think Synaptic (not the default package manager in Kubuntu, but it can be installed) has the option in preferences to automatically install updates.
