### Post by scratman on 2009-10-02
First of all I'm turning a hideous shade of green, that's an awesome rig.

Secondly, burn the ISO to a CD, and then leave it in the drive when it's burned.  Restart your PC, but go into BIOS, make your machine looks at the CD/DVD drive before the hard drive for a boot image.

Restart the PC, let the Live CD loads up, and make sure things look OK.  If everything's fine, select Install Ubuntu, select the partition you have left free, and then install.  If you hit any problems after install, come back here and ask again.
