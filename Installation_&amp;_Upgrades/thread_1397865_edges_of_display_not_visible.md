Hi,

I have just set up my new Acer R3610 Atom/Ion netbook attached to my TV via HDMI. I installed UNR 9.10 and the NVidia restricted driver 185.

Unfortunately, the edges of the screen are cropped off and invisible. This is a problem regardless of which display mode I use.

The Windows7 NVidia driver has a configuration option that sets the visible extent of the display.

How can I configure UNR to fit the actual space on the screen?

Rick

### Post by rickbeton on 2010-02-06
OK it turns out this is a Sony Bravia TV 'feature' not related directly to the HDMI output from the Ubuntu PC.

By default, the Bravia expands the screen to overfill the pixels available. There is a configuration option called "full pixel" that switches to a 1:1 pixel size. With it, the edges are fully visible.

The relevant menu on the Bravia is Setup> Screen Settings> Display Area.

Rick
