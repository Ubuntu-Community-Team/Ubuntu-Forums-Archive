### Post by Thelasko on 2008-04-18
[Wikipedia ]("http://en.wikipedia.org/wiki/Intel_Core")says you should run the standard version as the T2050 is a 32-bit processor.

A few words of advice:

1.  Don't install random software from the internet.  Use the repositories! (either applications>add/remove software or the command ```
sudo apt-get *name of program*
```)

2.  Use the command "sudo" to do root user commands (change system settings add and remove software etc.) from the command line.  This is to prevent viruses from installing themselves on your system and it works! (not that there are many Linux viruses but this is the reason why)

3. You probably won't be able to get Compiz running with your Intel graphics card (no big deal it's just for show)

4.  Always search the forums before posting, 9/10 times somebody has asked the same question.

5. If you should find yourself staring at a black screen with a blinking cursor and don't know what to do type:```
startx
```If that doesn't work type:```
sudo dpkg-reconfigure xserver-xorg
```Write that down!

That's my crash course of the most common problems new users have.  Don't feel intimidated it's very easy.

### Post by Thelasko on 2008-04-21
From what I understand the Intel graphics driver cannot do 3d and 2d video at the same time.  Games mostly use one or the other at any point in time.  The Compiz problem occurs when the Intel driver tries to render video on the 3d cube.  This usually results in the video player crashing.  Compiz is disabled on computers with the Intel graphics driver by default.  There are ways to enable it and ways to get it to partially work.  Search the forums.
