I am playing with Kubuntu, what do I have to do to install Synaptic?

### Post by amingv on 2009-10-28
On an terminal, run

```
sudo aptitude install synaptic
```

This will install it. After it finishes you can find it in K Menu>Applications>System>Synaptic Package Manager.

Nothing needs to be uninstalled.
