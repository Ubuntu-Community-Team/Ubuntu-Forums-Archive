### Post by lemming465 on 2009-01-19
Let me guess, nvidia video chip with their legacy binary driver?  Try turning off compiz visual effects (Preferences -> Appearance -> tab "visual effects", change the radio button to "None").  There are already bugs open in launchpad about it (I filed one of them myself), so if this resolves your problem, you don't need to follow up.

The open source "nv" video driver doesn't have this bug, but lacks 3d acceleration support.
