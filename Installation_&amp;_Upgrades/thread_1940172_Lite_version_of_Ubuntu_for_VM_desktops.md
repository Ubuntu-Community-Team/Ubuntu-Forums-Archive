### Post by sudodus on 2012-03-13
I suggest that you select the ***Lubuntu*** flavour with LXDE which is a desktop environment with a very small footprint. The default application programs are are very small, but you can install whatever software you want from the Ubuntu repositories. Download the iso file of the present version 11.10 and the coming 12.04 long time support version which is in beta stage now and test them 'live' before installing.
