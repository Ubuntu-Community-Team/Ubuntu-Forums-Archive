### Post by carlosfocker on 2007-03-30
If your just hooking the DSL modem straight to your computer you should only need to follow their instructions on how to physically hook up the modem to your pc. This usually consists of plugging the Ethernet cord from your computer to the modem, Plugging in the phone line into the modem (with filter if needed), and plugging in the power to the modem. Sometimes on first use of the DSL service you maybe brought to a web page where you need to enter in account info. They used to give you a cd that you had to run but im not sure thats done anymore

### Post by flyup on 2007-03-30
If you have a westell versa link modem just make sure its on and connected before you start linux. by on I mean you should have a power light, an e(1-4) light on if hooked up by cable a dsl lite(steady) and an internet lite on. If internet light is not on then press and hold reset button on back of modem for 20 secs. This will put your modem online without having to connect. Verizon will tell you how to set it up for windows (always on mode) and it works for ubuntu
