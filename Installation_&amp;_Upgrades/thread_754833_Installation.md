### Post by njparton on 2008-04-14
You downloaded a .iso file?
 
This is a CD image and you can burn it directly to a CD using programs such as nero (or a free alternative if you search for one).
 
Remember to use the burn image option in your chosen CD burning software otherwise you'll just end up with the .iso file on your CD - buring the image in nero etc actually extracts the files contained within the .iso at the same time.
 
Leave the CD in your optical drive and then boot from it :)
