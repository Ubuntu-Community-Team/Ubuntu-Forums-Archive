### Post by audaz on 2005-04-13
You need to make sure that you don't just burn the ISO file to a cd.

I don't know which program you use to burn cds, but in Nero you use the burn image to disk option. That will make the cd bootable.

Then just stick it in and restart it. Make sure your system bios will check for cd before the hard drive.

I hope this helps.
