### Post by dstew on 2010-03-01
> is there any chance to get my data back???It depends...

If you simply deleted the Windows partition, but did not write new data over it, you might be able to recover it using a program like Testdisk. See [this Ubuntu document]("https://help.ubuntu.com/community/DataRecovery") for details.

If you by mistake installed Ubuntu in a new partition that was created out of the space of the old Windows partition (the "Use entire disk" option), I am afraid you will not be able to get your data back. Hopefully you have a back-up of your data on another disk somewhere.
