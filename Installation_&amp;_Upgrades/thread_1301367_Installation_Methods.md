### Post by macaholic on 2009-10-26
The most advanced and detailed system is Synaptic, it gives you many options and will list every package from sources you've enabled.

Sources, often referred to as repositories or repos, are indexes of packages and their dependenccies (programs and libraries and their requirements to be installed). Source manager allows you to add third-part sources for software not always offered in the default Ubuntu repos. These are usually in the form of a deb line, ie "deb [http://archive.ubuntu.com/ubuntu/](http://archive.ubuntu.com/ubuntu/) karmic main" or, in 9.10 specifically, with PPAs from [launchpad]("http://launchpad.net/"),something like "ppa:deluge-team/ppa"

Also in 9.10 there is now the "Ubuntu Software Center" which is, from my viewing, similar to Add/Remove, with the potential for a future application storefront and other improvements.
