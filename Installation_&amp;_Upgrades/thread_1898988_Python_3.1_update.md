### Post by oldfred on 2011-12-24
If you have a python3 installed, it should be in addition to a 2.6 or 2.7 revision that Ubuntu uses for internal things. You never want to un-install the standard version but can install additional versions without issue as long as you do not install from outside Ubuntu's repository  and have it then overwrite the standard install.

Or the upgrade should be just fine.
