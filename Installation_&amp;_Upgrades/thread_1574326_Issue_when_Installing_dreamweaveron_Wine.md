Hi,

   Just want to let you guys know that i have found a work around for this issue.


[LIST=1]
[*]I reinstalled the Wine application.>>> sudo apt-get  --purge wine
[*]Installed Microsoft data access components 2.7
[*]copied odbc32.dll and odbcint.dll from the windows machine to .wine/c/system32 folder in linux.
[*]configured the Wine application with odbc32.dll as native windows.
[*]Installed Dream weaver on the machine
[*]and works fine.
