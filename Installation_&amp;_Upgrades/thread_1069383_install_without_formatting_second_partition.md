### Post by Mark Phelps on 2009-02-14
Safest thing to do would be to download the GParted Livecd, boot from it, and use it to reformat the partition you want to use to Ext3 format.  It will launch a GUI tool with the same view as the partition tool in Ubuntu.

Best place to find the GParted Live CD, and a bunch of other stuff as well, is the distrowatch web site.
