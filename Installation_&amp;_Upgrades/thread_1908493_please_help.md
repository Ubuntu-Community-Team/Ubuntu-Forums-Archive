### Post by darkod on 2012-01-14
Not everyone is having issues. Don't forget that the forums are usually for your problems, so it might look like it when reading here.

And the other poster didn't say you can not upgrade, he said you can not 9.04 to 10.04. A non-LTS release can only upgrade to the next release, if I'm not mistaken. With LTS releases you have the option to jump directly to the next LTS.

You could upgrade 9.04 to 9.10 to 10.04. Although personally I am doing clean installs and not upgrades.

The issues you mentioned above seem like hardware issues. You should always be able to boot from bootable CD for example, no matter which OS is installed on the HDD. In fact, you should be able to boot a CD even without a HDD in the machine.
