### Post by merlinus on 2009-08-10
If you did not do so, checksum the .iso and burn to a CD-R at no more than 4x.  If you get to the opening menu, then check the cd for errors as well.

If all this still leads to failure, you might try the text-based alternate install cd.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
