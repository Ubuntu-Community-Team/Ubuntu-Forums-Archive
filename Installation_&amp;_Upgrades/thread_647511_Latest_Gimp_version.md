### Post by PilotJLR on 2007-12-22
Brand-new versions of applications are usually not rolled into existing releases... for example, this is why Ubuntu Dapper LTS still uses Firefox 1.5

So you will need to not use the repo's and download/install from gimp.org instead

Updates are usually for bugfixes and security issue fixes
