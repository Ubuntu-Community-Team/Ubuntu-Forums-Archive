### Post by sikander3786 on 2010-11-25
> **igotsanevo4g said:**
> What do you mean?
As soon as the computer starts booting from the disc, press any key and you'll gain access to a menu. It lists the option to "check the disc for defects". Check it!

[https://help.ubuntu.com/community/LiveCDBootOptions](https://help.ubuntu.com/community/LiveCDBootOptions)

And you might also need to check the integrity of downloaded image used to burn the disc.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If needed, burn a new disc and at the lowest possible speed this time (I use 4x).
