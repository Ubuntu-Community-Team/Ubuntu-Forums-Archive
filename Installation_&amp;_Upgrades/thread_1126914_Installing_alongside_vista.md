### Post by Mark Phelps on 2009-04-15
Since you don't want dual-boot, what CAN be done is that Ubuntu can be installed INSIDE Windows.  This is not dual-boot, it is a form of virtualization where Ubuntu is installed as an application in MS Windows.  

It's done using Wubi.  See the link below for the Wubi Guide:

[https://wiki.ubuntu.com/WubiGuide]("https://wiki.ubuntu.com/WubiGuide")
