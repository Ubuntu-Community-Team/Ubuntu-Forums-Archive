Johndoe:

Yes, Windows and Ubuntu/Linux MAY be able to co-exit, key word MAY.

Therefore, my recommendation is that if possible you get yourself an oldish machine and start out by experimenting with installing Ubuntu ALONE on it before you try to start doing any DUAL booting on your Windows work/production machine.
