### Post by csevcik on 2011-05-14
I am also very disappointed with U 11.04, I have 6 machines running Ubuntu and 1 with wubi. I have 5-6 years of experience with Ubuntu.

But the introduction of Unity is a disaster for me. Many of the things I did with Gnome are messy in Unity. I cannot run Crossover, for example, which runs nice from "Classical Ubuntu" in the same machine, some of the icons in Unity are just a gray space with a question mark. And to make things worse in one of the machines with an nVidia Quatro 500 card the driver (173) doesn't work neither on Unity nor on "Classic Ubuntu". I am froced to use the "experimental" open driver which is just barely acceptable, and under which some applications (Ghemical, for example) do not work. The Wine Windows emulation is also faulty under 11.04, many applications that worked nice now do not work or crash (Astrolog 5.4, for example).

Running on "Classic Ubuntu" is no solution either, it is not "Classic" it is "Crippled Ubuntu", just try Compiz, none of the old features (cube cube rotation using a mouse or active window selection) work anymore and old bugs persist (need to reload the windows manager which randomly collapses for example). 

All this seems a dictatorial way to force a shift to Unity which I very much resent. Prehaps is time to quit Ubuntu for good.
