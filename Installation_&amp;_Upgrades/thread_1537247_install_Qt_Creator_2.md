### Post by wkulecz on 2010-07-23
Unless you've a burning need for Qt Creator 2, its safest to install the one in the current repos and hope for an update soon.

I've found the Qt stuff to have a rather fragile set of dependencies over the years so I've quit chasing the bleeding edge.

OTOH, after looking at the link, you download a *.bin file which the instructions seem to suggest you run it as a normal user after making it execuatable, so you should be able to install it for your use only without it messing up anything else.
