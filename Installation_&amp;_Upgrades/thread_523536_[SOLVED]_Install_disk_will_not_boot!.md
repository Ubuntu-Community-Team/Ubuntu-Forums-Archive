### Post by aysiu on 2007-08-12
You have to burn it as a disk image.

WinRar will try to associate the .ISO as a WinRar file, but it is still an .ISO. Just ignore WinRar completely.

Burning it properly isn't just about using the right program (both Nero and InfraRecorder will work). It's about picking the right burning *options*. Do not burn it as data. Do not burn it as a "bootable disk." Burn it as a *disk image*.

More details here:
[http://www.psychocats.net/ubuntu/iso#burning](http://www.psychocats.net/ubuntu/iso#burning)
