### Post by sudodus on 2014-02-10
10.04 is still alive as a Ubuntu Server (until April 2015). It means that the packages that belong to Ubuntu Server will continue to get updates (also security updates), but the packages, that belong to the desktop will not receive any security updates. I cannot recommend running it with a connection to the internet, particularly the web browser, email client and such programs. So you are 'on your own' concerning security updates.

Have you tried a some flavour of Ubuntu 12.04 LTS? If standard Ubuntu is too heavy for the hardware, maybe Xubuntu or some re-spin based on 12.04 will work for you: LXLE, Bento, Bodhi, Precise Gnome Classic Tweaks.

Edit: And of course you should try Lubuntu 13.10 (as suggested by Bashing-om) and ask detailed questions here if you have problems.
