### Post by Mark Phelps on 2009-10-17
If you quit cursing about Vista for a minute, you might learn that it uses lots of space that is not easily seen.  There is a hibernation file reserved by default that uses as much space as your system memory.  Every since Windows Update uses as much space as needed to save a copy of all the files it replaces.  Event logs take up space. System Restore also allocates disk space for shadowing your drives -- to make copies of files and to store restore points.

Your actual installation space is about 22GB -- not bad for a Vista installation.

If you were to Google for Vista performance improvements, you would find some links that give you advice regarding services to disable (including hibernation) that would free up some disk space.

I'm not a fan of Vista, either, but your writing a post that reads like a tantrum is not going to encourage folks helping you.
