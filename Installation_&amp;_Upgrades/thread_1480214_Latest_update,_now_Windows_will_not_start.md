### Post by darkod on 2010-05-11
There are two different terms used but with very close meaning.

Update is when you receive updates for the version you have.

Upgrade is upgrading to newer version after it's released.

If you actually upgraded to 10.04 it is probably the issue you were reading about. In that case look here:
[http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Boot_Sector](http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Boot_Sector)
