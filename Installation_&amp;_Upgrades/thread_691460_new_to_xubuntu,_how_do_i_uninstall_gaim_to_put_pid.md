### Post by zvacet on 2008-02-08
First you need all repositories to be open,so go to the systtem>administration>software sources>chek all boxes under Ubuntu software tab and all boxes under updates tab.Reload.Now go to the system>admin>synaptic package manager>in search box type pidgin and when you find it mark it for installation and press apply.More updated version of Pidgin you can find [here](http://www.getdeb.net/search.php?keywords=pidgin).You can download it in your home dir and just double click on it.
