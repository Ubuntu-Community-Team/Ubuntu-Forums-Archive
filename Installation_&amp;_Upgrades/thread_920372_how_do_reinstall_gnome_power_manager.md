### Post by Partyboi2 on 2008-09-15
when gnome-power-manager was uninstalled it probably uninstalled ubuntu-desktop as well, so reinstall ubuntu-desktop which will reinstall gnome-power-manager.
When ubuntu finishes loading press Ctrl+Alt+F2 to get to a terminal then login and reinstall ubuntu desktop
```
sudo apt-get install ubuntu-desktop
```
