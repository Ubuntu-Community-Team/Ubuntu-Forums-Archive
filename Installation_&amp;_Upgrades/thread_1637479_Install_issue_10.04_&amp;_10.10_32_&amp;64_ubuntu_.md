### Post by sikander3786 on 2010-12-04
Hi. Welcome to the forums :-)

At the screen where you see two icons at the bottom of your screen, press any key. A menu should pop up. Select "Check disc for defects" and press ok. If the result it ok, press F6 and select "Nomodeset". Then press Try Ubuntu without making any changes. Hopefully it should boot this time. If not, please provide a little more details as to let us know what happens after the two iconed screen?

[https://help.ubuntu.com/community/LiveCDBootOptions](https://help.ubuntu.com/community/LiveCDBootOptions)
