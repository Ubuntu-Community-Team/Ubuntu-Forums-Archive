1. There is Open Office in Linux to replace Microsoft office, it is compatible with MS Office files. You can read files created with MS Office and when writing files you have option to save in Open office format (which MS Office can read) or save them in MS Office format (which Open Office can read as well). 2. Whether to keep windows or not is a personal choice. There is no shame in using windows and don't be led to believe that there is anything "wrong" if you choose to keep windows on your machine.
