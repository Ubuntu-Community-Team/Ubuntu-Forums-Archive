I have the same problem. I went to the CNETUSA site, and I found a generic driver for my specific model of the card, but it is designed for a 2.4 kernel.... I know Ubuntu 6.06 uses 2.6 by default. Here's the URL I found:

[http://www.cnetusa.com/newwebsite/techsupport/driverdownload.htm](http://www.cnetusa.com/newwebsite/techsupport/driverdownload.htm)

If you download the EXE on windows, it will self-extract to a folder you can choose, and under that there will be a folder called "LINUX" and inside is a tarball. There's a readme file in there. (If you can't read a tarball on Windows you can download 7zip, [http://www.7-zip.org/](http://www.7-zip.org/)) The readme has instructions but they seem very complicated.
