### Post by daou on 2006-12-10
Try this [guide]("http://wiki.beryl-project.org/wiki/Install/Ubuntu/Edgy/nVIDIA").
It is somewhat similar. The "wget" command that you got stuck at installs a gpg key. This isn't necessary. The key is used to authenticate updates, but if you don't have it, you will just get warnings which don't really matter.

And the server might be down. Try the wget command another time if it doesn't work right away. The guide I gave uses a different server.

Use method 1 in the guide to install your nvidia drivers. Use the first repository (for i386/x86_32 architecture). Do not use the Amd64 version because you are running a 32bit kernel.

### Post by daou on 2006-12-10
You can also check if your current nvidia drivers are working by running this in your terminal:

```
 glxinfo | grep direct
```

If it responds with "direct rendering: Yes", they should be working.
