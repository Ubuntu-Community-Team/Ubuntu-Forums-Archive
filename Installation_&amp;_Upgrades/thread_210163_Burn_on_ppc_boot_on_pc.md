### Post by adwait on 2006-07-06
The image is an exact copy of how the CD should be with all the options etc. built into it so technically shouldn't make a difference whether it is burnt on a mac or whatever, it should still be bootable on a pc if you have downloaded the pc version of the CD. Check the MD5 sum of the image you downloaded and compare it with those published on [www.ubuntu.com](www.ubuntu.com)
