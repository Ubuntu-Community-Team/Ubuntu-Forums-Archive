### Post by oldfred on 2011-02-12
If you do not install any proprietary drivers, it should just boot. 

The main issue is usually video drivers, but defaults should work, perhaps not optimally until the proprietary driver is installed, if needed and with an older computer it may not.

Did you burn CD at slowest speed that you could? If it is an older computer did you download the 32bit version? Does the CD boot to run as liveCD. You may want that to be able to do repairs anyway.
