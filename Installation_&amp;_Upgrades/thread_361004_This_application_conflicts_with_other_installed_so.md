### Post by jpeddicord on 2007-02-13
That error is because of dependencies of other packages in the system. To install Nvu, go to System > Administration > Synaptic and search for 'nvu'. Click the checkbox and select Install. It will probably ask you about installing/removing other packages. If you are fine with what it gives you, click OK, then click Apply on the toolbar.
