### Post by LaRoza on 2007-05-29
If you are using linux, there would be no "A:\>" prompt, that is a Window's syntax.

Try booting from the disk again, with these steps:

1. Insert bootable disk into cd tray.
2. Turn off computer or shutdown.
3. Turn on computer.

At this point, it should work, if you are not seeing  Ubuntu or a Linux error message, start the computer up again, but this time, access the bios settings and select your cd player as the boot device.

-edit How you access your bios is different, usually, you press "esc", "del","F1" or some combination, there will be a message telling you what the key(s) is before the system boots an OS.
