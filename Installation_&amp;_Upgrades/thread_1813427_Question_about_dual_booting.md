### Post by ajgreeny on 2011-07-27
Is this a proper partitioned dual boot or a wubi installation of ubuntu inside windows?

If you have the two systems on different partitions you can go ahead and re-install vista, which will probably not even see the ubuntu partitions.  Just to be safe, however, make sure you have backups of everything first.

Once installed, you will find that windows has taken over where grub used to be so restore grub by following the instructions on the **Grub2 basics** link in my signature
