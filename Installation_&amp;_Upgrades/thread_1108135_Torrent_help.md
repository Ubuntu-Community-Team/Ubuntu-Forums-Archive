### Post by graysky on 2009-03-27
Wait... are you running Ubuntu or Windows?  If you want a torrent client for Ubuntu that looks more like utorrent I would suggest deluge.  Version 0.5.x is in the Intrepid repos (1.1.5 is in the juanty repos) or you can download/compile from the [deluge page](http://download.deluge-torrent.org/source/).

To install the 0.5.x version on Intrepid:

```
$ sudo aptitude install deluge-torrent
```
