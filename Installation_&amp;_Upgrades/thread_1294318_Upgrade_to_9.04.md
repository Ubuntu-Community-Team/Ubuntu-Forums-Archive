### Post by ajgreeny on 2009-10-18
You should certainly upgrade to the newest version, 9.04, or wait for 9.10 later this week, but either way, you can't do it in one jump, I'm afraid, so probably best to backup all your personal files and start from scratch by downloading the install/live CD.

Distro version updates can only be done via the update manager from one version to the next, or from one LTS version to the next LTS version.  From 7.04 to 9.04 is 3 jumps so not possible, as far as I'm aware.
