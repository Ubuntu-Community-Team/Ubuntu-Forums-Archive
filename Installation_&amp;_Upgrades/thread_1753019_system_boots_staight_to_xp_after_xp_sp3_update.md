### Post by Quackers on 2011-05-08
Try booting from the Ubuntu 11.04 live cd/usb and select "try ubuntu".
When the desktop loads open up a terminal and copy/paste these two commands in one at a time, pressing enter after each line.
```
sudo mount /dev/sda7 /mnt
sudo grub-install --root-directory=/mnt /dev/sda
```
If that reports "no errors" please reboot. This time Ubuntu should load directly. If it does open a terminal and run ```
sudo update-grub
``` and watch to make sure that the Windows Loader is picked up as grub.cfg is run.

### Post by drs305 on 2011-05-08
You do not have Grub installed in the MBR of sda. To install it, boot a Natty LiveCD, mount your Ubuntu partition (sda7), and install Grub:
```

sudo mount /dev/sda7 /mnt
sudo grub-install --boot-directory=/mnt/boot /dev/sda
```
Do not include the partition number in the second command.

Note this is a slight variation of the commands presented in Post #2, but those should have worked as well. Since you have installed Natty, make sure you are using a Natty CD.
