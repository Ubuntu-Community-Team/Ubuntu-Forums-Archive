### Post by lisati on 2012-08-15
Do you have update manager set to check CDs? On my machine, running 11.04, the setting can be found by opening update manager, clicking on "settings" (you might be prompted for your password), and opening the "Ubuntu Software" tab.

1.- To introduce in the required CD, DVD with ubuntu 12.04 OS
2.- Or going to software sources, under administration menu and in the tab corresponding to ubuntu software, uncheck in the bottom part the CD-ROM software installable from CD-ROM.
