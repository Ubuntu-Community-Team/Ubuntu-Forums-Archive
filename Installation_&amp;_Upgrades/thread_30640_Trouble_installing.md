### Post by OJe on 2005-04-30
FIX: the LCD display was connected through, DVI interface, this was clearly not working good with the linux installer. Same error happened when trying to install Fedora Core as well. 

Solution: use old VGA cable instead of DVI. I'm surprised that this actually has something to say.. Maybee it's just something with my Hyundai LCD screen. :/ oh well
