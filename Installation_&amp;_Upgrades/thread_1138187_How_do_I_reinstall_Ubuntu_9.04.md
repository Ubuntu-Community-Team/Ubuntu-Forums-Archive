### Post by Partyboi2 on 2009-04-26
Without knowing what type of pc you have, try booting the machine and pressing one of these keys to enter the bios.
Delete
F1
F2
F10
ESC
Then look for boot order, boot sequence or something like that and make sure that the cdrom is set to boot first.
[http://www.whitecanyon.com/how-to-change-boot-order.php](http://www.whitecanyon.com/how-to-change-boot-order.php)
