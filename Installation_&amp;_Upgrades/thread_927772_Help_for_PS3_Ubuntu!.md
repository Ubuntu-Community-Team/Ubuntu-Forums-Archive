### Post by tarps87 on 2008-09-23
The none-live and alternative cd's are the same thing, they are text based installers. Which version of Ubuntu you use is up to you, id suggest Ubuntu, Kubuntu or Xubuntu. Edubuntu is more based towards schools.
Ubuntu is the default with the gnome environment
Kubuntu uses the kde environment
Xubuntu uses xfce and is designed for slower systems.

I plan to install Ubuntu this weekend when I get my ps3 :)

Edit: If you are using a standard TV (not HD) you should use the alternative/none-live cd to install it
