### Post by Elfy on 2012-01-01
IF you are having trouble with the partitioning part I would seriosuly suggest that you ensure that you have both checked the download's md5sum (unless you got it from a torrent) and the disk's integrity.

If you still have issues perhaps a screenshot might cast some light on this - I've never had the installer not take any notice of what I've told it.

I assume this is not the dev 12.04 unreleased version.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
[https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)
[https://help.ubuntu.com/community/Installation/CDIntegrityCheck](https://help.ubuntu.com/community/Installation/CDIntegrityCheck)
