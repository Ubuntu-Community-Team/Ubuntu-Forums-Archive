### Post by critin on 2012-06-05
If it won't boot any computer, there's a problem with the burn.  Check the  md5 sum and burn at the slowest speed possible.  Some burners only go down to 4, but that should work.

Be sure to push the 'Boot From' key on the Bios page, or change the Bios boot order if necessary to boot from CD.

Use a USB stick instead.

> 
 it shows all the Ubuntu files but no option to start the installation.



It won't 'run' like Windows does.  It must be 'booted' from start-up.
