### Post by Paul T. on 2009-06-16
I downloaded all the upgrades and experienced no problems, no lost progs or files etc. However I've since done a clean install of 9.04 and this is much better, system runs much more smoothly so I would recommend backing up all your important files and going this route.

### Post by merlinus on 2009-06-16
If you install rather than upgrade, then all your add-on apps will have to be reinstalled.  Also, unless you have a separate /home partition, their configuration and setup files, and your personal data, will be lost as well.

You can of course backup the important files.
