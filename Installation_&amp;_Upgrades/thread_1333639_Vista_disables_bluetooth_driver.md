I have installed Ubuntu 9.10 - the Karmic Koala on my Dell E1505 running Vista Home Premium.

I did get my wireless working and my bluetooth mouse working fine.  

I found that after I boot back into Vista, then back to Ubuntu my bluetooth driver is disabled/hidden and I can't find a way to get the bluetooth driver to show up again.

When I open Hardware Drivers the Broadcom STA Wireless Driver is still in the list but the driver for the bluetooth is gone.  

I did not run any updates after installation thinking this was causing the problem, but it's just the fact of booting into Vista.  As long as I just boot into Ubuntu the driver for the bluetooth is fine.

Any ideas?
