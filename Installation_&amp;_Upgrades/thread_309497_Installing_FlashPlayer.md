### Post by SyvanX on 2006-11-29
You can copy them to your plugin directory:
sudo cp libflashplayer.so flashplayer.xpt /usr/lib/firefox/plugins

I always had trouble with the manual and repository installations of flashplayer.
If you're using firefox on x86, try going [to the flash test site.]("http://www.adobe.com/products/flash/about/")

Then there should be an option to "Install Missing Plugins" at the top of the firefox window right below the menubar.  That works better for me for some reason.
