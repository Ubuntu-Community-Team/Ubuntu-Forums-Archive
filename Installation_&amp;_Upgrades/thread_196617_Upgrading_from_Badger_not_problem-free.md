### Post by hillsh on 2006-06-14
The upgrade from Badger to Drake was/is not problem free.  Here are the problems I have come up with so far (my system uses an AMD64 processor with an ATI Radeon type video card):

1. X would not start because of problems with fglrx driver - fixed by deleting and reinstalling the linux-restricted-modules and xorg-fglrx drivers in the correct order (?) - linux-restricted first.

2. My only pointing device is a Wacom Graphire 4 tablet that worked fine in Badger, but now only the stylus works as a pointing device - the Graphire mouse does not move the cursor - and very poorly as well - I have not been able to fix that problem yet, will probably just plug a PS/2 mouse into the system for a workaround.

3. Shutting down and restarting the system does not work - have to do a hard reset to get the system to start up again - this never happened in Badger.

I have a dual boot machine running Windows XP - have no idea why anyone would choose a Linux system over a Windows system to get real work done - the only reason I use Linux is to develop software for a Nokia 770 Internet tablet - otherwise I would never touch it. Of course one can argue that it is free, but if I billed out for the time I have spent getting Ubunto to work on my system I could have purchased ten copies of Windows XP!!
