### Post by jabeez on 2007-12-21
It is very straightforward to install via Adept. Open Adept (it will ask for password, which you would have set during install), look towards the top of the window and there is a search box, type in firefox. You don't even have to push enter, the list of matching programs updates automatically. Look for Firefox, click on the arrow to the left and it will expand to a description and click the install button. Then click the apply buttons at the top of the window and it will install. Voila. (jeez it sounds way more complicated than it actually is when you try to write it out:))
