### Post by mcwnz on 2009-05-25
Just installed vanilla 9.04 jaunty on a VIA VB8001 = CN896. Same problem as you. Seems that there is an issue with the mouse pointer in Openchrome.
Bug appears to be known. A workaround is to enable SWCursor ([https://help.ubuntu.com/community/OpenChrome](https://help.ubuntu.com/community/OpenChrome))
I confirm that this fixed the issue for me.
