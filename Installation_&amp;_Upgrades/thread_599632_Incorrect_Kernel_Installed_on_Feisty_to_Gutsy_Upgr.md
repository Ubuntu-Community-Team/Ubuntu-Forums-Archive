### Post by peter_ryan on 2007-11-01
I completed the kubuntu feisty to gutsy upgrade on my D620. I ran into a few issues. 

Most notably the installed hanging which caused me to resume it manually using dpkg. Not that big of a deal.

The second issue I needed to resolve was the upgrade installed the latest 386 kernel instead of the generic which led me to only use one of my cores on my laptop. I fixed this by uninstalling the 386 kernel.
