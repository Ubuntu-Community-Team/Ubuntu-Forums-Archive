### Post by monty95 on 2009-02-08
Bought a 4GB SanDisk Cruzer Micro USB flash drive, to use for installing Ubuntu using "Copying Files - The Easy Way" from page
[https://help.ubuntu.com/8.04/installation-guide/i386/boot-usb-files.html](https://help.ubuntu.com/8.04/installation-guide/i386/boot-usb-files.html)

First attempt failed: machine to be installed to hung at flashing cursor during boot.

Used a Windows-XP machine to remove the U3 software, following the instructions at
[http://www.u3.com/support/default.aspx#CQ3](http://www.u3.com/support/default.aspx#CQ3)

Repeated "Copying Files - The Easy Way" and it worked.
