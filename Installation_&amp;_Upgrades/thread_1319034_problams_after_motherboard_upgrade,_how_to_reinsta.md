### Post by ricardo.gloe on 2009-11-08
There are probably other ways you can do this, but the most user friendly is:

install remastersys, make sure you install the Karmic version from [http://www.geekconnection.org/remastersys/ubuntu.html](http://www.geekconnection.org/remastersys/ubuntu.html). 

Use remastersys to backup your system, burn the iso to a CD/DVD and then use it to install. You can also put it on a pen drive using UNetbootin. 

But I can't say for sure the motherboard upgrade and the slowness are related, much less if a reinstall would solve it.
