### Post by A. J. Rimmer on 2009-04-26
I recently installed Intrepid using Wubi and am very impressed by how easy it was and how well it works.  Essentially, it makes a large file on your hard drive and treats it as a virtual partition.  However, unlike true virtualization (e.g., Sun xVM VirtualBox), Ubuntu sees your real hardware, not simulated hardware supplied by the VM.  Also, if you don't like the installation or something doesn't work, you can just go to Control Panel and use Add/Remove to uninstall Ubuntu and remove the "virtual partition."

I have two laptops set up to dual boot Hardy Heron and XP (which I only use to run my Hauppauge HVR-950 on one and my wife uses MS Office on the other).  I installed Intrepid on the Windows partition on one of them so I could check it out and help a friend who has just installed Intrepid as his first experience with Linux.  I prefer to stay with the LTS releases as much as possible, but like to see what's new in the latest releases, so Wubi is perfect for that.
