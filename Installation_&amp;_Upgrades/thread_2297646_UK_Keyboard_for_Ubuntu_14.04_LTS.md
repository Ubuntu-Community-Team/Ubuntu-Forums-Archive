### Post by grahammechanical on 2015-10-05
I do not understand by "no longer works." Is the UK keyboard layout present? Was this upgrade from 12.04 to 14.04 a fesh install?

There should be a an icon inn the top panel.  A new install with have English (US) as the default keyboard layout. The language icon in the top panel has a menu and Text Entry Settings will load a dialog that will let us add an keyboard layout under Input source to use. There will be a long list of languages/keyboard layouts. We click the plus ( + ) sign. Once we have added the keyboard language layout of our choice we can remove the one we do not want by selecting it and click the minus ( - ) sign.

Another way to do this is from System Settings>Text Entry.
