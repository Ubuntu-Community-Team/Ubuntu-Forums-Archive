### Post by adwait on 2005-06-03
1) Ensure that you have not simply copied the iso file you downloaded from the net to the CDROM. You need to specifically select the option such as "Burn from Image" or something to that effect in your CD burning software.
2)You need to be able to boot from the CD on your computer. If you put the CD in your drive and then restart the computer, and nothing happens , then the BIOS is probably not checking CDROM for a bootable disk. To change this, press Del when the computer is checking its memory (immidiately after you start the computer). Now go to the General settings sections, and you should find something called as "Boot Sequence" or something to that effect. There ensure that you have the CDROM drive before your hard disk.

### Post by C.B. on 2005-06-03
Aside from making sure your machine boots from the cd, you need to make sure the installation file you downloaded was copied correctly onto the CD as an ISO file. Apparently this doesn't happen automatically. I am in the middle of doing this myself, so the following is newby (as opposed to sage) advice.

Go to [www.ubuntulinux.org/wiki/BurningIsoHowto](www.ubuntulinux.org/wiki/BurningIsoHowto) . Scroll down to "Burning/writing an .iso Image to CD with Windows XP." There you will find out there is a missing component in the XP file transfer/CD burner thingy which you'll need to download and patch. Instructions are pretty straightforward. Like I say, am in the middle of it all myself, but seems to be going okay . . . so far. Of course, I may have to wipe some egg off my face and repost.
