Since you're using Vista, you might want to read over this link first:
[http://apcmag.com/5046/how_to_dual_boot_vista_with_linux_vista_installed_first](http://apcmag.com/5046/how_to_dual_boot_vista_with_linux_vista_installed_first)

[http://www.getautomatix.com/](http://www.getautomatix.com/)
you'll get some naysayers here on the forum for using Automatix, but it's worked great for me on many Ubuntu versions and Debian(Etch)...it'll get you up and running in no time.

I have a Nvidia card, but there are several howto's around for installing ati drivers:

Dump AOL and get yourself a plain jane ISP that doesn't use any proprietary software.  Copper.net, Fry's.com, totalspeed, 550, etc.
