### Post by lotster on 2009-06-29
Hi!

From what I've seen on some other web sites, it seems that it may be caused by a damaged disc / corrupted ISO.  You could try using another disc.

Also, are you installing the installation from within Windows?  I have heard of this problem occurring if Ubuntu is installed from within Windows.  If that's the case, try to boot to the disc, run the install from the boot menu, and when prompted for partitioning information just point it to your second hard drive.  (It leads to better performance, too.)

I hope you get it sorted!
