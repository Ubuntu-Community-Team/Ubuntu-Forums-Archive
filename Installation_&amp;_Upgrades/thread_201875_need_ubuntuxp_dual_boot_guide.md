### Post by e_james on 2006-06-22
I'm just a beginner myself but the best I know at this time is you DON'T use the Live / Install CD for dual boot installations. It's likely to wipe the XP partition if you try. Use the Alternative install CD instead. I just did. The partitioning was no problem. Initial user permissions is my problem.
