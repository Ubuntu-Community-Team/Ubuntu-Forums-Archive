Check your BIOS to make sure the CDROM drive is hooked up properly.  (It almost certainly is, but you need to cover all bases.

If you have access to an external CDROM drive that you can connect and boot from, I'd consider disengaging the problematic drive and installing from an external drive.  

I have successfully installed Ubuntu from the hard drive, but I don't recommend it.  You might look into this further.  I'm sorry, I could not begin to describe for you what I did to get this to work. 

Try a reburn of the Xubuntu ISO.  Try a different CD-R first   If that doesn't work, try a reburn using a different burner.  If that doesn't work, try using a Linux Live CD.  I've solved more than one flaky install by scrapping the install CD and starting over.
