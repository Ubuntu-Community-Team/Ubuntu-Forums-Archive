### Post by Agent.Logic_ on 2009-04-10
I had a similar problem, but I don't know if it might be related. More specifically, what video adapter/chipset does the old computer have? My laptop is fairly new, albeit built with a sh**ty video adapter from Taiwan (surprise, surprise!) to make the laptop cost cheaper than if it had an Intel one. Anyway, I realised my problem was a display driver issue. You might want to hit F4 on the LiveCD menu and choose "safe graphics" option so that Ubuntu does not care about the hardware as far as display adapters go. That will fix the blank screen issue.
