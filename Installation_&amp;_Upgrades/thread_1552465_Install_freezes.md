### Post by mörgæs on 2010-08-14
You could try the alternate (text-based) installer as a first step.

If that does not work: A minimal Ubuntu lets you install all packages by hand. Start by installing the core system with this CD
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

and afterwards you add packages by hand, like for example 
[http://ubuntuforums.org/showpost.php?p=9287016&postcount=223](http://ubuntuforums.org/showpost.php?p=9287016&postcount=223)

When the system is running, you can use Synaptic like always.
