### Post by mikewhatever on 2007-02-06
The problem may be, that the minimum requirements of Ubuntu LiveCD installer are 192 MB of RAM. You should be able to install Xubuntu, but get the Alternate install CD from the [Download Page]("http://www.xubuntu.org/get").
If you want to try another light distro, check [This Page.]("http://ubuntuforums.org/showthread.php?t=354201")
There is a good guide on [how to dual boot Ubuntu with Windows using Alternate installer](http://users.bigpond.net.au/hermanzone/). It can help you with Xubuntu setup. Alternate installer is the text mode installer with no fancy graphics.
You do not have to dump Windows, unless you want to. In case you do, formate or delete the partition it is on. Note, that you'll loose ALL data on that partition. If you choose to keep Windows, try getting help on [Windows Security Forum](http://www.theeldergeek.com/forum/index.php?showforum=22)
