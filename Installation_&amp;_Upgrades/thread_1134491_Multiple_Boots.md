### Post by yeats on 2009-04-23
You *can* do that, and I did that myself for a while, but I decided that it is just as simple to have both GNOME and KDE on the same Ubuntu version. From Ubuntu:

```
sudo apt-get install kubuntu-desktop
```

If you're a purist like me, you'll have to edit the menus in GNOME/KDE to remove the gtk/qt entries (respectively), but it works better than tri-booting for that purpose alone :-).
