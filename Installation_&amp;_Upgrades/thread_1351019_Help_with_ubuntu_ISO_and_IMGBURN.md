### Post by phillw on 2009-12-10
> **Krozard said:**
> I tried making an install disc using IMG burn. But after it finished it said something about the track file being longer on the file than on the disc, leading it it not letting me install from...this happened twice.  Help please.

Hi ensure you get your image from here --> [http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download)
(I'm assuming you want the desktop (standard) version of 9.10 for a standard 32 Bit computer - which is the most common one)

Ensure you Burn no faster than at 4X speed. That thread also covers various ways of burning the cd - those images WILL fit on a cd.

Once it is burnt, boot off it & select the 'test cd' option (I think it's the last option) It will take a while.... Assuming that checks as okay - you're good to go !!

(If you don't have the check cd option, then ensure you do check it --> [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM))
