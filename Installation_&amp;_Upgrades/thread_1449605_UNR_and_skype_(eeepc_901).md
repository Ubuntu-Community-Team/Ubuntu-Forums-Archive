### Post by VeeDubb on 2010-04-08
I haven't used the UNR, but my guess is that it's actually a matter of which audio device is selected.  

Open sound preferences, click on the 3rd tab, which is "Input" and select the input for your webcam's mic.  By default, it's probably set to some internal input that's part of your soundcard, but not actually accessible on a netbook.

On regular Ubuntu, you access the sound preferences by left clicking the volume applet in the task tray and click "Sound Preferences."  I have no idea if it's that simple with UNR or not.

### Post by VeeDubb on 2010-04-10
Upgrading skype is pretty straight forward.

1. Go to their website
2. Download the .deb file of their latest version
3. Double click the .deb file


That's pretty much it.  If it's newer than the one that was installed from the Ubuntu repos, it will tell you it's newer, but not from the repos and ask if you really want to replace what you've got.  I'm running the latest release direct from skype on 10.04 beta, and it works just as well as it did on 9.10.  Not a lot a changes, but a few extra features like desktop sharing can come in handy
