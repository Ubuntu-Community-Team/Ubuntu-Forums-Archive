### Post by eltonw on 2010-05-06
I have been running Ricardo Salvetri's patched kernel which  enabled my LUCID installation to reliably connect to the wifi network. Today, Update Manager popped up with a list of updates. I downloaded and applied all that were available.


:evil:**[SIZE="3"]BIG MISTAKE!![/SIZE]** I should have *de*-selected the 2.6.22.22/23 kernel. Thankfully, I managed to re-install the salvetri patched kernel. [linux-image-**2.6.32**-21-generic_2.6.32-21.32rsalveti1_i386.deb] which got me back online.

I am currently connected using the **2.6.33** kernel. As such, I would advise other Asus EEE owners / notebook owners or others with connection problems to _*[COLOR="DarkRed"]steer clear of the 2.6.22.** series.[/COLOR]*_[-X
