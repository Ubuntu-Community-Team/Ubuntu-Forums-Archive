### Post by DarkN00b on 2006-10-13
You could just run the LiveCD. It loads everything into memory and doesn't touch your harddrive. Make sure you have plenty of memory though, 256 meg minimum. I installed it on my laptop w 256 megs and it just crawled, took almost an hour to install. Once installed though, it runs like a champ.

Pop in the LiveCD, and reboot. If your computer can boot from CD then you're good (afaik-all modern compters can). If it doesn't you may need to set your BIOS to look for bootable CDs before booting from HD.
