### Post by oldfred on 2018-08-30
I believe in a new clean install and then copy /home, list of installed apps &  perhaps some settings from /etc, but different system may need different settings anyway.
It becomes a good test that your backup procedure includes everything you need to fully restore system.

Others may suggest full image backups. If system is similar, it should just work. But if different video or wi-fi then you may have drivers issues.
