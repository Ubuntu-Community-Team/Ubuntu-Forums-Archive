### Post by SoftwareExplorer on 2009-09-06
Go to System->Administration->Software Sources. Go to the updates tab. Under show distribution releases, set it to normal releases.

---

### Post by SoftwareExplorer on 2009-09-06
The reason you have to do this with 8.04 is that it is a Long Term Support (LTS) release. That means that it is supposedly more stable. They didn't want to suggest upgrading to an unknowing user who wanted only LTS versions. Don't worry, 9.04 is quite nice. (Unless you have an older intel graphics chip). To get to the latest version, 9.04, you will have to upgrade to 8.10 and then 9.04.
