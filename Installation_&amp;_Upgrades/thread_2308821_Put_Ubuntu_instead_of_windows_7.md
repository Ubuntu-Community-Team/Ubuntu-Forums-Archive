### Post by mojo-jar on 2016-01-05
Hi. When you boot the Ubuntu installation disk, it will give you several options for install.
One if I remember correctly is to use the entire hard drive and another is to install Ubuntu and Windows side by side.
If you choose the entire hard drive, Ubuntu will use the entire drive and Windows will be lost. 
Be very careful when you choose that option since Windows will not be recoverable.

### Post by Bucky Ball on 2016-01-06
Welcome. Let your fingers do the walking. Do a search for 'ati radeon 4200 ubuntu'. You will find that card [appears to be fully supported]("https://help.ubuntu.com/community/RadeonDriver#Fully_supported").

Suggest you create bootable install media, boot from it, 'Try Ubuntu' at the options screen, and take it for a test drive, see how it goes. This is a 'live' session which won't change your hard drive in any way.
