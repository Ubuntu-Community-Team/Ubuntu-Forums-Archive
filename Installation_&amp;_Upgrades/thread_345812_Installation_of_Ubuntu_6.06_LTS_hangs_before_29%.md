### Post by wpshooter on 2007-01-24
First I would suggest that you download and burn your own ISO image of Dapper 6.06.1 from the Ubuntu website (DESKTOP/live) version.

Make sure you burn it at 4X or less.

Then make sure you check the integrity of the CD by running the verification function found on the initial Ubuntu boot screen menu.

Run the memtest from the same menu.

If you still have problem then try running the CD using the video safe mode parameter or the F6 option.

If that still does not work the consider downloading and burning the ALTERNATE (text based) installation CD.

Good luck.
