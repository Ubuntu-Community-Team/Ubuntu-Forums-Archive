### Post by TheFu on 2012-02-13
* Download the full OS ISO file, not wubi.  Wubi is when you want to run both Ubuntu AND Windows on the same file system.
* Burn it to a CDROM or DVD.
* Put that into your PC, boot it, and test that the most important things work.
* If everything works ... mostly, then install it to the HDD. You'll need to create 2 partitions, maybe more.  

If you get lost, come back and ask specific questions OR find an installation guide online - there are hundreds. Youtube [https://www.youtube.com/results?search_query=ubuntu+11.10++installation&oq=ubuntu+11.10++installation](https://www.youtube.com/results?search_query=ubuntu+11.10++installation&oq=ubuntu+11.10++installation) has a few great Ubuntu installation videos too.  Even old videos are helpful since they cover the important partitioning topics.

Enjoy.
