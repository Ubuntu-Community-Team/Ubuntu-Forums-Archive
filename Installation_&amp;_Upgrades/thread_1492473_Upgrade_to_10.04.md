### Post by Kim Foltz on 2010-05-24
I have separate partitions for /boot and /home. What happens to them if I upgrade from Karmic to 10.04 through the Update Manager?

---

### Post by darkod on 2010-05-24
There should be no problem. The only thing you need to be careful about is if you get a window asking where to install grub2, select the correct disk where you want it installed on the MBR, and make sure you don't select any partitions.
