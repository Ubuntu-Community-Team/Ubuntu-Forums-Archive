### Post by tal007 on 2009-08-31
Absolutely, if you have insufficient RAM, you will run into that problem. Yesterday one individual was having the same problem as you - can not bring up the installation menu due to insufficient RAM. He was trying it on a Pentium iii with 256 MB. I don't know what kind of system you have. Someone said that he installed it by creating a SWAP space prior to running the installer. He used 1 GB for SWAP.

Before you try, make sure to read the requirements from the Ubuntu site. Things like :

CPU type : 32 bit , 64 bit
Ubuntu : Desktop / Laptop
Ubuntu : 32 bit version / 64 bit version
