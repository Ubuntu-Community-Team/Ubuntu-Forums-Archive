### Post by Pumalite on 2007-12-29
With 512 of RAM and Nvidia card you'll have no problems with a Live CD. Let Grub install to MBR and you'll have dual boot. And if Vista doen't show up, your menu.lst can be edited easily. If you are installing in the same drive, make sure you allocate space for Ubuntu with the Vista partitioner first, then install Ubuntu.
[http://www.howtoforge.com/dual_boot_windows_xp_vista_ubuntu_feisty_p2?](http://www.howtoforge.com/dual_boot_windows_xp_vista_ubuntu_feisty_p2?)
