### Post by lykeion on 2010-12-05
If you look around the web for the Jahshaka project you will notice that it's now called CineFX. But I think you are heading towards a dead alley if you try to compile and install it; the sources haven't been updated since 2007.

This is what I'd do:
1) Take a look at the Linux alternatives to Adobe Premiere [here]("http://www.linuxalt.com/"). 
2) Look at the home pages and see if some alternative have the requirements you're after.
3) Search Ubuntu Software Center to see if it's in the repositories. If so install it and try it out.
