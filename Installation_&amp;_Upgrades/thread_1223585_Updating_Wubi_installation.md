### Post by Mark Phelps on 2009-07-26
yyylv is correct because lots of folks take the default size parms for installing in Wubi, which generally allocates just enough space to do the install.  Use the Partition Editor to check to see how much free space you have in Ubuntu.  If there is no entry for the Partition Editor, you can try installing GParted through synaptic.
