### Post by tehwa on 2005-11-14
Gnome-look has lots of themes, not limited to metacity:

[http://gnome-look.org/](http://gnome-look.org/)

After you have downloaded the theme .tar file to your desktop, you can install any of them (metacity, windows borders, icons...etc) through the system menu:

System > Preferences > Theme

And clicking on "Install theme". This will add new theme files in the *.tar format, you will not need to untar them first.

A tutorial exists for aspiring metacity themologistorians:

[http://developer.gnome.org/doc/tutorials/metacity/metacity-themes.html](http://developer.gnome.org/doc/tutorials/metacity/metacity-themes.html)
