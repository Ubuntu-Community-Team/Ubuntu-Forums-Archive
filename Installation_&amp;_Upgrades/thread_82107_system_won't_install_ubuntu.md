### Post by kairu0 on 2005-10-25
Good news: your CD is probably OK.

What you need to do is BOOT FROM THE CD. Put it in your drive, restart, and it will probably start to load the install.

If it goes straight into Windows like normal then you'll need to enter your BIOS and tell it to boot from the CD before hard drive.
