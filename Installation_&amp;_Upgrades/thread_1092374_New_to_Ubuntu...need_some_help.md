### Post by avtolle on 2009-03-10
FWIW, it sounds like a bad burn (and yes, check the md5sum of the iso before burning). Be sure to burn the iso as an image, that is, do not burn it as a data file, nor just copy it. If you have another computer handy, please insert the cd you have burned and check it to see whether there are multiple files present thereon, or whether it shows just one large file. If the latter, you've not burned it as an image.
