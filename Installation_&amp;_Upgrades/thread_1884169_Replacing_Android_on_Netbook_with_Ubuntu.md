### Post by Paddy Landau on 2011-11-27
I don't think Ubuntu would fit; the specs are probably too small even for [Lubuntu]("https://wiki.ubuntu.com/Lubuntu").

AFAIK, Ubuntu can only be booted from USB or CD. You would have to find out how to get into the BIOS to allow booting from USB. Try pressing and holding F12 when you boot the machine from a cold start; if you're lucky, it will display a menu allowing you to choose where to boot from.
