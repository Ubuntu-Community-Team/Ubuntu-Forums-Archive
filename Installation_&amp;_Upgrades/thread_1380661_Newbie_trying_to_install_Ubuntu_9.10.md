### Post by raymondh on 2010-01-13
you can make a request from canonical or the ubuntu store or ... you download the .iso file and burn it (image ... not copy it).  That is known as a LiveCD.

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

When you start-up your computer, there is an option to either QUICK-BIOS or ENTER BIOS.  Preferably quick bios, you make your CD-drive FIRST TO BOOT (ahead of the HD).  YOu then load your liveCD and enter.

The liveCD allows you to TRY UBUNTU WITHOUT ANY CHANGES (known as a live session). Use that option to test out the ubuntu version vis-a-vis your system (wifi, keyboard, Fkeys, etc).  When happy, there is a icon to install.  If not satisfied, just exit and close the liveCD .... no harm done.

Best to read up on install methods.

[https://help.ubuntu.com/9.10/index.html](https://help.ubuntu.com/9.10/index.html)
[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)
