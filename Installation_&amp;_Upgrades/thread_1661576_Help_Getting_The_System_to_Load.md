### Post by tommcd on 2011-01-12
If you have installed XP on the problematic computer, just open the Device Manager and look under at the graphics driver. This should tell you what graphics card is in the computer and what driver it is using.

One way to avoid graphics problems is to install Ubuntu using the alternate install CD. This is a text based, install CD that does not need a graphical desktop to istall Ubuntu. You can get the alternate install CD here: [http://www.ubuntu.com/desktop/get-ubuntu/alternative-download#alternate](http://www.ubuntu.com/desktop/get-ubuntu/alternative-download#alternate)
