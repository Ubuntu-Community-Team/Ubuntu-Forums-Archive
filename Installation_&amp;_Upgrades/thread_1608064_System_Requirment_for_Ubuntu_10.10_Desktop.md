### Post by ajgreeny on 2010-10-28
You will probably get ubuntu running on that machine, but it will be rather slow with only a 600 Mhz PIII and 512 MB ram.

I suggest you have a look at xubuntu, with the xfce desktop or even lubuntu with lxde, both of which will run fairly well on that spec compared to the gnome desktop of ubuntu.

If you want to look at other distros which may suit you better, go to [http://distrowatch.com/](http://distrowatch.com/) and search for those with lower requirements, eg, those mentioned above, vector, puppy.  I am sure there is something you can find, and if you really want the ubuntu family, and to keep it simple but learn all about linux, try the minimal install CD of ubuntu, and then add what you want or need one by one.
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

### Post by nearadyn on 2011-01-16
Anyone wanting to use Ubuntu on older machines which are really too old for the latest version should look at WattOS. Just google it. WattOS is a distro which is based on Ubuntu, but stripped down and made super efficient so that it works very well on old machines. I have it running on a 4GB hard drive with 256MB of ram with no problems. It works well on faster computers as well. Put it on a quad core with lots of ram and you won't believe how fast your boot up and shut down times are!
