### Post by wpshooter on 2006-11-17
> **Rocket Boy said:**
> What's the best way to get a computer like this to boot from CD if it doesn't already?

Go into BIOS and make the first boot device the CD-ROM drive instead of the hard drive.

I would try regular Ubuntu/gnome on this computer first.  I think she will like the interface and the combination of applications that are installed by default.

I would try installing from the DESKTOP CD first.  If problems with the DESKTOP CD then try the ALTERNATE CD.

Make sure you check the integrity of your Ubuntu CD by running the verification test found on the initial Ubuntu boot screen menu.

Good luck.
