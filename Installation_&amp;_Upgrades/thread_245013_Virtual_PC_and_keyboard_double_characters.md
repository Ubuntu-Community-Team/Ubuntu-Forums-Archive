### Post by fdlplyr on 2006-08-28
I found the answer myself.  The issue was that Virtual PC had some compatibility issue with my dual core laptop (Dell Latitude D820) and Advanced Power Management.  There is a Microsoft hotfix for this problem:

[http://support.microsoft.com/?kbid=899525](http://support.microsoft.com/?kbid=899525)

This solved the issue of the Virtual PC pausing over and over again, I also noticed it fixed the smoothness of the mouse movement, I did not realize that before the mouse was a little jittery.

I have the patch if you need it!
