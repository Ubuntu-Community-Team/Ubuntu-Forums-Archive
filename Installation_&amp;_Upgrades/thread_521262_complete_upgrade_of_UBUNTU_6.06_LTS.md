### Post by Jimmyfj on 2007-08-09
In my opinion the best way to upgrade from older versions of Ubuntu is to get the iso file you want, say 7.04, being the latest, burn a Live Cd and install your system from scratch. 

If that is not an option you could use apt-get update/upgrade/dist-upgrade. I'll consider this the insecure way for an upgrade. Should you choose this way of upgrade, be sure to have plenty of time available for it.
