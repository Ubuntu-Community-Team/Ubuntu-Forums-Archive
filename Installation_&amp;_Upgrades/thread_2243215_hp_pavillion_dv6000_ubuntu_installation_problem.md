### Post by mooreted on 2014-09-07
Simply copying the ISO to a hard drive does not create bootable install media. You need to create a bootable thumbdrive or burn a bootable CD.

Here are a couple of methods to try:

[http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/desktop/create-a-usb-stick-on-windows)

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
