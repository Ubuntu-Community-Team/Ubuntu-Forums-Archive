Yes. Even with this hdd dead you can use ubuntu from the cd, so called LiveCD, just make sure you adjust BIOS to boot cd first (in fact doesn't really matter with dead hdd :) ).
Boot with the cd and select Try Ubuntu. That will load ubuntu from the cd and you can even surf, check if audio and video work fine, look around, etc.
After the new hdd is installed, boot with the cd again, select Install Ubuntu and use the Use Whole Disk option if you want to have only ubuntu on the disk. Ubuntu will take care of everything. If you want dual boot with windows it's better to install windows first.
