### Post by deadflowr on 2020-04-19
Yes.
Once any version of any release is installed, as long as you keep updating it, it'll become that release's stable version.
Doesn't matter if you installed after it is officially released or if you installed from the earliest daily build image, it's all the same release.

Keep updating it.
