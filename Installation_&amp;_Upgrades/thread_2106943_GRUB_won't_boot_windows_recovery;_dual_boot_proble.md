### Post by Mark Phelps on 2013-01-20
You could try pressing F12 or F8 -- on some PCs, this brings up a boot selection menu where you can select the CD/DVD drive, USB, and other things.  That's a one-time selection and can work when the default boot setting is not what you want it to be.
