### Post by kaamos on 2006-01-12
installing the package "g++" should get you going. You can install it with synaptic ([https://wiki.ubuntu.com/SynapticHowto](https://wiki.ubuntu.com/SynapticHowto)) or with the command
```
sudo apt-get install g++
```
