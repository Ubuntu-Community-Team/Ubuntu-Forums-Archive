### Post by localzuk on 2007-02-10
Updates to dapper will simply be security updates. Updates to edgy will also be security updates - but this release had much less time to be polished than dapper so was generally more unstable when released. Updates to feisty can be feature changes and fundamental changes to the release - as it hasn't been released yet.

Personally, I have stuck to Dapper - as I now only use Ubuntu for server installs (4 at the moment) and long term support is a major advantage with this.
