That is a good question.

No,

There are two sides to Debian and three versions.

There is the Stable version which is slow to change and very relable.  This is important if you want to be sure that your computer will not crash while handling your important tasks.  Woody was released in 2002 and still runs the packages that were brand new at the time (mozilla 1.0)


The other side of Debian is the unstable version.  It is pretty much as cutting-edge as it comes.  It is properly named unstable.

The third version of Debian in the Testing version which is made up of packages from Unstable which will one day become stable.  Currently, Debian Sarge is Testing, but is slated to become Stable in a few days.  The Release crittical bugs for Testing is usually in the hundreds, but before a release, the packages are frozen so that the bugs can be worked out.

Right now, I think there are something like 20 RC bugs to be fixed before release.

Ubuntu, like testing is based on Debian Unstable.  So the fact the Sarge will release will not really affect Ubuntu.
