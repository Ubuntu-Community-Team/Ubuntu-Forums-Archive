### Post by mipesom on 2011-02-03
Adobe air updates itself. Cause of the strange naming(?), Ubuntu always wants to "upgrade" from the actual version to its old version "1:2.0.4.13090-1maverick1". How can I permanently exclude this downgrade before I get crazy? :shock:

UPDATE: Found a solution. In Synaptic package manager I selected the adobe air package and have chosen "Force version.. " from the package menu, selected the actual version (2.5.x). To make that change permanent I have chosen "Lock version" from the package menu again then. Downgrade terror gone! :)
