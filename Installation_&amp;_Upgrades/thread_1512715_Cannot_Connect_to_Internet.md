### Post by Icarus315 on 2010-06-18
If it is a wireless device:

Plug a network cable into your wired connection on your laptop.  This should get you an Internet connection.  Run Update Manager.  Update Manager needs to be run at least once for Hardware Drivers.  Go to: System > Administration > Hardware Drivers.  Your wireless network card driver may be listed there.  Activate it and reboot.  Attempt to connect to the Internet using the wireless device to test.

Edit: When you run Update Manager, check for updates to build the database in your local computer.
