### Post by Johnsie on 2009-03-03
Turns out the problem was due to Ubuntu trying to use Compiz. I stuck the hard disk back into the other computer, disabled compiz and transfered it back. Probably not the best way of disabling compiz but it worked and now my machine is running fine.
