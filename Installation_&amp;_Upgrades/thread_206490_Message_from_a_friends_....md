### Post by anando on 2006-06-30
Hi all,

This is a message from my friends in Australia about his experience in installing Dapper 6.06 on his computer (IBM NetVista). He requested me to post this message  on the Ubuntu forum ... so here follows :

```
Managed to install this from a downloaded version. Really enjoying this.
Actually this mail is from Evolution.

Actually I got some problems while installing some were trivial and
human error. But the Major one was that after I finished the install and
restarted the machine X windows wouldn't load. So all it would give was
the command prompt. I think this might frighten new users to cry foul
and abuse Ubuntu. Actually is pretty simple and I think this problem
will affect all Intel Machine (i used a IBM NetVista) which come with
pre-configured Windows. [B][COLOR="Red"]The solution is to Change the Video Shared
Memory in BIOS setup from 1Meg to 8meg. (windows comes with a default
1meg).[/COLOR][/B] If you could let Ubuntu know about this and ask them to post it
on their Installation guide/documentation it would help a lot of people.

But honestly these guys have a wonderful job. I used the non graphical
install (i386-alternate.iso) and one site that can help is
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/). A beautifully described site
and will really help newbies.
```


Have fun guys,
Anando.
