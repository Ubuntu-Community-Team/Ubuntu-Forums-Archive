### Post by rillip on 2007-05-25
RAM is only used for current processes.  If you boot xubuntu, it can't use ram.

The determining factors as to whether you keep Windows or not are:

1.) Have you tried all your hardware from the LiveCD? For example, some people have a hard time with printers, webcams, etc.  Try out all your hardware first and make sure it either natively works, or that you are willing to invest sufficient time to get it working.

2.) Are you running many Windows only programs?  Most, like MS Office, Windows Media Player, Internet Explorer, etc. have Linux equivalents or better.  If you're running a lot of Windows only software, like games (probably not too new of ones with those system specs) then you should probably stay on Windows.
