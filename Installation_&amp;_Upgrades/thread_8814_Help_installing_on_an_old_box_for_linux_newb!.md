### Post by az on 2004-12-21
Do you have a correctly burned cd?

Try reinstalling.

Or, if you are lazy, try doing 

sudo apt-get -f install

That will attempt to fix things it they are fixable.  If you have a bad cd, there is not much you can do but burn yourself another cd.

Also, don't panic.  Linux is one of the world's most advanced operating systems and the graphical environment (the one where you use a mouse) is optional.  It is not like in windows where a text screen represents another crash.  You just have not gotten a working X server running yet.
