### Post by mörgæs on 2012-08-01
Now I see. You are talking of *the* application CW and not *a* CW application.

After installing CW: If you enter ```
man cw
``` at the prompt you will see the help pages. Here you find the example 

```
echo "UNIX CW SOUNDER" | cw -w 25 -t 700
```

Try to change the parameters like

```
echo "free as in speech" | cw -w 15 -t 500
```



Cheers and 73 from a former 2M amateur.
