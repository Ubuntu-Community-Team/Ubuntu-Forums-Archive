### Post by Leif on 2005-04-30
I don't know of an ubuntu floppy, but there is a debian one : [http://www.debian.org/distrib/floppyinst](http://www.debian.org/distrib/floppyinst). Once started, just change the /etc/sources.list file to the default one (look at ubuntuforums.org to see how the original should look), then go from there.
