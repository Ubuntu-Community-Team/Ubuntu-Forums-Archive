### Post by snowpine on 2009-04-13
Hello Pierre,
You need to enter your computer's bios menu and change the boot options.

When you first turn on the computer, it should tell you which key to press for the bios menu. It is usually something like F2, F12, etc. Go to the page with the boot options, and change the boot order so that the CD drive is the first device on the list. Save your changes and reboot. Sorry I can't give more specific directions, but it varies from manufacturer to manufacturer.

Also if you have access to another computer, try booting that with the Ubuntu CD. This will tell you whether the problem is with the CD or the computer. (Or both!:))
