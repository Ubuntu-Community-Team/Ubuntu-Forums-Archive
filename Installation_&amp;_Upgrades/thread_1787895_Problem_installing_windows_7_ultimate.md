### Post by Mark Phelps on 2011-06-22
You should be able to stick in the DVD, boot from it, and install.  But since you cant' do that, you need to boot from an Ubuntu LiveCD and use Gnome Partition Editor (or Disk Utility) to REMOVE all the existing Linux partitions on the drive.

If you still can't boot from the DVD, either it's damaged or your CD/DVD drive is bad -- neither of which we can fix here.

Also, if you want detailed help with Windows 7, you should go to sevenforums.com -- a Windows 7 forum.
