Gobuntu is only free software as defined by the FSF.

Kubuntu uses KDE and KDE applications

Xubuntu uses Xfce and applications that are associated with it.

Ubuntu uses GNOME and GNOME apps

You probably don't want Gobuntu, but it is like Ubuntu mostly.

If you install Any one of the others, you can get the desktop environments of the others on the same installation (you chose which one to use when you log in)
