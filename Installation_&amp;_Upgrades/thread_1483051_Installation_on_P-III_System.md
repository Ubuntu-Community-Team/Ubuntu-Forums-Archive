### Post by cascade9 on 2010-05-14
Just for your information- the 'chipset' is actually the chips on the motehrboard,a nd the Pentium and Celeron CPus both use the same chipsets ;) 

A P3 with 512MB of RAM is about the minimum requirements for ubuntu. 256MB is quite a bit under that. To run with 256MB you will need to do a minimal install (either for ubuntu or xubuntu) or try lubuntu. 

Minimal installs, or lubuntu, will make the P3 run faster as well. ;)
