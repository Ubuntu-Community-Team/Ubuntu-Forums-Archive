### Post by Joe Ker1086 on 2010-07-19
I think the problem you are having is with the boot order. I would guess at some point it stopped looking to the cd drive first. Check the boot order when the computer is booting up. depending on the computer there might be a key you can press to boot off a selected device or you might have to get into the bios (usually f2 or f8) and move the cdrom above the hard drive.
