### Post by Grenage on 2011-09-27
To be honest, I'm struggling to understand what you mean.

If you want to take a copy of your existing install and make it bootable on CD/USB, then you could use a tool such as Remastersys.

If you want to make a simple live Ubuntu USB device, you can either use the existing CD and make an ISO using Brasero, then you can use the guide in post two.

If you can't make an ISO from the CD, you can simply format the USB stick, make it bootable, and copy the files from the CD to the USB device.
