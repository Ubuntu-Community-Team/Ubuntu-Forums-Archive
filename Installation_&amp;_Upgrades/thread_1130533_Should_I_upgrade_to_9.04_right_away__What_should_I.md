### Post by acimmarusti on 2009-04-19
If intrepid is working well for you, I suggest you stick with it for a while. New releases often break things that worked before. I know for example that newer versions of pulseaudio highlight instabilities in my sound card linux driver and cause glitches in my audio playback...

If, however, there is something broken in intrepid that you never got working and you want it badly (especially new hardware, since the kernel is newer for jaunty), you might want to try upgrading.

### Post by ptn107 on 2009-04-20
You should read the 9.04 overview[1] and the full list of release notes[2] to see if any of the 'known issues' will affect you.  In the end it's really your choice though.  If you find that none of the known issues for 9.04 apply to you, then why not upgrade.  Upgrading the day of the release or waiting will not matter.  Most of the release critical bugs and minor known issues would have been solved by the days preceding the release (most [but not all] post-release updates are security related or minor bug fixes anyway).  Again, read the release notes.  Most people prefer a clean install over an upgrade (myself included).  Either way you should always back-up your data before moving to a new release.

[1][http://www.ubuntu.com/getubuntu/releasenotes/904overview]("http://www.ubuntu.com/getubuntu/releasenotes/904overview")
[2][http://www.ubuntu.com/getubuntu/releasenotes/904]("http://www.ubuntu.com/getubuntu/releasenotes/904")
