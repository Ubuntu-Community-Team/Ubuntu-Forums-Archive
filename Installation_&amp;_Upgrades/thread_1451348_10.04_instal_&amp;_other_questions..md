Hi Colintivy

I think going the LTS route is very good unless you love the joys of seeing new features regularly and risking a lot on alphas/betas :)

As for the /home backup that is easy to do :) Just plug in your USB drive and copy all in /home to it. ie when you plug it in and it opens, goto PLaces => Home Folder, CTRL -A and copy all into the USB ramdrive folder.

I would stick with ext3 for now to be honest as it is proven and safe I have found (personal opinion :)

For the keyboard have you tried the setting in:
System => preferences => keyboard (both general and layout options, many configuration changes are hidden in there ;)


Please let me know if this helps and keep on Ubuntuing :)
