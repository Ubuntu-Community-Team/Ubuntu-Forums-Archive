### Post by lechien73 on 2010-06-25
You can install Grip from a repository by typing:

```
sudo add-apt-repository ppa:futurepilot/ppa
sudo apt-get update
sudo apt-get install grip
```

Or you can just download the individual package for Karmic here:

[https://launchpad.net/~futurepilot/+archive/ppa/+sourcepub/976254/+listing-archive-extra](https://launchpad.net/~futurepilot/+archive/ppa/+sourcepub/976254/+listing-archive-extra)
