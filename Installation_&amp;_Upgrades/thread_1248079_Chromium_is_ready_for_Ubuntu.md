### Post by Solrac924 on 2009-08-23
For those looking for a 64-bit browser that's a whole lot quicker than Firefox or Opera, here's Chromium. It's not quite Google Chrome, but it's practically it. It's an in-progress build of Chromium on Linux. 

add in System > Administration > Software Sources:
deb [http://ppa.launchpad.net/chromium-daily/ppa/ubuntu](http://ppa.launchpad.net/chromium-daily/ppa/ubuntu) hardy main
deb-src [http://ppa.launchpad.net/chromium-daily/ppa/ubuntu](http://ppa.launchpad.net/chromium-daily/ppa/ubuntu) hardy main

enter in terminal:
sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 4E5E17B5
sudo apt-get update

in Synaptic Package Manager, find Chromium-Browser & "Mark for Installation"

enjoy! :)
