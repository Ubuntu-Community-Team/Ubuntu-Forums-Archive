### Post by LewisTM on 2012-05-22
This has been investigated in this post
[http://ubuntuforums.org/showthread.php?t=1895108](http://ubuntuforums.org/showthread.php?t=1895108)
I'm not sure if it still applies in 12.04.
In brief, there is not much difference. The default XFCE theme is more basic but as soon as you change it, it will apply to the Xubuntu session theme and vice versa. Panels are the same.
