### Post by ajgreeny on 2015-01-07
How are you installing wordpress?

It is in the repositories so should need nothing more than command ```
sudo apt-get install wordpress
``` or to open the software-center and search in that.

If you have downloaded a version of wordpress from elsewhere you are best advised to forget that and use ubuntu's own repos for the installation.  This is also true of just about everything else you might need to install, not just wordpress.
