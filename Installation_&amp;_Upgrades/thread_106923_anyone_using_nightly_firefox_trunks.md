### Post by hhh on 2005-12-22
I have been testing Branch since March and just switched to trunk now that 1.5 is out and the devs are taking a break.

The short answer is no, it's not safe, since extensions are not being updated to  be compatible. It's almost definitely an extension that's causing you memory leaks in 1.5, BTW, probably Flashblock. If you grab a trunk build right now it is very stable though. If you were happy with 1.0.7 you can go back to that, but you'll have to start with a clean profile and reinstall your extensions. See the mozillaZine "Build" forums...
