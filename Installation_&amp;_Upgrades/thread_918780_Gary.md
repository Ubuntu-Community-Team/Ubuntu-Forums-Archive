### Post by wpshooter on 2008-09-13
You need to download the image from the Ubuntu website, then make sure that you **BURN** that image to a CD and not just **COPY** it as **DATA** to a CD.

Make sure that you burn it to the CD at as slow a rate as possible (preferrably at 4X or slower).

Then once you get the image BURNED to the CD, boot to that CD and make sure that you check the integrity of the Ubuntu CD by running the verification function that is found on the initial Ubuntu boot screen menu.

P. S. - I would suggest that you download the Gutsy version instead of any of the other versions of Ubuntu.  I think you are likely to have better luck with it than any of the other versions.

Good luck.
