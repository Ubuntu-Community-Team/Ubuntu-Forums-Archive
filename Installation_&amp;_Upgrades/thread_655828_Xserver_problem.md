### Post by TerabyteUK on 2008-01-02
ok problem solved

finally for anyone who wants to know...

when running
sudo dpkg-reconfigure -phigh xserver-xorg
you'll probably be picking the wrong band from the driver list.

solution, at the prompt
write 

```
sudo apt-get install 915resolution xserver-xorg-video-intel
```
This installs the 'intel' driver

Now run 
```
sudo dpkg-reconfigure -phigh xserver-xorg
```again

this time you should pick intel from the list, press enter and the screen will turn black

Wait a few seconds (maybe not but I did) At this point press CTRL+ALT+DEL, wait a few moments for the machine to restart....

then hope that the xserver starts up this time, it did for me.

Thankyou all, I am now a happy bunny
