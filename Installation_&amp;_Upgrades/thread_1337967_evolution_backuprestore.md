### Post by kenklay on 2009-11-26
My son had the same problem and it seems it had to do with the fact that he stored the backup file in a directory called "evolution backup".  According to a thread in Suse forums, [http://forums.opensuse.org/applications/425760-evolution-11-2-backup-restore-problem.html](http://forums.opensuse.org/applications/425760-evolution-11-2-backup-restore-problem.html), Evolution can not handle a directory with a space in its name.  My son is up and running now. [http://ubuntuforums.org/images/smilies/icon_biggrin.gif](http://ubuntuforums.org/images/smilies/icon_biggrin.gif)
