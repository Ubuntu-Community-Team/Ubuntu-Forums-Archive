### Post by darkod on 2009-11-27
I'm not sure about the CDs they ship, but on the website the Desktop 32bit version is offered by default. However, you can click on Alternative Download Options and it will offer the 64bit version.
Alternatively, all images can be found on [http://releases.ubuntu.com](http://releases.ubuntu.com) including previous releases. In the image filename there is either i386 or amd64 to inform you whether it's 32bit or 64bit.
