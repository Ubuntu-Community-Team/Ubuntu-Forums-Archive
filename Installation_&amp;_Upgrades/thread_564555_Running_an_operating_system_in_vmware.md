### Post by DerHesse on 2007-10-01
you need to install an OS in the vm first. You do need vmware workstation od Vmwar Server to do so. (VMWare Server is freeeware)

If you are running vmware player, you cannot install but need to run an existing vm.
You can find downloadable appliances at [http://vmware.com/appliances/]("http://vmware.com")
