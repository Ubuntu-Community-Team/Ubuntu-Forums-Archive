### Post by sikander3786 on 2010-12-06
First problem I see in your output is Windows SFS partitions. And that is not a minor problem.

SFS means you've got Windows Dynamic Partitions and Linux is not good at recognizing them. Not sure how you got Ubuntu installed besides Dynamic but you needed to convert your partitions to Basic before installing Linux.

Another problem is that there is no Swap partition (it is recommended). But that shouldn't cause any problems in booting Windows.

Third, sda1 is not being recognized, not sure what it contains (??).

Other than that, Windows boot files seem intact and it should boot normally.

The best workaround here would be to boot a Windows 7 install/repair disc and try to repair your startup. You'll need to do that 3 times at least from the disc and then you might be able to boot Windows successfully but Ubuntu would no longer be bootable.

As this is a fresh install of Ubuntu, I would recommend first to try to convert your Dynamic Partitions to Basic, using the Partitioning Tool of Windows itself. You need strong backups before doing that. Windows partitioning tool was intended to resize/convert Windows partitions without losing any data at all but who knows?

I am not sure if converting Dynamic to Basic will keep Ubuntu partition or will get rid of it. Whatever, as this is a fresh install, it won't hurt to do a fresh install again.

You can wait for some other member replies also if you want some consensus to be developed on what I said ;-)

### Post by oldfred on 2010-12-07
Windows offically says you cannot convert back. You have to backup and erase drive & recreate basic partitions. But some third party tools may work, but you best have good backups.

SFS converting:
[http://www.sevenforums.com/tutorials/26829-convert-dynamic-disk-basic-disk.html](http://www.sevenforums.com/tutorials/26829-convert-dynamic-disk-basic-disk.html)
You can use a third-party tool, such as Partition Wizard 4.2. to convert a convert a dynamic disk to a basic disk without having to delete or format them.
The Partition Wizard software for Windows is supposed to be able to convert dynamic disks to regular partitions without data loss, so it may be what you need to get around this problem; however, I've never used it and so I can't be sure it will work.
Dynamic volume is a Microsoft proprietary format developed together with Veritas (now acquired by Symantec)
