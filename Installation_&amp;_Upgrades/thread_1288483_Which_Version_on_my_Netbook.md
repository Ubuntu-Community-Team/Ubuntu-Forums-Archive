### Post by p2bc on 2009-10-11
Well it depends... want to save resources or not. A lighter weight distro like Xubuntu will use less battery and resources than say Ubuntu. Ubuntu which even use less resources than XP. And there are even lighter ones like you mention Netbook remix.

Running Wubi, is running Linux through Window, kinda like a virtual machine. You install it like a application.

[http://wubi-installer.org/](http://wubi-installer.org/)
[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

So you will be using some Windows resources as well as Ubuntu resources.

Netbooks are not the most powerful machine, sorry but it is true, they are build for portability and convenience, so power (ie resources) are limited. So to use Wubi you need to leave Windows installed. So in essence running a dual boot system. If this is what you want, and you have the room on the hard drive, then you would be better off doing an actual dual boot. This way when booting your Linux Distro you are only using enough resources to run Linux, and when you boot Windows, you will be using more resources, but only while you are running Windows.
