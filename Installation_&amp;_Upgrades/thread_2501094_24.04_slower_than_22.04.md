Thinkpad 430s is my preferred hardware, runs Lubuntu 22.04 well.

I've now upgraded 3-4 of them with Lubuntu 24.04, and they all have the same problem, which is that they run slower, and at higher temps, than 22.04, and also needs frequent rebooting. This persists with 24.04.1.

I see that this is well reported with servers using 24.04 vs 22.04, but have not seen laptop issues mentioned.

Not a big problem, 22.04 is LTS, good til 2027, but I wonder if eventually I'll have to find a new my beloved T430s's behind: they're tanks, and I've found the newer hardware not nearly as reliable.

### Post by TenPlus1 on 2024-09-25
Ubuntu 24.04 does use more memory which could cause your older system to use virtual memory more which slows down the system.  You could maybe edit the /etc/sysctl.conf file and add this line to the end and reboot to see if it helps:

vm.swappiness=20
