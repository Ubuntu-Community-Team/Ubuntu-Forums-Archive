### Post by Klaidas on 2006-06-01
Well, LiveCD + installer is like "express installation" - easy, but doesn't have too many options
The alternate CD is for more advanced users, or users who need more installation options - like install OEM mode, etc :)
Here's list of its advantages:
[LIST]
[*]creating pre-configured OEM systems;
[*]setting up automated deployments;
[*]upgrading from older installations without network access;
[*]LVM and/or RAID partitioning;
[*]installs on systems with less than about 192MB of RAM.
[/LIST]
