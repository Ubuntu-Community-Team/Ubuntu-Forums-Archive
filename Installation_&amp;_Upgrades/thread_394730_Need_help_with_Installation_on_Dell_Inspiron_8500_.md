### Post by saphil on 2007-03-27
I have to go take some more Dell training.. but it sounds like, maybe, this might be a bios issue.
Can you boot from ANY OS disk?. like Windows, for instance??
If your bios boot order is not set to "see" the CD, it might put out an error like this.
I have a dell desktop on which the bios cannot see any cd during boot, even if I hit [f12] and choose CD/DVD out of the menu.  I think in my case it may well be a hardware issue, like the bios has been hit by static.  If you cannot boot from any OS CD, you might have the same issue.  I am considering just putting in a different motherboard and chipset, but in a laptop, that is not as simple.
