### Post by Cheesemill on 2013-01-13
No, there isn't. Point releases are only made for LTS versions of Ubuntu (12.04.1, 12.04.2, etc).

You can however use the mini.iso to install. This is a small CD (around 30MB) that only contains the files needed to load the installer. All other packages are downloaded directly from the repositories meaning you start with a fully updated system.

[http://cdimage.ubuntu.com/netboot/](http://cdimage.ubuntu.com/netboot/)
