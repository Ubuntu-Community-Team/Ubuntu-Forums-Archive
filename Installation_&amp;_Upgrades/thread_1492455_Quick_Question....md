### Post by tgeer43 on 2010-05-24
Under normal circumstances you will need just one partition for the Ubuntu installation. As for the filesystem, you can use any one of a number of filesystems but generally it is ext3 (or ext2) for 9.04 and earlier. With 9.10 and above you also have the option of using the newer ext4.

Hope this helps,
