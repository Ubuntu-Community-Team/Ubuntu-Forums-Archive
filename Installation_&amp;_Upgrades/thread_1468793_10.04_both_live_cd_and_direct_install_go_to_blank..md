Thank you, I succeeded with the "Solution 2":

1) “Try Ubuntu without any changes” selected, then press F6
2) now press ESC
3) write i915.modeset=0 xforcevesa at command line end

This way I was able to have the live cd working and then install and have 10.04 working properly after reboot.
