### Post by chris_nava on 2008-11-03

### Post by slakkie on 2008-11-03
The last install was.. two months ago on a workstation at work and I got the ISO from a co-worker (we all run Ubuntu at work). That was the first re-install of Ubuntu on a box in 3 years. 

I hardly download ISO's, I upgrade my boxes via the command line. I normally wait before upgrading (measure of caution, get the initial bugs resolved). My latest upgrade was 8.04 to 8.10 which went bad and before that I had all my boxes running 7.04 and one on 6.04. The 6.04 got upgraded to 7.x and then to 8.04. The others followed the same week.

I think I will download the netinstall of 8.04 this week to restore the machine where the upgrade to 8.10 went bad (or I need a different solution).

I don't order CD's. I think it is waste of money, I don't use them and if I need a ISO its cheaper to download it, and burn it. And it is quicker, no need to wait.
