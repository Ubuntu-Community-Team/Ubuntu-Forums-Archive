### Post by oldtimer7777 on 2011-12-03
Have you tried just using a Live Flash Drive of Ubuntu 11.10 to boot from?

You need a Flash Thumb Drive / Flash Drive to copy the Ubuntu iso file onto.

Here are instructions on how to download and install Ubuntu on a USB Flash Drive:

[http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

[https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/](https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/)

See if you can get your computer to work with the USB Flash Drive of Live Ubuntu 11.10 first, and see if your hardware is completely supported.   Resize your partition with the installer to shrink Windows down to a smaller size so you can install Ubuntu on your HDD and do a side-by-side installation of Ubuntu. The installer does everything for you, but it will prompt you which way you want to go with it. Don't use Wubi because I have found it to be more problematic sometimes then  doing a normal regular installation from a USB Live Stick of Ubuntu 11.10. You will simply need to boot your computer from that USB Flash Stick of Ubuntu you created, and you will be ready to rock and roll.
