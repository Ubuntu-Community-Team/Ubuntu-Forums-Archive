### Post by empthollow on 2007-02-04
try changing the driver line in the device section from vesa to ati like this
```
Section "Device"
    Identifier    "ATI Technologies, Inc. ATI Default Card"
    Driver        "ati"
    BusID        "PCI:1:0:0"
EndSection
```
i had a similar problem when i first tried ubuntu, cant remember the exact error i got at that time though.
