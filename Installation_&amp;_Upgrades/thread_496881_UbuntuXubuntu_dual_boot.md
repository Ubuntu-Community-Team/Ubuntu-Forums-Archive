### Post by Odrodzona-Sarmacja on 2007-07-09
On Ubuntu you don't need to have couple partitions to host all three guis (xfce,gnome,kde). 
You only need to download and install their packages with synaptic package manager.
During login there is an option in which any of the installed gui-sessions can be chosen.
