### Post by NonPapa on 2007-05-18
Hi there,

Sorry, I'm only able to give you a partial reply, as I'm fairly new to Ubuntu.

**Removing Evolution** seems not possible if you don't want to risk that
some other programmes/processes stop working. I asked the same question
a couple of months ago, and also found a number of postings on this forum
which made me come to this assumption. Not nice, but not a disaster, I think.
You don't have to *use* Evolution.

**Removing pre-installed games**: is easily done by using the add/remove option
from the 'Applications' drop-down menu.

Hope this posting was helpful, even though it only partially answers your questions.
