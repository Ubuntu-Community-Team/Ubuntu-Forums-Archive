### Post by lloyd_b on 2010-03-15

> i have ubuntu 9.04 installed on my Desktop I want to add a new ram bar whine I do i have a CRC error msg before system starting up and the system didn't start how can I add the new ram bar without installing my system from scratch

Could you be more specific about that CRC error - what exactly is failing?

It's quite common to get an error after installing more memory (because the amount of memory no longer matches the amount stored in BIOS).  Usually, the system can correct this on it's own, but you may need to go into the BIOS setup (F1 on boot-up will get you into the BIOS on many machines, but consult your machine's manual to be certain), and then save the current settings to permanently clear this error.
