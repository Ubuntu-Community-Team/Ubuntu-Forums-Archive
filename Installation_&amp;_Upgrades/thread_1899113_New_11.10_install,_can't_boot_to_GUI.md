### Post by jsmtrd on 2011-12-22
A few ideas:


[LIST]
[*]Restart or reboot your computer a few times (making sure you remove the install disk before).
[*]Make sure you installed the desktop edition of Ubuntu not the server edition.
[*]Log in at the command prompt with the user name and password you setup during install. Type startx to see if that will start a windows session. and report back what happed.
[*]Make sure the disk is good. There should be an option to check disk when you boot off it.
[*]Reinstall Ubuntu. It may be something did not get installed correctly.
[/LIST]
