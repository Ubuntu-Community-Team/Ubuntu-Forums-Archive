### Post by cstudent on 2005-12-13
Since your Win98 OS is on hard drive 1, all you have to do to install Ubuntu on hard drive 2 is burn the Ubuntu .iso file to a CD (or order a free CD) and restart your machine with the CD in the drive. Your machine will have to be setup to boot from CD first before the HD. If it's not already you can see if you can change the settings in the bios when the machine starts. Once the Ubuntu install starts you should be able to walk through the install using the default options that come up. After install you'll see the grub menu when you boot up. Ubuntu will be your first choice and 98 will be at the bottom. Just use the arrow keys to move up or down and pick which OS to start.
