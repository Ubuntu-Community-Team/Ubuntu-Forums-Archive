That's pretty much correct. It was pretty highly publicized (and commented on here in the forums) that Gnome-shell was a plugin for the default Gnome 3 window manager and therefore *couldn't* work with Compiz. This is pretty much that same way that Unity is a plugin for Compiz, so you couldn't, for example, combine Unity with Fluxbox.
