### Post by mingus on 2005-06-26
A crc (Cyclic Redundancy Check) error usually indicates a hardware or media failure.

The first easiest thing to try is to re-burn the install CD at a slower speed, using disc-at-one and finalize.

Also been known to happen is the drive is not set up in the BIOS as LBA, and when a RAM stick is getting flaky (can be checked with memtest86).

You'll find plenty on info googling.  But first, just burn a new CD.
