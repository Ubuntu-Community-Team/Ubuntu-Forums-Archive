### Post by dearingj on 2010-02-10
I've got this MacBook Pro with a broken LCD. I'm thinking of setting it up as a server for my home network. I have an external display which I've hooked up to it, and I can boot into Mac OS X without any problems. I'm trying to set up a dual-boot with Ubuntu, but I can't figure out how to make any of Ubuntu's installers use the external monitor. I've tried booting from a desktop CD, a server CD, and the alternate installer CD. All CD images were downloaded with BitTorrent and burned at the slowest speed that Brasero would allow (on a different computer also running Ubuntu).

EDIT: Managed to get it installed. I got the computer to boot from an Ubuntu livecd. A small portion of my laptop's built-in screen was still working, and I used that to enable VNC. I was then able to connect using another computer and run Ubuntu's graphical installer.
