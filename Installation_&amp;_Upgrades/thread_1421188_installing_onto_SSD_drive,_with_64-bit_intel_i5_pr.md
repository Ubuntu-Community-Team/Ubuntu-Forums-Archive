### Post by yogesh.girikumar on 2010-03-05
Hi,

I would recommend a fresh install. If your system supports booting off an SDHC, then you might as well be able to boot off it. If you do a fresh install, you'll have the ability to use the card elsewhere on a different system which supports booting off the SDHC. This is a cool thing.

Also be sure to install the 32 bit version as a 64bit version will create problems if it is shared with other systems which are of x86 architecture.

Yogesh.
