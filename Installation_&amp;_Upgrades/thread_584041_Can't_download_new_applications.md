### Post by JPWheless on 2007-10-20
Never mind, I helped myself.


If anybody else is/was having this problem, I had to enable more repositories:

System -> Administration -> Synaptic Package Manager

In SPM:
Settings -> Repositories: Check all of the repositories you want/need.
