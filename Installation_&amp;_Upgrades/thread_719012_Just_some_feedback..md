Hello,
Today I went from XP to Ubuntu 7.10 and I thought I'd give some feedback in case it may help others doing this also.

- Install process when excellent, no issue at all. Format took longer then expected (hung at 5%, but after 15mins started moving).

- Once installed, we had our first issue. No sound. The volume icon was displayed at the top right indicating the Hardware was found. NVidia CK804 (Alsa) was what was detected. [ISSUE FIXED - SOUND WAS MUTED SOMEWHERE. SUGGEST YOU RUN alsamixergui AND ENSURE NOTHING IS MUTED]

- That was the only real critical HW issue I got. Next, Media Player. I found it very hard to get a good Media player, however I have now found Amarok.

- Could not play WMA files. [FIXED, ararok installed some codec when I tried to play a Mp3 file and this worked for WMA also]

- E-mail - needed a client that had a good junk mail filter [This is still open, I'm using Thunderbird and ThunderBayes, however I'm not happy - it only identifies Junk, but does not move them I need to play with the rules/filters I think]

- MSN messenger and Gmail. [Fixed installed Pidgin]

- Cisco Systems VPN Client [Fixed, installed KVpnc and something else when prompted - can't remember what exactly it was]

- Need a trace route command [Still open :  tracert [www.google.ie](www.google.ie)
The specified type of tracerouting is allowed for superuser only  -- Need to Google this]  [FIXED - COMMAND IS traceroute [www.google.ie]](www.google.ie])

- Needed root password [Workaround type: sudo passwd root]

- Connect to Dell AIO 810 Printer shared on Network [ Still open, Need Driver - may not be possible]



I hope this is slightly helpful to anyone...
