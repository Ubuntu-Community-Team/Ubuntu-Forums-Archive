### Post by lovinglinux on 2010-11-04
It is a known issue with libmoon. Just uninstall it and you should be fine. It is required for Moonlight (Silverlight plugin), so this will be removed as well.

```
sudo apt-get purge libmoon
```

To reinstall Firefox:

```
sudo apt-get install firefox
```
