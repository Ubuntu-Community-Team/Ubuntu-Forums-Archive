### Post by Cheesemill on 2009-05-11
Ubuntu Server by default does not include a GUI, it is command line only.
It also uses a different kernel that is optimised for server use.

Check [https://help.ubuntu.com/community/ServerFaq](https://help.ubuntu.com/community/ServerFaq) for more info.
