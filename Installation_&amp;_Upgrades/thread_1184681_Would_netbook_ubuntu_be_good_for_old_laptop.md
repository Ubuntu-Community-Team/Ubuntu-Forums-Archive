### Post by aysiu on 2009-06-11
The Netbook Remix isn't really designed for older hardware, and I don't even know think it has a small footprint. It's mainly designed to have large icons for launching applications and then to auto-maximize applications that launch.

If you want something for low-end machines, try CrunchBang Linux (based on Ubuntu). You might also want to look into Puppy Linux or Damn Small Linux (not based on Ubuntu).

Short answer: No.

Long answer, quoted from [https://help.ubuntu.com/community/Installation/SystemRequirements#Netbook%20alternative%20(Ubuntu%20Netbook%20Remix):](https://help.ubuntu.com/community/Installation/SystemRequirements#Netbook%20alternative%20(Ubuntu%20Netbook%20Remix):)

> Ubuntu Netbook Remix should run reasonably well on a computer with the following minimum hardware specification.

    * 700 MHz x86 processor
    * 384 MB of system memory (RAM)
    * 6 GB of disk space 

I dare to guess your old laptop hasn't got 384 MB RAM ... ;-)
