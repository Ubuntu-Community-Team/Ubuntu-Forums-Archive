### Post by Frogs Hair on 2012-05-21
If you were using unity and can't get a terminal from AWN try Crtl + Alt +F1 to enter TTY . Enter your user name, password and use the following. ```
unity --reset
``` wait until the process is complete and re-enter the desktop with Ctrl + Alt + F7 . If an error report is running wait until finished and unity should load shortly after.
