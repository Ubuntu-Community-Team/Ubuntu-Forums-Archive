### Post by Pynalysis on 2011-10-13
> **borgward said:**
> Why is 32bit recommended? My laptop is 64bit.

32 bit OS will run on 64 bit machines but not the other way around. Some software packages are not yet ported over to work on 64 bit OS - have not had many issues with this using 64 bit Ubuntu. Would recommend 64 bit - especially if your laptop has 4 or more gigabytes of ram - and if you run into problems with compatibility switch over to 32 bit.
