### Post by Mark Phelps on 2009-07-22
> **TheNerdAL said:**
> Will it be the same as making a separate Partition? 
Obviously not. Separate partition allows for more flexibility in file systems, partition arrangement and sizes.  But other than that, not much difference.

> Same performance and speed and protection?
Depends on the machine capabilities.  If you're running a modern 2-core or 4-core machine with 2GB of high-speed (DDR2 or DDR3) memory, the cycles used in Ubuntu to deal with Wubi won't even be noticeable; if you're running an ancient P-III 800 with 512MB of SDRAM (don't laugh, some folks are), Wubi is likely to be MUCH slower than a separate install.
