### Post by Mark Phelps on 2010-03-12
Since there is no "downgrade" from Win7 to XP, do the following:
1) From inside Ubuntu, reformat the Win7 partition as new NTFS volume
2) Boot from WinXP CD and install it.
3) Boot from Ubuntu LiveCD and reinstall GRUB
