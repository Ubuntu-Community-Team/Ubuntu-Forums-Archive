### Post by christhemonkey on 2006-12-17
I take it that this is after your install?
EDIT: Obviously...
> just thought i'd be practical and include the photo of what the screen looks like after my install.


Can you type <ctrl>+<alt>+<F1> which will take you to a terminal, then log in.

Type the following command:
```
cat /var/log/Xorg.0.log | less 
```

This will let you view the X servers log file.
(press up and down to navigate and press q to quit)
There might be useful lines that say something like:
> [EE]Error screen vsync out of range

Try finding out your exact monitor sync rates (often in the manual with it, or obtainable online) then enter them when going through this command:
```
sudo dpkg-reconfigure xserver-xorg 
```
And then select the Advanced option for entering your refresh rates.
