### Post by Mark Phelps on 2009-08-11
Wubi needs to have the Ubuntu CD or an ISO to install from.  Since it was probably unable to locate the ISO file, it was prompting you for the Ubuntu CD -- not for a blank CD.

You should read the Wubi guide:  

[https://wiki.ubuntu.com/WubiGuide%C2%A0]("https://wiki.ubuntu.com/WubiGuide%C2%A0")
