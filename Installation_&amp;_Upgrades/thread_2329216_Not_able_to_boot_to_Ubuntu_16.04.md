### Post by yancek on 2016-06-29
Windows 10 pre-installed will be using UEFI so you would have had to install Ubuntu UEFI.  Did you do that?  Explained at the link below which you can check to see if you used the correct procedures.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)

You can go to the site below and get the boot repair software and run it from a CD or from the Ubuntu DVD and post the link to the output.  Do not try to make any repairs but select the option to Create BootInfoSummary.

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
