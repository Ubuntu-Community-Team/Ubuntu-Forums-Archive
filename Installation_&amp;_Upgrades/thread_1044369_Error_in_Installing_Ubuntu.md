### Post by taurus on 2009-01-19
Did you tell the installer to use the whole harddrive when you installed Ubuntu?

What's the output of this command from a terminal?

Applications -> Accessories -> Terminal
```
sudo fdisk -**l**
```
That is a lower case letter L, not number 1.

Otherwise, you can try testdisk to see if you can recover your data.

[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)
