### Post by K.Mandla on 2006-05-11
Which release are you using? If you're using Breezy, I would suspect a flawed CD. The kernel should be fine, so long as the CD burned right.

If you're using a Dapper release, try the Beta 2 CD to install it. But the Dapper daily releases are sometimes ... inconsistent. I tried to install Xubuntu off the 2006-05-10 daily CD and got messages similar to what you describe.

Also, try to be patient with the grub install progress bar. I know once or twice it has sat and sat and sat ... and then mysteriously pops back to life. 

Good luck!
