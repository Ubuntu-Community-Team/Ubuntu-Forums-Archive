### Post by Frogs Hair on 2011-03-11
Methods vary from computer to computer or even from board to board . You should find the method for your computer first and improvise if needed from there. Make sure the update has features that you need . (release notes)

Example: if the update is for hardware support that you will never use there is no reason to update. My mother board requires a floppy so I can update from Windows or Linux other options include disk or usb. Some computers have update software but this tends to be a Windows only feature.
