### Post by kdormuth2 on 2014-08-24
To fix the grub, I followed these steps:
cd /mnt
mkdir boot
cd boot
sudo mount /dev/sda3 /mnt/boot
sudo grub-install --root-directory=/mnt /dev/sda
sudo umount /mnt/boot
