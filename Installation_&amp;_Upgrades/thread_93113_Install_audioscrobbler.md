### Post by AlexMartinius on 2005-11-21
You can install "xmms-scrobbler - XMMS plugin that sends your track information to audioscrobbler". Just type sudo apt-get install xmms-scrobbler in a terminal or search for the package in synaptic. Then you probably need to restart xmms and you can setup the plugin (enter username and password for Audioscrobbler).
