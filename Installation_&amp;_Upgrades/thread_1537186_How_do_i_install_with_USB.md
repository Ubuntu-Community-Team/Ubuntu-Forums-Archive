### Post by amsterdamharu on 2010-07-23
Unetbootin can extract an iso to harddisk, you need an os to run unetbootin so if there is no os (windows or linux) on the harddisk then it will be hard to run unetbootin.

After the iso is extracted to harddisk you will boot and have an option to "run the cd" from harddisk. If you wish to install however the partition where the iso file is extracted cannot be resized because it is in use so when you want to install it will be tricky.

Installing on a computer with no media (except harddisk) will be hard unless you have an os and some room (unused partition) on the harddisk
