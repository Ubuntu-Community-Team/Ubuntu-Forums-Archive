### Post by nepaliman on 2007-07-13
hi everyone i want to share what i had learned.. 
 to install realplayer download realplayer from internet ... (for linux)
then  you open terminal  and follow following steps..



( step one)       $ cd Desktop



(then type )

 $ chmod a+x RealPlayer10GOLD.bin

then 

Run the installer by typing

 $ sudo ./RealPlayer10GOLD.bin

and enter your password of your computer
#

Follow the prompts provided to finish installation.
do all u can !!
