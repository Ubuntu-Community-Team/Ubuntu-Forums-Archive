### Post by Partyboi2 on 2009-08-16
Hi. make sure you have check the download Ubuntu iso's [[COLOR=Blue]md5sum[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM") that they match. 
When you burn it to a disk as a iso file, make sure to burn at x4 or less speed to reduce the chance of data corruption.
Once you have booted to the Ubuntu main menu use the option to 'Check the cd for defects'.
