### Post by dakal on 2008-06-22
Assuming the laptop can be coaxed into having Internet access, I don't see why this wouldn't work: [https://help.ubuntu.com/community/Installation/WithFloppies](https://help.ubuntu.com/community/Installation/WithFloppies)

If it doesn't have a network interface, but you have a CD-ROM drive that Linux will work with over the USB interface, you probably could use largely the same guide but put the CD-ROM into the apt source list before starting the Ubuntu installation. That would probably take some fairly heavy wizardry and lots of finger-crossing to get to work, however...
