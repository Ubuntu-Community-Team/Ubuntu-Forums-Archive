### Post by hardcandy on 2005-01-14
Is ubuntu already installed? If so "apt-get install lilo" then look in the /usr/share/doc for the lilo example config file.
If not installed, try running an expert install (type expert at the boot prompt). It will let you choose which boot loader you want to install.
