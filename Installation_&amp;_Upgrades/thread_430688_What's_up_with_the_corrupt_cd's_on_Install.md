### Post by hoos30 on 2007-05-02
I'm new here and I've been trying to install Feisty on a four year old Dell P4 desktop for the past three nights.  I'm happy to report that I finally got it last nite (or this morning, as I fell asleep in the middle of the install), but it was a major headache and bad first impression.

I found the answer on this forum, so I wanted to share and possibly encourage an update in the help documentation.  Basically, I was getting a bad burn on the iso that I downloaded from the mirror.  I did the checksum, turned down the burn speed on InfraRecorder to 8x, checked the resulting CD for errors, but still ended up with the corruption.

I ended up burning the CD using the same procedure on another, newer drive and that did the trick.  Maybe the help docs can be updated to possibly warn for this or the "Check CD Intergrity" program can be fixed so that it actually detects a bad CD.
