### Post by m.musashi on 2006-02-06
Ubuntu has a good install program. It asks for several basic pieces of info (language, keyboard and such) and then scans the hardware. If it finds another OS (or maybe even if it doesn't) it brings up a partitioner program. If you make the wrong choices you will lose Windows but if you choose to partition it works fine. I've done this on 4 or 5 computers now and never a problem. All are dual booting Windows and Ubuntu.
