### Post by dstew on 2008-06-13
Looking at Ubuntu packages, there are several optical character recognition programs. I see gocr-gtk and tesseract-ocr. The tesseract package is command-line only, but there is a good description of how to use it [here]("http://www.howtoforge.com/ocr_with_tesseract_on_ubuntu704").

You can download and install these packages using the Synaptic package manager (System --> Administration menu). In Synaptic, under the Settings tab, you need to make sure the Universe repository is enabled.

I don't have any direct experience with either of these programs.
