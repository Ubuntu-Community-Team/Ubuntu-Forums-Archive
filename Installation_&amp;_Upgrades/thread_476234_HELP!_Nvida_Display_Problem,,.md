first thing you want to do is backup your xorg.conf (or any conf file) before you start changing things
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf-bak
```
then you can, with relative safety, edit your conf
```
sudo gedit /etc/X11/xorg.conf
```
add your "1280x800"  to the lines for each of the six modes making each look like this:
```
Modes "1280x800" "1024x768" "800x600" "640x480"
```
save the file and restart X (or you may need to reboot, not for sure).

in the event that something goes wrong and you cannot get a graphical login you will still be able to restore your current xorg. from the terminal login prompt enter your user name then password and then do
```
sudo cp /etc/X11/xorg.conf-bak /etc/X11/xorg.conf
```
this will restore your xorg.conf to the one you started with, so you will at least be no worse off than when you started.
