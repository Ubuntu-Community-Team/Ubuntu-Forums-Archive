### Post by Rocket2DMn on 2008-03-11
You should just be able to put the windows disc into the cd drive and boot from it (and therefore install from it).  This may require you to change the boot sequence in your BIOS to put the CD drive ahead of the Hard Drive, or to hit a key during the early stages of the boot sequence to either boot from the CD or choose from a list of boot methods.
