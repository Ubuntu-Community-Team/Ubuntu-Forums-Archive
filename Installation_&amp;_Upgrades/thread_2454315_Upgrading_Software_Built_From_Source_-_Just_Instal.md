### Post by rebeltaz on 2020-11-26
If I have software that I have built from source that I want to update, do I just need to build the new version and install it as if I were just doing it for the first time or is there something else that I need to do first/instead?

---

### Post by Impavidus on 2020-11-27
There are no rules. Typically, you can just build the new version and install it, overwriting the old version. Sometimes there were files in the old version that are no longer present in the new, and you have to remove those files manually or run the uninstall script that was hopefully provided with the software to remove the old version before installing the new.
