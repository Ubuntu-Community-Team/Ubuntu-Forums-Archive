### Post by Bucky Ball on 2011-10-18
If you want to install it alongside your current install you are talking about a dual-boot (two operating systems). If you want to install just the packages then you probably need to add the repo.

In the first link on that page, click the installation link. There's also this:

[http://www.mythbuntu.org/repos](http://www.mythbuntu.org/repos)

Just a matter of finding out which packages to install (yes, through software centre or Synaptics).
