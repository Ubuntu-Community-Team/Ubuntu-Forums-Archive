### Post by puneit on 2008-02-29
You are "double clicking" what? .exe files??? Not all work in Linux. Windows and Linux are different entities. 
Instead of the "exe" what you have here is the **.deb** Deb stands for Debain which is one can say the basis of this distribution. So in order to install programs by double clicking you need to double click .deb files.
How do get deb files... that has already been answered. 
Now your next question would be I want application X but its not there in Ubuntu. Ubuntu has a lot of equivalents for windows programs.
Read this [https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows)
and this [https://help.ubuntu.com/community/SoftwareEquivalents](https://help.ubuntu.com/community/SoftwareEquivalents)
