### Post by flakron on 2007-10-24
I had the same problem

this is how I solved it

open the terminal

[COLOR="Cyan"]Applications -> Accessories -> Terminal[/COLOR]

enter the command

[COLOR="Cyan"]sudo su[/COLOR]

it'll ask you for the root password, give the password

then enter this command

[COLOR="Cyan"]nano /etc/apt/sources.list[/COLOR]

where ever you see this comment

[COLOR="Cyan"]# Line commented out by installer because it failed to verify:[/COLOR]

uncomment the line below 

example : 

[COLOR="Red"]before[/COLOR]

[COLOR="Cyan"]
# Line commented out by installer because it failed to verify:
# deb [http://al.archive.ubuntu.com/ubuntu/](http://al.archive.ubuntu.com/ubuntu/) gutsy main restricted
# Line commented out by installer because it failed to verify:
# deb-src [http://al.archive.ubuntu.com/ubuntu/](http://al.archive.ubuntu.com/ubuntu/) gutsy main restricted
[/COLOR]

[COLOR="Red"]after[/COLOR]

[COLOR="Cyan"]
# Line commented out by installer because it failed to verify:
deb [http://al.archive.ubuntu.com/ubuntu/](http://al.archive.ubuntu.com/ubuntu/) gutsy main restricted
# Line commented out by installer because it failed to verify:
deb-src [http://al.archive.ubuntu.com/ubuntu/](http://al.archive.ubuntu.com/ubuntu/) gutsy main restricted
[COLOR]

P.S. I uncommented these lines too

[COLOR="Cyan"]deb [http://archive.canonical.com/ubuntu](http://archive.canonical.com/ubuntu) gutsy partner
deb-src [http://archive.canonical.com/ubuntu](http://archive.canonical.com/ubuntu) gutsy partner[/COLOR]

for no particular reasons


I hope it helped
