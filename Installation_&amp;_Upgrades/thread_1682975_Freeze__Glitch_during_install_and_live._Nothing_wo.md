### Post by korleon on 2011-02-06
I've experienced this before with Karmic.  I had a corrupted download.  The ISO was just a few MB less than what was listed at the Ubuntu download site.  Try checking that.

I only have 32 bit machines and my functioning 10.10 iso is 693.2 MB

There is also a MD5Sum.txt inside the ISO make sure that matches.  The documentation is here:
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
