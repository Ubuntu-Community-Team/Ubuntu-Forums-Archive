### Post by malt^ on 2009-08-18
Well ubuntu isn't windows, so if your trying to install a window program it gets a tad tricky but do-able. At your screen top is the"System" button which opens a drop down screen. 
Go to "Administration" and then go to "Synaptic Package Manager".
In "Syn-Pac-Man" scroll down thru the software list to "Wine".
Check the box for it (left side) and at the top hit the "apply" button.

It will then install "Wine" a window emulator to run most window programs to a point. Once the wine is install you should be able to open that install program for your phone set-up.
Hope this helps....
