### Post by Lisiano on 2013-03-07
Have you tried telling it to boot directly the CD by using a Boot menu while booting? Generally it can be found either inside the Bios menu (In the Exit section) or by pressing one of the F keys.
Also, while your PC might be saying it can't boot from a USB, it actually could using [Plop](http://www.plop.at/en/bootmanagers.html). You can also use it to boot your CD if you fail booting using the Boot Menu.
