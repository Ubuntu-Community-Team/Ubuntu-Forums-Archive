Seriously, why upgrade?  Is there something you really need in Edgy?

I can understand the appeal of playing with the new and shiney, but for a working system the maxim "if it ain't broke, don't fix it" is really good advice.

I have two systems at home - a work system (dapper) and an older machine that I experiment with (currently running Edgy).  I will eventually upgrade the working system but before I do so all vital apps will be running on the Edgy box.  I understand this is not practical for a university student, but if you really want to upgrade and can scrape together a bit of cash you could try getting an external drive and installing Edgy on that, as the previous poster suggested.

### Post by Dre1 on 2006-11-01
I've had some troubles using the 'apt-get distro-upgrade' method. First, the upgrade halted for some reason and it took me some while to find out how to proceed (apt-get upgrade -f). Then the Xserver didn't start because of a mismatch between the server module version and ABI module version. And finally I've had problems to get gdm running.

So overall, the upgrade wasn't a nice experience and I would recommend to wait a few days until these problems have been sorted out. (Nevertheless, I think the upgrade is worthwhile.)
