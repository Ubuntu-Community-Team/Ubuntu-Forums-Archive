### Post by aysiu on 2005-12-10
I'm still not sure I get what exactly you're looking for.
As Ubuntu installs, it tells you exactly which packages it is installing as it's installing them. I happen to be installing it on a computer right now, and it's giving me messages like *preparing xterm* and *preparing xclock*.

If you want a list of packages, you can go to [http://packages.ubuntu.com](http://packages.ubuntu.com) to see what's included with ubuntu-desktop, kubuntu-desktop, and xubuntu-desktop.

This site has a step by step walkthrough of the entire installation process (with screenshots), including all the questions you get asked:
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)

I have a PDF user guide that explains a lot of the basics of installation and preinstallation:
[http://ubuntu.xgn.com.br/ubuntu_user_guide_05.12.pdf](http://ubuntu.xgn.com.br/ubuntu_user_guide_05.12.pdf)

The Wiki has a starter guide:
[http://help.ubuntu.com/starterguide/C/faqguide-all.html](http://help.ubuntu.com/starterguide/C/faqguide-all.html)
