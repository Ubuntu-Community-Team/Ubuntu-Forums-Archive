### Post by raul_ on 2007-06-23
sudo - executes a command in super user mode without needing to switch accounts

apt-get  - executes the  package manager

install - it's a parameter for the previous command, that well...tells it to install

build-essential - it's name of the package to install. this package in particular contains the basic packages to compile things on your own

PS: you could just open [http://www.getdeb.net]("http://www.getdeb.net"), download the pidgin package, and double click it ;) much easier
