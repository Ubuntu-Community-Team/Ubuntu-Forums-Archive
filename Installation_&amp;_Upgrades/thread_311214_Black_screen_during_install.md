### Post by wpshooter on 2006-12-08
If you are not going to be running O/S other than Ubuntu then I would recommend getting **KILLDISK** program and **WIPE** your hard drive clean and then retry installing from the DESKTOP CD.

[http://www.killdisk.com/](http://www.killdisk.com/)

If it still will not install the download and burn the ALTERNATE CD version and install from that.

Make sure your burn your CDs at 4X or less and make sure you check the integrity of your CDs by running the verification test found on the initial Ubuntu boot screen menu.

Good luck.
