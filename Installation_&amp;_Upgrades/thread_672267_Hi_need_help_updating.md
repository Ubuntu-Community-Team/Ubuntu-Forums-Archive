### Post by zvacet on 2008-01-19
if you mean how to change server 

> To use your local mirror you can add "cc." before archive.ubuntu.com (cc = your country code)

For example 

deb [http://archive.ubuntu.com/ubuntu](http://archive.ubuntu.com/ubuntu) dapper main restricted universe multiverse

change to 

deb [http://us.archive.ubuntu.com/ubuntu](http://us.archive.ubuntu.com/ubuntu) dapper main restricted universe multiverse
