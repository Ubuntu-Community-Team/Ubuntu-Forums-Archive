### Post by Therion on 2008-09-25
Put the disk in the drive and let it spin up.
Open the CD using Windows Explorer.
If there is only ONE file on the CD (ending in .iso) you've not burned the CD correctly.  

If you see numerous files and a few folders on the CD you've burned it correctly but it's possible the "burn" itself was bad.  
Try re-burning the CD at slooooow speed, like 8x or even 4x and try installing again from the new CD.
