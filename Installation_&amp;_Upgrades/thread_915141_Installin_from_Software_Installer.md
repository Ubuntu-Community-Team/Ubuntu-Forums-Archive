### Post by tuxxy on 2008-09-09
To run the apps you installed, open terminal and type their name in but remember its case sensitive so for example to run virtualbox you would type

```
VirtualBox
```

To prevent having to do this each time you would like to run an app, you can right click on your applications menu and add a launcher to your menu using the command for each app like above
