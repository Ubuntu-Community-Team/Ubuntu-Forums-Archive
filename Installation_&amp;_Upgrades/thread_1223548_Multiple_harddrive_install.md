Hi

I am going to install Ubuntu 9.04 Server using a computer
that has several harddrives. So i wonder how do i in the install
direct it to use the swap on another harddrive.

The reason for this is becuse i would like to use the smallest drive
so that the others can be used for other stuff like space for my server.

So it would be like this.
First drive Main Ubuntu install
Second drive My servers space
Third drive swap

---

### Post by Hobgoblin on 2009-07-26
Select the manual partitioning option.

Put / on the 1st drive

Put /swap on the smallest drive

Put whatever mount point you will use for your server on the other drive.  Or leave it and assign that later.
