### Post by wooster on 2007-11-17
What kind of server are you looking to set up? You can install ubuntu from the desktop cd and install what you need for the server you are looking to set up.
Here is a link that will help you with the server part. [https://help.ubuntu.com/ubuntu/serverguide/C/index.html]("https://help.ubuntu.com/ubuntu/serverguide/C/index.html") to see how to set up various kinds of servers.

If you install from the server edition run this command to install the ubuntu desktop. 
```
sudo apt-get install ubuntu-desktop
```
