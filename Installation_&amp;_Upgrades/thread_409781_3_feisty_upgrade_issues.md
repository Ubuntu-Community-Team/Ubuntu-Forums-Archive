### Post by hardyn on 2007-04-15
1) it is possible that the boot is choking on the detect of something... run the disk verify util at the boot menu, make sure that disk is good.

2) synapics, not sure... the 3 screens, that is an overscan.  your display is likely a 60hz refresh, and it is running at 70hz (or higher) for some reason, i have seen this before, 1024x768 boot up is normal... i have never had any of the ubuntu disks run the correct resolution off the live disk.

3) youd think... but no... i have had the same thing happen.

you might want to try the alternate install disk with the text based installer so you can get the screen parameters right

ps. what notebook are you running?
