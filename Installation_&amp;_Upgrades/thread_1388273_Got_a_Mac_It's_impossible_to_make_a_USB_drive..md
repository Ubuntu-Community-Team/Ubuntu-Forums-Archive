### Post by unclejames on 2010-01-22
Fact: There's no .img version of the latest Ubuntu Netbook Remix. The only files available for i386 machines are .iso.

Fact: If you have a Mac, it will only create bootable USB drives from .img files. There is no way of creating a bootable USB drive from an .iso.

Fact: It's impossible for me to install Ubuntu onto a netbook, if the only other machine I have access to is a Mac.


It would be difficult for me to understate how frustrating this is for a Mac user.
