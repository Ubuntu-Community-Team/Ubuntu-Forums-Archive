### Post by howefield on 2016-11-30
Do you use the google-earth package ?

I'd suggest purging it and removing the ppa for it.

```
sudo apt-get purge google-earth
```
```
sudo mv /etc/apt/sources.list.d/google-earth.list* ~/Documents
```

I'm guessing at the name of the package, if it isn't google-earth then change to suit.

There is a more graceful method to removing a PPA which is to use the ppa-purge application, it isn't installed by default. Once installed use sudo ppa-purge ppa_name to remove the ppa and put all related packages back to default.
