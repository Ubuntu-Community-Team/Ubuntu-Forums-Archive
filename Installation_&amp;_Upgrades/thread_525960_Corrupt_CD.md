### Post by dabl on 2007-08-15
To check the quality of the downloaded ISO file, you check the md5 number of the downloaded file against the md5 checksum number for that particular file that is on the Ubuntu download site alongside the file.  Usually, your CD burning application will "announce" the md5 sum when you tell it to "burn as ISO".

Then there is the question of the CD burn quality.  You need to set the burn speed down to 4X for this particular CD.

So, with a good download ISO file, and a slow-burned CD, you should be good to go.  :popcorn:
