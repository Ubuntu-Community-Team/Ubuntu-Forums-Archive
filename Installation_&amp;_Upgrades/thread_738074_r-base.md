### Post by Voynix on 2008-03-28
R is a terminal based statistics package. You need to type ```
R
``` in a terminal window to launch the programme. There are some gui frontends (rkward or jgr, both available in the repos). There is a [good discussion]("http://ubuntuforums.org/showthread.php?t=33052&highlight=spss") on more visual alternatives in the forums.
