### Post by Quackers on 2011-01-31
It appears that grub has not been installed. You also have a couple of Ubuntu boot files missing.
I suggest that you boot from the Ubuntu live cd/usb and select "try ubuntu" then when the desktop loads, make sure you have an internet connection.
Then I suggest you open up a terminal and copy/paste the following commands into the terminal, one line at a time, pressing enter after each line.
```
sudo mkdir /mnt/temp 
sudo mount /dev/sda6 /mnt/temp
for i in /dev /dev/pts /proc /sys; do sudo mount -B $i /mnt/temp$i;  done
sudo chroot /mnt/temp
```
At this point, if the commands ran successfully your terminal prompt should change from ubuntu@ubuntu to root@ubuntu
Now you should go to the guide below and scroll down to the heading "Why Chroot", in red.
As we've already done the first few commands, scroll down to the heading at no 2
"2. Confirm an Internet connection"
Then enter those commands (not the bit after the # in the first command) as described there all the way to No 7, but just the first command in No 7 (the second one is for people who have a separate /boot partition - you do not).

EDIT!  Sorry, I forgot the guide! Here it is :-)
[http://ubuntuforums.org/showthread.php?t=1581099](http://ubuntuforums.org/showthread.php?t=1581099)


If no error messages appear anywhere (as per the guide details) reboot, and, hopefully Ubuntu should boot. You may or may not get a grub menu with Windows listed. If you do not get that menu, when Ubuntu loads, open a terminal and run ```
sudo update-grub
``` and watch to see that the Windows Loader is picked up. If it is, reboot and select Windows from the grub menu, to check that it loads ok.

Take care to let all commands finish before copy/pasting the next one. If you get stuck anywhere, or are not sure of something, just ask. It is better to get it right first time :-)

As this is as a result of an unfinished install, Ubuntu may not work anyway, but we'll see!
Good luck!
