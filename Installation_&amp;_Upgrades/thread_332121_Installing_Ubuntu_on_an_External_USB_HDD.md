### Post by danilo287 on 2007-01-05
first you should go into the laptops bios settings and check the boot sequence/order (make sure that the hard drive is plugged in when you start the laptop up).  If it can boot from the hard drive it will show you in the boot sequence.  Move the external hard drive infront of the internal hard drive for the boot sequence if it even allows you too boot from the external hard drive.  Once you do this insert the Ubuntu install cd and when u run through the install you should be able to select the external hd.
