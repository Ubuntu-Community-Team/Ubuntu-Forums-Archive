### Post by fantab on 2013-05-29
Try [**Boot-Repair**]("https://help.ubuntu.com/community/Boot-Repair"). It should fix the boot issue. Just run the 'recommend repair' and more often than not you are good to go. If that does not help then post the link generated for "Bootinfo Summary'.

Good Luck...
