### Post by lisati on 2009-04-05
Ubunutu doesn't normally run .exe files (they're usually designed for Windows and/or MS-DOS). If they're regular applications, you might have to use Wine or similar to get them to work. Another option is is to look for an Ubuntu-friendly equivalent.

See [here]("https://help.ubuntu.com/community/Wine") for more information on Wine.
