My son and I recently reloaded a recycled machine with Hardy. It has some quirks, one of which is the sound. The machine came with an Audigy sound card, one with all the doobries you could possibly want. The install has recognised and configured this card, along with the main board sound. The Audigy has so many channels it results in an impressive audio mixer display which is substantially wider than a 1600x1200 monitor can handle. In fact, you would probably need one of Apple's 30" widescreen displays to be able to see it all in one go.

However, apart from directly after the install, when sound went through the Audigy card during post-install testing; once the machine was rebooted, only the on-board sound is working, and nothing I yet have been able to do, will get sound output routed via the Audigy card. What we suspect may be happening is some sort of misconfiguration which occasionally shuts down sound altogether, especially from anything running in Wine. This is not actually a problem, because we want to remove the Audigy and use it in another recycled computer, destined to be a media machine, probably with 64Studio.

My question is: how do I go about removing the packages/configuration installed for the Audigy, without going the time-consuming and download expensive path (we did a lot of post-install work) of a reinstall?
