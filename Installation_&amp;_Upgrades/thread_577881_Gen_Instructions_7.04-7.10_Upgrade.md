### Post by Sef on 2007-10-16
Read [Gutsy Upgrades]("https://help.ubuntu.com/community/GutsyUpgrades").


> What if we have multiverse / universe restricted codecs installed? will anything break?

There could be a problem with restricted drivers, especially for video cards.

> Any taboo applications that are certain to cause problems?

Someone else will have to answer that.

> I am awaiting my disk from Canonical. Meanwhile I should make certain everything is updated. All applications as well as system. What else? Back-up system to external drive?

Back up your home directory before installing Gutsy.

> Is this a good time to add in a larger hard drive or do it after install?

You can add it now.   Gutsy should pick it up.

> Put the new CD in the drive and it will automatically detect previous install and then what? Any pitfalls to avoid?


Yes, it should detect all of your partitions.  If you have a back up, then there should be no problems that can't be solved.
