### Post by MARP1961 on 2011-10-11
Welcome,

Ubuntu works in a different way to Windows. You will have to learn a new way of doing things. You cannot install software from the net by downloading exe. files. Software for Ubuntu is provided in stores or repositories accessed by the Ubuntu Software Centre (search in the menus).

Your next step should be to fix yourself up to the internet using a network (ethernet) cable and get yourself an install disk for the latest version of Ubuntu (11.04 Natty Narwhal). Your present version 9.10 (Karmic Koala) is no longer supported by critical updates. Once 'Natty' is installed and connected to the internet by cable then open up Update Manager and do the updates. We can then help you get your wireless working.

Post us a few details about your computer specifications.

### Post by MARP1961 on 2011-10-12
OK it recognises your wi-fi signal. Left click the network manager and see if ENABLE WIRELESS and ENABLE NETWORKING are checked. If they are then click on EDIT CONNECTIONS and click the wireless tab then your wi fi router.Next, make sure you have entered your WPA/WPA2 passkey (the same one that works for Windows) correctly and make sure you have enabled ALL USERS and AUTOMATIC CONNECTION.

If you do this, it should attempt to connect to your wi-fi router once you unplug any network/ethernet cable you might have plugged in. 

Sometimes you will be repeatedly prompted to enter a keyring password and this can be another reason for connection problems. It might be better not to have your computer set for automatic logon if this is the case.
