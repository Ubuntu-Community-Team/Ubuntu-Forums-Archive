### Post by Lacrimstein on 2007-09-11
Yes, perhaps its best if you unplug those drives. Install Ubuntu, see if it works fine, then plug those drives back in. You will be able to browse them, but before you write to them, check their file system - if it's NTFS, you will need to install ntfs-3g. Otherwise, the data there will be read-only
