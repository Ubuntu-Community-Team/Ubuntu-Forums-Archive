### Post by Bucky Ball on 2014-01-24
Welcome. Download the 12.04 or 13.10 ISO, burn image to disk (or USB), boot from the install media and install. That easy! ;)

[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

Your only issue might be whether the old specs can handle the new OS. If not, there are ways around this, like trying Xubuntu, Lubuntu or a minimal install (or another, even lighter, distro).
