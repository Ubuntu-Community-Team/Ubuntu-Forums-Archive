### Post by logos34 on 2007-06-10
If I'm not mistaken, it is only possible to upgrade one release at a time.  (That would entail 3 successive upgrades!).  In any case, the only free ShipIt CD's available are 7.04 Feisty, although you can purchase older releases elsewhere.

If at all possible, the easiest course of action, I think, would be to backup all your data and home directory, then do a fresh install of Feisty using the shipit cd.  That would leave only the recent updates.  You could probably download the updates to a usb flash drive using the computers at a local library with fast internet, a cybercafe, etc., or just do it over dialup (they can't be that big).
