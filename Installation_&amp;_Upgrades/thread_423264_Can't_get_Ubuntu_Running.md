### Post by rsambuca on 2007-04-25
There are a couple of things it could be:

1.  As the previous poster indicated, it may just require a simple bios change.  When booting your computer, just after the first couple of boots, press esc, or F12, or whatever the key is to enter your bios.  You can check with your computer manufacturer's website for that.  Then change the boot device section to CD drive first.

2.  You may not have burned the CD as an image instead of a regular data CD.

### Post by rsambuca on 2007-04-25
Open the CD in Windows explorer - How many files and folders do you see?  if you only see one file, you burned it incorrectly.  If you see a few different folders and files, then you burned it correctly.

If you only see one large file on the CD, then you will have to burn another installation CD.  Instructions on how to burn the CD properly can be found [[COLOR="Red"]here[/COLOR]]("https://help.ubuntu.com/community/BurningIsoHowto").
