### Post by guiverc on 2020-08-07
Existing users of 20.04 have been running 20.04.1 for days (including flavors).

The release primarily relates to the release of the ISO for new installs
[http://fridge.ubuntu.com/2020/08/06/ubuntu-20-04-1-lts-released/](http://fridge.ubuntu.com/2020/08/06/ubuntu-20-04-1-lts-released/)

The release does not actually mean existing users of 18.04 LTS will be offered the upgrade though, as the wording is that it will be offered **after** release of 20.04.1
You'll note the following wording in the release (see prior link)

> Users of Ubuntu 18.04 LTS will **soon** be offered an automatic upgrade to  20.04.1 LTS via Update Manager.  For further information about  upgrading,

This is to ensure maximum stability.  A decision will be made on when this occurs, at earliest it'll be next week, but may occur later too if it ensures stability, as it's assumed LTS users prize stability.
