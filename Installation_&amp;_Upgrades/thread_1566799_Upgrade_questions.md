### Post by Rubi1200 on 2010-09-03
You can upgrade an LTS release to another LTS release:
[https://help.ubuntu.com/community/UpgradeNotes#From%209.10%20or%208.04%20LTS%20to%2010.04%20LTS](https://help.ubuntu.com/community/UpgradeNotes#From%209.10%20or%208.04%20LTS%20to%2010.04%20LTS)

However, many people recommend a clean install.

Make sure to backup important data before choosing either option.

In my opinion, try 10.04 as a LiveCD first to make sure it is compatible with your hardware.
