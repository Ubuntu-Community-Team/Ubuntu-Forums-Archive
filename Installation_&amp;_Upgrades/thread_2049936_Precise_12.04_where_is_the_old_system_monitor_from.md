### Post by Karlchen on 2012-08-29
Hello, doktorOblivion.

gnome-system-monitor is still gnome-system-monitor. This is true e.g. on Ubuntu 12.04 (Unity Desktop) and Linux Mint 13 Cinnamon Desktop.
The only edition which seems to bring along a pretty lean task-manager which reminds of ancient Windows task-managers seems to be Linux Mint 13 XCFE. So the same might be true for Xubuntu 12.04.
Anyway, if gnome-system-monitor is not present, then you are free to install it through the Software Centre or through Synaptic.
