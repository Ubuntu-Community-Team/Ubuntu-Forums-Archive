For some reason, maybe update/package manager cannot find a server.... or the files needed from the server you've chosen in software sources. I've seen this on 64-bit files.

Try a different server.  Go to software sources (system > admin) and the tab UBUNTU SOFTWARE will allow you to choose a different server.

Regards,

Raymond
