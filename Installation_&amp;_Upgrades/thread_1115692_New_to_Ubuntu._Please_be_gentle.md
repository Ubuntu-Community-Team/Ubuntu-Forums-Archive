### Post by whitefort on 2009-04-04
Hi.  One of the nice things about Ubuntu is that you can actually install it in XP, just like installing any Windows application.

When this is done, when you turn on the PC you'll be offered a choice of whether to boot into Ubuntu or windows.

I've found this the easiest way to get people to play around with Ubuntu, as it doesn't involve any scary disk partitioning or anything like that.

There's also the security of knowing that if you want to get rid of Ubuntu for any reason (WHY???!!!!), all you need to do is open the XP control panel and uninstall it like any other windows app.

It's been a while since I've done it, but if you put a Ubuntu CD into a PC that's running XP, it will walk you through the process.

Hope this is some help!
