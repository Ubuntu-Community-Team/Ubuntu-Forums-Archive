next time you install do two things:

1. [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM") the downloaded iso prior to burning it as an image to CD.

2. Burn the CD at 4x-8x speed. When you boot the CD choose "check disk for defects" prior to installing or trying Ubuntu from the CD.
