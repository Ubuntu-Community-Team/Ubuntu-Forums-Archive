### Post by skippy81 on 2006-05-12
If you are more familiar with windows than with linux then I suggest you use windows to analyse the disk.   Delete unused partitions in order to make a large block of free (unpartitioned) space.  When you run the ubuntu installer you can tell it to automatically set up in this unpartitioned space.  That way you don't run the risk of getting mixed up and writing over your windows partition.
