### Post by Lord Landis on 2007-04-19
While it really is a matter of personal choice, I'd set him up with Kubuntu for now.  It ships with more of the stuff that a 9-year-old would like, including several 'edutainment'-type games.  Also, the interface is more customizable w/o needing to resort to Beryl/Compiz.

Of course, with that said, you could always install Ubuntu and then apt-get KDE after the fact.  That would allow him or anyone else to easily switch back and forth between the two simply by logging out and then changing the session.  Since most apps will run just as well under either desktop, there's no need to worry.

Personally, I think KDE presents a more polished interface and since Amarok is a KDE native app, I tend to run with it.
