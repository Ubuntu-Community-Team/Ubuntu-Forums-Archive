### Post by graphius on 2010-10-29
You are trying to run ubuntu from within windows in a sort of virtual machine. Before it can install itself, it downloads a fresh version of ubuntu (the iso image). I am not sure why it does this when you have the cd in the drive, but then again I am not a developer.
I do think it is better, once you have played with the live CD to make sure everything works, to install ubuntu on its own partition beside windows rather than with WUBI.
