### Post by kidders on 2007-07-24
Hi there,

The differences between the server & desktop editions of Ubuntu are subtle at best ... for most users, the only major distinction is whether a graphical environment (which poses major security & performance issues) is present or not. So, I can't help wondering whether installing a Ubuntu server, only to turn it right back into a Ubuntu desktop, would be worth the trouble.

For your situation, the first thing I would recommend is to decide what you want from your server box. Ubuntu's server installation is designed to be lean, performant & stable. The more you deviate from the way it was intended to be configured, the less you will gain from the attention that has been paid to its design.
