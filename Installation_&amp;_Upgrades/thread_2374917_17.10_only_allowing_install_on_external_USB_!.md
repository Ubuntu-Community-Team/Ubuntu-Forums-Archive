### Post by oldfred on 2017-10-20
Most Windows 7 installs use all 4 primary partitions allowed in the 35 year old MBR configuration.
But you can make one primary an extended partition and use it as a container for as many logical partitions as you want.
Post this:
sudo parted -l

### Post by yancek on 2017-10-20
> I have 3 physical internal HDs, all NTFS with plenty of room

Having 'plenty of room' if it is inside a windows partition isn't going to work.  You need free/unallocated space 'outside' your windows partitions.  You can shrink one or more of your windows partitions from Disk management to accomplish this. More specific advice will need the output of the command asked for by oldfred.
