> How do I uninstall it?
Usually you would be able to uninstall it as usual.
Try opening the Uninstall wine sofware (search Wine in the dash)
If it doesn't work, I'm afraid you'll have to delete the hidden wine folder (.wine): Open the file manager (Files AKA Nautilus) > CTRL+H to see hidden folders. It'll however remove all installed Windows software (but not WINE itself, the next time you open it or try to install other Windows software the folder will be recreated).
