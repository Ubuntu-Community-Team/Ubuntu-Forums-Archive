### Post by Pumalite on 2009-09-07
Download ImgBurn:
[http://www.imgburn.com/](http://www.imgburn.com/)
Install it
Go to Mode>ISO>Burn
Choose 4x or less for burn
Burn a new CD
Do not use CD-RW
Download a new ISO if you have to
