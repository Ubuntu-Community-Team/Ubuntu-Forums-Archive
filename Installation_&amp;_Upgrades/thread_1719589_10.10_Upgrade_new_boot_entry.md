### Post by Rubi1200 on 2011-04-02
Hi,
yes this is normal. After a kernel upgrade a new entry is added to the GRUB menu (I believe the maximum is 7).

It is advisable, for troubleshooting purposes, to keep at least one spare kernel on the menu.

If you would like a GUI app. to customize the menu, including deciding which entries appear, see this tutorial by drs305:
[http://ubuntuforums.org/showthread.php?t=1664134](http://ubuntuforums.org/showthread.php?t=1664134)
