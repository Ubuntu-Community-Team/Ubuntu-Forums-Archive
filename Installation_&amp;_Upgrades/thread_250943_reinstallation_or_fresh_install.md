### Post by facefur on 2006-09-05
I agree with Anonii.  Back up your home folders, and anything else important.  Try using the alternate CD in "Text Mode" install.  You can define all new partitions in the partitioning step, reformat them, and have a clean installation.  I created /home and /usr partitions on my machine in addition to the / one.  I also dual boot, so there is a windows partition as well.

Once the install completes, you can copy the backed-up files to whatever partition/folder you like.
