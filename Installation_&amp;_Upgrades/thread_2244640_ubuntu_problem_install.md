### Post by yancek on 2014-09-17
Since you second image shows a boot menu for an Ubuntu installation medium, I suppose you are trying to install it?
The first image is not your boot menu, that is the BIOS setup.  You have "Legacy" set for Boot Mode and if you have windows 8/8.1 and it came pre-installed, you should set it to EFI/GPT and use that option for Ubuntu when you install.
