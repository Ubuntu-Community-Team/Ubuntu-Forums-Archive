### Post by lemming465 on 2010-05-09
Yes, you can upgrade multiple PC's from one set of package downloads.  Easiest method: download the appropriate "alternate install" CD, and follow the directions in the file on the root.

Second easiest method: follow directions for [setting up a local APT cache server]("https://help.ubuntu.com/community/Apt-Cacher-Server").
