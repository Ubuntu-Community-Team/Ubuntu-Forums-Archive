### Post by cinematography on 2007-08-31
I found the option. It was in PCMan's preferences under the Desktop tab. You can set it to "show icons on desktop" and it kind of takes over the default desktop. :D And then you can change the folder on the taskbar at the top to open with pcmanfm instead of Thunar.
