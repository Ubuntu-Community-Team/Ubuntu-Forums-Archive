     I installed ubuntu 9.04 in my desktop. if I want to upgrade to ubuntu 9.10, will all installed packages be deleted? pl. reply

No, your Linux programs will be upgraded if they are supported by ubuntu to the versions that come with 9.10, such as FF3.0 to FF3.5.

If you have a slow or flaky internet connection, you may want to download the 9.10 alternate installer, burn it, then use it to start the upgrade by inserting it into the drive while 9.04 is running.

Please be aware that sometimes upgrades do break.

Let us know how it goes and if you have any more questions.
