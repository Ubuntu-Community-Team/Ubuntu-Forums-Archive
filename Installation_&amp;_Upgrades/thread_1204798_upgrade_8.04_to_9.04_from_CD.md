### Post by Ekeluo on 2009-07-05
If it's supplied with a magazine it's most likely a live cd, which can't be used to upgrade an installed system. You'll need an alternate (text-based install) cd which you can download from the ubuntu website for that purpose. I assume this version (9.04) works fine for you and recommend that you install from scratch.
