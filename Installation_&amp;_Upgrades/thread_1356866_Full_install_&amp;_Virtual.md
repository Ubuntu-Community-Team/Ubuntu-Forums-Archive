### Post by Dennis N on 2009-12-16
Certainly you can have Ubuntu in a dual boot setup and also running in a virtual machine. With VirtualBox running in Vista, it is essentially a Vista application and can access the internet as such. Ubuntu running in that virtual machine should connect as well since VirtualBox serves as a go between. I don't see how this will help the dual boot Ubuntu installation, however.

Good Luck.

### Post by snowpine on 2009-12-17
You would have two separate Ubuntu installs in this scenario: the dual boot, and the virtual machine. Updating one would not update the other.
