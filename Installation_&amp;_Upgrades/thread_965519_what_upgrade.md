### Post by KenMac on 2008-10-31
Hi. I guess because Hardy is an LTS, normal upgrades are disabled by default.  It is easy to get however:
1. Go to *System > Administration > Software Sources*
2. In the *Software Sources* window click on the *Updates* tab.
3. Select *Normal releases* in the *new distro releases* drop down.

Then start Update Manager and the option to upgrade should be available.
This link has a good description:
[http://www.howtoforge.com/how-to-upgrade-ubuntu-8.04-to-ubuntu-8.10-desktop-and-server](http://www.howtoforge.com/how-to-upgrade-ubuntu-8.04-to-ubuntu-8.10-desktop-and-server)

Good luck!
