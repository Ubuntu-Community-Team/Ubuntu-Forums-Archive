Terminal... aka gnome-terminal
Top
Ifconfig
Apt-get
Joe
Midnight Commander
Rdesktop
Alsamixer
Lshw
And generally the terminal applications are VERY useful.

Synaptic
Exaile
Compiz (cube + atlantis kicks ***)
Fusion-Icon - this one is a must if using compiz
Cairo Dock - i use it with the -c option - it still has cool effects and uses 1/5 or 1/10 memory as with opengl - its damn stable this way and works on most hardware without glitches

Skype - only with alsa

Wine

OpenOffice
