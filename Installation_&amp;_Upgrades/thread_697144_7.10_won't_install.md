### Post by Rocket2DMn on 2008-02-14
Yeah, this is a fairly common problem.  You can install with the alternate install cd from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the alternate desktop cd box at the bottom.  This will give you a text based installer without a live session, but it is very intuitive and easy to use.
