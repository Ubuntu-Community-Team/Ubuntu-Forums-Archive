### Post by aysiu on 2005-10-15
Windows 2000 installation is hell and it's all text-based. Windows XP leaves you with a completely empty desktop (with the exception of the Recycling Bin) and no quicklaunch toolbar activated.

Please do not compare Linux to Windows. Thanks.

I'd say if you want an install like Windows 2000, Ubuntu's installer is similar and actually better--it's text-based but really explains what's going on step by step. If you want an install like Windows XP, maybe Mandriva, Mepis, or SuSE might be your cup of tea.

Honestly, though, you shouldn't pick a distro based on the installer. I'd pick it based on one or more of the following:

1. Hardware compatibility
2. Package management
3. Community Support
4. Cost
5. Ease of acquisition

You may find these two links helpful in finding a good distro:

[http://www.zegeniestudios.net/ldc/](http://www.zegeniestudios.net/ldc/)
[http://distrowatch.com/dwres.php?resource=major](http://distrowatch.com/dwres.php?resource=major)
