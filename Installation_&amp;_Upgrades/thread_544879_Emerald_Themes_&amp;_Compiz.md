### Post by Inxsible on 2007-09-07
> **snakeminidez said:**
> I recently installed Linux onto my computer and I was wondering how to install Emerald Themes and I also was curious to know if it is necessary for me to download Compiz 0.5.2 since I just installed Linux.
Emerald works with Compiz Fusion and Beryl. so you don't necessarily need Compiz Fusion, you can do with Beryl
But Beryl is no longer in development as its merged into Compiz to become Compiz Fusion. 
So you should look into installing Compiz Fusion and then you can install Emerald by
```
sudo apt-get install emerald libemeraldengine0*
```
