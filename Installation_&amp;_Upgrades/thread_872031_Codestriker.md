### Post by Partyboi2 on 2008-07-28
Download the file from [[COLOR=Blue]here[/COLOR]]("http://downloads.sourceforge.net/codestriker/codestriker-1.9.5.tar.gz?modtime=1215166127&big_mirror=0") to your Desktop then open a terminal (Applications>Accessories>Terminal) and extract it
```
tar xvzf codestriker-1.9.5.tar.gz 
``` then change into the newly created codestriker-1.9.5/bin directory
```
cd codestriker-1.9.5/bin
``` 
then start the installer and answer the questions displayed on screen.
```
sudo ./install.pl
```
