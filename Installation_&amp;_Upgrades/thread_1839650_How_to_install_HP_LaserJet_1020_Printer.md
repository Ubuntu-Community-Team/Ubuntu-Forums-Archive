### Post by chip616 on 2012-02-04
This worked for me using Kubuntu 11.04.

During the install it asks you to unplug and replug the printer or to reboot your machine and then run the hp-setup command.  I had to do the latter, as my printer did not show up in the discovered devices list by just unplugging and replugging it.

You now end up with an 'hp' icon in the kicker for the HP Device Manager software, but at least the printer works.
