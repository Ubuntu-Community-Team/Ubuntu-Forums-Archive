### Post by mikewhatever on 2009-09-20
Since you want to reinstall Ubuntu, that implies overwriting the current installation. You can't do it while the current installation is running, and need to use the installation cd. That said, deleting your Windows partition from Ubuntu is not a problem, just install gparted from the repositories. If all you want is to delete the Windows partition, gparted may safe you time and effort of reinstalling, and you could use the freed partition for storage and backups.
