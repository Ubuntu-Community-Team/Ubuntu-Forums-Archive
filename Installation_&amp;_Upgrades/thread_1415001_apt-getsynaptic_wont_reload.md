### Post by mbm450 on 2010-02-24
I just had the same issue with Ubuntu 9.10  The way I fixed mine was to go to >system>administration>software sources, type password and then from the download from box click the down arrow, select other and then click best option. It will do a search and highlight the best choice, then all you do is select it. I just did it 30min ago and now my synaptics manager is working fine again. Hope this helps, Good luck
