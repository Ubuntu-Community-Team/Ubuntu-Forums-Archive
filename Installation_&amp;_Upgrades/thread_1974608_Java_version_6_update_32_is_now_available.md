### Post by Cavsfan on 2012-05-06
The latest version of Java - version 6 update 32 just became available.
The [COLOR=Red]red link[/COLOR] in my signature will check your version and provide an option to download the new version.
Download the ".bin" file and not the ".rpm" file. I have a 64 bit system so I downloaded **jre-6u32-linux-x64.bin**.

The [COLOR=Green]green link[/COLOR] will provide instructions to remove the previous version and install the new version.
For 32 bit machine, the instructions are down a ways on the left, just scroll down until you see **HOW-TO FOR 32 BIT UBUNTU**.
For 64 bit machines, on the right side look for **HOW-TO FOR 64 BIT UBUNTU**.

If you just follow the instructions carefully, you will have no problem.
That site has already been updated, so you just have to copy and paste where applicable.
