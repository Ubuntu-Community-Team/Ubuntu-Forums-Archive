Welcome to forums Jon,

If you have upgraded to Natty (11.04), I hope this thread by hedgehog1 may help you:
[http://ubuntuforums.org/showthread.php?t=1741293](http://ubuntuforums.org/showthread.php?t=1741293)

If a previous one, then I guess "Autohide" is somehow enabled for the top panel. Just disable it in panel properties, and to add the icons just right-click (on the panel) > Add to Panel > select "Clock", "Indicator Applet", "Notification Area", "Indicator Applet Session",.... (and whatever else you need) from the opened list.

If this doesn't help, please mention exact version you are using. Some screenshots may be great!
