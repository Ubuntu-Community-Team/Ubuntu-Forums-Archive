The best way is this:

1) Install first windows on a partition of its own.
2) Then install ubuntu on a partition of its own.

The most important thing to know is that if you have a single hard disk with a single partition which has windows on it and install ubuntu on it then it will of course delete windows (unless you use wubi - see the wubi subforums here). 

Ubuntu is a independent OS from windows so it won't cause any blue screens etc. for windows.
