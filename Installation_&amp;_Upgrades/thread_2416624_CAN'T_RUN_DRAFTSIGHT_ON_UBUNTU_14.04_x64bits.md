### Post by TheFu on 2019-04-12
Welcome.
Support for 14.04 ends any day now. Trying to solve this issue on that release is counter productive.
[https://wiki.ubuntu.com/Kernel/Support](https://wiki.ubuntu.com/Kernel/Support)
Since DraftSight is commercial, contacting that company for support is the best solution. BTW, a company that can't be bothered to use a real HTTPS cert is one that I would trust running any software on my system. There isn't any reason for a self-signed cert these days (free exists and works) and hasn't been for 2+ yrs. A commercial cert is only $20 and they don't have one of those?

Move to 16.04 or 18.04 and work on getting it working there or change to a different CAD package: [https://itsfoss.com/cad-software-linux/](https://itsfoss.com/cad-software-linux/)

According to this: [https://www.draftsight.com/media/system-requirements-draftsight-2019](https://www.draftsight.com/media/system-requirements-draftsight-2019) , 16.04 is the only Ubuntu it is supported on.
