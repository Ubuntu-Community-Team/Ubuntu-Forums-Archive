### Post by StuartN on 2009-04-25
> **piovertwo said:**
> What would be the most straightforward, easy and hassle-free method of upgrading from Ubuntu 8.04 to Ubuntu 9.04?

In System->Administration->Update Manager there should be a selection at the top marked "Upgrade" for the New distribution 9.04.

If this isn't shown, then check in System->Administration->Software Sources under the Updates tab, and set "Show new releases" to "Normal releases".

This should be painless. If you have proprietary hardware drivers or unusual settings / software then you could download the Desktop ISO image, burn it to CD and boot from it in Live CD mode to test how Ubuntu 9.04 recognizes your system before upgrading. You can also install software in Live CD mode to test compatibility.
