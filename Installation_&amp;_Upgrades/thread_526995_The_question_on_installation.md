It should be .iso file, but your WinZip may be able to open it - which is what you DONT want to do. You need to burn the .iso file as an image to CD or DVD. That's an important part - you do not burn it just like any one file, but as an IMAGE file. How you do that, differs depending on your burning software. After burning .iso file and opening the CD/DVD you will see lots of files in it. Then set your computer to boot from CD/DVD and boot from it.
