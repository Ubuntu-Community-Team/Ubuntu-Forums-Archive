After some time I gain control again (Around 2 minutes) and can select another option that will do the same "hanging".

I can't see any loading or anything. It is just stuck'd at the boot screen. And every now and then I can hear the CD spin for 5 seconds and then it slows down again.
I have tried to make multiple copies and download the .iso file from multiple servers but still no luck.

Also I have tried the "Check CD for defects" but again it just hangs. I have tried press F4 and used the "Safe mode", and the same thing happens.
In other words if I select anything it just begins to hang. The only thing that works is the "Boot from first hard disk".

Any idea why this happens?
Brian.
