### Post by Frogs Hair on 2013-06-04
Many applications are community and not Canonical  developed and so bugs need to be reported to and checked for at launchpad . There was a reason Evolution (Gnome Application) was replaced as the default mail client and again Launchpad is the only place where known problems are documented. I don't know of any other site or filter you could use. M.S. Windows had a compatibility page , but it was up to the software developers to test and post if their applications were compatible with new versions. Though code is checked before applications enter the Ubuntu repository, the responsibility for testing lies with the developer/s.
