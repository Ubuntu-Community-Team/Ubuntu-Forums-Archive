### Post by mikewhatever on 2010-09-30
You can't get software with apt-get if it's not available from a repository, and it doesn't look like the version of mantisbt you want is available. Instead, check out the installation instructions:
[http://www.mantisbt.org/manual/manual.installation.php](http://www.mantisbt.org/manual/manual.installation.php)
