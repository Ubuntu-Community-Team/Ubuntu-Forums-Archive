### Post by l_synapse_l on 2009-01-07
I have the Acer 4530 as well but there is no sound coming from my laptop. I boot with the kernel parameter "pci=nomsi" don't know whether it is relevant but you could give it a shot.

I initially installed Ubuntu 8.10 with "acpi=off","pci=nomsi" and "pnpbios=off" as I was getting a few kernel panics. After that the install went smooth.
