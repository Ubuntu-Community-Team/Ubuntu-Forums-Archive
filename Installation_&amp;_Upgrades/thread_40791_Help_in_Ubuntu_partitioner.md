### Post by Martin Witte on 2005-06-10
You can resize your windows partition - if it is not completely filled with data, you need  5Gb - 10 Gb for Ubuntu - with e.g. Knoppix, see e.g. [http://users.tkk.fi/~tkarvine/linux-windows-dual-boot-resizing-ntfs.html](http://users.tkk.fi/~tkarvine/linux-windows-dual-boot-resizing-ntfs.html) for an howto. But you have to be careful: backup everything you do not want to lose before starting with this!
