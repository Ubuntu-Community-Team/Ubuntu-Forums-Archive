### Post by Benchrest on 2005-07-30
Yes.  I am running an IBM laptop with 20gb hd. Mine has windows 98SE and the IBM Windows repair disk forced me to have two partitions under Windows. C ( 1.7 gb) and D ( 17.x gb). C has my operation system and D some programs and data files. First I backed everything of importance up. Then I defrag C & D. In particular to get all the data on D moved to the start of the drive. Then I booted the Ubuntu install disk. Took the simplest install steps. It gave me an option for dual boot. I gave D enough room to not overwrite the tail end of my data. The rest I made into two Linux partitions. The second of less than a gb for swap. I'm only a beginner myself but go slow and it works. Grub installs on the C drive of mine in the boot record I believe to allow dual boot.
