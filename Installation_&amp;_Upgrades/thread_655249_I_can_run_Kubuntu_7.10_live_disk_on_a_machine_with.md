### Post by laxmanb on 2008-01-01
I checked the release notes for Kubuntu 7.10 and I read it requires 384MB of RAM. My computer has 256 MB of RAM and runs Kubuntu 7.04 fine, though the install was slow. If anyone's got a computer with 256MB of RAM, could you tell me if I'll be able to get the Kubuntu live CD running on my computer??

---

### Post by Pumalite on 2008-01-01
To boot your Live CD you need 512 MB of RAM or make a 500 MB /swap partition in your hard drive.

---

### Post by Seisen on 2008-01-01
It will run but it will take a while for it to load and probably seem to act like its unresponsive unless you do what pumalite says and create a swap partition.

### Post by PmDematagoda on 2008-01-01
Actually it would not run that bad. I ran the Xubuntu Live CD on an Intel machine with 256Mb RAM and it worked really well, but it was not terribly responsive, so I would recommend that you use the Live CD **only** to install Xubuntu.
