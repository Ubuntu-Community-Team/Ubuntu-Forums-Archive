I tried the beta version long time ago. If I remember correctly is more like a server version than a independent version. It works very similar to transmission-daemon (which I really recommend BTW). You run the executable on the terminal and access the interface on the web. Something like:
```
http://localhost:8080
```
[Here]("http://forum.utorrent.com/viewtopic.php?id=84274")'s more information.

Note: the best way to leave it running is to use the program 'screen'.
