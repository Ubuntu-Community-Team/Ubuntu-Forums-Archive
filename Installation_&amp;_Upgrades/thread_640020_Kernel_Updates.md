### Post by rsambuca on 2007-12-13
The short answer to your question is NO.  Very few versions changes occur to software packages other than security patches for the most part.

Honestly, if you really like to have the latest versions of packages without waiting, then definitely Ubuntu is not the best distro for you.  I would recommend a Rolling Release Distro, where all packages are continuously updated and upgraded.  The plus side is no waiting for updates, the downside is the occasional breakage.

Some of the more popular Rolling Release Distros are Debian Sid, Gentoo, Arch, and Foresight linux.  There are many others that a web search will find.

### Post by PmDematagoda on 2007-12-13
If you do want to use the latest kernel in an easy way then I suggest that you use [KernelCheck]("http://sourceforge.net/project/showfiles.php?group_id=199755").

But bear in mind, most of the current applications and packages on Ubuntu might not work with the new kernel, so that means you will have to do a fair amount of work to get the programs running well on the new kernel, trust me, I did this on my own Ubuntu OS and I lost the sound system, so right now I am trying to recompile ALSA from scratch which is taking a while.
