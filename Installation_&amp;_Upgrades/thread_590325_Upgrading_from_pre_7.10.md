### Post by ThrobbingBrain66 on 2007-10-24
If you downloaded the Alternate CD, you can use that to upgrade:
[https://help.ubuntu.com/community/GutsyUpgrades?highlight=%28gutsy%29%7C%28upgrade%29#head-93ac2e597b9e0c5ff78111d4fd2bbe34a35799c7](https://help.ubuntu.com/community/GutsyUpgrades?highlight=%28gutsy%29%7C%28upgrade%29#head-93ac2e597b9e0c5ff78111d4fd2bbe34a35799c7)

If you didn't, you have a few options:

1. Get a copy from a wired internet connection
2. Get a friend to download/burn a copy for you
3. Get some help fixing the current problem.

Here's a link to get started:
[https://help.ubuntu.com/community/NetworkDevices#head-a54b0d9ad3156fc55d6af2185e0a2f96f9d5d5b9](https://help.ubuntu.com/community/NetworkDevices#head-a54b0d9ad3156fc55d6af2185e0a2f96f9d5d5b9)
