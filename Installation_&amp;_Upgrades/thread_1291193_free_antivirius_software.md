### Post by themattbeballin on 2009-10-14
Out of all the Antivirus software that I have tried on Linux.. 

AVG 8.5 for Linux seemed to only have a terminal scanning, whereas I prefer a A/V to have a GUI.

ClamAV/ClamTK - I was never able to get it to update.

Avast! Linux Home Edition - Was my best experience... It updated and had a GUI.

The .deb link is the following... [http://files.avast.com/files/linux/avast4workstation_1.3.0-2_i386.deb](http://files.avast.com/files/linux/avast4workstation_1.3.0-2_i386.deb)

But you should make sure you get a Avast! Registration code. Here is the link:

[http://www.avast.com/eng/home-registration.php](http://www.avast.com/eng/home-registration.php)

and then I found a tutorial on the install.... 

Which was here.. [http://www.howtoforge.com/virus-protection-with-avast-on-ubuntu-gutsy-gibbon](http://www.howtoforge.com/virus-protection-with-avast-on-ubuntu-gutsy-gibbon)

I'm well aware that tutorial is for Gutsy. But I have been using that tutorial from Gutsy to Jaunty.

Hope this helps!
