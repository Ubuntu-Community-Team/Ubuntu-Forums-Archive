### Post by Lars Noodén on 2012-05-02
We'll need more information, which package and from which site? But first two comments:

Tarballs usually come with a file called [font=Courier New]README[/font] or [font=Courier New]INSTALL[/font] which contains instructions on what is needed to successfully install the program.  

More importantly, have you checked the Ubuntu software [repositories](http://packages.ubuntu.com/) for your package?  You can use either Synaptic or the Software Center to search for packages.  By using the package manager, you automatically handle things like installation/removal, updates, upgrades and dependencies.  All of those must be done manually if you go and install from a tarball.
