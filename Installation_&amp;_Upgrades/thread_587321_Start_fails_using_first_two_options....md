### Post by rampazzg on 2007-11-20
I had the a similar problem on a Dell Inspiron 4100: first a couple of errors (Buffer I/O error on device fd0, logical block 0), then a system freezed;
after looking at the BIOS Setup (BIOS Version A03) on Page 3 of 7  i changed the Boot Configuration option **Boot POST** from **Minimal** to **Thorough**, and at the next booting from CD the system went up (with a notable delay...), despite the couple of errors remained unchanged (may be this is the cause of the startup's delay...).
No NTFS Hard Disk saw at startup, but when i opened the file browser i have been able to open (may be better mount...) the NTFS volume. No problems at all with the automatic mounting of an USB Memory Stick.

Hope this will help.
