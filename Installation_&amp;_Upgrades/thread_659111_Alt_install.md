### Post by LaRoza on 2008-01-05
The alt disk installs the same Ubuntu as the live disk, but does it with a simpler interface. It is still GUI.

You have a few more options on it than the Live disk, and the alt disk is more reliable as it doesn't require the live disk to work, and it uses less RAM and is faster.
