### Post by sudodus on 2014-02-08
0. Do you ***try*** Ubuntu (trying to run it live without installing) or do you start the installer directly?

1. Have you checked that the download was good? Check the md5sum of the iso file
[URL="https://help.ubuntu.com/community/UbuntuHashes"]
https://help.ubuntu.com/community/UbuntuHashes[/URL]

2. Is it another version of Ubuntu (compared to the last time you installed it)?

3. Is it a new computer (compared to the last time you installed Ubuntu)?

If one of these is different, maybe you need a boot option

[https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

-o-

4. Try ***Unetbootin*** to make the USB drive bootable. There are versions for Ubuntu as well as Windows.

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
