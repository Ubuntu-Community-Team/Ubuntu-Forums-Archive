I assume you are using the installation program from [www.xfce.org](www.xfce.org)  It sounds like you installed it to your local home directory, which is probably not what you want.  If you type in "sudo" (without the quotes) before running the installer (ie sudo install-program-name), you can install it in the proper directory.

Of course, you could just install it through Synaptic, although that might only be 4.2.1 instead of 4.2.2.
