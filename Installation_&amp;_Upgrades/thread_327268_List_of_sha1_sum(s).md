Here is my sha1 sum for my Edgy ISO file fir x86 computers(and for my edgy disc):

a1dc38c7314c7309404ba0ed77ea6041a0f3acf0     ubuntu-6.10-desktop-i386.iso

The MD5 sum was correct and edgy installed properly. Therefore the sha1 sum must also be correct.


How to get sha1 sum (.iso file):[LIST=1]
[*]Open terminal
[*]Type: ```
sha1sum /location/iso_file_name
``` remembering to replace /location/iso_file_name with the location of the iso file. For example:  ```
sha1sum /home/Bob/ubuntu-6.10-desktop-i386.iso
```
[*]Hit enter
[*]the first set of numbers is the sha1 sum[/LIST]How to get sha1 sum (CD):[LIST=1]
[*]Open terminal
[*]Type: ```
sha1sum /dev/*name_of_drive*
``` remembering to replace /*name_of_drive* with the name of the drive the the CD is in. For example: ```
 md5sum /dev/cdrom 

```
[*]Hit enter
[*]the first set of numbers is the sha1 sum[/LIST]Post your md5 sum.
