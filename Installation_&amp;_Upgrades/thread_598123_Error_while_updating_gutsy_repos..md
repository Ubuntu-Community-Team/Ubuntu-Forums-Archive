### Post by zvacet on 2007-10-31
You have mixed repos.
```
gksudo gedit /etc/apt/sources.list
```

and make this line look like this 

#deb [http://10.3.140.50/ubuntu](http://10.3.140.50/ubuntu) feisty main restricted multiverse universe
#deb [http://10.3.140.50/ubuntu](http://10.3.140.50/ubuntu) feisty-security main restricted multiverse universe
#deb [http://10.3.140.50/ubuntu](http://10.3.140.50/ubuntu) feisty-updates main restricted multiverse universe
#deb [http://10.3.140.50/ubuntu](http://10.3.140.50/ubuntu) feisty-backports main restricted multiverse universe

#deb [http://download.tuxfamily.org/3v1deb](http://download.tuxfamily.org/3v1deb) feisty eyecandy
#deb-src [http://download.tuxfamily.org/3v1deb](http://download.tuxfamily.org/3v1deb) feisty eyecandy

#deb [http://archive.ubuntustudio.org/ubuntustudio](http://archive.ubuntustudio.org/ubuntustudio) feisty main


#deb [http://packages.medibuntu.org/](http://packages.medibuntu.org/) feisty free non-free

Save and close.

```
sudo aptitude update
```
