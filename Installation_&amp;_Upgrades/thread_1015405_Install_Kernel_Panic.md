### Post by jeffyoung on 2008-12-18
Hello Everyone, this is a solution to a problem I faced while i was trying to install ubuntu on my old HP Pavilion desktop.  I kept receiving this error when i tried to install:

kernel panic - not syncing : attempted to kill idle task

It took me forever to figure out this simple problem.  It you are receiving this error and don't know what to do about it, instead of installing, run the memory test.  If you get a ton of errors back (I had over 20,000), then the problem is a deep-fried RAM stick.  If you have 2 RAM sticks, trying removing one of them, and see if it will work then.
