### Post by tijmz on 2008-07-07
You should be fine installing all. If you don't want unnecessary downloads, you can remove applications you don't want before updating, using the Add/Remove programs option. That way, you won't upgrade any applications you're not going to use anyway.

Beside that, it is a good idea to install all the updates upon a fresh install, since there have been a lot of fixes since Hardy Heron came out.
