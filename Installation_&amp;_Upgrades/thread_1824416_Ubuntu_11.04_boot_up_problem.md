### Post by drs305 on 2011-08-13
Pacerola

Welcome to the Ubuntu forums.

Either your CD drive is not working correctly or your BIOS is not set up to boot the CD first. Enter BIOS during boot (this entails pressing a key such as DEL, F1, etc) and ensure the system is set to boot the CD drive first.

If it is already set to boot the CD drive first, listen to see if the CD spins up during boot.

If you can get to the Ubuntu desktop, please go to the following site and download/extract/run the boot info script. Post the contents of RESULTS.txt, which will show us the status of your boot files.
[http://bootinfoscript.sourceforge.net]("http://bootinfoscript.sourceforge.net")
