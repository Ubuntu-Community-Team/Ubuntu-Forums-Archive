> **junooni10 said:**
> 1. I clean formatted my netbook & Installed ubuntu netbook remix from my flash drive. So thats the only OS on my netbook at the moment But I really want to install win7 as well.

If you really want to do it you should have done it the other way around. Install Windows 7 first and then Ubuntu at its side. The reason is while Ubuntu (and Linux distributions) have no problem with having more than one OS installed side by side. Windows, obviously, doesn't like it. This doesn't mean you can't do it, just that it's more complicated and needs extra care since is error prone. Nice users already wrote for you how to do it in the [documentation on how to dual boot]("https://help.ubuntu.com/community/WindowsDualBoot#Installing%20Windows%20After%20Ubuntu").

> **junooni10 said:**
> 3. Can I sync my iphone 3gs via itunes 9.0?
iTunes hasn't been released for Linux yet. I guess you can e-mail apple asking them if they could do it. Since you are at it, tell them we'd be most thankful if they also released the source code. In the mean time you have two alternatives. You can
[LIST=1] Install wine (through the Ubuntu software center in the applications menu). This should allow you to install and run some windows programs. Check [Wine's application database]("http://appdb.winehq.org/") to search for which applications work and instructions on to install each specific application.
[/LIST]
[LIST=2] Install windows inside Linux (in a virtual environment) and install iTunes there. Again, you find how to in the [documentation of iTunes in Virtualbox]("https://help.ubuntu.com/community/PortableDevices/iPhone/iTunesVirtualBox")
[/LIST]
[LIST=3] Find open-source alternatives to iTunes. There's in the Ubuntu documentation, stuff about [iPhone]("https://help.ubuntu.com/community/PortableDevices/iPhone"). I must say that I never used an iPhone but I've successfully used gtkpod to sync my iPod in Linux.
[/LIST]

However, I can't stress enough on how you should search for an open-source alternative since, as I once read, Linux (and Ubuntu) shouldn't be just another way to run windows software. Not that I have anything against wine and virtualbox, it's very useful when there's really no alternative and the later is much more than just to install windows.
