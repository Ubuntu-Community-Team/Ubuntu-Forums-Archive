### Post by solar george on 2007-02-18
Try downloading the alternate install cd. It uses a text based installer that is documented [here]("https://help.ubuntu.com/6.10/ubuntu/installation-guide/i386/index.html")

### Post by meng on 2007-02-18
Another vote for the Alternate CD. You need at least 192 MB RAM (and probably at least 256 MB RAM) to run the LiveCD.
