[QUOTE=wEEtoZ]If i reboot computer with the live cd in the cd drive it ask me if i want to install it[/QUOTE] The live CD isn't capable of installing Ubuntu on your hard drive. ### Post by Haegin on 2006-01-28
If you have had cds sent to you by shippit (in a nice case thing with 2 cds with ubuntu logos on them) then you need to put the install cd in the drive. If you use the live cd it will ask some questions and then will load linux running from the cd drive. This does not change anything on the hard drive. It may look like an installer at first bt it wont change your hard drive.

The install cd on the other hand will install linux. It will ask more questions and will change your hard drive (at a certain point - before the "partitioning disks" step u can always wimp out).

If you want to install ubuntu you NEED the install cd (there is no other way). If you just want to try it use the live cd (you cannot damage your system using the live cd unless you do some really odd stuff with it when you first load it up). Also remember the live cd will be alot slower than a proper installation on your hard drive.

Hope this helps.
