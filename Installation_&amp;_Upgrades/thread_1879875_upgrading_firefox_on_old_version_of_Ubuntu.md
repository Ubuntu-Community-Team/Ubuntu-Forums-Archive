### Post by snowpine on 2011-11-12
Welcome to the forums!

You can always get the latest Firefox from [http://firefox.com](http://firefox.com) regardless of which Ubuntu version you are using.

However I recommend you use a supported release (10.04 or newer) so that you get support, bug fixes, and security patches.

You may find that a lightweight desktop environment such as Xfce (Xubuntu) or LXDE (Lubuntu) runs better on older hardware.
