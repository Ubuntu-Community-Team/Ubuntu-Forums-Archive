### Post by kellemes on 2008-02-15
Yes, it seems the image isn't errorfree..
See following link for burning and checking md5 sum.
[https://help.ubuntu.com/community/BurningIsoHowto]("https://help.ubuntu.com/community/BurningIsoHowto")

Also.. the liveCD is a rather big beast, could be too big for your system. You can also try the alternate-installcd, this often seems to work where the livecd fails.
To get it you need to tick the checkbox somewhere at the bottom of the downloadpage.

Let us know if you have questions..

Edit: The time it takes for the system to install depends on the hardware.. but a system that's able to handle Ubuntu will need somewhere around 20 minutes I guess..
