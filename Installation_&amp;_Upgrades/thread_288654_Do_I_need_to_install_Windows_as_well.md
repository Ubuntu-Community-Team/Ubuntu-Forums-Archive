### Post by Rhubarb on 2006-10-30
If you can get the internet connection working from the live CD (before you install Ubuntu), then you'll know you can easily go online once you've installed it.

If you normally connect via wired connection (ethernet cable), then there's a very good chance Ubuntu will detect the network card and will be able to automatically connect you to the internet.

I apologise if I'm not making much sense here, I've only just awoken this morning :-D

-Put the Desktop Ubuntu CD in the drive, reboot your PC, press enter to the 1st option ... wait for a bit until the desktop comes up.
-Check to see if the internet is working, if not you can configure it from the menus:   System --> Administration --> Networking
-If all is good, then install Ubuntu

### Post by frying_fish on 2006-10-30
No the really important thing is how do you connect to the internet, if you go through a ethernet connection to a wired modem / router I would say 99% chance it will work fine and be able to get stuff from online.

If you connect purely wirelessly and don't have the ability to connect wired into some form it may be a bit more difficult to do as not all wireless chipsets are recognised and work "out of the box"

Thirdly, if you use a USB modem it will be slightly different again, but I would recommend finding guides on the particular modem for how to do it first so you have a reference to work with.
