### Post by stychman on 2006-06-30
The alternate-install CD will install Ubuntu without the Live CD part of it.  You could also download the server-install and then update your system to include a graphical interface.  If you decide to do a server install you can take a look at this [http://www.ubuntuforums.org/showthread.php?t=206782]("http://www.ubuntuforums.org/showthread.php?t=206782") on what to do after the server is installed.
