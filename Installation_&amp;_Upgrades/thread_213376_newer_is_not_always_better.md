### Post by nkhansen on 2006-07-11
If I were you, I would try Debian (debian.org), since it is much better geared towards low-end hardware.

If you really need ubuntu, maybe you should try xUbuntu (Ubuntu with a XFCE desktop). Older distros not always perform better. 

Alternatively you could do a server-install, and add the things you need.

EDIT:
To do a server install with Dapper Drake, you need the alternate CD instead of the desktop one.
