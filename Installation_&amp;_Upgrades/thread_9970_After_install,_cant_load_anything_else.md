### Post by wallijonn on 2005-01-03
Networking:
Computer -> Sysytem Configuration -> Networking -> add

Printer:
Computer -> Sysytem Configuration -> Printing -> Add Printer

Hopefully your printer will be on the list.

You are probably trying to install a Windows game onto Linux. There is a help file / faq on getting video drivers installed along with installing Wine. Use the search function, or go to [http://ubuntuguide.org/index.html](http://ubuntuguide.org/index.html)

There should be a thread concerning accessing Ubuntu from Windows.
