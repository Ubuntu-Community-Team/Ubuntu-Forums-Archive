### Post by mackyman on 2007-02-04
I have about the same problem, but I think I might have figured out what the problem is.

Older laptops (and computers in general) tend not to take burned CD's too good ( they may take them a bit better if they are burned with as low speed as possible ).

You might get a better result by using a alternative CD, as that will reduce the aumont of used system memory during install ( less graphics on old computers == love ).

Hope this helps atleast a bit.

//Macky
