### Post by Partyboi2 on 2009-04-12
Looks like you need to add a # in front of the cdrom line in your /etc/apt/sources.list or open up Software Sources (System>Admin>Software Sources) and on the first tab make sure that the "installable from cdrom/dvd" is unticked..
