### Post by 2F4U on 2012-04-12
Is the BIOS seeing the hard disk? To me it looks as if either your hard disk isn't connected properly or died. The message PXE-E61 Media Test Failure, Check Cable is an indication that the BIOS is not able to find a boot device and, as a last resort, attempts to boot from network (PXE boot).[SIZE=2]
