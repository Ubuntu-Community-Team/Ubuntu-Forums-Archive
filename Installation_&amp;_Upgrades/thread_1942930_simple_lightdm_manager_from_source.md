since 12.04 doesnt have simple lightdm manager in its repo ill make it so i can set that stuff.

nab this file with your browser save to $HOME/Downloads

[https://launchpad.net/~claudiocn/+archive/slm/+files/simple-lightdm-manager_0.2-public7.tar.gz](https://launchpad.net/~claudiocn/+archive/slm/+files/simple-lightdm-manager_0.2-public7.tar.gz)

then alt + f2

gnome-terminal

run

then 

```

sudo apt-get install python-distutils-extra python-launchpad-integration

```

then

```

cd $HOME/Downloads
tar -xf simple*
cd simple*
cd bin
./simple*

```
