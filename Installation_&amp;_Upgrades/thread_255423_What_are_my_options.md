### Post by ajgreeny on 2006-09-11
1  Download the iso file to the computer on the net then copy the iso to the usb hard disk.

2  Take the usb disk to the computer with the burner and burn the disk image, (don't just copy the file to the CD, but make sure you burn a disk image)

3  Use the burned disk to boot into live ubuntu and then install.  I assume the computer without CD burner has a CDRom drive, just no burner.

Alternatively, of course, if you don't mind swopping CD drives, you could just swap them over and do the job without bothering about the usb drive.
