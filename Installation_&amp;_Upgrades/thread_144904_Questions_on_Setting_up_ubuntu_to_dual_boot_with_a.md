Hmm I don't know if such a lists exist - but hey try the LiveCD and then you'll see if it works or not. Just try out everything that you think you'll do with your Ubuntu install, like burning CDs, listen to music, plug your USB stick or camera in, surf the web etc. The LiveCD is identical to the installed version of Ubuntu, so when you install it later, you can be pretty sure that everything works as you expected.
