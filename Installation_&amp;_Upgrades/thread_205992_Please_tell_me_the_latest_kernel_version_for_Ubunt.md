### Post by 23meg on 2006-12-01
It may help if you indicate why exactly you'd like to upgrade your kernel. 

The latest Ubuntu kernel available will only be available for the development version (currently Feisty); stable releases won't get new kernel versions (security updates only). The latest in Feisty is 2.6.19-7. 

You can always compile your own kernel from kernel.org, and if that's what you want to do, the latest stable version is 2.6.19.
