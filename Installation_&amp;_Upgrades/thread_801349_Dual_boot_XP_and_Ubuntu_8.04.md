### Post by tamoneya on 2008-05-20
assuming that windows xp is already on the 20 GB drive all you need to do is pop in the CD and during install when you are prompted for where to install tell it to us "full disk" and select the 40 GB disk.  Ubuntu unlike windows is smart enough and nice enough to set up the dual boot for you.  Nothing else you have to do at the point except sit and let it complete the install.  Its a lot easier than it used to be.

### Post by jtrindle on 2008-05-20
Install XP first.  It's probably easiest if the disk you want XP on comes up as your C:\ drive.

Once XP is happily installed, install Ubuntu.  It should find XP and add it to the Grub menu, allowing you to select either OS on boot.
