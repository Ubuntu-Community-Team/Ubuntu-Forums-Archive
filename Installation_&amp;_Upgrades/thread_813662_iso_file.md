You don't have to extract it yourself. Use Brasero (or any Linux Burner you have installed) to burn it to disk. If you're using Brasero, choose the "Burn Image" option.

Just be sure that you checked the MD5 hash of your ISO file before burning it. Use a lower burn speed (4x) maybe.
