Dual booting with Vista is actually very easy, however, there are a couple of important points.  First, make sure you resize the existing Vista partition using the utility in Vista.  If you use the ubuntu partitioner or gparted, Vista will not boot afterwords.  Other than that, just follow the instructions on the ubuntu installer and you should be good to go.
