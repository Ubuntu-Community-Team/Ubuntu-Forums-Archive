### Post by bertan2 on 2014-01-03
UEFI is just the new replacement for the traditional BIOS.

In your UEFI/BIOS, turn Secure Boot off and select Legacy rather than UEFI mode.

25GB is fine for Kubuntu. If you want to practice without affecting your real computer, download and install VirtualBox and practice the install process there. At a certain point, it should give you a choice to install your new operating system alongside any existing operating system.
