### Post by Mark Phelps on 2009-06-16
If you're going to use Wine, you need to familiarize yourself with the CodWeavers compatibility database, which lists the results of testing different apps. The page below shows the results of AIMP2 as "untested", meaning, no one knows if it works or not, and there are no aids available to assist:
