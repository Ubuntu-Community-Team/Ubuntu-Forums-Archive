### Post by jken146 on 2008-04-02
I guess you're referring to downloading the source of a program and compiling then installing it yourself.  This can't be automated very well.

In most cases, you'll find what you need in the repositories or in binary form somewhere like [http://getdeb.net](http://getdeb.net)

### Post by logos34 on 2008-04-02
There's usually a .deb available somewhere, like jken146 said.  But if not, installing from .tar is usually not much harder than running

./configure
make
sudo make install

[http://monkeyblog.org/ubuntu/installing/#source](http://monkeyblog.org/ubuntu/installing/#source)
