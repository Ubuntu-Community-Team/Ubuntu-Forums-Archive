### Post by sudodus on 2014-11-02
It might work well, and there might be issues depending on what tweaks there are. I'd suggest that you have a common data partition with personal files (documents, pictures, video-clips etc). Then the home directories (or partitions) will be rather small, and it will be no problem to have two of them (one for each version). You can install the new version to a ***copy*** of the current home (directory or partition) and inherit a lot of settings. Possible conflicts now can be fixed and can be avoided in the future.
