### Post by jasmuz on 2007-11-10
Hey there!!

If what you are looking for is installing GNU/Linux in your iPOD as an alternative to the firmware Apple gives you by default, then check this out: [IPodLinux]("http://ipodlinux.org/Main_Page")

But if you wanted to know if you can use your Ipod as a mass storage device in order to boot GNU/Linux off it, then sunny.. IT WONT WORK!!

The reason behind this, is that the original Ipod system is mounted on an HFS partition wich dosent allow to be used as a conventional memory device.

I hope this helps you out a tad.
