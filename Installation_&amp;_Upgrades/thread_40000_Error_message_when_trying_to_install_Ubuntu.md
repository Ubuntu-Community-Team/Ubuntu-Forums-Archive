### Post by C.B. on 2005-06-07
If the VM thing is too tricky, you could try sticking a 2nd (cheap) hard drive in your machine and install Ubuntu there. You only need about a 1 G drive. Window stays as is on your main drive, and you don't have to worry about partitioning issues, etc.

That's what I did. I'm new to Linux and not a computer whiz, and everything installed fine. Only post-install printer and sound issues remaining.
