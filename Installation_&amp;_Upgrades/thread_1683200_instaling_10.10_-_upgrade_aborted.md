### Post by mikewhatever on 2011-02-07
The url you posted gives the 404 file not found error. Apparently something is wrong with that particular mirror at the moment. Either wait for a few hours, then try again, or try a different mirror.
You can select a different mirror (out of about 300) in the Software Center, Edit->Software Sources.
