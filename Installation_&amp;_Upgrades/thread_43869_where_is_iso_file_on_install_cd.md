### Post by netrattler on 2005-06-23
If you downloaded from the link on there website it's already in as and ISO image file [http://us.releases.ubuntu.com/releases/5.04/ubuntu-5.04-install-i386.iso](http://us.releases.ubuntu.com/releases/5.04/ubuntu-5.04-install-i386.iso) , From Nero all you should have to do is exit the wizard and click on Recorder -> burn image -> and select the ubuntu-5.04-install-i386.iso file -> click burn.
