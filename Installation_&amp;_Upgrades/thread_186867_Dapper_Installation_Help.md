### Post by orev on 2006-06-02
You should be able to run the installer inside either operating system.  I don't have any reference on which operating system (Breezy or WIN98) is lighter on the RAM usage, but I would try booting into each OS to see if the installer will fly....

If it doesn't work on either OS, are you sure the CD is good?  Maybe try a re-burn at a lower speed if possible....to make sure you have a good copy.

My next advice would be to use the text-based installer....which isn't really very hard, but you'd have to download another disk image.

I have installed Dapper on a 600mhz machine with 128mb of RAM using the text based installer....so I'm sure it can be done.

Hope this helps.
