### Post by snowpine on 2010-08-01
Hi m138, you can easily resize your partitions, a reinstall is not necessary. :)

1. Back up all data on Windows and Xubuntu to an external drive (just in case... better safe than sorry!)
2. Defragment your Windows partition
3. Boot with an Ubuntu or Xubuntu Live CD
4. Use the Gparted Partition Editor (in the System menu) to resize the partitions

Good luck!
