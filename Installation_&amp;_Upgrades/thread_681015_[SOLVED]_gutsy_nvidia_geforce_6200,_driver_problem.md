### Post by TBD on 2008-01-31
i've solved it...

it was this bug: [https://bugs.launchpad.net/ubuntu/+bug/107646](https://bugs.launchpad.net/ubuntu/+bug/107646)


to avoid the API mismatch (hence loading only the nvidia-glx-new -who is hovewer the only installed- ):

**sudo nano /etc/default/linux-restricted-modules-common**

(or the editor of your choice)

and set this entry:
[B]
DISABLED_MODULES="nvidia nvidia_legacy"[/B]

i wonder why this bug hasn't yet been solved...
