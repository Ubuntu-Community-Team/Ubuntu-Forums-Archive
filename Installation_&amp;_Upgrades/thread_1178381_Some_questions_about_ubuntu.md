### Post by drs305 on 2009-06-04
Hi welcome to the ubuntu forums.

If you are the one installing the system, you will will have access to all the files on the system - that is you will have access to all permissions.  If that isn't what you are asking, please restate the question. You will be able to see your Windows files with just a bit of work setting it up.

You won't use Internet Explorer on a normal installation of linux. Many use the default browser, Firefox, which can import your bookmarks from Internet Explorer.

You can try the Ubuntu LiveCD without installing ubuntu. Just boot it up and elect to try Ubuntu without changing your system.

[https://help.ubuntu.com/9.04/installation-guide/i386/index.html]("https://help.ubuntu.com/9.04/installation-guide/i386/index.html")
