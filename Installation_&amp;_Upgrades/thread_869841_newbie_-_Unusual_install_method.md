### Post by Potatoj316 on 2008-07-25
About your bios password, it is possible to get back into your bios but its difficulty can varry.  If you have a laptop (which you do right?) it might be harder.  You can try to remove the CMOS battery which will reset defaults (mine did this, but it reset the language to chinese) or you could go looking for backdoor passwords.  There are a lot of possible backdoor passwords but they might not work on a laptop (laptops are made to be more secure) you also could get the ultimate boot CD (UBCD) which as some bios password cracking programs but because you cant boot from CD this might not help.  If you have a floppy drive (probably not) you can get someting on your floppy drive that allows you to boot from something other than the HD (you need to be able to boot from floppy for this)
