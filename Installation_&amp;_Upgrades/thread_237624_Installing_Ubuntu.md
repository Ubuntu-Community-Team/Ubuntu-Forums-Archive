### Post by randell6564 on 2006-08-16
> **Seldoms said:**
> Once I install Ubuntu, will I be able to keep windows on my pc along with Ubuntu?

How will I choose which one to boot up in?

How much memory will this take up?

Yes, it's called 'dual-booting'.  But if windows is the only OS on your box right now, and you only have one HD, then you will need to resize your windows installation and partition a space for ubuntu.

On the other hand, if you have two seperate HD's then just stick the ubuntu disk into your cdrom and it will recognize windows and give you options for installing. for dual-booting make sure to install Grub to your MBR.

As far as "memory", They will be two seperate OS's. They will not be sharing resources as you can only use one OS at a time.
