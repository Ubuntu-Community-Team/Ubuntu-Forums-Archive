### Post by Mark Phelps on 2010-06-15
> **kasei said:**
>  ... Should I uninstall Windows 7 starter and then install Ubuntu or should I try to install it side-by-side with Windows 7 OR do a trial version?... 
Since you might, at some point in future, wish to have Win7 back, I would not remove it from your PC.  You probably do NOT have a reinstallation DVD, since those are not provided anymore.  Instead, I would recommend you dual-boot having both Win7 and Ubuntu.  But if you decide to do that, be sure to use the Win7 Disk Management utility to shrink the Win7 OS partition to make room; do NOT use the Ubuntu installer to do that.

> How much space will it take up? 
That's entirely up to you, but 30GB provides LOTS of room.

> Will it slow the computer down?
No.  However you install, each OS will be running on its own.
