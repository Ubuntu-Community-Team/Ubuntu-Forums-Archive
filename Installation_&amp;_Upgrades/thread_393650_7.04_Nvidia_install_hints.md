### Post by dcstar on 2007-03-26
I just installed the 7.04 Beta desktop system (full install on a fresh partition), and found that it installed the basic VESA driver for my (old) Nvidia video hardware.

After I downloaded the nvidia-legacy package and then ran:
```
sudo dpkg-reconfigure xserver-xorg
```
to set it up, I found that it wasn't auto-detecting my video hardware.

I subsequently found that the package required to do hardware detection is not installed by default in this release, so you will need to install the **discover** package.

Once this was installed my video hardware was detected ok, and I was able to get it all set up correctly.

The release seems ok, (apart from the **nvu** package no longer existing in the repositories.....), and I am looking forward to using it over the next few weeks to see if any other issues arise.
