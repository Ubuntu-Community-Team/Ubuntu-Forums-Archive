### Post by stoneheart on 2009-03-23
I got the idea from this blog post:  [http://tuxoblog.blogspot.com/2007/06/ubuntu-light.html](http://tuxoblog.blogspot.com/2007/06/ubuntu-light.html).  I used a server install cd of Hardy Heron and followed the directions provided.

sudo aptitude install x-window-system-core gnome-core gdm gnome-media gnome-system-monitor gnome-system-tools gnome-volume-manager gnome-utils gnome-app-install gnome-screensaver synaptic firefox usplash usplash-theme-ubuntu ubuntu-artwork

The x-window-system-core package is now named xorg, but everything else worked.  Then I added the fast-user-switch-applet package to get rid of the sole annoying error message I got on login to the Gnome desktop. 

The resulting system after adding the multimedia goodies like Flash and DVD libraries is screaming fast on my IBM Netvista P4 1.8 gigahertz with 1 gig ram.  Youtube works fine, and commercial DVDs play readily.  Not bad for a free computer I got from work.  I'll give the system to my little son and let him cut his teeth using a PC on it.

Anyway, this is the way to go instead of running Xubuntu.  I much prefer Gnome over XFCE, but there's a lot of bloat in the gnome-desktop package I will never use.  This way gives me the best of both worlds.
