### Post by dstew on 2008-05-21
First you download the .iso file to your hard disk. Then you use Infrarecorder to burn a disk image, using the .iso as the source file. Select Action --> Burn Image in the Infrarecorder menu, and select the downloaded .iso to burn. Use a slow burn speed, 4x at most.
