### Post by snowpine on 2008-08-19
> **gzambran said:**
> Can I customize what applications will be installed with the Standard Ubuntu Installation disc? If not, how can I do this?

Two ways (that I know of) to do this. One is to install a minimal command-line system, then add the apps you want (see [http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)). The other is to set the system up how you like, then use Remastersys to create your own custom Live CD.

Good luck!
