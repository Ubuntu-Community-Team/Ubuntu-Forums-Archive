### Post by catlett on 2006-06-22
That command installs ubuntu/gnome. Are you changing sessions during login? In the bottom left hand corner there is "Options" Hit on that and choose "change session". Slect "gnome". Login as usual.

It will ask if you want to use gdm as default session manager. You can deny it to keep Xubuntu's session manager or chabge to ubuntu. Also it will ask if you want gnome as the default session or just this session. Again accept or deny depending on if you want gnome as the default or not.

If it didn't install you may have a repository issue but I doubt ubuntu-desktop wasn't available. If it isn't because of changing sessions, post ther output of this ```
sudo mousepad  /etc/apt/sources.list
```

### Post by catlett on 2006-06-22
You have all the UBuntu repositories. You gave the correct command. Let's try installing with synaptic package manager. I haven't been in Xubuntu in a while. "Is synaptic package manager in the menu?" If so open synaptic.
If it isn't in the menu enter this in  the terminal```
 gksudo synaptic
``` When synaptic opens hit on "search". Enter ```
ubuntu-desktop
``` in the search box. It will do a search and list "ubuntu-desktop". Right-click on it and select "mark for installation". Then hit on apply at the top. Let synaptic run the installation.
Log out. On my login page (I am running Dapper) there is "options" in the bottom left. When I hit on that a menu pops up. One of the choices is "Select Session". When I select that a window pops up with options. Mine says 
```
Last Session
Default System Session
Gnome Session
Failsafe Gnome
Failsafe Terminal

CANCEL         CHANGE SESSION
```Your menu should have "XFCE" in it as well. Select Gnome and hit on "change session". This will switch you to Ubuntu which is Gnome.

If this doesn't work, try to post any errors you are getting.
