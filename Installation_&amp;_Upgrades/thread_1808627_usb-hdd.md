### Post by ajgreeny on 2011-07-20
There are no separate usb .img files available for ubuntu.

Use the standard .iso image file and copy it to the flash drive with startup-disk-creator or unetbootin.

If this is a normal usb external hard drive, however, you can use the normal installation method, but choose the correct /dev/sdb, or whatever it is, when you get to the disk preparation page of installation.

Come back again if this does not make sense to you.
