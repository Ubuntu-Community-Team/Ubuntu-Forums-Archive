### Post by kavenien on 2006-04-30
.ISO is a disk image format. You need to burn it off on a CD with whatever software you have. If you have Windows XP and a CD-R or better drive, but no extra burning software, you can use [ISO Recorder]("http://isorecorder.alexfeinman.com/isorecorder.htm") with the native burning capabilities. Other OSs should already support burning ISOs.
