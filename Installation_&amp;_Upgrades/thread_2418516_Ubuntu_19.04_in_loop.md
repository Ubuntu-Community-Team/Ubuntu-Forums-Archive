### Post by oldfred on 2019-05-08
If you just run the report, it makes no changes to system.
And with two installs best to let someone review it, before. 
Sometimes better to only use its advanced mode so you have more control over what changes it does.
With one install or dual booting on same drive as Windows, then Boot-Repair works well. But multiple installs or multiple drives usually need advanced mode.
