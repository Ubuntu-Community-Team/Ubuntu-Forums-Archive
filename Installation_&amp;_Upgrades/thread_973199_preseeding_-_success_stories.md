### Post by GarryOwen on 2008-11-06
You can use kickstart with Ubuntu. 

For hardy - grab the mini.iso from [http://archive.ubuntu.com/ubuntu/dists/hardy/main/installer-i386/current/images/netboot/]("http://archive.ubuntu.com/ubuntu/dists/hardy/main/installer-i386/current/images/netboot/")

and boot it with the parameters:
root=/dev/ram ks=http://192.168.0.10/configs/hardy.cfg

The layout of the kickstart file is the same as for RedHat/Fedora
Sample one below:

```
#Generated by Kickstart Configurator
#platform=x86

#System language
lang en_AU
#Language modules to install
langsupport en_US --default=en_AU
#System keyboard
keyboard us
#System mouse
mouse
#System timezone
timezone Australia/Brisbane
#Root password
rootpw --iscrypted $1$Y9rdhdfhjshddfskilCOvZ.jdXsz/
# Initial user: test 
# Password: test
user test --fullname "" --iscrypted --password $1$d2Lsjh4Bz$fd3C768iFz8poqI.BfcJDG1
#Reboot after installation
reboot
#Use text mode install
text
#Install OS instead of upgrade
install
#Use Web installation
#url --url http://archive.ubuntu.com/ubuntu/
url --url http://192.168.0.10/ubuntu/

#System bootloader configuration
bootloader --location=mbr 
#Clear the Master Boot Record
zerombr yes
#Partition clearing information
clearpart --all --initlabel 
#Disk partitioning information
part swap --size 512 
part / --fstype ext3 --size 1 --grow
#System authorization infomation
auth  --useshadow  --enablemd5 
#Network information
network --bootproto=dhcp --device=eth0
#Firewall configuration
firewall --enabled --ftp --ssh --telnet --port=21:tcp
#X Window System configuration information
xconfig --depth=32 --resolution=1024x768 --defaultdesktop=GNOME --startxonboot
#Package install information
%packages
@ ubuntu-desktop
grub


```

Although for intrepid, kickstart doesn't seem to work, but preseeding is fine. Let me know if you need more info on that

Cheers
