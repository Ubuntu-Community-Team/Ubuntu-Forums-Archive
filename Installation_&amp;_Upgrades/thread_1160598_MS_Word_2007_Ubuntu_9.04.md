Anthony, 
If you really need this, your best bet is to install VirtualBox and use it to create a Windows (of your choice) virtual machine. Then you can install Word 2007 into that virtual machine and set the virtual machine to seamless mode so you can copy and paste back and forth between ubuntu and windows and have Word 2007 "appear" to be like a linux application.

Requires you to have a license of Windows to install, which I assume won't be an issue if you already have a license for Word 2007 / Office. If you google or check youtube you'll find lots of videos for how to do the Virtual Box part. 

Good luck.
