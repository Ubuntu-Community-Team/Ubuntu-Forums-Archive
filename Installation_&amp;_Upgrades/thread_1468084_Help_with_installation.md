### Post by tommcd on 2010-05-01
> **Pycnopodia said:**
> 
I am trying to install the netbook version of the new ubuntu. I have  tried everything it has suggested on the homepage, plus using a external  CD drive to try and install it. But it has never occurred.
Can you post a link to the instructions you are using? Also post whatever errors you may be getting, or the specific place where the install fails. Describe what you did.
If you have an external CD drive, and are able to create an install CD, that would be the easiest way to go. Be sure to change the boot menu in the BIOS of the computer so that the cdrom drive is selected as the *first boot device* in the BIOS.
To install UNR from USB stick:
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
The UNR iso image can be downloaded from here:
[https://help.ubuntu.com/community/Installation/FromUSBStick#Ubuntu%20CD%20or%20ISO](https://help.ubuntu.com/community/Installation/FromUSBStick#Ubuntu%20CD%20or%20ISO)
[http://www.ubuntu.com/getubuntu/download-netbook](http://www.ubuntu.com/getubuntu/download-netbook)
To burn the image to a CD for install:
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Be sure to burn the CD at the slowest possible speed. When you first boot the CD, be sure to run the option "check disc for defects" to make sure the CD is good.
I have never installed UNR, but there should be an option during the install to shrink your Windows partition to install UNR along side of Ubuntu.
