First off, I would have thought the default for the installer would be to set the machine up for dual boot, not so. IT wanted to wipe everything. So I went to manual setup. This is when the real problems started.

It recognised two NTFS partitions already on there but I could see no way to resize them, and new way to even create a new one. My options seemed to be to overwrite them or go back to the "guided" option. Which I did in the hope that some "behind the scenes way" it would resize everything and put Ubuntu alongside Vista, but it did not and simply wrote over the top of it :(

Also, the new import settings+files part of the installer could not find any Vista profiles. Maybe this is well known already?

My machine is fine, I backed everything up first but would like to know how to go about it differently next time. Is what I'm trying to do not possible or did I miss something?

cheers guys!

I have vista dual booted with feisty
what I did was just create a partition with vista's partition manager in the control panels
if I were on vista at the moment, I'd tell you exactly where it is
but you'll have to go into classic view and select something that says administrative something (forgot what it's called)
and you can go to the disk some where on there to resize your vista partition to make enough room for ubuntu
