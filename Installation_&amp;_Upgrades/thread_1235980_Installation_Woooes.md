### Post by running_rabbit07 on 2009-08-09
Do you have the ability to boot said laptop? If yes then try [Unetbootin]("http://unetbootin.sourceforge.net/"). 

If no then you may have to find a docking station that has a CD ROM drive or if the BIOS will allow it an external CD ROM drive. Otherwise you will have to do a network boot which will take a little skill. 

I haven't been able to set that up yet but I have a dead laptop that has a dead cd rom and the most recent install locked up and all is lost until I get the network boot set up.

Good luck.

Edit: What are the specs on the system, so we can help make sure you get the best release?  RAM/CPU and GPU are really important roleplayers in deciding which install will be best.
