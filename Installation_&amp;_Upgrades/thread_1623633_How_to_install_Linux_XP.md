As long as your image size is less than 4.7GB, you can burn it on a regular DVD. Even if it is a CD image. Although that'll be a waste of a DVD to burn a cd image on it. If the size is greater than 4.7GB, then you need to get a dual layer DVD with the burner capable of burning it. Dual layer DVDs have around 7.2 GB capacity I think.
