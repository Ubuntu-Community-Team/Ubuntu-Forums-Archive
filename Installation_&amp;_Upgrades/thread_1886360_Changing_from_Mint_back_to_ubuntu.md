### Post by fantab on 2011-11-24
> **Borg said:**
> Dual boot set up with Win 7 and Mint, don't get on with Mint and want to replace it with Ubuntu again. Is it just a case of **running the Ubuntu set up and installing in the same partitions I've set up for Mint, after formatting them, or will this not work with Grub ?**


When you've booted with Ubuntu LiveCD go to **Disk-Utility** and format Mint Partitions, you don't, however need to format SWAP. Reinstall Ubuntu as you normally do. There won't be any issues with Grub.

You are good to go.
