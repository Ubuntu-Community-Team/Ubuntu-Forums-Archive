Hi,

I've just done a clean install of 12.04 from my old 10.04LTS.  The "Home" is a separate partition and was not touched, and is still all there.  My own account is working fine, and I can get root if I need it.  But the other users were not set up during the install, and when I add them now, the original password does not work.  
I'm not sure if I'm doing this right; in "System Settings" "User Accounts" I unlock and then set up a new user (Add User) with the original name, and it gets created, never asking me for a password.  But when I go to log in as that person, the original password is rejected.   What am I missing please?

Thanks for any help.

### Post by momist on 2012-04-28
OK, I'm dumb, or maybe too old.

I've found the answer, when the new user is created there's a box to the right which announces that the user is disabled.  If you click that it drops a menu allowing the entering of a NEW password.  Simply set that to the old password and the user will be able to log in and access their existing files.  Simples.  :P
