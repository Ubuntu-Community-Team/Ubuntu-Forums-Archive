Hi TheNad,

you can create a new primary partition, format it as ntsc, then go ahead with the install.  This may break your grub for booting ubuntu so follow this how to on how to recover after the install: [https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows)
