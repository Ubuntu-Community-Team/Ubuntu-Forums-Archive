### Post by cariboo on 2008-12-26
Did you try starting the LiveCD in safe graphics mode? If not at the menu press F4 and select safe graphics mode. This will have no affect on the installation, but at least you should be able to boot to the desktop and install from there. I haven't needed to boot in safe graphics mode for that last 4 releases, but after the installation everything works the way it should.
