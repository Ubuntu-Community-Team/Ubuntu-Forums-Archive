### Post by sudodus on 2016-08-28
I tried Multisystem maybe some years ago and it worked for me. I don't know how to plays with 16.04 LTS. If you have problems with French (like me), maybe you should try another tool to create a multiboot system.

If you intend to use it for many distros, also distros that are very different from Ubuntu and Debian, I suggest that you try some other tool, for example

- Multiboot-USB
- YUMI (from Windows)

If you will use it only for Ubuntu, flavours and re-spins based on Ubuntu, you can try

- [One pendrive for all PC (Intel/AMD) computers - single-boot dual-boot multi-boot](https://ubuntuforums.org/showthread.php?t=2259682)

You can also make it yourself according to the following general 'grub-n-iso' links

[Arch: Multiboot_USB_drive](https://wiki.archlinux.org/index.php/Multiboot_USB_drive)

[https://help.ubuntu.com/community/Grub2/ISOBoot](https://help.ubuntu.com/community/Grub2/ISOBoot)

[Grub2/ISOBoot/Examples](https://help.ubuntu.com/community/Grub2/ISOBoot/Examples)
