### Post by labinnsw on 2009-06-07
If the Ubuntu CD is the live CD, you can select the option to Try Ubuntu, (Boot into the Live CD) and from there use the partition editor (System >> Administration >> Partition Editor) to delete the Ubuntu partition and then reboot and try installing again.
