### Post by Hedgehog1 on 2011-04-28
For best results, use the Upgrade feature from the LiveCD/LiveUSB.

[IMG]http://img848.imageshack.us/img848/874/upgradetonatty.png[/IMG]

This gives you an upgrade that with will finish even if the Ubuntu servers get swamped, and gives you a repair tool if you need it later.

**Please backup your documents/data before you do any install or large upgrade.**

Also - you can load Natty/11.04 and still use the 'Classic&#8217; UI: [**_Natty Info: Your UI options_**]("http://ubuntuforums.org/showthread.php?t=1741293")

There is no rush to upgrade; so plan it for when you have a bit of time.
