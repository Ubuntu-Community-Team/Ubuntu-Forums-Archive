### Post by sudodus on 2014-01-03
Is it important with a CD? In that case you can [download the Ubuntu ***mini.iso***]("https://help.ubuntu.com/community/Installation/MinimalCD") and from that install xubuntu-desktop. If you don't find it as a menu option, you can use the terminal command

```
 sudo apt-get install xubuntu-desktop
```

### Post by Bucky Ball on 2014-01-03
The minimal install installs the 13.10 latest kernel. That is all. You install the rest. Installing xubuntu-desktop is pretty much equivalent to then installing Xubuntu into the minimal. That's it.

You can go even more stripped back and just install xfce4 (Xubuntu's desktop environment) and then only the apps you want/need. That's my setup. Zippy.
