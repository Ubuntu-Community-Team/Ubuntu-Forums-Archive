The alternative installer is not a liveCD. But for any system windows, Ubuntu or other, you should have a liveCD just to do repairs or boot another way to check system.

You can also download the gparted liveCD which will let you  see partitions on drive.

From a live Linux CD run this.

```
sudo fdisk -l
```
