### Post by SectionThree on 2006-03-02
If you're running KDE, please note that I'm running Gnome and have next to zero experience with KDE. With that said...

How to disable passwords completely:

1. Open up Login Screen Setup in the System/Administration menu

2. In the General tab, check the "Login a user automatically on first bootup" option and select the user from the menu below.

3. Hit the close button and you're done.
