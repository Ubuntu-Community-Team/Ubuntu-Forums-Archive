### Post by sikander3786 on 2010-09-16
Just install Ubuntu and it should automatically pick up Windows 7 boot loader thus enable the dual boot for you.

The default boot loader included in Ubuntu Lucid is GRUB2.

[https://help.ubuntu.com/community/Grub2](https://help.ubuntu.com/community/Grub2)

If you prefer a Windows based boot loader, try EasyBCD (still GRUB is way better than it).

[http://neosmart.net/dl.php?id=1](http://neosmart.net/dl.php?id=1)
