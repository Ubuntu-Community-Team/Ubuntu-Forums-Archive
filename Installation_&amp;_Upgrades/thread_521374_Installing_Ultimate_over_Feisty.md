### Post by Irony on 2007-08-09
If you are a bit vague on how to do this then boot up into windows and find computer administration (I think it is in accessories) and delete you current ubuntu and swap partitions.

Put your live CD in and shut down (don't reboot or it will hang) and start up and then install to the freespace (the deleted partitions).
