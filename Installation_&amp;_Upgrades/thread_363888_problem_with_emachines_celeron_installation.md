### Post by mikewhatever on 2007-02-17
The little thing/pin, as you call it, should be set the according to manufacturer's instructions. If it's not, the CD-Rom may not work at all, not just booting from it. The operating system and the BIOS will not see it. That's as far as the jumper issue. 
To boot from a CD-Rom that works, you have to access the BIOS, find the boot sequence, and make sure the CD-Rom is before the hard drive, then save and exit.
