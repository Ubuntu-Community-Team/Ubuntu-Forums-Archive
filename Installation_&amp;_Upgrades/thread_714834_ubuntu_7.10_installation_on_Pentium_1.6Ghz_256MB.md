Boot the cd and pick the "check CD for defects" option.

Also, you should be able to get through the install with 256 megs or ram.  However, is any of your ram allocated to an onboard video card?  If so, decrease that amount so that you can free up some more ram.  

You will really be more happy with more ram, though.  Is there any way you can buy another stick of ram?
