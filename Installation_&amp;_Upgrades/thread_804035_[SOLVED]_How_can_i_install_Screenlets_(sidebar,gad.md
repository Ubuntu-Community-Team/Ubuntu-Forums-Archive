### Post by damis648 on 2008-05-22
In the panel:
System>Administration>Synaptic Package Manager
Search for "screenlets", when you see it, double click it. Click apply if a window pops up. Click apply at the top, click ok and screenlets will install. Just configure them from System>Preferences>Screenlets.
Download screenlets from [http://gnome-look.org/?xcontentmode=6700](http://gnome-look.org/?xcontentmode=6700)
