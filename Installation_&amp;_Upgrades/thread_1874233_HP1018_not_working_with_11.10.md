### Post by Seb71 on 2011-11-08
It seems that the server from which the plugin has to be installed is down.

The plugin can still be found on various sites on the Internet. You need to search for "hplip-3.11.7-plugin.tar" archive (containing "hplip-3.11.7-plugin.run" and "hplip-3.11.7-plugin.run.asc").

After downloading the archive, extract the files and install the plugin from downloaded files.
