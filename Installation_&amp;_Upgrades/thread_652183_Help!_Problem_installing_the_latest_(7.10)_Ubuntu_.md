From the boot up menu on the live cd, select the option to verify the media.

Before attempting to install it, you should make sure your media is valid.

If the media passes the verification test, then it is probably safe to continue with the installation process.

Boot the LiveCD and then select the first option for installation.

Once it goes through the progress bar screen, it might turn black for a moment, let it sit there for a couple of minutes, as it might just be that your machine is a little slow or something. However, if after a few minutes it doesn't seem to continue, than I would suggest downloading and burning the Alternate install cd for Ubuntu.

Go to the following URL:
[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

On the form select the Desktop version of "Ubuntu 7.10 - Supported to 2009".
Then select: "Standard personal computer (x86 architecture, PentiumTM, CeleronTM, AthlonTM, SempronTM)".

Next, select a location to download from, from the location selector.

Then at the bottom of the page there is a checkbox that says:
"Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer."

Check that box.

Now click the "Start Download" link.

Once that is downloaded, burn the cd.

Boot the cd and then verify the media.

Once the media has verified, boot it and install it.

The installer on the Alternate CD is text based and should resolve the installation issue with graphic installer that you are having.
