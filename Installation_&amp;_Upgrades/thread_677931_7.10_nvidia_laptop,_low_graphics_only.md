### Post by Pumalite on 2008-01-25
This is how you install a driver:
Ctrl+Alt+F1
username
password
sudo /etc/init.d/gdm stop
sudo sh /path/to/the/driver/NVIDIA-Linux-xxx.run
Accept the License
Let the driver compile the module into your kernel
Let the driver reconfigure xorg.conf
sudo /etc/init.d/gdm start
Is that how you did it?
