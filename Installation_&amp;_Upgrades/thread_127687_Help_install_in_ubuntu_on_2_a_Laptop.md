### Post by Derek Djons on 2006-02-09
Well it should be no hassle. Make sure you have enough free space left on your harddrive. 

When you start installing Ubuntu Linux you will automatically be guided through the partitioning phase. There you can select the installation which fits you. You'll have to select the option which is called or looks like: "Use (most) free space." This option will gather the free space left on your harddrive and form a new partition. If you like you can even edit it's size.

After some other settings and file installations (which will be all automatic and guided) the installer will search for other Operating Systems installed (automatically) and it will inform you. 

So basically, you don't have to do anything except to select the right partitioning options.
