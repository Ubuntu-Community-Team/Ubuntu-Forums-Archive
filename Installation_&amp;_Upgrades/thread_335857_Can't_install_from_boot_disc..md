### Post by meng on 2007-01-10
1. You need to set the BIOS to boot from the CD before the hard drive. The key to enter BIOS immediately the computer boots is variable on different systems, f1, f2, esc, del, or something else. Sometimes the computer helps you out and tells you which key to press, but you have to be quick.
2. (It sounds as though you may have done #1) Then it's likely that your CD is badly burned. Perhaps you just copied the ISO file to disc (mistake, you need to extract from the ISO file and create a complete filesystem on the CD) or else the .iso download was corrupted or else you burned too fast and should try again at a slower speed. Check this out:
[www.psychocats.net/ubuntu/iso](www.psychocats.net/ubuntu/iso)
