### Post by yappie on 2010-01-02
OK problem solved.......I'm posting this result in case someone else runs into this split screen problem.
I thought my old laptop was under powered but i got it to install and now it works perfect without a split screen.

What I did was searched for a while finally came across someone else with the same problem in the tech section.

When installing I hit F4 and set it to safe graphics mode and it works perfect.

Granted my old P3 laptop isn't as fast as my dual core laptop but it works pretty good and I'm happy with it.
