### Post by sikander3786 on 2010-09-25
You mean while booting from the CD you can save changes or not? If I am right, you cannot. Quite obvious. The changes you make are included in the install only in the case if you perform the installation in the same session. Once you restart all the changes are gone.

If you don't want to install Ubuntu on you HDD for some reason, you can install it to a USB Flash Drive and it will be able to save the changes.
