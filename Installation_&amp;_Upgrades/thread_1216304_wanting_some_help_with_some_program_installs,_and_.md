### Post by Partyboi2 on 2009-07-18
Hi, qbittorrent is in the jaunty repo after you have upgraded you can either search Synaptic (System>Admin>Synaptic Package Manager) for 'qbittorrent' and install or you can install it from the terminal (Applications>Accessories>Terminal) with
```
sudo apt-get install  qbittorrent
```
