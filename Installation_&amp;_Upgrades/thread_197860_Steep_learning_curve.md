### Post by zxee on 2006-06-16
Oh boy-this is an issue and I wish I could automatically insert some excellant posts about this here. You need the correct codecs such as libdvdread installed and a lot of big distros don't install those by default because it happens to be illegal. There are distros that do play dvd out-of-the-box, but ubuntu isn't one of those-obviously. There are ways here to get those installed but you'll be breaking the law if you are in a country it's illegal to do so in (like the US).
Take  a look at automatix and if you don't want to do that check out  pclinuxos.
