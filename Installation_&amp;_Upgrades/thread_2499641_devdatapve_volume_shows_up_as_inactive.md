### Post by euol on 2024-08-06
First you can try to reactivate the logical volume.

To check for issues:
sudo vgck pve
sudo lvcheck /dev/pve/data

To activate the volume:
sudo vgchange -ay
sudo lvchange -ay /dev/pve/data

To check file system integrity:
sudo fsck /dev/pve/data


as for back up, you can do for example:
sudo tar -cvpzf /backup/config_backup.tar.gz /etc
