### Post by shamrock_uk on 2006-03-11
Just choose a server install, then when you get dropped to the command line, type

```
sudo apt-get install xfce
```

or ```
sudo apt-get install xubuntu-desktop
``` if you want a fully functional desktop.

### Post by eriefisher on 2006-03-11
Once you've installed the server just do a "sudo ap-get update", and once your packages are updated do "sudo apt-get install xubuntu-desktop". That should give you XFCE4 desk top. Then repeat with what ever pakages you would like. The choices are almost unlimited.
