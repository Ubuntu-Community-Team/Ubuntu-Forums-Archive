### Post by StormJosh on 2006-12-25
Not that I'm an expert by any means buttttt...

In order to create a boot disk, usually a special program is required. Essentially, you need to burn the ISO file onto a disk and make sure that the bios has booting from CD set to the top priority. 

Here's how to burn the ISO: [https://help.ubuntu.com/community/BurningIsoHowto]("https://help.ubuntu.com/community/BurningIsoHowto")

And you need to access your BIOS options when the computer is rebooting to set the CD boot to the first priority.

### Post by NeoLithium on 2006-12-25
Well, if it has less than 256MB of RAM; you'll need to use the alternate install disk; just trying to make the installation as easy as possible.  They both do the same install, the only difference is that the Live CD shows you the desktop ahead of time and gives you a nice graphical install; but it runs the CD and desktop through the RAM.

The alternate install does the same thing; but it goes through a text install; and is just as easy to use, from what I've seen.
