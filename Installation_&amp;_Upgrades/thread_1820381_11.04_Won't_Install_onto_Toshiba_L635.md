### Post by jeffreyldavidson on 2011-08-07
I have a Toshiba L505 and had a hard time due to their bios. I had to add acpi=off to the grub menu to get it to install. However, Toshiba has updated their bios and now the 32 bit will install. I still cannot get 64 bit to install. Check the Toshiba web site to see if there is a bios update for your machine first and then see how it goes.
