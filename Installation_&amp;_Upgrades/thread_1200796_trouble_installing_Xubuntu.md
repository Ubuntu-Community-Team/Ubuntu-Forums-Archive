### Post by Bucky Ball on 2009-06-30
Try the alternate install CD and see if that works (not Desktop install). Sounds like they might all be installing but having trouble with your screen or hardware.

A good way to check if your hardware is going to work is to run the 'Try Ubuntu' and run it from the Desktop disk (LIVE). If that runs with no problems, your good to install. If it doesn't, most likely a hardware compatibility issue and that can be worked around or not.

Also at the 'options' when booting from the Desktop CD you will have an option to 'Check disk for defects'. Do it.

Could you post the exact model of your Pavilion or try a search for:
> 
HP Pavilion 'Your_Model' ubuntu

... replacing 'Your_Model' with, well, your model! :)
