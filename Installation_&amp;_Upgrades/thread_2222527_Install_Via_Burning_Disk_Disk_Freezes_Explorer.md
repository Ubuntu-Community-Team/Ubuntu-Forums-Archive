### Post by lisati on 2014-05-07
> **Justin_Priester said:**
> I used Cyberlink software that came with my laptop.
It has been a while since I've used any software from Cyberlink, and I don't currently have a copy on any of my machines. Does the version you have include a "burn image to disk" option?

Edit: It is important that you don't just copy the downloaded ISO file to a CD or DVD. The file is designed to be written as a "disk image" that is then used to boot the computer. More information can be found here: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
