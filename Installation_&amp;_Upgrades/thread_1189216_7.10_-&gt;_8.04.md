### Post by Mark Phelps on 2009-06-17
You typically do an in-place upgrade using the Upgrade Manager.

But first, you need to open Synaptic, go to Settings --> Repositories --> Updates tab.  Down at the bottom, check that Release Upgrade says Normal Releases.  If not, set it to that, close Synaptic, and relaunch Upgrade Manager.
