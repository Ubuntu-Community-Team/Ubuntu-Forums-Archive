2.  If you don't want to use the LiveCD anymore when you install something, edit /etc/apt/sources.list and place a # in front of the line (usually near the top) that has CD-ROM in it.

Applications -> Accessories -> Terminal
```
gksudo gedit /etc/apt/sources.list
```
