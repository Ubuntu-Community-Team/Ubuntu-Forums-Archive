I finally got my 2 monitors to work. I did a quick search but didn't find any threads that applied.

after installing 11.04 for the 2nd time.
I changed the classic view not unity..
then I installed the latest Nvidia drivers..270.41.06 not the 173.x in the distro..yes a manual install..

download the drivers first.
reboot..
press ctrl-alt-F1 for a terminal window
cd to the driver location
sudo ./driver name to install..you might need to enable the +x
reboot

all is working in 11.04 here :)
