### Post by adambh on 2010-03-27
If you want to reformat your hard-drive, I recommend Gparted. It's a bootable tool that allows you to shuffle around, add delete and reformat your partitions. It's available on the [Ultimate Boot CD]("http://www.ultimatebootcd.com/") or [on it's own]("http://www.icewalkers.com/Linux/Software/526550/GParted-livecd.html").

I had trouble installing Ubuntu from it's graphical install and found the text-only install to be much more simple and just as user friendly.
