### Post by taurus on 2008-02-10
You need to get into the BIOS and turn off the onboard graphic controller.  Then, boot into recovery mode from GRUB menu and reconfigure your X server again.

```
dpkg-reconfigure -phigh xserver-xorg
shutdown -r now
```
