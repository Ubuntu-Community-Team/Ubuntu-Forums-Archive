### Post by taurus on 2008-02-06
Not sure which program you use to burn CD in windows (I assume you burn it from windows) but you need to burn that ISO image as an image file, not a data file.  And don't unpack the whole thing and just copy all those files and directories to a CD.  Doing that won't boot.

Again, you need to look at your burning app and look for an option to burn as image file and pick the ISO image to burn it.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28burn%29%7C%28iso%29)
