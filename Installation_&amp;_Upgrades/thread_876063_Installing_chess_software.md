### Post by DirtDawg on 2008-07-31
Scid can be started by using the command 'scid' in a terminal. If you want to add it to your menu, simply right-click the Applications menu and choose 'Edit Menus'.

As for Jin, right-click the tar.gz file and choose to extract it. Then enter the new directory created, double click the file named 'jin', and choose 'run'. I think you need Java installed for Jin to run.
