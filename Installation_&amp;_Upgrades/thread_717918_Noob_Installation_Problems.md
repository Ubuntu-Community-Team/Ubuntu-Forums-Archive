### Post by taurus on 2008-03-07
You can use gparted, System -> Administration, and format your harddrive to ext3.  Then, click the install icon on the desktop.  When you get to the partition window, tell the installer to use the whole harddrive.

Otherwise, just click the install icon and the installer will erase Vista and install Ubuntu on that harddrive.  Remember to tell the installer to use the whole harddrive and Vista will be bye-bye.

### Post by cocoKnut on 2008-03-07
When booting from the LIVE cd, you are only temporarily using it without installing;however, if you wish to install ubuntu; once the LIVE cd has booted up, the installer (graphical) is right on the desktop. Just click on it and follow the instructions.

The default option I believe allows for dual booting with windows but you will be granted the option to choose to do that, erase your entire disk and have ubuntu only and probably more advanced options. So, that's all really up to you.

I hope that answers your question.
