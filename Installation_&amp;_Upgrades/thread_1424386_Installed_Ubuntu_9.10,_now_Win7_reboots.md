I installed Ubuntu 9.10 on a system that had Win7 installed.
Now when I select it on grub, it starts booting (the Win7 animation begins), then the system reboots.
What can I do?

---

### Post by leoh on 2010-03-07
Solved.
Started the system into safe mode, lots of activity on the HD, but failed to start.
Then rebooted and it did the HD diagnostic which is expected after the Ubuntu installation.
So, solved. Thanks!
