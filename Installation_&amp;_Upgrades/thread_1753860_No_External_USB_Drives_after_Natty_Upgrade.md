### Post by binary00mind on 2011-05-09
No clue what computer you have (personally I never upgraded, always do clean install) I don't know what packages you have installed. 
If you have "Hard Info" installed, generate report and see which controllers are recognized (UHCI, OHCI, EHCI, AHCI etc) 
You can try to install these packages. 
portmap 
discover (with dependencies)
irqbalance (not needed in all computers) 
mount all 
maybe testdisk (just to take a quick look what tesdisk sees)
all HAL extras 
maybe...multipath (also not needed in all computers) 
I may have missed something. 
Dmraid only if you know what are you doing...(but that has nothing to do with Flash Disks...mhhh)
before you boot make sure that your BIOS sees all the Hard Drives and/or Flash Disks. 
Then boot into Ubuntu. 
Do you use LVM(s) I have no clue how many OS's you have installed and what partition(s) layout you have. 

Good Luck
