### Post by MAFoElffen on 2023-10-13
First check this
```

grep 'Prompt=' /etc/update-manager/release-upgrades

```
The valid values are either "lts" or "normal". If "lts" you could have to use the "-d" optio to check for development releases. But that will not work yet...

Usually the 'do-release-upgrade' path is not turned on until 1 week after the release. That is to enure that things are STABLE and working smoothly for that. The release is on October 20th. Today is Oct 13th. The Mantic Repo is currently frozen (as it usually is the week before a release), and they are moving packages around to get ready for the final release.

Not released yet.
