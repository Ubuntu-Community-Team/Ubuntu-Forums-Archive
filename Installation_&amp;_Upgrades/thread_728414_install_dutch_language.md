### Post by erginemr on 2008-03-19
No problem with that.

You should install the Dutch language support packages "language-support-nl, language-pack-nl and "language-pack-gnome-nl" from Synaptic. When you select them, they will fetch all other dependent packages, too.

One you have done this, you should be able to see and select Dutch from the language selection tool in Main Menu -> System -> Admin -> Language Support.

Now restart your computer to reflect the change. You should also select Dutch from the gdm (login) screen as your default language. (You may need to login and logout more than once.)
