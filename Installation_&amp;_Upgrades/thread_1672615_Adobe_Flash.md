### Post by lovinglinux on 2011-01-21
The most popular ppa for flash 64bit is [sevenmachines]("https://launchpad.net/~sevenmachines/+archive/flash").

However, I actually never used it, because I develop an extension for Firefox, called [Flash-Aid]("https://addons.mozilla.org/en-US/firefox/addon/flash-aid/"), that can check for flash updates, download directly from Adobe and install it. Additionally, the extension removes conflicting plugins and applies some performance tweaks.
