Are you using a cd/dvd?

If true, did you verify the disc first?

Before you burned the disk, did you compare the checksums?

This sounds like a bad copy of the distro (either from the download or the burn)

It could also be hardware related, but before you start chasing wild geese, verify your .iso (with sha1 or md5 whatever is listed) and then verify the burn using the built in utility on the first screen you see when you run the setup.
