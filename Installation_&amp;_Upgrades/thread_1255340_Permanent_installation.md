### Post by Mighty_Joe on 2009-09-01
What you are looking for is a "persistent" install.  The [Live USB Creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") on the Ubuntu Live CD can make one for you.  
[This post]("http://ubuntuforums.org/showthread.php?t=1193680") lists many alternatives, including my preference, a full install to the USB drive.
