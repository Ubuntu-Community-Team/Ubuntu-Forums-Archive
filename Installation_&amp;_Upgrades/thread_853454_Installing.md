### Post by pofigster on 2008-07-08
After you burn the CD - place it in the CD tray of the computer you're installing it onto.  When the computer boots up, it will boot off of the CD.  Then, you can run the "Live CD" to try out Ubuntu or you can "Install Ubuntu"  In either case, it will bring up appropriate menus that will walk you through the installation process.

When it asks about your drives, just use the "Guided - Use Whole Disk" option and it'll clear off Windows 98 and install Ubuntu onto the computer.

Hope this helps!
