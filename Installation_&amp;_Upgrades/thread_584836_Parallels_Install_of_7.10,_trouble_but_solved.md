Running on Mac Pro Dual CPU 3GHz, 8Gigs memory
Running Latest Parallels 3.x
While installing 7.10 it could not start in graphics mode. I. Using CTRL-ALT - and one of the F keys to get into command line. I then ran sudo dpkg-reconfigure xserver-xorg

I selected a modest setting of 640x480, simple Monitor 17inch, memory 32MB
Then went back to the main session which timed out t and gave it another try.,.
Graphics came up and I proceeded with a normal installation.

Up and running now.
