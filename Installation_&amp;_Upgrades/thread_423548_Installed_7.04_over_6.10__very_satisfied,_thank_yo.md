### Post by Browser_ice on 2007-04-26
I have just installed Ubuntu 7.04 over my 6.10 and I gota tell you, I am very satisfied on how it went. Just one minor problem which I can live without. Thank you !


Before the complete 7.04 installation, I had :
2 Hard disk (disk1:XP o/s partition + Ubuntu o/s partition, disk2: 5 XP partitions + 1 Ubuntu swap partition)
dual boot Win-XP Pro + Ubuntu 6.10

After the complete 7.04 installation, I now have :
2 Hard disk (disk1:XP o/s partition + Ubuntu o/s partition, disk2: 5 XP partitions + 1 Ubuntu swap partition)
dual boot Win-XP Pro + Ubuntu 7.04


Installation was very easy. 
-I did manual paritioning where I simply kept everything but first: deleted paritions of my Ubuntu o/s and swap, then re-created them with format option on. 
-then in the import option, I chose my XP ID with firefox favorites and wallpaper
-rest of the installation went smooth

After booting in Ubuntu, I checked the Grub to make sure it didn't mess it up with the previous install. It was perfect but I still needed to tell it that my XP is the default O/S (for my wife). I then had to change the options list for my screen resolution (bigger then 1024*768). To my pleasant surprise, on my desktop, all my XP partition drivers were there and accessible, COOL !!!

I rebooted with XP to make sure it is ok and it is.

The only problem (so far) that I have to report is my XP firefox favorites that didn't get imported. My Ubuntu Firefox toolbar is there but blank. I don't realy mind as I remember all my links anyway.

Thx guys for a smooth 7.04 install  !!!
