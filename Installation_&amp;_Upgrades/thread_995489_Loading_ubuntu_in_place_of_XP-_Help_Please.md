### Post by taurus on 2008-11-27
If you want to wipe out XP from the harddrive, just boot Ubuntu from the LiveCD and install it.  When you get to the partition screen, tell the installer to use the whole harddrive, wiping out windows from your machine.  That's all you have to do.
