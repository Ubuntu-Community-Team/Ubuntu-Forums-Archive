### Post by Quackers on 2011-05-12
You need to go into your computer's bios to set the boot order so that the usb flash drive boots before the internal hard drive.
There will be a key (or keys) that you need to press during the boot process which will take you there.
Sometimes it's the Del key, sometimes it's F2 and sometimes it's F11 or even something else. You need to find out what the correct key is for your system.
