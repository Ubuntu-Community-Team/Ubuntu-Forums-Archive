### Post by bcbc on 2010-12-09
The max size for wubi installs is about 30GB (limited by the wubi installer). If you want a bigger install, and you are able to install direct to partition, it's better not to use Wubi. 

If you decide to use Wubi, don't update packages grub-pc and grub-common - they tend to break wubi installs.

PS I think that ubuntu.com still has the 10.04.1 version of wubi.exe. If you want the 10.10 version, either burn the .iso to cd and install if from there, or you can get the 10.10 version here: [http://releases.ubuntu.com/maverick/](http://releases.ubuntu.com/maverick/)
