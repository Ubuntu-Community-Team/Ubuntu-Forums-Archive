### Post by ian-weisser on 2015-12-13
Windows Fastboot is simply incompatible with a dual-boot or multi-boot system.
There is no resolution, hack, nor workaround for it. Incompatible.
 
That's a feature you paid Microsoft good money for, so enjoy it.
See [http://askubuntu.com/questions/452071/why-disable-fast-boot-on-windows-8-when-having-dual-booting](http://askubuntu.com/questions/452071/why-disable-fast-boot-on-windows-8-when-having-dual-booting) for details on why Fastboot only works on Windows-only systems. The same answers apply to Windows 10.

Fastboot WILL work if Windows is a VM host or VM guest instead of dual-boot, of course.
