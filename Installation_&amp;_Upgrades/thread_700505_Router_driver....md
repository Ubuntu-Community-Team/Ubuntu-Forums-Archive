### Post by DrewMac on 2008-02-18
OK I'll reiterate.

1. Plug the router into a power source and plug the modem into the WAN port (it shows this in the Quick Start Guide).

2. Plug an ethernet cable into one of the four ports on the router and plug the other end into the computer you wish to get online. (Don't worry you will be able to go wireless later.)

3. Open up your web browser and type 192.168.0.1 in the address bar. This will bring up a router splash screen.

4. Log in to the router. (The username and password should be in the manual. If they are not in the manual try "Admin" as the username and "admin" as the password [if that password doesn't work, try leaving it blank])

5. When logged in, click the "Setup" tab in the web interface. Follow the instructions there to configure your wireless network.
