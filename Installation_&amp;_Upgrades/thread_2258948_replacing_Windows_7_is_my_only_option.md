### Post by yancek on 2014-12-31
ZevenOS is apparently a derivative of Ubuntu and if it uses the same installer as Ubuntu, you should see an option "Something Else" which is a manual install.  That should give you the option to select a specific partition to install to.  You could also have a bad download or a bad burn if you burned to a DVD.  You don't indicate how you are trying to boot to install?

### Post by Impavidus on 2015-01-01
Best to make sure first that there is enough unallocated disk space available and, if using MBR partitioning, you don't already use 4 primary partitions. If both are OK, you should have the option to partition manually.
