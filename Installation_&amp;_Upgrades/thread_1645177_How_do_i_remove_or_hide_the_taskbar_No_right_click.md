### Post by sikander3786 on 2010-12-14
I suspect you are running the Netbook Edition. If yes, it ships with Unity and it is under heavy development and not mature and feature-rich yet. We'll have to wait for that to gain all those features by the next 2-3 Ubuntu releases (I hope).

A simple workaround will be to switch to the standard Gnome environment. Log out, click your user name, sessions option will appear at the bottom of screen, select ubuntu-desktop and login. You can customize that according to your needs.

### Post by sikander3786 on 2010-12-16
You are Welcome :-)

> I dont understand why theres a desktop edition option within the netbook edition ubuntu though...

Basically, Unity is built on and runs on top of standard Gnome DE. So it is always there.

Another reason might be because Unity is 3d and requires you to install the proprietary graphics drivers (for ATI and NVIDIA) in order to function properly. Hence, with a fresh install of Ubuntu Netbook, you by default boot into Gnome in order to install the drivers and then you can switch between the two.

There might be other reasons which I am missing here.
