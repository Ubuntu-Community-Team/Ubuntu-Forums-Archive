### Post by lisati on 2009-04-08
One way is to use the update-manager on the system->administration menu to upgrade the version of *ubnutu, and then install the new desktop, e.g. ```
sudo aptitude-install ubuntu-desktop
```
This way will help avoid loss of data, but don't be scared to make backups anyway.
