In my opinion regarding the current 64 bit situation, for the vast majority of users, there is no reason to use 64 bit Ubuntu, and there is no reason not to use 64 bit Ubuntu.  In other words, if your 32 bit install is working fine, you won't gain much by changing.  If you are building a new computer, go with a 64 bit OS.

64 bit can help if you are memory starved and need 4GB+, but unless your running under heavy loads of numerous VMs you can get by easily with 2-4GB (even if you lose 512MB of your 4GB due to video card memory mapping).

The speed gains of a 64 bit OS for most aren't worth it either.  Maybe 1%-5% on encoding apps which might mean an mp3 encodes on 58 secs instead of 60, or a movie in 9m30s instead of 10m.  You'll see a nearly 0% speed increase for daily tasks such as browsing, email, office apps, etc. since these are user-input limited.
