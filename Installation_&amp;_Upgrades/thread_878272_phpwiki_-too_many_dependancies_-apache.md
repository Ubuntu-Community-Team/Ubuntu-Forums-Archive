Ubuntu 8.04 using synaptic, I went to install phpwiki and the dependancy list includes apache, php5, php-pear and mysql-common.

This may be fine for a meta package however I am running phpwiki because you can run it with flat files... no mysql is required.

> 
[http://c2.com/cgi/wiki?PhpWiki](http://c2.com/cgi/wiki?PhpWiki)
PhpWiki can run on relational databases (MySql, PostgreSQL, etc), DBM files, or flat files.


When installing php applications such as phpwiki I think installing the entire lamp server with it way overkill. I think you should just add a comment about installing your lamp first.

If you want to change the existing into a phpwiki-server fine but phpwiki should be a little thinner of an install.
