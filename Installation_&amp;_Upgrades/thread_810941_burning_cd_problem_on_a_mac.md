### Post by IHATEDLINK on 2008-05-28
> **grilla said:**
> i have no trouble downloading ubuntu on my mac. i am trying to burn a cd for my pc. it appears that i must be doing something wrong during the burn because the pc doesnt see the linux disk. the computer sees other disks at startup, so i know i have that right. i have tried burning the cd using the newest version of toast and a program called burn. neither works. please tell me how you a mac user burns a linux cd for the pc.

Did you downloaded the right Ubuntu version? (¡386 for 32-bit processors, macs have 64-bit)
If you downloaded Hardy Heron (8.04) the just insert the CD while you are using Windows (if you are) and a nice menu pops out asking what way do you want do install Ubuntu. Just click the first option and then click the "Help me to boot from CD" button.
Are you SURE you selected the right options of your BIOS?
when you are booting-up type the BIOS config key of your computer. (Usually DEL, ESC or an F as in F12, there's usually a text line that says " press KEY to change boot settings").
Did you burn the CD correctly?
You can't just "copy-paste" the ISO file. You can't make a "data" CD with it. You HAVE to select an option like BURN IMAGE FILE.  
Does the CD boots on other PC's? Try booting from it with your mac. If it doesn't burn the CD again this time as an IMAGE FILE
Finally, is your BIOS capable of booting from a CD? If not you are out of luck.
