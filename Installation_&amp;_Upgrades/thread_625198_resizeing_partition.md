### Post by miltonr on 2007-11-27
Click on the Live Saver (help) icon on the desktop menu bar. Type "Disk Partitions" in the search box and press the Enter key.
A whole set of instructions on how re-size a partition is found on section 2.2.1 paragraph 4. Hope this helps.
