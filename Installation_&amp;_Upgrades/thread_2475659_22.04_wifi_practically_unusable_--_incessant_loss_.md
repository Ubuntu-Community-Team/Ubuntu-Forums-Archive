[https://ubuntu-mate.community/t/wifi-disconnects-over-time-until-reboot-is-performed-18-04-4-lts-qualcomm-atheros-ar9285/22095](https://ubuntu-mate.community/t/wifi-disconnects-over-time-until-reboot-is-performed-18-04-4-lts-qualcomm-atheros-ar9285/22095) has some things to try and the wireless-info script that would help others (not me) to help you.
