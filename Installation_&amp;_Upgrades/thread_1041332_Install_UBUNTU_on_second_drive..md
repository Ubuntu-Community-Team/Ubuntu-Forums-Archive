1) I would recommend Ubuntu because most people here use Ubuntu, so help tends to be faster.  The backends are the same, but the Ubuntu desktop uses Gnome, and the Kubuntu desktop uses [KDE]("http://kde.org/").

2) Dual boot is where you have either two operating systems on [one hard drive]("http://www.psychocats.net/ubuntu/") or on [separate hard drives]("http://users.bigpond.net.au/hermanzone/").  

3) [GRUB]("http://www.gnu.org/software/grub/") starts up initially when the computer is turned on.  If you have a dual boot, then it will offer you a choice of what operating system to boot into.

4) We were all in your position at one time.  I would recommend running the Live CD for a bit to get used to Ubuntu (or Kubuntu, if you prefer.)
