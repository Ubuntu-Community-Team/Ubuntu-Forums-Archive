### Post by PeonDevelopments on 2010-09-29
Ubuntu, at least recent versions, use ubiquity, therefore only the ubiquity commands will be recognized if you decide to use that installer. I do not know if you can use both ubiquity and d-i together.
If you need d-i options, you may have to go with a command-line install (using ascii windows)

You might want to look up what ubiquity options are available to you.
