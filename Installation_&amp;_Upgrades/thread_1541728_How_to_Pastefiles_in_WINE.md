in order for paste to work you need to copy something but keep the source folder(copy)open and the destination folder(paste) open as well as linux has a bug(if it is a bug)where if you copy something and you close the source before pasting that something then the copied item will be lost and you will have to keep doing it but you will get the same result so keep the source and destination both open then once the pasting is done then close the source and destination assuming your done with those locations but under Windows it does not have this bug but it is minor at best.
