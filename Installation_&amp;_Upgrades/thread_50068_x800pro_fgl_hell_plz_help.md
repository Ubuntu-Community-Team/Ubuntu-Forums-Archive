### Post by evilghost on 2005-07-19
Here is my xorg.conf when I was running a 9800 PRO.  Looking at your xorg.conf, you're still using the ATI driver.  Wrong, you need to be using the fglrx driver if you want 3D support ;)

You'll also want to install the newest ATI drivers from ATI's website, but you're need build-essentials and the kernel-headers.  There are some good posts out there on how to do it, it took me about 30 minutes, including the research.  First time I had ever done something like that, ever ;)

```

luser@400sc:/etc/X11$ cat xorg.conf.ati
```
