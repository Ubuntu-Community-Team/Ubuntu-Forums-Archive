### Post by Pumalite on 2007-09-19
tsclient: (Terminal Server Client) is a frontend for rdesktop and other remote desktop tools. It comes installed with Ubuntu and is a GNOME application. Notable visual options include color depth, screen size, and motion blocking.

tsclient and rdesktop are both found in Synaptic as is Java.
