### Post by jobezone on 2005-04-12
You see, you don't just burn the ISO file onto a CD, like you would with any other file.

Go to [http://www.ubuntulinux.org/wiki/BurningIsoHowto](http://www.ubuntulinux.org/wiki/BurningIsoHowto) and you'll find instruction in the section called "Burning/writing an .iso Image to CD with Windows XP" (I assume you are using windows xp). There, it will mention a software you can download, which after installed will allow you to just right-click the ISO file, and burn it the "correct way".
