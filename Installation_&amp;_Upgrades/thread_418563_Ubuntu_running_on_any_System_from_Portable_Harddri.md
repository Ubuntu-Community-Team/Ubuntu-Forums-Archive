### Post by carpex on 2007-04-22
Other option is to install Ubuntu on your external HD, and to boot from this hard drive. However, this will only work from computers that have a bios that support usb booting. Useful info can be found here: [http://www.pendrivelinux.com/](http://www.pendrivelinux.com/) and here [http://ubuntuforums.org/showthread.php?t=80811](http://ubuntuforums.org/showthread.php?t=80811).
