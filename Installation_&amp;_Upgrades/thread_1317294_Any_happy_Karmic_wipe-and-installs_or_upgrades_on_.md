### Post by 1clue on 2009-11-08
I've more or less solved this.

In spite of what has been said on the sticky for Karmic upgrade experiences, you still need the alternate installer for AMD64 to install on RAID.

That said, if you are familiar with the Jaunty install for this platform, then it's pretty much the same thing with the Karmic one.

Extras:  Some things work much better.  Sound and WebCams for example.
