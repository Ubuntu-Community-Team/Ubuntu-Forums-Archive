### Post by por100pre1 on 2007-10-14
It's possible, no upgrade is 100% error proof. Backup all your user folder files (even hidden files, those are the settings) to removable media or external hard disk drive. Disable any external repositories (if any) before upgrading, and be sure to have an ISO already burned in case you need to make a fresh install.

Hope this helps, sorry for the delay.
