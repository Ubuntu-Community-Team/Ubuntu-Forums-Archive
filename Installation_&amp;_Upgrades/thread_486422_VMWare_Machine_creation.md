### Post by sublimation on 2007-06-28
Just as a heads up for anyone having trouble trying to run their own VMs
 [http://www.easyvmx.com/]("http://www.easyvmx.com/")
will allow you to create an image to run in VMWARE player, and therefore have a simpler interface than trying to set up server. you can use the original easyvmx with the old version of VMWARE player in the add/remove programs. It's a good way to get your feet wet until you decide you need something more customizable. 
Cheers!
