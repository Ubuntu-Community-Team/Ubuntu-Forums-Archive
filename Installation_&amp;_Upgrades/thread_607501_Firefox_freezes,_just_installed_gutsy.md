Please excuse me if this is an old thread.


Anyways I think I figured out how to fix the problem. In Firefox go to  Edit : Preferences : Advanced: then go under the tab "Network". At the bottom there is the cache setting. Set it to around 20.

[[IMG]http://img412.imageshack.us/img412/6972/snapshot12qx1.th.png[/IMG]](http://img412.imageshack.us/my.php?image=snapshot12qx1.png)

This seemed to work for me. I rarely get any freezes now.

I hope this helps!
