### Post by engla on 2006-03-24
Nope, the default installer installs all of the base system. This requires some disk space.

There are other install options on the install disc, and will be in the future too  -- "expert" or "server". Expert will ask more questions, and server will install just a base system where you have to add further needed packages yourself.

You should be able to get a fine ubuntu system with a server install, but it will be cumbersome..
