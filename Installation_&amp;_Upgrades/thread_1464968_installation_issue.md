### Post by smellyman on 2010-04-28
Pretty easy if you already have a partition ready.
 
Boot up with the Ubuntu CD and when the intallation asks where you want to install, choose "specify existing partition" and choose the partition you have free.
 
Ubuntu will load Grub2 which is the boot loader, so when you boot your computer after installation, Grub will load and you can select to boot into Windows or Ubuntu.
