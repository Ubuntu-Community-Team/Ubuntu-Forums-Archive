Have you tried the  one in the ubuntu repo?
[http://deepbluespaces.blogspot.com/2008/07/installing-tomcat-on-ubuntu-linux.html](http://deepbluespaces.blogspot.com/2008/07/installing-tomcat-on-ubuntu-linux.html)

If the file you want to move is on the desktop you can do
```
 sudo mv /home/username/Desktop/filename /usr/local
```Change "username" to your username and change "filename" to whatever the file is called.
