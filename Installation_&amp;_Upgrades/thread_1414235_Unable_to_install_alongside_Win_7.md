### Post by phillw on 2010-02-23
Hi and welcome the the forums.

Wubi is installed like any other windows programme - So, if it has gotten poorly and you cannot access it use Add/Remove programmes from Win to remove the old one. You should then be able to re-install. A note of caution, there had been issues reported with running Wubi with Win7.

If you can spare, say, 10GB of hard disk space, I'd suggest going for a dual-boot approach. Whilst Wubi does run very well for many - for some it causes problems.
