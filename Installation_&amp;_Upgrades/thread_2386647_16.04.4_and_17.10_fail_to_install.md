### Post by TheFu on 2018-03-09
I had a similar issue. Had to load new firmware onto the motherboard ... seemed to be part of the Intel CPU bug fixes.  After that,16.04.4 loaded fine.

Also, you do need a FAT32 partition for UEFI if the system is booting via EFI.

If you don't hibernate, I think 8G for swap is more than desired.  4.1G is the size, IMHO.
