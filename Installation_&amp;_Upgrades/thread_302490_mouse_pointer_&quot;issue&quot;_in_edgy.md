### Post by mgmiller on 2006-11-18
Try changing your mouse pointer to something else.  System>Preferences>Mouse--pointers.  Select a different one and see if it works ok.  You can then try changing it back.  Sometimes this will clear problems.  There are also optional mouse pointer themes available online from places like gnomelook.org.

Good luck.
