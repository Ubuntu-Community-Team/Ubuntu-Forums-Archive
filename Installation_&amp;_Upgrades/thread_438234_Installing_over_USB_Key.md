### Post by michaelmuir on 2007-05-10
I am sure this is possible.  A colleague of mine has been using this arrangement to allow his students to run Ubuntu V5.X on laptops that have Windows installed.  Using the USB key avoids the need to have two partitions.  I think he used a machine that was running linux to format a key, then installed the kernel and/or other components.  Once you have the OS installed on the key, your machine will be able to boot from it as long as the BIOS is set to check for a USB device with a boot sector.
Good luck with the new version.
