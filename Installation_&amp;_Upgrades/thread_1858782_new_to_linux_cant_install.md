### Post by MARP1961 on 2011-10-13
Does it have a CD/DVD drive (or have you one you can plug in via USB)?

If you have this option, then just burn the iso image with suitable CD burning software. One you have done this you will have a bootable CD. Make sure your netbook is set to boot from this device first.

I know that in Ubuntu you can create a bootable 2GB flash drive but I'm not sure how you do this in Windows (I'm sure someone here will know).

Incidentally, if you wait until later on today you can download the latest (and some say the best so far) release Ubuntu 11.10 Oneiric Ocelot.

### Post by MARP1961 on 2011-10-13
