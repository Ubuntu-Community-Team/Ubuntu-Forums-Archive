### Post by Melab on 2024-02-25
Suppose that one is on a recent release of Ubuntu. Now, suppose that one wishes to install a tuple of kernel image, kernel modules, and kernel headers from an older release of Ubuntu. How would one do this? The package names must always contain something of the form "X.Y.0-Z-generic".[FONT=courier new][FONT=courier new][FONT=arial]

On launchpad.net, one can find various Debian packages of presumably older kernels, but the problem is that their file names do not match the specified format.[/FONT][/FONT][/FONT]
