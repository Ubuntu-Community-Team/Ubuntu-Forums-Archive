### Post by cskeide on 2006-06-19
As far as I know, universe and multiverse are not officially supported, but you will still receive updates in the same way as with the main repository. The restricted repository is supported, but contains non-free packages.

> Canonical does not provide a guarantee of regular security updates for software found in universe but will provide these where they are made available by the community. Users should understand the risk inherent in using packages from the universe component.

The quote is taken from [http://www.ubuntu.com/ubuntu/components](http://www.ubuntu.com/ubuntu/components)
