Hi njothi;

Well install isn't that difficult,but there can always go some thing wrong,by the user,computer malfunction or even a power failure.
That's why we always advice to make a proper backup of your data.

Next thing of importance is your hard disk and how it's partitioned.
Do you have room,or can you create enough space to install ubuntu.
You have to realize you need about 10GB of space,divided into two partitions,to make a proper install.
So the current layout of your hdd is quite important.

You can download the GParted live cd ,burn it to a disk,and boot from this disk.

[http://gparted.sourceforge.net/](http://gparted.sourceforge.net/)

Now you have a partitioner with a graphical interface,which is some what easier to use.
With this tool you can resize your partitions [defrag windows first] and create new ones.
Keep in mind your windows should be on the first primary partition,don't mess with it.
Ubuntu doesn't care if it's located in a primary or logical partition.

If you understand what I'm talking about,you can have a go.
If you think this is all abacadabra to you,you'll better reconsider the fact if you want to install ubuntu.
