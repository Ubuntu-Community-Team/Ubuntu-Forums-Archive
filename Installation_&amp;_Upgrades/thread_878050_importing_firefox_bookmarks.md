### Post by ubshyam1234 on 2008-08-02
Hi,

You can import it using a tool provided by firefox as a plug-in. it'll take a little time to set up.
But its absolute wonder if u start using it. its called [Foxmarks]("http://www.foxmarks.com"). Just register for a new account.
Now, u got to install the plugin to your firefox browser. [Click]("https://addons.mozilla.org/en-US/firefox/downloads/file/33149/foxmarks_bookmark_synchronizer-2.1.0.12-fx.xpi") this and download and install thru firefox. Restart firefox. Input the settings.. It syncs all bookmarks to an internet server. use it in your firefox 3.0 by again doing the 2nd step( plugin installation alone). All your bookmarks are safe and ready.

Happy browsing.. I know its complicated than the previous post. but its worth doing.
