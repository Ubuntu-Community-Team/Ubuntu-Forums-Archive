### Post by Old_Grey_Wolf on 2012-05-13
Did you check the MD5 hash? How to check the MD5 hash is at [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
You will need to substitute the iso filename you downloaded for ubuntu-11.10-dvd-i386.iso in the instructions.

If you downloaded 12.04 from an alternate torrent site you don't need to check the hash.
