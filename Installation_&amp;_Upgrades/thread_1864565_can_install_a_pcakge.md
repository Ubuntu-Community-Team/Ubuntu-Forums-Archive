### Post by dino99 on 2011-10-20
download from:
[http://www.w3.org/Amaya/User/BinDist.html](http://www.w3.org/Amaya/User/BinDist.html)

then install from the download folder (via cd/to/that/folder):

sudo dpkg -i amaya_11.3.1-2-ubuntu_i386.deb  (example for the 32 bits package)

Thats it :)
