### Post by presence1960 on 2009-08-14
to add to running-rabbit's suggestions:

Did you [MD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM") the iso you downloaded prior to burning it to CD as an image?

Did you [burn]("https://help.ubuntu.com/9.04/switching/installing-burning.html") it as an image to CD at no more than 4x speed

Did you choose "check disk for defects" when booting the CD prior to attempting to install?

if all that checks out see [boot options]("https://help.ubuntu.com/community/BootOptions")

if all else fails use the [alternate text based installer]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate") which by the way has more options with support that the Live CD does not have. You just can't run Ubuntu from it as you can from a live CD.
