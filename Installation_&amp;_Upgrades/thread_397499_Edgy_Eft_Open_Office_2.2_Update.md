### Post by Timokl on 2007-03-30
> **louisd11 said:**
> Hi, I am running Ubuntu Edgy Eft and i saw that they came out with 2.2 of OpenOffice but, when I do "check updates" it doesn't find it. And it says I am using OpenOffice 2.0.??? So how do I update it to 2.2?

You have to upgrade to the next Ubuntu version "Feisty Fawn".

Everything you install through Ubuntu's software repository (meaning using Synaptic, apt-get or Automatic Update) is prepared by the Ubuntu team especially for Ubuntu. So they have to ubuntuize ;-) OpenOffice from the OpenOffice project and put it into the repositories. However, Ubuntu will not get an update to a newer version, only bug fixes will be provided for one Ubuntu release.

If you don't want to wait until the middle of april for Feisty Fawn, you can update OpenOffice by hand. See [http://www.ubuntuforums.org/showthread.php?t=397233]("http://www.ubuntuforums.org/showthread.php?t=397233")
