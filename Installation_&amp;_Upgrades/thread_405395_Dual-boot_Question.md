### Post by BDwinsAlt on 2007-04-09
My computers hardly use ANY swap at all.  If you have 512 MB of ram or more you probably don't even need it.  I have all kind of servers and games on mine and it runs fine.  Install ubuntu on the hda3 partition.  As long as it installs only on that partition, it won't erase windows.  Just make sure you don't select to erase it all.
