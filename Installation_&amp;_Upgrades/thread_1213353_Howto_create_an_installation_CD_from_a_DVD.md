### Post by kilowattradio on 2009-07-15
> **jheis said:**
> Anyone?

 You will have to create an ISO from the DVD and mount that in Linux. A free good tool to partition a hard drive is called PARTED MAGIC and you can burn that to a CD-R. You can do a google search for how to mount and install an ISO image. You could also copy the DVD image to a USB stick and mount it from there in Linux. Honestly your best solution is to find a person with a DVD burner and burn it. I think kinkos will allow you to use their computers for $10 or so as a last resort. The install via ISO image take some expertise and a lot of typing on the command line.
