### Post by solar george on 2007-04-29
A desktop cd boots up a live cd system (one running from the cd) which allows you to try ubuntu before you install. However it does have occasional hardware problems and runs very slow on old computers.

The alternate cd uses a text-menu based system to install ubuntu, it's not the most user friendly design but has more reliable hardware support than the desktop cd.

N.B. If you post about problems installing from the desktop cd  the first suggestion you will get will probably be to use the alternate cd.

### Post by stealth17 on 2007-04-29
The reason the GUI version can get slow for computers with less then 256mb of ram is because the Live CD attempts to load as much as possible into the ram since it is really slow to read everything from the CD all the time. If you have less then 256mb of ram, the speed decreases exponentially.
