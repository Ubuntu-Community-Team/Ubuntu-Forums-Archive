### Post by presence1960 on 2009-08-08
you don't have to touch your ubuntu other than backing up just in case something goes wrong. Follow the instructions in the link merlin gave you. If you install windows to the same hard disk as Ubuntu you will have to restore GRUB and then you will be good to go. The windows bootloader will overwrite GRUB. It is really simple to restore GRUB, if you have problems post back here.
