### Post by g00f on 2009-05-24
Go to [http://www.vmware.com/download/ws/](http://www.vmware.com/download/ws/) and download the <filename>.bundle file, assuming you have the rights to do so. Then:

chmod 755 <filename>.bundle
sudo ./<filename>.bundle

That's it.
The icon will appear in the Applications, System Tools menu.
