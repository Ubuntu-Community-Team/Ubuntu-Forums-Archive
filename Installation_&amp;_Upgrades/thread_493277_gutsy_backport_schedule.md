### Post by rbmorse on 2007-07-05
My experience with Ubuntu is that unless there is a serious security related issue they don't backport. 

For example, I have a lot of reliability problems with Evolution 2.10. 2.11 (with fixes) is in the Gutsy pre-release, but it won't install into Feisty. Since I don't lose data (it's a benign crash) there doesn't seem to be any impetus to backport the support libs required by Evo 2.11 into Feisty. 

In fairness I should note here that Evo 2.11 is considered to be a development release and I suppose I can see why there is no rush to backport a dev release into a finalized distro. So, I'm not saying the devs are doing anything unreasonable in this regard, it's just bad karma for me and that sometimes sucks.
