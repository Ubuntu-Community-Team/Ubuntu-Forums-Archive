### Post by angryfirelord on 2007-11-21
> **tub3r said:**
> Thanks, is there something i'm missing for compiz or anything else there for it to work?
Yes, you have to do three additional things for it to work.

-Re-open your xorg.conf and find the word *Composite*. Change it to say this:
```
Option          "Composite"     "Enable"
```
-Look for something called *Section "ServerLayout"* and add this under it:
```
Option "AIGLX" "True"
```
Save your file and open up this one:
```
sudo gedit /etc/xdg/compiz/compiz-manager
```
Add this to it:
```
WHITELIST="nvidia"
```
Log out and log back in. 3D effects should now be enabled.
