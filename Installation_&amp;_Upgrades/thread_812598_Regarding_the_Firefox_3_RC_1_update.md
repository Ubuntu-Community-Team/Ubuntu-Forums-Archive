### Post by bapoumba on 2008-05-30
As of today, the packages are in the -proposed repositories.

Users should remember that these repositories are for testing purposes and may lead to breakage:

[From this link]("https://help.ubuntu.com/community/UbuntuBackports#head-c8ff4dccb06c553c4f03777dd5dcbeb5caa38a32"):
> -proposed is the testing area for -updates. A number of people must give positive feedback on these packages before they are allowed into -updates. This repository is recommend to ONLY interested in helping to test updates and provide feedback. Since they are in effect testing updates, there is a higher chance of defective updates in this repository.

[General discussion thread]("http://ubuntuforums.org/showthread.php?t=797212")

This sticky will be removed as soon as the packages hit the main repositories :)
