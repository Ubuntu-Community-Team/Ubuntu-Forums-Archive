### Post by az on 2005-11-28
You did not mention how  big of a hard drive you have, but you can install the default ubuntu system and run it on those specs.  It wil run slow, but it will run.

You can add on icewm, or xfce by enableing the universe repositories in synaptic and installing the packages.  After which, you log out and then chose the icewm or xfce4 session in the login manager.

Install the menu package, since icewm will have nice menus if you do that.
