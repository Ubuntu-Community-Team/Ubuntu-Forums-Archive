### Post by darkod on 2012-02-21
Regardless of the error, do you plan to upgrade version by version since 10.04? Many issues can come up during three consecutive upgrades.

First make sure you have done all the UPDATES (not UPGRADE).

Then, you can try the upgrade again but I would recommend to wait little bit longer when the new 12.04 LTS comes out in late April. Since 10.04 is also LTS, you can select in the update options to offer only LTS upgrades and can upgrade to 12.04 LTS directly.

### Post by ajgreeny on 2012-02-21
You should still be able to upgrade your distro version to 10.10, and in a couple of months to 12.04, but I would suggest that the huge changes between 10.04 and 12.04 make it better to perform a clean install when 12.04 is out in its final form.

If you are not getting the option to upgrade to 10.10, go to System >Administration >Software Sources and in the updates tab check how your updates are setup.  See screenshot.  Choose "Normal releases" to upgrade to 10.10.
