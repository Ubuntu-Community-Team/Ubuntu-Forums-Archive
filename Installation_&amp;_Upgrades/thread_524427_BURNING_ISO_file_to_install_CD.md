### Post by az on 2007-08-14
Insert the cd into the drive and you should be prompted as to whether you want the packages on the disk to be added to the list of repos.  If you don't get this, then the cd is not a proper archive of packages.

Once you add the packages to your list, you should be able to install them using synaptic.
