### Post by ariek on 2006-06-26
Hi Steve,

The default installation (started from your Desktop cd session) takes care of everything. Grub, the boot loader, searches for other OS's which will be added to the boot menu options.
Ofcourse, you should take care of the needed Ubuntu space as well, which is also easily configurated from the default installation.
Here's the Wiki article about installation:
[https://help.ubuntu.com/community/Installation/I386]("https://help.ubuntu.com/community/Installation/I386")

Good luck!
