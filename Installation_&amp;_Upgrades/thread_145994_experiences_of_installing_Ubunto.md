### Post by seanieb on 2006-03-17
My experiences of installing Ubunto. I have installed a Linux distro before, but it’s been a few years (but in my memories it was much easier)

The first few setup screens were fine. 

But then I wanted to resize my hard drive, so I could dual boot between windows and Ubuntu . I was using the instructions on the website:
[https://wiki.ubuntu.com/WindowsDualBootHowTo](https://wiki.ubuntu.com/WindowsDualBootHowTo)  and
[http://users.bigpond.net.au/hermanzone/p3.htm](http://users.bigpond.net.au/hermanzone/p3.htm)

But I was still having problems.

When I enterer the size I wanted to reformat my windows partition too, I would click yes then I would return me to the main partition disks over view, but there were no new partitions or free space. There was also no error information informing me what had gone wrong.

I was trying to install to a Dell Inspiron 5100.

Points the install of Ubunto falls down on:

-	Displaying progress. No progress bar. Even a bad one would be better than none at all. Or even a “step 1 of 15” type thing.
-	The back button doest take you back!! It takes you to the master list of install steps. The back button should take you backwards.
-	The partitioning screen is too technical and the layout is bad. There is no mention of dual booting with windows. Most people who install for the first time will be doing so with the intentions of keeping windows on their machine. If there was a “keep windows on your machine and install Ubuntu also” option this would make everyone feel less tense around the partitioning screen, safe in the knowledge that they wont accidentally erase windows.
-	There seems to be no context when typing in the “New partition size:” which partition are you referring to? The new one or resizing the old one? (I actually know, but your average Joe will not)
-	Lack of error messages when there are problems with the portioning.
-	It’s fine to refer people to web pages for instructions, but if you’re installing on your only pc, then the information should be displayed on the install screen. The instructions are minimalist and sometimes not relevant.

To combat these problems get 10 newbie linux users into a room and observe them attempting to install a dual boot system without help, take notes and feed back. And then go back to the drawing board.

I see these points as being huge problems for Ubuntu. It would be much easier to get people with existing windows installed on their machines to change over it was easier to install. You might be producing the best OS in the entire world, but the majority of the world will never know if they can’t install it.

I hope this helps

Yours sincerely Sean Byrne

PS keep up the good work Ubunto is the best.
