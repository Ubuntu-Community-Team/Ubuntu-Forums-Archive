### Post by lsemmens on 2011-12-12
I don't know the answer to your question but the process that I would follow is this.

Does it boot into Live (Try Ubuntu) mode? i.e. not installing it
Try your USB stick in another machine
If the above fail i'd then try a CD version 
If that works you can install from there
Otherwise re-download an re-create your image
and try again.

I'm running a 32bit version of Linux from USB on a Toshiba Lappy that currently has no HDD with no issues boot time is approx 30secs.
