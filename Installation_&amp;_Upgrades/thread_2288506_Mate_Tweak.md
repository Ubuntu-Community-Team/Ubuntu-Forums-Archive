### Post by Dennis N on 2015-07-28
What version of Ubuntu MATE are you using? There is one bug report that MATE Tweak does not start when using the Numix theme. 
Bug reports are here: [https://bugs.launchpad.net/ubuntu-mate](https://bugs.launchpad.net/ubuntu-mate)
You can report a bug (or add to an existing bug report) at the above page by logging in to Launchpad at upper right (register if needed first) and click on "report a bug".

I am using Ubuntu MATE 14.04 and haven't had any problems starting MATE Tweak. There have been many updates since it was installed.
