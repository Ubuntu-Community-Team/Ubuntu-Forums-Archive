### Post by darkod on 2010-02-21
During startup there is usually button you can hit for Quick Boot menu, like F12, F10, etc. It should say it on the screen.
Another way is to enter BIOS and change the boot options CDROM to be before HDD. Save and exit. For entering BIOS there should also be button specified on the screen or similar, like F2, etc.
