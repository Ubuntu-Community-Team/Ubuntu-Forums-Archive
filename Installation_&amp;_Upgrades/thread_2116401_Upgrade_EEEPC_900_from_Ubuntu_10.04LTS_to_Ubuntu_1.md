Procedure for upgrading ubuntu 10.04 LTS on eeepc 900

get the iso - burn to DVD or USB as per instructions. I assume you already know how to do this - seeing as you would like to upgrade

if you get synaptic errors relating to your sources lists do: [http://ubuntuforums.org/showthread.php?t=863742]("http://ubuntuforums.org/showthread.php?t=863742") and go to #4

then if you get MMap errors with APT::Cache-Limit go to: [URL="http://askubuntu.com/questions/219523/dynamic-mmap-ran-out-of-room-when-trying-to-sudo-apt-get-anything"]
http://askubuntu.com/questions/219523/dynamic-mmap-ran-out-of-room-when-trying-to-sudo-apt-get-anything
[/URL]

then run Update Manager and you should be able to complete the upgrade to 12.04LTS

I hope this helps someone. It took a little looking around and a little frustration to get this sorted.
