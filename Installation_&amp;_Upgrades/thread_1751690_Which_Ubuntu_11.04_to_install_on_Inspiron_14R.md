### Post by Dutch70 on 2011-05-06
Hi Angela

I would definitely go with the 64-bit version if your computer is capable. 

You will need to shrink your hdd from within windows to leave room to create partitions for Ubuntu. There are plenty of tutorials on the net that will help you shrink your hdd. You'll need to do things like defrag, turn off system restore, but it's best you follow a tutorial.

You will also need to make sure you don't already have 4 primary partitions.

What else do you need to know?
