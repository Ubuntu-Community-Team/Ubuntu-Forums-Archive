### Post by Bucky Ball on 2013-06-27
*Thread moved to **Installation & Upgrades**.*

Welcome. Which bit don't you understand?

Download the ISO (12.04 LTS Desktop might be a good place to start), 
burn the image to a DVD, 
switch off computer, 
boot up and go to the BIOS or press whichever key takes you to a list of the boot devices, 
put the Ubuntu install DVD in the slot,
Instruct machine to boot from CD, either in BIOS or select from device list,
Follow your nose.

That should get you started. You will need to be more specific as to which bit you're having trouble with exactly. ;)
