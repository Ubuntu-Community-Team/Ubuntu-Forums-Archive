### Post by 73ckn797 on 2009-12-20
Boot from the LiveCD and run Gparted to reformat or repartition and reformat the drive.
You could also boot with CD and the install will ask/give you a choice of doing the same (reformatting the partition that has the current OS on it) as part of the installation process.

Back up any data you do not want to lose.
