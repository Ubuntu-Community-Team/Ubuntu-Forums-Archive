### Post by grahammechanical on 2012-11-09
If you go to the power/cog menu and select About this computer it will load the Details page. There you will see a panel - bottom right that will tell you if the System is Up-To-Date. A few minutes after booting it will change to Checking for Updates and if there are any it will offer to Install Updates.

To put it another way the 12.04 software Updater is now are part of the Details page that is the front page of system Settings. Everything is done automatically, except the actual installing of the updates.
