```
gksudo gedit /etc/X11/xorg.conf

Looks like not everything got updated when you ran the install.  
The kernel is using an earlier rev of the software than the X server is using.  I don't think editing files will help.  You need to bring everything up to the latest rev or drop everything back to the earlier rev.  Its a driver issue not a configurartion issue. 1.0-8776 is the latest rev 1.0-7184 is from August.  Use your spare PC to go the Nvidia website and go to download drivers.  Find the linux drivers section and checkout the Readme.  Anyone got another way to update the file that is out of date?
