### Post by rcarring on 2006-06-14
Some users here have laptops with XP but no restore/recovery disks. I have used the information on this page to create working, reliable bootable XP reinstall disks that use the i386 folder in the root of C to provide the XP files.

[http://www.nu2.nu/bootcd/wxp/](http://www.nu2.nu/bootcd/wxp/)

The procedure has to be done from XP and requires a cdburner drive.

Read all the info carefully, and perhaps print out the instructions too.

Edit:

slightly off topic but if you are ordering a new laptop insist on a spec that is Vista capable and do not be fobbed off with machines that are not: they will cost more but in the long run (expecting you will use the new machine for at least three years before buying a new one) it is better.
