### Post by .nedberg on 2010-05-16
Get the AMD64 version, it is for all 64bit systems. I use it with my Intel i5. The naming is a bit "strange"...

Your processor is listed here: [https://help.ubuntu.com/community/32bit_and_64bit](https://help.ubuntu.com/community/32bit_and_64bit) as one that should use the AMD64-version.
