### Post by Frogs Hair on 2011-12-12
There is always risk when installing a kernel not designated for the operating system .  The latest stable version would probably be the best choice because it includes the most recent updates . You will need to lock the kernel version once installed or it will downgrade when normal kernel updates come for 10.10 . Ubuntu will install updates for the kernel you are supposed to have not the one you are using .
