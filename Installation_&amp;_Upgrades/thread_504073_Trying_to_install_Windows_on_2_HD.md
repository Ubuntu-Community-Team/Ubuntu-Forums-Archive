### Post by Pumalite on 2007-07-18
Your 80 GB drive is probably sda. If thats the case, Windblows will probably always want to be there. Besides that, Winddoze always demands a primary partition, where as Ubuntu is happy anywhere. Make sure you install windblows first and Ubuntu last.
