### Post by ubfan1 on 2024-07-02
Check your memory with the free command.  How much is free/available and is swap being used?  Lubuntu or Xubuntu have a smaller footprint and might help.  Swapping off an USB3 port with an M.2 NVME SSD might be almost as good as more memory (which is the first thing to try).

### Post by grahammechanical on 2024-07-02
The installer for Ubuntu 24.04 LTS offers two install options Essentials (default) = a web browser (Firefox?) and basic utilities (including Gnome terminal) or Extended Selection = the usual Ubuntu applications; office tools & web browser. - no games.

With a basic (Essentials) install you then use the terminal to install the applications you need and change browsers.
