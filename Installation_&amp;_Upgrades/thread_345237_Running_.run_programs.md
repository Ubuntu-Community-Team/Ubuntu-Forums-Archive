### Post by amperSandII on 2007-01-24
I recently downloaded the demo of the game majesty, titled majesty_demo.run and i have no idea how to start it up. In an old thread someone suggested typing sh majesty_demo.run in the terminal but that did not work. Help would be appreciated.

---

### Post by 23meg on 2007-01-24
Make the file executable if it isnt. Right click, go to "Properties" and check the "Allow executing file as program" box in the "Permissions" tab. Then run it with ```
sh majesty_demo.run
``` If you're not in the directory where the file is, navigate to it with the "cd" command.
