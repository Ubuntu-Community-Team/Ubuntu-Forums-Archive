### Post by forestpixie on 2007-11-24
open it for editing

```
gksudo gedit /etc/apt/sources.list
```
delete the two lines at the bottom which start 'sudo' - save and close, then update

```
sudo apt-get update
```

see if there are problems then - if there are post them here

if you want to install zsnes

```
sudo apt-get install zsnes
```
