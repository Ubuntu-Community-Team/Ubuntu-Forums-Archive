### Post by AceMilo on 2008-05-06
Steam does run just fine in ubuntu with wine.  First install wine, then you're going to need to find an old steam installer, maybe oldversion.com has it, just google around you should find one quickly.  It doesn't matter how old it is, just make sure its an .exe not a .msi file.  After that just update steam from itself and login and install your cod data.  You should be good to go after that.
