### Post by Pollywoggy on 2007-06-14
I had to partition a hard drive so that I could install a second distro and I have been using gparted from a Freespire CD to do that type of job, but it has not been easy to move partitions and I was not always able to shrink a partition from either side.

Enter System Rescue CD.  

I used this today and was easily able to shrink partitions from either the left or right sides.  I think the reason might be that the gparted on the CD is a newer version.

You can get System Rescue CD at [http://www.sysresccd.org](http://www.sysresccd.org) and it is based on Gentoo but is distro independent.
Also, Linux Magazine has an article about System Rescue CD, at [http://www.linux-mag.com/id/3357/](http://www.linux-mag.com/id/3357/)
You need to register on the site to read the article, and it's free.
