### Post by darkod on 2010-01-20
You installed wubi, not ubuntu. There is quite a difference between wubi and full install.

As for the default OS in windows bootloader, try this:
Right click Computer icon on desktop, select Properties. In the right menu select Advanced System Settings, then the Advanced tab and under Start up and Recovery click the Settings button. That will allow you to select default OS and timeout.

I'm currently on win7 but on vista the options should be called the same or very similar.
