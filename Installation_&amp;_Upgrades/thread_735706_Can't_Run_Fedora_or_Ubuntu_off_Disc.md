### Post by ridgeland on 2008-03-30
My first effort into Linux was Fedora Core 4 which I could not get to boot.  A forum explained my problem.  I had tried to copy the download to the CD which resulted in a Data-CD and not an iso image.  Sounds like you are burning it as an iso image and not a Data-CD.
Have you tried to boot something like a Windows OS install CD just to verify the system boots CDs properly?
Download and try booting "SystemRescueCD".  It's my go-to CD when I have problems.
[http://www.sysresccd.org/Main_Page](http://www.sysresccd.org/Main_Page)
It's small so the download will be faster than Ubuntu or Fedora.
