### Post by Linux_oid on 2007-04-02
**Local installation** happens when you install packages for one user only. Most of the time a package is installed for the person who sigh up when ubuntu starts, 

When you have more then one user on a box (computer), it's better to install globally, so every user can use the same package immediately.  [There]("http://www.ubuntuforums.org/showthread.php?t=389500") is some info about global installation.

1.  Click *Download .tar.gz file* on [B][URL="http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash"]
[/URL][/B] site. 
2. When window *Opening install_flash_player_9_linux_tar.gz * appears, choose  *Save to Disk* and click OK.
3. Wait untill download is completed.
4. Go to **Places** (top of your screen), click it, choose **Home Folder**, and click  it.
5.  File Browser opens home directory (<user name> - File Browser) where file ***install_flash_player_9_linux.tar.gz*** has to be. 
6. Double click it. 
7. Archive manager should start and open folder *install_flash_player_9_linux*.
8.  Highlight *install_flash_player_9_linux* and click extract. 
9. In ***Extract*** window just click ***Extract***
10. Go to the **File manager** (see #4). Make sure  you have a folder *install_flash_player_9_linux*.
11. Double click it.
12. Double click *flashplayer-installer*.
13. In window ***Do you want to run "flashplayer-installer", or display its contest?*** you have four choices. Click *Run* (not *Run in Termina*l).
