### Post by robert shearer on 2008-09-25

> i need to upgrade to firefox 3, i have downloaded firefox 3.0.2.tar.bz2

First I am going to assume that you are using Ubuntu  and are a newcomer.
Welcome.
Firefox 3 ships as default for Hardy 8.04 but otherwise...

Installing apps in Ubuntu is different to Windows.

On the panel click   System=>Administration=>software sources
You will be asked for your password.Type it and enter.

The Software Sources window opens and on the first tab Ubuntu Software put a tick in the first four boxes and close. (they may be ticked already if so just close)

You will be prompted to update the sources.Make sure your internet connection is on then click on the reload box.

You only have to do those steps once.

Now, after the reload is finished click  System=>Administration=>Synaptic package manager  
   and this will open (it may ask for your password again)
Top right,   click on the search icon and type firefox in the search box that opens. Click on Search.

When the search stops look through the list of packages found until you find firefox.(just plain firefox)

Right click on it and in the box that opens select Mark for Installation.
(If a box opens showing other packages that are needed then click on Mark)

Click on the Apply icon in the top menu bar and again Apply on the box that opens.

After the Download and install is over then go to Applications=>Internet and firefox will be on the menu there.

You can use Synaptic to search for all sorts of applications and automatically download and install them following this procedure.
Hope this helps and post if you have Questions.
