### Post by Rubi1200 on 2012-02-18
Hi and welcome to the forums :-)

You can't install Wubi using USB.

The best thing to do is save the wubi.exe and relevant Desktop ISO in the same folder (e.g. on the Desktop) and then run the executable file.
