### Post by ajgreeny on 2006-12-06
When it freezes do you get any error messages or does everything stop, including the keyboard and mouse?  Are you using the live CD?  If so perhaps worth trying the alternate install CD which gives you more options, but is still easy to use.  What computer are you trying to install onto, some more specs would perhaps help.

Grub is the boot control software that normally takes over from the windows master boot record, and is placed onto the boot sector of the hard disk.  If you dual boot it is this software which gives you the option to boot into windows or Ubuntu.  There are other boot software options available for some other distros of linux, such as lilo, but I think most now use grub (**GR**eat **U**niversal **B**ootloader).  Don't be afraid of it, it works extremely well.
