**Kernel-source:**
In case you want to build a kernel using the same sources as the one in Ubuntu.

**Kernel-headers:**
This allows you to compile aditional 3rd party modules (drivers) into the kernel without compiling a new kernel by yourself. Useful for propietary nvidia and ati drivers.
