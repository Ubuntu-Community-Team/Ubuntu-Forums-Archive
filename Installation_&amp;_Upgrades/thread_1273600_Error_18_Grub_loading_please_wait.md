[http://members.iinet.net.au/~herman546/p15.html#18](http://members.iinet.net.au/~herman546/p15.html#18)

Error 18 is when the BIOS cannot access as the OS has exceeded the cylinder limit.  The limit depends on the vintage of your BIOS/computer.  See above linked.  

Before trying to create a boot partition, try to install within the limit first (see if installing within the first 137GB works)

Good luck.  Back-up.
