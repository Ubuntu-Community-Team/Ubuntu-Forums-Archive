Pretty much the only way is to reinstall from scratch using a 64 bit disk.

As to whether you should or not, I can only say that I have no issues whatsoever running 64 bit so I would disagree with your last sentence. A perusal of these forums will probably yield as many people decrying 64 bit as there are for it. This is probably not the best place to get a definitive answer.

Having said that, you state that you have an "extremely reliable 32 bit version". Why risk it unless you have concrete benefits, eg do you have more than 4 gig memory, do you encode/re-encode media, and so on.

Were it mine, I'd likely hold what I had till Karmic comes out and upgrade to 64 bit then. Actually, who am I kidding ? I'd put 64 bit on right now and try it. 

Make sure you are in a position where all your data is safely backed up, then try it. Then you'll know.

### Post by tenghoward on 2009-08-26

> How easy is to upgrade to the 64 bit version? Will the installer handle it, or will I need to delete or tell it to install on top of the 32 bit version?

Thanks again

It is NOT possible to upgrade to 64 bit from 32 bit. However, you can install them side by side or install 64 bit on top of 32 bit.

Should you decide to install them side by side, you only need to create separate partition(s) for 64 bit. No need to create another swap partition.
