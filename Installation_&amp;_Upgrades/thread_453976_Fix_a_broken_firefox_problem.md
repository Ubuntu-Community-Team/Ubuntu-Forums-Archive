### Post by ffderrick on 2007-05-25
Thought I'd drop a quick line about a problem I had with firefox crashing.
I used apt-get clean to get rid of old package downloads and then reinstalled everything that I thought had anything to do with firefox or web browsing. I use synaptic search to find all installed programs with firefox in the description or title.
This fixed my problem and I'm happily typing this message with firefox.
Opera is OK, but I need my extensions.
