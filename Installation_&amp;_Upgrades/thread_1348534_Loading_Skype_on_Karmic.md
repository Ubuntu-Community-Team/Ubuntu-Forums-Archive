### Post by gradinaruvasile on 2009-12-07
Go here:

[http://www.skype.com/go/getskype-linux-beta-ubuntu-32](http://www.skype.com/go/getskype-linux-beta-ubuntu-32)

Download & install that file.

Test:
Open a terminal (applications -> accessories -> terminal). Type:

skype

See if launches as it should.
If all goes well, quit skype and launch it from application -> internet -> skype.

I dont know what is supposed to be the problem with the AMD machine. There is nothing special about running any OS on it. 

If you think about the "AMD 64" "architecture" thing, forget about it. That expression has *absolutely* nothing to do with AMD, it means 64-bit instruction set plain and simply. And it works with Intel and AMD alike.
