### Post by ajgreeny on 2011-08-23
If using wubi you can install XP to the whole disk and not worry about formatting it at all, the installation will make the partitions needed.  Later when you add the wubi ubuntu installation, that will not require partitioning as the wubi application will make a virtual drive for ubuntu as it installs;  all you need to do is decide how large it should be.
