### Post by mcduck on 2006-01-01
You should install audio codecs.

First do this: [http://makuchaku.info/amnesty/#extrarepositories]("http://makuchaku.info/amnesty/#extrarepositories")

and then this: [http://makuchaku.info/amnesty/#codecs]("http://makuchaku.info/amnesty/#codecs")

After that your mp3 files will work, as will most of the other media files. Install also mpg123 if you want sound file preview to work in nautilus (moving mouse over a mp3 file's icon will cause the file to play)
