### Post by doas777 on 2009-04-24
yes with an if, and no with a but

all I can say is try the live cd first, but assume that a hardened install will be better (since you can;t install restricted drivers or configure anything on the livecd).

if you laptop is new, then use jaunty. if it's a few years old, experiment between feisty and hardy/intrepid. hardy worked better for my hardware than intrepid, but jaunty seems better than either of em.

with any pc (but laptops most of all), there is a bit of a gamble with hardware support. I would create a seperate home partition so that trying differant versions doesn't require much more than installing the os and apps. 

good luck
