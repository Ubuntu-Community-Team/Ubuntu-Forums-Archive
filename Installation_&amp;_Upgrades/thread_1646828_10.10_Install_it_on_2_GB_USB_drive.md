### Post by dabl on 2010-12-16
> **Derek10 said:**
> Hi!

 Is it possible to install in a 2 GB pendrive? 

No -- not the standard released system.

You might try limiting the number of packages to be installed with this:  [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

especially if you can find a way to install the "server" only, with no desktop environment.

Here are some more ideas:  [https://help.ubuntu.com/community/LowEndSystemSupport](https://help.ubuntu.com/community/LowEndSystemSupport)

If you can figure out how to install LXDE on the "server" version of Ubuntu, you might get lucky and get it to live in 2GB, with a GUI.

But I dunno if you can get it into 2GB, even doing those things.  You might be better off looking at a minimalist distro like slax, if you can't get a 4GB USB stick.

### Post by C.S.Cameron on 2010-12-16
You can do a persistent disk image install to 2GB flash drive no problem.
The install will take up 0.7 GB leaving about 1.2GB for the casper-rw persistence file, (or partition).

Try Startup disk creator from the Live CD or Universal from pendrivelinux.com

I am using a UNetbootin install on my EEE PC with a ext3 casper-rw partition.
