### Post by efflandt on 2012-04-13
i386 (or x86) is 32-bit.

amd64 is 64-bit (doesn't matter if AMD or Intel, it is just that AMD was first to come up with x86 compatible 64-bit cpu)

---

### Post by grahammechanical on 2012-04-13
It is to avoid confusion like this that from the release of 12.04 the recommended version will no longer be i386 but amd64.
