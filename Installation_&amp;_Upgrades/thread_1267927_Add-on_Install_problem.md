### Post by quixote on 2009-09-17
Firefox has preferences to block popups and warn about software that tries to install.  The latter should only be warning, not stopping it, but those are the two things I would check.  The first one is under Edit > Preferences > Content, Block popups.  The second is under Edit > Prefs > Security.
