### Post by Therion on 2009-06-25
Well it's going to take you hours to do another online upgrade too so I'm not sure I see the difference...  

Options:  If you don't want to do a fresh install using the standard-install CD from a download, which is what I really strongly suggest you do, you can order a free 9.04 standard install CD and then do a fresh install using that. 

You can download the *Alternate Install* CD and do an upgrade using that.  This would be my second choice.

I think there's a way to to do an upgrade from the Desktop Install CD but it's more complicated and I'm unfamiliar with it.

The choice is yours but my suggestion is that you download the 9.04 standard install CD via a Torrent (since your connection is slow you'll want the automatic integrity-check that torrents provide) and do a fresh install from that.
