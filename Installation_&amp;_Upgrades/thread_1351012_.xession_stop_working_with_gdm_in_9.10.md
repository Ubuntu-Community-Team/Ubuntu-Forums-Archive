### Post by leronen on 2010-10-13
Had the same problem. Solved by creating file /usr/share/xsessions/xsession.desktop, with following contents:
```
[Desktop Entry]
Name=Xsession
Comment=User-defined ~/.xsession
Exec=/etc/X11/Xsession
```
