### Post by kansasnoob on 2010-12-31
TestDisk and PhotoRec:

[http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step](http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step)

I'm not at all proficient with it, but if does work. Note where it says, "For lost/deleted partitions or deleted files from a FAT or NTFS file system, try TestDisk first - it's usually faster and TestDisk can retrieved the original file names."

You will of course need to use a Live CD or USB.
