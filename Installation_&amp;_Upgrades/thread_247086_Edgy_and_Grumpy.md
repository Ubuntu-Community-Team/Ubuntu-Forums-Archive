### Post by matthew on 2006-08-30
For those wondering what the OP is talking about: [https://wiki.ubuntu.com/UbuntuDownUnder/BOFs/GrumpyGroundhog](https://wiki.ubuntu.com/UbuntuDownUnder/BOFs/GrumpyGroundhog)

As far as I know this has not been fully implemented yet. Last I heard the chatter was that grumpy was to be a repository of the latest cvs builds of software and would in no way be considered stable...rather just a place for the devs to pull down the latest version of something for testing, etc.

Finally, to your question, as far as I have seen and know there are no grumpy repos at the moment, and if there were I would recommend staying far away from them unless you are involved in Ubuntu development.
