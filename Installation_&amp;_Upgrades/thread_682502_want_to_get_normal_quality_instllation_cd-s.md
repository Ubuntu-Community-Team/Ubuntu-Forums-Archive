I'm not sure I really understand what you're asking, but if you're looking for the checksums, you can find them here - [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)
Here is how to run the checkums on the ISO file:
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If the md5sum checks out, then you can burn the ISO as an image to a CD-R, but do it at a slow speed like 4x to help prevent write errors, and use quality media if possible.
