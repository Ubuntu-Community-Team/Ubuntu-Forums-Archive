### Post by Rocket2DMn on 2008-01-24
If you think the CD is ok and that it's your computer, you can try the alternate install cd which is a text based installer without the live session - it tends to solve a lot of problems associated with the LiveCD.
Go to [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and check the box near the bottom for alternate cd.
When you burn the image, burn at a slow speed like 4x to help prevent write errors, and use quality media if possible.
