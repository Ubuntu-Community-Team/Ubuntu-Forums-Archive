### Post by aysiu on 2006-12-03
A live CD on a 450 MHz computer (that probably has about 64 MB or 128 MB of RAM)? Not going to happen.

For systems like those, you should use the Alternate CD instead of the Desktop CD.

I have a 766 MHz computer with 128 MB of RAM, and I can't install off the Desktop CD--it requires too much memory.
