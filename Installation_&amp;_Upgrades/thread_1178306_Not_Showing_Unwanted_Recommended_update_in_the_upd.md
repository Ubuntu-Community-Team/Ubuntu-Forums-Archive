### Post by mcduck on 2009-06-04
You can lock the package version in Synaptic (select a package, then Package/Lock Version), but the truth is that you really should either update the package when there's updates available, or uninstall it.

There's no actual option to hide updates for a package, if you have something installed it only makes sense that you also use it and want to get updates for it. If you don't care about the package, why have it installed at all?
