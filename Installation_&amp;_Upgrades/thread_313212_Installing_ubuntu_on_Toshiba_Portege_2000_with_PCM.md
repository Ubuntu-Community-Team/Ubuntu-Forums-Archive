### Post by fishymark27 on 2007-03-10
Did you do the Text install or the OEM install? In text mode, my Portege 3110CT installed fine, its just that the pcmcia slot doesn't work at all on the xubuntu install. Works fine in windows 98 still though :(

Actually if you press escape at start up and then change the pcmcia settings in the bios, then you can install xubuntu fine. Toshiba also have updated bios installs on their website.
