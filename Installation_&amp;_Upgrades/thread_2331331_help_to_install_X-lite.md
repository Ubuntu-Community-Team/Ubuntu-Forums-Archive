### Post by deadflowr on 2016-07-21
Your best option to try would be to get the Windows version of X-Lite, and then install the windows emulator program, [WINE]("https://help.ubuntu.com/community/Wine").
Then try to install the package in wine.

Looking at [WineHQ]("https://appdb.winehq.org/objectManager.php?sClass=version&iId=16413") for any tests of X-Lite show it worked, albeit seven years ago.

Other then that, your options are to install Windows as a dual-boot or inside a virtual machine.
Windows version is the only way you could do it, as the mac version would be uninstallable, (it's incompatible with linux, currently).

Best of Luck
