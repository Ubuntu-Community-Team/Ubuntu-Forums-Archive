### Post by darkod on 2012-11-28
In your place I wouldn't try accessing the linux partition from windows at all. Windows can barely work with its own ntfs partitions, it can only mess up linux partitions.

If you need to frequently exchange data between the OSs, you should have one ntfs data partition. In that case both OSs can read it easily.
