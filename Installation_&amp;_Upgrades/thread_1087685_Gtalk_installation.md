### Post by Paul41 on 2009-03-05
As far as I know there is no Google Talk version for Linux but Pidgin works with GTalk and is installed in Ubuntu by default.

For the codecs open the terminal and run:

```
sudo apt-get install ubuntu-restricted-extras
```

That will give you what you need.
