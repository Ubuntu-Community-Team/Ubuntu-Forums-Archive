**Question:** What does GNU stand for? 

**Answer:** The name &#8220;GNU&#8221; is a recursive acronym for &#8220;GNU's Not Unix&#8221;.

**Question:** What is Ubuntu?

**Answer:** A GNU/Linux distribution, based on Linux, which is the kernel for the GNU/Linux OS. Linux is a POSIX implementation, like the Unix kernel, but Linux is not the same as Unix, so this is the wrong place to ask about the Unix installation process.

**Question:** Therefor I am at the wrong place, but isn't it nearly similar?

**Answer:** Yes, the unix and linux installation process maybe close together, since they share the POSIX base, but on the other hand, it is a total different story.

**Question:** So why don't you explain the process to me and stop this stupid Q/A game?

**Answer:** I have better things to do the next 2 days!
