### Post by DJ_Peng on 2008-10-09
This is what the script shows within Amarok:

> Usage:
Upon startup, the script will look for the "amarokFS" executable in your PATH, and if not found, will prompt you for its location. Once the script is running, you will have an "AmarokFS" menu entry in your playlist popup menu; now simply select "Run Amarok Full Screen" to activate AmarokFS.
If you have multiple displays, you can choose which screen to use by selecting the appropriate menu option from the list of choices. I'm unable to get it working within Amarok itself, but I'm a GNOME user so there's so cross-desktop issues to be dealt with. I was able to install the package nixie liked to which put an entry on my Sound & Video menu that let me start Amarok FS. I have to set up my playlist in Amarok itself, but the FS app does let you run with the playlist without having to show your desktop. To get out of FS mode just hit the Esc key.

I do recommend experimenting with different FS themes to find one you like best.
