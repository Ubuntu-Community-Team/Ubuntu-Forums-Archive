### Post by jcs296 on 2005-06-18
Slow down; most of what you are trying to do is very easy, but should be done in the right order.  Once you have an internet connection established, you can add mp3 support without any need for makefiles and compiling (or even fetching the codecs manually).  Instead you can easily add on applications by following the steps at ubuntuguide.org (please use this guide, as it is well tested and very comprehensive).  

But first things first - get the internet connection working. I haven't set up a dialup connection with ubuntu, but from looking at the properties setting I would suggest that you be sure to fill out all the information on the 'general' tab, as well as try out different ports in the 'modem' tab.  Also make sure you used the correct dial type.  If none of this works, I'd suggest not moving on to mp3 support yet but looking for other posts in these forums discussing dial-up issues, because all your other goals will be easier once the ubuntu system has working dialup.
