### Post by lukeiamyourfather on 2009-09-16
If you have a spare hard disk then you could install Windows on that. I usually disconnect other disks in the system when I install an operating system and then setup the boot loaders after the fact. If you're using GRUB its pretty easy to add Windows from a second hard disk. Other option would be to resize your current Linux install to make room for Windows on the same disk which would be my second choice. If a hard disk is used for each operating systems then you can remove one completely and not interfere with the others. Cheers!
