### Post by howefield on 2012-10-31
> **Christopher111 said:**
> So will my windows 7 apps work in ubuntu?

No, but you will be able to install an application called Wine which may allow some of them to be installed in Linux successfully. But you are probably better off finding and exploring the linux alternatives to your Windows applications if you can.

> should i get an anti virus for ubuntu.

No, but you can browse this page, there are valid reasons for using an anti virus from Linux which might apply.

[https://help.ubuntu.com/community/Antivirus](https://help.ubuntu.com/community/Antivirus)

>  IS there any firewall with ubuntu.

Yes, [https://wiki.ubuntu.com/UncomplicatedFirewall](https://wiki.ubuntu.com/UncomplicatedFirewall)

> Will i be able to run my NFS Games on ubuntu.

Maybe, see point one. Browse the Wine application database to get an idea of how likely you are to succeed.

eg, [http://appdb.winehq.org/objectManager.php?sClass=version&iId=1903&iTestingId=12012](http://appdb.winehq.org/objectManager.php?sClass=version&iId=1903&iTestingId=12012)

> First i'm gonna test it for a week or 2 and once all falls into place i will finally remove windows 7.

Good luck :)

Remember it doesn't need to be all or nothing, dual booting is a good way of keeping what you know and are comfortable whilst you take time to learn something new, Linux isn't Windows.

### Post by snowpine on 2012-10-31
I recommend that you run Ubuntu from a Live DVD or Live USB for a week before you install or make any changes to your computer. I'm not sure you fully understand what Ubuntu is therefore I think you should take it for a lengthy test-drive to see if it will work for your needs. It is a completely different OS from Windows and therefore most of your applications will not work. You need to make sure that a) you can find Linux alternatives to these apps; and b) all of your hardware is well-supported, before you take the plunge and install Ubuntu. 

Good luck, let us know how it goes! :)
