### Post by stoomaroo on 2007-10-06
I have an ATI card.

I ran the following :

sudo aticonfig --resolution=0,1680x1050

Done.  I have my login screen back!

Shout-out to these guys: [https://bugs.launchpad.net/ubuntu/+source/gdm/+bug/16472](https://bugs.launchpad.net/ubuntu/+source/gdm/+bug/16472)
