### Post by ubfan1 on 2018-05-06
Sometimes the installer cannot get on the network (no driver etc.), so install without trying to update (no network connection), then deal with the networking issue.  Sometimes you might need an ethernet cable to connect to get a wireless driver.  Did you hashcheck the downloaded ISO file used in the install media, and did you run the media check on that install media?
