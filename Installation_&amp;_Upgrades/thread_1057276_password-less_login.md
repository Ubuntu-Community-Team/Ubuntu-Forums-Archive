### Post by boof1988 on 2009-02-01
I use Ubuntu 8.04 and this is how it can be done on my machine:

System >> Administration >> Login Window

This brings up *Login Window Preferences*.  Chose the *Security* tab and then Uncheck the box for *Enable Automatic Login*.

Of course if you have a different distro, this may not be the correct menu path.

### Post by Partyboi2 on 2009-02-01
If you have the laptop set to automatic login you can change this by going to Login Windows (System>Admin>Login Windows) and on the security tab untick the box under "Enable Automatic Login"
