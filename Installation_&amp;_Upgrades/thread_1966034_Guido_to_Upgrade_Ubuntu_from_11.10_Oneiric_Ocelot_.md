### Post by tkoopa on 2012-04-26
Just wanted to mention, always make a backup. I upgraded from 11.10 to 12.04 using a live-usb and encountered an issue during the upgrade. It got stuck on the "replacing previously installed packages" process. So I disconnected the network which allowed it to finish. However all my installed software had gone ( tomboy, ecplise, mysql, apache, shutter, gimp, ... ) so I have to restore everything ( luyckily my /home was on another partition so when I re-install most programs use my old settings/data )
