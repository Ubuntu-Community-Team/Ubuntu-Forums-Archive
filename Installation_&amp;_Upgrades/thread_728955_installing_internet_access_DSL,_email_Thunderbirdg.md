As ajgreeny mentioned, this is a lot to take on at one time, but let's see if we can get you started.

If your computer is networked with a router you should be able to access the internet already by allowing the computer to use DHCP when it boots. If it is wired directly to the DSL modem, that's a case that I'm unfamiliar with as I've never been down the DSL road.

Using GMail with Thunderbird is a pretty straightforward process. First, go into your GMail settings and enable POP3 and/or IMAP. Once that is done, you should see a link that can walk you through the setup process (it's very much like setting up any other e-mail account).

As for the floppies and CD drive... if they are not working, you may want to submit your hardware specs here. It's possible, though unlikely, that the equipment is not supported. Just open up your file manager of choice and look for the device in question. For example, if you are using the standard Gnome load, click Places ~> Computer at the top left of your screen and you *should* see the CD drive and floppy.

Check back if you need anything else.
