### Post by darkod on 2010-05-14
It seems you found a solution but just to note that from what I've read around, software raid is said to be a better solution than the fakeraid (mobo bios raid) anyway. Especially if using a single boot, that would be the preferred option by many. The problem is when dual booting, windows can't use the raid also.

Of course, in your case we are talking about a server so the dual boot is off the menu anyway. :)
