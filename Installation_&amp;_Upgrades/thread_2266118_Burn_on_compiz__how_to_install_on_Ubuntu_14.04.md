### Post by grahammechanical on 2015-02-20
Compiz is the compositor/window manager used in Ubuntu and it will be around at least until sometime after Ubuntu 16.04 is released.

CompizConfiguration Settings Manager (CCSM) is a tool for tweaking settings and enabling special effects. It is no longer being developed by the original developer. Many of the special effects are now no longer included in the CCSM package. To get them we have to install compiz-plugins. It is in the Ubuntu Software Centre. These plugins are not officially supported.
