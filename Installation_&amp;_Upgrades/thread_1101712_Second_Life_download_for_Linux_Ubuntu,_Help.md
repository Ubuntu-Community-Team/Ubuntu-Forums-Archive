### Post by mikewhatever on 2009-03-20
Given that no installation packages are provided for Linux at secondlife.com, the guides you refer to probably suggested compiling from source. Simple is a relative concept, and compiling from source is not quite simple for an inexperienced user.
The good news is, there is a pre-compiled package for Ubuntu.
[http://www.getdeb.net/release/3124](http://www.getdeb.net/release/3124)

Just download the package to your desktop and click on it. After entering the password, it will start downloading the viewer from the home site. There is no progress bar, so just wait.
