### Post by scruffyminds on 2009-03-29
A much more sane approach would be to use VirtualBox instead.  Pick one for the host OS (Windows or Linux -- it doesn't matter.  VirtualBox is available for both) and install the others in VMs.  I have VMs with Ubuntu 9.04, OpenSolaris, Fedora 10, Windows XP and even MS-DOS 6.2 on my Ubuntu 8.04 laptop.
