OK i found a way to make xrandr permanent. You simply open a terminal , type ( sudo gedit /etc/profile ) it will open your profile file folder. At the bottom write what ever you want xrandr to do. For example i  wanted dual screen so i added ( --output VGA-0 --mode 768x1024 --output DVI-0 --mode 1024x768 --left-of VGA-0) look at this web page on how to use xrandr ( [http://www.intellinuxgraphics.org/dualhead.html](http://www.intellinuxgraphics.org/dualhead.html) )  and that made my dual conf. of xrandr run every time i start my PC. I must say that my xserver crashes ever now and then but it may be do to the fact that i am running the drivers that were installed buy default when i installed Gutsy. so the hole 3D Desktop+effects+Compiz+dualscreen may be a bit to much. Even thou i added a virtual 2560x2560 line to xorg.conf. my xserver kept crashing so i lowerd my res. to 1024x768 and i can use Gutsy with a occasional crash when least expected:( but I'm still working on finding y it dose that.
