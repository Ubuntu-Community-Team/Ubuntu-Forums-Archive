### Post by varunus on 2005-07-17
Installing software by heading to the developer's website is the windows way of doing things.  Ubuntu has a much nicer way to install software:  synaptic.

Go to System->Administration->Synaptic Package Manager.  This will list a large amount of packages you can install.  Almost every program you'll need can be obtained there, easily.  All the dependencies will be taken care of.  Its very very nice.

That said, I recommend you take a look at [www.ubuntuguide.org](www.ubuntuguide.org).  They have some nice software you can start with, and also tell you how to enable the "extra repositories" in synaptic (which i recommend you do first).
