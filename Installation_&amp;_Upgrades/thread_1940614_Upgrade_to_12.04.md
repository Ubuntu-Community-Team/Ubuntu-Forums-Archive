### Post by Mark Phelps on 2012-03-14
Don't use Wubi, but I do know, that in the past, doing such version upgrades would trash the Wubi installation -- which wouldn't be so bad if there was a way to roll-back to the previous version, but there isn't.

I've been told that such problems don't happen anymore, but Wubi is NOT designed to be used for the long haul -- across version upgrades.

If you are keeping it around that long, you should look into the alternative of having it be a separate install.  Those tend to handle version upgrades better.  See the details for doing this in the linked Wubi Guide:

[https://wiki.ubuntu.com/WubiGuide]("https://wiki.ubuntu.com/WubiGuide")
