### Post by etripp on 2010-04-26
Tried out ubuntu using wubi.exe. Liked ubuntu so would now like to remove windows and install ubuntu only. How do I do this from wubi?

Reason I ask this is because my Acer laptop would not boot from the iso disk. (where there is an option to install completely)

---

### Post by P4man on 2010-04-26
afaik you have to boot from a cd or usb. You can not uninstall windows while running windows (and if you use wubi, your OS is hosted by windows filesystem).

If you cant get a cd to boot, try a USB stick. Use unetbootin to make it bootable:
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
