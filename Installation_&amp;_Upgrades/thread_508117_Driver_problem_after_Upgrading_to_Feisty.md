### Post by linuxwizard on 2007-07-24
Ubuntu 7.04 (Feisty)

Go to > System -> Administration -> Restricted Drivers Manager and see if it is enabled. If enabled you should be ok nvidia drivers installed. If not enabled,  you can enable and it will install newest nvidia driver.

[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)
