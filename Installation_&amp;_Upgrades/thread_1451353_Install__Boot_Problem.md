### Post by oldfred on 2010-04-10
Do you have more than one drive? Normally f12 lets you choose which drive to boot from, not which partition as a one time choice. You can go into BIOS and set which drive as a permanent choice. 

New installs often install grub to sda even if you have system set to boot from sdb. BIOS will show it by hard drive type & size not sda, sdb.

If that is not the issue post this so we can see your entire boot configuration:

Boot Info Script courtesy of forum member meierfra
Page with instructions and download:
[http://bootinfoscript.sourceforge.net/](http://bootinfoscript.sourceforge.net/)
Be sure to highlight and use code tags (#) to make it easier to read when you post the results.txt.
