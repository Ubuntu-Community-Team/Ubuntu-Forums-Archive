### Post by an.echte.trilingue on 2006-11-29
Use the liveCD.  Do the install process up to the section where you format your drives.  Format the drive Ubuntu is on.  Cancel the install.  Get your Windows system recovery CD.  Boot with it in the drive and go to the recovery console.  Issue the command FIXMBR.

That is it.

However, I must reiterate, it is normal that Ubuntu will have some glitches for you qt first.  Installing an operating system is a very complicated task, even in windows.  After all, the copy of windows that most people have on their recovery CDs already has all of the bugs ironed out for them.  If, however, you take the time to fix these glitches you will find a new world of computing will open up to you that does not have the walls the windows world does, and there are many.

Take care,
-mat
