### Post by mgol on 2009-10-09
Ad. 1) If You don't boot the system it won't upgrade, obviously. ;) But if You run it even after a long time, it'll notify You about new updates You can install. I'm not sure what the question is...

Ad. 2) First of all, You can enter Your computer name during an installation process (or change it later). There should be no problems regarding Your internet connectivity on a new system - e.g. if You have a network where there is a MAC address filter, under Linux You see the same addresses...

Ad. 3) You can already make a USB image of the Ubuntu installer, I wouldn't wait for Unetbootin etc. However, 9.10 comes with a lot of new features (new Usplash, GRUB2, major kernel update, Kernel Mode-Setting etc. - You can see it at [the Ubuntu site]("http://http://www.ubuntu.com/testing/karmic/beta")).
