### Post by jonest1 on 2009-12-26
You should have a .icons directory in your home.  The . in front of icons means it is hidden.  In nautilus, go to View -> 'Show Hidden Files' to view the hidden directories.  

I have installed icons into the .icons directory and they work.  One caveat is that I'm not sure if .icons is there by default.  If you cannot find it, create a new directory named .icons and you're off and running.

BTW, there are also a system-wide icon directories, but they are typically not used for installing downloaded icons (/usr/share/icons).
