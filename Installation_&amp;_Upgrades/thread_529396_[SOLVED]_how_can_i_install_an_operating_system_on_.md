### Post by luisromangz on 2007-08-19
You create a new virtual machine, setting up the hard drive space, memory etc. Then you set up the machine to use an iso file or a physical cdrom unit and to mount it at boot. You start the virtual machine, and select to boot from the cdrom when it asks you to press a key if you want to select the boot device.

Then it should work as the setup will in a physical machine.

### Post by Scorpuk on 2007-08-19
If you are using VMware Player 2.0 then you can create your virtual machine here:

[http://www.easyvmx.com/](http://www.easyvmx.com/)


Select the icon to the right (easyvmx v2.0)

Rememebr to add in the second CD-ROM drive for the ISO file. (you may need to edit the .vmx file to enter the correct details for the iso file name and location. Unless you have a CD you can use in the drive of course. :)


Also best not to select the 3D option when creating your virtual machine.
