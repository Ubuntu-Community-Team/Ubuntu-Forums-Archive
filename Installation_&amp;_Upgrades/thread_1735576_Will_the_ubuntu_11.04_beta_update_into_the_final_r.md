### Post by howefield on 2011-04-21
You won't need to reinstall, updating as normal will give you the released article. 

Using lsb_release -a in terminal will give you something like this at the moment.

```
Distributor ID:	Ubuntu
Description:	Ubuntu Natty (development branch)
Release:	11.04
Codename:	natty
```

You'll know you are no longer using the Beta when the description line changes by dropping the words.. (development branch).
