### Post by Elfy on 2008-10-27
Indeed it is - if you have one of the variants you can install the others by either

```
sudo aptitude install ubuntu-desktop
sudo aptitude install kubuntu-desktop
sudo aptitude install xubuntu-desktop
sudo aptitude install edubuntu-desktop
```

When you get to the login window - use sessions to change what you wish to run
