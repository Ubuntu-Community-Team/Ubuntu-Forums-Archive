### Post by MAFoElffen on 2024-03-10
In Android Store: [Flasher]("https://play.google.com/store/apps/details?id=com.karthek.android.s.flasher&hl=en_US&gl=US")

  
[LIST=1]
[*]Connect USB stick with OTG cable. 
[*]Enable OTG on phone. 
[*]Connect OTG cable with phone. 
[*]Open Flasher app. 
[*]In Flasher app: 
[/LIST]
[INDENT]
[LIST]
[*]Select USB stick to flash 
[*]Browse and select ISO image which you want to boot. 
[*]Tap write button to start writing image on USB stick. 
[/LIST]
[/INDENT]
 
Test USB.

Flasher works in a r/w mode like 'dd' in Linux...
