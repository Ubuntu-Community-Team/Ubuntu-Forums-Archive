### Post by grahammechanical on 2013-08-24
When you boot Ubuntu and it loads to a desktop wallpaper without the Launcher or the top panel, then right click on the desktop and select Change Desktop Background. That will open System Settings. Now, load Software and Updates and open the Additional Drivers tab and experiment with another video driver. That might solve the problem.

How did you install Skype? Through the Software Centre?
