### Post by oldos2er on 2011-01-03
Did you try KDE by installing kubuntu-desktop? The only difference between Ubuntu and Kubuntu is the desktop environment, everything "under the hood" is the same; so if you removed the gnome packages you have installed, you'd essentially be left with Kubuntu. See [http://psychocats.net/ubuntu/purekde](http://psychocats.net/ubuntu/purekde)

Direct links:

32-bit here: [http://mirrors.cytanet.com.cy/linux/ubuntu/releases/10.10/ubuntu-10.10-desktop-i386.iso](http://mirrors.cytanet.com.cy/linux/ubuntu/releases/10.10/ubuntu-10.10-desktop-i386.iso)
