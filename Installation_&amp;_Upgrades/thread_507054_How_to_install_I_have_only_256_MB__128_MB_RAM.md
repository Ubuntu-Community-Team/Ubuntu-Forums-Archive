### Post by Neobuntu on 2007-07-22
256MB (actually a bit less) install Kubuntu Alternate CD and use Konqueror for your browser. You will be surprised. If that is not fast enough boot a Puppy Linux 2.15CE CD.

With 128MB, Puppy Linux 2.15CE. Just boot the CD. Running from memory (128MB) is automatic. Freeded up CD drive after boot is automatic. Saved changes is automatic (after telling it where). Use a Flash drive for backup. There is little advantage to NOT  running Puppy from CD. This makes install nill. Reason? SPEED!

Kubuntu is the best do all OS on the planet. It works best on fast, newer computers. 

Why not Xubuntu? I found it weird and limited. Sorry, that's my OPINION. While puppy is perhaps more "different", the gains are worth it and it makes a fine full featured desktop. The odd little utilities work and work well with attention to no nonsense usability. If you look hard enough you will find Puppy less "perfect" but if it does your tasks then there you go. New Puppy versions are already out, but I can not speak to them.

Of course, it should go without saying, if you can get more memory, do so (and run Kubuntu although 256MB is enough). I assume the hardware is not upgradeable or not worth it. I understand this.
