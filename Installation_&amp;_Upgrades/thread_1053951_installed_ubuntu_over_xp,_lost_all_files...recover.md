### Post by maxhavoc on 2009-01-29
Eek, sorry to hear that. Unfortunately since you installed a new OS with a new file system over an existing file system you have lost everything and it will be practically impossible to recover it. Your only possible solution it to send the drive to a commercial data recovery company which will charge at least $500.

You might want to try SpinRite at [www.grc.com](www.grc.com). It's the most low level data recovery tool that I know of and might help.

### Post by Mark Phelps on 2009-01-29
If you have access to a Windows box, I would do the following before you spend MAJOR bucks to have your data retrieved:
1) Download GetDataBack from the link below:
2) Install GetDataBack on the windows box
3) Hook up your reformatted drive to the windows box -- but be sure you don't write to it
4) Run GetDataBack to see how well it locates files. It won't actually recover them because it's a trial version -- but you can see how well it does without having to buy it.
5) If it does a good job of finding your files, purchase it online, activate your trial version, and redo the recovery.

BTW, I'm not affiliated with them in any way -- I've just used their products numerous times and found them to work very well.

[http://www.runtime.org/data-recovery-software.htm]("http://www.runtime.org/data-recovery-software.htm")
