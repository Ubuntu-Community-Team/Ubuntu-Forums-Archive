### Post by bcbc on 2012-10-31
> **jcurtiswood said:**
> Possible to install this new variant via Wubi? Not sure if I want to keep it just yet and don't want to have to rebuild BCD if I choose to remove.
It's not supported by Wubi. You might be able to trick wubi into installing it, but not directly from the ISO. 12.10 wubi only installs Ubuntu/Kubuntu/Edubuntu/Lubuntu/Mythbuntu

Re tricking Wubi, here's an example of what I mean: [http://ubuntu-with-wubi.blogspot.ca/2012/10/tricking-wubi-installing-xubuntu.html](http://ubuntu-with-wubi.blogspot.ca/2012/10/tricking-wubi-installing-xubuntu.html)
