### Post by Paul41 on 2009-06-12
The new version of iTunes does not work in WINE. You have several other options though. GTKpod as already mentioned, Rythmbox (the default music player in Ubuntu), and Banshee just to name a few.  All of these work like iTunes.

VirtualBox is a way to run other operating systems in a virtual machine.  So you could run Windows inside of Ubuntu.
