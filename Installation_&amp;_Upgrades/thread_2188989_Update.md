### Post by grahammechanical on 2013-11-20
You do nothing. Update manager will do all the work. That notice is telling you that the package Mulit-protocol Instant Messaging Library will be updated in the future but it is not yet in the repositories. That is why there is no tick mark against that package.

Packages that will be upgraded have tick marks against them and we can un-tick them if we wish to prevent the package from being upgraded. But in this case the package is not yet available for downloading. Expect this situation to change over the next few days.
