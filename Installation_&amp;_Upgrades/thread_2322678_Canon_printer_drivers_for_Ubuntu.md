Have a look here:  [http://askubuntu.com/questions/690419/canon-pixma-mx494-driver-close-enough-for-mx492](http://askubuntu.com/questions/690419/canon-pixma-mx494-driver-close-enough-for-mx492)

Which refers to this link for downloading:  [http://www.canon.co.uk/support/consumer_products/products/fax__multifunctionals/inkjet/pixma_mx_series/pixma_mx494.aspx?type=drivers&language=&os=Linux%20%2864-bit%29](http://www.canon.co.uk/support/consumer_products/products/fax__multifunctionals/inkjet/pixma_mx_series/pixma_mx494.aspx?type=drivers&language=&os=Linux%20%2864-bit%29)

After downloading the Linux driver, you need to download from the same site the "IJ Printer Driver Ver. 5.10 for Linux (Operation guide)"
which contain the installation instructions for various linux distributions (including Ubuntu).

If you have questions or problems with specific commands, let us know what the command is and what "unexpected" response you are getting.
