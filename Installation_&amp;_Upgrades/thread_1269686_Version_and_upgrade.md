### Post by sthestrup on 2009-09-18
Thanks, found out my kernel build was for 8.04.. 

To others in same situation (newbies like me):
The reason for 8.10 not showing in update manager was my setting in synaptic was wrong. It was set for "Long term releases" instead of "Normal releases". After that the 8.10 update was available from Update Manager.
