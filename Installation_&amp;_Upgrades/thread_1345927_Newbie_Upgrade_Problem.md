> **AJH101 said:**

No, the Ubuntu upgrade process is very nice. When you update your system with Synaptic/apt, it will search the mirrors for software you have, and upgrade everything that is outdated unless you specify certain packages to be held. This will keep all software you have installed up to date and current. 

You can also upgrade from one release to another using this. On my laptop I have not done a clean install since 7.10, I just upgrade via the update manager a few weeks after the new release is out.
