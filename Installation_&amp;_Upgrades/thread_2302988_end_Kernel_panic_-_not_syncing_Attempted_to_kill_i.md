### Post by michas2 on 2015-11-15
I solved this problem. In a bootable window, select probably the "systemmd" or smth similar. After that you'll see a window with an error "ACPI PCC probe failed starting wersion 219". You have to use a shortcut "ctrl + alt + f1", after that you will be logged into the system. That the error did not occur you should update "Kernel" and "Grub".
