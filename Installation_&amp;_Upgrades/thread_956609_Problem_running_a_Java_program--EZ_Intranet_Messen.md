### Post by jespdj on 2008-10-23
Which Java are you using? There are several implementations of Java available on Ubuntu 8.04. The one that is the most compatible is Sun Java 6. Install it like this:
```
sudo apt-get install sun-java6-jre sun-java6-plugin
```
Check if it is the Java that is used by default:
```
sudo update-alternatives --config java
```
It looks like you are currently using GNU Java, which is a slow and incomplete implementation of an older version of Java.

### Post by jamesstansell on 2008-10-25
The OP _was_ running the GCJ Java, but apparently doesn't need the browser plugin.  Even though GCJ works very well in some situations, ezim reqires java 6, so gcj will not work.

MightyFrog, the error message said 'file not found.'  I'm not sure if that was for ezim.jar or if was for a file inside the archive.  When I rn using gcj I get a different error, along the lines of class not found, for one of the swing classes.

openjdk-6 runs ezim ok on my machine (ubuntu 8.04.1) but sun-java-6 works just a little bit better for the message area.

The sun-java6-plugin package will pull in the sun-java6-jre package, which is the one you really need here.
