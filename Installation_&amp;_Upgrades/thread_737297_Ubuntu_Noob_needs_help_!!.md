### Post by PmDematagoda on 2008-03-27
If you have 4Gb of RAM then you would have to acquire Ubuntu 64 bit instead of Ubuntu 32 bit in order to use all 4Gb of RAM since Ubuntu 32 bit does not support more than 3.2 Gb of RAM(except if you compile the kernel with support for 4Gb of RAM).

You can play WoW on Ubuntu 32 bit but I am not 100% sure if you could do that on Ubuntu 64 bit due to the different architecture.

You can refer on using WoW on Ubuntu using this [page]("https://help.ubuntu.com/community/WorldofWarcraft").
