### Post by readybrew on 2012-07-10
I ran into a problem today while trying to create a new VMWare virtual machine running Ubuntu Desktop.

This was using VMWare Workstation 8.0.4 build-744019 on Win7 x64

The first ISO I tried was ubuntu-12.04-desktop-i386.iso which I downloaded today from the web site.  No matter what settings I tried changing in VMWare Workstation or in the VM's BIOS, I always got "Operating System not found" error.

I decided to try the next oldest LTS which was ubuntu-10.04.4-desktop-i386.iso.  I reset the BIOS of one of my new VM's to factory defaults and re-pointed to the new ISO.  Everything worked fine the first time.
