### Post by sudodus on 2014-09-30
Welcome to the Ubuntu Forums :-)

1. Please specify your computer

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card -- the graphics card can often be hard to get running
- wifi chip/card

It will make it easier for us to give relevant help.

2. Please tell us what version (14.04.1 LTS or other), and what flavour (standard Ubuntu, Lubuntu, Xubuntu ...) that you have downloaded.

3. Check the downloaded iso file with ***md5sum*** according to this link

[https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)

4. Learn more about installing from USB at this link

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

5. And finally, do *not* install at once.

[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it]("http://ubuntuforums.org/showthread.php?t=2230389")
