### Post by kle on 2007-12-10
When you get the menu from the install cd run the option that checks your CD. 
If it is OK, you can continue using the CD you have. 

Laptops tend to be a bit tricky. When running your CD chose the last option  (6?) and add to the string you find there: noapic irqpoll
That should get you started. 
 
If you intend to install the CD, not just run it, yuu should read up a bit about dual boot. There are a lot of good directions available. Personally, I prefer this one.
[http://users.bigpond.net.au/hermanzone/p14.htm](http://users.bigpond.net.au/hermanzone/p14.htm)
Good luck
