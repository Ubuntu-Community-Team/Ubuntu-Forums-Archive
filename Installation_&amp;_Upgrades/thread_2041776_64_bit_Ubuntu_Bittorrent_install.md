### Post by Moose on 2012-08-13
Just use one of the torrent downloaders on the Ubuntu Software Centre. If thats too difficult then install Wine and then download Bittorrent. I have done this because when i had Ubuntu and Windows dual booting i used Bittorrent to download to my windows partition. But you can use Bittorrent on Ubuntu installed on wine just like you can on windows. But before use you have to choose your output (finished downloads ) folder before you use it because it will think your pc is windows and try to download to a nonexistant folder.
