### Post by az on 2005-04-08
Yes.  Change all that is Warty in synaptic wo Hoary.  That is, for every repository, edit it and change Warty to Horay.  Update and then do a smart upgrade.

To use the cd, just insert it and it should upgrade.  In general, the cd is another repository you add to synaptic.
