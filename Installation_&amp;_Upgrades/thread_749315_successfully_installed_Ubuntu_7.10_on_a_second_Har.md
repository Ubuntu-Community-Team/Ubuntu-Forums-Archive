### Post by Damsongs on 2008-04-08
hi there ubuntu users,Im fairly new to all this, like most of us,  i was very curious about this new system, being a windows user for over 11 years, I had a little bit of trouble installing Ubuntu on a second drive at 1st  with Windows XP being the 1st OS which is on Drive C , I went through the installation process from the Live CD, 
i found it pretty quick. I installed Ubuntu on my Local Drive D which is 80 G, and I chose to install it with the guided option, there is an option where you can  install the GRUB loader, I decided to load the grub on my windows drive, when installation completed and rebooted, I got this message Error 17, cannot mount selected partition, there was NO option to boot windows, i was very worried at this time, but I found a way to get into ubuntu without having the grub menu bother me with the error 17 message, all I had to do was hit F12 at bootup time and it gave me an option to boot from either Drive C or Drive D, I chose drive D to load ubuntu and it worked fine, but at this time I still couldn't boot windows, so I downloaded super grub disk.iso  and burnt that on a rewritable CD and whacked that in, then i chose to uninstall grub from the MBR, and I rebooted again and i got my windows back, I can now get into both OS's windows loads by default but if I need ubuntu, i hit the F12 key at boot time and choose Drive D to load Ubuntu. just thought i'd share my experience installing Ubuntu with XP
