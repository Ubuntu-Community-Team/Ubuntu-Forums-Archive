I did a clean install from a 19-April daily build of the release candidate on 22-April.  Right after install, it saw 94 updates.  I installed them and left the machine till today.  I expected there to be more stuff to get it up to final, but there were only 2 updates.  Apparently, it upgraded itself to final on 22 April.  Runs great.

On my main machine at home, well, look at my signature line.  I will soon be adding >8.04 to the end of the list.  I have had no problems with the last 2 upgrades.  The first few were "interesting" and forced me to learn a bit.

I would say if you have a good working install of Gutsy and don't have many 3rd party repos, especially for your video drivers, it will probably just upgrade with little to no drama.
