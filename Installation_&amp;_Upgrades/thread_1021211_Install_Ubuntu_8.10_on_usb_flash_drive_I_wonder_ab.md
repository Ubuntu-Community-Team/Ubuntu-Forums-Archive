I have an OCZ Rallye 2GB. It is supposed to be high-speed, the specs from the manufacturer claim "Up to 28MB/s (read), 15MB/s (write)". I put Ubuntu 8.10 live CD (it needs cca 640MB + I set 400MB for the persistence file which still leaves about 1GB free for whatever) on it via "Create USB startup disc" tool. The boot time is acceptable and when it finally reaches desktop, it is definitely faster than the regular live CD.

So unless you like waiting, my advice would be to get tha fastes one you can afford. :)
