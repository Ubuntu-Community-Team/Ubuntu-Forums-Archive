IMHO, if it ain't broken don't fix it.

Run 9.10 as LiveCD mode, have a good poke around around your system - sound / video / network etc.

Yes, you can take a 'photo-copy' of your existing installation - I always do before I do either re-partitioning, major hits to my LAMP system or from my 9.04 --> 9.10 upgrade.

My backup script is based on this thread -->

[http://ubuntuforums.org/showthread.php?t=35087](http://ubuntuforums.org/showthread.php?t=35087)

It seems to have all points covered :D

For backing up of your FF stuff, what list of additional programmes, there's threads for those, but that thread will give u your complete system 'in a tin' that can be put back as if nothing had happened.

Backing up packages is covered here ---> [http://ubuntuforums.org/showthread.php?t=1057608](http://ubuntuforums.org/showthread.php?t=1057608)

FF has options to export bookmarks, for passwords u need to add PasswordExporter (it's on about version 1.2)
