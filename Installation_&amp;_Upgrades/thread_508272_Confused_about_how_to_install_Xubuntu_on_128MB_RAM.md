### Post by SinoDave on 2007-07-24
I originally installed Edgy to an old AMD machine with 128MB RAM, then recently upgraded to Feisty (though with so many problems that I may do a clean reinstall soon).

My experience was that the desktop CD was so slow as to appear broken, so I used the alternate install CD. Presumably the desktop CD would have worked if I had allowed it the time to run, but we're talking about 30 minutes or so just to boot the thing! You have to bear in mind that it has to create a RAM disk to install from, so you're not even getting 128MB or swap space.

My suggestion is to go with the alternate install CD. It's text-based, but very intuitive.

Good luck!

-SinoDave
