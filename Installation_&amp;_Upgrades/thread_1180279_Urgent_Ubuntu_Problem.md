### Post by Mighty_Joe on 2009-06-08
You are a little light on the details, but I can tell you Unetbootin does not install Ubuntu (or any other Linux distribution).  It creates a live CD on flash media (see the [FAQ]("http://unetbootin.sourceforge.net/#faq")).  
What you *probably *want to do is use Unetbootin to create a bootable Live CD on a USB flash drive, boot from it and install Ubuntu from the flash drive, but again, you are a little light on the details of what you are trying to accomplish.
