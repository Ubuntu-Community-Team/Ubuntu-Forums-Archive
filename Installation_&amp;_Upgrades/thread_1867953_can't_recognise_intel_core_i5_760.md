### Post by oldfred on 2011-10-23
Welcome to the forums.

You are installing the wubi version. The amd-x64 version is correct for your system as that version is for all the current standard Intel & AMD 64 bit chips.

Wubi is a way to have an extended test of Ubuntu without having to create partitions. But is is running inside the Windows NTFS partition and relies on the Windows boot loader to start.

[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)
[https://help.ubuntu.com/community/Wubi](https://help.ubuntu.com/community/Wubi)
