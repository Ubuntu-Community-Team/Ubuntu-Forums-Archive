### Post by JBAlaska on 2007-11-03
Try converting the rpm to a deb with alien;
```
$sudo apt-get update
$sudo apt-get install alien
```

```
$sudo alien -k name-of-rpm-file.rpm
```

```
$sudo dpkg -i name-of-deb-file.deb
```
