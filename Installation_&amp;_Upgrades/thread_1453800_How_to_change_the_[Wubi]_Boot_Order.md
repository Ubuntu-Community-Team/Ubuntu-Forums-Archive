### Post by lisati on 2010-04-13
Please clarify: are you dual-booting or do you have a Wubi install? 

As I understand it, "Wubi" installs and "dual-boot" aren't the same thing, and the boot-up process for each kind of installation is slightly different. "wubi" uses a portion of your Windows partition, and a dual-boot installation uses a separate partition for each OS.
