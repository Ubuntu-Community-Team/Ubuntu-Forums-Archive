### Post by Mark Phelps on 2011-12-07
To the OP ...

If all you want to do is remove Vista and replace it with Ubuntu, then you don't need the Vista password to do this; instead, you can do the following:
1) Boot from the Ubuntu desktop CD
2) Choose the Try Ubuntu option
3) Open the Disk Utility and delete all the partitions
4) Reboot from the Ubuntu desktop CD and install to the drive.

If your PC has a BIOS password, you can't reset that using password reset utilities as they only work on OS passwords.
