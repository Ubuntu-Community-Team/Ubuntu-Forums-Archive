### Post by tgoose on 2007-05-29
I don't know, but I'd say it more likely that some other component (hard disk or RAM) is failing than your optical drive: optical drives *expect* errors in reading so they're often more forgiving of them. The easiest thing to do would be to get hold of an external optical drive to try installing from, but if you can't find one to borrow then it is possible to boot from LAN. I'm almost positive this is not a simple procedure, though. It's also possible to boot from an ISO on the hard drive with the method described on [http://ubuntuforums.org/showthread.php?t=457318&highlight=boot+iso](http://ubuntuforums.org/showthread.php?t=457318&highlight=boot+iso) , but that of course requires some Linux flavour to already be installed, so it's not a lot of use unless you could try it with some small, easily installable distro like DSL that manages to install fully before anything breaks.
