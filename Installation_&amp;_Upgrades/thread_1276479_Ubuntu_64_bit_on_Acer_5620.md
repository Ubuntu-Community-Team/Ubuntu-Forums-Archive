### Post by Elemen7s on 2009-09-27
Basically the 32Bit OS can only read up to 3GB of RAM and the 64BIT OS can read more than 3GB.

So if you dont have more than 3GB you shouldnt install the 64Bit. The 64 Bit uses more Ram than the 32Bit, so if i were you i wouldnt install the 64Bit as you only have 2GB
