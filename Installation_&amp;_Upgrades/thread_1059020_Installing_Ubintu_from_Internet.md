### Post by taurus on 2009-02-03
You can certainly install intrepid from the LiveCD (desktop CD).  Do you see the install icon on your desktop?

By the way, make sure you run the check cd for defects when you first boot from the CD.  If there are errors to the disc, no need to go further because it will bomb you out eventually.  

1.  Make sure you do the checksum on the ISO image right after you downloaded to check the integrity of the file.

2.  Burn it at a slow speed like 4x.

3.  Run the check cd for defects option at the initial screen.

[https://help.ubuntu.com/community/BurningIsoHowto?highlight=(burn)|(iso](https://help.ubuntu.com/community/BurningIsoHowto?highlight=(burn)|(iso))
