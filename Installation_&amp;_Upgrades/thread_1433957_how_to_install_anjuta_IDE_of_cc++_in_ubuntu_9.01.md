### Post by UCSBGauchosRock on 2010-03-19
Open Terminal/Ubuntu Software Center/Synaptic Package Manager...

Terminal Directions
1.Open Terminal 
2.sudo apt-get install anjuta or....
3.sudo aptitude install anjuta

Ubuntu Software Center
1.Open Software Center
2.Search for Anjuta
3.Click Install

Synaptic
1.Open Synaptic
2.Search Anjuta
3.Click and select "Mark for Installation"

Cheers
