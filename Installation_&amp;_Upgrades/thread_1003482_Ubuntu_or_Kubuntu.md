### Post by OrangeCrate on 2008-12-06
Personally, I like Gnome over KDE. I guess it's a matter of whether you like the crisp look of Windows (KDE), or the softer look of a Mac (Gnome). I prefer the later. But, to answer your question...

The internals are the same, they're both Ubuntu inside. Frankly, it's a matter of personal choice. But, if you install KDE over Ubuntu, you'll have access to both by simply switching sessions at log-in. If you install just Kubuntu, that's all you'll have.

Instructions to add KDE to Ubuntu are here:

[http://psychocats.net/ubuntu/kde](http://psychocats.net/ubuntu/kde)

### Post by m_duck on 2008-12-06
If you're going to be using KDE for sure anyway, it would make sense to just install Kubuntu. This is the same as Ubuntu underneath but has KDE as its DE along with KDE apps. If you are wanting to play around with Gnome as well, you could install Kubuntu, then install the "ubuntu-desktop" package, or install Ubuntu and then install the "kubuntu-desktop" package. The DE can then be selected on the log in screen.
