### Post by Rocket2DMn on 2008-04-02
How did you do your upgrade?  Did you use the update manager (this is recommended)?  You can see more here about that upgrade you did - [https://help.ubuntu.com/community/DapperUpgrades](https://help.ubuntu.com/community/DapperUpgrades) - the part about changing sources.list is not well supported and can cause problems.
Right now you are very far behind in the distros.  I think there is going to be a 6.06->8.04 upgrade system when Hardy is released next month, but I wouldn't bank on it working 100%.  I would suggest doing a fresh install, either with Gutsy right now, or with Hardy when it is released.

As far as all these problems go, you can try removing the packages and reinstalling them (except the (k)ubuntu-desktop metapackages).
