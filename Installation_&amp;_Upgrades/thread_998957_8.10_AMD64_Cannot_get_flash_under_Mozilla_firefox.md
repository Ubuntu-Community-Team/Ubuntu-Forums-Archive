### Post by Jammy4041 on 2008-12-01
Hi, and may I just say welcome to the forums!

there is a x64 flash player in the ubuntu repos.

(Here it is) [http://packages.ubuntu.com/intrepid/flashplugin-nonfree](http://packages.ubuntu.com/intrepid/flashplugin-nonfree)

Just try opening a Konsole and type:

```
sudo apt-get install flashplugin-nonfree
```

You should then be able to get flash running.

If that doesn't work, try downloading the AMD64 version of Flash from the download link above.
