### Post by Mark Phelps on 2009-02-17
Longer answer (in case you want to know why NOT) ... Linux requires the partition to be formatted as Linux-swap; MS Windows requires partitions formatted as FAT32 or NTFS.  Reformatting the MS Windows partition would remove everything in it.
