### Post by darkod on 2010-02-28
First boot with the ubuntu cd and select the Try Ubuntu option. That will load and run ubuntu from the cd and it will show potential issues with hardware. Check whether everything is working, ethernet, audio, graphics with correct resolution, etc.
If you notice any issues search for fixes so you're ready. Most problems can be solved.

If everything is working as expected in the live desktop, and after you have copied all data you need from XP, you can either click the Install icon on the live desktop, or boot with the cd and select Install Ubuntu in the main menu to start the installer.

Unless you need specific setup, in step 4 of the installer simply select Erase and use whole disk. That will wipe the hdd and install ubuntu on the whole hdd.
