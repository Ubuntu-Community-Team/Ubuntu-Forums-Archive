### Post by az on 2005-12-19
So, your system is not able to boot from cd?

SBM (Smart Boot Manager) is available on the install cd.

I think it is in the install folder.  You can use rawrite to make it into a bootable floppy.  It boots and detects your bootable hardware and shows you the options it found.
