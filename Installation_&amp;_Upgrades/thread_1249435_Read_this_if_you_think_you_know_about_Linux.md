### Post by tgalati4 on 2009-08-25
The PrtSc (printscreen) button is the quickest way to make a snapshot of the entire desktop.  Alt-PrtSc will make a snapshot of just the active window.

Googleearth has a flyby utility where you can define Points A and B and it will fly between them.  Then you can post a link (or attachment) with just the flyby points.  This is sort of like a bookmark in googleearth.
