### Post by bscbrit on 2007-02-03
No, its not silly thinking - they are both linux.  However, you could still experience problems if the configuration of each installation is not identical, or at least similar.  Worth a try but make a backup of the existing fstab before you make any changes and that way you can always use the recovery console to correct it again.
