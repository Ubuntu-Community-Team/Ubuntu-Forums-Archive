### Post by C.S.Cameron on 2018-11-04
Mkusb +1, It can make either Live or Persistent installs.

Or if you are on Windows, UNetbootin or Universal should work for a Live or Persistent install.

If you use SDC for a Live, (non-persistent), install note that any extra space on the flash drive will be unusable.
