### Post by lemming465 on 2009-11-01
> [does]update manager ... attempt to keep the changes i have made to 9.04
Yes, for configuration files that have changed it will offer to keep the old one, replace it with the new one, or let you do some kind of merge.

> my download speed when i upgrade is an order of magnitude slower than normal
Probably the servers are overloaded with eager upgraders and downloaders.  It will likely be better in a week or so.  Bleeding edge types can avoid the rush by upgrading around the beta or release candidate stage, but need to be prepared to cope with extra breakage, file good bug reports, and live with hundreds of megabytes of extra package churn as the release approaches.

It is possible to run upgrades using the *alternate install* CD's, which can cut the on-line time some if you don't have zillions of extra packages installed.
