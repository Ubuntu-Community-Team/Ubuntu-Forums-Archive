### Post by Mark Phelps on 2011-04-20
It's NOT a program -- it's a Linux distribution, known as a Distro.

And, unless you are experienced with installing and using Linux distros, I recommend you avoid it -- as it is a Beta and very buggy.

You would better, if you're new to this, to install Ubuntu 10.10.  You can always upgrade that to 11.04 later -- after they release it and work out some of the major bugs.
