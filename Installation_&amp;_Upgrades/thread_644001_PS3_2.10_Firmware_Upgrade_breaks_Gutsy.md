### Post by Ben Cook2 on 2008-03-12
You Can Fix That Without A Reinstall By Entering At The Kboot Prompt: ```
old
``` In Order To Allow You To Just Hit Enter Again At The KBoot Prompt, Run: ```
sudo gedit /etc/kboot.conf
```
And Change The Line Default From: ```
default=linux
``` To Read: ```
default=old
``` Note This Will Not Restore Wifi, It Just Restores Your Ability To Boot By Using The Old Kernel. :evil: DO NOT CHANGE Any Other Lines! Just The Line default. :evil:
