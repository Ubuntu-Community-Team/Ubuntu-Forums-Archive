Check to see if the Bios on that old lappy even supports USB boot.  If not, I don't see how you'll be able to install from usb stick. (unless maybe there's a Bios update that adds that feature).

Other options:

-take out the hard drive, hook it up to another machine and install ubuntu,  Then put it back.
