### Post by Frogs Hair on 2011-02-02
What may  be a better idea is install the Kubuntu desktop environment from your current installation and when you log into Ubuntu you will have the option to switch between them.

After installation below the greeter box when you type your password the choice will appear , select what you want to use and press enter.   

sudo apt-get install kubuntu-desktop
