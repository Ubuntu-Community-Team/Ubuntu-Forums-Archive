### Post by ugm6hr on 2008-05-30
The "Desktop" CD is for fresh installs.  The "Alternate" CD allows upgrades or fresh installs.

If you have a separate /home partition, the "Desktop" CD will allow you to do a fresh install without losing any data or settings from /home (just use the same username / password).

If not - I don't think it's possible.
