### Post by darkod on 2010-07-02
Yes.
Open Software Sources in System-Administration and look around at the options (I can't remember exactly right now). There is an option whether to offer upgrades on new versions, or only on new LTS versions. If you select only on LTS you will not get prompted to upgrade until the next LTS is released.
