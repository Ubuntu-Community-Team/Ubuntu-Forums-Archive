### Post by Partyboi2 on 2009-06-05
Hi, you could move your /home to its own partition then do a clean install. If you do decide to go that way when you are installing 9.04 you would need to choose the manual partitioning option and reset the mount point for /home as well as / and choose the option to format / but do select the option to format /home. 
[http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)
