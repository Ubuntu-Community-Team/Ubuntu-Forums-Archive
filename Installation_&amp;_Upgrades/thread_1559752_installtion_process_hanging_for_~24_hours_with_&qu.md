### Post by uRock on 2010-08-24
It is either a bad cd or possibly a bad HDD. Restart the disk and hit a button at the first purple screen to get the menu you need. Select the option to test the CD, if the CD is good, then boot it into live mode and go to System> Administration> System Testing and let it test your hard drive.
