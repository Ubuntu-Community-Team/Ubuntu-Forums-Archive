I also leave the default update monitor turned on that watches all installed packages and lives in the system tray. Very handy.

### Post by aysiu on 2007-01-12
Updating and upgrading are not the same thing.

Updating means you're checking for more up-to-date versions of the packages you have installed--these are usually security updates.

Upgrading means you're changing to a new version of the entire operating system--you would upgrade from 6.06 to 6.10, for example.

[https://help.ubuntu.com/community/EdgyUpgrades](https://help.ubuntu.com/community/EdgyUpgrades)
