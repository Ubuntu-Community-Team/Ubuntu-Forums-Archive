### Post by dandnsmith on 2009-06-07
Download the iso file for (eg) Ubuntu 9.04 desktop,
burn it as an iso to a CD,
boot the CD,
select the *run Ubuntu from the CD* option.

I seem to remember that the ubuntu downloads section has helpful tips on how to create the CD (in case you haven't done it before).
