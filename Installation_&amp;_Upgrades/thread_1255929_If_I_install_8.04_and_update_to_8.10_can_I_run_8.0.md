### Post by tommcd on 2009-09-02
If you install 8.04 and then upgrade to 8.10, then 8.04 will be overwritten with 8.10; and 8.04 will be gone. 
It is possible. although not easy, to downgrade back to 8.04 from 8.10. Your mileage may vary though. See this:
[https://help.ubuntu.com/community/DowngradeHowto](https://help.ubuntu.com/community/DowngradeHowto)
Your best option would be to install 8.10 (or 9.04, which is what I would recommend) to a separate partition on your hard drive. Then you would have 8.04 and 8.10 (or 9.04) on separate partitions. You could set up grub to boot either version.
