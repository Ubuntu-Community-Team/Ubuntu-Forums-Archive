### Post by erickb on 2006-12-31
Hi,

This is more a comment than a question. I'm installing Ubuntu for the first time, after doing many other Linux installs & many Windows installs. It was so smooth, I started thinking to myself, "Is this finally Linux for the desktop?" I decided to pretend that I was a complete novice, and see how the process felt. It was pretty darn smooth, with only a couple of UI glitches when setting the time:

 - First, it asked for my "location". I couldn't set my actual location (Boston). That's OK, though, because I think all it really wanted was my time zone, so New York was good enough. Maybe call this "time zone" instead of "location"?

 - Second, it kept showing the wrong time, because my time zone hadn't been set. I tried using the "set time" dialog, but that dialog wasn't synched with the information displayed on the installation dialog (in particular, it didn't show my selection of New York city). In fact, Ubuntu had detected everything just fine - I just had to click "next" and the proper time and date showed up. If this were broken into two screens - set time zone, then set time (if need be) - this might be less confusing.

Those two glitches aside, it looks great so far!

Eric
