### Post by lemming465 on 2008-11-25
If it took more than an hour, it has probably run into something that will never finish.  It's time for the power button (Ctrl-alt-F1 to swith to a text console if you aren't on one already followed by Ctrl-alt-delete is an alternative).  The install will have failed, but the hardware itself should be OK and the disk partitioning should also be OK.

Before trying the install again, run the CD self-test to verify it downloaded and burned and reads correctly.
