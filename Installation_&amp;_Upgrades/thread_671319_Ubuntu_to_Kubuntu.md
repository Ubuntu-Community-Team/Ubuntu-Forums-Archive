### Post by atoponce on 2008-01-18
There are a number of options available to you.  You could install Kubuntu in a virtual machine, dual boot, or just install the 'kubuntu-desktop' package.

As far as performance, this is a big debate in the Linux community as to which desktop is more lean, Gnome or KDE.  I think you'll find the answer to be how productive you are in your desktop environment, what bugs you and what doesn't.
