### Post by ajgreeny on 2015-01-28
Try using the install medium again but just boot to a live desktop; don't try to install again.

Now use the browser to come back to this thread and go to the **Boot-Repair** link shown in my signature and follow the instructions to run the utility's Boot-info script; do not run the default boot repair at this stage.
This will give you a pastebin link that you can post back here in this thread, and which may help us to see what is going on.
