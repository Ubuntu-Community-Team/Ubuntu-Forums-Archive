### Post by Qrk on 2006-05-04
I don't think it will work. The difference between x86 and ppc isn't trivial; it affects all the applications, not just the kernel. It isn't like i386 and i686, where you can run i686 apps on a i386 kernel and visa versa. i386 compiled applications won't work on ppc, however.
