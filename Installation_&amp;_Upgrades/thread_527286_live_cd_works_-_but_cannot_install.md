### Post by merlinus on 2007-08-16
You do not have enough RAM to install from the live cd.  Use the text-based Alternate Install.

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Tick the box below the green Start Download text, then click the text to begin.

Preferably, try xubuntu.  It will run much better on that machine.
