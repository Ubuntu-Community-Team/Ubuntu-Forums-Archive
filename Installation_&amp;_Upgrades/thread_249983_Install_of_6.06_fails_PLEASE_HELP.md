### Post by daou on 2006-09-03
It's a text-based install. This means the installation won't boot into the Ubuntu desktop and won't load as much stuff. Quite easy, kind of like a Windows installation.

You can get them from a lot of places, for example here:
[http://iso.linuxquestions.org/version.php?version=362]("http://iso.linuxquestions.org/version.php?version=362")

Make sure you choose the alternate CD, not the desktop CD which is probably what you have.
