> I am completely new to Ubuntu. Have newly assembled computer not formatted yet. Also have Ubuntu cd. Do I just start by booting from cd?

Yes. Then you can either install or run the CD without installing, which is what I'd recommend. That way you can make sure everything works. If it does double-click on "Install" and go to it. One thing I forgot. If your system won't boot from the CD go into the BIOS and change your boot order so the CD is first.
