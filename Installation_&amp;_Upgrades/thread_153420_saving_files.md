### Post by RRS on 2006-03-31
A third partition formated as FAT32 can be used by both windows and Ubuntu for file storage, editing, etc.

At the partitioning section of the instalation select "manual"

[This site]("http://users.bigpond.net.au/hermanzone/index.htm") will walk you thru the entire process.
