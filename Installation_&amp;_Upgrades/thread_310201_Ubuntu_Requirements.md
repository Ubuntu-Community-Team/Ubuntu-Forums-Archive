### Post by d3v1ant_0n3 on 2006-11-30
I'd say [ this](http://ubuntuforums.org/showthread.php?t=259901) thread gives a good example of how flexible ubuntu (linux in general, really) can be about hardware requirements.

You should be able to run Ubuntu with less than 256 Mb of RAM, but the more the better. @56Mb is required for the live cd, however.

Ubuntu is very good about dualbooting- it will find your windows partitions and should be able to resize them well without damaging your windows install.
