### Post by ronparent on 2009-09-03
Uninstalling is as simple as deleting the unwanted ubuntu partitions. If you reinstalled grub with the new ubuntu, you might have to remove the lines for booting the old install from the /boot/grub/menu.lst. How you handled the situation so far, is as good as any, especially if you need to retrieve data from the old system.
