### Post by lutrafobic on 2005-12-31
in Gnome:   use the 'Synaptic Package Manager' - located in the Menu under System/Administration to install new stuff/programs.

in 'terminal': ```
 sudo apt-get install wine
```

or (if you downloaded a .deb file):   ```
 sudo dpkg -i  the_file_you_downloaded.ded 
```
