Hi there,

> **LoganHarbour said:**
> 1. Make ubuntu believe the external HD is a real HD, same as windows.Unlike Windows, Linux makes no distinction between internal and external devices (beyond very low-level stuff, obviously).

> **LoganHarbour said:**
> 2. Partition the external HD giving 80GB to ubuntu, and then the remaining space to another partition that windows and ubuntu can read and write to.You can partition external devices any way you want to. :-)

> 4. Could ubuntu be able to also read my regular laptop drive, and write/read to it.That depends on what filesystem you're using, but it's unlikely you're using one Linux doesn't support.
