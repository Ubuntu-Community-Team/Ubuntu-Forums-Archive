### Post by john280z on 2006-06-06
davidbd,

   Using ubuntu-6.06-server-i386.iso and doing a Server install is aprox 430 Meg Hard drive space. I have done several Server installs using P1/233MHz/64 Meg Ram with no problem.
   For even LESS Hard Drive usage the Xubuntu-6.06-Alternate-i386.iso will require about 325 to 330 Meg Hard drive space.
