### Post by apparle on 2009-08-03
Linux has a desktop environment(DE) installed on top of it.
All the GUI you see is decided by your desktop environment

The default Ubuntu comes with GNOME DE

Kubuntu comes with K DE

Xubuntu comes with Xfce DE

Each DE comes with its own set of applications although you can mix various DE 
For more info
