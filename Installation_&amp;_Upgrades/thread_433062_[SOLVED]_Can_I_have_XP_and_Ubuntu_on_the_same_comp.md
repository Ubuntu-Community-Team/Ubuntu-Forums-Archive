### Post by davidgarcin on 2007-05-04
yes, the Ubuntu installation CD allows you to safely resize your Windows NTFS partition and make a new Linux partition for Ubuntu in its place.  It will also install a bootloader named GRUB which will allow you to choose which OS to boot when you turn on your computer.  It's fairly straightforward, but do backup first, just in case.
