### Post by gsmanners on 2010-01-09
Make sure you combine the settings files into one tar.gz file, then unpack that to the target home folder. That way, you preserve the permissions and ownership flags.

See: [http://library.gnome.org/users/file-roller/2.28/file-roller-creating.html.en](http://library.gnome.org/users/file-roller/2.28/file-roller-creating.html.en)
