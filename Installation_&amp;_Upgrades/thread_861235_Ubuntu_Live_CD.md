### Post by snowpine on 2008-07-16
Definitely check out those links Puma gave you!
And to answer your question about partitioning, there is a partition editor on the Live CD (under the System menu). Using a Live CD is the best way to partition, because the hard drive is "unmounted" (not in use) when you are running the Live CD.
Definitely educate yourself on what you need to do *before* messing with the partition editor, so that you don't have any more "accidents." :)
