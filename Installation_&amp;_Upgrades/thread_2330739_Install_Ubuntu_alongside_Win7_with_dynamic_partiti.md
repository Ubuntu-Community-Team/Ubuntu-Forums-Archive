> **akshayjn said:**
> I have not any primary partitions instead i am having all the dynamic partitions.


They are what are causing the problem. Dynamic partitions are a proprietary Microsoft format to which you can only install Windows - You cannot install Ubuntu on a drive with dynamic partition. Here is a link to a post which contains more information and some useful links.

[http://ubuntuforums.org/showthread.php?t=2325331&p=13492758&viewfull=1#post13492758](http://ubuntuforums.org/showthread.php?t=2325331&p=13492758&viewfull=1#post13492758)
