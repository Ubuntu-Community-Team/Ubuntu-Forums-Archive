I like it better when I use 2 HD's. XP pro on one HD and Ubuntu 10.04 on the other HD.

In my case I installed grub on the second HD so XP is completely unaffected.

For ASUS motherboards I press F8 during startup, chooses whatever HD I want to boot off from and boot into XP or Linux.

Pay attention during installation for an "advanced button". This is where you tell the installation prog where to install grub.
