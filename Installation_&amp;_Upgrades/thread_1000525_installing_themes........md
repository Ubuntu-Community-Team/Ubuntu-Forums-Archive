### Post by aysiu on 2008-12-03
If you're downloading a GTK, Metacity, or icon theme from [http://www.gnome-look.org](http://www.gnome-look.org), then just download the file (do not double-click it or extract it or use a terminal command on it). Then go to System > Preferences > Appearances and click the Install button and find the file you downloaded.

If you downloaded a GDM theme, go to System > Administration > Login Window and install the theme there. Same deal - do not extract or double-click the download. Leave it as a compressed file.
