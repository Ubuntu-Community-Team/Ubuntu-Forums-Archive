### Post by Partyboi2 on 2008-09-10
If you haven't already done so you will need to set the jumpers on the hard drives so that one is set as Primary and the other one set to slave. You may need to physically pull the hard drives out and look at the diagram showing the jumper settings as different hard drives have different jumper settings. Or if you know the brand you can probably google for the jumper settings. Once you have them correctly setup boot a ubuntu cd and restore grub.
