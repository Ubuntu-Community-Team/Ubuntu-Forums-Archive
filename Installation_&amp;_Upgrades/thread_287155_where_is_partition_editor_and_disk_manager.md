### Post by tpg on 2006-10-28
You can install the partition editor (gparted) by running
```
sudo apt-get install gparted
```
which will then appear under System->Administration.

I have no idea what happened to the disk manager, I'm not sure what the package is called, so can't find it in the repository. :-k
