If I choose one directory to be synchronised with "Ubuntu One", this directory will be synchronised with both computers, so with the big laptop and the netbook.

I assume the hard disk of my netbook would be filled until it is full and then I would get an error message. I admit that I have not tried it but that is what I expect.

My question:
Is there a way that a specific directory of the big laptop is synchronised with Ubuntu One but not synchronised with my netbook? If so, how can I do it?

### Post by pfnorris on 2010-10-22
As far as I am aware this is not currently possible on Ubuntu One. The key word here being synchronization. I had a similar need and could not find a way to achieve it under U1.

However, other services are available that provide facilities additional to sync, such as backup, which is really what you are describing. One that I found which did just what I needed was Spideroak. With this I could back up any folder from any subscribed PC to the service, and then choose separately which if any I wanted to sync, and specify the specific PCs to sync with.

If you want to have look it's [here]("https://spideroak.com/"). Like U1 it offers a free 2GB basic service.

Regards

Nobby

P.S. Just for clarity I'm not affiliated with it in any way. I just think it's a pretty useful service.

### Post by ellis rowell on 2010-10-24
I use Ubuntu one for storage and synchronising.

Files which are stored go into folders as they would on your machine. If they need updating at any time you update the file (in the Ubuntu One folder on your desktop). This will then update on Ubuntu One which will in turn update on other machines which you have synchronised on Ubuntu One when they log on.

I have three data files which get regularly updated (one updates every day). This is usually done on my desktop, the laptop only gets used occasionally. When the laptop connects to the Internet it automatically updates the files on that. The process works the same in the opposite direction. I have the applications on each machine. My present usage on Ubuntu One of 2GB allowance is 25% (500+MB).
