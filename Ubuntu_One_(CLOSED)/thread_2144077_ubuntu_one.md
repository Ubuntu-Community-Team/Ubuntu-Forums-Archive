### Post by Irihapeti on 2013-05-11
If you go to the web interface and click on "My account" (top right-hand corner) and scroll down to "Devices" (near the bottom), is there any mention of your 12.04 system there?

If there is, delete it. 

Then on your 12.04 system, go to the passwords and keyrings application and remove any authentication token that relates to Ubuntu One.

That leaves you with a clean slate. Now try logging on again and see if it works better this time.
