The avi file is in the* top level* of your storage. 

I have a folder that I synchronised to test U1 then 'unsynchronised' again using the FAQ at

[https://wiki.ubuntu.com/UbuntuOne/FAQ#How%20do%20I%20stop%20synchronizing%20a%20folder%20outside%20my%20~/Ubuntu%20One%20folder](https://wiki.ubuntu.com/UbuntuOne/FAQ#How%20do%20I%20stop%20synchronizing%20a%20folder%20outside%20my%20~/Ubuntu%20One%20folder)

I then followed the instructions at 

[https://wiki.ubuntu.com/UbuntuOne/FAQ#How%20do%20I%20remove%20a%20folder%20I%20previously%20selected%20to%20synchronize%20with%20Ubuntu%20One?](https://wiki.ubuntu.com/UbuntuOne/FAQ#How%20do%20I%20remove%20a%20folder%20I%20previously%20selected%20to%20synchronize%20with%20Ubuntu%20One?)

and found that the empty folder disappeared in U1. You need to check how to use the u1sdtools for removing a file as opposed to a folder.
