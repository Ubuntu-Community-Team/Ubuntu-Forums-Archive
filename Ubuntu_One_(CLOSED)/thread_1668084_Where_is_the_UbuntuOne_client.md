### Post by Jackson Tan on 2011-01-16
According to [this tutorial]("https://wiki.ubuntu.com/UbuntuOne/Tutorials/FileSharing#Sync%20status"), the "!" icon indicates that the file is not synchronised. I presume this meant that it is in the queue to be synchronise; the sync-daemon is dealing with other files at that time.

I'm not sure which Ubuntu One client you're talking about. Perhaps you're referring to the Ubuntu One indicator, which has an installation instruction [in this thread]("http://ubuntuforums.org/showthread.php?t=1594301").

### Post by duanedesign on 2011-01-21
The Ubuntu One Indicator is no longer present in newer versions. It was replaced with the Ubuntu One Preferences. Found by going to Me Menu --> Ubuntu One or System --> Preferences --> Ubuntu One.

You can find info about installing a third party U1 Indicator and getting more info about what U1 is doing in [this sticky thread.]("http://ubuntuforums.org/showthread.php?t=1594301")
