Yes, computers hooked up to the same Ubuntu One account should sync files with the Ubuntu One cloud as well as to each other. We have been experiencing some performance issues with our messaging infrastructure that may make the sync between computers slow. The best way to force the sync between the two computers is to disconnect the Ubuntu One client and then connect.
