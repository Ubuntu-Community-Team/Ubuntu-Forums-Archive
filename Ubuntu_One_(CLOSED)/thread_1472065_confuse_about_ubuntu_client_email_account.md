Files in the "~/Ubuntu One/Shared with Me" folder are files that someone else shared with you. If you want to share files with Jack, then you would right-click on a folder synchronized with Ubuntu One and select "Share on Ubuntu One..." and put the email address/contact in that you want to share with. The same applies for Jack. If Jack wants to share with John, then he would do the same. If what you really want to do is sync the same files across multiple computers, then you should just setup each computer with the same Ubuntu One account (same email address).
