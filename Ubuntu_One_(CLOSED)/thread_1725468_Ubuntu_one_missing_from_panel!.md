### Post by duanedesign on 2011-04-13
Ubuntu One no longer has an applet. Their is a [third party app indicator]("https://launchpad.net/one-indicator") you can install. If that is what you have you can start it with:

```
/usr/bin/ubuntuone-indicator
```
