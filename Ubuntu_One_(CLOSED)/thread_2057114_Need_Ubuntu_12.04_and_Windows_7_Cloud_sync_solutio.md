### Post by Dragonbite on 2012-09-13
Currently the one "official" release that is cross-platform is Ubuntu One.  

I haven't tried the 3rd party projects for connecting to Google Drive but Ubuntu One does work on Windows, slightly interestingly.

For example you can have your Pictures folder in Windows sync with your Pictures folder in Ubuntu.

If you don't need the cloud component you can easily synchronize between two machines using FreeFileSync.

Edit: Insync lists only Windows, Mac and Mobile as supported OSes.
