### Post by Irihapeti on 2012-03-18
I want to be able to upload some files to Ubuntu One so that I can publish them, but I don't want them to sync to my computers.

I've looked through the documentation, but I can't see a way to do this. Any file that I upload ends up in my Ubuntu One folder on my local machine.

Can someone tell me 1) if this can be done and 2) what I've overlooked?

---

### Post by duanedesign on 2012-03-20
The Ubuntu One folder is always syncing, you can not unsubscribe it. You could remove your computer from your Ubuntu One account. Then upload a file using the web interface. Since no computers are connected to the account the file will not sync anywhere. You could also create a folder in your Ubuntu One account. Open the Ubuntu One control panel on your computer and select the Folders tab. Then deselect the option 'Sync Locally'. Then upload files to that folder in the web interface. Since Sync Locally is not selected files in that folder will not sync to that computer.
