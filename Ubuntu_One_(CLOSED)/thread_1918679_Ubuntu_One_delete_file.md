### Post by duanedesign on 2012-02-02
Yes. All changes made locally are synced to all your devices, and cloud space. This works both ways. Changes made in the cloud space are synced locally.

This is why we generally do not recommend using Ubuntu One as a back up solution. The exception being the [Deja Dup/Ubuntu One]("http://duanedesign.net/blog/?p=264") integration in 11.10. This does more of an upload to the cloud space, rather then a sync.
