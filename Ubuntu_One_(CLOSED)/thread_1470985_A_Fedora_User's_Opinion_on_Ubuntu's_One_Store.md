### Post by thewOndErEr57 on 2010-05-03
hi Ubuntu'ers.

I'm a die hard Fedora fan, since the days of Fedora Core 6. 

Yesterday I was asked to fix a PC for a friend, so I did the natural thing, by installing Linux. I chose to install Ubuntu 10.04, because I read time and time again about Ubuntu's user friendly feel, and also because I wanted to have a look myself at the advances in the Ubuntu experience (the last install I'd seen was 6.10

I was *amazed* with the installation procedure in 10.04. But was more amazed with the Ubuntu One integration with Rhytmbox. I love listening to music, and love linux. What a great way to contribute to Linux - buy music you love, and in doing so, doing "your bit" for the cause.

I loved the idea so much, I wrote a blog posting about it. Find it here:
**[Ubuntu One Music Store]("http://linuxsoftwareblog.com/?p=478")**

Leave any comments you may have on the posting, I'd be interested to see if there is mutual applause for this latest offering, or if I've overlooked any issues? 

Thanks,

Rob
