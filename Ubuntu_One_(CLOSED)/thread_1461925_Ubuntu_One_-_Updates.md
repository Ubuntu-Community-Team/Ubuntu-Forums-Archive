A number of problems people are seeing right now are related to issues on the server side. We are experiencing rapid growth in our user base (good problem to have!) and as a result the system is struggling to keep up. The good news is that we are currently rolling out some server side updates that will go a ways in solving these problems. There is no silver bullet to fixing the scaling issues, so we'll continue to improve this situation on a continuous basis. I apologize for the instability you're seeing in the service and appreciate your patience.
