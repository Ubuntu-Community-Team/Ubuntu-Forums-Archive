### Post by gpeck157 on 2012-10-05
This is the response from Ubuntu One support: 

"Those aren't duplicate folders. The files in ~/.local/share/ubuntuone
are a local cache that Ubuntu One keeps on the computer to help keep
track of changes to files. It's best not to mess with this folder in
general.

The ~/.ubuntuone/Purchased from Ubuntu One is where music purchased from
the Ubuntu One store are stored."
