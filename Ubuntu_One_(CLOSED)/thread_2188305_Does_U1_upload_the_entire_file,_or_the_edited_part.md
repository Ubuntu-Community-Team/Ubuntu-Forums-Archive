### Post by thnewguy on 2014-01-19
Not 100% sure, but I think Ubuntu One uploads the whole file. I'm mostly basing this on the status messages u1sdtool displays when it shows the file transfer queue. It appears as though the new version of the file is uploaded under a temporary name, the old version is deleted and then the new file is saved under the proper name.
