### Post by Paddy Landau on 2011-12-18
Are you doing this for backup or for some other reason?

If it is a simple one-off, you can use an FTP program. The best one I know of is FileZilla, which I believe is available in Ubuntu Software Centre.

If you want a proper off-site on-line backup system, I recommend [SpiderOak]("https://spideroak.com/"), which I have used for a long time. It offers only 2Gb free, but it is cheap to add 100Gb.
