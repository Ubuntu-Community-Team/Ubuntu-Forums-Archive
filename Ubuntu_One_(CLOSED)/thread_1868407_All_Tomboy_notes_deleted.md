### Post by Lars Noodén on 2011-10-24
> **Retor said:**
> But a BIT sceptical of Tomboy now.

There are other options, if you are interested.  They include Gnote (which is a C++ version of Tomboy), [BasKet](http://basket.kde.org/) is very nice, [Knotes](http://pim.kde.org/components/knotes.php) is simple and to the point, and [Zim](http://zim-wiki.org/) is a desktop wiki.  So if you like wikis, zim rocks.  As bonus, none of the four are built on Mono.  So there are more choices than just Tomboy.
