### Post by krackersk on 2010-04-16
Basically you can sync stuff between different computers which are attached to the account.

Any files that are in your "Ubuntu One" folder in your "home" folder will be uploaded to the web, stored there and downloaded to any other computer attached to the account. Just copy files into this folder and it happens automatically.

You can also sync other things like, Tomboy notes, Evolution contacts, bookmarks and you can buy music!

check out:
[https://wiki.ubuntu.com/UbuntuOne/Tutorials](https://wiki.ubuntu.com/UbuntuOne/Tutorials)

for some more info

### Post by thebear78 on 2010-04-21
> **aion4217 said:**
> Just copy folders into the Ubuntu-One directory in your Home folder.

That's one option. Many people asked for it so Lucid users can now select any folder within their Home folder and mark it for synchronization. If the same folder already exists on any of your other synchronizing computers, the contents will merge. So you don't have to drag ~/Documents or ~/Music to the Ubuntu One folder. You can just mark them to synchronize. You can also disable this if you would like to stop synchronizing a specific folder.

Another thing is that you cannot mark a folder for synchronization if one of its child folders is already synchronizing. For example, if you first set ~/Documents/School to synchronize, you cannot then set ~/Documents to synchronize as well. The way to do this, though, is to disable sync for ~/Documents/School and then mark ~/Documents for synchronization.

We'll soon publish a lot more information about this on the Ubuntu One website. We hope you enjoy this new Ubuntu One feature!
