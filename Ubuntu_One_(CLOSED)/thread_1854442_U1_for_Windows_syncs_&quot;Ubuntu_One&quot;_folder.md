### Post by Alderman on 2011-10-04
U1 is installed on my Win machine.  It synchronzes items in the ubuntu one folder, but not folders i add using the Windows client.

OS: Windows 7 Home Premium Service Pack 1. 64-bit version.

My problem, described in terms of steps to reproduce.

1. Install Ubuntu One and set up account. 
2. Note that installation creates %USERPROFILE%\Ubuntu One\
3. Files and folders placed in %USERPROFILE%\Ubuntu One\ sync with cloud. Files placed there are copied to cloud, files uploaded to cloud from web page are copied down. All is good. 
4. Create folder %USERPROFILE%\SomeFolderName\.
 5. Using Ubuntu One Windows client application, add %USERPROFILE%\SomeFolderName\ to sync list. I do this by clicking the "Add a folder from this computer" button. 
6. %USERPROFILE%\SomeFolderName\ appears on web console and in Windows Client. "Sync Locally" box is checked. 
7. Add files to %USERPROFILE%\SomeFolderName\
8. %USERPROFILE%\SomeFolderName\ does not sync with cloud. Files are not copied to cloud.
 9. I stop/restart the Ubuntu One processes on the Windows machine, also disconnect/reconnect the windows client. No sync. 
10. Check %USERPROFILE%\Ubuntu One\. It still works and syncs with the cloud.

So what am I doing wrong?
