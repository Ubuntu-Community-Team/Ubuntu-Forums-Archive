### Post by duanedesign on 2010-12-12
> **dawiba said:**
> I found it. The Purchased Music folder is a hidden folder #-o

For technical reasons it has to be a hidden folder. In case anyone stumbles on this thread looking for that folder it is:

~/.ubuntuone/Purchased from Ubuntu One

You may have to go to the menu View --> Show Hidden Files or Ctrl + h to see the .ubuntuone folder.
