### Post by Kelvari on 2012-07-27
Actually, the simple way of stopping the deluge that you claim to have experienced, egeezer, is to just delete the old files that you no longer want. If there are files that came down that you wish to keep, copy them out of your Ubuntu One folder and into another folder. Ubuntu One automatically syncs between any and all devices that you have set up - delete a file on one, and you delete it on all of them.
