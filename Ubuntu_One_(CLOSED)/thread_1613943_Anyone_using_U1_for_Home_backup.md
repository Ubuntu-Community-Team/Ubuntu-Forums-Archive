### Post by solitaire on 2010-11-05
If you install any Backup Software you can point it to save in the "Ubuntu One" folder and exclude that folder from backups.

Then the resulting file will be synced to the U1 Cloud...
But it's not a "nice and clean" way to backup.

There are other 3rd party options designed for online backup...
