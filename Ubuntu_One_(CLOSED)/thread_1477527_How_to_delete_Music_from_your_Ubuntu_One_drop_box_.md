I had trouble getting this to work today.  

I'm not using Ubuntu One for anything but purchasing music.  I have my own method of backing things up which is off site already.

The music needs to be deleted out of

.ubuntuone/Purchased from Ubuntu One/

Obviously you need to make sure that you've saved it somewhere else first.

Rhythmbox will try to tell you that it's in 

.local/share/ubuntuone/Purchased from Ubuntu One/

That, however, is a sym link.
