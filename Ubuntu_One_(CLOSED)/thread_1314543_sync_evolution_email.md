Is there a way to use Ubuntu One to sync Evolution email folders so that it is possible to see all  of the stored messages regardless of the computer being used?  Or, for that matter, is there a way to do it without using Ubuntu One?

### Post by tcole on 2009-11-05
The best option in that regard (right now) is probably to use an mail provider which offers IMAP.  Then your mail can be on the server, but you can also configure Evolution to keep a local mirror.
