### Post by slickymaster on 2014-04-08
AFAIK one of Ubuntu One's purpose was to provide access to one's data from anywhere, any time. With encryption, that would not be possible.

From the [Ubuntu One help FAQ]("https://one.ubuntu.com/help/faq/what-security-and-privacy-policies-does-ubuntu-one-have/")'s: > We do not store your files encrypted in our data storage since we need them unencrypted in order to send them to the people you choose to share with. If you are concerned about storing your files unencrypted in the Ubuntu One cloud, you could always store the files already encrypted so Ubuntu One never sees the plain text files. Doing so may prevent the proper functionality of some Ubuntu One features such as multiple computer synchronization, web browser access, and sharing with others.
