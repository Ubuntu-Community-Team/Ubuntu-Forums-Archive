### Post by pfnorris on 2010-10-22
As far as I know, before you can do any synchronization you have to sign up for an Ubuntu One account [here]("https://one.ubuntu.com/"), something that is not always immediately apparent.

Once you have done that the first time you launch the Ubuntu one client from System-Preferences, it should prompt you to register your machine by bringing up an option in your web browser. You should then be able to synchronize folders.

Even then, you may not see any obvious signs of activity (I don't). I don't even see the icon indicators mentioned above, even on folders that I know full well have been synchronised. However, shortly after starting a sync the folder appears on the files tab, on my online dashboard on the Ubuntu One web site.

It does work (on the whole) but I think is best described as a work in progress at present.

Hope this helps

Nobby
