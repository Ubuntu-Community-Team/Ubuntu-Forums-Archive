Currently, this won't work. But, we are planning on providing this functionality in Lucid Lynx and our PPA. Instead of being confined to only syncing files in the ~/Ubuntu One folder you'll be able to tell Ubuntu One which folders and files to synchronize. It's a very popular request and we're excited to start offering it in the not too distant future.
