### Post by afrowildo on 2010-08-03
The biggest advantage, at least in my mind, that U1 has over Dropbox (and other similar services) is the open framework that allows application developers to sync data and preferences between different computers running the same app. It's the same thing that syncs up your Tomboy and Firefox data, and any developer can tap into it in the same way. I've been browsing the [Dropbox addons directory]("http://wiki.dropbox.com/DropboxAddons") and I haven't seen anything provides that sort of functionality.

### Post by asavik on 2010-08-05
Ubuntu One's main advantage is that it's open and free software, while Dropbox isn't (albeit neither costs anything up to 2 GB storage.)

Dropbox' main advantage is that it works.

Ubuntu One's main disadvantage is that it doesn't work.

For example, the only thing the client in the taskbar is capable of, is freezing. Ubuntu One doesn't warn you when something has gone catastrophically wrong, but hides the error in a log file somewhere. So you think tour files are safe, while they're actually anywhere but in the cloud. And so on a.s.o.

(I subscribed to Ubuntu One for about a year before I finally gave up on it.)
