### Post by VooDooSyxx on 2012-11-17
Not really. I don't see anything in the terms and conditions that would prevent someone from uploading a basic html page to their account and then publicly sharing the U1 URL. That said, Ubuntu One is not a web server or web hosting account and is not going to have an apache server with php or python etc. available. You would more than likely want to just use a cheap web hosting account for hosting a web page. There are plenty out there for very low cost or even free.
