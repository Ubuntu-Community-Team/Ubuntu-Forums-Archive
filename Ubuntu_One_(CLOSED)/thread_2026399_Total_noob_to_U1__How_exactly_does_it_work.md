### Post by Shadius on 2012-07-15
I might've found an easier way rather than extracting each folder. You can go into System Settings, under the System category, click Backup (this seems to be the Deja Dup Backup Tool) and you're given the options of "I want to restore files from a previous backup..." and "Just show my backup settings". Choose the first option to restore from a previous backup and select the folder you want to restore from.
