### Post by duanedesign on 2010-01-27
The workaround for this is as follows:
Upon receiving OpenID failed error with 'nonce' info, go to [https://one.ubuntu.com/](https://one.ubuntu.com/)
The login attempt _was_ successful, the problem is that the page is not redirected properly.


The Ubuntu One Team is aware of this problem and is working on a solution if not already completed one. So hopefully soon this will be a distant memory :)
