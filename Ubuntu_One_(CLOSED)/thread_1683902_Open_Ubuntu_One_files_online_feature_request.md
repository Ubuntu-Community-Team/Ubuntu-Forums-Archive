### Post by Dragonbite on 2011-02-08
I have run across a cloud storage site (Box.net) that includes (at cost) a service that allows users to open a file in their cloud storage using Google Docs, and as far as I know it saves the file back into the storage space, not Google Docs.

What about something like this with UbuntuOne and maybe Zoho Office?

This would benefit Ubuntu Users because they would [LIST=1]
[*]Not have to download the Windows client, they could do most of what they need online.
[*]It would be cross-platform so Mac users also benefit without needing to reinvent the client application.
[*]Allows for updating files while away, which are updated the next time Ubuntu syncs.
[*]Makes using UbuntuOne more beneficial to mobile and cloud-based users (tablets, Chrome OS, etc.) because the necessary programs are always available, nothing has to be downloaded.
[/LIST]
Ideally, this would work best if a Zoho account is not necessary as the files will be stored in UbuntuOne and not requiring an account promotes a single sign-on like environment.

Where can I submit this idea?  Does this sound like something anybody else would be interested in?
