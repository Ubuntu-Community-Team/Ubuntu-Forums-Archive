### Post by duanedesign on 2010-03-18
Ubuntu One is a great program/service.

What sets it apart from other Sync Program/Services is its application integration with Tomboy, Evoloution, and Rhythmbox. I think this program and service is very ambitious and I am glad to see it progressing.

I wrote a Python script that saves a list of the applications on my computer ( In a CouchDB like your Contacts get saved). It then syncs that list using Ubuntu One. Then the script installs those programs on my other computers. :)

I also use Ubuntu One to keep a copy of my etc/ folder handy as well as many of my config files (.emacs, .vimrc, .bashrc, .zshrc, etc.).

The guys who produce 'Shot of Jaq' use the Ubuntu One share folder to share files of the current podcast they are working on. Using Ubuntu One they are able to produce a great show from different continents.
