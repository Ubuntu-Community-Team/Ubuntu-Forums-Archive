Oops - managed to find a way of doing it :oops:

For anyone else who's in this situation, there is an option within the Ubuntu One application to allow files/folders which are already uploaded to be synchronised locally.  I obviously didn't see this option as I was trying to share the folder with Ubuntu One via Nautilus.
