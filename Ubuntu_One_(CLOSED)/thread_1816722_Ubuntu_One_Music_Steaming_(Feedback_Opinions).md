### Post by Beacon11 on 2011-08-02
Nahh, if you think of Audiogalaxy as a viable alternative, I would recommend rolling an Ampache server ([http://ampache.org/](http://ampache.org/)). It's really easy to setup, and free. In addition, if you change a few settings and add a different flash player, you can also stream your videos. Let me know if you want some help.

### Post by derklempner on 2011-08-03
I've been using Google Music Beta for about a week now, and I can verify that it's worth the wait for an invitation.  Streaming is done via web browser and your Google account.  The music player's UI reminds me a lot of Rhythmbox, so getting used to it was a breeze.  And the Google Music Manager is now available as 32- and 64-bit **.deb** packages.  I've even figured out how to start it from the command line with a password flag (-p) so I can start it up at system boot (after initial setup and account credentials have been entered).
