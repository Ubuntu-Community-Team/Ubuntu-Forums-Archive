Yes, Windows users will be able to access folders you share with them. The user will need to sign up for an Ubuntu One account (if they do not have one already), sign-in to the Ubuntu One web site and then should be able to access the shared folder(s) via their web browser under the "Shared with Me" folder. 

We do not currently have a Windows client so accessing files through the web is the only current option for Windows users. If you or others are interested in helping us port the Ubuntu One client to Windows we'd love to chat with you. :)
