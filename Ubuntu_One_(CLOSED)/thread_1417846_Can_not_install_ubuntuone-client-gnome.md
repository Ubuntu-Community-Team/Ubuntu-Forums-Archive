### Post by duanedesign on 2010-03-01
mullinsn2000,
What versioin of Ubuntu are you using? The latest release is Karmic Koala (9.10). In Karmic it is preinstalled. All you have to do is go to Applications > Internet > Ubuntu One. You only need to add the PPA if you are using Ubuntu Jaunty Jackalope (9.04). If you are unsure which version of Ubuntu you are using you can find this out by going to Applications > Accesories > Terminal and entering the following command:

```
lsb_release -d -s -c
```

*NOTE:* there are some nice tutorials on the wiki that are useful.
[https://wiki.ubuntu.com/UbuntuOne/Tutorials](https://wiki.ubuntu.com/UbuntuOne/Tutorials)
