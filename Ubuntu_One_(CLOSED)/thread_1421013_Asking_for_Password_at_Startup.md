### Post by joshuahoover on 2010-03-04
> **DedMoroz said:**
> I dont have this prob on my home computer, but here at work, I recently reinstalled UbuntuOne to fix some issues I was having (fixed) but now upon startup U1 is asking for my password on the default keyring. How do I fix this so it doesnt ask me every morning? Thanks!

Please see this FAQ: [https://answers.edge.launchpad.net/ubuntuone-client/+faq/839](https://answers.edge.launchpad.net/ubuntuone-client/+faq/839)

Since it looks like you're on Jaunty, you'll want to:

1. Open Applications->Accessories->Passwords and Encryption Keys
2. Click on the Passwords tab
3. Right-click on the Passwords folder and select "Change Password"
4. Set the password to be the same as your computer login

Thanks!

Joshua
