### Post by duanedesign on 2010-01-28
Did you get 1.0.3 installed.
You can get it by setting your Software Sources to karmic-proposed. if you do not want all the other proposed packages you can install the ubuntu One Stable PPA by running the following in a Terminal (Applications > Accesories > Terminal).

```
sudo add-apt-repository ppa:ubuntuone/stable && sudo apt-get update && sudo apt-get upgrade
```
