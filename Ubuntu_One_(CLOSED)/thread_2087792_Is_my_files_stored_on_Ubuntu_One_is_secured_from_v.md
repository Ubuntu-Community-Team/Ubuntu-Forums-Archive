### Post by Frogs Hair on 2012-11-27
If file sync is in use and an infected item is added to the synced folder it will be uploaded to Ubuntu One.I find nothing to indicate Ubuntu One scans for malicious content prior to upload, but I could be wrong. 

Prior to adding items to a synced folder from Windows it is possible to right click and scan with Security Essentials or another AV program to check for _known_ viruses.
