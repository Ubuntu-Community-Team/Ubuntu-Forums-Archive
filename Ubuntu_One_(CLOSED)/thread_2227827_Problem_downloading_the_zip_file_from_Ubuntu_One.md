### Post by Sleepy-zz-John on 2014-06-17
Well they did say they would restore the  download.zip service as soon as possible,  but the clock's ticking toward July so I can understand the anxiety.  

I've just solved this problem myself with mover.io.    They are a kind of temporary cloud themselves which you can use to transfer stuff from one cloud to another.   So I chose another cloud from their list of available destinations,  and had them move each folder I had on Ubuntu One to the new one.   They can move folders recursively but you do have to order each top folder to be moved one-by one.  Having done this I've now got  all my Ubuntu One folders stored safely on another cloud so I'm no longer panicking about July.     Folders with large content and multiple sub-folders do take some time to transfer,  it's not instantaneous,  but very much quicker than trying to download everything manually.
