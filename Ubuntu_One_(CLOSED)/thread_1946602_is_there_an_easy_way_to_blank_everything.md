I use ubuntu one to backup one of my machines. Ive decided that i wanted to change the folders that are backed up and thus want to delete everything and start again. Is there an easy way to totally wipe everything in my Ubuntu one personal cloud or do i have to go through it file by file?

### Post by howefield on 2012-03-25
Whilst connected to Ubuntu One, simply delete all the files in the synced folder to tell the server to delete them.

---

### Post by duanedesign on 2012-04-06
You can also visit the [website]("https://one.ubuntu.com/files/") and select 'More' next to the folder name and select 'Stop Syncronizing'. This will delete the folders from the cloud but leave them locally. This will not work for the Ubuntu One folder you will need to move the files out of the folder like Howefield mentioned.
