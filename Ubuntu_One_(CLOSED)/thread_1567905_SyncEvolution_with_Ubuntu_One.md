### Post by duanedesign on 2010-09-08
currently Ubuntu One contact sync is disabled between Evolution and the cloud. However you should be able to sync your mobile with the cloud. You can get the latest on the services status at:

[https://wiki.ubuntu.com/UbuntuOne/Status](https://wiki.ubuntu.com/UbuntuOne/Status)

**UPDATE:** Currently the Nokia N900 is not supported by Ubuntu One Mobile Sync. The reason is their are some bugs in the Nokia N900 implementation of SyncML. Those bugs have been reported but not acted on yet.
