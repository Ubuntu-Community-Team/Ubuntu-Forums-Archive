### Post by jaxån on 2009-11-15
Depends on how you want to have it set up.

Should work either way.

One account if you trust the account owner of you account data (and password).
Two accounts if you want to separate the two accounts data (and don't trust it with your password).

It's like if you read Google mail.  Do you want some or all account on your machine use the same mail account or not?
