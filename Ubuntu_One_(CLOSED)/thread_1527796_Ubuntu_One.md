### Post by Revolutionary101 on 2010-07-09
There is no dedicated link to it but you can find the folder in /home/user/Ubuntu One. To make that folder a bookmark just open the folder and go to Bookmakers>Add Bookmark. That will make the Ubuntu One folder appear on your Places menu.
