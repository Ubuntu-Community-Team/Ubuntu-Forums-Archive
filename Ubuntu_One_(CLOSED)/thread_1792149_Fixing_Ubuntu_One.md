### Post by mkarnicki on 2011-08-31
>  It works fantastically within Ubuntu, just the Windows version is not given some attention ATM.

Believe me guys, we have a highly qualified team of engineers working on the Windows client every day. I'm sure Ubuntu One for Windows will eventually make users happy.

Dropbox was created in 2007, Ubuntu One in 2009. Think of that, and the fact that Dropbox is _files_, whilst 2 years younger (and certainly not perfect, but improving on a daily basis) Ubuntu One is _files_, music _streaming_, _notes_, _contacts_, some of these services both on Desktop (Ubuntu, and **beta** Windows) and Mobile (Android Files and Music, iOS Music). [https://one.ubuntu.com/services/](https://one.ubuntu.com/services/)

I know some of you like Dropbox, so do I (well, maybe not so much after the day when it turned out their promises about no access to user data turned out rubbish). But, all in all, please keep in mind Ubuntu One is 2 years younger, and we're doing our best to make it of great value and quality to users.

In case you ask, I don't develop the Windows client, by fellow co-workers are. I'm currently responsible for Android apps.
