### Post by imanumber on 2011-04-29
I'm running 10.04 LTS and am syncing almost 20 gigs of stuff with Ubuntu One.

I'd like to do a fresh install of 11.04 and I have my /home directory on a separate partition so I won't have to format it.

When I install 11.04 and put in my login credentials, will UbuntuOne recognize that since my folders are all in the same places that they are the ones that should be synced?

Also, will it be smart enough that it won't need to re-download all ~20 gigs of data (when it is already on the machine)?

---

### Post by imanumber on 2011-04-30
So, in case anyone has the same question...

I installed 11.04 and had it format my root partition, but not my /home or /usr/local partitions.   During the installation I gave the new user the same name as the previous installation's user and after everything came back up Ubuntu One worked exactly as expected.

All my previously synced files remained synced and it didn't have to download/reupload anything and adding new things to the directories that are synced cause the new files added to be synced as well.

So everything worked out as it should!
