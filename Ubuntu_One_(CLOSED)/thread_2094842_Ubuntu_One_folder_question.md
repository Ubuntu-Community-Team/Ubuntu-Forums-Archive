### Post by deadflowr on 2012-12-15
> What is the purpose of the Ubuntu One folder? 

It's just the default folder.
Not everyone who uses Ubuntu One shares more than that folder.
It's a dedicated folder, where as all other folders you add to the  list of shared folders can later be removed, the Ubuntu One folder cannot be removed.
There is no need to move your other folders into the Ubuntu One folder.
