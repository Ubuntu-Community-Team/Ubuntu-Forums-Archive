Sorry to hear Ubuntu One isn't working properly for you. The best way to get a problem like this solved is through filing a bug. You can do this by right-clicking on the Ubuntu One Client and selecting "Report a Problem". It will also be helpful if you can attach ~/.cache/ubuntuone/log/syncdaemon.log to the bug. Note that attaching syncdaemon.log will show filenames you are attempting to sync with Ubuntu One. If you do not want this to be public, please mark the bug as private and the bug report will only be available to you and the Ubuntu One team.
