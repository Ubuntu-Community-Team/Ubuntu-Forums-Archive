### Post by pfnorris on 2010-10-31
> **Jolicoeur said:**
> I just opened the home folder, in there is a folder called Ubuntu One and inside that it says shared with me.  Is this the folder that I drop files into?  If it is can it be added to the task bar like drop box?

Anything that you place in the Ubuntu One folder should automatically sync with the service and any other subscribed computers you have, once you have set up your account.

In addition you can right click on any other folder (Or files) in your home folder and the context (pop up) menu will offer an option to 'Synchronise on Ubuntu One'.
