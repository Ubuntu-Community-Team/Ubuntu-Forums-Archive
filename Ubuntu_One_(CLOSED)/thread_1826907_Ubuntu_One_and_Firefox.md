### Post by lovinglinux on 2011-08-17
I would recommend using Firefox built-in Sync feature, instead of Ubuntu One. It can synchronize bookmarks, history, tabs, passwords and preferences. You can access this feature from the Preferences dialog.

I know is not a solution if you really want to get that functionality on Ubuntu One, but with the built in Sync you are sure to get it working on every Firefox release.
