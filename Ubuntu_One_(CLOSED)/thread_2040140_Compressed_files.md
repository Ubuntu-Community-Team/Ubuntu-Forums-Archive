### Post by irv on 2012-08-10
Ubuntu One creates a directory in your home folder when you set it up.
Find the files you want to compress and select them.
Right click on the selected files and compress them. (it will ask for a file name) just give it a name and remember it.
When it is done compressing, find the file and move it to the Ubuntu one directory. It will sync with the cloud. If this is a large file it may take awhile to sync.
Go to your machine you want to copy the file to and setup Ubuntu One. 
Again it will create a directory in your home folder and sync with the cloud folder.
That's all there is to it.
