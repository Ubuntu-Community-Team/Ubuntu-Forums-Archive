### Post by tgalati4 on 2014-02-13
The purpose of cloud storage is for syncronized storage of files.  What were you exactly trying to do?  As you have seen the delete function is very efficient.  How many coffees did you consume during the download process?  

To recover the files on Android, you would have to plug into a Linux machine and use some file recovery tools.  They should still be there if you haven't done too much on your phone.  Once files are deleted, the free space can be used by other applications, so either pull the memory card, or turn the phone off until you can perform the recovery.

### Post by Irihapeti on 2014-02-13
You would need to tell the device not to synchronise that folder with U1 (I'm assuming it's outside of the Ubuntu One folder). I don't know how you do that with the Android app.

If the data is inside the Ubuntu One folder, you need to move it somewhere else. If you just move it, instead of copying it, it will delete what's on the server. Which will, in turn, delete what's on any other synced device, so make sure that's what you want before you do it.
