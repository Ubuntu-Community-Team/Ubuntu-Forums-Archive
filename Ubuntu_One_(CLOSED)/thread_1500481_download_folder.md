### Post by duanedesign on 2010-06-03
You can download files from [the web interface ]("http://one.ubuntu.com/files")

If you add the computer to your Ubuntu One account the Ubuntu One folder and any User Designated Folders in your $Home will be synced. So all [computers you have added ]("https://one.ubuntu.com/account/machines/")to your Ubuntu One account will have the same files in their ~/Ubuntu\ One directory.
