### Post by duanedesign on 2011-07-22
To add a folder to your Cloud Folders list Right Click on the folder in Nautilus (file Manager) and select Ubuntu One > Syncronize This Folder. Then the folder will appear in the Ubuntu One Control Panel and you can check and uncheck the folder to sync and unsync the folder.
