> **DaComboMan said:**

You should be able to follow the tutorial here: [https://wiki.ubuntu.com/UbuntuOne/Tutorials/Notes](https://wiki.ubuntu.com/UbuntuOne/Tutorials/Notes) 

The difference is how you open Tomboy and accessing Tomboy->Preferences from right-clicking the Tomboy icon in the taskbar. You'll also likely need to allow Windows firewall to unblock the port on initial setup. It should prompt you for this automatically, assuming you have Windows Firewall turned on.
