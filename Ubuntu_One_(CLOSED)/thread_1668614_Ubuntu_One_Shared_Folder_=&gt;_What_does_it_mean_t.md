### Post by mkarnicki on 2011-10-26
This has been addressed. Old published files still have the same URL (otherwise we would really break U1 experience), newly published files have a more complicated URL, for instance:

[http://ubuntuone.com/4ScPyLBj3SGJUsMCcDVYbW](http://ubuntuone.com/4ScPyLBj3SGJUsMCcDVYbW)

We hope this provides more balance between publishing files and allowing access by random people just by playing with URLs.
