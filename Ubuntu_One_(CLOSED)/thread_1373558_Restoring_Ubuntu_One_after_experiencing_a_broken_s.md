### Post by jonest1 on 2010-01-05
I have thought about this very issue with only having one system.

I believe you may be safe if you remove your computer from the subscription using the web interface.  Then, when you start Ubuntu One it will be a new registration, but your files should remain in the cloud.

I believe you will be save regardless, as the service will see the new installation as another computer.  This would be similar to adding a second computer to the Ubuntu One service.  The only problem may come in if you want it to have the same name.

Your files should be available in Ubuntu One when you re-register your computer.  The launchpad id you have is usually an Email address/password if I remember right.

I have multiple computers associated with my account and the files are on all my computers.

So, you should be good.  Of course, the files are also available through the web interface.

Also, bear in mind this is only the first release of Ubuntu One and more features, etc. are planned and on the way.
