### Post by Lisiano on 2012-05-19
I don't know much about KeePass, but if it let's you set a db for your keys, just make one in the Ubuntu One folder and point devices to that file on other machines. If it has it's own config folder then right-click that folder and press Ubuntu One -> Sync folder, on other machines just open Ubuntu One and tick that folder for syncing (Remove it beforehand or the files on the 2nd machine will be uploaded and replace the files on the 1st machine)

EDIT: You can get a public link from the same right-click menu or you can use Ubuntu One for Windows/Android if you need it there as well.
