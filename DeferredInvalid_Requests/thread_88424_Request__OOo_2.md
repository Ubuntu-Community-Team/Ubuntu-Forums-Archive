### Post by varunus on 2005-11-11
If you can't wait, one of the devs was kind enough to set up a repo with it for Breezy.  Add this line to the end of your /etc/apt/sources.list:

```
deb http://people.ubuntu.com/~doko/OOo2 ./
```

Then hit reload in synaptic, and you'll notice that openoffice can be upgraded to the full 2.0 version.  :)

Enjoy.
