### Post by sciurus on 2005-11-21
The <a href="http://www.xiaprojects.com/www/prodotti/kxdocker/main.php">latest release</a> is 0.39, but Breezy only has 0.35. Lots of changes have been made, such as French and German language support.

Features added on KXDocker 0.39
Engine: buble are done with more accurate timings
Engine: No more icon cache, so KXDocker use ONLY 40MB of ram, it's only to store active icons
Command parser (1/10) parsed commands
New Engine without moving KXDocker, so we are ready for composite manager, WHEN it WILL be stable.
A lot of new features on plugin managenment, like ttl, rolling and Plugin of Plugin Manager!! (look at GIPContrack attached to xARP)
New translation: FR
New translation: DE
Highlight icon configuration

other Fix-changelog on KXDocker 0.39
Some background bad draw fixes
More fast engine, try to lower sleeps or disable interpolation
bad redraw when set desktop=0 and disable fasthide be more fast
xCommand bug: onDropExec run another cmd, please make carefull
