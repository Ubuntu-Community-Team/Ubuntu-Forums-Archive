### Post by slickymaster on 2016-08-16
Ubuntu Membership via Forums Contributions is a way of recognizing people whose main activity is on the forums.

**_[SIZE=3]How to apply:[/SIZE]_**


[LIST=1]
[*]Create a Launchpad account (See [this tutorial]("https://help.launchpad.net/YourAccount/NewAccount"));
[*]Sign the Ubuntu Code of Conduct (Use the detailed [Code of Conduct Signing Assistant Tutorial]("https://wiki.ubuntu.com/Forums/CoCSA_Tutorial") to help you);
[*]Create a Wiki Page ([https://wiki.ubuntu.com/your_wiki_username](https://wiki.ubuntu.com/your_wiki_username)) with some basic information about yourself, i.e. who you are and what you've done on the forums that's relevant to gaining Membership. It doesn't need to be elaborate;
[*]Start a thread in the [Ubuntu Membership Applications]("https://ubuntuforums.org/forumdisplay.php?f=404") sub-forum requesting a review of your application and title it: **Ubuntu Membership application: Your_Forum_Name**;
[/LIST]


Example of a membership application> I apply for Ubuntu Membership.

My wiki page is at [https://wiki.ubuntu.com/your_wiki_username](https://wiki.ubuntu.com/your_wiki_username)

My Launchpad profile is at [https://launchpad.net/~your_lauchpad_name](https://launchpad.net/~your_lauchpad_name)


If you have any doubts with any of the steps of the process, please consult the [Ubuntu Membership via Forums contributions]("https://wiki.ubuntu.com/Forums/Membership") wiki page, which covers not only all the potential questions an applicant might have, but also contains a tutorial for the Code of Conduct signing assistant, complete with screenshots to guide applicants through the steps involved in signing the Ubuntu Code of Conduct, in order to make it as easy as possible.

If, due to the present community wiki lock down, you got stuck and thus unable to set up your own wiki pages, please feel free to get in touch with a [member of the staff]("http://ubuntuforums.org/showgroups.php/"), through a PM in the forums, and ask for help regarding it.
