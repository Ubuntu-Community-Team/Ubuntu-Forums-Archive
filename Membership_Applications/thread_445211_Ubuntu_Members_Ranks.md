### Post by Technoviking on 2007-05-15
[IMG]http://ubuntuforums.org/images/rank_ubuntumember_2013-02.png[/IMG]

The Ubuntu Forums has added user rank graphics for people who have applied for and received Ubuntu Membership though the Ubuntu membership process. If you are an Ubuntu Member would like a user rank graphic added to your profile, please add your forum username to the home page/description part of your Launchpad account and send e-mail to [EMAIL="ubuntu-forums-council@lists.ubuntu.com"]ubuntu-forums-council@lists.ubuntu.com[/EMAIL] with your Launchpad URL.

**Please allow 72 hour for this request to be processed.**

This is an extra only for **[Ubuntu Members]("https://launchpad.net/~ubuntumembers")**. You must be an Ubuntu Member and on the ubuntu-member team in Launchpad. For information on how to become a Ubuntu Member please visit: [https://wiki.ubuntu.com/Membership](https://wiki.ubuntu.com/Membership) and [http://www.ubuntu.com/community/processes/newmember](http://www.ubuntu.com/community/processes/newmember). Please consider joining, it is an easy and fun process.
