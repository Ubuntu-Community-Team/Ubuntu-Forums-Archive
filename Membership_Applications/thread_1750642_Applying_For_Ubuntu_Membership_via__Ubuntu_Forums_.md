### Post by s.fox on 2011-05-05
[SIZE=3]How to Petition for Membership:[/SIZE]

1. Create a wiki page.
2. Create a Launchpad page. Although not mandatory, many people use the same identity/nick on Launchpad and on the forums.
3. Sign the Ubuntu code of conduct. This has traditionally been one of the more difficult steps. FYI: A bug has been opened on Launchpad to make the process easier.
4. Create a thread in the [Ubuntu Membership Application]("http://ubuntuforums.org/forumdisplay.php?f=404")s section requesting a review of your application. Be sure to include a link to your wiki and launchpad pages.

For all information see [Ubuntu Membership via Forums participation]("https://wiki.ubuntu.com/Forums/Membership")
Enclosed are updated guidelines for signing the Ubuntu Code of Conduct and creating a Wiki page.

[SIZE=3]Sample Forum Post:[/SIZE]

> 
I would like to apply for Ubuntu membership, my wiki page is here:

**<insert_your_wiki_page_link_here>**

and my Launchpad page is here:

**<insert_your_launchpad_page_link_here>**

Thank you

**<insert_your_forum_name>**
Good Luck in your application :)
