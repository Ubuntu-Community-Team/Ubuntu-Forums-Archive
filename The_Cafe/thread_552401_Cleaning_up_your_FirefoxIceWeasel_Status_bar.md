### Post by RAV TUX on 2007-09-16
So it took a bit of research on Dark Themes for Firefox/IceWeasel to finally find the look I like without having to create my own theme. 

Please see this thread as a reference: [http://ubuntuforums.org/showthread.php?t=550452&highlight=dark+themes](http://ubuntuforums.org/showthread.php?t=550452&highlight=dark+themes)

I finally found:

**[Cylence Theme 2: Black Edition Beta 2]("https://addons.mozilla.org/en-US/firefox/addon/4334")          **by** [ Warboy]("https://addons.mozilla.org/en-US/firefox/user/89381") **

which for the most part runs very excellent.

I found two Firefox extensions that help mimic some default attributes found in the Opera Web Browser:

1. Secure Login 0.8.13 which mimics the Opera wand tool;
**[Secure Login]("https://addons.mozilla.org/en-US/firefox/addon/4429")** by[B] [Sebastian Tschan]("https://addons.mozilla.org/en-US/firefox/user/100849")

[/B]Please see this thread as a reference: [http://ubuntuforums.org/showthread.php?t=469076&highlight=Secure+Login&page=2](http://ubuntuforums.org/showthread.php?t=469076&highlight=Secure+Login&page=2)

  2. Scroll Search Engines 0.5 which mimics the ability in Opera to right-click>select term for search> and be able to choose between all your search engines listed;
         **[Scroll Search Engines]("https://addons.mozilla.org/en-US/firefox/addon/3696")**          by[B] [Einar Egilsson]("https://addons.mozilla.org/en-US/firefox/user/61607")

[/B]Please see this thread as a reference: [http://ubuntuforums.org/showthread.php?t=550297&highlight=Scroll+Search+Engines](http://ubuntuforums.org/showthread.php?t=550297&highlight=Scroll+Search+Engines) 

Both of these extensions and the above theme work excellent.

Important to note I tried Speed Dial for Firefox that mimics Opera's speed dial but found that it crashed my Firefox regularly and I would highly advise against using this Firefox extension;
         **[Speed Dial]("https://addons.mozilla.org/en-US/firefox/addon/4810")          **by** [Josep del Rio]("https://addons.mozilla.org/en-US/firefox/user/9201")**
Again I highly advise against using this Firefox extension, beyond being dysfunctional it is also highly aesthetically displeasing to view.

Instead of Speed Dial I use the YubNub Command Line;
         **[YubNub Command Line]("https://addons.mozilla.org/en-US/firefox/addon/4714")          **by** [BlueJ774]("https://addons.mozilla.org/en-US/firefox/user/116588")**
for those who are use to 'sudo', you should feel right at home using the YubNub Command Line for the web.

So for the most part I finally have my IceWeasel in Elive Gem 1.0 looking and functioning the way I like it to.

The only thing I didn't like was having the status of my web pages I was loading displayed in the status bar. I do not want to remove the status bar because it frames the bottom of the Web Browser nicely. I just want any and all icons and status notification on it gone. To achieve this final customization I used:

Organize Status Bar 0.5.2
**[Organize Status Bar]("https://addons.mozilla.org/en-US/firefox/addon/1759")          **by** [y5]("https://addons.mozilla.org/en-US/firefox/user/123")**
Which works nicely to remove the default status notification on the status bar and make for an overall clean status bar.
