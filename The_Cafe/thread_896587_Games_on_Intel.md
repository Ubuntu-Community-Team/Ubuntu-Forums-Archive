### Post by Kvark on 2008-08-21
I have a G31 board with GMA 3100 which is almost identical to the GMA 950 in 945GM boards. I think the only difference is that GMA 950 supports DirectX 9 while GMA 3100 supports DirectX 9.0c.

My experience is that it runs the 20 or so games from the repos that I tried perfectly fine but maybe there is a more demanding one in the repos that I haven't found yet. I haven't tried any that are not in the repos on Linux.

I also tried about 20 closed source games and demos on Windows XP. Based on my experience I'd say you can see if it'll run a game by asking "Is Shader Model 2.0 a minimum requirement?" If no then it'll run fine in high settings. If yes then it either runs fine on medium settings or crashes entirely, Google to make sure. If the minimum is higher than 2.0 then forget it not a chance.

So forget about playing new games on it but plenty of games that are a couple years old are still fun. The best thing is having completely bug free desktop effects out of the box.
