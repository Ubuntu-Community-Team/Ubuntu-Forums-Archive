Power usage is the same no matter what if no parts of your rig (usually the case) aren't energy star qualified.

You're better off getting a really good machine now, and later on, buying yourself an android tablet or something for web surfing via wi-fi.

If you can, the ultimate way to save power would be the following:

1. Get your gaming rig configured completely with Windows.

2. Buy the android tablet. (preferrably one that can support 32 GB or more of space either in itself or on MicroSD, so you don't need a flash drive, they take up power too.)

3. Set up Windows to accept SSH tunneling and connections to allow your tablet to connect and manage the computer remotely.

4. Unplug the monitor, remove it from your desk when everything is set up so the tablet will not need re-auth for every connect.

5. Use the tablet as a monitor, keyboard and mouse (Note, you will need right-click functionality, and your keyboard must still be active for in-game interfaces.)

6. ????

7. Profit!

other notes: Don't use a gaming laptop, always go desktop with integrated sound and ethernet. Do not use wireless USB dongles on the desktop, it takes more power than an ethernet cable and connection (plus, your ethernet chipset will be active unless you turn it off in the BIOS, and still consume power.) if you need to use voice programs, you will need to execute the program via tablet, but have the microphone in the desktop unless you use Skype for Linux or other voice chat system that has a linux client.

Keep in mind, you will still need to set up a dedicated card for your computer (IE: NVIDIA GTX 470) it will do the processing for your tablet, because the tablet is connecting remotely. Also use an ATX Case that has a spot for a front 120mm fan, and a 120mm back fan. Front fans carry the cool air into the bottom of the tower, as it heats up via the video card, it rises, then is pulled out either the side or back vents. The more/better dedicated fans you have, the less the video card has to use its own fan.
