### Post by Spice Weasel on 2010-10-28
An advantage to using Fedora over Ubuntu is that as well as being released 2 months earlier per version they have a rolling version (Rawhide). There are also lots of different officially recognised "spins" providing different desktops and preinstalled software. You can find these on the Fedora website.

It's much better on a desktop (without an ATi card) than it is on a laptop. If you want to try Fedora on a laptop with no free software drivers available, you're insane. Remember this as well: it's not based on Debian and is aimed at home use, not for workstations and servers. If you want a stable free Red Hat distro try CentOS. Fedora is NOT going to be as stable as Debian or Debian-based distributions, and it won't spoon feed you as much as Ubuntu. No non-free codecs are provided by default (and free software drivers are used over non-free ones if they are available. This may cause problems for users of Nvidia cards.) In any case, if you need non-free codecs and the non-free Nvidia drivers, you need to enable a separate repository. This is called RPM Fusion:

[http://rpmfusion.org/](http://rpmfusion.org/)
