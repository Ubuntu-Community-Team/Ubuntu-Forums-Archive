### Post by samalex on 2011-04-29
Saying you want to be an IT Professional is like saying you want to work with Cars... you can paint them, fix them, build them, design them, tune them, sell them, etc.  You first need to choose the area you want to focus on and run that direction, and any direction you go will itself run into multiple directions :)

Also it depends on whether you want to tackle this endeavor on your own, go through a college or university, or just get certifications.  I personally think this depends on your age and how fast you want to get into the workforce.

And job demand is something else to look at.  Some areas, like mine, are black holes for Linux, Unix, and open source development, so knowing Microsoft's way of doing things is a must.  Other areas are quite different.  So you'll need to assess the market in your area, unless you want to move, and see what employers want.

One major thing to note is there's a huge difference between Hardware and Software, and IMO the $$$ is in the software.  So if you really want to get your foot in the door start learning Linux, Mac, and Windows (yes Windows!) both client side and server side.  It's not hard to learn the basics of all of them (install, configure networking, etc).  VirtualBox is wonderful for this.  This way on a resume it makes you look very versatile.

Then if you want to get into programming firstly start learning the Object Oriented Programming concepts, which is an often overlooked step.  Once you earn this choose a language you're comfortable with and start going at it.   You don't have to be a pro, just learn it well enough to create some basic apps with hooks to the file system and database.  Then from there start creating other apps with similar functionality in other languages using a variety of IDE's and Databases.

Again you don't have to be an expert in all of these, but putting on a resume that you have 'experience with' a large amount of stuff is always impressive.

Then if something REALLY sparks your interest run with and get some certs if you want.  

Anyway, just my two cents on it --
