### Post by cardinals_fan on 2009-03-20
> **gjoellee said:**
> If you compare Xubunut and Ubuntu, then yes. Xfce is however based on GNOME but heavily modified and a lot of stuff removed. Xfce is a lot faster then GNOME!

That's wrong.  Xfce began as a simple Xforms toolbar mimicking CDE in 1996 - before GNOME ever existed.  Xfce remained Xforms-based and therefore only quasi-open-source until 1999 (version 3), which used GTK+.  The project still uses GTK+ today (it was updated to GTK+ 2 with version 4.0.0).

While some optional Xfce apps use GNOME libraries, all the core components are not based on GNOME.
