### Post by torry_loon on 2008-06-09
Why should a child be taught to use only one OS or application?

There is no reason that proprietary systems (such as Windows, Mac OS X or MS Office) can't be used alongside free/open source systems (such as Linux or OpenOffice). Even if the computers in the school dual boot Windows and Linux and the children spend 80% of the time in Windows, they are still getting experience in using Linux.

Many software applications require some retraining each time a new version is released. By teaching the children to use individual hardware and software products instead of teaching the concepts that can be transferred to similar or newer products, you are continually trying to keep up with the technology and teaching the same old things in a slightly different way.

Many of the children who are starting school now will probably have jobs that do not exist now.  To assume that Microsoft Windows will continue to be the dominant OS for the next 13 years is very short-sighted.
