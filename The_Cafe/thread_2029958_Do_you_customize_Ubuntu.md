### Post by rg4w on 2012-07-20
Based on admittedly limited and informal usability testing here, and in the absence of published usability testing reports from Canonical, I regularly make these changes to each installation of Ubuntu I do for myself and my friends who prefer my setup:

1. Edit the metacity theme file for Ambiance to increase the grippable border of windows for easier resizing.

2. Use the built-in Appearance System Settings panel to narrow the Launcher width.

3. Use MyUnity to change the behavior of Launcher icons to be filled only for open apps, making the distinction clearer and the shapes of unopened apps more easily recognizable (the autofill color for things like the USC gives it a muddy appearance which makes the shape difficult to discern).

4. Use Compiz Config to turn off the auto-maximizing of re-opened windows, leaving the user in control of window size.

5. CCSM again to turn off the sticky-windows behavior so that the user is in control of window placement.

6. CCSM to turn off the auto-maximize behavior when a window is dragged near the top panel.

7. Use the System Settings panel to turn on two-finger scrolling (why isn't this detected and when available enabled by default?)

There are probably a few other tweaks, but those are the first I do on a new install.
