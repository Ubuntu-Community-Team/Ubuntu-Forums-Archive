### Post by kostkon on 2008-03-17
Indeed, I got an update for it!

Anyone else who wants automatic updates for *Skype* can add its *Debian* repository (the *Skype* site actually offers the same file for *Debian* and *Ubuntu*, so there's no problem using this repository):
```
deb http://download.skype.com/linux/repos/debian/ stable non-free
```
You can find help on how to add repositories into your software sources at [this page from the *Ubuntu documentation*]("https://help.ubuntu.com/community/Repositories/Ubuntu").
There's more information about using *Skype* in *Ubuntu* [here]("https://help.ubuntu.com/community/Skype").
