### Post by tgalati4 on 2015-07-14
I have not used the BBC wx app, but perhaps you have a custom location in your location listing.  Try deleting all locations.  Then set one location using Postal Code only.  Uncheck any preferences in the app that say "Use Current Location".  Any location that requires GPS will normally force the app to ask the phone for GPS location (and that framework is now part of Google Assist).

If that doesn't work, then I would follow the suggestion by ubuntuman001 and just find another wx app.  There are several.  But yes, I want an Ubuntu phone as well, but they are not available in the US yet.
