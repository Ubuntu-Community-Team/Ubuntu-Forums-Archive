### Post by cwrann on 2010-04-11
It' been a while since I've had to post a question in the forums (May '09) which goes to show how far the OS has come with regards to ease of use. 

I've been using Ubuntu since Fiesty and have faithfully upgraded with every new edition, it just keeps getting better.

I make a web comic called Aquarium Drinking and wanted to give a shout out to Ubuntu in it. It is written in Open office, and colored, lettered etc. in Gimp all on an Ubuntu OS. In addition I use Filezilla for an ftp and Wordpress for publishing, now that I think about it "Aquarium Drinking" is created using 100% open source software without even intentionally trying. 

Anyway, I gave Ubuntu a little "easter egg" type of shout out in one of my recent pages and thought people here might get a kick out of it. Just an image on a t-shirt, just enough to let those who know how great Ubuntu is that I know how great it is. 

Check it out, click on the link below.
[Aquarium Drinking]("http://aquariumdrinking.com/comic/chapter-2/page-23-3/")
