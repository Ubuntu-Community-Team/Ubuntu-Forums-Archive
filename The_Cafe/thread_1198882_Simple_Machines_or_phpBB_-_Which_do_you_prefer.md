### Post by leef on 2009-06-28
This is certainly something you should think about, once you have chosen the software, you will more or less be stuck with it. Sure you can get database converters,etc but if you go down that route, you are asking for trouble and people don't like change.

Personally I would choose SMF over phpbb because SMF has a plugin system which makes upgrading much less of a hassle and it is much easier than hacking the code yourself and forever updating hacks, etc. You are most likely more secure with SMF simply because updating is easier. I choose phpbb for my last website (simply because it was what I was used to) I have been regretting that decision for the last year or so, lol.
