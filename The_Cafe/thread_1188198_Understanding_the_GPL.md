A big, red [SIZE="5"][COLOR="Red"]**no**[/COLOR][/SIZE].

The fact that a license is compatible with the GPL does **not** mean you can relicence GPL code under them. **Only** the entity who owns the copyright on a piece of code can change its licensing terms.

[http://www.fsf.org/licensing/licenses/gpl-faq.html#WhatDoesCompatMean](http://www.fsf.org/licensing/licenses/gpl-faq.html#WhatDoesCompatMean)

> It means that the other license and the GNU GPL are compatible; you can combine code released under the other license with code released under the GNU GPL in one larger program.

All GNU GPL versions permit such combinations privately; they also permit distribution of such combinations **provided the combination is released under the same GNU GPL version**. The other license is compatible with the GPL if it permits this too

### Post by ssam on 2009-06-15
all software has a licence that you have to comply with if you want to copy it (you need to copy the data to RAM to run it). most have restrictions along the lines of 'you can only run this if you pay this much money' or 'you may not make any modifications'.

GPL, has almost no restrictions. you can run, modify, study, redistribute the code as much as you like. the only restriction is that if you modify it, the modified version must have the same freedom.

(you can make your own private modifications, if you do not redistribute the resulting program)

(gpl3 has some extra bits about patents and such)

this is good because lots of people want to take code other people have writen, make a small improvement, sell the result and prevent the original project using the new code. (examples [http://gpl-violations.org/](http://gpl-violations.org/) )

BSD code has the additional 'freedom' that you can remove the freedoms from a derivative  work. some people think this is freer, some think it is less free. i guess it depends on the definitions.

if you want to make some software, either write it from scratch, pay someone to write it for you, or find something to base it on and comply with the license.
