### Post by PriceChild on 2007-01-13
If you are the original poster in a thread, edit your first thread.

Click advanced edit.

Chose the "check this box if my thread is resolved" box and press submit (NOT preview)

All is done and it will have a [RESOLVED] tag on subforum view :)
