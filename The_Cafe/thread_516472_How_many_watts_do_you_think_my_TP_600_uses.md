### Post by bernied on 2007-08-03
Have a look at the power supply, that should tell you the maximum amount supplied to the machine. This is not the same as the consumption, because the power supply will lose some as heat, but it's probably about the same because the laptop won't be running on full power all the time.

So, my laptop is 65W, so for one day 65W x 24h is 1560Wh or 1.5kWh per day. Electricity prices are weird and complex, but it looks like about 10p per kWh, so 15p per day, which is about a fiver a month. Though, if it's not up to much, then maybe it would only use half of that.

### Post by Syke on 2007-08-03
If you already have it running, right-click on the battery icon and select "Power History", then under Graph Type select "Power History", and you'll see a chart of how much power the laptop is using.

Somewhere around 10 to 20 watts is pretty normal for a lot of laptops.
