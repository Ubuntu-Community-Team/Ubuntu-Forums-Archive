### Post by sub2007 on 2008-05-25
Yep as mentioned there are various protocols. You would need to sign up for at least one and the other person would have to have an account with the same protocol.

The main protocols are:
- AOL Instant Messenger (AIM)
- Windows Live Messenger (normally known as MSN)
- Yahoo! Instant Messenger
- Skype
- ICQ

Pidgin can connect to all of these protocols (I think) except for Skype, which has a Linux version available. Skype is mainly used as a voice protocol, for making calls (it has a free Skype-Skype call as well as text chat). You can sign up for as many or as few of these services as you want, it's just matter of selecting a log on name and password. When you have those you just put them in to Pidgin and it will connect.

To chat to a friend you need to get their user name and add that to your instant messenger's friend list. Different protocols handle adding friends differently, on MSN you have to approve a new friend request and on AIM you will automatically see when someone is online if you have added them. When you see them online you can send them an instant message and chat with them in real time.

Pidgin, quite nicely, will handle ALL your protocols (except for Skype) and so if you do have friends on different networks you will only have to run Pidgin and you can chat with them all.

### Post by tashmooclam on 2008-05-25
To just add a bit to Sub2007's post. 

There is a website [www.meebo.com](www.meebo.com) that allows you to use a bunch of the chat accounts at their site. So, it's convenient.

I think the most convenient account to have is AIM, because you can use it in various other chats, such as Apple iChat for example.

If you have a gmail account (a good idea), others who have a gmail account will be able to chat within the email page with you. 

You may find that many people are using MSN, but you can find them on Meebo when you know their "name". 

There is a program similar to Skype that is less well known called Gizmo5, it also has chat and is open source and can also chat with the various accounts. It also is good for making phone calls. 

Have fun.
