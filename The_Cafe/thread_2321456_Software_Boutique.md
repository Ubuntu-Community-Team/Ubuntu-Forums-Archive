### Post by lah7 on 2016-06-18
The latest versions of the Software Boutique offers a search function, but (currently) only within the Boutique's selection.

You would need to "opt in" (subscribe) to the updates (in the interface) or add the PPA. as I think the one in the Ubuntu archives will be a fair bit behind:

```

sudo add-apt-repository ppa:ubuntu-mate-dev/welcome
sudo apt update
sudo apt install ubuntu-mate-welcome

```
