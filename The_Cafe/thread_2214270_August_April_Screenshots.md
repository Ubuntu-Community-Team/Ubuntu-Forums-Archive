### Post by coldcritter64 on 2014-04-15
gtk-theme      DebianRed
Style              Light-industry
Icons              Nuvola theme (a lot of the xfce4-panel icons are personalized ones, the screenshots don't show the icon set very well).
Default Font    URW Bookman L Demi Bold 11

2 Desktops running on Debian 7 (Wheezy) XFCE (with gnome-shell/fallback installed alongside the basic xfce install).

First is xfdesktop with a "customised ubuntu-conky" and screenlets, the screenlets clock and ringsensors are set up to overlap the conky (which puts its border around them). gdevilspie keeps the conky below all windows and the screenlets are set as normal windows (no above or below settings activated, keeps them above the conky but will drop them below other normal windows)

[ [IMG]https://dl.dropboxusercontent.com/u/30874719/UF/1_xfdesktop_thmb.png[/IMG] ]("https://dl.dropboxusercontent.com/u/30874719/UF/1_xfdesktop.png")

Second is the Nautilus desktop which appears after a toggle switch, on the xfce4-panel with the workspace switcher (or via an easystroke mouse gesture), is operated. 

[ [IMG]https://dl.dropboxusercontent.com/u/30874719/UF/2_nautilus-desktop_thmb.png[/IMG] ]("https://dl.dropboxusercontent.com/u/30874719/UF/2_nautilus-desktop.png")

Third is after a wmctrl script is operated to shift the nautilus desktop down half a screen (this is all over 2 monitors, both are 1920x1080). 

[ [IMG]https://dl.dropboxusercontent.com/u/30874719/UF/3_both-at-once-pt1_thmb.png[/IMG] ]("https://dl.dropboxusercontent.com/u/30874719/UF/3_both-at-once-pt1.png")

Last is after the another wmctrl script has shifted xfdesktop down half, nautilus is reset to fullscreen with a mouse gesture, and the desktops are toggled to put xfdesktop in front again.

[ [IMG]https://dl.dropboxusercontent.com/u/30874719/UF/4_both-at-once-pt2_thmb.png[/IMG] ]("https://dl.dropboxusercontent.com/u/30874719/UF/4_both-at-once-pt2.png")

Check the screenlet clock seconds hand for screenshot timings ;-)

If you're wondering why anyone would want 52 workspaces; changing the workspace is set up, through a memory resident wmctrl script, to change the background wallpaper as well on changing to the next workspace.

Now all I need to do is find enough widescreen wallpapers to fill about 80+ more spots, only got about 8 of the 52 spots per desktop filled so far :P
