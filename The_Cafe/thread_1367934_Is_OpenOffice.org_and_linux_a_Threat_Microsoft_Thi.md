### Post by Dr. C on 2010-01-02
OpenOffice.org is a very serious threat to Microsoft. It is also the "shock troops" of FLOSS. The GNUs and the Penguins come much later to finish the job. When a home user buys a Windows computer it usually comes with one of the following:

1) Microsoft Works
2) A trial (60 days or so) of Microsoft Office
3) No office suite
4) Starting this year starter versions of Microsoft Office riddled with ads
5) A trail for a Microsoft competitor for example WordPerfect.
6) OpenOffice.org

In cases 1-4 if the user does not buy or pirate a full version of Microsoft Office before a FLOSS advocate gets to them they literally become the low hanging fruit for the promotion of FLOSS and eventually GNU / Linux. One simply introduces them to OpenOffice.org on Windows and after they get over the shock that it is gratis and legal the response is almost always gratitude. This then opens the door further FLOSS for example Firefox, Thunderbird etc., and the concepts of Free Software and Open Source Software. The stage is also set for an eventual introduction to GNU / Linux, for example Ubuntu, especially if the user has accepted many different FLOSS applications. In case 1 one of the selling points is, paradoxically, compatibility with Microsoft Office since many Microsoft Office users have not installed the plug-ins to read and write Works files. Case 5 can also work only in this case the full version is much cheaper so there is a much greater chance that the users has already purchased the full version. Case 6 is strangely the hardest since one is not offering anything the user does not already have.

In the enterprise the situation is different since the user already more than likely has Microsoft Office. The opportunity comes at upgrade time where Microsoft will face stiff competition from companies like IBM and their OpenOffice.org derivatives. [http://symphony.lotus.com/software/lotus/symphony/home.nsf/home]("http://symphony.lotus.com/software/lotus/symphony/home.nsf/home") Again first install Lotus Symphony on Windows and then migrate to Ubuntu or another GNU / Linux. IBM by the way has been successfully selling to the enterprise for well over 100 years. Oracle might get into the Office software act also, especially if they buy Sun.
