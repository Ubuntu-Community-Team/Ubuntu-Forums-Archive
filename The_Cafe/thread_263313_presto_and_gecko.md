### Post by Bloodfen Razormaw on 2006-09-23
Presto is Opera's engine.  It generally has better standards support, especially in the area of CSS, which is Opera's specialty.  But the most common stuff is supported in both.  Presto is also much faster and lighter.  However it is not free software like Gecko.

Most of the differences between them will only appear in obscure corner cases.  99% of web sites should render almost identically between the two.  If you do some very specific things with position, Gecko won't handle it right.  Presto tends to be less accepting of very badly coded sites though, so you might find some differences if someone really screws up.  Also, Opera is much more stingy about using quirks mode, which may cause rather minor rendering differences.
