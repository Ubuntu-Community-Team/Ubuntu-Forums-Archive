The idea is simplicity!

So there is one search field and when I do a search it should present all the sites (which I should be able to add) that got hits. 
The user should then press on the hits and it will open the search in a new tab showing the search site.
The user can also press a category - lets say "chemistry" - and tabs for all the sites (with hits) in that category will open.

I do not want to steal any users from the search sites. I only want to make it more easy the user do search multiple databases.

It needs to be a widget so I can easily embed it on the forum..
