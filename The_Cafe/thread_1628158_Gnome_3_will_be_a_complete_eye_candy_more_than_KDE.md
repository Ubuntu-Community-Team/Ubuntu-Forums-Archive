### Post by dawhippersnapper on 2010-11-22
I've been using gnome-shell for a while and deployed it onto a few workstations to see how users adapt.  Once people are trained to know the shortcuts and they understand that they can make different workspaces and keep all of the applications they need grouped together for tasks, a taskbar isn't a complaint anymore.  

The only thing people don't like is the hover over upper left to open an application.  To help with this I have added a mac clone dock at the bottom.   I think gnome-shell is a good idea, and very efficient if you'll take the time to appreciate it.  The only thing I'm not crazy about is activity hover over drop down menu.   And yes I know there are shortcuts for that.  

There is also some hesitation with the alt-tabbing, and I'm not too fond of the alt-tabbing behavior.  So far I much prefer gnome-shell over unity.  Unity has been a resource hog, with no customization at all.  

At least gnome-shell still follows gconf and gnome-menu rules.  I know this is planned to be changed, but it works well for now.

### Post by ilovelinux33467 on 2010-11-24
> **joepie91 said:**
> Odd that that repo wasn't installed by default. If you choose to use GNOME during the OS installation, you'd expect it to add the GNOME factory (if it exists, which I didn't know) instead of KDE factory.
And just curious, what kind of problems does the Packman/VideoLan combo cause?

Packman and Videolan have the same packages. This can cause problems when you install a package from Packman and one of those package dependencies are in Videolan. The openSUSE forums receives lots of posts with multimedia issues due to the fact that users have both Packman and Videolan repos enabled. My recommendation is to disable the Videolan repo and switch packages in YaST to the Packman repo. Here is the openSUSE multimedia guide link: [http://forums.opensuse.org/english/information-new-users/new-user-how-faq-read-only/407184-multi-media-restricted-format-installation-guide.html]("http://forums.opensuse.org/english/information-new-users/new-user-how-faq-read-only/407184-multi-media-restricted-format-installation-guide.html") - It has guides for 11.0, 11.1, 11.2 and 11.3
