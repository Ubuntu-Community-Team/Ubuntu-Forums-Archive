### Post by ubuntu27 on 2008-08-17
Apt-ListBugs is script that warns us which (and how many) bugs are reported for a program before we want to install it. It works for Console/Terminal and Synaptic Package Manager.

With Apt-ListBugs we will save time. And the best thing of all is that it is easy to use since it is executed automatically when we want to install a program.

[[IMG]http://img205.imageshack.us/img205/2628/aptlistbuguy5.png[/IMG]](http://imageshack.us)




Apt-ListBugs is in the Debian and Ubuntu repository, you can install it by 

```
sudo apt-get install apt-listbugs
```
