[URL="https://imgur.com/rru7dya.png"]
  [IMG]https://imgur.com/rru7dya.png[/IMG]

It's for arch, but archive can easily be extracted and program files run right from the folder.  [https://drive.google.com/open?id=1YgOTvgpToZy9fPPUaeNxo6Xr-jy_Lrcx](https://drive.google.com/open?id=1YgOTvgpToZy9fPPUaeNxo6Xr-jy_Lrcx)
Also, windows version [https://drive.google.com/open?id=1ZFEe4ySwWUoce2aj9xkze16f9zZXmxuo](https://drive.google.com/open?id=1ZFEe4ySwWUoce2aj9xkze16f9zZXmxuo)
And github repo [https://github.com/ZVVUbuntu/covid19info](https://github.com/ZVVUbuntu/covid19info)

Once you extract the package, open up the folder, cd into usr and then share and then covid19info
To run the app type python3 covid19info.py

Alternatively you can git clone the repo, cd into cloned folder and run python3 covid19info.py


Be sure to install requisite sudo apt install python3-pyqt5


Columns are
[COLOR=#b22222]Country[/COLOR]  [COLOR=#ff0000]Total Cases[/COLOR]  [COLOR=#a52a2a]New Cases[/COLOR]  [COLOR=#ff8c00]Total Deaths[/COLOR]  [COLOR=#daa520]New Deaths[/COLOR]  [COLOR=#2f4f4f]Total Cured[/COLOR]  [COLOR=#006400]Active Cases[/COLOR]  [COLOR=#008080]In Critical Condition[/COLOR]  [COLOR=#0000cd]Total Cases per 1 million[/COLOR]  [COLOR=#4b0082]Total deaths per 1 million[/COLOR]  [COLOR=#ff0000]11 is total tests done[/COLOR]

New cases and new deaths is for the past 24 hours. 


Also good resource is the covid map if you want to visualize the world situation.  [https://coronavirus-monitor.ru/](https://coronavirus-monitor.ru/)
