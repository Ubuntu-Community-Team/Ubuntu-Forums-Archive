### Post by xmastree on 2005-11-18
Welcome aboard, I hope you enjoy linux. Don't be afraid to ask questions, however stupid they may seem to you. Although it's always a good idea to search the forums first, chances are you question may already have been answered.
Also, there's [the guide]("http://www.ubuntuguide.org/") which is full of good advice.

Edit: I just noticed you're planning to dual boot, but have wiped Windows. You'll need to start over, install Win first, then install linux. Depending on your hard disk size, I would suggest using 1/4 as a primary partition for XP, 1/2 as a shared secondary partition for data and linux will install in the remaining 1/4 and will be able to read and write to the shared partition if it's FAT32 but read only if it's ntfs.
