### Post by Jucato on 2006-07-31
Saw it a few days ago and said "woah". I used it but didn't come up with any results.

I think the biggest problem would be is that this button is really just the Search button with a different button text. The problem? The default Search looks for **all** terms, with the exact spelling. 

So if you type in the thread title of "amarok crashes when playing MP3", what would be the chance of a post containing all the words "amarok", "crashes", "playing", "mp3"? While entering "amarok crash play mp3" or "amarok crashes playing mp3" in the search box at the top of the page yields some search results.

Try making a new thread (without click on Post new thread) and putting in different titles and clicking on the "Has this already been asked" button. See how "accurate" it is.
