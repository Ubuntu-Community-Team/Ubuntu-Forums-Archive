### Post by forrestcupp on 2012-05-21
The user interface from The Minority Report is getting ready to become a reality. [Leap Motion](http://www.engadget.com/2012/05/21/leap-motion-3d-motion-and-gesture-control/) looks pretty awesome. It's only going to sell for $69, and it's supposed to be 200x more accurate than the Kinect, or any other motion sensing technology out there. It creates a 4 cubic foot user space, and you'll be able to chain multiple devices together for larger user spaces. They say it will be easily integrated with anything out there with an onboard computer.

Hopefully we'll see support for Linux.
