New 9643.05 Legacy driver from Nvidia
[http://www.nvidia.com/object/linux_display_x86_96.43.05.html]("http://www.nvidia.com/object/linux_display_x86_96.43.05.html")

> Release Highlights

    * Improved compatibility with recent versions of X.Org X servers.
    *** Improved GLX_EXT_texture_from_pixmap out-of-memory handling.**
    * Improved compatibility with recent Linux 2.6 kernels.
    * Improved compatibility with SELinux systems

This includes the fixed texture_from_pixmap extension which was responsible for the black window bug. So hopefully us Legacy users will be able to use Compiz without black windows now :D
