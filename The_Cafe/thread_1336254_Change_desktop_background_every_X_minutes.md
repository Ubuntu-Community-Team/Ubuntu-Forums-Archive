### Post by Mr. Picklesworth on 2009-11-24
> **RiceMonster said:**
> Yes, I'm pretty sure GNOME 2.28 implemented that.

2.26, actually, but that isn't what's being asked for here. The time-lapsed backgrounds feature is awesome but it is designed for single cohesive packages of wallpapers (like the space ones, or Day of Ubuntu, or most of Fedora's backgrounds). The feature request is for the user to be able to simply select multiple individual backgrounds (of his own choosing) and have them be switched between in a slideshow.

Having just installed an amazing hdr-wallpapers package from the vdpau PPA, I can understand why people want this :)
