### Post by TheFu on 2022-06-13
Mozilla has forums. Ask there.
If you want process separation, there are many different methods.  I use firejail and run different browser instances inside 'private' containers.
You could setup a new userid to achieve similar results, if firejail isn't working.

Firejail doesn't work with snap packages. The container tools conflict with each other. If I move to 22.04, I'll likely purge the snap firefox and use a PPA version. Firejail protection is a core part of my work-safety.
