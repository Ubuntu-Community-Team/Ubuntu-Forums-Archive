### Post by MetalMusicAddict on 2006-12-17
Since this will no doubt be asked a couple of times I decided to create a thread.
> **damu said:**
> MetalMusicAddict, can you explain this decision a bit more. I don't get it. It seems Ubuntu Studio is about an open source system based on Ubuntu which purpose is to deliver authoring tools for audio, video and graphics. The first thing needed (and probably one of the only thing missing in Ubuntu now) for audio is an rt-kernel, and this thread was about that. 
Most of the other thread were very much about audio tools for authoring and hardware issues. It had nothing to do with most of the thread that we find in multimedia & video, which don't relate to authoring tools.
So how the previous forum was creating confusion and how is this one more appropriate?

Thanks

The Ubuntu Studio project has evolves since its inception. It started out as a WIKI page on how to tweak Ubuntu for audio production. It has since transitioned into becoming ints own installable disk whos focus is much more than audio now.

This change is something people have not realized even though Ive explained it many times.

[THIS]("http://www.ubuntuforums.org/showthread.php?p=1897932#post1897932") thread is just the most current thread to continue this confusion. This is also no fault of the thread creator. We have had many people coming to our development IRC channel thinking that because this thread was in the "Ubuntu Studio" section that it was our responsibility. The packages in the thread are in no way related to the work we are doing.

So with the constant confusion and in a effort to make a break from the old project I asked to have it removed.

We will have our first release in Febuary along-side U/X/Kubuntu.

If you would like more info ask here, join us on IRC at #ubuntustudio on Freenode or visit our [WIKI]("http://ubuntustudio.org/").
