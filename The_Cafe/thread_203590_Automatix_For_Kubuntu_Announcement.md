### Post by mstlyevil on 2006-06-26
**[COLOR="Blue"]The Automatix Kubuntu is out. It is version 6.2-6.[/COLOR]**

**[COLOR="Red"]All the reports I have recieved were positive so we have made it stable. Azureus is removed from the options for now until we can sort out a bug with it's execution.[/COLOR]**

**Here is what the newset version features:**

  01.  Installs KDE 3.5.3 (The latest version of the Kubuntu Desktop)
  02. Installs multimedia codecs
  03. Installs Firefox all Firefox plugins (java, flash, etc) (except Adobe reader and mplayer)
  04. Installs Amarok 1.4 (Updates Amarok to the latest version)
  05. Installs K9copy (CD/DVD software)
  06. Installs Kile (A KDE intergrated latex enviroment)
  07. Installs Knode & Kilibido (KDE news reader and klibido a usenet binary grabber)
  08. Installs Koffice 1.5.1 (A KDE Alternative to Open Office and the newest version)
  09. Installs Krusader (A popular KDE file manager)
  10. Installs Kftpgrabber (FTP client for KDE)
  11. Installs Superkaramba (Eye candy for KDE)
  12. Installs RAR, ACE and UNRAR archive support
  13. Installs skype
  14. Installs Acrobat reader 7 and firefox plugin for the same.
  15. Installs Frostwire (GPL clone of Limewire)
  16. Installs multimedia editors (Audacity (audio), Kino (video), EasyTag (ID3))
  17. Installs DVD (dvdrip) ripper
  18. Installs Mplayer and mplayerplug for Firefox
  19. Installs Xine-UI, Realplayer, and VLC player
  20. Installs Opera Browser 9.0
  21. Installs Avidemux (Video editing tool) (New version 2.1.0)
  22. Installs GIMP
  23. Installs Programming Tools (Anjuta (C/C++ IDE), Bluefish (HTML editor), Screem (Web Development Env.) and NVU (HTML editor))
  24. Installs MS true type fonts
  25. Installs Gaim Beta 2
  26. Installs Streamripper and Streamtuner
  27. Installs NON-FREE audio and dvd codecs
  28. Installs Laptop Wifi Network Tools
  29. Installs Checkgmail (A system try icon that alerts you to new mail in your Gmail account)
  30. Installs SUN'S JAVA JRE version 1.5
  31. Installs SUN'S JAVA JDK version 1.5
  32. Installs wine (u need to run winecfg manually after installation)
  33. Enables ejection of CD when CDROM drive button is pressed.
  34. Installs AMSN 0.95 (MSN client with webcam support)
  35. Installs Xchat (A very popular IRC chat client)
  36. Installs BUM (Boot-up Manager)
  37. Installs Swiftfox (A CPU specific version of Firefox optimised to be faster)
  38. Installs Swiftfox Pluggins (The same plugins as in Firefox)
  39. NVIDIA cards (Detects Nvidia cards and installs drivers) (Needs a restart)
  40. Installs Mozilla-Thunderbird 1.5 (US-only version) (no support for non-US-english language packs and enigmail)
  41. Installs Amule (A opensource Emule/Edonkey P2P clone)

I'm still open for more suggestions and I will try to figure something out on the Konqueror plugins that will not force a install of Firefox.

Please keep the suggestions coming in.
