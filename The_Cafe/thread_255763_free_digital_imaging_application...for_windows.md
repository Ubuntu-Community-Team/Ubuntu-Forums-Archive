### Post by Dragonbite on 2006-09-12
I'm using [Adobe Photoshop **Album SE**]("http://www.adobe.com/products/photoshopalbum/starter.html") for my picture (album) management. It also provides for some editing (like red eye, rotate, tagging, etc.)and some other things.  There is a free (SE) version and a paid version. 
For image manipulation, when I'm not using Photoshop (or Gimp) I am fooling around with [Paint.NET]("http://www.getpaint.net/index.html"). It includes the simplicity of Paint but with much more (filters, layers, etc.) so it is kinda-like an in-between of Paint and Photoshop/Paint-Shop-Pro. Part of it's claim-to-fame is that it is programmed in .NET and I think there was a mention on the Mono blogs that somebody got it to work in Mono (thus Linux, Mac, et. al)
