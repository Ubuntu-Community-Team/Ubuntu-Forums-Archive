### Post by epicoder on 2009-07-10
If any of you like the Brasillinux GDM theme (The Login Manager is watching you) but have a widescreen monitor, here is a widescreen version of it. To use this version, make sure you have gdm-themes installed. Take the attached image and run this code:

```
sudo mv -f [replace/with/image/path] /usr/share/gdm/themes/olho/olho.jpg
```
