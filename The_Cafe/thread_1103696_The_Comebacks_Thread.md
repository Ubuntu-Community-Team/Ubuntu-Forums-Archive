### Post by Giant Speck on 2009-04-01
I jump back and forth every now and then.

Ubuntu is my primary operating system, and I usually use Windows in order to use iTunes to sync my iPhone to my computer.

Usually, I hop back and forth between the two operating systems based on my interests at the time.  For example, when Windows 7 was released, I used it extensively for a couple weeks.  When I saw there were updates to the Murrine theme engine, I hopped back to Ubuntu so that I could work on my Macchiato theme.  When I got my iPhone, I hopped back to Windows in order to sync it with iTunes.  Finally, when Firefox 3.1 beta 3 was released, I decided to try it out on Ubuntu rather than Windows (since I only use Google Chrome on Windows).

Basically, I have no preference or partiality toward either of the two operating systems.  However, my use of either one of them is usually based on what interests me.
