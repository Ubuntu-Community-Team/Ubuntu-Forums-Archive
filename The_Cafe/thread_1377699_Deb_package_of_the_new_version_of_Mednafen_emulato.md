### Post by frenchn00b on 2010-01-10
Hello, 

[Mednafen emulator ]("http://mednafen.sourceforge.net/screenshots/")*has released a new version of its program. 

This program is very good and has high compatibility. alt+shift+1 (left side) can change the joystick input, and so on. 

**From source:**
```
apt-get install libsdl1.2-dev libcdio-dev   libsndfile1-dev   libsdl-net1.2-dev 

```
[tar.gz]("http://yellowprotoss.ye.funpic.org/website/mednafen-0.8-D-rc1/mednafen-0.8.D-rc1.tar.bz2")
then 
```
./configure ; make ; make install
``` 
  
  
Alternatively, one may find a deb for Ubuntu
[deb url]("http://yellowprotoss.ye.funpic.org/website/mednafen-0.8-D-rc1/mednafen_0.8-1_i386.deb")
 
```
dpkg -i mednafen_0.8-1_i386.deb
```
