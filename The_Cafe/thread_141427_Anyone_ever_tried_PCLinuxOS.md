### Post by nalmeth on 2006-03-21
I use the PClinuxOS liveCD when I go to work, because the work computer is really messed up with Windows :-?
PCLinuxOS comes with non-free codecs and plug-ins already installed (flash, mozilla-plugins, i think w32codecs etc) and a host of other non-free stuff. 
When I boot at work, everything works (even a funny ball wheel mouse), including automatically setting up a LEXMARK printer. That's right.
It's pretty slick, but I wouldn't install it personally. I tried a couple times, and the installer seemed so sluggish running with live evironment all the time, I think the installer just froze and crashed. Anyway, I use Ubuntu because I like APT, and I like the free idea.

PCLinuxOs is one of those companies that offers an "enterprise" or premium version at a cost, and while it would work great for a lot of people to start using linux (willing to cough up cash), only the liveCD is of any use to me.

### Post by Lunixfanboy on 2006-04-14
PCLOS is done by Texstar with support from the plf repositories. Texstar used to do a bangup job getting KDE packages ready for Mandrake before Mandrake did, and also had other items that Mandrake didn't do. Got to the point he felt they were taking advantage of him, so he split and came up with PCLOS, all the fineness of the texstar tweaks to KDE on a current Mandrake/Mandriva core. Only one flavor of PCLOS (freebie) but donations are always welcome.
