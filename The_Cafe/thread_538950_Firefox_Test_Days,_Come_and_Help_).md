### Post by starcraft.man on 2007-08-30
Hey there folks I figured I would draw people's attentions to the upcoming test days for Firefox 3 (one is tomorrow if your free). For those that don't know, test days are dedicated days when users get together in the Mozilla IRC room to just churn out tests and find issues with the current nightly build.

A good general article of this was recently highlighted on [Ars Technica]("http://arstechnica.com/journals/linux.ars/2007/08/30/becoming-a-part-of-the-firefox-quality-assurance-process") (good site for tech news). The links explain pretty much everything so no need for much expertise and friendly folk in the IRC room are sure to be available if you have any questions or issues the day of.

Participating isn't hard so new or veteran lets all get together to make sure the Linux (and others) version of Firefox is great :).

If there's any questions or problems before hand I'll be glad to try and answer. I'll try to be on but I can't guarantee I'm a bit busy of late >.>.
