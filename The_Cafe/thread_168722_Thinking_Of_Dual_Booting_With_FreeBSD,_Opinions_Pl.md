### Post by tseliot on 2006-05-01
It's fast but it takes more time to mantain it because, if you update the system to get the latest software from the Ports, you need to compile.

Make sure you install Bash.

I don't use it because (don't laugh) I need an easy and fast interface to connect my computer to my mobile phone through bluetooth.

I warn you that there are no ATI proprietary drivers on FreeBSD (but I use NVIDIA cards).

However it's a wonderful system and you can learn lots of things if you use it.
