### Post by heimo on 2005-10-22
To install vlc, you need to enable universe using these instructions:
[http://www.psychocats.net/sources.html](http://www.psychocats.net/sources.html)

Then you can use Synaptic Package Manager or apt-get to install vlc. (System->Administration->SynapticPackageManagement) or open terminal (hit alt+F2, type: xterm and hit Run) and write
```
sudo apt-get install vlc
```
