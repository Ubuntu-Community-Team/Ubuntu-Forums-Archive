### Post by EdThaSlayer on 2006-09-02
There is also a built in password manager(in GNOME)
Go to 

System->Administration->Users and Groups->click on one of the users[in the User tab]->Click on properties->in the password section click on the Generate Random Passwords->Generate the password and copy that one->use the password in any internet account you want!
