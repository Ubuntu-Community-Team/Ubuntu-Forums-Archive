So a better answer would be...

> Don't try to remove Linux. It's far too complicated, and in most cases Linux will tolerate Windows on the same hardware so you can dual-boot whenever you like.

You'll want to come back to Linux, anyway, when you find that none of your favourite apps are available in Windows, or that the names of equivalent apps are totally unfathomable.
For example, in Linux you're used looking in the menu for "Text Editor". In Windows, you'd have to search for the "Notepad" program...
