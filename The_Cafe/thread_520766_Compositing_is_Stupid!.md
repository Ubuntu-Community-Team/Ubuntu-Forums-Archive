### Post by igknighted on 2007-08-08
You are confusing compositing with excessively flashy tricks like compiz/beryl.  Compositing itself is just a different way for the computer to draw the desktop.  If you have a discrete graphics card, it is very useful because it offloads work from the CPU to the GPU, and can actually make your computer faster.  Xfce has a compositing window manager, and while it does allow you to do flashy things such as transparencies and wobbly windows, it has benefits behind the scenes as well.

So if you don't like beryl, thats fine.  But if you are going to rant, rant about the right thing.

### Post by starcraft.man on 2007-08-08
> **d3br074 said:**
> If it helps make it faster...then why do things get slower on my computers when running it?

Because things like Beryl/Compiz require a decent/recent video card and modern system (of course less than Aero). I guess maybe your system isn't powerful enough. Just like if you installed a game for the PC and only met the minimum spec (or under) you would see slow frame rate if you started turning on features. This is not a valid argument.

Oh and if you try Beryl and you want to make it run smoother on older machines change the following options on the general options page (Beryl Settings Manager > General Options > Main):

Texture Filter: Fast
Level of Focus Stealing Prevention: None (doesn't improve speed, just is an annoying feature that needs turning off in Beryl)
Detect Refresh Rate : Off
Refresh Rate: Manually Set to recommended for your screen (usually 60-75)
Sync to Vblank: Off

If you still have speed issues, turn off some of the high end animations/effects.
