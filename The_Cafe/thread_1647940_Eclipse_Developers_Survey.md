Who are they?
Mostly Java IDE developers.  Eclipse is an open source community of Java IDE developers.  It was created by IBM back in 2004.  1,365 respondents.  

Highlights of the survey:
Linux is now the most common deployment operating system.
Developers are shifting away from Windows and MacOSX to Linux.
Strong Ubuntu use not only in the desktop but also in deployment servers.

Blows away the myth that Ubuntu is for linux beginners, doesn't it?
Another myth busted here is that Ubuntu is NOT for servers.
