### Post by saracen on 2006-09-03
I just discovered [Tangerine]("http://www.snorp.net/log/tangerine").  From the website:

> Tangerine is an application that allows you to publish music over the local network, using DAAP. It runs on both Linux and Windows. There are several clients that you can then use to connect to it, such as Apple’s iTunes, Banshee, and Rhythmbox.

Music files can be specified either by a directory, or can be automatically discovered using Beagle (under Linux) or Google Desktop (under Windows). For typical configurations, a graphical tool is included.
