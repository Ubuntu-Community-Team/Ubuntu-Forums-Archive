### Post by directhex on 2009-10-11
Moonlight requires a bridging library, browser-specific, to read and write into the browser's DOM. So far, the only bridges written are for Xulrunner 1.8 (Firefox 2) and Xulrunner 1.9 (Firefox 3+)

So Moonlight sorta doesn't work on Webkit browsers like Chrome. It should be *really* easy to write for someone who knows Webkit's design properly (the FF2/FF3 bridges are only about 350 lines of code each), but it's not a priority item for the upstream Moonlight developers
