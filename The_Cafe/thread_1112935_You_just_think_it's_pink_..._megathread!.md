### Post by ayoli on 2009-04-01
I use this script in greasemonkey to turn back to a original background :

```
// ==UserScript==
// @name           ubuntuforums
// @namespace      ubuntuforums
// @description    ubuntuforums skin
// @include        http://ubuntuforums.org/*
// @include        http://*ubuntuforums.org*
// ==/UserScript==

// turns forums background back to original color
tags = document.getElementsByTagName("div");
for (var i = 0; i < tags.length; i++) {
	if (tags[i].className.toLowerCase() ==  "page") {
		tags[i].style.background = '#fff';
	}
}
```
