### Post by noelvh on 2009-10-10
I have been using Vbox for a few years now, and the only issue I have ever had was getting the USB to work. I have USB working, as well as sound and other things. For sound you can switch between Sound blaster, or ac97 sound cards in the options for the Virtual machine. Also make sure the setting is using the correct Linux sound. 
When you start Vbox, on the right scrol down to audio and select it. Make sure the check is in enable Audio. Then adjust from there. It dose take a bit to test the settings as you have to reboot allot thill you get it.

I hope this helps.
Noel
