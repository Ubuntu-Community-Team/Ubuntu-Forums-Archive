I haven't seen any discussion on this before (although I haven't actively looked, so apologies if this is already covered) but I wanted to know what people think about a lot of great Linux apps that have now been ported to Windows under the same license.

### Post by Solicitous on 2008-02-03
I can't really speak about hindering the number of people moving to Linux from Windows...if anything it probably helps if they've already had day-to-day exposure to commonly used programs, make them feel at home I suppose. 

I can say personally that I really like having Linux programs ported to Windows.  There was one stage where I had to run Windows for Uni, yet I was still able to run firefox, thunderbird, openoffice and pidgin (even though it was under a Windows environment).

The effect? I guess at the end of the day the market will decide what effect porting software from Linux to Windows will be.
