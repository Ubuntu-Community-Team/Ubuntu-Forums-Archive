### Post by smartboyathome on 2008-02-17
> **Vitamin-Carrot said:**
> What are the benifits of enlightenment over the standard gnome?

Advantages:
* It looks better
* It has animations for all sorts of things
* It is lighter and uses less resources
* It is easier to make themes for

Disadvantages:
* It doesn't have a system tray :(
* It can't use Compiz [1]
* It has to run either GNOME Settings Daemon or XFCE Settings in order to get some preferences to work (like GTK themes)

Overall, I prefer E17 over GNOME, and it is the first thing I install when I do a fresh install of Ubuntu.

[1] There is a plugin for E17 called ecomorph which is being worked on and already gives you a few of compiz's affects, but it isn't very many.
