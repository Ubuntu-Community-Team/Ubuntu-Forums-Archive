### Post by TuxLyn on 2012-06-04
Hi everyone, I hope this is right forum to post this in. I haven't been able to find contributions category. I've coded a tiny perl script that lets you view contents inside torrent files. If any one interested you can find it at [http://wiki.gotux.net/code/perl/btview](http://wiki.gotux.net/code/perl/btview)

Example:
```


./btview.pl Fedora-17-x86_64-DVD.torrent 

 btView v0.1 by Tux {2012-06-03}
 ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

   Created : 2012-05-29, 05:00:47

   Announce: http://torrent.fedoraproject.org:6969/announce

 Torrent Contents:
   Fedora-17-x86_64-CHECKSUM                
   Fedora-17-x86_64-DVD.iso


```
