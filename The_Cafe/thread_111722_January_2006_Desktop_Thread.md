### Post by darkmatter on 2006-01-04
[QUOTE=somuchfortheafter]can someone pleas explain the appeal of gnustep to me, i just dont get it. Is it mainly just for old machines because it looks like a stripped down version of flux/black box.[/QUOTE]

Technically, GNUstep isn't a desktop... it is an application development framework... based on the OpenStep specification created by NeXT (now Apple Computers)... It is a 'sibling' of Cocoa, one could say... It's not aimed at older machines as such... though it runs blazingly fast on them... its aimed more towards application developers who like the OpenStep spec...

What makes GNUstep so appealing, IMHO, is that it has the highest flexibility to power ratio I have ever seen in any environment. Also the fact that it has the most intuitive and elegant interface ever designed... (once again, my opinion... but one shared by many others)

### Post by tseliot on 2006-01-14
[QUOTE=ice60]hi, can someone show me how to change the panel like that? and are there other good panels?[/QUOTE]
Right click on one of the 2 gnome panels and select "New panel".

Then you can customise the new panel.
If you want to use a menu button like mine you have to download it (one of those buttons for KBFX from kde-look.org).

Then backup the current Ubuntu logo:
sudo cp /usr/share/icons/hicolor/48x48/apps/distributor-logo.png /usr/share/icons/hicolor/48x48/apps/distributor-logo.png_bak

Then replace the old logo with the KBFX button:
cd /home/your_username/folder_with_the_new_button

sudo cp name_of_the_new_button /usr/share/icons/hicolor/48x48/apps/distributor-logo.png

Now let's add it to the panel:
Right click on the new panel and select "Add to panel".
Select "Main Menu" and add it.

Type this to update the panels:
killall gnome-panel

Customise the rest of the new panel and remove your old panels.

If you want your Ubuntu logo back you have to type:
sudo cp /usr/share/icons/hicolor/48x48/apps/distributor-logo.png_bak /usr/share/icons/hicolor/48x48/apps/distributor-logo.png
