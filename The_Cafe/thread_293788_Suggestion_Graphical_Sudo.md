Apologies if this is the wrong place to post, I don't know where else to put this.

One feaure I propose is maybe a menu option, say in Accessories, let's call it Root Mode. When clicked, you're asked for your root password and after this you've now enabled Root capabilities for 15mins (or a specified time), perhaps some icon could let you know you're in Root mode with time ticking down? So now you can carry out tasks like editing files only Root can, or other graphical able tasks only root can (of course, apart from those that already have password prompts).
Maybe I'm wrong (I'm still fairly new...) and someone will point out why a graphical Root Mode is as bad as Linspire's Root mode by default, but I think if not suggested before, it wouldn't be a bad idea. The average Win/Mac man is still a point clicker. That doesn't mean out with Terminal though.

I'm interested in hearing responses; if only to learn of my flaws. :mrgreen:

### Post by ciscosurfer on 2006-11-05
You can accomplish this by issuing```
gksudo nautilus
```in a terminal or clicking ALT-F2 and entering the command.  

You can create a launcher that carries out this effect, place it on your desktop or the panel, add your favorite icon, et voila, graphical root access.
