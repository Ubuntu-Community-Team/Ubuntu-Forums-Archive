### Post by FuturePilot on 2007-11-12
Maybe so, but the benefits come back to you, the end user. Because it's open source anyone can work on the program, bugs can be fixed more rapidly, and new features can be added. Unlike closed source apps, bugs take longer to fix, new features don't get added as often and only a few people work on it which slows it down. Look at just about any open source project. It moves incredibly fast. Every couple of months or so, a new version comes out.
