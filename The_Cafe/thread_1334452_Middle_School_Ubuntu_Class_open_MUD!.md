Hello all.

-My name is Josh Beck and I'm a Technology Teacher in San Antonio Texas. For the last 3 years, I've been teaching my curriculum on Ubuntu, and I have one class I call 'Advanced Programming.' 

-The kids in the class have been writing a MUD. (Multi-User-Dungeon.)

-I have 25 this year that can use the shell, emacs, ssh, telnet, ftp like their lives (and their grades!) depend on it.

-The game is DIKU based, but modified like crazy.

-If you are into Linux or RPG's, check it out. It's a neat class project.

Thanks for supporting these students!
