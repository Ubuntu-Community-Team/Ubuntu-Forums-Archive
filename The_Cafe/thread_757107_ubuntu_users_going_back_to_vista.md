### Post by muncrief on 2008-04-16
For many years I used Unix and Linux at client companies, and at home, for engineering work only. I tried numerous times to use Linux as a desktop replacement but for various reasons never could.

But all that changed with Ubuntu, and the hard work that went into all Linux distributions by developers from around the world.

For about four years I've been running Linux as my primary OS, most of those running Ubuntu, which has consistently proved, in my experience, to be the most stable and just plain usable Linux OS to date.

While there are some programs that will only run under Windows, I bought VMware years ago before there were any truly viable open source virtual machine solutions to run them. But now there are many, including KVM (which will actually be included in the upcoming Hardy Heron release), VirtualBox, Qemu and its variants, Xen, etc. So if you need to run Windows programs you can simply install Windows as a virtual machine under Linux and you're set, no dual booting required. In fact I have a dual boot machine, but only need to use native Windows XP every six months or so, always for testing or debugging purposes.

But wait, there's something that is often even better than a virtual machine, and that is Wine ([www.winehq.org](www.winehq.org)). Wine runs Windows applications natively under Linux, and while it is a bit difficult to configure right now for novices, and it still can't run all programs, millions of people are using it to run Windows programs even as we speak without having to boot into a virtual machine. And there's a company called CodeWeavers that sells a Wine based package (CrossOver Office) that makes it easy for novices to install and use Windows programs.

One word of warning - there's also a company called Cedega that sells a low cost ($5.00 a month I think) subscription service to enable users to easily install and run Windows games, and it used to work quite well, but they starting concentrating on Mac's and stopped most work on Linux about a year ago, and are just distributing what most people consider garbage now. You're better off with straight Wine.

And one of the greatest things about Wine is that, unlike virtual machines, it can run Windows games. In fact, here's a list of all the Windows programs I currently run under Wine (and I'm not using CrossOver Office or any other third party Wine programs or scripts):

Windows Media Player 10
Firefox with Flash and Windows Media Player plugin
Opera with Flash and Windows Media Player plugin
Microsoft Office Word
Microsoft Office Excel
Microsoft Office Powerpoint
Need for Speed ProStreet Demo
FlatOut 2 Demo
Star Wars Jedi Academy Demo
Far Cry 2 Demo
Call of Duty Demo
Call of Duty 2 Demo
World of Warcraft
Steam
Half-Life
Half-Life 2
Half-Life 2 Episode One
Half-Life 2 Lost Coast
Portal The First Slice
Hald-Life 2 Deathmatch

All the above programs work great, except that the Windows Media Player Guide doesn't work and Firefox crashes when running some Flash pages. But with Linux Firefox, and Windows Firefox and Opera under Wine, I can browse any web content I've come across that doesn't explicitly require Windows Media Player 11. I can even watch videos at video.aol.com.

And one last note. Because of the incredible work that created Compiz and Beryl, Linux now has a 3D desktop second to none. In fact, Windows XP is now downright boring to look at, and Vista has difficulty simply drawing opaque window frames, and is widely acknowledged as one of the greatest technological blunders in history. And while Mac's do have a wonderful desktop and OS, they are ludicrously expensive, most can't be upgraded in any significant way, and simply serve as a testament as to why one should never buy computer hardware from a single source vendor.

I went through a ping pong stage of nuking my Windows partition and installing a Linux distro, running it for a while and then getting frustrated (mainly because I didn't know enough to make Linux work for me) and then switching back to Windows.

About a year and a half ago I installed Kubuntu for the first time and since then I've had only Kubuntu, Mythbuntu or Ubuntu running on my laptop and media centre. I don't see myself ever switching back to Windows on those machines. I even run Ubuntu in VMWare on my Windows machine at work.

The one machine that I still need to run Windows on is the one I use solely in my music studio. It seems that there aren't drivers for my hardware (M-Audio Firewire 1814) and there aren't any good music production applications (I use Cakewalk Sonar). If there were options for that setup then Windows would eat dust in that situation too.

Rock on Ubuntu 8.10! Looking forward to upgrading!
