### Post by ndefontenay on 2010-03-01
Hi.

You might want to have a look to the following in the Ubuntu application centre:


1) cheese (effects for your web cam).
2) If you want to install google earth, it downloads a script for you.
To run that script, right click your google script, select "start application with", in the command field, type sh. "sh". Or, open terminal (located in accessories) and type sh (google script name here).
3) You can install skype by downloading skype for linux ubuntu. Right click and choose install using synaptic.  (from memory but should be close enough).
4) Locate wormux in Ubuntu Application Centre. It's rather funny. A bit buggy but plays quite well anyway.


Have fun :)
