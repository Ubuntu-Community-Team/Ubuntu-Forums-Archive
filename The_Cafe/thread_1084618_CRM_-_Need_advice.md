### Post by andrewabc on 2009-03-02
I don't actively use any of these software programs. These are just lots of software that I found when looking for good business software.

[OpenERP](http://openerp.com/) - Open Source enterprise management software
[Sugar CRM](http://www.sugarcrm.com/crm/) - Open Source customer relationship management software
[Compiere](http://www.compiere.com/) - Open Source enterprise resource planning software
[Joomla](http://www.joomla.org/) - Open Source Content Management Systems
[Redmine](http://www.redmine.org/) - an open source flexible project management web application
[List of Open Source CRM software](http://en.wikipedia.org/wiki/Open_source_CRM_software)

Try out the demos on the websites. I think all the ones I listed except redmine has an online demo (all are free/opensource).
