### Post by 3rdalbum on 2009-07-28
> **Boss 999 said:**
> 1- More faster boot (approximately 10 seconds).

It's on the drawing board.

> 4- Newer version of Gnome please like Gnome 3.0.

Ubuntu always ships with the latest Gnome. Let's hope Gnome 3 is good.

> 5- DETECT ALL HARDWARE - Support for latest Nvidia Graphics Cards. Although it automatically donloads driver similar to the correct drivers, we need original drivers to get maximum graphics. Detect sound cards. My Creative 5.1 sound card was never detected.

I think you've misunderstood something. The Hardware Drivers program downloads the official Nvidia driver. If your sound card is not usable, it's because there is no driver for it. Support should come in time.

> 7- Bring the live experience of Plasma Desktop like that of Sabayon Linux 4.0 and higher. :o

Sabayon uses KDE, Ubuntu uses Gnome. You can't run Plasma on Gnome. If you want to use Plasma on an Ubuntu base, try Kubuntu.

> 8- More better Cpu & ram usage.

There are limits to how much RAM you can use for cache. If Ubuntu is only using 1.5 gigabytes of cache, it's because that's how much cache it can use without putting random bits of data into RAM.

> 9- A system Optimizer, an open source application, registry cleaner like that of  "Tune-up Utilities".

Windows and ReactOS are the only two operating systems with a "registry". Gnome has something similar, but it doesn't get clogged up and it doesn't cause slowdown. There is currently work being done on a kind of system optimizer.

> 10- Support for mobile to PC synchronization softwares like Nokia/Sony Ericsson PC Suites.

I thought there was already software to handle that.

> 11 - There should be in-build program to change the boot screen by the user rather than checking in the net how to do such a complicated operation. Boot screen should be available for download. "BOOT SCREEN SHOULD BE ANIMATED - it would be really lovely and awesome". There should be many Boot screens in this OS by default

If it only takes 10 seconds to boot up, why do we need an animated boot screen?

> 14- There should be better chat messengers designed for Ubuntu like Windows Live Messenger, Google Talk and Yahoo Messenger. "amsn" is designed for Ubuntu but the interface should be more catchy and made better than that in Windows.

That's a bit beyond the scope of Ubuntu itself, but there is work being done in this area, on Empathy.

> There should be a software like "Gnome Grub Editor" or G-Grub Editor for Ubuntu like "K-Grub Editor" in Kubuntu. "K-Grub Editor" is indeed a nice software.

There shouldn't be the need to edit your bootloader.
