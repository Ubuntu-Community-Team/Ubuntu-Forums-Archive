Apple Opens Up: Kernel, Mac OS Forge, iCal Server, Bonjour, Launchd

From [http://lists.apple.com/archives/Darwin-dev/2006/Aug/msg00067.html](http://lists.apple.com/archives/Darwin-dev/2006/Aug/msg00067.html) :
> 
A. Intel Kernel Sources
As of today, we are posting buildable kernel sources for Intel-based Macs alongside the usual PowerPC (and other Intel) sources, starting with Mac OS X 10.4.7. We regret the delay in readying the new kernel for release, and thank you for your patience.
B. New &#8220;Mac OS Forge&#8221; for Community Projects
C. New Open Source Calendaring Server
D. Apache-Licensed Bonjour and Launchd sources


I had already given up hope they would release the kernel sources for Intel-based Macs. I hope Apple will spend some more effort on open source from now on.

### Post by nursegirl on 2006-08-08
I am very excited about them releasing the source for the calendaring server. The Calendar apparently uses the CalDAV standard, but also integrates in Automatic Scheduling, resource scheduling and todo lists.

The OSS world and Apple could really work together to make this a standard to rival Exchange with clients for Linux, OS X and Windows. And then there would be no more reverse engineering headaches, as there have been for Exchange.
