### Post by engla on 2006-01-20
The problem is that this is not supposed to be a chat. Ubuntu has lots (and lots) of irc channels if you want to chat with ubuntu users -- this is not the same thing on the web, this is a forum.

A forum is a place for well-considered posts, and benefits alot from its searchability; a part of this is that the posts made by all members should be in some way complete, well-formulated pieces of a thread.

So.. you may have heard this before in any forum: "This forum is not a chatroom". Keep that in mind.
