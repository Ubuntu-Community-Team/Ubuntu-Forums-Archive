### Post by vasa1 on 2015-06-29
Apparently, it won't be possible to have something like this in Firefox 41 onward: ```
browser.newtab.url;file:///home/vasa1/Dropbox/MyLinks.html
```
This change, like many before it, will protect the user from malicious extensions.> The browser.newtab.url preference is abused and should be removed.[Source]("https://bugzilla.mozilla.org/show_bug.cgi?id=1118285")

An extension to restore just this feature is here: [https://addons.mozilla.org/de/firefox/addon/new-tab-override/](https://addons.mozilla.org/de/firefox/addon/new-tab-override/)

I'm hoping the dev of Classic Theme Restorer, which already has a lot of useful features, will provide this one as well.
