### Post by wannadumpwindows on 2008-08-06
It's called user agent switcher. Just search for it by name.

You can add other profiles too, so you can make firefox identify itself as any browser you want. Even an iphone or other mobile device.

### Post by FuturePilot on 2008-08-06
[User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/59")
