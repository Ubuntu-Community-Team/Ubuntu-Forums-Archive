### Post by pinguy on 2015-09-22
Pinguy Builder is a fork of remastersys.

The script can create a livecd of the installed system and works with *buntu systems.

You can either make a distributable livecd or a backup of your system.

Notable Changes:

[LIST]
[*]Single package for Script and GUI.
[*]Full EFI support including EFI system partition.
[*]Populate pool folder with grub-efi for offline installing.
[*]Update grub entry with the named used for the CD label.
[*]Uses XZ to compress the filesystem.squashfs for smaller ISO size.
[/LIST]
Version **3.*** works with *buntu systems 14.04 (may also work with 14.10. If not use version 4.*)
Version **4.*** works with *buntu systems 15.04+ (at the moment 15.10 beta1 has a dependency issue [xresprobe]("https://launchpad.net/ubuntu/+source/xresprobe"))
