### Post by jespdj on 2009-02-03
Can you explain in more detail what those issues are that you've had with the 64-bit version? How do you know these problems are specific to 64-bit?

Note that it happens very often that people blame problems on the fact that they're running the 64-bit version, while they really don't *know* for sure that it's because they're using 64-bit. Because of (unconscious) prejudice they too quickly conclude that it must be a 64-bit problem.

64-bit Ubuntu works just as well as 32-bit Ubuntu. I've been using 64-bit since Ubuntu 7.10.

Historically, Flash and Java have been a little bit harder to install on 64-bit than on 32-bit, but that hasn't been really true anymore for at least a year. Installing Flash on 64-bit is exactly the same as on 32-bit; you'll automatically get a wrapper installed to make 32-bit Flash work on 64-bit Firefox. And the IcedTea Java browser plug-in in combination with OpenJDK works for most websites that use Java.

It's going to be even better in the near future: there's now a beta version of 64-bit Flash 10, and in Sun's next update of Java 6 there's going to be a 64-bit browser plug-in. If these get into Jaunty, then the Flash and Java troubles will really be over.
