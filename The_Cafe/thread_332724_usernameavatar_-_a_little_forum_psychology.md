### Post by Randomskk on 2007-01-06
The base of my username, random, came from the first game I played multiplayer; Red Faction.
At the time kills were announced like "<someone> was killed by <someone>" and some people were using smart names like 'ur mum' to get the effect of:
[quote=Red Faction]
SomeGuy0345 was killed by ur mum[/quote]

Anyway I decided "random" for that, it worked ok, I stopped playing RF a few weeks later.
I kept the name though, and it's been fairly good, and I like it.
However, typically (this first happened WAY back when I started playing Runescape) Random is taken, so in that case I use randomskk. The SKK doesn't stand for anything, it's just an ergonomic extension to "random" on a qwerty keyboard.

Since alternating hands type keys quicker (see dvorak), the first letter is S. After "RANDOM", your left hand is on the D, so moving to S is very quick. Then you alternate back to the right hand, which is on M, and moving to K is easy. K is repeated since right hand's stronger, quickly to type the same key twice, and I didn't like "sk".

So in the end I get randomskk, which is quick to type (mostly it's all alternating hands (especially when using right-shift for capital R) ), I've never seen another randomskk (and neither has google), and I still use Random when I can.

My website's randomskk.net, but my email address is random@.


The avatar's an extension of a hobby; networking is fun and the avatar is a crop of a [picture](http://randomskk.net/uploads/servers/dark/Image154.jpg) of my switches in the dark.
