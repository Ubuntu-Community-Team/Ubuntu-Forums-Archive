### Post by forrestcupp on 2008-10-07
Kind of a misleading poll.  No, I wasn't scared of the terminal when I switched, but I prefer using the GUI.  When I first started using Linux, it was all command line, and X and DE's were just beginning, if you were technical enough to set a desktop up.

I used to love learning the command line.  But I also enjoy adapting to modern things when technology advances instead of hanging on to ancient techniques.  Today, we can do almost anything needed with a GUI.  It's more comfortable, and it looks more modern.  If I want to show off Linux, I'm not going to show them a terminal that looks like an ancient Commodore 64.  I'm going to show people that Linux is keeping with the times just as well as or better than any other OS out there.  I'm going to show them that you don't have to go to MIT to be able to figure it out.

But I do concede that when helping someone, it's easier for them to copy and paste commands.  But if I already know what I'm doing, I'm most likely going to use a GUI when one is available.

And we really should be beyond this debate.  We shouldn't be worried about whether or not to *have* GUI's.  We should be worried about how we can get them into 3D holograms instead of these antiquated monitors.
