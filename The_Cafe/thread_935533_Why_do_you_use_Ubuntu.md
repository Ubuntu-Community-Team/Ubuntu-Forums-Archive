### Post by jespdj on 2008-10-02
Here are my reasons to prefer Ubuntu over other Linux distros.

Ubuntu is easy to install and easy to use. I use my computer for serious work, so I need a stable OS that Just Works, and I don't have time to dive into all the technical details of Linux all the time.

Distros like Arch Linux or Gentoo are harder to install and require more technical knowledge. I don't want a "rolling release" distro, because it's less stable (there's a bigger chance that you install an update that breaks things, which is a business risk if you need your computer to do work).

I like the release cycle of a new release every 6 months and LTS releases that are supported for a longer period of time. With a new release every 6 months, Ubuntu stays on top of the latest versions of all the software. On the other hand, if you need stability you can use the LTS release for a longer period of time.

I personally prefer the GNOME desktop over KDE.

Ubuntu has a large and active community (especially here in the forums) - if you have a problem, ther are lots of people who can help you. The Ubuntu community seems to be larger and more active than those of other distros I've looked at.

I've briefly looked at OpenSUSE, but didn't really like it - maybe just because I'm used to the way things work in Ubuntu.

Fedora, as far as I know, has a more strict "100% open source" philosophy, which makes it harder to install Sun Java (which is the main thing I need for my job!), codecs for restricted formats and other non-free software.
