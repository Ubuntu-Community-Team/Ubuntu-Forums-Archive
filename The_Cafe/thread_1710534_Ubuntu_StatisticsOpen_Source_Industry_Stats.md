### Post by ubuntu27 on 2011-03-26
Please keep in mind that when people talk about market share, they are usually talking about the Desktop Market.

Linux is used the most in the Server market (Google and wikipedia uses Linux)

In the Web Server
Linux has a minimum of 64%,
Microsoft Windows has 33%,
BSD 2.4%....

In Super Computers:
Linux: 92.80%
Windows: 1%
Unix: 3.80%
