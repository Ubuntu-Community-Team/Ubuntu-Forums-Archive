### Post by Indras on 2006-09-16
There's an important saying I heard while I was shopping for a house a couple years ago:  A perfect house is one that is always under construction.

As soon as you stop working on it, it's no longer perfect.

The fact is, there's no such thing as a "complete" program.  Every program is a work in progress until it is abandoned.  Take Windows for example: Windows NT 4.0 has had at least six service packs, Windows 2000 has had four, Windows XP has two.  There are still bugs in Windows 95 that will never be fixed, since it is no longer supported.  Therefore, it was never actually "finished."

Heck, this is the reason we have version numbers on software.  AOL 9.0, Internet Explorer 7, Winamp 5, DirectX 9, Windows Media Player 11... these are all closed source projects.  They aren't finished, they never will be.  Eventually, people will lose interest, or it will become too expensive to track down the remaining bugs, so it is abandoned.
