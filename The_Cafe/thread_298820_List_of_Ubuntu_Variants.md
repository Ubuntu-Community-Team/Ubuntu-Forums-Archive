### Post by kopilo on 2006-11-13
**Ubuntu**
Base of Ubuntu distributions, comes with gnome.
**Xubuntu**
Based on Ubuntu but has an Xfcs interface and also aims to be a light version of ubuntu.
**Edubuntu**
Aimed at "bringing the spirit of Ubuntu to schools", uses the Ubuntu base whilst being aimed at and people going to educational institutions.
**Ubuntulite**
A version of ubuntu aimed at being both light and usable, or rather for the grandmother with an old computer who is unsure of how to use it.
**Fluxbuntu**
A version of Ubuntu with a fluxbox interface.
**Ubuntu CE**
Ubuntu aimed at christians, just basically packaged with different software, etc.
**Kubuntu**
Based on Ubuntu with a KDE interface.

If I've missed any out please add to the list, however if I've describe any variant incorrectly please correct me.
