Hello there.

I have been a linux user for one week (yes one big week). I've had so much pain installing Fedora core 3 on dualboot with winxp getting 3 different grub errors, assertions during install, not being able to mount cd, etc. That got me sorta worried about trying other distros that were not labeled as "for beginners". 

But today while surfing around I read alot of goods about Ubuntu, so I decided to order me a few copies with shipit. Good thing I couldn't wait and decided to burn it and install right away on this same hd. What a charm, honestly I was so astonished by how smooth it went! No need for fixes, no need to curse at grub for hours(it automaticaly detected the other two os) and it took me only like, what, 35-40 mins to install it get it to work perfectly? I will now definately stick with Ubuntu and throw away Fedora.  :razz: (Not it was that bad, but a pain in the ass for little me)

So, you guys will sadly have me around from now on!  ;-)
