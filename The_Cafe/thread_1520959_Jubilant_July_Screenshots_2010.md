### Post by hhh on 2010-07-06
@mmesantos1, if you install the development version of Docky, you can get rid of that mismatched Docky anchor icon. Remove Docky, add the ppa...
[https://launchpad.net/~docky-core/+archive/ppa](https://launchpad.net/~docky-core/+archive/ppa)
```
apt-get update
apt-get install docky
```Then set gconf-editor accordingly (scroll down to "ShowDockyItem")...
[http://wiki.go-docky.com/index.php?title=GConf_Settings](http://wiki.go-docky.com/index.php?title=GConf_Settings)
You can still access Docky settings by right-clicking the seperator in Docky (just have that option enabled before you remove the anchor).

        }

        }
