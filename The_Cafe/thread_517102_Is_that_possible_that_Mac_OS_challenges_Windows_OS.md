Not allowing the OS to be used on 'foreign' hardware makes a lot of sense, marketingwise. OS X has the reputation that it 'just works' and that you do not have to be a geek to get it to it work.

Once you loosen the restriction on the supported hardware, you will lose that very point. That, I believe, is today one of the most marked differences between OS X and - say - Ubuntu linux. Ubuntu also runs out of the box with a very wide range of hardware components. But, have you noticed the 'Hardware and Laptops' forum? How is it that this place is not only used  by many but desperately needed?

The decision to support only defined hardware environments might be beneficial to Apple at the moment If it is a sustainable remains to be seen. It is most certainly not the most straightforward strategy if gaining market shares is the object.
