### Post by kg4tah on 2009-03-07
I would suggest you download and start reading page 14 of this months full circle magazine ( issue 22).  It goes into detail on how to install a LAMP server on your ubuntu system.  It is as easy as eating a pie, LOL.

I have 8 Meg down speed and 512 k up speed.  I currently do not run a server however I have in the past (both a web server and quake 3 game server).  For no more traffic than I expected my 512k upload speed was all I needed for bandwidth however if you plan on having a high traffic server then the more upload bandwidth the better off you will be.

One point of caution, many isp's specifically prohibit running ANY server on your home system.  Some isp's run bots to scan the entire network for web servers and will send you a terms of service violation letter if one is located on your computer.

I never experienced that with roadrunner in my area however I saw in different forums for other roadrunner areas where this had been an issue for some.
