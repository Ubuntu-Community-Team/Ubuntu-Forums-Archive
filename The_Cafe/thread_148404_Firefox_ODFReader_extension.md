### Post by zero0w on 2006-03-22
Just found this [in Wikipedia](http://en.wikipedia.org/wiki/List_of_applications_supporting_OpenDocument) and search on Google: there is a new Firefox extension called ODFReader:

[http://www.alcoholicsunanimous.com/odfreader/](http://www.alcoholicsunanimous.com/odfreader/)
[https://addons.mozilla.org/extensions/moreinfo.php?application=firefox&id=1888](https://addons.mozilla.org/extensions/moreinfo.php?application=firefox&id=1888)

[http://wiki.services.openoffice.org/wiki/Firefox_ODFReader_extension](http://wiki.services.openoffice.org/wiki/Firefox_ODFReader_extension)

The extension is still very early in development and needs more effort to further its progress.

It will go a long way to help the spread of [ODF](http://en.wikipedia.org/wiki/OpenDocument) if everybody can read it without downloading the 70-100MB+ OpenOffice.org main program (at least on Windows or Mac), when a 20-200kb+ Firefox extension can do the job.
