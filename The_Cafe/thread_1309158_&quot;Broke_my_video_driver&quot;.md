### Post by 3rdalbum on 2009-11-01
I just thought I should remind people using Ubuntu that if you install a third-party driver from anywhere except the Ubuntu repositories, and don't take steps to register it with the DKMS system, it will stop working when the kernel gets updated.

There seem to be a lot of 'Karmic broke my video' threads at the moment, so please note that you should use official Ubuntu drivers during the update process or expect to have to recompile/reinstall your drivers after upgrading.
