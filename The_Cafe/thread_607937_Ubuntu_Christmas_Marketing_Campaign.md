Hi People,

I'm trying to get a Christmas marketing campaign started. Basically the campaign revolves around giving Ubuntu CDs to family, friends and colleagues as gifts for Christmas. Please see "[A Merry Ubuntu Christmas]("http://crunchbang.org/archives/2007/11/08/a-merry-ubuntu-christmas/")" for more information.

So far support for the campaign has been fantastic. If you'd like to help support it please consider **[digging the campaign]("http://digg.com/submit?phase=2&url=http://crunchbang.org/archives/2007/11/08/a-merry-ubuntu-christmas/&title=A%20Merry%20Ubuntu%20Christmas")**.

If you're a website/blog owner you can add the campaign image to your site with the following code:
