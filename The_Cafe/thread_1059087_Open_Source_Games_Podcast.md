### Post by MaxIBoy on 2009-02-04
I think it might have something to do with the hosting software he's using.

I'd suggest putting up a download in the Theora format.

EDIT: Also, you've got a couple technical inaccuraces: 
* A 500-gigabyte SSD was recently brought to market. There is no real limit to the possible capacity, but these things can get pretty big in terms of size. This 500 gig model is an external, it's not made to fit into your computer.
*"Open source" doesn't always mean free as in "free beer," it means free as in "free speech." Very often, the only way to get the whole thing for free is to download the source code and compile it, which can be a pain. And a lot of these things then require assets to run (so you can get the game engine for free if you compile it, but then you need maps, models, and textures which are copyrighted.) Quake 3 is a good example of this kind of game. It's open source, but you still need to buy it (or get a free mod.)


EDIT AGAIN: If you're doing FPS next time, try this one:
[http://icculus.org/alienarena/rpa/](http://icculus.org/alienarena/rpa/)
[http://icculus.org/alienarena/rpa/mediathumbs/aa2k8_24.jpg](http://icculus.org/alienarena/rpa/mediathumbs/aa2k8_24.jpg) <---screenshot too big to attach and too impolite to embed. Also note that the graphics aren't really cranked in that shot. Most people can't crank the graphics on this game. 

If you want a bleeding-edge version, where updates show up as soon as the programmers finish them, try checking out from SVN. (Instructions on the game's forums.) However, the stable release (v7.21) is recent enough to be reasonably accurate.
