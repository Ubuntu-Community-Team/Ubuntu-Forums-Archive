### Post by Rhubarb on 2008-01-06
If you read [http://fy.chalmers.se/~appro/linux/DVD+RW/](http://fy.chalmers.se/~appro/linux/DVD+RW/)
You'll notice that it says nothing about being able to play back DRMed blu-ray high definition movies.

Which means, in general, no, linux can not currently play back blu-ray movies you would buy from a shop.
If you have a blu-ray burner, you can burn data to a blu-ray disc, read from it, do what ever you like (except in some cases where the blu-ray disc contains DRM).

Please also note that known keys for blu-ray players that are found to be compromised do get revoked, so when a new movie title is released, it will not play with the compromised key.
While it is true that DRM is quite ineffective, and does get cracked soon after.  It is not something we should want to support where possible.
