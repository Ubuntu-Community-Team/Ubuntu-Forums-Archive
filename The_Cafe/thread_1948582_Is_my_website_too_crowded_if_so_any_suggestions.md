### Post by Paqman on 2012-03-29
Yes, I think it is. Too much going on at the top of the page.

You've got a bit of white space in there to separate items, which is good, but the fact that your main content box lacks contrast with your background and other elements on the page ruins the effect. Also, some elements are using gradients and drop shadows, and some are using blocks of colour, and there doesn't seem to be any consistency or logic to why. The various elements don't look like they're part of a cohesive whole.

I think there's a bit *too* much advertising above the fold. Your page is only about 20% content, the rest is all navigation and ads.

Overall though, you only need some minor tweakage to make it look quite nice.
