### Post by jhsu802701 on 2010-10-15
Windows 98 didn't seem that much better than Windows 95, and Windows XP didn't seem that much better than Windows 98.  However, Windows 98 clearly needed far more computing power than Windows 95, and Windows XP clearly needed far more computing power than Windows 98.  "Upgrading" a computer's OS inevitably means downgrading the speed and stability.  I've seen with my own eyes what happens when a computer originally equipped with Windows 98 is "upgraded" to Windows XP.  If I hadn't switched to Linux in 2007 at home, I might still be using Windows 98 today.  I've found the same to be even more true of other Microsoft programs.  I remember when my college switched from Word 5.0 to Word 6.0, I was thinking, "Why couldn't they leave well enough alone?"  Word 6.0 was agonizingly slow.  That said, Puppy Linux has greatly improved over the last few years.  I first tried Puppy Linux in 2007 (the version 2 days) and had difficulty getting it to work, so I used Damn Small Linux.  However, things worked much better in later versions, and I later adopted it as my main distro.  I have since moved to antiX Linux as my main distro, but I still use Puppy Linux as my backup distro.  I still like Puppy Linux better for burning CDs and for scanning documents into my computer.  I think the improvements in Puppy Linux are well worth the increased hardware requirements.
