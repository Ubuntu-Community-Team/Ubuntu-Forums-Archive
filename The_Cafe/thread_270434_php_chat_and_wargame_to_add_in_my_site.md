### Post by aipforum on 2006-10-03
For a chat, you really can't beat FlashChat.  It costs $5, but it's well worth it.
[http://www.tufat.com/script2.htm](http://www.tufat.com/script2.htm)

If you want a free option, I use a PHP/Ajax solution called YShout.  It can function either as a shoutbox or a chat room.  You can have it open in a window or just use an iframe.  Either way, it auto-refreshes.
[http://yurivish.com/yshout/](http://yurivish.com/yshout/)

(If you Google YShout, you'll see it sometimes listed as a shoutbox, sometime a chat system).

YShout is VERY easy to install.
FlashChat is a bit troublesome for the inexperienced.

Both should integrate with various CMS.
