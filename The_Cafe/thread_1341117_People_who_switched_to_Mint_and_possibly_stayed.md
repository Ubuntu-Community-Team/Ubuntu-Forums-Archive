### Post by Uncle Spellbinder on 2009-11-29
It's not all that different from Ununtu, to be honest. I am, however , really loving it. 

I know that installing the codecs necessary for DVD, avi/Divx/mkv, mp3, java, flash and so forth is not difficult. But it's really nice that these are all installed along with the OS. Especially nice for a newer Linux user. 

There are also some nice Mint-specific things and additions that differ from Ubuntu. The best way is to multi-boot, as I am. Nothing against Virtual Box, but I prefer a true install of any distro I want to try. You don't lose Ubuntu and can run Mint as well.
