### Post by maniacmusician on 2007-03-17
What I would do, personally:

1. Erase the hard drive
2. Partition the hard drive over again for a dual-boot. Give Windows 5-10 GB. Give Linux the rest.
3. Install XP on the Windows part of the drive; it's at least less troublesome than Vista
4. Install Ubuntu on the rest of the hard drive. It'll pick up your dual-boot situation and configure itself nicely. 
5. Now you have Linux, and a little bit of Windows if you decide you want to go back.

At any rate, I think that's better than keeping Vista on there. Even if something happens to Ubuntu, it's better to regress to XP rather than Vista.

### Post by sn0m on 2007-03-17
Mate divide your hard disc into four partition, ext3, fat32, swap and ntfs, you can nicely do that with gparted.
Then install xp on ntfs and linux on ext3, then you can use the fat32 partition to store data.
The advantage is that fat 32 can be accessed easily from both linux and xp and from my experience you would have to reinstall linux a few times, as well as the xp, especially if u like to tweek and tweek till u screw it up like i did. At the end of the day u can reinstall linux and not loose any data as they are safely stored in fat32 partition.
