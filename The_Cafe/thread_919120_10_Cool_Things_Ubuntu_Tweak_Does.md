### Post by SunnyRabbiera on 2008-09-14
> **iampriteshdesai said:**
> What are those scripts thing? I couldn't understand them. And what are those document templates?
I tried to change the location of user folders, but couldn't do so.

well for the scripts I have used in the past:
search within folder: allows a quick search function in the right click menu in the scripts, the main search tools search all over the computer but this script localizes it per folder.
open in gedit (as root): I think this is easy enough to understand, if you are in a folder owned by root you can right click on a text file and edit it as root. Using sudo in terminal becomes a lot less needed.
browse as root: browse a folder as root, easy enough to understand.
move to: provides a "move to" option similar to the "send to" option in windows.
