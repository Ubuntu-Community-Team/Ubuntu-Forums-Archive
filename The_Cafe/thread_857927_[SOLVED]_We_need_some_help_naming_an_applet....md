### Post by isaacj87 on 2008-07-13
Hello fellow Ubuntians,

For those who use Avant-Window-Navigator (AWN), some of you may remember an applet that allowed users to quickly and easily switch workspaces. It was called **BlingSwitcher**. I liked this applet because it was cool eye-candy and functional. Unfortunately, it was removed from Awn-Extras due to copyright/license issues and more importantly, it seems the original author has abandoned the applet. :(

Luckily, a kind user, sharkbait, has done a complete re-write of the applet and has fixed a lot of the problems that plagued the original. Plus, added some cool new features!

Only one problem, we can't seem to come up with a cool name for the applet! For now, it is only known as *Sin-nombre* (or "No Name"). I was hoping that you guys could help us out. The best I've got so far is Slickswitch(er). Yeah, I know...

**So throw out some names! I know you guys can come up with some good ones.**

Some features of the applet (to help get a better understanding):

-Quick switching with mouse scroll wheel
-It can detach and be placed anywhere on the desktop
-You can middle-click to show the desktop
-It will show you what windows are opened on any workspace

Below are some screenies of the applet to help (I chose a background so that people could see it better).
