### Post by Derek Djons on 2006-04-17
Well, just because of inner joy I had to post this. Today I compiled my very first GNU/Linux kernel. I even did quite some customizations using xconfig.

The only thing what failed me was the Intel 2200BG Wireless Lan adapter but I'll figure that out.

Using xconfig I found out the kernel chose to use generic processor drivers and quite some useless drivers I don't need to be in my kernel or even loaded as module. So now it's customized for a Intel Pentium M with quite some uncessary drivers disabled.

I've used only one [How-To]("http://www.ubuntuforums.org/showthread.php?t=157560&highlight=how-to+compile+kernel") Much thanks to MasterChief1234
