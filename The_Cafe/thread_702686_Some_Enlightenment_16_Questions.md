### Post by urukrama on 2008-02-21
> **aimran said:**
> 1. How do I make themes? The default theme is nice but I'd like something nicer. 

You can find themes at [freshmeat]("http://themes.freshmeat.net/browse/60/")

> **aimran said:**
> 3. When I run nautilus in terminal it loads gnome-extensions. This is annoying as it loads the gnome desktop. How do I avoid this? Change file manager?


You can launch nautilus with 

```
nautilus --no-desktop --browser
```

and it won't load the desktop.

You can also change file manager. Thunar is a great (and lighter) alternative.
