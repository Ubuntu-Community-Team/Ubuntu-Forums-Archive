### Post by gurnaik on 2009-07-27
On my Windows XP machine at work, I am using (not at the same time) and can recommend all of the following: 

Emerge Desktop: [http://www.emergedesktop.org/](http://www.emergedesktop.org/)

BBLean: [http://bb4win.sourceforge.net/bblean/](http://bb4win.sourceforge.net/bblean/) (I mainly use this, but version 1.16; I've had problems with 1.17)
Xoblite: [http://xoblite.net/](http://xoblite.net/)

LiteStep: [http://www.ls-universe.info/download.php?view.63](http://www.ls-universe.info/download.php?view.63) or [http://www.litestep.net/](http://www.litestep.net/)

SharpE: [http://www.sharpe-shell.org/](http://www.sharpe-shell.org/)


Shell switcher: [http://dx13.co.uk/programs/shellonv3.html](http://dx13.co.uk/programs/shellonv3.html) use to switch between all of the above (and Windows Explorer) at login
