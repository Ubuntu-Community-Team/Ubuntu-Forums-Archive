### Post by mips on 2011-07-13
I used the TinEye browser plugin before this, now I use the Google browser extension and when I want to search for an image I just right click on it and select search with google, no need to drag & drop etc.

Extensions for Chrome/Chromium & Firefox can be found here [http://www.google.com/insidesearch/searchbyimage.html](http://www.google.com/insidesearch/searchbyimage.html)

The above page also provides you with the 4 different methods you can use to search for an image.
