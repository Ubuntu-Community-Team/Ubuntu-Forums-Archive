### Post by dzon65 on 2011-04-06
It uses a small gui of course, but it's easy in use.

#!/bin/bash

sel=`zenity --title="Screenshot Options" --list --column="" "Whole desktop" "Delay screenshot" "Select area"`

if [ "$sel" == "Whole desktop" ]; then
   scrot -d 1 %T.png -e 'mv $f ~/Bureaublad'
printf "### screenshot - Done ###\n"
elif [ "$sel" == "Delay screenshot" ]; then
   num=`zenity --entry --text="Delay Time"`
   scrot -d $num %T.png -e 'mv $f ~/Bureaublad'
xmessage "screenshot done"

elif [ "$sel" == "Select area" ]; then
   scrot -s %T.png -e 'mv $f ~/Desktop'; notify-send "Screenshot done"
printf "### screenshot - Done ###\n"
fi
exit 0


Btw, "Bureaublad" means desktop.

[COLOR="Red"]**Warning:**[/COLOR] Misuse of the dd command can destroy a system. Make sure you have the in file as the correct path to the iso, and the outfile the correct device name for your usb device. Obviously this will erase the usb device when done (it will need reformatted). Also note it needs the device itself not a partition. For example on mine the device is: **/dev/sdb** (not /dev/sdb1). A DVD is probably safer (but if done right the dd way works :) )
```
sudo dd if=/path/to/debian-live.iso of=/dev/usbdevice bs=1M
```

Assuming you run KDE :D 


Install these things from the repos (search):
Lancelot widget
plasma-menubar
appmenu-gtk (this lets gtk programs use global menubar)


Install from ghns:
Kneda (think that is what its called)

Install from source:
```
hg clone https://bitbucket.org/panzi/smooth-tasks
```

Assuming you have the dependiencies installed you can just run the commands in the INSTALL file.


Once Installed:
Create a new panel,centerized throw smooth tasks on it.
adjust smoothtasks icon size to 60%
throw on any shelf widgets you like ( I am using +newdocument, inbox, system) adjust spacing with spacers.
toss two additional spacers on each end with flexible size to balance.

If this sounds complicated, its really not. Most of the configuration and installation is all gui-friendly. the only thing is you need to install smooth-tasks from source. :)
