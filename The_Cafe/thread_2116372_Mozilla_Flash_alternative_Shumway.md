### Post by BigCityCat on 2013-02-15
Shumway is a Flash alternative for Firefox being developed by Mozilla. I have heard some things about it but there hasn't been a lot of noise about it lately. What I found interesting is that there is a plugin for it and it works pretty well. I disabled flash in my browser and Shumway is getting the job done. I think you have to be using a Firefox version higher than 18. It's the best flash replacement I have seen so far. At least for Firefox anyway.

You can get the extension here.

[http://mozilla.github.com/shumway/](http://mozilla.github.com/shumway/)
