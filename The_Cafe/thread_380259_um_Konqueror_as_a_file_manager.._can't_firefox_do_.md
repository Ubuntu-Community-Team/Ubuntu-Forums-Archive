### Post by Erunno on 2007-03-09
Konqueror is not the best browser, that honour goes to Opera ;-)

Konqueror uses small, independent components called kparts which are embedded in its shell. Firefox can't do that because it was not made to use the component architecture of KDE and probably never will as it is intended to be OS/DE independent. There are not only components for web and file browsing but also for reading PDFs, editing files, viewing pictures etc. It can be useful if you happen to have a lot of documents open while working as it is easier to manage documents via tabs contrary to managing them via the taskbar but it really depends on your own personal workflow. Most of the kparts can also be started in applications of their own if you prefer it that way.
