### Post by dtfinch on 2010-05-09
I used to write and sell screensavers, about 10 years ago. I still get support emails for them. XP broke them all when it came out, and Vista's UAC broke them again. Some people would complain that it slowed down their servers, which shouldn't be running screensavers in the first place. Then when I made them run at idle priority, other people with a lot of other stuff in the background would complain that they were jerky.

I don't use screen savers on my own systems, apart from blank screen or logo. Best case, they do nothing. Average case, they compete with other background programs for resources. Worst case, some 3D screensavers would crash your system after X hours. It was so common that Intel's display drivers for Windows would disable hardware acceleration for any program with a .scr extension. [http://www.intel.com/support/graphics/sb/CS-004684.htm](http://www.intel.com/support/graphics/sb/CS-004684.htm)
