### Post by sumguy231 on 2008-01-14
If you're interested in learning more about how a complete Linux system is put together (and have a LOT of free time) then building a Linux From Scratch system (I recommend doing this in a virtual machine, by the way) is a good idea:
[http://www.linuxfromscratch.org/](http://www.linuxfromscratch.org/)
It's basically a free online book which tells you how to make a system from scratch. I recommend using their LiveCD as a host system so you don't have to worry about gathering all the requirements.

No actual coding necessary but in "real life" so to speak most distros do have programmers to fix bugs in upstream software packages and contribute to them.
