### Post by TheFu on 2022-07-05
I click "Settings" to see if any threads I'm involved with have updates by others.
After dealing with those, I'll query for "Unanswered Posts" in the Quick links to see if any subjects jump out.
If I have more time, I might query for all "new posts" or "Today's Posts" ... if that is available.

Placement of new posts into the correct forum by users is seldom correct, so I ignore that, unless it is in the "New Users" forum, which I avoid. Over the years, I've learned that new users ask GUI questions, which are outside my skills (I use fvwm, not any DE), so unless the question is specific to a program, it is likely an answer from me would be confusing.  Frustration for everyone involved.  Everyone thinks just a little bit differently.
