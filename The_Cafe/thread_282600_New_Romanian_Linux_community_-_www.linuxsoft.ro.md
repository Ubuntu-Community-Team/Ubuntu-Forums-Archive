### Post by i_m_meen on 2006-10-29
Ok, to clear the air. This is a general Linux site, not a distribution specific site. ubuntu.ro is a great site, but it is only for Ubuntu users. 
Linux Soft is dedicated to all Linux users, including Ubuntu users, but not only to them. Let's not forget that Ubuntu isn't alone, in its development: it's based on Debian, it takes all sorts of programs from Fedora/Red Hat, and sends back its contributions. So Linux is all about interactions, with all sorts of communities.

So there should be a site for Romanian Linux users from any distro. There already are some, but Linux Soft wants to be the alternative for those who don't feel at home on those sites.
My 2 cents.
