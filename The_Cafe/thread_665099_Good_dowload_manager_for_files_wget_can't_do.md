### Post by ice60 on 2008-01-12

but, i also use a great program called Desktop Data Manager, it's a download manager, clipboard manager and screenshot program, i use it for the clipboard manager and load it at boot, it keeps a history of both clipboards and shows pictures too if you've copied any to your clipboard. 

i don't know much about the download manager though, but i'm sure it will do what you need.
