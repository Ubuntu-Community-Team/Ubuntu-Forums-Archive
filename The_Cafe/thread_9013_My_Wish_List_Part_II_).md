Bored at work so I'll throw in my $.02

1 - While I think Firefox is a great piece of software, I don't think Thunderbird is quite ready for primetime yet and really prefer Evolution when running Linux.

8 - XMMS absolutely.  Maybe ubuntu could take a page from Red Hat and include a copy of XMMS in the base system without the MP3 libraries?

9 - K3B is a great program.  I'd love to see a version that integrates more smoothly with the ubuntu default theme.
