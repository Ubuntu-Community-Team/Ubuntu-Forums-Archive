### Post by gletob on 2009-12-20
So I go to pick up the Lucid Alpha 1 to play with in VirtualBox and I see the normal .torrent links, and .zsync files.

So I decided to try out Zsync by running 

```
zsync http://cdimage.ubuntu.com/releases/lucid/alpha-1/lucid-desktop-amd64.iso.zsync -i ubuntu-9.10-desktop-amd64.iso -o lucid-desktop-amd64.iso
```

And bam 50% of the Karmic ISO file was usable for download.

Now of course it's still best to grab your ISOs from the torrents to reduce server load but if you want to get the ISOs fast or just want to upgrade your current ISO to the newest version ZSYNC is the way to go.

BTW I'm sure a lot of you might already know about this, but I think it's really cool, and just had to share it.
