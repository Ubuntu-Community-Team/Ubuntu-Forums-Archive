### Post by chessnerd on 2009-08-21
When I investigated other browsers I noticed that IE loaded images all at once and that Firefox loaded them from the top down. The IE way was slightly faster, but the Firefox way felt quicker because you started getting data right away. I think Firefox may have changed it to save a few milliseconds of load time for images/pages.

Personally, I really like using Opera because it has an intelligent loading engine (is that a term?) in that it loads text first, then it starts to format everything, then it loads images (unless, of course, the web page is designed to not work that way) and it has a built in feature that allows you to decide whether or not to load images at all. It works great on my grandparent's dial up because the text is usually what you want to see first. Of course, if you have cable or faster then this doesn't matter as much. :)
