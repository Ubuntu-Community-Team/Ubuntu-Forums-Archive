### Post by 3rdalbum on 2009-04-19
Boot Camp is three things:

1. A partitioner
2. A set of drivers for Windows
3. A utility that patches your iEFI to include features that Apple had disabled in their machines.

It was possible to run Windows on the Macintosh before Boot Camp was released - in fact it looks like Apple released it in response to the hackers who got it working on their own.
