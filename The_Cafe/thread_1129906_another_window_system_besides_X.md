### Post by ddrichardson on 2009-04-19
Its not unreasonable to suggest that developing a new windowing system is akin to developing a different kernel. Gnome, KDE, XFCE and so forth are all built on Xorg.

Linux applications are built on them so really you're asking to pull the rug from underneath all those efforts and have to start from scratch with application support.

Besides, Xorg works.
