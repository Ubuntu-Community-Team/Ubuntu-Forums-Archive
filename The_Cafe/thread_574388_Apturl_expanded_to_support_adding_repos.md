### Post by smartboyathome on 2007-10-12
I was just wanting some input on what others thought of an idea I had been thinking about ever since I found out about apturl.

Maybe the function of apturl could be expanded to include adding repos to your sources.list. This would make it easier for many to install apps. It could be a security threat (but, then again, 3rd party repos already are), so maybe there could be something like "Do you want to add this repository to your sources.list? Repositories could contain harmful software and are not supported by Canonical or Ubuntu developers." There would be a yes or no box like the current apturl, and then would ask for your password and would add the repo to your sources.list. Maybe it could even allow installing keys to the repos by clicking.

This is just an idea. Post your opinion.
