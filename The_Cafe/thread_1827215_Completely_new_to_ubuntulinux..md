### Post by Erik1984 on 2011-08-18
Welcome aboard. Well I use Ubuntu for my daily computing needs. I'll give an overview of tasks and programs I use for them, underlined programs are installed in Ubuntu 11.04 by default.

- Browsing the web: _Firefox_
- Listening Music an online radio stations: _Banshee_
- Work on thesis: Latexila for LaTeX and _Evince_ for PDF documents
- Creating text documents: _LibreOffice Writer_
- Creating useless spreadsheets: _LibreOffice Calc_
- Programming and scripting: Geany, _Nano_
- Torrents: _Transmission_
- Light gaming: _Aisleriot Solitaire_, OpenTTD, Battle For Wesnoth and many more.
- Creating graphics (I'm not good at that): Inkscape
- Editing Video: OpenShot

Please note that for every task I mentioned alternatives exist in the Software Center, you just have to try out which ones you like best. Maybe you won't find any that is as good as the proprietary solution you're used to. In that case you could try to run your Windows software through Wine but sometimes there is no Linux alternative. Therefore Is suggest if you have Windows installed to keep it around (by a dual boot setup or by running ubuntu as Virtual Machien first) until you feel absolutely sure about going with Ubuntu/Linux.
