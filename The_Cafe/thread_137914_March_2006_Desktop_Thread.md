Does anyone know where I could get some OSX-like icons? :)[/QUOTE]

Here goes the best set I have found. It is called exquisite, and it is located on gnome-look.org. To install them download them then go to System-->Preferences-->Themes. Click on the Install New Themes button and then navigate to where you downloaded the icon pack. Select the icon pack and hit OK. Then all you have to do is select the Theme Details button, click on the Icons tab and select exquiste and close the themes box. You will then have it installed.

People, **please stop posting ridiculously large pictures**. First of all, you're breaking the format of the forum (which is annoying.)
