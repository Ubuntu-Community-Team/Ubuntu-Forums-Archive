### Post by kio_http on 2012-10-01
> **lucazade said:**
> is there any ppa for oxygen-transparent in quantal or it is still necessary to patch and compile it by hand?

No ppa that I am aware of,  but you can use a script to handle the installation for you.

Do

```
sudo apt-get install build-essential cmake  libxrender-dev libx11-dev git kdelibs5-dev kdebase-workspace-dev
```

Then save scrpt from [here]("http://kde-look.org/content/show.php/?content=127752") in a directory and run with

```
sudo perl oxygen-setup.pl
```
