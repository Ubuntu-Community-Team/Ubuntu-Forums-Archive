### Post by goodtimetribe on 2007-10-24
> **henriquemaia said:**
> Thanks. I have ex falso installed and I like it, but there are some features that I miss. I was wondering if this was more complete.

DEFINITELY not feature packed. The interface is great and i do like how it's able to easily change specific tags on multiple files, but THAT's all it's good for. It can't do WMA, and since it has no ability to REMOVE files from it's queue, you have to exit the program after you're through with one batch and then reopen it in order to use it with a clear queue. however, the first and only feature i mentioned above is definitely worthy of the bother of installing it, and also worthy of further development. I don't know that I've found a suitable replacement for what it does do well.
