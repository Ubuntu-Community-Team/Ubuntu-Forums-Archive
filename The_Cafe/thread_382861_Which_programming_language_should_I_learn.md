### Post by reacocard on 2007-03-12
Python. I have only recently begun programming myself, and I have found python to be both reasonably fast and easy to use. Also, as it is an interpreted language, programs can be run nearly unmodified on any platform python is available for. The core python runtime is only a 10 meg download too, so it's lighter than java, and there are tools allowing you to wrap everything needed to run your program into one windows executable as well. Python is also suitable for server-side scripting, so you'd be able to use it for that as well.
