### Post by Ozor Mox on 2008-11-06
Try this, I find it works for me:

1. Backup all important data.
2. Run upgrade.
3. Try out Ibex. If everything works proceed to step 4. If things are broken, proceed to step 5.
4. Do a little dance. Make a little love. Get down tonight, doo doo.
5. Attempt to fix or work around broken things with the help of the forums, documentation, bug reports etc. If successful, proceed to step 4. If unsuccessful, proceed to step 6.
6. Reinstall Hardy, restore all important data backed up in step 1, and stick with what works.
