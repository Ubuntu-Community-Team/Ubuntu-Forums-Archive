Okay, to improve the user experience, the "System" &#8594; "Configuration" needs some cleaning up. The visual and audio settings could be placed together and polished.
[David D Lowe]("https://launchpad.net/~flimm") or flimm is already working on a new [theme manager]("https://launchpad.net/epidermis/") called *Epidermis*. This will be a great improvement, but unfortunately it won't be ready for 8.10 :(
Epidermis will handle:

    * Wallpapers
    * Metacity window border themes
    * GTK widget themes
    * Icon sets
    * GNOME splash images
    * Mouse cursor themes
    * Grub bootsplash screen
    * Usplash bootsplash screen
    * GDM login screen
