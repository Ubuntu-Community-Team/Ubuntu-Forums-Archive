Just wondering if anybody is still using [Google X]("http://q4q.org/")?

What is(or was) Google X?

> **Google X** was a project released by [Google Labs]("http://en.wikipedia.org/wiki/Google_Labs") on [March 15]("http://en.wikipedia.org/wiki/March_15"), [2005]("http://en.wikipedia.org/wiki/2005") and rescinded a day later. It consisted of the traditional [Google]("http://en.wikipedia.org/wiki/Google") search bar, but it was made to look like the [Dock]("http://en.wikipedia.org/wiki/Dock_%28computing%29") [user interface]("http://en.wikipedia.org/wiki/Interface_%28computer_science%29") feature of [Apple]("http://en.wikipedia.org/wiki/Apple_Computer")'s [Mac OS X]("http://en.wikipedia.org/wiki/Mac_OS_X") [operating system]("http://en.wikipedia.org/wiki/Operating_system").
 Google has not released any official statement as to why the project was shut down. Apple has [patent]("http://en.wikipedia.org/wiki/Software_patent") protection covering the implementation of the Dock effect, together with [trademark]("http://en.wikipedia.org/wiki/Trademark") protection for the graphical design elements of the Dock.
 Since Google X has been taken down, several mirrors still exist.
[http://en.wikipedia.org/wiki/Google_X](http://en.wikipedia.org/wiki/Google_X)

Here is a complete [list of Google X mirrors]("http://mirrors.goox.se/").
