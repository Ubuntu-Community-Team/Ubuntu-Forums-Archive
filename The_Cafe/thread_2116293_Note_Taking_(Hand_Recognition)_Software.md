### Post by Favux on 2013-02-16
Hi ATXChris,

Unfortunately there really aren't any handwriting recognition app.s for linux that I am aware of.  Evernote did have a web demo a few years ago for linux and I think there is another proprietary app. with a web demo also.  Xstroke development halted years ago and it no longer builds.  Although I believe some Distros had it building long after it stopped in Ubuntu.

Most folks make do with Xournal or Jarnal and CellWriter.  CellWriter has letter recognition.

You can get it using the Ink2Text project.  That uses Wine and Windows XP tablet PC libraries.
[http://sourceforge.net/projects/ink2text/?source=directory](http://sourceforge.net/projects/ink2text/?source=directory)

Installing it looks like a project.  Don't know if it works in Precise or later.  I saw someone wrote some detailed installation instructions for Natty 11.04 in the project's HELP.  The author also had a thread or two on it in this forum.
