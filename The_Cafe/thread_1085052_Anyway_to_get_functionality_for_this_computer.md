### Post by Ozor Mox on 2009-03-02
Xubuntu would definitely not work if DSL doesn't. My only recommendations are to use the Ubuntu alternate CD to do a command-line installation using the text-based installer. That should get you a base system on which you can install a light window manager and some lightweight apps. You could also do a Debian net install for pretty much the same effect.

The only other distribution I can think of that is lighter than DSL is Slitaz (I think I spelt that right), maybe that would work?
