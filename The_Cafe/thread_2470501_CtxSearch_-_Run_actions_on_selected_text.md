CtxSearch - Run actions on selected text
Most desktop applications - text processor, web browser, terminal - allow the user to selected text, and do actions on the text, using a context menu.  I wanted to add my own personalized actions to these menus.
Unfortunately, there is no standard way to this.
This is a solution to this problem. Run CtxSearch in the background. Selection a region of text in your favorite application. This will wake up CtxSearch, which will show a floating icon near the mouse position.
