### Post by Paqman on 2013-05-26
The term "web app" can describe a lot of things. Web apps running on things like Java have been around for years, and have nothing to do with what DE you use. These days you'll see more stuff like AJAX, and arguably any modern site doing a lot of client-side scripting through js is a "web app". So that would include stuff like web mail.

Or do you mean desktop integration of online services, such as the way Gmail and Twitter can hook into Ubuntu now?
