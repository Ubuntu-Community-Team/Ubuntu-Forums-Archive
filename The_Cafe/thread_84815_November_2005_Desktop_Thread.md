### Post by matthew on 2005-11-01
[quote=YourSurrogateGod]Heh... the fox taking a bite out of the e. BTW, how did you get that Mac OS like setup at the bottom?[/quote]
I'm using [gDesklets]("http://gdesklets.gnomedesktop.org/"). That one is called [starterbar]("http://gdesklets.gnomedesktop.org/categories.php?func=gd_show_app&gd_app_id=210"). Lots of cool eyecandy on that site.

The main gdesklets program can be installed using synaptic and then individual desklets downloaded from the web site. Installation of the individual desklets is easy, just start gdesklets form the terminal
```
gdesklets
``` and then drag the downloaded desklet's archive icon into the open gdesklets window.

### Post by mcduck on 2005-11-14
[QUOTE=benplaut]can you post the relevent section from .devilspie.xml to get conky working correctly? tried forever, still didn't work :confused:[/QUOTE]
Sure, here's my .devilspie.xml, hope this helps :)

```
<?xml version="1.0"?>
<!DOCTYPE devilspie SYSTEM "devilspie.dtd">

<devilspie>

  <flurb>
    <matchers>
      <matcher name="DevilsPieMatcherWindowName">
        <property name="application_name" value="conky"/>
      </matcher>
    </matchers>
    <actions>
      <action name="DevilsPieActionSetWorkspace">
        <property name="pinned" value="TRUE"/>
      </action>      
      <action name="DevilsPieActionHide">
        <property name="skip_tasklist" value="TRUE"/>
      </action>
      <action name="DevilsPieActionHide">
	<property name="skip_pager" value="TRUE"/>
      </action>
    </actions>
  </flurb>

</devilspie>
```
