### Post by drs305 on 2010-04-04
Yes, I have the Oracle logo on OO 3.2.

Not an editorial comment, but if you don't want to see the OpenOffice logo you can run this command to turn the logo off:

```
sudo sed 's/Logo=1/Logo=0/' -i /etc/openoffice/sofficerc
```
Reverse the 0 and 1 to turn it back on again.
