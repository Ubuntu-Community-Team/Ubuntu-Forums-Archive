### Post by sandyd on 2010-05-17
When you go to office live workspaces right now ([http://workspace.office.live.com](http://workspace.office.live.com))
you will likely be faced with the unsupported OS window.

Heres an easy fix.

1. Download User Agent Switcher
2. Restart
3. Go to Tools -> User Agent Switcher -> Edit User Agents.
4. Create a new one with the data below
```

Description: Firefox
User Agent: Mozilla/5.0 (Windows; U; Windows NT 6.1; ru; rv:1.9.2.3) Gecko/20100401 Firefox/3.6.3
Platform:win32

```
Leave everything else as they are.

now, go to tools -> user agent switcher, and select firefox.

have fun using it.
