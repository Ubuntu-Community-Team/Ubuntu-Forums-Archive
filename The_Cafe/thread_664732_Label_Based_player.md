I was wondering why there isn't a music player that uses labels to sort and create playlists for your music.Like how Gmail uses labels for your mail.

For example you have a slow Brazilian rock song that you added recently. 

You could add labels: Slow, Rock, Brazil and New

So then you can create a Slow playlist that will look for the label slow. 

Similarly you can create Brazil, Rock, and New playlists. 

You also mix labels to create a variety of playlists. 

So does anyone else think this would be useful?

Also is there a player that has a similar feature.
