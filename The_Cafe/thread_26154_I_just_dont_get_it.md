### Post by bryan986 on 2005-04-12
Well, ive tried ubuntu for a while, and linux just isnt for me,  I just dont get what is so great...linux in general is such a hassle

Here are some points

- What is the deal with so many different locations for applications? I would prefer one folder for each application, and one folder to hold all the applications...why does one application need to be spread accross 5 different directories? This is totally confusing and a waste of time when looking for files or managing programs

- Why does half of the functions require the terminal? Spending half the time in linux fumbling with the terminal is terribly anti-productive, and much more difficult than it needs to be, a simple GUI would be extremely helpful in most situations

- A universal package system would be nice, sure you can use source code but that involves a whole nother level of compiling, editing configuration files, unessesary work.  then there such things as rpm, deb, etc but sometimes those dont work, or they only work half-assed, creating broken pages lying around your system, they might not work on your system, or you have to convert them to another package format, and then edit configurations for another hour, what is the problem with letting a program decide where everything needs to go and letting it do the configuration

- Lack of hardware support is ridiculous too, sure they claim that most hardware works, but in most cases, this is at half capacity. even with video drivers its confusing, do I get the ati driver in synaptic or do I get the one from ati.com? ok so I install the deb package, now I have to spend an hour editing the xorg.conf, and then another hour figuring out how to get dual monitors to work? WTF? when I try to get my wireless usb adapter to work, I end up having to spend another hour to get ndiswrapper to install my windows driver, and after that it just freezes and crashes my sytem!?

- My windows sytem runs FOREVER, I have not seen it crash or hang in a year at all while my linux has crashed and hung numerous times in a single weekend, and I thought linux was supposed to be more stable?

- If im looking for an application to do something particular on linux, half the time nothing exists yet, and then the other half is installation hell again with configuring and installing. want java to work, gotta get the package from sun, spend a half an hour to get it to run without errors, and then copy some obscure file manually to get the plugin to work in mozilla. I could not get irc to work at all, with gaim or with xchat, not to mention the fact that you needed a third party plugin for xchat just to get the thing off the screen and into the system tray

- what the hell is the deal with sudo. Why on earth would you restrict the computer from the person who is using the computer?!?! this is so stupid, I want to use the computer without having to type in passwords, run sudo commands, just to use the more advanced functions of MY computer. try to copy a system file, run a system program, oops, gotta type in a password, gotta run it as sudo. so you say this protects me? from what? myself? how the hell am I supposed to know what a package is going to do, it does not tell me until it is installed and I let it run. so you say only install from trusted sourcs? well lets just go ahead and limit me to 5% of all linux software...THANKS. i just want to use my computer with FULL ACCESS, not some semi-user junk. linux is suppose to be secure right, well if that is true, then what is super user for when nothing can get onto my computer on its own? yeesh...

- not to mention linux hell. linux is already dying. too many distrobutions for its own good. this is the same thing that happend to unix. the distrobutions get so far apart from each other, that they will no longer be compatible and totally defeat the purpose of linux itself. the linux community is a giant mess of code that is mashed around to fit with peoples ideas of what linux is supposed to be, it keeps combining and diverging apart, further increasing the mess that is linux

- why do you think apple and microsoft are so successful? because their stuff JUST WORKS. sure it costs, but you get what you pay for, thinks work the first time you do it, it is easy to use, you dont have to know about the innerworkings of the system just to use simple functions, there is a wide userbase, a wide range of programs, a wide range of hardware, really, how hard is it to install virus and firewall protection on your computer, frankly  I like having this level of control over my system rather than trusting something in the operating system to do its job without knowing anything about it, bottom line is that I want to use the tools on my computer, not have to spend half the time working with the computer itself

ill try linux again another few years down the line when it is a REAL operating system
