### Post by Nuno_Abreu on 2016-02-04
To fix firmware on AMD processors, mainly for stability and efficiency. When an update rises, it is applied at boot time.
It is rare when this is updated, though. There will be no harm, eventually!
There is also a version of this for Intel processors, it is called Intel microcode.
