### Post by natrixgli on 2008-04-25
As someone who sets up Ubuntu PC's for other users I can say that while many people like to call Compiz eyecandy, it makes a lot of concepts much easier for new users to understand.

For example, the idea of virtual desktops/workspaces/viewports is much easier for new users to understand when it's in 3D using the Cube. Even simple things like switching between windows (super+tab) is more appealing for users when it's done in 3D and makes them more compelled to learn how to do it.

Many of the users I support are older, and don't quite grasp multitasking. Imagine if you didn't know how to use ALT+TAB to switch between windows - now having it look cool adds a bit more incentive to use it.

Also the Scale plugin is GREAT for older, inexperienced users when configured to show windows from all workspaces! 

So in conclusion, eyecandy has helped me teach low-tech users to multitask and not lose track of their windows. Heck, it helped me sell Ubuntu to my superiors in the first place, and cut our number of Windows desktops down to a small minority of users!
