### Post by linux_tech on 2009-02-25
Operating system installs are not often bloated. However, the restore (system recovery) disks that come with a new pc or laptop are often loaded with all kinds of 30-60 day trial or subscription type software (bloatware).  Ubuntu does not have any bloatware.

Most software offers you 2 install paths- express (typical) or custom
If you choose express you get whatever applications, protocols, and utilities the manufacturer want to include. But if you choose custom you can often choose whether or not to include them. 

Ubuntu offers several installation choices from minimal to full so it really is up to you how much you install. If you want a minimal install, you can download one here-
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

### Post by issih on 2009-02-25
As you mentioned web browsers, do bear in mind that one of the features of windows is that it keeps large chunks of internet explorer loaded in memory at all times, as it is so intrinsically bound into the OS (somewhat illegally...at least in the eu) This isn't the case in ubuntu, so the first time you launch firefox it will take a little time, once it is run the first time it will probably fire up much faster as it will be cached. You could look into using preload, I think that is what it is called, which attempts to analyse your pc usage and make sure the things you use a lot are kept cached in memory if possible.

Secondly these little atom pcs are not very pokey at all, and I highly doubt that a 1.6 ghz atom would live with an equivalent speed of almost any other architecture, so it being a bit slow doesn't really surprise me.

Thirdly XP is actually pretty snappy as a fresh install without any thing installed on it. The problems come once you add AV software and install a few bits and bobs. Once the registry gets a bit bloated and you have a few extra things starting at boot (e.g. messenger, skype, av, whatever else) the time to boot xp gets dramatically longer (and I count booted as when the machine stops whirring away, and will let me launch something, not when the desktop appears).

Ubuntu probably is about on a par with xp as a fresh install, maybe a smidge slower, but it won't slow down the same way, and it is a far more advanced base install, in terms of security, visual effects, just about everything.

As has been mentioned, it may be unfair to bring vista into the equation in that you could never run it on that machine, but in a sense that is the point, ubuntu is more equivalent to vista than xp in terms of capability, and yet it will run on that box, and it will run well.

To be honest, ubuntu probably is a bit bloated in some senses, but that is because it is trying to be an OS that can do just about whatever is asked of it, on as wide a variety of hardware as possible, it certainly is possible to build a far more focussed stripped down distribution, and many exist, but ubuntu is a fine starting point, and wil probably be all you need.
