[http://en.wikipedia.org/wiki/Lesser_GPL](http://en.wikipedia.org/wiki/Lesser_GPL)I hate to say it, but I don't think you're right about this.  Any software released under the LGPL is open source.  It's the *other* software that it *links to* that doesn't have to be open source.  That's the difference.  With the GPL, you are limited to what you can dynamically link to because it has to be compatible with GPL.  LGPL is the same as GPL, except the other software you link to or that links to it doesn't have to be open source.  So if Symphony were LGPL, it would have to be open source, but it could link to other software that is not.  Obviously, that's not the case.

A similar example that may help to understand, even though it's under a different but similar license, is the wxWidgets license.  wxWidgets is open source, but anyone can link to it and use it to create a program that can be proprietary.
