### Post by billgoldberg on 2009-03-17
First thing I do is add the medibuntu repo. After that I install:

- p7zip-full *(install rar, zip, 7z, ... support for fileroller)*
- abiword *(light weight, .doc capable, word editor)(and remove all open office stuff)*
- win32codecs *(wma, ... support)*
- ubuntu-restricted-extras *(flash, java, mp3, ... support)*
- libdvdcss2 *(dvd support)*
- non-free-codecs *(divx, ... support)*
- totem-mozilla *(makes firefox play wma, divx, quicktime videos)*
- emesene *(MSN client)*
- xchat *(IRC client)*

Sure over time I install some other stuff, but right after install I feel that is what I needed to get up and running.
