### Post by HermanAB on 2007-07-20
As far as desktop systems are concerned, you are quite safe.  There was a Linux virus a few years ago but it doesn't work anymore.  It didn't even work then - Linus had to patch the Linux Kernel to make the virus work...

Server installations have different problems.  Issues are usually caused by badly written PHP code on web servers.  Other times people use exceedingly simple 4 character passwords and get hacked.  I repair one or two hacked servers per year.  There are viruses that try to break into system using SSH.  This only works if the system has a bad quality password.

So in short, for desktop systems - don't worry about it.  For servers, be careful.  Always use looooong passwords.  Mine are 16 characters semi-pronounceable.

'Hope that helps!

Herman
