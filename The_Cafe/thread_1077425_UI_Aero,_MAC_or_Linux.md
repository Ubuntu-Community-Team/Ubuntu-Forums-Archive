### Post by Giant Speck on 2009-02-23
Saying that one user interface is best is as foolish a generalization as saying that one operating system is the best.  No user interface exists today that meets the preferences and needs of every computer user on the face of the Earth.

Some people like user interfaces that are dependent on graphical representations.  Some like minimalistic user interfaces.  Some don't even use a GUI at all, as they prefer the benefits of a command line interface.

I like the default user interfaces of Vista, Windows 7, and Mac OS X for what they are.  I believe they are excellent user interfaces by themselves.  They are very solid, are fairly consistant, and are very stylish.  However, the customization of all three of these interfaces is very limited.

I do not like the default look of GNOME, nor do I like the default look of KDE.  However, I like the fact that I have the freedom to customize GNOME (and, to some extent, KDE) to my will, as I have made numerous themes for GNOME and have released them on GNOME-Look.org.
