### Post by ssam on 2009-07-28
in some ways more recent versions are faster. i am pretty sure jaunty will boot faster than any previous version, on most machines.

but you will benefit from turning off some features. make sure desktop effects are turned off (system->preferences->appearance). remove things like bluez and non-latin fonts. reduce the time firefox keeps history (it used to keep 30day, but now does 90). install flashblock to stop webpages loading lots of flash ads. use abiword instead of openoffice.

if the machine still only has 256MB ram, then i strongly recommend upgrading that. going from 256 to 512 will make a big difference.
