### Post by aysiu on 2009-06-10
I've been using Firefox since 2004 and probably will for the next couple of years at least.

I've tried just about every web browser anyone's ever suggested to me, and none behaves properly except Firefox.

This is all I want out of a browser: [list][*]Allow me to have a save-for-later tab[*]Switch tabs via Control-Tab and not Control-PageDown[*]Be cross-platform (Mac, Windows, Linux)[*]Find in page as you type[*]Search from the URL bar[/list] It's actually quite a simple list, but I haven't found another browser that does all that. Opera and Chrome do not allow me to have a save-for-later tab. Epiphany and Galeon use the Control-PageDown keyboard shortcut, and they are also not cross-platform (nor is Konqueror). Opera pops up a search box when you're searching for text. Safari isn't available for Linux, but it also is annoying because it doesn't search from the URL bar.

Believe me, I've tried them all, and Firefox is the only browser that suits my needs.
