### Post by Hur Dur on 2011-01-12
Puppy Linux would be so damn slow on 64mb RAM. But, if you can add another 64mb, it should run pretty well.

I recommend:

ConnochaetOS (Pretty much Arch Linux for i586)
Slackware 
Damn Small Linux (No longer updated)
Damn Small Linux-Not (Pretty sure it is still being updated)
Debian Net Install - I've heard of it running on 32mb RAM.
CRUX - I've heard of it running on 16mb RAM.
Tiny Core - Pretty much as minimal as you can get. Should run quickly on 64mb RAM. 
TTYLinux
Arch Linux + Openbox (I'm not sure if the Pentium MMX is i686, and you probably do not have enough memory for the installation. So, you should take out your hard drive, install Arch from another computer, and then put the hard drive back in.)
SliTaz - You need to install this one on your hard drive using a different computer. Tazpkg crashes a lot on 64mb.

Non Linux:
KolibriOS - Runs quickly on 16mb RAM

Check out:
[http://wiki.laptop.org/go/Minimal_Linux_distros](http://wiki.laptop.org/go/Minimal_Linux_distros)
