 What you downloaded is a CD image file (.iso) Burn it to a CD using your favorite CD burning program under windows, make sure that boot prioritites are set so that CD has higher priority than hard drive (it is done in BIOS setup - you know, that little thing that comes on before the oeprating system starts. It ususally says something like "press F2" for setup...)

THen put the CD in the drive and reboot. Answer the questions, and you are ll set.
