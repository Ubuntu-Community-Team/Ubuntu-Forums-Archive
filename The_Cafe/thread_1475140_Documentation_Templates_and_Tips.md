### Post by red_Marvin on 2010-05-06
Depends on what kind of program it is, and what kind of documentation you are talking about.
E.g. if it is a console program and you want to document how the user would interact with it, you should write a man page. That would need something like troff/groff if I'm not mistaken.
If it is more a report LaTeX might be appropirate.
If it's documentation of user interaction with a gui program; LaTeX or html depending on how you want to publish it.
