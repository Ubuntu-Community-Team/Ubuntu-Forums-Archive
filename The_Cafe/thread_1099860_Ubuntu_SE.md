You have two options, to go back to a standard Ubuntu gnome splash, run:

```
gconftool-2 --set /apps/gnome-session/options/splash_image -t str /usr/share/pixmaps/splash/ubuntu-edgy-smooth.png
```

There are other pixmaps in the /usr/share/pixmaps/splash directory to choose from.

To get rid of the gnome splash altogether, use:

```
gconftool-2 --set /apps/gnome-session/options/show_splash_screen --type bool false
```
