### Post by cl333r on 2009-05-22
We already have Chrome 2, sort of, it's called Chromium, version 2 and is fast as hell. 3 (major) problems:
1. Not stable yet.
2. No extensions, think of adblock.
3. Doesn't support the HTML5 <video> yet. When implemented not clear which codec be chosen.
Good news:
1. Firefox 3.5 daily build compared to Chromium is from 3.5 (sunspider) to 20 times slower (google's benchmark). To me, Firefox's TraceMonkey is at a dead end.
2. 99% acid 3 test compared to Firefox 3.5 daily build - 93%.
