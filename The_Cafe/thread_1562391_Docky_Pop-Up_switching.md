### Post by koleoptero on 2010-08-30
If you're using compiz (have the effects thing in appearance enabled) then install simple compizconfig editor by:

sudo apt-get install simple-ccsm

then open it from the system -> preferences menu, go to the edges tab, and set the bottom left or right edge to "Scale: initiate window picker for all windows"

then whenever you want to switch to a window just point your mouse to that corner and see the magic. (as long as it's not minimized, which you'll find you'll have no need to do ever again while using this).
