### Post by Spr0k3t on 2011-03-22
> **Philsoki said:**
> Wouldn't that help the server load as well?

It doesn't work like that actually.  A dormant account is not tapped for any server load.  It has to do with pageviews per second.  A thread with several hundred posts by a single user is going to have the same load on the server if the thread was done by several hundred users.  It's a cacheable resultset.  A better way to reduce the server load is to move threads to archives when they become dormant for a period of time... still searchable, but with less graphics and features of an active thread.
