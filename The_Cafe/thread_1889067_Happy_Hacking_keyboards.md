### Post by castrojo on 2011-11-30
I've owned three of these. 

It's a great vi keyboard but can be annoying if you need to use F keys a bunch since you need to add an Fn key to the mix. The size is great, it's a good kb for an htpc or home server where you don't want a huge keyboard around.

They're good quality, worth the money if you're into it.
