Hi all

Just wanted to share this in case anyone was wondering
I noticed while in the (file/network browser) on 12.04 you use be able to go to the top menu and add bookmark to your shared folders and your network folder (bookmark) would appear in your left pane file browser window.

On 13.04 it's different and subfolders are not the same as the main shared folder either
So on 13.04 You can right click on the main share and bookmark the shared folder. However you can't do this for subfolders inside the shared folder because those folder will not appear in the network location window

For subfolders you have to select the (configuration icon looking thing) to the right of your file browser screen. If you mouse over it a windows will appear that says (location options) Just click the icon and select (bookmark this location)
This create a bookmark to any subfolder within the shared folder that you want.

Most people likely won't need this but if you have a fairly deep folder structure and have many subfolders such as Documents/oldbackups/photos/2000christmasparty and so on.

Then bookmarking subfolders is pretty cool but different on 13.04 the previous versions just FYI

Hope this help
