### Post by jkxx on 2010-09-15
Foobar2000 through Wine with output set to 48KHz and foobar set up with the SSRC resampler.

Disclaimer: I use OSSv4 instead of ALSA.

Full story: I have an M-Audio Revolution 5.1 card that's connected through a relatively high-end headphone amp and a pair of Audio Technica A900 headphones. I like my music to sound good and that set up gets the job done when fed the proper audio data.

Unfortunately the default alsa+pulse uses low quality resampling which makes any music played sound abysmal. Removal of both alsa and pulse is necessary, as well as replacing the audio stack with OSSv4. Once OSS is installed though, the audio quality goes through the roof.

As a plus Wine sees all OSS devices which are conveniently mapped to the different output ports on the sound card, making it very easy to target any one output.

I really like Rhythmbox though I have not figured out how to configure the default output in it, as the only option seems to be "default", which happens to be the front output, which happens to be the output with the lowest audio quality in this sound card.

So in short OSS+Foobar2000+SSRC resample = awesome-sounding music :D
