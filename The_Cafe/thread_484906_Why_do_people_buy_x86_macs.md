### Post by phrostbyte on 2007-07-23
Mac OS X uses a "universal binary" executable (inherited from the Mach kernel), which can run code for multiple CPU platforms. Because of this a single executable can run natively on PPC and x86 (if it is compiled that way).

Windows XP 32-bit will only allow up to 3 GB of RAM to be used for user space programs. Even so by default the maximum is 2GB. Even when you have 4GB of RAM it is impossible to utilize it all without PAE (extends the address bus to 36-bits), which is not supported in XP. XP and Vista 64-bit have applications can address the max amount of memory supported by the OS, which is usually pretty high.
