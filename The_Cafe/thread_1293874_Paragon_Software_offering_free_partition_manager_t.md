I got the Paragon software the last time they had a free download.

For my personal usage I prefer booting System Rescue CD and using gparted, but it is nice to have alternatives. Unfortunately the last situation I had where I would have liked to try the Paragon stuff because gparted was telling me there were no partitions on the disk, the Paragon rescue disk would not even boot, ended up having to dig around for my Partition Magic floppies. 

This was on an old 486 system, tried using Clonezilla first, even tried doing the clone on a newer system, but apparently the newer system recongized the HDD differently, so ended up having to do everything on the 486, ddrescue to clone the drive, then Partition Magic to expand the partition.

All that just to get some check-in software for a fitness place to work, DOH!!!
