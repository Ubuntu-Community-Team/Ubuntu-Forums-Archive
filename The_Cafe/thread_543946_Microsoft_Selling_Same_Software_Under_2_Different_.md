### Post by Kingsley on 2007-09-05
> Expression Web is focused on the needs of professional Web designers seeking to build high-quality, standards-based Web sites for companies. It provides support for integrating [XML]("http://en.wikipedia.org/wiki/XML"), [CSS]("http://en.wikipedia.org/wiki/Cascading_Style_Sheets") 2.1, [ASP.NET]("http://en.wikipedia.org/wiki/ASP.NET") 2.0, [XHTML]("http://en.wikipedia.org/wiki/XHTML"), [XSLT]("http://en.wikipedia.org/wiki/XSLT") and [JavaScript]("http://en.wikipedia.org/wiki/JavaScript") into sites. It requires the [.NET Framework]("http://en.wikipedia.org/wiki/.NET_Framework") 2.0 to operate. Its sibling is [Microsoft SharePoint Designer]("http://en.wikipedia.org/wiki/Microsoft_SharePoint_Designer"). Expression Web uses its own standards-based rendering engine, different from the browser-based FrontPage, which uses Internet Explorer's [Trident]("http://en.wikipedia.org/wiki/Trident_%28layout_engine%29") engine. [[1]]("http://en.wikipedia.org/wiki/Expression_Web#_note-0") Microsoft claims that Expression Web's rendering engine currently provides the most accurate standards-compliant rendering on the market [[2]]("http://en.wikipedia.org/wiki/Expression_Web#_note-1"), especially CSS rendering.
