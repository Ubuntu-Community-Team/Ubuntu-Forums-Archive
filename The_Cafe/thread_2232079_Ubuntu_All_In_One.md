### Post by SurfaceUnits on 2014-06-29
Project name: **Ubuntu AIO** (All In One)

 Project product name: **Ubuntu AIO ISO**
 Plan is to bring Ubuntu and some of the [official derivatives]("http://ubuntuaio.wordpress.com/ubuntu-aio-flavors/") (Kubuntu, Ubuntu Gnome, Xubuntu and Lubuntu) on one iso file that can be burnt on one DVD or [USB flash drive]("http://ubuntuaio.wordpress.com/instructions/").  Every one of them can be used as Live system, with no need of  installation on hard drive, or can be eventually installed on computer  for full Ubuntu or any other system experience. More information on [About us page]("http://ubuntuaio.wordpress.com/about-us/").
