### Post by perspectoff on 2010-09-20
> **Mike'sHardLinux said:**
> Don't forget, you probably need some kind of database for the backend of the forum software. I am not sure if there are any free hosts that give you a database, too. I don't know if vBulletin supports SQLite, but I doubt it.

Drupal has forums. It is free. It can use either free MySQL or PostgreSQL databases as the backend. You can host it yourself on your own equipment using Apache2 (part of LAMP) or even NGINX. Of course, it runs on Ubuntu server (also free).

Everything is free. Even this advice. 

I realize these forums are run by vBulletin, but that doesn't mean you must run it. It appears to use MySQL and PHP, both which are part of the free LAMP stack.

I know of no free hosting anywhere. It costs money to run a host server, just for the Internet access. Even running your own server you have to pay for Internet access.
