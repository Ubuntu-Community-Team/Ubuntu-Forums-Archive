### Post by Poyntz on 2008-12-12
Two suggestions:
1. Back up and reinstall
- 1.1 Back up pidgin data to a folder (look for pidgin, pidgin-data, libpurple0 and libpurple-bin)
- 1.2 Do a complete removal of pidgin via synaptic
- 1.3 Reinstall Pidgin

2. Get the next release of pidgin. Try:
```
sudo apt-get update && sudo apt-get upgrade
```
