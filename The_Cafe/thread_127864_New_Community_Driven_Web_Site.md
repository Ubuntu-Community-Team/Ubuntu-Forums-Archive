### Post by galgoz on 2006-02-10
MyGNUWorld.com is a new web site for the OSS Community.  It will be a web site dedicated to introducing and helping people that are new to the GNU world.  We, (and my we I mean me) are looking for volunteers from the people that are familiar to experts in the FOSS world to help with the collaberation of the web site.  That is the broad purpose of the site.  The site will use Joomla as the base for the website so we can spend more time developing and writing content than coding the actual site.  If you would like to help (all are welcome) please respond here and register on the site.  You will be given admin rights and we will go from there.  If you have suggestions for direction, content, or even different software than Joomla please feel free to add them.  We will definately be promoting Ubuntu on the site as one of if not the main OS recommendation.  Hope to see you on the site.
