### Post by dpm on 2011-02-22
> **jwbrase said:**
> But as I understand, they're generally translating *away* from English, and the best results are generally obtained when one translates into one's native language.

Generally, though, in the Free Software world translation goes in the English -> Native language direction, as developers choose English as the original language for the application messages. This allows having one common denominator as the source language for all volunteer translators.

In any case, I'd recommend you get in touch with the German translation team: [https://launchpad.net/~ubuntu-l10n-de](https://launchpad.net/~ubuntu-l10n-de)
