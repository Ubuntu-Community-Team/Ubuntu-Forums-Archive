### Post by groudzero on 2012-07-04
I did not know where to go with this one on the forums, but I needed some input. *(This may be a very long winded, but here me out) * 

So my friends and I were talking about how cool it would be to build a _Linux based game console_  with a specialized OS and then realized it was really possible.  Build a  small form PC, install a bare bones OS and play. So we first came up  with a pros/cons list about consoles.

Pros:
1.  Drop and play games.  (just put the disc in and go)
2.  Easy to use interface.  (simple setup anyone can navigate and play)
3.  Portable and easy hook up

Cons:
1.  Can not upgrade hardware  (hard drives and controllers are about it)
2.  Price of games and extras  ($40 to $60 for a new title)
3.  Can only use approved peripherals  (try switching your SNES controller with a PS3)
4.  Program coding is unavailable to non-developers
5.  Console only as good as the OEM feels as it's profitable

..... and that's just what I wrote down....

As  many of you know any game console is just a computer with a very  limited function range.  Plus, many of the same consoles are not even as  good as a off the shelf computer from any big box store. So why hasn't  anyone build one yet?  There are two I know of.  EVO and PANDORA.  Both  seem to miss the mark or never released.  

So here is the main.   Really combine the best of console and pc.  Build a perfect pc that is  built and runs as easy as a console, but have the flexibility of a pc.   So here is a list of what I think it should have as features.

[LIST]
[*]'Drop and Play'
[*]Wireless WIFI for wire free gaming
[*]Multiple USB for your own favorite control device
[*]Compact design
[*]Easy to upgrade off the shelf hardware (memory, drives, CPU and etc...)
[*]Open Source coding and software for first time game designer to the most advanced developer
[*]Can run games on any PC and vice verse
[*]HDMI, DVI and VGA video outputs for hi-def
[*]Blu-ray optical drive
[*]PCIe upgradeable for video and audio
[/LIST]
The only thing that would make this possible is the 'Drop and Play' OS.  

I  have found Ubuntu to be a great platform for various mods for media  players and sound/video editing.  Just a change here or there would not  be to far of a stretch to make a 'D&P OS'.  Before you say anything ***I am not a coder.***   Nor do I know C++ or anything like that.  I know some games need to be  downloaded or need special drivers to play.  Why not run the game from  off disc out of the install file or a special 'game converter' program  that burns it to disc for the 'D&P OS' to easier to read?

I  have no idea if this would be impossible or easy as cake to do.  That is  why I am posting.  The games are out there.  The hardware is too.  But  all on all...

Can a operation system like Ubuntu be changed or modded to work in/as a game console?

Like what I said earlier, long winded, but just need the Ubuntu/Linux community input on this.

Thanks.
