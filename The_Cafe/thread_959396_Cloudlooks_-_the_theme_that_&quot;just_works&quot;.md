### Post by Mazza558 on 2008-10-26
I got fed up of Futurelooks so I decided to make a theme that is functional but still looks nice. I have now been using it for  nearly a month and haven't found anything which needs tweaking. 

[IMG]https://dl.getdropbox.com/u/29948/Cloudlooks/Untitled.jpg[/IMG]

It's designed so that you can focus on getting things done without anything getting in the way. It's called Cloudlooks and you can get it here:

[http://www.gnome-look.org/content/show.php?content=91945]("http://www.gnome-look.org/content/show.php?content=91945")

If you really like it, Digg it here: [http://digg.com/linux_unix/Ubuntu_Gnome_finally_has_a_theme_that_just_works]("http://digg.com/linux_unix/Ubuntu_Gnome_finally_has_a_theme_that_just_works")
