### Post by BrokenKingpin on 2011-12-13
Personally I find Gnome 3 a more appealing interface for a media center/TV or a tablet. I can't picture either interface on a mobile phone... that would require an entirely new DE IMO.

Out of all the DEs I would say that KDE would be the best for a media center front end, as you can configure that thing for almost anything. even the current netbook interface works pretty good for that. The only downside is that it is a bit heavy for a media center front end.
