I've noticed that all of people here seem to have netbooks. As we all know, netbooks tend to be less powerfull, and many these days seem to be heading towards cloud computing, where strain is taken off the local computer, which only boots a minimalist OS, and browser, which heads to a virtual desktop. 

Desktop Two is an online virtual desktop, something similar to what would be used in cloud computing, and made by the good guys over at Sun =]. I use it alot when moving between computers, expecially between school, home, internet cafes and elsewhere, lets me access certain apps, including OO.o., and an online virtual harddrive, so handy for accessing files on the go. 

Just thought I'd share the link :D Not sure if its still a beta though....
