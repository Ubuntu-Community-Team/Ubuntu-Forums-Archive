### Post by tbodine on 2007-03-03
I have seen so many different posts by different people about how they are starting to use GNU/Linux(often only referred to as Linux -- shame on you), asking how they can get their Windows applications running in GNU/Linux so they can be *free* of the monolith, and asking others if they should abandon Windows altogether.
Now, being a free forum, these posts are fine, and usually informative, which is good. But I think people tend to get the notion that GNU/Linux is, in and of itself, anti-Microsoft.
It's not.
GNU/Linux and the true hackers that designed it, along with most all of the free software that is used on it, is anti-proprietary software, which just so happens to be Microsoft's business.
Most true GNU/Linux users don't bash Windows every chance they get, they would be all bashed out in a minute if they did -- people who love GNU/Linux love freedom, not the fact that they are saving money by using this operating system.
In a post earlier I commented on one person asking others if he should give up Windows and switch to only Ubuntu, I voted that he should, but I only really care if he does, if he gives up the proprietary software also. If he doesn't do that, I really don't care if he uses Ubuntu at all or not.
Using GNU/Linux as your operating system alongside proprietary software is like a vegetarian eating meat. It just shouldn't happen.
At school I have to use Windows, or if I'm lucky Mac OS X, to do my school work, because I'm not allowed to boot from a LiveCD to do it. Sometimes I complain that I'm using a proprietary piece of junk, that I would much rather bring in my own computer to do the work, but since I can't do that, I settle with what I have. I do, however, agree with Richard M. Stallman when he said that if his job forced him to use proprietary software he would quit that job. I'm not afraid to touch a Windows machine, but I'm definitely not going to get paid to use/develop anything proprietary, and I'm not going to use any proprietary software on my own time.
Thank you for reading, hopefully now the Windows debates will die down some, and licensing issues might start up, which are much more GNU-like and fun.
