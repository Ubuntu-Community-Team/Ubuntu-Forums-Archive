### Post by machoo02 on 2007-09-13
I use Streamtuner from time to time.  It's nice because it makes it easy to find lots of different streams at once.  Since I listen to mostly college/independent radio it makes finding stations from across the country a breeze.

I think the program you are looking for is [Streamripper]("http://streamripper.sourceforge.net/").  It's in the repos as well.

### Post by machoo02 on 2007-09-13
It looks like you already have Streamripper installed (it's the program that is doing the actual recording in the terminal window).  Looks like elive was nice enough to set up both for you.  You can configure which ripping program you use with Streamtuner under the Edit->Preferences menu.
