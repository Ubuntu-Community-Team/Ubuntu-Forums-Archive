### Post by Grant A. on 2009-07-27
> **Mr. Psychopath said:**
> [http://invisionfree.com/](http://invisionfree.com/)

InvisionFree is still around, but it has been superseded in features and power by Zetaboards ([http://www.zetaboards.com/](http://www.zetaboards.com/))

If you want a free subdomain, you can get one from FreeDNS ([http://freedns.afraid.org](http://freedns.afraid.org)), enter it into the domain manager of your zetaboard's control panel, and set it as your default.

**TIP:** On your subdomain, set the CNAME to: "http://sx.zetaboards.com/" (x stands for the server number shown in your board's URL, like this one: s**6**.zetaboards.com/Your_Board_Name (not a real website!). In this board's case, the subdomain's CNAME would be set to: "http://s6.zetaboards.com/"

If you don't want a subdomain, you can always buy a domain name from Yahoo! ([http://www.yahoo.com](http://www.yahoo.com)), set its CNAME to the same as you would the subdomain's, enter it into the domain manager of your zetaboard, and set it as the default.

You can find cool themes for your zetaboard here: [http://zbthemezone.com/index/](http://zbthemezone.com/index/)



Good luck! :KS
