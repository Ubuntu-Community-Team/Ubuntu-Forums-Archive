### Post by Blindraven on 2007-08-10
Hey guys,
This might be pretty useless to some of you but for those who like to search the forum/threads a lot and don't like that annoying "You must wait 15 seconds" feature, then here is a really obvious (but you'd never think it) way :
Just go to create a new thread and put whatever you want to search for as the topic, it will search for those values (after you click out of the topic box and in to the message box) and return instant results every time with no delay.
Now you can search for what you want with no delays as much as you want ;)

Nothing drastic, but might be useful to a few of you :)
