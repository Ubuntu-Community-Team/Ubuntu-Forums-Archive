### Post by PartisanEntity on 2007-04-25
As far as I know (and noticed) Roaming Mode will move on to the next network if it can't connect tothe first one. I noticed it here at home when I try to connect to my home wifi network, if it can't connect, then it will connect to my neighbours network.

### Post by ssam on 2007-04-25
network manager does not work with all cards (eg the rt2500).

in these case you have to disable network roaming, and use system->administration_.network like you did in edgy.
