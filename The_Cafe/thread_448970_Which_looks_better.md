### Post by maniacmusician on 2007-05-19
I've been messing around with page sizes for my Switching to Linux book. Since it's going to be distributed mainly in PDF/Website form, I figured that sticking to the standard 8.5x11 size wasn't necessary.

So I came up with the following size; 9.54x8.5. 

It's rectangular-ish, but it's not as tall as 8.5x11. It allows for more information to fit horizontally while making the pages look a little shorter. 

So, which do you think looks better in PDF format? 

8.5x11: [http://www.cutlersoftware.com/UnderstandingUbuntu/docs/Switching_to_Linux1+2.pdf](http://www.cutlersoftware.com/UnderstandingUbuntu/docs/Switching_to_Linux1+2.pdf)

9.54x8.5: [http://maniacmusician.googlepages.com/Switching_to_Linux_new.pdf](http://maniacmusician.googlepages.com/Switching_to_Linux_new.pdf)
