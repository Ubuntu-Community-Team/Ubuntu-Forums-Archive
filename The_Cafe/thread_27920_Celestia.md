### Post by ioliver on 2005-04-18
Wow, I just used synaptic to install Celestia, and it's utterly fantastic!  If you have any interest in space, and a reasonable graphics card, then you'll love this application!

See - 
[http://www.shatters.net/celestia/](http://www.shatters.net/celestia/)

In particular, enjoy the screenshots.

The version in the repos is rather old.  I just downloaded the latest version and built it from source. Searching their forums, docs and FAQs tells you what else to add to satisfy dependancies. It was pretty easy, and I've got no real experience in this area.

I've now spent hours wandering the solar system, recreating eclipses, working out where to sit on the moon to see the earth rise, and seeing how the constelations look from different stars.

Does anyone know the best way to request that a more up to date version is included with Breezy?  This program is an *excellent* way of answering the "OK, what can this computer do?" question.
