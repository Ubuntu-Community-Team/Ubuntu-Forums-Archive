### Post by Metallion on 2011-07-01
I'm a developer for Axsh  co., LTD and I am proud to announce the release of Wakame-vdc 11.06. Wakame-vdc is a powerful open source software solution that implements IaaS. It's written 100% in ruby and licensed under LGPL v3.0.

In other words, Wakame-vdc is a virtual data center. Installing Wakame on your Ubuntu server will give you a full suite that stores virtual machine images, launches instances, handles their networking and provides security.

Main features of Wakame-vdc
 * Intuitive web interface, powered by Ruby on Rails
 * Amazon EC2 ami compatibility
 * Can run on one Ubuntu server or multiple Ubuntu/Solaris servers
 * Firewall handled by dynamic security groups
 * Attachable volumes and volume snapshots
 * Ssh authentication by private/public key pairs
 * Instances with Global External NAT Mapping Exposure
 * Administrator Command Line Interface
 * Block Storage to Object Storage Snapshot Capability (IIJ GIO FV/S and Amazon S3)
 * Multilingual support (Currently English and Japanese)
 * Easy to use installation CD coming late July

Currently the only way to try Wakame-vdc is by either installing our ruby gems or getting our source code from github and running vdc.sh. As mentioned, we will release an easy to use installation CD later this month.
