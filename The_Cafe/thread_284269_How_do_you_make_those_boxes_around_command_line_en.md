Actually, if you click on the GO ADVANCED button when posting, there are options for including files, images, quotes, code (that's what you're asking about), php, html, etc.

As chaosgeisterchen said, just enclose the text in the appropriate tags:
```
To create this box around stuff, type [tag]your text[/tag] and replace tag with **code**, **quote**, **html**, etc.
```
> To create this box around stuff, type [tag]your text[/tag] and replace tag with **code**, **quote**, **html**, etc.
[html]To create this box around stuff, type [tag]your text[/tag] and replace tag with **code**, **quote**, **html**, etc.[/html]

Hope this helps.
