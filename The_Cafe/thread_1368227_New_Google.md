### Post by ceramicm on 2009-12-30
This has been [out]("http://www.neowin.net/news/main/09/11/25/google-now-testing-new-search-interface") for a while now. Google is testing an updated design for their search pages. It will randomly appear in the wild, or you can test it yourself:

1. Enable cookies

2. Go to [www.google.com]("http://www.google.com")

3. Paste the following data into the address bar (overwriting the URL):
```
javascript:void(document.cookie="PREF=ID=20b6e4c2f44943bb:U=4bf292d46faad806:TM=1249677602:LM=1257919388:S=odm0Ys-53ZueXfZG;path=/; domain=.google.com");
```

4. Click Refresh

You should now see the updated Google design. To revert back, you can delete the cookie that is created.
