### Post by Atomic Dog on 2008-01-22
I can comment only about the pentuim dual as we do not use AMD chips in workstations or laptops, only in our servers do AMD's lurk...If it is the board/processor combo at Fry's for $80 then it's not a bad deal.  You can do the processor pin mod to bump up the speed significantly without any worries about frying the system (your processor goes up to 2.3ghs and FSB to 1066).  I have a machine in our accounting office with the pin mod and it never shuts off... 3 months and no problems + it benchmarks considerably better with the mod.

The pentuim dual does only have 1/2 meg per die cache, but for most applications it is fine.  I wouldn't use it for hard core gaming, but for a general desktop system or workstation it will be fine.
