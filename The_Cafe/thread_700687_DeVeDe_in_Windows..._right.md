An installer for DeVeDe Windows can be found at [http://www.majorsilence.com/devede]("http://www.majorsilence.com/devede").

I have tested it on Windows XP and 2003.  I have been told that it works on 2000 and Vista.

The current version for windows (3.6.1) fixes drag and drop and uses the Windows native file chooser. 

While it is possible to use the devede package from rastersoft on Windows, you have to hunt for and install many dependencies.  Anyway the packages on majorsilence work better on Windows.
