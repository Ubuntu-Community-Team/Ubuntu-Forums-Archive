### Post by futz on 2009-10-26
1975. Bought and read every issue of Byte magazine - wanted one bad but had no money. Couldn't actually touch or use one till 1977 - a TRS-80 Model 1 at school. In 1981 I bought my first machine - a TRS-80 Color Computer with a whopping 16K of RAM and cassette tape storage. Upgraded almost immediately to 32K. Within two years or so I was programming in assembler and C and had a 5-1/4" floppy drive on it. Then came a 30MB MFM hard drive. BASIC was long gone then - was running the OS-9 operating system (Unix) in a 128K RAM machine.

I had (still have) a TRS-80 Model 3 as well.

Didn't buy a PC till 1994 - a 486DX-33. Bought it cuz I wanted to run a MIDI sequencer and didn't want to write my own. Since then it's been an endless stream of better and faster boxes.
