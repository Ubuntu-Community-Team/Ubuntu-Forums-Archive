### Post by .Maleficus. on 2009-07-21
Just to be clear - web application != PHP.  There are many, MANY web applications written in other languages like Python or Ruby.

Anyways, using your shared hosting as backup is probably against your provider's terms and conditions - read through it and make sure that it isn't.  If it's not there are plugins like [Web File Manager]("http://drupal.org/project/webfm") that should do the trick.
