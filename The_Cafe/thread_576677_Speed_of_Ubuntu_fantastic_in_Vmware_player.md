### Post by funrider on 2007-10-15
virtualbox works well too.
2GB might be not enough since you do a lot of graphic. Ram is cheap now so you should get a good deal.

i am using ubuntu 64 bit with both virtualbox and vmserver (bot are free), never really compare the speed.

vbox with virtualtool can let you seamless access to both host and guest system, but usb support is a bit complicated. (either give up guest or host usb function)

i like vmserver more since i can do more "hardware" configuration than vmplayer. i can also download and use the vm image from vmware.com right away without doing the installation and configuration by myself.

hope this helps.
