Moonlight released for Linux

The developers behind the Moonlight project have now made available the first public release for download.

Moonlight is the Linux implementation of the Microsoft’s Silverlight technology which is officially available only on Windows and Macs.

Microsoft Silverlight has been designed for Rich Internet Applications and provide an alternative for Adobe Flash platform.

The developers said that the released edition of Moonlight supports the Silverlight 1.0 profile for Linux.

They are now working on transitioning APIs to support the upcoming Microsoft Silverlight 2.0 edition.
