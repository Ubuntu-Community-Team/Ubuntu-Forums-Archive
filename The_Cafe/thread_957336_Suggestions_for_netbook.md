### Post by Don S on 2008-10-24
I know you're asking for a netbook with Ubuntu preinstalled, but the Asus Eee is fairly easy to configure, to work with Ubuntu. There's a special kernel made for the Eee that supports all the hardware in it out-of-the-box.

I've got a 901'er myself with "Ubuntu Eee" - a netbook remix specifically designed for the Eees - and it is really easy to get to work. The only problem I have, is that switching on/off the wireless card, can only disable it and you have to go into the BIOS to enable it again (I spent 8 hours, before I found that out...).
