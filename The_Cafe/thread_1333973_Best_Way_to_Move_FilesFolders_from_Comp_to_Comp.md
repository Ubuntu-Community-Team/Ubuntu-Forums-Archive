### Post by the yawner on 2009-11-22
Are you both using Ubuntu? If so, just install an ssh server on either machine. Then on the other machine running click Places>Connect to Server, on the dialog, change service type to SSH, and enter the IP of the workstation running SSH, and enter a valid user credentials of said machine.

Now of course I'm assuming you have a running local network at home?
