### Post by tatermonkey on 2008-06-24
```
tate@tate-desktop:~$ sudo aptitude moo
[sudo] password for tate: 
There are no Easter Eggs in this program.
tate@tate-desktop:~$ sudo aptitude -v moo
There really are no Easter Eggs in this program.
tate@tate-desktop:~$ sudo aptitude -vv moo
Didn't I already tell you that there are no Easter Eggs in this program?
tate@tate-desktop:~$ sudo aptitude -vvv moo
Stop it!
tate@tate-desktop:~$ sudo aptitude -vvvv moo
Okay, okay, if I give you an Easter Egg, will you go away?
tate@tate-desktop:~$ sudo aptitude -vvvvv moo
All right, you win.

                               /----\
                       -------/      \
                      /               \
                     /                |
   -----------------/                  --------\
   ----------------------------------------------
tate@tate-desktop:~$ sudo aptitude -vvvvvv moo
What is it?  It's an elephant being eaten by a snake, of course.
tate@tate-desktop:~$
```
