[quote=ubuntu_demon]
[Here’s]("http://fridge.ubuntu.com/atom/feed") the feed for the fridge. And [here]("http://planet.ubuntu.com/rss20.xml") you can find the feed for the planet.

You can follow all official maillinglists as rss feeds [here]("http://dir.gmane.org/index.php?prefix=gmane.linux.ubuntu"). Some interesting ones for everyone : [announce]("http://dir.gmane.org/gmane.linux.ubuntu.announce") and [security-announce]("http://dir.gmane.org/gmane.linux.ubuntu.security.announce").

You can follow the changes to Ubuntu right [here]("http://media.ubuntu-nl.org/rss/"). This one is especially handy if you follow Edgy’s development.

[A nice feed]("http://absolutebeginner.wordpress.com/feed/") for absolute beginners.

[Here’s]("http://ubuntudemon.wordpress.com/feed/") the feed for my blog.

A nice program to read all these feeds is liferea. To install it (without the $ sign):
```

$sudo apt-get install liferea

```
[/quote]
