### Post by lovinglinux on 2009-11-18
I have played America's Army 2.x for a couple of years, when I was a Windows user. It was really nice and realistic. I had a lot of fun with it.

The new America's Army generation game, AA 3, was released this year surrounded by controversies. It uses Unreal 3 engine, it's Windows only and it sucks. The US Army, which produces the game, has no plans about making a Linux version. The last Linux version was 2.5 and was ported by a developer on his free time, so it wasn't really officially supported. 

I was able to play AA 2.8.x using Wine, with some limitations. The mouse movement doesn't work well, but the biggest problem is that AA uses PunkBuster, which does not work under Wine. So you get kicked of official servers.

I couldn't install the new version using Wine.
