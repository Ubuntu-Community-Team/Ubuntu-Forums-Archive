### Post by dalee on 2006-02-28
Hi,

QCad is a fair cad program. Not the best, but not the worst I've used either.

It will do architectal drawings, though it seems to be aimed at quick and dirty. Rather than full and elegant. The symbol libary does have some windows and doors and a few interior items. But you would have to create your own symbols to add to it, (as would any drafter with most cad programs). If you didn't install it when you installed QCad use synaptic to find qcad again. Then install it.

Otherwise there isn't much out there yet for Linux based cad programs.

dalee
