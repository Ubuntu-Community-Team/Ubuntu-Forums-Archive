### Post by RAV TUX on 2007-11-19
Also, for more information here is the Developers Blog:
[http://shlomil.blogspot.com/2007/11/ubuntu-hebrew-remix.html](http://shlomil.blogspot.com/2007/11/ubuntu-hebrew-remix.html)

> 
IMO the script I wrote is far from perfection. The initial goal was to easily create an Ubuntu "Remix", and this goal was achieved, but the script can be easily be modified to create the next [Kazit]("http://kazit.berlios.de/") (Hebrew), or [Kinneret]("http://kinneret.berlios.de/main/index-en.html").

The LiveCD defaults to a fully "Hebrewalized" UI but you can easily boot to a "Hebrew Enabled" environment by hitting F2 and choosing English in the initial boot menu.Well done Shlomi!...while others have talked about developing a Hebrew version of Ubuntu you have actually done it.

I too would like to see a return of [Kinneret]("http://kinneret.berlios.de/main/index-en.html"), with it's beautiful goal of developing a distro in 4 default languages Hebrew, Arabic, Russian, and English. Ubuntu Hebrew Remix would be an awesome base for a "New" Kinneret!
