### Post by malspa on 2013-07-07
I'd suggest a bit more detail about what you're installing. For the first one, the post says, "My first distribution is Linux Mint with the Cinnamon desktop."

The reader would want to know which release you used. Also, looks like you did not install Linux Mint Debian, but it would be important to note that if you did.

As for myself, I like to spend at least a few months, and preferably a couple of years, with each installed distro, if possible, to really get a feel for how things go with that distro over time. But I guess no Linux reviewers take that approach. Wouldn't be very popular, I guess, but someone who did that could certainly tell you a lot more about a given distro than someone who takes a short look at a distro.
