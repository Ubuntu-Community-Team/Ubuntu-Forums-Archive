### Post by aysiu on 2008-12-17
> **jrusso2 said:**
> Exactly as I had predicted here.  Also Linux returns are ten times that of Windows.  When will they learn what makes a usable desktop for average users?
Yes, when will they actually use a full-fledged distribution instead of the crippled ones that come preinstalled?

Even Dell's Ubuntu uses some strange other architecture that isn't compatible with the normal repositories, so you have to force the architecture to install certain packages.

There are several strikes against Linux netbooks, but the main two are:

1. People who are not already Linux users have no idea what they're getting into. If they try to install a .exe and fail, instead of learning how to properly install software, they'll just return the Linux netbook and get an XP one.

2. People who are Linux users get annoyed that Linpus isn't really Fedora and Xandros isn't even the normal Xandros (or isn't fully compatible with Debian). I've read that the SuSE that comes on the HP Mini-Note is just a vanilla SuSE that doesn't even recognize all the hardware it's preinstalled on. Of course, Linux users can always install and tweak their own preferred distribution, but then they get long boot times.

And the solutions are:

1. Actually have salespeople be informed about the Linux options so they can tell people intelligently what the pros and cons are compared to XP.

2. Install a real distribution that isn't crippled in any way. You can put a "simple" mode on top, but it should basically be the normal distribution with an easy way to Add/Remove programs and no passwordless *sudo* and no severely outdated programs (six months behind is normal; a year and a half behind is not).
