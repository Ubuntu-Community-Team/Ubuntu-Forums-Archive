### Post by MaxIBoy on 2009-02-07
In Firefox, hit "ctrl+y" (I think it's "ctrl-j" on Windows.) This brings up the "downloads" dialog. Right click the file you want, and select "open containing folder." If you didn't save it anywhere and just "opened" it right away, it will have been saved to somewhere in /tmp. /tmp is cleared on every reboot, and I think it also might be cleared if it gets too big, so your downloads might not be there anymore.


I have my browser set to ask me for the location to save every download instead of just picking a default location. Instructions for changing that preference have already been posted.
