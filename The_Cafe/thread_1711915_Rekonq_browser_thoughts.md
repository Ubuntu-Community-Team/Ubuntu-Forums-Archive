Lancelot   - ```

sudo apt-get install yakuake
``` , and lastly
oxygen-transparent - [http://kde-look.org/content/show.php/oxygen-transparent?content=127752](http://kde-look.org/content/show.php/oxygen-transparent?content=127752)

Some people don't like oxygen-transparent, but personally, I've found it to add a whole new layer of awesomeness, if you give oxygen, just the tiniest hint of transparency.
