Is there a FOSS version of iTunes?

There are many independent labels that give out free music:
- [http://www.enorgis.com/pmwiki/pmwiki.php](http://www.enorgis.com/pmwiki/pmwiki.php)
- [http://www.allegro-music.com/labels.asp](http://www.allegro-music.com/labels.asp)

Not sure how big is the demand for OGG or any other non-propriety music files, but there is demand for it:
- [http://www.gizmodo.com/archives/open-letter-to-apple-ogg-for-us-010020.php](http://www.gizmodo.com/archives/open-letter-to-apple-ogg-for-us-010020.php)

There are various independent labels out there so what I have in mind is a iTunes + Synaptic hybrid. Where you can add a repository/music server from various music labels to a source list. Ability to check whether there are new addition to the music server. 

While a music file is playing. A URL to the label official website should be displayed + the song title + artiste name. If a user is intereted in buying a song from that label he/she just needs to click the URL. Clicking on the URL opens a browser redirecting user to the label online transaction website/portal.

Just whacky ideas, maybe someone already implement this. Then I apologize for wasting everyones time :)

Inspired by this [snub the RIAA](http://www.kuro5hin.org/story/2003/10/22/02551/056) article.
