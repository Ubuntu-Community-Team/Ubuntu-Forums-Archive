First of all, I wasn't sure of the correct forum to post this in, so my apologies in advance if it should have been posted somewhere else.

I've written a custom wallpaper switcher for ubuntu that uses the image on the front page of [http://wvs.topleftpixel.com/](http://wvs.topleftpixel.com/), which changes daily. The program is a bash script, which utilizes zenity heavily in order to provide a more user-friendly interface. Included in the attached zip archive are the install and uninstall scripts, as well as a readme, the actual program, and a cron job. Everything can be installed easily with the install script, and can just as easily be removed with the uninstall script. If you wish, please download it and test it out. Any feedback is welcome.
