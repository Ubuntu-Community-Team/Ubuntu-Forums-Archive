### Post by helpme on 2006-06-06
Well, I don't think it's really an eithor or situation. I'd recommend setting up a dual boot. This way you can stay with XP and at the same time explore Ubuntu.

About having no net access, this certainly is far from ideal for Ubuntu. If you decide that you really like Ubuntu and use it more often, I'd recommend thinking about getting some wireless hardware that works with ubuntu.

Also, I haven't really followed it, but I think there are some people working on an add-on CD. Seeing that you don't have net access, you might want to look into getting this.

And don't forget that you can now install Ubuntu from the LiveCD, so you don't even have to install it if you just want to give it a spin.

Have fun!
