### Post by Therion on 2009-06-17
I think every disto falls into one of three categories:  

**The Spartan Install**, who's philosophy would be "Provide as basic a framework as possible, let the end-user decide what they want on their system and then install it."  See: Arch Linux.

**The Median Install**, who's philosophy would be "Provide an enriched experience from the get-go by pre-loading a basic spread of applications that the typical user is going to want/need so we can get them up and running as fast as possible."  See about 90% of Linux distros: Ubuntu, Fedora, SUSE.

**The Tricked Out Install**, who's philosophy would be to pre-load all the bells and whistles right from the start.  See: Mint, Ultimate Edition

Each approach will appear to a certain percentage of the population and for different reasons.
