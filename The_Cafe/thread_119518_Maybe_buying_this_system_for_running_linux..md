### Post by futz on 2006-01-19
[QUOTE=red_Marvin]What do you think about these parts?[/quote]
Decent.

> ABIT KN8-Ultra, nForce4 Ultra,Socket-939 ATX, S-ATA, GbLAN, Raid, Sound, PCI-Ex16
PC3200 DDR-DIMM 512MB CL2.5 Minne 184-P
That's a dual channel board, so be sure to get two sticks of RAM. If you're going for 512mb, get two 256mb sticks and plug one into each channel. Dual channel (or interleaving) is faster for the same reason RAID-0 is faster than a single hard drive.

> Athlon 64 3200+ 2.0GHz Socket 939, 512KB, BOXED with fan
If you can afford it, I'd definitely step this up a notch or two.
