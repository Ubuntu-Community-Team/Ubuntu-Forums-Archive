### Post by Corfy on 2009-12-07
I wish I could say Warty Warthog. I first installed Linux in February 2005 and had a hard time deciding which distro to try. I heard good things about the Ubuntu release, but being so new to Linux, I was nervous about trying a newly created distro. I didn't want to risk it disappearing on me.

So I went with a more established distro, Debian 3.1 Sarge (which was then still "unstable", but only about four months from going stable). It took two weeks of installing, re-installing, web surfing, Google searching, and Forum asking before I finally got it to work right on my system, and even then I half-stumbled across the solution on my own. Still, once it was running well on my system, I was hooked on Linux, but a tad leery of Debian.

After hearing more and more about Ubuntu, I decided to give it a try (I figured it couldn't be a worse install than Debian, and I figured I knew how to fix the install problem I had before). That would be 5.10 Breezy Badger. To my surprise (and delight) the install went flawlessly. I've been with Ubuntu ever since.
