### Post by bigboy_pdb on 2010-09-12
**I've written out a few explanations. They may look confusing, but if you read through them carefully and more than once they should make sense at some point. You can ask me to explain further.**

I'm going to explain the process using different equations because it's better to have other examples. Also, I'll split them into multiple posts.

**_Example 1_**

First let's solve "y + 3 = x" for y. Solving for y means we want to isolate y (or have y on it's own).

Since y + 3 is the left side, we'll call it LS and since x is the right side we'll call it RS. LS and RS are both numbers. The equals sign tells us that they're the same number. This means that if we take 3 away from LS and 3 away from RS, the new numbers are still the same because we've done the same thing to both old numbers. This means that we can still say they're equal. So let's write that out as a statement:
LS - 3 = RS - 3

Remember that LS just represents y + 3 and RS just represents x so let's write it using those values instead:
(y + 3) - 3 = (x) - 3

I put brackets to show that three is being taken away from the the whole number (this is sometimes important).

Let's simplify everything:
(y + 3) - 3 = (x) - 3
becomes:
y + 3 - 3 = x - 3     (when I remove the brackets)
which becomes
y + 0 = x - 3     (when I subtract 3 from 3)
which becomes
y = x - 3     (since anything plus zero is itself)

Since y is on its own, we've solved for y.

### Post by bigboy_pdb on 2010-09-12
**_Example 2_**

Next let's solve "4y = x" for y. Again, solving for y means we want to isolate y (or have y on it's own).

Since 4y is the left side, we'll call it LS and since x is the right side we'll call it RS. Again, LS and RS are both numbers and the equals sign tells us that they're the same number. This means that if we divide LS by 4 and divide RS by 4, the new numbers are still the same because we've done the same thing to both old numbers, meaning that we can still say they're equal. So let's write that out as a statement:
LS/4 = RS/4

Since LS just represents 4y and RS just represents x we'll write it using those values instead:
(4y)/4 = (x)/4

Again, I put brackets to show that the whole number is being divided by 4 (which is sometimes important).

Let's start simplifying:
(4y)/4 = (x)/4
becomes:
(4y)/4 = x/4     (when I remove the brackets from the right side)

<***IMPORTANT SIDE NOTE***>
At this point I want to mention a few important things.

When any number is being divided by another number you can rewrite it as two numbers being multiplied. For example, x/4 is the same thing as x * (1/4), or in words x times one quarter.

Also, when we're multiplying numbers by one another we can rearrange them. For example, 3*2*x is the same as 2*3*x or x*3*2 (and so on). That means that x * (1/4) is also the same as (1/4) * x.
</***IMPORTANT SIDE NOTE***>

Based on what I said, we'll look at the left side of the statement we left off with:
(4y)/4 = x/4

(4y)/4 can be written as (4y)*(1/4). Since 4y is just 4 times y, we can write 4y as 4*y. This means that (4y)*(1/4) can be written as 4*y*(1/4).

I mentioned when we're multiplying numbers that they can be rearranged. So 4*y*(1/4) can be written as 4*(1/4)*y.

Now we have:
4*(1/4)*y
Since 4 times a quarter (or 4 times one over 4) is 4 over 4, we get:
(4/4)*y
Since 4/4 is 1 we get:
(1)*y
Removing the brackets we get:
1*y
Anything times 1 is itself so 1*y is just y and we're left with:
y

Continuing with the example, since we were just simplifying the left side, we can rewrite theses steps within the equality statement as:
(4y)/4 = x/4
(4y)*(1/4) = x/4
4*y*(1/4) = x/4
4*(1/4)*y = x/4
(4/4)*y = x/4
(1)*y = x/4
1*y = x/4
y = x/4

Since y is on it's own we've solved it and y equals x over 4.

**_Example 3_**

I'm going to combine the concepts I've mentioned into one example. See if you can follow it.

```

5y + 1 = x
(5y + 1) - 1 = (x) - 1     Subtract 1 from both sides
5y + 1 - 1 = x - 1         Remove brackets
5y + 0 = x - 1             Simplify left side
5y = x - 1                 Simplify left side
(5y)/5 = (x - 1)/5         Divide both sides by 5
(5y)*(1/5) = (x - 1)/5     Rewrite denominator on left side as fraction
5*y*(1/5) = (x - 1)/5      Rewrite 5y to include multiplication sign
5*(1/5)*y = (x - 1)/5      Rearrange multiplied numbers on left side
(5/5)*y = (x - 1)/5        Simplify left side
(1)*y = (x - 1)/5          Simplify left side
1*y = (x - 1)/5            Simplify left side
y = (x - 1)/5              Simplify left side

```

### Post by bigboy_pdb on 2010-09-12
**_Example 4_**

I'll give one more example using different operations and that has more on the right side. Also, this is an example where it's important that you write the brackets around the terms on both sides before performing an operation on both sides. Otherwise, you'll get the wrong answer.

```

y/6 - 7 = x + 4
(y/6 - 7) + 7 = (x + 4) + 7   Add 7 to both sides
y/6 - 7 + 7 = x + 4 + 7       Remove brackets
y/6 - 7 + 7 = x + 11          Simplify right side
y/6 + 0 = x + 11              Simplify left side
y/6 = x + 11                  Simplify left side
(y)*(1/6) = x + 11            Rewrite denominator on left side as fraction
y*(1/6) = x + 11              Remove brackets on left side
6*(y*(1/6)) = 6*(x + 11)      Multiply both sides by 6
6*y*(1/6) = 6*(x + 11)        Remove brackets from left side ONLY!
6*(1/6)*y = 6*(x + 11)        Rearrange multiplied numbers on left side
(6/6)*y = 6*(x + 11)          Simplify left side
(1)*y = 6*(x + 11)            Simplify left side
1*y = 6*(x + 11)              Simplify left side
y = 6*(x + 11)                Simplify left side

```

You'll notice that for one explanation it says "Remove brackets from left side ONLY!". That's because "6*(x + 11)" is not the same as "6*x + 11". You can try substituting a few numbers for x to see this.

You can also simplify this example further if need be using distribution laws.

```

y = 6*(x + 11)
y = 6*x + 6*11
y = 6*x + 66

```
