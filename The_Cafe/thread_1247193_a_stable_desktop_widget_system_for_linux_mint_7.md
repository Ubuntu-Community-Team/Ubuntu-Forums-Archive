KDe 4.3 is much more stable. And you can run the kde widget/plasma system inside gnome (search google). 

However if you don't want to do that you have three choices.

1. Google gadgets - find the .deb on the getdeb website. Great choice of stable widgets and a sidebar if you want.

2. Screenlets - A good choice of integrated widgets such as music applet.

3. Gdesklets - Haven't tested it much but it looks similar to screenlets.
