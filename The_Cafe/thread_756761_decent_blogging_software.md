### Post by Levander on 2008-04-16
I'm interested in customizing my blogging software.  And, I'm more interested in ruby than PHP.  So, I'm looking at mephisto and typo.  typo is definitely under more active development, but mephisto already has a multi-site feature working (multiple blogs with one mephisto install).

I was told in IRC that the typo group implemented multi-site, but the lead developer yanked it all out because it was starting to make the code look sloppy.  If they can't implement multi-site well, this kind of concerns me as to the quality of code in typo that I'd be customizing.  So, I'm going to look into mephisto first.  I could be wrong here, this opinion is just based on antecdotal evidence.

There are other really good blogging tools than Wordpress that are written in PHP.  But, I don't remember any one of them.  Maybe check in #wordpress on irc.freenode.org.  Those guys are aware of the other ones.  Don't be shouting out how annoying wordpress is though.  They won't like that.
