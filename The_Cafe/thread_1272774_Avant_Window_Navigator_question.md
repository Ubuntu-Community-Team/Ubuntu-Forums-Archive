Some apps do this, some don't.  See, if you keep clicking the OO launcher, it keeps launching a new document.  Now, try that w/ FF.  All it does is minimize/maximize the current open window.  I find it teensy bet annoying, since sometimes I like to have two browser windows open.  It's got to be in the parameters of the apps or something.

I think the same is true for the terminal launcher, minimizes and maximized the current open window rather than launchine a new one.

I get around it by having key bindings for apps like that - super+w launches FF, super+t launches the terminal, etc..
