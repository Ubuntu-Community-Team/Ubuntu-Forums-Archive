### Post by igknighted on 2007-06-25
For those using Opera, check out the Opera Show function.  When viewing a web page in fullscreen mode (F11) it can split the page up into slides and become just like a powerpoint presentation (or OO.o Impress), except it is using html documents formatted with CSS.  Check out this page for more details: [http://www.opera.com/support/tutorials/operashow/](http://www.opera.com/support/tutorials/operashow/).
