### Post by Gorlist on 2010-01-30
Good day, ive been drumming over an idea about having a second user interface on one desktop.. The idea would be to use the second screen output (which of course would go to another monitor or TV), and allowing a another user to work on the same desktop but using a different mouse/keyboard, or specially graphics tablet.

Theirs either two options, have duel mouse cursors going and giving the other person full access - or more preferably the main user could define the program(s) that can be used, and just specify the USB input for that program, and drag over to the second monitor.

The reason behind my thinking is my computer is mainly for art and graphics, ive got a number programs installed that wouldn't run on my other systems. Sometimes it would be handy to let someone else matte paint at the same time etc, but still allowing me to get on with my normal work, or collaborate more fully.

Just an idea... perhaps it can already be done?
