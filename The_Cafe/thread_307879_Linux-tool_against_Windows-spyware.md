### Post by greggh on 2006-11-27
> Targeted for system managers and expert users, the Alternative Operating Software Scanner (AOSS) is patent pending technology...

> &#8220;We have created a novel solution to the spyware problem by building an alternative operating system based on Linux software. This means that we can boot the system with an entirely different operating system and ferret out spyware problems,&#8221; Clausen said.

The software - to eventually be incorporated into Spyware Doctor&#8212;works by putting the Windows Operating System to sleep; enabling it to remove the most persistent spyware (such as rootkits and keyloggers) that launch and hide themselves in the OS when Windows is booted.

&#8220;By putting Windows to sleep, we are turning off the life support for the embedded spyware and getting a clear shot at removing it with AOSS,&#8221; Clausen said.

The typical approach of using the Microsoft® Windows® &#8220;Safe Mode&#8221; to fix stubborn spyware problems will not work because these new complex threats hide in the kernel of the computer and report false signals when the operating system is engaged.

The AOSS software is distributed as an &#8220;ISO image&#8221; that must be burned onto a CD. The user then has to boot the system with that live CD. AOSS is specifically designed to detect and remove most components of a threat&#8212;those which specifically make it function. Then, Spyware Doctor cleans up what is remaining.
