### Post by gradinaruvasile on 2010-04-23
Just get Debian testing or, if u like living on the edge, sid. Most packages that are in Ubuntu are in the main repos (no PPAs required), only 1 extra repo is required for multimedia. But then everything is kept at the newest version, excepting the buggy packages.

I switched from Ubuntu to Debian testing and never had any problems, in fact its more stable - maening that everything is working as it should, unlike Ubuntu sometimes - even upgrading certain Gnome core packages (2.28 to 2.30) went without a problem. There are a few bells and whistles missing in the default install compared to Ubuntu (like splash screen etc), but i dont miss them.
