### Post by kagashe on 2007-11-24
I was curious how many packages are required on a Linux distribution for surfing through Firefox. To check I installed Ubuntu server and X-window-system (it is DapperCD) and started X-server through startx command. It opened but there was a protest "Unable to start x session, no session manager,no window manager and no terminal emulator found.

Then I thought let there be only terminal emulator so installed xterm.

When I again tried startx there was no protest this time and a terminal was open in front of me on top of x-server.

Then I installed Firefox and typed firefox in the terminal and WHAT?

Firefox obliged.

Offcourse, since there is no Window Manager the three symbols on right hand corner minimise, resize window and close are not there but I can close it through File/Close or File/Quit

I am posting here from such a Firefox session.
