### Post by fragos on 2009-07-19
I've been running my business using Ubuntu for years while sharing files with MS and Apple systems. The applications will be different than in the MS world but I'm able to do all that I need to and to  save in formats that that MS aps can read. Your only problem will be if you are required to run a proprietary MS application. There's no problem with Office documents as all the formats are supported by OpenOffice. Unlike MS, almost all applications can save directly in PDF format which I prefer to use for documents I deliver to customers. I edit graphics and video on Ubuntu for the web sites I build. I've found a Linux alternative for every application I use.
