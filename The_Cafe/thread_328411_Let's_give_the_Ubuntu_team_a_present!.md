### Post by robzon on 2006-12-30
Hello there,

this past year the Ubuntu team have done a great job with both Dapper Drake and Edgy Eft. They have been working really hard to deliver this high-quality software to our desktops.
Sooo here's the idea: let's show the Ubuntu team how much we appreciate and support their work! Let's all donate a small amount of money - let's say 5 USD. It's really very little compared to what they've done, yet I'm sure it will help to improve the software faster and make it easier to use for everyone. Remember that time == money, so by donating some money you in fact donate some development time that will be spent on pushing Free Software forward.
If you don't want to donate - that's fine, nobody requires you to do so. If you want to donate more - go ahead.
Anyways, here's the link where you can find info on how to donate:

[http://www.ubuntu.com/donations](http://www.ubuntu.com/donations)

If you donate drop a post here, so we can encourage more people to do so ;)
If at least few thousands of Ubuntu users donate 5 USD, we would get a total of at least 20000 USD, that's surely enough to get a lot of those long-awaited and demanded things like "restore from trash" working :)

There's another thing you can do: spread the word to local communities by translating and posting this message on unofficial country-specific Ubuntu forums.

To moderators: if it's possible, please set this sticky and link to this post on other forums, and unstick/unlink the post after 2-3 weeks.

I wish all Ubuntu developers a happy, very productive new year, and all Ubuntu users Free, free and bug-free software to suit all your computing needs ;)
