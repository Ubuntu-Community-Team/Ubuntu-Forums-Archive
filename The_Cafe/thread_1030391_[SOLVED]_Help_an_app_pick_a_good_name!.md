### Post by Vadi on 2009-01-04
An open-source (GPLv3) **screenshot program is looking for a new name**, so we thought we'd come here to ask for some good suggestions :)

Currently, the program can do a lot of nice things related to screenshots - take a screenshot of a selected area, a window, a compiz workspace, or fullscreen. You can also apply nice effects to the screenshot - for example, give it a soft border, add sepia, or make it seem like a polaroid pic. After you're done, you can upload it to a website (imageshack, ubuntu-pics.de, or your ftp server).

We're looking at using "Snap", but that name *might* be trademarked, so we're holding off on that for now. "screeny" and "screenie" are also pending, but we're hoping for something more serious ;)

So if anyone has any suggestions for a good name, please let us know. So nobody gets bashed for having a crappy name later on :KS

update: thanks to [damis648]("http://ubuntuforums.org/showpost.php?p=6492345&postcount=3") and everyone else for helping out. A revamped website & name will be out soon :)
