### Post by deadlydeathcone on 2007-02-01
> **kko1 said:**
> Hello!

I'd like to hear how people manage to keep their bookmarks in any kind of order - what do you do, how do you save them, how do you categorise and organise them?

I just use single directory levels. they're much easier to find and keep in order that way, and works well with bookmark system Epiphany's has going on.

> **kko1 said:**
> What happens for me in Firefox, specifically, is that I find it too clumsy to directly add a new bookmark in a right folder, when the browser only lists the five latest folders for alternatives and you would have to fetch the right folder through a tedious process involving lots of mouseclicks. So, I usually just let them end up at the end of the list, or if I have a number of tabs open that I want to bookmark I bookmark them together in their own folder. Still not in the right place, necessarily, just at the end of the list... :( So, in Firefox I currently have a longish list in the bookmarks menu that I need to scroll to get to the right parts. I haven't really taught myself to use the bookmarks sidebar either, at least not yet.

The del.icio.us extension helps a lot, as the default system isn't really all that great. When you bookmark a page you can just type in the correct tag or pick one that it suggests for you, which is a good improvement and also has the benefit of keeping bookmarks synced on different machines.

The best way I've found to actually find them again is with the extension [myurlbar]("https://addons.mozilla.org/firefox/1722/") (if you've used galeon or epiphany this will be really familiar). With this you can just type  a part of the bookmarked page title or url into the address field and it will immediately find it. Typing a tag also brings up all of bookmarks in in the same category.

The best that I know for finding and navigating between bookmarks that I know of I can't find a Firefox extension for (if any knows of one, please tell!), and as far as I know is for Epiphany only. In short it lets you add two buttons to the toolbar; one that lists your most visited sites, and another that list all of your bookmarks that share a tag with the site you're at. It's really handy, as if you have a mailing list or programming tutorial bookmarked it makes it really effortless to switch to other sites in the same category, and is really fast and intuitive.

### Post by beast2k on 2007-02-01
The best way I have found is (in Firefox) use the bookmarks toolbar folder and create folders in it so the folders show up on the toolbar each folder has a catagory. For example Hardware, Software, bittorrent sites, Linux sites, gaming, etc etc. Then go to [https://addons.mozilla.org/firefox/3880/](https://addons.mozilla.org/firefox/3880/) and install the "add bookmark here" extension so every one of the folders on the bookmarks toolbar will have a button within it that says "add bookmark here". This way you put all the bookmarks in the correct folder as you go. Otherwise I find my bookmarks will be a massive list that scrolls on for pages.

### Post by cowlip on 2007-03-13
I think this is an interesting extension for those temporary pages you might want to revisit in a day so, but not permanent bookmarks:  [https://addons.mozilla.org/firefox/2886/](https://addons.mozilla.org/firefox/2886/)
[img]https://addons.mozilla.org/images/previews/tark__temporary_bookmarks_-3.jpg[/img]
> 
Tark (Temporary Bookmarks)
by Rollin Crittendon


More previews&#8230;
One use of Tark to harvest bookmarks that are of interest now but may not be in a few weeks.

Tark uses icons to function as an egg-timer to show show much time is left on a "Tark'd" bookmark. The bookmark-icons that are "Tark'd" will change as weeks elapse (see previews).

After time elapses the bookmark will still be in your bookmarks for whatever you want to do.

If a Tark'd bookmark is UnTark'd any icon associated with that URL from before is restored in the bookmarks menu.

Install the toolbar button from the menu via "View->Toolbars->Customize...".
