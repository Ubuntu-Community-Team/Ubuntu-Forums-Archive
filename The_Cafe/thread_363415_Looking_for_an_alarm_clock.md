### Post by Paerez on 2007-02-16
If you learn about cronjobs, it is worth it. You have to edit a text file, but you can schedule tasks to be run at specific times. Then you just put in a command to run:

xmms /path/to/my/wakeup/song.mp3

and you get woken up by your song,
