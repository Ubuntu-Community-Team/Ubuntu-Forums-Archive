### Post by COMECON on 2013-03-15
> **georgelappies said:**
> Cool, how do you get the 13.04 theme into 12.04? I would really like to have the new nautilus skin etc. in 12.04 as well as the new Ubuntu button ;)

It's easy! For the Ubuntu Touch launcher icons, see [this](http://www.omgubuntu.co.uk/2013/03/ubuntu-touch-style-launcher-icons-for-the-ubuntu-desktop). Note that if you use 12.04, when you see "unity/6" you will have to go to "unity/5". 
For the BFB Raring icon, see [this](http://www.omgubuntu.co.uk/2012/11/how-to-install-the-13-04-unity-launcher-design-updates). Again, use the unity/5 folder if you use 12.04.
I was a bit worried about having both "tweaks" at the same time, because installing one will override the other. There's no problem! What I did was to copy the Ubuntu Touch-like files as it's said on OMG! Ubuntu!, and then copy only the file launcher_bfb.png that you get in the second link. That makes them work lovely!
And if you want the new Nautilus (Nautilus 3.6) (which I do NOT recommend until we're forced to use it in Raring), you can't as far as I know on 12.04. On 12.10 you can just add the GNOME team PPA as explained [here](http://www.webupd8.org/2012/10/how-to-install-nautilus-36-or-patched.html).
If you have any other doubts feel free to PM me :)
