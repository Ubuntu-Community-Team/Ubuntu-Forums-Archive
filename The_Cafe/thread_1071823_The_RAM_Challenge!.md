So for Xorg after you set up system after the install (especially networking) download the latest pkgsrc tree and just "cd /usr/pkgsrc/x11/modular-xorg-server" and then "make install clean". ### Post by kk0sse54 on 2009-02-19
> **linuxisevolution said:**
> I can't cd to /usr/pkgsrc/wm/fluxbox or anything. NetBSD says /usr/pkgsrc does not exist. Any help?

You need to download the pkgsrc tree, refer to this documentation for getting it set up [http://netbsd.org/docs/pkgsrc/getting.html#getting-first](http://netbsd.org/docs/pkgsrc/getting.html#getting-first). After you get the main branch of pkgsrc set up I'd also recommend if you have the space to download the wip branch as that will greatly increase the amount of software available.

Otherwise in order to save space (no idea how much you have but if you only have 16 MB RAM I bet it isn't too much or if you just prefer binary packages vs compiling) you can just install binary packages which are only available for the main branch and are fairly up to date.
You just need to make sure you have your PKG_PATH set first 
```

su
export PKG_PATH="ftp://ftp.netbsd.org/pub/pkgsrc/packages/NetBSD/i386/<insert version #>/All"
export PKG_PATH
pkg_add -v fluxbox
```
