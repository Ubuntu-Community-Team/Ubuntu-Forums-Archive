### Post by ubuntu27 on 2010-09-09
Hello friends and Ubunteros.

There is a bug in Network Manager Applet (nm-applet) that affects everyone. 

This bug is about nm-applet not showing-up for the second or third user when the first user does not log-out. It is reported as [bug#284596]("https://bugs.launchpad.net/ubuntu/+source/network-manager/+bug/284596") at [Launchpad.net]("https://launchpad.net/")

Many people including me believe  this is a bug of high importance specially in a multi-user environment.

This bug was discovered 3 years ago and it is still not fixed.

The bug report mentions: "This bug affects 68 people".


I ask you to please log-in to Launchpad and then **add yourself** to the _count of people who are affected by this bug_ so that we could show how important it is for this bug to be fixed.

Here is the URL:

[https://bugs.launchpad.net/ubuntu/+source/network-manager/+bug/284596](https://bugs.launchpad.net/ubuntu/+source/network-manager/+bug/284596)
