### Post by MoebusNet on 2010-09-01
Judging by the number of threads & Google searches on the Internet, I'm not the only person who needs to be able to view plain, ordinary text documents on their Android phone without going through the Cloud. Now I've read enormous amounts of fanboy comments like "Why buy an Android phone if you're not going to use the Cloud capabilities?" Reality check: most of the planet Earth's surface is not accessible by smartphone to the Internet or Cloud. Deal with it.

My solution to this dilemma has been to plug my Android phone into my Ubuntu PC using the provided USB cable. When I do this, a screen pops up on my phone that offers several options including "Disk Drive". I select "Disk Drive" which mounts my Android phone's internal and SD cards as drives visible in "Places" on the Ubuntu desktop. I use Open Office to generate memos to myself and save them in the .doc format to my SD card.

I select these files on the phone using the "Astro" Android file manager app downloaded (free) from Android Marketplace. I simply select the file on the SD card, open (with the included Quickoffice app) and read it. I can also use the included Quickoffice app to view the text file location directly and open it.

Note: the Quickoffice app that reads my memos only seems to read .doc formatted files, not .odt. This works on Android 2.2 OS on my HTC Incredible phone. I'm using Ubuntu 10.04 Lucid Lynx. YMMV
