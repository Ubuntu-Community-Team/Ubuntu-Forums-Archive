### Post by saulgoode on 2010-05-22
When you are adding the drop shadow to the main text, it appears that you are using the Fuzzy Select Tool (the "magic wand") to select the text and missing the interior parts of the letters "R" and "O". Instead, use "Alpha To Selection" on the text layer and then add the drop shadow (this will result in a more natural shadow inside those letters).
