### Post by Arktis on 2006-06-03
I am posting this from the live mode of the dapper install disc (using wireless internet!).  Everything works wonderfully.  I am very pleased with it.  I especially liked the video clip in the examples folder on the desktop which has Nelson Mandela talking about the meaning of ubuntu.  Though the music was kinda corny, it gave a nice impression.

A wonderful job was done with this, and it's awesome to have a fully functional desktop without doing ANYTHING.  The only thing it is missing is 3-d acceleration, which I KNOW is possible with a live cd because of the xgl demo live cd that came out a while back.

I have tested out a lot of live cd's and this one is really top notch.  Good job, guys.  Wow.  I can really carry my ubuntu cd around with confidence, ready to give a demo to whomever might be curious, with the ability to launch a nice easy install if need be.

If anyone else has thoughts about this, post away!  I'd be interested to see what YOU thought about the live mode in dapper, and how it compares to previous incarnations and various other live cd's.
