Some of you might recognize me, but for those that don't I'm the author and maintainer of the Breezy Starter Guide (available in your Breezy help system). I'm currently working on the Desktop Starter Guide for the Dapper Drake release and on behalf of the Documentation Team I would like to ask the forums community for any input they would like to give to the guide.
