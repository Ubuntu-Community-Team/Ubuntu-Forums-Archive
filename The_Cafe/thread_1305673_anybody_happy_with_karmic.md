### Post by gmjs on 2010-02-15
> **Crossbow said:**
> I did a clean install. So far so good except that Firefox is really slow. I was advised to upgrade to 3.6, but for some reason 3.5.7 doesn't search for upgrades.

It's something to do with the way Ubuntu installs it--forcing you to stick with the latest version available in its repos.

If you want to use the latest version, just grab a copy from Mozilla's website ([http://www.mozilla.com/en-US/]("http://www.mozilla.com/en-US/")), extract the file and run the 'firefox' script in the folder to start it.  You should find it just works.

If you want it to feel more like it's 'installed' (if that makes sense :D), I tend to copy the folder to /opt/ and then change the symlink (/usr/bin/firefox) to the new one.

e.g.

```
$ **cd /usr/bin**
$ **sudo ln -s /opt/firefox/firefox**
```

Hope that helps.
