### Post by Dragonbite on 2009-01-29
My laptop, desktop and server are running Ubuntu. I use Windows at work. I've had this grand idea of being able to work from home and all that but when the time comes to do anything like that I'd rather spend my time with my kids.

I did have Windows going for a while.  Unfortunately in running Windows I had to also update and run the anti-virus, anti-spyware and windows update which often caused me to not be able to do anything on the computer for about an hour (or more) while this stuff ran.  If I was on every day then it probably would have been different.

Meanwhile I had set up a dual-boot system with Edubuntu for the kids and to let me play with Linux (like learn LTSP).  I figured it would give the kids some games to play without me paying for all these games and then have them hanging around the house uselessly.

Then it was a combination of removing almost all of the computers so we could re-finish the wood floors plus the Windows machine in the basement giving BSOD and spinning down the hard drive at boot-up!  So after some looking around, I decided I was going to try and convert everything to Ubuntu and see if we can make a go with that.  

The biggest hurdle was my wife who only used the computer once-in-a-while so she was easily frustrated at trying to learn each time where to find her stuff and how to do it. Finally she conceded when she became Room Mom for our second child's class and found the rest of the world communicates with email a lot more than she thought!  Combined with the convenience of the laptop in the kitchen she could use and she has slowly learned how to do what she wants.

Kids are happy with Linux and as far as I know have not had any issue with using Windows in school.

As for myself, I am spending more time working with the computer rather than working ON the computer which was one of the goals. Now I use the computers for email, surf, IM, presentations (for the computer club presentations), and teaching myself web technologies such as PHP, Drupal, Ajax, and general networking issues.

Some advantages of using Linux instead of Windows is ...[LIST]
[*]No licensing fees have allowed me to explore the option of LTSP (thin clients) and file/web/router/firewall/filter servers
[*]Ease of running a server since I am new to networking in general. I have the chance to see better what is going on and why and still haven't had to get too technical to do it or use obfuscating applications that I "think" I know what's going on.
[*]Able to place a firewall/router/content filter right behind the modem so all internet use in the house is filtered.
[*]Fooling around with various applications and uses without having to plunk down money on licenses or applications that I may find are not worth it!
[/LIST]
