I am simply amazed at myself.  I ran Smart Boot Manager because i was going to reinstall windows 98 (ethernet card wasn't detected in XU Edgy) and then reinstall Xubuntu 6.06 or 6.10 to have a dual boot.  But since I had just recieved the Kubuntu 6.06 that i ordered from shipit for fun and to test it on my newer computer.  So I ran SBM, i was going to run the windows installer, but sbm said there was a disk error(i assume sbm can't boot windows install CDs) so i tried the other cd drive, which just happened to be the one i popped the Kubuntu CD into. Lo and behold, it ran(albeit very slowly)!I'm definitely going to install it since it is much more complete than Xubuntu and install KDE core so it runs faster.
