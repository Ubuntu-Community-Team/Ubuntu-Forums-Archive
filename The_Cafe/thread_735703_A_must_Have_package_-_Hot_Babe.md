### Post by glennric on 2008-03-25
Add the line 
```
deb http://packages.medibuntu.org/ gutsy free non-free
```
to the end of the file /etc/apt/sources.list
Then type
```
sudo apt-get update
sudo apt-get install hot-babe
```
You will get a warning about gpg verification, but it is safe to go ahead and install.
Then the program will be in Applications->Accessories
