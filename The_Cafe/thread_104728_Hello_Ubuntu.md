What I've done so far...and why I'm loving kubuntu....

I've installed apache, php, mysql, firefox 1.5, unrar and learned how to unrar with a passworded rar file which goes as follows since I didn't find the answer in this forum: unrar x -p*xxxxxx* your_rar_file.rar

I had a little frustration with my session not quitting apps.  I kept getting an error when trying to access KDE config menu's that required root priveleges.
