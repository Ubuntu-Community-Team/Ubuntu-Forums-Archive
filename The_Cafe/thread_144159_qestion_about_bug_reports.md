### Post by Xian on 2006-03-13
If you are confident the problem is located upstream then you filed the bug at the correct place. However, if it might be a distro problem then Malone would be where you need to go, and really if the Ubuntu devs are able to determine that it needs to go upstream as well they most likely will send the applicable patch along.

I guess that's just a long winded way of saying that I would normally file the bug at Malone (if it is a Ubuntu package), and let the devs determine if it needs to progress upstream.
