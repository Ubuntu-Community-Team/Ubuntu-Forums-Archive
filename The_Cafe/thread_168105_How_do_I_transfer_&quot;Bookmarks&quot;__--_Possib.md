### Post by lucia_engel on 2006-04-29
If you're using XP:

(You may need to show hidden folders first to see Application Data: Tools > Folder Options > "View" tab > check "show hidden files and folders")
Go to C:\Documents and Settings\YOURUSERNAME\Application Data\Mozilla\Firefox\Profiles

There should be a folder with random numbers and letters, click on it. A file called bookmarks.html should be there.

Now you can back this up somewhere (FAT partition, USB drive, online storage). You should be able to import these bookmarks using File > Import from Firefox

### Post by aysiu on 2006-04-29
Burning your bookmarks to CD is a bit extreme.

You can just open Firefox in Windows, go to Bookmarks > Manage Bookmarks > File > Export and then save bookmarks.html to wherever. Instead of burning it, you can even email it to yourself (it's a small file).

Then open up Firefox in Ubuntu, go to Bookmarks > Manage Bookmarks > File > Import and then select the bookmarks.html file you emailed to yourself.
