### Post by cyberhood on 2011-05-15
I'm using [GNU IceCat]("http://www.gnu.org/software/gnuzilla/") 4.0.1 (essentially Firefox [without]("https://secure.wikimedia.org/wikipedia/en/wiki/GNU_IceCat") the logos)

[AdBlock Plus]("https://addons.mozilla.org/en-US/firefox/addon/adblock-plus/"): ESSENTIAL, customizable malware ad blocking.
[Beef Taco]("https://addons.mozilla.org/en-US/firefox/addon/beef-taco-targeted-advertising/"): opt-out of cookies permanently.
[BetterPrivacy]("https://addons.mozilla.org/en-US/firefox/addon/betterprivacy/"): ESSENTIAL, gets rid of [LSO "super" cookies]("https://secure.wikimedia.org/wikipedia/en/wiki/Local_Shared_Object").
[FastestFox]("https://addons.mozilla.org/en-US/firefox/addon/fastestfox-browse-faster/"): helps me browse faster and more efficiently.
[FireClam]("https://addons.mozilla.org/en-US/firefox/addon/fireclam/"): [ClamAV]("https://secure.wikimedia.org/wikipedia/en/wiki/Clamav") scans downloads for viruses.
[FlashBlock]("https://addons.mozilla.org/en-US/firefox/addon/flashblock/"): lets me choose what flash animations I want to see.
[GoogleSharing]("https://addons.mozilla.org/en-US/firefox/addon/googlesharing/"): ESSENTIAL, helps us all become liberated from Google's spying. **PLEASE** add this one, the more the merrier!
[HTTPS-Everywhere]("https://www.eff.org/https-everywhere"): provides encryption to the communications with a major number of websites.
[NoScript]("https://addons.mozilla.org/en-US/firefox/addon/noscript/"): ESSENTIAL, allows JavaScript, Java and other executable content to run only from trusted domains of your choice.
*EDIT* And now, thanks to this thread I'm also using:
[Close'n forget]("https://addons.mozilla.org/en-us/firefox/addon/closen-forget/"): helps stamp out cookies
[Element Hiding Helper for Adblock Plus]("https://addons.mozilla.org/en-US/firefox/addon/elemhidehelper/"): fights text ads
[Add to Search Bar]("https://addons.mozilla.org/en-US/firefox/addon/add-to-search-bar/"): Make any pages' search functionality available in the Search Box.
[Browse By Name]("https://addons.mozilla.org/en-US/firefox/addon/browse-by-name/") re-enables the pre-Firefox 4.0 Browse By Name Feature, that I absolutely loved with the previous Firefox and have missed since upgrading! This is an awesome add-on!
[Flash Video Replacer]("https://addons.mozilla.org/en-US/firefox/addon/flashvideoreplacer/"): Really great work **lovinglinux**! Thanks a ton for this. Now I'm one step closer to freedom!

I also use various searchbars:
[Youtube (SSL)]("https://addons.mozilla.org/en-US/firefox/addon/youtube-ssl/"): utilizes [SSL]("https://secure.wikimedia.org/wikipedia/en/wiki/Secure_Sockets_Layer") to protect searches between You and YouTube.
[Scroogle SSL Search]("https://addons.mozilla.org/en-us/firefox/addon/scroogle-ssl-search/"): search Google anonymously.

Please consider making a donation to the developers of [FOSS]("https://secure.wikimedia.org/wikipedia/en/wiki/FOSS") add-ons and programs

All of these add-ons and [more]("http://www.gnu.org/software/gnuzilla/addons.html") are available for GNU's [IceCat]("http://www.gnu.org/software/gnuzilla/"), which is what I'm using as [Mozilla has trademarked their name & logo]("https://secure.wikimedia.org/wikipedia/en/wiki/Mozilla_Corporation_software_rebranded_by_the_Debian_project").
