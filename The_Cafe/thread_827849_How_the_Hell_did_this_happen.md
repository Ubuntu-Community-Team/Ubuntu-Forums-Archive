### Post by ellis rowell on 2008-06-13
Last night I switched on the computer. It failed to read the system disk, being tired after a day out I decided to try this morning. This morning, same thing. I thought about it and decided to try the BIOS settings. On entering the BIOS I checked the settings for Hard Disks. there were three settings for order of access (1. Add-in Cards 2. slave HDD & 3. system HDD), it should have been set for system HDD but had changed to Add-in Cards, I changed it back to system HDD and everything worked again. So how did the setting get changed?

### Post by bomanizer on 2008-06-13
Last time when I ran into problems like this I tought about a dead mobo battery. I haven't had a desktop PC for a while, so I don't know if those batteries are used anymore, but there was a time when batteries were used in order to supply a small amount of voltage for the mobo. This voltage would keep the bios settings. With no voltage, the bios would default to factory settings when powered on.

Hope this solves your case :)
