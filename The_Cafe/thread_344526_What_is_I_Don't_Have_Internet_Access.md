### Post by gwm on 2007-01-23
Hi,
I am trying to get up and running with Ubuntu. The way I understand Ubuntu, is that it is designed to be helpful to the disadvantaged communities in the third world. Down here in Southern Africa, we live in a third world country. Internet access is expensive!!!

In the spirit of Ubuntu, a CD or DVD with things like Alien, SSH, GIMP's help file etc on them would be much appreciated. For example, at our access speeds, GIMP help is a two hour download and we pay for our telephone calls by the second!! We don't have things like cable modems. Broadband is available, but costs between 10% and 20% of an office worker's wages each month. At this rate, one might as well buy Windows because it is cheaper than an Ubuntu CD plus all the phone bills.

Ubuntu's policy on open source means that the release is hamstrung by missing essentials that are way beyond the average non techie beginner. The result is a desktop version that doesn't have peer networking, so that printer can't be shared. To have a basic working environment thus requires various gymnastics like hours in the forums, trying to decode techie speak to discover what needs to be installed. This defeats the spirit of Ubuntu.

The average user installs Ubuntu, hasn't got internet, can't use what he has, doesn't know why, concludes that this new fangled thing is useless and cans the whole exercise. I've watched this happen with some pretty gifted youngsters I know. I realize that these extras would push the content beyond a CD's capacity. That is why I appeal for a second CD containing some of the other items and an install procedure that has a non internet option which installs from the second CD.
