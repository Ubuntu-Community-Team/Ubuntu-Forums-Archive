### Post by mcduck on 2011-05-12
I switch quite comfortably between Unity, normal Gnome, XFCE, LXDE and my Openbox setup. I suppose it depends on a person how easily you adjust to such changes.

I also use different programs with different tools and shortcuts for same purposes all the time, regularly switching between Photoshop/Gimp, Illustrator/Inkscape, Blender/3DS MAX (and of course switching between different operating systems). :D

Anyway, the only suggestion I have is to not to try changing the different environments to look the same, things are actually easier when the difference between the interfaces is clearly visible. If you make a KDE desktop that looks like Gnome it's much more likely you'd try using Gnome shortcuts as well. The more different the desktops look from each other the easier you should find adjusting yourself to each environment.
