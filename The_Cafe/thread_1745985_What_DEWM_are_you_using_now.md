### Post by Spice Weasel on 2011-05-01
I am using FVWM, you know, that desktop that I shamelessly advertise on the forums.

Reasons why... Hm. It's very stable, has been along for a very long time (I think it's the oldest WM that is still developed besides TWM) and is unlikely to change for a very long time. It has an amazing number of features for such a small program, customization ability that is unmatched to this day, powerful virtual desktop support, mouse gestures support and can imitate any desktop you can think of (besides desktops that utilize 3D).

Mine is set up to look like BeOS/Haiku but behave slightly differently (and my window borders are completely different)
