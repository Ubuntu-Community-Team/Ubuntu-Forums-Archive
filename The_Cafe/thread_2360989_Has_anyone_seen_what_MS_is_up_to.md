Users find and install distros , safely, quickly, and reliably via the Windows Store app.

Once installed, however, distro&#8217;s should be treated as command-line tools that run outside the UWP sandbox & secure runtime infrastructure. They run with the capabilities granted to the local user &#8211; in the same way as Cmd and PowerShell do.

This is why Linux distro&#8217;s don&#8217;t run on Windows 10 S: Even though they&#8217;re delivered via the Windows Store, and installed as standard UWP APPX&#8217;s, they run as non-UWP command-line tools and this can access more of a system than a UWP can.

So what should I use?

If you want to run all your dev tools, distros, shells, etc. on a machine running Windows 10 S &#8211; like the sweeet new Surface Laptop &#8211; then upgrade it to full Windows 10.  You&#8217;ll then be able to run Linux distro&#8217;s, Cmd/PowerShell, install dev tools, debuggers, profilers, packet sniffers, etc.
