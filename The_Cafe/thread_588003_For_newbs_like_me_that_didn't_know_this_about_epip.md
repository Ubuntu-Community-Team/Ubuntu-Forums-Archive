### Post by episodic on 2007-10-23
Epiphany is gnome's integrated browser. I always loved it for it's simplicity and speed. However, I was alway loathe to use it due to no adblock. I never knew there was a package called epiphany extensions which ads adblock (among other things). I just figured it out the other day. The extension uses a filterset g block that is updated from the internet upon your click. 

Now epiphany is my default browser. BTW if you have java problems - mine seem to disappear in epiphany

### Post by Darkhack on 2007-10-23
> **n3tfury said:**
> i hate it because it looks terrible.  opera is much faster at rendering pages on my box and looks a ton better.

Epiphany uses GTK+ and integrates with GNOME so it has the native look of whatever theme you are using.  Try changing the theme if you don't like it.  Opera is faster for now, but Epiphany is slowing gaining support for WebKit which is even faster than Opera.  The upcoming months are going to be a really exciting time for Epiphany as development continues further with the GTK+ port of WebKit.
