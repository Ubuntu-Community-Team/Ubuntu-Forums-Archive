### Post by scottuss on 2009-03-30
There are loads of discussions about which is better but this is the list of why for me it is better than Ubuntu (again, this is personal and others may disagree, some things may not affect other users in the same way, we are all different!)

My machine is a quad core 2.4ghz with 2 gig ram:



1) Boot time: Arch boots soooo much quicker than Ubuntu
2) Applications start quicker
3) I can customise the system in more ways, I can pick which modules are
   loaded etc
4) RAM usage is much lower
5) Rolling release, updates to applications (new versions) are available    unlike Ubuntu which has "feature freeze"
6) Arch runs MUCH faster on my EEE 701 than Ubuntu did


Don't get me wrong, I still like Ubuntu, I just don't use it as my main OS any more. It's a natural progression than many go through, start off in Linux on Ubuntu and then move on to other things.

### Post by Netsu on 2009-03-30
Arch gives you a lot of control over what do you want your OS to be and how it's functioning. By default you configure everything by text files but yet the developers try to keep it simple. The installation itself is a great way of learning how your Linux is working.
In short, it gives you WAY more control than Ubuntu but at the same time it's quite easy to learn how to use it properly, thanks to the minimalistic approach.

### Post by Simian Man on 2009-03-30
I wouldn't say Arch really gives you more control so much as it *forces* you to take control.  You can build a minimal install and edit config files in most distributions, but you have to with Arch.

I myself use Fedora.  It is similar to a rolling-release such as Arch in that they push new versions of most programs in older releases, so you don't have to wait for new releases for the latest and greatest.  However, like Ubuntu, it provides a nice desktop experience out of the box - slightly better in my opinion.

I never use Ubuntu myself, but suggest it for those new to Linux because it is at a nice point on the software curve where it's mostly stable but has mostly up-to-date stuff.  Plus the online resources are great.
