### Post by steveneddy on 2009-07-14
If you are new to Linux in general and Ubuntu specifically, I recommend using versions 8.04 (long term support) or the 8.10 version (better driver management for wireless and video cards). Using a more stable version will help you get things running and also assist you in keeping your sanity.

Don't use KDE (Kubuntu) at first. Use Ubuntu which used Gnome as the DE (desktop environment)

Get rid of the ATI card and buy a good Nvidia video card

You also may consider buying a system already built with Linux supported hardware.

Hardware that runs well on Linux:

Intel Core 2 Duo, or quad core processors

Nvidia graphics cards

I would Google for Linux compatible desktop or Linux compatible hardware and read the list.

Getting the right hardware and a stable version of Ubuntu should give you a better experience.

Try version 8.10 first to see how your ATI card reacts.

I believe ATI drivers are coming along, but I run Nvidia so I can't tell you that with all confidence.

If you find something that works, stick with it.

I am using 8.04 Hardy on my laptop because it is the LTS version and I need/want stability/reliability and performance, which i get with this little rig.

Once all the pieces are in place your experience will be better, I promise.
