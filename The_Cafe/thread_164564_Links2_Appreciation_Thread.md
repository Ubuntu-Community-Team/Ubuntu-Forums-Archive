### Post by 3rdalbum on 2006-04-23
Man, Firefox and Epiphany have been getting me down. Thanks to all the new CSS features that every web designer has started using, those two browsers have been crashing more frequently than any version of Windows. Plus, they eat system memory for breakfast.

Yesterday I discovered Links2, and although it doesn't support SSL and can't seem to post to the Ubuntu Forums, it rocks in many other ways:

[LIST]
[*]It's fast
[*]It's a small download
[*]It has a small memory footprint
[*]It anti-aliases all text
[*]It has the ability to send false referrer and user-agent headers
[*]It doesn't (/rarely) crash
[*]It has a kind of popup blocker
[*]It supports some Javascript
[*]It can be run in text-only mode too! (but I prefer w3m for that)
[/LIST]

Post your appreciation for Links2 (or your other favourite lightweight browser) here!
