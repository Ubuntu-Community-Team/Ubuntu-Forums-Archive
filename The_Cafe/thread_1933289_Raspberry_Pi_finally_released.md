### Post by ve4cib on 2012-02-29
> **forrestcupp said:**
> But you still have to buy a monitor, adapter to hook it to the monitor, keyboard, mouse, speakers, SD card, and anything else you need to make it usable. If you're going to buy all of that, you might as well go ahead and get something that is more powerful than an early model Android phone.

You're really limiting your definition of "usable."  Computers don't need monitors, speakers, keyboards, and mice to be usable.  Anyone who's worked with embedded systems can tell you that.

If you're planning on using it as a controller board (like for a robot or as a remote webcam unit) you don't need any of those besides the SD card.  Set up a DHCP server on the 'Pi, plug in an ethernet cable, and use VNC or SSH to write log in, run programs, etc...  Once it's set up you can unplug everything and let it do its processing.  Hook up the necessary USB devices and a power supply (which can be as little as 6 AA batteries) and you're set.

These are really designed to be educational/hobbyist devices.  Not general-purpose computers that will replace your laptop.  If you can't think of a fun way to use one then don't buy one.  Nobody is forcing you to.
