### Post by Yfrwlf on 2008-07-03
[http://marketshare.hitslink.com/report.aspx?qprid=9&qpdt=1&qpct=4&qptimeframe=M&qpsp=100&qpnp=15](http://marketshare.hitslink.com/report.aspx?qprid=9&qpdt=1&qpct=4&qptimeframe=M&qpsp=100&qpnp=15)

[http://www.xitimonitor.com/](http://www.xitimonitor.com/)

Of course, all these are very inaccurate, by only monitoring certain websites, and of course website visits among other factors don't give you an accurate number of how often, or how much use, Linux actually sees on desktops/laptops/etc.

For example, W3Counter shows Mac steadily dropping recently, while Hitslink shows it steadily rising.  Some sites put Linux around 0.5%, others around 1%, and still others around 2%.  The fact is, Linux does get a lot more use than these show, but how much more, who knows?  Regardless, I think there is about 50-100 million Linux desktop users worldwide, given the high use in countries other than the U.S., and all the switching to Linux that has been going on.
