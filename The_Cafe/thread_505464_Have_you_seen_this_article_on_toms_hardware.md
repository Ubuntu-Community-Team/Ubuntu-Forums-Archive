### Post by ddrichardson on 2007-07-20
Fair handed article. One thing always intrigues me though when anyone is discussing migrating to Ubuntu:

> The default GNome environment is more than adequate by any standards I've been able to come up with.  It is comparable in function to Windows and any new users should not have any issues migrating to it.

Comparable in function? This issue seems to come up time and again that there is a learning curve with the GUI.

As anyone who has moved on with Windows since its older incarnations must have noticed the Windows GUI itself changes radically between incarnations. I've never seen why people don't remark on how difficult it can be to find things that you were used to in the last Windows version - the Control Panel layout for example between 2000 and XP.

Individuality in the window manager is IMHO one of Linux's greatest strengths.
