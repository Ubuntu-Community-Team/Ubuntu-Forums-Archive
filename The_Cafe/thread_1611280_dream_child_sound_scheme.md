### Post by brandenmikal on 2010-11-01
I decided to create my own sounds and turn it into a sound scheme for ubuntu.

It's a cute child sound theme I made to use with the "daydream" wallpapers from microsoft. 

give it a go and let me know what you think!

note: MUST BE LOGGED INTO ROOT ACCOUNT in order to add schemes.

directions:

1. download zip file and extract to usr/share/sounds

2. system>preferences>sound

3. select the "dream child" theme from the drop box


enjoy!


sounds were created by me and are free to be remixed; copied; shared; and so on.
