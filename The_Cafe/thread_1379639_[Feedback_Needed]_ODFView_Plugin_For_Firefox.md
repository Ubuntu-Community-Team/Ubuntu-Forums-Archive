One of my friends who is currently taking up M.S. Computer Science in the University of the Philippines is developing a project.

As a requirement for her degree, she's asking for feedback/review of the project.

It would be great if any of you could go and try out the project and maybe report some bugs you find, in the spirit of FLOSS development (I'm told that the code will be released as GPL).

The feedback form and more information about the product can be found on this website: [http://odfview.webs.com/](http://odfview.webs.com/)
