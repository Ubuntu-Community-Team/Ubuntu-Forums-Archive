Why does no one know about search keywords? In the menu where you select the search engine for the search bar, there is "Manage Search Engines". You can add keywords for each search site. Like 'g' for google, 'w' for wikipedia, 'y' for yahoo, and so on. Then you can type in the _address bar_:
```
g lolcats and rolfdogs
```
and it will seach with Google.
