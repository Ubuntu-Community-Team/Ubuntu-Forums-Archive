### Post by thraxy on 2009-01-12
I just had a rather amusing thought while reading through a few of the usual "Windows vs Linux", "is windows better", "why switch" type of threads.

It made me think of how things would have been, had the situation been a bit different.

For the sake of argument, lets say:

- Linux was developed 10 years earlier
- All computers come with some sort of Linux distro pre-installed
- All hardware drivers are open-source, written specifically for Linux
- All software and game developers create with Linux in mind
- All viruses are downloadable in .dep, .rpm, source file and whatever various flavours there are

Now in 1991 a company called Microsoft comes along with their brand new take on what an operating system should be.

They include:
- Expensive licensing
- Closed-source
- User by default uses admin account
- Support forum that is supportive to the limit of the weight of your wallet

Now the rest is up to you. What would you do? Would you switch to Microsoft? What would be the benefits of this move? Would windows be an overnight success story?

This is all just speculative. We can thank Microsoft for leading the way in many cases such as GUI. The world wouldn't be the same without them. The point of this exercise is to simplify the way we compare the two, Linux and Windows.

Also, if the personal computer trend had been sparked by an open-source initiative, would there ever have been room for a close-source alternative?

Next time you're faced with the question, which is the most superior, try speculating how one would do a mile in the other ones shoes.
