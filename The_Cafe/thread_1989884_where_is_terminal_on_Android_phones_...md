### Post by 3rdalbum on 2012-05-29
> **asifnaz said:**
> As you know Android is Linux

Android is a Linux kernel, plus some basic GNU utilities, plus a completely new userspace layer on top. It doesn't use GTK, Xorg, Gnome etc so don't expect it to be able to run "Linux applications" or even Linux command-line tools. All programs running on Android need to be written for Android because it's basically completely different to your regular Linux distros.
