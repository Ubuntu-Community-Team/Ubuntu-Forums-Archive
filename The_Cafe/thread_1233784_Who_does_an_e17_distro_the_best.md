### Post by thisllub on 2009-08-09
I have been a sometime Opera user for a couple of years.
It currently has a bug that makes it unusable.

Go to a site that it doesn't like (at least 2 of my regulars)  and it will go into a frenzy of eating memory until it kills your computer.

I built Ecomorph but it still has some serious problems not working on multiple desktops and losing windows.

The key to having a good E17 setup is to customize the keystrokes.
It is possible to use E17 without a mouse at all if you have the Vimperator Firefox plugin.
I can live with E17s few idiosyncrasies. It leaves KDE and GNOME for dead in utility, customisation and speed.
