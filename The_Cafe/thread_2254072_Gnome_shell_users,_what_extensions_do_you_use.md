### Post by buzzingrobot on 2014-11-29
Dash to Dock, Wheel Scroller to Change Workspace, Disable Oversized OnScreen Graphic During Workspace Change, Add Quit Button to App Icons in Dash/Dock, Kill Top Left Hot Corner, Dynamic Panel (makes it transparent), Small Icon in Panel, Better Volume Indicator, TopIcons.

These essentially convert the stock Gnome Shell interface into a conventional Dock with static top panel interface.  I find that much more useful.  

(I wouldn't use Gnome without at least Dash to Dock. I only use Gnome on Fedora installs because Ubuntu Gnome is always a release behind the curve and adding the PPA's has been a problem. This is not really the fault of the Ubuntu Gnome team, which I'm sure could use more people and resources, and which has to cope with the fact that the Ubuntu release cycle and the Gnome release cycle are out of synch for their purposes.)
