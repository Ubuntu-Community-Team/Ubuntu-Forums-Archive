### Post by mT3 on 2008-12-01
I recently installed Ubuntu 8.10 and was very surprised how fast it runs on my desktop compare to my windows xp. And my XP is tuned and tweaked as fast as possible but still get killed by Linux.

My PC specs(kind of out dated, and fast with Ubuntu!)
Pentium D 3.4Ghz
3GB DDR-677 ram
160GB Hard Drive
Nvidia 7900GS 256mb

First thing I've noticed after using Ubuntu for a long time is... I don't have to restart my computer because it's getting slower! On the XP I had to restart after using it for a while, otherwise the hard drive will get too excited for no reason.

Another thing I noticed is moving files around on Ubuntu is so much faster then XP, and you can still do other stuff while doing that. Impossible on XP... One thing at a time

Also I was worried Ubuntu might not play some of my video and music files, but so far I haven't found any files that can't be played.

For what I do with my computer, basically internet, music, video, papers..I don't see the reason using XP again! Even though I still have it just in case when I need to use some programs that only runs on XP.

Linux/Ubuntu I love it!
