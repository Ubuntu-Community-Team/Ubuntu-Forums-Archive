### Post by Horseboy on 2011-04-24
GTKpod won't work with your iPhone. Since iPhones/iPod Touches/iPad's run on a completely locked up Mac OSX based platform, no program would be able to communicate with it. You'd have to "jailbreak" it, which is a simple process of getting root access via exploited vulnerabilities in the kernel or the bootrom. But GTKpod still wouldn't be able to communicate with it, but I'm sure there's a way it can be done.
