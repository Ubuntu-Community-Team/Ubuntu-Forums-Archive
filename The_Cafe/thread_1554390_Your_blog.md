### Post by keithpeter on 2010-08-17
Hello All

My personal and professional Web site is [_here_]("http://bodmas.org/").

It used to be published using the WordPress script (for 5 years), and I think WordPress is a great open source project, to which I donated money.

Then I began to realise that I wasn't posting even once a week, nearer once a month. So I changed over to using some [_bash scripts and lftp_]("http://bodmas.org/pages/20100221--web_how_this_site_is_published.html") to publish my Web site from my desktop PC.

If I was starting a Web site now and not 10 years ago, I would probably be using wordpress.com or google pages rather than paying for a domain!

As fatality_uk says, the really useful and successful blogs I see tend to be about one topic that the author knows a lot about.
