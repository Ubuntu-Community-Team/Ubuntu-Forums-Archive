### Post by mikewhatever on 2012-04-26
At some, purely abstract, level Debian, Ubuntu and Mint get, more or less, the same updates. However, since Debian and Ubuntu have separate repositories maintained by different people, and since package versions and naming may differ between the two, updates may also differ on the bit by bit level.

Furthermore, various Ubuntu derivatives get different updates for different packages they have included, for example, Kubuntu gets updates for KDE related packages, and Xubuntu for XFCE, etc.

Linux Mint doesn't have separate repositories, and uses either Debian or Ubuntu ones, so that its updates should be exactly the same as the parent distro gets.
