### Post by gn0me on 2005-08-02
Well, I had just installed Ubuntu and was looking to install WINE.  Wanted to install the ie6 tools and such (for use with other applications in WINE), so I figured I'd browse the Microsoft website rather than just use WineTools to do it.  Found this thing about requiring a copy of "Genuine Windows" in order to download some of the programs on the website.  Downloaded the program offered on the site and got a genuine little keygen program they created, ran it in WINE, got my key, pasted it in the site.

Visited the site, went to a "Genuine Windows" required download.
[img]http://www.gn0me.org/microsoft_before.png[/img]

Ran the GenuineCheck.exe in WINE, got my key, pasted it in..
[img]http://www.gn0me.org/microsoft_after.png[/img]

Now apparently I'm running Genuine Windows!
Sure I am! ;-)
