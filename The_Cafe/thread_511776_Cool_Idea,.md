### Post by runningwithscissors on 2007-07-28
> **Yaffle said:**
> Is it possible to use USB stick(s) / SDcards as ram? I would of thought that they have a fast enough transfer rate, Also there is somethink similar to this in vista called "cash". :P
Yes, but considering that most flash memory only allows a very few number of writes (relative to hard disks), the device would be useless after some heavy swapping.

Also, I think hard disk I/O is faster, so you're better off having your swap on the hard disk.

So no, it's not a very "cool idea" at all.
