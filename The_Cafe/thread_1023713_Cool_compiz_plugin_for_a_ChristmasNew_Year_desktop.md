I've found a great Compiz plugin, it is called 'Snow' and guess what it does? Right! It adds falling OpenGL snowflakes to your desktop! :) Open terminal and type the following commands to install it (I assume that you have Compiz up and running):```
sudo aptitude install compizconfig-settings-manager compiz-fusion-plugins-unsupported
```Go to CompizConfig Settings Manager (in System) and turn on Snow plugin! It is activated with WIN+F3!
