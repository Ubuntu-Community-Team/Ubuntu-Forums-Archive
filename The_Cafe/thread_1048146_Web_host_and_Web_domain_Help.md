### Post by cb951303 on 2009-01-23
There are 2 ways
1. Easier: Forward via DNS
2. Harder: Domain Transfer

1. Go to 123-Reg.co.uk domain control panel. You should see 2 DNS server inputbox. Type the DNS server addresses of JustHost.com there. Wait 1 day. Done.

2. Ask your EPP (auth) code from 123-Reg.co.uk. Unlock the domain transfer from their control panel. Go to JustHost.com domain transfer control panel. Type in the auth code and domain name. Wait 1 week. Done.
