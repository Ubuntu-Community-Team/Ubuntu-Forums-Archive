### Post by Kai Summers on 2009-08-12
OK, so I started using Ubuntu back in the Hardy days but always kept a dual boot with windows xp, purely because Flash videos over the net seemed to be appalling quality in Full Screen running in Ubuntu, and my favourite Online Poker account through Full Tilt would not work in Linux even through Wine. Thirdly I am a Photoshop fan which again does not render well through wine
About a month ago I installed the alpha test of Karmic Koala and now have the latest 3.5 version of Firefox and latest version of flash installed. I have to say whatever you guys have done has worked!! Video streams are fluid and the version seems so stable, that I have now completely renounced Windows altogether. The dual boot is gone and I have freed up several GB's and reallocated the partition to my ext3 linux partition. The latest development version of Wine runs my Full Tilt account. Then I came across gimpshop which mod'd my GIMP GUI to be more like photoshop. So this year has seen Ubuntu hit the hattrick for me and I am very pleased.
[B]
So this post is a thank you to the community that has helped develop Ubuntu to its present state y'all :guitar:!!!!![/B]
