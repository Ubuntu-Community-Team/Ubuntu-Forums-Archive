### Post by koleoptero on 2012-02-04
> **dagroves said:**
> What Icon set is that and how did you get the Unity Dash icon to change?

The icon set is Faenza-Ambiance. To get it run in a terminal:

sudo add-apt-repository ppa:tiheum/equinox && sudo apt-get update && sudo apt-get install faenza-icon-theme --assume-yes

To change the dash icon go to /usr/share/unity/4/ and replace the launcher_bfb.png with the one you want. I used the one from Faenza which is in /usr/share/icons/Faenza/places/48/

### Post by koleoptero on 2012-02-04
> **Ringi said:**
> Why dont I get any tracks in Cmus?

In library view you see the artist, hit space to expand with the albums sorted chronologically and when you scroll to an album you'll see a tracklist on the right. :) Hit tab to focus the tracklist and scroll through the songs there.

Pro tips: Highlight a song or album or artist and hit y to add to the playlist view (3) to make a playlist with what you want to hear, or hit q to add to the queue view (4) where they go away after being played and then playback continues on the library or the playlist (depending on the mode). Use m and M to change modes (artist, album, all, library, playlist). And s and r for shuffle and repeat.

EDIT: Super Pro Tips: numbers 1 to 7 change views. View 7 has all the keybindings, dig them and see what you want to use. It takes several times to memorize the keys but once you do every single other way of navigating your music will feel clumsy and cumbersome.

> **viperdvman said:**
> If someone can make a proper Mac/Unity style global menu for GNOME Shell, then I'd likely switch over :D

I'ld say, take a look at the source code. Only the JS file is important I believe.

The most important part of the code is the bottom: the enable and disable functions.

```

function init() {
    let leftChildren = Main.panel._leftBox.get_children_list();
    oldAppMenuButton = leftChildren[leftChildren.length - 1];
}

function enable() {
  Main.panel._leftBox.remove_actor(oldAppMenuButton);

  button = new MyAppMenuButton();
  Main.panel._leftBox.add(button.actor);
  Main.panel._menus.addMenu(button.menu);
}

function disable() {
  Main.panel._menus.removeMenu(button.menu);
  button.destroy();

  Main.panel._leftBox.add(oldAppMenuButton);
}

```

There you see that the button is created with the MyAppMenuButton function. In that function, a lot of sub-buttons get added to that one. Instead of adding sub-buttons to the main one, it should be changed to adding buttons directly to the panel.

It's a bit of work to go through the entire code, but a good scripter (with some time) should be able to do it.
