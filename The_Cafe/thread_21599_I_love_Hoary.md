### Post by jdodson on 2005-03-23
[QUOTE=carlc]What makes Hoary so different? Does anyone care to list some notable differences between Hoary and Warty? I am happy enough with the warthog and at this point that I have no huge desire to upgrade.[/QUOTE]

sure.

gnome 2.10.  [http://www.gnome.org/start/2.10/notes/rnwhatsnew.html](http://www.gnome.org/start/2.10/notes/rnwhatsnew.html) this might seem like a trivial release, it is not.  the new changes are amazing.

default human clearlooks theme upgrade(this is seriously awesome).

xorg - this, like gnome 2.10 is a showstopper, xorg much improved video card support native.  if i did not require 3D i would not go proprietary driver!  the proprietary driver does not afford me a higher resolution anymore, the default is great!

better icon/menu integration - a grip of warty was that it hardly ever added the icons to the gnome menu.  i have found that most of the programs get added, some still don't, but it is much nicer.

speed - it is faster.

ndiswrapper - on the install disc!

latest and greatest - of the coolest programs.  some programs that i find noteworthy: firefox 1.0.1, kino .7.5(this program is great!), battle for wesnoth, gaim, openoffice.

update manager - this is a bit easier to use than synaptic and it is pretty automatic.

new default background - this is a pretty cool new background.

this release is about polish and it is an amazing job sofar.
