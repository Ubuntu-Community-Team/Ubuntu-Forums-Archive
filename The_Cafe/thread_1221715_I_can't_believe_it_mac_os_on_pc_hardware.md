Were you really that surprised that it's possible? Why?

The current Intel-based Macs use exactly the same hardware as PCs running Windows, Ubuntu or another OS. The only real difference is that Macs have an [EFI](http://en.wikipedia.org/wiki/Extensible_Firmware_Interface) instead of an old-fashioned [BIOS](http://en.wikipedia.org/wiki/BIOS) (although I'm not 100% sure of this).
