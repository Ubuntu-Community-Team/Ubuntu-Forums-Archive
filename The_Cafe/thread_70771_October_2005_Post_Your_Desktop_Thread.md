[QUOTE=antidrugue]Can you run multiple X session at a time?
Exemple: can I open another tty and from there open a new x session, while the "default" one stays open?[/QUOTE]

Open the **Applications** menu.
Open the **System** submenu.
Select **New Login**.
Press CTRL+ALT+F7 to switch back to the original, and CTRL+ALT+F8 to switch back to the new one.

### Post by dickohead on 2005-10-05
A nice simple and effective way to "mod the hell out of your desktop" is by using gDesklets - either use synaptic, or:
```

sudo apt-get install gdesklets, gdesklets-data

```
And then to get them to start when your system does, add the program "gdesklets" to start at login by adding it to sessions under the system menu.
