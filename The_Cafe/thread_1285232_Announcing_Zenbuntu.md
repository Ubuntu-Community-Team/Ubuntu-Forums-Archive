### Post by bodhi.zazen on 2009-10-07
Announcing Zenbuntu:

[IMG]http://zenbuntu.net/zenbuntu_logos/new_Zenbuntu4_246-66.png[/IMG]

I would like to announce Zenbutnu to the commuinty.

Zenbuntu is a custom Ubuntu respin with a Buddhist Theme.

The goals of the project are :

   1.  Community - We would like to continue the tradition of an Ubuntu "Buddhist Edition". This is achieved primarily via themes and the inclusion of Buddhist Bookmarks in Firefox. The Buddhist content is "soft" and for non-Buddhists should be non-intrusive.

   2. Security - Zenbuntu includes an active firewall (iptables configured with GUFW) and AppArmor is configured out of the box (with custom profiles for network aware applications).

   3. Compatibility with Ubuntu - Zenbuntu uses the standard Ubuntu repositories and we hope to release Zenbuntu in sync with Ubuntu.


Additional information :

[Home page]("http://zenbuntu.net/")

[Zenbuntu Features]("http://zenbuntu.net/features.php")

[Screen shot (custom xsplash)]("http://zenbuntu.net/Screenshots/xsplash.png")

[Launchpad]("https://launchpad.net/zenbuntu")

[Zenbuntu contribitors]("https://launchpad.net/%7Ezenbuntu-contributors")

IRC - We have a channel on IRC on freenode (irc.freenode.net) or Ubuntu Servers at ##Zenbuntu :twisted:

All are welcome to join the Zenbuntu Team. If you would like to become involved with the project please send me a PM or join us on ##Zenbuntu.

There is a Beta iso available from :

[Zenbuntu 32 bit]("http://bodhizazen.fivebean.net/zenbuntu-beta-i386.iso")

[Zenbuntu 64 bit]("http://bodhizazen.fivebean.net/zenbuntu-beta-amd64.iso")

Thank you to [fivebean]("http://fivebean.com/") for hosting the iso.
