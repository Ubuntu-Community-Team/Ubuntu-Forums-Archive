### Post by kwacka on 2009-04-25
It isn't possible to even measure the number of Linux users, let alone numbers for each distribution.

Number of downloads might give some idea, but how many computers is one download installed on? How many are downloaded, installed and removed when the user found they couldn't run a particular game?

Distrowatch is useful in that it gives an idea of the interest shown in each distribution by the visits to the site, but no more than that.
