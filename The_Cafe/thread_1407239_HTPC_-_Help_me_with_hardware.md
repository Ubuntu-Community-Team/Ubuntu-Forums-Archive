### Post by cariboo on 2010-02-15
Right now with the state of ATI drivers, that video card is a bad choice. I just picked up an Nvidia 210 with 512Mb ram with VGA, DVI, and HDMI outputs for the same price as the ATI card you are looking at. I personally would stay away from ATI until they start doing drivers properly.

I have an atom 320 with the 945 chipset media center pc. I used the Karmic mini.iso to install a CLI OS, then added LXDE and XBMC. My motherboard only has one pci slot, so I use the VGA output connected to my 32" HD LCD, which has a VGA input. I get the native 1360x768 resolution, videos run with no tearing and audio gets converted to Dolby Pro Logic 5.1 surround in my Technics receiver.

I've experimented with many different setups, and for me this one is the best one yet.
