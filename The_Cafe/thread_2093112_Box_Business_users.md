Here's something valuable that I can contribute. How to connect your Box account to Nautilus:

1. Open Nautilus
2. Go to File -> Connect to Server
3. Select HTTPS Web DAV
4. Type in your Box credentials and click connect

This will connect you to your Box account in Nautilus so you can upload or download your data using Nautilus. It does not provide real-time folder and file synchronization features.

Here's something else that's valuable. Box is thinking about developing a Linux Sync software application client, but they put this on their back burner. If more people register and log into the Box forums and tell them that they want Linux Sync to be shipped out sooner, then they will do it and Linux users will get Box synchronization and other features too.

Finally, they offer one of the lowest prices for cloud storage sync and file sharing only if you require enterprise grade features and administrative management capabilities like I do currently on the market.

If you are looking for more personal and casual sync and file sharing, then Ubuntu One is highly recommended, but you will need Ubuntu 12.10 or an Ubuntu One client for Microsoft Windows or Apple Macintosh on each PC or mobile device. Otherwise, DropBox is the best supported and easiest alternative solution, but it is one of the most expensive providers.
