### Post by mostwanted on 2006-11-25
GoboLinux has a "virtual" file system layout similar to that of OSX, using symbolic links to create fake names for the major file system folders for easier navigation. It also has some funny source packages which compile automatically when they're put in the program directory (like how OSX applications install by drag'n'dropping them into the Applications folder):

[http://en.wikipedia.org/wiki/GoboLinux](http://en.wikipedia.org/wiki/GoboLinux)

OSX-like program installation with a twist of package repositories can be obtained by using Klik. Klik circumvents your package manager and file system completely and gives you a single file you just double-click to run or throw in the trash to uninstall:

[http://klik.atekon.de/](http://klik.atekon.de/)
