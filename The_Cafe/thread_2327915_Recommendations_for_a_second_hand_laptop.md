### Post by kurt18947 on 2016-06-17
I agree too with 'business class' machines. I'm partial toward Thinkpads, there are packages in the repositories that add functionality to Thinkpads. It's pretty easy in the U.S. to find off-lease corporate machines T410/T420 1st gen. i5 for < $150, maybe quite a bit less. Take some of the savings and install a nice SSD, makes for a sweeet machine. Lenovo makes available online the hardware manual that details how to remove/replace components. The only downside I find to Thinkpads is wifi cards are whitelisted in the BIOS so it's not easy to replace the Intel 62xx MiniPCI cards that can be slow -- I typically get 25-35 Mb./sec. on an N only router running DD-WRT. I've tried various fixes with little success. If speed were a problem I'd install a faster USB WiFi adapter but I don't find it an issue.
