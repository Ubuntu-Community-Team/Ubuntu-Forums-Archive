### Post by Enverex on 2007-02-04
> **Sunnz said:**

Yes, but it's a real version of Windows, so you need a full license. You need a second license if you've got a copy of Windows installed for dual-boot as well as installed in a VM, you need a second license if you're using it on a second PC, etc.

That is a pretty absurdly stupid bit of reasoning. :confused: Do you think that the VM is just playing a movie of Windows rather than running it, and that it only coincidentally happens to correspond to your input?

Anyway, Windows XP and earlier require a full Windows license for each VM installation.  Vista liberalized those licensing terms and now has provisions specifically for VMs.  If you use Vista Enterprise or Vista Ultimate, you may have four virtual installations of Windows (but they must all be on a single real machine).
