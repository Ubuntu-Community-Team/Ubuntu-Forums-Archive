### Post by ubuntu-geek on 2005-07-21
This is the first of its kind at ubuntuforums. We are looking for talent to create a new/updated style for forums using the same style as ubuntu.com. We have permission to use their style sheets, ubuntu logo etc as we are currently doing so on the forums.

DEADLINE: Sept 1, 2005

PRIZES:
First Prize: $100
Second Prize:$50
Third Prize: $25

The first prize winners style will become the new forum default. Second and Third place winners styles will be optional additions users can choose.

CONTEST DETAILS:
1. Must be a valid vbulletin style based off the color theme/style at ubuntu.com. Information can be found below to get started.

2. Default XML Template and DEFAULT Vbulletin Graphics sets are available for download to match sizes and layout.

3. New forum graphics including icons, buttons etc must accompany your theme.

4. All CSS/HTML must be compliant with major browsers. IE, Firefox, Safari etc.

5. Style must match ubuntu's theme and flow well with the ubuntu culture.

Getting Started:
[http://www.vbulletin.com/docs/html/graphics_kit_vb3](http://www.vbulletin.com/docs/html/graphics_kit_vb3)
