### Post by sanderd17 on 2011-09-03
> **hhh said:**
> Linux blog? Aren't there a ton of distros out there that DON'T use apt?

Otherwise, nice.

Right: great tip for the OP.

@OP: if you want to write for Linux in general, try a distro with a complete other philosophy. Ubuntu tries to do everything out-of-the-box. So I would suggest you try a do-it-yourself distro like Arch.

In Arch, you basically start with nothing but a basic command line (from which you have to configure your internet and package manager) and a bunch of information in the Arch Wiki.

If you try a distro from the complete oposite point of view, you will be able to write about general Linux stuff and you will also be able to understand Ubuntu more.
