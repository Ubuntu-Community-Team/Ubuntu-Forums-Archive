### Post by pqwoerituytrueiwoq on 2015-09-06
if you plan to overclock i would go a little higher on the PSU
[http://www.newegg.com/Product/Product.aspx?Item=N82E16817151094](http://www.newegg.com/Product/Product.aspx?Item=N82E16817151094)
there is little price difference between a 1tb and a 500gb drive, a 1tb seagate barracuda should also be faster as it has a higher density platter IIRC
if you plan to run linux on this build i would recommend going nvidia over amd (some of AMDs drivers are so bad on linux intel GPUs can beat them)
if this is a budget gaming system i would suggest the intel Pentium anniversary CPU that is unlocked if you plan on overclocking
if you want a liquid cooler you should look at a H100i or better, look at a NH-D14, NH-D15, or a Hyper 212 EVO

if you do go with this AMD build you will not have much of a upgrade path, i upgraded from my Phenom II to a Z97 system cause i knew if i waited any longer i would have to buy new ram, all i would need was a new CPU/mobo
the only stuff you will be able to carry over are the PSU, case, and disk drives, yea you could keep the GPU but by then i doubt you will want to keep using it, you will have probably upgrade it by then
