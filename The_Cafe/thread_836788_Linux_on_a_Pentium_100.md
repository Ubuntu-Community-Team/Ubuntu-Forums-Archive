### Post by dragobr on 2008-06-21
Hello eveybody! 

I posted some time algo about a computer I have here.. it's a compaq presario 4410, with a pentium 100 mhz, 64mb of ram and a 10gb hd...

So I decided to put Debian on it... I was playing with it a little yesterday and now I have a file server and a web server with php (lighttpd + php5-cgi).. I changed the getty for fgetty, optimized the web server a little, uninstalled some things, stopped some services, and when I finished, it was all using just 8 mb of ram on idle... 

I tried to use xorg and wmii.. it ran well, but it was just for testing and I uninstalled it after some minutes of fun... 

Now i just installed gnump3d for mp3 streaming.. using now something around 17mb of ram when streaming music... 

Now I need cash to buy a larger HD.. I tried, some time ago, a hd with 250gb.. the bios complaint, but it was ok with the OS.. unfortunately, I need to use this hd on another computer..

Do I need to say how happy I am with this PC? It's very old and it's working like a charm.. well, i just thought that this story might be interesting to someone :P
