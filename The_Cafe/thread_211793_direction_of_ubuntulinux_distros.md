### Post by aysiu on 2006-07-08
Ubuntu certainly has screenshots of its future releases before the release dates. Dapper officially came out on June 1, but I think there were screenshots for it even in late April.

Microsoft is making a big deal about Vista because it's been almost six years since their last major release. Ubuntu gets released every six **months**, so the screenshots don't come out months in advance--usually weeks in advance.

I don't know where you get this idea that Linux applications aren't "visually appealing." Yes, *gtkpod* looks like shit but so does FileZilla. You can't really tell me that Internet Explorer looks better than Thunderbird or that MSPaint looks better than Kolourpaint. 

You have to compare like to like--what's out now is Windows XP, **not** Windows Vista. So compare XP to Dapper Drake.

When Vista finally comes out (whenever that is), compare it to whatever is out for Ubuntu then. It may be Edgy Eft. It may be something else.

In any case, no one I know says, "Yeah, I love Linux, but I just can't use it because Windows looks much better." I've never met anyone like that. Sorry. In fact, if you read [this thread](http://www.ubuntuforums.org/showthread.php?t=88023), you'll see that looks are rarely mentioned.

The difficulty in figuring out how to install something new usually has to do with culture shock. If you're using to the Windows way of doing it, a centralized repository will blow you away. Get used to the centralized repository, and you won't be able to go back.

As for the "simple enough for everyone to perform installs," that hasn't happened even for Windows yet. That'll be the day when you buy a computer from Dell and they say, "Here's a Windows installer CD. Install it yourself. Anyone can do it. We won't include any drivers or DVD playback programs or codecs. Good luck." Please!
