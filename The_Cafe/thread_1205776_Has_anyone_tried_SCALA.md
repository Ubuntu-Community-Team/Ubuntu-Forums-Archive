Scala is not a Java-like language. It runs on the JVM, but it is not much like Java. Scala is more advanced and more elegant than Java. It's not an easy language, but it's powerful and has object oriented as well as functional programming features. I like it and it would be nice if it will become more popular in the future, I'd like to do a Scala job one day.
