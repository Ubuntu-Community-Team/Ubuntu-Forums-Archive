### Post by T.J. on 2015-12-07
Giving you a specific answer depends on what desktop environment you are using.  KDE and Gnome, for example - load their own cursor sets.  If I am not mistaken, that is or is very close to, the default cursor set that X11 uses when no other set has been applied.  Since most "modern" desktops override the default set, you may have difficulty getting it, but it is possible.
