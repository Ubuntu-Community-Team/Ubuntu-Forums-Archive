### Post by buzzingrobot on 2014-11-05
I've owned MacBooks and ThinkPads...

Buy the MacBook if you intend to stay with the Apple universe. There's a lot of good software and support there. Most Linux software is available for OS X, either directly or via ports. In addition, you can run X on a MacBook.

Buy the ThinkPad if you know you want to jump in the Linux pool and stay there.

Getting Linux running on a MacBook can be easy, or very difficult.  The hardware changes, often in subtle ways, from release to release, and much of the guidance I found on the web applied only to that person's specific hardware.  The open source video drivers won't be as good as the proprietary drivers that ship with the MacBook, and the proprietary drivers from AMD or Nvidia for Linux probably not, either. 

All the character-based traditional Unix tools are there, or installable, in OS X.  It's a Unix.

Getting Linux running on Lenovo hardware is usually not difficult, but checking the hardware specs would be appropriate.
