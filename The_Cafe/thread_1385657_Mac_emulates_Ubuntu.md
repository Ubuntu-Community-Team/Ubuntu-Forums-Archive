### Post by MaxIBoy on 2010-01-19
Mac OS X had the first non-prototype 3D compositing window manager, ages ago. Certain elements of the UI have transparency effects on them, using this technology. This "Afloat" thingy simply allows you to set *more* of the UI elements as transparent. 

In fact, versions of Windows going back to XP have 2D compositing, including window transparency. It's just that "out of the box," these capabilities aren't used. However, similar to "Afloat," there are 3rd party tools to set window transparencies on XP. 

And assuming a well-defined UI, it is easy enough to set windows as always on top (think "pop-up bubble notification.") 

Your link is just a 3rd-party tool which, when installed, adds some useful capabilities. Nothing wrong with that. Similar to Compiz, actually.
