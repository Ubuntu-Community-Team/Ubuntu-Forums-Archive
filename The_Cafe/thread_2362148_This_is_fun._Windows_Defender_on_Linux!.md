### Post by Paddy Landau on 2017-05-24
> Google Project Zero's Windows bug-hunter and fuzz-boffin Tavis Ormandy has given the world an insight into how he works so fast: he works on Linux, and with the release of a personal project on GitHub, others can too.

Ormandy's project is to port Windows DLLs to Linux for his vuln tests (“So that's how he works so fast!” Penguinistas around the world are saying).
Here's more [about this project]("https://www.theregister.co.uk/2017/05/24/windows_defender_linux_port/"), which he's put [onto Github]("https://github.com/taviso/loadlibrary").

I don't have the technical know-how to do anything with this, but I bet some people do.

It's fun to see a Windows developer using Linux to develop his Windows app! I wonder how many Linux developers use Windows to develop their Linux apps?
