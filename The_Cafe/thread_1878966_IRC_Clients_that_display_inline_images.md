### Post by zerwas on 2012-11-22
Pidgin is able to display inline images. It can also [play videos]("https://code.google.com/p/pidgin-embeddedvideo/") from Youtube links etc. within the chat window.
That being sad, Pidgin is still an instant messaging application and no IRC client.

[grove's]("https://grove.io/") web client also supports inline images, but it's not Free Software.
