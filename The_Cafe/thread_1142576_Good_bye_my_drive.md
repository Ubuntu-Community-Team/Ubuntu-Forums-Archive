### Post by az on 2009-04-29
The drive seems to be recognized by the bios, which means you may be able to image the drive.  If there is any data on it you want to save, there is hope...

See [http://help.ubuntu.com/community/DataRecovery](http://help.ubuntu.com/community/DataRecovery)

You will need another drive onto which to write the image;  it needs to be the same size or bigger than the original.  

Use gddrescue to image the drive. It will read and re-read the disk until it gets a perfect image.  It can take a few hours or a few months.

Good luck.
