### Post by RAV TUX on 2007-10-01
Simple concept with not so simple goal to make a live Linux CD with Haiku pre-built in Linux to run in QEMU and easily install on the hard drive. Ideally it would include Haiku development tools.

**[SIZE="3"]Haiku Builder Live CD[/SIZE]**

If you would like to help put this new Linux live CD together please post here, developer discussion will continue at ##cafelinux (freenode).

the target is to provide a haiku builder cd/dvd

The goal of the Haiku Builder Live CD is ultimately to help in the development of the Haiku OS by facilitating a easy to use Linux live CD.
