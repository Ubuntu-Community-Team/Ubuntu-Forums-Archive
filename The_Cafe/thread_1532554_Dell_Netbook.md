### Post by snowpine on 2010-07-16
I bought a Dell Mini 9 with Ubuntu 8.04 preinstalled, about 1 year ago. It was a special "Dell branded Ubuntu remix" that was really outdated and not very compatible with "regular" Ubuntu. 

I would suggest simply buying a netbook you like with Windows preinstalled, then install Ubuntu 10.04 yourself.

Make sure you check the hardware compatibility list to see if the specific model you're looking at is well supported: [https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks](https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks)

System 76 sells netbooks with "regular" Ubuntu installed; I would choose this over the "Dellbuntu" netbooks if I was looking for out-of-the-box Ubuntu joy.
