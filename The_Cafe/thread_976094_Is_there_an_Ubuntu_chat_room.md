### Post by zmjjmz on 2008-11-09
Install XChat (from Add//Remove)
When you first start it you should see a server listing. Connect to Ubuntu Servers, and join the channel #ubuntuforums using the command ```
/join #ubuntuforums
```
You can do more stuff, like having it autostart, but I'm too tired to explain it now (it's some simple GUI switches, you can probably hunt them down).
