### Post by chris4585 on 2008-06-30
Well.. my idea is pretty straight forward, for anyone new to Ubuntu, it can be quite scary.  You know how right after you install Windows, it offers a very annoying tour in a balloon popup.  Why not do something similar for Ubuntu, but instead of a balloon, why not have a popup dialog Saying "Congratulations You have successfully installed Ubuntu.  Would you like a tour of your new operating system?"  giving the user some information, such as if you would like to try this later go to the system menu, and whatever else, but very noticeably why not have a "Check this box if you want to download videos explaining Ubuntu," or have a alternative option right under it explaining things in text and a graphical way.

and this dialog/popup box would be very easily accessible from synaptic & available in the system menu somewhere.. 

I feel like Ubuntu should have this!

Brainstorm Link: [http://brainstorm.ubuntu.com/idea/10545/]("http://brainstorm.ubuntu.com/idea/10545/")
