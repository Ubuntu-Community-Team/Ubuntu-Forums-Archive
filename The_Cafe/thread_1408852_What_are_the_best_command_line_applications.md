### Post by tjwoosta on 2010-02-17
My personal favorite setup is 

wmii (tiling window manager)
xterm (terminal)
vifm (file manager)
vim (text editor)
htop (system monitor)
ncmpcpp (music player)
mplayer (movie player)
cdrecord (burning disks)
irssi (irc client)
bitlbee (im to irc gateway, allows you to handle IM's with your irc client)
mutt (email)
rtorrent (torrent client, also I use the vi keys and color patches)
ushare (upnp media server, seves media to my xbox360)

For a web browser I do have w3m for cli based browsing, but I mostly use uzbl. Its gtk, but its still pretty light and is all keyboard oriented.

Now this setup is cli based, but wmii for some reason doesnt seem to be the lightest wm Ive ever used. Fluxbox actually runs with about 30mb less ram. I just prefer wmii because I like tiling and I love the way the wmii guys do it.

With this setup I never have to use the mouse. I can do anything and everything with just the keyboard, which is very convienient for me. Not to mention that everything uses vi  movement keys except htop (which im still trying to figure out)
