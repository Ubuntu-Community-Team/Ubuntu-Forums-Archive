### Post by 3rdalbum on 2012-01-23
Cores aren't all there is to it.

Intel's CPUs are more efficiently designed - each core is significantly faster than an AMD core.

Intel's CPUs do Hyperthreading too - while a core is waiting for data from a thread, it can start executing another thread in the meanwhile so it stays occuppied. This improves performance by about 30% alone.

It is not so much about workload or server market, as it is about the way programs are written. It used to be the case that every generation of CPU would come with faster clock than the previous, but for some time now this is no longer the case. Now CPUs come with more and more cores and if you want to tap into that power, you need a multi-core app. Currently most desktop apps do not take good advantage of multi-core, but this is slowly changing. Intel CPUs are still faster on the desktop, but this may not be the case over the long run, hence many people call the Bulldozers "Ahead of their time".
