### Post by finticio on 2021-08-09
[COLOR=#000000][FONT=Helvetica]After working on this hobby project for almost a year, I am so proud to finally say that it's complete. Fintic, the name of this project, is a desk size LED matrix ticker that could display all kinds of information. From stocks, crypto, forex prices, to news, weather, sports and even custom uploaded images, GIFs and messages, all controlled via a web server! I plan to add even more features in the future. [/FONT][/COLOR]
[COLOR=#000000][FONT=Helvetica]
[/FONT][/COLOR]
[COLOR=#000000][FONT=Helvetica]The hardware consists of two 64x32 P3 LED Matrix panels, a Raspberry Pi 3 A+, the Adafruit RGB Matrix bonnet, a 5V 14A power supply and a custom 3D printed frame to fit everything inside! [/FONT][/COLOR]
[COLOR=#000000][FONT=Helvetica]
[/FONT][/COLOR]
[COLOR=#000000][FONT=Helvetica]The code was written in both Python and C++, and is primarily based on the Hzeller's RPI RGB Matrix library. Check out some examples of the ticker in work below  :) [/FONT][/COLOR]
