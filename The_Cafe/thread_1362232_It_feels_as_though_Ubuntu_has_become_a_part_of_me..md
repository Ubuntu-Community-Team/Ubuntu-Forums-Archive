### Post by BslBryan on 2009-12-22
I distro hop with some regularity.  I rather like Arch (of course), as well as Gentoo and Fedora, among many others.  Mainly the big guys though.  However I've decided I'd like to try something other than the beta for Windows 7, so I grabbed a copy for $30, utilizing my student discount.

When I go from OS to OS, I don't really mind the change from Ubuntu, and quite often I embrace it, pushing everything I can to the limits of my ability.  I always come back to Ubuntu, however.  

My change to Windows has nothing to do with not liking Linux, thinking it's not ready for desktop use, etc.  I own a computer store that advocates Linux, I translate and help the project as much as I possibly can, and I'd love to soon become an Ubuntu member, if I get enough support.  But just that I am installing Windows on a computer that's almost exclusively run Linux, I feel a strange loneliness that I can't begin to comprehend.  Like I'm saying goodbye to a best friend before departing to some vacation spot.  I'd never thought I'd have such an attachment to something like an operating system.  

I'm sure I'll be happy with Windows, you know...  I mean, I'm at a level where viruses don't really scare me, and I'll be glad that I can run software without trying to make it work, but it's really made me decide that when I come back, I'm going to try to work on advocacy.  Because when someone can become so attached to a big bundle of software, that's when you know that you've made the right choice.  

And I wonder how many exclusive Windows tinkerers just haven't yet given Linux a try, because I'm sure that a handful will realize that Ubuntu or any Linux OS can be a huge part of your life, just as it is to me.  

Usually I don't even make a big deal about distro hopping.  I might post it in my Sig, if I feel like it.  But this hop has driven me to post about it.  I'll still be around here, posting just as much as I usually do (probably more so since I'm on break at college now), but I think I'll feel more like a visitor or a tourist if I'm not looking through the eye of gnome or some other window manager, running an operating system that has no boundaries.

Ubuntu has become a part of me, so much that I know that it will be my choice, for better or worse, for a very long time.
