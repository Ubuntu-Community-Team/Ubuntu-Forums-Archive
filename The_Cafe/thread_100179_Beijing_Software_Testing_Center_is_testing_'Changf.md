### Post by jianggw@bsw.net.cn on 2005-12-07
From December 2 to December 15, Beijing Software Testing Center have done a testing for Changfeng PC. “Changfeng PC” is promoted by Beijing government, which 16 hardware and software providers ally to develop for rural areas. Changfeng PC is installed with Linux system named RedFlag5.0 and other applications such as agriculture expert system, priced at around RMB 3,000 per unit. Now there is only a Chinese version. An international version of Changfeng PC will be developed in the near future. Beijing Software Testing Center has done a great deal of testing on linux and its applications so far.
