64-bit Mozilla Java plug-in has been committed to Java 7 (JDK and JRE 1.7).

However, Hardy Heron will have IcedTea's gcjwebplugin, which is based on OpenJDK6, which is the fully FOSS version of Java 6.  So it should be much more stable and usable than Gutsy's IcedTea plugin, which is built on OpenJDK7 unstable development code.
