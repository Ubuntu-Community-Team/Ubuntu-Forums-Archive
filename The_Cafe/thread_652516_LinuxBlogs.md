### Post by Dr Small on 2007-12-28
I have created a little project that is to be a big list of blogs with people posting about Linux. If you have a blog and post about Linux (in general, or about a certain distribution), you are more than welcome to add your blog to this list.

There are currently 8 categories, but more can be created if necessary.

If you have any suggestions to make towards it, just let me know, and I'll see if I can implement it to it :D

[http://php.8ez.com/linuxblogs/](http://php.8ez.com/linuxblogs/)

PS> The submit button (at the top) is to add new links to the list, and the report button is to report broken links or blogs that are not posting about linux, and shouldn't be on the list.

Have fun :)

Dr Small
