### Post by nocturn on 2006-03-27
I have spoken with the Schoolbell developers ([http://www.schooltool.org/products/schoolbell)](http://www.schooltool.org/products/schoolbell)).

Schoolbell is a Calendaring server sponsored by the Shuttleworth foundation, it is in the Ubuntu repos.

Currently, it only supports WebDAV for calendars, which is not supported by Evolution for writing (so you can only view them).

The developers currently have no time/money to implement CalDAV in Schoolbell to make it work with Evolution.  

So I was wondering if some people here have enough interest in a small calendaring server project to issue a joint bounty for someone that writes this code.

The developers have said that they would accept contributed code for this.

So, please post here if you would like to participate in this.
