### Post by PrimoTurbo on 2007-03-05
[img]http://img248.imageshack.us/img248/3881/ideamr8.jpg[/img]

Basically you have a web like structure of folders, all connections are represented. When you double click a folder a contents window opens and shows you all of the contents of the folder, this window is resizable, when you resize it the icons/icon text of files is resized at the same time. 

Furthermore the whole web of folders is movable/flexible, so you can arrange a number of folders in the same area, despite the fact that they might be located in completely different physical places, you will see arrow points where these folders actually exist.  You can then open each one of them up and a contents windows will open, you can mass rename, mass move/copy very quickly with out having to minimize or cascade anything or have overlapping windows.

All the data is presented to you, if you need to find more folders all you have to do is move your mouse left/right/down/up. At the same time you have excellent control in zooming out and in around the web of folders.

Best way to imagine this is like the movable map in the game DEFCON, or pretty much any strategy game with it’s movable camera, except instead of units you have folder and files.

What do you guys think? Has this every been done before?
