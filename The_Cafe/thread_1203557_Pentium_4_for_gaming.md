Forget the directx 10 games, they won't even run at lowest settings :(

On the other hand, the open-source linux shooters aren't that demanding usually, if you get a nice deal with the graphics card you will likely be able to run (almost) all of them at medium to high settings. I have a 512 MB DDR 2 GeForce 8 8600 GT video card myself, and it's really nice for these games. It can also run all of the commercial linux games at very high settings with no hicks. Prey and doom 3 for example both run very smooth at ultra graphics with max FSAA and at 1680x1050 resolution. You can get it for below $100;-

An extra gig of memory would certainly be usefull. Given how low RAM prices are these days, you might consider being two gigabyte of 800 mHZ DDR2. This is because if you install an 800 mHZ module into a computer which already has another, say, 533 mHZ module, both will run at 533 mHZ if you don't replace both.
