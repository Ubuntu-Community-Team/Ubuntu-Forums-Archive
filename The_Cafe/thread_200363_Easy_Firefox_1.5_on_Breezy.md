2. Install a recent version of WINE (through the official WINE repository is best).

3. Run Firefox in WINE.

It works perfectly, even when displaying Flash movies. Now let's see someone run Windows Firefox through WINE on Ubuntu on the Browser Appliance on VMWare on Windows!
