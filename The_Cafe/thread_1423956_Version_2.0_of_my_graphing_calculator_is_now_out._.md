### Post by RussianVodka on 2010-03-07
I just released version 2.0 of my open source, cross-platform graphing calculator. This is gona be the last release for a while, since I think I've accomplished everything I set out to do.

Many many changes and new features since last release. Try out them all.

All feedback welcome.

[Screenshot 1]("http://epoblaguev.com/images/graphCalc/calc.png") | [Screenshot 2]("http://epoblaguev.com/images/graphCalc/graph.png")

Some of the Features:
[LIST]
[*]Does NOT require installation.
[*]Works on Windows, Linux, and Mac.
[*]Evaluate mathematical expressions.
[*]Store and reuse variables.
[*]Simultaneously graph multiple equations.
[*]View table of values of an equation.
[*]Find min/max values of an equation.
[*]Plot and connect points.
[*]Save and load the state of the application.
[/LIST]
