### Post by PapaWiskas on 2006-06-21
Just a suggestion from a former Windows user, who relied heavily on Windows easy to use gui software to do this stuff.

Get Tovid, it has a GUI but the GUI is in dire need of some updating since it is several releases behind, so just get the Tovid program.

Once you have it installed, you open up a terminal and just type a few key things....

For example.... change to the directory where your video files are stored.  If you video files are stored in a folder called videos you would type

cd videos

Then once you are in the directory, for example you have a video called sample.avi, so to convert to a DVD compliant .mpg that K3B can burn as a DVD Video disc you would type the following...

tovid -in sample.avi -dvd -ntsc -out sampleenc

You do not need to put an extension on sampleenc.

Then make the menu by typing the following.  keep in mind you can type whatever you want in the " "

makemenu "Sample Menu" \
-out Sample

Then you need to make the disc structure...adding menu to associated files you converted.

makexml \
-menu Sample.mpg \
         sampleenc.mpg\
-out SampleDisc

Once that is done you can create the Video_TS folder simply by typing...

makedvd SampleDisc

Then you can use K3B to burn these files to a DVD and off you go.

Granted you menus will not be pretty but they are functional.

Now there are other programs out there like DVDStyler, Devede etc that will make fancy menus and such....

My point is that there are ways to get to your destination, you just have to learn a little bit more about which road to take.  Above all else dont give up!!!
