### Post by Mr. Picklesworth on 2008-10-14
Speaking of forking projects for no good reason, it was not very sensible to name this MPX which a two second Google search tells us is the name of a project for [multi pointer XInput]("http://wearables.unisa.edu.au/mpx/").

So this project has achieved two things so far:
-Further crowded the already jam packed Linux media player market.
-Confused anybody interested in the very worthy endeavour of having multiple mice with individual pointers on one X server and multitouch supported across the desktop including on legacy applications.

When I saw the name "Audiosource" I was expecting some kind of magical air mouse style input device that used high pitched sounds and desktop microphones, which could be used in groups. Alas, no :(

Other than that, I suppose the low resource folks amongst us are happy. It looks pretty. Does it come with the GTK theme?
