### Post by lovinglinux on 2010-06-03
Mozilla is conducting a survey to know what users like and dislike about add-on practices. For example I don't like when an add-on automatically opens the developer site after an update. Although this can be useful and not a problem when installing a single add-on, sometimes I install several add-ons in a row. So when I restart the browser, there are several new tabs with sites that I don't want to visit.

Please [take the survey]("http://spreadsheets.google.com/viewform?formkey=dC13c3pTUzVNUGZ1dkxZd2NZY1Vub0E6MQ") and help make Firefox add-ons better for you.
