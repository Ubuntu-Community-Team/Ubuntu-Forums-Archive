### Post by mrgnash on 2009-08-29
There's nothing wrong with using LyX in the beginning, but you'll probably want to move beyond it at some point. I started with LyX myself, and gradually started hand-coding more stuff, before moving to Gedit, and then Vim (both of which have excellent plugins for LaTeX :) ). 

This might also be of some help: [Gentle Introduction to LaTeX -- Modern Greek Translation](http://www.ctan.org/tex-archive/help/Catalogue/entries/gentl-gr.html#Download)

### Post by tezer on 2009-08-29
For me the best Latex editor is Gedit with the latest Latex plug-in (0.2. rc2). It has auto-completion, view of a document structure, it can generate dvi-previews with links to the text you're editing, it even can embed the R scripts.
Very nice and clean editor. 
As for Lyx. It's nice, but I wouldn't say it's 100% Latex compatible... It's just "simplified Latex".
