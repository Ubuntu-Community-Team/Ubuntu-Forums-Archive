### Post by Washer on 2008-03-04
I really tried to work with ubuntu's default search (tracker, is it?) before giving up & installing google desktop search (GDS). They're not even remotely close. I like to think I'm good at tech & I still couldn't get tracker working properly. GDS was up & running perfectly in minutes. Is there anything like it for OSS?

I should mention I ran into the resource hog issue, but it was pretty easy to google & fix. It happens because GDS is repeatedly trying & failing to read some huge image file. Just run 
```
ls -l /proc/$PID/fd
```
where PID is the process id number & blacklist that folder.
