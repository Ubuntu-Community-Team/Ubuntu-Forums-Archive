### Post by madjr on 2008-05-27

you should create a website dedicated to this to start recruiting.

show us some demos of the stuff you have made (even if they are simple puzzles)

you might also want to find groups off the net that already do this kind of work.

IMO recruiting at the community cafe without having nothing solid is just a waste of time.

**start off the work and they will come**.

That's how **FOSS works**, someone has to start the project and then others join.

you should start with 1 project at **sourceforge** and then start advertising it to interested people who may want to be part of main development or other stuff like testing, mod creation, art work, 3d design, etc.

in fact you can start helping out in other Open gaming projects, make some friends and get some experience with the process. Those same people might become your main developers.

i don't think you're taking this project very seriously. I don't see nothing in the table.

At least create some mockups of your game idea (if you don't have an idea yet, then start brainfarting).
