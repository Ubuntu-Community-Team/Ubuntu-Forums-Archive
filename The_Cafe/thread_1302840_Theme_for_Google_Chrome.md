### Post by CoolHand on 2009-10-27
I thought I would post this in case anyone was interested.  I have been playing with Google Chrome lately and though it has a few things that annoy me I like its speed enough to use it.  One of the issues I have with it is that it doesn't theme to my GTK theme on its own.  Because of that I have been playing with the theming on Chrome and came up with a first version to share.  I use one of the standard dark Ubuntu themes so this goes ok with it.  I may do more as the orange is too prominent for my taste still but like I said.  If you like it help yourself.

I put it in a tar.gz archive as the forum won't take a .crx file.  Just untar it and open the crx file with Chrome to install. (file:///home/user/theme-name.crx)
It's basically a zip file so maybe in the future they will allow uploads of crx files.
