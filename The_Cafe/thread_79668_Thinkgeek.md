### Post by RastaMahata on 2005-10-20
[quote=majikstreet];)

Anybody know any good sites to learn binary?[/quote] just use 2^x ;)
The main point is to use powers of two to write any number.
For example, use the following simple table:
2&#176;=1
2&#185;=2
2&#178;=4

Each power of two uses a space in an array.. so 2&#176; is the first one, 2&#185; is the second one, and so on. The trick is to turn on (1) or off (0) the powers :P

For example, if you want to write the number 3 in binary, you have to do 1*2&#185;+1*2&#176;=3. That means you have to turn on the first 2 powers. So 3 in binary is 11.

To write 2, you have to turn on the second power of two (2&#185;), and leave the rest off (2&#176;). So 2, in binary, is 1*2&#185;+0*2&#176;=2. So 2 in binary is 10.

Do you want to write 5? 1*2&#178;+0*2&#185;+1*2&#176;=4+0+1=5. So 5 in binary is 101.

I hope you understand, as this is the first time I explain math in english :(
