### Post by angryfirelord on 2008-02-01
Hard to say. I don't think Microsoft will ever sponsor a *completely* free platform (they'll tie it into SQL Server or .NET somehow), but I see this plan as a not half-bad idea.

I could be wrong, but let's look at it this way: When you have a bunch of servers running Microsoft products and a bunch a employees knowing only Microsoft tech, it makes it much harder to get to an open platform. However, if some of these servers were WAMPs, then it would actually make it a lot easier to switch to Linux since the apps would be the same. The only difference would be the OS and since Windows Server Enterprise still costs big $$$, I see a bright light for Linux adoption.
