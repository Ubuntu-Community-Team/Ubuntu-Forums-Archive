### Post by Stormy Eyes on 2005-05-19
[QUOTE=wowbuntu]And why the need to ONLY have KDE apps????[/QUOTE]

For the same reason that vanilla Ubuntu has only GNOME apps: Kubuntu is KDE-centric, a community project by and for Ubuntu fans who prefer KDE over GNOME. As such, Kubuntu installs only KDE apps by default. If you want other apps, you can use apt-get to install them.
