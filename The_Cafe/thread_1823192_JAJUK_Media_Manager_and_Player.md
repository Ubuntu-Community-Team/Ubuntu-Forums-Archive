### Post by bokgeneraal on 2011-08-11
p, li { white-space: pre-wrap; }  [FONT=Ubuntu]Ever since I started using Ubuntu I had been searching for a program that has a media manager equivalent in quality to that of Windows Media Player 10 ( and above ). All of them have dissapointed me. Of course VLC ( being the swiss-army knife of media players) does the job and does it well. But browsing through your collection is  a hassle. Finally I've come across Jajuk. It has the same ease of use as Windows Media player. The only features lacking from it is synchronisation and an equaliser. It also uses mplayer to play videos. One of my favourite media players for Linux and I highly recommend this to anyone.:o[/FONT]
