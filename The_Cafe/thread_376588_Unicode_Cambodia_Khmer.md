### Post by simosx on 2008-08-30
There is a strong open-source initiative in Cambodia, Khmeros,
[http://www.khmeros.info/](http://www.khmeros.info/)

The proper way to install Khmer language support is to enable it from System/Administration/Language support. This will pull in things like fonts automatically.

It is quite possible that some things may be missing from the existing Khmer language support in Ubuntu. Helping out with testing, so that a default installation for Khmer works out of the box, would be really appreciated.

### Post by felixdzerzhinsky on 2010-05-04
I think a big part of the problem is that Gnome is not as well internationalized as KDE. 

[http://l10n.kde.org/stats/gui/stable-kde4/team/km/](http://l10n.kde.org/stats/gui/stable-kde4/team/km/)

Kubuntu may work better in Khmer at this point in time.

There is a Translation page on Launchpad:

[https://launchpad.net/~khmer](https://launchpad.net/~khmer)

But not a lot of activity.

KDE  is probably the best option at the moment for Khmer language users.


The Khmeros people are very OpenSuse oriented.
