### Post by CharlesA on 2010-04-15
Shouldn't have anything to do with MTU, since it will be at 1500 bytes when it goes out onto the internet.

The only reason to have a higher MTU is if you are running Gigabit ethernet on hardware that supports a higher MTU. (I haven't been able to set the MTU on my gigabit NIC using Linux).

For me, both Windows and Linux download/upload at the same speed.
