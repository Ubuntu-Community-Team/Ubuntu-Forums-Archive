### Post by southernman on 2007-10-21
My overall experience has been better than expected. The only real issue I've had with Ubuntu, can't really rest soley on Ubuntu's shoulders.

See, I have this motherboard ASUS A7N8X. It doesn't play well with Ubuntu as far as the onboard network port. Between it and the /etc/udev/70-net-persistant-rules it can be troublesome for those less determined to resolve the issue. Once you edit the file with the appropriate driver and adapter name, all is fine... 

I'll add too, this problem isn't specific to the Ubuntu OS and didn't happen until Gutsy. I first hit this snag when toying with GENTOO.

Overall - Ubuntu is astounding in my book.
