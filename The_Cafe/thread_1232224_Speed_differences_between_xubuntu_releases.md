Yes, the boot times are considerably faster, but even just the general responsiveness of applications is better.

If you have only 512 MB of RAM, I don't think you can load a ton of applications at once.

My point was mainly that the Windows and Mac conventional wisdom of "newer release requires more resources" doesn't apply to Linux. Newer releases usually take the same resources, use fewer resources, or just use the resources more efficiently.

If you're really worried about applications freezing up on you, I would advise using lighterweight applications--Sylpheed instead of Thunderbird, Dillo instead of Firefox, AbiWord and Gnumeric instead of OpenOffice, etc.
