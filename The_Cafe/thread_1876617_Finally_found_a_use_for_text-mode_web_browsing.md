### Post by SteveDee on 2011-11-06
I was searching for information earlier today when I found myself on a LinuxForums.org page.

There seemed to be an annoying advert in a frame blocking much of the text I wanted to read. Clicking on the ads "X" closed the advert ...but wait a moment...its also shut the page!

Loading the page again I read the small print: "Log in to disable this ad". Not a chance!

Opening the page and using <ctrl><u> allowed me to see the page source, so I could then clamber through the mound of HTML and read all the comments on the page.

This set me thinking that maybe I needed a text-only browser for occasional "emergency" use. However, a quick search for Firefox add-ons led me to "Textise", so I can now get the text for any page from within my default browser.

While not exactly a convert to text-mode browsing, I'm beginning to see some benefits.
