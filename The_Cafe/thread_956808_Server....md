### Post by bobbocanfly on 2008-10-23
Why would you want to use one? 

Serving files around your home network
Hosting your own website (if you have a vaguely ok broadband connection, serving a personal blog should be fine)
Folding @ Home (or anything else like that)
Cache server (would make browsing the Internet faster, especially if you have multiple people on the network that visit the same kind of websites)

I run one that I test my Ubuntu package uploads on, making sure they all build etc, so that you guys dont download broken packages ;)
