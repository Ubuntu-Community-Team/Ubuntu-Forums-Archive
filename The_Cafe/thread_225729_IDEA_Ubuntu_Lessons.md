### Post by tomplast on 2006-07-30
Before anyone starts to throw comments to me about an allready exisiting wonderful guide I would like to say that this idea is merely an extension to the allready existing guide.

System documentation is amazingly easy to comprehend but it's still mainly text pieces. My idea would be a site called Ubuntu Lessons or something like that where people could watch video/flash-demonstrations of how to do things in Ubuntu, this doesn't have to be taken only from the System documentation guide, it could cover other common things as well :).

The only problem that I see is localized demonstrations. We get a quite good compression rate of Ogg/Flash-movies so if the bandwidth and space were enough localized demonstrations wouldn't be a problem.

To create these demonstrations Istanbul or xvidcap is two great tools up to the task.


Please tell me what you think about this.
