### Post by marshmallow1304 on 2010-03-29
> **themarker0 said:**
> I wouldn't know how to start even :P I think Wiimotes run on bluetooth no?

Yes.  You'll need a bluetooth adapter and a couple packages ([see here]("https://help.ubuntu.com/community/CWiiD") for details).  Once you've learned how to edit the configuration files, you can use the wiimote as a mouse pointer and a game controller.  It's great with SuperTux 2 and SuperTuxKart. :D

Also, you can get homebrew on your Wii and install Media Player CE (or one of the other available players), then use it to watch shared media over the network.  Edit: Also, DVD playback.
