### Post by drs305 on 2012-06-18
If you started the thread, it's up to you to decide when to mark it solved. You can consider SOLVED to also be a "no longer seeking input" although in this case a note indicating there is no solution might be helpful (see below).

The SOLVED tag does two things - it allows 'helpers' to concentrate on threads still needing input. Secondly, it helps those looking for answers narrow their searches to threads more likely to contain solutions.

If you give up on finding a solution in an especially long thread, you could mark it SOLVED but add a note in the first or last post that no solution was found. That can help save a lot of time going through a thread without a solution.

Also remember a SOLVED tag can be reversed from the same link that added it.
