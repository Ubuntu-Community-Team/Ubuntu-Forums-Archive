### Post by Tatty on 2009-01-19
MythTV is probably what you want for your media centre.  Mythbuntu is basically just a customised ubuntu distro with mythtv integrated out of the box.

MythTV is in the repos - its pretty easy to install if you install the mythbuntu control centre first, then let that install and configure it for you.

There are lots of TV tuners that wont work in ubuntu as they dont all have linux drivers.  You may want to do a bit of research before buying one.  I have a nova-t 500 dual DVB tuner, though it took me a bit of googling to find a howto on how to install the driver.  (I would post it here but I seem to have lost the link)

Yes it can work with touchscreens, ive seen a few interesting youtube vids of people running ubuntu with a touchscreen.


**EDIT:** There is also the Linux Media Centre Edition (Linux MCE) which is very similar to windows media centre.  However from my limited personal use of it, it was a little bit slow and buggy on my machine.  I believe you need to invest in some made-for-purpose hardware if you want it to function well.

### Post by Tatty on 2009-01-19
> **stonewall12345678 said:**
> but i thought Myth TV was another OS completely.
forgive me if im wrong

No, MythTV is just a linux TV/PVR application.  Mythbuntu is basically a variation (flavour) of ubuntu, which comes with MythTV pre-installed and a few other tweaks.  

Afaik, all the parts of mythbuntu are in the official ubuntu repos, so switching between a standard Ubuntu install and a Mythbuntu install is nothing more than installing/uninstalling a few packages.
