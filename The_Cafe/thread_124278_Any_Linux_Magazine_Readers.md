### Post by kewl1uk on 2006-02-02

I actually installed it but on a different computer - there's no way I'm getting rid of Ubuntu. It's basically LinEx which is sponsored by the government of Extremadura - an autonomous province in Spain - with 150 games added. The desktop was a sight to behold which appears to be a mix of Gnome and KDE with game drop-down menus in the top bar and the KDE bar at the bottom. Looked ok but the desktop was crammed with icons. A surprise was that Java, Flash, Codecs and Firestarter was included and also updated. Doesn't appear to be an automatic update. I accessed the update from the Linex menu which then disappeared after I updated. I uninstalled it because it didn't detect my floppy and I couldn't find any means of configuring hardware on the desktop and I didn't have the inclination to start messing with fstab. It will install alongside Windows on the same disk but doesn't automatically resize the windows partition but includes a partitioning tool. Has both the Anaconda installer and a text installer. I chose the Anaconda.

Getting back to Linux magazine I think I might unsubscribe although I like the articles. The DVD becomes somewhat redundant when you're sticking with a distro, containing as it does Linux OSs.
