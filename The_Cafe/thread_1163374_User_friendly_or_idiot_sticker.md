### Post by Bodsda on 2009-05-18
===*!=*!==Begin Rant==!*=!*===

I would like to know if anyone else is getting rather annoyed with the so called user-friendliness of computers. I find that nowadays most programs will ask you to confirm actions by default, which I believe is just wrong and a step in the wrong direction. 

While I have no problem with programs prompting me that closing it could result in unwanted behavior such as terminal running applications. But I think that this feature should be disabled by default. 

Now before everyone starts shouting out the Ubuntu philosophy and how its aimed at average joe, let me just say that I'm not criticizing Ubuntu for this 'feature'. 

I dont understand why anyone would want a piece of code to be telling them what they are doing is wrong when it obviously isnt. If i want to close the terminal running firefox its because I want firefox gone as well. If I am a careless person who forgot that it was running an app then thats my fault and I should face the consequences, I should have to enable prompting manually.

It seems to me (nobody take this personally) that the people who are in need of these constant prompts have brought that plague upon me. Just because you need to be prompted does not mean I should be. 

/me is considering spending the night working out how to disable prompts on every app he uses and then write a script to enable/disable them all automatically. 

===*!=*!==End Rant==!*=!*===
