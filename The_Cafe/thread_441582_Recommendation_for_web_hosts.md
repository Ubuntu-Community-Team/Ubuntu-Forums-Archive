### Post by -Phi- on 2007-05-12
I've run websites on [Bluehost]("http://www.bluehost.com/"), [Dreamhost]("http://www.dreamhost.com/"), and [Servage]("http://www.servage.net/"), and haven't really had problems with any of them. Bluehost is big and offers PostgreSQL as well as MySQL databases, Dreamhost is friendly and I have a soft spot for their [blog]("http://blog.dreamhost.com/"), and Servage's included domain names can be country specific (I wanted a .ca).

Then again, I haven't heard of pixelpost so I don't know what you need for it.

None of those links have referrer thingies on them, so I don't get monies if you pick one of them.
