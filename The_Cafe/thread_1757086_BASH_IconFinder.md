What is BASH Icon Finder?

BASH Icon Finder is a script I developed to collate all the icons on the system with particular strings in their path and present these visually on a single webpage. Icons can then be dragged and dropped onto receptive GUI elements on the desktop or the location cut-and-paste.

:) I have created a detailed image-rich tutorial at [http://scripting.fossworkflowguides.com]("http://scripting.fossworkflowguides.com") under the 'Workflow Tutorials' section. I invite you to visit the page and comment on the tutorial and try out the script. The source code is provided in an attached ZIP.

I use this script all the time and find it extremely useful. Either in customizing a launcher or giving a link a new look that might be more relevant than the MIME default icon used by Ubuntu.

Any feedback would be greatly appreciated, either through the tutorial webpage feedback form or via this post.
