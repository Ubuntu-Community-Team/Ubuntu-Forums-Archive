### Post by kevdog on 2009-01-31
I ran across this neat little command line script called apt-history that allows one for example at the command line to the history of packages installed/upgraded/removed via the apt system.  I love the simplicity of the script.

apt-history install
apt-history upgrade
apt-history remove

This utility may be found here:
[http://linuxcommando.blogspot.com/2008/08/how-to-show-apt-log-history.html](http://linuxcommando.blogspot.com/2008/08/how-to-show-apt-log-history.html)
