### Post by 3rdalbum on 2010-03-11
If you can program in Java, you can program in Python.

I'd probably go so far as to say, if you start programming in Python you'll never go back to Java :-)  Python is twice as quick to write and twice as quick to debug as Java. It's dynamic-typed - you don't need to declare variables and their types, just put whatever you want into the variable. And there are HEAPS of libraries to let you do certain things quickly and easily; and of course the Ubuntu repositories have Python bindings for almost everything.
