### Post by aamukahvi on 2006-03-23
Maybe this has already been posted, maybe not.

According to Mikrobitti magazine (finnish), Windows Vista will not make it to the Xmas market. The delay is due to additional testing and polish, the magazine states.

This, in fact, means that it is Dapper +1 that is "competing" with Windows Vista.

For those of you equipped with Finnish, read ahead for the full story, courtesy of Mikrobitti ([http://www.mbnet.fi/]("http://www.mbnet.fi/jutut/uutiset/?from=etusivu#U2035"))
> **Uusi Windows ei ennätä joulumarkkinoille.**

Microsoft aikoo taas kerran viivyttää uuden Windows Vista -käyttöjärjestelmän julkaisua. Näillä näkymin ohjelmiston kuluttajaversio tulee kauppoihin vasta vuonna 2007.

Yhtiön mukaan viivästys johtuu siitä, ettei Windows vastaa Microsoftin tavoittelemaa laatua. Kaikkia käyttöjärjestelmän eri versioita ei ole vielä ehditty testata halutulla tarkkuudella, ja ohjelmistotalo suunnittelee myös viime hetken parannuksia käyttöjärjestelmän tietoturvaan.

Alun perin Windows Vistan piti ilmestyä vuonna 2004. Sen jälkeen käyttöjärjestelmä luvattiin sekä yritysten että kotikäyttäjien saapuville vuoden 2006 toisella puoliskolla. Yritykset saavatkin jo omat versionsa käyttöjärjestelmästä marraskuussa, mutta kuluttajat joutuvat odottelemaan tammikuuhun 2007.

Marraskuussa julkaistaan kaksi Vistan kuudesta eri versiosta: Windows Vista Business ja Vista Enterprise. Tammikuussa julkaistavat kaikki neljä kotikäyttäjien versiota ovat Windows Vista Starter, Vista Home Basic, Vista Home Premium ja Vista Ultimate.

Windows Vistan myöhästyminen saattaa olla kova paikka tietokonemarkkinoille. Tietokonekauppiaat olivat toivoneet, että uuden Vistan julkaisu marraskuussa siivittäisi entisestään vilkasta joulukauppaa. Nyt käyttöjärjestelmän viivästyminen saattaa päin vastoin hillitä kuluttajien ostohinkua.
