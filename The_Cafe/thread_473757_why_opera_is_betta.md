### Post by silent1643 on 2007-06-14
For me i love the Opera browser.. and when compared to firefox i can't see using the internet without it.

Firefox always seems to load slower than Opera on start up.

Some java script crashes FireFox, for example if you embed any sound (on mouse over for example) using java script the whole browser just closes down.

After i installed some basic extension in FireFox - every little image and icon like you would find on ebay or a forum loads increadabliy slow - even after i remove all the extension and reinstall firefox!

The general feel of opera's website and support forums are much more appealing and their users are alot more friendly.

Opera's wand tool & speed dail are the best features i have found in an internet browser in awhile. I love the wand feature, it keeps all my passwords and stores them and doesn't allow access unless you have the master password. Perfect. The speed dial is awesome and saves my wife and kids the time of typing the website url in every time they want to go somewhere.

Finally the only thing i can complain about opera is that some websites are not fit for opera browsing. Example would be ebay when trying to sell something, the editor doesn't allow you to cut and paste html code in your ad if you are in Opera, but in firefox it works fine.

### Post by crimesaucer on 2007-06-14
> **pjalegria said:**
> Opera is very ugly...

Default Opera may look better (betta) than default Firefox when using the default Human gtk-2.0 theme...but once you change your gtk-2.0, your default Firefox will change with it, and if you have a nice gtk-2.0, then you can have a much better looking Firefox theme than any Opera theme available.

But with Opera, it doesn't use gtk-2.0 themes, or your system's colors. You can only use their Opera themes that aren't that nice (way too much plastic), and the custom color choices suck. There are a few themes that are nice and simple, but they still look so boring with all of the gray menus. 

Plus, widgets suck compared to extensions, java and flash are a hassle for a beginner, so is the embedded media compared to the media player plug-in Firefox.

The tool bar and tab system are worse than Firefox, and most web pages don't write their code to look good in Opera...plus to get the simple things that make Firefox nice, like Google thumbnails...you have to find a userscript and sometimes those don't work properly as stated in their forum...and then even the automated Opera updating didn't work for me so I just removed it and never looked back. 

Yeah...Opera was fast...but after I have done every optimized tweak available in about:config, I find Firefox just as fast.

Plus in Firefox...userChrome.css is so simple...changing icon packs and menu folders is also really easy...it uses my gtk-2.0 perfectly...all of my colors match the rest of my OS theme...I have the Ubuntu Forums extension...StumbleUpon...gmail skins.....grease monkey....stylish....View Source Chart....web developer tool bar...FireFTP...Gspace....del.icio.us....media player connectivity....the NBA scores...down them all...Fasterfox....gmail checker...measure it...Forecastfox enhanced....tab mix plus...cute menus....mr. local install...scrap book...and an easy to use search box with 18 different search engines...plus...having Adblock Plus with the Filterset makes using Firefox more of a pleasure than Opera.
