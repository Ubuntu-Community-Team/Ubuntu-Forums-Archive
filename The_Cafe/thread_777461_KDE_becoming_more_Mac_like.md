### Post by tbrminsanity on 2008-05-01
I found that KDE (prior to 4.0) was more Windows like in that the "look and feel" followed the same ideas as Windows (same top right icons, similar colour scheme, same positioning of the main menu).  This was probable a good thing since most new users to KDE were Windows users (people are more likely to adopt something new if it is somewhat similar).  As for adding Mac like "look and feel" features like border less windows and a brushed steel look, this will only help to attract Mac users to KDE plus it will also help to grab more Windows users that may be considering switching to Mac.

These changes may also have been done to assist the KDE on Win and KDE on Mac projects.  I don't know I would have to ask a KDE developer.

It's the basic concept behind plasma. Where Compiz has virtual desktops, KDE4 has containments, except that each containment has its own designation. So you can label one "work" or "schoolwork," and have it customized to facilitate the task that it was created for.

So you can have different containments for the different ways in which you approach your computer. What I was saying was that I'd like to be able to have more than one virtual desktop per containment, if that's not already possible.
