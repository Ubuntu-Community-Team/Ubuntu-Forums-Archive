### Post by BobLand on 2008-08-18
I like SearchMe.  It's not fast but it displays the home page of the web site related search string.  One very nice feature is stacks.  

I have a stack for News, Linux, MMA, Woodworking and a few others.  This way I can browse pages directly on one screen without having to jump back and forth or open new windows or tabs.

It is sloooow so it is not for intensive research.  However, for sites you go to often just to see updates, it is excellent!

Started using Cuil.  So far, it is a vast improvement over its initial debut.
