### Post by HappinessNow on 2009-08-26
After losing my favorite Firefox add-on: [Book Burro]("http://www.bookburro.org/") (due to a lack of it being updated); which has saved me hundreds of dollars in textbooks in the past. It is essentially needed once again with the Fall Term coming soon, so I have to start searching for the least expensive place to buy textbooks. This is what Book Burro does best by using the ISBN, it searches for the lowest possible price for textbooks (or any books). 

Unfortunately, the Book Burro Dev has yet to update this Firefox add-on to work with the latest version of Firefox. So I noticed that it also works in Flock (I thought Flock died years ago, I guess not), so anyway I installed Flock, once again after many, many years.

What The Flock! I found an awesome Flock Theme! [Aero Flock]("http://www.virtusdesigns.net/?page_id=639") makes this Mozilla based browser much more tolerable then the lousy default look. While Flock is an overly bloated version of Firefox, I am impressed. Most importantly 'Book Burro' works flawlessly.
