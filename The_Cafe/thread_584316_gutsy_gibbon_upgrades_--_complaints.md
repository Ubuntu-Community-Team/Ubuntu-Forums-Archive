### Post by diablo75 on 2007-10-21
Many seem to have problems upgrading if they had Beryl on their system before starting.  This problem has recently been fixed (at least for me, anyway) after an update for the tzdata package was sent out.  This allowed me to install compizconfig-settings-manager using "sudo apt-get install compizconfig-settings-manager" at the terminal.

Though I must complain too, that this problem should have been foreseen and dealt with before the release.

Another problem I had after the upgrade was VMware server not working because the kernal changed.  Shouldn't something like this be fixed automatically via the update manager?
