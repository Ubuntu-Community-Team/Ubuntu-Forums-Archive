### Post by Bucky Ball on 2015-05-18
I'd go for a straight install of Xubuntu or [Xubuntu-core]("http://xubuntu.org/news/introducing-xubuntu-core/") (preferably the latter, but you will need to be able to get online with a cable unless you are lucky enough to have it pick up your wireless during install). Xubuntu-core doesn't come with very much and you install most things manually. But you do get the Ubuntu kernel and xfce4 and a few other bits and bobs to get a working desktop and get you started. 

If that machine works ok with Ubuntu, it will work better again with a smaller, more lightweight install, Lubuntu or Xubuntu (I prefer Xubuntu but each to their own ... I actually do [minimal installs]("https://help.ubuntu.com/community/Installation/MinimalCD") and add xfce4, synaptic and take it from there ... if you're feeling game and want to continue with the learning curve, give a mini a whirl). 

Good luck, thanks for sharing, and have fun! ;)
