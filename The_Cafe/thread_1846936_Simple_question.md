### Post by Alwimo on 2011-09-19
You'll be able to install Gnome Shell just by typing "sudo apt-get install gnome-shell" into the terminal, giving your password when prompted and entering 'y'.
 
Easy! :) 
 
But it will come with Gnome 3 installed, which is different from Gnome Shell, running underneath Unity. Unity, like Gnome Shell, are shells that go over something else. 11.04's Unity was a shell for Gnome 2. 11.10's Unity and Gnome Shell go over Gnome 3.
 
Being able to switch between Unity and Gnome Shell easily is the thing I'm most excited about in 11.10. :)
