Hello everyone reading this! I created a new linux OS! Im only 13 and this is pretty big!To run this hunk of linux you
will need:

-Virtual Box (Or any kind of Virtual Computer Emulator)
-My OS (Of course)
-Computer with at least 6 GB Of space!

Once you have Lets get started
First Download my OS

Download Link: [http://www.mediafire.com/download/b76mi9ld9kq3gjo/L.e.P_OS_1.0.5.i686-1.0.5.preload.iso/](http://www.mediafire.com/download/b76mi9ld9kq3gjo/L.e.P_OS_1.0.5.i686-1.0.5.preload.iso/)
Open Virtual Machine Emulator
First Create a new machine (If youre using Oracle VM Virtual Box)
Then, Make it Linux and open SUSE (whateva the VER is)
Next, Click Add or make
Go to Settings (Right Click Your new Machine in Left Bar)
Go to Storage.
Click on CD/DVD Drive Under IDE
Then Click the little CD next to Attributes
Find my OS
Go to System now
Then Change the Boot Order to CD/DVD to The first one.(It boots of a CD/DVD if you wanted you could burn it to a disk!)
Finally Press OK
Then Press START
There you go!!!!

If presented with black screen WAIT its loading. (Its loading my assembly and Suse Studio BTW great OS creator! susestudio.com)

Thanks if any problems or you need help starting the OS Email or leave a comment

BTW Graphical MODE is CTRL-ALT-F7

Password : password
