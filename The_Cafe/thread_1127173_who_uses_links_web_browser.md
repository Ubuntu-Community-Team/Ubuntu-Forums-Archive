### Post by .Maleficus. on 2009-04-16
I don't use it all the time but I use it pretty frequently.  I got used to it by just using it a lot- I used it last night to read the Gentoo Handbook while installing Gentoo on my server, I immediatly install it on a fresh Arch install so I can download packages before Yaourt is on their.  My suggestion is to just keep using it and power through the learning curve.  It's like Vi; weird to get used to but easy once you know it.  Also, a really good thing to do is simply press Escape and check out what's in the menus and read the shortcuts for each item.

Oh, and I use eLinks.  It's got more features than Links (if you only want text) but cant render pictures like Links 2 can.
