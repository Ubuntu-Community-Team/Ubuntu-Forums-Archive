### Post by jack24 on 2009-07-23
It would be great if more hardware companies would routinely consider Linux when developing their products.  I have a couple of suggestions that I'd like to get some comments on.

It's amazing how easy it is to install Ubuntu and use it on almost any computer -- even my Sony laptop with all of its proprietary stuff. I just helped my son install Windows on a computer and we ran into more issues than I've had installing Ubuntu in years.  The main Linux problems seem to be accessories, like phones, networking equipment and the like.

I'm in a state legislature and we get tons of email on issues from people who send it automatically from a website. For instance, it will say: "Concerned about this?  Click here to send a message to your representative."  They enter a bit of info and the messages goes out.  What if we had something similar we could use to easily notify hardware companies that we're using Ubuntu/Linux and would like support?

To go another step, the message could include a link to someone who could help the hardware company develop a driver for Linux.

Any thoughts?

Unfortunately, I don't know enough about writing software to work with them, but I'd could help set up and maintain a site to coordinate it.
