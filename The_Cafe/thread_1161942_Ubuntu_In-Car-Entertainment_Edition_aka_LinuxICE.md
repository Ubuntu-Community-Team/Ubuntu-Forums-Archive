### Post by kev000 on 2009-05-17
[OpenICE]("http://openice.org"), a team dedicated to pushing Linux as the OS for the car, is working on creating LinuxICE2, a slimmed down version of ubuntu specifically tailored for use in the most mobile of environments: your car.

Because the vehicle is different from the desktop, and even from current mobile environments (see [why here]("http://tripzero.blogspot.com/2009/05/mobilest-of-environments-car.html")), some custom software has been developed to make it easier to use Linux in the car.

If you have any interest in way-awesome technology and cars, you may want to contribute to the project.  Daily live [images]("http://linuxice.openice.org") are being built so testing and debugging is always welcome.

If you know how to code (c++), the OpenICE team is currently working on building a customized mobile toolkit + SDK using Qt and Clutter.

Here's a concept demo video of what's in development: [http://www.youtube.com/watch?v=mEfrnDOxcUw](http://www.youtube.com/watch?v=mEfrnDOxcUw)

The OpenICE team is playing with a couple interesting concepts including [compiz integration]("http://www.youtube.com/watch?v=FWuQeUH7dMA"), [multi-process desktop]("http://tripzero.blogspot.com/2009/04/multi-process-mobile-desktop-concept.html") to take advantage of multi-core systems, [car engine/system integration]("http://www.youtube.com/watch?v=GbiMpJNHE24") and more.

We are looking for:
* Coders
* Ubuntu boot speed hackers
* Kernel hackers
* Testers

If you want to learn more, check it out: [http://openice.org](http://openice.org)
