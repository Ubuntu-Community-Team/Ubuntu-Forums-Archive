### Post by theviewfromtheground on 2008-06-09
I am thinking about getting a Lenovo x61s and have been evaluating if this is worth getting.  It seems that the answer is probably no.

If I were to get this, I think I would use it as a swap device.  Perhaps that goes against its stated goal of acting as a drive cache, but my memory utilization tends to be high because I must often run vmware and generally need gobs of memory around. 

Plus, setting up multiple, prioritized swap partitions is [  ]("http://tldp.org/HOWTO/Partition/setting_up_swap.html#multiple_swap_areas").  I don't have tons of time to maintain my system, so if I'm going to utilize it, I want it to be easy to deal with.

There is a [Slackware thread]("http://www.linuxquestions.org/questions/slackware-14/best-utilization-of-intel-turbo-memory-in-slack-609345/") about this.
