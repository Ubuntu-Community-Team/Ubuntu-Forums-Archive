### Post by RAV TUX on 2007-09-30
>    a video game by [Jason Rohrer]("http://jasonrohrer.n3.net/")

Cultivation is a game about a community of gardeners growing food for themselves in a shared space.

 Cultivation is quite different from most other games. It is a social simulation, and the primary form of conflict is over land and plant resources---there is no shooting, but there are plenty of angry looks. It is also an evolution simulation. Within the world of Cultivation, you can explore a virtually infinite spectrum of different plant and gardener varieties.

All of the graphics, sounds, melodies,and other content in Cultivation are 100% procedurally generated at playtime. In other words, there are no hand-painted texture maps---instead, each object has a uniquely "grown" appearance. Every time you play, Cultivation generates fresh visuals, music, and behaviors. You can read more about what I was trying to do with Cultivation in [this essay]("http://www.northcountrynotes.org/jason-rohrer/arthouseGames/seedBlogs.php?action=display_post&post_id=jcr13_1169054799_0&show_author=1&show_date=1")

Read my [open letter]("http://www.northcountrynotes.org/jason-rohrer/arthouseGames/seedBlogs.php?action=display_post&post_id=jcr13_1168438267_0&show_author=1&show_date=1") to the other Slamdance Finalists about the controversy surrounding [SCMRPG]("http://www.columbinegame.com/").

Cultivation is free---if you think it's cool, please [support my work]("http://hcsoftware.sourceforge.net/jason-rohrer/supportMyWork.php").[http://cultivation.sourceforge.net/](http://cultivation.sourceforge.net/)

### Post by Sandlst on 2007-10-01
Cultivation seems pretty cool, I haven't had the chance to play it yet, but if you want to install it from source on feisty, the dependencies can be installed with ```
sudo apt-get install libglut3-dev build-essential 

```Then download the source package from their site, extract it then run ```
./runToBuild
```from inside the folder, it will install to that folder.
