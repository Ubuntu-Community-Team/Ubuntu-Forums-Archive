### Post by Mr. Picklesworth on 2008-05-16
My favourite torrent search site is [Torrentz]("www.torrentz.com"). It's really useful, because identical torrents duplicated across sites are all merged under one entry. For example, if I search for Ubuntu 8.04 there are several torrents listed, the entries for which lead me on to all of their respective pages on the actual hosts.

It keeps stuff really tidy, has a library the size of all those torrent sites combined (but still sorted in an orderly way), and has some nice features so I can really quickly find if something is worth my while.
