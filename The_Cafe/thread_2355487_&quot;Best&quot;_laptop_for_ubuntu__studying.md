### Post by faraco3 on 2017-03-19
General guideline if I want to buy laptop for using Linux -

* 64 bit architecture
* Has 4GB RAM or more
* Has a minimum of 1 or 2 USB keys for files and installation backup 

Other than that, is optional.


PS: You don't want to compile big applications on 32 bit in case you want to try (for example - openjdk/rakudo jvm). You either run out of memory and crash, or take a lot of time to finish.
