If you could build a browser any way you liked, what features would you include? How would you design the GUI?

These are my ideas

> 
# - User sign in with full encryption of user's profile
# - Compatability with ALL major platforms which support Python using the same
# code
# - Universal sign in credentials for the entire web using gpg public keys
# - Embeddable Peer-to-Peer file sharing using file hashes as URLs
# - Tabbed browsing
# - Popup control
# - 3D browsing
# - Ability to write third-party add-ons using the Python scripting language.
# - Open sourced under the GPL or CC
