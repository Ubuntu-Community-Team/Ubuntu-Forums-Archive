### Post by Bou on 2008-06-08
Please give [this]("http://ubuntuies.com/index.php/2007/12/07/guia-del-instalador-en-pendrive/") page a look. It's a Spanish distribution based on Ubuntu, whose main feature is that it gives an easy way to create live USBs.

Basically, you burn a LiveCD and find an option to install the system to an USB:

[IMG]http://ubuntuies.com/img/Instalador/00InstaladorEnEscritorio.png[/IMG]

Then you follow the install wizard and voila! you get a Live USB you can take anywhere and even save changes. Basically, you can take your Ubuntu with you anywhere you go :D

[IMG]http://ubuntuies.com/img/Instalador/03SeleccionaDispositivo.png[/IMG]

[IMG]http://ubuntuies.com/img/Instalador/05InformarTamanoPedirParticion.png[/IMG]

[IMG]http://ubuntuies.com/img/Instalador/11InstalacionCorrecta.png[/IMG]

Wouldn't it be great to have something similar with the official Ubuntu?
