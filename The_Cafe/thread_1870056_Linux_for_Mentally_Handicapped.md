### Post by decktrio on 2011-10-26
**Does anyone has experience setting up a pc for a person who is mentally handicapped (either EMH or TMH)?**

My younger sister is TMH (trainable mentally handicapped), and I've been thinking that maybe a computer would help her self-esteem a bit. (Plus I have a old pc laying around.)

To go into specifics about her disability... she will probably never learn to read or do math (unless it's with manipulates, like an abacus). However, she can memorize icons very easily, so an icon-based gui would work wonders**. She's in the process of learning "life-skills", or survival, sight words. Her class has some windows computers that the students can choose to use throughout the day, but Windows intimidates her; I think it's the wordy menu. An icon based menu is ideal for MH people. 

[B]Has anyone tried something similar with a family member?
If you have, what distro/how did you configure it, to encourage them to use it? How did you deal with log-in screens? What apps did you install?
[/B]

**An icon-based gui would work wonders on *anyone *who's mentally handicapped.

### Post by RaganN on 2011-10-26
I once set up a computer for a developmental center using Windows by making desktop shortcuts to everything people would need (I noticed they had difficulty running the Start Menu) and increasing the icon size to make it clearly visible (some of the clients had vision problems, and I noticed they were unable to see the icons), a few other changes like increasing double-click time.  But mainly I emphasized using desktop icons, and it worked very well.  You could setup Ubuntu with launchers on the desktop for the necessary applications, to get the same effect.  I think a dock program like Cairo Dock might also work, since it visually sorts the most useful programs into icons.

### Post by Alwimo on 2011-10-26
Icons on the desktop would be good.
 
With Gnome 2 and possibly other DEs, you can put an icon on the desktop and right-click on it and choose to resize it. You can then drag the markers at the corner to make the icon as large as you want if you think bigger would be better.
 
If she only uses a few programs, that would be ideal.
 
Gnome Shell is great as an icon-based gui. Just move the cursor to the top left hand corner and then select "Applications" and all the application icons show up. Although some will probably share the same icon. Uninstalling any programs she's not interested in would help a lot.
