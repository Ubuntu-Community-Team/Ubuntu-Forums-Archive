### Post by llelectronics on 2008-06-26
Anyone that want to have a fully, fast and easy to use IceWM Desktop should test ZevenOS Beta 5. (known also as the former Zebuntu)
It's based on Ubuntu 'Hardy' (8.04) and tries to reimplement a BeOS kind of look. 
We created a lot of own written tools like MAGI 2(a application- launcher-replacement for the desktop with an easy installation assistent) that allows older people and computer beginners to do what they want to do without standing in their way. An OneClick Installer that allows you to install whole packages with one simple click (i.e. Gamerspack with lots of Opensource Games and tools gamers need, like xfire, ts2 ...) . Gencoder and Videtis handle your video conversion and VideoCD/DVD creation. And a bunch of IceWM graphical configuration Tools (IcePanel, IceMenu, IceDesktop,IceShutdown,...) (Of course all of the tools are licensed under the GPL)
One of the Systems goal is to be fairly runnable on a Pentium 200 MHZ with 128 MB RAM so that older computers are useful again. Useful is a good point. We are using a lot of GTK and Gnome based (mainly lightweigt) applications so that the system is easy to use and fast enough. Some of them are:

Epiphany (Gecko Browsing fast as hell ;) ) 
Sylpheed (also known as Claws-Mail) 
Quod-Libet (Music Management the easiest and fastest way)
Ristretto (Fast and Easy photoviewing)
Brasero (Super easy burning application)
Geany and Gambas 2 (for development experts and beginners)
Abiword and Gnumeric (Fast and Easy to use Office Apps)
Avidemux (Easy video editing)
Mplayer (Playback 'anything knowable') 

So test out the new ZevenOS and send us your feedback. You can try it without installing, because its coming on a 688MB ISO LiveCD Image. 
If you want to help our project please kontact us. (we also speak english, even if the website is not completely translated yet. Simply post in our forum :D ) 

And don't worry we won't go down so soon, as we are alive for more then 6 months now :P :lolflag:

ZevenOS Beta 6 Update is ready for Beta 5 with a lot of new features and some changes (see list below). Enjoy :)
