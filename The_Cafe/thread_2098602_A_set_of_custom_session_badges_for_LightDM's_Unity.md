### Post by zombifier25 on 2012-12-27
By default, LightDM's Unity Greeter displays icons for the selected session (an Ubuntu logo for Unity for example). But the greeter only came with 3 icons by default for Ubuntu, GNOME and KDE (4 if you count the mysterious "recovery session", and more if you count unofficial icons scattered in various desktop environments, such as Xubuntu and Pantheon), which for some are not sufficient.

So, after knowing [how to create custom badges for unity-greeter](http://askubuntu.com/questions/121352/how-to-change-circular-icon-to-indicate-chosen-session-in-unity-greeter), I decided to create a set of badges for DEs and WMs used in Ubuntu and share them with you guys, from the most popular to the really obscure (as long as they have icons). Also included as a bonus are badges for Windows and Windows 8 (if you, like me, run virtual machines in a custom session), and [ChromeOS](http://www.omgubuntu.co.uk/2012/11/how-to-run-chromeos-in-ubuntu). This is the link to download:

[http://ubuntuone.com/4PbkiHdiQOet6WNfz4AtXx](http://ubuntuone.com/4PbkiHdiQOet6WNfz4AtXx)

To use it, extract the archive and copy all the images inside (as root) to the folder /usr/share/unity-greeter .
So I hope to receiving feedback from you guys. I'm in no way a graphic designer though, I'm just doing something everyone can do, but nobody does.

NOTE 1: Because me and my slow Internet are not really in the mood of "downloading a couple dozens on WMs to my machine to test", I need someone to test this out for me and see if the badges look nice. A screenshot is appreciated :)

NOTE 2: I'll definitely have missed some DEs or WMs (while introducing a bunch of WMs that almost nobody used)
