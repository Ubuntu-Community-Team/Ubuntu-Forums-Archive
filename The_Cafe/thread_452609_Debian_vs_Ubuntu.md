no, not really. 

Ubuntu is a compromise between the bleeding edge--represented in the Debian world by Debian Unstable (Sid)--and the stodginess of Debian Stable.

So why not use Debian Testing?  Well, until fairly recently, the Debian release cycle has been positively glacial:  features would arrive in Unstable but not move to Testing for ages;  Testing took even longer to move to Stable.  This is all very well and good for server admins, but not so much fun for desktop users.  Desktop users, after all, have different needs:  oddball devices--mice, displays, and the like-- power management, etc.  

In general, I like to think of Ubuntu as Debian On A Deadline.  Semiannual, predictable releases, incremental change, and none of the vicious Debianista elitism.

### Post by KIAaze on 2007-05-24
> **tompickles said:**

One very nice thing abount Ubuntu is the Desktop Effects. Is this a Ubuntu thing, a Gnome thing, or a Debian thing - or sumthing else entirely?!

The desktop effects are available for all GNU/Linux distros. ;) (except maybe some special small ones or other)

There are two different 3D "window managers":
-[Compiz]("http://en.wikipedia.org/wiki/Compiz")
-[Beryl]("http://en.wikipedia.org/wiki/Beryl_%28window_manager%29")

The one used by default in Ubuntu Feisty Fawn is Compiz.
Beryl started as a fork of Compiz and has a lot more effects than Compiz, and a lot more configuration options.
However it is generally considered more unstable (but it is quite stable and unstable effects can be disabled ;) ).

As of March 30th Beryl and Compiz have decided to merge their projects back into one. Much work has yet to be done, but the beryl forums are no longer active as the project is being shutdown in favor a new merged project. The Compiz team has decided to work with the Beryl team and implement many of the changes beryl made. (that's from the wiki ^^)

Both window managers can then be used either with [AiGLX]("http://en.wikipedia.org/wiki/Aiglx") or [XGL]("http://en.wikipedia.org/wiki/Xgl") depending on the hardware.
You should find out by yourself which one performs better for you.
For me it was Beryl with AiGLX. However the default Compiz now also works OK for me, but I don't know if it uses XGL or AiGLX.
