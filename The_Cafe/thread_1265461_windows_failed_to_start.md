### Post by NormanFLinux on 2009-09-13
Look in your bios and see if ACPI is enabled. If it isn't, put a check beside the box. Then see if Windows starts. If not - use your install disk to perform a repair. As a last resort, reinstall Windows.

### Post by NormanFLinux on 2009-09-13
Are you running Windows Vista? You may need to download and burn a recovery disk to make Windows bootable.


[http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/]("http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/")

A recovery disk isn't a Windows installation -it lets you recover from an error that prevents Windows from booting. And if you can't boot Windows, you can't perform a repair or do an install.

Hope this helps.
