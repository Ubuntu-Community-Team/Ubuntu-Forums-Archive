### Post by Duncan J Murray on 2009-11-13
I had an idea which I've posted onto the ubuntu-brainstorm website - how about a tea-coloured theme for ubuntu?

After Karmic came out, I did feel sorry for ubuntu getting stick for it's brown theme.  However, I think ubuntu should definetely stick with the colour it was born with, and use it to mark it out from Windows and OS X.

After all, there are lots of 'nice' brown colours around (even if there are possibly a greater number of less nice browns as well).

This was one I found off the internet which I quite liked.  There are plenty on google images.

[CENTER][IMG]http://i156.photobucket.com/albums/t24/HobbesOxon/TeaPourB.jpg[/IMG]
[/CENTER]
 
I was thinking of translucent glossy title bars, with those range of orange hues that tea has.

I think the best thing about it is the way tea changes it colour all the way from a light yellow, through to a deep orange, and then deep reds and browns as it gets stronger, providing plenty of different hues for various visual touches in ubuntu.

If you like this idea, then vote for it below, and then maybe we'll see it in Lucid Lynx!

[/B]All best,
