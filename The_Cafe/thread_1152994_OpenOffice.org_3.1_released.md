### Post by jespdj on 2009-05-08
It will be available in the next version of Ubuntu in October.

Don't count on the repository for 8.04, 8.10 or 9.04 to be updated. Only security updates and important bug fixes are added to the repositories for existing Ubuntu releases.

Ubuntu is not a [rolling release](http://en.wikipedia.org/wiki/Rolling_release) distro.
