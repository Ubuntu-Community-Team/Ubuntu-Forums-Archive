### Post by Toz on 2011-09-21
I'm also looking into this. **googlecl** exists in the repositories. It's a command-line utility that allows you to interact with Google Docs.

```
This package provides a user-friendly command-line interface to
 some of the Google Data Protocol (gdata) APIs
```

For more info, see: [http://code.google.com/p/googlecl/]("http://code.google.com/p/googlecl/") and [http://code.google.com/p/googlecl/wiki/Manual#Docs]("http://code.google.com/p/googlecl/wiki/Manual#Docs").
