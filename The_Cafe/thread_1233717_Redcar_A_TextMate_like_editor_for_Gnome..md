### Post by theApokalypsis on 2009-08-07
I have been trying out various editors for various languages and scripts/markup such as Geany, Bluefish, even Aptana. I came across Redcar, a infant-stage programming editor under development written in Ruby/Vala with many TextMate (Mac) like aspects. Some great aspects

- tree style file manager
- huge language bundles(ex Rails commands) and highlighting.
- Lots of themes including, for anyone that watches Railscasts, the sweet textmate theme Ryan uses in his casts is included as well
- tons of built in snippets for many languages

[http://redcareditor.com/](http://redcareditor.com/)

I have been using Geany so far, but it has some good potential and I figured worthwhile posting about on the forums. Got it running nicely on 9.04 from instructions here

[http://github.com/danlucraft/redcar/blob/eeebf739365d8bfd0e06ed001bd6b7960d76daa3/INSTALL.md](http://github.com/danlucraft/redcar/blob/eeebf739365d8bfd0e06ed001bd6b7960d76daa3/INSTALL.md)

[http://ubuntuforums.org/showthread.php?p=7746834#post7746834](http://ubuntuforums.org/showthread.php?p=7746834#post7746834)
