Hello everyone,

I recently brought an old PC out of the closet and loaded it up with Hardy. The installation worked great and I've got everything updated, but now there is just one nagging question on my mind:

What can I use this old pc for anyway?

It is a 900MHz Celeron Coppermine with 512MB of RAM and a 30.0GB hard disk. It has an integrated graphics chip by Intel. As you can see, it doesn't have the greatest specs, but I'm convinced that there must be some way to put this hardware to use.

For what do you, the Ubuntu forum goers, use your own old PCs? I'm especially interested to hear about your own setups or zany projects. Have you upgraded an old machine for a project? Are there ways to tweak Ubuntu for better performance on older hardware?

### Post by issih on 2008-06-12
Install mediatomb and use it as a upnp server for music/media.

Throw a dvb card in and set it up as a myth tv backend (this will work happily on even very modest hardware with dvb as there is no need for video encoding).

Grab a nice big harddrive (stupidly cheap at the moment) and set it up as a nas box for everyone else on the network, you could even use nfs if its a totally linux setup.

Basically anything you can think of ....chances are it will basically be used as a server of one form or another, but its good to have the option to play about huh :)
