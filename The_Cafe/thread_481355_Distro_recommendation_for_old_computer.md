### Post by mig5 on 2007-06-22
I'm running Xubuntu with 512mb memory and 2500 processor (1.8ghz core), it runs very fast.  However it takes quite a while to start up (over a minute), but I think that's more to do with the fact that most of the startup is identical to Ubuntu.  Once you get to X windows the XFCE startup is about 5 seconds.  I also looked into Fluxbuntu, as apparently they have taken out some of the Ubuntu startup stuff that slows things down.  However Fluxbuntu is still in test builds at the moment so I decided to put it off for a while.
I don't recommend Puppy Linux, I tried a version once and it seems to run everything as root.  I also like Damn Small Linux, but I couldn't get it to work my wireless internet, so I gave up on it prematurely.
