### Post by forrestcupp on 2012-09-15
Lotus Symphony has some pretty awesome features. 

The main thing I don't like is that everything uses the same interface, so you can't set your custom window sizes for different things, like the spreadsheet and word processor. I like my spread sheet window to not be maximized because I mainly use it for a custom budget sheet that doesn't take up the whole screen. With LS, everything uses the same interface and window.

But it really does have some good features in the word processor. It's probably worth the download to check it out.
