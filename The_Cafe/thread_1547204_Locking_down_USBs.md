### Post by t0p on 2010-08-06
Check out **[Truecrypt]("http://www.truecrypt.org")**  It can be used on multiple OSes (Linux, Windows, etc) and offers a function whereby you encrypt the contents of a USB drive, which you can then decrypt at your leisure, using pretty much any computer.


Check out the docs before you embark on the project.  It makes pretty good reading.
