### Post by scragar on 2010-04-02
Lights out has been added to the debian gnome-games, so I'm assuming it won't be long till it finds it's way into the ubuntu repos, in preparation I've generated a small list of fairly common points in the game where you can get a bit stuck on where to go from, if you have any more feel free to post them in response, it would be nice to have a catch all location for help.

We'll start off easily with some basics to make sure people can understand the notations:
[SIZE="2"](X is turned on, green is to be clicked, # is blacked out)[/size]```
[b] [COLOR="Green"]X[/COLOR] X # # #
 X # # # #
 # # # # #
 # # # # #
 # # # # #
==============
 # X [COLOR="Green"]X[/COLOR] X #
 # # X # #
 # # # # #
 # # # # #
 # # # # #
==============
 # # # # #
 # # X # #
 # X [COLOR="Green"]X[/COLOR] X #
 # # X # #
 # # # # #
[/b]
```
Those are quite easy, and after playing a couple of games you'll pick up a few more advanced tricks(learning to solve most problems), but these are tricks which take longer to learn:
