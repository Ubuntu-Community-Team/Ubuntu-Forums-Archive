### Post by Eclipse. on 2008-04-09
No, grumpy groundhog was the name given for a Debian unstable like version of ubuntu, which would have be basically bleeding edge.

Grumpy Groundhog was orginally going to be the first version of ubuntu but grumpy didn't really give the idea the devs were wanting.
