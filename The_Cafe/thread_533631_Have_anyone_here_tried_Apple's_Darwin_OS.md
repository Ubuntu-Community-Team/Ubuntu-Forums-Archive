### Post by tgalati4 on 2007-08-24
I imagine that you can run Darwin with XDarwin (or xorg-server or X86) and then run Gnome Desktop on top of that.

I have a powerbook running Tiger (10.4.8) and I run XDarwin in one window with gnome-panel remotely from an Ubuntu machine.  All of the panel applets seem to work.  Window management works well.  I've never experienced any crashes of the xserver while using it.

If they work over an ethernet remote ssh connection, then I imagine they would work locally as well.

Of course it doesn't have the polish of Tiger, but does it work well.  If I had older, PPC hardware, it's worth the $129 to put Tiger on it.
