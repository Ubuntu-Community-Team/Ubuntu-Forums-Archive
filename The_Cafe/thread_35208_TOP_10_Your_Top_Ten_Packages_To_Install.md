### Post by telemetry on 2007-09-09
my advice would be for the beginner in feisty fawn (like myself):

get a cable modem and ditch your USB one and your network is autuomatically detected by ubuntu..

ubuntu will ask to install updates which you should do.

then go to system/admin and enable restricted drivers to activate and install your graphics card.

Go to your terminal in accessories and type:

sudo apt-get update

install konqueror (best and fastest browser for feisty fawn)

enable and choose your plugins in konqueror settings/options such as flash player (manually install them in a directory which you can direct konqueror to in its settings/plugins) which is quite easy in konqueror as synaptic won't do it and will drive you mad!

Next: type:

sudo apt-get install firestarter 
     
This will secure your internet behind a firewall...go to admin and configure firestarter once installed.  Google: firestarter configuring to find out how.  Then go to sessions and add firestarter to sessions startup to enable it everytime ubuntu loads up.  Add firestarter to sessions by simply googling: configure firestarter in ubuntu.  This will bring up lots of guides to give you the simple command line that will reload firestarter  (real easy).

it took a lot of work to find all this out and I hope it helps someone.

### Post by eksasol on 2010-03-22
Here are some small apps since the OP said '10 packages'. These should be useful for newer users, the names are in the form that would work with apt-get install.

**compiz-settings-manager**: This install both the advance and simple compiz settings manager.

**gnome-app-install**: It's the old "Add/Remove Applications" software center interface that came before version 9.10. Some people might prefer this version because you can select multiple apps at once, instead of installing things one by one.

**gparted**: Speaks for itself.

**gnome-alsamixer**: its a GUI for alsamixer, similar to Windows XP sound preferences screen. This will let you control your sound card much more than the default Sound Preferences. I prefer just alsamixer, but this one is more user friendly.

**ndsiswrapper**: Lets you install Windows drivers for your network card it failed to work with the linux driver.

**p7zip**: This should be included with Ubuntu by default.

**recordmydesktop**: A god-sent screen caster/desktop recorder app. Also no, I still couldn't get sound working.

**ubuntu-tweak**: It's sort of like a GUI alternative for gconf-editor.

**uck**: "Ubuntu Customization Kit", a GUI to remove and add extra packages to the Ubuntu ISO. You can remove extra languages and apps you don't need and add in apps you wants come preinstalled. The app can get iffy at times though, sometimes I had to uninstall and reinstall it to get it working again. It also require 5gb free space in your /home partition to extract all the files. Very useful if you want to slim down Ubuntu for netbooks.

**testdisk**: Help to recover lost data from deleted or formatted partitions. This package also installed "photorec" which is designed for extracting specific files types. Only downside is they don't come with a nice GUI so this is more for advance users. I'm interested if anyone know a GUI for them.

**wine** + **playonlinux**: speaks for themselves.

**unetbootin**: Let you create live USB of many different linux distros. A very, very useful tool. Although the one in the repository is not up to date (unetbootin.sourceforge.net)


Others:
**avidemux**: It's my favorite video editor/converter, though more suitable for intermediate and advance users. The only thing that kept this app from being complete is ability to encode Ogg Theora.

**gnome-color-manager**: System color management, useful for people with 8bit/wide gamut monitors or does artworks and printing. You'll still need a colorimeter to create a color profile for your monitor though. This should be included with Ubuntu by default.

**pidgin-otr**: It allows encrypted messaging between you and your buddy while using Pidgin, but you have to turn it off if your buddies don't also have pidgin-otr or they'll just see random characters.

**winff**: It's a GUI for ffmpeg, an extremely useful apps for batch converting audio and videos. However, you will have to install medibuntu packages to convert some videos.

**sl**: Display an animated steam powered train when you typed sl.

For FTP, I use the Firefox addon FireFTP.

This is the script I run after a new install:
```
#!/bin/bash

# My standard install packages.

# Filename: install.sh

# This script must be called as root.

apt-get update
echo y|apt-get install human-theme rdesktop
echo y|apt-get install gparted
echo y|apt-get install conky
echo y|apt-get install dopewars
echo y|apt-get install xserver-xephyr
echo y|apt-get install pyrenamer
echo y|apt-get install vlc
echo y|apt-get install mplayer smplayer smplayer-themes
echo y|apt-get install ubuntu-restricted-extras
echo y|apt-get install dvdstyler

# Other packages I might want to install
apt-get install wakeonlan
# echo y|apt-get install build-essential
# echo y|apt-get install gimp
apt-get install ethtool
echo y|apt-get install openssh-server


# Remove Ubuntu One and related packages
echo y|apt-get purge indicator-messages ubuntuone* couchdb*

# Install localepurge last, if installed earlier 
# it runs after every line in the script.
# It saves space on small hard drives by removing language packs.
# Very slow to run and called after every install/update,
# often multiple times.
apt-get install localepurge

# and cleanup
apt-get clean
apt-get autoclean
apt-get autoremove
localepurge
```
