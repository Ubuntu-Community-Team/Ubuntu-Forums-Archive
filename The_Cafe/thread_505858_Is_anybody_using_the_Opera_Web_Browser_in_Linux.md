```
sudo apt-get install polymer qt3-qtconfig
```

Launch qtconfig-qt3

Select polymer as your gui style.

N.B: you can just use [this method]("http://my.opera.com/Tamil/blog/tab-bar-below-address-bar") instead.

### Post by Akovia on 2008-02-29
I wanted to post a couple links that might help those who either are considering Opera, or are new to using it. When I first started using it I almost fell into the same trap as I see others are. Opera doesn't do enough to draw attention to the multitude of features it offers, and too many times people just assume it can't do it. I suppose they don't want to overwhelm new users and try to streamline it to "just work" out of the box. 

I have to admit when I started using it (not really by choice) I was turned off by the default theme, (ugly) and the weird toolbar that popped up from the address bar.(Start Bar) It came packaged as the default browser on the BartPE rescue disc I was using so I figured I'd bear with it. Once I realized I'd be using it I decided to read up on it some so I could get the most out of it. Long story short, once I realized that Opera had every single Plug-In I had to install manually every time I reinstalled Firefox built right in I was sold. Even though it has more features built in than any other browser I know of, it is a smaller download and one of the fastest browsers available.


Here's a link that summarizes the main benefits and features that alot of people are unaware of.
[COLOR="DarkOrange"]http://www.howtocreate.co.uk/operaStuff/aboutOpera.html[/COLOR]

That site has the link to the wiki which I also highly recommend. The opera site itself is of course a great source of information as well.
[COLOR="#ff8c00"]http://www.opera.com/products/desktop/features/[/COLOR]

Looking back on how I almost never gave Opera a chance since I was turned off by its looks (How shallow of me) I wanted to add a tip to change the skin.

R-Click on any toolbar and choose _Customize_
Click the _Skin_ tab if it's not already selected.
Click on *Find more skins*
Select a skin to try out (Highly recommend Shadowguard!)

Everything is fully automated so try as many as you like. When you click on a skin it will automatically apply it and then ask you if you want to keep it.
