### Post by Whistling Nixie on 2011-03-04
> **na12 said:**
> What app you using for top panel menu?
Right-click on the GNOME panel.
Select *Add to Panel...*>*Application Launcher*, and click on *Forward* at the bottom of the same dialog box.
Choose *Accessories* then click on *Add* down the bottom. Repeat the exercise for *Games* through to *Administration*. 

To get rid of the arrows, open up gedit. Paste the following:


```
style "panel-arrow-remove"

#the following removes the arrows from the panel
{
engine "pixmap"
    {
    image
    {
        function    = ARROW
        recolorable    = TRUE
        overlay_file    = ".arrow-blank.png"
        overlay_border    = {2,2,2,2}
        overlay_stretch    = FALSE
        arrow_direction    = UP
    }

    image
    {
        function    = ARROW
        recolorable    = TRUE
        overlay_file    = ".arrow-blank.png"
        overlay_border    = {2,2,2,2}
        overlay_stretch    = FALSE

 */
@namespace url("http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul"); /* set default namespace to XUL */


 
#appmenu-toolbar-button {
  list-style-image: url("/home/hhh/Pictures/menu.png");
}
#appmenu-toolbar-button > .toolbarbutton-text,
#appmenu-toolbar-button > .toolbarbutton-menu-dropmarker {
  display: none !important;
}

.tab-icon-image {display: none !important;}

.tab-text {margin-left: 2px !important;}
```
