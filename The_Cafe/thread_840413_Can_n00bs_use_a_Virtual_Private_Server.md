### Post by FFighter on 2008-06-25
It is not a big deal, really. Know your IP(s) and you will be fine :D. When using VPS, you will mostly be accessing via SSH. Of course, you can setup a FTP server and access specific folders through it, or even use scp. I started using VPS when I started programming in Rails. Deploying using Capistrano is sublime, so I kinda left FTP in the ashes and rarely use it anymore (only for very specific cases).

Don't be afraid to go VPS. You will only have more control and power. 

The only thing you should be aware is that you'll never want to use shared hosts again.

VPSLink is a nice company. If you screw up your VPS, you can re-install an OS image in a couple of seconds and some of them even have pre-configured componentes such as web server and other things.

Come back when you have more specific questions ;)
