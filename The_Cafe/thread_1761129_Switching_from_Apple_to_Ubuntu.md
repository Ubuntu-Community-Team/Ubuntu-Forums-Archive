@kaldor: Thanks for asking.  I actually just bought this machine. Currently downloading the ISO image for Pinguy OS.  Waiting for the final stable version of Linux Mint 11 to be released.  If I don't like either, I'll put in Ubuntu 11.04.  I was glad to see that now Ubuntu can get the 3rd-party restricted extras during installation.  

Although I've used Linux since 2007, I still consider myself a noob. :-k  So I lean toward the "out-of-the-box" distros.

We'll see how it goes.  My mind went numb from all the research I did into Linux compatibility.  Checking brands wasn't enough, I had to look at every component too.  This machine is all Intel: processor, wireless Internet card, and graphics card.  Seemed the safest way to be friendly with Linux.  Not having a high-powered graphics card wasn't a huge sacrifice, since I don't do gaming.

FYI: it's the **Asus Bamboo U43F model **sold by Best Buy.  The pricier bamboo versions (also sold on Amazon.com) have Nvidia cards.  Hope this info helps!
