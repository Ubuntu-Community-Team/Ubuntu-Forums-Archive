### Post by Bloodfen Razormaw on 2007-03-03
I wouldn't want to use either of them.  Look into Kolab as it will provide more integration to your end users, and gives you the peace of mind that it doesn't have PHP's infamous lack of security to contend with.  OpenGroupware.org is another option that you should look into.  The only catch to OG.o is that it has no e-mail server by default.  Then again, you didn't say you needed e-mail, so maybe that works for you.  Both Kolab and OG.o can be used from your colleagues' Windows machines if they use Outlook, using proprietary connectors.
