### Post by PurposeOfReason on 2009-05-05
Depends what frontend you use. Some decode with the GPU, others the CPU.

Best bet is get some mAtx board, throw in the cheapest dual/quad core you can. Get a good quiet heatsink. Make sure the board has some good integrated video, and about 2GB ram. You're set.

EDIT - Wait, you want a server that streams to a different box that plays the videos? CPU power is all that matters then. 2.5Ghz or higher dual core will be fine.
