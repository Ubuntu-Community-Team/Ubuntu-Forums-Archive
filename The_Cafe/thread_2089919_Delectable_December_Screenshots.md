### Post by LillyDragon on 2012-12-01
Gotta love long exposure photos, and that minimal desktop. ^^

Here are my desktops. Just conveniently decided to play with Unity a while ago, and now that I've played with it in GnomeTweak, MyUnity, and CompizSettingsEditor, Unity is, gasp, just as customizable as I like. ^_^ I can change the window borders and the theme independent of each other using any of my favorite GTK2/3 themes, and I have transparent windows setup too, so I'm quite happy with it right now. The only thing I wish it would do is have wallpapers for each desktop!

[[IMG]http://th08.deviantart.net/fs71/200H/f/2012/299/7/6/767612c7bfda8b0258fbf799fe7a23f4-d5j2jdl.jpg[/IMG]]("http://i1058.photobucket.com/albums/t410/johnluke728/XubuntuDesktop8.jpg")

**Xubuntu 12.04 LTS**

**Window Manager** : XFCE
**Window Borders** : Kikodi
**Desktop Theme **: OverglossedBlack on XFCE-Look.org
**Icon Theme** : Humanity Dark
**Taskbar Borders** : Vista Graphite Bars
**Programs** : XFCE extras, and Wallpapoz, for wallpapers on each virtual desktop.
**Wallpaper** : Solidarity, by `Rahll on deviantART - [http://fav.me/d2vqkhs](http://fav.me/d2vqkhs)
(That's just my wallpaper for the Workspace, Vocaloid fan art spans my other virtual desktops. =P )
**Additional Changes** : Messed around with the panel settings until it was just like Ubuntu on Gnome 2 back in the day, with some extras, so I'm quite happy with that.

[[IMG]http://i1058.photobucket.com/albums/t410/johnluke728/th_UbuntuDesktop5.jpg[/IMG]]("http://i1058.photobucket.com/albums/t410/johnluke728/UbuntuDesktop5.jpg")

**Ubuntu 12.04 LTS**

**Window Borders** : Crunchy Orange
**Desktop Theme **: Ambiance
**Icon Theme** : Ubuntu Mono
**Wallpaper** : Obscure, by *JoeJesus on deviantart - [http://fav.me/d360etw](http://fav.me/d360etw)
**Additional Changes** : Launcher set to auto-hide at 0.3 milliseconds of mouse-pushing for reveal, (Something I always wished Windows would do!) and launcher icons set to 32 to display additional icons without clutter. Also used Gnome-Tweak to put the Trash bin there, and created a Nautilus file manager launcher disguised as a "My Computer" icon. Which is recognized by both my XFCE and Unity desktops as executable.

Also enabled Opacity with Compiz Config, so I can make windows transparent with the scroll wheel on my mouse easily. ^^ I really want a traditional workspace switcher on the top panel, but the Ctrl + Alt + Arrow Keys shortcut gets me around almost as fast.
