### Post by NMFTM on 2010-05-07
In all honesty, I don't think doing commands (or anything computer related) from a book is really a good way to learn. I can only speak for myself, but I can't really learn anything about computers unless I can somehow apply it to something I want to do in real life.
 
Whatever it is that you do in the GUI, learn how to do it with the CLI. For example, VLC player comes with a CLI app called cvlc that you could use in place of the GUI version.
