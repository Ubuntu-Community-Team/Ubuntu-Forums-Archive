### Post by chessnerd on 2010-05-19
All of those distros are time consuming ones. They are good distros and I'm sure they are fun to set up (I'm hoping to set up an Arch system later this summer) but I have another idea:

Do the DEW (Distro Every Week)

Every week or two, pick a distro, and install it and mess around with it. Have a home partition installed so that your personal files remain constant. This will likely be a lot of fun because you will experience constant change. Also, it will only take a bit more time (overall) versus setting up and maintaining the other distros you mentioned.

It would teach you about the various ways that Linux is put together and allow you to then learn and, later, make recommendations about various Linux distros. Also, if you are unsure if Ubuntu is right, this could be an easy way for you to find the distro you've been hoping for...

A possible order could be this:

1. Fedora
2. TinyCore
3. PCLinuxOS
4. Slackware
5. OpenSuSE
6. Gentoo
7. Mint
8. sidux
9. Debian

That's over 2 months worth and you could obviously go on. I suggest this order so that you have a wide level of variety from week to week. Going from the typical, Red Hat-based Fedora to the ridiculously small TinyCore, from the easy-going Mint one week to the tougher, bleeding-edge sidux the next.

I haven't used all of these distros, so there is probably a much more interesting order, but the idea is still there for you to kick around...
