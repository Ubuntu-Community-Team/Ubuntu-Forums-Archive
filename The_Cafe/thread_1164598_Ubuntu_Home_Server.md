### Post by MaxIBoy on 2009-05-19
So, you if I understand you right, you're talking about some software which will automatically set up your server hosting software? Some kind of autodetect and autoconfig program? A front-end for graphically editing config files? 

Personally, SSH is the only tool I need for remote administration on my home server. I actually have no idea if a program fitting your description already exists. 

One thing that would really help is if someone wrote some good, up-to-date documentation for existing tools. Setting up the home server for the first time was something of a nightmare, but not because the command-line tools and config files were inherently difficult to use. On the contrary, if I had to do it all over again, I'd probably have it up and running in a half-hour at most. 

I just didn't know what was involved. It's like thinking you need a hammer to tighten a bolt. It doesn't matter how user-friendly the bolt is-- if it's not the right tool for the job, you're going to get frustrated, and if there's no source of information to clarify things, you're going to stay frustrated.

Just my two cents.
