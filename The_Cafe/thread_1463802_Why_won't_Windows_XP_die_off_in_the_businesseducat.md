### Post by kaldor on 2010-04-27
Here in Newfoundland, *everything* runs on Windows XP except for most of the servers which seem to like Solaris and RedHat.

XP is old now. It's getting painful to use now that I'm used to using Snow Leopard and various Linux/*NIX distros. Windows 7 is even a big improvement over the old software on XP.

I do not think money is an issue; the schools always buy the latest and greatest software for Windows XP when it is released.
