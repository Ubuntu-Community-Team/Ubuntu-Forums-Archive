yes ubuntu allows you to use passwords.  thought i think a more accurate telling is unbuntu requires you use a password(which is a good habit to get into).  what you have to do is type in your username and then after that it asks you to enter your password.  so what it does is changes the username box to a password box.

when you install ubuntu you have to enter a username and password, so make sure you don't forget what they are when you install.  if you have usernames and passwords set in winxp, it is somewhat similar to that.
