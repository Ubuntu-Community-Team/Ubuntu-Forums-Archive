### Post by rsambuca on 2007-10-29
The meta-package 'ubuntu-desktop' is the set of core packages/programs that the Ubuntu developers consider the standard ubuntu desktop installation.  By removing one of these core packages, the 'ubuntu-desktop' will be removed, but it doesn't really affect anything else as it is just a meta-package.  Do not confuse the  'ubuntu-desktop' meta-package with your Desktop Environment and Window Manager, which is Gnome.
