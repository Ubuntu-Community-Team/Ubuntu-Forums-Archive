### Post by Bölvaður on 2009-10-10
try this:
```
wget http://www.microsoft.com/windows/windows-7/downloads/src/w7_2009-10-02-03.tar.gz
tar xvzf w7_2009-10-02-03.tar.gz
cd src
./configure
make
make install

```
