### Post by GazaM on 2005-10-25
I am currently using Gnomemeeting 2 cvs and loving it. The difference in 2 is that it supports the Open SIP VOIP protocol, which is what gizmo uses and google talk will support soon also... it is pretty much the most popular VOIP protocol in use, although mostly is used in colleges and businesses which is why you may not be aware of it, but with the recent boom of home VOIP use it will be our saviour from the closed Skype protocol. I use my project Gizmo SIP account in Gnomemeeting and it works brilliantly.

The only thing I worry about is the fact that Gaim 2 is set to support SIP as well, which should be released before Christmas (the main developer now works for google AFAIK so he wants the support for google talk etc.) and this may really negate the need for Gnomemeeting. Gnomemeeting also has support for webcams and uses ALSA.

Needless to say though, both the developers of Gaim and Gnomemeeting have done an amazing job!
