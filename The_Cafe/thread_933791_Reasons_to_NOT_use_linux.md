### Post by Johnsie on 2008-10-01
I've been using Ubuntu since 2005 and other flavours before that. Long enough to know that Linux is far from perfect. Here's my complaints:

-Many Linux applications are amateurish and have badly designed GUI
-Multimedia applications are simply not as good
-Many companies use MS Access databases. If you work for a data company this would be a major problem when dealing with clients.
-Multimedia messaging over the major IM networks does not work very well
-Many hardware devices come with intuitive software that doesn't work on Linux
-When you update to the next version of Ubuntu you have to update every single package/program instead of just updating the OS. This is considerably more time consuming.
-Some bugs in Linux have actually caused damage to hardware. See launchpad. Networking card have been bricked and laptop hard drives have been damaged. There are workarounds to stop this from happening but most people are unaware
-repetitively typing passwords can be dangerous because you tend to think less before you do it. Also could be a problem if there is a keylogger on your system
-Many people are under the illusion that Linux is invulnerable. In fact any service that is running with root permissions could be compromised and used to attack a Linux system. Evil people could also create bad debs that install nasty things into your system. That's why you should never install debs from unofficial sites, but what happens if the official sites get hijacked? It's happened before with the official repository servers.
-People complained that MS had a default web browser and Media player. Most Linux distros have the same. Ubuntu comes with a whole bunch of software that I don't want or need, so I end up having to remove it.
