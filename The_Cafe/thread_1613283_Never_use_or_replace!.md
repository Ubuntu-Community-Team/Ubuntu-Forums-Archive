### Post by mcduck on 2010-11-04
> **3Miro said:**
> I disagree with using QT apps under a GTK environments as defaults.

I have never in my life used Empathy. Remove Evolution or at least don't tie it to gnome-panel so people can delete it. Also, I don't see why Avahi is installed by default.

There are no Qt apps on Ubuntu's Gnome by default. And I can't remember seeing a single GTK app by default on Kubuntu either...

You might want to actually try Empathy so you can know if you like it or not. (personally I liked Pidgin a bit more, but prefer Empathy for it's better integration with the desktop.)

Evolution isn't tied to gnome-panel, and can be safely removed. Gnome-panel, on the other hand, has one Evolution library as it's dependency. So all you need to do is to not uninstall that single library. (simply uninstalling Evolution will not uninstall that library anyway, so you'd only get problems removing Evolution if you'd blindly mark everything with "evolution" in it's name for removal.)

Avahi is included by default because it's integrated to GVFS. And it's also used by some included apps, like Rhythmbox. Besides, I really can't see a reason why such functionality *shouldn't* be available on a desktop OS by default.
