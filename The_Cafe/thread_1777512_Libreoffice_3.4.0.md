### Post by sandyd on 2011-06-07
Libreoffice 4.0 released.

Grab a copy at [http://www.libreoffice.org/download/](http://www.libreoffice.org/download/) (Early Adopters)

Remember to remove the libre/openoffice from ubuntu if it was installed from the ubuntu repos before installing the downloaded libreoffice.

If your lazy, cd to the folder where all the DEBS were extracted to (after downloading libreoffice), and run
```

sudo dpkg -i *.deb
```
to install all the debs in the directory at once.

There may be a seperate deb for the desktop integration in the 'desktop integration' folder. There was one in the OpenOffice binary packages in the past, but I don't use ubuntu anymore, so I don't know if libreoffice has it or not.
