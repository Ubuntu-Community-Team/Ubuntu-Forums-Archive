### Post by luoihoc on 2008-10-27
This addon contains all the best functions of some other dictionary addons for firefox. I have read some comments of users to these addons, so i try to write D4G so that it is easy to use and has useful functions. As you can see in the name of this addon, this dictionary addon is for geeks (i mean computer geeks), because you can add more dictionaries by using javascript. But normal users can still use D4G because there are a lot of high quality default dictionaries.
[img]http://img60.imageshack.us/img60/3194/snapshot1jp2.png[/img]
Some features of D4G:
- Trigger the translation via combination of mouse and keyboard (Ctrl/Alt/Shift + Left/Right mouse)
- Translation result showed at mouse position.
- Open dictionary data : this function based on the same function of OAK dictionary (you can still find it here but no support for FF3) . You can write javascript parser to filter the translation from websites.
- Support pronunciation : it's open too, that means, you can add your own pronunciation by using javascript to parse sound from web.
- Translate word in search bar : type word in search box of firefox and click button END on keyboard to translate.
In these pre-release version i used some icons and code of OAK. I just want to concentrate on the usability and functionality of D4G.
I really wait for feedbacks, so that i can change D4G and release the final version soon.
All bugs, new dictionary parsers and so on please post in this Google Group : [http://groups.google.com/group/dictionary4geeks](http://groups.google.com/group/dictionary4geeks)
Home page of this project: [https://sourceforge.net/projects/d4g](https://sourceforge.net/projects/d4g)
I will try to write a tutorial for parsering translation and pronunciation. Just wait.

I have submitted this addon to Mozilla and it is in sandbox. But you can still try this add on here
