### Post by macogw on 2008-03-23
It can make SVG or PNGs of the image-type charts (like the radial ones) and HTML pages for the "reports"

The report you'd want is Reports -> Web Page -> Narrative Web Site

The first page has a list of surnames.  Click a surname and choose a person.  It'll show their name, birthdate (which can be just "restricted" if they're alive to protect against identity theft) and parents, then there's a pedigree section under that, and below that is an ancestor chart.

### Post by Reverendspam on 2008-07-03
If you want other folks to also work on your family history / genealogy I would suggest using a program called phpgedview which will run on a webserver.

There is also a commercial program that is similar called TNG which I use on my own site.

You can also set these up to work locally on your home machine, but this could be a pain if you are not familiar with setting up a linux server environment.

If you have an ISP they will prpbably help you set up phpgedview or TNG. IT's a matter of uploading the program to your server space and running the configuration file.


The beauty of these programs is that other researchers can uplaod their gedcoms to your server and work on the same lines.

You can use Gramps locally on your desktop to keep a database of your info then upload the gedcom to phpgedview or TNG. 

The gedcom file is a specific format that most every genealogy program adheres to so those working with windows genealogy programs can also create gedcom files and uplaod to your website that is running phpgedview or TNG.

Good luck.

Here is my web site using TNG genealogy software  [http://mullismelange.com/tng/joe.php]("http://mullismelange.com/tng/joe.php")
