Now this opens up an interesting conundrum since Ubuntu is free to the masses therefore they should be entirely free to use whatever they like from Ubuntu.

The ubuntu logo is a trademark. Just because the source code of the software is open, doesn't mean everything on an Ubuntu LiveCD has a totally permissive licence. Firefox also has trademarked branding, for example.

So yes, these people are infringing on Ubuntu/Canonical's trademarks, and should be bodyslammed with the full force of the law. Or more to the point, they should sack whoever did their graphic design.
