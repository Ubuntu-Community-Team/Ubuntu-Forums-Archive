### Post by IYY on 2007-07-28
Has anyone tried the new Web2.0-esque website [Geni]("http://www.geni.com")? It allows you to very quickly and easily build your family tree, and also works like a private  social network for your family.

Plus, they now have Gedcom export, so after collaboratively entering your family data, you can always download it and keep it offline (there's a great genealogy program called Gramps for Linux, that supports Gedcom import).

I've been using this site for a few months, and now actually got a summer job doing interface design for them (my Linux skills really helped there). So if you have any feature suggestions, I'd be happy to hear them. :)
