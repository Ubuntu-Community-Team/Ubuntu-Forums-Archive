### Post by perspectoff on 2011-01-31
I hadn't realized that the iPhone only represented 16% of the smartphone market. Google Android #1 and Nokia Symbian #2, each have double the market penetration, about 32% each, of iPhone.

With all the hype about the iPhone, you would think it was a majority player, but it is actually 5th place.

iPad will go the same way, I'm afraid. The Samsung Galaxy is already growing exponentially. It's great to be first to the party (and I love a catchy name and to appear "trendy", too), but in the end I like something that works well and is affordable and which I can customize the way I like it. 

Suddenly it dawns on me why Shuttleworth is looking at making sure Qt is part of Ubuntu (and KDE, of course, has had the headstart in incorporating Qt into its framework). That's because Nokia's Symbian (which is Qt-based) has been the number one smartphone for a long time (Google Android only surpassed it this month).

To have vertical integration between Mobile OS platforms that use Qt and desktop/laptop/netbook platforms, Qt must be integrated into the desktop/laptop/netbook OS.

So a focus on Kubuntu (which uses KDE and Qt) is logical.
