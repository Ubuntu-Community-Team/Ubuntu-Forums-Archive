### Post by raublekick on 2005-11-16
I don't use anything but my wireless router to get online. I think the key you're talking about is [this.]("http://store.nintendo.com/webapp/wcs/stores/servlet/ProductDisplay?currency=USD&storeId=10001&jspStoreDir=NOASTORE&productId=95704&categoryId=11157&langId=-1&catalogId=10001&ddkey=SetCurrencyPreference")

The USB adapter will not work under Linux or Mac, but like I said, if you have a wireless router you should be able to get set up.

You can check router compatability [here.]("http://nintendowifi.com/consumerservice/routerSetup.do") Some of the routers require some tinkering, like I had to set some stuff for the 802.11b settings for it to work.
