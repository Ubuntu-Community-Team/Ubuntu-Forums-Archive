### Post by ice60 on 2006-10-23
this greasemonkey script makes youtube videos automatically large and stays on the original page. i've seen others in the past which open a new window, or work slightly differently. but i like this best!

[http://userscripts.org/scripts/show/3975](http://userscripts.org/scripts/show/3975)

for it to work you need to install the greasemonkey extension first, then click on the 'resizer' download link. greasemonkey will then popup and ask if you want to install it.
