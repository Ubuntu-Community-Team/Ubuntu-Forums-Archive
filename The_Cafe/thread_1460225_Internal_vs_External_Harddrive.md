### Post by Wee_Guy on 2010-04-22
It depends on what you want to use it for. If it's for backups, then it's probably best external (so if your laptop is stolen/damaged the backup will be intact), but if it's going to be used for storing stuff it would probably be best inside the laptop, especially if you intend to install an operating system on it. You should bear in mind, though, that another hard drive inside your laptop will likely have a detrimental effect on battery life.
