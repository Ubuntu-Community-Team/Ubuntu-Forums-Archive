### Post by Paqman on 2012-07-18
> **agnidev said:**
> can u list some features for me.i did not get that.

He's asking if you just intend to install different default applications or if you intend to modify and enhance something.

There's a lot of stuff you could do, but you'd probably score better and find it easier if you focussed on doing just one thing really well. You could build a good touch UI to make an embedded device to control something (Media streaming? Home automation? Industrial control?). Or you could make a distro optimised for users with a specific disability? Or one optimised for a certain set of tasks (pen testing has been done a lot, or you could do hardware stress testing, commisioning or maintaining a specific set of equipment, etc, etc)

Whatever you do make sure it fits the criteria for what your project will be marked on, but in general I'd say find one thing that you're interested in and hasn't been done to death and just focus on that. There's not much point IMO of just firing up Remastersys and producing a general desktop distro like Fedora or Ubuntu.
