### Post by SomeGuyDude on 2010-02-27
Sure thing! Here's what I call my "basic setup". Everything here SHOULD     be in the repositories. 

1) Install Exaile, VLC, GIMP, Deluge, and KDenLive. That'll take care of music, movies, image editing, torrents, and video editing (respectively).

2) Install every GTK engine you can find. That's what you need to change the theme. Specifically you'll want Aurora, Clearlooks, and Murrine.

3) Go to [www.gnome-look.org](www.gnome-look.org) and just look around. There are TONS of themes there. Also [www.4walled.org](www.4walled.org) for wallpaper. Just hit the "random" button and enjoy. WARNING: some walls are NSFW.

After that, just enjoy. There are other things you'll need to install to take care of various little behind the scenes things, but I'll let others who know what they're talking about more than me to handle it.

### Post by standingwave on 2010-02-27
You will find all of these  in the software center.

[LIST]
[*]VLC - video
[*] Audacious - audio
[*]GQview - fast image viewing program
[*]Filelight - view your file usage graphically
[*]Gnome Commander - an orthodox file manager I like
[*]Recoll - search within files
[*]Grsync - GUI for rsync that I use for backups
[*]Foobilliard - fantastic billiards game with good physics
[*]Gnu Backgammon - a very strong backgammon game
[*]Stellarium - planetarium program
[/LIST]
I like to replace the three panel drop down menus (applications, places and system) with the Main Menu, a single icon that gets you into the same menus. Or you can totally do without and just use <alt> F1. In either case, this frees up  space so I can run with just one panel. If you use Gmail you may  want to install  CheckGMail and add it to the panel.

### Post by gnomeuser on 2010-02-27
First of all, welcome to life full time on Linux. I hope you will find it both rewarding, fun and productive.

As for theming it depends on your preferences and is hard to give advice on, try out some different themes and see what works for you. I quite like the dust theme but in Lucid (the upcoming release) I have to admit that the polish of the current theme is excellent. The Application Indicator work has made a huge difference both in how I use my desktop but also how elegant it looks and feels.

Cool applications also depends on what you use and how you prefer going about working. E.g. if you like the approach gnome-do takes of giving you a sexy interface to search, start apps and what not then that is definitely for you.

Generally I tend to recommend switching to Banshee but please do use the banshee ppa the version in Ubuntu isn't kept up to date during the lifetime of a release due to policy which is a shame since Banshee continues to roll out wonderful new features on a regular basis. 

I also prefer Chromium to Firefox so I install that, in Lucid it is in the repos, on Karmic you need to use a ppa.
