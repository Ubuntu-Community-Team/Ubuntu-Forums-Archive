### Post by Kvark on 2005-11-01
Dunno about auto click but you can stop Firefox from popping up message boxes by doing this:
1. Type "about:config" in the address field.
2. Type "error" in the filter field on the about:config page.
3. Double click on the line called "browser.xul.error_pages.enabled" to change it's value to "true".
4. Restart Firefox.
