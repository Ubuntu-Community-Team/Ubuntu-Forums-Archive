### Post by t0p on 2009-11-20
Install the package "grokking-the-gimp".

```
sudo apt-get install grokking-the-gimp
```

Then go to /usr/share/doc/grokking-the-gimp/html/index.html and you'll find "Grokking the Gimp", a pretty good Gimp handbook in HTML format.  Click on "Contents" and you'll find a list of topics.  It also has an alphabetical index which makes finding stuff easy.
