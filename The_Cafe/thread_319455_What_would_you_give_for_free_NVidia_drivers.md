### Post by michaeljt on 2006-12-15
If postings I have read in various places are to be believed, a lot of people would seriously like to see free drivers for NVidia graphics cards.  There is a project (nouveau - nouveau.freedesktop.org) underway to create these.  However, I think that if enough people joined together to put some money towards paying someone to work full time on the project it might noticably accelerate it.

I have decided to try an experiment, and create a bounty which anyone interested in these drivers can contribute to.  The address is 
  [https://launchpad.net/bounties/free-nvidia-drivers](https://launchpad.net/bounties/free-nvidia-drivers)
A hundred contributions of twenty five dollars ought to pay for a week of developer time.  Two hundred for two weeks.  Someone familiar with the project should be able to make some progress in this time.  I am interested to see if how many contributions come.

If you think that this is an interesting idea, tell other people as well!
