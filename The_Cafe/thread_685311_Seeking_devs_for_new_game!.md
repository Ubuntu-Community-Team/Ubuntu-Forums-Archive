### Post by musther on 2008-02-02
I'm looking at starting a new game project, I love games with decent physics, I love bouncing cars around the place, jumping over hills and rolling about the place etc. So I'm looking for anyone interested in helping develop a game along these lines.

* Large 'arenas', generated landscape for infinitely varied play.
* Large, bouncy, fun to drive, 4WD, monster truck type vehicles for tearing about the palce.
* Different game modes
* Races (through checkpoints, timed etc)
* Deathmatch (destroy the other trucks by crashing, flipping etc)
* Team deathmatch
* Capture the flag?
* Weapons?
* Realistic physics, but with crazy vehicles and unrealistic possibilities (and unrealistic fun)
* Modifiers (change gravity, suspension, parameters for generating terrain etc).

So, I'm fairly experienced at designing graphics (mainly 2D, but can learn 3D), but I'm not a coder, so I'm primarily looking for somebody who is (or some people who are).

All ideas welcome.

There are great open source physics libs we can use, and great graphics libs too.

If you're interested, let me know!
