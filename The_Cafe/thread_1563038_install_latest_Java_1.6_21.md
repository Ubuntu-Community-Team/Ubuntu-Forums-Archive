Since the partner repo is not being updated for the moment, those looking to stay current on Sun Java finally have an excellent option to install/update to latest Java via a dedicated Java PPA thanks to Ferramosca Roberto

sudo add-apt-repository ppa:ferramroberto/java && sudo apt-get update
