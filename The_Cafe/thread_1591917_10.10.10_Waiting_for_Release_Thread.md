### Post by WRDN on 2010-10-10
For those taking the upgrade path from 10.04 to 10.10, it's worth noting that, the button to upgrade to 10.10 won't appear in the Update Manager unless "Normal Releases" is set in the Software Sources tool. To find this:

- Click System -> Administration -> Software Sources
- Select the "Updates" tab
- Select "Normal Releases" from the drop down under "Release Upgrade"
- Click Close

Now in the Update Manager, "Check" for new updates, then upgrade to 10.10.
