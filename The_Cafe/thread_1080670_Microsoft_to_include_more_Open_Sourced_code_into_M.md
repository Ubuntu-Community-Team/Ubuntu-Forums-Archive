### Post by MaxIBoy on 2009-02-25
> **chris200x9 said:**
> Am I the only one who read that and immidiatly thought microsoft is just doing it to lighten their load, i.e using GPL'd code.
They're not using GPL code.

There are open source licenses that do not require derivative works to be open source.

For example you can legally rip off code under the BSD and MIT licenses without releasing your source. It's well-known that Windows NT uses some BSD code for its TCP/IP stack, but Windows NT is still closed source. 

That's why the Free Software Foundation doesn't like you to use the term "open source" in the same sentence as "GPL--" they're not one and the same.
