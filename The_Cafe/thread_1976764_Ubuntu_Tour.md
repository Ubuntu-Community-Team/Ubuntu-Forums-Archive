### Post by Shadius on 2012-05-09
Hey all, 

I'm curious to know how come there isn't a Ubuntu Tour in Ubuntu. I've seen the tour on the Ubuntu site, but why not put the tour in Ubuntu so that new users can access it and get acquainted to the Ubuntu experience? 

My idea is to have a prompt giving the user a choice to take the tour now or do it later. I also find the Ubuntu online tour a bit lacking. In my opinion, it should be a little more interactive, like having the user perform some basic tasks such as installing/uninstalling a program via Ubuntu Software Center; a basic introduction to Ubuntu.  

What do you think? Share your thoughts!

So I thought it would be nice to have something that gives the newcomer a tour of Ubuntu built in and from there they would begin the experience and possibly continue using Ubuntu, thus gaining another Ubuntu Linux user!
