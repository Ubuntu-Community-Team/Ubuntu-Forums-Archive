### Post by endersshadow on 2006-04-27
Works great on Dapper!

I just made this with it:

[img]http://i31.photobucket.com/albums/c390/endersshadow7/jackbauer2.png[/img]

You will need the dev files for pygtk, glib, and gtk (off the top of my head), and oh yeah, make sure you install libxml-parser-perl.  This is a partial, but as best as I could make it, list:

```
sudo apt-get install libart-2.0-dev libglib2.0-dev libgtk2.0-dev python2.4-dev python-gtk2-dev libxml-parser-perl python-gnome2-dev python-gnome2-desktop-dev libtiff4-dev
```

That should give you all you need for install...though like I said, not 100% on it.  Good luck!
