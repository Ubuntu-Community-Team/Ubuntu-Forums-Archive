### Post by JDShu on 2010-03-19
Actually, Blender has its own game engine. So its technically possible to make a game with it without programming (they use some weird module system). However, you could say its underdeveloped and lacking in important features. For this reason, even the official Blender game project used a different game engine than the blender one.
