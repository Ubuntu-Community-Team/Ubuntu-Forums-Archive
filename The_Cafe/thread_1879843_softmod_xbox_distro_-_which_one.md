### Post by jjex22 on 2011-11-12
I did this myself quite a while back - before even my linux days as was - I didn't really know what linux was, i was just following a guide to see if I could do it.... didn't last long but it did begin my interest in how pc's worked which eventually led me here (hooray!)


I used yellow dog if I remember, but that's all but dead these days. From what I can tell gentoox is the only currently supported version around and being based on gentoo you really can set it up how you like and learn a lot about linux at the same time. Other than that you're looking as BSD - netBSD'd be the way to go in my opinion over free bsd as it's aim is to run on anything where as free bsd is user experience based and is struggling to support even ppc these days. Both should be entirely possible but you're not going to get a glossy debian installer for gentoox, and not much better for NetBSD.

Just to give you a realistic impression of what you can expect - basic office and internet use only - you can game, but think of it as 90's gaming at best, I remember getting a hacked version of age of empires to run (I'd imagine under wine)on the box I set up tho, but at the end of the day if you want to play games, it is an... ahem xbox!
