Hi everybody,

I need some advice on setting up a secure, web based, simple to users but sophisticated for admins [details below] file sharing system to be used between many remote sites. I'd like to know if anyone has set up anything similar who might be able to offer some advice.

We need:

- an open source, linux based solution
- easy enough for non-technical users
- granular access control [user/group permissions etc.]
- browser based, preferably no client installation necessary
- support for mass-uploading of multiple files

To give an example, KnowledgeTree DMS ([http://www.knowledgetree.com/](http://www.knowledgetree.com/)) is an excellent project, however a little overqualified and complex from the non-technical user's point of view.

I think what I'm looking for is something halfway between KnowledgeTree and MediaWiki; i.e. something with very simple, usable functions but more geared toward file sharing as opposed to Wiki page writing.

Does anyone here have experience with similar packages that might fit this project?

Thanks in advance for your suggestions.
