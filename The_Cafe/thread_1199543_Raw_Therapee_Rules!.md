### Post by HappinessNow on 2009-06-29
Today I was using Apple's Aperture to work on some raw images, unfortunately to my surprise Aperture does not support RW2 files (the raw files used by the latest Leica and Panasonic cameras [and it has been this way for almost a year now!]). The quick solution was to use [UFRaw]("http://ufraw.sourceforge.net/") and [GIMP]("http://www.gimp.org/") to convert the images to jpeg so Aperture could read the images. Basically if you want to work in Raw and own a late model Leica or Panasonic Digital Camera, Aperture is basically useless.

After some research I found [Raw Therapee]("http://www.rawtherapee.com/")! the Open Source alternative to Aperture. Raw Therapee not only is compatible with RW2 images it is also much more intuitive then the slop Aperture set up that Apple puts out.

Raw Therapee Rules! Thank You!

Also; another interesting note: CS3 can not read RW2, CS4 can but if you don't have CS4 your screwed, luckily GIMP came to the rescue; as per usually!
