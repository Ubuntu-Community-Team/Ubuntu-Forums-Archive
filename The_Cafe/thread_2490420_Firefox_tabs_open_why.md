The tab bar is an abomination.  I hide it with custom CSS and have tabs shown vertically.  We all have wide-screen monitors, which browsers would stop wasting vertical space.

Why have more than a few tabs open?  Because you are working on something.  

I only bookmark things I'll want to use more than 30 times a year.  Some of my internal-only webapps are in use constantly, so I want those pinned for easy access. The same applies to a few external websites.

Then there are all the others.  Say I'm shopping for a new "dewdad" and there are 10 companies with different dewdad versions.  I'll need 10 tabs to help compare each, assuming some review sites aren't trustworthy.  Plus, sometimes dewdads come with and without extra features, so I may decide that getting an external capability with a "thingamajig" that is separate would be better.  Now I need to find a thingamajig that is compatible with the dewdad I'm leaning towards at an overall price I can stomach from a seller that isn't a scam.  Easily, this workflow would get 20 tabs open.  Sometimes I want to think about it for a few hours, perhaps overnight, so I'd leave the main tabs open.

Plus, I still want to check the NOAA weather ...  ooooh, today is a perfect day to drop the fall fertilizer on the lawn.  Not too hot, with some rain in a few hours and not too hot tomorrow or Sunday.  Or control playback of some music ... another tab (sometimes).  I usually use a CLI music player mpc/ncmpc, but not always.

Same goes for having multiple browser tabs open.
