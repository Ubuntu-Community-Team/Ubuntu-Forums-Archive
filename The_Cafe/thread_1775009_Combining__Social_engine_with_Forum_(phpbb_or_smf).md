### Post by Lucradia on 2011-06-04
Well, that depends on what features of a social thing you want to add. For phpBB 3, there are always portal systems you can add to it, or maybe an RPG Modification, or if you need a gallery, you can add coppermine.

Pretty much anything that you find bundled nowadays can be replicated with individual modifications. The only things I don't like about either phpBB3 or SMF is that the official mods database requires you to use MODX (or in the case of SMF, some weird code that allows you to install it automagically), and I've stopped writing phpBB3/SMF mods since. I mean, if people can't learn how to find little pieces of code, why admin a forum? I know, I know, it's about efficiency, but I learned MOST of my PHP3/MySQL skills by going through code like that. Same goes for HTML3, and JavaScript 1.2 and CSS2.
