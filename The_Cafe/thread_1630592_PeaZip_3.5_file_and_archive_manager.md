### Post by Giorgio tani on 2010-11-25
PeaZip 3.5 was released today.
The new website of the application, with all information about latest release, is [http://www.peazip.org](http://www.peazip.org)
Downloads for Linux, as well as most common hints for Linux users, are on [http://www.peazip.org/peazip-linux.html](http://www.peazip.org/peazip-linux.html) page.

The project is now hosted on Google Code, which also provides the issue tracker, on [http://code.google.com/p/peazip/](http://code.google.com/p/peazip/)

As you can find linked in homepage, the new repository for translations is [http://code.google.com/p/peazip-translations/downloads/list](http://code.google.com/p/peazip-translations/downloads/list) 
You can start from here if you are willing to contribute or update a translation (your help is very welcome!) or if you are searching a newly contributed language file that is not yet in application's packages.
