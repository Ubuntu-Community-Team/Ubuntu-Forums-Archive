### Post by whizman on 2012-10-25
My DiskFit utility may be useful as a starting point for someone who would like to build upon it to develop a more complete application.

Or, for those familiar with command-line/file-based utilities, the current DiskFit utility could be useful.

[http://whizman.com/code/diskfit.tgz](http://whizman.com/code/diskfit.tgz)

C, C++, & Java; AGPL3.

Basically, diskfit outputs qualifying file-sets, and the user chooses one-disk file-sets to assemble into the disk-set - assisting the user (but not fully automating) toward both:
* making a complete disk set containing all the files;
* file priority order for first/earlier disks in the disk set.

More detailed information is in the diskfit.tgz:Doc/diskfit.txt file.
