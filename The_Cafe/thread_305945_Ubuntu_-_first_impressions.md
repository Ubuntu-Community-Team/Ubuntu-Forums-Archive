### Post by CJNyfalt on 2006-11-24
I switched yesterday over from FC5 to Ubuntu Edgy, and I like the switch.
Some of the reasons I switched:
- bittorrent not supported in Fedora
- No firefox 2.0 in FC6
- CD bloat, FC6 requires six CDs!
- I couldn't get my nvidia card working
- slow boot
- upgrade of FC = reinstall

I like what I see in Ubuntu, but there are some annoyances:
- The installer ASSumed that my CMOS clock was set to local time, and it required research before I found out how to make Ubuntu realise that it should be set to UTC. :mad: 
- The GRUB menu screen in ugly compared to Fedora.
- Emacs default fonts was very pixelated and ugly to look at.
- VIM didn't work correctly out of box, syntax highlighting didn't work until I installed more packages, but it took a long time before I realized that was the problem. ](*,)
