### Post by forrestcupp on 2006-08-28
For all of you video editing enthusiasts, I found an editor that may be worth checking out.  It's called zs4, you can find it here:
[http://www.zs4.net/features](http://www.zs4.net/features)

It looks like it probably can do about anything imaginable, unlimited video/audio tracks, motion, chroma key, lots of built in filters ...  The example videos are pretty impressive.  It's cross platform so they have versions for Linux, Windows ...

It's not opensource, but it is free to use for any purpose.  It looks like it might be more intuitive than Cinelerra.  I haven't tried it out yet, so don't ask me how to do anything.
