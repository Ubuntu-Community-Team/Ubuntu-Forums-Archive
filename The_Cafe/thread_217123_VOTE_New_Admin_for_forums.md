### Post by ComplexNumber on 2006-07-16
> **dabear said:**
> Please tell me why kassetra quit? And what's the difference between an admin and a moderator? Shouldn't the remaining admins decide this issue?
there are 3 levels:
-moderator
-supermoderator
-admin


a moderator can't ban, and only has access to various forums.  supermoderator can ban, has access to all forums, and has most of the facilities that an admin has...except that the supermoderator can't save the changes. for example, in the moderator/admin control panel, it will show everything about the user's settings and profile (except their password). however, the supermoderator cannot change any of that because there is no way to save it. so whilst, for example, a supermoderator can change a persons usertitle, only an admin has the facility to save the changes for them to take effect.
admins can also make changes on the boards layout - moderators and supermoderators can't.


i dare say the moderators and admin will discuss it between themselves, if they feel there is a need to. if kassetra brought some special skills (eg designer, coder, etc), then she may need to be replaced with someone else who has her skillset.
