### Post by ki4jgt on 2011-11-22
Because I have a netbook, I don't have a full sized keyboard. One of the first thing I do is adjust the keyboard shortcuts on my keyboard. Before Ubuntu started to require multiple keys for macros, I had set my super key to hide everything on the desktop. It was a quick exit all strategy which kept prying eyes to themselves (I live with very nosy people). I **used** to be somewhat private. Now, I've set some other keyboard shortcuts to increase my efficiency and I thought it would be interesting to see what shortcuts people have assigned to their own systems.

Currently I have no audio controls on my netbook, so I setup the following macros to deal with this problem.

CTRL + UP = Starts or resumes Banshee
CTRL + Left = Goes to previous track
CTRL + Right = Goes to next track
CTRL + Down = Pauses Banshee

Though this is a bit trivial, I would like to hear of other's customization to their shortcuts.
