### Post by Ender Black on 2007-04-24
Hmmm... although there is still raging debate about what is or is not Web 2.0 I think most agree it is change from static developer created content to user generated content.  For example - [www.ubuntu.com](www.ubuntu.com) = Web 1.0.  [www.myspace.com](www.myspace.com) = Web 2.0.  Digg, Slashdot, basically any social based user generated content is what is moving us toward Web 2.0...whatever the hell that is.  

But, I think an online only OS is not too far away and the ones who will pull it off will be Google.  Google is simply awesome but the amount of private data they are aggregating is getting to be scary.  With an online only OS they would be able to watch your every move, every website, every application you launch.  Umm...no thanks.
