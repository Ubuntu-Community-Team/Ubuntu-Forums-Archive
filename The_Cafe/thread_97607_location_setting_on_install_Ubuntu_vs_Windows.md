### Post by jnoreiko on 2005-12-01
Yesterday I had to help a friend install Windows XP (I've shown him Linux, but I'm not sure he's ready for it. I've put Firefox & GIMP & OOo on his machine for now...)

I had to tell the Windows installer I was in the UK THREE times: for the time zone, for my keyboard, and for something else I don't remember now. For that, I had to click on three 'Details...' buttons in the same dialog box.

Today I installed Ubuntu on a new drive.
It asked me my language, then suggested a short list of possible countries, and then an even shorter list of time zones.

Whoever created the interface for the ubuntu installer, I'd just like to say nice job :smile: 
It may not be as pretty as the MS installer, but it's a LOT nicer to use!
