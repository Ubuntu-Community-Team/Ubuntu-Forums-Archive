I would never PAY for a game like that, but I did learn a lot about how the UN operates. I'm sure teachers will have a use for it in their classrooms, but they could have done so much more. If I were to make a free game to promote the UN, I would do something more like SimCity, only with a tiny, poverty stricken villiage instead. It's no America's Army, but it's a start.
