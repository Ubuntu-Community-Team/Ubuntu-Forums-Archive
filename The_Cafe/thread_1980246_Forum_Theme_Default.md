### Post by Technoviking on 2012-05-14
By popular demand the new Ubuntu forum theme is now the default. To assist people in moving to the new theme, it will be the only one selectable for a couple of weeks. After that Ubuntu and Ubuntu-White will be selectable. The Clean2/3 theme will be deleted to keep the look more uniform.

Please fill out a bug report on Launchpad against the forums ([https://bugs.launchpad.net/ubuntuforums.org/+filebug](https://bugs.launchpad.net/ubuntuforums.org/+filebug)) if you find any bugs.

Thanks for the kind words and feedback, changes will continue to happen to improve the new Ubuntu theme.
