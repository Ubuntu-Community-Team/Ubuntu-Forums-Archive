### Post by Redache on 2010-06-29
1. How is Linux different from Windows?

Linux is a derivative of Minix, which itself is a derivative of Unix. Windows uses the NT kernel and is based on an architecture of (mostly) Microsofts design (Although there are some elements of POSIX compliance). Windows comes with everything integrated in one, Linux lets you choose the different components of the OS. Technically, Linux is actually just the Kernel where Windows is the whole OS. GNU/Linux is the more correct term for Linux as an OS.

2. Why isn't Linux just like Windows?
Because they have different ways of doing things, Windows gives you an OS that is fairly locked down and all the tools you need to get up and running straight away( and a pretty big Monopoly to boot). Linux on the other hand has as many options as there are users. Linux Distros are a way of combining various Software combinations to create an OS that you can use from the off. 

3. Why doesn't Linux get viruses/malware?

This is a double edged question. 
- A lot of Linux distributions operate on a non admin by default policy this means that things that do anything to the System(apart from your Home directory) require elevation (i.e. they need to attain Admin rights). This makes it hard to engineer a virus as the user would have to knowingly elevate its permissions to run.

- There is a lot of discussion over whether Linux just isn't big enough on Consumer PC's to be targeted. If it was then we'd probably find out if Linux is truly secure from attack (on the server side it's proven that Linux is reasonably secure). 

Basically, most of the attacks on a Linux system would require social engineering to work. On Windows there is more opportunity to not have to use social engineering (this is changing though).

4. Why/How is Linux free?

Because most people who develop for Linux do it to scratch an itch, or they are employed by companies that sell support for Linux to businesses. It's a different model to a lot of traditional software, one that gives away the underlying product for free but you pay for support. There are also plenty of political/ethical reasons as to why Linux is developed for free.

5. Is Linux illegal? / Why do people think Linux is illegal?

No.
People think it's illegal due to a lot of shady marketing and the assumption that something that's free has to have something wrong with it. It's hard to accept that human behavior can sometimes surprise you and that people genuinely believe in advancement through knowledge.

6. Why are there so many distributions? / Why not just one distro?

Because there are different users. People want something different from an OS, distros spring up because people have a new vision for how a GNU/Linux OS should behave/look. It all gets down to software choices and sometimes how the kernel is modified (like Distros for multimedia production and so on).

A One True Distro would ruin the choice available and would go against the ethic that drives Linux development, as much choice as possible.

7. What makes 64-bit Linux better/worse than 32-bit?
