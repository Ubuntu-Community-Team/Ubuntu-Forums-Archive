### Post by NightwishFan on 2009-08-18
I like XFCE a lot. It is simple and fast, and improved a lot lately where it can really begin to compete with KDE and GNOME. Especially the control panel. I use GNOME though because nearly everything is run by drag and drop, which is magnificent.

It is odd that Xubuntu actually uses more RAM than Ubuntu on average, though it is still lighter. You have to study Linux memory concept to understand what I mean. For example the Listen player uses 55mb when Rhythmbox may only use around 20-30mb, but it does not 'need' all 55mb and could do with less.
