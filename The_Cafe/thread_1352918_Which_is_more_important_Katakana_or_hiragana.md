### Post by miggols99 on 2009-12-12
Well, if you're talking about learning, hiragana is more important (but katakana is still important to learn). Katakana is used more in websites because it is used for emphasis, the equivalent of using italics or underline in English. Also, it depends on the kind of website you are visiting. If it is a news website for example, kanji will be used a lot more. If it is targeted to a younger audience, kana (hiragana/kanakana) will be used more.
