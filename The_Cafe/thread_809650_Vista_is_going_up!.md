It seems like use of Linux is rising, but that may be an observational bias on our part. ### Post by Babbage on 2008-05-29
Another thing that prevents ordinary (non-technical) users from installing Linux are the restore partitions that computer manufacturers put on hard drives. Paths to these partitions are written to the Master Boot Record, so users can access them by pressing an F key at boot time. This can make it more difficult to configure a dual boot. As can the various Media Direct and Office Direct features, that allow direct application access without booting, found in Dell laptops among others. There should be agreed protocols for the installation, use and configuration of these type of features, to allow them to be used across different operating systems.
