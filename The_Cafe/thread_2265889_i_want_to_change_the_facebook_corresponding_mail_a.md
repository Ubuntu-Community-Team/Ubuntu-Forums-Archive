### Post by djchandler on 2015-03-05
I have changed one of my accounts' email without cell-phone verification, but I have had it 5 years and have many friends and family connected to the account (obviously a real person). If you are having a difficult time, it could be that your account has been flagged as using a pseudonym. I have a newer FB account using a fake name that could not be verified because the cell phone provider was not listed. Eventually any issues I had resolved after using that account heavily to post on a third party's site as well as building a profile for that identity on FB. I use both Google Chrome and Firefox. The only time I use IE is in Windows 7 and the site uses Silverlight to stream video, such as Amazon.

The browser you are using shouldn't make a difference, but security settings may be making a difference. Alternatively, if you are using AdBlock, NoScript, or another plug-in for Mozilla browsers that protects privacy, one of those may be blocking a script or FB app. Either way, make sure you are not blocking any FB domain (facebook.com, facebook.net, fbcdn.net) while trying to change an email. Make sure you are not deleting facebook cookies.
