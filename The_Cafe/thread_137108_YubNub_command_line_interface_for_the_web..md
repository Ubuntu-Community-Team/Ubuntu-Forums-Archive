### Post by purdy hate machine on 2006-02-27
Here is an interesting idea, a command line interface for the web. 
[http://www.yubnub.org/](http://www.yubnub.org/) has a huge list of commands, a list of which can be viewed  by typing &#8220;ls&#8221; as the prompt.  These range from everything from a Google search to making a Paypal payment. 
There&#8217;s an article on [NewsForge]("http://internet.newsforge.com/internet/06/02/13/2032208.shtml") about it or just try it out for yourself and see what you think.

A few example commands:

**playgame **
Starts up a random online game for you to play.

**loto (state)**
search for state lotto results 

**g Linux**
Performs a Google search of Linux

**Wp Linux**
Performs a Wiki search of Linux
