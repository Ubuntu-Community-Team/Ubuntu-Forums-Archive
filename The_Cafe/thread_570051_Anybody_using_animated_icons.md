### Post by thisllub on 2007-10-07
The easy way 
You edit the .desktop file for the app

e.g.

In ~/.e/e/applications/all
I have the following firefox.desktop

[Desktop Entry]
Encoding=UTF-8
Name=Firefox
GenericName=Web Browser
Comment=Browse the Internet
Exec=firefox
StartupNotify=true
StartupWMClass=Firefox-bin
Icon=firefox.edj
Type=Application
