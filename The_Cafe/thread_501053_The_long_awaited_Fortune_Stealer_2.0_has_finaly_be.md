### Post by RussianVodka on 2007-07-14
[Download yours here!]("http://download285.mediafire.com/nj3p0n1q5rtg/ehpjyjy1nmm/FortuneStealer2.0.zip")

I was bored the last couple of days, so I decided to put my self to use and brush up on my Java skills. So I went and rewrote a program I made a while ago in C++. I gave it a graphical interface with pretty buttons, and a built in downloader (the old one used to make system calls to a windows port of wget).

Anyways, what does this program do you ask? Basically, it is a program that with a click of a button allows you to download "fortunes" from ThinkGeek, and read them at your own leisure.

For instructions on how to use, read the Readme.

So that's it, go download it, try it out, and tell me if it works as intended.

Edit: Just realized that I used forward slashes to separate folders in code. I don't know if that'll cause a problem in Windows. I think what'll happen (unless java auto-corrects this) is that instead of files being in the "Data" folder, they'll end up in the main folder as "Data/fortune_list" and what not. But since many of you use Linux, that problem shouldn't effect you.
