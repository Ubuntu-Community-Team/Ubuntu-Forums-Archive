I don't have first hand experiences with the H300, but I've had great experiences with rockbox on my H120, and I know that quite  a few people are using it on H3x0's. That said, you have to install a bootloader on the device. It's not hard if you follow the instructions, but it could potentially ruin your device if you're really unlucky.

### Post by Project 318 on 2006-04-04
I use Rockbox on my H340.  It's much better than the stock firmware with all the great WPS. 

There have been NO reports of Rockbox ruining a player.  What Rockbox does is patch the default firmware with a bootloader.  From the bootloader you can boot into Rockbox or the default firmware.
