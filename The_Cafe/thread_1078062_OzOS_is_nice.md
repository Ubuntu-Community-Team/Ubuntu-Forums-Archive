### Post by smartboyathome on 2009-02-22
OzOs is based on Ubuntu, though it is also ported to Arch in the easy-e17 package. OzOs isn't an official Ubuntu build, but maintained by a community including Tux Aubrey, Rui Pais, and RAV TUX/OCUUNS/etc (he changes his name often). E17 is a window manager with other things built in, including shelfs and a file manager. They like to call themselves a desktop shell like on Windows.
