I've just spent the last couple of hours setting up a Windows XP Machine and using it. I'd forgotten just how much I dislike the OS. I won't go into great detail about this but on an AMD XP 2600+ with 512MB RAM and 60Gig HD 128MB NVidia 5500 AGP the thing really doesn't run very well at all.

So as you'd expect I was glad to get back to my Ubuntu Network. The reason for the name of the post well here goes.

I thought I'd do an experiment and have a laugh to see how much better Ubuntu runs on the same box so Installed Ubuntu Gusty and decided to do some load testing.

Heres what I've done.

Set compiz to box rotate with most effects on. SSH'd into my old media server (Dual PII 350MHz 1GB RAM) and set the computer up as a client. I am at present streaming the films Clockwise & L.O.T.R. The Two Towers from the server also streaming music from the server and running two ssh sessions and top in terminal not forgetting writing this using Swiftweasel all on the same PC that would only run Windows XP in mediocrity at best. Oh and running Record My Desktop file attached. Take a look. The only reason it's jerky is because of Record MY Desktop records jerky. Everything was running smooth as silk.
