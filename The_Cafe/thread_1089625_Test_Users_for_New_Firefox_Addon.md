### Post by Baris Derin on 2009-03-07
Hello all,

I have developed a new Firefox add-on. The add-on lets users install backgrounds and wallpapers directly on Firefox. Although Firefox add-ons are known to be cross-platform working, the new add-on only works flawlessly on Windows specific operating systems. The add-on high probably having file and directory permission problems on Ubuntu.

Normal Ubuntu users can not and do not send bug reports. Because Ubuntu Forums brings many power users together, I thought it would be great place to find some test users for the addon.

I want Ubuntu users to try this new add-on. But before releasing the add-on to Ubuntu public, I have to fix all current Ubuntu specific bugs. So could you install the new add-on and send the possible bug reports, please?
