### Post by RudolfMDLT on 2007-06-01
Hi,

I am interested to see how many people would like Ubuntu's Bluetooth capabilities expanded, and for what reason. I've been able to send files since edgy, but the method was a little crude and big(>2Mb) files mostly ever dropped the file transfer. I know it is possible to do a lot already, but in order to get things done a lot of tweaking of a system is needed and usually it only works with certain cell phone models or other devices.

I personally would like to see Ubuntu's BlueTooth capabilities expanded.

For me necessities would be:

1) The ability to browse the files on my phone
 [INDENT]KDE already has this feature but Gnome not[/INDENT]

2) To be able to use my cell's 3G or GPRS as a way to access the web with my Laptop or PC.
[INDENT]As far as I  know there are a few "private" projects on the topic but it's still very under developed[/INDENT]

I would consider the following Luxuries:

3) Syncing my cellphone with Evolution or Thunderbird's Calender and/or email

4) Being able to control Amarok from my cell
[INDENT]This is possible with certain Sony Ericson's as far as I know[/INDENT]

It'll be interesting to see what other Ubuntu/Kubuntu users want to use bluetooth for.

Cheers,

Rudolf
