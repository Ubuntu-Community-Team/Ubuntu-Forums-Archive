### Post by KingBahamut on 2005-09-29
MEPISLite 3.3.2-test01 is available in the testing subdirectory at the MEPIS Subscriber's Site and public mirrors. This release includes the pr2 version of the upcoming DCC base kernel. The 2.6 kernel was updated because performance and compatibility with this kernel has been excellent. Also this kernel uses initrd to support SCSI and USB bootable partitions. **This is the first step toward MEPIS converting to only building and using standards compliant kernels.** The HP printer support has been improved through the introduction of the packages from the HP Linux Printing Project.


Im anxious to see how this standardization affects the distros that are members of it. Has anyone cut this ISO down yet? Any visable improvements (cant expect much with a test release, but still) ?
