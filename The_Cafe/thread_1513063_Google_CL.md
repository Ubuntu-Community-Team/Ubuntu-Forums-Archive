```
Blogger

Calendar

Contacts

Docs
$ google docs edit --title "Shopping list"
Picasa
$ google picasa create --album "Cat Photos" ~/photos/cats/*.jpg
Youtube
$ google youtube post --category Education killer_robots.avi
```
