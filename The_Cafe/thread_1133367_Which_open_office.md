### Post by MaxIBoy on 2009-04-22
Ubuntu currently ships with OpenOffice 2.4. OO 2 is still being updated with bugfixes, but it's not getting new *features*.

OpenOffice 3 is in active development, receiving features *and* bugfixes. OO 3.0 wasn't considered stable enough for Ubuntu 8.10 (Not that it actually wasn't stable, it was just untested.) OO 3.1 came out too recently to be included in Ubuntu 9.04 (I think. I'm not at all sure, I might be wrong, 3.1 might be in 9.04.)

Ubuntu's policy is to not update packages unless they contain important security flaws or showstopper bugs. So if it A) allows an attacker to crack your system, or B) sets your computer on fire when you run it, then it will be updated with fixes. But if it lacks functionality which was added in a more recent version, it probably will not be updated. This is designed to improve stability. For some people, the stability is worth it. Others don't mind if the computer locks up once every two weeks, as long as they get the new features. It's a personal choice.

Right now, you just have to install Open Office manually in order to get the recent version. Be sure to remove all traces of the old version before installing the new one! Otherwise it will conflict.
