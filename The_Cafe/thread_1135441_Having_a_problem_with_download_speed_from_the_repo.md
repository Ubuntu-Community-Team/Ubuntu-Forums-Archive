### Post by wersdaluv on 2009-04-24
Downloading software from the official repositories is much slower lately. It's probably because of Jaunty's release. I tried the local and main server but both of them offer the same results. I do not know if this option is available for all versions; I found out about this on Jaunty. 

Go to System->Administration->Software Sources. On the Ubuntu Software tab, select "other" for "Download from:". Click "Select Best Server" then servers will be tested. It will give you the best server for your location then just click "Choose Server".

Enjoy faster software downloads!
