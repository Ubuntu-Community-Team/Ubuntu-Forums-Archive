### Post by chessnerd on 2010-06-25
It all depends on what the OS is designed to be for...

If it uses LXDE, you could call it **Flite OS** because LXDE's logo is a bird and it is a lite distro.

If you want it to be for Windows-like you could call it **FenOS** or **Fenee OS** (fenêtre is French for "window").

If your OS is supposed to be an easy-to-use SuSE varient you could call it **SoftLanding OS** or **SoftSystem OS** because SuSE used to stand for "Software und System-Entwicklung" meaning "Software and System Development."
