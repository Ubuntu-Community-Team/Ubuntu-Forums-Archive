### Post by steveneddy on 2009-07-07
Maybe VirtualBox along with Ubuntu will do them some good.

[COLOR=Blue]**If they already own a Microsoft key**[/COLOR], you could DL TinyXP and TinyVista. They are little .iso files that boot quickly and install great in VirtualBox.

I use TinyXP in VirtualBox on my Ubuntu laptop (**[COLOR=Blue]I do own several XP licenses[/COLOR]**) and use it to do things that can't be done easily on Ubuntu.

If something bad happens, I just delete the virtual hard drive and start over with the TinyXP .iso file.

I like the fact that I can store the .iso file on my laptop and boot to it when necessary.
