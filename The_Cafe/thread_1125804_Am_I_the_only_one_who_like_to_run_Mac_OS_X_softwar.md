I really do not like having to wait for years at times for WINE to get up to speed with the latest dll files and software available for MS Windows. However, there are many other types of software like certain games that I enjoy and the Tax software I have been using for the past 8 years that there really is no open source alternative you can get it for Windows or OS x and that is it.

Anyways enough of my ranting, let me know how you feel and if this seems like a good idea or a bad idea.

I just curious how many people would like to see this happen besides myself.

### Post by 3rdalbum on 2009-04-14
PearPC is a PowerPC emulator. You still need a compatible version of the Mac OS in order to use it, the speed is low, and you probably need to make an image of Macintosh ROMs in order for it to work.

Implementing OS X on Linux would be probably no easier than implementing Windows on Linux. Sure, there's some bits of BSD in OS X, but we're on a GNU/Linux system, not a BSD/Linux system, and OS X programs are written against the Cocoa or Carbon API not the kind of libraries you see on Unix.
