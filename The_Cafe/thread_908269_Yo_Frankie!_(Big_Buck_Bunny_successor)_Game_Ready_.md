The Yo Frankie! game, made in blender, is ready for testing (i.e. practically finished).

[http://www.yofrankie.org/](http://www.yofrankie.org/)

If you install the 'zeroinstall-injector' package you can install by running ```
0launch http://crystalspace3d.org/downloads/YoFrankie/YoFrankie.xml
```

Alternatively download the 'big package':
[http://crystalspace3d.org/downloads/YoFrankie/YoFrankie_run.sh](http://crystalspace3d.org/downloads/YoFrankie/YoFrankie_run.sh)
