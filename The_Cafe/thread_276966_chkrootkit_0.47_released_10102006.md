### Post by lotusleaf on 2006-10-13
**[chkrootkit 0.47 is now available!](http://www.chkrootkit.org/)** - [download](http://www.chkrootkit.org/download/)
*"chkrootkit is a tool to locally check for signs of a rootkit"*

> (Release Date: Tue Oct 10 2006) This version includes:

    * chkproc.c
          o some bug fixes, thanks to Lantz Moore
          o use of getpriority() to identify LKMs, thanks to Yjesus(unhide) and Slider/Flimbo (skdet)
          o new rootkit detected: Enye LKM
    * chkrootkit
          o new test: crontab
          o new rootkits detected: Enye LKM, Lupper.Worm, shv5
          o more ports added to the bindshell test
          o some minor bug fixes
