### Post by vanadium on 2016-05-22
I stopped using Autokey as it does not work reliably anymore (last update: 2012). It used to work until Ubuntu 15.10 provided you disabled ibus input. However, in 16.04, I found that disabling ibus cripples the US international keyboard, where dead keys would work in some apps, not in others. So the choice became either Autokey and a crippled keyboard or no Autokey and a properly functioning keyboard. I have moved to [url=http://www.linux-magazine.com/Issues/2014/162/Workspace-Text-Expander]DIY text expansion using xdotool, xsel and a bash script/url]. Works very well and I never looked back.
