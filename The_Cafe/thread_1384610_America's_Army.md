### Post by lovinglinux on 2010-01-18
No. America's Army does not have a Linux version anymore. That link is for the old 2.5 version. After that, there was 2.6.x, 2.7.x, 2.8.x, which were all for Windows only. The current version is 3.x, which uses the "new" UT 3 engine and it's a completely new game, also without a Linux version. There are no plans for supporting Linux again. In fact, that Linux version was ported on spare time by a developer, which I guess no longer works for the Army, since most developers contracts ended and weren't renewed after the 3.0 fiasco.

That being said, you can still install the 2.5 version and I guess there is still some servers running it, but finding a server full of players won't be that easy I guess.

BTW, I have played AA for more than a year. It was the best and only game I kept playing for so long. But since I switched to Linux, my only experience has been a couple of rounds on the new version using Windows in dual-boot.
