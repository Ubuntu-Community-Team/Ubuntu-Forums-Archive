### Post by BobCFC on 2008-02-06
Things that I noticed about Hardy:

When I first booted into vesa mode it auto-detected the native resolution of 1680x1050 not some silly 800x600 default.

The new gnome is snappier and more responsive, launching small apps instantly and browsing big folders using Naultilus.  Even though I have 8gb ram  -  ms delays used to really bug me.  Other small improvements such as dialogs / icons.

It is possible to enable compositing for metacity using gconf.  This means you can have drop shadows without needing compiz.

The latest NVIDIA restricted drivers are there (169.09) even though they only came out last week.

The repos are up to date including Firefox3 even though it's only a beta and Miro 1.1.1 etc

As you say new default apps such as CD burner/ bit torrent.


Over all an evolution not a revolution.  Which is what you'd expect considering this is an LTS release with 5 year support.. stability is the priority this version
