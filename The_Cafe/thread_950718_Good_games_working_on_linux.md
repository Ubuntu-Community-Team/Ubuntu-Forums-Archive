### Post by joninkrakow on 2008-10-17
> **crazyfuturamanoob said:**
> 
So I want at least better graphics than those 1990's graphics. And addictive, working gameplay, like soldat or cs 1.6.

[SIZE="1"]Think I am a bit too choosy?[/SIZE]

Yes. Since you have rejected mid-90s graphics, you have cut yourself out of probably one of the best-playing games available. You probably know Halo. Marathon was spiritual predecessor to Halo by Bungie. Like Halo, it's a trilogy, also like Halo, it is a first-person shooter. Unlike Halo, it's open source now (was open-sourced when Bungie got bought out by Microsoft. The source is  called "alephone".

You can find a rpm for it on the source's web site, [http://source.bungie.org]("http://source.bungie.org"). You will need alien to convert it to a deb file. Don't forget to download at least the first scenario--Marathon. 

Unfortunately, it can be a bit complicated to get alephone running. When you first try to run alephone, you will get errors. I use "apt-cache search" in the terminal to search for the library that was the source of the complaint. You will need to do this a few times before you will finally get alephone to run. When you try to run it, don't forget to add the path to the game files you downloaded. 

You seriously ought to set aside your pickiness and try alephone. ;-) I'm sure that you will find Marathon a satisfying game to play. IMO, it has the best story of any game from its era, and better than most today, in fact. Just don't go to the marathon.bungie.org/story/ web page before you play--it contains spoilers--no, on other hand, they aren't real spoilers, but give you a sense of the depth of the game's story. :-)

-Jon

(and apologies to all who are tired of me pushing this game every time games come up.) ;-)
