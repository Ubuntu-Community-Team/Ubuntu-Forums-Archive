### Post by hatedsoul on 2010-05-05
First activate "medibuntu repository" in synaptic package manager and then download and install [[COLOR="Red"]This[/COLOR]]("ftp://88.191.250.18/3v1deb/pool/gutsy/3v1n0/pytube_0.0.8-1_all.deb") with all its dependencies.This is an old version of PyTube and it doesn't supports capturing browser catche.Purpose of installing this version is just to resolve all dependencies required for PyTubes.This will Install all additional file required for PyTube.Then go to "Control Centre" search for "PyTube" (You Tube Downloader) in Search box and remove it.Then download [[COLOR="Red"]PyTube 64 Bit[/COLOR] ]("ftp://130.239.17.6/archlinux/community/os/x86_64/pytube-0.0.11.5-1.pkg.tar.gz") or [[COLOR="Red"]PyTube 32 Bit[/COLOR]]("ftp://130.239.17.6/archlinux/community/os/i686/pytube-0.0.11.5-1.pkg.tar.gz"). Now extract the tar.gz file and past the "usr" folder from extracted folder to root directory.Press merge all.Now hit alt+F2, enter "Pytube".Hit run and you have done it.
  Pytube 64 Bit tar gz is also attached below.
