### Post by Idefix82 on 2011-01-10
Regarding video tutorials for the terminal, I think that video is the worst possible medium to learn the cli. Firstly, as has been said before, it's important to try the commands yourself, so pausing and continuing all the time becomes a pain. Secondly, cli tutorials are as much for reference as for first learners. So having things written down somewhere is much more useful than a video, which you would never use to look something up.

Usually, video tutorials are well suited to explain something that is inherently visual. There are plenty of video tutorials showing off the compiz effects. But one could create a tutorial, e.g. showing other customisation options for ubuntu. For example, how about a tutorial about completely changing the look of ubuntu, including gtk themes, metacity themes, icons, gedit colour scheme, installing cairo dock or awn, conky, screenlets and so on. It would be hard to make a really useful and good such tutorial (in particular, to actually change the gedit colour scheme or to write a conky script, the user would have to go elsewhere and learn this, and you would have to set things up in advance, so as to show off the functionality on the video), but if you succeed, I imagine that many people will benefit.
