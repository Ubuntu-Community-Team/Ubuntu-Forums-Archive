### Post by Lord Xeb on 2008-05-31
HDAPS (Hard Drive Active Protection System) is used by IBM/Levono in their laptops to keep the HDD safe by stopping the HD when a sudden bump, movement, etc is detected. Thinkwiki's tutorial doesn't work because linux doesn't support it in the first place. There needs to be more awareness about this problem and I would like to see some improvement. The HDAPS utils do not work. They only make the motion sensor work. The actual stopping of the hard disc still doesn't happen.
