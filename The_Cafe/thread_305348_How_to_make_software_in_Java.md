### Post by Circus-Killer on 2006-11-23
just to correct the previous post, AFAIK sun's java is not GPL. but there are other java implementations that do fall under the GPL, however won't have the same support as sun's java.

i prefer to develop using sun's java. but thats just my personal opinion. a few nifty apps/text editors to use include eclipse, jgrasp, and scite. scite and eclipse can be installed via the universe repositories, whilst jgrasp can be downloaded from [http://www.jgrasp.org/]("http://www.jgrasp.org/").

As for learning the actual language, there are online tutorials, however i think it would be best to buy a java text book. text books are never a waste of money, and you'll always find yourself referring back to it.

a good howto on installing the different java environments can be found [here]("https://help.ubuntu.com/community/Java").
