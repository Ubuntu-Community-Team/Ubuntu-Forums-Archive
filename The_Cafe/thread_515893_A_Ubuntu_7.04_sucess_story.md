### Post by CHFFriday on 2007-08-02
This is an Ubuntu  success story. I know there are many success stories but I think this one is special.

To begin, the PC is a IBM 600X laptop. Nothing fancy just a CD Rom and built in Agere Winmodem. 

This is what I have done with this little low powered Laptop:

Wireless is up and running with a PCMCIA D-Link DWL-G650 card. Plug & Play

LAN connection is up and running with a D-Link PCMCIA Lan card. Plug & Play

Have setup a Network Printer (Konica Minolta). This was most difficult.

Setup VPN to remote into the office VPN Server. Use from my home and other places. Works Great!

Use WINE to run VPN Ulta.

Have setup the Agere Winmodem. Don’t use it but it may come in handy. Just wanted to see if I could do it.

Used all of the Open Office apps to create and edit various MS Word and Excel documents. Have done this when connect via LAN and VPN.
 
Have set my Ipod. Not the best but it works.

Have setup my Palm Zire. Not the best but it works.

Setup a Local printer for home use. Just plug in the USB and it works.

Setup USB external hard drives. Plug and Play.

Installed KMyMoney and checkout a few other banking programs.

Installed a Windows program that I use. It is running from Wine.

Total time involved two weeks.  

This is to give everyone a quick idea of what you can do with Ubuntu.

CHFFriday
