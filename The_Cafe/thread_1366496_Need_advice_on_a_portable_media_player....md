### Post by garryknight on 2009-12-29
**7.  Samsung YP-Q2**
    **Price:** ~$150
    **Screen:** 2.4" QVGA (320x240)
    **Memory:** 4G, 8G, 16G
    **FM:** Yes, can record broadcasts
    **SD Expansion:** No
    **Battery:** max stated 50 hrs. In practice, more like half that, but depends on how much video you watch.
    **Audio:** MP3, WMA, Ogg, FLAC
    **Video:** WMVand MPEG4 when encoding with EmoDio software, almost any using ffmpeg
    **Images:** Picture viewer for jpegs, haven't tried other formats
    **Linux:** Putting the Q2 into MTP mode, it appears to Linux as a camera or PMP device; putting it into MSC mode, it appears as a removable storage device so you can drag and drop MP3s, JPEGs, videos, etc.

    **Other:** built-in mic for voice notes or lecture recording; no built-in speaker; comes in black or white, no other colours; 2 built-in games; can download podcasts; adjustable playback speed; clock but no alarm mode; basic file management; text file viewer; configurable EQ which can also be set to change according to the type of music you're listening to; no line-in recording

**Notes**
1. You can see the full specs _[here]("http://www.samsung.com/us/consumer/mobile/mp3-players/mp3-players/YP-Q2JCB/XAA/index.idx?pagetype=prd_detail&tab=spec")_.
2. Converting videos for use with the Q2 is simple using ffmpeg. I've written a howto _[here]("http://ubuntuforums.org/showthread.php?t=1346980")_.
3. The Q2 organises MP3s based on the embedded tags. For example, it reads the Album field in all of the files to make them available in the Albums list, no matter how you organise the files on the Q2. I ensure that I organise my MP3s into directories - something that the Samsung EmoDio software fails to do, dumping them all into the root directory. I also make sure that all of the relevant MP3 fields are filled in, i.e. Title, Artist, Album, Year, Genre, and I set up album cover artwork using Amarok, which embeds a small image into each MP3 file; the Q2 looks particularly good displaying the album artwork though the screen is configurable.
4. Playlists can be set up using the Samsung EmoDio software (Windows only) or you can create up to 4 playlists using the Q2 itself. This isn't a big limitation if you tend to play a whole album at a time.
5. The great thing about the Q2 now is its price. I've never heard such good sound out of something so inexpensive. The drawbacks are a) the proprietary data/charging cable, and b) the slightly fiddly controls, which don't really take that long to get used to.

I hope this all helps someone as I would think that the OP would be happier with something more like the [Cowon iAudio 7]("http://www.cowonglobal.com/") which is more feature-packed, including line-in recording, longer battery life, clock alarm, and more.
