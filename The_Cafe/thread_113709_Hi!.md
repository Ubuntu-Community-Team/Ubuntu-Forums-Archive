The main ubuntu website suggested that I post a hello message, so I am following up on that.  I have been using Ubuntu for about 8 months now and love it.  Great user community and support...

The best way to learn linux is just to install it as your primary system, afterwhich you are forced to deal with whatever issues arrive.  Same with any O.S., but the open source community has been great to work with and well worth any investment that may be in question. Keep up the great work everyone... I am with you.
