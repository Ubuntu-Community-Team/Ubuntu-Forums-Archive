### Post by plb on 2011-06-18
So I've made a note app similar to iPhone one in appearance but many more features. Here is a market link and features listed below. Look forward to feedback!

[https://market.android.com/details?id=com.flufflydelusions.app.notesclassiclite&feature=search_result](https://market.android.com/details?id=com.flufflydelusions.app.notesclassiclite&feature=search_result)

At present:
* Export notes as text to SD
* Share
* Export/Import entire database
* Google translate notes
* Text to Speech audio playback of notes
* Add notes to Google Calendar
* Add notes to status bar
* Create Home Screen shortcuts of notes
* New Note Homescreen shortcut
* New Note Statusbar entry
* Modify timestamp
* Note Details: create date, modification date, word count, character count, line count
* Search note contents 
* Search database
* Image support
* Video notes
* Paint notes
* Private notes
* Audio notes w/ voice recording
* File attachments e.g PDF, MS Word, MP3, PowerPoint, etc
* Geotag + Address look up support
* In note content search with hi lighting and search & replace options
