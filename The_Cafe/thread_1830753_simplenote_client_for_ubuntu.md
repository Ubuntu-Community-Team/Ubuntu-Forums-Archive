### Post by cpbotha on 2012-05-25
I'm a heavy simplenote user, so I've tried syncpad on chrome (search can be slow), resophnotes with wine (sometimes freezes) and even the dropbox syncing option of simplenote.

In the end, none of these made me 100% happy, so I coded up nvPY: An ugly but cross-platform simplenote client inspired by nvALT and ResophNotes. Get it here: [https://github.com/cpbotha/nvpy](https://github.com/cpbotha/nvpy)
