### Post by punkrockguy318 on 2008-07-02
Hello!

I as well as the fceux team as working on a new version of fceu, the versatile NES emulator called fceux.  This will be the first official release of FCE in years, and I want to be sure that the SDL port works smoothly, especially in Ubuntu.  

This is where you can contribute.  If you have NES roms to test with, go ahead and grab the latest subversion code from our repository:

```
  svn co https://fceultra.svn.sourceforge.net/svnroot/fceultra fceultra  
```

fceu is the directory of the fceux base.  Install this with scons.

gfceux is the directory of my GTK2 front-end.  You can use it out of the directory itself by running

```
python ./gfceu
```

Please give me your feedback on sound, video, or anything that you may run into.  I'd like to resolve all issues I can before the release.

If you have any further questions about compiling just ask away in this thread.
