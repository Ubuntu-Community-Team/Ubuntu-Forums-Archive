### Post by mystics on 2015-09-05
I always have at least two workspaces. They're very convenient for moving windows out of the way when they won't be used for a while and separating windows that are dedicated to different tasks. There are also times when I choose to run a "daytime" and "night time" workspace. In other words, one workspace has a more vibrant wallpaper while another has a darker, more mellow one for very low-light environments.

After that, just a few things that work for me:
1. I always put separate wallpapers on each workspace. This makes it easier for me to keep track of them. Generally, these wallpapers also have a theme to them so I don't have an excessively jarring shift when switching.

2. At least on Xfce, I find options like dragging to different workspaces and scrolling through them to be difficult to manage. As a result, I generally right click on windows and select their workspace if I want to move them, and if I want to switch, I use a keyboard shortcut (in my case, Ctrl+Alt+Directional Arrow, which I think is the default on Xfce).

3. If I'm working on a single task that requires too many windows to keep track of in Alt-Tabbing, I generally move less relevant ones to another window. For instance, when programming, I often have a text editor, web browser, file manager, and terminal all open. I'll often move the file manager and terminal to a separate workspace, or I'll move the web browser instead of the terminal if I'm debugging. Overall, I've generally found it easier to manage Alt-Tabbing when less-relevant windows are out of the way but easily accessible through switching workspaces. That may just be the weird way my mind works, though.

### Post by mystics on 2015-09-06
> **mikodo said:**
> Hi, yoshi2 (like your moniker).

Thanks, for your input. I don't know about the window titlebar "shade" function. What/where is that?

The "Shade" or "Roll Up" option removes the content of the window so that only the title bar remains. There are a few ways to get to it on Xfce.

The first is to right-click and select "Roll Window Up". When you want to "unsahde" it, right click it and select "Roll Window Down".

The second method is by adding a button to the title bar. You can do this in Settings -> Window Manager -> Style. The button looks like an arrow. I believe that, by default, it is labelled under hidden buttons. Just drag-and-drop it where you want on the window in relation to the other buttons.

The third option is to configure keyboard shortcuts. You can go to  Settings -> Window Manager -> Keyboard and locate the "Raise  window", "Lower window", and "Raise or lower window" options. From here,  you can give it the keys you want.

The final is by going to Settings -> Window Manager Tweaks -> Accessibility and selecting "Use mouse wheel on title bar to roll window up". The nature of this option is self-explanatory.
