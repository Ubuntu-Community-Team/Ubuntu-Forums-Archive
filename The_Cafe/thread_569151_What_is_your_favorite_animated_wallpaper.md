You edit the .desktop file for the app

e.g.

In ~/.e/e/applications/all 
I have the following firefox.desktop

[Desktop Entry]
Encoding=UTF-8
Name=Firefox
GenericName=Web Browser
Comment=Browse the Internet
Exec=firefox
StartupNotify=true
StartupWMClass=Firefox-bin
[SIZE="5"]Icon=firefox.edj[/SIZE]
Type=Application



When viewed with the e17 file manager the animated icon appears
