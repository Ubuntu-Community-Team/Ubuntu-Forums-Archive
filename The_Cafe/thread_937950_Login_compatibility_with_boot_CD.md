I had a fairly interesting idea today, and that was having the ability to use the default Ubuntu install disc and be able to log into an already existing account currently on any disk.


This could serve many purposes, such as:

*Easily preserving/manipulating data, and their permissions, during a system failure.
*You could make a limited customizable and portable account on a flash drive so you could have "your computer" anywhere.
*Experimenting with multi-source applications.  For example, maybe if you were to run OpenOffice, that would start from the CD but if you were to use Wine, that might work from the HDD.  I'm sure this could somehow help with development in some way.


So, does anybody like the idea of doing this, or know how to?
