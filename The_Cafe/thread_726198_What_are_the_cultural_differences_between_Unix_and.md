**_What are the cultural differences between Unix and Windows programmers?_**

What are the cultural differences between Unix and Windows programmers? There are many details and subtleties, but for the most part it comes down to one thing: Unix culture values code which is useful to other programmers, while Windows culture values code which is useful to non-programmers.
This is, of course, a major simplification, but really, that's the big difference: are we programming for programmers or end users? Everything else is commentary.[COLOR="Red"][ Read more]("http://www.joelonsoftware.com/articles/Biculturalism.html")[/COLOR]
