### Post by weekend warrior on 2007-04-08
Yes, one of the nicer things in Thunar; been documented for a while now.

From >> [Thunar FAQs]("http://thunar.xfce.org/pwiki/documentation/faq")

> Thunar currently features basic support for so called &#8220;mouse gestures&#8221; in its icon view. You can use these &#8220;mouse gestures&#8221; by holding down the middle mouse button (usually the mouse wheel) while the mouse pointer is on the background area of the icon view component (any area that is not covered by an icon or a text). Now you can move the cursor into four directions to perform certain actions, which are described below. 

Left - opens the previous visited folder
Up - opens the parent folder
Right - opens the next visited folder
Down - reload the current folder
