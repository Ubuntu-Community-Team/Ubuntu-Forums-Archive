### Post by Dragonbite on 2009-04-28
I use Google mail and calendar. I connect to email via IMAP, import or sync with Google Contacts and Sync with Google Calendar.  I have a firewall/router/content filter on my network which I suspect is causing me problems with full bi-directional calendar controls.

I'm using Thunderbird with Lighting, partially because my default view is with the right panel hidden, my Inbox in the middle and my calendar list on the right side.

I can also set it up the same in Linux or Windows for consistency. This proved helpful in moving the family over from Windows to Linux.

Evolution does just as good a job, and is easy to set up with Google Calendars but I'm not so keen on the interface.

At work, I use Outlook 2007 which is very powerful. That's great for work (Exchange server) but overkill for home.
