### Post by Pablo Mouzo on 2007-10-21
First, sorry about my English, is not my first language, I hope you understand what I mean.

I'm an Ubuntu user, Ubuntu is the first distro of Linux I've ever used.

I'm disappointed with the last release, something is wrong here. I'm having lots of problems with the Desktop Effects (Compiz Fusion I think), problems with the configuration (I set something and random things happens), it works really bad. So I was wondering... Am I the only one with problems? No... Some friends of mine had the same problems. It's really integrated with the "system", I try to use Beryl instead and I fail.

But, this isn't what disturb me. If there really are big problems with this effects, why put them in the release? Are you trying to drag people to Ubuntu by giving them shiny mirrors and fancy desktop effects? That's disturbing.

If you are trying to add lot's of functionalities, they must work first!
( In Spanish sounds better: "Si desean agregar funcionalidades, primero tienen que funcionar!! ^^ ).
I want my computer to work fine. If I have to choose between it working just fine so I can use it, or spend hours configuring some fancy add-on... I choose it working fine.

Please, don't loose the linux way. Don't try so hard to put things that have problems into the distro to make it fancy. Don't release a distro that's no ready, that could be really harmful to Ubuntu and it could lose lots fans for just a few good intended but silly mistakes.
You know what happens when something has a lot of shiny things but works like hell ( Yes, I'm talking about Wista ;) ). People use it for a few days, cry, curse and looks for other options. And in Linux world, there are lots of options. So please, KISS ( Keep It Smooth & Simple ).

Saludos,
Pablo Mouzo.
