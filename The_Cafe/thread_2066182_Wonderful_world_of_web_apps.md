### Post by Mikeb85 on 2012-10-04
Well, I might be a little late to the party, but I've discovered just how useful web apps have become.  I used to use only native apps, but lately I've discovered how great web apps are, especially when it comes to syncing between Ubuntu, Windows, and Android.  

I'm currently using Feedly, Google Docs/Drive/other services, Evernote Web, and a bunch of others.  

Are there any web apps that others have found useful?  What are you guys using?  

As an aside, I didn't believe it before, but now I truly do believe that web apps are the future.  As we get more and more fragmented between platforms (various Linux distros, Windows, OSX, iOS, Android, etc...), and as each of us uses more and more platforms, web apps will be the only way to truly connect all these devices.  

So, what is everyone's favourite web app(s)?
