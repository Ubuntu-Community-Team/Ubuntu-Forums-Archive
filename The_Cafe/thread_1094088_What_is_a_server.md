When people say server they might mean several things: the server software (Apache, Samba, IIS, JBoss etc.), the server OS (RHEL 5, Ubuntu server edition, Microsoft Server 2008, etc) or the server hardware.

You can run server-software on your desktop-computer on top of your desktop-OS, or you can choose to run the server-software on a server-computer running on top of a server-OS :)

### Post by Peter09 on 2009-03-12
While any machine can be a server (and serve a service of some kind to its clients), in general servers are machines specifically configured to store and transmit data. They often have limited User interfaces (GUI's), will have lots of secure disk storage, often have multiple network connections, may be clustered or load balanced, and also tend be be configured for security and speed, not user friendliness.

### Post by Johnsie on 2009-03-12
A server is a computer that runs software that other computers can connect to.

With Ubuntu Server you can host your own website. Ubuntu server comes with program called apache which outputs webpages to any computer that can connect to your computer.

Rather than jumping straight into Ubuntu server I think you should just use the regular Ubuntu to host some webpages. You can do this by entering the following command into the terminal:

> sudo tasksel install lamp-server


Once you feel more comfortable with Ubuntu andthe command line you should be more comfortable moving on to Ubuntu-Server. Ubuntu-Server is pretty much the same thing as Ubuntu-Desktop, just without a graphical desktop and with the server software enabled out of the box.

Don't forget, there is a server forum on these boards if you have any quetions or need help with anything.
