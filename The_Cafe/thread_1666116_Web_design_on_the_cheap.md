### Post by linuxforartists on 2011-01-14
> **sanderd17 said:**
> 
If you want a lazy solution: use work that already has been done for you. Use drupal or wordpress. They have a lot of extensions/modules/... and a lot of themes. If you use Drupal, stay with version 6 for a while. Only a minority of the modules have been ported to version 7.

Drupal also has "basic" themes which you can configure from inside your website via your browser.

I agree, content management systems (CMS) are a good route.  Stay away from using office software to design websites.  GIMP and Inkscape would be good for creating mock-ups of what you want your site to look like.

If you're a beginner, I highly recommend going with WordPress for now.  I mean the self-hosted version at WordPress.org.  Great tool for building and managing a website.  It's easy to change text and content later on if you need to.  For a hand-coded website, that can be more difficult.

You can make quite professional-looking websites without much money or time.  Use "themes" (pre-made designs).  There are lots of free WordPress themes, but it's worth the money to buy a "premium" theme.  You can buy a nice theme for US$30-50.  Then you're free to customize to your heart's content.

Extra features can be added through installing WordPress plugins.  But be careful to check the "See what others are saying" comments to see if they have any bad reviews.

Like others have said, it's worth learning some basic HTML and CSS so you have a good foundation in web design.  While WordPress and other tools can minimize coding, there will always come a point when you need to dig into code to install something or fix a problem.  

This book helped me a lot: [Head First HTML with CSS & XHTML]("http://www.headfirstlabs.com/books/hfhtml/").  The diagrams and illustrations made the concepts clear.  The first computer book where I really understood how web design worked.  The Head First WordPress book is great too.

Drupal is a more advanced CMS, so I'd save that for later.  That's for building large-scale news websites, social networks, and similar big projects.

Good luck!
