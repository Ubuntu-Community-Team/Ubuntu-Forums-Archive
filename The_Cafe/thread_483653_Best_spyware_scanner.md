cwshredder

### Post by macogw on 2007-06-25
Mix Soybot S&D and AdAware SE.  What they cover overlaps a bit, but you need both to fill in each one's gaps.  Then add AVG Free Anti-virus and run AVG's free rootkit detector just to be safe.  Finally, install CCleaner and clean up the registry and junk (big speed boost).
