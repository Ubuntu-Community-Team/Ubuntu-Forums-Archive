### Post by TechSonic on 2006-02-02
Last night I was able to configure Ubuntu 5.10 to use a Wireless card thats 'Not Supported'.

D-Link AirPlus G 54  DWL-G510 802.11g
[http://www.dlink.com/products/?sec=1&pid=308](http://www.dlink.com/products/?sec=1&pid=308)

Tell it to use Firewire device when installing! Without it, it wont work.. Not sure why.
Go to > Networking
Wireless > Properties
Go to the part about DHCP and enable that.  Now give the card 10 seconds and then try surfing a page.


Have fun. It was recently said that this card was not compatible.  I found an easy and non in-depth fix for it.

Note: This card setup will NOT work in older versions of Ubuntu.  So far it's been 'setup' and working under Ubuntu 5.10.
Current Fedora 4 and FreeBSD releases do not detect this device.

Any comments?
