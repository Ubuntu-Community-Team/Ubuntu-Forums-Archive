CCLeaner is highly Windows specific. It does three things: Scan Registry, Clean Disk, and Uninstall Software.

Linux has no registry (which is a big binary file that is easy to corrupt), Linux apps typically look after themselves. Apt can clean its own cache, the browsers do their own cleaning, and such. And the package manager deals with all aspects of software installations and uninstallations.

CClearner is a great app for Windows, but it has no use in Linux.
