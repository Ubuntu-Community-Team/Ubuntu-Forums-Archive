### Post by ryno519 on 2007-08-27
Mono is a terrific tool and the .net framework from a developers standpoint is incredibly useful. The upcoming generation of programmers are being raised on the .net languages (C#, VB and managed C++ to a lesser extent) more and more. If we (the GNU/Linux community) make it as easy as possible for these developers to move over to GNU/Linux from Windows and allow them to program in their preferred language, using their preferred framework, all this means for GNU/Linux is more applications. I don't see anything bad about that.

We programmers have nothing to fear from moonlight. The only thing we programmers have to fear is sunlight and caffeine free coke.
