### Post by amingv on 2008-03-14
Actually, they're not version names. They're codenames for the development version and are later "Officially" named after the version number. (If you see the main ubuntu webpage, they do not call it "Gutsy Gibbon", but 7.10.

[https://wiki.ubuntu.com/DevelopmentCodeNames](https://wiki.ubuntu.com/DevelopmentCodeNames)
