### Post by beastrace91 on 2011-01-09
For those who are unaware, Bodhi Linux is a minimalistic, Enlightenment based distro built on top of Ubuntu 10.04

The Bodhi Linux team and I have just released our sixth release and first ever beta release. This version includes a number of package updates including Firefox beta 8 and the latest Enlightenment SVN build as of 01/08/10. For a full change log see [here]("http://www.bodhilinux.com/forums/index.php?/topic/177-bodhi-014-released/"). 

One of the most important changes to this new release is how the initial system layout/colors are configured. For an overview of how it now works see my blog post [here]("http://jeffhoogland.blogspot.com/2011/01/bodhi-linux-hits-beta.html").
