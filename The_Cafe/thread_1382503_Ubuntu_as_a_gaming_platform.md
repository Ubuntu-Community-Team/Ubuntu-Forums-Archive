Ubuntu it's by itself friendly: what can provide the community with is stability, compatibility, reliability (etc) of the platform... then the ones which need to be *friendly* are the game developers+hardware manufacturer(providing decent proprietary driver).


IE: nothing forbidden the game-developers to build their own live-cd distro and compile an OS (from zero, with linux kernel) with just the modules/stuff their product need to run (openGL, audio, internet/networking)!

copy on HD only:
1.driver/modules (which may need to be updated)
2.swap partition (cache and such)
3.save games/settings

then to run the game you just to put the disc in and reboot machine (console way)
