### Post by aysiu on 2008-09-06
My host is ICDSoft, and I love them. Their pricing isn't as competitive as some other hosts (I've heard good things about Site5, but I've never used them). Nevertheless, I've had really good service from them for multiple websites for years.

They don't insist all your files be part of a website, and they even have a special folder just for private files:
[https://www2.suresupport.com/faq.php/17/135](https://www2.suresupport.com/faq.php/17/135): > **What is the Private directory for?**
The **/private** directory is a directory with a special purpose. You can use this directory to store files or create directories which cannot be accessed via the web. (e.g these are files and directories outside of your webroot).

Any files and folders under the **/private** directory cannot be accessed via the web, however you can access them using FTP or using scripts on your web site. You can use the **/private** directory to store files which contain sensitive information such as mysql connection details for example.
