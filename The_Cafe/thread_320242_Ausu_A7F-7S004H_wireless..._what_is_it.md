### Post by mips on 2006-12-17
That series seems to be using Intel wireless chipsets. Why do I say this ? Well  they seem to be centrino certified which means it must have a intel wireless chipset. On the website there is only one driver for the entire A7F- range so that also leads me to believe it is intel.

### Post by mips on 2006-12-17
Ok, I downloaded the driver file for the A7f- series of laptops.

Inside the zip file is an 'apps' & 'driver' folder.

The driver folder only contains two .inf files, NETw39x5,inf & w29n51.inf

I opened the .inf files and the results are as follows:
w29n51.inf=     Intel(R) PRO/Wireless 2915ABG Network Connection Adapter
NETw39x5.inf=  Intel PRO/Wireless 3945ABG LAN Adapters

So there you have it. That laptop will come with one of the above Intel chipsets. Hope this clears it up for you.
