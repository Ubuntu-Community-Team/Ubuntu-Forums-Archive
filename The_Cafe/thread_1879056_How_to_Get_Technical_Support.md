### Post by plethora on 2011-11-11
IE used to be quite the menace. Less so now, fortunately.

Netgear used to sell a router billed as "open source". By this they wanted to encourage people to buy it and install an open-source version of the software, e.g., DD-WRT or Tomato Linux. The first step in doing this is, of course, to log into the router's default software and reflash.

And you would naturally expect that they would let you use your Linux desktop machine to log in and initially configure the router.

Not so, apparently! To do the initial configuration of this allegedly Linux-friendly "open source" router, they required you to use IE.
