### Post by 23meg on 2009-04-06
You can also set up a filter in your mail client or server to save all incoming Launchpad bug mail to a specific folder, and check that folder only when you want to know what's going on regarding your favorite bugs. It's also possible to set up multiple folders for bug mail from different projects, and do more advanced tricks using the mail headers that Launchpad provides. See [https://help.launchpad.net/Bugs/Subscriptions](https://help.launchpad.net/Bugs/Subscriptions) for details.

Note that Launchpad also exposes Atom feeds for each bug report, as well as for all bugs in a project, distribution or package, and all bugs related to a certain person or team. The feeds for individual bug reports are not populated with comments, but depending on your setup, can still be useful for checking the progress.
