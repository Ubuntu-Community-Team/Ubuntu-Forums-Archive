### Post by squakie on 2014-06-20
For others following this thread, I posted earlier today in my thread something others need to be aware of: 

If you are running a 32-bit version of one of the 'bunto's, the default u1_downloader downloaded is 64-bit and will give an error about not a valid exec.  There is a 32-bit version which you can download [here]("https://help.ubuntu.com/community/U1/Downloader/All/Architectures").
