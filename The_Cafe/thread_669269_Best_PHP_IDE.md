### Post by binarybit on 2008-01-16
Free:
[http://www.eclipse.org/pdt/](http://www.eclipse.org/pdt/)
PHP Development Tools framework for the Eclipse platform.
Many in the industry use this platform, including IBM. A contender to the VS.NET IDE. Zend is transitioning it's products to this base.

[http://www.aptana.com/](http://www.aptana.com/)
Aptana Studio Community Edition, JavaScript-focused environment, will do PHP, I think it's also based on the Eclipse Platform.


Commercial:
Zend Studio is probably #1 for a PHP IDE.
