### Post by saulgoode on 2008-04-08
> **nonewmsgs said:**
> i was adjusting my my file system with sudo and the stupid files were now root only, and i was having issues then i was like wait chmod 7....7...7 and perhaps overly excited, but i get like that sometimes late at night.

If what you're speaking of is changing all of your files to make them "world-readable", I would caution that in a few situations this will cause problems. Certain security-related files (private access keys and passcodes, for example) will be ignored by the system if they have the wrong permissions set -- the assumption being that if something private is available publicly, the system has been compromised.

Generally speaking, this is not a concern for the average user, I just would discourage the idea of recursively going through your home directory and changing all the permissions to "777". And I definitely would not recommend it outside of your home directory.
