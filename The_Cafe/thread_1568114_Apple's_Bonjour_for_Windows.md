### Post by floatingworld on 2010-09-04
This has probably been mentioned before but I have to gush about [Bonjour for Windows]("http://support.apple.com/kb/DL999"), Apple's Windows implementation of the zeroconf standard, which works with Linux's avahi.

I just got it working on my Windows system and so I can FINALLY just type "ping myserver.local" to hit my Ubuntu server without using static IPs or anything... I used to have to go fetch the list of DHCP clients from my router to be able to find it.

For anyone else who tried version 1.0 without success Apple released a 2.0 version around June or thereabouts which is what I needed to get it working.
