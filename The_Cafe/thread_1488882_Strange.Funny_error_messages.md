### Post by donkyhotay on 2010-09-29
Never seen it in real life because I use linux, but I've seen screenshots elsewhere showing that when the GNU HURD completely crashes it says it "Bought the farm". Sometimes when working on my own software I'll put interesting messages into the error logs. Most often when I'm trying to track to confirm whether a value is getting a value that it shouldn't get for what I'm using it for I'll do something like:

if (variable == an_impossible_value):
logging("If you're reading this you're hosed")

to confirm my program isn't coming up with values that are "impossible" and part of my problem. Of course if I get that message it usually tells me exactly what's wrong. Those are always temporary for fast troubleshooting and removed before distributing... unless I accidently forget of course.
