### Post by Bragador on 2006-02-15
> TransGaming Technologies today announced the release of Cedega 5.1, which features support for some of the newest blockbuster PC titles such as **Sid Meier&#8217;s Civilization IV**, FIFA 06 and Need for Speed: Most Wanted.Wine Games

Cedega allows games originally created for the Windows platform to run on Linux, straight out-of-the-box. Other popular titles supported on Cedega 5.1 include **Battlefield 2**, Dungeon Siege II, City of Villains, Madden NFL 2006, **World of WarCraft**, Half-Life 2, Guild Wars, and many others.

Among the many enhancements included in Cedega 5.1, Linux gamers will also benefit from a range of new features and fixes including an updated Cedega user interface, installer and copy protection improvements, support for wide screen monitor resolutions, and better handling for international keyboard layouts.
