### Post by Jucato on 2006-07-19
Eye candy is not evil as long as your system can handle the amount of eye candy you are using. There are some styles/themes that are also less resource intensive.

If you're interested in using a very lightweight desktop while still getting (almost) all the GNOME/GTK+ goodness, you could definitely try out Xubuntu (although I can't personally recommend using Thunar just yet).

If you do consider eye candy to be evil, you might be more at home with using lightweight window managers instead of full desktop environments.
