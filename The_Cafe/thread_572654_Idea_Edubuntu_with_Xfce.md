### Post by maybeway36 on 2007-10-10
This was proposed a while ago, but apparently evaporated...

Basically, it would provide a metapackage for installing Edubuntu with Xfce. This would be very helpful for schools with old computers (i.e. all schools), since Xfce is designed to run fast.
It's actually not too hard to make an Edubuntu-themed Xfce anyway; you just install edubuntu-desktop and xubuntu-desktop, then change the style to EdubuntuColors, the icon theme to Gartoon, and the desktop background to /usr/share/backgrounds/edubuntu_default.png. (And probably take away some of the desktop icons so it's not so confusing.)
