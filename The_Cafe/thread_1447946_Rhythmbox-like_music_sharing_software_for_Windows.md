### Post by ssam on 2010-04-06
the protocol is called DAAP, lots of software supports it:
[http://en.wikipedia.org/wiki/List_of_software_using_Digital_Audio_Access_Protocol](http://en.wikipedia.org/wiki/List_of_software_using_Digital_Audio_Access_Protocol)

note: older versions of iTunes would work with any DAAP program, but new versions check that it is iTunes on the other end. So if you want a mix of programs avoid iTunes.
