How do you do a thumbnail?

If you use an image uploading service like omploader or imgur, they give you a thumbnail.

You can "nest" the img and url bbcode tags, like ```
[[img]path_to_example-thumb.png[/img]]('http://example.com/image.png')
```

using a smaller image in the img tag, and the larger one as the link.

### Post by ZarathustraDK on 2011-05-18
> **potat said:**
> Is this a built-in feature? I'm not using 11.04 yet but I've wanted to do this in Compiz for a really long time.

If you install the extra plugins package for compiz you'll get a "Wallpaper"-plugin in CCSM that you can enable and configure to accomplish this.

You also need to disable Show Desktop in gconf-editor, otherwise Gnome-desktop will cover up the compiz-wallpaper, you'll also loose your desktop icons by doing this. There is no easu way to keep icons on the desktop and getting different wallpapers at the same time.
