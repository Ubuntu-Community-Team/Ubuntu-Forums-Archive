### Post by lovinglinux on 2010-10-20
Firefox 4 will be (already is) an awesome browser. It has many interesting new features, a much better performance and some radical UI changes. Since pleasing everyone is almost impossible, some of these changes are making some people angry or disappointed. Fortunately, Firefox is the most powerful browser in terms of extensibility/customization and with such large community of extension developers, there are already extensions to bring some features back.

The idea of this thread is to share those extensions.  If you don't have Firefox 4 and want to try it, see my tutorial [http://firefox-tutorials.blogspot.com/2010/08/firefox-4-download-install-customize.html](http://firefox-tutorials.blogspot.com/2010/08/firefox-4-download-install-customize.html)

So here are some extensions I was able to find:

[**Status-4-Evar**]("https://addons.mozilla.org/en-US/firefox/addon/235283/") - brings back several statusbar (defunct) features and allow to hide or personalize the address bar target link display and loading bar (awesome extension).

[**Link Target Display** - ]("https://addons.mozilla.org/en-US/firefox/addon/55724/") displays link targets in a tooltip at the bottom of the browser window if the status bar is hidden (***this one if for those who don't want Status-4-Evar, but don't like the link target display on the address bar. This conflicts with Status-4-Evar***).

[**Browse By Name**]("https://addons.mozilla.org/en-US/firefox/addon/222531/") - re-enable "Browse By Name" awesome bar feature, which is what allows to type *ubuntuforums* in the address bar and enter this site directly, instead of being redirected to a Google search result page.

[**RSS Ico**n]("https://addons.mozilla.org/en-US/firefox/addon/241029/") - adds the old FF 3.X RSS Icon behavior back into FF 4, which was removed from the awesome bar and placed as a new button in the toolbar.
