### Post by dizee on 2008-05-01
The Heron is excellent, and Xubuntu's purple/blue gradient one is great aswell (if a bit Mac OS-ish ;) ).

BTW you can change the panel text colour by adding an entry to your .gtkrc-2.0 file
```
style "panel"
{
fg[NORMAL] = "#FFFFFF"
}
```
where #FFFFFF is the html colour code.
