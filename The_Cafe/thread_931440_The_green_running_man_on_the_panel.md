### Post by twisted_steel on 2008-09-27
There is a SVG version located at

```
/usr/share/icons/gnome/scalable/actions/system-log-out.svg
```

There are also some fixed size versions at

```
/usr/share/icons/gnome/16x16/actions/system-log-out.png
/usr/share/icons/gnome/22x22/actions/system-log-out.png
/usr/share/icons/gnome/24x24/actions/system-log-out.png
/usr/share/icons/gnome/32x32/actions/system-log-out.png

```

According to [this page]("http://svn.gnome.org/viewvc/gnome-icon-theme/trunk/COPYING?view=markup"), the entire theme is GPL v2.
