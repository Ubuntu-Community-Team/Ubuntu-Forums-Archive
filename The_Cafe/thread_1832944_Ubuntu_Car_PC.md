### Post by BrokenKingpin on 2011-08-25
I would recommend running KDE (Kubuntu, or custom KDE build on top of Ubuntu). I know KDE is a bit heavy, but you can really customize the UI. Firstly, set the UI layout to the netbook view. This view works great with a touch screen, and makes it quite easy to launch applications. From here you can customize the desktop widgets and workspace to do whatever you want... media controls, etc. 

You could also look at KDEs Marble application/widgets for navigation:
[http://edu.kde.org/marble/](http://edu.kde.org/marble/)

Alternatively you could run a lighter DE/WM like Xfce or LXDE and then just run a full screen media center that has a good interface for touch controls.
