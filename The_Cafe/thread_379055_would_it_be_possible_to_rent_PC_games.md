### Post by XeroZohar on 2007-03-08
Well, one way it could be done is temporary activation codes, used on downloaded games, or even special CD/DVD install discs.  With the moves to digital distribution I see this possibly working with things like Steam.

I'm not sure this would be the best method given that there are activation code crackers out there, though, unless the codes were randomized and the type (full, rental) only identified on a server, so you didn't know what kind of code you were getting until it was too late.
