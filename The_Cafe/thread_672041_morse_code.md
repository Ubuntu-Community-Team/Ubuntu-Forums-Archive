### Post by Namtabmai on 2008-01-19

Binary is a simple on/off | 0/1 concept, where as morse code has on/off/long/short as well as different length pauses between.
The wikipedia entry on [morse code](http://en.wikipedia.org/wiki/Morse_code) lists 5 different states.
