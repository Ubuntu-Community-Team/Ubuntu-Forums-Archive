### Post by jlb.think on 2010-01-18
The best things about Ubuntu are that is does what I want, and it has the programs I like.  I program as a hobby and it is much easier to program on Linux than in windows.  There are better tools, more help, and it is easier to get a good IDE.  Personally I use gcc and VIM, but if I was to use and IDE I would take Eclipse any day over Microsoft Visual Studio.  I had to use Visual Studio for a class of mine and trust me, VIM is a better an more versatile editor, and I got work done a lot faster than my teachers and classmates using Linux than Windows.

The downsides are lack of support by vendors.  I have a blackberry and use it for tethering and I use berry4all.  Before my blackberry I couldn't tether to linux, I don't remember the model of phone I had, but I could never successfully tether it, and I couldn't find anyone on the internet who had.  I had to use Windows to surf the internet for about a year.  Talk about a terrible paranoid experience!!  I was constantly worried.  Fifty million security programs, constant scans, etc.  I managed to make it a full year without a single piece of malware making it onto Vista, which is my personal windows record.  I couldn't make it more than six months with XP without having a problem.  So windows security has improved a lot, but I'm glad to be back online with Ubuntu where I haven't had a mal-ware problem four years running (I've never had a malware problem in eight years of using Linux actually).

EDIT:  If you think it is hard to get some things to work under Linux now, you should have been around eight years ago.  Ubuntu is an incredibly easy and robust operating system.

### Post by aysiu on 2010-01-18
Depending on what you want to achieve by posting "criticism," there are various different appropriate places to do so.

**Brainstorm**
If you have an idea for how to make Ubuntu better, post it to [http://brainstorm.ubuntu.com](http://brainstorm.ubuntu.com)

Truthfully, most ideas that are any good have already been posted, and a lot of terrible ideas have been posted as well. Who knows? Maybe you do have an actually original idea. Suggest it. See what others think. See what the Ubuntu developers think. Maybe it'll happen.

**Bug Reporting**
If you think there is a defect in Ubuntu (not a strategy or implementation you disagree with but a behavior you believe the developers did not even intend to have happen), report it as a bug to the developers:
[http://www.ubuntu.com/community/ReportProblem](http://www.ubuntu.com/community/ReportProblem)

**Venting**
If you don't actually want to make a real, concrete and useful suggestion, there are various places you can vent. On the forums here, there are two places to do so. If you want to talk about a bad personal experience you had with Ubuntu, post it in the [Ubuntu Testimonials and Experiences subforum](http://ubuntuforums.org/forumdisplay.php?f=103). If you want to post a general rant about Ubuntu in general, post it in the [Community Cafe](http://ubuntuforums.org/forumdisplay.php?f=11).

Then, of course, there are plenty of places outside of the forums you can vent. Get a blog. Find a forum dedicated to hating on Ubuntu or Linux. They exist.

**Contribute**
Don't forget, too, that you can actually do things to make Ubuntu better apart from just criticizing it. You can make good documentation for it, donate money to it, or support it in other ways (especially if you're a programmer).

I tend to lean more toward suggesting folks do something productive, but I also recognize there is a place for venting and just letting off some steam. If you're frustrated, vent away. Won't change anything in Ubuntu. Maybe it'll make you feel better, though.
