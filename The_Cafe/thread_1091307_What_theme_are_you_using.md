### Post by OrangeCrate on 2009-03-09
Slateman-Murrine from gnome look:

[http://gnome-look.org/content/show.php/Slateman-Murrine?content=85844](http://gnome-look.org/content/show.php/Slateman-Murrine?content=85844)

To apply themes to root applications, use these commands at the terminal:

sudo ln -s ~/.themes /root/.themes
sudo ln -s ~/.icons /root/.icons
sudo ln -s ~/.fonts /root/.fonts

[http://tombuntu.com/index.php/2007/09/18/theme-applications-running-as-root-in-ubuntu/](http://tombuntu.com/index.php/2007/09/18/theme-applications-running-as-root-in-ubuntu/)
