I'm afraid I have pestered you guys with Pipsqueak before, but I'm not about to resurrect a three-month-old thread to say "done". Pipsqueak is based on ttylinux, so it's somewhat familiar to some of you, but a lot has changed in the startup systems and a few of the apps. Anyway, enough yaking. It includes the latest kernel, 2.6.34, it's a 15MB ISO, and you can download it here: 

[http://geekinsnthings.homelinux.org/pipsqueak](http://geekinsnthings.homelinux.org/pipsqueak)

The download speed is somewhat slow, so sorry about that.
The new kernel hangs sometimes when installed to a hard disk in VirtualBox. QEMU is fine. This release is 1.0.0 stable, "Chipmunk".

Hope someone finds it interesting/useful.
