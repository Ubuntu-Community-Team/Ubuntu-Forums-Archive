### Post by Diluted on 2009-11-22
I would recommend you start with a business plan. Second, you will need some sort of terms and conditions to protect yourself. You can either get a lawyer to do this or use services like HostLegal.

As for which hosting to start with, it depends on how proficient you are with Linux administration (assuming you are using Linux). If you aren't confident, then start off with a reseller.

There's a lot more you can read about a hosting business. I'd highly recommended you head over to Web Hosting Talk and look around their forums. Not only are there tutorials for this stuff, there are also customer testimonies for current web hosts and it's worth looking at them to see the problems in the hosting business.

That being said, shared hosting is regarded to be hard to break in nowadays. If you think you can find a niche to differentiate yourself from dozens of web hosts popping up every day, then go for it. Be sure to read about overselling first though.
