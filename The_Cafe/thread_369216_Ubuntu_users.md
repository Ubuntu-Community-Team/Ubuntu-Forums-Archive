> 1] Linux Vs Windows User Graph

I doubt you'll find any statistics for 1, 2 and 3. The reason is that Ubuntu (and other distributions) are free of charge, so no one can know how many users there are. There is this site:

[http://counter.li.org/](http://counter.li.org/)

But I don't think that every Linux user is registered there (I'm not). Here are some more estimates:

[http://counter.li.org/estimates.php](http://counter.li.org/estimates.php)

There's something similar for Ubuntu as well, but I don't think most users are registered on it either:

[http://ubuntucounter.geekosophical.net/index.php](http://ubuntucounter.geekosophical.net/index.php)
