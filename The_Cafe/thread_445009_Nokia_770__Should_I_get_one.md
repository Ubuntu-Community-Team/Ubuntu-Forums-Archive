### Post by rggavubt on 2007-05-15
I have a 770 and like it very much.  I don't like that you can't install Java on it.  I want to play some crossword puzzles that require Java and can't.  The 770 has now been replaced with the N800.  I would purchase the newer one.  There is lots of software you can install on the device.  The 770 has an opera browser.  There is more available software for the 800 and software development for the 770 is slowing down.
