### Post by Oliv76 on 2008-08-25
Hi,

I am pleased to introduce my project: Cooperation-iws which is a nomad Intranet Web Server project that relies on Ubuntu Live System.

The aims of Cooperation-iws are to provide a turn key Web Server for Intranet Purpose, you could take with you on a removable media (Usb Key, Hard drive, Cdrom).

Cooperation-iws offers therefore web services of open source web applications like Wordpress, Mediawiki, Trac, KnowledgeTree, Nuxeo, EzPublish, ...

Cooperation-iws is available with 2 Demo Live CD /DVD and live-scripts based on Reconstructor ([http://reconstructor.aperantis.com](http://reconstructor.aperantis.com)) to easily create your own server or to transform Demo CD into Live USB.

The project is subject to GPL2.

More info on the blog of the project: [http://www.cooperation-iws.eu/](http://www.cooperation-iws.eu/)

See you on the web,
