### Post by libssd on 2010-05-25
> **mamamia88 said:**
> I have a samsung n130 netbook running windows 7.  at 1 time the most i have open is firefox and maybe one other application.  do you think it would be worth $50 to add an extra gb ram?
I'm not sure I understand the question. Do you want to add more RAM for Windows 7 or for Ubuntu? 2gb seems to be the practical minimum for Windows 7. For Ubuntu, 1 gb should be ample. 

Looking at memory in use, System Monitor is currently showing ~350mb out of 1 gb, zero in swap. I rarely see RAM usage spike about 700 mb, almost never see swap being used. FireFox has a reputation as a memory hog; you might want to look into Chrome, which is much faster and lighter on its feet.
