[B]Microsoft honoured for Inuit language software
[/B]
Last Updated: Tuesday, February 16, 2010 | 3:35 PM CT

The Nunavut government has recognized Microsoft Canada for its work in developing Inuktitut-language software.

As a result of the software giant's work, Windows computer users can freely download versions of Windows XP and Vista, along with Microsoft Office 2003 and 2007, in the Inuit language.

The Inuktitut versions of Vista and Office 2007 were launched last spring...

Nope, no Inuktitut translation, but there *is* a translation project on Launchpad:
[https://translations.launchpad.net/ubuntu/quantal/+lang/iu](https://translations.launchpad.net/ubuntu/quantal/+lang/iu)

It would be interesting to see a push towards supporting languages like that. It would need some extra resources and a lot of outreach, since the actual work would mostly need to be done by people outside the usual user/translator community.

Building the [Haida translation]("https://translations.launchpad.net/ubuntu/quantal/+lang/hai") would be another opportunity to spread some good will. Like a lot of others, the Haida language is barely holding on - it has a number of native speakers measured in tens. It's very sad. I bet it would really help to offer an operating system that supports their language, even if the translation is incomplete.
