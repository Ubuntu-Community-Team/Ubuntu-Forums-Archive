### Post by CharlesA on 2010-11-11
If you want to keep a backup of Windows 7 in case you need to send the machine in for service at a later date, I'd suggest just imaging the drive and then wiping the hard drive.

1) Most OEMs don't include install discs anymore since they all have recovery partitions on the machine that will wipe the drive and reinstall the OEM install of the OS.

2) It's up to the OEM if they want to provide recovery disks, most of the systems I've looked at include a way to create yer own recovery disks inside on Windows.

3) Recovery partition vs install disk, see above.

4) Clone the drive first, that's what I do with any new machine I get.

5) Spelling it phenetically makes sure that they spell it correctly, letters can sound the same and be spelled differently.

6) Then don't use Samsung products if you don't like their customer service.
