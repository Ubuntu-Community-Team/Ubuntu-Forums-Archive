### Post by BC59 on 2011-12-06
Is available to download the latest Wally version with Ubuntu 11.10 Unity integration.

Wally Features:
[LIST]
[*]change wallpaper manually / automatically (on a given interval)
[*]rotate images according to EXIF information
[*]set wallpaper position (centered, tiled, etc.)
[*]you can set downloaded photos to be at least 1/4, 3/4 or larger than the desktop size
[*]stores downloaded images on the disk so you can use them later on
[*]more
[/LIST]

You can download it from here:
[http://www.becrux.com/index.php?page=projects&name=wally](http://www.becrux.com/index.php?page=projects&name=wally)

It's a .deb file installed with double click and through Ubuntu Software Center. The Center has an old version of Wally which lacks Unity integration.

When installed open the Dash, right Wally and click on the icon.
Then you can see the small icon on the top bar. Click to change the settings to Play automatically on application start.

Now Wally has to be added to the Startup applications. Open from the upper right Menu the Startup applications. 
Then from the Dash find Wally and drag and drop the icon inside of the open window of Startup applications.
