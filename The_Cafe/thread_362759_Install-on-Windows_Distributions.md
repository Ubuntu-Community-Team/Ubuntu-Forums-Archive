### Post by Thrashers7989 on 2007-02-16
Anybody ever heard of/tried these?

The most available one is Topologilinux - [http://www.topologilinux.com](http://www.topologilinux.com)

Then there's smaller, defunct distros that do this. Google WinLinux, PhatLinux, DragonLinux

Topologilinux was a scary one. First of all, complete pain in the butt to install nVidia drivers. You need the drivers to run Gnome, otherwise, you had more than 4 GB of XWindows software installed, but only a command line to work with. I figured out how to work it in 4.0, but once 5.0 came out, they switched XFree86 with Xorg which made it even more impossible to install any video driver.

So I gave up on Topo for a while until I got a laptop with a generic display driver. I gave Topo another try. I was able to get XWindows running without compiling any drivers, but then I ran into more problems. Apparently, this distro did not like my network and refused to connect via DHCP. Then there was compiling. I don't know if Topo is capable of compiling source code. That might sound stupid, but I never successfully compiled anything. A friend of mine who is very experienced in Linux command line down-to-the-core looked at it and after an hour of investigating, was able to compile one stupid little program after much tweaking. Not even he could figure out how to install my wireless network card. The issue wasn't me being a n00b, apparently, it was a distribution that had so many damn bugs and glitches and compatibility issues.

The support is TERRIBLE. There's one new post on their support forums every month and you could see that their front page is not updated very often. They don't even have a dedicated FTP server...constantly switching from donor to donor.

Topologilinux gave me a bad impression of Linux altogether. I hated Micro$oft, but I was intimidated by Linux, and I couldn't install Mac OS X on a PC. I stuck with Windows suXP for a good while. Long story short, got Vista, hated it with a rotten passion, and checked out Ubuntu. And here I am.

Anybody else try these distros and have horror stories?
