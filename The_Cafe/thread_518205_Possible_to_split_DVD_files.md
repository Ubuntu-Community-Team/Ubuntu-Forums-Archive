### Post by gn2 on 2008-09-01
You could just rip it as an .avi with [**dvd::rip**]("http://exit1.org/dvdrip/") which will result in a much smaller file size.

dvd::rip is in the repositories:

```
sudo apt-get install dvdrip
```
