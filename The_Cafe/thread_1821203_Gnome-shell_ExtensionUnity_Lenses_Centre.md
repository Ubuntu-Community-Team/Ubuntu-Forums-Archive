### Post by akand074 on 2011-08-08
I think they should really have a spot in the software centre/it's own software centre for extensions and lenses. Or at least have a nicely designed web page like firefox addons/chrome extensions for them. Currently I find it very difficult to find out what lenses/extensions (even indicator applets) even exist and then I'd have to add a new ppa for **each** one I want to try. They should really organize it and make it easy to access. That would even give people more incentive to make them, especially if they provide decent developer tools. Those things, especially gnome-shell extensions have huge potential. I think Ubuntu should set one up themselves (into their repositories perhaps) that store Unity lenses, gnome shell extensions, even KDE activities and such all in one place. I'd be okay if gnome/ubuntu/kde made their own just for theirs, point is we need good extensions and an easy way to find them and install them.

Has anyone heard of anything in regards to this topic? I've seen the gnome's extension page but that hardly counts.
