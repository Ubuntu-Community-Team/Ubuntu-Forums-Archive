### Post by WorMzy on 2010-06-07
[QUOTE="http://en.wikipedia.org/wiki/Getdeb"]
In recent May, 2010 the server for Getdeb has crashed and as of June 4, 2010 the server is down. The server crashed on February 28, 2010 and was down until March 9, 2010. The official blog for Getdeb gives a public notice of downtime and states they hope the sources will be up and running in a few days.

As a Ubuntu software portal software package repository and source, the downtime of Getdeb, as with other repositories, cause slight panic to Ubuntu software portal users as they can not install or update software that is provided through the Getdeb.[/QUOTE]


I know it's a wikipaedia article, and therefore not 100% reliable, but it looks like they're having server issues.
