### Post by regeya on 2006-09-02
Nothing very exciting, in fact pretty standard colors for a Ubuntu system, but this colorscheme/theme is something I'll put somewhere at some point, when I get done with it.  I'm intending it for people like me, who're using a gimpy old box as their main axe, yet don't want to go full-on with a spartan windowmanager or even the XFCE route (though XFCE is excellent.)

Colorschemes inspired by Human, leaning more toward the earthen tones than orange.  I'm using Openbox instead of Metacity, and I'm using Thunar instead of Nautilus for local filesystem browsing.  The Openbox theme is modified from the standard mist theme, and the gtk theme is modified from the SouthSide scheme to be more Human, with ThinIce scrollbars.  Icon theme is the near-ubiquitous NuoveXT theme.

Since it's a mix of Mist and Human, I'm thinking of calling it Humid.  ;)
