### Post by lswb on 2008-08-24
Well, if you need some excitement in running ubuntu, here are some things to try:

Use Network Manager to create and/or connect to an ad-hoc wireless network

With Network Manager running, connect to an ISP using ppp and then have evolution start up in online mode.

Get a Win CE or WM device and have it sync to evolution or your PIM of choice using a USB cable. Then do it again using bluetooth.

From the gnome desktop, activate the normal logout-shutdown-etc. window, then close it by using the escape key. Now try to restart X by pressing Ctrl-Alt-Backspace.

Install the latest Adobe flash player, then start firefox and go to [www.paypal.com](www.paypal.com)

If these are not enough, I can suggest some more!
