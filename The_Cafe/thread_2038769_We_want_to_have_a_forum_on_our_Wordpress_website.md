### Post by coffeecat on 2012-08-07
[Mingle forum ]("http://wordpress.org/extend/plugins/mingle-forum/") is a plugin for Wordpress which allows you to have a forum on one of your Wordpress pages. It's basic but does the job within its limitations. When I tried it on an admittedly short-lived website earlier this year, the skins that were then available for Mingle clashed nastily with the theme colours I had set up. I got round that by hacking the various gifs in the images folder to at least match the site colour scheme.
