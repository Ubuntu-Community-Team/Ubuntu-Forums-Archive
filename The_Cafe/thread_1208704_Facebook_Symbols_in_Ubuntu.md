### Post by FuturePilot on 2009-07-09
Try Application > Accessories > Character Map. You can copy and paste them from there. Or if you know the Unicode number you can use Ctrl+Shift+U+unicode#. For example Ctrl+Shift+U+263A is a smiley face [SIZE="4"]&#9786;[/SIZE]
