### Post by Paqman on 2008-09-01
> **jeyaganesh said:**
> Web builders should aware of this.


They are. Screen width is a fairly controversial issue in web design.

One problem is that the human eye doesn't really follow very long lines of text well. Conventional wisdom is that narrow columns of 6-8 words per line is easiest to follow, which is why newspapers are formatted that way.

However, a lot of the conventional wisdom doesn't work the same in print as it does on screen (generally because screens have much, much lower resolution than print) Also, many people publishing web content have no formal training. So you tend to see a huge variety of styles across different websites.

Only time will really tell what the best way to format text on the internet is. Personally I prefer fixed-width columns of no more than 10 words, but a lot of people like to use dynamic formats and leave it up to the user to define their own width by resizing the browser window.
