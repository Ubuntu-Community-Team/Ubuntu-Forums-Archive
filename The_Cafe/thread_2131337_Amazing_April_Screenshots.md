### Post by proycon on 2013-04-27
> **m4gnus said:**
> Gorgeous, thank you for the post. I recently have been wanting to switch to openbox as well. 
Did you run into any issues? I notice you're on xubuntu 12.10, that's exactly the switch I wish to do on my laptop.

Depends on what you consider 'issues'. Openbox is quite bare (I always thought too bare but I now like it!) so it does all require some manual configuration to get it precisely the way you want it, mostly editing openbox's rc.xml, an autostart script that starts tint2 or whatever dock you prefer (if any), sets a background (feh), and other stuff you might like. You may also need to add menu entries or key bindings for suspend/hibernate/screensavers etc. One special tweak I'm using is  [http://openbox.org/dist/tools/setlayout.c](http://openbox.org/dist/tools/setlayout.c) , it configures your workspace layout to be 2D (3x3) instead of 1D. Allows for easy navigation using super+arrows. I also configured keybindings that allow windows to be quickly 'tiled'. And I really like synapse, it doesn't get in the way and is a great tool for launching apps and all kinds of recently used files. Another thing that doesn't get in the way and which I use heavily is guake, the drop-down terminal. As login manager, I use slim instead of lightdm.

The only thing I still really miss is a fancy zoom function (I used to be on compiz, without unity, and I loved the zoom function)...

I'm using the xubuntu branding currently.. but without XFCE it isn't much xubuntu of course. It doesn't really matter what ubuntu variant you're using, just take the configuring into your own hands.

Since this is a screenshot thread, a minor tweak from my last post, besides having upgraded to 13.04 today: I moved the tint2 panel to the top :) All the rest still looks the same as in my [previous post](http://ubuntuforums.org/showthread.php?t=2131337&p=12584025#post12584025).
