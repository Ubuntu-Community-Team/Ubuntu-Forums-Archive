### Post by meh_phistopheles on 2011-10-28
i know this is solved, but i would just like to voice my opposition to this apa template thing. my justification is that the only thing you need to know about setting up an apa document is 1 inch borders with a 1/2 inch header, double spaced, and 12 point times new roman font.

that's it. aside from the initial page setup though, you're going to need to know how to properly cite and create a reference page, and you are not going to learn those things from a template. so since you have to read about citing in apa anyway, and since the initial page setup is so simple, you may as well just figure out how to setup your page yourself so you have the skill if you ever need to use a different style of paper format.

btw, here is a good reference for citing in apa: [http://owl.english.purdue.edu/owl/resource/560/01/](http://owl.english.purdue.edu/owl/resource/560/01/)
