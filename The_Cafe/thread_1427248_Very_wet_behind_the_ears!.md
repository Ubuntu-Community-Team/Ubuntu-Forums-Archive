### Post by Paddy Landau on 2010-03-11
I've just read your blog.

If I may make a couple of hints for the brand new newbie, to make your life easier!

To install a new program, you don't look for it on the Internet and download it and install it (that's the Windows way). Instead, you open your package manager (System > Administration > Synaptic Package Manager) and install from there. (There are a few exceptions to this rule, but they're rare.) Packages with the Ubuntu logo have been tested and are virtually guaranteed to work. Most other packages seem to work, not that you're likely to need them.

Likewise, to uninstall a program, you go (again) to the package manager and uninstall from there.

You can run *some* Windows programs on Linux, but not all. If you want iTunes, you can run version 7, not the latest version. iTunes was very carefully designed to run only on Windows and Mac. If you're interested in learning how to install iTunes version 7, post here and I'll show you the easy way to install it.

Finally, if you run into problems, search these forums, because the chances are that someone else has already asked. Google is also a good friend. If you're still stuck, post a new thread in the ABT (Absolute Beginner Talk) forum.

Yeah I tried to post on your blogspot but forgot my username & password. annyway.. I'll post it here

```

sudo apt-get install ubuntu-restricted-extras

```

(That's assuming you installed the 32 bit version)

That *COMMAND* (NOT code) simply installs various codecs, fonts  and flash that are not permitted to be distributed by default.

The sudo bit means you are donning your administrators hat and as so it will demand your password which you wont see on your screen as you type it in.
