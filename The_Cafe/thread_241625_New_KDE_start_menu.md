### Post by RavenOfOdin on 2006-08-22
This argument about similarity to XP design is idiotic.  Microsoft doesn't have a monopoly on design ideas, never mind that they would like to.

Here are the differences I see, and I'm only reacting from the "sneak preview" shot:

**KDE4**: Search bar on top, and Help bubble.
**XP**:  Find and Help buttons on bottom right.

**KDE4**: Menu divided into top, middle with 5 tabs on bottom, and bottom. Applications accessible through upward-moving structure without folders.
**XP**: Menu divided into top, top left (for pinned programs), bottom left (for recent programs), and right for computer functions. "All Programs" 
accessible on bottom, which springs forth a tree structure of folders to the right.

**KDE4**: Leave button as 5th tab on right.
**XP**: Switch User and Log Out buttons on extreme bottom.

**KDE4**: Listing "user xxx on xxx" on bottom next to lock screen button.
**XP**: User listing replaced by little .jpg on extreme top left with user name next to it.

That's all I can remember at the moment, but I'm sure I can think of more.

For my part, I like this new design.
