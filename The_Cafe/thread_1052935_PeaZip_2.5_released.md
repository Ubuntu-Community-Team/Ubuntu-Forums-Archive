### Post by Giorgio tani on 2009-01-28
PeaZip 2.5 was released earlier today, full change log on [http://peazip.sourceforge.net/](http://peazip.sourceforge.net/)

Context menu was reorganized, and "Open with" submenu allows to define up to 16 custom applications and scripts to run selected files (or archived items) with.
The application can be minimized to tray icon, and can alternatively display extended characters in archived file names or replace them with "?" jolly character (to avoid possible issues between archive's and system's character encodings).

In the package, in Freedesktop_integration folder, are now featured Nautilus scripts to launch main application's function from context menu in Gnome.
