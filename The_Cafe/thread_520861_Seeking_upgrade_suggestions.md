### Post by ssam on 2007-08-08
> **jimbob said:**
> I guess it is possible to go into my sources.list and change all the Feisty names to Gutsy and do an upgrade but it has been my experience in the past that this does not appear to get everything

that is not a recomended method.

you should always read the release notes eg [http://www.ubuntu.com/getubuntu/releasenotes/704](http://www.ubuntu.com/getubuntu/releasenotes/704) . these explain the recomended upgrade method (there is a comand line method as well GUI).

Also if you want the updates to go smoothly then be care with installing third party stuff, or doing customisations to the system. for example if you have install graphics drivers from somewhere other that the ubuntu repos, dont expect them to work after the upgrade.
