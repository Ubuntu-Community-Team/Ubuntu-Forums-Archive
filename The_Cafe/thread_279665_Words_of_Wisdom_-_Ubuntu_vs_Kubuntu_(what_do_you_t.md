### Post by Engnome on 2006-10-18
If you need a webserver but do not feel comfortable setting it up using the command line you could install a ligth window manager like fluxbox Icewm or maybe even XFCE core. Running a server with a complete (k)ubuntu-desktop might be easier but will waste alot of resources.

For more info about this: [https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)

For a comparison between GNOME and KDE: [http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)
