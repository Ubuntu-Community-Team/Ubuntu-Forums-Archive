### Post by lovinglinux on 2010-07-26
I have been following the discussions about Firefox here for a while, since Firefox 3.5 was born. By that time everyone wanted it and soon tons of threads asking how to install Shiretoko appeared in the forums. When Firefox 3.6 was released, it was no different. This inspired me to create [my tutorials]("http://firefox-tutorials.blogspot.com") and since then I have been more and more involved with Firefox in these forums. 

One series of threads that bothered me considerably, was the complains about the fact that Ubuntu doesn't provide updates for major software versions, until a new version of Ubuntu is released. Some of the discussions were reborn over and over, to a point that I couldn't even reply anymore.

Well, because of changes in Mozilla's update policy, [Firefox 3.6 has been pushed to all supported Ubuntu versions]("http://ubuntuforums.org/showthread.php?t=1500861"). What is interesting about this entire story, is that now lots of users wants to downgrade Firefox. I have seen several threads asking how to do it and also a few complaining that Firefox 3 and 3.5 were much better.

I understand that nobody wants to use a software that doesn't work as expected and everyone wants the latest versions of everything, but this situation is at least a lesson to all those people that complained that Ubuntu is not a rolling release and that they can't get the latest versions of their favorite applications from the regular repositories.

I don't know who is to blame for this situation. Mozilla has released two versions in a very short period to address issues in Firefox 3.6.4 and 3.6.7, mostly because of the plugin isolation feature that was pushed to 3.6 series instead of Firefox 4. But I believe the fact that Canonical didn't develop Hardy, Jaunty and Karmic with Firefox 3.6 in mind, has a big role in this situation. On top of that, there is also Adobe, which keeps screwing Linux users, by dropping support for flash 64bit and by releasing a crappy 32bit plugin. 

That being said, my journey with Firefox has been joyful for the entire ride. I have experienced some minor issues here and there, but overall Firefox has been improving in every major release, specially in regard to performance. Some of you might have already seen my posts, saying that I have more than 60 extensions installed and I have no performance issues whatsoever. I don't use a particular version of Firefox all the time, since I do a lot of extension testing and also tests to help other users. I have already used several different versions and sources of Firefox, like official and not official ppa repositories, binaries downloaded from Mozilla, source code which I compiled myself and even Swiftfox. 

Soon we will see the birth of Firefox 4, which is already at beta stage and is looking really good. Although the new interface hasn't been introduced into the Linux version yet and most of the extensions are not compatible with it at the moment, Firefox 4 will certainly shed a new light into the browsers war. It is lightening fast. I know, not as fast as "the other browser", but I don't care, since it is fast enough for me and is the only browser that fulfill my needs.

Anyway, besides venting a little bit, I would like to know what is your experience with Firefox 3.6.7 and what version/architecture of Ubuntu you are using. I'm posting a poll, but please describe your experience, since polls are very limited.

If you use "the other browser", please don't start a browser war here.

**Note:**

Anyone looking to downgrade, keep in mind Firefox 3.6.8 is now available through the official repositories and it is indeed better than 3.6.7. Although I don't have issues with both versions, 3.6.8 seems to be more responsive and faster. If you have serious issues with it, then instead of going through the trouble of downgrading, grab whatever version of Firefox that rock your boat from Mozilla, extract it and run it from your home directory, until a better version come through the official repositories. More info on how to install Firefox versions manually are available at [http://firefox-tutorials.blogspot.com/2010/05/installing-other-versions.html](http://firefox-tutorials.blogspot.com/2010/05/installing-other-versions.html)
