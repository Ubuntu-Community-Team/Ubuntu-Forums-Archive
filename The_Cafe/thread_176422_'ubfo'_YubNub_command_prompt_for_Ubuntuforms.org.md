### Post by RAV TUX on 2006-05-14
I set this up a while back. Just wanted to share it with everybody here.

If you use YubNub the command prompt for Ubuntuforum.org is 'ubfo' (ub-for ubuntu,  fo- for forums). Very easy to remember.

If you use firefox, you can install the extension: 'Search Engine Ordering', to add YubNub to your search engine que. 


1. first add the extension 'Search Engine Ordering' 
[https://addons.mozilla.org/firefox/2164/](https://addons.mozilla.org/firefox/2164/)

2. Then add YubNub to your search engine que by going to the YubNub website then using 'Search Engine Ordering'
(right click on the search box then select 'add this search to the searchbar', then you can customize your search title, I shortened mine to YubNub).
[http://www.yubnub.org/](http://www.yubnub.org/)

3. Your done, now type in 'ubfo' to get here.

(you can also add YubNub by going to 'add engines' then doing the advanced search for YubNub and add it this way, the only problem is you don't get the YubNub icon with it, just a random 'Y')


Here is an extensive guide for installing YubNub:

[http://www.yubnub.org/documentation/describe_installation](http://www.yubnub.org/documentation/describe_installation)

(including but not limited to; Konqueror & Opera integration, as well as Linux/Python integration.
