### Post by Isaac_x on 2007-08-09
I mapped my Menu key to Super.
<Super>+L - nautilus ~/documents
<Super>+<return> - close window
<Super>+O - run command dialog
<Super>+/ - gnome-terminal --geometry 155x30
<Super>+P - main menu
[IMG]http://img377.imageshack.us/img377/8161/20070803gt6.png[/IMG]
<Super>+<equals> - zoom in
<Super>+<minus> - zoom out
<Super>+<keypad direction> - move window to direction
<Super>+C - qalculate
<Super>+F - kayali
<Super>+M - maximize window
<Super>+<up> - scale windows
<Super>+<left> or <right> - switch desktop <left> or <right>
<Super>+<Ctrl>+<left> or <right> - switch desktop with window

And many more, but those are the main ones.
