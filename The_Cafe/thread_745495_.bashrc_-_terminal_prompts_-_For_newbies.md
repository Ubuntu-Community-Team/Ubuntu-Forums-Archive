### Post by Bruce M. on 2008-04-05
Well, after working for hours today, looking at webpages and playing I have my first "customized by me" prompt.

Nothing fancy, it's my first attempt. And because of that I like it, I'll keep it. OK, it may get rem-ed out in the future, but I'll keep it.

First thing I did was add this to my .bashrc file so I would know what colours I could use. I can't remember where I got it, I've been to a million places today, but **to the unknown author (*Edit:* [Ice60]("http://ubuntuforums.org/showpost.php?p=3800461&postcount=4")): Thank you!**

```
# Define a few Colours
BLACK='\e[0;30m'
BLUE='\e[0;34m'
GREEN='\e[0;32m'
CYAN='\e[0;36m'
RED='\e[0;31m'
PURPLE='\e[0;35m'
BROWN='\e[0;33m'
LIGHTGRAY='\e[0;37m'
DARKGRAY='\e[1;30m'
LIGHTBLUE='\e[1;34m'
LIGHTGREEN='\e[1;32m'
LIGHTCYAN='\e[1;36m'
LIGHTRED='\e[1;31m'
LIGHTPURPLE='\e[1;35m'
YELLOW='\e[1;33m'
WHITE='\e[1;37m'
NC='\e[0m'              # No Color
```

Next I took the commented "colour" prompt and copied it to:
```
# My Test for a prompt
```
and played with that until I got this:

```
# My Test for a prompt
PS1='${debian_chroot:+($debian_chroot)}\[\033[00;37m\]\d at: \t \n\[\033[01;36m\]\u @ \h\n\[\033[01;33m\]Issues the command\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$\[\033[01;37m\] '
```

I know some will say, "How basic." because you've been using Linux for years. But it's my #1. Wait until there is a #20 and lets see the difference.  :)

Have a nice day
