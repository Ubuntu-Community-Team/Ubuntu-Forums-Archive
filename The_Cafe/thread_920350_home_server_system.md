Just to let people know about [www.ubuntuservermanager.org](www.ubuntuservermanager.org) which is a web interface i have been developing for around a year on my own.

Looking for people interested in testing documentation and helping with development because this is a large project which is developing quite slowly on its own, you don't need a lot of experience just be willing to learn.

This will eventually be a system which will be suitable for home and small business its based on a plugin system for each service you need, and a profile system for creating default configurations for different use cases.

If your interested let me know or talk direct on freenode using irc on #ubuntusm.

The project is also hosted on launchpad at [https://launchpad.net/usm](https://launchpad.net/usm).
