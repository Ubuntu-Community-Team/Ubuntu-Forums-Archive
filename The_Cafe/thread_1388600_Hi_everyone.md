### Post by k64 on 2010-01-23
Personally, if hardware support is your preference, Karmic is the best way to go without getting an alpha release. Ubuntu 9.10 "Karmic Koala" supports more hardware than any of its predecessors. I actually (uesd to, before upgrading to Lucid Alpha) use it on BOTH of my computers. Yes, it worked perfectly, with very few crashes.

If you want more luck with hardware, use Karmic. If you want stability, use Hardy (8.04 LTS). If you want to see the next LTS release, use Lucid (10.04 LTS Alpha 2).
