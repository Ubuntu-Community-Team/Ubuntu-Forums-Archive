As long as I am not lucky in Installation & Upgrades forum I am going to post the third Rescatux release here (I am also renaming the title just in case).

Rescatux is a rescue live cd but it is not like the others... Rescatux offers you wizards (currently only the Recover grub wizard) so that you can achieve your rescue tasks.

Rescatux 0.03 released

[LIST]

[*] [Download ISO 0.03 (358 Megabytes)]("http://prdownload.berlios.de/rescatux/rescatux_cdrom_2010_07_09.iso")

[*] [Other downloads]("https://developer.berlios.de/project/filelist.php?group_id=11211")
[*] Features
      [LIST]
          [*] It only has one option: Install grub into the mbr.
      [/LIST]
    [*] Improvements
      [LIST]
          [*] Now grub is installed in a wide variety of installations (no hardcoded grub-install paths).
          [*] It should work with separated /boot partitions (non standalone but associated to another system).
          [*] Grub Installation check improved
       [/LIST]
[/LIST]

Please report your experience on fixing Ubuntu boot after reinstalling Windows and loosing Grub boot. Please report what it is your Ubuntu distribution (8.04,9.04,9.10,10.04,...).
