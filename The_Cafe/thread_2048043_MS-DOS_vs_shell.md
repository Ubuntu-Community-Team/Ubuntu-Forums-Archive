### Post by Paqman on 2012-08-26
DOS is long-gone, the last Windows version to use it was Windows ME. Windows does have a command line available, and it uses a mashup of DOS commands and some other stuff, but it isn't running the actual system the way the old Windows/DOS hybrid used to do.

MS have beefed up the command line ("power shell") in Windows lately, but it's still not as powerful as Bash et al.

Someone would have to be a pretty keen user of the command line on Windows for the Linux command line to be a major selling point though.
