> **Hobomank said:**

Keep Rockin':guitar:

P.S. whats the deal with Unity though guys? Gnome 3 rocks, why not use it lol

Glad to hear your story.

Technically, Unity is based on Gnome infrastructure; so Ubuntu is using Gnome 3. Just not Gnome Shell.

You can install Gnome Shell on Ubuntu easily enough; it's in the repositories. Once installed you can activate it on the login screen.

I think Gnome Shell is awesome too, and there are things in it I'd like to see in Unity. But in the end Unity is just a little more out-there and has some really good ideas that maximise usable screen space and bring the Linux desktop bang up-to-date. Gnome has a Not Invented Here attitude toward accepting code from Ubuntu which I think is why Ubuntu built its own desktop shell rather than building on top of Gnome Shell.
