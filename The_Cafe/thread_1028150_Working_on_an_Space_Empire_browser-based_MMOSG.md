### Post by CaptainNet on 2009-01-02
Hello all,

After eating a tonino's pizza (imitation pizza-esque death disc), I have started to regain the energy I need to stay up all night and work on this MMOSG.

Here is the idea.

It is a browser-based Space Empire MMOSG which will be released entirely under the GPL, and GNU-FDL.  Anyone with a web server will be able to set up their own game server for this game or a game mod thereof.

I have set up the wiki for the project at:  [http://www.nexxux.org/wiki](http://www.nexxux.org/wiki)

The web-application will be written entirely in PHP, HTML, CSS, and Javascript.  It will use a MySQL 5.0+ database.

Currently I am seeking volunteers to help out with the game. There will be a need for game art (including futuristic structures, spacecraft, etc) and people to double check the code driving the application.

All of this is possible through the wiki which I will be shortly beginning to edit to add content to.

Anyone is welcome to join in on this little project if they like.  All you have to do is register for the wiki and begin using the discussion pages/editing pages/etc.

Thank you,

~Joel
