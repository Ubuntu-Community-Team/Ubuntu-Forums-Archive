### Post by Tomosaur on 2006-09-22
I think you may be a little out of your depth here. Creating a simple web-site is simple with WYSIWYG editors like NVU. However, once you start talking about dynamic stuff - a forum, irc chat etc, you're getting into a lot more technical stuff. The page itself will be written in HTML, although there are drag and drop editors which will allow you to design your page visually and it will create the HTML code for you. Java-Applets are embedded into your web-page, they are not coded directly into it. Javascript is not Java, it is mainly used for adding dynamic elements such as drop down menus and fancy rollover effects to your pages. Javascript can be coded right into your web page.
