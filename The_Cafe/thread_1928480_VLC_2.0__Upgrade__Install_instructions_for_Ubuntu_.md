 Check out all the changes on the [VLC 2.0 Release Page]("http://www.videolan.org/vlc/releases/2.0.0.html").  NOTICE: Linux (Ubuntu) download is currently NOT available on that  page. If you are using Ubuntu 12.04 (Precise) all you need to do is do  an update. For everyone else on the current stable Ubuntu 11.10  (Onereric), there is a PPA.
  
Before you install, take heed from VLC:
"Yes, there will be bugs in 2.0.0! 2.0.1 is scheduled soon. Report bugs on our bugtracker!" (from [VLC Twitter]("https://twitter.com/#%21/videolan/status/170953930572963840"))
  
  [SIZE=2][COLOR=Red]**VLC 2.0+ PPA**[/COLOR][/SIZE]
Just add **ppa:n-muench/vlc**           to your system's Software Sources.

WARNING: this PPA includes more than VLC packages - it includes other video packages.
  
NOTICE: If you have mplayer, gnome-mplayer or any other mplayer  derivative installed, remove it prior to upgrading as there seems to be a  conflict.
  
TIP: Removing VLC along with deleting mplayer and gnome-mplayer prior to  adding the above PPA and installing VLC 2.0,  made for a trouble free  upgrade.
  
As always, you are using this PPA at your own risk.

  Thank you Nate for the PPA.
