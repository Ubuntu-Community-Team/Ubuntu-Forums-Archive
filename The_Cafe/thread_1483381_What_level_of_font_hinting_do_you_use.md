### Post by dca on 2010-05-14
Don't think of me as a looser, but I install Tahoma (and also remaining ms-corefonts) separately, set that as default, set rendering to 'Monochrome' and on laptops and small displays - I set the size to 8.6 (which equates to 8.59961 in settings) and set window title font to 7.6 (7.59961)...  I only do that because for some sick reason I make my Ubuntu installs look as close to Windows XP (still use top/bottom bars) as possible including icon packs, yadda, yadda.  It makes it more special that way when people ask, 'how you got your Windows desktop to look like that'???  I reply, 'Easy, it's not Windows'...
 
Oh, I leave 'hinting' at Full.

### Post by 2cute4u on 2010-05-14
I don't use font hinting at all, and never could understand why anyone would use it.  It makes font's look thin, light, ugly, and hard to read.  Font hinting is a leftover artifact of 20th century,  low resolution, bitmap technology.
