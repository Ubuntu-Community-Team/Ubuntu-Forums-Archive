### Post by Bombenbach on 2010-02-06
Try iCalParse (freeware)

[http://forum.xda-developers.com/showthread.php?t=544181](http://forum.xda-developers.com/showthread.php?t=544181)

it can import both local and remote .ics files! Tested on Touch Pro 2 with WinMo 6.5. I 
used it to sync my phone against a local (on a storage card) .ics calendar from Lightning and against a remote password protected .ics calendar from a Horde (kronolith) server of my college  and in both cases it worked like a charm!
