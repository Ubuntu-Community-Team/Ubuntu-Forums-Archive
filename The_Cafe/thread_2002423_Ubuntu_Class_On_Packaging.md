Hello there I will be teaching a Class On How to find fixs on the Ubuntu Forums. And how to make Package (Debian(.deb)) For these fixes. Think about it as a crash course in packaging. What we will be doing. 

We will be looking a a b43 driver(wireless broadcom) and the awful "missing firmware" Bug. There is a [Fix]("http://ubuntuforums.org/showpost.php?p=10932045&postcount=4") for this. We will be making a package that does all the work for the end users.

Things that are needed Teamvier you can get deb for that [Here]("http://www.teamviewer.com/en/download/index.aspx")
[How to install Teamviewer]("http://www.youtube.com/watch?v=qoY4e7TGTNI")

Other tools that are needed
ubuntu-dev-tools , dpkg-dev, dh-make, 

```
sudo apt-get install ubuntu-dev-tools dpkg-dev dh-make
```
