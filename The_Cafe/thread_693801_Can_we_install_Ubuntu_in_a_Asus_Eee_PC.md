### Post by moephan on 2008-02-12
I bought one when they first came out. I originally installed vanilla gutsy, but I recently installed eeeXubuntu using a usb thumb drive and instructions from here:
[http://wiki.eeeuser.com/ubuntu:eeexubuntu:home](http://wiki.eeeuser.com/ubuntu:eeexubuntu:home)

I also wen through the page of customizatoins here:
[http://wiki.eeeuser.com/ubuntu:eeexubuntu:customization](http://wiki.eeeuser.com/ubuntu:eeexubuntu:customization)

I didn't go through the motions of getting the camera to work yet, but there are instructions on the second link for getting it set up. There are also instructions for installing fusion if you want the desktop effects (I didn't). I did run the scripts to get the on screen display working for the function buttons.

I found the whole process of installing eeeXubuntu to be easy and well documented.

HTH

Cheers, Rick
