### Post by schopenhauer1962 on 2012-04-02
Nearly 14000 writers are active today in a new ***Script Frenzy*** event [(http://www.scriptfrenzy.org)]("http://www.scriptfrenzy.org") There are a lot of beginners and I suppose that some of them are Linux Ubuntu users.  I was asked which software to use when writing a script.  The easiest software for writing a script is ***Trelby*** ([http://www.trelby.org/download](http://www.trelby.org/download)) (available in Debian package). On the other hand I would recommend ***Basket Note Pads*** (Ubuntu software) to use as support, it is absolutely the best software to use for ideas, brainstorming, collect info from the internet, etc .... 
When you start reading the info from script Frenzy in pdf there is **Okular **also from the Ubuntu software collection. It allows note taking and has a lot of options, it's great software. 
On the other hand, I wrote my first novel on **Abiword**, using **Basket Note Pads**. 
So if you feel creative and want to join us on Script Frenzy ( April 1 ===> 30) or you want to write a novel on Nanowrimo ( November 1 ====>30) Ubuntu has all the possibilities !
