### Post by lm8 on 2023-07-31
This year is the 20th anniversary of Software Freedom Day on September 16th, 2023.  It's a worldwide event to increase awareness and educate others about the value of Free, Libre and Open Source Software.  It would also be a great platform to discuss Ubuntu and introduce new users to the operating system.  If you'd like to attend a local or virtual Software Freedom Day event or would like to volunteer to help out, please see https://www.softwarefreedomday.org/  There's also a #SoftwareFreedomDay channel on matrix if you have any questions.  Thanks.
