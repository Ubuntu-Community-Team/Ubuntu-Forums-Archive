### Post by patrickballeux on 2016-04-15
Hi guys,

I've started to work on ScreenStudio 2.x (Current version: 2.1.2) a few weeks ago after loosing the source code for version 1.6.  I had to start from scratch but it was a good thing as I took the time to re-think a few ideas.


The main features of ScreenStudio 2.x is to display a side panel instead of having overlays on your screen.  It does work quite well on older computer if you do not push the framerate too high.  The main goal of ScreenStudio is to use as less CPU time as possible and keep things simple.

Currently supported features:
- Local recording : FLV, TS, MP4, MOV
- RTMP Streaming: Twitch, Youtube, Hitbox and many others
- Optional side panel to display interactive content (HTML based)
- Optional webcam integration into the side panel
- Audio mixing from Mic + Internal
- Auto-hiding when your start recording/streaming
- and many others features...

It's not as powerful as WebcamStudio or OBS and will never be.  Layouts and overlays are limited to keep the CPU usage as low as posible but it does provide enough flexibility and customizations for streaming great looking content.

Binaries are available at [http://screenstudio.crombz.com]("http://screenstudio.crombz.com")
Sources are also available on GitHub: [https://github.com/patrickballeux/screenstudio]("https://github.com/patrickballeux/screenstudio")

Feedback is needed to help improve ScreenStudio and you help would be really appreciated.

Leave your comments below!

### Post by patrickballeux on 2016-05-08
ScreenStudio 2.2.0 is available!

Features:
- New compositing engine
- Added Privacy mode while streaming
- Manual panel size when no webcam is selected
- Better layout handling (Top, Left, Right, Bottom)
- Mouse follow up (Virtual webcam)
- Other stuff I may not remember ;)

This is a major milestone for ScreenStudio.  While trying to keep the CPU usage to a minumum, this release is opening a lot of possibilities like WebcamStudio a few years ago,

Hope you like this new version.  Please tag your videos with #screenstudio if you are using it when streaming or uploading to Youtube.

Thanks for all your support.
