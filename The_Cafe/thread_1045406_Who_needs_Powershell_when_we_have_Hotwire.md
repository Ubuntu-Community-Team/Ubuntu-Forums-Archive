In a search for a terminal that would magically replace Nautilus as my desktop background (help appreciated. I don't want to use devilspie) I discovered something far cooler: Hotwire!

[http://hotwire-shell.org/](http://hotwire-shell.org/)
It's in the repositories, too.

Like Powershell, it is aware of data types being returned by commands, rendering it really really useful at presenting information. For example, the ls command gives you a proper directory listing with icons :)

It also has a really pretty GUI.

All very technical, but I think I'll enjoy tinkering with this one.
