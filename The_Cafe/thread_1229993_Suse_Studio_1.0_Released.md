### Post by jrothwell97 on 2009-08-18
Sorry to bump, but I received my invitation today.

It seems good enough, if rather sluggish (I can forgive them for that: the server load is very high and it's a closed beta at present.)

I've only run the images using Testdrive, and not on an actual machine yet, but there's actually a startling level of control over the system. That said, I would still like to be able to lock the root user, and I'd love a finer level of control over the individual configuration files.

However, in comparison to other online Linux system generators, it's superb. I'm building the second iteration of my appliance now, and I think I may give this one a run on a real machine.
