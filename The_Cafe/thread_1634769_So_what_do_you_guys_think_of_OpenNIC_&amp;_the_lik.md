### Post by SoftwareExplorer on 2011-05-23
I'm using it! I especially like it for ssh on my local network--I can get something short so I don't have to type as much. I also like the .geek domain names :)

I run an ubuntu desktop as the router/WAP for my home LAN and have the dhcp give out an Opennic-enabled DNS server. 

What strikes me as Ironic is the ISPs that run DNS servers that try to be helpful by resolving unknown queries to a search page but can't be bothered to be helpful by being able to try resolving Opennic domains. (Of course, search pages make money. If you send clients to a search page instead of an Opennic Domain, you make more money. )
