### Post by Jiraiya_sama on 2007-11-29
The Kde4daily qemu image lets you run a version of kubuntu running an svn version of KDE4, with daily updates, to follow along on KDE4's progress and help with bug reporting.

This is a conversion of the original qemu image to run on virtualbox.
