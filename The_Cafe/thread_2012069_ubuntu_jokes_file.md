I don't know what the plugin was, but it probably got its content from the ***fortune*** program. Install ***fortune-mod*** from the software centre and run it from the command line:```
fortune
```From memory, it has a load of command line options to help choose what sort of thing you want from it as well, so it's worth checking:```
man fortune
```
