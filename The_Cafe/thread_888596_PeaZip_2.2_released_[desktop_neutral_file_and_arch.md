### Post by Giorgio tani on 2008-08-13
PeaZip 2.2 was released yesterday; it is available as installable DEB package and also as standalone application not needing installation.

In both cases in peazipdirectory/res/altconf.txt you can set absolute or relative path of application's configuration files if you keep the application in a path that is not writeable for the current user, or i.e. if you want to store configuration for multiple users in a shared path for administrative purposes. 

The new version supports UTF-8, localizations (so far only English and italian, if you would like to contribute see the documentation), bookmarks archives and paths etc...

You can see full changelog on [http://peazip.sourceforge.net/](http://peazip.sourceforge.net/)
