### Post by YeOK on 2010-07-02
> **mister_playboy said:**
> Since I'm using KDE I'm less than excited about having to install all these Gnome deps for Opera to upgrade.  I'm gonna hold off for the time being.

Opera is not GTK based! 

It uses its own lib called Quick, it will use gtk / kde widgets if it finds them, you can also tell it what to use in about:config. If it can't find gtk or kde, it will simply draw itself directly to X.

The Linux version was the last to use QT, its the reason they took so long to catch up to the Windows / mac version and IMHO, it was well worth the wait.

So, it will require less dependencies now than it ever has before.

[Opera Dev blog post]("http://my.opera.com/ruario/blog/unix-10-50-evenes-work-in-progress").
