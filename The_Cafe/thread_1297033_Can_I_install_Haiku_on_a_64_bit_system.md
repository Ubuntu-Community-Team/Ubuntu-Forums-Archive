### Post by jespdj on 2009-10-21
Yes you can because x86-64 systems are fully compatible with x86 32-bit systems.

Just like you can install 32-bit or 64-bit Ubuntu or Windows on your PC, you can install 32-bit Haiku OS on your PC. It will just not use the 64-bit extensions.
