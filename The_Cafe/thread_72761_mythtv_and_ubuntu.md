### Post by Joshua on 2005-10-07
I like that idea.  But I also don't think there is any reason to reinvent the wheel.  KnoppMyth rocks at what it does, and Ubuntu rocks at what it does.  I want my computer to be able to do everything for me and I like the way Ubuntu handles those things so far.  I like working in an Ubuntu desktop, and I like the decisions that Ubuntu makes so that I don't have to.  I also think Ubuntu is no less suited for a MythTV box than KnoppMyth.  I think that if there were an Ubuntu-based MythTV project, it's focus should be to make sure that anyone who wants, can type ```
sudo aptitude install mythtv
``` and have a DVR up and running without any trouble.  In other words, it should make sure that recent versions of MythTV are included in each release and that installation is as trivial as possible.  Not to have another live cd distribution that recognizes everything and automatically sets up a minimal desktop.
