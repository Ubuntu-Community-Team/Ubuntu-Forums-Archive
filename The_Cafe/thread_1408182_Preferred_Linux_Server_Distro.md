### Post by scottuss on 2010-02-16
It depends what you want it for:

**For production:** Redhat (or Centos), Debian Stable or Ubuntu Server. All rock solid, reliable and great support. Plus loads of Enterprise stuff is built on top of these for a reason!

**For home use:** Arch. Sounds crazy? Sure, it's bleeding edge but fast as hell, has no GUI overhead on a default install and has the latest version of all the server packages you could want for a nice home set up.

Edit: I read yestereday that Lucid will feature the latest version of likewise-open which makes it easy to authenticate Linux (and other OS's) with Active Directory.
