### Post by Computer Guru on 2007-11-13
EasyBCD 1.7.1 has just been released. It is a minor (bugfix + optimizations) build that addresses a couple of issues that have cropped up in EasyBCD 1.7 since its release 2 months ago. 

EasyBCD is a Windows Vista bootloader management tool that lets you boot easily and quickly boot into Windows Vista/Longhorn Server, Legacy Versions of Windows, Linux, Mac OS X, and BSD.

EasyBCD provides a one-click graphical interface to repairing the Windows Vista bootloader, managing, creating, and renaming existing entries, and gives you complete control over the Windows Vista boot process.

Some of the changes:
[LIST]
[*]Compatible with the new Wubi Gutsy Gibbon releases. 
[*]EasyBCD now warns users if NTLDR or Bootsect.dos was not found in the process of adding a XP or 9x entry. 
[*]The WinPE section has been completely overhauled&#8230; under the hood. No more delays, just raw, sheer performance to match the speed of WinPE 2.0 itself. 
[*]Rewritten (no longer caches responses) update checker.
[/LIST]

What EasyBCD does:
EasyBCD is a Windows tool that makes it easy for Windows users to dual-boot with Ubuntu in a way that doesn't involve mucking around with the GRUB shell, setting up the MBR, or putting your data at risk. It lets Windows and Linux blissfully live together without any bootloader/dual-boot related problems.
