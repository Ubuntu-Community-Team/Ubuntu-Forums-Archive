### Post by ve4cib on 2010-08-04
I don't believe it exists.  And I'm not surprised that it doesn't.

Creating a mobile phone OS is a pretty long way from creating a desktop/notebook/netbook OS.  It's a tablet OS that has to be heavily-optimized for a very small screen, ideally handles rotation via an accelerometer, and handles all the cellular data/voice/SMS traffic functionality.  Add to that the fact that every handset requires a different set of modules for the hardware, the relatively low memory and CPU power, etc... and it all amounts to a very large amount of work.

Ubuntu's main focus has been on more traditional computers and limited support for mobile devices.  In another few years we may see an official "Ubuntu Phone Edition," but such a thing is likely a long way away.

That having been said, there are other distros out there that are intended for use on phones.  MeeGo (currently under development by Intel and Nokia as the successor to both Moblin and Maemo) is a good example.  Mer (a community-driven mashup between Maemo and Ubuntu as far as I know) is another example.
