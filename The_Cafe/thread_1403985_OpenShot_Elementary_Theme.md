### Post by Excedio on 2010-02-10
Not sure how many OpenShot users are here, but to those of you that are, I've created an Elementary theme for OpenShot. I'd love for people to give it a try and provide some feedback. :-)

How to install the theme:

[LIST=1]
[*]Download the tar file
[*]Extract to desktop
[*]sudo mv /home/[username]/Desktop/Elementary /usr/share/openshot/themes/
[*]Open OpenShot and go to Edit > Preferences > Theme
[*]Select Elementary
[/LIST]
Thanks for those who try it out. :-)
