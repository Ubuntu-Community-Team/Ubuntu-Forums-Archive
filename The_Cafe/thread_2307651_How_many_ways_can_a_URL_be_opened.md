Hello,

I was wondering about the different means through which a URL could be opened and came up with these six:

1) Manually entered: type written into a device
2) Hyperlink: found on web site, email and similar
3) Spoken: via supporting devices
4) QR: Scan qr
5) NFC: Touch NFC tag
6) URL Forwarding: bounce from one website to another

Would the community like to tell me if I forgot any others?

Thank you very much :) !
