### Post by NightwishFan on 2009-04-26
A little tip:

Under system - preferences - software sources there is a location where you can autodetect the fastest server. The update should be fairly fast on a decent connection. No more than 30 mins to download and 20 to update.

Choose "download from" and pick other.

### Post by SonicSteve on 2009-04-27
I have my /home folder on a separate partition, this means that I download the ISO file, burn it, and install it.

I use the manual option for setting up partitions during the install. I told it to use the system partition  with EXT4, reuse the same swap area, and mount the home folder on the same old partition where it was with 8.10. After the install I only had to install codecs, a few apps, and I was done. So not counting the 20min to download the ISO the whole process was a bit more than an hour, in the end I have a fresh install and it's clean.
