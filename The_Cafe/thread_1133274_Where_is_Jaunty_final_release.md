Sorry if I missed something earlier, but you shouldn't need to do any of this. The Ubuntu installer disc handles it all for you.

Essentially, you just pop in the disc and start the installer, and let it handle the partitioning for you. By default, it will shrink your existing Windows partition, create the two new Linux partitions (swap and main), set up the boot loader to handle booting into either Windows or Ubuntu, install Ubuntu, and optionally copy some settings (like bookmarks and such) from Windows.

It lets you decide how much space to let Windows keep and how much to give to Windows; it will show your partitions in a graphical gauge, and you can drag to pick the size to give to Linux. Couldn't be easier.
