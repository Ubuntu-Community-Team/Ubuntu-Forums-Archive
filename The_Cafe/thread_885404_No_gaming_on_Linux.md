### Post by StumpyMcDonut on 2008-08-30
It'd be wonderful if more games came out for Linux, but it won't happen soon, even with the support OSX is picking up. It's to do with the lack of money, but also the lack of users that would actually bother with it - plus the fact that OpenGL doesn't stand up against D3D quite yet, and that's simply because D3D was specifically oriented at gaming. GL has only 1/2.5 of its calls that are useful for gaming (I think, don't quote me), and I have no idea about D3D, but I'm assuming its a higher percentage.

It seems that our choices are limited - we can either assist with the development of OpenGL (Those that can) and attempt to orient it more towards gaming, attempt to get MS to make D3D multiplatform (IMO I wouldn't mind paying the same price as a new vista license if it enabled the D3D api on Linux if you wanted it), although that's never going to happen (Nor is even possible). Then again, we can simply buy the games that exist for Linux in the hope that it will encourage devs to release ports. Perhaps we could push for "Cider for Linux", I'd be happy with that insofar as it allows games to be played, although the only long-term solution is to get native games up, and that needs the support of major games companies.

Another thing to note is that WINE has NEVER come close to native performance in any games I've tried (And believe me, I've tried a few). 20-65 FPS On WoW (This is a ~2004 game) with an 8800GTS 512 (OpenGL vers)? Believe me, it just isn't worth it, regardless of the moral high ground I'd have. Despite the abysmal OpenGL api on Win, and in WoW, its still almost double that, and maybe four times faster using native D3D. Same with CnC3, I get flawless performance with everything and max AA native, but in wine I have to turn AA off and other settings down a bit just to make it playable

The best thing we could push for would be a native version of Photoshop (That would make a hell of a lot of people switch over that are currently dual-booting, trust me), and other such commercial software, because our chances of getting good gaming are slim unless we have the userbase to support it. OSX has a big enough userbase, and they've only recently been complaining about games, and it can't be that important, as many Cider-powered games are a joke.

Fyi, I use XP for gaming and PS, and Ubuntu whenever I can for anything else - I even try and reboot if I just want to use the internet (Sounds stupid, but I'm in love with Ubuntu and I try and use it as often as possible, I even try and put up with the awful performance in WoW if I'm simply browsing or chatting while playing it)
