### Post by LowSky on 2010-12-25
You could set up a media server on the computer. mythtv or even something simple like mediatomb run using UPnP.

Many devices like Sony Playstation 3 and Xbox360 can see these on wifi and not so well known items like Western Digital WD TV Live Plus HD Media Player, ViewSonic VMP71 NexTV, and Popcorn Hour A-210.

There are many other devices that can do this too like Western Digital WD TV Live Plus HD Media Player, ViewSonic VMP71 NexTV, and Popcorn Hour A-210.

### Post by sgosnell on 2010-12-26
There are a number of bluetooth options if the network isn't available.  Several companies make cheap bluetooth transmitters and receivers that will connect through standard 1/8" stereo plugs.  They are handy for streaming music to stereo equipment that isn't network enabled, and are actually rather cheap, comparatively speaking.
