### Post by monsterstack on 2009-05-07
There is an educational application aimed at high school kids for learning the *very* basics of programming called *Guido van Robot Next Generation*, which you can get by running

```
sudo apt-get install gvrng
```

Its syntax is like a simplified Python, whose syntax is pretty easy to begin with. It allows you do move a little robot around the screen, and create mazes for him to navigate and such. I found it incredibly boring and useless as I only found out about it after I'd already learned most of the basics of Python anyhow. It would be a good place to start for the completely uninitiated.

why_the_lucky_stiff has done some good things with making learning Ruby fun, as others have already noted. If you want to learn Ruby from scratch you might want to try this [great place]("http://tryruby.hobix.com/") [tryruby.hobix.com], which allows you to go right ahead and punch lines of Ruby right there in your browser, whilst a tutor guides you through the basics and the significance of the thing you're doing there.
