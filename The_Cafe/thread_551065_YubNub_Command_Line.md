### Post by RAV TUX on 2007-09-14
Awesome extension for Firefox/IceWeasel:

         **[YubNub Command Line]("https://addons.mozilla.org/en-US/firefox/addon/4714")**

          by [BlueJ774]("https://addons.mozilla.org/en-US/firefox/user/116588")     


Even though you can add YubNub to your Search Engine bar using          [Add to Search Bar]("https://addons.mozilla.org/en-US/firefox/addon/3682") by [Dr. Evil.]("https://addons.mozilla.org/en-US/firefox/user/4924")

If you are like me and use YubNub command prompts as often as you use sudo and apt-get in the terminal. This places a very small bar at the bottom of the browser window that gives you a very quick and easy to use YubNub Command line.

[[IMG]http://cafelinux.org/OptickleArt/albums/userpics/normal_YubNubCommandLine.png[/IMG]]("http://cafelinux.org/OptickleArt/displayimage.php?album=lastup&cat=0&pos=0")

remember the YubNub Command Line for UbuntuForums.org is:

```
 ubfo
```
