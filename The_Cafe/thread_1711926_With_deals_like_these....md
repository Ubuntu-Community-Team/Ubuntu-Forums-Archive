### Post by Dark_Stang on 2011-03-21
I never buy laptop computers online for a few reasons...
1. I don't get to see it before it's at my door
2. Spending 2 hours on the phone with tech support with a person who's going over a flow chart before getting a UPS tag issued is annoying (warranty service)
3. Dead pixel policies
4. How do I know that Linux is going to run on it? If it's on display in a store I can just pop in a live CD and see how it goes.

Yeah, I end up spending a little more (probably $100-$150 more for one in a store). But knowing that I have exactly what I want is worth it to me.

Now if we're talking about a desktop, then I like to just build. And if we're talking about a server, I would just go with a virtual host and run whatever I want on it.
