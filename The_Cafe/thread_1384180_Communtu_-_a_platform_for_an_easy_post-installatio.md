### Post by KnutHB on 2010-01-18
Communtu is a web2.0 internet platform, designed to assist users to easily install a bunch of software after they have installed Ubuntu. The program packages are stored in bundles, which can be installed as metapackages. 

If you want to install software which is only available in a repository other than the standard repositories of Ubuntu, Communtu automatically adds the repository to your sources.list and also adds the corresponding GPG-keys.

Moreover, you can not only install already existing bundles on your system - you can create your own bundles and, if you want to, make them available for other users. 

Communtu is designed both for users who just installed Ubuntu for the first time and for system administrators who want to install one selection of software on multiple computers.

Currently we are testing a new feature: create your individual Ubuntu-CDs! With this feature it is possible to add your own software selection to the CD and test it as a usual Live-CD and install it as a normal Ubuntu (with the software you have chosen previously). 

Communtu is free of charge and can be used without registering. 

We also have an IRC-channel on Freenode: #en-communtu .
