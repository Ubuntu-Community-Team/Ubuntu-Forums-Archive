### Post by wildman4god on 2009-04-10
I have found a couple of websites that pretty much do the smae thing as add/remove in ubuntu, however these sites offer a lot more features and information on the apps, they also seem to be more user friendly.

[http://cnr.com](http://cnr.com)

This site has paid software as well as open source software all in  one covenant place, looks very professional. Setup by xandros you just download the client and off you go, it doesn't rely on repositories so it has more software, looks a lot like iphone app store which people are more familiar with. works with all popular distros, it's also supposed to have music and video's for sale, and has the ability to install web apps to your desktop. Also has commercial games for linux.

[http://appnr.com](http://appnr.com)

web based version of Ubuntu's add/remove, uses Ubuntu's repos pretty much only works with Ubuntu, however it seems to be more user friendly and its easier to find stuff like codecs.

I can't yet use linux where I am, i am working on that though, so I would ask if ya'll would be so kind as to test them out for me, the xandros look promising though especially if they have a good selection of media to buy.

Also speaking of xandros, they will be releasing presto "instant on os" april 13 and I tried the beta and it looks solid for what it was intended, check it out [here]("http://www.prestomypc.com")

it also has access to the cnr store built into it and it booted up in seconds (excluding my desktops post) it also includes real player with most if not all codecs, also comes with a free 14 day trial of real player's media store services, also real player finally got their player looking half-way decent and with more functions (last i checked it was just helix player with real player logo and codecs) also has flash built in, every thing works out of the box, their desktop os also has stuff like that built in, you have to pay $99.99 for it though, presto will retail for $19.99 when released.
