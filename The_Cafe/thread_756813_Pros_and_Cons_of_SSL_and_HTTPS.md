### Post by HermanAB on 2008-04-16
SSL is secure.  There is a slight risk of a man-in-the-middle attack; SSH handles that with the 'known-hosts' file.  Other than that, the only downsides are speed and the need for many IP addresses, which can be an issue if you are doing web hosting.
