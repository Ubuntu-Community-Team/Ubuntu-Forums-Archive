[http://www.ribosi.com/index.php?main_page=softwiki](http://www.ribosi.com/index.php?main_page=softwiki)

The main feature of this wiki is the ability for users to search a Windows application and see alternatives for Linux. So, someone searching a Windows alternative of "MS Word" might get a response of Abiword, OpenOffice.org and Kwrite.

It's brand new, so there's very little software in it now. By my own efforts, this will continually grow. With some community involvement, this will change drastically. Software doesn't have to be open source to be included, but it does have to run on Linux - this is the only stipulation for inclusion.
