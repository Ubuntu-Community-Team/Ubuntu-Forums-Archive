### Post by Bruce_MAYO on 2017-01-04
The password manager fpm2 was apparently deleted from the Ubuntu repositories in 2015. I've used this program for several years, and I like the large free-form area that lets me save a lot of information about a site. The coding doesn't make much use of exception handlers, so if the program doesn't find a file or the format it wants, it just lets a segmentation fault happen without any feedback. Apart from this, I haven't seen any bugs. You can still install it by downloading the .deb-file from launchpad.net/ubuntu/wily/i386/fpm2/0.79-3 and installing with gdebi. I'm using it in Xubuntu 16.04 on two very different machines without any problems.
