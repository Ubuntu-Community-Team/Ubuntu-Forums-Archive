### Post by M_the_C on 2007-01-10
> **Handssolow said:**
> Am I right that if you run Ubuntu live just from CD nothing is saved when the session is ended? 
That is correct.  Once you shutdown a LiveCD everything is lost.
> **Handssolow said:**
> 
Is there any such Ubuntu animal?
The Puppy Linux CD only takes up about 100MB, Ubuntu takes up the whole CD, so there would be no space to save the sessions, although you could save session information to the hard disk.

There is a way of doing it with a [USB Pen drive]("https://wiki.ubuntu.com/LiveUsbPendrivePersistent").
