### Post by 1clue on 2012-01-04
Dude, you need to check out the specs on your cell phone.  It's probably faster than that box.  Seriously.  My phone has a dual core 1.5 ghz processor with 1g RAM and 48g storage.

If you're just looking for a challenge then that's one thing, but it looks like you're trying to actually use this thing for something.

If you're not willing to max out your RAM I would expect to get a long string of disappointments.  RAM is about as cheap as I've ever seen it.  My office recently bought 2x4G ddr3 memory modules for everyone's laptops for $80 per set.

The problem is that Linux evolves along with hardware.  If you were to take a Linux from a few years ago that system might have worked reasonably well.  If you're going to use a modern distro it better be a micro one.  If you want a window manager on it maybe look into blackbox or fvwm.

I seriously doubt you're gonna run Firefox.  I have two windows open, one is using 527.1m and the other is using 258.9m.

A brand new desktop box comes for a couple hundred bucks.  There comes a time when you just gotta retire the old junk.
