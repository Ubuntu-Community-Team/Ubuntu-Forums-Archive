Comparing mac to ubuntu is nonsense, mac produces several products, the OS is only one of them, not to mention that the word "mac" occurs in several other contexts, "big mac"... (same thing for "ubuntu" but much less so). Comparing "windows" to ubuntu is even worse, windows and linux would be a better comparison if windows was not also a generic word for a common object...

Probably a better comparison would be Ubuntu to OSX or "Windows Vista" ("vista" by itself is also a generic word) 

[http://www.google.com/trends?q=ubuntu%2C+windows+vista%2C+osx&ctab=0&geo=all&date=all](http://www.google.com/trends?q=ubuntu%2C+windows+vista%2C+osx&ctab=0&geo=all&date=all)
