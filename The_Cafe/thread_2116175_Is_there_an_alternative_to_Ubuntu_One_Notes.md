### Post by lads on 2013-02-15
Dear all,

The Ubuntu One Notes service is going to be shut down a few days from now. I happen to use this service quite heavily through Tomboy and my work will be negatively impacted. The Tomboy folks are working on an alternative but this is something that won't come overnight; I don't expect it to be up before the service gets shut down.

So my question is if there is an alternative to the Ubuntu One Notes service, that provides a similar experience (independent of the client software).

Note: I'm not asking for a file synchronisation with Ubuntu One. I'm pretty aware of that work around and its problems.

### Post by ssam on 2013-02-15
i use tomboy syncing to ssh. you just need a server somewhere that you have ssh access to. you may need to install the fuse sshfs package on the clients.

### Post by tgalati4 on 2013-02-15
I find zim works pretty well over a network with multiple users working on the same notebook file.

Host the zim file on a server and put a zim launcher on everyone's desktop that points to the server notebook file.  You will need to set up the server mounts on each machine.
