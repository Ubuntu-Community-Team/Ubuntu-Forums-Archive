### Post by praveesh on 2010-03-15
There are some funny features hidden in some open source softwares . To reveal some of them ,type  
1. In firefox type about:robots in the address bar
2. Type alt+f2 and type gegls from outer space :you can play a new game not in your computer
3. Type alt+f2 and type free the fish :you can see a fish floating in your Gnome desktop
4. In the terminal type apt-get moo  : you can see a cow inside the terminal 
5. In terminal, type aptitude moo
then type aptitude moo -v 
then aptitude moo -vv   and so on increasing the number of 'v' s

### Post by doorknob60 on 2010-03-15
I think there's something where if you type something into OO Calc, you can play space invaders or something. Don't remember it though.

Here:
>   The GAME() function

Putting =GAME() into any cell results the text "say what?" in that cell.
Frogger

Putting =GAME("Froggie") into any cell results the text "Froggie" in that cell. Putting =GAME("Froggie") into any other cell afterward results in the text "oh no, not again!"
Tic-Tac-Toe

Put the =GAME(A2:C4;"TicTacToe") formula into A1 and press ENTER. A version of the Tic-Tac-Toe game is playable.
Space Invaders

Put =GAME("StarWars") into any cell and press ENTER. A version of Space Invaders is playable. Although if you try to play it again, the message "oh no, not again" appears. To play again, you must fully close OpenOffice.org.

    This easter egg has been removed since DEV300_m72.
