### Post by CrusaderAD on 2009-12-05
I figure because Empathy has taken over the IM software with Ubuntu (probably because Pidgin is lacking developers) I wanted to make Empathy look prettier. So here's some instructions on how to get more than just the 4 or 5 standard themes. In fact, 65 themes!

Download the deb file here: [http://tecnicoslinux.com.ar/livecd/empathy-chat-themes_1_all.deb]("http://tecnicoslinux.com.ar/livecd/empathy-chat-themes_1_all.deb")

   1. Find the installed files in &#8220;/usr/share/empathy- chat-themes&#8221; . You can do it manually or in nautilus &#8211; click on pencil below the back/forward icons and copy the path without quotes.
   2. Next open your local folder on new nautilus window: Click Places/Home Folder.
   3. From the View tab select &#8220;Show hidden files&#8221;
   4. Go to &#8220;.local/share&#8221; folder and make new folder &#8220;adium&#8221; then make another folder inside adium called &#8220;message-styles&#8221;.
   5. Final step, copy all files from &#8220;/usr/share/empathy-chat-themes&#8221; (Ctrl+A to select all and Ctrl+C to copy) and copy to &#8220;.local/share/adium/message-styles/&#8221; (Ctrl+V to paste)
   6. That's it, you can found empathy themes in edit/preferences/themes in empathy window

Happy messaging!
