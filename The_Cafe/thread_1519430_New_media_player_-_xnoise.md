### Post by steveneddy on 2010-06-28
Ran onto this new to me media player.

Seems cool and very fast - and there is an Ubuntu PPA available.

Here is the link:

[http://www.xnoise-media-player.com/](http://www.xnoise-media-player.com/)

To install the PPA do this:

```
sudo add-apt-repository ppa:shkn/xnoise

sudo apt-get update

sudo apt-get install xnoise xnoise-plugins-core
```
