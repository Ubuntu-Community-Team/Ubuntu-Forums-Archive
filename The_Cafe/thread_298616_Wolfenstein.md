### Post by barney_1 on 2007-02-14
Just stumbled onto this thread.  What a blast from the past!

Sorry about your installation woes.  About 7 or 8 years ago I installed this from the original disc (discs?), the burned my installed directory to a DVD for backup.  I was able to copy it onto my drive and play it in DOS box.  Excellent!

Howto: Wolfenstein 3d in Dosbox:
(this assumes you have a copy of an already installed directory, but you should be able to install it the same way)

1. Copy all files to a directory:
```
mkdir ~/wolf3d
cd ~/wolf3d
cp /media/cdrom0/wolf3d . #change the directory in this line to match where your wolf3d files are right now

```

2. Install dosbox:
```
sudo apt-get install dosbox
```

3. Run the program:
```
dosbox ~/wolf3d/WOLF3D.EXE -fullscreen #this runs in fullscreen and assumes your executable file is called WOLF3D.EXE
```
