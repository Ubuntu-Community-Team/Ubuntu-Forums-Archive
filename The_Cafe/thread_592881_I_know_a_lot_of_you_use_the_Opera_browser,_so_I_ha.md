### Post by sunexplodes on 2007-10-26
Is there any way to change the way it decides to display tabs after closing one?

For instance, when I close a tab, opera shows the last active tab. I'd like it to show the NEXT tab, as I often go to a site such as reddit, open 3 or 4 links in a tab, then view them sequentially.

---

### Post by daynah on 2007-10-26
Tools > Preferences > Advanced > Tabs > Cycle in tab bar 

Is that what you're looking for? It may do it in "order" but the backwards order to what you'd like, I'm not sure, I've never cared what order  the next one comes up in. Try it! :)

### Post by racoq on 2007-10-26
this is the best way:
 tools > preferences > shortcuts > Edit keyboard setup> create a shortcut under category applications for instance:

Input context and shortcuts - d ctrl      
Actions - Close page & Switch to next page


after that just do CTRL + D to close the tabs and it will work ;)
