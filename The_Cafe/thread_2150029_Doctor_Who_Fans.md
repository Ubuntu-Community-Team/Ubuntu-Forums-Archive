Edited script YET AGAIN!!

This time I added a scream and uped the count for which the angel is standing still. I've also added better path detection which should fix the problem of it not finding it's directory. The images and sound file aren't mine. I found the sound file on a public domain site. Be sure mplayer is installed before using.

If you'd like to install extract the zip into your home folder, open your startup programs and add this command:

```
python /home/username/WA/script.py
```

If you would like to disable the scream, remove or comment "#" this line out of the script.py file:

```
os.system("mplayer " + str(cwd) + "/3.mp3")
```
