As elements with a higher atomic number (or mass number) have more nucleons, it is logical for the respective atoms to have a bigger size. If an atom has a large atomic number, it will also have more neutrons and more electrons than the atoms with less protons. This means there is a bigger nucleus and more shells, hence the total atomic size is larger, even if it has a smaller mass.
