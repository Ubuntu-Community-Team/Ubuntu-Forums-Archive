### Post by disturbed1 on 2006-06-01
If you haven't removed Ubuntu yet, run synaptic and enable the universe and multiverse repos. This will give access to all of the GStreamer codecs for Totem (good bad and ugly). But I use VLC as defualt, some use Mplayer, it's a personal choice. Much like your IM client. There are some others available too.

Totem plugin is also in synaptic, but it still isn't that good. The Mplayer plugin works better.

About your USB mouse, check in your BIOS, and enable usb for DOS devices and asign an IRQ to USB. Some mother boards have a funky boot with USB devices, this will usually work.

Linux (in general) isn't for everyone just yet, but it's getting closer. I must say it's much easier today than it was just a year or 2 ago. I can fully understand where you're coming from. Not everyone wants to have to try this, try that, right this script, change this config, just to get a working set up. But these things will only improve from user input such as yours;)
