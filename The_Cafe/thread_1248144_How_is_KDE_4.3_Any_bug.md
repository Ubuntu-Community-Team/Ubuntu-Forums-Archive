### Post by Firestem4 on 2009-08-23
I've been using KDE 4.3 on all of my Linux computers and I absolutely love it. its rock-solid compared to the previous releases. Like everything in this world it's not perfect, and there are some bugs. But it is much more stable than it was. (Mind you I am running on Jaunty backports which is not supported.)

Overall performance is better and a lot of programs tend to work seamlessly with the rest of the system. As the KDE Developers themselves pointed out there isn't one huge improvement in any area rather its the combined improvement of the overall desktop that has made a large impact. Try it out, it has some great new features too that I really love. 

An example is in a Folder View (widget or desktop style; ) if you hover your mouse over an icon it does a quick preview that you can navigate instead of having to bring dolphin up. It works seamlessly and is very useful.
