### Post by techrush on 2010-11-05
Curious how many ubuntu users out there use a webOS device ? Its powered by a linux kernel and I find it to be a really nice user experience especially  how Palm and now HP are supporting the homebrew community. They essentially hand you the device with root access enabled. No nasty jailbreaking or rooting like on an iphone or android device. 

I was able to install a custom kernel with some new memory management options and the ability to overclock my cpu! There are also tons of tweaks you can make to the UI via "patches".

To top it off if you are a web developer like me all the apps are written in a custom javascript framework (based on node.js now i believe) and html.

What do you guys and gals think ?
