Hi praveesh. You idea is good, but  I guess the real problem lies in the ability of the authors to incorporate the resizing feature into their plasmoids. There are also plasmoids that don't make sense when sized down to the panel height, so the usually behave differently when added to the panel. It's the case of the default Folder View, which is fully displayed when added to the desktop, but iconized when added to the panel.

There are some plasmoids that I would like to add to the panel, but they are too big and don't resize. For example the Plasmacon. Perhaps it could be a one-line terminal when added to the panel.

Anyway, what I did with the Plasmacon is to put it in the dashboard and configure plasma to use a separate dashboard. Then I added the "Show Widget Dashboard" widget to the panel and assigned a keyboard shortkey. So whenever I need a terminal, I just hit CTRL with my smaller left finger (don't know the name of it in English :) ) and SPACE with my thumb. ;)
