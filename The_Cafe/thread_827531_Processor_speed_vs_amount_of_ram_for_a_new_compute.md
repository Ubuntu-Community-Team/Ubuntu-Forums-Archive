processor speed actually matters less than you'd think. Also, even though ubuntu says it's only using 600-700MB, the remaining RAM is still being used - to cache recently used pieces of data. This means that even though the first load won't be any faster, more RAM lets you hold more data in this cache, which means that future loads of this data will be MUCH faster. For this reason, upgrading RAM can often lead to a 'snappier' system than upgrading the processor will. However, past 2GB of RAM you're not going to see much additional benefit, so at that point I'd advise going to processor instead.
Also, LiveCDs see more benefit from large amounts of RAM than they do from a fast processor, so if speed there is a factor the RAM may be more important. Again, past 2GB you won't see much difference.
