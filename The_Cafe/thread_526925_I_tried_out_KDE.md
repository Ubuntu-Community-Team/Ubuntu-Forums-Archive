### Post by Chaotic Thought on 2007-08-16
I don't run KDE, so I can't say from personal experience, but there are countless themes and theme engines that allow you to change how it looks. A good site to check out is kde-look.org.

As far as the speed, I think you're right--Gnome and GTK2 do seem less responsive than Qt (and Windows). I still use Gnome and primarily GTK2 applications, but it's just one of those things that is a little irritating and especially noticeable on older hardware.

In any case, this is Linux so it really doesn't matter too much what environment you use. You can run Amarok (KDE) in Gnome for example and Totem (Gnome) in KDE without trouble. And with some effort you can even run Nautilus in KDE, and kdesktop in Gnome (kdesktop draws the desktop icons for KDE). Or I know many people don't like DEs in general and just run fluxbox or some other window manager and put the applications they like on the menus.

Anyway, the point is you can choose what's right for you.

### Post by miggols99 on 2007-08-16
You can change the font sizes. Go to:

Control Centre >> Appearance and Themes >> Fonts

Now change them to a smaller size. You can click on a button like "Configure all" (I'm using Xfce at the moment - slow comp) which lets you change them all at once.
