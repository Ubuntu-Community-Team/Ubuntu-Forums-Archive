### Post by Petro Dawg on 2012-08-29
Well, if you are using linux, then most any wallpaper site is safe.

I really like [http://www.desktopnexus.com/](http://www.desktopnexus.com/) as it will size the images to the resolution of your screen automatically.  Has a pretty good selection.

I don't much mess with themes, but if you install MyUnity (if you are an ubuntu user) or install advanced settings by installing the gnome-tweak-tool you can change a lot of your theme settings.  

```
sudo apt-get gnome-tweak-tool
``` 

I believe that is the terminal code.
