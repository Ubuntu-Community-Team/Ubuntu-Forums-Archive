### Post by iPCGuy on 2009-11-18
If you have ever tried to install Ubuntu 9.10 using Wubi in a dual boot, you will notice it wont boot into Ubuntu, it fails. (Error 15 I think it is called: Cannot find certain Grub boot files) As of now there is no real fix but i found a way to get around this mess. OUTCOME: get 9.10 (Or latest version) running in sucesful Wubi Dualboot with windows.

Here is how:
-Install 9.04 jaunty first:
[http://releases.ubuntu.com/jaunty/](http://releases.ubuntu.com/jaunty/)
-Finish the install with Wubi like any other (on Windows partition or your own that is above 30gigs)
-boot into Ubuntu, finish the install. THen go into Update service, and click install the latest version of Ubuntu (As of now 9.10).

Now u will have Ubuntu 9.10 (Or the latest out there when you read this topic) sucesfully dualbooting with windows working around Error 15 Where the files are not found.
