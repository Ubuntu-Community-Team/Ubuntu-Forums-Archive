### Post by Maupertus on 2010-11-02
Although I'm not as afraid of Mono damaging my free software rights, I am concerned by Canonical choosing Banshee for an other reason.

When I started using Ubuntu (Hoary) I slowly came to respect the common held belief in Linux-circles that you should use programs best suited to the task at hand. It is very easy to integrate certain features into one bloaty piece of software, but in the end, this will only lead to something that may have the capabilities to do everything and Granny's dishes, but it will be so incomprehensible and resource heavy, that no-one will actually benefit of these features.
Is it really so bad to use different programs for different uses, especially if they integrate easily? No and it is Ubuntu's focus on the integration of different applications into its user focussed environment that I've always loved.

I've never had a problem with Totem, I've stuck with Evolution and everytime I use Thunderbird I think people don't understand the wonder that is Evo. But most of all, I've switched to Foobar on Windows because it is the only player out there that gives me the same ease of use and solid performance that Rhythmbox does.
I've run Rhythmbox on everything, even the toaster I've got running Sarge, and it manages large libraries exceptionally well. I've never had any problems with it. I don't need an EQ, because I use a good soundcard and stereo system on my main computer. I don't want visualizations because...well I've never understood why anyone would want them. I want some organisation, an easy UI and for the program to just work.

Banshee has done none of these things for me. I'm now trying the 1.8 release because of UDS and I've had problems all the way. Both my netbook and AMD64 have problems running it at a decent speed. I have no idea why I want all those mem-hogging album art things as my main focus and why in gods name do I want Banshee to manage videos?

For me, Linux is choice and freedom of choice. Because of that, you shouldn't pile on the most feature rich software in an initial install. You should package the things that work and give the option for those users that feel that this is to bare bones to improve on it. With the new Software Centre, this has become even easier that before (and come on, it was pretty easy before USC) 

So I guess what I'm saying at the end of this rantish post is: I get the move to Unity, LibreOffice seems logical as well, these are 'political' choices that can go wrong, but at least have a thought and idea behind them. But why would you ever include a piece of software in the default install that's completely contra the philosophy of the rest of your distro. Ubuntu just works, and should remain to do so.

I apologize for the length.
