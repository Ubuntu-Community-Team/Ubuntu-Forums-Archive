### Post by 3Miro on 2011-10-04
> **Throne777 said:**
> Am I right in thinking that I can't repartition my hard drive with 10.10 already installed because it'll throw up kernel panics?

Repartitioning a harddrive with an OS already installed can be tricky. I would make sure to backup everything important first.

Having said that, I have repartitioned my HDD with various Linux distros installed without getting kernel panic. Depending on how you do it, you may have to edit /etc/fstab and reinstall grub, but that's about it.

If I have one partition with 11.04 (or 10.10) and I resize it from the back creating another partition, then install 11.10 on the newly created partition, everything should work fine.

Again, it depends on what you are doing and how. In either case, you should make sure to backup all important data.
