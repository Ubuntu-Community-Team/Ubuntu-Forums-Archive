### Post by ibuclaw on 2008-05-16
if it's a site (.com, .co.uk, .org, .biz, .co.nr), firefox takes you to that site. Else it sends out a search query to your DNS Address. If no DNS is found, it uses google.

I have [OpenDNS]("http://opendns.com/") as the DNS Address for my Router. So all Firefox searches go through [http://guide.opendns.com/](http://guide.opendns.com/) instead of google.
