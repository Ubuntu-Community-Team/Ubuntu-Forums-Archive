### Post by master_kernel on 2007-11-28
I stumbled upon a post that was so cool, I just had to make a new thread. Use the following code to generate an HTML-formatted version of your system specifications:
```
sudo lshw -html > ~/hardware_info.html && firefox ~/hardware_info.html
```
Then upload it to the server of your choice so we can all see it!

Here's mine: [http://kcheck.sourceforge.net/other/hardware_info.html](http://kcheck.sourceforge.net/other/hardware_info.html)

Note: Free *instant* webhosting can be found here: [http://xthost.info/](http://xthost.info/)
