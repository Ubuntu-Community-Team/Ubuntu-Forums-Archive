### Post by natrixgli on 2008-04-26
Trixbox = very stable, easy to maintain and configure. (CentOS based)

VoIP Providers = unpredictable... 


I recommend taking this to the Trixbox forums and searching there, you'll get better results. Trixbox has web management for nearly every aspect of the system. Personally I like it best when used with Sangoma analog line cards. 


Note that if you're looking to do telemarketing you probably won't get much help ;) It would be like posting to the Postfix list asking how to send spam...
