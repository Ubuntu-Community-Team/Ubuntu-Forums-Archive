### Post by sudodus on 2015-03-10
Terminology is really fancy :KS

But if you want something really light-weight, you can brush up old xterm with some options. There is a huge list of options in ```
man xterm
```

It looks pretty nice with

```
xterm -fa default -fs 13
```

and even better with

```
xterm -fa default -fs 13 -foreground black -background beige -sb -rightbar -title "Hello World"
```
