Don't Know what Ubuntu user days are ?
User Days was created to be a set of courses offered during a one day period, 
to teach the beginning or intermediate Ubuntu user the basics. To get them started with Ubuntu. 
User Days is a series of online courses where you can:

               ***** learn how to install Ubuntu
               ***** find equivalent programs in Ubuntu
               ***** learn how to get help
               ***** learn the basics of how to use Ubuntu
               ***** learn how to get involved in the community

SO Come on by on Irc learn some things. Or teach if you like. There are still open spots. For more information you may have a look at this link 
[https://wiki.ubuntu.com/UserDays](https://wiki.ubuntu.com/UserDays)
