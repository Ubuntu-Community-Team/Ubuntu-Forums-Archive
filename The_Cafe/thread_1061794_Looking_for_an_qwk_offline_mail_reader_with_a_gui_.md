### Post by wmcbrine on 2009-02-06
The last QWK reader in the world, TTBOMK, is the one I maintain: [MultiMail](http://multimail.sf.net/). ([Get it here for Ubuntu](apt:multimail). Note that the command name is "mm". And you'll probably want to read the man page.) And even that project is nearly dead now. It started as and remains a Linux program, though it's since been ported to other OSes. However, it's not quite GUI; it's curses-based.

I've only ever seen one GUI QWK reader for Linux, and you probably don't want it. You can find it [here, as "X11-QMR"](http://www.ibiblio.org/pub/Linux/system/mail/mua/). Note the dates. Some other (non-GUI) Linux QWK readers are [here](http://www.ibiblio.org/pub/Linux/system/bbs/mail/).
