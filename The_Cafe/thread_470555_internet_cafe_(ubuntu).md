### Post by prizrak on 2007-06-11
Not that you need it anymore. I don't think Ubuntu can really work in an internet cafe, at least not 100% of the machines. One of the things people go to those places for is gaming and lets face it, aside from a handful of games there is really no way to run that. 

FYI:
Best thing to do in such a situation is a terminal server setup with the workstations working as dumb terminals and a central server holding all of the stuff. Obviously should be clustered with mirrored RAID NAS system to avoid a single point of failure but easiest to manage once up and running. Also requires very basic hardware for terminals. Of course this would be for a something that won't do any gaming.
