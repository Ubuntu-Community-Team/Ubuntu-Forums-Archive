### Post by snowpine on 2011-12-21
"Stable" in Linux development simply means that features are "frozen" and no surprising new features will magically appear. "Stable" in this case does not refer to the number of bugs present in the final release. Bugs continue to be identified and patched for a certain interval after the release date; for most Ubuntu releases this is 18 months (60 for LTS).

Ubuntu is a community project; if you feel there are bugs and instabilities, have you reported them through the proper channels so that they're fixed in the next release?
