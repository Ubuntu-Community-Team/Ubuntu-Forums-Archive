govts and military use systems that meet certain criteria through certification such as in the US the orange book or now common criteria, in UK then ITSEC or common criteria.

Linux or Windows or whatever if certified to a given level cna be used, depends if they want MAC or DAC, SELinux for example providing MAC and Windows providing DAC.

Windows is not insecure, it is the administrator or the security policy or design which makes it so within the environment where it is.

Windows has flaws yes, which can be hardened or made worse.

As for which one is harder to crack, again all comes down to version if in vanilla state etc and often entry is through a given service and its vulnerabilites and not the OS itself.

The military and govts use Windows alot, do you think they would if it was so insecure ? i mean its not a cost issue is it ;-)

it is all academic, it comes down to who is running/administering the system, sec policy, asset value etc
