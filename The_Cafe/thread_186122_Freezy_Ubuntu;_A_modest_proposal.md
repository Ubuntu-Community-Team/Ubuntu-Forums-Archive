### Post by dufnutz on 2006-06-01
The name has been changed from gnubuntu to ubuntu-libre and the website is can be found here [https://wiki.ubuntu.com/Ubuntu-libre](https://wiki.ubuntu.com/Ubuntu-libre)

The name was changed to appease RMS, because he felt ubuntu should not be associated with gnu if it provides non-free software, even if this particular edition does not.
