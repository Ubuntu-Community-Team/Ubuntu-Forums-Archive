### Post by koenn on 2008-07-28
I think it has to do with the fact that the learning curve for system administration on Windows is less steap. A desktop power user can, with some training, start with easy sysadmin tasks and grow in to it. 

Linux/unix system administration is a bit harder, apparently, and often made more complex by the additional requiremnt that businesses often want Linux servers to integrate in a Windows environment. 
Often, they out-source the linux system administration to external, specialized firms, system integrators or infrastucture integrators. Maybe you can look for those, then check their websites for job offerings.
