### Post by *grin* on 2007-11-18
I downloaded 'bluefish text editor' from the synaptic package manager yesterday evening for pretty much the same reason.  I've not explored it fully, and I'm not sure it will give you all the power of Dreamweaver, (bearing in mind you're not fluent in HTML) but I can say that it has a lot of pre-built code and icons for things like images and anchors.  

It may suit your purposes esp. if you're going to continue in Dreamweaver and use this for small edits.

### Post by Kadrus on 2007-11-18
You can try using Bluefish
```
 sudo apt-get install bluefish

```
Try using Quanta:
```
sudo apt-get install quanta

```
You can try using Screem:
```
sudo apt-get install screen

```
And for CSS editing use CSSED
```
 sudo apt-get install cssed

```

Hope that helped :)
