### Post by DigitalAxis on 2007-07-02
[IMG]http://www.chara.gsu.edu/~riedel/images/DigitalAxis.jpg[/IMG]

The basic philosophy here is, I have a widescreen laptop, but most of my data (PDFs, etc) are still vertical... so why not rearrange the space a bit?

It's standard Kubuntu, with:
* The wallpaper replaced with a 1920x1200 crop (by me) of [this Hubble image]("http://hubblesite.org/newscenter/archive/releases/2007/16/image/a/")
* The panel made vertical, 72px wide and a second identical panel added (I wish KDE would let me make those taskbar buttons taller)
* All major fonts replaced with [Bastardus Sans ]("http://www.dafont.com/bastardus-sans.font") (which is pretty with the default antialiasing, but doesn't come in bold or italics)
*Kicker-applets installed for the mixer and storage media applets
*The new [Crystal iconset]("http://www.everaldo.com/crystal/") by Everaldo (a few icons, such as the AIM one [really Pidgin], the 'web' one [Opera] and the Laptop [actually Home] were swapped for Crystal icons just because they looked better)
*Panels locked so I can hit logout by just scrolling to the top-right corner of the screen.

Since I posted this I've reduced the size of the default Konqueror font; it looks better to me now.

I'm not sure if they are the same that you installed, but with these the reflection effect is included.

You can't access reflection settings from the gui yet, you have to run gconf-editor, go to apps/avant-window-navigator/bar and add these keys (if they're not already there): "angle" and "icon_offset"

### Post by ayoli on 2007-07-09
> **el mariachi said:**
> 

Is there a way to make the upper panel's text white?

edit your gtkrc file, and put something like this at bottom of file:
```
# panel stuff
style "mypanel"
{
  xthickness                    = 0
  ythickness                    = 0
  # uncomment line bellow and put path of a png for panel background if you want
  # bg_pixmap[NORMAL]             = "panel-bg.png"
  fg[NORMAL]                    = "#ffffff"
  fg[PRELIGHT]          = "#ffffff"
  fg[ACTIVE]                    = "#353535"
  fg[SELECTED]          = "#353535"
}

class "*Panel*"        style "mypanel"
class "*Notif*"        style "mypanel"
class "*Tray*"         style "mypanel"
class "*Applet*"       style "mypanel"
class "*Manager*"      style "mypanel"
class "*Netstatus*"    style "mypanel"
```
After, that logout/in gnome should reload your theme and do the trick.

### Post by ayoli on 2007-07-09
> **el mariachi said:**
> where can I find it? I used the search function but I don't know which one is it..

themes are located in two places:
wide:
```
/usr/share/themes/
```
or userspace:
```
~/.themes
```

found the theme you use in one of these two place, this is a folder with the name of your theme:
```
<yourtheme_name>/gtk-2.0/gtkrc
```

hope that helps

### Post by alphane on 2007-07-10
The black quits for me after less than a second, so I'm not too bothered about that one.  The flickering is to do with double buffering.

If you're comfortable editing your xorg.conf, open it up and under the section called "module"
add in the following:

**REMEMBER TO BACK UP YOUR XORG.CONF - JUST IN CASE!**

```
	Load	"dbe"
```

"dbe" = double buffer extension (at least in my head it does).

Now, when I edited it, logged out and logged in, it screwed over my desktop (couldn't see anything, all text disappeared etc) but a reboot seemed to fix all that.

If it doesn't work, just remove the line, or restore your backed up xorg.conf.

(Sorry, I get hyper paranoid with xorg.conf, I've been thrown to command line so many times...)

### Post by TheKid965 on 2007-07-31
*How did you make VLC look like Windows Media Player? *

1.  Download the WMP11 skin, available [here](http://www.videolan.org/vlc/skins.php).

2.  In VLC, Settings -> Switch interface -> Skins 2.  

3.  In the new interface, right-click anywhere but the video window, Select Skin -> Open Skin, then select the WMP11 skin you downloaded.  

4.  Enjoy your WMP-lookalike VLC.  8^)
