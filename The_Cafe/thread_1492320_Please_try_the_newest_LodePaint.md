### Post by Zeemvel on 2010-05-26
> **del_diablo said:**
> What does this application have that GIMP needs?

It fills some gaps that Gimp doesn't have here and there. It's made more tailored for pixel editing (rather than publishing) with such things as a color per mouse button, arrow keys that move the mouse one pixel and such. It's easier to see the exact RGB value of the color of each pixel, and such.

It's got a totally different philosophy than Gimp, so that it can be complementary to it.
