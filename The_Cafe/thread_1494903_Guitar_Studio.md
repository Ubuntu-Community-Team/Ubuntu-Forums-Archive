### Post by cchhrriiss121212 on 2010-05-27
There are a few good guitar based FOSS apps: guitarix (an amp simulator), rakarrak (realtime fx), tuxguitar. You could do all the rest you want with jackd (sound server), Ardour and Hydrogen (drum sequencer). Linux pro audio is set up to be a modular affiar, so that each thing you do is a seperate program connected with jack, as opposed to windows DAWs which have an all in one approach.

> You won't find anything that comes even remotely close to Cakewalk on Linux.
Ardour, qtractor and LMMS are all remotely similar.
