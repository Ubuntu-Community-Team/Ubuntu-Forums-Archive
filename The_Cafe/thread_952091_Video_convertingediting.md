### Post by Ubuntiac on 2008-12-15
For anyone reading this, if you're going to try Kdenlive, I would recommend *against* the version in the repo's which is over a year old, grabbed from SVN and not intended for public release and has been since almost completely rewritten.

Kdenlive is great, but just make sure you're using at *least* version 0.7 before trying it. There's a great GUI app for building the latest version from source for you at Kde-Apps.org called the "Kdenlive Builder Wizard". It also sounds like in the next couple of weeks we're going to see packages for Intrepid. Just read the link in my sig for further details.
