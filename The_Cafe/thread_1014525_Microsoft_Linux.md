### Post by jespdj on 2008-12-18
> **Grant A. said:**
> TBH, Microsoft should embrace Linux. What ever happened to extend, embrace, and then innovate? It seems now like it's extend, embrace, and ultimately extinguish.
Microsoft has done this numerous times in the past with standards: take a standard, add your own proprietary extensions to it, make everybody use the proprietary extended version and kill off the competition that way.

They can't do that with Linux because Linux is GPL'ed, which means that they can't add proprietary (closed source) extensions to it. The GPL ensures that if you do extend Linux, you must publish the source code of your extensions. Microsoft obviously doesn't want to do something like that, they want full control over their operating system and what the user can do with it.

Ofcourse they're never going to support an operating system like Linux, which they can't make money on and which competes with their cash cow Windows.
