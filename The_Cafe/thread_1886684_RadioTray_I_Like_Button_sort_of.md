Hi everybody,

Just want to share with you a little script that I did this morning, I suppose it can be helpful for others...
I'm used to listen music through internet radios with radio tray, but when I like something, I never had the idea to write down the name of the artist / song for further investigations. 
So I did a small shell bash script than extract from the notify osd log the name of the last song played in radio tray, and then copy it to a file.
In my computer, I associated it with a keyboard shortcut.
You'll find attached the script.
I hope it will be helpful for some of you.
