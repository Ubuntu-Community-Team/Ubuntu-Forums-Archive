### Post by Jucato on 2006-06-19
It's really up to you. If you love extensions/add-ons and prefer more compatibility with websites, you'd be better off with Firefox.

If you prefer desktop integration and some nice features (spell checking, integration with KGet, better bookmarking features) then I suggest you try Konqueror. 

I personally use Konqueror for both browsing and file managing. Except for certain times when Konqueror either fails to load sites properly or crashes when loading some sites. That's when I use Firefox. I don't quite mind using Gmail in Konqueror (use browser spoofing). Besides, I use KMail mostly so I rarely visit GMail, except to erase some rubbish. :D
