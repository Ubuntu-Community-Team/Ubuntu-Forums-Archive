### Post by soiled skivvies on 2008-10-11
It's the most generic thing you will ever see and it does take some terminal downloads to make it all run the right way, but yes you can set it up to dual boot, and ask you which to boot, the gaming side or the OS side.  Wiring mods on the controller ports to make a usb port out of it.  The wires are the same colors, so no soldering needed, just grab the extender cable and cut it.  You can add extra ide drives, cd burning, hard drives, etc.  Unlike most pc's, the video out supports tv and looks amazing just from composite cables (red, white, yellow).  p.s -->  it has vga support ;)

BTW, with a cat6 cable direct connect to your computer, the ftp name and password is XBOX, in lower case.  The gaming side runs in root mode, but when you boot the desktop, it doesn't run in root mode.  it's fun to play game on it when you're so used to pc games using a real keyboard.


note:  I also got the xbox360 controller keyboard to work on windblows xp.  Fully supported, just some soldering ;)
