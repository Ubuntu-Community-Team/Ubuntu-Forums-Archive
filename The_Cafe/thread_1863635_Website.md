You can't make web sites with photoshop.  The corresponding image editor in Linux is GIMP.  I find it superior, for my needs, than Photoshop.

### Post by DeadSuperHero on 2011-10-18
Honestly, I feel that just using a good content management system like [Drupal]("http://drupal.org/") is enough. Alternatively, learning a web framework such as Django or Ruby on Rails is a great way to roll your own system and build it up from scratch.

Just make a good mock-up in Gimp, invest in learning some CSS (WC3Schools.com is a great resource!), and find the best way to translate your work from an image into CSS. There are some great starter theme engines to build from and get things going.

Plus, the Block system in Drupal is terrific for setting up layouts with a minimal amount of PHP code, and the [Views module]("http://drupal.org/project/views") is terrific for basically creating any sort of visual output you want.
