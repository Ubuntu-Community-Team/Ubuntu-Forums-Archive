### Post by jamesstansell on 2009-12-29
Hi Gizenshya,

What you're talking about is javascript, not java.  Java and javascript are different things.  (The difference is a whole different subject.)

From a brief glance at the link you typed in, it sounds like what is happening is just making use of web browser functionality.

Does closing the browser tab restore your system functions?  If not, what about closing the entire browser?  If the functions are still messed up then you probably have something different than the javascript code you found.

For right-click being dis-abled (or altered) it's typical to allow a modifier key to re-enable it.  Typically something like alt or ctrl, held down before you do the right-click.

For some of this stuff it makes a big difference which browser you're using, so (for example) MSIE on ms/vista could have different behavior than Firefox on ubuntu.  Or, even for the same browser different versions sometimes behave differently.  (Usually the newest releases are the most 'correct'.)  Also, sometimes the plugins and other addons that you've installed will make a difference.

Sorry for rambling.  If you want to know if other people have the same problems then post  information about what you're seeing, starting with the browser in use and the web site that causes the issue.

Sincerely,

-james.
