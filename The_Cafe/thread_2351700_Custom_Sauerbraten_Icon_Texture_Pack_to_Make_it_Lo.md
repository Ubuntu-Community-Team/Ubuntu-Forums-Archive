Note:
Before modifying the textures or files in your game make sure to  close the program to prevent corrupting your game files. This should be a  relatively easy modification to make even if you have no idea how to  "mod."

Installation Instructions:
First you need to download the "Sauerbraten New Textures" package and  then change the ownership of the contents in the directory /usr/share/games/sauerbraten/ to your current user. To do this go to your terminal and type in the following command sudo chown (your username) -v "/usr/"  and repeat this step for all of the files that you will be modifying or  if you have a right-click option with root commands use that. Then you  need to enter the directory /usr/share/games/sauerbraten/data/ and copy the contents from your downloaded files directory ~/Sauerbraten New Textures/opt/sauerbraten/data/  and paste it into the "data" folder for your game and replace all of  the conflicting files. Then you need to enter the directory /usr/share/games/sauerbraten/packages/icons/ and copy the contents from your downloads directory ~/Sauerbraten New Textures/opt/sauerbraten/packages/icons/ and paste them into the "icons" folder for your game and replace all files. Finally you need to enter the directory /usr/share/games/sauerbraten/packages/hud/ and copy the contents from your downloads directory ~/Sauerbraten New Textures/opt/sauerbraten/packages/hud/ and paste them into the "hud" folder for your game and replace all files.

After this is all done your game should look beautiful and less rusty!
