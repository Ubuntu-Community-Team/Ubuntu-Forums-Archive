You would use the dd command:
```

dd if=/dev/pendrive1 of=/dev/pendrive2

```
pendrive1 represents the source disk, and pendrive2 represents the destination disk. Of course, you will need to know the actual device names of the USB drives. Make sure you don't mix them up, or you could loose data.
