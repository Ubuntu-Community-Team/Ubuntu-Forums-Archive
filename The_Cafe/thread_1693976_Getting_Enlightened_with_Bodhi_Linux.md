### Post by beastrace91 on 2011-02-23
Nearing it's first stable release Bodhi Linux is a "minimalistic, enlightened, Linux desktop". Focusing on user choice and easy customization Bodhi's use of the Enlightenment desktop also makes it suitable for older systems. Also provided are weekly snapshots of the Enlightenment desktop built right from the E developer SVN along with the widest selection of EFL related modules and applications any distro can offer. Of course a picture is worth 1,000 words - so here are some shots of some of the 42 default configurations Bodhi offers:
