### Post by crun on 2005-10-01
The Matrix code alphabet was specifically designed for the movie:

[http://www.animallogic.com/film/matrix/thematrix.html:](http://www.animallogic.com/film/matrix/thematrix.html:)
> The Matrix code was designed around a paradox: it had to be familiar enough to reflect a modern computer code yet original enough to appear futuristic. To this end, an entire alphabet was created, a combination of eastern and western characters incorporating numerals, letters and symbols. Designer Simon Whiteley sourced references from manga comics, combining Katakana script with chopped up alpha-numerics.

Since it is a real alphabet, it must have been tempting to slip a few hidden messages in there somewhere :)
