### Post by doorknob60 on 2012-08-09
If you are using Virtualbox you aren't supposed to use your laptop's audio drivers, because Virtualbox uses emulated audio hardware. All you need to do is enable audio in Vbox's options and install Virtualbox additions (you probably don't even need that, but you should do it regardless). All that assuming you use Virtualbox anyways, but I think VMWare does it similarly.
