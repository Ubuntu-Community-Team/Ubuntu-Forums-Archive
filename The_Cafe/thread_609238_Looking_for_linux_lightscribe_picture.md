The LaCie software works with any drive that does lightscribe as far as I know. There are basically 2 programs for linux that do lightscribe labels. 

one is from LaCie and is commonly known as "4L"

The other is from lightscribe.com and is known as Simple Labeler.

The software at lightscribe.com seems to be updated more frequently, but both of these apps burn in different ways so it's handy to have both.

### Post by wizard10000 on 2011-05-21
LaCie's lightscribe labeler is IMO the best Linux labeler out there.  You can run it on a 64-bit system if you have ia32 libs installed and install with dpkg -i --force-architecture.

The labeler needs to be run as root but that's not a huge deal.  I've used it for years.
