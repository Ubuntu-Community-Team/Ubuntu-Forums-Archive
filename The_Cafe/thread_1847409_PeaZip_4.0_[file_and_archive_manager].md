### Post by Giorgio tani on 2011-09-20
I'm glad to inform you that PeaZip 4.0 was released yesterday, you can find all downloads, complete change log, and documentation on [www.peazip.org]("http://www.peazip.org")
Packages are compiled for GTK2 (recommended) and Qt widgetset, both works on Gnome and KDE, the application is desktop neutral.
Alongside DEB package for Debian / Ubuntu systems it is available a portable version that does not need to be installed.

Any help in improving existing localizations is very welcome, package for translators is: [http://peazip-translations.googlecode.com/files/peazip-4.0.about_translations.zip](http://peazip-translations.googlecode.com/files/peazip-4.0.about_translations.zip)
