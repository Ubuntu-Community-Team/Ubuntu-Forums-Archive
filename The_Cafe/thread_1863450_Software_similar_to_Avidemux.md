### Post by alphacrucis2 on 2011-10-17
For some reason they couldn't get the package to build on Oneiric so it was dropped from the repos as they ran out of time. You could try the natty package instead. Alternatively download the daily source for the experimental 2.6 version (works well for me). 

Download 2.6 nightly source:

[http://www.avidemux.org/nightly/source/](http://www.avidemux.org/nightly/source/)

Instructions here to compile, build packages etc:

[http://www.avidemux.org/admWiki/doku.php?id=build:install_2.6](http://www.avidemux.org/admWiki/doku.php?id=build:install_2.6)

Or 2.5.4 version from natty:

[http://packages.ubuntu.com/da/natty/avidemux](http://packages.ubuntu.com/da/natty/avidemux)

Here is the launchpad entry to read about the Oneiric problem:

[https://bugs.launchpad.net/ubuntu/oneiric/+source/avidemux/+bug/831096](https://bugs.launchpad.net/ubuntu/oneiric/+source/avidemux/+bug/831096)

Looks like it has now been sorted out but it doesn't appear to have landed in the repos as yet?

Reading the bug again, it sounds like the fix may only be getting delivered to the precise (12.04) repos at this stage.
