### Post by MasterNetra on 2009-10-08
Pardus for me isn't a keeper, at least not quite yet. Though overall it does seem to behave quite well, more so then I've had a KDE distro behave. However, that said it package upgrade can use a bit more work (or their repo) as for whatever reason I cannot download all the upgrades from it, it trys to load the next screen but fails and instead just hangs with the busy cursor. While with other updates it had no issue. Almost tossed it as useless to me even earlier with how you have to setup a wireless connection, me being a gnome user I was expecting the clicking network icon and selecting the network it was only after some playing around did I learn you got to create a profile and scan for the target network from it. I don't remember if that is for all KDE or just this and perhaps a few other distros but that would minus points for newcomers. Either way the issues with the upgrades kills it for me, otherwise it is a well polished and seems to be a stable KDE Distro.
