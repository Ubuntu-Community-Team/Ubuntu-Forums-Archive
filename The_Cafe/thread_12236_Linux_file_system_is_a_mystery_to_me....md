[QUOTE=mrgardon]Can anyone explain to me how this Linux file system works?[/QUOTE]
 Sadly its still confusing to me.  Check this for a helpign hand: [http://www.linux.org/lessons/beginner/l4/lesson4b.html](http://www.linux.org/lessons/beginner/l4/lesson4b.html)

Just remember:
~ is a shortcut to your home folder.
/dev/ for device nodes..kind of like /windows/system I guess?
/etc/ for configuring your programs
/usr/share/ for programs for all the computer's users.
/opt/...umm the dumping ground I guess?

anything with a "." in front is a hidden file.
