### Post by el_ricardo on 2008-04-09
[http://ubuntulite.tuxfamily.org/](http://ubuntulite.tuxfamily.org/)

this is my new favourite lightweight distro, installation involves installing a command line based system then simply running a script which automatically gets a bunch of packages, deletes a bunch of packages (i think) and sets everything up beautifully for you. It runs openbox by default, with fbpanel, and a really minimal collection of software.

I think i like it because it gives the option of building up the system you want from the ground up, choosing exactly what software you want.

has anyone else used this? I personally love it! it's finally resurrected my old laptop!
