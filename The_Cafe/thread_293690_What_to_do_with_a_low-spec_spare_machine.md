### Post by VinylPusher on 2006-11-05
So, I just cobbled together a machine from various parts and ended up with:

P2-450
384MB PC133 SDRAM
Nvidia Riva TNT AGP video card (I kid you not!).

I've also got a collection of various HD's between 60 and 120GB in size.


Now that I've got the thing, I don't know what to do with it. Thoughts so far: -

1) Mailserver.
2) Caching DNS server.
3) Webserver.
4) File server.

The thing is, I don't really need any of the above.

A mailserver would give me what kind of advantage? I have hosting to handle most mail and two gmail accounts for everything else.

Caching DNS server would be a waste of power. My page load times are fast enough already.

Webserver. Again, I already have hosting. Plus, I've only got about 80KB/s outbound bandwidth.

Fileserver. I've got a 300GB drive in my workstation. That drive is a lot newer than the spares I'd be putting in the new box, so using it as a backup seems a bit daft. I've got a DVD-RW for backups in any case.


What shall I do with it? It's not really worth much to sell (I've no monitor, keyboard or mouse I can sell with it). Bearing in mind that I watch my power usage, so anything I *do* use it for, it's got to earn its way!
