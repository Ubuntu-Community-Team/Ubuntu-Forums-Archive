### Post by Jose Catre-Vandis on 2006-04-03
Try here for answers
```
http://windowsxp.mvps.org/index.html
```

If nothing else works, and you have a proper install CD (not a recovery cd that wipes everything) then do a repair install. This will preserve your settings and files.

Make sure your PC can boot from CD first (fix this in the bios if needs be)

Insert Xp CD
let it boot
Select Install Windows
Press R to repair XP
Add product key, network and regional info as required
Wait 30 minutes
Job done
