### Post by Keyper7 on 2009-11-04
The [recent discussions about Seahorse](http://ubuntuforums.org/showthread.php?p=8240973) made me wonder... who does actually use it? For changing and removing saved passwords, the average user usually goes directly to the app. For synchronizing remote keys... well, the average user does not do that at all, I think.

So I got curious and decided to set up this poll. I'd likely to politely request two things before you reply:

- forget about security
- forget about bloat

I'd like you to assume that all your security concerns about Seahorse are completely fixed and that it occupies exactly 0Mb in your hard drive. I'd like you to be objective and base your answers only in **functionality**. Answers like "I would never use it because the current implementation is insecure" and "I'd like it removed from the default because my hard drive is small" would distort the results.

And remember: Seahorse != Gnome Keyring. Removing the former does not remove the latter.
