### Post by pc_magas on 2015-02-23
Actually I use codeigniter. I **LOVE** the documentation very straightforward, clean. Just build for dummies. Also Codeigniter do not need by default fancy mod_rewrite and plays even if it is on a subfolder instead of symphony and is actually very basic, neat and simple.

I also used for a while and Yii framework that is strongly inspired by Ruby on rails. And I loved Codeigniter cause it does not jave jquery and other stuff like bootstrap (but I miis the CRUD functionalities that Yii has and auto code generation)
