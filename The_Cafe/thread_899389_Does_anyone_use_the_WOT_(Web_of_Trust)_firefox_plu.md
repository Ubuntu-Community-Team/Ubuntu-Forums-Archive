### Post by benny bronx on 2008-08-24
While I have not used WOT, it has been discussed in the security forum I belong to, and the consensus is that it does what it claims to do, is stable, and does not noticeably effect browsing speed. However, the site ratings are defined by the users, and may be inaccurate at times.  Also, from a security standpoint, it might be a bit of overkill in that we are running a secure OS, and FireFox 3 has built-in security features that will alert you to attack and phishing sites.
