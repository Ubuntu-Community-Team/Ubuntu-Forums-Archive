### Post by PreviousN on 2008-05-06
There are some pretty easy softmods to do to the ps2. If you've got a fatty, you can use an original ps1 disk and have it boot into linux/a diff bootloader just by having a certain save file on your memory card. 

You can put in a hard drive if you have the network adaptor, and there are some pretty good open source media center programs floating around for it. Specifically, this program known as SMS is pretty cool. 

Do a google search for "ps2scene" and you'll find some cool stuff out.
