### Post by eriqk on 2007-01-30
> **kpkeerthi said:**
> Where to get the deb of 2.3.14?

I'm not sure if there's a .deb, but it's not hard to compile (it just takes a while).
Make sure you do ```
sudo apt-get build-dep gimp
``` first, though. This will sort out the dependencies.
You could also do ```
./configure --help=short
``` in the Gimp folder to see what compilation options there are.

Groet, Erik
