```
touch ~/.dmrc; chmod 644 ~/.dmrc; chmod 750 ~

which will create the .dmrc file if it doesn't exist, set the right perms on this file, then correct perms on your home folder as well, just incase(DO NOT USE 644 or 755, 644 will break your home folder till you fix it, and 755 is insecure).
