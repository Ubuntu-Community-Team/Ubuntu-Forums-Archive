### Post by aysiu on 2006-12-12
*aptitude* by far.

Most of the other Linux applications I use either have Windows ports or Windows equivalents that almost as good, just as good, or better.

These are the programs I use that all have Windows ports: Thunderbird, Firefox, OpenOffice, GIMP. I love Gedit, but Notepad++ isn't so bad for Windows. I love Kolourpaint, but MS Paint will do, I guess.

There is no substitute in Windows for *aptitude*, though. I can use *aptitude* to install twenty programs with a single command or upgrade my entire system (including newer versions of all the installed applications) every six months with one program instead of twenty setup.exes and ten reboots.
