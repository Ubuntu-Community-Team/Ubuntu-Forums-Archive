### Post by keithpeter on 2010-09-02
Hello Infection

Lubuntu has sylpheed 3 as the mail client. You can see attached image files displayed below the text of the e-mail. I use sylpheed by choice as I like ascii e-mail!

Text is always plain text. If the sender has their program configured to send ascii then html as a mime attachment, then you read the plain text one.

If they have not set their e-mail program up properly so it just sends html, then you shouldn't get the 'angle bracket soup', just the text, but html entities (e.g &emdash; ) are written as codes.

The other possibility would be to install Opera as an alternative Web browser. It has an e-mail client that a lot of people like, and is supposed to be lightish on memory.

Cheers
