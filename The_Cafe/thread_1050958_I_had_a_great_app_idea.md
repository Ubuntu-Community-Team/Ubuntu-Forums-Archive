### Post by davbren on 2009-01-26
well... i think its great. I'd love some help with it in the future if anyone is interested.

The name of it (for now) is "Homework"

It would be aimed at school kids of any age who need to use a pc for their work.

Some might suggest that a word processor is the best tool for that. I disagree. I feel that a desktop publishing tool is better for the job. 

So how about a lightweight application in the same frame as Abiword?

We all know that MS Word is jam packed with features. Most of which we don't use. If can collect together the key features, like Abiword, then create a decent way to manipulate pictures and diagrams (much like smartart in '07) 

I was thinking an app in the picasa mould. Preset options that are known to work but with the flexibility to fiddle if needs be.

Preset templates, that are really useful. Effective tooltips that genuinely help with document composition. Sets of rules that should be followed for a certain use. For example when creating a poster, there are many ways to convey an advert. Why not incorporate this knowledge into the app...?

What do you think?
