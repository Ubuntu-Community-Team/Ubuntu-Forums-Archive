If you're not happy with the [new  Google Image search]("http://lifehacker.com/5592011/google-image-search-updates-with-a-cleaner-look-and-improved-functionality") and would prefer to stick with the old search  as your default, reader [The_Doc]("http://lifehacker.com/people/The_Doc/")  offers [this simple  URL tweak]("http://lifehacker.com/comment/26359109/"). The new Google Image search has a Switch to basic version link at the  bottom of the page, but it doesn't stick, so if you prefer the old  view, you'll have to scroll all the way down there every time. The_Doc  offers the solution:[INDENT] If you don't like the new Google Image Search design, add:
 &sout=1
 to the URL to use the the "basic" (i.e. old) version. Update your  bookmarked keyword searches accordingly.
[/INDENT][http://lifehacker.com/5594912/get-back-the-old-google-image-search-with-a-url-hack](http://lifehacker.com/5594912/get-back-the-old-google-image-search-with-a-url-hack)
