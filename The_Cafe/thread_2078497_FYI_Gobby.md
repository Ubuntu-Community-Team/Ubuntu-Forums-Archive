### Post by ki4jgt on 2012-10-31
A couple of years ago I asked you guys if there was an editor which could be used to collaborate coding like Google Docs. I recently went on my search again and found Gobby. I believe it relies on Jabber or IRC to callaborate between different coders (DON'T HOLD ME TO THAT). The code is instantaneously on screen of all users. Each user gets their own text highlight color to show who added what along with the cursor of each user being shown on screen. No GIT required. Just thought I'd throw this out there for any other user wanting to find something a little similar to google docs with coding.
