### Post by t0p on 2009-04-04
> **falkTX said:**
> I am already an Ubuntero, so this means I'm close...


Actually I don't think being an Ubuntero makes you particularly close to being anything.  [To become an Ubuntero you just need to sign a copy of the Ubuntu code of conduct]("http://www.ubuntu.com/community/processes/newmember").  Whereas to become an Ubuntu member, you must do stuff that actually *promotes* Ubuntu.

I think the documentation on this matter is a little unclear.  The [page about membership]("http://www.ubuntu.com/community/processes/newmember") does seem to imply that Ubuntero is a membership rank. But to be a member, you must make a "significant contribution to the Ubuntu community".  And digitally signing the Ubuntu code of conduct is *not* a significant community.  In fact, on the page about membership it says that "Ubuntero" is another came for "Ubuntu activist".  Sorry, but I fail to see how digitally signing the code of conduct makes me an activist.  When I answer questions on this forum, *that's* activism.  Signing the Ubuntu code of conduct... that's just signing a document.
