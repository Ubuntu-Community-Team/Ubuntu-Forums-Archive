### Post by cgroza on 2011-06-05
Just wanted to share this Firefox plugin:
[https://addons.mozilla.org/en-US/firefox/addon/firemacs/](https://addons.mozilla.org/en-US/firefox/addon/firemacs/)

I really got tired of using the mouse to browse  and scroll around.
I  miss the emacs shortcuts when posting on this forum.
So I found myself an emacs plugin for firefox(link above).
It does not include all the shortcuts but it offers the basic ones plus some more.
So if you really like emacs, this may interest you.

It is called Firemacs.

### Post by lovinglinux on 2011-06-05
You might also like [Vimperator]("https://addons.mozilla.org/en-US/firefox/addon/vimperator/").

Although I use shortcuts for some things, I still like to use the mouse. What I do to make my browsing experience easier, is to use [FireGestures]("https://addons.mozilla.org/en-US/firefox/addon/6366/") add-on. It can considerably reduce mouse movement across the screen.

I am currently using Firefox with hidden tabs and access tab groups via FireGestures shortcut (left+right mouse click). Is very easy to navigate this way.
