### Post by 3Miro on 2011-06-17
What's the point? Ubuntu 11.10 comes with Gnome 3 and Unity interface. Furthermore, you can install both:
```
sudo apt-get install gnome-shell
sudo apt-get install gnome-session-fallback
```
gnome-shell gives you the Gnome-shell interface available at login. gnome-session-fallback gives you the classic Gnome 2 panel/metacity interface. With all the libraries, we are talking about 20 or so packages.

There is no point in making a separate distribution for 20 packages.
