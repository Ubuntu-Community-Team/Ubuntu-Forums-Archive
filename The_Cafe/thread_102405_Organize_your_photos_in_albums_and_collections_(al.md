### Post by anil_robo on 2005-12-12
Hi Everyone! I just found out a cool app in Ubuntu! The name is Albumshaper.

[[IMG]http://img351.imageshack.us/img351/1048/albumshaper5dg8sg.th.jpg[/IMG]](http://img351.imageshack.us/my.php?image=albumshaper5dg8sg.jpg)
Install it by typing this in a terminal window:
```

sudo apt-get install albumshaper

``` 

To use it, just type albumshaper in a terminal window and press enter. Alternatively, you can add a launcher to a panel or desktop, with command being "albumshaper".

It's pretty easy to use. Enjoy!
