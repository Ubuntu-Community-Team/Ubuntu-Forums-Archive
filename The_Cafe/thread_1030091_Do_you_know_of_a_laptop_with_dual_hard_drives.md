### Post by TBev0 on 2009-01-04
I've got a Sony Vaio BX297XP which is a cpl of years old now with dual hard drives, 2x 100GB. It runs pretty well, but is a bit of a chunky *******. I dont mind though because i dont use it as a laptop, but rather as a desktop replacement. I usually install the OS on one drive and the /home partition on the other although it doesnt really make a difference in my case as all my data is stored on a server. I've never really used the dual HDD's to their potential.

The 17inch screen @ 1920x1200 is the best feature of this laptop for me though by far. I have a fairly beefy server running headless that holds all my storage and runs a couple of VM's. The cool thing is i can get like 4 VNC windows up on the screen and watch em all at once on the laptop monitor.

If you have a server where you can store your data on and run a few VM's, i would go for a higher screen res any day and gladly loose the extra hdd.. - You can always plug in a USB HDD if you need to.
