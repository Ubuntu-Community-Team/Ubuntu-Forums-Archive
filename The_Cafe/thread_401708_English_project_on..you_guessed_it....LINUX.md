### Post by kevinlyfellow on 2007-04-05
1. Why Linux? 

I use linux because I feel that I can stay more organized with it (at least that was the start).  Then I realized that I didn't need to spend as much time fixing it when I broke it (I break operating systems all the time trying to figure out how things work).  I remember one instance when I broke the windows boot screen and therefore needed to reinstall the whole operating system.  But whenever I broke linux, there was always a way that I could avoid reinstalling it.   I also use linux because I can make more useful shell scripts than windows.  

2. How did you first hear about Linux?

It was long ago that I had heard about it, 2000 maybe.  I just remember a friend let me borrow redhat, but my computer (66 MHz processor 16 Mb ram) was way too slow for it.  I was very interested in linux and when I got a new computer when I graduated in 2002 I bought a "The Red Hat Bible" with cds in it.

3. Which distribution do you prefer Ubuntu, Kubuntu, Xubuntu (anything else...) and which desktop environment (KDE or GNOME).

I like Ubuntu, but I always have a copy of Puppy Linux with me also.  I have always preferred gnome, although sometimes I like to play with icewm.

4. Do you like Linux better than Windows or the Mac OSX and why?

Yes!  If you break windows, you need to reinstall.  If you break linux, you just need to fix it.  I don't like the lack of control one has over Windows and also the lack of good virtual desktop support makes life difficult.

I don't like OSX for two major reasons:  I find it very confusing and more importantly, to get Mac software I need to buy Mac hardware.  Macs are expensive and in my opinion, ugly.  I would be happier to buy hardware and software separately from each other.

5. Does anyone else you know use Linux (you don't need to put specific names XD).

Yes!  A friend of mine from High School, a friend from when I was an undergrad, and I've helped a friend at my current college install Ubuntu.  I know another student who runs ubuntu and two others who are interested.  I have a professor who is fanatical about linux.

6. Do you make use of the CLI?

Yes!  Its like talking to the computer.  Its often easier for me to figure out how to do something in command line instead of clicking everywhere.  It is faster, especially if you are working with multiple files.  But pictures have there place and so doesn't written words.  Some tasks are terrible to do in the command line others are a joy.

7. What do you like best about your favorite distribution?

Ease of install.  That is how ubuntu got me to use linux (after a dry spell from redhat).  I just wanted something that would be usable (even if everything didn't work immediately out of the box) from the start.  I also really like how they chose a smart set of programs instead of every program under the sun.  I can install any software that didn't come with the distribution easy enough.  And that brings me to a third thing I like best :-):  software installation is a breeze!

8. Did you partition your drive with Windows (or the OSX) and Linux? Or did you just go all out Linux?

Up until about 4 months ago I have had windows installed, although I rarely rarely used it.  When I bought a system76 computer and moved, I left my windows install behind.  The only thing windows was useful for was playing video games.

9. What does Linux mean to you as an operating system.

It means a smart way of computing.  I really believe linux will be the future of home computing because of the security and flexibility that it has.  Open source development is superior and the open technology being developed will soon grow past proprietary counterparts.  Windows does the least that it can to get by and fills in the holes with marketing.  Linux has stubbornly clung to ideals that should be prevalent in the computer world; open source code, easy configuration, and listening to the users.  That is why it is a smart way of computing.

10. Post anything else you can think of that, just, well makes Linux great (or just better than Windows or OSX)!

Linux is not ready for everyone in this world.  There are things that would currently annoy the average user, like installing the latest versions of some software.  These things will be fixed and it will soon be ready for the average user.  This is something that makes linux great though: it will be fixed.  Some other operating systems do the bare minimum to keep their customers happy, but linux is built partly by its users.  If it strays away from what the users want, then it can just be forked and will be made for the users again.  Linux is not owned by a company, and there is no real bias in the software (compare to Vista's DRM layer, which shows a bias towards big business).  

Also, I remember back when I was using windows 95, I hated it when software would need to be upgraded.  It meant that my computer would run slower with no real advantage being added.  With open source, I have found that it has been the opposite.  I get excited about upgraded because most of the new features are worthwhile and usually it does not mean a big performance hit (often new software will be faster!)


BTW, you may consider reading "Just for Fun" by Linus Torvalds for you English class.  Good luck with your project!

### Post by zoetrope666 on 2007-04-05
1. Why Linux? 

As a student, I found the constant need to upgrade my computer and software a hugely expensive task, and felt disillusioned by the fact that each time I upgraded the new operating system didn't really seem that different from the old one. Even when there was a big difference made between the systems (for instance, 98SE and XP are very different on a lot of fronts, particularly their aesthetics) I felt annoyed that many of the core problems recurred (crashes, freezing up, viruses, spyware, defragmentation, etc.).
