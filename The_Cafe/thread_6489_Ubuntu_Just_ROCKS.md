### Post by jdodson on 2004-11-29
[QUOTE=bukdor]Now if only there was Ubuntu for servers! :D[/QUOTE]

your wish has been granted:)  you can use the regular ubuntu cd to install a "server".  you can remove the xfree packages(if you wish) and install apache/php/mysq/postgres/ssh/sftp/ftp/samba/etc/etc/etc all via apt-get.  you can also configure them all via the console.  i personally have setup apache2/php/mysql/ftp/dhcpd on my desktop/server box and it works great.

remember that the line between "desktop" and "server" is blurred in gnu/linux.  the only difference between the two is package selections really and firewall rules, etc.
