### Post by motin on 2008-01-02
You know what you can do to spread the word? Inform the maintainers of the [http://hardware4linux.info/download/](http://hardware4linux.info/download/) page that the MEPIS .deb works on Ubuntu as well:

1. Download the MEPIS package to your desktop
2. Double-click and install
3. Run "sudo hwreport /tmp/report" in a terminal
4. Upload the file "/tmp/report.tar.bz2" to [http://hardware4linux.info/upload/](http://hardware4linux.info/upload/)

Maybe you also should request that this package is included in universe for Hardy?
