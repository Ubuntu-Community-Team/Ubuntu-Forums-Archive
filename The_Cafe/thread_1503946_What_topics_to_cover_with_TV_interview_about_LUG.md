### Post by samalex on 2010-06-07
Hi,

Our local LUG has all but dried up, so we're working on a 'reboot' of sorts to try and relaunch the group as more of an Open Source Users Group instead of focusing exclusively on Linux.  In doing this I hope to get some media attention by possibly going on one or more local morning news programs, but I wanted to know what you guys suggested we talk about since the time frame is generally only 3-5 minutes.

I don't want to get too technical or get into software bashing, but I do want to cover the benefits to home users and small businesses to show how they can use Open Source software just as they use Closed Source software but without the added costs.

My thought would be to give a quick overview of licensing saying "all software is licensed, but the differences is what freedoms or restrictions are tied to those licenses" and segway into restrictions that go with Microsoft, Apple, Adobe, etc where you can only install it one one system, can't share with friends, costly, and so forth.  After this I'll go into Open Source software and how the cost is generally zero and you can do whatever you want with it.

I'm just not sure how to address the literal open source vs closed source aspect of it where closed source code is under lock and key and open source is there for anyone to modify. I thought about using the car analogy with "you wouldn't buy a car with the hood welded shut" which I think most people can relate to, but is there any other?  

Thanks for any insight.  If we do get the group kicked back off I hope to work on getting some airtime going later this summer or early fall, but we'll see how it pans out.  

I also want to have TONS of material and case studies on Closed Source vs Open Source on our website so we can redirect people to it.  In this regard we can post more technical info, but does anyone have suggestions on good case studies that aren't funded by Microsoft or FSF?  I want something unbias.

Take care --

Sam

### Post by LowSky on 2010-06-07
You should look to talk about software that runs on any type system, for example OpenOffice, VLC, and Pidgin. This way people can try it on their Windows/Mac machine.

I wouldn't bring up the word Linux at all, but maybe briefly bring up Ubuntu being a free community supported Operating system as a great replacement to older machines where Windows doesn't work so well, and how its safer form viruses.
