i find moka with faba is pretty tasteful. found it recently. definitely alot better than gnome default

for the newbies:
you don't actually need to install using sudo (and installing from new sources with sudo should be avoided as much as possible)
you can just download the .zip from github, unzip it, and just move the folder named 'Moka' to /usr/share/icons and it will work
then you do the same as above with 'Faba'
faba links to moka, so you select faba as your icons and it will use both together, as long as they're both installed

both are available here:
[https://github.com/moka-project/moka-icon-theme](https://github.com/moka-project/moka-icon-theme)
[https://github.com/moka-project/faba-icon-theme](https://github.com/moka-project/faba-icon-theme)
