### Post by Bicx on 2009-06-02
Hey everyone,

I just wanted to make an early plug for my proposed target shooting web application which I currently call [RangeTracker]("http://launchpad.net/rangetracker/"). 

From the Launchpad summary:

"RangeTracker is a community-based web application which brings together target shooters from around the world in a friendly competition of best shots and tightest groups. Using a specialized algorithm, RangeTracker simply requires a user to submit a digital image of a paper target in order to calculate meaningful bullet group statistics. These statistics can be compared to other targets, allowing a shooter to monitor personal skill level or join in a competitive scoreboard of 2 or more other shooters."

Although the project is still in the conception stage, I would like to give people the opportunity to get in on the early design and development phase. I am interested in recruiting experienced and inexperienced developers alike. Because this is my first OSS project hosted for community development, a lot of the process will be played by ear (as far as team planning, repositories, ect...). 

If you are interested, please contact me via the [Launchpad page]("http://launchpad.net/rangetracker/"). I look forward to any feedback, and I will continue to seek support as the project grows.
