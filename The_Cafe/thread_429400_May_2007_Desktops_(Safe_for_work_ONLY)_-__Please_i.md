Under any window manager, run the command icewm --replace metacity

You can replace icewm and metacity with any window manager of course.
You have to know I didn't make this theme, I only ported it: [http://gnome-look.org/content/show.php/Kore+Suite?content=54701](http://gnome-look.org/content/show.php/Kore+Suite?content=54701)

draw_shades yes

draw_outline no

# Draw borders around text
draw_borders no

# Stippled borders?

# border margins

# border width
border_width 1

# Default colors and also border colors

default_shade_color black

# Text alignment, other possible values are commented
#alignment top_left
#alignment top_right

gap_y 12

uppercase no

# number of cpu samples to average
# set to 1 to disable averaging
cpu_avg_samples 2

# number of net samples to average
# set to 1 to disable averaging
net_avg_samples 2

### Post by bonzodog on 2007-05-13
> **Opeth115 said:**
> please post you wallpaper i like it a lot! and also maybe ur conky.rc?


The wallpaper is called Zen Mist, it's part of the Zenwalk Wallpapers for Zenwalk Linux - can be found here -
[http://www.zenwalk.org/modules/myalbum/](http://www.zenwalk.org/modules/myalbum/)

As for the conkyrc, it's based on one by K.Mandla, with some adjustments.

```

double_buffer yes
update_interval 3.0
background yes

own_window no

use_xft yes
override_utf8_locale no
xftfont Dejavu Sans:size=9
xftalpha 0.8
draw_shades no
draw_outline no
draw_borders no
uppercase no
use_spacer no

border_margin 9
border_width 0

default_color white
default_outline_color black

alignment bottom_left
gap_x 9
gap_y 9

TEXT
${font Zekton:bold:size=26}${time %H:%M}${font}
${font Zekton:italic:size=8}${exec ddate}${font}
${stippled_hr}
${font Zekton:size=9}Down: ${downspeed eth0}Kb/s | Up ${upspeed eth0}Kb/s | Cpu ${cpu}% | Ram ${memperc}%

```

### Post by crimesaucer on 2007-05-22
Well, I turned my desktop icons off, and then made a setting in my .conkyrc to not use a separate window, because I always had a problem during the start up of my system when the Beryl started and the Beryl splash screen would show, that and my panel loading and my Conky would always make my desktop wallpaper not show, and my desktop icons would be doubled up on top of each other.

So fixing this part:

```
# Create own window instead of using desktop (required in nautilus)
own_window no
own_window_transparent yes
#own_window_type normal
own_window_type override
own_window_hints undecorated,below,sticky,skip_taskbar,skip_pager
```

...fixed my Beryl and Conky starting so my Desktop wallpaper loads correctly, but then the icons would not show unless moused over, and since I don't use them anyway, I turned the setting to NONE in the Settings Manager-->--Desktop-->--Behavior.

Now my desktop works perfect, and I just put a Thunar Launcher on my panel that is all my icons were in the first place. I don't know how it works with docks.

### Post by kvonb on 2007-05-26
I've never liked KDE on Ubuntu, it lacks the "cohesiveness" of Gnome, things like that annoying wallet thing which if you don't use it you have to enter your password continuously, and the menu is just far too cluttered and confusing.

It looked very "childlike" to me, so I stayed away from it.

However, I think I found the secret to using K: the lower the resolution the better it looks! :).

I installed it on a spare machine for a laugh and actually found that it works really well with a little tweaking.  The computer is in a dark room so I went for a darker theme.

The things I like about it are:

1. Fonts.  The default font seems to make a smaller resolution much more workable, I can get so much more on a 1024x768 screen than Gnome at the same res, and the fonts are very clear and legible.

2. Desktop Effects.  KDE has built in effects that create drop shadows and transparency without all the problems that Beryl/Compiz _**can**_ give, it's simple, fast, and looks very nice, I especially like the icon "zoom on click" thing.

3. Handles on panels!  This is my one pet hate in Gnome, those damned "handles" on each end of a panel that you can't get rid of.  KDE has the option to turn the rotten things off, oh joy!  And yes I have tried all the messing about with gtk files and it does not work :(.

4. KBFX.  Yes I know it is very "Windows-ish" and can be annoying and unproductive but I think it looks good and it can be configured and modified very nicely.  If you remove 90% of the annoying KDE menu items and get rid of the descriptions (takes a long time but is worth it to reduce the visual GBH!), you are left with a slick, simple menu.

So have a look at these screenshots, the only thing left for me to do is find a nice "black" window border theme.

Regards, Kev :)
