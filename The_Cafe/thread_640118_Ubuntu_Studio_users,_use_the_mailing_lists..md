### Post by MetalMusicAddict on 2007-12-13
If you are a Ubuntu Studio user and wish to stay up-to-date with the latest goings-on please subscribe to the mailing lists.
[LIST]
[*][Ubuntu Studio-users]("http://lists.ubuntu.com/mailman/listinfo/Ubuntu-Studio-users")
[*][Ubuntu Studio-devel]("http://lists.ubuntu.com/mailman/listinfo/Ubuntu-Studio-devel")
[/LIST]
News will always hit here 1st and technical help is more common.

On a personal note, browsing, distilling info, and the ever-increasing off-topic chat here have made it hard to continue to be active on the forums. Along with the need to spend more time with the family.

So hard-core users please sign up for the mailing lists and [your voice will be easier heard]("http://www.youtube.com/watch?v=9tqxtEFAJEU").
