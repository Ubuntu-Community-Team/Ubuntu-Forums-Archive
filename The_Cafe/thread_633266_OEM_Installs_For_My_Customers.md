### Post by Vince4Amy on 2007-12-06
With my business I do a Linux install service, which also includes some essentials for people who are new to it.
 
I always set up their machines in the following configuration:
 
Kubuntu 7.04 
Upgrade to the latest Kernel
Install Any Required Drivers
Install Codecs 
Install WINE
Install msttcorefonts (You'll be shocked how many times people ask where's Arial, if I was to not install this).
Install Evolution
Install Firefox and Flash
 
The best part is installing this on a Brand New System build. Especially the ones which look really nice, you could make a nice system and a nice OS to go with it.
