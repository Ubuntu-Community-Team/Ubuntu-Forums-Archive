### Post by Lucradia on 2011-04-16
1. Ubuntu is GNOME-based. Many of the applications are from GNOME (IE: Epiphany,. etc.)

2. Lubuntu is an LXDE Spin of Ubuntu. It aims to be lightweight (even more so than Xubuntu, as LXDE has no Window Manager, but integrates well with openbox, etc.) Lubuntu uses mainly LXDE Apps, XFCE Apps with few requirements, etc.

3. Gobuntu is a spin that only uses free software, as such, has non-free repos all disabled (IE: You can never install flash-nonfree, non-free codecs such as windows codecs, etc.)

Please note that Pidgin has gnome requirements only because the devs require gconf for proxy info.
