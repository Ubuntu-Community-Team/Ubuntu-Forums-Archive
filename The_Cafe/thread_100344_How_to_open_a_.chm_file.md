### Post by Sublime Porte on 2009-05-24
xchm is a little bland.

The best two apps for chm files would have to be kchmviewer (a KDE app, but there is a no-kde version) and chmsee.

sudo apt-get install chmsee
or
sudo apt-get install kchmviewer-nokde
