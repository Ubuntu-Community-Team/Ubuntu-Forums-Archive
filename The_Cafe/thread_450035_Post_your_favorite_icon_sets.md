### Post by kop316 on 2007-05-21
For installing Buuf,


[http://www.deviantart.com/deviation/38433725/](http://www.deviantart.com/deviation/38433725/)

[http://www.deviantart.com/download/38433725/buuf_theme_tar.bz2](http://www.deviantart.com/download/38433725/buuf_theme_tar.bz2)


1. Download the .tar.bz2 from there.

2. Extract the file, and it wil have another tar file.

3. create a folder in your .icons folder called Buuf and extract the contents of the second tar file there.

4. Go to your themes, and it should be here.

Enjoy!
