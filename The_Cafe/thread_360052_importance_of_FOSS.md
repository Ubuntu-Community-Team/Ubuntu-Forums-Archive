### Post by Somenoob on 2007-02-12
Note: abbreviation = "Free/Open Source Software"

I just bought a set-top box/digital video recorder device for my television and it obviously has a lot of semantic errors. The software is configured by default and searches for channels automatically the moment it's activated, so the average user probably would not complain. I wanted to make some changes, it's software is stored in a hard drive that's plugged behind it(unusual). Everything is in a executable form, they could easily have the source code in a separated file. If they would have made it available I would not be dealing with these bugs right now. for example: The record schedule doesn't work, the channel categories are out of place, and the GUI design is ugly.

As a programming hobbyist I could have benefited from such an experience. Now, I know how RMS felt when he had to fix the printer.
