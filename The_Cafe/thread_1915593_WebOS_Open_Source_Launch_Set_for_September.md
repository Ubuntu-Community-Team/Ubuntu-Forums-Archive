### Post by Dry Lips on 2012-02-15
update:

> **webOS governance model announced, more open than Android**
HP has published a [blog post]("http://developer.palm.com/blog/2012/02/sams-blog-open-webos-governance-model/")  that describes the Open webOS governance model. The project will adopt a  similar approach to that of the Apache Software Foundation. Code will  be developed in public repositories and key community contributors will  be able to earn commit privileges. The standard of inclusiveness and  transparency set by this governance model will make the Open webOS  project more open than Google's Android open source project (AOSP).
[http://arstechnica.com/business/news/2012/02/webos-governance-model-announced-more-open-than-android.ars](http://arstechnica.com/business/news/2012/02/webos-governance-model-announced-more-open-than-android.ars)
