About the Rescue-Remix:
Ubuntu Rescue Remix is a GNU/Linux live cd that provides the data recovery expert with a command-line interface environment equipped
with the best free-libre, open source data recovery and forensics tools available.

Since Ubuntu uses many filesystem and hardware detection libraries in its live cd, it makes sense to make a remix of the live cd for data recovery use.
