### Post by rcarring on 2006-06-02
I was going to start a little Vista thread of my own, but this is as good as any. Note: I am not running Vista, so most of this is based on what I have read.

1) The aero interface is very similar to the interface called Energy Bliss on Tablet PC. 

2) The minimum system requirements of 512mb ram, 15GB disk space free and 128mb ram recommended for aero experience effectively mean that machines older than say a year aren't going to be able to run Vista in any flavor without at least upgrading the memory. Laptops can't have their video card upgraded, so far as I know -- my Toshiba Satellite that dates from November 2002 has a mere 16mb dedicated video ram on an ATI radeon card -- playing City Of Heroes was a challenge.

3) Before I read about the system requirements, I was under the idea that if a machine could run XP SP2 then it could run Vista, seems I was wrong.

4) By specifying somewhat higher system requirements, I wonder if its Microsoft's intention to a) make people upgrade or buy new b) if they can't run Vista, they can't be supported so the number of people using tech support drops dramatically.

5) As with all prior releases, after a while, support stops for the old version. I expect within five years maybe less, there will be thousands of ex-corporate pcs that cannot legally run the os they came with and are too weedy to run Vista or Vista plus One.

6) At which point, Linux is the best and obvious option as an operating system. I would like to see Ubuntu marketed like this as well -- got an old pc, can't run won't run Vista then try the true roasted flavor of Ubuntu.....
