### Post by ugm6hr on 2009-07-19
I'm not really a micro-blogger, but do use it to follow some website updates, since Gwibber is more convenient than using RSS feeds.

However, I have thought it would be good to set up a private (i.e. not publicly visible) multi-way discussion between work colleagues.

Obviously, micro-blog sites are generally publicly visible, so any posts would be seen be anyone looking at contributors profiles.

Twitter allows protection, but is closed source.

Jaiku is now open source, but Gwibber support is a bit rubbish.  The new API system doesn't seem to work properly, and the latest 0.92 doesn't offer Jaiku at all.

Identi.ca doesn't offer protected posting.

Have to say, of all the above 3, I like Jaiku best.  But Gwibber is my client of choice, so I may as well get used to Twitter now :(
