### Post by HappyFeet on 2009-06-18
> **doas777 said:**
> some of the first responders to bugs are a bit confrontational from what i see on launchpad, so I've always been reluctant to raise my hand.

If you are "professional" about reporting bugs, aka, go about it in the right manner, any first responder will not be confrontational. I've never had anyone be confrontational with me. Just make sure the bug is repeatable and you provide as much information as possible.

For example, if application X gives you a problem, just start the application via the terminal, and if it happens again, you can copy and paste what was shown in the terminal. 

Doing:
```
dmesg
```
can also give clues as to what is happening.

I've reported a couple bugs that had no error messages or anything amiss in the logs. But I went to launchpad and reported it anyway. Guess what? Other people saw my post and commented that they too had the same problem, and it was fixed soon after. YOU *can* make a difference. Just be very professional and to the point. No emotions are needed in bug reporting. If I can do it, anyone can.
