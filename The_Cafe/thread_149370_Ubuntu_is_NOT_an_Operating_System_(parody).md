### Post by maruchan on 2006-03-23
Lately I've heard a lot of people around here calling Ubuntu an "operating system." I know what Ubuntu is, and it is NOT this!!! Don't tell new users Ubuntu is an operating system, or it will lead to birth defects in their offspring, car accidents involving cement trucks, and having a useable, hassle-free desktop experience, among other things. We need people to earn their desktop experience by being doused in the face with the HARD TECHNICAL DETAILS about ten times a day.

What Ubuntu IS:

Ubuntu is an African word that holds a together a collection of logical bits like a kernel, display drivers, and some games. And a window manager. That's ALL it is. It is NOT an operating system. Plus there are bits that float on top of other bits, like the way you can have a window manager float on top of your default display subsystem. But everything is free to move about a bit, e.g. a file browser that is the default from any other desktop environment can be used in a separate desktop environment if you like its features.

Plus, if you use GNOME, which is the default desktop environment in Ubuntu, you can have integrated Beagle search, which, even though the KDE people claim they can use too, it isn't the same. 

Also, if you want you can run Ubuntu in a virtual machine, which is like...a machine, but it's not real so we say it's virtual. And Ubuntu is free, but only free as in either 1)Beer or 2)Freedom. Other kinds of free, including 3)LCD Sports Wristwatch With Mail-in Offer and 4)Toothpaste Sample With Two Proofs of Purchase do NOT apply to Ubuntu. 

Anyway, I could go on forever, but if you explain it to people THAT way, believe me, you can go home feeling better about yourself at the end of the day. THAT is what Ubuntu is all about: Telling people the truth.

(We don't want to hide or hold back credit from where it is due, either, so it can't hurt if you mention GNU, RMS, Canonical LTD, or Debian in there.)

Trust me, all of this is much better than calling Ubuntu an "operating system" and having people get all confused about it.
