### Post by jhonan on 2011-08-04
While analogies might be useful in everyday speech, overuse can get tedious - a bit like eating the same flavour crisps all the time.

The funny thing about using analogies in a technical forum is that you are comparing a concept that we all understand to a very similar concept that we also all understand. Rendering the analogy useless. Like a teacher explaining a user interface to a group of students, and as an analogy he talks about how a washing machine works.

But my favorite analogy abuse is when the analogy is more complex than the concept it's trying to explain. Imagine teaching someone to drive and comparing the controls and process to these of flying a plane.. except it's in two dimensions and not three... and there are less controls to worry about.

I'm going to 'post' this now. Which is a bit like posting a letter, but this one is going to a group of people. Like posting a letter to a newspaper for publication. But the editors can decide to delete it, or move it to another part of the newspaper.
