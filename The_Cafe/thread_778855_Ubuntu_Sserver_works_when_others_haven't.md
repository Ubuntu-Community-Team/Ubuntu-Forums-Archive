### Post by Dragonbite on 2008-05-02
After trying to set up a home server with openSUSE (10.3) and CentOS (4.3) I have to say the first really successful one is the current installation of Ubuntu Server 8.04 (Hardy Heron) LTS!  

Great job, guys!

I have to admit, I am fairly new to networking and when I was dropped into a CLI after the installation I had no idea what to do at that point.  I'm not afraid of the command line, I just didn't know what to do!

Then I ran across [[COLOR="Blue"]HowToForge.com[/COLOR]]("http://www.howtoforge.com/") and a [[COLOR="Blue"]HowTos[/COLOR]]("http://www.howtoforge.com/howtos") on [[COLOR="Blue"]Simple Home File Server (Based On Ubuntu)[/COLOR]]("http://www.howtoforge.com/ubuntu-home-fileserver")! Perfect!

Although I don't agree with everything they said to do (like setting up the root account), it's clear enough that I could follow what was going on and edit as necessary.

In one night I set up the Samba server and it was visible by all of the computer on the network! Even better, it was stable when previous attempts either was not fully visible or was available one moment and not available the next!

This morning, before getting ready for work, I was able to use another HowTo ([[COLOR="Blue"]The Perfect Server - Ubuntu Hardy Heron (Ubuntu 8.04 LTS Server) - Page 3[/COLOR]]("http://www.howtoforge.com/perfect-server-ubuntu8.04-lts-p3")) and set up the LAMP server and tested that PHP worked!
