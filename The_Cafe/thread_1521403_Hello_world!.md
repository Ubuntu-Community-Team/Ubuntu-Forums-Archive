### Post by TimTimTimma on 2010-06-30
Hello everyone,

I dunno if this is the correct polace to post this but; I am a new user to ubuntu 10.04. I have occasionally tinkered around with ubuntu in the past (last time I installed ubuntu I had like 5.0) and boy has it come a LONG WAY! 

A little story about how I got here real quick because I think it is worth telling. I got in to work at 4am this past Saturday was talking with a buddy of mine at work about Apple and we got onto the topic of Mac OS X Snow Leopard and he mentioned that Hackintosh was a reality now and that you could install Max on your PC. 

Overjoyed later that day I went to partition off my 300gig HDD on my laptop using Windows 7 (which is a pretty decent OS) and it FROZE! I had to hard boot the computer and when it came back up dos said it could not find any operating system. (I did not get a chance to get any backups either!!)

Trying to salvage the situation I went ahead and tried to install iAtkos v1.0i. After a successful install it would not boot because darwin sucks hard. Knowing I need some sort of OS on my machine I tried to install WinXP but because my system is 64bit that did not work.

Next I tried a distro of SuSe I got back in 05. That did not work.

Tried Ubuntu 7.10 and X Server would not start during installation. 

FINALLY I live booted using DreamLinux and found the latest release of ubuntu and downloaded and burned it to a dvd on my wifes computer and installed it.

I must say I am VERY pleased with how it looks, feels and most importantly WORKS! I though for sure getting all my hardware to work with ubuntu was going to be nightmarish but everything works! Even my hp officejet 6500 networked scanner/faxer/printer! Hell the feeder in the damn thing works too and thats WITHOUT the factory disc that comes with it!

I have found it very easy to install various programs I need or want and even better, FOR FREE! I am beginning to wonder if anything costs money with ubuntu!

I've played around with compiz and am enjoying some very fun and useful visual effects. I've even used the terminal to download, install and remove various programs. I have also set up my ubuntu one account and can't wait to take advantage of 2gigs of online storage!

I am still learning a lot about ubuntu and linux still but I can honestly say that this lemon of a situation has been turned into delicious lemonade!
