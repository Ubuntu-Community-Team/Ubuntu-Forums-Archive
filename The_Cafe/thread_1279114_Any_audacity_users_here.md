### Post by dummy910 on 2009-10-01
[http://forum.audacityteam.org/viewtopic.php?f=28&t=7577#p30003](http://forum.audacityteam.org/viewtopic.php?f=28&t=7577#p30003)
> To lower the pitch gradually you need to use Audacity 1.3.x and add a time track. You can then use the envelope tool in the time track to adjust the speed ("Mix and Render" the track to make the pitch change permanent, then remove the time track so that it does not affect other tracks).

Note that this will change the playback speed, so **it adjusts both pitch and tempo - it's like slowing down or speeding up a record.**Check Google too:
