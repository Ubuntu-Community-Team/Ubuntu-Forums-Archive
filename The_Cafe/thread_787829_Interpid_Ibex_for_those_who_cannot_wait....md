### Post by ibuclaw on 2008-05-09
For those who are die-hard Ubuntu users.
You'll be glad to know that Intrepid is here!

You can get it using this handy little hack...

Hit **Alt+F2**
```
gksu gedit /etc/apt/sources.list
```
Now type in **Ctrl+H** (or your text editors' command of choice for replace... ie: Nano is "Ctrl+W, Ctrl+R")
And type in:
```
hardy
```
In the "Search For" field, and:
```
intrepid
```
In the replace field.

Continue to replace all matches, save the changes and quit.

You are now upgraded!
One last thing to do:
```

sudo apt-get update
sudo apt-get dist-upgrade

```
