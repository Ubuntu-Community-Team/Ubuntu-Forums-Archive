### Post by ksennin on 2009-11-25
I had been debating over what OS to use on a couple of old machines for simple family use at home, so that would not be too slow yet retain the many advantages of Ubuntu. So after asking a bit around in a previous thread [http://ubuntuforums.org/showthread.php?t=1232224](http://ubuntuforums.org/showthread.php?t=1232224) , I decided to do some actual comparisons on the machines themselves using every available ubuntu-related release I had at hand, and check CPU and RAM usage .

Some limitations:  
-No full installs, only live cd running.  Actual performance may improve after full install, so values are not meant to be system recommendations, yet the relative performance comparison should be valid.
-The effect of running the different included applications is not considered, as you can install and run lighter applications in the heavier distros.  The systems are allowed to sit idle, with just the system monitor open.
-When tested, Crunchbang results were taken from the on-screen conky readouts, while in others, the gnome monitor was opened and used.  This may distort comparison.

The machine used on the first batch of tests is a Dell GX150 desktop, Pentium3-933Mhz with 384mb of ram, integrated video card.

UBUNTU  7.10:      6% CPU 134.7MB/374.8MB + 0 swap
XUBUNTU 7.10:      5% CPU  99.0MB/374.8MB + 0 swap

UBUNTU  8.04:     28% CPU 162.4MB/374.6MB + 0 swap
XUBUNTU 8.04:     32% CPU 134.0MB/374.6MB + 0 swap
CRUNCHBANG 8.04:   4% CPU 103.1MB/374.5MB + 0 swap
UBUNTULITE 8.04:  40% CPU 105.0MB/374.5MB + 0 swap   

UBUNTU  8.10:     42% CPU 183.8MB/373.4MB + 12.7MB swap
XUBUNTU 8.10:     43% CPU 177.2MB/373.4MB +  7.7MB swap
CRUNCHBANG 8.10:   7% CPU 108.1MB/373.4MB +  0 swap

XUBUNTU 9.04:     40% CPU 164.6MB/369.2MB +  0 swap

UBUNTU 9.10:      35% CPU 167.8MB/369.1MB +  9.6 swap

Comments and more results to come as testing continues, on this, and in the second test pc.

Differences in CPU usage are particularly surprising so far.
