> **ubunterooster said:**
> I seriously don't think many will want it as setting up each command does not seem to simplify things IMO, but add more steps.

The idea behind CLI Companion is to provide a tool with a decent default command list. This will help people not familiar with the Terminal get used to what it can do. For those who are experienced with the Terminal you can store commands that you come across but do not use all the time, or often enough to have memorized. The idea is not to store *every* command you run. It is a terminal with a 'command dictionary' attached. Before making CLI Compnaion I used a Tomboy note for this. The goal was to create something that put those commands somewhere that they would be easier to get to. Maybe some have a better memory then me, but i can not remember every single useful command I have ever come across. :)
