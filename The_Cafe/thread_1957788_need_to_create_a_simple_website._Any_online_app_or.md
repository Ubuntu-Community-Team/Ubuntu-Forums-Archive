### Post by lovinglinux on 2012-04-13
Wordpress.com doesn't allow to install plugins and as far as I remember, doesn't allow to install themes either. So, I would recommend getting a domain, a web hosting and installing Wordpress on it.

If you have no experience with this, look for a hosting plan that offers cPanel and Fantastico. The last one is a tool that allows you to easily install several applications, like Wordpress, without configuring anything. You can have a default web site running in less than a minute.

Most hosting plans also include domain registration btw.
