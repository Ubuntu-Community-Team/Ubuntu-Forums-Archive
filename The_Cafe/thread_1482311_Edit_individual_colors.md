### Post by Merk42 on 2010-05-13
First find whatever icon you want to edit, probably in /usr/share/icons

if it's an .svg you need a vector editor like Inkscape (available in the repositories)
if it's a .png you need something like GIMP (also available in the repositories if you don't already have it)

### Post by arsenic23 on 2010-05-13
The best thing to do would be to copy whatever icon them you are using and edit the individual icons, then rename it something slightly different and install that and use it.
