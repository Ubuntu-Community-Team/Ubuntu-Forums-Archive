### Post by Interestedinthepenguin on 2007-11-02
I've created another Stylish userstyle for the Ubuntu forums.  Black, this time.  
[http://userstyles.org/styles/3992](http://userstyles.org/styles/3992)

The code was too long, so, I had to upload it as two, separate files.  Just load the first piece in Stylish, remove the brace at the end, follow the link to the second piece, show the code, copy it (leaving out the first two lines), and paste it at the end of the first code-piece.

(Also can be used in Opera, and Greasemonkey.)

Enjoy!

Feedback is welcome, and let me know if you find something I missed.
