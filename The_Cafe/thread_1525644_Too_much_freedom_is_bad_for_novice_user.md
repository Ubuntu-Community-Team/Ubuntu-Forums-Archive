### Post by Johnsie on 2010-07-07
I've been experimenting on a computer at work used by novices and so far I haven't had any problems like that. The advantage has been that I haven't had to worry much about spyware.


I think this experiment has only been successful because the main thing they use is a web browser. If they started  trying to do other things then I might run into a problem. 

Evolution Problems I did have:

-Teaching them Evolution as an Alternative to Outlook was a little bit tricky. Evolution is ugly and not very user friendly. IMO doesn't interface well with Ubuntu.

-The email alerts are not as good as on Outlook

-Evolution does not support Exchange very well compared to Outlook. 

-The user directory works but doesn't display a list of all staff

-Emails don't appear instantly the way they do in outlook. (This might be adjustable)
