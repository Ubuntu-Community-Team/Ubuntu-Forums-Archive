### Post by Legendary_Bibo on 2010-06-11
I can see that they added only software for kids, but the color scheme just isn't kid friendly. Kids aren't going to know how to change the panel bar color, or anything like that. They need to make a very colorful, shiny, fun looking environment. The only effect from compiz that's needed is  Wobbly Windows, and we can make the desktop a bright green, Red top Panel Bar, Blue bottom panel bar, yellow windows borders or something. Make it look goopy. Change the icons. Remove the network bars (I don't know if it shows up with a wired connection). Remove the evolution thing. Don't have the cube enabled but have the 3D effects when choosing different workspaces, probably use the sphere. Kids like bright colors, so that's why I'm going on about the aesthetics. I will also speak blasphemy, but make more launchers so that kids can fix their own computer problems if they come across it so they don't have to learn the terminal just yet. Which is what I'm working on. We could use "fun" looking texts all around.

My first experience with computers was Win 95 and it was bland. Give kids a fun experience only achievable by Linux, and you catch their interest. As my programming teacher used to tell us; "Keep it simple, stupid". Everything is simplified and organized, but it could use just a few tweaks to work with younger kids.
