### Post by Nostrafus on 2008-08-03
So I have a dual boot system, I've got my Windows partition for gaming, and I've got my Linux partition for everything else.

I've been off and on using different distro's of Linux for the past 10 years, and every time I make the switch back to windows because I would get frustrated with a lack of "stuff to do" I don't mean like playing solitare, or same-gnome, those are always fun.

But before OpenOffice.org came along there wasn't a well built, easy to operate document creator.  Before GIMP there wasn't anything to do the higher end of graphics editing, things like that.

So I'm sitting here doing some writing, and my brother tells me that a new release of a game I'm beta testing is available, and I've gotta download the newest core.  I finish up what I'm doing and swap over to my windows partition.  I start the download, it says "19 hours", well it is 9 1/2 gigs.

Then I stared at my monitor for about 5 minutes and I had the thought... "Now what do I do?"  I seriously could not thing of anything to do with Windows since the game I'm currently playing wasn't able to be played.

Anyone else had this weird realization?
