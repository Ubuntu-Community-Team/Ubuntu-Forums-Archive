### Post by Bachstelze on 2010-09-03
> **Khakilang said:**
> Is it the kernel that what make unix Unix? And the rest are desktop management, GUIs, file management, window management etc..?

It is a large number of things, but GUIs and file managment are not among them. Basically, it's 1) an API (that is, a set of functions that the OS must make available to programmers) and 2) a set of utilities (i.e. programs) the OS must make available to users. All of them must respect strict rules for the OS to be labeled as UNIX.

If you're interested, the whole thing is here: [http://www.opengroup.org/onlinepubs/9699919799/](http://www.opengroup.org/onlinepubs/9699919799/)
