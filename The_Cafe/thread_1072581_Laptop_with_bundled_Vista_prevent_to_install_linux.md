Find out exactly what is on your 3 partitions. 


What it probably has on them:

A.) A Recovery partition that is NEEDED to work with your DVD/CD's that you have to burn yourself.
B.) The main Vista with all of the important files needed to run. 
C.) A data partition that is only for personal data. (not sure about the boot flag though)

(B and C could be done in a weird way. I say this because of the boot flag on the data partition of C)


Leave A.) the /dev/sda1 "Recovery" partition alone.


You probably can resize the main Vista partition to a size that is just enough for everything that is needed for Vista to run, and a few GB's of download space for new programs and other stuff.


The same can be said for the Data partition (since it has the boot flag I would leave it alone.... just resize it down).


To install Linux, make a 4th Extended partition using the Ubutnu install CD using the freespace that you made available by shrinking down your 2 Vista partitions. Install Linux as an Extended/Logical partition.


Use the Vista Partitioner to do this: [http://apcmag.com/how_to_dualboot_vista_with_linux_vista_installed_first.htm?page=2](http://apcmag.com/how_to_dualboot_vista_with_linux_vista_installed_first.htm?page=2)


Also defragment Vista afterwards.
