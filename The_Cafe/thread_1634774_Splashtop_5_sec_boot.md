> **Gremlinzzz said:**

Still I haven't found any. The only two downsides I've noticed up till now are these:
- It needs Windows to run (its files are stored in a Windows partition, more or less like Wubi I guess)
- It can't save data on the HD if I don't go wrong. While trying to save the latest Catalyst drivers and the latest AVG for my fersh install of W7, it asked me to add an USB drive to write to.

Other than that, I don't plan to use it very often but just in case of emergency/quick check of my mail, so it's ok. Of course don't expect any complete OS when you're told about Splashtop. I agree it's kind of very basic ;)
