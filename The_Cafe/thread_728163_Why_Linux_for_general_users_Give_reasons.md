### Post by justin whitaker on 2008-03-18
> **NightwishFan said:**
> Photoshop is not good. The cs3 is like $1000 isn't it? 90% of the photoshop owners pirate it. :)

Yes, but like Windows, the piracy contributes to their ubiquity. 

I'm with aysiu on this: there should be no evangelism about Linux. If someone wants to learn, teach them. If they don't, then let them be. 

You know, I never really liked the "use Linux because it's free" argument. I think that it is a non-starter for most people, like the person the OP talked to.

There is no cost in using Windows. All PCs come with some version of Windows on it, and if they do not, there are plenty of hacked versions of XP and Vista out there to download.

And as for free as in the source code....the ratio of desktop users to source code users is quite large. In fact, I would hazard a guess (and I don't know where to find this sort of stat) that most FLOSS users are just that, users, not developers.

I think the argument for Linux, as a whole, is that it is just a better than the competition.

[LIST][*]Linux can be made to run on just about everything, and with as little or as much, software as you want to use. 
[*]You can tailor your system to run as efficiently as possible on your hardware. 
[*]You can change and configure the desktop interface  to match your work style, instead of having an interface forced upon you.
[*]You can decide when and how you update, not when the central server says to. 
[*]You are not bound by upgrade cycles. If you don't like what how the version of Linux you are using is developing, you can always choose another. No one is going to force you to upgrade. 
[*]User and admin capabilities are separated out, which allows you to use the computer without fearing that some virus or accidental crash is going to screw something up. 
[*]Patches are submitted more quickly than on closed source systems.
[*]FLOSS is completely legal. You can use it however you want, so long as you comply with the GPL. There is no hidden agenda, you aren't renting the software, your data is your own. [/LIST]

Those are the sorts of things I think are better selling points than "OMGZORZ it's Free!"
