### Post by phillychease on 2009-12-31
> **sudoer541 said:**
> it looks nice! how did you make the reflection?
 
get the pic/text you wanna reflect
duplicate layer
Rasterize Type or Rasterize Layer if its a text layer
vertical flip
Rectangular Marquee Tool and set to feather 10px
highlight the part you wanna delete
and set to 50% opacity!
