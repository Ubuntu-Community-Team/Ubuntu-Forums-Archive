### Post by themusicalduck on 2010-05-27
If everything is working as it should (it sounds like it is if fire effects and cube effects are all working) and there are none listed in Hardware Drivers then you don't need to install any drivers.

Almost all of the drivers Ubuntu uses are included in the kernel already.

Not sure what the best place to look for extra Compiz documentation is, but I found the Compiz wiki which might be helpful - [http://wiki.compiz.org/](http://wiki.compiz.org/)

Also, you could use google to find out about any specific Compiz feature.
