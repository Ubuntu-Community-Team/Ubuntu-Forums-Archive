### Post by xXx 0wn3d xXx on 2006-10-26
Did anyone ever realize that it's in the Dapper + Edgy repositories ? I am using it on Edgy now and it works really well. If you want to install it, do so like this:
> sudo apt-get install smartpm smartpm-core

then run:
> sudo smart update
Useful Options: [sudo smart option}

> Action commands:
    update
    install
    reinstall
    upgrade
    remove
    check
    fix
    download
    clean

Query commands:
    search
    query
    info
    stats
