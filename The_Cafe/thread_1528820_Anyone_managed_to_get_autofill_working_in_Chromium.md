Think of Chrome as the closed source browser built off Google's Chromium project. Basically Chromium + official Google branding + Flash. Chromium is the open source chunk of Chrome, where all the development action happens. Every now and then there's a stable snapshot, which becomes the next Chrome release. So, if you have the daily build of Chromium, that's the bleeding edge.

There are now four Chromium PPAs being lovingly maintained on Launchpad, where the Stable and Beta repositories have releases from the same points as Chrome's stable and beta releases:
[https://edge.launchpad.net/~chromium-daily](https://edge.launchpad.net/~chromium-daily)
