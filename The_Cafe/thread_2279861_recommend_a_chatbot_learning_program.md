### Post by monkeybrain20122 on 2015-05-27
There is something called libbot-training-perl in the repo, this is the description

```
Markov bots like Hailo and AI::MegaHAL are fun. But to get them working
you either need to train them on existing training material or make your
own.

This module provides a pluggable way to install already existing training
files via the CPAN. It also comes with a command-line interface called
bot-training.
```
