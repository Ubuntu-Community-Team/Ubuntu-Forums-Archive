### Post by newbie2 on 2007-07-23
> Last month, GNOME developer Alex Graveley taunted us with a vague teaser screenshot and revealed a scant handful of details about the project he has secretly been working on with his friend and colleague, Mono developer Chris Toshok. Now the cat is finally out of the bag: the project is called Pyro, and it's a desktop environment built on top of Firefox.

Distributed as a Firefox add-on, Pyro renders the entire desktop as a web page and leverages window compositing functionality to display native Linux applications alongside dynamic HTML. Pyro aims to provide extreme desktop Internet integration by facilitating a unique fusion of conventional native Linux software and versatile modern web technologies like Javascript, CSS, and SVG.

In a blog entry, Chris explains how Pyro renders native Linux applications inside of an HTML environment:
