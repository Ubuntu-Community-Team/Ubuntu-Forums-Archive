### Post by brandenmikal on 2010-11-01
To go with the "Eerie Autumn" theme that Microsoft recently released (yes, it is still legal to do this. I already checked)

I compiled a working sound scheme. It's as easy as downloading the file, unzipping, and then copying the entire folder to: usr/share/sounds/

If you're new to this; look for the "sounds" folder in your File System (normally labeled as File System or sometimes the name of your drive such as C:/ or D:/ )

Enjoy everyone!
