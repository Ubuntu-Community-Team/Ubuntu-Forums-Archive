```

wget http://tkhtml.tcl.tk/hv3-linux-nightly-08_0203.gz
  gunzip hv3-linux-nightly-08_0203.gz

```
you have the file **hv3-linux-nightly-08_0203** in the directory **Hv3**, so you go to Hv3 and run ./hv3-linux-nightly-08_0203. you probably worked that out though :)
