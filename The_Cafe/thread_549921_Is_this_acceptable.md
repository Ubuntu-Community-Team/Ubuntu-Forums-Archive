### Post by pevans_om on 2007-09-13
In our country we have a major lack of bandwidth. So I am wanting to start a service to allow people to select from a bunch of free software, demos, patches, mods, drivers, opensource and linux distros. They can select whatever they want, which is then burnt onto a dvd and posted to them. They are charged for the DVD.

So, is this acceptable?
Would it be okay to charge for a dvd like this, since we're essentially charging for the service of downloading the files on behalf of the person?

Some community input would be appreciated.

### Post by Scruffynerf on 2007-09-13

### Post by Tomosaur on 2007-09-13
Yes, it's perfectly acceptable :)

However - you should make sure that each piece of software you're distributing allows this. If the software uses the GPL licence, then there should be no problems, but not all open-source software DOES use the GPL.
