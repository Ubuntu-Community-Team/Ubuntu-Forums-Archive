### Post by kvonb on 2006-11-16
m.mushashi:

Yes unfortunately you have to mess around with config files :(, however here are the instructions:

[http://wamrfixit.homeip.net:8000/modifications/modification-desktop-text-colour](http://wamrfixit.homeip.net:8000/modifications/modification-desktop-text-colour)

If you are using Beryl (the 3d guff), then it is trickier because it doesn't use the standard themes. 

Here's what I did:

Look in ~/.themes, there should be some folders in there, check each one for a "gtk-2.0" folder, in that there should be a "gtkrc" file.  That is the one you need to edit.

If you can't find one, install a theme (from gnomelook.org) that has (glossy-p, greenheart, and nuvola-light-blue-gray have them), then use that theme (you might have to quit Beryl first), edit the file per my link, and re-run Beryl.

I did a shifty, it seems that when in Beryl you can only change "controls" and "icons" because Beryl becomes your theme manager.  So I set my theme as "nuvola-light-blue-gray", then copied the gtkrc file from the glossy-p folder over the nuvola-light-blue-gray gtkrc file!

This gave me the nuvola-light-blue-gray controls (which I like), and also changed my text to white :).

Hope that made sense!

I also like using the ubuntu system font (beware though it doesn't show capitals!), you can install it through synaptic, search for ttf-ubuntu-title (it is installed default in Edgy).

The icons I like at the mo are called "glass" if you are interested.

With those panel backgrounds, I use "menubar-02.png" for the top menu, and "menubar-01" for the bottom (one is reversed) to give it a "shining from the middle" look.

All I need now is some real-time snow effects :D.

### Post by lapsey on 2006-11-27
> **BLTicklemonster said:**
> nicely assimilated desktop, I must say.






How did you get the transparency in icewm? and where might one obtain more  decent themes for icewm?

conky and gdeskcal use pseudo-transparency, where they take the desktop supplied by icewm-session and crop and move it within themselves. That's part of those apps. Eterm can do the same thing.

I mention icewm-session specifically because that provides icewm with things like the tray and the desktop image. GDM should launch that anyway.

The window manager theme is a blue variation that I made from [IceBuntu](http://themes.freshmeat.net/projects/icebuntu) : the file provided there also gives the GreenBuntu (ubuntu-lite) and IceHybrid (vista-like) themes, which are the *only* IceWM themes that look good. Save them to ~/.icewm/themes/

I have attached the contents of ~/.icewm/ and ~/.conkyrc . This includes the modified IceBuntu, ~/.icewm/themes/IceBuntuBlue



To get the GTK2 apps themed like this in icewm without anything like gnome, you will need
gtk2-engines-ubuntulooks tango-icon-theme

and a .gtkrc-2.0 file:
```

style "user-font"
{
font_name="Bitstream Vera Sans 10"
}
widget_class "*" style "user-font"

gtk-icon-theme-name = "Tango"
gtk-theme-name = "Human-Blue"
```
