### Post by Cavsfan on 2017-05-08
As a matter of fact Arch Linux has 2 versions of compiz. The 0.9 version which is newer and I'm using the 0.8 version which is actually better IMO.
It has a Fusion Icon that actually works. You can easily change window managers from Compiz to Xfwm4 or Metacity, reload window manager and access emerald theme manager.
Also, you can change window decoration from Emerald to GTK+ window decorator.
Here is what I have installed:
[LIST]
[*]ccsm 0.8.14-1 
[*]compiz-bcop 0.8.14-2 
[*]compiz-core 0.8.14-5 
[*]compiz-fusion-plugins-experimental 0.8.14-1 
[*]compiz-fusion-plugins-extra 0.8.14-3 
[*]compiz-fusion-plugins-main 0.8.14-1 
[*]compiz-gtk 0.8.14-5 
[*]compizconfig-python 0.8.14-1 
[*]fusion-icon 1:0.2.3-2 
[*]libcompizconfig 0.8.14-1 
[/LIST]
One of he nicest things I like about Arch Linux is that it never needs upgraded to a newer version, it's always pushing the cutting edge updates.

Since I've been using it, I've seen the kernel go from 4.8 to 4.10:
```
[cavsfan@Le-Beast ~]$ uname -r
4.10.13-1-ARCH
```

Dropbox link for full resolution viewing:

Some details showing in the first shot above...
i3-dmenu-desktop activated and with customized colours at the top left portion of the image.

i3blocks is installed and configured to generate the bottom bar and its display. i3blocks had to be compiled from source to work on Trusty, from 15.10 on it is available from the repositories as far as I'm aware.

The system tray on the right side of the external display's bottom bar is showing vlc, guake, alarmclock, network manager, onboard (on screen keyboard), artha and easystroke, running from start up. Compositing is performed by "compton" for applications like onboard which can be set up with a transparent background (not on show here but is set up in such a manner on this install).

The workspaces have fontawesome webfont icons attached to them for applications assigned to specific workspaces. Workspaces with only a number and a full colon symbol can have any other non assigned applications opened on them. The original config was set up for 10 workspaces, I added an additional 10 spaces for assigning applications to a specific workspace keeping the original 10 spaces as unassigned workspaces.

The background image is maintained at each start up using the application "nitrogen"; in the i3 config file the command "nitrogen --restore" re-enables the image at each start up. The image is a stitched together set of 6 photos I took while working at Camp Anderson in the Hamersley ranges region of northern Western Australia way back in 1993. I lived in that rail maintenance camp for 6 months back in 1993 working as an Electrical Fitter/Mechanic for the rail maintenance contractors.
