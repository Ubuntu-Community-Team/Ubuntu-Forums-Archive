### Post by Tomosaur on 2007-03-20
Closed is good for Macs because Macs are sold as a full 'experience' rather than a product. As soon as you combine Mac with anything else, the holes start to appear. Microsoft already tried to fight this case (that all PCs need to run Windows) and failed. It's a strange situation with tech. A combination of propietary and closed doesn't work very well. A 'whole' system (ie, all closed, or all open) will work wonders, but combinations of the two tend to have problems. A Linux system on completely open hardware (or at least hardware which has open drivers) will work amazingly well. It's when you throw in propietary rubbish that things start to get messy. The same is true for Windows  - but Windows has the odd exception that using ONLY MS products leads to problems, security, and performance wise.
