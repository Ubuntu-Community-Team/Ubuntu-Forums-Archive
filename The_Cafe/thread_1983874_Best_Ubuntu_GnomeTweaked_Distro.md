Arguably the best distro for gnome-shell is fedora because it is the native desktop and most of the gnome/gnome-shell designers and developers work for Red Hat.

However, I **much** prefer a debian based distro so I use ubuntu. Gnome-shell is trivial to install and reasonably well supported on 12.04 although some compatability issues (e.g. [bug 965921]("https://bugs.launchpad.net/ubuntu/+source/gnome-control-center/+bug/965921")) are quite annoying and show a clear disregard by ubuntu to maintain gnome-shell compatibility.

I installed debian testing upgraded to unstable yesterday to see what gnome-shell was like in the very latest debian but it was a poor result. Still running gnome-shell 3.2 in debian unstable and it lacked a lot of the aesthetic polish ubuntu provides.
