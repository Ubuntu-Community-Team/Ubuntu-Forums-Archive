Don't bother. You'll waste time and energy getting nowhere at all.

Personally (and these come only as personal recommendations) I like

**FFmpeg** - Take the time to learn this from the command line. Once you have converting videos and ripping audio can be done with very little effort

**Amarok** - KDE music player, version 2.2 git is looking good but it's no V1.4.10

**K3B** - for burning/ripping disks. So good a sizeable proportion of GNOME users use it

**Avidemux** - for converting videos if ffmpeg doesn't work

**wicd** - Network Manager

**digikam** - Photo database and small editor

**Kalzium** - The Periodic Table of Elements

**KMPlot** - To plot graphs, mainly functions

**DeVeDe** - DVD Author for putting avi files onto a disk for standalone players
