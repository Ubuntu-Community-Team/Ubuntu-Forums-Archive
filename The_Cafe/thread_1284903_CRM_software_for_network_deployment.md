### Post by Bufke on 2009-10-15
Have you looked at SugarCRM?  [http://www.sugarcrm.com/crm/](http://www.sugarcrm.com/crm/)  It's OSS and can run on a Ubuntu server.  It's completely web based, though does have some integration with phones, outlook, etc if you buy the commercial version.  Uses MySQL as a backend so it shouldn't be hard to import data and it is very customizable.
