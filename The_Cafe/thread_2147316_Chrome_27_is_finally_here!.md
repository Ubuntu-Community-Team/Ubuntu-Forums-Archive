### Post by Gavin77 on 2013-05-21
I had a problem with Chrome 27, flash videos would play with too fast audio making it sound high pitched.

If anyone else has this problem I fixed it by editing /etc/pulse/daemon.conf and changing

```
; default-sample-rate = 44100
```

to 

```
default-sample-rate = 48000
```

(note the removal of ; )

I read that Chrome 28 dev is supposed to include a fix for this but I tried it and was still broken.
