### Post by PrimoTurbo on 2008-06-13
I think the whole better for the environment attitude is flawed.

While yes Linux might not have packaging, but Linux users tend to download linux and burn it to a blank cd. All of this uses power. Also Linux users tend to try various distros repeating this processes. Furthermore there are repos which means there are tons of servers shelling out updates and software all of this needs electricity.

So in reality it could be that linux is less environmentally friendly then windows. But I doubt OS are the ones causing any real damage.
