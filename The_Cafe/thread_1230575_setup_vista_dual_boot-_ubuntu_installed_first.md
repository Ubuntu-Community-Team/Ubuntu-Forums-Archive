### Post by blur xc on 2009-08-03
Well, I followed these instructions [http://apcmag.com/how_to_dualboot_vista_with_linux_linux_is_already_installed.htm](http://apcmag.com/how_to_dualboot_vista_with_linux_linux_is_already_installed.htm) and after a couple of hours on my wonderful Saturday (too damn hot to do anything else anyway) I was happily dual booting away!

I chose the Grub boot loader option, and that was relatively painless.  The only sticky spot was getting vista to select the partition I made for it- that part of the instructions was a teensy bit vague, but I fiddled around and got it going.  Oh- and once it was running - my stupid usb keyboard gave me troubles.  It turns out I didn't have the "enable usb keyboard" option set in my bios...

The only extra thing I did was create an extra ext3 partition for my home folder which I still need to move over.  That sounds simple enough from the psychocats tutorial.

I was hoping a clean ubuntu install w/ vista in virtualbox would have fit my needs, but that has been troublesome.  That's a subject for another thread though...
