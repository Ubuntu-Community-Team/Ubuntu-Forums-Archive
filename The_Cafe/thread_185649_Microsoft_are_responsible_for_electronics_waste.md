Things to keep in mind:

M$ is very happy with forcing people to use newer windows releases, such as breaking backwards compatibility. Newer windows invariably requires stronger hardware.

Casual users treat computers the same way they treat, for example, a pair of shoes. They don't see that typical windows pc gets bloated with spyware and anti-spyware/anti-virus monitors. For them, computers just wear away. They think it's simply time to buy new one, because current machine gets less and less responsive.
