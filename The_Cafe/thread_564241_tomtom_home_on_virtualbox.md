### Post by tcdrewiv on 2008-01-09
I've used tom tom with virtualbox and vmware server using XP and 2000pro....
Its pretty simple.  
I prefer virtualbox over vmware server because your USB connection can be made after your virtual machine is running w/ virtualbox. Sometimes in the process of transfering data the device may need to be restarted and that can be a pain if you are using vmware.  If you use vmware server you have to connect prior to starting the vm or it will not see the USB device.

Make sure your virtual hard drive is large enough so you can back up the tom tom (you should have about 2GB of room just for tom tom) to the hard drive in order to install new maps.
