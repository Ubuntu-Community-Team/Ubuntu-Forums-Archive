### Post by AlbinoButt on 2007-08-12
I should preclude this by saying that **this is just my personal opinion**.

I considered getting a System76 laptop but there were a few things that put me off:
[LIST=1]
[*]The hardware was just rebranded (Asus hardware, if I recall correctly).
[*]The keyboard has a WIndows key(!). This is a small issue for some people, but the use of the Windows logo/key requires a royalty fee to Microsoft. It shows that this system isn't built from the ground up for Linux. (I should also say that Dell has a Windows key on their Linux laptops.)
[*]The price was very steep for the hardware that you get. I can understand that you're also paying for the fact that they provide Linux support, but I'd prefer to get a better value elsewhere and install Linux myself.
[/LIST]

The one thing that almost outweighs all that other stuff for me is the fact that you're supporting Linux. But the way I see it, I can support Linux by contributing to the community and donating to organizations that create FOSS instead.

### Post by dscherry on 2007-08-12
Hi,
We've had a Dell desktop that shipped with Ubuntu for a few weeks now.  I had no problem getting questions answered from Dell hardware support and absolutely excellent Dell-Ubuntu tech support from a group in Canada - no extra fees.

The installation on the hard disk works just fine.  Before we started heavy use, I wanted to see what I'd be facing if I had to reinstall it.  I was amazed that the Ubuntu Feisty CD that shipped with the pc couldn't recognize all of the hardware!  Dell-Ubuntu get's credit for having an immediate work-around, but Dell should still be able to ship a PC that is entirely compatible with the OS it ships with.  (This wasn't a major problem - the network driver has to be 'available' on the harddrive before booting with the CD - then modprobed for use - not a real biggy, just peculiar).

I'm happy enough with Dell and their support, but when it comes time to replace my laptop, I'll most likely try System76, but only if I can first determine if all of the hardware gets recognized by the standard OS and perhaps a couple of other liveCD's.

I know that doesn't help with choosing a laptop, but the subject of Dell-Ubuntu support came up, and I felt I'd comment on my own experience.
