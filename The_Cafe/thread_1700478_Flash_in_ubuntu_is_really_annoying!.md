### Post by Cracklepop on 2011-03-05
Using 32 bit Flash with 64 bit Ubuntu is known to be a bit iffy. 

Better to use 64 bit:
Download here: [http://labs.adobe.com/downloads/flashplayer10_square.html](http://labs.adobe.com/downloads/flashplayer10_square.html)
Extract the .so file from that, and put it in ~/.mozilla/plugins
Restart Firefox. 

It's that easy, and working great here.
