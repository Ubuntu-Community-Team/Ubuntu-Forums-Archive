### Post by fluteflute on 2008-08-23
I prefer file/folder names - keeping most of my music in the folder structure:```
/Files/Music/__album__/__number__ - __title__
```

However occasionally I group things into more folders (normally for artists if I have multiple albums):```
/Files/Music/__artist__/__album__/__number__ - __title__
/Files/Music/Jamendo/__album__/__number__ - __title__
```

Also, single tracks (normally downloads where I only want the one track) go in:```
/Files/Music/Misc/__artist__ - __title__
```

Of course ID3 tags are also important (especially for Last.fm scrobbling) but I find them to be very limiting (for example multiple authors) - perhaps not so much the actual ID3 standard, but the lack of full implementation in media players.
