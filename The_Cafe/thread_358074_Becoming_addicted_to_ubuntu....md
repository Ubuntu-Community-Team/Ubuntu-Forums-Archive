### Post by snowboard975 on 2007-02-10
I first installed it a month ago to connect to a school server that provide icfb, the Cadence circuit simulation software. 

Other guys use Windows XP and access their files on the server with "ftp"
But I can use the server as if it is my own filesystem by "ssh"ing it.

They use an X-server emulation program that crashes sometimes to run the circuit simulation program on the server. 
But I can directly run the program on ubuntu that never crashes.

And they're suffering switching back and forth among dozens of circuit windows while I move through only a couple of windows through neatly arranged workspaces.

The most thing that I love with Ubuntu is that it supports customizing keyboard shortcut to maximize, minimize, or move a window! 

Moreover it has the synaptic package manager that I couldn't see before in Windows XP. It's very convenient. I could find all most all programs that I need there! And they're all malware free!

I was surprised many times to find out some powerful features that I couldn't use in Windows XP. I think I am becoming addicted to ubuntu and cannot help stop using it in the hope of finding another amazing features.
