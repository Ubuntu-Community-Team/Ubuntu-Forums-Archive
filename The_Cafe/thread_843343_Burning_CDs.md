### Post by Tatty on 2008-07-01
> **Silpheed2K said:**
> Thanks for the response..
I was guessing that as long as the raw data is handled as raw data there shouldnt be any difference (instead of puting it through something like DirectShow and re-recording it to another format in a sense)

Well it depends what you mean by raw data.  If you are burning a wave file to CD then it should end up with the same data on the CD no matter what you burn with.

If you are burning something which has been compressed such as mp3,mp4,wma etc then it will very much depend on the algorithms used to decode the audio.
