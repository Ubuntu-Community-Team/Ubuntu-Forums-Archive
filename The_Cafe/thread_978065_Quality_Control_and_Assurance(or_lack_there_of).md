### Post by igknighted on 2008-11-10
This is open source... so the simple answer is no one has looked into the bug that knows the answer.  No way of knowing if people aren't looking into the bug, or if the solution is simply not known.

Honestly though, this is nothing new.  Bugs in all software (windows, mac, linux, and more) often sit for years like this.

I wouldn't worry too much about this, the telepathy framework is soon going to be the main instant messenger framework for linux, so this should be solved.  I believe that kopete will someday use telepathy, but in the meantime Psi should give you a decent QT based preview.

EDIT: That is a forum post, not a bug report.  Developers don't read forums, only bug reports.  So if no one reported it... it wont get fixed.
