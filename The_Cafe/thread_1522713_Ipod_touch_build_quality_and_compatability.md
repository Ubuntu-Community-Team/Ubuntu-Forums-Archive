### Post by murderslastcrow on 2010-07-02
From what I know it just works out of the box with Ubuntu, but I've heard you need to sync it or hook it up to iTunes on another computer to write the initial database, then Ubuntu can access it from then on.

However, this may have been fixed by now. But yeah, iPad even works with Ubuntu 10.04.

There are community documentation pages on this. [https://help.ubuntu.com/community/PortableDevices/iPhone](https://help.ubuntu.com/community/PortableDevices/iPhone)

That should give you what you need- the main thing you'll miss out on is the ability to buy/add apps from the App Store in iTunes. However, Ubuntu does have utilities for organizing these, and if you're willing to jailbreak your iPod you can get third-party applications without using iTunes for anything.

However, if I were you I'd run a minimal VirtualBox with XP just to get that kind of stuff. If you use Microsoft's official ISO, you can demo XP legally for 30 days, which should be long enough.

However, if you have a friend who's willing to let you use their iTunes, that's an option.

If all you want is the music/video/picture/third-party stuff, Ubuntu will suite you perfectly well.

### Post by LMP900 on 2010-07-02
The glass touch screen is very strong. It goes in pockets and bags without a case and has not been scratched. The metal back scratches very easily, however. I've read about issues with the buttons on the headphones (32GB & 64GB models only) but I think it's only dust or lint building up and can be fixed by cleaning it. Overall, the build quality is excellent.

I can't comment too much on compatibility, since I sync my iPod to my Mac once, then it's on it's own from there (except for OS updates, backup, and transferring purchases). I wouldn't count on Ubuntu being compatible for a long time, since Apple has the nasty habit of actively restricting outside access to your device. I would only recommend the product if you have access to iTunes especially if you will be purchasing applications or media so you can back it up properly and update the OS.

Keep in mind that the current 8GB model is actually the same as the 2nd generation model, minus the outside shell. It did not receive the processor and RAM upgrades. This means that only the current 32GB and 64GB models support the big iOS 4 features like pretend-multitasking.
