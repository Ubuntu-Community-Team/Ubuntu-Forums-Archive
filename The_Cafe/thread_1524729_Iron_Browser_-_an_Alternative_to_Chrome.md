### Post by apb2390 on 2010-07-05
As many know, Chrome is a fast and reliable web browser. However, some of us don't like the fact that Google may be snooping on us. A community of these users have gotten together to re-build Chrome without those worries.

[Iron Browser]("http://www.srware.net/forum/viewtopic.php?f=18&t=1031") is a very viable alternative, having all the speed and features of Chrome, but without security issues. As Chrome is open source, it's nice to be able to do this.

Comparing Iron to Chrome yields minuscule differences in speed, so there's no loss in performance for all of us multi-taskers.
