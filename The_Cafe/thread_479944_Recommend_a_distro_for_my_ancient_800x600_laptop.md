### Post by ThinkBuntu on 2007-06-20
Zenwalk. Absolutely use Zenwalk. You'll learn more about their project at [their website]("http://www.zenwalk.org/"), but in short, it is a Slackware-based, Xfce-focused distro that's blazingly fast and very stable, while including all the latest software. Modern things like wireless have given me hassles with it, which is why I use Debian instead, but for a computer like yours, I can't imagine a better solution. Debian could also work if you use their [Xfce base install (direct download from kernel.org)]("http://mirrors.kernel.org/debian-cd/4.0_r0/i386/iso-cd/debian-40r0-i386-xfce-CD-1.iso"). It works from a single CD, like Zenwalk. With Debian, you also can install from floppy disks and download the OS as it installs (called a Netinstall). Check out their [Getting Debian]("http://www.us.debian.org/distrib/") page for more info.
