### Post by mikodo on 2015-08-06
Thanks folks.

I still haven't found how to search for and open a file ina folder, with one shot.

Key press open Thunar and running "Find" to look for files, is a pain, >> Key press open Thunar > Run Find for Documents, <Enter>, Run Find for folder, <Enter>, Run Find for file  in the folder, <Enter>, is too long a process.

So far, the best I can come up for me is:

Key Press Thunar > Toggle (use the "up key"), to Documents, <Enter>, Toggle to file, <Enter>. If it is in a folder, Toggle to it, <Enter>, Toggle to the file,<Enter>. 

(That's faster than taking my hands off the keyboard, to negotiate the "click-a-round", with the mouse).
