### Post by sadaruwan12 on 2009-04-18
Hi all,

 I;m going to start a small poll between Deluge torrent client and qBittorrent client. First I'll let you know about the qBittorrent client I actually came across this by accident  while I was searching for a alternative for the current client I was using (deluge). When I installed this it just blew me away with the simple and polished GUI it presented to me and setting up the connection for the maximum download speed was a peace of cake. With out diving right in with this one did small test with both the clients (Deluge V1.1.6 and qBittorrent v1.2.0).

The test torrent was a game called Command and Conquer Red Alert 3 upraising-RELOADED.torrent size 5.1Gb's time 1/2H for both the clients to get the Maximum speed it can.

Well surprisingly qBittorrent took over from Deluge like lightning and started to hit the highest download speed it can get on my 512Kbps ADSL. Then I stopped qBittorrent and started with the Deluge even after the 1/2 was over Deluge was only toping 20.5Kbps while qBittorrent came up with 47.9Kbps constantly.

I was really surprised to see this but qBittorent stole my heart away with it sleek and simple GUI and with it's speeds. Also it have many good option's like "Spoof Azures to avoid ban" if I want this I could enable it other wise leave it as it is. Both client use the same libtorrent from Arvid Norberg.

Here is the list of features in qBittorrent
    *  A polished Graphical User Interface written with Qt4 toolkit
    * UPnP / NAT-PMP port forwarding support
    * Encryption support (compatible with Azureus)
    * Multiple simultaneous downloads / uploads
    * Download from RSS streams
    * An integrated search engine
    * DHT (trackerless) support
    * uTorrent Peer eXchange support (PeX)
    * Directory scanning with automated download of torrents inside
    * Good internationalization (~25 languages supported)
    * Selective download of files in a torrent
    * Media File previewing while downloading
    * Torrent trackers edition and prioritizing
    * Seeders / Leechers displayed in download list
    * Styles support
    * Files prioritizing in a torrent
    * Torrent download in correct order
    * System tray integration
    * Tracker authentication support
    * Unicode support
    * Torrent downloading directly from their url
    * Connection though a proxy
    * IP Filtering similar to the one in eMule or PeerGuardian
    * Torrents queueing and prioritizing (new!)
    * Web user interface for remote control (new!)
    * Azureus spoofing to avoid ban (new!)
    * Allow to buy your downloads thanks to Sharemonkey (new!)

And Now in Deluge

# Web UI
# BitTorrent Protocol Encryption
# Mainline DHT
# Local Peer Discovery (aka LSD)
# FAST protocol extension
# µTorrent Peer Exchange
# UPnP and NAT-PMP
# Proxy support
# Private Torrents
# Global and per-torrent speed limits
# Password protection


In qBittorrent the RSS is also there but we've to enable it if we want it and the other thing is we don't have rely on a torrent web site to find a good torrent the built in search function will give you the best torrent available just type the name and start searching. Hope you guys get a chance to install and test qBittorent it's very easy to install and use this might be a very good replacement for those who miss the speed and the simplicity of uTorrent.
