### Post by lovinglinux on 2010-05-13
The test measures the time necessary to launch the browser, open a web page completely and close the browser completely.

[LIST]
[*]Before initiating each test, the browser cache was deleted.
[*]All browsers used had clean user profiles and were vanilla distributions.
[*]Tests were conducted with a fast web site and a slower one.
[*]The completeness of the page loading process was determined by the browser status message and by visual observation of the content, thus being subjected to imprecisions.
[/LIST]

[http://lovinglinuxblog.blogspot.com/2010/05/browser-speed-test.html](http://lovinglinuxblog.blogspot.com/2010/05/browser-speed-test.html)
