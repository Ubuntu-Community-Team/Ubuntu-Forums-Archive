### Post by jpkotta on 2007-02-19
XFractint is the fastest I've tried, and computes a lot of fractals.  If you build it yourself and play with compiler settings, you can get up to 15% speed increase too (the ones I used are CFLAGS='-march=pentium4 -O2 -mmmx -msse -msse2 -malign-double -mfpmath=sse').  The problem with it is that it's still a DOS program with some X code hammered in, and it's kind of hard to use (this is from someone who used to use DOS Fractint).

I remember a easier to use generator called Gnofract.  I think I was unimpressed with it because it didn't draw enough fractals or something like that.

Xaos is very fun.  You want to try that one.  Should be in the repos.
