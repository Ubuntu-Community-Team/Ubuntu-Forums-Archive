### Post by aysiu on 2007-04-19
Well for Ubuntu problems, here are two tips.

1. **If you get an error, Google the error message.** To make sure you get error messages, launch the application from [the terminal](http://www.psychocats.net/ubuntu/terminal) instead of from the launcher icon. For example, if you wanted to see errors from Firefox, you'd type ```
firefox
``` in the terminal. You'd then highlight the errors, copy them, paste them into the Google search box, and see what results show up.

2. **Search the Forums**. Google search results are pretty good for the Ubuntu forums. Let's say you want to do know what a good FTP client is. Just type ```
site:ubuntuforums.org good FTP client
``` in the Google search box, and it'll give you relevant results from these forums.
