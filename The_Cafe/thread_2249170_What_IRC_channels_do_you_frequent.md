### Post by slickymaster on 2014-10-21
> **markodd said:**

Does anyone know of a XCHAT extension/addon (whatever it's called) that enables different colours for different users? I want to be able to distinguish between users more easily.

XChat is highly customizable. Colored nicks can be enabled via Settings, Preferences, Interface, Text Box Appearnce. Changes take effect when XChat is restarted.

List of all available plugins for Xchat: [http://xchat.org/cgi-bin/disp.pl](http://xchat.org/cgi-bin/disp.pl)
XChat 2.0 Plugin Interface documentation: [http://xchat.org/docs/plugin20.html#menu](http://xchat.org/docs/plugin20.html#menu)
