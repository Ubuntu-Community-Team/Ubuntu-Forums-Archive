Yandex has been offering 10 GB of free storage for several months now. They have been invitation only, and their English version used to be in development, but this time I checked, the English version seems to work, and anyone can sign in.

Benefits for us, Linux users, it that we can easily connect with the Yandex cloud through WebDav using Nautilus. No need to install anything. Very clean.

To anyone interested, here's the link: [http://disk.yandex.com/](http://disk.yandex.com/)

*(Yeah, I know, Russian company, and Russians are scary :p But it's the leading Russian web search company, has been in business since I don't remember when and is quite reliable. Personally, I like them.)*
