### Post by mridkash on 2008-01-02
you can use the copy command to update files which are old and copy files which are missing.

```
cp -u SOURCE DEST
```

cp is copy command
-u is the option to update files
SOURCE is the source directory
DEST is the target directory

see man cp for more info
