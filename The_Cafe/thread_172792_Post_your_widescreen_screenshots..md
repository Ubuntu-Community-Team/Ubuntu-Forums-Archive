### Post by GazaM on 2006-05-12
If those pics are of your current widescreen laptop setup, then I know what's wrong. You're screen is running at 1024x768, which is NOT a widescreen resolution! Everything is probably looking stretched and unclear for you.

This was the same for me when I first installed Ubuntu on my new laptop, and I couldn't seem to change the resolution from the gnome screen-resolution program, as it was only giving me the one option. I quickly found that I needed to use a program called 915resolution to allow my laptop to run at 1280x800 (the native widescreen resolution of my laptop). If you have an intel integrated graphics card it is probably the same for you. The package is in the repos and very easy to use.
