### Post by albertfuller on 2010-03-29
> **ikt said:**
> Plymouth was screwed during alpha 2 to alpha 3, there have been times when updating will cause ubuntu to not boot properly etc

If you're really not willing to report bugs etc or be prepared for some potential breakage I'd just wait it out till release, and even then be sure to read the 'notes' page they put out with regards to the issues that you might encounter.

I have been doing Ubuntu installations for many years now and I still don't have confidence in the update feature. I don't favor the gui installation either.

My approach is to:
1 - do a get-selections on my system (2500 packages)
2 - I always save off any files I change/update to a config folder
e.g. /etc/fstab -> /home/myname/myconfig/u9.10/etc_fstab
3 - backup my home folder to dvd
4 - download and burn new version
5 - run the text-based install
5.1 - checkout partition features I might wish to implement
5.2 - if I had to do something on the home partition, I reload it from backup
6 - run set-selections to get back all my (2500) installed packages
6.1 - review my backup config files with the current file and add back my changes if still relevant
7 - do safe-upgrade and enjoy beverage of choice (wine ... cheers!)
8 - when all is done: put on a big stupid grin for an hour and explore the new system

P.S. when I hear people speak on the serious stability of the upgrade feature, across a few versions, I will be willing to go that route. Until then get/set selections is the cleanest "upgrade path" that I know of. 

Ok it might be a bit slower... but with a good vintage, time is the luxury that is needed to best savor life ... I mean, a good wine.

Have fun...

### Post by frncz on 2010-03-30
I have updated regularly to 10.04 using
sudo apt-get update && sudo apt-get upgrade
since Alpha2 or 3.
I have done this on 4 machines, including a very old laptop (384Mb Pentium 3) where it is rather slow.
Lucid works much better than Karmic. It is very reliable and stable. I have not had WiFi problems which I had with Karmic. The interface is prettier and feels contemporary. I have not lost any data.
I still have Flash issues in firefox on a couple of machines which I don't understand, and I wish openoffice spreadsheet or gnumeric had a functional 'solver' function as in Excel. I also wish evolution could cope with Microsoft Exchange and outlook calendar.
