### Post by jml on 2009-04-07
The custom Xandros distro is only meant for distribution with EeePC's that originally came with Linux preloaded.  Having said that, you might try calling ASUS customer support and see if they will send/sell you a copy of the Xandros restore disc for your particular model if one is available.

The default user interface is an interesting change of pace from other linux distros, but I have to agree with the other comments.  If you have any Linux experience you will quickly run into limitations.  I did.  The official ASUS/ Xandros repositories do not hold many additional applications. And ASUS does not seem too intent on posting security updates either.  You can add other repositories such as the official Xandros repository, but since the EeePC version of Xandros has custom "Tweeks," applications from the official repository can cause breakage. 

I have used Ubuntu, Easy Peasy and eeeBuntu on my 900HA (which also came with Windows preinstalled,) and I must say that eeeBuntu meets my needs the best, with Ubuntu a close second.  Just my two cents worth.
