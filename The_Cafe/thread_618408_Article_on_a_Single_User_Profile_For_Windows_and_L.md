There is an article at;

[http://www.goitexpert.com/entry.cfm?entry=singleprofile](http://www.goitexpert.com/entry.cfm?entry=singleprofile)

That shows you how to set up a "single user profile" between Ubuntu 7.10 and Windows.  Basically you are sharing the resources between the two OS'es so that you don't have to transfer files, folders and settings back and forth.

Essentially you are accessing folders and bookmarks etc from the samme folder in Windows and Ubuntu.  I tried it and it works awesome!  It is saving me so much time!
