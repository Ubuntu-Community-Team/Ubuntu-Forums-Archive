### Post by Burstaholic on 2012-01-31
I've found a Teamviewer host on Linux has a serious issue if you use the web client - it will not receive 'ENTER,' which makes working with a Linux machine very difficult (TeamViewer support has confirmed this as a known issue).

I'm assuming the people posting about using it successfully are using the standalone client application to connect, which I haven't tried.

So if the web client is the reason you're looking at TeamViewer, which it was for me, that won't get you very far on Linux. It seems the standalone app works fine though, so just be aware.
