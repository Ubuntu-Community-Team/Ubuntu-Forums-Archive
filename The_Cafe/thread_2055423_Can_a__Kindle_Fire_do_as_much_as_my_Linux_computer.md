### Post by forrestcupp on 2012-09-09
Android has good media players; you don't really have to use VLC in this platform. You'll be much better off if you use the apps that are tried and true on Android instead of trying to force it to use the apps you use in Linux (although some Linux apps have good Android versions).

If you seriously need to type a lot, you'd be better off getting one of the cheaper versions of the ASUS Transformer and save up to get the keyboard attachment. But the honest truth is that tablets are great *supplements* to computers, but not good replacements.
