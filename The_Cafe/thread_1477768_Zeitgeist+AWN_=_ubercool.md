### Post by shafin on 2010-05-09
You have to see it to get it. Basically its an AWN applet which provides a sort of recently accessed files for all related applications. More like Win 7 jumplists, but better than that. For instance if you saved an image using firefox, it'll be available with one click after you open any image related program.
Great for working with projects.
