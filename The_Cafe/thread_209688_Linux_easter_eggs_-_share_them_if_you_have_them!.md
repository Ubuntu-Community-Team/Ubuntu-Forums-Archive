### Post by s_h_a_d_o_w_s on 2006-07-05
This is just a little thread i'm starting to list all the easter eggs I have found and that other forum users have submitted. Enjoy!

1) In the terminal, type:

```
apt-get moo
```

Then type:

```
aptitude moo -v
```

```
aptitude moo -v-v
```

and so fourth...

credit goes to merlyn and aamukahvi

2) Press Alt+F2 and type: 

```
free the fish
```

Credit goes to monchichi

3) Go to about:mozilla

Credit goes to monchichi

4) Type: 

```
sudo apt-get install cowsay
```

then:

```
cowsay -f head-in.cow ouch
```

Credit goes to leech

5) Type:

```
ddate
```

to see the date in some weird format.

If youhave any more easter eggs, please submit them for viewing. I'll add them here. I hope you enjoyed this little collection of easter eggs.
