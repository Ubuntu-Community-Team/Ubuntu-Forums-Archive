### Post by bsharp on 2008-12-01
I'm bored, and nothing like this showed up on the first page of Google, so here we go:

What is/are the Firefox addon(s) that you just can't live without?

Here's mine:

1.[Video DownloadHelper](https://addons.mozilla.org/en-US/firefox/addon/3006) - Flash video downloader that supports the most sites I've ever seen, and it queues videos too.

2. [User Agent Switcher](https://addons.mozilla.org/en-US/firefox/addon/59) - Get around (most) IE-only website browser checks. Never had this one fail on me.

3. [Firebug](https://addons.mozilla.org/en-US/firefox/addon/1843) - Web page debugger.
