### Post by gtsoftware on 2009-07-25
I've been waiting for this Ubisurfer to be released for months so when I got the in-stock notification from Maplin I ordered and now I have one.

It's linux based with a number of installed applications. Firefox for wifi connection browsing and their own (decompressing) browser for GPRS connection.

I'm really pleased to see python 2.5 there too

Nice little machine, quite slow but small, good screen, decent keys. I'm happy so far.
