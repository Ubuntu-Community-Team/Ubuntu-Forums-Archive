### Post by monsterstack on 2009-05-29
Introduce him to the terminal. Seriously. Make him open an a terminal and type in a command that is useful. I don't know, something like,

```
sudo apt-get update
```

You can make him an alias so all he need type is "update" or something later, but the important thing is that he knows the terminal is there, and the basics of what it is. For when he does find he needs some help, there is a very good chance it will involve typing something into the terminal. For the first couple of days using Linux, I kept seeing people give help commands such as "Just type 'gcc -o bl;ah weirdo weirdo.c' once you've downloaded the weirdo libraries!", and I remember thinking, "TYPE IT WHERE!?"

Another thing, explain the main differences about how installing programmes works. Show him all of the ways he can install programmes, and make sure he knows that fundamentally, they are all variations of the same thing (apt). Explain to him why he won't simply find a "Program Files" directory. Tell him that if he's installed something and cannot find it, then he can look for it in the Gnome Menu, or check /usr/bin, or by running the name of the application in the terminal. Perhaps a launcher programme would be useful: he can type terminal commands into that without even realising he's doing it.

### Post by Icehuck on 2009-05-29
Advice?

Play with whatever distribution you use for about three weeks. You have to take the time to get used to a **new way** of doing things. After that go ahead and make a decision on if you like it. 

1. OpenOffice.org is not a drop in replacement of MS Office. It is similar but different and it won't always correctly interpret MS Office documents.
2. Media(music and movies) transfer no problem. Ipods and Itunes are not linux friendly.
3. If he actually likes linux don't go telling everyone and anyone about it. Respect that others use systems to suit their needs not yours.
4. Linux is not the best thing since sliced bread. Ever sliced your own bread every time you wanted to eat? Exactly.
