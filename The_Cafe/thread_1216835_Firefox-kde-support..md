### Post by Islington on 2009-07-18
I love Ubuntu but sometimes using Kubuntu is like hammering a rusty nail in my foot. Before reading further: THIS THREAD IS NOT ABOUT THE SUPERIORITY OF ANY DESKTOP ENVIRONMENT. 


There needs to be better integration of Firefox in kubuntu. Konqueror is great, but Firefox tends to win with its massive arsenal of addons.

The problem is that making Firefox integrated into kde is trivial. Change the theme, change the file selector; change the applications for the different file types that Firefox encounters. All this has to be done manually. And removing these changes is even more tedious.

Is there a way we can build a community package that does all these things automatically, much like the firefox-gnome-support package? Would anybody else be interested in such a package? How would we go about making this? I dont know anything, *anything*, about packaging but I am willing to learn.
