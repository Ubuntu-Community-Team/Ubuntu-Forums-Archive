[Micro Python]("http://www.kickstarter.com/projects/214379695/micro-python-python-for-microcontrollers")

> Micro Python is a lean and fast implementation of the Python programming language that is optimised to run on a microcontroller.  The Micro Python board is a small electronic circuit board that runs the Micro Python language.  The aim of this Kickstarter campaign is to make Micro Python open source software so you can use it in your own projects, and also to fund a small manufacturing run of Micro Python boards so that you can own one for yourself!
