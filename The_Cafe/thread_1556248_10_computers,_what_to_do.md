If you want to run a single desktop session over all the computers you could just do an Ubuntu Cloud installation on all of them ([http://www.ubuntu.com/cloud/private/deploy](http://www.ubuntu.com/cloud/private/deploy)) and then try to get an NX server running on the cloud, discussed here: [http://www.starryhope.com/linux/2010/ubuntu-desktop-in-the-cloud/](http://www.starryhope.com/linux/2010/ubuntu-desktop-in-the-cloud/)

Also check:
[https://blueprints.launchpad.net/ubuntu/+spec/desktop-lucid-desktop-cloud](https://blueprints.launchpad.net/ubuntu/+spec/desktop-lucid-desktop-cloud)

And:
[https://secure.wikimedia.org/wikipedia/en/wiki/Linux_Terminal_Server_Project](https://secure.wikimedia.org/wikipedia/en/wiki/Linux_Terminal_Server_Project)

If I am correct, you should be able to harness the combined power of all the machines as a single desktop. Of course you can only really use a remote desktop connection to it, but you could maybe configure one machine to act as a thin client to login to the cloud.

After that, you can basically harness that power for anything you are doing.
Or give away accounts to friends/family/strangers to test out Ubuntu without having to install it..
