### Post by Mehall on 2009-03-31
Joomla!!

(the first exclamation mark is part of the name)

If you want a blog, Wordpress is a good option actually, and you can add an "about" page easily (see [my blog](http://blog.mehall.co.cc) for an example Wordpress, or see [here](http://razor.mehall.co.cc/) for an example Joomla! with a nice free theme from RocketTheme.)

Both are content Management Systems, and easy to use. (If you can use a word processor, you can make an article, and installation of the system ain't hard either.)

You want a forum? PunBB or phpBB3 (I like PunBB, but it's got a much smaller community, less custom stuffs like themes.)
