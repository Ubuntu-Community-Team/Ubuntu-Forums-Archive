Yes, I think you can say that, if you have the skill, you can play the OSS music on your bass, tune it, change it as you like and play it again to others as you like... As long as you say to others "look, I've found this OSS music, it's free and here's how I like it. If you want it, pay me!". I don't know if it really fails. Look at the wine/cedega/crossoffice example. Wine is distributed (source and binaries) with GNU license, but cedega and crossoffice use wine code for them, change it and "sell" their modified code. Of course they can't sell the GNU code (wine), because it's already free, but the point is you can sell modified code (tuned music).

There are plenty of other example on this in the open-source world, beginning with distros (like Red Hat and others), and ending with wine and many others programs.
