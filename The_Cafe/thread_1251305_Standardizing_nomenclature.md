### Post by Mark_in_Hollywood on 2009-08-27
I propose that the alliterative (and cute) naming convention in Ubuntu be dropped. Jaunty Jackalope, Breezy Badger, etc., only make searching for solutions more difficult. Some posters (across the entire internet, not just this forum) use the full binomial name. Others do not, using only Jaunty or Hoary, etc. Still others use the release number -- 8.04 or 5.10 without the binomial name or partial name. On top of being more definitve, they also are easier to tag, etc. i.e., fewer keystrokes.

As the binomial names are "code words" for the actual release number, if we must have "code names" for alpha, beta, RC and final releases, let us have numbers, please.

thanks for your time.
