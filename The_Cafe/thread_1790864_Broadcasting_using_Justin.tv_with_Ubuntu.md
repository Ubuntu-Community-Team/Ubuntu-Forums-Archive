WebcamStudio: [http://www.ws4gl.org/](http://www.ws4gl.org/)

It can turn a number of things, such as screencasts, or whatever else, into a virtual webcam device, that should be detected by any program supporting webcams, such as the Flash-based Justin.tv streaming thing, so it should hopefully work (though I tried in the past and it didn't work that great for me at least, but worth a try for sure)
