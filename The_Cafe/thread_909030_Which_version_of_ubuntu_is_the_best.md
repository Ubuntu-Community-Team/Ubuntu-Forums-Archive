### Post by snowpine on 2008-09-03
I am assuming you want to stay within the Ubuntu family (since this is the Ubuntu Forums). Here are some options for that computer:

Official releases: Xubuntu 8.04
Unofficial "remixes": Crunchbang 8.04, Fluxbuntu 7.10

If you step outside the 'Buntu family, there are a million options, ranging from Debian (quite similar to Ubuntu but a bit faster) to Slitaz (very lightweight and fast, but lacking many of Ubuntu's features). 

Do you have an existing Ubuntu install that you wish to keep, or are you starting from scratch? Because Xubuntu is very easy to install on top of Ubuntu (sudo apt-get install xubuntu-desktop), while the others are trickier.

ps Upgrading ram from 256 to 512 (or even 384) makes a huge difference.
