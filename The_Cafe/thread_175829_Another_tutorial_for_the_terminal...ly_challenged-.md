[QUOTE=confused57]Does the swap partition have to always be at the end of the partition table?  For example could you have partitions:  Windows/ext3/swap/free space  ?[/QUOTE] I know it does *not* have to be at the end. I don't know, however, whether it is *advisable* for it to be at the end or in the middle. I've never experienced any performance issues with either (the middle or the end). I've never put it at the beginning, because I think Windows likes to be at the beginning.
