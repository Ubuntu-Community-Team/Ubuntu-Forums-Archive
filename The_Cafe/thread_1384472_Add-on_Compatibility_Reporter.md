### Post by lovinglinux on 2010-01-18
Since Firefox 3.6 is about to be released ( get [rc2](ftp://ftp.mozilla.org/pub/mozilla.org/firefox/releases/3.6rc2/) here ) I would like to share the existence of the [Add-on Compatibility Reporter](https://addons.mozilla.org/en-US/firefox/addon/15003?src=external-fxfirstrun). It is a very nice extension created by the Mozilla team, that allows to override add-on compatibility and thus allowing to install extensions that haven't been updated to be compatible with newer Firefox versions.

This is very useful, because several extensions doesn't need new code to be compatible, just a change in the compatibility declaration.

It also has another feature that allows to disable the extension if it doesn't work as expected and report why it doesn't work. You can also report if it does work as well. This is very useful, because the reports are posted in the extension developer area and thus helps to alert the developers about issues with newer versions of Firefox.
