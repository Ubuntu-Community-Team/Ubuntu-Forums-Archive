### Post by lao_V on 2005-09-07
I've just tried this distro and I am extremely impressed with the installation procedure. I found Anaconda text installer and disk druid a lot better then Ubuntu's installer. Being able to chose a "group" of packages is great.

I got the system up and running in less then 10mins. However, by default, the system is completely lockdowned - no ssh and can't su -. Takes few minutes to resolve that. The documentation on the trustix.org site is growing and getting better I think.

I've tried to use Ubuntu as a server. It is pretty good but installation takes much longer. I don't like the second stage - "configuring packages" which takes long time.

However, I'm gonna be trying out both for a while to see if Ubuntu can really fulfill my needs as I don't really like RPM based distros.
