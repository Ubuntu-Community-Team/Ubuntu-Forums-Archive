### Post by LxP on 2006-11-11
> **migla said:**
> Is there a way to set up amarok locked to fullscreen, without window borders and with reduced features?

You may wish to investigate a window manager called [Ratpoison]("http://www.nongnu.org/ratpoison/").  At its simplest, it is a keyboard-driven window manager that forces windows to the size of the entire screen.  It doesn't draw things like title bars or maximise/minimise/close buttons, and its keyboard shortcuts are by default very different from the norm, so the uneducated user would have a really hard time closing it down or switching to another program.

Perhaps the best way to go about this would be to create a new user account, with this account specifically set up to use Ratpoison and launch Amarok.  I have successfully used this approach with a web browser.
