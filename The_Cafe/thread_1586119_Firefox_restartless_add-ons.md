### Post by lovinglinux on 2010-10-01
Mozilla has been working for a while on [a new add-on framework]("http://lovinglinuxblog.blogspot.com/2010/08/meet-next-generation-of-firefox.html"), which brings some advantages, like restartless installation.

If you are using Firefox 4, you can find those add-ons now by searching AMO site using the tag "jetpack":

[https://addons.mozilla.org/en-US/firefox/tag/jetpack](https://addons.mozilla.org/en-US/firefox/tag/jetpack)

Have fun.

If you are not using Firefox 4, then see my [installation tutorial]("http://goo.gl/y92e").
