### Post by -grubby on 2008-08-06
To register on freenode, change to the nick you want using :

```
/nick nickname
```

Then, register it :

```
/msg nickserv register yourpasswordhere youremailaddresshere
```

Then, you must activate it, instructions are provided in the email

After that, in order to identify (If you're using your nick) : 

```
/msg nickserv identify yourpassword
```

If somebody else is using it (for some reason) type : 

```
/ghost yournick yourpassword
```

Then identify using the above instruction.
