Try Slitaz or TinyCore, or if you have the time and experience, Crux.

Also, consider running Linux with X, and use only command-line applications. It might be a little difficult at first if you are relatively new to the console, but you can get a lot done on the cli with much less resources.

If you need to install something on that computer, just connect its hard disk to another computer with a CD drive and internet, install your favourite distro on it and connect it back to the original computer. I've done this several times before and it saves a lot of time and headaches.
