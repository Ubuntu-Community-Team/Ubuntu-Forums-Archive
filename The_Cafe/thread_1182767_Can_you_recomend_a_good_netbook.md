### Post by Ferrat on 2009-06-09
Compaq Mini 730EO works great if it's a small easy extra computer you need for mobility and functionallity
Intel Atom N270 (1,6GHz)
1024MB RAM (Specification says this is max but they lie! I put in a 2GB module today and it works great)
80GB HDD, you don't really need more IMO unless you are using it as a primary computer but then you can use external HDDs
Intel 945GME (Specification lying again, says 128MB shared max but after installing Ubuntu it can use 256MB)

Cost (in Sweden) just under 3000SEK ~(270&#8364; or $375) 

Ubuntu 9.04 Netbook remix from a USB stick installs without any problems and everything works out of the box except the speaker sound but that's an easy 1min fix, other than that everything works great, bluetooth, WiFi, Special keys and buttons all work directly after install and no problems with sleep mode :)

All and all a great netbook that's slightly cheaper than most others and preform just as well or better, also it has a 10.1 screen and a 92% keyboard which is lovely. 

Only thing that's not 100% is the touchpad IMO, a bit flimzy but no problems with it and I hate touchpads top start with so :P
