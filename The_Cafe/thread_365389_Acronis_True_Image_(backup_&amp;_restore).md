I sure hope nobody minds my pumping a product but, this one has saved my bacon so many times. I installed Acronis True Image on Windows and made an emergency CD from within Windows. Using the emergency CD I have backed up and restored Ubuntu a few times. One feature I really like about it though is restoring the MBR which, in the past has always been a hassle. The program costs about $50.00 but, worth every penny to me.
I have just given Sabayon Linux a try and well I'm back in Ubuntu. Sabayon installed the GRUB but, never included Ubuntu in it, just Sabayon and Windows. I used Acronis to restore  the Ubuntu grub.
