### Post by nickburns on 2007-10-06
I have found a nice way to go from a pst (Outlook) to Evolution.

1.  Just install thunderbird on the windows box.
2. import you settings and mail into thunderbird.
3. Go to Docs and Settings/User/AppSettings/Thunderbird and get all the mBox files (no extension)
4.  Save them to ubuntu box
5. Import them into Evolution.
6. Enjoy
