### Post by akniss on 2005-11-10
Hi all.  I've been using ubuntu for a few weeks now, and I have to say... if I could french kiss an operating system, i would!  I've been a Windows user since high school, so linux has been an intimidating move for me.  Up until today, I've not been able to make the switch because I could not get my  wireless connection working at home.   I thought it was that Ubuntu didn't set up my wireless card correctly.  I am in a hotel room right now, and Ubuntu connected to the Wi-Fi here with absolutely no trouble.  upon searching the forums for a few seconds, I realized it is my home wireless gateway that is causing the problems, not device drivers.  As soon as I get home I'm buying a different wireless router to use instead of the Actiontec gateway provided by Qwest.  I think you have a new Ubuntu convert for life!

if anyone is interested, I'm using Breezy on a Dell inspiron 600m notebook.  AND EVERYTHING JUST WORKS!!!  Goodbye Microsoft!!!
