### Post by kevinf311 on 2007-02-28
I took the easy way out. I figured that disabling the alt-text windows that come up would disable them everywhere. 

So instead I went to Extras >> Window Thumbnails >> Taskbar and chose to have the thumbnail always on top. This lets the thumbnail be unobstructed by the alt-text. 

There is probably a more graceful way to do this, like only disabling the alt-text for the window bar panel, but this was a quick fix that gave me the results I desired. Hope this helps!
