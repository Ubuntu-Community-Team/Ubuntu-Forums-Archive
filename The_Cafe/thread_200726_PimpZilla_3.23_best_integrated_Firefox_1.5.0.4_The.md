### Post by RAV TUX on 2006-06-20
After searching for a long time for a Firefox theme that integrates the best with the Ubuntu Human Theme I have found PimpZilla 3.23 to be the best. Especially if you set your custom tab options to disable "Hide the tab bar when only one website is open".

This leaves up the beautiful leopard print on the tabbar up which blends nicely with the Ubuntu Human theme browns and also metaphorically works well with the African origins of the word and the founder of Ubuntu. The rest of the theme has a fuzzy beige faux fur look to the dashboard and gold icons and scrollbar that sparkle.

This theme should be the default Firefox theme with the Ubuntu Distro. Simply Beautiful and best integration with the Ubuntu Human Theme Available (and yes a have tried the supposedly integrated Ubuntu Human themes for Firefox that are horrid).

Perhaps they can include this in the Edgy Eft.
