### Post by pootan on 2012-03-05
First one I always add is the Mycroft search plugin which is a database of search plugins:

[https://addons.mozilla.org/firefox/addon/mycroft-project/](https://addons.mozilla.org/firefox/addon/mycroft-project/)


Then I add the the 1st and 3rd from this page:

[http://mycroft.mozdev.org/search-engines.html?name=ubuntuforums](http://mycroft.mozdev.org/search-engines.html?name=ubuntuforums)


and there are many other Linux search addons you can find there.
