the alternative text base installer cd has raid tools.

[http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)

choose a mirror

[http://www.ubuntu.com/getubuntu/downloadmirrors#mirrors](http://www.ubuntu.com/getubuntu/downloadmirrors#mirrors)

the raid tools aren't on the live cd because their is no space to accommodate these raid tools.

configure the raid bios and run the cd, your array should be detected.

if you plan on dual booting with windows' you may run into technical issues with it, if that's the case' leave well enough alone.
