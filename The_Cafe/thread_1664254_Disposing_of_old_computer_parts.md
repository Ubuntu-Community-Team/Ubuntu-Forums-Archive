### Post by Old_Grey_Wolf on 2011-01-10
> **mamamia88 said:**
> 1 more question guys.   1 of the ram modules says 128mb dimm ram.  the other one isn't labeled.  does that mean each stick is 128mb or it's 128mb combined?  came from an oem compaq i got for free

Each DIMM is usually labeled separately, and not as a pair. If you can get to the BIOS of the computer, it should tell you how much RAM is installed. If you have Linux installed on the computer or you can run Linux from a CD/USB, you can type in the command "lshw". Look toward the top of the output from the command to find the "*-memory" label with the total size of the memory detected listed below. If it says 256MiB then you have two 128MiB DIMMs.
