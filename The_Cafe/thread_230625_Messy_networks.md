I've found the easiest way to do this would be to build a new network in parallel to the existing one. First audit the existing network, then you remove what you can from that network, plan the new one, install & config it, do the wiring followed by cut-over. This allows for minimum downtime which is usually required in areas where you have a 99.99% uptime SLA.

In order to do the above you need spare equipment which is not always a problem in a big company. Alternatively make as much of the current equipement redundant by ofloading onto spare capacity elsewhere on your network so you can re-use the recovered equipment.
