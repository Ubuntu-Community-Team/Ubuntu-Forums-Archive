### Post by Frogs Hair on 2016-04-30
[https://github.com/moka-project/moka-icon-theme](https://github.com/moka-project/moka-icon-theme)

[https://launchpad.net/~moka/+archive/ubuntu/stable](https://launchpad.net/~moka/+archive/ubuntu/stable)

```
sudo add-apt-repository ppa:moka/stable

``````
sudo apt-get update
```
```
sudo apt-get install moka-gtk-theme moka-icon-theme

```
