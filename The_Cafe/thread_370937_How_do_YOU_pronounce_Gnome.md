### Post by Darkhack on 2007-05-19
The correct pronunciation according to the developer is "guh-nome", although "nome" is very common and if you verbally said it that way to a Linux user they probably wouldn't raise an eyebrow to it.  Although I was very very surprised to see the number of people in this thread saying "jee-nome".  I have never heard anyone say it like that.  When I was a beginner I said "nome" but then I found out the correct way was "guh-nome" but I went back to "nome" because I was very shocked to find many of my friends were unfamiliar with this pronounciation although if I were to ever meet a real Gnome developer in person, I would switch back to saying "guh-nome".  So it really depends on whom I'm speaking with.  To summarize... 

GNOME = guh-nome (nome is acceptable, but incorrect)
GNU = guh-new
Ubuntu - oo-boon-too (the u is pronounced like a double o in almost every language but English (I HATE the English language, but that rant is for another time))
Linus = lin-us (Finnish)
Linus = lee-nus (Swedish - Torvalds is part of the Swedish speaking minority in Finland)
Linus = lie-nus (English - think Charlie Brown's friend from Peanuts)
Linux = lin-ux (Torvalds expressed in the documentary Revolution OS that this is the only way to pronounce the name of his operating system although he has no particular preference when it comes to his first name.  You can use which ever accent you like.)
