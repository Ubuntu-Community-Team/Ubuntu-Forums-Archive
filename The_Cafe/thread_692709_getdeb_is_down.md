Taken from [www.getdeb.net/sponsors](www.getdeb.net/sponsors) :

The following organizations are currently supporting GetDeb with mirrors:

[LIST]
[*] [http://www.akl.lt/](http://www.akl.lt/) - Open Source for Lithuania
[*] [http://darkstar.ist.utl.pt](http://darkstar.ist.utl.pt) - Portuguese open source software mirror
[*] [http://www.diotto.net/](http://www.diotto.net/) - Diotto.Net
[*] Dotsrc.org - Open Source Hosting
[*] ENDPOINT.NU
[*] [http://www.fastbull.org](http://www.fastbull.org) - Fastbull Association
[*] pebosi.net - German mirror
[*] [http://www.heanet.ie/](http://www.heanet.ie/) - HEAnet Limited, Ireland's Education and Research Network
[*] [http://labby.co.uk/](http://labby.co.uk/) - LabRat's mirror
[*] [http://neacm.fe.up.pt/](http://neacm.fe.up.pt/) - Núcleo Estudantil da ACM
[*] [http://www.internap.com/](http://www.internap.com/) - Internap Network Services Corporation
[*] [http://www.sofrench.net/](http://www.sofrench.net/) - SoFrench.net
[*] [http://www.tuxli.ch/-](http://www.tuxli.ch/-) Tuxli.ch
[/LIST]

For example, to use the mirror from students in "Instituto Superior Técnico" in Ubuntu 9.10, you can create a file as **/etc/apt/sources.list.d/detdeb-games.list** with this content:
```
deb http://darkstar.ist.utl.pt/getdeb/ubuntu karmic-getdeb games #GetDeb games for Karmic
deb-src http://darkstar.ist.utl.pt/getdeb/ubuntu karmic-getdeb games #GetDeb games for Karmic
```
And also a file as **/etc/apt/sources.list.d/detdeb-apps.list** with this content:
```
deb http://darkstar.ist.utl.pt/getdeb/ubuntu karmic-getdeb apps #GetDeb applications for Karmic
deb-src http://darkstar.ist.utl.pt/getdeb/ubuntu karmic-getdeb apps #GetDeb applications for Karmic
```
