Hi all,

I'm developing an Ubuntu derivate distribution [http://www.cloudusb.net](http://www.cloudusb.net)
which installs on a USB stick: personal data privacy is important, but
must be widely available and easy to use!

The solution I developed is:
- a USB key containing your operating environment and your data
- a protected folder so nobody can access you data, even if you lost the key
- a backup program which keeps a copy of your data on an online disk, with doub\
le password protection

Have a look ... and any comment is welcome!

bye
giammy

P.S I was thinking to call it CloudU(S)Buntu :-)

I was thinking about the idea of cloud computing (the one you are
misleaded to I imagine) of a user sitting at his desktop, using a
"cloud" of resources all around theweb, and I thought that in a
"complete" cloud computing environment, not only the resource
can be all over the world, but the user but be able to be all over
the world, with his working environment, in a format as small 
as possible ... and so the name :-) 
Probably my line of thought is not very linear ...
