### Post by 23meg on 2007-06-04
[File a bug]("https://launchpad.net/ubuntu/+filebug-advanced?field.tags=upgrade") with the "upgrade" tag. The link will take you to a page where the tag is readily placed. See the [other requests]("https://launchpad.net/ubuntu/+bugs?field.tag=upgrade") if you need examples. The same procedure detailed in the sticky [Requesting New Packages for Inclusion in Gutsy Universe]("http://ubuntuforums.org/showthread.php?t=414355") thread applies, except that the tag is "upgrade" instead of "needs-packaging".
