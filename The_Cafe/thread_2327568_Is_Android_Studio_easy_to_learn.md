### Post by Nytram on 2016-06-17
If you haven't coded for a decade it will probably take some time to learn how to code for a phone or tablet. You will need to be competent with object oriented programming and Java. Then more specifically it takes time to learn the android api and other things such as the gui layouts known as activities in android. If you're a decent coder you can pick these things up pretty quickly. The ide and sdk are free to download and use, you mention widows and you can use it on that platform, but it's also available for linux.

I've only written one app so far, but I'm currently working on another one. I think the best place to start is the official Android developer site:

[https://developer.android.com/training/basics/firstapp/index.html]("https://developer.android.com/training/basics/firstapp/index.html")

Hope this helps and good luck, it's fun writing apps for one's phone.

### Post by grahammechanical on 2016-06-22
You folks might be interested in Ubuntu Make. It will install Android Studio and keep it up to date. See the section How to use it.

[https://wiki.ubuntu.com/ubuntu-make](https://wiki.ubuntu.com/ubuntu-make)

It is also possible to run Android apps on Linux. Install Google Chrome. Then install Google Arc Welder and use it to test the apk and it will load on Ubuntu.

I am using Arc Welder to run an Android app. It works well. The first purpose of Arc Welder is for Android app developers to test their app on Google Chrome OS but it works just as well on Linux. The requirements are Google Chrome + Arc Welder + any apk file.

Regards.

### Post by misfitpierce on 2016-07-05
Android Studio may actually look pretty intimidating at first because of how much crap they got cluttered everywhere but with its simple android autocomplete commands and built in visualizer its actually probably one of the easiest for android dev. You dont need to know what everything does inside the app so dont focus on stuff you dont need atm and if you ever need to know something you can even trigger a Google This command from inside the app. Pretty simple and straight forward. Java itself tho is a pretty simple language once you get the basics down so... I'd def go with android studio for android developing specifically and its updated pretty often.
