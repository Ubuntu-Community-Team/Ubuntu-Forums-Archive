### Post by lz1dsb on 2010-08-24
I'm using WRT54GL for years now running DD-WRT. I'm quite satisfied with it. I run Wireless Distribution System on three routers and it's quite nice. I'm able to cover very large area, without the need to use any cables. 
If you want something more user friendly, check out DD-WRT and Tomatoo. 
I believe that OpenWRT is more close to the Linux environment we're all used to (though all third party firmwares are Linux based) and gives you more opportunities to play around and reconfigure the system. DD-WRT for example is more commercial, if you need more fancy features, you'll need to buy that package, but as far as I know with OpenWRT you can compile any additional package you need, just like in any other Linux machine, if you have enough resources off course ;) I've been running only the free package so far...

Cheers,
