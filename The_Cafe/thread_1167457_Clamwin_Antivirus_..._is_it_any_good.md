### Post by handy on 2009-05-23
I use ClamAV on my headless, IPCop/Copfilter firewall/router/proxy/& some other services, box.

I don't need it, as we have no windows machines on our LAN, but seeing as it doesn't slow down our internet speed, my thinking is why not use it.

I have ClamAV set to update its brain-file daily, which it does, after which it emails me the confirmation, it will also email me if it finds any virus.

If there was a windows system on the LAN I would subscribe to one of the other anti-virus applications (which cost money) that are available via the IPCop/Copfilter add-on, which incidentally allows you to run multiple anti-virus app's simultaneously.

It has been so long since I've had windows around that I've forgotten all about the joys of virus infestation. :)
