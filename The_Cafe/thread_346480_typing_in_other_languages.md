### Post by seshomaru samma on 2007-01-26
I think you should post this in the Absolute Beginner section
To write Chinese, Japanese and Korean (CJK) you need to install SCIM
I am not sure what version of Ubuntu you are using and how you installed language support but if you are using Dapper or Edgy installing any CJk language package should automatically  install SCIM . SCIM is activated by pressing ctrl+space
Sometimes it doesn't work in an English session, for this you would need to play with the im-switch
But first if CTRL+space doesn't bring up SCIM
open a terminal and write
```
gedit
```
write click anywhere in gedit and look if you have SCIM under 'input methods' if you do choose it and press CTRL+space again
If any of these doesn't work , post again....
good luck
