### Post by The Titan on 2008-03-25
Are you interested in joining an open source community project? Want to develop your PHP coding skills, or just practice them? This is for you.

[COLOR=Sienna]*[SIZE=4]What is it?[/SIZE]*[/COLOR]
I am developing an open source CMS called Fireware. For those who don't know a CMS, or Content Management System is like a wizard for web pages, that allows you to build or maintain a web page through a Graphical User Interface. It has templates, modules and other utilities to make web developers of all experience levels lives easier.

[COLOR=Sienna]*[SIZE=4]Other open source CMSs exist, why this one?[/SIZE]*[/COLOR]
Fireware, is intended to be a learning project, many open source projects exist, but many are too complicated for beginners to develop their skills the most important way, through practical implementation of their skills. This project well be paced so the beginners can ask questions and start to contribute. In addition, many complain that existing CMSs are not what they are looking for, an this is an attempt to provide an easy-to-use-alternative.  We want to have help not only for the administrators but the clients as well and an admin area that morphs to your needs.  We also want to keep the source very straight forward so it is easy to modify, given the user has some PHP experience under there belt.


[COLOR=Sienna]*[SIZE=4]Can I help?[/SIZE]*[/COLOR]
You certainly can. We are looking for :
-PRIORITY: People that want to help get the deveopment wiki going!!!
-Php coders
-Graphical artists
-Testers/Constructive critique

Note: Not all of these skills are necessary, nor do you have to be an expert.

Of all skill levels. The requirements are that you are eager to learn and contribute, and patiently work as a team. I am also looking for experienced programmers, and ask that they will be willing to provide others with knowledge.
