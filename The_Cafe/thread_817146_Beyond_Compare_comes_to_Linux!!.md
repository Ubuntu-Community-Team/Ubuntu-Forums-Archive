### Post by KingTermite on 2008-06-03
I've seen various file diff tools for Linux, but from the screen shots I could see they were mediocre at best. For a number of years at work we have used a very inexpensive shareware file diff tool called "Beyond Compare". I've still never seen anything that even came close to it, IMO.

I decided to email them yesterday and inquire if they had a Linux version (not expecting much). Well, sure enough they have a major new software update (haven't had one in a few years now) and there is a Linux version. 

It's not free or open source, but from my experience using it, the tool is well worth the paltry fee they ask for it. It's one of the best tools a developer can use for comparing source code.

[http://www.scootersoftware.com/beta3/download.php](http://www.scootersoftware.com/beta3/download.php)


P.S. I don't work for Scooter Software nor am I compensated in any way. This is just a tool that I love and wholeheartedly endorse.
