### Post by Jackp90 on 2008-11-11
After you get compiz fusion you need the following manager to set it up. Open a terminal and type:

> sudo apt-get install compizconfig-settings-manager

to run compiz hit Alt+F2 and type compiz

A cool add on for compiz is awn aka avant, to get it type:

> sudo apt-get awn-manager awn-applets awn-applets-python-core awn-applets-python-extras

And then there are the applets:

> sudo apt-get install gdesklets

These are better applets for your screen though . . .

> sudo apt-get install screenlets

Hope this is what you were looking for.
