### Post by Rokkross on 2013-02-26
xfce + awesome as the window manager is pretty neat. Can't really say much as to where you would "get" this as it's a somewhat specialized set up. Basically I set the saved session I was using to not start xfwm4, and put "awesome" in the list of startup applications. The GTK theme I'm using is just blackbird, and the theme.lua file I'm using is just a modified version of the "dunzor" (I think) theme, which is included in this collection: [COLOR=Blue][https://github.com/mikar/awesome-themes](https://github.com/mikar/awesome-themes)[/COLOR]

As for how I removed awesome's menu, well, I'm not quite sure -exactly- what I removed, because I just removed anything relevant to "mymainmenu" or "mylauncher". Here's what I get when I compare the differences between my current and my backup:
```
$ diff rc.lua rc.lua-backup
93a94,101
> mymainmenu = awful.menu({ items = { { "awesome", myawesomemenu, beautiful.awesome_icon },
>                                     { "Debian", debian.menu.Debian_menu.Debian },
>                                     { "open terminal", terminal }
>                                   }
>                         })
> 
> mylauncher = awful.widget.launcher({ image = image(beautiful.awesome_icon),
>                                      menu = mymainmenu })
171a180
>             mylauncher,

```If you're looking for just a tiling WM, there's a more comprehensive [COLOR=Blue][guide to setting xfce up with xmonad]("http://www.haskell.org/haskellwiki/Xmonad/Using_xmonad_in_XFCE")[/COLOR]. I just went this route because I really wanted a dynamic window manager.
