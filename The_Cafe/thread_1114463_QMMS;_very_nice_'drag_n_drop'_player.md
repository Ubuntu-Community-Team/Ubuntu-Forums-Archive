### Post by BGFG on 2009-04-02
Was copying some music to a flash drive for a friend and as i browsed the music, using the very cool quick preview built into Gnome, i really felt like listening to some of the songs. Which made me thing It would be a cool enhancement to nautilus if you could have a simple preview column to the right of the screen built into the file manager that could play songs. A quick simple playlist while you perform file operations :)

Anyway, I did not feel like launching Exaile, my primary media manager, so i took to add-remove intending to install XMMS. Well QMMS came up and i decided to try it. It's great. great sound quality and very nice and simple. Built in QT and very light, 4 packages less than a meg in all.

I often like to browse my music folder directly rather than going through a media manager so players like audacious, XMMS and QMMS suit me quite nicely. So check it out :)
