### Post by glamajamma on 2007-02-26
I just recently noticed the automatic deb installer that pops up in firefox, when you click on a deb on a webpage.  I think this is a very bad idea.  I think the simple user would be better served if when they clicked to install software it could only que up software already in trusted repositories.  If someone wants to install a binary from an untrusted source they can go through the trouble of installing as debs have always been installed.  The installation model in place right now is too easy to take advantage of via social engineering.  I don't see creating some file extension that que's up trusted software in synaptic too difficult.
