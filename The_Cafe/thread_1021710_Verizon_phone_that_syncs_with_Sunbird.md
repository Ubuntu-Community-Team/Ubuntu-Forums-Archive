BirdieSync works very well, if you're in Windows.  It's not free, though.  It's like $40 or something.

I do sync my Verizon XV6700 PocketPC phone with Ubuntu.  But I don't use Sunbird.  You can only sync it with Evolution, which I'm growing to like a lot.  I'm starting to like Evolution more than Thunderbird with Lightening.  There were some issues with Thunderbird that Evolution doesn't have.

If you want to sync a pocketpc phone with Evolution in Ubuntu, you need to use Synce to connect to your device and Multisync to sync it with Evolution.  [Here are instructions](http://www.synce.org/moin/SynceWithUbuntu) on how to set it up.  It works flawlessly for me.

A word of advice.  Back up your contacts before you set up a partnership with Ubuntu because you'll lose them all.  But since you're getting a new phone, I guess you won't have any contacts yet.

Edit:
if you follow those directions, it asks you to install a package called **librra0-tools**.  It couldn't find that package when I tried.  I had to install **librra-tools** without the '0' instead.
