### Post by OrangeCrate on 2008-03-20
If you're using Firefox, you can use either Gmail Manager or Gmail Notifier. Both will notify you of new mail.

[https://addons.mozilla.org/en-US/firefox/addon/1320](https://addons.mozilla.org/en-US/firefox/addon/1320)

[https://addons.mozilla.org/en-US/firefox/addon/173](https://addons.mozilla.org/en-US/firefox/addon/173)

For the Ubuntu desktop, I just looked at cgmail, which is in the repositories. I liked it, but I don't really need one outside of the browser. I know there are a couple of others for the desktop in the repos too, but I'm not familiar with them.

[http://cgmail.tuxfamily.org/](http://cgmail.tuxfamily.org/)

Anyway, one of these should do the trick for you...
