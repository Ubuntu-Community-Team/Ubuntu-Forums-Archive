3ddesk
```Your desktop should respond by shrinking and showing the other three. at this time they'll be grey, so kind of ugly.
use left/right arrows to move, then enter to select another and it should come back to full screen.
Next time you run it, that one will also look right in the multiple view. Once you've loaded them all, you're off.

Somewhere on these forums is a thread about how to bind it to a F key. Or you can create a launcher in a panel. :cool:
