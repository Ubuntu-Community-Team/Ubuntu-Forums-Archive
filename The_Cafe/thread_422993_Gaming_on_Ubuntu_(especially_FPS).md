### Post by forrestcupp on 2007-04-25
If you don't mind commercial games, there is a good linux installer for Soldier of Fortune II.  There is also an installer for Return to Castle Wolfenstein and Medal of Honor Allied Assault.  The only thing about the MOHAA installer is that the audio doesn't work perfectly, which is kind of annoying.

There's a decent racing game that is in the repos called TORCS.

With wine you can get some of your other games to work too.  Go to [www.winehq.org](www.winehq.org) and find the HowTo section.  You can find an applications database there to see what works.  

There is also a website called frankscorner.org that tells how to get a lot of games/apps working with wine.

There are other resource like what someone listed above that will tell you open source games.
