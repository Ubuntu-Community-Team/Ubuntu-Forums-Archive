### Post by snowpine on 2009-01-26
Hi Sidewalkcynic, 7.04 is no longer "officially" supported, but the repositories have moved to ubuntu-old. In other words, you can edit your sources list using the following command:

```
gksu gedit /etc/apt/sources.list
```

Then change every line to say ubuntu-old, for example, change this: 

```
deb http://archive.ubuntu.com/ubuntu feisty main restricted
```

to this:

```
deb http://old-releases.ubuntu.com/ubuntu feisty main restricted
```

I think that will allow you to install the packages you need, until such time as you are able to upgrade to a more recent version. :)
