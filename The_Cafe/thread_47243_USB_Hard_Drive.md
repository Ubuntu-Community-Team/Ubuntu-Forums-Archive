### Post by apollo2011 on 2005-07-07
You can install Ubuntu on an External USB Hard Drive.  You don't need the Live version, the live version is basically for trying it out and demoing it.  You don't install that.  So you can download the Install CD and install it like normal and the Partitioner in the setup should see your external hd.  I am not sure how well it will work when you move from machine to machine.  You might have hardware incompatibilities, but all the USB hardware should work from machine to machine.
