Hey, Folk!

ImpAmp is a program written within a web page for improvised sound tech. The idea is that you can assign an mp3 file to a letter on the keyboard so pressing that key will instantly start playing that sound file. Voila! Instant gratification sound effects keyboard that works within firefox,

Spread the improvised love far and wide!

I'm really hoping people can take a look at it, have a play around with it, add files or change settings, and make suggestions for improvement.

You can download the latest version (version 1.3 "Fringe 08" ) from
[http://launchpad.net/impamp](http://launchpad.net/impamp) either under the "Latest Downloads" title
or click on "External downloads"

A demo version is available at
[http://www.see.ed.ac.uk/~s0453136/impamp/demo](http://www.see.ed.ac.uk/~s0453136/impamp/demo) (if you'd like to test
it out before downloading it)

Have fun!
