### Post by Spacecaptain on 2006-09-22
to change the hair colour, the best i can think of is to select it in the picture and cut and paste it to a different layer.
for this kind of complicated selections, there is a very cool function in GIMP:
use the "Create and edit paths" tool from the tools palette and draw a path around the area you want to select (in this case the hair).
then go to the "select" menu and click on "From path".

now cut, paste to a different layer. you will be able to change the colour by using the Layer->Colors->Hue-Saturation method.

have fun! :)
