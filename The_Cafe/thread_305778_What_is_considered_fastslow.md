### Post by voided3 on 2006-11-23
Alrighty, here is where I open a can of worms. When I talk to people about computers, opinions vary about the capabilities and how "obsolete" the system is. I've heard anything from "with 2.0ghz or less you're pretty limited" to "I got Windows XP to run on a Pentium One". I currently own 6 computers ranging from a '96 Compaq Presario @ 200mhz to a dual core 2.16ghz Macbook Pro, but I personally enjoy toying with older PCs and finding uses for them, so I believe that as long as a computer has a significant use (web browsing, music playback, word processing/printing) it isn't obsolete. What are your opinions on this? Personally, this is how I see the speed/usability ratio with the amount of RAM factored in that's typical for those systems:

--150mhz or less: 32mb or less- Basic word processing, extremely basic GUI or command prompt, possibly music playback  
64+mb- same, but add very basic web browsing and music playback

--150-300mhz: 32-64mb- add normal web browsing and basic video playback 
64-256mb- add older 2D/3D games (graphics dependent too)

--300-500mhz: 64mb+- add higher desktop resolution and 32 bit color (if supported), DVD playback

--500-1000mhz:  128mb+- more advanced 3D apps

--1.0-1.5ghz: 128-512mb: same as above, but faster
512mb-1.5gig: add average graphics intensive games

--1.5ghz-2.5ghz: all RAM sizes same, but faster load times/FPS

--2.5ghz+: currently no real cap on capability, if graphics/RAM allow (i.e. everything but insane games that need SLi and 2+gig of RAM)



Albeit this list is a little sparse, I think it's a reletively decent outline. Other huge factors (amongst many) are the system's operating system, graphics card, internet connection interface, and hard drive. Also, you sometimes see computers with large RAM amounts on older processors, such as a 333mhz comp with 768mb I saw on ebay. For another example, DOS would be much faster than WinXP on a 300mhz computer, but from a general use standpoint, would you guys agree or disagree with the above? I'm not out to make arguments, I'm just curious to hear opinions and more so personal experiences. Any thoughts/comments?

### Post by fritz_monroe on 2006-11-23
Personally, I think that usability starts to fall off with systems that are slower than about 400MHz.  While slower systems can actually do the things you point out, people start getting frustrated with the wait times involved.

I also feel that people are too quick to upgrade.  Their system starts getting slower because of the spyware and adware.  They don't know that they could clean up the system and get it working better.  A 1 GHz system works just fine for what most people would use it for.  Granted, it won't run the latest and greatest games, but web surfing, MP3 playback and word processing it's fine.

The problem with slower systems is that Windows is so hardware intensive, and with Win98 being officially retired, that spells the death for many older computers.  While Linux runs great on these systems, I think most everyone will agree that Linux isn't quite for everyone yet.
