### Post by cdahmedeh on 2009-09-26
Hello,

This is something I've noticed with Ubuntu and other Linux distributions that use GNOME and KDE environments; the icons and text are always very big. There is lots of space in between items, toolbars take important space, etc. On my production system, I have a custom .gtkrc file in my home folder and have done some configuration changes. Here I compare my current Jaunty installation with custom options, and a Karmic Install in VM with the default options. The distribution version isn't really important, just for reference purposes. I decided to attach a few pictures with some comparisons so that you can see what I mean.

Basically, I think that having smaller text and icons makes the desktop feel larger and cleaner, something that many users will appreciate. It will reduce scrolling in many cases, especially those with small screens like netbook owners. It's also very hard to change the the icon sizes of toolbars, menus, etc, they need be to via the gtkrc files. It would be interesting for sure to have an option to change this.

What do you other users think ?
