### Post by TheFu on 2012-08-29
2 weeks ago, found a $100 AMD E350d APU / motherboard / case deal at NewEgg.  $30 for 4GB of DDR3 RAM locally, an old SATA HDD and XBMC is happily working running 1080p content.  The embedded ATI GPU rocks for everything I've tried to play from mpeg2, mpeg4, avi, wmv, flv, mp4, mkv, h.264, xvid, divx, .... whatever.

Using a 3 yr old $17 media center remote with it. Works perfectly.

The only negatives were
* Blue pwr LED on the case is blindingly bright.
* Case PSU is loud like a jet engine.  $35 PicoPSU on the way-completely silent.
* Case fan open for PSU fan was off center blocking 30% of the output. Unimportant with the PicoPSU.

BTW, it runs** Ubuntu 12.04 + XBMC **with **Nav-X** and **FreeCable** plugins.
**Not bad for $165.**

Ubuntu 12.04 was snappy before XBMC was loaded, but this isn't a desktop replacement. I can see using it router, NAS, home server, or media center PC, but not for CPU intensive needs.

If I built these for $250, would anyone buy it pre-configured with XBMC pre-installed?
