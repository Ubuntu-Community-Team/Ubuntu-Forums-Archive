### Post by frenchn00b on 2012-07-14
Hi,

I have found relatively comfortable to play the TV on the console by using FBDEV (no need of X11).

It works like this. Get the file and run it with:


```
sh onlinetv-nox.sh --dialog
```

or 

in debug mode:

```
sh onlinetv-nox.sh --dialog --debug
```

It is cool but still some more channels and update would be appreciated from the community.

You can contact us at Freetuxtv for more info.

Cheers
