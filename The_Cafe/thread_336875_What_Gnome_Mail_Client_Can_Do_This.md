### Post by aysiu on 2007-01-12
> **NoVista said:**
> Umm, I didnt try that..

Ok, if i install TBird on Gnome, it will still find OE?
If you've already imported to Thunderbird on Windows, you can just copy C:\Documents and Settings\novista\Application Data\Thunderbird in Windows to /home/novista/.mozilla-thunderbird in Ubuntu.

P.S. Application Data and .mozilla-thunderbird are hidden folders.
