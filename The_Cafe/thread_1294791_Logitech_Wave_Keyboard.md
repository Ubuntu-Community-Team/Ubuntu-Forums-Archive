### Post by etnlIcarus on 2009-10-18
Went the, "Cordless Desktop Wave". It's the wave keyboard + LX8 mouse. 

The wave isn't listed when configuring your keyboard. I've currently got it setup as a, "cordless desktop LX-300", which gets some of the media keys working but not all of them. I don't use them, anyway, so I haven't sunk a great deal of time into finding the closest layout match.

The keyboard is quite comfortable and none of the keys have died on me yet, going on aftr 8 months of use. Haven't had to replace the batteries yet, either. After a while, key presses get a bit noiser but it's still quieter than your average $10 keyboard.


The LX8 Mouse is a pain in the ****. For some reason, possibly due to the receiver being shared between both the keyboard and mouse, HAL incorrectly identifies the mouse as an, "unknown button", making it impossible for me to configure it's behaviour.

[[IMG]http://img43.imageshack.us/img43/892/screenshotxi.th.png[/IMG]](http://img43.imageshack.us/img43/892/screenshotxi.png)

Played around with a few custom .fdi files but all I ever managed to do was break the mouse completely. Tried the HIDPoint driver, which gave me configurability but introduced input lag. Plus, even with the HIDPoint driver, the tilt-wheel still sucks; registering as a left/right arrow key press, rather than scrolling horizontally.
