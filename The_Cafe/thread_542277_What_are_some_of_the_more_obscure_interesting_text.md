### Post by Nekiruhs on 2007-09-03
[Scribes]("http://www.getdeb.net/search.php?keywords=Scribes") is my absolute favorite text editor, simple, clean. and functional. It has templates which are great for usability. If I type in a shell script
```
if
```It highlghts it, and when I press tab it expands it to
```
if [ [COLOR="MediumTurquoise"]condition[/COLOR] ]; then
       [COLOR="MediumTurquoise"] statements[/COLOR]

```
Then I type and it automatically goes into condition, then hit tab and you can change the statements. 

You can add templates too. It supports many languages (programming). I love it.:guitar:
