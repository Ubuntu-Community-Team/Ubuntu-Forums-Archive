### Post by Superkoop on 2008-08-02
I've used Songbird quite a bit, (made a few feathers for it also), but it's still pretty beta. Lots of features aren't in it yet, and it's just not ready as a default media player, like others are. And it is rather heavy on the system resources still, (rhythembox is at 59mb atm, where as Songbird is always as high as firefox, without even browsing the web).

By 0.8 or later, as long as they have the ram usage down to sub-75mb, and they have a majority of the features in, it should be ready to be used as a default media player. (I am guessing this will be by mid 2009 based on their current development progress)
