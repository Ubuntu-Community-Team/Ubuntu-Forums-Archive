### Post by t0p on 2008-10-17
++The computer security industry will change.  Linux's permissions system makes it difficult for viruses to spread.  So other forms of malware will become predominant (such as trojans that use *social engineering* to spread).

I'm not claiming that viruses will disappear.  A few proof-of-concept designs are kicking about, I'm sure they'd be improved and released into the wild. And some coders will find other ways to get round the permissions problem.  But this will certainly be a shock for the security industry.  Without all the windows-related disasters, there won't be any reason for so many security firms to exist.
