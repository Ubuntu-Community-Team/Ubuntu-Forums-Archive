**Dragonbite:** It is a lot easier if you use a simple theme to base it on. At first I tried Zen, but it didn't work as I expected. So I used [Foundation](http://drupal.org/project/foundation) and it worked great! If you want a fixed width theme you can try a different base theme.

It did take quite a while to get things looking right. If you have all the images ready it won't take too long ;)
