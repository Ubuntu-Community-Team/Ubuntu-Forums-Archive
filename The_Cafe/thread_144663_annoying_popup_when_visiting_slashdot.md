### Post by ComplexNumber on 2006-03-14
WinFixer is malware. read about it here:

****

> **Firefox Popup**

 The [Mozilla Firefox]("http://en.wikipedia.org/wiki/Mozilla_Firefox") browser is less vulnerable than [Internet Explorer]("http://en.wikipedia.org/wiki/Internet_Explorer") to initial infection by WinFixer. However, once installed, WinFixer is known to exploit the SessionSaver extension for the [Firefox]("http://en.wikipedia.org/wiki/Firefox") browser. The program causes popups on every startup asking the user to download WinFixer, by adding lines containing the word 'WinFixer' to the prefs.js file. The prefs.js file is located at:
 **Windows:** C:\Documents and Settings\_username_\Application Data\Mozilla\Firefox\Profiles\_profile_\prefs.js
 **Linux:** ~\.Firefox\Profiles\_profile_\prefs.js


[http://en.wikipedia.org/wiki/WinFixer]("http://en.wikipedia.org/wiki/WinFixer")
