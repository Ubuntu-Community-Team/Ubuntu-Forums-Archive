### Post by NightwishFan on 2011-07-28
A good example is ReactOS. You could call it 'fast'. However there is no way that it would beat Linux in raw benchmarks. It is simply light and possibly reactive, not 'fast'. Or even better it could be like saying Windows 95 is faster than Windows 7. Windows 95 could likely not even make use of advanced modern cpu features to make it run as well as it should.

Basically, benchmarks (even real world ones) only mean things to the folks that are looking for a specific result. A 'rule of thumb'. The rest is all marketing.
