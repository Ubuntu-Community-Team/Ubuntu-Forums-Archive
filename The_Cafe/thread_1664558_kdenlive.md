### Post by davidvandoren on 2011-01-11
It used to be very bugged but they worked a lot on it lately so it seems to be very stable now.

I like it very much since it has no problems dealing with all kind of different formates. 
Actually Cinelerra and KDenlive are the two video-editors I use a lot.

However, I am very excited about [lightworks]("http://www.lightworksbeta.com/") coming to Ubuntu. I have tried it on windows and it seams to have a lot of features. Still importing is not supported for most formates yet. Hopefully this will change once the source code is public and it's ported to linux. 

Then we will have a number of decent video-editors to choose from.
