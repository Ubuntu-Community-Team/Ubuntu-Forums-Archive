### Post by Qrk on 2005-12-15
I've had three OS's on at a time. In fact, they were Fedora, Ubuntu and Windows. I liked Ubuntu the best, so it was the default one.

I wouldn't share a home directory between Fedora and Ubuntu, though. I used a separate partition labled ~/documents in which I stored all my files between the two Linux Distros. 

While its fine to keep the same ~/ between upgrades; there is so much configuration stored in there (all those hidden files) that some of them are bound to conflict. Not to mention that Fedora defaults to different permissions in the ~/ than Ubuntu.

The only problem I ran into is that Fedora and Ubuntu could configure Grub to see windows, but had a harder time seeing each other. So everytime I reinstalled Fedora, I had to manually change grub.
