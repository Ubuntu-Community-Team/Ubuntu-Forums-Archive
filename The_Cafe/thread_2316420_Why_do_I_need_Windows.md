### Post by buzzingrobot on 2016-03-08
Potential reasons:

1.  You need/want to use specific Windows applications. 
2.  You need to create documents in Windows-specific formats LibreOffice, etc., cannot provide.
3.  You need/want to use Microsoft's online services in Windows.
4.  Cortana,
5.  Updating firmware in digital toys like cameras, etc. 

I'm sure others exist.  But, it all boils down to needing Windows because you need to do something with it you cannot do elsewhere.

(Running tax software on Windows can be something that that might merit keeping a Windows install around for once-a-year use.)
