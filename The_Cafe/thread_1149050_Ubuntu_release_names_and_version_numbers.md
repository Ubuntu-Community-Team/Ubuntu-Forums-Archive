### Post by bikeboy on 2009-05-04
I don't think there's any particular reason why 'W' was chosen first up, it was just a chosen codename. From then on the alphabetic system started.

As for the numbers, these are the official release names and coincide with the release date - 9.04 is equal to 2009, April. When 9.10 s released it will be October 2009. Ubuntu is released every 6 months in April and October, hence the .04 and .10 alternation.

### Post by PacSci on 2009-05-04
"Warty" came about as "Warty Warthog." Mark Shuttleworth, aka "sabdfl", wrote:

```
lifeless: how long before we make a first release?
sabdfl: it would need to be punchy. six months max.
lifeless: six months! thats not a lot of time for polish.
sabdfl: so we'll have to nickname it the warty warthog release.
```

"Hoary Hedgehog" was them trying to keep the "something somehog" theme. Then, they released Breezy Badger, which broke the theme. Dapper was just picked, then someone came up with the idea to sort it by the alphabet, leading to Edgy, Feisty, Gutsy, Hardy, Intrepid, Jaunty, and (coming soon) Karmic.
