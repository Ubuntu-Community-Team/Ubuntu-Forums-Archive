### Post by gnomeuser on 2010-05-24
Problems with the current Banshee podcast plugin includes:

Occasional problems updating feeds (feeds will have updates which Banshee for some reason doesn't register).

Occasional problem can be triggered where "download every episode" is no longer honored.

Slight problems with non compliant rss feeds (not really Banshees fault).

These should be addressed in the upcoming podcast-ng plugin, if not at least the refactored code should at least be in a much better position to get them fixed.

That being said of the mediaplayer podcast extensions I have tried Banshee currently is my favorite.

The only reason I haven't spent time investigating these issues in depth is that with podcast-ng coming it seems like time that could be invested better. Sadly merging has taken a while longer than expected.
