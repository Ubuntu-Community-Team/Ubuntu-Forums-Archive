### Post by Vadi on 2008-03-19
I'm a bit worried about the announce that'll be used for the 8.04 release describing the new kernel; because for the past few alphas, it has only been this:

> Linux kernel 2.6.24

Alpha # includes the 2.6.24-11.17 [WWW] kernel based on 2.6.24.3. This brings in significant enhancements and fixes that have been merged in the last few months into the mainline kernel, including:

    *dynticks support for amd64, bringing the same power savings already available on 32-bit systems to 64-bit laptops and desktops
    *the "Completely Fair Scheduler", a new process scheduler introduced in Linux 2.6.23 that provides improved interactive performance.

That's all nice and all, but how about the mention of more wireless drivers, this brining support to more wireless cards? Because when you first give Ubuntu a try, and your wireless isn't working, thhe last thing you care about is improved interactive performance. Some just give up on Ubuntu right there. The mention of more wireless cards being supported though will be more tempting to give it a try than these two points combined.
