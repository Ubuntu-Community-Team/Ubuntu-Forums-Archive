I can't answer your question about Omns question but to schedule it every 10 minutes you'd have to set up 6 entries within crontab which lets you schedule automatic tasks.

In the terminal type this in.

```
cron -e
```

Then we're going to add the entries into the crontab so they're executed at every 10 minute mark of every hour

```

10 * * * * "/path/to/script or just the command without quotes"
20 * * * * "/path/to/script or just the command without quotes"
30 * * * * "/path/to/script or just the command without quotes"
40 * * * * "/path/to/script or just the command without quotes"
50 * * * * "/path/to/script or just the command without quotes"
00 * * * * "/path/to/script or just the command without quotes"

```

### Post by hhh on 2011-07-14
@Avian, sure thing. The GTK is based on Clearlooks. The Xfwm theme is based on Totem. To remove everything from the titlebar, open Xfce's window manager, drag the buttons off and set the font-size to 0. I packed them both up with the wallpaper here...
[http://www.mediafire.com/file/ro724afa0haudg1/Carbon-Totem.zip](http://www.mediafire.com/file/ro724afa0haudg1/Carbon-Totem.zip)

Let me know if anything screwed up.

@mips, it's Carbon, part of the GNOME-colors Extra Variations pack available here...
[http://gnome-look.org/content/show.php/GNOME-colors?content=82562](http://gnome-look.org/content/show.php/GNOME-colors?content=82562)

I'd recommend installing the regular GNOME-colors icons as well as the gnome-colors-common icons fill the icon set out nicely. (Not shown in the screenshot, I installed them after I took it.)

-edit- I forgot to pack the xfce4-notifyd theme. It's just as easy to create it yourself. Create the folders and file ~/.themes/Notify/xfce-notify-4.0/gtkrc and copy/paste the following into it...```
style "smoke-window"
{
    bg[NORMAL] = "#505050" 
    XfceNotifyWindow::border-color = "#ffffff"
    XfceNotifyWindow::border-radius = 0.0
    XfceNotifyWindow::border-width = 1.0
}
class "XfceNotifyWindow" style "smoke-window"

style "smoke-text"
{
    fg[NORMAL] = "#ffffff"
    GtkWidget::link-color = "#a7a7a7"
}
widget_class "XfceNotifyWindow.*.<GtkLabel>" style "smoke-text"

style "smoke-button"
{
    bg[NORMAL] = "#6F727A"
    bg[PRELIGHT] = "#ffffff"
    fg[NORMAL] = "#ffffff"
    fg[PRELIGHT] = "#505050"
}
widget_class "XfceNotifyWindow.*.GtkButton*" style "smoke-button"
```

Switch themes via Xfce Settings Manager>>Notifications.

sudo add-apt-repository ppa:tiheum/equinox && sudo apt-get update && sudo apt-get install equinox-theme faenza-icon-theme --assume-yes

(any) & !(type=Gnome-terminal)
