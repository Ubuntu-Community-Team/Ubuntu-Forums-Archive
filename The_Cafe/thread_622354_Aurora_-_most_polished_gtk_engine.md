### Post by Mazza558 on 2007-11-24
Every gtk theme or engine I try seems really buggy or unpolished compared to Aurora, and using Aurora makes my desktops look smooth and professional, more so than even the legendary clearlooks, and in my opinion, KDE4's Oxygen theme. Every theme I've tried so far on gnome-look just pushes me back to Aurora.

A link for those who are unaware of Aurora:

[http://www.gnome-look.org/content/show.php/Aurora+Gtk+Engine?content=56438]("http://www.gnome-look.org/content/show.php/Aurora+Gtk+Engine?content=56438")

Some themes prevent me from using them because they use a background for items on the gnome bars, which is annoying if you want a transparent taskbar. Others don't let you change the window colours, meaning that if they're too grey/ <insert colour here>, you can't do much about it. My one and only pet peeve with it is the colour of tooltips, and AFAIK, there's no way to change it.

Anyone else love Aurora? :)

### Post by Polygon on 2007-11-25
here is a guide that i wrote on how to compile it:

download the theme

save it to your desktop, extract it to your desktop (right click > extract here)

you will then have a folder on your desktop. double click it, and then you have two more package files. Right click > extract on "aurora-1.2". you will get another folder called aurora-1.2

now open the terminal, type (ALL ONE COMMAND! Deviantart ruins the formatting)

cd Desktop/56438-Aurora-1.2.tar.bz2_FILES/aurora-1.2

After that, type

./configure --prefix=/usr -enable-animation

then

make

then

sudo make install (or sudo checkinstall if you want to make a deb file for easy removal, checkinstall must be installed first)

After that, go back to the aurora-1.2 folder, extract the package "gtkrcthemes", then a folder of the same name should appear, click on that.

you should see three folders now, each of these are themes. simply open up your home folder, press ctrl+h to show hidden files and see if the .themes folder exists (note the period infront). If it doesnt, create it, and paste all 3 folders inside of it. (if it already does exist just paste em inside)

THERE! now you can go to system > preferences > themes and select Aurora Midnight as your theme and it *should* work. to get the same window border as me, simply just go to my metacity/window border link, download it, and drag the package file over the themes window where you select themes, it should ask you to install and click yes. Then you can select it and have a nice pretty desktop
