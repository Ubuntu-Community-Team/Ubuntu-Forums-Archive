### Post by slimdog360 on 2006-10-20
fuji s6500fd. Im using it because the guy gave me the wrong camera, a better, more expenive camera.
I originally wanted the fuji s5600 which is quite cheap and takes a great picture, also has a massive zoom.
If money isn't really that much of a problem id recommend the s6500fd. The batteries last for ages and has lots of great features. 
It has a mechanically linked zoom on the lens barrel and the same with the manual focus. You can set about a gazillion settings so it certainly is worth it.
The 6500 is about $600 Australian, but thats because it just came out this month, and the 5600 costs around $370 Australian. So both are reasonably cheap.

They are both slr-like cameras, so they are shaped like a slr, but if you wanted a point and shoot like your old one then I guess Im no use.
