### Post by Atomic Dog on 2008-02-13
Seagate and western digital are equally good (or bad depending on how you look at it).  I believe they all park the heads nowadays.  

I did have a rash of 200GB WD drives blow up within a month (all dated dec 2003).  Very odd, but generally speaking I have seen them all blow up.

I have recently installed both seagate and WD drives without incident.  Too bad you can't find a model with firewire.  I noticed better performance through firewire, but USB 2.0 will be more than fine for your needs.

### Post by m0eman on 2008-02-13
I own the Seagate 320gb and so far I'm very happy with it. I did have a problem setting it up with linux. It has a power saving feature that only works with windows, so the drive would become inaccesible after 15minutes of no activity. But using a simple command in linux will disable the feature for good. Also, it came with an ntfs partition, so I recommend using a different type if you want to use it with linux. Other than that, I've been using it for about 2 months with absolutely no problems.
