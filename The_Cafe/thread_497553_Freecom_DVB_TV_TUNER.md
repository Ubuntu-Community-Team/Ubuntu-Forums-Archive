### Post by MonkeyBoy on 2007-07-10
I have a very similar card called an Avermedia DVB-T tuner.  The problem with this, and with that Freecom one too, is that they rely on their Windows program to install the firmware onto the device.  With a windows box this means that you install the program then the device gets its firmware which stays in the device until you unplug it. When you plug it back in it reinstalls the firmware and works again.  

With linux you have the problem that you generally have to find a different method of getting the firmware into the device.  This is a bit complicated and I never did it but relied on my windows partition and didn't unplug the device ever.

Kaffeine could use the Avermedia device and will scan for channels etc but could never find as many as the windows program.

Previously I had an internal TV Express card which was analogue but worked very nicely with tvtime.

I reckon you should look into cheap internal cards that don't use this lame firmware system.
