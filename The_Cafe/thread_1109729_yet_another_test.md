### Post by ugm6hr on 2009-03-29
Do you use a clipboard manager?

Remember, the default Gnome / Ubuntu has no clipboard manager, so if you copy from one application, and then close that application, the copied item in clipboard is then removed.  When you then paste elsewhere, you will get unusual results (sometimes previously copied data).

If you want to change this behaviour, install a clipboard manager.  If you already have one, it may have a bug.  I have used parcellite, which is a text only clipboard.
