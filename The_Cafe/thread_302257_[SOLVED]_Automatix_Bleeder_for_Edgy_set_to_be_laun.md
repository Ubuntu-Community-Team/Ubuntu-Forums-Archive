### Post by arnieboy on 2006-11-18
[Automatix2 Bleeder]("http://getautomatix.com/wiki/index.php?title=Automatix2_Bleeder_for_%28K%2CX%29Ubuntu_6.10_i386") for Ubuntu 6.10 (i386 only) has been released.
Currently it has the following features:
   1.  Installs ATi graphics drivers for select ATI cards
   2. Installs latest beta nVidia graphics drivers
   3. Installs Xgl/Beryl (for nVidia cards only)
   4. Installs Glipper (Gnome clipboard tool) and adds it to Gnome startup
   5. Installs GyachI (full featured Yahoo Messenger client)
   6. Installs PDF printing capability
   7. Installs Brasero (Disc burning utility)
   8. Installs Exaile (Gnome music manager)
   9. Installs xDVDshrink (DVDshrink for linux)
  10. Installs Gambas (Visual Basic IDE lookalike)
  11. Installs GnomeTweakUI (power tool for Gnome users)
  12. Installs Optimization scripts (Optimizes certain gnome utilities) 
To install (after making sure you have the correct repositories added) do from terminal
```
sudo apt-get install automatix2bleeder
```
