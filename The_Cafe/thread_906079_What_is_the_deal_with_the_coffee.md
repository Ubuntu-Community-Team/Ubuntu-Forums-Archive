### Post by barbedsaber on 2008-08-31
The easist way would probably to boot into a gparted livecd, find the ntfs partition (windows partition) wipe, and use the free space to make a new ext3 partition. then install mint onto the new ext3 partition. The fastest way to get gparted would probably be with a torrent, try [http://www.btmon.com/Applications/Unsorted/GParted-Clonezilla_2_0.torrent.html](http://www.btmon.com/Applications/Unsorted/GParted-Clonezilla_2_0.torrent.html)
hope that helps.
