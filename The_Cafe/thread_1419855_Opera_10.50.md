### Post by chessnerd on 2010-03-09
The day the final version came out I put it on every single Windows install which I have control over (6). Opera 10.50 works great on 7, Vista, XP, and even Windows 2000 (although it lacks the window border integration).

I think that Opera 10.50 is the best browser I have ever used. It's fast and feature rich, but yet light. It gets out of your way and let's your web page control most of the screen space thanks to it's new interface. Everything seems slick and clean and intuitive.

Opera mouse gestures have been a must have for me since I started using Opera a year ago, but now Opera Link is becoming the same way. How did I ever survive when I had to re-add bookmarks and Speed Dials to all 6 computers?

I'm really looking forward to 10.50 on Linux and I'm hoping for a release in mid-April or early June. Opera 10.10 is fine, but 10.50 is epic and I'd really like to use it on Ubuntu soon.
