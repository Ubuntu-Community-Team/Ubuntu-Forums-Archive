### Post by k64 on 2010-01-10
If Windows 7 Starter is crippleware, so is Linpus Linux Lite - only Linpus is crippleware Fedora!

To clear this up for you: Linpus Linux Lite is a Linux distro that was made by Acer for the Aspire One back around the time of the AOA110. Several things are crippleware about it: You can't customize the menus and panels, you can't change the desktop background, you are automatically logged in at all times and don't have the option to change it and make it prompt for a password. You also can sudo without a password!

On top of this, other netbook distros such as Ubuntu Netbook Remix give you the option to switch to a "Classic Desktop". Not Linpus Lite! You are stuck with the crippled-down UI, and have no options.

### Post by Warpnow on 2010-01-10
It can be modified. You can access a terminal via thunar "open terminal here" in the file menu. Then type sudo xfce-settings-show, and you can activate the right click menu.

From there, you can access synaptic, or just use the terminal to use yum to install from the fedora repositories.
