### Post by Omnios on 2023-10-01
I have been thinking about how Linux and Linux Desktops are represented within Movies and really like the idea of starting a new Linux Desktop environment based on something like what is seen in movies. So first off to work properly it will have to be very powerful and easy to use from the command prompt with the use of tab and other hotkeys to use the environment. It would have to be a hybrid of windowed and tab based desktop environments. So far thinking you would have the desktop with features like a hybrid text terminal would pop up in the middle of the desktop screen with the ability to send it back into a portion of the desktop. A big thing would be user contributed content similar to how Wikipedia works for content as a community. Also there may be things like a internet radio shall we say sort of an app for the environment that would parse data and info. Another aspect would be the development of a hybrid text terminal app to use various terminal programs with a custom graphical like interface to use there apps. This would also allow for new text terminal programs to be added to the project. Also could implement system reporting features allowing for custom system information as a powerful tool both in the background and in a pop up window/tab that may be both powerful and useful, generally something that is way more useful than windows task manager that can organize programs pertaining to essential core programs and programs that are not essential so as in a manner that helps a user manage there computer.    

 The hole idea is to develop a new type of desktop user interface that is highly usable and to achieve similar visual affects to what we have seen in the movies.
