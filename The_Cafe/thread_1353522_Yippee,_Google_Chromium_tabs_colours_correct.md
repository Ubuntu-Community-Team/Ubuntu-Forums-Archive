### Post by mybunche on 2009-12-12
> **cmat said:**
> I really want to try it but I'm under the impression that you must build it yourself.

No cmat. Add the Launchpad PPA to your third party software sources, install chromium via synaptic, then you'll get auto updates everyday which you can install if you want. I have been doing this for a couple of months now and I have seen the progress and it's been 100% stable for me. Also the daily build version has bookmark syncing now, just recently added actually. If you have a netbook you may find it performs better because it uses less CPU resources. Give it a go.

The launchpad PPA is:
[https://launchpad.net/~chromium-daily/+archive/ppa]("https://launchpad.net/~chromium-daily/+archive/ppa")
