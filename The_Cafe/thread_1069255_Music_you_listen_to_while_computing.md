and since I use mpd for my music player, I use mpc and can use the above command to randomly add artists/albums to the current playlist.

I wrote a small script to add 1 album at random, keeping a list of the last 40 added so they don't get repeated soon:
```
#!/bin/bash
cd $HOME
IFS=$'\n'
list=$(find Music/ -mindepth 2 -type d|sort -R|head -5)
for album in $list
do
    if ! grep "${album/#Music\//}" .mpcrnd > /dev/null
    then
        mpc add "${album/#Music\//}"
        echo "${album/#Music\//}" >> .mpcrnd
        trim=$(tail -40 .mpcrnd)
        echo "$trim" > .mpcrnd
        exit 0
    fi
done
```
It depends on mpd and mpc. The **-mindepth 2** can be changed to **-maxdepth 1** to add by artist instead of album.
That is, if your collection is sorted:
$HOME/Music/Artist/Album
