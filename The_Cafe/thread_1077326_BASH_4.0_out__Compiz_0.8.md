### Post by mcduck on 2009-02-22
What did you expect with Bash? It's a command-line shell, so the way it looks is meaningless. Actually most of the time you can't even *see* Bash at any point when using it. You only see the command prompt, which is pretty much what ever you define it to be.

If you are interested in what's new i Bash 4 then read this: [http://www.bash-hackers.org/wiki/doku.php/bash4]("http://www.bash-hackers.org/wiki/doku.php/bash4")

edit: If you want to change what you see when using Bash, and don't want to bother learning how to change the command prompt yourself, install Bashish. It's available in repositories and includes a bunch of themes for the command prompt with a simple ncurses-bases tool for changing them.
