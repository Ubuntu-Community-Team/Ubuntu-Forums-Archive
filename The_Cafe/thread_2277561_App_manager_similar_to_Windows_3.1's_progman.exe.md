### Post by CantankRus on 2015-05-11
The file browser **pcmanfm** has an applications entry in the side pane.
[ATTACH=CONFIG]261917[/ATTACH]

You could turn off the side pane and toolbar buttons and create a launcher to open pcmanfm to the applications menu...
```
pcmanfm menu://applications/
```
[ATTACH=CONFIG]261918[/ATTACH]
