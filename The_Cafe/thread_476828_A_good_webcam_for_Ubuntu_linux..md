### Post by _sAm_ on 2008-11-24
> are there any logitech or equviliant ones that would work in ubuntu.
If you go to logitech.com and search for Linux you will find several webcams that will work out of the box on Ubuntu. Logitech are writing for some webcams(with a _tiny_ font) that it will work under Linux if the **UAC** driver is enabled/installed.

You can read about UAC here; [http://en.wikipedia.org/wiki/USB_video_device_class](http://en.wikipedia.org/wiki/USB_video_device_class)

Ubuntu 8.10 use the Linux kernel that have support for UAC.
