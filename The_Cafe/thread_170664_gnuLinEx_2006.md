### Post by tseliot on 2006-05-05
gnuLinEx 2006 seems kinda cool:
[QUOTE=]The new release is based on Debian "sarge", but includes many updated components, such as kernel 2.6.16, X.Org 6.9.0, GNOME 2.14.1, OpenOffice.org 2.0.2, Firefox 1.5 and other popular software. It also ads support for many modern hardware devices.[/QUOTE]

It's a Spanish distro (from Extremadura) but you can select other languages by pressing F2 in the Installation screen (It has a graphical installer).

It has an interesting application a la Automatix, called "Actualizar LinEx", that lets you install all the codecs, java, flash, dvdcss, etc.
[IMG]http://cvs.linex.org/~rapisardi/quehaydenuevo2006/actualizarlinex.png[/IMG]

Another interesting feature is Aptzilla i.e. a Firefox extension that lets you install applications and games (there are more than 140 games!!!) directly from your browser.

It also sports quite an eyecandy control panel:
[IMG]http://cvs.linex.org/~rapisardi/quehaydenuevo2006/paneldecontrol.png[/IMG]

If you want to check it out, this is the link to the rest of the features (it's in Spanish):
[http://cvs.linex.org/~rapisardi/quehaydenuevo2006/]("http://cvs.linex.org/~rapisardi/quehaydenuevo2006/")
