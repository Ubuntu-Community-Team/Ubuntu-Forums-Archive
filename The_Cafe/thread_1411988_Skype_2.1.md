Can't say on about how long it's been since an update occurred.  

However, comparing Windows versions to Linux versions is comparing apples to bananas.  Each side of any application is developed independently of each other.  So while a Windows version of a program may be on version 4, the exact same functionality on the Linux side of things may be still version 2 (or even 1!).  This has to do with the OS change schedule, the development needs, functionality changes (if any), and the fact that *nix versions usually start much later than their Windows counterparts.
