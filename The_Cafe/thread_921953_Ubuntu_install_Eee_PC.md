### Post by awalsh on 2008-09-17
I would recommend [Ubuntu-eee]("http://ubuntu-eee.com"), for my eee 1000 it worked right out of the box. Or you could just use the latest copy of Hardy + [the custom kernel for eee pc]("http://www.array.org/ubuntu/index.html") which should get most features working nicely in case you do not not like the netbook remix which comes with Ubuntu-eee. 

I changed from Xandros because to me it felt like a child's toy with that interface, plus it was horribly outdated and a real pain to add new things to. Just my opinion though, im sure many people leave xandros on with no problems. If you are careful during the install it is possible to leave the restore partition, in place so if you do not like it you can always go back. I accidently erased the restore partition so no going back for me :) unless I buy an external cd drive
