### Post by kaldor on 2010-03-08
This is mainly aimed at Ubuntu, but could also apply to Linux in general.

When I first started using Ubuntu it was my first Linux experience. It was on Gutsy (7.10).

I loved it, but there were loads of problems...

1) Panels would move even when locked. People would often use my computer and try to move a window (like on Windows) by grabbing what they thought was the window. This would cause my panel and all its icons to clutter up. I'd be unable to reposition it, so I had to delete each icon and then move the panel back. 

2) LOTS of crashing. No idea why. Just lots. Too much. Stopped after I installed 8.10

3) Kernel updates regularly damaged my installations. 

4) Several times I contemplated going back to Windows due to so many issues with everything. Random firefox freezes, etc.

By the time 9.04 rolled around... all my woes disappeared. Everything "just worked". This didn't last long though, because now Karmic is pissy when it comes to sound and likes to cut out and mess things up. 

All in all though, Ubuntu is solid enough for me. It could be better, mainly with sound. I don't use any other OS except for OS X on a MacBook (needed to choose between Windows or Mac for a school course.. obviously I went for the UNIX choice). 

For me, Ubuntu has come pretty far in such a short time.

### Post by aysiu on 2010-03-09
Started with Ubuntu 5.04 (Hoary Hedgehog).

You couldn't edit the Gnome menu by right-clicking it.
There was no bootsplash.
No Wubi.
No *ubuntu-restricted-extras* metapackage.
No Software Sources (you had to edit the /etc/apt/sources.list file manually).
No Hardware Drivers in the menu.
No Add/Remove or Software Center (only Synaptic).
Separate installer and live CDs.
No Ubiquity (ncurses text-based installer).
No slideshow tutorial during installation.
No NTFS write support.

So, yeah, it's come a long way, certainly.
