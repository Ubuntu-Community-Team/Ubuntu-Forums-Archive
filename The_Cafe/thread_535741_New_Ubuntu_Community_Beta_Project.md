### Post by Beamerboy on 2007-08-26
Hello everyone,

I was talking with some of the community members on irc today about what to do with my ubuntu-users.org domain name.  I was originally going to setup a forum but as pointed out there are already many forums, so after some discussion I am considering the following.

Social Networking sites are taking the internet by storm at the moment and it would be nice to give something back to the Ubuntu community.  So next week once I finish my exams, I will be working on setting up a beta for ubuntu-users.org social network site.

I will offer people the chance to host their blogs under [http://username.ubuntu-users.org](http://username.ubuntu-users.org) and possibly have a [email]username@ubuntu-users.org[/email] email address.  It will be a beta project for now and will therefore be limited numbers until I can crunch  the hosting requirements, costs etc. but if you are interested in taking part in the beta please leave a response to this thread.

I cannot guarantee that everyone will get a chance to be involved in the beta but I do promise to put the names into a virtual hat so everyone has a chance to be involved (as opposed to first come first served).

It will initially be donation based and if enough donations come in to cover the costs it will remain that way.  However, if costs start to escalate I will put some banners on the main website to try and cover the shortfalls.  I do promise never to force banners on users individual blogs so we can keep the community as ad free as possible.

Think of is as kind of a facebook for ubuntu users with free email thrown in.

Let me know what you all think and remember to add your name to this thread if you want to be involved in the beta.

Have fun :)

Paladine

Please Digg this here: [http://digg.com/linux_unix/Ubuntu_Users_Social_Network_BETA](http://digg.com/linux_unix/Ubuntu_Users_Social_Network_BETA) so we can get the word out.
You can help get this news published on slashdot by clicking this link: [http://slashdot.org/firehose.pl?op=view&id=267879](http://slashdot.org/firehose.pl?op=view&id=267879)

UPDATE
I have just launched the User Interface Design Competition.  See this thread for details : [http://ubuntuforums.org/showthread.php?t=536593](http://ubuntuforums.org/showthread.php?t=536593)

UPDATE
If you want to be involved in the development in any capacity, whether it be marketing, programming, security, support, graphics, translations or anything else, please follow the instructions below.

Firstly thanks for expressing an interest, as stated this is going to be an Ubuntu Users project developed and maintained by Ubuntu Users.  Now that my schoolwork is out of the way for the next 6 weeks I have dedicated my life to the project and the first thing we need to do is get all the people who want to be involved, together in one place.

To this end please could you login to the freenode irc network and join the channel ##ubuntu-users.org

You can access the network either using xchat or another irc client and using the command:

/server irc.freenode.net

Later this week we will be having the first official meeting to setup the different teams and allocate people to them.  So please join the channel and more instructions will be forthcoming.

Regards

Paladine

### Post by JeevesBond on 2007-08-27
Don't put my name down for the testing (although it sounds great am lacking time :( ) but thought I might suggest you use Drupal as the platform for this: [http://drupal.org](http://drupal.org)

It's a fantastic CMS that can easily be tailoured to do what you need. It's very modular, so you can easily rip out the bits you don't want. Some sites using Drupal:

[list]
  [*] [The Onion](http://www.theonion.com/content/index)
  [*] [Warner Bros Records](http://www.theonion.com/content/index) (yes I got an RIAA joke into the [announcement thread](http://drupal.org/node/162162) :) ).
  [*] [Linux Journal](http://www.linuxjournal.com/)
[/list]

Granted, I don't know off-hand of any big sites that use Drupal's many social networking modules, but there's certainly an active [Drupal Social Networking Group](http://groups.drupal.org/social-networking-sites). You'll be needing to look at [modules](http://drupal.org/project/Modules/name) like: [buddylist](http://drupal.org/project/buddylist), [invite](http://drupal.org/project/invite), [user relationships](http://drupal.org/project/user_relationships) and [organic groups](http://drupal.org/project/og).

It comes with database and file handling abstraction (see: [http://api.drupal.org](http://api.drupal.org) ) so you shouldn't get SQL injection problems or people cracking the site using file uploads. Almost as important: it's standards compliant.

Anyway, thought I'd show Drupal off a bit as this looks like a great project and I really hate duplicated effort and the not invented here syndrome. :)
