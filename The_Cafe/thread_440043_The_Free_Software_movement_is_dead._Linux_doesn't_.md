### Post by az on 2007-05-14
> **leg said:**
> He also said that 67% of servers run Windows. I don't think that is true the last I heard 70% of servers were running nix.

Microsoft has the majority marketshare in business servers.  A business server is for running a database, or corporate email and filesharing.  Most Web servers run apache which is FLOSS - that's where FLOSS has a majority marketshare.  A percentage of those webservers run apache on Windows.

GNU/Linux has a lot to offer the business server world.  It can become expensive to buy licences for running you database server (you have to pay per client in some cases!)  I guess that the webserver market is more volatile - you can set up a webserver and run it for a year or two and then tear it down and start over again.  For a corporate databse, you just want it to work and be reliable to years.

Microsoft's email server (Exchange) isn't so good at actually doing mail, but it is excellent in terms of interoperability with other microsoft products (authentification, shared calendering, etc...)  So if your whole company is spread over several continents uses exchange to keep all their people in contact, they probably would be less keen to change.

But as the GNU/linux offering in this space gets more polished (it is as complicated to set up a mail/database server today as it was to set up a web server a few years ago) that will be a major market for linux adoption.
