### Post by perspectoff on 2007-08-11
I have both Xubuntu and Ubuntu (Gnome) on my quadruple-boot Win XP Pro, Debian, Ubuntu, Xubuntu.

I might use Xfce (Xubuntu) on a very old computer, but otherwise, no. The desktop is way too limited. 

I have tried to convince several friends to use Xubuntu (XFCE). None have agreed and all have preferred to buy an inexpensive new computer for under $300 and use Ubuntu rather than convert an old computer to Xubuntu. 

Is XFCE better than a command line environment? Yes, perhaps, but it is not going to make many converts from Windows. So what is XFCE's niche? As a thin client desktop, maybe.

That's how I intend to use it in a large network if there are some really old computers with minimal hard drives. On the other hand, it costs only about $50 for a new hard drive these days. It costs far more money in time to make XFCE work to a satisfying level.
