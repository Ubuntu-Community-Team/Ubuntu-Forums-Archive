### Post by drizek on 2006-06-05
Use ubuntu+enlightenment or another lightweight window manager. Just make sure you set up whatever that WM uses to launch apps to launch her web browser/chat client. And get more ram from newegg, 256mb is only about 30 bucks shipped.

Id also recommend using Opera over any other browser on a computer with those specs. Opera runs very well even on my 300mhz celeron.

And try the Elive enlightenment live cd. It allowws you to use either e16(stable) or e17(tons of eyecandy, pretty unstable). The good thing about it is that the whole thing comes preconfigured and you dont have to mess around with configuring the WM, which can be a pain with some of the more lightweight ones. I ran Elive, from the cd, on a 300mhz P2 with 64mb ram and it ran pretty decently all things considered. Once installed to the hdd it should fly on that computer, even without a ram upgrade(although it is still recommended). If you like it, then you can just install it to your hdd. It is debian based so it shouldnt be too different from ubuntu.

### Post by K.Mandla on 2006-06-05
[QUOTE=Donshyoku]It is a Celeron 650MHz, 128MB PC100, 6GB HDD, and a 24x CD-ROM.  [/QUOTE]
I would suggest Arch, but only if you're fairly handy with Linux on the whole, and well versed in hardware. I keep an old 750Mhz P3 as a streaming audio machine, and it runs unbelievably fast with an Arch version of xfce-svn. Boots faster than my Athlon XP 3000+. But it's a trick to install. 

Otherwise, I'd go with Slack. I think it does a better job handling older hardware, from what I've seen. 

Just as a side note, my experience has been that anything over 800Mhz can handle Xubuntu without too much sweat. Below that you need Slack, and below around 300Mhz, I'd go with either DSL, Puppy or Feather ... (but I've only used DSL much, so the other two might be better).

Just my $0.02. :rolleyes:
