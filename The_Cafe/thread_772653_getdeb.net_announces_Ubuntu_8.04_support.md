### Post by Vadi on 2008-04-28
Via the blog entry ([click]("http://blog.getdeb.net/2008/04/building-for-ubuntu-hardy.html")) getdeb.net is now making Hardy packages too! Just make sure it says "You are browsing software for Ubuntu Hardy" on top of the page, if you're on 8.04 now. If it doesn't say that, then click "select other version" and pick the appropriate one.

If you'd like to help in making packages, they've got a guide up here ([click]("http://wiki.getdeb.net/Building_Packages")).

This is just the time to thank them if you've ever gotten a .deb off getdeb.net too I suppose :)
