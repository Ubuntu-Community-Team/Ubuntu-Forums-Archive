### Post by murderslastcrow on 2009-12-23
The upgrades for the Wii tend to be added as updates, not new consoles. Or new controllers. Nintendo hasn't been known to release too many new IMPORTANT features in their revisions, and typically don't create too many versions of anything but their portables, without defining it as an entirely new platform (Nintendo DS, for example, has a few different sizes and one with a camera/webcam, but DS games work for each version).

I don't see how they would plan to improve the Wii really, except maybe a camera, which I don't see being on the console, but a third-party add on.

It just seems unlikely for a stay-at-home console from Nintendo to need a new version. They make more money from add ons either way.
