I've noticed there isn't much good quality and free budgeting software such as  'YNAB'. There are some options, like Buddi, but I didn't find any of them very attractive or intuitive. So I started my own. 

Now, I've finally reached a point where I'm happy to make it available online. It's not perfect, but I think it's pretty good. I use it myself all the time. It's written using C++ and Gtkmm 2. 

Please check it out and tell me what you think. 

[http://sourceforge.net/projects/dough/](http://sourceforge.net/projects/dough/)
