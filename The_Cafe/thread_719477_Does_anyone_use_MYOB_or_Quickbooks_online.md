### Post by daaussie on 2010-05-20
We have used MYOB on ubuntu successfully for a while now.
The way you do it is install WINE from the Apps/Software menu.
Then you double click the myob setup file (i do it by copying the CD content to the desktop first).
The program should install.
Then you create a desktop shortcut by going to Apps/Wine/Programs/MYOB
and right clicking "Add to panel or desktop".

There are only 2 problems I am trying to sort out.
We access the MYOB file on multiple dumb terminals/workstations from a server drive with SMB client. It doesn't lock the file, and hence we have multiple users on it. Sometimes multiple access corrupts the file, so we want to switch this ability off, but can't work out how too.

Also, the invoices don't print our logo on the invoice, all other text is fine, just the logo. I think its got something to do with Quicktime needing to be installed and am looking into this.

Good luck!
