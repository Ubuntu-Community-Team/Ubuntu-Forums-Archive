### Post by ubuntu-freak on 2008-08-24
[Linux Game Database](lgdb.org) is a site which lists native Linux games and more. The site gives full game descriptions and specifications, has screenshots of each game, and allows you to search for games, rate, and comment on them.
 
Edit: Thanks for posting the other Linux gaming sites, I'll post the links below to make it easier for everyone.
 
[Linux Game Database](lgdb.org)
 
[Tux Games](http://www.tuxgames.com)
 
[Ubuntu Gamers Arena](http://gaming.gwos.org)
 
[Linux Games](http://www.linuxgames.com)
 
[The Linux Game Tome](http://happypenguin.org)
 
[Play Ubuntu](http://www.playubuntu.com)
 
[Linux Game Publishing](http://www.linuxgamepublishing.com) 
 
[GarageGames](http://www.garagegames.com)
 
[live.linuX-gamers.net](http://live.linux-gamers.net/?s=home)
