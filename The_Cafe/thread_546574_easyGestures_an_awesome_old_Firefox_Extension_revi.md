### Post by RAV TUX on 2007-09-09
Anyway I tried this extension out years ago, and I thought I would revisit it.

Overall it's pretty awesome if you like to use your mouse for Web Browser short cuts.

This would be a great extension for a tablet computer also.

**[easyGestures]("https://addons.mozilla.org/en-US/firefox/addon/29")**


by default it comes with the middle click button to pop-up the easyGestures menu. In Linux you will want to change this, I changed mine to only pop-up when I depress the  'Ctrl' key and the left-mouse button.

easyGestures is nice because it is fully customizable.
