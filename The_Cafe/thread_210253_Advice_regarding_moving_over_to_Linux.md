### Post by krazyd on 2006-07-06
[QUOTE=avarice]I would like to ask if say, i were to completely switch over from Windows to linux would i face any major difficulty, and what about programs. Will i be able to instally most windows based games and programs on linux using the emulators available.

are they stable, and any other advice to give for switching over?[/QUOTE]Short answer? If the programme doesn't have a linux native equivalent (and many do :)) then you most likely will have trouble moving from windows. WINE certainly isn't stable for many programmes, and windows games (unless specifically designed for cross-platform compatibility, eg. UT2004) often don't work well if at all in linux.

Your best bet is to google around for the programmes you may need, and see if there are linux equivalents.

Oh, and apt-get is very nice for installing software, provided you know exactly what you're looking for. For example
```
sudo apt-get install firefox gimp inkscape openoffice
```
will install all these apps at once... nevermind they're installed by default.

Just to make sure: typically, when you're about to install something, you need to provide password. It's the password of the first account you created during installation, even if it wants 'root' password.  But it's probably better to stick to Adept at the moment. You can also install Synaptic (using Adept) and try it.
