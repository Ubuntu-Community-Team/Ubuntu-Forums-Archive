### Post by TheFu on 2018-09-11
64-bit is what I'd want for a huge number of reasons on any computer with 3G or more of RAM.  32-bit Windows doesn't have the same level of security as 64-bit Windows.  [https://www.howtogeek.com/165535/why-the-64-bit-version-of-windows-is-more-secure/](https://www.howtogeek.com/165535/why-the-64-bit-version-of-windows-is-more-secure/)

It is unclear which OS your other questions are about.  I can't say anything about Windows. Not my expertise.

Depending on where you are in the world, Linux desktops could be politically impossible. It has been suggested and deployed in a local school system here. About 4 months after the deployment, something failed somewhere and the local administration called some "computer experts", who decided that wiping the authentication server and installing Windows would be a good idea.  That broke everything, so rather than call for help from the phone number on the side of the servers, the Linux experts, they decided to wipe all the desktops and install Windows.  

Linux was never re-introduced, since clearly it was the entire problem from the start.  The "fix" included hundreds of new MSFT licenses and CALs and support contracts and Office software and A/V software, but no new hardware. This was before Google-Docs or other "cloudy" offers that worked for schools.   

More and more schools are using Chromebook hardware with school-oriented google services.  ChromeOS is tied to the hardware, so it can't be run on former Windows machines.

BTW, a hint for posting questions - split up huge walls of text into paragraphs. Makes it easier to read and more replies are likely if it is easier to read and clear.
