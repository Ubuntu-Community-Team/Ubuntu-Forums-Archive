### Post by murderslastcrow on 2011-03-25
Because the Cocoa framework, Aqua graphical framework, and Quartz display server don't work natively under Linux, it's impossible to run applications that need those under Linux ie. most OS X software.

I think it'd be awesome to get Aqua/Cocoa working on Linux and use it next to Qt4 and GTK+ 3, and it would probably be less work than Wine. But then you have to ask the question- outside of Adobe software, what Mac-specific software is there that Linux users would really want in the first place?

So yeah, stick to the Windows version of CS5, it works pretty well under Wine. Is there anything specific to the Mac OS X version you wanted?
