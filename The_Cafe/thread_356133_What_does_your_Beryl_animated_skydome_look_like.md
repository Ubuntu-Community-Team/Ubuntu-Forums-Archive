### Post by FyreBrand on 2007-03-22
> **Akrash said:**
> Hey,
I'm trying to get the skydome. Can someone link a how to for the skydome.Setting the skydome is really easy.

First download a skydome.  You can search the beryl forum and you will find threads with several skydomes.  Or you can go to kde-look.org and do a search for 'skydome' and a couple will pop up.  There is one really cool galaxy skydome at kde-look that I like.

Next you need to configure beryl to use the skydome:

1. Open Beryl Settings Manager (right click the beryl gem in the system tray)

2. Select the Desktop Section -> Desktop Cube

3. In the Desktop Cube settings area there are several tabs.  One of them is labeled 'Skydome'.  Select it.

4. There are 3 settings you need to enable or set in the Skydome section:
[INDENT]a. Click the checkbox labeled: Skydome  -- It's the top most checkbox in the section
b. Browse to your downloaded skydome graphic -- This is the second configuration option on the page.
c. Click the checkbox labeled Animate Skydome: This checkbox is just below where you set the location of the graphic file.[/INDENT]

5. Transparency:  Click the tab labeled transparency.  Set the 'Opacity During Move' to a low setting.  Mine is set to 25.  Anywhere between 20 - 40 should work okay.  Just experiment and find what suits you.

6. Zoom Out: In the Desktop section, below Desktop Cube, is a section called 'Rotate Cube'.  Select it.

7. Select the 'Behaviourl' tab if it isn't selected already.  There are two subsections in the tab frame: 'Edge Flip' and 'General'.  Open up the 'General' subsection.

8. The 'Zoom' setting is the fourth from the top.  Set it to 15.  Just like transparency you can set this to a lower number.  Just experiment and find the setting you like.  The higher the number the smaller the desktop is when you rotate your desktop and the more skydome you see.

Well that should be it.  Enjoy.
