### Post by whee on 2007-04-27
Look at the Platinum lists and Gold lists on [http://appdb.winehq.org](http://appdb.winehq.org) .

- Platinum applications in Wine: applications which install and run flawlessly on an out-of-the-box Wine installation

- Gold applications in Wine: Applications that work flawlessly with some DLL overrides or other settings, crack etc.


If i'm not mistaking there is also a silver and bronze list. (you get the idea)
