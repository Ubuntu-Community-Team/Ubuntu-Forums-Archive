### Post by DoctorMO on 2006-07-23
One of my friends wanted to create both video and normal iso's this is a quick script I pulled together in perl (requires libgtk2-perl) which hooks into the Gnome folder mime-type to create an iso of any folder. it detects if it's got a VIDEO_TS directory and creates a video/dvd iso.

If there is interest I will fix this script up more and it could do with some testing on other machines. (doesn't even have a make file yet, run sudo ./install && ./install-local)

[http://www.postmaster.co.uk/fs/doctormo/Public/isocreate.tar.bz2](http://www.postmaster.co.uk/fs/doctormo/Public/isocreate.tar.bz2)
