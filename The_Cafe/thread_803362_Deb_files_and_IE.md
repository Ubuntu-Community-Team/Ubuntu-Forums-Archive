### Post by Zafrusteria on 2008-05-22
This corruption could be caused by the process of copying from MS to the memory stick. If the copying program is not treating the deb as a binary file and is doing a CR/LF conversion. This happens with some FTP clients.

IE 5 or 6, cannot remember which, had a known bug where it added a .txt to the file name this could account to the above problem.
