### Post by LaRoza on 2008-06-17
> **Redrazor39 said:**
> Also, do I install things in Kubuntu the same way as ubuntu? There's an add/remove thing and then I can use .deb files? Or is there a different package format that has it's own installer?

It is the same. I use the command line anyway, so I don't know about the GUI front ends.

If you want to check them out on Ubuntu run:

```

sudo apt-get update && sudo apt-get install kubuntu-desktop

```

Or, for the smaller KDE package (doesn't have all the Kubuntu apps):

```

sudo apt-get update && sudo apt-get install kde-core

```

You select what you want when you login (under "Sessions")

Kubuntu/Ubuntu/Xubuntu are just variants of the same OS. You can install the same software on each.
