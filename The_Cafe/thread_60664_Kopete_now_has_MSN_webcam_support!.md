It's not difficult to use Kopte svn together an 3.4 KDE following this: 

1) Install libjpeg-devel 
2) 
svn co -N svn://anonsvn.kde.org/home/kde/branches/KDE/3.5/kdenetwork 
cd kdenetwork 
svn up kopete 
svn co svn://anonsvn.kde.org/home/kde/branches/KDE/3.5/kde-common/admin 
make -f Makefile.cvs 
./configure --enable-debug 
cd kopete 
make 
su -c "make install"
