### Post by thisllub on 2009-06-12
I run it on my laptop because Linux doesn't do power management right (hangs without ACPI=OFF etc..) and BSD doesn't do VirtualBox.
When a VLC port exists and I can get OpenBox to run I won't really miss anything. the 2009.06 release fixed Gvim but I am still stuck with Gnome.
OS seems to handle heavy loads much better than Linux too. It may be because I haven't rebuilt a server priority kernel but I find that once Linux fills my 2GB of RAM it bogs down, sometimes so badly I need a power cycle to get going again.
Firefox is the chief culprit.
When you run Solaris and BSD you realise a whole set of problems with Linux that you never knew about.
