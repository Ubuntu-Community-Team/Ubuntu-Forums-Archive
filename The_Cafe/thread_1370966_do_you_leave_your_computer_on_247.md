### Post by RandomJoe on 2010-01-03
I have some that stay on (file server plus a few other things on one, MythTV box on another, a third is my "off-grid" low-power computer) and the ones I use directly get shut off.

The difference in power depends on the design of the system, of course.  My file server is a P4 2.4GHz w/ five HDDs, the Myth box is a similar P4 system with three HDDs.  Both of those run about 100W idle, or 2.4kWh per day.  Electricity here is roughly 10 cents per kWh, so about 50 cents per day to keep both of those running 24x7.

The off-grid machine is an Atom system, pulls about 25W so only 600W per day.  No cost on the electric bill, it runs off my solar system.  Of course, the up-front cost for the solar system was rather high!
