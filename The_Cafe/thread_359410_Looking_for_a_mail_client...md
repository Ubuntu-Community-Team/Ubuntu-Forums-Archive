### Post by Paerez on 2007-02-12
IMAP's advantages over pop3:
[http://en.wikipedia.org/wiki/Imap#Advantages_over_POP3](http://en.wikipedia.org/wiki/Imap#Advantages_over_POP3)

gmail doesn't do imap:
[https://mail.google.com/support/bin/answer.py?answer=10339&query=imap&topic=&type=f&ctx=search](https://mail.google.com/support/bin/answer.py?answer=10339&query=imap&topic=&type=f&ctx=search)

Read through gmail's help for more info.

You can have gmail gather the mail from the server through pop and remove it from your server (putting it only on gmail). Then gmail can manage deleting and storing etc.

EDIT: Forgot to mention gmail can also send using your address, so it will say From: [email]You@yourserver.com[/email].
