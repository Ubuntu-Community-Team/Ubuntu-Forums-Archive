### Post by RAV TUX on 2006-06-18
Patrick Verner has announced the availability of an updated version of [GParted]("http://distrowatch.com/gparted") LiveCD: "GParted LiveCD 0.2.5-1. I think this might be the best version of the live CD released so far. Many issues have been fixed and I spent some time making it look better. Junel Mujar made a very cool wallpaper for the desktop. Changes: updated to Linux kernel 2.6.16.20, Parted 1.7.1, e2fsprogs 1.39, GTK+ 2.8.19, Fluxbox 1.0rc, udev 094; Added xfce4-panel 4.3.90.1 to Fluxbox, Thunar 0.3.0beta1; rebuilt X.Org to be smaller; the source for the CD is located in /usr/src/." Read the rest of the [changelog]("http://sourceforge.net/project/shownotes.php?release_id=425366&group_id=115843") for further information. [Download]("http://prdownloads.sourceforge.net/gparted/gparted-livecd-0.2.5-1.iso?download"): [gparted-livecd-0.2.5-1.iso]("http://surfnet.dl.sourceforge.net/sourceforge/gparted/gparted-livecd-0.2.5-1.iso") (29.4MB, [MD5]("http://sourceforge.net/project/shownotes.php?release_id=425366&group_id=115843")).
