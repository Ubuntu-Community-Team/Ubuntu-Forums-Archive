### Post by kurt18947 on 2012-04-13
I needed to scan about 150-200 pages into several files and  save them to CD.  I started out with SimpleScan.  It worked but was pretty slow.  The job was multiple pages/file and I wasn't sure how to get Xsane to do that.  I remembered I had gscan2pdf installed. Started it up and it worked beautifully.  I have a Brother MFC-6490CW and thought I was going to be stuck with Brother's Windows software.  Gscan2pdf was as fast as Brother's Windows scanning software and for file management Nautilus >>>>>> Windows 7's file manager. 

Xsane and SimpleScan work very well for a sheet or two but don't seem like a very good choice for larger archiving jobs.
