### Post by Xbehave on 2009-11-20
how much ram are you using now, if it's more than 1GB(or 896 if you have onboard gfx) it means that:
* programs were doing extra work to avoid using more ram
* you were using the swap more

Irrespective
* you now have more catching to reduce io usage
     - apps often eat CPU while waiting for disc data
     - whenever discs are accessed the CPU has to stay awake to get the data back
* (if you have on-board gfx) your bios has probably given it more ram (probably doubled it to 256), this means your GPU can do more work without passing data to/from the cpu
