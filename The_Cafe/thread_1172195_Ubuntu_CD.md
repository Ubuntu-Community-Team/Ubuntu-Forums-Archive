If you don't mind the download of gnome (I'm guessing you might, if you ordered CD's) you can run (in a terminal):

sudo apt-get install ubuntu-desktop

That command will install the full default set of Gnome and all the apps that come with it in Ubuntu.

And Kubuntu generally ships faster since there is less demand for that cd.
