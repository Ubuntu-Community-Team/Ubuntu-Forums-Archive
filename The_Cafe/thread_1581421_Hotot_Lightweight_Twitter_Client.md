### Post by jusuchin85 on 2010-09-24
Has anyone tried the new Hotot? It's a Twitter client; lightweight and has extensions for future extensibilities. It's still new though, but I believe it has the potential to become better as it matures.

First read about it from [WebUpd8]("http://www.webupd8.org/2010/09/hotot-new-lightweight-twitter-client.html"). Decided to try it out since there's no other Twitter client that can satisfy me (well, apart from [TwitterGadget]("http://www.twittergadget.com/"), that is). Also, one of my Twitter friends have tried it himself and I too agree with him (about the future of it).

Here is a screenshot of how the interface looks like:
[[IMG]http://img168.imageshack.us/img168/3958/screenshotglz.th.png[/IMG]]("http://img168.imageshack.us/i/screenshotglz.png/")

Installing:

```
sudo apt-get install python-webkit python-notify python-keybinder mercurial python-distutils-extra
``````
cd
``````
hg clone https://hotot.googlecode.com/hg/ hotot
``````
cd hotot/
``````
python ./hotot/hotot.py
```**UPDATE:** There's a PPA for Hotot: [https://launchpad.net/~jimmyxu/+archive/hotot/]("https://launchpad.net/%7Ejimmyxu/+archive/hotot/") (Thanks, [FuturePilot]("http://ubuntuforums.org/showpost.php?p=9886234&postcount=2")!). You can also try to install from it. It still is the same version.

If any of you are trying this out, give a shout out here! :)
