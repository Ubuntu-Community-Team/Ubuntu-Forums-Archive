A little explanation is in order:

> **BB**

**Group/Artist**

AA Group

**Party**

**Description**

demo only use Ascii art from various library (fractal , 3D torus , photo ..) 
with sound . A very good Ascii art Demo . The demo focus on Realistic ascii art , but have some schematic ascii art include (like invaders ascii style )
Really great . there is an article on wikipedia too showing some screenshot : [http://en.wikipedia.org/wiki/AAlib](http://en.wikipedia.org/wiki/AAlib) 

**URL**

[http://aa-project.sourceforge.net/bb/](http://aa-project.sourceforge.net/bb/)

**Compatibility Notes**

I run on debian linux and derivative (knoppix , dreamlinux ... ) without problem . it was on repository . Just apt-get it . here is the compatibility from official webpage : 
    * Portable demo - supports many unixes + DOS
          o Accelerated support for Linux console and X11
          o 50 VGA / SVGA modes
          o Two dual monitor modes
          o Sound supported for
                + PC-speaker
                + Gravis ultrasound
                + Sound scape
                + Sound blaster
                + Linux
                + SGI
                + SUN
                + Digital Alpha
                + AIX
                + HP-UX 
    * Many resolutions
    * First demo running at two monitors! 
  Stuff required to compile BB
      * UNIX version
          o AA-lib 
    * DOS version
          o DJGPP
          o AA-lib
          o vga
          o text
