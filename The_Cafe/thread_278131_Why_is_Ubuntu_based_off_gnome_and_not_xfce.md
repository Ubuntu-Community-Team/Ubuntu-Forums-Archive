### Post by IYY on 2006-10-15
XFCE is very nice, but not nearly as feature-complete as Gnome. Some people are fine with that -- especially if they are using older machines -- but it is not what the general public wants. You have to draw the line between 'bloat' and 'feature rich' somewhere.

If you want a truly snappy desktop, use IceWM (as I do on some machines). It is far snappier than XFCE, and leaves Gnome in the dust.

### Post by IYY on 2006-10-15
You really don't have to get a new distro for every window manager. To get XFCE from Ubuntu:
```

sudo aptitude install xubuntu-desktop
```

To get IceWM:

```
sudo aptitude install icewm icepref iceconf
```

To get Fluxbox:
```

sudo aptitude install fluxbox
```

Ubuntu is not equal to Gnome!
