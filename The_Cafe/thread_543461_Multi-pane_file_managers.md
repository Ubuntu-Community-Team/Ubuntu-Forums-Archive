### Post by epimer on 2007-09-05
Part of my work over the past few weeks has involved sorting masses of files in unsorted directories (3 years' worth...!) across the networked machines into some semblance of order. I've been using the Windows 2k/XP machines at work for this, and found [mtexplorer]("http://www.mtexplorer.com/") to be incredibly useful - it's a lightweight Explorer alternative which can have up to 6 panes open.

I'm working on my own laptop now, and would like to find a file manager with similar functionality for Ubuntu (GTK preferred, but not essential); seamless handling of network shares is a necessity. I know there are a few 2 or 3 pane file managers around, but what about more?

### Post by qalimas on 2007-09-05
Konq can have as many open at once as you'd like (I've done up to 40 just to show that off).

CTRL+T splits horizonally
CTRL+L splits vertically
CTRL+R closes current focused pane

One more reason I love Konqueror...
