### Post by finferflu on 2008-04-21
fuscia, 
you *can* use Youtube with Elinks. I have created a [little script](http://openroom.shellpenguin.com/index.php/2008/01/01/video-di-youtube-senza-il-flash-plugin/) that uses youtube-dl, and you can incorporate it within Elinks. If you can read Italian (or if you can use Google Translator), I have written a [little howto](http://openroom.shellpenguin.com/index.php/2008/01/02/smanettando-con-elinks/) (it's split into 2 parts, one is for copying the current URL to the clipboard, and the other to watch the videos). 

As for the colours, you can toggle between the original page colours and the default Elinks colours using the key "%". 

Yes, Elinks is an advanced web-browser, able to compete with Firefox, for that matter :D 
I use w3m as a wrapper, which is what it is, in the end, and I use within scripts mainly.
