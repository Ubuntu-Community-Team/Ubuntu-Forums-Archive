### Post by Architeuthis on 2006-11-16
My maths teacher handed me over a special cd box today (it contains 2 cd's) This box is being spread and composed by the Flemish government. The letter included with this cd says: "this cd is made to help schools choose qualitative software":
[[img=http://img369.imageshack.us/img369/6770/00001fb5.th.jpg]](http://img369.imageshack.us/my.php?image=00001fb5.jpg)

For those of you who can't read dutch, the cover says "free software for education" (free as in freedom of speech). 
One cd includes opensource software for windows the other one includes....

[img=http://img49.imageshack.us/img49/5927/00003hz9.th.jpg](http://img49.imageshack.us/my.php?image=00003hz9.jpg)

Yes, this box includes ubuntu 5.10! (yes my government is a bit slow)
Whats even more special is that these cd's are delivered to ALL schools in Flanders! This is a big opportunity for ubuntu to become well known and gain market share.

Ubuntu FTW!
