### Post by cariboo on 2010-08-16
Just create a regular bug against rhythmbox @ bugs.launchpad.net. You'll have to create a launchpad account. Once that's done, press Alt-F2 and type:

```
ubuntu-bug rhythmbox
```

Check to see if the bug has been created already, if not fill in the details, and you're done.
