### Post by gnomeuser on 2009-11-08
The way Banshee is designed and implemented ensures that the frontend is merely a plugin that can be replaced. In fact the current Cubano implementation (Banshee's netbook UI) is around 2000 lines of code.

I presume thus that since all the library functionality is granted by the underlying Banshee technology one could swing into action using the Qyoto QT# bindings and write a simple KDE friendly frontend for it.
