### Post by thisllub on 2008-07-21
#1 This is an Acer Travelmate 5610.
It never worked properly with the pre-loaded XP so I deleted Windows altogether. I ran Ubuntu for about 10 months now I use Arch.
Excepting the ACPI issue the computer has been 100% error free for 7 months.

Good points;
17" screen 2GB ram full size keyboard,wireless built in, value at time of purchase 18 months ago.


Bad points;
ACPI doesn't work with Linux so battery lasts 20 minutes max.
Curiously there is no ACPI problem with BSD but the 3945 network driver for FreeBSD is flaky and unreliable.
Wouldn't buy again.

#2 My daughter has a Lenovo R61 with Ubuntu Hardy. / Vista.
2GB ram 15" screen.
At just over 1/3 the price of the Acer this was a great buy.
I shrunk the Vista partition and installed Hardy at her request the day he got it. Runs beautifully and gets close to 4 hours battery life.

I am convinced that a 15" screen is the maximum needed for a laptop.
The difference in price between a 15" model and a 17 will get you a 24" monitor as well and you really only need the bigger screen for using it in an office.
