### Post by OffHand on 2010-02-18
This version is mostly a bug fix release. The network code has been made more robust to work in different processor types and operating systems and there is a now a wizard to help new users configure Nicotine+ more easily. The minimum PyGTK version required is now 2.12.

[Complete changelog]("http://www.nicotine-plus.org/browser/tags/1.2.15/doc/CHANGELOG")

[Download]("http://sourceforge.net/projects/nicotine-plus/files/nicotine-plus/1.2.15")

And as sugar on top we now provide a .deb installer  :popcorn:
