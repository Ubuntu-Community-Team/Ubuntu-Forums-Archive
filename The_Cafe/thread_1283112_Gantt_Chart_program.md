### Post by jonian_g on 2009-10-05
There is OpenProj. It is better than Microsoft Project and it can also handle Primavera file formats.

From site:

"The OpenProj solution has Gantt Charts, Network Diagrams (PERT Charts), WBS and RBS charts, Earned Value costing and more."

It is a very good application. I can dare to even compare it with Primavera Project Planner. We use it at my job and we are absolutely satisfied with it.

It even has a .deb package available for download.

[http://openproj.org/product-overview](http://openproj.org/product-overview)
