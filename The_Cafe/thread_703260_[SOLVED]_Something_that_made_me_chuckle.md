### Post by aaaantoine on 2008-02-21
> **stooshbunutu said:**
> lol, I think we all know it blaitently is

Nah, a Windows *Emulator* would simulate the entire Windows environment.  Wine is a "translation layer".  It reads the binary that was programmed for Windows and translates it for use in the native environment (Linux, BSD, etc).

There are no real Windows Emulators per se, except for virtual machines (which emulate computers rather than a single OS).

GTK+'s name doesn't come into GNOME, and GNOME has nothing to do with GIMP's name.
