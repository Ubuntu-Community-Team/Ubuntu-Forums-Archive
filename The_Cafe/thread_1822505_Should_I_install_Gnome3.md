### Post by IWantFroyo on 2011-08-10
Gnome 3 is a pretty good DE, but it has faults.

Pros:
-Pretty
-Window Snap (I know, but at least it has it by default).
-Easier to see what's what when you have more than one of the same program open.

Cons:
-You need a good graphics card, and a lot of RAM if you multitask.
-It isn't very efficient if you have a lot of windows open.
-Not very customizable.
-If you have limited screen estate, the two thick bars at the top will likely get on your nerves.
-If you have a big screen, it will be more of a pain with the "hot spot" on the left side and the window controls on the right.

All I can say is, try it. If you don't like it, don't use it.

### Post by ninjaaron on 2011-08-11
I've use Gnome3 on Fedora 15, Ubuntu 11.04, and Ubuntu 11.10 alpha.  Obviously it runs the best in Fedora, but it also works well in Ubuntu 11.04.  It just completely breaks gnome classic and Unity, but it's not hard to remove with [FONT="Courier New"]ppa-purge[/FONT].  It's still kinda broken in 11.10 (as are a lot of things)

### Post by sanderd17 on 2011-08-11
I've first tried Fedora and Gnome-Shell, but I couldn't get used to the package management in Fedora. Then I discovered gNatty (Gnome-Shell natty remix by Cbowman57 - see above), and it ran fine until I did a partial system upgrade (those are always dangerous to do). The result was that I was left without any kind of network (not wired or wireless). Now I'm on a backup Mint installation and waiting until I have enough free time to Download and install a new GS OS.

The time that I used gNatty, it worked flawlessly. I believe GS is intuitive, fast and very customisable (not always via graphical menus like in the CCSM, but that will come). You do need a well supported graphical card to use GS.

Maybe I would suggest to wait until Oneiric comes out. By then you will have a lot more extensions and themes, and GS won't give that much problems with Ubuntu (although my problem wasn't GS related, I've also seen it with a non-GS installation).

GS is already my favourite DE, but the number of themes and extensions have to grow and the Ubuntu support has to become better before I advise GS to the majority of people.

@nijaaron: on gNatty I had Unity and Gnome-classic and Gnome-Shell sitting next to each other (some magic done by cbowman :P )
