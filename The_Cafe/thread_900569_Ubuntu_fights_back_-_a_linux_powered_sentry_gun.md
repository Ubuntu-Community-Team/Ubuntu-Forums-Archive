### Post by cfinger on 2008-08-25
I thought this might be appreciated here- [http://zero-op.com/mercenary](http://zero-op.com/mercenary)

[img]http://zero-op.com/images/had_sentry.jpg[/img]

We've designed and implemented a paint ball sentry gun using a stripped down version of ubuntu, C++, and Intel's OpenCV computer vision library.  It boots off flash memory, which we plan to leave unencrypted so you could ssh in to your turret and play around if you wanted to.

USB support is left in the kernel so you could add wifi, etc.  At some point it would make for a cool experiment in mesh networks :)
