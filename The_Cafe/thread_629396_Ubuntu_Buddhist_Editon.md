### Post by amerikkanu on 2008-01-09
Just in the way of an fyi, if this project goes forward...  I've had issues with Sanskrit input in linux for a while now and finally solved the remaining nagging issues in [this]("http://ubuntuforums.org/showthread.php?t=646207&highlight=sanskrit+input") tips 'n trick post.  Basically, the default phonetic Devanagari IME (scim - ITRANS) is problematic in a few respects (e.g., typing "rh" yields &#2353;&#2381; instead of &#2352;&#2381;&#2361;&#2381; , etc.)  So I tweaked that input method and streamlined various character combinations.  I also created a new SCIM input method for Sanskrit transliteration, which is as intuitive as any I've used on windoze, etc.  It can easily be extended so that all the relevant languages can be transliterated with the same input method.  (Since both input methods go beyond ITRANS, they ought to have a table documenting their character combinations--which would be easy to draw up from my post--if they're included in your distro.)

Good luck with your project, and if there's anything I can help out with, Sanskrit-wise, feel free to contact me.  &#2358;&#2369;&#2349;&#2306; &#2349;&#2357;&#2340;&#2369; &#2404;
