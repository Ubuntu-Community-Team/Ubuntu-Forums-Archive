### Post by m83 on 2008-06-18
I also use Azureus and when it changed to Vuze I was confused and found it really bloated. Then I found out that you can change the default UI to be like the old Azureus. All you have to do is go to Tools->Options->Interface->Start->Display Vuze UI Chooser and select Classic. That way the Vuze content is not loaded and Azureus is fast and lean again.
