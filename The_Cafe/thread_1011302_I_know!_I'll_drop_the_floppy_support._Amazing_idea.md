### Post by Dr. C on 2008-12-14
Since there may be users who are looking for floppy support in intrepid ibex this has worked for me (for 3.5 in 1.44MB floppies, I have not tried this with the 1.2MB or 360KB 5.25 in floppies) : 

```
gksudo gedit /etc/modules
```

and add to the file the following line

```
floppy
```

save and restart

This is from  Michael.Godawski's post in [http://ubuntuforums.org/showthread.php?t=1007271]("http://ubuntuforums.org/showthread.php?t=1007271")
 
As for the last time I used the floppy, about a month ago when I moved some data from a 24 year old 286 with 1MB of RAM to an 8 month old Quad Core with 8GB of RAM using a 3.5 in 1.44MB floppy disk. The floppy remains as a link with the past
