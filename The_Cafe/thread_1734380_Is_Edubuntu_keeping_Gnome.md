### Post by Artemis3 on 2011-04-20
Try not to be confused ;) Ubuntu 11.04 comes with Gnome 2.32; Gnome Shell happens to also work with that. But in the next version (11.10) if Ubuntu uses Gnome 3, only Gnome Shell is available. Since they don't like it, they made Unity to replace Gnome Shell, but the "Classic" shell will be gone when they ship with Gnome 3.

For that reason, I'll just use Xubuntu because XFCE is the closest to the gnome 2 classic shell.
