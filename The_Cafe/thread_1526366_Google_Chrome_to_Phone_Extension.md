Read about Chrome-to-Android: [http://techcrunch.com/2010/05/23/chrome-to-android-push/](http://techcrunch.com/2010/05/23/chrome-to-android-push/)

Download Chrome-to-Android: [http://code.google.com/p/chrometophone/downloads/list](http://code.google.com/p/chrometophone/downloads/list)

Step 1: Get Chrome browser ([http://www.google.com/chrome/](http://www.google.com/chrome/))
Step 2: Make sure you have Android 2.2 on your phone
Step 3: Go to the Chrome-to-Android Download page ([http://code.google.com/p/chrometophone/downloads/list](http://code.google.com/p/chrometophone/downloads/list))
Step 4: Install the Chrome extension to Chrome
Step 5: Install the Android app on your phone

Updates:
For Firefox Users, 

You can use the following extension: [https://addons.mozilla.org/en-US/firefox/addon/161941/](https://addons.mozilla.org/en-US/firefox/addon/161941/)

BONUS Feature: If you have text selected in your browser when you send it to your Android  phone, that text is automatically copied into the clipboard on your phone.[http://www.google.com/support/forum/p/android/thread?tid=75f3b38794a17cc8&hl=en](http://www.google.com/support/forum/p/android/thread?tid=75f3b38794a17cc8&hl=en)




> Google Chrome to Phone Extension is a project consisting of a Chrome Extension, Android App, and supporting AppEngine server that enables users to send links from their Chrome desktop browser to their Android device using Android's Cloud to Device Messaging service.

Features:

Send links to browser
Google Maps links launch the Google Maps app.
Selecting phone numbers in Web pages, launches the dialer with the number pre-populated.
Selected text populates the Android clipboard (long-press on text boxes to get paste option)[http://code.google.com/p/chrometophone/](http://code.google.com/p/chrometophone/)

I just installed this, which is available on Google Chrome and on Android 2.2 or later.

Anybody else using Google Chrome to Phone Extension?
