### Post by artir on 2009-04-16
I made a program similar to fala(fala.sourceforge.net), capable of saying what you write or save the recording in a file, plus unicode and pulseaudio support.

The code is here [http://launchpad.net/cerva](http://launchpad.net/cerva) in a bazaar branch.

This is my 1st useful python program and I made it to practise with pygtk,glade and calling external commands.
If somebody can tell me how to use the "po" files, please tell me :popcorn:
