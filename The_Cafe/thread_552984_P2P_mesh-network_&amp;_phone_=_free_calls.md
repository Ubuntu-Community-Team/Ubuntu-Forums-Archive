### Post by PartisanEntity on 2007-09-17
A Swedish company by the name of [TerraNet]("http://www.terranet.se/") has developed a p2p-like mesh network that would allow people to make calls for free using special phones reports the [BBC]("http://news.bbc.co.uk/2/hi/technology/6987784.stm"):

>     The TerraNet technology works using handsets adapted to work as peers that can route data or calls for other phones in the network.

    The handsets also serve as nodes between other handsets, extending the reach of the entire system. Each handset has an effective range of about one kilometre.

    This collaborative routing of calls means there is no cost to talk between handsets. 

My thoughts:

[LIST]
[*]Privacy: Since my calls will be forwarded through other phones in the network, would uses be able to listen to my calls? Perhaps using a hacked phone?
[*]Battery: All the phones in the network would not only have to periodically monitor the network strength but also forward calls, how would this affect battery life given the current level of cell phone batter technology?
[*]Availability: To turn this network into a viable alternative to GSM there would have to be a wide circulation of these phones, so how much will they cost?
[/LIST]
