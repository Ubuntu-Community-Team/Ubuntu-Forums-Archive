### Post by Xzallion on 2008-06-11
It is really hard to understand your english, and unfortunately the title doesn't make a lot of sense.  It should read something like "What do you imagine the computer world would be like if all software was open-source?"

I doubt that things would be faster, because we are currently pushing hardware about as far as it will let us.  Development on some things would be faster, on others it wouldn't be.

Wine wouldn't exist, because if Windows was open source then we wouldn't need a compatibility layer.

Companies and individuals already have to pay programmers to develop software for their needs, otherwise there wouldn't be any programming jobs at all.

Really the only benefit would be support for various devices like scanners, cameras, printers and the like.
