### Post by ssam on 2010-04-29
UPDATE: the announcements are up now :-) [http://fridge.ubuntu.com/node/2030](http://fridge.ubuntu.com/node/2030)

some people are posting links that claim to be the ubuntu lucid release. but there has not been an official announcement yet. please don't download anything until you see the announcement on ubuntu.com, because:

* it may not be the final version. (there is work on fixing a last minute bug, if you download early you might get the buggy version)
* the devs need the bandwidth so they can test the ISOs before release.
* the ubuntu servers cannot handle everyone downloading from them, so there is a network of mirrors. these mirrors need to download ubuntu from the official server before the release can be announced. if lots of people are hammering the official server this will take longer.

so relax and wait.
