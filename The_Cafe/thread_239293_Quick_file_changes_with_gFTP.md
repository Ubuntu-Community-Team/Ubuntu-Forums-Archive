### Post by sapo on 2006-08-19
gFTP has this feature, you just have to configure it properly.

in the gFTP window, go to: FTP -> Options 

and in the general tab, fill the Edit Program with: gedit

then you just have to right-click a file and the click on edit, it will open up the file on gedit, when you save the file and close gedit, gFTP will ask if you want to upload the file.
