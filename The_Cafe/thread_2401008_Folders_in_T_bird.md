### Post by amanchesterman on 2018-09-12
It's because Gmail uses labels rather than folders to organise messages.
I believe the reasoning behind that is that if your mail provider uses folders, any given message can be in only one of them. You might create a folder called 'important' and another called 'friends' for example, but a message can be in only one of those folders at a time. In Gmail, the message could be labelled 'important' and 'friends' at the same time. Gmail thinks this is a big plus.
Thunderbird doesn't use labels, it uses folders. When you create a Gmail account, it creates folders corresponding to the labels, but it still provides the standard folders that it uses by default for every account. So 'sent' is a standard Thunderbird folder, 'sent mail' is a folder that corresponds to the same Gmail label.
I suggest you read [https://support.mozilla.org/en-US/kb/thunderbird-and-gmail](https://support.mozilla.org/en-US/kb/thunderbird-and-gmail) and [http://kb.mozillazine.org/Gmail](http://kb.mozillazine.org/Gmail) to understand this better and to learn how to set up Thunderbird in the way you want it, so that folders aren't duplicated.
