### Post by inportb on 2008-05-12
> **FuturePilot said:**
> You can't run Compiz if you're using a WM.
XFCE is probably the lightest DE that you can use with Compiz.

No, but you can run a compositing manager with a window manager. Compiz just has the compositing manager and window manager rolled into one.

> **SomeGuyDude said:**
> What's the lightest DE or WM that can run Compiz/AWN?

xcompmgr is a small compositing manager you can use to add compositing capabilities to just about any WM. You can then run AWN on top. If you think xcompmgr is too old/unstable, check out E17+Bling.
