### Post by louis_nichols on 2006-04-17
[QUOTE=Majin_Buu]Thanks for the info, but what's "kwin"?

kwin
kwin: unable to claim manager selection, another wm running? (try using --replace)[/QUOTE]

kwin is kde's window manager. the window decorations, that include the minimize, maximize and close buttons.

If you right click on the upper window border, a menu will popup, with several options. You can go to "advanced" and choose "hide border". pretty neat effect ;)

So then, if you go to "special window settings", you can set changes there, such as "no border", that will automatically take effect whenever that window is opened. So, saving a kde session (or, in my case, a gnome session - I use kwin instead of metacity - the gnome window manager) with a terminal open and configured this way, means that every time you login you will have a borderless terminal on your desktop.

I also set the session properties of the terminal to have a transparent background (mind you, that will actually be pseudo-transparent - just imitating the wallpaper - you won't see any windows that go under the terminal window), and hide the menubar and scrollbar. And... also configure kwin to remember the last position and size of the window, as well as keep it below others. And there you have it: terminal on desktop. Enjoy!

I've found that in Dapper you have to put the icon in the folder that contains the icon theme you are using:

/usr/share/icons/[icon theme you are using]/48x48/apps/distributor-logo.png
or
~/.icons/[icon theme you are using]/48x48/apps/distributor-logo.png


After this, do:
```
sudo gtk-update-icon-cache
sudo killall gnome-panel
```
