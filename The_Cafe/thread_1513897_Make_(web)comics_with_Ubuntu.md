### Post by alexan on 2010-06-20
Essential tool
[LIST][*]Gimp - The very basic to start anything, it provide you all the tools from basic resizing image, to digital inking and coloring
**Installation: [alt]+[f2] apt:/gimp**
[*]Inkscape - This is what do you need to make vectorial comics, or manage your bitmap with more handy tools for design them (logos, balloons, lettering). See chapter "Vectorial vs Bitmap" for more
**Installation: [alt]+[f2] apt:/inkscape**
[*]Xsane - Simple scan it's simply not enough
**Installation: [alt]+[f2] apt:/xsane**[/LIST]
Other tools
[list][*][OpenClipart](www.openclipart.org) database (balloon, graphic and other stuff Inkscape (SVG) compliant
[*][Ubuntu social group](http://ubuntuforums.org/group.php?groupid=787)
[*]Blender - If you plan to make a 3d comic, you need to be basic proficient with this one. [Start here](http://wiki.blender.org/index.php/Main_Page) and don't panic, it's more easy that you may think (notice: if you use 3d models made by others, always check for the license for it's use)
**Installation: [alt]+[f2] apt:/blender**
[*][makehuman](http://www.makehuman.org/) - If you need 3d character to put in your comic, you may need a generator of these character. (notice1: blender fully compliant //notice2: GPL3.0 licence apply)
**Installation: [download the deb package](http://sites.google.com/site/makehumandocs/download)**
[/list]

Vectorial vs Bitmap:
Comics made in vectorial graphic are restricted to particular tools, which make your work little flexible when come to use it with other software (blender, gimp...). But once you get used to it, you'll get some really cool advances. SVGs have no problem of resolution/size, one other advance with webcomics is that the _text inside your page is readable by software_ (example: [ctrl]+[f] > search word > text in balloon found)

Faq:
Q: Can I put/use opensource-free stuff in my comics and eventually sold or make pay subscription?
A: Every tool/media you do use come with own license, you must read to know what you can and what not. For specific license question, ask in this thread: experts of Ubuntu community will answer



This thread is for those who want use Ubuntu to make comics. This first post will be constantly updated with the best software and resource available: only one main big rule. Only free software that can be used on Ubuntu.
