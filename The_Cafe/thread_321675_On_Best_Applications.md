### Post by wersdaluv on 2006-12-19
There are some applications that are hard to do away with.

Here is my list:
Firefox- The best web browser (my opinion)
Gaim- Instant messenger
AmaroK- The best Media Player for Linux that I know
Kontact- Personal Information Manager (comes with KDE's basket and more)
Synaptic Package Manager-You know what it is... Damn! We can't live with without it! 
Terminal- runs almost everything
SuperKaramba- Eye candies
Automatix2- Graphical Installer Script

Applications that I love but I cannot run:
Games!- It's hard to work with wine for newbs and not everything is supported by wine
Drivers!- I have problems with printer drivers, scanner drivers, etc. There are ways (maybe) but those ways are yet to be figured out by me
Microsoft Office!- Yes, you will say that there are substitutes but nothing beats MS Office's features and it's way easier if we are working with the same file formats that everybody else is working with


I think that's all. I want to see your own lists and comments on mine. Thanks! :)
