### Post by xArv3nx on 2009-10-08
yesterday I tried using the new gnome shell everyone keeps talking about, and i find it slick, and very fast. it seems to have some type of nice animations going on with minimizing windows, etc., and the best part is it doesn't seem to interfere with fullscreen games.

i'd say it's much better than kde 4.0, but then again it doesn't seem to be a complete rewrite (the base system of gnome 2.24 on jaunty is still there). i've replaced my main desktop with the new shell, that's how much I like it (it's very usable even at this stage in it's development). i found it very easy to pick up and just go, as well.

i do have one problem, however, and that is that the window borders always stay as "Human", even if I change them in the gnome appearance properties. is there a fix, or is this because no one has made and "new gnome shell" window borders yet?
