[FONT=Franklin Gothic Medium]> **RAV TUX said:**

I use:

**Games:** KSirtet, KMahjongg, KReversi, KBattleship, KMines, Kolor Lines, KPatience (I probably enjoy procrastination too much)
**Graphics:** KolourPaint (yes, I used to like a similar M$ program too...), KPDF
**Internet:** Konqueror (although I use it more as a file manager), KMess, KTorrent
**Multimedia;** Kaffeine, Amarok, K3B, Rosegarden
**Office:** KWord, KSpread, Kile, Kivio, Karbon14, KNotes
**System:** Adept Manager (I end up finding all sorts of fun packages by accident)
**Utilities:** Ark, KAlarm, Kate, KCalc, KCharSelect, KJots

If I had to pick favourites, I'd probably only narrow it down to: Amarok, Kaffeine, Kile, KolourPaint, KCharSelect, Kate, KMess, Konqueror.

The main thing I feel is missing is a "heavy" web browser. I love Konqueror as a file manager and light web browser, but some websites these days are just beyond its capabilities. I wish something had come of the Qt port of Firefox idea that was around a few years ago (IIRC some smart aleck even suggested a good name -- Kitsune).

However, there is one KDE application I absolutely hate: Dolphin. I find it tragicomic that its developers seem to equate "usability" to a pretty interface lacking several useful features* of what they're trying to replace. If I wanted that sort of usability, I'd use GNOME.

---

* At least I presume it's lacking them, and they're not just hidden somewhere impenetrable. Here are a few of my main annoyances (feel free to tell me I'm wrong!):
[/FONT][SIZE=1][FONT=Franklin Gothic Medium]1) No top/bottom split view. (Try the combination of both sidebars plus split view on a 4:3 monitor!)
2) Only allowing a two-pane split view. (The shortcut key works as a toggle, rather than adding a third pane.)
3) That Previews view instead of Konqueror's Image view. (The effect of this is that you can neither see what files are in the directory, nor what they look like.)
3a) The strange, counter-intuitive workaround of not using the useless Previews view, but turning on the right sidebar.
4) Its predilection for firing up KView instead of just showing me the darned picture.
5) The total absence of Multi-Column, Tree, and Info List views, along with four more that I admit I don't actually ever use.
6) Hovering changing the selection, instead of displaying a properties tool-tip.
7) Leaving the Up, Home, and Refresh buttons off the toolbar by default. (I know this is fixable, but this is a dreadful, user-unfriendly default setting.)
8) The Navigation Bar in some horrid clicky mode by default. If I want to type, say, /lib/firmware/ into it, I'm likely not to be in the mood for messing around with menus.
9) Not letting me sort by File Type, let alone Owner, Group, or Permissions.
10) In fact, failing to display those things in the Detailed List view by default. (Don't the developers understand the word "detailed"?) With defaults like these, it's far quicker to jettison the dogfish of a program than configure every setting.[/FONT]
[/SIZE]
