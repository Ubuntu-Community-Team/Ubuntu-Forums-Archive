### Post by Tundro Walker on 2007-07-26
If you want to get serious about it, follow the 3 (or 4) rules to code-names...

1) positive adjective/descriptor, that shows some moxy while not being too egotistical (gutsy, feisty, dapper...)

2) animal noun/name, usually something off the beaten path (warthog, gibbon, drake, fawn..)

3) must use alliteration (both words start with same letter)   

4) (not sure how strict this is, but) words can be 2 syllables max each

So nothing too funky, like "supercalafragiliciousexpialidocious salmon"
Or non-animal, like "succulent sandwich"
Or negative/derogatory, like "salacious salmon"

But, things like...

"poised peacock"
"saucy salmon"

...there you go.

Of course, it's not as humourous if you follow rules.  I kinda like the idea of a "Horny Housewife" version.
