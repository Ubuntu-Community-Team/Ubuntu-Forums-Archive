A great free eBook that teaches you all the fundamentals of the command line is a book called: "The Linux Command Line" By: William Shotts.

I can be downloaded here: [http://www.linuxcommand.org/tlcl.php](http://www.linuxcommand.org/tlcl.php)

It takes a kind of hands on approach to learning where the author has you work through exercises as you read.
