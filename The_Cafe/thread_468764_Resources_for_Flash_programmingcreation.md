### Post by izak on 2007-07-16
For open source flash tools head over to [www.osflash.org](www.osflash.org).

I've been successfull in compiling a flash game I'm working on in ubuntu, but is is somewhat of a mission. Two different compilers have to be called in succession (mtasc and swfmill). swfmill has to be built from source for ubuntu. Theres supposed to be an eclipse plugin for actionscript but I haven't got it working yet; still using plain old scite but that doesn't do all the correct highlighting and the usefull tab comletion of class names. 

Still, kudos to all the great people making open source flash development a reality!
