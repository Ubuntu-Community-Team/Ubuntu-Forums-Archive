### Post by MaxIBoy on 2009-09-25
Just found this:
[http://www.linuxjournal.com/content/tech-tip-really-simple-http-server-python](http://www.linuxjournal.com/content/tech-tip-really-simple-http-server-python)

First, CD to the directory you want to share files from. Then:
```
python -m SimpleHTTPServer
```(requires python to be installed.)
Now you can pull up a web browser on any other computer, type in the IP address and port 8000 of your server (for example 192.168.1.103:8000) and you can browse the files!


Honestly, is there anything Python *can't* do?
