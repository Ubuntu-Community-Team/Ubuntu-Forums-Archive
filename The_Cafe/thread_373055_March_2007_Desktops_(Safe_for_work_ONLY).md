March 2007 Desktops - Posting Guidelines

* Stay within the board rules, no matter how hot you may believe the wallpaper/screenshot to be.

* If you are going to quote, there isn't a need to quote an img-embeded picture - Just don't do it. Please just reference the post by page number/page link/etc.

* Off Topic Posts will be edited or removed.

* The desktops thread is not to be used as a means to post explicit or suggestive images (they must be safe for your kids and/or boss to see). Please help to keep the thread relevant and safe to browse for everyone.

* The moderators of this forum reserve the right to edit or remove your posts as necessary so they may comply with our Community Ruleset.

* Please use reduced-size thumbnails, where possible, out of respect for people running on lower bandwidth or screen resolutions.

* In the interests of keeping the thread on topic it is strongly urged that you format your posts in the following order:

Themes: Name/where you got it (with link would be nice).

Wallpaper: Name/where you got it (with link would be nice).

Icons: Name/where you got it (with link would be nice).

Programs: Name/where you got it (with link would be nice).

Important Note: Any image posted that is over 800 x 600 will be removed! Use a thumbnail or use a text link to your image. If your image is removed, this is why.

### Post by finferflu on 2007-03-21
The command for taking screenshots is (note that you have to have imagemagick installed):

```
import -window root screenshot.png
```
if you want to take a shot of the whole desktop, and save it in your home folder as screenshot.png

If you want the shot to be delayed for a number of seconds you can do:

```
sleep 10; import -window root screenshot.png
```
this will delay the command of 10 seconds. 

note that if you do not use the option "-window root", you will be able to take pictures of single windows, by clicking over them.

i got it from here -

```

the bits you need are these -

# Define a few Color's
BLACK='\e[0;30m'
BLUE='\e[0;34m'
GREEN='\e[0;32m'
CYAN='\e[0;36m'
RED='\e[0;31m'
PURPLE='\e[0;35m'
BROWN='\e[0;33m'
LIGHTGRAY='\e[0;37m'
DARKGRAY='\e[1;30m'
LIGHTBLUE='\e[1;34m'
LIGHTGREEN='\e[1;32m'
LIGHTCYAN='\e[1;36m'
LIGHTRED='\e[1;31m'
LIGHTPURPLE='\e[1;35m'
YELLOW='\e[1;33m'
WHITE='\e[1;37m'
NC='\e[0m'              # No Color

# WELCOME SCREEN
#######################################################

clear
for i in `seq 1 15` ; do spin; done ;echo -ne "${WHITE} USA Linux Users Group ${NC}"; for i in `seq 1 15` ; do spin; done ;echo "";
echo -e ${LIGHTBLUE}`cat /etc/SuSE-release` ;
echo -e "Kernel Information: " `uname -smr`;
echo -e ${LIGHTBLUE}`bash --version`;echo ""
echo -ne "Hello $USER today is "; date
echo -e "${WHITE}"; cal ; echo ""; 
echo -ne "${CYAN}";netinfo;
mountedinfo ; echo ""
echo -ne "${LIGHTBLUE}Uptime for this computer is ";uptime | awk /'up/ {print $3,$4}'
for i in `seq 1 15` ; do spin; done ;echo -ne "${WHITE} [http://usalug.org](http://usalug.org) ${NC}"; for i in `seq 1 15` ; do spin; done ;echo "";
echo ""; echo ""

i changed the WELCOME SCREEN to this though -

# WELCOME SCREEN
#######################################################

clear

echo -ne "${LIGHTGREEN}" "Hello, $USER. today is, "; date

echo -ne "${CYAN}";echo ""

```
