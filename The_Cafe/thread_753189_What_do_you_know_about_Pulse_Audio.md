### Post by edm1 on 2008-04-12
Im loving the ability to control individual streams and move them between devices. It is very nifty indeed. You may want to install the pulse audio guis;

```
sudo apt-get install padevchooser paman paprefs pavucontrol pavumeter
```

I then added 'padevchooser' to my startup commands to get the tray applet to start automatically.
