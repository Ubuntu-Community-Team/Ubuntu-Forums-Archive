USC is far from complete. Pretty much every common use case scenario is covered on the specification page in great detail: [https://wiki.ubuntu.com/SoftwareCenter/](https://wiki.ubuntu.com/SoftwareCenter/)

For example, correcting reviews is here:

[https://wiki.ubuntu.com/SoftwareCenter/RatingsAndReviews#Correcting_a_review](https://wiki.ubuntu.com/SoftwareCenter/RatingsAndReviews#Correcting_a_review)

The problem is that they haven't been implemented in code yet and that, AFAIK, previous Ubuntu releases don't get the new version of USC.
