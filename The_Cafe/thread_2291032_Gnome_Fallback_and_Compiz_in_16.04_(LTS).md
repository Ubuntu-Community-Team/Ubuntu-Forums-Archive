### Post by Z80A on 2015-08-17
On quite a few PC's, I have installed Ubuntu 14.04 LTS and added [Gnome Fallback Session]("https://apps.ubuntu.com/cat/applications/gnome-session-fallback/") (gnome-fallback-session), [howto link]("http://www.howtogeek.com/189912/how-to-install-the-gnome-classic-desktop-in-ubuntu-14.04/"), along with [Compiz Config Settings Manager]("https://apps.ubuntu.com/cat/applications/compizconfig-settings-manager/") to fix an [ALT-TAB issue]("http://askubuntu.com/questions/498315/no-alt-tab-in-14-04-gnome-flashback-session"), to get a kind of light weight Ubuntu version. It makes standard Ubuntu quite manageable on low-end or old PC's with a very traditional user interface while still being part of mainstream Ubuntu.


I don't want to stir up the Unity discussion/war, but I am quite happy with Gnome Fallback Session and could easily see be using that for years to come. Looking ahead, what are the odds that Gnome Fallback and Compiz will be supported in a coming LTS (16.04)?

### Post by grahammechanical on 2015-08-17
Why not?

Ubuntu 16.04 will be using Unity 7 and it will be running on the X server, LightDM and Compiz. I am running 15.10 development release and I have Gnome Flashback session installed. Apart from Unity I get two other login options. Gnome Flashback (Compiz) and Gnome Flashback (Metacity). I fully expect to upgrade to 16.04 development release this November and still have Gnome Flashback session working as it should.

 If you are thinking about old PCs then the Metacity option would be better than the Compiz option.

At some point it will have to be accepted that old hardware is not up to the job of running Ubuntu. And there is Xubuntu, Lubuntu and Ubuntu Mate as alternatives. The biggest obstacles, in my opinion, to using older hardware is not the CPU or the amount of RAM but the video adapter not having the power to do the job and the adapter being so old that it is no longer supported by the newest proprietary video drivers.

For those who want what is called the traditional User Interface then I suggest that they install Xubuntu, Lubuntu or Ubuntu Mate from the start.

Regards.
