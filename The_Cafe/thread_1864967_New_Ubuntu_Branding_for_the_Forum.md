### Post by TeoBigusGeekus on 2011-10-20
Talking about customization of ubuntu forums, here's a nice javascript (thanks to tjwoosta).

The threads you've participated are marked with a dot. Either because you're old and you can't see the dot, or you're simply tired of it after all these years, here's how to change it.
First create six icons (about 30x30 pixels) with gimp.

[[IMG]http://imgur.com/WHfC2s.jpg[/IMG]]("http://imgur.com/WHfC2.jpg")

The 6 icons should reflect the different icons used by the forums on threads you have posted:
1)thread_dot (the normal dot icon which indicates that you've participated in this thread)
2)thread_dot_hot (as above, but the thread is hot)
3)thread_dot_hot_new (as above, but the thread is also new)
4)thread_dot_new (as above, but the thread is simply new)
5)thread_dot_hot_locked (as No2, but the thread was locked)
6)thread_dot_locked (as No1, but the thread was locked)

You can use anything you want; I personally took my avatar and customized it for each situation.

The next step is to convert your images to URI (Base 64). Here's a couple of sites to help you:
[http://www.scalora.org/projects/uriencoder/]("http://www.scalora.org/projects/uriencoder/")
[http://www.abluestar.com/utilities/encode_base64/]("http://www.abluestar.com/utilities/encode_base64/")

Now open the attached javascript (line wrapping or you'll be lost).
Every icon starts with a line similar to this:
```
img[src*=\"images/statusicon/thread_dot.gif\]
```
Directly underneath it, is a line with the actual icon, in URI format, enclosed by parentheses.
All you have to do is replace my URI code in the parentheses with your URI code, ie. to replace the first icon of the script (thread_dot.gif) with yours, replace the line that goes
