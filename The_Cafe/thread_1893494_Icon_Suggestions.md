### Post by Frogs Hair on 2011-12-10
Place the extracted folder in usr/share/ icons in the file system or create a .icons folder in home / hidden folders . Be aware that some of the older icon sets don't work with Gnome 3 . You will also need install the Gnome Tweak Tool / Advanced Settings for making theme and Icon selections .
