### Post by MaxIBoy on 2008-10-19
Armed Assault 2 (from a brief Google search) will not work well under Linux. You'll need a copy of XP. Be prepared to <BipBipBip> shell out $90 for an XP CD that will only install once:
[http://www.newegg.com/Product/Product.aspx?Item=N82E16832116511](http://www.newegg.com/Product/Product.aspx?Item=N82E16832116511)
Trust me, you do NOT want Vista for gaming. It just doesn't work well. (Have you ever tried to set up a LAN game on a Vista box?) 

Partition up your hard drive using a linux liveCD, give XP a 60 gigabyte NTFS partition. Make sure there's a swap partition equal to double the size of your RAM, and make sure it's the first partition (you want swap to be first because the first partition reads/writes the fastest.) Then install XP. Then go back into Linux and install your distro of choice in the remaining partition.



As for the hardware, this is roughly equivalent to my system specifications. I've been happily gaming on these parts for a month or so, I've had absolutely no problems.

The recommended specs are crap. They do that only so that people will buy the game. Truly, the recommended specs can only play the game on LOW settings. Trust me, I have one of those PCs.

If you want to run it well, get:
Intel Quad-core (QX6850)
Nvidia cards (8800+) in SLI (Make SURE you have HUGE amounts of GPU dedicated memory)
4+ Gigs of RAM (Corsair)
Top-rate motherboard.
SATA HDD @ 7200 RPM
Power supply > 500W

Also, if you don't have a monitor already, make sure you don't get something HUGE. I made that mistake, and I can't play anything high on native resolution.
