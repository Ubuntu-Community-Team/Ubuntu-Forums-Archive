### Post by CocoAUS on 2007-05-07
Debian is fantastic.  The only problem is at the moment, unstable isn't quite so up to date (lots of work went into putting out the new stable Etch), the unstable nvidia driver is broken, etc.  But I do like it more than Ubuntu.

As for more customizable distros, I'd say Arch is pretty awesome.  It takes a LOT of time and learning, though.  If you need something that just work, Arch probably isn't it.

And I'd probably be using GoboLinux above ALL of these if it had more recipes available.  Gobo is doing some true innovation and moving open source operating systems in the right direction, whereas most others are simply reinventing the wheel, so to speak.

And, as a side note, I sort of hate Gentoo.  It doesn't really offer any advantages, as you can ALWAYS build ANYTHING you want from source, regardless of your distro.  Gentoo simply requires too much work and doesn't yield anything worthwhile.
