### Post by Derek Djons on 2006-06-30
It has been a while since I checked the Ubuntu forums on a daily base. How come? I've finally bought that Mac I was chatting about in some older posts. It isn't exactly the model I wanted (the Apple iMac Intel 20 Inch) but after some good thinking I realized an Apple Mac Mini G4 with 80GB + SuperDrive + WIFI/BT would suffice.

I didn't had to learn about Mac OS X and the Mac I bought since I know that all from experience and work. So it was immediately using it for some duty. I bought the Mac to:

- Surf on the internet
- Email
- Write letters / brochures
- Manage my iPod and my whole Music Library
- Organize and (minimaly) edit my photographs

I must say the Mac and Mac OS X just works as I expected. I'm very happy I bought it. On a daily base I save a lot of time with the build in applications and possibilities for batch editing. Now that I have a Mac Mini with Mac OS X and a notebook with Ubuntu Linux I can finally say: "No more dual booting and no more Microsoft Windows XP". Not that Microsoft Windows is a bad OS, but Mac OS X and Linux are just more handy, transparant and easy to adjust.
