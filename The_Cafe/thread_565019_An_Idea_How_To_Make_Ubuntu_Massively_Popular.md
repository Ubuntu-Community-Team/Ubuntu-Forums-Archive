### Post by aysiu on 2007-10-01
At this point, I would be *very* hesitant to just give a stranger a Ubuntu CD. That's a disaster waiting to happen.

Does this person know to set the BIOS to boot from CD? Does this person know the live CD won't affect her hard drive? What if she has a lot of Windows-only applications she uses (ones that don't have comparable Linux counterparts)? What if she has Linux-incompatible hardware?

I wouldn't expect most people to be able to install an operating system themselves. I wouldn't give a Windows installer CD to someone, and I wouldn't give him a Ubuntu CD either.

Distribute these Ubuntu CDs the way AOL distributes CDs and be prepared for an onslaught of "I got this Ubuntu CD free and it f'ed up my computer" threads in Absolute Beginner.
