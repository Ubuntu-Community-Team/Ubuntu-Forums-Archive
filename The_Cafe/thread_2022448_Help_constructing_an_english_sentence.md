### Post by papibe on 2012-07-11
If your main concern refers to the '44th', AFAIK, there's no simple way to ask that.

A practical approach would be to give context or examples:
```
George Washington was the 1st president of the US. Which one is Barack Obama?
```

Where 'one' is an [ordinal number]("http://en.wikipedia.org/wiki/Ordinal_number_%28linguistics%29"). But using that wording is very uncommon.

If you were to be among engineers or alike (think 'The Big Bang Theory'), using 'ordinal' would be very well understood. For instance:
```
What is the ordinal number that represents the position of Barack Obama in the succession of US presidents?
Which ordinal number president is Barack Obama?
Ordinally which president is Barack Obama
```

A more common structure is
```
where does X fall in this list/sequence?
```
But in this particular example (Barack Obama) you could something you won't expecting: the last! (valid nevertheless).

Just a few thought from a former ESL student.
Regards.
