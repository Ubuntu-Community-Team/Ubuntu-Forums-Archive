### Post by daz4126 on 2008-05-06
Everybody always asks what this is. I've been thinking for a while that the thing that might get people interested and keep them is a great backup tool.

If you read the forums there are loads of threads asking how to do this and they are always met with a variety of answers. Lots of these answers use the command line and look complicated.

It would be great if somebody could build a front-end gui where you could simply create a 'snapshot' of your system with the click of a button. You could then restore you system by using the live CD.

Other cool features would be incremental backups, scheduled backups, separate backups of different areas such as /home, /opt, /usr etc... and the ability to specify which software that you always install after a clean install. This might be useful if you want to start again from scratch, you simply push a button and it goes and gets all your favourite programs from synaptic.

I've never seen a soltion/app that does all this in a simple fasion and all in one package. I'm sure it is possible and feel that it would be of great benefit to lots of Ubuntu users. I guess that the backend would use some sort of combination of part-image, r-sync and maybe even subversion for version control??

If this was on the live CD and it was also able to backup Windows partitions, you could even 'sell' it to windows users as a great backup tool. You never know, they might even enjoy using the live CD so much that they decide to never actually restore Windows!!!

Any thoughts/suggestions - can it be done?

DAZ
