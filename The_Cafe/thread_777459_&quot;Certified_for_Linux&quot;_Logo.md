### Post by clubsoda on 2008-05-01
Buying parts for a linux machine is a tricky business.  You have to have an idea of what kind of widget you might want, print out a list of which ones work with linux and have that list with you when you go shopping, ready for cross-checking.  Even then, you can easily be deceived by manufacturers who modify their designs so as to obtain a "certified" logo from some major software company, and then sell the modified parts with the exact same model number as before, breaking any linux compatibility in the process.

Getting manufacturers interested in our 5%(?) slice of the market may be a challenge initially but we only need one player in each hardware category to start making a difference.

It seems like there is already [a logo](https://www.redhat.com/apps/isv_catalog/VendorProfile.html?cat=isv&vendor_id=1154) designed for this purpose...
