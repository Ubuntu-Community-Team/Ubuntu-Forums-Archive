### Post by Mornedhel on 2010-01-11
> **gletob said:**
> Dunno, never had a Alt Gr button, just two alts.  I feel left out. (This is one of the few things, I like about my native language english, the fact that there aren't any accented letters.)

Not all layouts use Alt Gr to input accented characters.  Think of Alt Gr more as another modifier like Shift.  You can use it to access unusual characters.

The top row of my layout (France alternative) looks like this :
```
²&é"'(-è_çà)=
```

Shifted :
```
³1234567890°+
```

With Alt Gr :
```
¹&#711;~#{[|`\^@]}
``` with &#711; being a dead key, that is, I can use it for accents : &#462;ž&#283;&#345;&#357;. Notice the presence of frequently used characters, {}[]| and ~.

Shift Alt Gr :
```
¸&#731;É&#728;—–&#8209;È™ÇÀ&#8800;±
``` with ¸ and &#731; also dead keys.

Languages with few possible accented letters (e.g. French) usually set them on separate keys, so the most used accents are directly usable: éèçàù are all single keystrokes, ^ and " are simple dead keys (the examples for the top row are never actually used in French).

Languages with few generic accents on more letters (e.g. Spanish) will have a few single keystroke dead keys so you only need to press those and then the non-accented letter to make the accented letter (‘ then e gives é). Languages with almost no accents at all (e.g. English) have an advantage in that they don't need to spend useful keys on those and have more punctuation available at a single stroke.

Alt Gr works just fine on the right side, if you had Alt there you'd soon complain about needing the other hand for Alt-based shortcuts. If it really bugs you, you can remap those with some xmodmap magic, but I can't remember how right now.
