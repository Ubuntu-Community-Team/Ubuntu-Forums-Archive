### Post by murderslastcrow on 2010-09-18
Depends on the hardware- if it's well supported, Linux (even KDE 4) manages memory and scales better than OS X. However, if you're using it for specialized purposes, like database management, you may want to look at another distro, like Chakra, or Debian Stable. Depends on your needs, really.

But, especially if you plan to use e17 or fluxbox for desktop management, Linux will consistently deliver optimum memory management. However, if you have enough RAM, OS X has similar performance. Windows probably isn't a good idea for any serious data crunching.

For more accurate information, and proof of what I'm saying, I'd refer you to Phoronix for their extensive benchmarking, which pinpoints this kind of data.
