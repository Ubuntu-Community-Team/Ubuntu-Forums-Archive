### Post by wildman4god on 2009-01-23
I was researching all the media players there are for linux and I came across a new one called MPx/AudioSource (MPx is its development name, Audio Source is it's offical name) AudioSource is an eveolution of BMPx which is an eveolution of BMP (beep media player) which is a fork of XMMS (a winamp 1 clone) anywho, the player is looking sweet, the just released their first stable release (0.01) right before Christmas 2008. I don't think people know about it, you can get its predisesor BMPx in the ubuntu repos, anywho if this project keeps going it looks to be a good media player. Check it out:

[http://mpx.backtrace.info/wiki/AudioSource](http://mpx.backtrace.info/wiki/AudioSource)

I don't think the website is compleatly upto date, it says only alpha 6 is released but i found another site that confirms 0.01 is released and you can download the tarballs here:

[http://freshmeat.net/projects/mpx/?branch_id=75210&release_id=290825](http://freshmeat.net/projects/mpx/?branch_id=75210&release_id=290825)

let me know what you think of it, also I hear it is to use clutter, which is kinda like plasma for kde, Clutter will also be used by some of the gnome games included with gnomes next release (2.26) I think this is just a start of clutter being integrated into gnome to overcome its lack of eyecandy.
