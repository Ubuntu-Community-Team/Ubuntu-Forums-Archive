### Post by James_Lochhead on 2009-04-17
The other day I was thinking (as you do!) about semi-free (as in freedom) software and how it could benefit the consumer. I think a lot of the benefits of Open Source software could apply to semi-free as well given the right set of laws.

I believe an example is in order to help explain:

Lets take three companies X, Y & Z. X wants to release proprietary  software with public source code and a license that allows consumers to modify the code (main direct advantage of OSS) as long as they don't distribute it. However, the managers at X know that if X releases the source code Y & Z will snap up the source code and use it in there product. 

Time passes and Y & Z release their products (closed source). The managers at X would like to know if Y & Z have infringed their copyright by looking at Y and Z's source code. However, the X managers cannot legally do this because the products are closed source (**note: I am not a lawyer, this is just my interpretation of how the laws work**).

Y & Z did use X's source code. They have gained an advantage in the market because of this and are therefore making more profit and have a higher market share. X's managers decide they cannot take loses like this and X's next product is closed source (not as good for the consumer).

**What if all off the shelf software (assume X, Y and Z product off the shelf software) was open source due to laws?**

The problem described above would not happen. X could check if Y & Z infringed copyright **like for most other products in the world today.**

This approach is obviously not perfect. A lot of companies would squeal about security concerns (*cough* Microsoft). However, by definition a lot of software that is publicly available is basically open source anyway for someone willing to break laws (i.e. reverse engineering). Hence, that argument doesn't stack up.

Certain software could be immune from the laws, such as defense software (obviously), very specialized software and software built for corporations/other large organizations.

I couldn't see this ever being implemented in the US. However, I think this might be the way the EU will *eventually* go.

Thoughts?
