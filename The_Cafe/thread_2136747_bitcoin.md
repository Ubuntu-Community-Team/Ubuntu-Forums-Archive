### Post by haqking on 2013-04-18
to reiterate the 2 posts above me yes mining is also pretty pointless these days without dedicated hardware or a server famr or part of a pool.

However you didnt actually mention mining you only mentioned the client which is basically a wallet.

You dont need to mine to use bitcoins or to use the wallet so if you want to install the client then go ahead but you wont earn any bitcoins by doing so but you can buy some which will then come into your client (wallet) which you can use to make purchases online anonymously from places which take them, however the client itself will do nothing but enable you to send and receive coins.

Mining is a little differnt which requires mining software and a decent hardware configuration, which at this point in time is not really worth starting, but there are plenty of places online to scavenge them for free but will take a large chunk of your everyday to make any real return
