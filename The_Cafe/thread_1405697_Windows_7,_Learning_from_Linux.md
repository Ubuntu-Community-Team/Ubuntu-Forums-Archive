### Post by carbonbased on 2010-02-12
A couple of weeks ago I installed Windows 7; now dual-booting with Ubuntu 9.10.  I had been using WinXP(3) for many years, before going to Ubuntu totally.

What I see in Windows 7 is that, compared to WinXP, MS has learned an awful lot from Linux. In fact, it's incredible how many features one knows from Linux that are now in Windows 7, and were not in WinXP. I've been running it 64bit, as I do Ubuntu. One thing I think I can say merely by observing over a few weeks is that 64 bit Ubuntu is faster than 64 bit Windows 7, more snappy. Win 7 Ultimate puts about 18 Gb on the disk--I think Ubuntu took up less than 4 Gb.

Just a some observations. MS learning, obviously, from Linux. MS always has had to mimic its competition to make progress in ideas.
