### Post by Johnsie on 2006-11-29
I've submitted a feature request for songbird-inclusion in Ubuntu. Right now the Ubuntu developers are saying that Songbird is not ready to be included in Ubuntu but it looks promising in the longrun. For the time being I'm suggesting that Ubuntu users who like Songbird should make efforts to contribute to Songbird so that when Songbird is released it works well on Ubuntu. This can be done by reporting bugs and making feature requests at [http://bugzilla.songbirdnest.com/](http://bugzilla.songbirdnest.com/) when necessary.
