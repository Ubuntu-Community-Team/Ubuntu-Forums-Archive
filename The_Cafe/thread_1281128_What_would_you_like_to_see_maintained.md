### Post by earthpigg on 2009-10-03
take over giver!

[http://tombuntu.com/index.php/2008/12/19/simple-desktop-file-sharing-with-giver/](http://tombuntu.com/index.php/2008/12/19/simple-desktop-file-sharing-with-giver/)

it was created during one of those 'weekend of coding' things for a prize...

...afaik, ***it has no active maintainer or developer***.

i use it all the time, install it on every system i have, and love it. i dread the day that some new kernel version or some-such makes it no longer work.

such a simple, elegant, and great app... with no maintainer.

if i where dictator of the world for a day and could choose one app to start coming preinstalled in Ubuntu by default, this would be it.

[IMG]http://tombuntu.com/wp-content/uploads/2008/12/giver1.jpg[/IMG]

[IMG]http://tombuntu.com/wp-content/uploads/2008/12/giver2.jpg[/IMG]

once you use it once and see how ***not a single bit of configuration is needed for it to work***, you will fall in love.

works across distros, too, with no config needed... well, at least between ubuntu, ubuntu-derivatives, and arch.

i think there are some issues with the Windows version, but i honestly dont care about that :D
