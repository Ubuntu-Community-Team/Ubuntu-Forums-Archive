### Post by cwoodworth on 2007-06-03
The GP2x is fun enough with older emulators. But the problem is that it isn't powerful enough for some of the juicy emulators like GBA/PSX. If you're thinking about plugging in an external hard drive and playing movies and such on it, think again. I have had the GP2x since September, and still have not been able to get USB host functioning properly. It's really hard to find information on it.

On the other hand, it is cheap, and fun to mess around with; so it's a tough call.

### Post by avik on 2007-08-08
Okay, so it's been a while since the last post, but as I just got my GP2X yesterday, I want anyone to miss a good opportunity.

The point of this system isn't playing games; it's *making* them.  That's why it's open source with regards to software and firmware, and why so many great development tools are available for it (most notably SDL).

If you want a media player, sure the GP2X is really nice with it's version of mplayer, but the real strength is if you want to actually develop for it.
