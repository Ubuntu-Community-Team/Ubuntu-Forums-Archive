### Post by DoctorMO on 2006-10-27
linux won't stop supporting ppc; it supports machines and hardware that one one person uses in some university somewhere it's hardly going to rip out support for a massive user base.

No Linux tends to keep hardware support going.

What could drop is the level of developer support. for instance if program X won't compile for ppc then user will have to get the developers to fix it or fix it herself (at least the option is open to her); the fewer developers using PPC the fewer programs will be tested and tweaked to work with it.

although on the whole gcc and various other libs are pritty good when it comes to compiling on ppc.
