### Post by neu5eeCh on 2012-07-13
Just thought I'd post this here for anyone else who uses radio tray. I've gotten a small but useful collection of classical stations. I'd be interested in Blues, if anyone has a good collection. Look under:

~/.local/share/radiotray

open your** bookmarks.xml** file with something like leafpad

Look for **<group name="Classical">**

You can either cut & paste the following:

> <group name="Classical"><bookmark name="KDFC" url="http://provisioning.streamtheworld.com/pls/KDFCFM.pls"/><bookmark name="Classic FM" url="http://media-ice.musicradio.com/ClassicFMMP3.m3u"/><bookmark name="WCPE" url="http://www.ibiblio.org/wcpe/wcpe.pls"/><bookmark name="CINEMIX" url="http://cinemix.us/cine.asx"/><bookmark name="Swiss Classic Radio" url="http://www.radioswissclassic.ch/live/aacp.m3u"/><bookmark name="Radio Mozart" url="http://listen.radionomy.com/radio-mozart"/><bookmark name="Baroque Around the Clock" url="http://shoutcast.omroep.nl:8068/"/><bookmark name="Canada Classical" url="http://91.121.166.38:7210/"/><bookmark name="KDFC FM 102.1 Bay Area" url="http://c1.mp3.liquidcompass.net/KDFCFM"/><bookmark name="Minnesota Public Radio" url="http://classicalstream1.publicradio.org:80/"/><bookmark name="Venice Classic Radio" url="http://109.123.116.202:8020"/><bookmark name="VPR" url="http://vprclassical.streamguys.net:80/vprclassical64.mp3"/><bookmark name="WITF-FM" url="http://sc2.lga.llnw.net:80/stream/witf_mp3"/><bookmark name="WYPR-All Classical Baltimore" url="http://live.str3am.com:9190/wypr-hd3"/></group>Or you can edit the group name to something like **<group name="*Your Name *Classical">** and just append it to the file _before_ the final:

[B]</group>
</bookmarks>[/B]
