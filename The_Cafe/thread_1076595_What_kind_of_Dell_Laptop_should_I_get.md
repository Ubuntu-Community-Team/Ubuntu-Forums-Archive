Dell sells laptops with Ubuntu pre-installed but, since you want to dual boot with windows, you should probably (as it is much easier to set up the dual boot when win is installed first) go to Dell's site and look up a machine that has the specs/price you need then make note of which hardware (particularly wifi and vid card) are offered with Ubuntu and then order a win box with hardware that you know will work or at least work with minimal tweaking.
