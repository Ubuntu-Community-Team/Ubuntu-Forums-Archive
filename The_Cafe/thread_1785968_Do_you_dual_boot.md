### Post by Old_Grey_Wolf on 2011-06-19
Not a simple answer. :)

I have 10 computers. I dual boot 4 of them. They run Windows 7 or Vista and a Linux distro, either Debian, Ubuntu, CentOS, or Mint. The 6 remaining computers run a single OS or they use virtualization to run another OS. 

3 of the remaining 6 computers do not have large enough disk drives to make dual booting an acceptable choice for me. Those computers run a Linux distro; because, Windows 7 and Vista use a lot of disk space, and I didn't have a Windows License to use on them. They are Frankenstein computers I put together from spare parts. I built them for my grandchildren to use when they moved into my home. The grandchildren don't care that they run a Linux distro, they do not know what OS is being used, and they use Macs at school. 

The other 3 computers have plenty of RAM and fast dual-core or quad-core processors; therefore, I use virtualization on those. I've used VMware, VirtualBox, and KVM. The computer I am typing on at the moment has virtual machines for CentOS 5.5, Fedora 13, Fedora 15, Ubuntu 10.10, Ubuntu 11.04, Vista, and Debian. The host operating system on this computer is Ubuntu 10.04 LTS.
