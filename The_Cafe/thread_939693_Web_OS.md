### Post by zvis on 2008-10-06
Firstly I can confirm that G.ho.st ([http://G.ho.st](http://G.ho.st) pronounced 'ghost' - acronym of Global hosted operating system) is the best "WebOS" (disclaimer: I'm the CEO :-) )

Actually we're taking a bit of a different approach to the others - we don't create our own Web apps but instead we aggregate leading Web apps like Google Docs, Zoho, Yahoo! Zimbra etc.

For those who like Zimbra and Zimbra desktop every G.ho.st account includes a Zimbra account (so you can use Zimbra desktop with it).

As a guideline if you use one computer (say a Ubuntu laptop) you don't need a WebOS.  If you don't carry a laptop and use different devices say at Internet Cafes or school libraries (including by the way if you want to access your email, files etc. on the mobile phone) you might like the service

Cheers

Zvi
