### Post by murderslastcrow on 2009-10-27
No, no, no. That's not what I'm referring to. If any previous Windows users have tried to change the theme of their setup, the most common way is through "Visual Styles," which are supported by Wine in Wine Config. This allows you to apply Visual Styles that look similar to your Gnome/KDE theme in order to make things look more integrated. It actually has worked quite well for me in the past.

On deviantART, there is one user in particular who releases GTK themes as well as Visual Styles of the same themes so you can have the impression that Wine is using your GTK theme (it's quite nice actually).
