### Post by JAPrufrock on 2008-01-06
I use Google docs and spreadsheets for files that I want to work on at home and at work.  That way I don't ever have to copy the files to a CD, memory stick or mail them to myself.  It works great.  I absolutely love it.  I also use Gspace for storing files whose formats are not supported by Google docs and spreadsheets.

Disadvantages- There is a size limit, and so you can't upload very large files.  If you have a slow modem or slow CPU, it can be very frustrating.  The spreadsheet isn't as complete as OOffice or Excel, but works very well for small to medium size files for straightforward data storage, maintenance and computation.
