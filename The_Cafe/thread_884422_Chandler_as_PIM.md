### Post by FFighter on 2008-08-11
Chandler is great. I have been using it for several months, and I confess I thought on giving up several times because of the bugs and quirks.

However, I thought that instead of whinnying, I would help by at least reporting bugs and participating on the mailing list. Guess what? Many of the bugs I have reported got fixed and the 1.0 version is very usable. I use it to implement GTD (search google for the best method of implementing GTD in Chandler, currently) and I'm even thinking about developing a plugin to better formalize the GTD support in Chandler. 

Don't forget that it is an open-source project. The platform, source code and API is totally open and documented at the wiki. If you want, you can adapt it to your needs, by forking it or using the plugin architecture. 

Another awesome thing is that it runs transparently well in the three major platforms: Our beloved Linux, Mac OSX and Windows.
