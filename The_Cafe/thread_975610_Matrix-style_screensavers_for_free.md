### Post by klange on 2008-11-08
Open a terminal, full screen it, remove the menu bar, [FONT="monospace"]sudo apt-get install cmatrix[/FONT], run [FONT="monospace"]cmatrix -b[/font], enjoy. I install cmatrix on every Linux device with a screen, just for kicks. (-b is "partial bold", which looks best)
