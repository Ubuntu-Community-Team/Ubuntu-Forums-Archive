### Post by mips on 2012-09-04
> **Sarys said:**
> Okay I found GLmatrix how do I install it?

```
sudo apt-get install xsreensaver-gl
```

And then just select it in your screensaver preferences.

If you are using Unity/Gnome3 you will have to jump through a few hoops to get it working though, see [http://askubuntu.com/questions/64086/how-can-i-change-or-install-screensavers](http://askubuntu.com/questions/64086/how-can-i-change-or-install-screensavers)
