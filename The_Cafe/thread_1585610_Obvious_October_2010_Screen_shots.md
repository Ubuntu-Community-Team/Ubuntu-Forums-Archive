Ratpoison is very similar to Screen. It's 'default' behavior is to have every window maximized, one on top of the other. It doesn't tile unless you want it to. I suppose you know how to install AWN and make it look like that (it's easy). Use their ppa to install the latest: [https://launchpad.net/~awn-testing/+archive/ppa](https://launchpad.net/~awn-testing/+archive/ppa)

GTK: [http://skiesofazel.deviantart.com/art/Orta-184118297](http://skiesofazel.deviantart.com/art/Orta-184118297)

copy the panel-dark.rc from elementary's folder and paste it into the gtk-2.0/apps/ folder in the orta theme, and then open the gtk-2.0/gtkrc file in that same theme, and at the bottom change the "include /apps/panel.rc" to "include /apps/panel-dark.rc" to get the dark panel.

Then install the zuki theme from here: [http://lassekongo83.deviantart.com/gallery/#/d2rieu9](http://lassekongo83.deviantart.com/gallery/#/d2rieu9)
And select it's zuki dark window border instead of the one that comes with orta (which is the same as elementary).

Then install from synaptic (or whatever) the indicator-applet-appmenu, run "killall gnome-panel", and add it to the panel to get the menus there (you might have to logout and back in to have it working properly).

Then install the window control applet from here: [http://gnome-look.org/content/show.php/Window+Applets?content=103732](http://gnome-look.org/content/show.php/Window+Applets?content=103732)
kill the panel again, and add those too (maximize a window so that they're visible, right click on them, and set them to appear always, and control all windows, not just maximized). Then set their theme to custom from the preferences and select the appropriate buttons one by one from the folder where the zuki-dark theme is installed (probably ~/.themes/Zuki-dark/metacity-something/). Be patient here, it might take some effort. Then select the option to remove the decorations of maximized windows.

Install cardapio from their ppa: [https://launchpad.net/cardapio](https://launchpad.net/cardapio)
and use that for a main menu (much much better)

If you use dropbox, install the beta version that supports custom icons: [http://forums.dropbox.com/topic.php?id=26578](http://forums.dropbox.com/topic.php?id=26578)
Just extract and move the .dropbox-dist folder in your /home/user/ directory (provided that you also have nautilus-dropbox in gnome, it should work fine, if not make sure the dropboxd file in there is executable and add it to your startup apps (should work outside gnome too)).

Use the faenza-dark icons (google it).

And get the wall from here: [http://toetag.deviantart.com/art/Swimming-with-the-fishys-43171943](http://toetag.deviantart.com/art/Swimming-with-the-fishys-43171943)

Font is segoe UI semibold size 8, but ubuntu size 8 works fine too.

Also install simple-ccsm, open it, go to the edges tab, and set the bottom corners to expo, and scale: initiate window picker for all windows. (I have them set to left and right respectively). Now move your cursor to those corners and see why compiz is so damn great.

If I forgot anything let me know.

EDIT: For firefox install firefox elementary 3: [http://browse.deviantart.com/?qh=&section=&global=1&q=firefox+elementary#/d30bb6h](http://browse.deviantart.com/?qh=&section=&global=1&q=firefox+elementary#/d30bb6h) (read the instructions) and then use this persona with it: [http://www.getpersonas.com/en-US/persona/5146](http://www.getpersonas.com/en-US/persona/5146) (or whatever else you feel suits you).
