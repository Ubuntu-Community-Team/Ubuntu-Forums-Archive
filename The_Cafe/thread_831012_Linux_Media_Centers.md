### Post by geoken on 2008-06-16
I haven't tried them all so I can only give limited opinions.

I'm using the current Elisa release (0.3.5) and a build of Entertainer that I checked out a few weeks ago. Elisa was obviously easier to set up since they have .deb's and Ubuntu repositories. Entertainer had to be built and had a lot of dependencies which also had to be built.

The two have equally pleasing UI's. Both had nice themes with cool transitions. Elisa seems to be further along in terms of stability and plugins. Elisa is at what I would consider a usable state, while Entertainer was still in it's early phases and I wouldn't consider using it on my main HTPC box. 

LinuxMCE is further ahead then Elisa and Entertainer but it's UI pales in comparison. One reason I stopped Using Myth was because I didn't like it's UI compared to the simplicity of Elisa. If you care about home automation and recording TV then the others can't compare to LinuxMCE. If you don't care about these, your going to be wading through a lot of UI bloat to get to the core functionality.
