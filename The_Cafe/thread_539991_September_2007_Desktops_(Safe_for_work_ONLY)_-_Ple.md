### Post by ~LoKe on 2007-08-31
September 2007 Desktops - Posting Guidelines

* Stay within the board rules, no matter how hot you may believe the wallpaper/screen shot to be.

* If you are going to quote, there isn't a need to quote an img-embeded picture - Just don't do it. Please just reference the post by page number/page link/etc.

* Off Topic Posts will be edited or removed.

* The desktops thread is not to be used as a means to post explicit or suggestive, and provocative images (they must be safe for your kids and/or boss to see). Please help to keep the thread relevant and safe to browse for everyone.

* The moderators of this forum reserve the right to edit or remove your posts as necessary so they may comply with our Community Rule set.

* Please use reduced-size thumbnails, where possible, out of respect for people running on lower bandwidth or screen resolutions.

**[COLOR=DarkRed]*To avoid having users having to ask you for them. we request  that you format your posts in the following order:[/COLOR]**

Themes: Name/where you got it (with link would be nice).

Wallpaper: Name/where you got it (with link would be nice).

Icons: Name/where you got it (with link would be nice).

Programs: Name/where you got it (with link would be nice).

Important Note: Any image posted that is over 800 x 600 will be removed! Use a thumbnail or use a text link to your image. If your image is removed, this is why.

```
#!/bin/bash
echo ""

echo " 3  -> DI.fm - Ambient"
echo " 4  -> DI.fm - Chillout"
echo " 5  -> DI.fm - Deep House"
echo " 6  -> DI.fm - Drum 'n' Bass"
echo " 7  -> DI.fm - Lounge"
echo " 8  -> DI.fm - Trance"
echo " 9  -> DI.fm - Vocal Trance"
echo " 10 -> ETN-fm - Progressive"
echo " 11 -> Soma FM - Indie Pop Rocks"
echo ""
echo "Choose station:"
read radio
if [ "$radio" = "1" ]; then
mplayer rtsp://sr-rm.qbrick.com/broadcast/cluster/encoder/02038_p3.rm
elif [ "$radio" = "2" ]; then
mplayer rtsp://sr-rm.qbrick.com/broadcast/cluster/encoder/02038_p3rockster.rm
elif [ "$radio" = "3" ]; then
mplayer http://www.di.fm/mp3/ambient.pls
elif [ "$radio" = "4" ]; then
mplayer http://www.di.fm/mp3/chillout.pls
elif [ "$radio" = "5" ]; then
mplayer http://www.di.fm/mp3/deephouse.pls
elif [ "$radio" = "6" ]; then
mplayer http://www.di.fm/mp3/drumandbass.pls
elif [ "$radio" = "7" ]; then
mplayer http://www.di.fm/mp3/lounge.pls
elif [ "$radio" = "8" ]; then
mplayer http://www.di.fm/mp3/trance.pls
elif [ "$radio" = "9" ]; then
mplayer http://www.di.fm/mp3/vocaltrance.pls
elif [ "$radio" = "10" ]; then
mplayer http://www.etn.fm/progressive-high.m3u
elif [ "$radio" = "11" ]; then

### Post by andrek on 2007-09-23
Add:
```

    PS1='${debian_chroot:+($debian_chroot)}\[[\033[01;32m\]\w\[\033[00m\]] '
```
to your .bashrc file [ /home/username/.bashrc ], under '# set a fancy prompt (non-color, unless we know we "want" color) ...blabla' line
If you have a distro which uses 'su' command to make you root, add this to /root/.bashrc
```

export PS1='${debian_chroot:+($debian_chroot)}\[[root: \033[01;32m\]\w\[\033[00m\]] '
umask 022

```

I'm using the "screen" terminal emulator.  In .screenrc, I have this:

> #!/bin/sh

# ~/bin/nowplaying

is_playing_music=`mpc | head -2 | tail -1 | grep playing`

else
  hostname
fi

```

```
Paste in the following:
> defflow auto
defscrollback 5000
altscreen off
autodetach on
msgwait 2 # 1 second messages

# show MPD info in status bar
backtick 1 5 5 $HOME/.scripts/mpd
hardstatus alwayslastline "%{=b W}%{k} %0c          # %1` # %= %w "

vbell_msg "[[[ ding ]]]"
vbell off
startup_message off

```
mkdir .scripts && cd .scripts && gedit mpd
```
Paste in the following
> #!/bin/sh

# ~/bin/nowplaying

is_playing_music=`mpc | head -2 | tail -1 | grep playing`
if [ -n "$is_playing_music" ]; then
  mpc --format "%artist% - %title%" | head -1
else
  hostname
fi

My conky config is attached
The wireless status part is (in the conky config):
```
Wireless:  Down: ${downspeed ath0}k/s | Up: ${upspeed ath0}k/s
```
Replace ath0 with whatever your connection is.
If you need more help with conky try the [HOWTO]("http://ubuntuforums.org/showthread.php?t=205865").

### Post by herbster on 2007-09-28
> **rkakkar said:**
> hey... i'm facing another problem... if i put 'conky' in my sessions, the minute my desktop loads, it disappears.. but if i manually run conky from the terminal, it stays.. what could be the problem?

In terminal, make a script:

```
gedit conkystart
```

Paste this into it:

```
#!/bin/bash

sleep 5 &&
conky &
```

The "5" after sleep is the number of seconds you're telling it to wait before running conky. Change it to 3 or 9 or however many is fair for your start-up speed and system.

Then make it executable:

```
 sudo chmod u+x conkystart
```

You should really check out this thread: [http://ubuntuforums.org/showthread.php?t=281865](http://ubuntuforums.org/showthread.php?t=281865)

### Post by Lux Perpetua on 2007-09-30
Here's a little "themeinfo" script I just cooked up:

```
#!/usr/bin/python

import xml.etree.ElementTree as Etree
from os.path import expanduser

prefixes = {
    'theme'               : 'Metacity Theme: ',
    'titlebar_font'       : 'Titlebar Font: ',
    'gtk_theme'           : 'GTK Theme: ',
    'icon_theme'          : 'Icon Theme: ',
    'document_font_name'  : 'Document Font: ',
    'font_name'           : 'Application Font: ',
    'monospace_font_name' : 'Console Font: '
}

def text(node):
    return node.find('stringvalue').text

def display(doc, d):
    for node in doc.findall('entry'):
        name = node.attrib['name']
        if d.has_key(name):
            print d[name], text(node)

try:
    gtk = Etree.parse(expanduser('~/.gconf/desktop/gnome/interface/%gconf.xml'))
    display(gtk, prefixes)
except IOError:
    print 'No GTK information'

try:
    wm = Etree.parse(expanduser('~/.gconf/apps/metacity/general/%gconf.xml'))
    display(wm, prefixes)
except IOError:
    print 'No Metacity information'
```My output: ```
GTK Theme:  alien933k
Icon Theme:  Human
Document Font:  Luxi Sans 10
Application Font:  Luxi Sans 10
Console Font:  Luxi Mono 8.5
Metacity Theme:  BlendedSmallDoubleRound
Titlebar Font:  Luxi Sans Bold 10
```It assumes you have Metacity. It should be similarly straightforward to get it to work with other window managers, though.
