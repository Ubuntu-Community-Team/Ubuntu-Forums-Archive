### Post by flygin on 2010-01-21
thi-scripts 0.1
helper app for the command line

[http://thi-scripts.sourceforge.net/](http://thi-scripts.sourceforge.net/)
thi-scripts are a framework to get help on the console. Meanwhile the man pages explain all option of a single command, thi-scripts are intended to give guidance for processes that need one or more commands with plenty of options. thi-scripts offer the commands to the user. The user decides which commands to use.
The special thing about thi-scripts is that the user is not *inside* the application: the console is always free. The user is not *in* the application, the application is *with* the user!

have a look!
