### Post by hotani on 2006-10-18
just posted this in the wrong thread but here is a little step-by-step for installation:

Grab install file from [this adobe page](http://labs.adobe.com/downloads/flashplayer9.html). Then:
```

shell> cd /path/to/download/
shell> tar -zxf FP9_plugin_beta_101806.tar.gz 
shell> cd flash-player-plugin-9.0.21.55 
shell> sudo mv /usr/lib/firefox/plugins/libflashplayer.so libflashplayer.so.backup
shell> sudo cp libflashplayer.so /usr/lib/firefox/plugins

```

Now go [here](http://www.adobe.com/cfusion/knowledgebase/index.cfm?id=tn_15507) and check your version.

1) sudo apt-get remove flashplugin-nonfree --purge

[LIST]
[*]/usr/lib/firefox/
[*]/usr/lib/mozilla-firefox/

[/LIST]
