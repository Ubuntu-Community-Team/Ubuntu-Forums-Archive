### Post by forrestcupp on 2012-09-26
I use Google Calendar for that. It can generate code to plug into your web site. It's pretty easy to use, but I have some complaints about it. On your web site, if the day's box is too small to fit all of your text, it won't word wrap, so the entry only shows what will fit in the box. They used to allow you to put a <br> tag in the title so you could manually make it fit, but they stopped allowing that.

But other than that complaint, it is pretty easy to use for collaboration.
