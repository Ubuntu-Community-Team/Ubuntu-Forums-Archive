### Post by Dragonbite on 2009-11-05
Is there any way to get the Gnome Slab menu in Ubuntu? If you don't know, the slab menu is like what openSUSE has, where it is similar to KDE 4's main menu, but is in Gnome.

---

### Post by FuturePilot on 2009-11-05
```
sudo apt-get install gnome-main-menu
```

Right click panel > Add to Panel > Gnome Main Menu

---

### Post by Rifester on 2009-11-05
Yes, you can find it in Synaptic under gnome-main-menu.
