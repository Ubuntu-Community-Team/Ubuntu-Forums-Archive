I not only use virtualization, but I don't know what I'd do without it.

Never stop using it.

Three things about it that I find indescribably useful.
[LIST]
[*]Portability. I can move/copy my client to any computer on my network without having to reinstall on new hardware. This includes all my virtualized Windows installs.
[*]Being able to run multiple clients on one set of hardware means that I don't need to have separate machines for my mail, web, ftp, and network file share servers. This means I can isolate my external web traffic from my internal and gives me a great deal more security than running everything on one machine.
[*]Backups are as easy as zipping up the vmx folder. If the machine crashes or is hacked, I can pull my zipped machine out of backup and be back online in less than a minute.
[/LIST]

I virtualize both Windows and Ubuntu. Windows for the times when I need to work with MS Office or upload music and movies to my phone. I use several virtualized installs of Ubuntu to handle all my incoming web traffic.

Already basically said this, but my reason for virtualization is for centralizing everything on one hardware platform.  My single virtual host currently handles two Windows, and 4 Ubuntu server clients.
