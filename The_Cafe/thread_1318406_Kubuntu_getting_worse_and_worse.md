### Post by QOLIM on 2009-11-07
It wasn't really clear where this belongs, so I post it here.

Being a kubuntu user since 4 years,
the last year I have wondered why I did it when I upgraded my distibution.

Since the last versions of kubuntu it has all gone downwards: slow startup, errors, simple menus being gtk apps (I'm talking about the distro, not kde which has improved a lot).
And now I upgraded to the latest version yesterday I noticed startup time has increased alot, my brightness controls don't work anaymore (fn+f7/f8 now increases/decreases the sound volume) and the computer won't shut down/restart etc when selected in the menu.
I have to do it with console commands.
And my keyboard settings always change back when I try to edit them.

I have the impression more work is done on ubuntu but far less on kubuntu these days, what makes the quality very low these days.
If I can't fix these problems soon I think I'll have to get another distro.
