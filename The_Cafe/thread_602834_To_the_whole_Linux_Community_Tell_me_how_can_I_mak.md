### Post by ubuntu27 on 2007-11-04
icett. You can make any GNU/Linux distribution e.g Ubuntu **look** like Windows, Mac, MacOS X, and any other. 

You can change the appearance and everything, but it will still be GNU/Linux.  

If you are using UBuntu (GNOME), then in [gnome-look.org]("http://www.gnome-look.org/") you can find themes, icons and other stuff to personalize the appeareanse of your GNU/Linux.

If you are using Kubuntu (KDE), then use [kde-look.org]("http://www.kde-look.org/")


There are some GNU/Linux distribution that tries to be like Windows. [Linspire]("http://www.linspire.com/") (previously called Lindows) is one of them.

[http://www.linspire.com/](http://www.linspire.com/)

There is also [XPde]("http://www.xpde.com/"). Well, this one is just a Desktop Environment though.

[http://www.xpde.com/](http://www.xpde.com/)


Beware though, Linux distribution that tries to imitate windows not just by appearenace but by its funtions... could be dangerous just as windows is. e.g  having root account by default (something that Windows does)

If you want to just make it look like Windows, but not change the way it interacts with the user (and not lower the security), then stay away from those distros.
