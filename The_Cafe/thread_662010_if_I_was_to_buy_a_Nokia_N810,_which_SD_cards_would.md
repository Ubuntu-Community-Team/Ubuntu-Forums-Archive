### Post by DalekClock on 2008-01-08
If you're unlikely to put much on the memory card, then just an ordinary SD card will do fine. Mini and Micro SD cards are really only for mobile phones and if you're unlikely to fill over 1GB than an SDHC is too big.

As for the two devices, the N800 has an Opera based browser while the N810 uses Mozilla, which works better with most web pages.

### Post by Onyros on 2008-01-08
> **alwiap said:**
> thanks for the replies. I am actually leaning toward buying a n800 because of mainly the cost, and the fact that it can do mostly everything the n810 can (I was intrigued by the GPS, but it seems you have to pay $$ to get actual functionality like I would want).  If I don't like typing on the screen, I can buy a bluetooth keyboard.Or an USB keyboard, there's a hack for using the OTG USB for other devices (keyboards, external disks, etc).

And the only differences between the two now are physical: keyboard, GPS and the storage. N800 users can upgrade to Internet Tablet OS 2008, which has the Gecko based browser and the upgrade in CPU speed from 330MHz to 400MHz. Yep, the processor in the N800 and N810 is exactly the same, it was just downclocked in Internet Tablet OS 2007 to save energy. Even the RAM is the same (128MB).

I actually prefer (and have) an N800. I don't like the N810's keyboard (have a mini keyboard which I connect through the USB port), don't use GPS and wouldn't want to dish out additional money so I could use it, and I prefer the N800's storage model.

I use an 8GB SDHC card internally and use the external slot with a few other cards I have. The N810 has 2GB of internal storage but only one external slot available.

As it is the N810 is not supposed to be a replacement, it's just a different version. And considering the price, the N800 is very, very appealing right now.

If you do go for the bluetooth keyboard, get the Stowaway foldable keyboard. It's not as expensive as some of the alternatives and is a great complement for the N800. I preferred the USB mini keyboard especially because of UTF-8 characters I use extensively typing in Portuguese which aren't present in the Stowaway keyboard... and the fact that it was much cheaper, too.
