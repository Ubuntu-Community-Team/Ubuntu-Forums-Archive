### Post by RAV TUX on 2006-06-27
[[IMG]http://img59.imageshack.us/img59/5870/amiga9cw.jpg[/IMG]]("http://imageshack.us")

Fun test with interesting results, I am researching Amiga OS 4.0 now.

looks interesting:

[[IMG]http://img59.imageshack.us/img59/441/workbench25hw.th.jpg[/IMG]]("http://img59.imageshack.us/my.php?image=workbench25hw.jpg")

AmigaOS has spawned at least two "clone" operating systems over time.
 [AROS]("http://en.wikipedia.org/wiki/AROS"), or *Amiga Research Operating System* is an attempt to clone the AmigaOS API in a portable open-source operating system. Although not binary compatible with AmigaOS (unless running on 68k), users have reported it to be highly source code compatible.
 [MorphOS]("http://en.wikipedia.org/wiki/MorphOS") is a PowerPC native operating system, originally created when the future of the Amiga looked uncertain. It provides binary compatibility with system-friendly AmigaOS applications. A version which runs on Classic Amigas with PPC accelerator cards has been released.
 Although not strictly Amiga related, a fork of the [FreeBSD]("http://en.wikipedia.org/wiki/FreeBSD") 4.8 release, called [DragonFly BSD]("http://en.wikipedia.org/wiki/DragonFly_BSD"), has been created by a former FreeBSD developer and Amiga programmer [Matt Dillon]("http://en.wikipedia.org/wiki/Matt_Dillon_%28computer_scientist%29"). DragonFly BSD aims to make the BSD core more like the Amiga architecturally, featuring a message-passing kernel and allowing for a very efficient and virtually [mutex]("http://en.wikipedia.org/wiki/Mutual_exclusion")-free SMP design.
 [AtheOS]("http://en.wikipedia.org/wiki/AtheOS") was inspired by AmigaOS, and originally intended to be a clone of AmigaOS. [Syllable]("http://en.wikipedia.org/wiki/Syllable_%28operating_system%29") is a fork of AtheOS, and includes some AmigaOS and [BeOS]("http://en.wikipedia.org/wiki/BeOS") like qualities.
