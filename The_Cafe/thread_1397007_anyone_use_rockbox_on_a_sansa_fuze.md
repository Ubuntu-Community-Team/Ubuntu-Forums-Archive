### Post by clonne4crw on 2010-02-06
I have been running Rockbox on my V1 Fuze for quite some time now, and I love it. To answer your questions:

1. Yes, you still have to use tags on the files if you want to use the database.

2. Battery life used to be really bad with Rockbox, but with the later versions it is pretty good

3. There is no USB support for Rockbox, so when you plug it in, it loads the original firmware for charging and transferring files. When you disconnect it, it will begin that "Refreshing your media" thing, but I just hold down the Power switch until it shuts off, considering I never use the Sansa firmware anymore, and it doesn't really matter.

Last time I checked, Rockbox only works on the V1 Fuzes, so you might want to make sure that what you got is a V1.

Hope this helps!
