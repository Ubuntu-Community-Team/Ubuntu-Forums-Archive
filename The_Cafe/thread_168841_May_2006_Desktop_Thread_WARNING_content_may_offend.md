### Post by mcduck on 2006-05-06
[QUOTE=K2712]Quick question:

My screen resolution is 1600x1200 and when I take a screenshot it saves in .png format and is about 1.4 MB in size, but the upload limit is ~975 KB.

Is there a quick way around this?[/QUOTE]
sure. Here's one way to convert them easily to smaller jpg files.

First, you need to install Imagemagick:

Open terminal and run 'sudo apt-get install imagemagick'

Now, take a screenshot with 'Prt Scrn'-button (or 'Take Screenshot' from gnome's menu), save it to desktop and now you have a file called Screenshot.png sitting on your desktop. Open a terminal and run this command:

'convert ~/Desktop/Screenshot*.png -quality 86 ~/Desktop/Screenshot.jpg'

..and you'll get jpg image on your desktop, same resolution but smaller file size. :)

You can also save that command to a text file (something like ~/.screenshots2jpg.sh), then make the file executable and make a shortcut for it in your panel.. then you only have to click a button in your panel, and all screenshots on your desktop would be converted to jpg files..
