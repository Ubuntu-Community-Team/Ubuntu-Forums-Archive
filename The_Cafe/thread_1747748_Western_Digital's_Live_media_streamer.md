Anyone else using a WDTV live? I have around 500GB of media files stashed on a separate internal HDD and this is probably the easiest way to watch your media files on a decent (read: a lot bigger) screen, unless you built a HTPC for this purpose. Virtually anything from DVD res to 1080p media works, apart from .avi files with too many warpoints and/or QPel. As you'd expect it runs a custom Linux kernel on a Sigma 8655 processor, and you can get the source code for it too.
