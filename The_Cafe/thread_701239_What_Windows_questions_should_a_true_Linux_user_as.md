### Post by Sunnz on 2008-02-20
Well the filesystem thing is a bit harsh, Linux itself can't be installed on a NTFS partition, of course that's not really Linux's fault as NTFS lacks proper documentation... but for the record you cannot install Linux's boot partition on anything else other than ext2/3 neither. (Installing the boot partition on reiserfs does not work.)

However there is a freeware driver for ext2 files system for Windows, fs-driver. It can be used to mount ext3 partitions, since ext3 is just ext2 with added journaling... not sure if it has a free license though, it does not seem to be open source.
