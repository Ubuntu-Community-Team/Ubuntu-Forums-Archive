### Post by chessnerd on 2010-10-18
Well, if you want some practice with little coding projects in Java I'd go work on some Coding Bat exercises ([http://codingbat.com/java](http://codingbat.com/java)) and do the warm-ups. After you've gotten back into the swing of things, I'd start with simple projects.

A good project dealing with simple GUI coding is to make a Notepad or Calculator app. If you want something more dealing with graphics and such, you could try cloning an old game like Space Invaders.
