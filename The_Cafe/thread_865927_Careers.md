### Post by stmiller on 2008-07-22
Tech support - every company, university, institution, government, business, everything has IT support staff. Most places run Linux servers for different stuff, so having some Linux chops may help get a job.

Sys admin - if you are good with Linux on the server side, there are admin jobs out there to do Linux full time. Apache, Sendmail, DNS, all of that stuff.

Places to look for job openings? If you are in the US, craigslist.org is good. Also going to local LUG meetings are good for finding a job.

It's a windows world for lots of stuff out there, but in the server room the machines are all running Linux or Solaris. So having Linux chops theses days is very valuable.
