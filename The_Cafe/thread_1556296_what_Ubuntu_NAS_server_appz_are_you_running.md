### Post by handy on 2010-08-19
Your question is ambiguous to the point of being incorrect.

You need to be more careful with the definitions of the words you are using.

As far as NAS is concerned, I used FreeNAS for well over a year, via NFS on my LAN. As a system it served me perfectly.

I stopped using it for no other reason than the machine I was using it on is an Athlon64 3500+, which was using far more electricity than I wanted to.

Currently I use a Netgear ReadyNAS Duo, with one 1.5TB drive in it for the purpose of system backup. It is also a torrent slave, a job it does very well. Since it uses only 4W of electricity at idle it is a very cost effective NAS/torrent slave solution.
