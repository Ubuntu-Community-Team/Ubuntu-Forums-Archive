### Post by MonkeyBoy on 2007-05-26
I was just in town and happened to find a magazine called "Build a Great Website" which was headed "Complete Toolkit For Windows & Linux"

Intrigued, I had a quick look and discovered that the first couple of pages detail how to install VirtualBox onto Windows and then how to run Ubuntu on it!

The rest of the magazine is a load of NVU and Gimp howtos with some PHP, Ruby, etc stuff thrown in for good measure.

Having thought about it this is a legendary bit of marketing as it comes with Ubuntu on disk and definitely does what it says on the front but insists on making windows users run a virtual Ubuntu desktop.

I was so impressed I bought it.
