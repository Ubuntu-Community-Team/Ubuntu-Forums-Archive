There are some things that PCManFM can't do (like extracting .rar files, opening the trash folder with the empty trash button, etc), but it does most of the things I need a file browser should do.

I can make Nautilus simple by disabling some features ( [http://library.gnome.org/admin/system-admin-guide/2.22/performance-8.html.en#performance-7](http://library.gnome.org/admin/system-admin-guide/2.22/performance-8.html.en#performance-7) ). On the other hand, I don't want to keep disabling features. I'll just use PCManFM if I'm going to do simple things.
