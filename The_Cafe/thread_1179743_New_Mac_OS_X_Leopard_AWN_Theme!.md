### Post by Tipped OuT on 2009-06-06
I wasn't sure where to post this really. But I wanted to share this with everyone, this is my creation, the first ever actual OS X Leopard dock theme, for AWN. This is version 1.00, so I'm planning on making updates as time goes on, and I hope people will actually use this for there Mac OS X themes. I would really like to spread this around to people, if you know any way I could, please let me now as soon as possible. Thanks, enjoy. :)



[U]
Installation[/U]:
-Extract the zip file.
-Right click on the extracted folder and select "Create Archive".
-Create a .tar.gz
-Add the .tar.gz file to AWN in the "Themes" section and enjoy.

**NOTE: The theme was designed for 1280 x 800 resolution. If the theme does not look as indented, please make a request for your resolution and it shall be done.**
