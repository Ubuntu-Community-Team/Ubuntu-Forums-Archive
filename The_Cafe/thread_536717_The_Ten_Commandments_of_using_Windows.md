### Post by izanbardprince on 2007-08-28
I listed them in Epinions thing's to know about Windows section:

[http://www.epinions.com/content_5087535236](http://www.epinions.com/content_5087535236)


1. I am the operating system, thy Windows, ye shalt have no other operating systems installed but me, and thou shalt not curseth when thy Linux partition is nukethed upon a Windows upgrade. 

2. Thou shalt read and agree to the eight pages of Ye Olde End User License Agreement, if thou hast enough papyrus, thou may print the sacred text in it's entirety. 

3. Thou shalt always use Internet Explorer, even when thou browseth thine unclean sites, for the true believer worries not about using an anti-virus suite, if thou dost load thine browser of choice, thou shalt have to continue using ye olde Internet Explorer's interface to browse thine hard disk. 

4. Thou shalt not play thine DVD's, lest ye have to purchase a DVD codec for the sum of forty gold pieces, thou shalt also not play thine unclean Ogg Vorbis media, or encode unto. 

5. Thou shalt submit to Windows Genuine Advantage when thou downloadest add-on's from Microsoft, even if it wanteth to eat thine babies and givest thou's operating system a rectal probing whenever it wanteth, thou shalt not disable the many system services that connect to Microsoft servers and transmit encrypted data packets, lest ye be smitten by the almighty Digital Millenium Copyright Act. 

6. Thou shalt never get sicketh of the malfunctioning anti-piracy software, and be driven to actually pirate a copy of Windows just to evade this subsystem. 

7. When thou browseth thine installed software, thou should maketh sure that every other entry is some kind of security software. 

8. Thou shalt never complain about having to use low level disk maintenence tools to correct problems that a modern file system should not have. 

9. Thou shalt never complain about how difficult it is to actually runneth a 64-bit Windows operating system on thine 64-bit processor. 

10. Thou shalt pay the sum of ninety-nine, to three-hundred ninety-nine gold pieces every time Microsoft releaseth another monstrosity, and thou shalt never complain when thine license cannot be transferred to another machine, nor should thou attempt to improve thine operating system beyond what Microsoft has seen fit to provide, also upon receiving the new version of Windows, thou may not question why thou would needest double the system RAM the last version demanded.
