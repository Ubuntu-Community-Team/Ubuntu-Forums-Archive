### Post by skymera on 2009-04-28
To get the GNOME default notification messages do:

```
 sudo apt-get install gnome-stracciatella-session 
```

Logout (or restart X) 
Click options (bottom left usually) > Select Session > GNOME (without Ubuntu extensions (or similar))
