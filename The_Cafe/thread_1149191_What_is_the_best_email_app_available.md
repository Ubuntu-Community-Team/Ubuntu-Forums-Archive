### Post by Mehall on 2009-05-05
I like Thunderbird.

It can be setup to use independent inboxes for POP3 accounts, but it automatically separates all IMAP accounts, and I prefer IMAP anyways, so I use all my accounts (except hotmail which doesn't support it) via IMAP to keep them separate, while still only needing one app to access them all at one time.

Evolution probably does the same, but TB has the cross-platform advantage, so i ahve TB Portable on my pendrive for when stuck on windows.
