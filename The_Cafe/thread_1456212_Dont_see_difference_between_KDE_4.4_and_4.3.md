### Post by RiceMonster on 2010-04-16
Here's some different things off the top of my head:

- Automatic usb drive mounting
- "aero snap" like feature
- Tabbed windows
- wallpaper can be changed via drag and drop
- Mouse bidings on desktop can be changed
- Plasmoids can be put in the system tray
- Plasma theme dialogue is now under appearance settings
- Date and time can be changed from the clock plasmoid


I guess it is a bit of a maintenance release, but I think it improved a lot of the gripes I had with 4.3
