### Post by James7 on 2010-06-05
brightside is a cool bit of software that makes your corners 'hot'.

I use it so that if I put my mouse pointer in the upper left corner, the screensaver kicks in; if I put it in the upper right, the volume is muted.

Downloading on Ubuntu is easy:

```
sudo apt-get install brightside
```

At the CLI, use 'brightside-properties' to set the properties. If you want it to run automatically on startup, put it in System/Preferences/Startup Applications.
