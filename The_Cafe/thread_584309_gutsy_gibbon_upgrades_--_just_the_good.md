### Post by tyggna1 on 2007-10-20
Now that Gutsy has hit mainstream--and been out for a few days, I think it's time we talk about what we like about it.   If you want to complain, go [here]("http://ubuntuforums.org/showthread.php?p=3586383").   For this thread, just post what's different about Gutsy that you like.


I'll start:
Wireless works!
Improved Bluetooth support (don't use, but nice to know it's there)
Smaller kernel = less RAM overhead usage for OS.
More GUI-based troubleshooting tools (device manager, better network manager, awesome printer management--ect)
Dual-screen support made easy, and GUI.
Wine installed by default (great for new converts)
Bug report tool in applications menu (as opposed to previous versions, it was hard to find, and compare to microsoft where it doesn't give you a choice to report a bug or not, this is a way nice feature)

Improved open-source java compiler -- not fantastic, but much better than the feisty version
