### Post by Eisenwinter on 2009-06-14
That depends on your experience.

I recommend Arch. Only the basic components will be installed (kernel, shell, package manager and some programming tools), you add everything else (including X and a window manager and etc) by yourself.

The key is minimalism. The system suits your needs exactly, because you're the one who builds it.

### Post by Pogeymanz on 2009-06-14
The beautiful thing here is that you can always reformat/go back to Ubuntu, so try a really different one. I don't recommend Debian because it's too similar to Ubuntu. I recommend Slackware, Arch, Gentoo, because they are very different from Ubuntu. Maybe even something like OpenSUSE or Mandriva, just to see a different take on how desktop Linux should be.

Personally I tried Arch to learn more about Linux, but I ended up liking it MUCH more than Ubuntu, so I switched permanently.
