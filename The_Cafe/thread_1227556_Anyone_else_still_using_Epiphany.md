### Post by Katalog on 2009-08-05
Just noticed that they had switched to the 2.27 branch in the epiphany-webkit PPA. Since Midori had been working out so well for me lately, I decided to give it another shot. I purged epiphany-gecko from my system, added the PPA to my sources and installed the webkit version. Right now it's at version 2.27.5. I've been using it for about an hour and a half now, and so far no crashes (which is a first, compared to the other times I've tried epiphany-webkit in the past). 

Some new changes in this version: 
Finally able to customize toolbars.
Fullscreen mode actually works without any crashing or screen tearing.
Scrolling is much smoother.
Haven't yet noticed any strange rendering on any of the sites I've visited so far.
Can't recall if this is really new and worked in past versions as well (been a while I've tested it out) but smart bookmarks seen to behave in the same manner as the gecko version

Things which have still not yet been implemented:
[s]No extensions (though I hear they are on the way soon, which I hope is true as I miss them terribly).[/s]
Still doesn't remember passwords, even though the feature is present in the Personal Data menu.

Otherwise, solid as a rock and *very* quick so far and I'm typing this post from it right now. We'll see how it holds up over the next week or so, but I hope they dump the tired old gecko version and make this the default very soon. Hopefully by the time Karmic hits the streets we'll finally see a full featured version, extensions and all.

**EDIT:** I was wrong! There *is* a set of webkit extensions available. Not quite as many as the gecko version, but they're still there. Time for more testing!
