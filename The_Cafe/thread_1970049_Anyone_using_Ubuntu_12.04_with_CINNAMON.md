### Post by Uncle Spellbinder on 2012-04-30
There's a Cinnamon PPA for Ubuntu 12.04. A stable and unstable PPA. I'm running Cinnamon stable.

```
#### Cinnamon [Stable]
## sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 28949509
deb http://ppa.launchpad.net/gwendal-lebihan-dev/cinnamon-stable/ubuntu precise main 
deb-src http://ppa.launchpad.net/gwendal-lebihan-dev/cinnamon-stable/ubuntu precise main 
```

Cinnamon really feels nice. Looks great as well.
