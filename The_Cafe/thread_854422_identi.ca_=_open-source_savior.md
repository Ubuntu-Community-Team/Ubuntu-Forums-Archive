### Post by shanepardue on 2008-07-09
Thank goodness someone is here to save us from the outages and spam of Twitter! [Identi.ca]("http://identi.ca") is an open-source micro-blogging platform that is poised to be a Twitter-killer (hopefully) and I'm pretty excited about it. Here are some of the features they hope to implement soon.

    * SMS updates and notifications
    * A Twitter-compatible API
    * More AJAX-y interface
    * Maps
    * Cross-post to Twitter, Pownce, Jaiku, etc.
    * Pull messages from Twitter, Pownce, Jaiku, etc.
    * Facebook integration
    * Hashtags
    * Image, video, audio notices
    * Automatic url-shortening
    * Multilingual interface (using Gettext)


My username is HeadlessHorseman, please share your usernames and help get this project the attention it deserves!

### Post by atoc on 2008-08-02
GWibber is awesome. VERY customizable.

to install it -
install bzr from the repos.
Also make sure these are installed 
[LIST]
[*]python (2.5)
[*]python-dbus (0.80.2)
[*]python-gtk2 (2.10.4)
[*]python-gconf (2.18.0)
[*]python-notify (0.1.1)
[*]python-simplejson (1.7.3)
[/LIST]
In my case the only thing I had to install was python-simplejson. The rest were already installed.

run this at the command line:
```
bzr branch lp:gwibber
```

Follow the instructions in the install doc in the gwibber folder.

I didn't bother installing it. I just run it from my home folder.
