### Post by presence1960 on 2009-05-12
> **mafaldaspeaks said:**
> How is this separate partitions done so I can do it for the next time ubuntu releases a new version?

When ready to do clean install: First backup all your data. Then you want to repartition your Ubuntu partition. Use an Ubuntu Live CD or gparted Live CD to delete your Ubuntu partition. Then create 3 partitions that will be used as "/" , swap & "/home" for the install. (If you already have a swap partition you may want to keep it. If not  right click on swap partition and select swap off. Then delete the partition.) The sizes of the partitions will be determined by your needs. Click apply and create them. When you go to install Ubuntu choose "manual" when you get to the partitioner section of the install process. Now choose each of those partitions by highlighting them and choosing Edit. Choose the proper mount point for "/" which is "/", choose "/home" as the mountpoint for the separate home partition and designate the swap partition as swap for file type. You can use ext 3 or ext 4 (or any of the other Linux file systems for that matter) for file type of "/" and /home. When install is complete transfer your data you want in home from your backup.

Now if you ever do a clean install again your /home will be left intact by choosing "manual" at the partition screen again. Install Ubuntu to the partition used as "/" on the old Ubuntu. Choose your home partition by highlighting it and choose Edit. Select the mountpoint as /home. This will insure your home partition automounts when you start Ubuntu and that another home is not created in "/". **_Be very careful to make sure the format partition box is NOT TICKED for the /home partition._** This will preserve your data on /home. **_Always have a backup anyway just in case something goes wrong-ALWAYS-ALWAYS backup!_**. Proceed with the install.

This should have you set up good.

Here is a link for partitioning basics and how to's : [https://help.ubuntu.com/community/HowtoPartition](https://help.ubuntu.com/community/HowtoPartition)

here is a guide from psychocats on creating a separate home partition on a current install that does not have one: [http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)
