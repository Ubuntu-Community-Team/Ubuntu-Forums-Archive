### Post by stmiller on 2009-01-25
No viruses, but out-of-date packages do have security vulnerabilities. If you are behind a NAT router at home, you are pretty much safe.

Though after managing some public facing Linux machines, there are people trying scripts and backdoor hacks 24/7 on these machines I have found. So while there are no viruses like Windows viruses that come via email attachments, if you are behind in security updates or careless about security with any OS, you can have issues.

There is a Linux program rkhunter that searches for known scripts, rootkits, and backdoor hacks. It's pretty cool.
```

sudo apt-get install rkhunter

```

update it with:
```

sudo rkhunter --update

```

run it with:
```

sudo rkhunter -c -l

```
