### Post by kevdog on 2008-07-20
Id like to make a list of all the pidgin plugins available (a master list). If submitting new plugins, it would be great if you could submit the source page or project page along with any known documentation of how to install the plugin.

Here are the ones I know about:

**OTR (Off-The-Record)** [http://www.cypherpunks.ca/otr/](http://www.cypherpunks.ca/otr/) -- need libotr
**Pidgin Encryption** - [http://gaim-encryption.sourceforge.net](http://gaim-encryption.sourceforge.net)
**Pidgin Extended Preferences** - [http://gaim-extprefs.sourceforge.net/](http://gaim-extprefs.sourceforge.net/)
**Purple Plugin Pack (51 various Plugins)**: [http://plugins.guifications.org/trac/wiki/PluginPack](http://plugins.guifications.org/trac/wiki/PluginPack)
**Bot Sentry - Prevents IM SPAM** - [http://sourceforge.net/projects/pidgin-bs/](http://sourceforge.net/projects/pidgin-bs/)
**Pidgin RhythmBox Plugin** - [http://jon.oberheide.org/projects/pidgin-rhythmbox/](http://jon.oberheide.org/projects/pidgin-rhythmbox/) [http://ubuntuforums.org/showthread.php?t=622264](http://ubuntuforums.org/showthread.php?t=622264)
