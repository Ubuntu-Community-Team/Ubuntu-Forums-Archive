### Post by Old_Grey_Wolf on 2010-01-09
There is a Perl script that creates HTML webpages it is WebbedGed [http://www.rawlins.org/Pedigree/WebbedGed.html](http://www.rawlins.org/Pedigree/WebbedGed.html).

Since it is Perl, you can customize the code. The Perl script I use was a modification of one of the scripts predecessors. The predecessor is mentioned in the GPL License.

First modification I made was to replace the navigation buttons with my standard navigation header that the rest of my website uses. I also added an "email me" in JavaScript (SPAM deterrent) to the bottom of each page it generates.
