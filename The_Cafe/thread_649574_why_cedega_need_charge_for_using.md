### Post by Ceemee on 2007-12-25
> **horry said:**
> i am new guy. i have a confuse about cedega. the softwares should be free for using on linux, isn't it? why cedega need charge, is it conflict with the spirit of GNU?

The spirit of GNU is that software be free to use how you see fit, and modify how you wish.

Cedega isnt charging you for the software, its charging you for continual updates, support, and for the promise of continual rapid development.

Cedega is exactly the kind of buisness model that Open Source Software supports and promotes.  Verses a Microsoft style buisness model - ie: Sell the software once, only patch security issues, and make the user purchase a large upgraded software next time for up-to-date features.

Yes your still paying for the most up-to-date features, however if MS went out of buisness tomorrow no one could pick up where they left off and keep supporting the software - anyone could pick up where Cedega leaves off.
