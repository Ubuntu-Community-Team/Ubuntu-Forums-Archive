Hi all,

I wanted to let you know that [Wubi](http://www.cutlersoftware.com/ubuntusetup/wubi/en-US/index.html) is now in beta. Wubi is a Ubuntu (7.04) installer for Windows. If you still have Windows XP/2K laying around and a bit of free disk space (no need for a dedicated partition or CD rom), give it a spin and let us know how it goes! If you need to report a bug pls use [this thread](http://www.ubuntuforums.org/showthread.php?t=338279). Feedback welcome!
