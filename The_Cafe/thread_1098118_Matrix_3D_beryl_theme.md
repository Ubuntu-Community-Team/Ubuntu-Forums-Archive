### Post by steveneddy on 2010-06-22
Goodness people - again, **Beryl is [COLOR="Red"]DEAD[/COLOR]** - and has been merged with Compiz Fusion which is INSTALLED BY DEFAULT on all Ubuntu releases since 8.04 Hardy Heron.

Beryl started as a branch of Compiz and a couple of years later Compiz and Beryl re-merged to form what we know today as Compiz Fusion.

[http://wiki.compiz.org/AboutCompizFusion](http://wiki.compiz.org/AboutCompizFusion)

[http://www.compiz.org/](http://www.compiz.org/)

The best place to find a load of Ubuntu and Compiz themes is

[http://www.gnome-look.org/](http://www.gnome-look.org/)

[Here is a Matrix theme]("http://gnome-look.org/content/show.php/Pwnage+Matrix+Conversion+Pack?content=123039") to start you off - be sure you read the **Read Me** file to help you install it.

Also open Synaptic and search for compiz by typing **compiz** in the main window.

You will probably want to install compizconfig settings manager by copying and pasting this into your terminal:

```
sudo apt-get install compizconfig-settings-manager
```

After installation you will find in in 

System --> Preferences

This will show you all of the options in Compiz and help you get the eye candy you are seeking set up.
