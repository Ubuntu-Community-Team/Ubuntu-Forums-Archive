### Post by RAV TUX on 2007-07-15
> Here are the first two screenshots from the upcoming release of SlackBot 0.6.3 with initial (and still experimental) support for Wolvix GNU/Linux. 

Wolvix 'Hunter' 1.1.0 RC2 requires only a few extra packages that can be downloaded by SlackBot using slapt-get (package management utility included in Wolvix). After those development packages have been installed SlackBot will compile fine. If you want to give it a try you must wait for either RC3 or the final Wolvix release since one package needs to be updated. Right now only the minimal GNOME install has been tested, but more tests will follow after SlackBot 0.6.3 is out later today :-)[http://www.mkanet.de/](http://www.mkanet.de/)
