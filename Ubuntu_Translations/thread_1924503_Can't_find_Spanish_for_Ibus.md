### Post by AV-2 on 2012-06-02
Hello, 
Since this is the first, and so far only relevant, result on Google, I am bumping this old topics to report my findings:

IBus purpose is to manage input from keyboards completely different than the latin set, (like chinese).

As described in this bug:
[https://bugs.launchpad.net/ubuntu/+bug/637766](https://bugs.launchpad.net/ubuntu/+bug/637766)

The package that adds languages like french, is kinda broken. If you want to input Spanish, you are going to be better served by the system default keyboard input manager, where there is an ample repertory of languages to pick from.
