### Post by Isamu715 on 2012-10-31
Language packs usually should get updated along with other system updates. If you want to update the language pack only use:
```
sudo apt-get install language-pack-xx
```Remember to replace "xx" with your desired language code(like "en" for English, "de" for German, etc.). It will either tell you the package is already at the latest version or upgrade if a newer one is available.
