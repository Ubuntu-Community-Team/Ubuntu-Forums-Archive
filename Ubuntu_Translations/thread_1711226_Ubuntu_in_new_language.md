### Post by dpm on 2011-03-22
If yours is a separate language and you can justify it is, you can always create a new translation team and request its addition to the Ubuntu Translators group. This is the first requirement for adding new languages to the distro.

Here are some simple instructions on how to start a translation team:

[https://wiki.ubuntu.com/Translations/KnowledgeBase/StartingTeam](https://wiki.ubuntu.com/Translations/KnowledgeBase/StartingTeam)
