Hi,

You can start translating Ubuntu into Slovenian here:

[https://translations.launchpad.net/ubuntu/maverick/+lang/sl](https://translations.launchpad.net/ubuntu/maverick/+lang/sl)

However, I recommend you to get in touch with the Slovenian team first:

  [https://launchpad.net/~ubuntu-l10n-sl](https://launchpad.net/~ubuntu-l10n-sl)

You'll also find more information about the team on this article:

[http://ubuntu-news.org/2010/11/12/ubuntu-translations-interviews-andrej-znidarsic-slovenian-translation-team/](http://ubuntu-news.org/2010/11/12/ubuntu-translations-interviews-andrej-znidarsic-slovenian-translation-team/)
