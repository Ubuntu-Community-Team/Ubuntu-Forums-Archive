### Post by Sylos on 2012-12-09
As far as I am aware there isnt a final version of Ardour 3 (yet) so it hasnt been packaged. I think FalkTX may have packaged it in his PPA but I havent tried it. 

If you search Ardour 3 beta 5 in google then you should find the beta 5 available from Ardours own website. The file you download will do all the installation for you and install itself alongside Ardour 2 (meaning you wont loose ardour 2). It worked for me with beta 3 (although I had problems loading saved sessions - a problem that has likely been fixed). 

Its beta so the usual disclaimers will apply.
