### Post by Nosphky on 2016-08-14
After making a clean install of 16.04.1, some extensions I restored to LibreOffice Writer would no longer work.  I also installed the sigil package and this would not show human readable html code.

These difficulties were entirely related to missing java runtime (jre) and missing libreoffice-java-common.

I am surprised these 2 packages were not included as standard and I don't recall having had the same problem with 14.04. Perhaps they are not considered 'free' software although I thought I had selected to install non-free components.

Also, it's rather strange that java was not shown as a dependency when I selected the Sigil package in Synaptic (which itself was not included in 16.04.1).

I just add this thread in case others have similar problems.
