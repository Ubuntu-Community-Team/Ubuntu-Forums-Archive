Flexbuilder works with Eclipse, an IDE that requires Java. In other words, you should install the latest Java, then the latest Eclipse, and then you can probably install Flex Builder on your Eclipse IDE.

Flex is a very different way to develop Flash. If you don't mind approaching Flash development like you might approach developing an application (writing code without much of a GUI), you might like it.
