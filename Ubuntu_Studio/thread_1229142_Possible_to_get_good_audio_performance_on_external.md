### Post by kayosiii on 2009-08-02
It will work OK up to a point. I run everything here from an external (esata) and it works fine. I have run from USB before the main difference is the boot time. It won't work as well as internal drives if you want to record many audio tracks simultaneously and tasks where HDD IO is the bottleneck could be noticably slower (Video decoding/encoding could be effected).
