### Post by Stochastic on 2009-07-10
It looks like you've run into this bug: [https://bugs.launchpad.net/ubuntu/+source/idjc/+bug/355222](https://bugs.launchpad.net/ubuntu/+source/idjc/+bug/355222)

and it looks like someone has packaged a newer version of idjc that functions normally in Jaunty.  It can be downloaded from this repo:
deb [http://ppa.launchpad.net/bjo-nord-west/ppa/ubuntu](http://ppa.launchpad.net/bjo-nord-west/ppa/ubuntu) jaunty main

On another note, you may want to look into other dj software if you keep having issues with idjc.  Some popular ones include Mixxx, DJPlay, and Xwax.
