### Post by prokoudine on 2010-03-02
Hey,

It looks like [Ramen]("http://ramenhdr.sourceforge.net"), the only free standalone node-based compositing tool, is not quite well-known yet. Which is a shame, because it's a quite handy tool written by a guy who's been doing VFX professionally for 10+ years.

Anyway, if you are not quite familiar with it, here is a review + tutorial: 
[http://www.libregraphicsworld.org/articles.php?article_id=14](http://www.libregraphicsworld.org/articles.php?article_id=14)

I think it's too late for the next UbuntuStudio, but at least there is a PPA with (sort of) nightly builds: [https://launchpad.net/~gabriel-inv/+archive/ramenhdr](https://launchpad.net/~gabriel-inv/+archive/ramenhdr)

BTW, the project will apply for participation in GSoC.
