### Post by rylleman on 2008-10-23
Digikam is very good for organizing your photo collection.
It has very good tagging abilities and it writes the tags directly to the image files so you won't loose them if you change software or do some editing. (All other organizing softw. saves the tags separate from the images in databases making them specific for that software and image. If you edit an image and moves it the tags for the copy will then be lost. With digikam they follow the image).
With properly tagged images you find the one you're looking for in seconds.
It also has basic image editing capabilities and RAW format support.

### Post by mutzinet on 2008-11-06
Digikam is in my opinion unbeatable for organizing pictures.
Qtpfsgui is a fantastic tool for HDR photography.

There are also a bunch of commercial software available for linux that I use on a daily basis:
- Bibble (raw editing, important update expected at the end of the year)
- Lightzone (great interface for postprocessing photographs. Lightzone is also a powerful raw editing software, but less "workflow oriented" than Bibble)
- AutopanoPro (very efficient to automatically stitch handheld shot panoramas)
