### Post by eye208 on 2008-06-19
> **Stochastic said:**
> .avi == yes.
.ogg == maybe, depending on the computer/player (it can be done, but if it's a public computer, the proper codecs might not be installed).
The answer is "maybe" in both cases because you need codecs for the AVI too (Xvid, ffdshow etc.). Installing a codec requires admin rights, but VLC can be run anywhere and will play pretty much anything.

If you care about both file size and quality, the best option is H.264 video and Vorbis audio in Matroska (.mkv) containers.
