Hmmm, I think you're talking about Multisession discs.

The CD has been finalized, I don't think it's possible to write on it anymore.


If you plan on burning small files then re-using the disc, then you should choose not to finalize the cd and make it a multisession.  You can add and delete files in multiple sessions this way.
