### Post by babarosa on 2009-11-08
Hi niceguy123,

just for your information, here [http://kompozer.net/download.php](http://kompozer.net/download.php) you can download the latest version of kompozer with your language pack already included.

On the bottom of the site you can download the dictionary for kompozer. You can download Handcoder, which you will need to launch a text-editor to work on the html- respectively CSS-code, get it from here [http://kompozer-web.de/forum/download/file.php?id=262](http://kompozer-web.de/forum/download/file.php?id=262)
For cleaning the code before editing it with an external text-editor via handcoder, install the package "tidy" - it is in the repos.

Now to your question:
Unpack and copy your complete folder "kompozer" for example to /usr/local/share as root.
Generate a starter with a link to the file "kompozer" in the folder "kompozer". Icons can be found at /kompozer/icons.
