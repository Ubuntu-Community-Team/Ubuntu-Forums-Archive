### Post by jejeman on 2013-11-04
If you want to mix audio recording and MIDI, than LMMS is not good. I don't use it, but as I saw it once there is no audio recording in it. And, some how, that is only program that has instruments as plugins.

I use Muse as for audio recording and MIDI sequencing; all instruments are in standalone instance of linuxsampler, plus stand alone synth Yoshimi. And some stand alone plugins...
Another program to use as audio and MIDI sequencer is Qtractor. Ardour 3 has this functionality, but MIDI is still not that good.

So, no really easy of use as Logic with plugins, you will probably need to use standalone programs, and with use of stand alone programs you need to find a way to manage sessions, and that is another story...

### Post by su:bhatta on 2013-11-05
Between jejeman & cub they have covered most of it. 
But I would like to add  that, I prefer Ardour. 

Also, if you are looking for midi accompaniments (like Band-in-a-box) you can try **a program called MMA** or Musical Midi Accompaniment: it is available as a debian package(installable through Software-Center) from here: [URL="http://www.mellowood.ca/mma"]www.mellowood.ca/mma

**But MMA is a command line program**, If you want to use a** GUI to MMA you can download and compile either Linuxband** from here: [/URL]

[http://linuxband.org](http://linuxband.org)

or you can download** Lemma** from here : [URL="http://welltemperedstudio.wordpress.com/code/lemma/"]http://welltemperedstudio.wordpress.com/code/lemma/

For simple midi recordings I have found Audacity to be quiet effective & as luctor pointed out Rosegarden is also pretty neat.
