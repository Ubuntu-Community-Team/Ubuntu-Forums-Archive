### Post by Dougie187 on 2008-08-27
Here is a link.
[http://www.ubustu.com/globe/2007/05/23/add-ubuntu-studio-to-an-existing-ubuntu-install/](http://www.ubustu.com/globe/2007/05/23/add-ubuntu-studio-to-an-existing-ubuntu-install/)

More simply:
Just type the following into a terminal.
```
sudo apt-get install ubuntustudio-desktop ubuntustudio-audio ubuntustudio-audio-plugins ubuntustudio-controls
```

There are also two packages for graphics and video called ubuntustudio-video ubuntustudio-graphics but i figured that you would just want the audio ones.
