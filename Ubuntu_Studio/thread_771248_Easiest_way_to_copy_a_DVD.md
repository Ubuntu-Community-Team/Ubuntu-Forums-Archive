### Post by unoodles on 2008-04-27
if you are planning on re-burning this, you should just create a .iso image with gnome-baker or brasero. If you are archiving it, you should use k9copy or dvd95. If you want a video file like an .avi, .mpg or .ogg, use dvdrip or acidrip
