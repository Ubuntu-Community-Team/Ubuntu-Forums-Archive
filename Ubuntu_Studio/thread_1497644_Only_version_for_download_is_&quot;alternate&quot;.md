### Post by Scott L on 2010-05-31
Yes, at this time the Ubuntu Studio developers only support the "alternate" installation ISO.

To explain more, the term "alternate" does not refer to the scope or content of what is installed, rather it describes the *mode* of installation.  In this case, this is the text based installer in contrast to the GUI installer.
