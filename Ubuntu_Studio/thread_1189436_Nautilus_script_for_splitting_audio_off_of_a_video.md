### Post by epicoder on 2009-06-16
This is a simple shell script that splits the audio off of a video.

To use it, set the permissions of the file to executable and move it to ~/.gnome2/nautilus-scripts. Right click on a video and choose Scripts -> splitaudio.sh. A file called out.mp3 will appear on your desktop.
NOTE: This script requires MPlayer.

This script is small enough that I don't care what you do with it. Don't bother giving me credit.
