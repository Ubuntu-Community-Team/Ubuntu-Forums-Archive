### Post by AutoStatic on 2009-12-22
Hello wbm, the simplest way would be to use **aconnectgui**. You can install it with Synaptic, it will install in Applications - Sound & Video - Aconnectgui. If you run it you can hook up your Midi keyboard to ZynAddSubFX as shown in the attachment. I used a virtual keyboard in this case, just replace it with your Keystation. In case you don't have ZynAddSubFX, it's in Synaptic also.
Step by step:
- Start Aconnectgui
- Start ZynAddSubFX by pressing Alt+F2 and entering the command **pasuspender -- zynaddsubfx**
- Press the cable icon in Alsa Sequencer
- Connect ZynAddSubFX to your Keystation with your left mousebutton
