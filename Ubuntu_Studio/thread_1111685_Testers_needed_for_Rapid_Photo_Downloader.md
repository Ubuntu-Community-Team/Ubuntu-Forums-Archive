### Post by ngaio on 2009-03-30
Greetings fellow photographers,

[Rapid Photo Downloader]("http://damonlynch.net/rapid/") is an application for professional and amateur photographers, designed for use on the GNOME 2 desktop. It can download photos from multiple memory cards and portable storage devices simultaneously. It provides many options for subfolder creation, image renaming, and backup.

[IMG]http://damonlynch.net/rapid/screenshots/download.png[/IMG]

[IMG]http://damonlynch.net/rapid/screenshots/prefs-rename.png[/IMG]

[IMG]http://damonlynch.net/rapid/screenshots/prefs-backup.png[/IMG]

Ubuntu packages are available for testing. The program is stable. 

Some links:
[Home page and screenshots]("http://damonlynch.net/rapid/")
[Documentation]("http://damonlynch.net/rapid/documentation/")
[Download]("http://damonlynch.net/rapid/download.html")
[Bug reporting]("https://bugs.launchpad.net/rapid")

All feedback is welcome. Together we can make this project a success.
