Such a request would be best presented to the Rosegarden mailing lists.  Most likely a forum of that nature should be hosted on the Rosegarden website (a la ardour's forums).  Only if the Rosegarden devs disagree with the need/cost/benefits ratio would the forum need a generous third party to step in and volunteer a community forum for it - and this would best be spawned from the Rosegarden mailing lists.

These forums have yet to set a precedent of creating any sub forums for specific applications, so I doubt it's here that you should be making the request.  There are however, many Rosegarden users here that would be happy to help you with your problem if you describe it in a separate thread (the title of this one really wouldn't suit a troubleshooting thread).
