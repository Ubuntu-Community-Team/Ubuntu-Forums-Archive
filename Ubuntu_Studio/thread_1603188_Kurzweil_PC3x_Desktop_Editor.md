### Post by trulan on 2010-10-23
It wouldn't be possible to use their software directly on Linux, no.  You would either need to use Wine or run it inside a windows virtual machine or something like that.  But, a quick search of the Wine AppDB database turned up no information on that program, so I doubt it would work even under wine without a lot of effort.

There's a good chance that your Kurzweil would work very well with some of the excellent audio production apps available natively in Linux though, such as QTractor or Rosegarden just to name a few.  I realize that's probably not the answer you were looking for though...

First of all, try running it from WINE, of course. If it doesn't work, the only "native" project I know of is [KurzFiler]("http://kurzfiler.sourceforge.net/"). It's a Java editor of Kurzweil files. Latest version was released about a year ago, so if it doesn't work for you, chances are that the developer is still around.
