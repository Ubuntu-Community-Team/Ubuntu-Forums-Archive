with a multi-timbral synth, you can stack wildly differing sounds,
up to your cpu limit, to create wonderful moving sounds on each single
keypress.

gated sounds for pulsing rythm
sweeping sounds for motion
pads for underlying emotive themes
arpeggios for glissands, and harmonic/rythymic sequences
legato/portamento lead sounds for interesting melodies
panning/delay/phaser/flanger/tremolo/reverb sounds for spatial effects

Yoshimi is the linux synth for this (it's an updated zynaddsubfx)

For each new sound, press the up-arrow widget to move to sound #2,
change the sound #2 midi channel to #1, and pick a new sound

This requires some hours picking and matching sounds, but can provide
great results.

Hydrogen 'kits' hold up to 32 samples, which don't have to all be drums, so sampled chords, leads, bass lines etc can all be placed in the pattern grid to build up new songs.
