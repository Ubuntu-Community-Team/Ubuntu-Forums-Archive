### Post by ibjsb4 on 2013-04-29
Want to try XFCE?  You can add it to your current system and choose which desktop to run at boot (login) by clicking on the icon at the login screen.

[Open a terminal]("https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal") and enter:

```
sudo apt-get install xfce4
```

You can add any Ubuntu desktop (Lubuntu, Classic, Unity ..) to your system and give it a test spin.
