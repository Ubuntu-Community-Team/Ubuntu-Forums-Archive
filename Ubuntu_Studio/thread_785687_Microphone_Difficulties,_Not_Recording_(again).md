### Post by Michl on 2008-05-07
For what it is worth, I haven't been able to
get my microphone to work in SOund Recorder 
since Feisty. However the mic works in Skype.
To make it work in Skype, I cannot set Sound In to
Default Device, but need to set it to one of
the other available options, which named the
mic.  ALso, I had to uncheck Allow Skype to
set mixer levels.  This, by the way, took care
of the static problem that I had.  (I'm
referring to Options -> Sound Devices in
SKype.)
