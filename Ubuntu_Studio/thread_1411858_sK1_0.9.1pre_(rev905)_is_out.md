### Post by prokoudine on 2010-02-20
[sK1]("http://sk1project.org") team has released preliminary builds of upcoming v0.9.1. The project is known as the only free/libre vector graphcis editor that exports CMYK PDF files ready for commercial printing with all the crop marks and color bars, spot colors too.

There have been over 200 commits since the last release, quite a lot of bugs were fixed, 64bit compatibility was improved a lot. Some of the new functionality like gradients, RGBA and CMYKA doesn't have UI yet, but that will be fixed for the final version.

Another thing that is new is that a lot of source code was refactored and subsequently separated into a library called sk1libs. sK1 now depends on this library, and so will the very next version of UniConvertor.

Ubuntu packages were created with dpkg, so you can install them by double-clicking -- all dependencies will be resolved.
