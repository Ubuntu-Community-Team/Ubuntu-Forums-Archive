### Post by jejeman on 2014-02-19
Ubuntu Studio is Ubuntu with XFCE and some preinstalled programs and settings for audio, video and graphic production.

Only real need for Ubuntu Studio is if one is doing audio production, for all the rest there is no need to go for Ubuntu Studio, well, only if you are lazy to install programs.

It is all explained here
[http://ubuntustudio.org/](http://ubuntustudio.org/)
