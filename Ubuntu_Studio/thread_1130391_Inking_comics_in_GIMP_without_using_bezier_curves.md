### Post by lukeiamyourfather on 2009-04-20
It sounds like you're going for a style that would be much better handled by a vector graphics application instead of a raster graphics application. GIMP is a raster graphics application and deals with pixels at a specific resolution, this is better suited for photographs and other very detailed images. Inkscape is a vector graphics application that is resolution independent and can create clean lines and shapes using coordinates and algorithms to create images, much better suited for comic like illustrations and large images with less detail than a photograph.

Its easier than it sounds, the points and algorithms are handled in the background. All you have to do is draw using the tools and work with the layers. Cheers!
