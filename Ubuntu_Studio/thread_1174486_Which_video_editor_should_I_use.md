### Post by cotcot on 2009-05-31
Cinelerra is high level. Needs quite some time to learn. If you spent the time it will be honoured. You do not have Blender (Video Sequence Editor) in your list. It is high level as well.

Avidemux is useful to learn anyway. It is a good toolbox.

Jahshaka : sure you want to learn an editor that has no development since a couple of years (2006) and is diffucult to install ?

Lives : medium level 

Kino is useful to learn anyway because it is often used to capture video from camcorders.

Kdenlive : fine if you get it working stable. Good starter if cinelerra and blender take too much time to learn.

Open Movie Editor : no comments
