### Post by variona on 2009-11-26
Which Midi applications do you use, which are you happy with?

I use primarily Rosegarden(RG), after trying many parameter combinations, I finally came up with a Jack configuration that allows me to sync RG with Ardour on a stable basis (It seems RG didn't like the -u parameter that unlocks the memory of some libraries; see: man jackd. Another RG misbehaviour I encountered: If I use the separate transport window, the main window, once minimized, will not come up again until you close the transport window first). But having configured to my needs and knowing about the 'Vanishing Window' and how to make it reappear I'm all smiles again.
While searching this forum I was pointed to QTractor which seems to be a very promising alternative to RG.

Stochastic pointed out, in another thread, that RG is not in the Studio Installation as a combination of other programs is offering the same functionality: Seq24,Denemo and Ardour.
IMO there is nothing to be said against Ardour, but I'd like to know are many of you using either Seq24 or Denemo?
Muse is a program that did not start at all, trying from a terminal only offers errors([https://bugs.launchpad.net/ubuntu/+source/muse/+bug/488844](https://bugs.launchpad.net/ubuntu/+source/muse/+bug/488844))
