### Post by Stochastic on 2008-12-05
if your card is based on the Phase 22, then you might want to take a read through this thread: [http://ubuntuforums.org/showthread.php?t=899689](http://ubuntuforums.org/showthread.php?t=899689)

I also found this [http://www.hydrogenaudio.org/forums/lofiversion/index.php/t27594.html](http://www.hydrogenaudio.org/forums/lofiversion/index.php/t27594.html)> If you use Linux and have an Envy 24 HT-S based card (like the Chaintech AV710) be aware that this card will work perfectly, provided you use alsa-drivers newer than 1.0.6rc1.

So if your card is an Envy24HT-S chipset then any modern Linux distro should support the card.
