### Post by andrew.46 on 2011-12-03
Could be a little late but the best way to multiboot with Lilo is to install Lilo to mbr and boot slackware from this. Then install Ubuntu to its partition and install grub to the root of this Ubuntu partition rather than the mbr (there is an option to do this during installation). Then *chainload* the Ubuntu installation from Lilo. Avoids a lot of mess and is easier when Ubuntu changes its kernel...
