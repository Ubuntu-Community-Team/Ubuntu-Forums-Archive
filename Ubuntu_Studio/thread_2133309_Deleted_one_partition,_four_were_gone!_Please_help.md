### Post by ibjsb4 on 2013-04-07
For starters, do not use that computer any more.  You can add to the problem by accessing it.

Use another machine to create a [live cd]("http://www.cgsecurity.org/wiki/TestDisk_Livecd")

Then just boot the cd in the problem machine.  With me, it was automatic after that.  It found the partition I deleted and was able to restore it.

Hope its that easy for you.
