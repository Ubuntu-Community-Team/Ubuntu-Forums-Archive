### Post by GameGod on 2008-08-07
Hi guys,

I just wanted to let everyone know that [Mixxx 1.6.0]("http://www.mixxx.org") has been released! 

Mixxx not only has great MIDI controller support, but also lets you use Serato, FinalScratch, and Traktor vinyls with any old soundcard. That means you don't have to buy an expensive kit to use vinyl control. :)

Anyways, the latest version can be picked up on the main site here:
[http://www.mixxx.org/](http://www.mixxx.org/)

GameGod
