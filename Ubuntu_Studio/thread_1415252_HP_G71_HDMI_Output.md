### Post by Raval on 2010-02-24
Hi, I have a HP G60t laptop. I got HDMI to work a few minuted ago. I'm using an Samsung 40" 1080P TV.

First select the HDMI channel the cable is connected too. 

Then click on display under System->Preference. 

Then select "mirror monitors" 

You should see your Ubuntu desktop.

When I select "detect monitors" my TV comes up as a 7" Samsung. 

This solves Video but now we need to set audio.

Under System->Preference click on sound

Under Sound select Hardware and for the profile option select "Digital Stereo (HDMI) Output"


If your laptop has a mic then you can select "Digital Stereo (HDMI) Output + Analog Stereo Input"
