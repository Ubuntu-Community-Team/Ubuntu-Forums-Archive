### Post by Arpage on 2012-05-08
*BUG REPORT*

On a fresh install of Ubuntu Studio 12.04 x86 the X-Fi Extreme Gamer is not being recognized. I've tested this on my pc as well as a friend's who also has this soundcard.

Ubuntu Studio 12.04 x64 DOES recognize the soundcard and it functions correctly.

Just a heads-up for those using the X-Fi Extreme Gamer.
