### Post by jejeman on 2014-01-02
Okay, now we have sda. Let's first repair boot, for partition recover you should open new thread, but not here in ubuntu studio forum because it is a general problem - you will get better help.
From the LIVE system, which MUST have same arch as installed system, and for the best results the same GRUB version, try to reinstall GRUB into MBR. Notice here you need to instal it on sda NOT on sda1.
1. Boot LIVE CD
2. Mount sda1 partition
```
sudo mount /dev/sda1 /mnt
```
3. Install GRUB to MBR of sda
```
sudo grub-install --root-directory=/mnt/ /dev/sda
```
4. Reboot
5. Now, you should boot system from HDD. Update GRUB
```
 sudo update-grub
```
