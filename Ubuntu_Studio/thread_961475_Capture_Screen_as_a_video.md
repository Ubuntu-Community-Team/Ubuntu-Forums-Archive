### Post by theorganloft on 2008-10-30
I am using Gtk-RecordMyDesktop on Ubuntu 8.04 Hardy Heron. To install do:

sudo apt-get install recordmydesktop gtk-recordmydesktop  

See my Youtube video: [http://www.youtube.com/watch?v=kWPyr7WcIic](http://www.youtube.com/watch?v=kWPyr7WcIic)

or look for TheOrganloft:guitar:

I went through some pains of setting it up because I use an M-audio Revolution 7.1 sound card. I had to set it up with Jack and that took some careful thought. 

However if you just have a stock or internal sound card, it should work well out of the box. 

To enchance the experience, I installed a Logitech Pro 9000 usb webcam and Ubuntu's Cheese.  See the You tube on how effective this is in Desktop recording. 

Good Luck
