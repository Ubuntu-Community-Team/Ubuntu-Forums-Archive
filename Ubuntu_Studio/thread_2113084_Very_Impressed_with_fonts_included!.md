### Post by callmebruce on 2013-02-06
I know this forum is for folks having problems and needing help. I just wanted to say that I have been using Scribus for a while, and was not happy with stock fonts and how my documents looked in Scribus. I am VERY pleased with the fonts included with Ubuntu Studio and the ease of using Font Manager to view and select fonts. It made a world of difference in a boook cover my son and I mocked up for a potential book on tourist railroads.

I'm having problems with sound and working my way through them - but the main reason I went with Studio is for working on a book with my son, and the fonts and set up in Studio make for a much more polished document. The cover we were able to do using Scribus and GIMP looks a lot more professional, and might convince someone to pick up a hardcover or read an e-zine (not sure if we're going print on demand or just an e-zine PDF - but having fun either way).
