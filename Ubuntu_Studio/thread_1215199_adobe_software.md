### Post by kayosiii on 2009-07-16
Ok ok no need to shout...

If you can deal with having Adobe Software that is one or two versions out of date (Say CS2) you can run the software using wine. Otherwise you will have to resort to using a Virtual Machine and a copy of Windows or MacOS...
I am currently experimenting with VirtualBox for these purposes.

Adobe currently has no plans for native linux versions of their software beyond, acrobat reader, flash player, and some of their more technical flash based scripting products (flex). Adobe will probably not consider the move until it is apparent that it is profitable that they do so. The more people that contact them and let them know that they would prefer using their products under linux the sooner this will happen.
