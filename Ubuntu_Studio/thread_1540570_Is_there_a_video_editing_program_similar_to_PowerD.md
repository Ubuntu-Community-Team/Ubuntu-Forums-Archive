### Post by lisati on 2010-07-28
There are a handful of video editors for Ubuntu that I'm aware of, but I don't currently use: pitivi, kino, and openshot come to mind.

Each has its own merits, but I don't think any of of the ones I've looked at are a direct drop-in replacement for Power Director. I've used a copy of the related Power Producer that came with a DVD burner. It was OK for occasional use but I didn't really like it. I've also tried a demo version of Power Director, but didn't stick with it because I was already comfortable with other software I'd purchased.

If you're using KDE, then kdenlive is the way to go.  If you're using Gnome, or any Ubuntu that is not Kubuntu, I'd try Openshot.  It's still a fairly new program, but it is built on the same framework as kdenlive, and seems to be pretty stable.
