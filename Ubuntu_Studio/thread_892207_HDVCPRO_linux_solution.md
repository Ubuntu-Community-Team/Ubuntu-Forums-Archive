### Post by tilllt on 2008-08-22
I think you are mixing things up here. 
There is DVCPRO HD, a 100Mbit HD format codecby Panasonic. 
And there is HDV which is basically MPEG2.

DVCPRO HD apparently is in development for linux:
[http://archives.free.net.ph/message/20080604.181914.772fb79c.fi.html](http://archives.free.net.ph/message/20080604.181914.772fb79c.fi.html)

and here is a thread about CS3 and DVCPRO HD
[http://forums.creativecow.net/thread/3/882170](http://forums.creativecow.net/thread/3/882170)

Apparently there is no official DVCPRO HD Quicktime Codec for Windows - but apparently you can use the Avid HD Codec for this, havent tried it myself though:

[http://avidtechnology.custhelp.com/cgi-bin/avidtechnology.cfg/php/enduser/std_adp.php?p_faqid=75079](http://avidtechnology.custhelp.com/cgi-bin/avidtechnology.cfg/php/enduser/std_adp.php?p_faqid=75079)

good luck
