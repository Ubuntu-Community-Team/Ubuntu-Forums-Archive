### Post by petersofi1976 on 2014-06-26
Hello. Is there any way to add the Keyboard layout switcher indicator in the top panel?. I tried through Panel - Panel Preferences - Items - Add new items to this panel, but there is no item to be added for showing the Keyboard layout. I remember that in Xubuntu there was an indicator for the Keyboard layout which you could add in a panel. How can i add this item to a panel in UbuntuStudio? Thank you.

---

### Post by petersofi1976 on 2014-06-26
Ok, here is how i solved this issue. I went to System - Synaptic Package Manager and i searched for xkb. In the results i selected xfce4-xkb-plugin and marked it for installation. Then i was able to find the panel plugin (keyboard Layouts) and add it to the top panel. Hope this helps someone else too.
