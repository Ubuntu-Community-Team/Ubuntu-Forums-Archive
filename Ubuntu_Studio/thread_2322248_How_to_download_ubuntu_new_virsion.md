### Post by ajgreeny on 2016-04-27
If you want the default Ubuntu for a 64bit system use the **Download** button at [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

If you want one of the other DE types Xubuntu, Lubuntu etc etc, or to download via a torrent use the Alternative downloads just below the Download button.

The different DE types are also listed at [http://www.ubuntu.com/download/ubuntu-flavours](http://www.ubuntu.com/download/ubuntu-flavours)

### Post by yoshii on 2016-04-30
in case you have trouble burning a DVD disc from the .ISO file(s), you might have luck setting up a blank USB drive using a freeware called Unetbootin.  
My DVD drive recently broke down (it kept going asleep during start up) and I was able to boot instead using a USB flash drive.  You have to first change your computer's BIOS/UEFI settings though.  There's a setting in there to change the "boot order".  You set it up to first boot from the USB drive and then save and restart.  If it can't find a startable (bootable) USB drive, then it just goes on to the next thing in the list until it gets to something that starts.  So there's not much risk there.  

Unetbootin can be found at [http://portablefreeware.com](http://portablefreeware.com) or from other sites.
