### Post by CrocoDuck on 2013-04-03
Hi. I think that software synths are actually very good. If the midi controller you are going to use is fully compatible with ALSA then you are not going to experience troubles. Having a lot of controls is not going to be an issue for software and system if you use controllers that are compatible with the ALSA driver. I use a M-Audio Keystation 88es and a M-Audio Keystation Mini 32. You see, my keyboards doesn't have a lot of buttons and sliders, but they works just great with any kind of  function I bind on their controls. I feel very confortable and they works just out of the box with all functionalities. I also have a FCB1010 foot midi controller that works just perfectly using an M-Audio Uno. Check the Alsa Matrix [http://www.alsa-project.org/main/index.php/Matrix:Main](http://www.alsa-project.org/main/index.php/Matrix:Main) . If you find the controller you are interested in into the alsa matrix then you can have deep info about the support. Notice that not all the working pieces of hardware are listed in the Alsa Matrix, so search on the web too. Here a good place for finding infos: [http://linuxmusicians.com/](http://linuxmusicians.com/) .Class compliants devices should work, but make further investigations before spent your money (I always buy things I'm almost sure are going to work).
