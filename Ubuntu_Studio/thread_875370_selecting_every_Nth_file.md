### Post by ipburbank on 2008-07-30
Ok, so I was looking at a video, and I wanted to make a time lapse, using about every 8th frame. I used ffmpeg to get all of the frames into a folder, but then the next part was selecting every 8th file... Here is my simple solution:

Open the folder up in a file browser, and view the files as icons. In my case I opened the window up so that it was 7 icons wide, then clicked and dragged selecting all but the first column, and deleted the selected files! Much simpler that a shell script I feared I was going to have to write!
