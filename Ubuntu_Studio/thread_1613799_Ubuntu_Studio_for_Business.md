### Post by cchhrriiss121212 on 2010-11-05
Hello Tim,
I think that Studio can handle all the things you need, but (unlike an iMac) I imagine it will take you a while to setup and get adjusted. 

1. Debatable depending on your usage and hardware, but Studio is about as stable as any other mainstream OS if you know what you are doing. In my experience, the simpler the OS is, the less there is to go wrong. On Linux that means no flashy compiz and something other than KDE.  
2. Not an issue as Ubuntu can use the same codecs/containers found on other systems, open ones like ogg or closed like mp3 and h264.
3. If you want a sound card for recording, research before buying, but manufacturer-wise: m-audio are good for PCI, edirol for USB and focusrite for firewire. Go with an Nvidia GPU, with your budget I would get a GTX 460.
4. Ubuntu gets new releases every 6 months, avoid upgrading as it will only be a headache. If you must upgrade for whatever reason wait at least a month after release date. 10.04 will be supported for sometime so get things set up the way you like with that and stick with it, new software can be had through PPAs.

$1500 seems like a lot to me as you are a student (!), and I would recommend you build the machine yourself to save a bit. My system only cost me around £330 ($500) last year and it could handle everything you are planning here.

For recording music I would go for LMMS which is easy to set up, if you find it too basic then you can look into jackd and qtractor and all the other jack based audio goodies. Jack is generally more complicated but most users will agree that it is worth it.

For video editing I only have used Openshot which seems OK to me but I mainly use my machine for audio stuff. There is also Blender which I hear good things about. If you are using a VM I'm not sure that audio or video production will work very well at all so keep that in mind when testing.

Good luck!

### Post by prokoudine on 2010-11-05
> **Timbothecat said:**
> But I need to know if the video editing software can do what premier pro does -not saying a whole lot, I know, but video editing has been one area in which Linux has struggled in my small amount of experience

If you are up to Premiere-like software, OpenShot is the wrong choice — it's more like iMovie really. In general you are down to Cinelerra and Kdenlive. Both are a bit unstable (Cinelerra's lastest release in October was all about bugfixes btw), and Cinelerra has more features. Kdenlive is shaping up nicely lately, most recent version features usual tools for color analysis (vectorscope, rgb parade, histogram) and grading (levels, curves and SOP/sat — implementation of color decisions list). Blender's video sequencer is neat, there are some analysis tools as well, and of course you can do node based compositing, but in general it's not very well suited for video production.

> **Timbothecat said:**
> - and also if there's anything that comes close to being the equivalent of GarageBand that I could use for samples and recording etc.

The closest you can get is probably Qtractor. Note that sampler is a standalone thing, though you can use e.g. LinuxSampler as a virtual instrument (that is, as a kind of VSTi/AU).
