### Post by ian-weisser on 2014-09-29
If you are requesting a new feature, open a 'wishlist' bug in the bug tracker, launchpad.net. Or search for and post to their mailing list.
That's how you best communicate with the (mostly volunteer) developers if you are not actively contributing code.

However, it's already supposed to be trivial to install any desktop environment (including KDE and Studio) on top of the Ubuntu standard base, so this seems like an idea that is already implemented.
Multiple desktop environments (inclufing KDE and XFCE) are supposed to work seamlessly side-by-side.
It's also possible that the developers have not tried what you want to do, and won't know the answer. It's doubtful they will try it for you.

I encourage you to experiment: Backup your data, and try adding kubuntu-desktop to your existing install.
It's free, and usually quite easy to do. Undo is not as easy,but possible, and we are here to help.
Nothing is supposed to break, so if you discover any problems please file a bug report.

Ubuntu needs testers and testing feedback (like this) more than it needs feature ideas.
