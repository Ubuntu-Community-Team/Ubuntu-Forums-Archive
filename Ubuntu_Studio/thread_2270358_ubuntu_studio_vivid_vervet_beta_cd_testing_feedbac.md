### Post by Lars Noodén on 2015-03-22
Cool.  There is a place to provide feedback if you log in to the ISO (disc image) tracker and add your comments for a specific version.  Beta was a while ago and some issues might have been fixed by now, though some might not.  So it would be of best use to give the latest daily snapshot a try:

[http://iso.qa.ubuntu.com/qatracker/milestones/326/builds](http://iso.qa.ubuntu.com/qatracker/milestones/326/builds)
[https://wiki.ubuntu.com/Testing/ISO](https://wiki.ubuntu.com/Testing/ISO)

You can log in with your Launchpad id and associate Launchpad bug numbers with a specific image.
