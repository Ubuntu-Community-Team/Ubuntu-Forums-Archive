### Post by deadflowr on 2012-08-23
The difference between Ubuntu and Ubuntu Studio is Ubuntu is a standard linux operating system, used for multiple purposes, such as web browsing, office work, and other stuff. Ubuntu Studio is a specialized operating system focused on multimedia creation.
