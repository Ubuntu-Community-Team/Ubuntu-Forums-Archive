### Post by cchhrriiss121212 on 2010-03-10
Depends what you want to use it for, US is divided into 3 sub categories: audio, video and graphics.
If you choose ubuntustudio-audio you will get a realtime kernel installed (essential for audio work), which can cause minor troubles on some systems. Other than that there is nothing to mess up, just uninstall the metapackage if you don't like it and all the programs will be gone.

If you are interested in audio recording/production and are serious about it (not just tasting the waters), then I would recommend to get a dedicated partition and installing Ubuntu Studio on its own.

For example, UbuntuStudio does not include the network applet, because it apparently messes up with the RT kernel.  Ubuntu Studio is optimized to get these tasks to work as good as possible, so my advice would be that you take that route.

I hesitated to post the same advice a few hours back for one reason only - us Linux newbies have a very steep learning curve on all fronts (from GRUB right through to Ardour). Having said that, the LiveCDs and installers are so good these days that, with some careful attention from a reasonably competent user during the install process, they can carve their own partitions very successfully.

Having a lean, real time kernel in its' own partition is definitely the way to go if you're serious about audiovisual production. Just make sure you have a net connection on a vanilla Ubuntu partition whilst you're installing - that way you can call on this incredibly helpful forum for any issues that might arise.
