### Post by SavagePT on 2011-06-04
For what i want, the best i found so far is MTVCGUI. It can be found here:

[http://code.google.com/p/mtvcgui/](http://code.google.com/p/mtvcgui/)

Just make sure you have PyQt4 installed, download, decompress anywhere, open terminal on decompressed folder and run ./run.py

You could right click the run.py file and choose execute, but it's better to open terminal to get mencoder feedback.
