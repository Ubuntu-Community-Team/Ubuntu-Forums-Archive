### Post by jejeman on 2011-05-19
try
```
sudo tasksel
```
Then choose what you need, with "space" this time ;), then go OK.

in 10.04 the tasks are named like this:
2d/3d creation and editing suite
audio creation and editing suite
LADSPA and DSSI audio plugins
Video creation and editing suite
