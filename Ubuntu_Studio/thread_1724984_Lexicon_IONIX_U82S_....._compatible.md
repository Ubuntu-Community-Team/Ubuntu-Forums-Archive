### Post by AutoStatic on 2011-04-10
> **poodoopealeoaph said:**
> hmm... well if there is a very limited supply of usb sound cards that work with ubuntu what are they?All USB1.1 class compliant soundcards, and there are still a lot of those availabe. USB2 soundcards that are reported to work:
[list][*][Cakewalk/Roland UA-101]("http://www.roland.com/products/en/UA-101/")
[*][Edirol/Roland UA-1000]("http://www.roland.com/products/en/UA-1000/")
[*][M-Audio Fast Track Ultra]("http://www.m-audio.com/products/en_au/FastTrackUltra.html")
[*][M-Audio Fast Track Ultra 8R]("http://www.m-audio.com/products/en_us/FastTrackUltra8R.html")
[*]E-Mu [0202]("http://www.emu.com/products/product.asp?category=610&subcategory=611&product=15186")/[0404]("http://www.emu.com/products/product.asp?category=610&subcategory=611&product=15185")/[TrackerPre]("http://www.emu.com/products/product.asp?category=610&subcategory=611&product=17511")[/list]

> **poodoopealeoaph said:**
> Also, is firewire more compatable?No not really. Very few manufacturers provide specs and/or testing devices to the FFADO devs so fully supported devices are scarce. Luckily a lot of FireWire devices use the same chipsets (like DICE) so with some small modifications to the FFADO source some other devices work too. Other devices are partly supported thanks to reverse engineering.

Best,

Jeremy
