### Post by Ahnab on 2009-08-25
Hi all! I'm quite new to Ubuntu (using 9.04) and having a strange problem. I use Blender (and Maya sometimes) for my animation needs. The problem I'm facing is that whenever I press the ALT key in either Blender or Maya, instead of doing what the key is supposed to do (deselect objects and viewport navigation respectively), the window header snaps to the mouse cursor position and starts moving with the mouse; as if the ALT key is used to initiate some kind of window-moving mode. My question is there any way of disabling this window-moving thingy?

I'm using Ubuntu 9.05. Blender 2.49a. Maya 2008. All 32-bit.

Thanx for any help you guys can throw my way!

---

### Post by mcduck on 2009-08-25
System/Preferences/Windows, change the "Movement Key".

If you use Compiz you'll need to change this in CompizConfig Settings Manager instead, you'll find the option in the Move-plugin's settings.
