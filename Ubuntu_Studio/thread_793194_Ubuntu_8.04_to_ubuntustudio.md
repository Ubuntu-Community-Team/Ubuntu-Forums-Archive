### Post by nalmeth on 2008-05-13
You can install as little or as much ubuntustudio stuff as you like.

Just search for ubuntustudio in synaptic, or via command line:
apt-cache search ubuntustudio

There are meta-packages for video, audio, graphics, and there are theme packages aswell. Installing a meta-package is like installing a bundle of items. 

I always install the normal ubuntu, then install the theme, menu, realtime kernel, and then *just* the applications I want (usually just audio). I don't do the whole bundle.
