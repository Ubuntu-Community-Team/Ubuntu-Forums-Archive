### Post by lswb on 2008-07-19
I've tried several brands of digital camera on linux boxes (though no Nikons yet) and they all have "just worked" including on an older system still running dapper. Linux will recognize the camera and start an appropriate program to view the pictures or interact with the camera.

The 2 modes you mention, with the camera set to the mass storage system, upon plugging it in to the computer, it presents itself as a simple usb drive, viewable as a directory in nautilus or other file manager, as if you had plugged in a thumb drive or flash drive with picture files on it. The other mode tells the computer that it is a camera.
