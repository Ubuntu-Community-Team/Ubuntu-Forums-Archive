### Post by tel93 on 2008-06-23
Try installing LMMS.

apt-get install lmms

It has great instrument sample libraries and you could easily add your own. 

However, it lacks a menu link and you need to start it from the command line with "lmms".

### Post by Stochastic on 2008-06-24
LMMS is most like what you'd want, but there are other ways too.  In fact the Linux way of doing things is generally to split tasks between various applications, and have them interact with one another.  You may also find that using external soft-synths (such as qsynth or zynaddsubfx) and linking them with your DAW of choice, will work much the same as the instrument libraries of Garageband.
