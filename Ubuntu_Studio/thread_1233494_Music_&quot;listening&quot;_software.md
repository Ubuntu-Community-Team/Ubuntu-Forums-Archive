### Post by Stochastic on 2009-08-06
A tuner will only work if the music is monophonic with no excess noise.

Programs similar to what you describe would be:
Baudline
SonicVisualizer
FreqTweak

But these only give an FFT spectrogram of the audio, not a note-labeled analysis.  Though this could be used to find the notes - if you have enough time on your hands.

There is also Melodyne's software for windows, but that costs money.
