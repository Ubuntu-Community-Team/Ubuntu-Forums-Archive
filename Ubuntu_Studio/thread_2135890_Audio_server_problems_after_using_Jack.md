### Post by zequence on 2013-04-16
Since on Ubuntu you're dealing with at least two, sometimes three audio systems - alsa, pulseaudio and jack, there may be problems when using them in combinations. It's really mostly about finding the correct way to control the servers.

You can find more information on pro audio on Ubuntu Studio here [https://help.ubuntu.com/community/UbuntuStudio/ProAudioIntro](https://help.ubuntu.com/community/UbuntuStudio/ProAudioIntro)
