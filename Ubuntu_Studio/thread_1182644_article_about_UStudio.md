### Post by dawiba on 2009-06-09
I'm now one of the 10,000!

Some good, valid points and also some good advice

I had trouble with UStudio at first as well. In the end I wiped my drive and reinstalled regular 9.04 then upgraded to Studio using the advice on the Ubuntu help pages. I've noticed a couple of others have had problems installing from the DVD.

Since then, with a couple of exceptions, everything seems fine. I've had two freezes while running the rt kernel, but none using the regular kernel. Also, these happened during normal use, not making music. So far, when I've been making music while running the rt kernel, everything is rock solid. Pulseaudio behaves as it's supposed to for me, as in, it gets out of the way when I start Jack. All the programs seem to run fine and after some initial tweaks, I'm making music again and enjoying it (although tinkering is quite addictive!).

I definitely agree that a couple of things should be set up 'out of the box' i.e. audio group and limits.conf settings. Judging by the posts in this forum, this would definitely help those (like me) new to audio on Linux and is something professionals like yourself would probably expect.

I tried Studio64 3.0 beta, but it just wouldn't run for me and I also tried AV Linux 2.0, but I couldn't get my wireless to work, so I'm sticking with US for now as it works for me.
