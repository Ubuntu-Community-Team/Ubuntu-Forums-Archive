### Post by skullmunky on 2008-10-31
Mostly blur and some other filters.  These have a lot of motion blur in them.  The zoom blur gets you the halo/exploding ray of light effect in the first one.  I don't remember how to do the nice smooth swoops but it's the same technique used for all the OSX and Windows backgrounds too, so you can google that pretty easily.  The background in #2 is a fractal, e.g. Filters->Render->Fractal Explorer, then a radial motion blur.

In the 3rd it's horizontal motion blur on the boxes in the background, horizontal on the logo to get thickness, then radial, also probably some different layer comp. modes like Multiply or Lighten.  Also probably perspective transform to help get the 3D effect.

A fun way to get started - load up the ubuntu logo, and try out everything in the filters menu.
