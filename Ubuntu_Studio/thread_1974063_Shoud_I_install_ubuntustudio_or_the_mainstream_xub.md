Hi
The low latency kernel or realtime kernel will not enhance your working experience if photography is what you are interested in. However, if you do choose to install ubuntustudio you will have alot of additional audio related software on your machine taking up disk space that could have otherwise been used to store more photo's.
I would say install a "slim" OS, then install just the software you need on top of that using a package manager (like synaptic) or ubuntu software center. As long as you have an active internet connection.
