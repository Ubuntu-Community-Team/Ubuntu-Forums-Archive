The first 30 seconds sell the song, so I would get some of the drum flourishes from later in the song, up to the front, and also a teaser of the sci-fi backgrounds, perhaps a riff before the percussion, then the main guitar rythym. Overall, a few good lead guitar/synth riffs

### Post by sgx on 2011-09-25
For guitar, a used 'fender mustang I'

[http://www.google.com/search?q=%22fender+mustang+I%22&hl=en&authuser=0&num=10&lr=&ft=i&cr=&safe=images&tbs=](http://www.google.com/search?q=%22fender+mustang+I%22&hl=en&authuser=0&num=10&lr=&ft=i&cr=&safe=images&tbs=)

can often be had for $60 (depending where you live, of course)

It has foolproof and quiet usb and line input, 12 excellent fender amp models, a nice range of pre/post fx, and can save 24 presets for easy dial-twist access. I save a
clean preset (twin reverb with no fx/cabinet) and the rest for a crazy variety of sounds. Before rakarrack ;) The bundled Fuse software now has a third-party linux coded implementation called 'plug', that provides the preset editing, in a simple gui.
(you also get bundled Amplitube Fender LE, which provides 3 more fender amp models, which can be used with the new Amplitube 3 Free, (27 nice guitar gear items) which works well in wine/wineasio and reaper. This Mustang is perhaps the best linux guitar interface, at any price!

Cheers

Hi, I have had great success using Reaper alongside with linux apps.
It's whole daw output is just another instrument, as far as linux apps are concerned. When a plugin (rarely) won't scan in Reaper, it often will work in Cantabile, which does not have automatic plugin scanning in the version 1.2. The windows version of EnergyXT2.5 also is a possible option.

If you are a guitarist, most of the free windows amp-sims will work
using the fst command (and presumably, the newer festige)

fst SoloC_FullStack_v1_05.dll  then connect in qjackctl, patchage, or jp1

DrumCore, KitCore, EZDrummer and Addictive Drums all work in Reaper. You can sample what you own, to use in Hydrogen.
