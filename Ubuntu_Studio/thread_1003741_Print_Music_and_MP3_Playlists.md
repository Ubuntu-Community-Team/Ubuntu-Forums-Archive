[LEFT]Hey everybody. I have been looking for a way to print music playlists for a good while now and I've found the solution.

There is a program for Windows called mp3tag that can export music from a directory to a written format such as .txt or .rtf. This program is compatible with Wine. I am a dj, therefore must print playlists of all my cds and this program works very well at doing that. 

Hope this helps out those that wanted to export music playlists. 

MP3Tag: [http://www.mp3tag.de/en/download.html](http://www.mp3tag.de/en/download.html)
