### Post by su:bhatta on 2013-09-20
I guess you have to add the Ubuntu repos if they are not already available in Mint.

then follow this link : [https://help.ubuntu.com/community/Ubuntu%20Studio%20Upgrade%20from%20Ubuntu](https://help.ubuntu.com/community/Ubuntu%20Studio%20Upgrade%20from%20Ubuntu)

P.S. : Actually am now logged in to my Linux Mint Nadia 14 installation & finding that Synaptic Package Manager has all the Ubuntu Studio packages and it can be installed from there.
Just search for 'studio' in Synaptic  and it list all Ubuntu Studio Packages.
Linux-lowlatency kernels are also available in Synaptic.
