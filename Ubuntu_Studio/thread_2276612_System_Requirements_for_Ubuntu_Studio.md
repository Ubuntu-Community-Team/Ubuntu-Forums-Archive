### Post by jejeman on 2015-05-05
Well, if you can't go for a faster CPU then okay. You are on bear minimum here. Guess you'll need to proxy the HD video.
Add more RAM to 4GB - this is a must.

My advice is:
Look at this table [https://www.cpubenchmark.net/mid_range_cpus.html](https://www.cpubenchmark.net/mid_range_cpus.html) find the newer CPU that you can afford which scores >2.5k
The Celeron you mentioned scores 1.5k

If you don't process video that much, you do more documentary style, or you don't do HD video, you don't need that much faster CPU.
