hi

I'm not sure if the emu 1616m works perfectly on Ubuntu. Reports say it does, but could be iffy...

What kind of plugins are that? VST plugins don't run natively on Linux. There are ways to make it run, via Winehq, Wineasio, FST and/or dssi-vst, but you need some luck with that and you don't have a guarantee that it works 100% ok.

I suggest to try the Linux plugins first.

For hardware support take a look at:
[http://wiki.linuxmusicians.com/doku.php?id=hardware](http://wiki.linuxmusicians.com/doku.php?id=hardware)

I would rather buy a full supported PCI or Firewire card.

Linux is great, but it can cost you some time to get used to it and to get the configuration/installation the way you like it...

Anyway, Good luck!
