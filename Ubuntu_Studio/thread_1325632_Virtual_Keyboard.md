### Post by bluesscream on 2009-11-13
Why not try ZynAddSubFX? Has some nice piano-sounds. A simple alternative option is hexter.
Guess you are on ubuntustudio, so you all you need is installed from start up.
I was confused a little by the title - virtual keyboard is a small program which displays a piano midi keyboard to be controlled by mouse and pc-keyboard (preinstalled in ubuntustudio too).
