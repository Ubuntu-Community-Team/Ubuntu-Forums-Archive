### Post by Stochastic on 2008-08-22
To check if a particular program works in wine, look at the wine app database:
[http://appdb.winehq.org](http://appdb.winehq.org)

As far as I can see Reason is listed with a silver rating while ableton live has a bronze rating.  I've never owned said softwares so I can't offer any more advice on getting them to work.

Edit: You could always run windows or OSX through a virtual machine if needed, but you'd need a copy of that OS, and it might be a touch slower.
