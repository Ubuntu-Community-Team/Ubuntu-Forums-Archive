### Post by DerickX on 2009-12-07
Install the latest LMMS and Qtractor

Start jack and qtractor.
Drag the LMMS samples in the filebrowser of Qtractor
The samples are in /usr/share/lmms/samples 

Qtractor can use DSSI (I suggest whysynth), LADSPA (even LV2, but that needs some special love) and VST.

Qtractor has an internal Jack connection tab, so it's also pretty easy to use zynaddsubfx and ams with it...


edit: see also [http://www.straightedgelinux.com/qtractor/](http://www.straightedgelinux.com/qtractor/) for a Qtractor tutorial
