Hi! I have Ubuntu Studio installed in my computer as a duel boot with Windows 7. Installing Ubuntu Studio can be done easily, by using the Synaptic Package Manager. Once opened, search for Ubuntu Studio, and then mark all of the Ubuntu Studio packages for installation, and you should be golden.;)

The community Ubuntu documentation is a great source of knowledge. It's very advisable to search there. For example, as for your question:
[https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromHardy](https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromHardy)

As the info is for hardy, I suggest to install the packages picking them from synaptic, just in case someone has changed its name (they seem ok to me though). Also, there is currently a problem with the last rt kernel, I'd go for the previous 2.6.31 rt kernel (it's offered in synaptic for lucid). It has been very stable for me.
