### Post by lasconic on 2010-08-06
Frescobaldi is a frontend for Lilypond. NoteEdit is for Linux only.
MuseScore is crossplatform and the community is great ([http://musescore.org/en/forum](http://musescore.org/en/forum)). If figure bass are only text in columns then you will be able to enter this in MuseScore like you can enter lyrics.
