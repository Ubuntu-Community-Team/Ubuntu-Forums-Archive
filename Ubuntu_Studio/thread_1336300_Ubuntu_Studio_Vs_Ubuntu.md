### Post by snowpine on 2009-11-24
Studio uses a different kernel (linux-rt) but otherwise your brother is correct.

You can install all of the Studio applications in "regular" Ubuntu (and vice versa).

### Post by Stochastic on 2009-11-26
Part of Ubuntu Studio's design is that it can be installed in Ubuntu and vice-versa; just like Kubuntu (from what I know about kubuntu).  The RT kernel is the main difference, pre-installed applications are certainly different, Ubuntu Studio Controls, and obviously the theme.  There are a few other minor differences/tweaks under the hood too.
