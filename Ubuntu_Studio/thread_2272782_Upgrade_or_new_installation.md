### Post by CrocoDuck on 2015-04-08
Hi,
no need to install a different distro. The packages that constitute ubuntu studio are readily found on Ubuntu software center. You can just add those to your existing Ubuntu installation. To make audio performances best, install [linux-lowlatency]("https://apps.ubuntu.com/cat/applications/linux-lowlatency/"). If you face issues, try [quickscan]("https://github.com/raboof/realtimeconfigquickscan") to fine tune the system. You said you are a beginner, so I link some website that will assist you for audio:

[Linux Audio Wiki]("http://wiki.linuxaudio.org/wiki/start")
[Libre Music Productions]("http://libremusicproduction.com/")
[Linux Musicians]("http://www.linuxmusicians.com/")

[This post]("http://www.linuxmusicians.com/viewtopic.php?f=19&t=9666") may interest you. Please, take your time before to modify your system and make sure you are understanding the meaning of your actions. You will learn and make your workstation best ;). In particular, don't make action you feel could damage your data. Also, make sure to browse the forums before post new questions, so that similar threads are not duplicated. For example, the fact that any Ubuntu installation can be straightforwardly transformed in Ubuntu studio could have been found [here]("http://ubuntuforums.org/showthread.php?t=2246697"). Anyway, if you choose to "upgrade" your Ubuntu to Ubuntu studio don't worry, it is just matter of few packages installation;).
