### Post by cchhrriiss121212 on 2010-05-12
I think what sgx and mango mean to say is that you should just stick with 9.10 as long as it works for what you need to create. Upgrading will likely cause you unnecessary trouble because: a) you tried it and it did not run smoothly, and b) it is still a fairly new release, which means it will be a while before the bugs are ironed out for users who don't have time to fix things themselves.

Also, your drive space should allow you to have more than one OS if you make separate root partitions with a shared /home. I can get along with 5GB for my root partitions as long as I don't install too many things.
