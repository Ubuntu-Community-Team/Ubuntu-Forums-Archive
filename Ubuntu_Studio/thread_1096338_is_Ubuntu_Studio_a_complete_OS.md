### Post by ad_267 on 2009-03-14
It's a complete operating system, however you can install the Ubuntu Studio packages on a normal Ubuntu installation (so both really). The 8.04 version included the real-time kernel but unfortunately this isn't available in the 8.10 Ubuntu Studio. The real-time kernel is pretty important for most audio/video applications so hopefully this will be included again in future Ubuntu versions.
