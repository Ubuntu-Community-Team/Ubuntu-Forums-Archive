### Post by Chilongola on 2011-05-02
To regain your one partion state I would say use a disk manger to nack up your drive as it is presently.  Then use the disk manager in Win 7 to create a new partion for Ubuntu and a small one - say 1.5 G for swap.  On installing Ubuntu select the third option.  Select your newly created partion to install Ubuntu.  And mark the 1.5 as swap.  If you do no like the results go back to restoring your originally backed up partion.

I believe that should do it.
