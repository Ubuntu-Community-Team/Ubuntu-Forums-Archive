### Post by oldsam on 2010-02-02
If you do not have any reason to get the very latest version of openoffice, you should just install the version from the repository, you can do this via the software-center searching for openoffice or via terminal:
sudo apt-get install openoffice.org

That's all! You don't need to install it from hand.
