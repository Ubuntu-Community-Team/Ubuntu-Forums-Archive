### Post by digitalghost1 on 2008-05-21
* Ubuntu 8.04/7.10
* Xara Xtreme version 0.7 revision 1692
Powerful Vector Drawing Tool similar but comparable to Adobe Illustrator!

[LIST=1]
[*]Go to the Xara Website and download the Autopackage:[http://www.xaraxtreme.org/download/](http://www.xaraxtreme.org/download/)

Recommended version:
» Download Xara Xtreme (version 0.7 Revision 1692) (Autopackage Archive - 19MB) 

[*]Once you have downloaded the file, right click the package file and select properties and go to the permissions tab, Tick the option 'Allow executing file as a program' then close the properties
[*]Double-click the package file to execute the install. Click the triangle icon to drop down the terminal window. In the command terminal window, press enter to accept the install (default is Yes). A prompt to enter in your root password a couple of times appears and then finish installing the program.
[*]The Xara Xtreme program now shows in your Graphics programs group
[/LIST]

Enjoy!
