### Post by sgx on 2012-10-27
Hi, Fender have a line of usb modeling amp interfaces
(12 amp models, and fx collection) that produce great sound,
and configure without hassles using qjackctl.
Record by usb, using headhones, while others sleep.
There is even a linux editor, to create or modify presets.
Also it has a well implemented headphone/line-out combo jack,
to record to any line-in. The Mustang is an input-only
interface, so your output will need to be your soundcard,
or motherboard soundchip.

On the $110 Mustang 1, 24 presets are on the main dial,
with fx knobs nearby. Deep editing is done using
linux Plug software:

[http://piorekf.org/plug/](http://piorekf.org/plug/)  (a debian package is linked
on the top right of the page)

forum discussion: [http://forum.musicradar.com/showthread.php?74982-The-Fender-Mustang-%28amp%29-Thread](http://forum.musicradar.com/showthread.php?74982-The-Fender-Mustang-%28amp%29-Thread)

youtube:
[http://www.youtube.com/watch?v=Wl2Q8SNVnPU](http://www.youtube.com/watch?v=Wl2Q8SNVnPU)

There is a nice software bundle, to aid Mac/Win/Wine users,
Amplitube Fender LE (works using Wine) Ableton 8 Live Lite,
and the Mac/Win Fuse editor and online-preset manager.
An excellent support forum is at Fender:

[http://www.fender.com/community/forums/viewforum.php?f=27&sid=f00e94ef54cd0219025027de0e8f2b6c](http://www.fender.com/community/forums/viewforum.php?f=27&sid=f00e94ef54cd0219025027de0e8f2b6c)
