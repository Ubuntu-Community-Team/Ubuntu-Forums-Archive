This distro is aimed to audio / video authors. Basically is the same as ubuntu plus some tweaks and extra packages which have nothing to do with gaming. You'd better use a generic ubuntu or some fancy distro based on ubuntu. That said, you can install sdlmame in US.

Install sdlmame following instructions in sdlmame4ubuntu site, repository tab.
