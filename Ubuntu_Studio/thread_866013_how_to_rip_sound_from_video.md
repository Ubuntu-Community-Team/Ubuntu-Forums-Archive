### Post by angelsguitar on 2008-07-21

I also use a nice GUI front end for it called WinFF, which you can get at [http://www.winff.org/]("http://www.winff.org/")

### Post by alegallo on 2008-07-21
A GUI frontend is a Graphical User Interface.
It is a graphical help, to avoid you learning the complete command options, and/or to show you all of them at once.

As an alternative option I suggest you Avidemux, which does more or less the same, but is somewhat more intuitive to me.

### Post by sg1983 on 2010-02-10
I know the thread is a bit out of date but AviDemux didn't realy work for me so I used VLC and it worked right away.

In VLC -> Media -> Convert/Save -> add file to convert -> then choose the format you want to convert it to and save it. Ready is the new sound file ;)
