### Post by CrocoDuck on 2016-08-12
It should be since it is the way it is meant to be performed. I don't think there are many ways to -permanently- loose access to windows, as any error in GRUB installation can be fixed by reinstalling it manually through a live. However, I _don't_ suggest to make distro advancement upgrades. I have been away from Ubuntu for some time now, so pick the following with a grain of salt, but distro advancement upgrades always broke my Ubuntu in the past, even if in single boot vanilla configurations. So my advice is actually to just keep on doing regular package updates instead. Sorry if I am being a sort of a party pooper...

EDIT: A somewhat related [post]("https://ubuntuforums.org/showthread.php?t=2333626&p=13530223#post13530223") with some advice from Bucky Ball.
