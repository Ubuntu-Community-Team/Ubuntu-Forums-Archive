Reaper has extensive keyboard combo editing, if there is a keypress to start something good on the linux side, you can try setting up something in Reaper to use the same keypress. There are linux apps that start
functions from one shared keypress on two apps when you don't want it to, luckily, not very many :)
If have bpm matching in hydro and reaper, and when hydro drums are playing, synths in reaper respond normally to midi keyboard input, and I don't recall hearing any lack of sync, arps and delay fx seem timely and normal.
