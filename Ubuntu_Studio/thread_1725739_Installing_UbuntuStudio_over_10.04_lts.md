### Post by Bucky Ball on 2011-04-10
All you really need to do is go to Synaptic Package Manager, look for and install ubuntustudio-desktop. That should install Ubuntu studio artwork, desktop, art, audio/video packages and you're done. No need for a reinstall or full scale upgrade. 

And no, I would stick to 10.04 LTS for stability and longer support life.
