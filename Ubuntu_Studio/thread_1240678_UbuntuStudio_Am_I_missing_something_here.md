### Post by cotcot on 2009-08-15
You should have Kino, Pitivi and Blender.

Kino is very good and stable but basic (not so much features and single track). It is often used to capture from video cameras.

Pitivi : I read that it is in early development. 

Blender : often unknown that it includes a video sequence editor (VSE) with lots of possibilities (my default editor)


You can get others with synaptic : kdenlive, cinelerra (add akirad repository), lives.

### Post by trulan on 2009-08-15
Did you install the software packages?  There are packages for audio, audio plugins, video, and graphics.  The installer is a little dumb on the page where you select these - you need to hit the spacebar to select each package.  If you hit enter it immediately goes on without allowing any more selections to be made.

But you can also install these packages in Synaptic.  Just search for Ubuntu Studio and you should see the audio, video, etc., metapackages.  Install them and you'll have what you were expecting.
