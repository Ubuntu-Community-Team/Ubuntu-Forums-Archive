### Post by cubeist on 2009-05-23
This suggestion isn't an ideal solution, but will probably give you the end result you are looking for.  There is a plugin available for gimp that allows you to slice an image into individual pieces for web use.  I have not tried it, but logically you could assign the sliced sizes to fit on one page each, save, then open each sliced image separately and print...


I can't find the plugin on my computer, but the gimp plugin registry should have it...


update...
the plugin is built into the gimp under filters/web/slice. It uses preset guides to do the slicing, so create guide lines where you want the cuts, then use the slice plugin, open, then print, voila!

Post back if it works.
