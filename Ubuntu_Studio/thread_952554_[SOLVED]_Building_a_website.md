### Post by Corelogik on 2008-10-19
Building a website has nothing to do with the operating system you use, except to influence the tools you use. All you really need is a good text editor and a web host. If you don't have the level of skill to design something by hand, there are a few places where you can make a website and host photos for free.

[Wordpress]("http://www.wordpress.org") Lets you create a blog for free with just a few choices and clicks.

[Flickr]("http://flickr.com") Lets you host photos for free.

As far as building a site from scratch, I know that Dreamweaver is the main app on the Mac, but I have no idea what the Linux equivalent would be. I'm still looking for apps myself.
