### Post by shaunthesheep on 2011-07-10
Since I upgraded to Ubuntu Studio 11.04 I have experienced repeated freezes when using Filezilla. A search revealed that disabling the new scroll bars solved it. I have done that. 

But when I checked System Monitor I found that whenever I launch just about any application or even Ubuntu Software Centre the CPU shoots up to 100% momentarily and then declines. A lot of the time it  is around 50% mark or over. Then it settles to around 20% when a programme(s) is open but I am not doing anything.

The CPU is an Athlon 2400 XP. 

I have 1GB ram and its is 5-6 years old system. I didn't notice tis happening before I upgraded. 

I have searched the forum and seen that dust in the CPU fan can cause this. The room where the computer is located is quite dusty.

Would using a vacuum cleaner inside the tower cause any damage?

### Post by jejeman on 2011-07-10

### Post by cchhrriiss121212 on 2011-07-10
Dust will cause a slower fan speed, leading to higher temperatures, but it will not have any effect on CPU usage.

System monitor actually uses a lot of CPU itself, so I wouldn't rely on it too much. Try using htop to get a more accurate reading.
