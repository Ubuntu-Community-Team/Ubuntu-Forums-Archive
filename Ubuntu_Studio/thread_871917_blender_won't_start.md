### Post by stoneage on 2008-08-12
You can start it in the Terminal with 'blender'. Probably it is a segmentation fault.
You can download a version from [www.blender.org](www.blender.org) which should run, or try the one at [www.getdeb.net](www.getdeb.net)

You can start the blender.org version in the terminal by going into the Blender directory and typing ./blender
