### Post by Mike_IronFist on 2009-09-30
Wow...it was as easy as installing lmms-vst from Synaptic Package Manager. Duh. Well, for any future users, the easiest way to get it installed is to either do so from Synaptic, or by opening a terminal and typing:
```
sudo apt-get install lmms-vst
```
Then entering in your password, pressing enter, and waiting for it to be done installing.

Afterwards, the instrument plugin VeSTige, in LMMS, serves as a VST host. Awesome.
