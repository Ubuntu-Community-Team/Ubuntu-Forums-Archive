### Post by Stochastic on 2008-05-22
This really should be posted to general help rather than multimedia production.

In ubuntu the user accounts you use everyday (even if there's only one) do not have any administrative privileges other than on their own files.  To change system wide settings you'll need to issue a command from the terminal window and prefix it with the command "sudo", which will give you temporary administrative privileges.  So ```
adept-manager
``` would become ```
sudo adept-manager
```.

Also, you've tagged this thread as [ubuntu] when adept is used by KDE, so I assume you mean to have tagged it [kubuntu].  Just beginner forum issues, nothing serious.
