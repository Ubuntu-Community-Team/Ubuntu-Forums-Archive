### Post by lwhitmore on 2008-12-12
I looked into video a while back and was a little bit disappointed with Kino and Open Movie Editor.

I'm not sure if this is classed as easy to use, but Blender actually includes a fully featured non-linear video editor.  It's pretty good.  Checkout this tutorial to get started [http://indiworks.wordpress.com/2008/07/25/blender-video-sequence-editor-quick-start/](http://indiworks.wordpress.com/2008/07/25/blender-video-sequence-editor-quick-start/)

One thing that I've found useful, is trying getting to grips with ffmpeg ([http://ffmpeg.mplayerhq.hu/](http://ffmpeg.mplayerhq.hu/)).   This command-line program is amazingly useful.  In many cases it seems that a lot of the video programs out there are simply front-ends for a some of the features that ffmpeg provides.  It's not that difficult and you can take it at yr own pace, picking up commands as and when you need them.

Another video-toolkit that I've used is gstreamer ([http://gstreamer.freedesktop.org/](http://gstreamer.freedesktop.org/)) - which I've found useful for adding real-time filters to video and for doing things like turning my camcorder into a webcam.  

No after-effects and no premier... but there are plenty of options available for video production.
