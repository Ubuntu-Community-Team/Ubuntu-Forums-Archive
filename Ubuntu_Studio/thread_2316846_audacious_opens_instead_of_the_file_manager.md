### Post by cfhowlett-a on 2016-03-15
It's a known issue.  still with us I see.  

Here's the fix: go into your settings manager >>  Preferred Applications >> Utilities >> File Manager.

Choose a different manager.  I only have Thunar installed so I chose a random program.  closed settings manager and launched the file manager icon.  Close that.  Now return to settings manager and reset Thunar.

Worked for me.
