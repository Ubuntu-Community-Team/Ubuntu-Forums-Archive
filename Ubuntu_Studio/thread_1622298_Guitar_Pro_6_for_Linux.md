### Post by Finn bjerke on 2011-01-25
I have purchased guitar pro 6. It has really good features and some flaws too.

1. It claims to import tabledit files and powertabs, this function is not working under Ubuntu 10.10
2. It does not work using 64 bit under Ubuntu 10.10

There is a solution to problem #1 Tuxguitar which is part of the ubuntu system and costs nothing can import/export. Tuxguitar have a wonderfull feature called "batch file converter" so you can take all your tabledit and midi files and convert them into guitar pro 5 format, which is easily imported in guitar pro 6.

I like tabledits user interface a lot its smooth and intuitive, guitar pro 6 have good features too and much better sound. There are man y great arrangetns of Rock, classical, jazz and celtic music written for tabledit and guitar pro.

The guiar pro folks are working on a solution to the import fil function under Ubuntu 10.10, the functions works under Ubuntu 10.04.  Guitar pro supports many instrumsntqal soudns, alternative tuning etc. The output looks really nice. 

Now play that guitar and have fun
