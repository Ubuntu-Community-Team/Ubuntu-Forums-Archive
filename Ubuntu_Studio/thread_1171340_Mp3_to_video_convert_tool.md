### Post by Bungo Pony on 2009-05-27
If you're going to upload a video to youtube, you should create a video. Use a video editor (like Open Movie Editor), add a .jpg, and add an audio file. Render your project, and you'll have a youtube-ready video.

However, I don't really see a point in using Youtube to host audio unless you really need the exposure. It's a fair amount of work to create a video. If you just need to host it somewhere for a website or blog, try using [http://www.houndbite.com](http://www.houndbite.com)
