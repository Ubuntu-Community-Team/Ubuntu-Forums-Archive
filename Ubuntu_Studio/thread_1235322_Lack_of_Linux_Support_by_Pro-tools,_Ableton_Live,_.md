### Post by sgx on 2011-08-31
Hi, some firewire issues are simply giving your user permissions to
use ffado, the latest firewire stack for linux. Google ffado, save some
how-to links, search this forum etc

Reaper works great for a linux daw using wine/wineasio. The windows version of energyXT 2.5 also works well. FLStudio has
also been working, but with lots of effort invested. You can install wine and wineasio, then try installing Studio One. From a terminal, type

wine /home/username/name-of-installer.exe   then
wine name-of-application.exe 

As long as pace, ilok or some exotic c/r system are not involved,
the installer should work. But few of the big commercial daws run without soon crashing. If there are problems, look in the
terminal output for missing .dll or other files, which you can add to
your wine setup, in the folder

.wine/drive_c/windows/system32

Cheers
