### Post by markbuntu on 2008-08-10
Ardour is in that package, it works great.
The Ubuntu Studio package includes the rt kernel, effects rack, everything which is a must for live recording. Also get the Ubuntu Studio audio-plugins.

At the bottom of this guide are some links to setting up jack, which you will need to use if you are serious about recording:

### Post by markbuntu on 2008-08-11
The thing with ardour is that you can ignore most of the options and it will still work great with the defaults, but if you need something very specific, it will be there for you. You do not need to decide every little thing.

Besides, if you want to get professional about your recording, you really need to learn ardour because every other professional software is just as complicated, if not more so.
