### Post by deadgobby on 2008-06-27
Please run audacity from the terminal. You can do this by going to applications>accessories>terminal. Once the terminal is open type in all lower case.
audacity
 The program will open up and try recording. If the program freezes up the terminal will give a error code on the cause of the problem. Then copy and past that information from the terminal to your reply here on this forum,
Gobby
