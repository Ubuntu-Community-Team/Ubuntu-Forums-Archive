### Post by texaswriter on 2013-08-23
If you can run open shot in a terminal and log the error(s) it emits when it crashes as well as the situation you are using it. 

I used OpenShot to merge some videos. Dragged video files from a file browser (dolphin), then dragged the videos into the timeline, and finally exported the output video.  

I also tried starting a new project, saving it in a valid location, then adding files via the file dialog "add new files" before dragging to a timeline and exporting the final video. 

Seems to work find. Run Openshot from a terminal, save your project file early and often if you have crashes, and report crashes here [https://launchpad.net/openshot/+filebug](https://launchpad.net/openshot/+filebug) 

Good luck
