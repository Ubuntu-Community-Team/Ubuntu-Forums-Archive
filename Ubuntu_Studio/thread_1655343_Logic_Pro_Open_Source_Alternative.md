### Post by cchhrriiss121212 on 2010-12-29
Qtractor will do all that. I don't know how it will compare to Logic Pro as I have not used it, and I don't know what you consider to be unacceptably hard. 
For a start you will have to install and understand the jackd sound server which is not exactly user-friendly, but once you get going it is mostly intuitive. There are some good tutorials out there on youtube, just take a bit of time with it and don't expect to be recording things in a matter of minutes.

If you have any trouble post back here and people will respond, any shortfalls in documentation for FOSS are often made up by the community support.

### Post by AutoStatic on 2010-12-29
Qtractor can't synthesize itself though, you will have to use plug-ins for that. But as Qtractor is the only DAW for GNU/Linux that supports basically all important plug-in frameworks (LADSPA, LV2, DSSI, LinuxVST and VST through DSSI-VST) you should come a long way.
And Qtractor's editing functionality is not brilliant, but there are other tools that can handle that.
