### Post by n5jyk on 2014-05-27
I hate you haven't had a response by now, and you have probably moved on or already purchased something. 

I have a VERY basic usb-midi interface and have had a few latency issues with it on linux, OS X, and windoz so its likely not a good choice for any platform. That said, if you haven't purchased anything yet, search the forums for usb midi, try to borrow one to test it out, and buy only from shops that allows returns.  Make sure you know how to edit init files to ensure its module is loaded in the correct order (to make sure it doesn't end up being the default interface for sound) just in case there are issues with that.

I have more issues with JACK than anything else.  Older computers are usable but if they have shaky/querky/dodgy components to begin with, sound latency WILL be affected if they are constantly resyncing the i/o bus.... just saying.
