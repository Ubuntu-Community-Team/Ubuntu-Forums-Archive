### Post by prokoudine on 2010-04-11
> **ubun2warrior said:**
> How can I view corel draw files( .cdr extension) on Ubuntu. I have tried wine but it does not work.

You want installing Inkscape AND UniConvertor. Please not that the current version of UniConvertor supports only paths and flat fills/strokes (that is, no gradients). Or you could install sK1 (vector graphics editor from which UC is created). They have fairly recent builds for Ubuntu.
