### Post by sgx on 2011-09-30
Hi, I would never update a working system. Luck, or the work
put into achieving stability, must be guarded. Instead, when curious,
use live media for testing, or install new distros on usbsticks or
spare/external hard disks. You can invent a test using some cpu intensive app to compare results between setups. I like always having
an rpm, deb, and puppy linux on hand, fully configured for audio production. If I mess something up, there is no panic :guitar:

I also horde debs and rpms, as there are times to manually install
a needed app, and the dependencies might be hard to find in a pinch.
Synaptic has an option to save those files in its cache, burn them to a dvd or usbstick just in case.
Cheers
