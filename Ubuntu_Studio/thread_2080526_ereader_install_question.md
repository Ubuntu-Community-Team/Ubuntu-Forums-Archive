### Post by mwinthrop on 2012-11-04
Here is the GTK website that explains what it is: [http://www.gtk.org/](http://www.gtk.org/)

I briefly looked up the FBreader program and see it is available in the Ubuntu software center, so the easiest thing is to install it using the Ubuntu tools.  The software center also links to the website you gave.  The FBreader repository site might allow you to have the most up to date version of FBreader.  So, if you want the most current version, you might want to use the FBreader repository rather than the Ubuntu software center version, though Ubuntu may update this program periodically as part of the LTS support  - don't know on that one.  Since Ubuntu maintains the program in its software repository, my opinion is the security risk for installing the FBreader repository is very low.  I have used other repositories external to Ubuntu, though I too try to be careful and have not had any trouble.  I only use a few external repositories, so perhaps that is partly why I don't get into too much trouble.
