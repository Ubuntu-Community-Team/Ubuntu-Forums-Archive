### Post by AutoStatic on 2010-08-06
From the ART site:
> The USB interface is fully compliant with the USB 1.1 specification and uses USB adaptive mode for playback and USB asynchronous mode for record. It will work with the USB audio device drivers built into Windows 98SE/ME/2000/XP/Vista and Apple OS9.1/OSX computers with native USB support. No special drivers are needed.So it's a class compliant USB1.1 audio device which means it should work with Linux.
