### Post by FakeOutdoorsman on 2009-03-06
I use Linux for video and audio conversion using FFmpeg and very rarely MEncoder.  I still use Adobe Premiere in Windows XP for editing (it's about the only time I use Windows these days), but the Creative Suite doesn't come close to matching the usefulness of FFmpeg when it comes to decoding, encoding, and transcoding sources.  One of my co-workers has Cleaner XL, but it has been dumped in favor of FFmpeg (and it's $595 cheaper). I also use FFmpeg to prepare edited content for distribution.  One the web end, Linux is used to develop and host the sites that present the media.

### Post by cotcot on 2009-03-07
Blender cannot render to flash. You can however easily convert most file type to flash (.swf) with ffmpeg.

Blender is probably less user friendly than the apps you use now. One of the reasons for that is the fact that it is a 3D animation program with a built in video editor. In creativity it passes the common commercial video editors due to the fact that you can include virtual animations in your video within the same program. Keyframing and node editor is included.
