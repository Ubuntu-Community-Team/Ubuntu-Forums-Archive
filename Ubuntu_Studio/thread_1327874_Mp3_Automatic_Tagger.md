### Post by Chronon on 2009-11-17
You could give Picard a try.  It's a MusicBrainz tagger that's pretty automatic.  You should double-check its matches before saving the changes to your files, though.

### Post by mobilediesel on 2009-11-17
[MusicBrainz Picard Tagger]("http://musicbrainz.org/doc/Picard_Tagger") is what I use, too. If it can't find the correct tags based on name it can also make an audio "fingerprint" and search that way.

I've had very few tracks that it couldn't find.
