### Post by bschultzjames on 2009-02-11
If you're thinking of buying the MicroKorg for the on-board synth sounds, then you're making a great choice.  I have had one for a number of years and am still in love with it.


As far as pairing it with Ubuntu, I'm a bit confused with your question.  If you'd like to run it as a midi controller, it does have midi capabilities...but if you're just buying it for that, you might as well save some cash and get an m-audio midi keyboard.  I have used it with midi before, and all you need to do here to ensure compatibility is run the midi through a compatible interface.  [anything by echo is great]

The synth sounds are amazing, and the keys take some getting used to [they're really small].  All in all, I've been very pleased with the MicroKorg.

### Post by bschultzjames on 2009-02-12
It's really about personal preference.  With keyboard controllers, I prefer something that at least has piano-sized keys.  The MicroKorg has mini keys and many people think it feels like a toy.  The Alesis doesn't have as good of synth sounds...but the keys feel great.


It really comes down to what you're going to use more:  Midi or standalone synth.  If you will mostly use this keyboard for midi, I recommend the Alesis.  If you will use this keyboard mostly for synth sounds, I recommend the MicroKorg [synth sounds are MUCH better].  


If you want something that's JUST a midi controller, check out the M-Audio Oxygen series.  They're a bit cheaper and they are supported via USB with Ubuntu.  If you use the MicroKorg/Alesis for midi, you'll have to use a midi interface to convert the midi into digital signal. 



Hope that makes sense to you :)
