### Post by sgx on 2011-10-11
Those are 'meta-packages', that install a collection, in one go.
You can also add or remove individual files, and their dependencies,
at any time. 

[https://launchpad.net/+help/soyuz/ppa-sources-list.html](https://launchpad.net/+help/soyuz/ppa-sources-list.html)

This link discusses adding extra specialized software repositories,
kindly maintained by ardent linux users. New, or latest-version
software, often is available in these, as an alternative to
learning the process of compiling the cutting edge source code.
