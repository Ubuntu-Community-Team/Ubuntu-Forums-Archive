Of course, you could add a low-latency kernel option to any other Ubuntu, but having Studio ship like that...is a big help.

### Post by Bucky Ball on 2016-03-26
UStudio comes with a LOT of stuff you might not need. For instance, if you are only after the audio stuff, you will get a HEAP of stuff you're not going to use, and vice versa. If you're only after photography things, you'll get all the audio stuff and all the other graphics stuff.

Solution: I installed UStudio once, years ago now, and since then, in other installs, have just installed the packages and the kernel into a minimal install. For instance:

ubuntustudio-audio will get you just the UStudio audio stuff. Have a look in Software Centre for 'ubuntustudio' and check the list of options that pops up. You can install quite a few 'sets' of UStudio software without installing the whole shebang (which is a very large install). 

I'd probably go about it something like this: Install Xubuntu-core 16.04 LTS, install the UStudio bits you want and the low-latency kernel, reboot and choose the UStudio kernel, you're there!)

Just throwing that in. Good luck. :)
