### Post by Stochastic on 2010-01-26
New versions of programs are generally not accepted into the repositories after the "Feature Freeze" for any given release.  For Karmic, that freeze occurred at the end of August 2009, for the upcoming Lucid release that deadline is set to mid February (so there's a tiny bit of time left to get those new versions into the Lucid repositories).

But it's rare to see a developer around these forums.  If you want to notify developers of out-of-date software in the development branch of the repositories, the best way to do it is file a bug like this one at Launchpad.net [https://bugs.launchpad.net/ubuntu/+source/muse/+bug/164719](https://bugs.launchpad.net/ubuntu/+source/muse/+bug/164719)
