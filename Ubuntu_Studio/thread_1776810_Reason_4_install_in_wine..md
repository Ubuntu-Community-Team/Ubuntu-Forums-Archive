### Post by sgx on 2011-06-07
Hi, try copying the installer to your home/username/.wine/drive_c folder,
and start it from there. But count yourself very lucky if it works
well enough to use. If there are problems, look in the wine terminal output for complaints of a missing .dll, which you can usually download and install.

I installed the demo last year, it started, and I was able to use a couple of instruments, but it did crash pretty soon. Wine has improved lots since then, so it's worth a try.
