### Post by Zimmer on 2009-07-13
The crux of the matter is that if you place a commercial CD into an Ubuntu machine and 'browse' the files you will see a listing of .wav files.
If you put the same CD into a Win machine  and browse  it will display .cda files .

If you have read the article I linked earlier you will see that the .cda files are just 'marker' files and contain no audio stream.

Indytim is right to ask whether or not you elected to ask K3b to create an 'audio' CD for you from the selected ogg files...

Bear in mind also that there are some CD players out there that do not tolerate certain types of media and may not play a DIY disc. Particularly old players can have a problem if you have used a CD-RW disc ...

Lots of info here.
[http://www.cdrfaq.org/faq02.html#S2-20](http://www.cdrfaq.org/faq02.html#S2-20)
