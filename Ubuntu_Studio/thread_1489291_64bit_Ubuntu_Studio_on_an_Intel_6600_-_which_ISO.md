### Post by trulan on 2010-05-21
The Ubuntu Studio download page for 10.04 Lucid Lynx is here:
[http://cdimage.ubuntu.com/ubuntustudio/releases/10.04/release/](http://cdimage.ubuntu.com/ubuntustudio/releases/10.04/release/)

For 64 bit, use alternate-amd64.iso
(All Ubuntu Studio iso's use the alternate [text-based] installer, as opposed to the live cd installer that regular Ubuntu uses.  And amd64 will get the 64-bit version, i386 is 32-bit.)

Or you can use the torrent, from the same page, if you prefer.
