### Post by pytheas22 on 2008-05-10
According to Wikipedia, .daa is a proprietary kind of image file owned by PowerISO.  Luckily, Wikipedia also says, "AcetoneISO is a free software available on Linux which can convert .daa files to ISO and then view them."  Go to [http://www.acetoneiso.netsons.org/viewpage.php?page_id=2](http://www.acetoneiso.netsons.org/viewpage.php?page_id=2) ; there are packages available for Ubuntu.  Then you can convert the image to a normal, open format.
