### Post by Marlonsm on 2009-11-28
You could save the file in whatever format Kdenlive prefers and make an .iso file with it using DeVeDe (available in the Software Centre).

This way, you take the .iso file to the Windows computer and burn the DVD there using a program like Nero or CDBurnerXP (this one is free).

*Don't burn the .iso file* in the DVD, select the option to burn a DVD *from* the .iso.
