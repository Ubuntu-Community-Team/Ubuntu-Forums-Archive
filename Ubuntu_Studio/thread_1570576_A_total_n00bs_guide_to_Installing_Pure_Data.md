puredata is a package in the repositories, so you install it like any other package. If you don't know how to do that, then there are several methods, including

[LIST]
[*]Ubuntu Software Centre (in the Applications menu)
[*]System->Administration->Synaptic
[*]In a terminal: sudo apt-get install puredata
[/LIST]
