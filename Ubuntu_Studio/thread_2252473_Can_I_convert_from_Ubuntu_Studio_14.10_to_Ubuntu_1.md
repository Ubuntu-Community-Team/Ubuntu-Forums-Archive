### Post by Bucky Ball on 2014-11-13

Definitely. This is best, else, using any of the methods outlined above, you will end up with ALL the Ubuntu Studio apps, which it sounds like you don't need, AND all Ubuntu apps, which you do want. Cluttered and bloated. Plus, you'll have two desktop environments, which you also probably don't want.

Fresh install is best, and only, way to have a true Ubuntu desktop install as bloat free as you can get.

@TheFu: UStudio has a TON of audio/video specific apps and other things that the average user definitely doesn't need and probably doesn't want or will ever use. Even though it uses xfce4 as the DE, the ISO is pretty massive because of these AV specific apps and other default bits and pieces. ;)
