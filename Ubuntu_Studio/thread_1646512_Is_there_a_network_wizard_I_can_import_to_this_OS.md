### Post by hueygunner on 2010-12-16
I tried to work with Network Manager and I also tried to install "wicd" but I couldn't do it.  I took the advice of an experienced user and installed Ubuntu 10.10 then upgraded it to Ubuntu Studio 10.10.  The key to doing this was to clear off my Ubuntu Studio 10.10 partition of a Windows 7 and Linux Dual-Boot system.  I used Windows 7 and Easeus Partition Manager to delete the Linux partitions for a clean install of Ubuntu 10.10.  When I tried to install Ubuntu 10.10 over Ubuntu Studio 10.10, the new installation defaulted to Ubuntu Studio instead of Ubuntu.  The key is to wipe out the partition for a clean install.
