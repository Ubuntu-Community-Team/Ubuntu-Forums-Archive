### Post by yoshii on 2015-06-24
If you are using FL Studio in Ubuntu Studio, you may have noticed the new option for FL Studio ASIO, instead of ASIO4ALL or ASIO.  On my system, FL Studio ASIO solved the long MIDI latency issue without requiring experiments with any of the other driver settings.  If you haven't tried it yet, give it a try because it might solve some of the overdub problems.  Just be sure to have done the standard DAW tweaks to optimize your Linux DAW (disabling unneeded services, etc).  What this basically means is you can overdub MIDI realtime without glitches.  

Thank you Ubuntu!  and Thank you Image-Line!  

FL Studio makes a good companion program to Reaper, which also can be run on Ubuntu Studio.  Neither of them run straight out of the box perfectly, but they each keep getting better for the most part and after the necessary tweaks and procedures are done, it works out.  Keep doing your research if you need to; it can pay off in the end.
