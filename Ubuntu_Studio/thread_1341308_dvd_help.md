### Post by howefield on 2009-11-29
Handbrake will take the video from the dvd, and my choice of editor would be Kdenlive, as a KDE application, installing into Ubuntu would pull in quite a few KDE dependancies, but there plenty of gnome video editors available as well.

If using Kdenlive, you'll want to add the ppa for it as the version in the repository is not supported by the kdenlive team on Karmic which you appear to be using.
