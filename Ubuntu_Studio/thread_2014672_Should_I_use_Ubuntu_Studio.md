### Post by jejeman on 2012-07-02
The bottom line is Ubuntu = Ubuntu Studio. Which means both have the same software available.

If you have already install all aplications that you need, and they are working great, no need for you to install studio.

Ubuntu Studio is audio oriented, so it has some configurations already implemented for that. For all the rest, 3d, video, there are no special configurations.

Differences are>
- ubuntu uses unity+gnome3
- studio uses Xfce

- Studio has lots preistalled aplications for multimedia creation.
- on ubuntu you need to do it your self
