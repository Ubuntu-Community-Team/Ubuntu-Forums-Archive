### Post by Pablo_F on 2010-12-04
It is strange that ubuntu root partition was on /dev/sdb1 during installation and now it is on /dev/sda1. This resembles The Prince and the Pauper.

I suggest you should try the following: Edit the /boot/grub/grub.cfg as administrator and replace:

Instead of
hd(1,1)
write
hd(0,1)

instead of
hd(0,1)
write
hd(1,1)

instead of
hd(0)
write
hd(1)

Just a guess, I hope this or something similar works. You can always try with rescatux.

Cheers, Pablo
