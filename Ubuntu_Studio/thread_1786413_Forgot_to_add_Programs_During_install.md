### Post by garvinrick4 on 2011-06-19
Yes you can use Synaptics package manager or Ubuntu software center to install anything
in Ubuntu repositories.
While in Synaptics under settings there is a repository selection make sure you put a check mark by all the repositories so they are open to download from. When you close software sources (where you set repositories) it will reload to add the new ones.
