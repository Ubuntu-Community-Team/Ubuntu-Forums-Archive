### Post by AutoStatic on 2010-02-11
From the Roland site:
> FPT (Fast Processing Technology) for ultra-low latency and fast, stable MIDI transmission
You can't use that in Standard OS-based driver mode. And 'Standard OS-based driver' means that this device works with a generic driver. Linux has such a generic driver, Ubuntu has it by default.
