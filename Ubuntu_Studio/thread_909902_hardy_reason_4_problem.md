### Post by greyspace1979 on 2008-09-04
You can't install Reason 4 on Ubuntu without Wine. Even then it is not guaranteed to work.

Go to Synaptic and search for wine. Install wine.  

Insert Reason 4 dvdrom into drive. Go to Applications/wine/Browse C Drive and find the Reason dvdrom in the left-hand pane. Go to Install Reason.exe. Install. Go through all that registration and such. 

Then go to APplications/Wine/Programs/Propellerhead/Reason/Reason.exe

And start it up. You may have to tweak the Wine settings but that is more than I can type up here. You may have to go to the Wine forum.

As the above poster said, you may find that Reason doesn't work as well as on a Windows box. Mine certainly doesn't.
