### Post by cchhrriiss121212 on 2010-07-03
Studio does not replace anything, it is just Ubuntu plus a few packages and repos on top with a different kernel.
If you go to synaptic and install the studio metapackage you will have studio 9.10, there is no real reason to upgrade to 10.04 just yet. You can find newer software versions from [here]("http://ubuntuforums.org/showthread.php?t=1350304") if you need them.
