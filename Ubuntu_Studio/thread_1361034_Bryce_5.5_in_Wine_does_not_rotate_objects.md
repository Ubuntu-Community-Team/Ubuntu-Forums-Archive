### Post by Madspyman on 2009-12-21
Bryce 5.5 has some spastic sliders. You're going to have to use the keyboard shortcuts in those cases. Mouse over a point on the model and hold Ctrl (you should see the rotation symbol) and click and drag to rotate the model using that point as an axis.
