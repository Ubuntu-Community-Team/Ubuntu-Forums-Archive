IT'S LOUD! warning, a lot of the patches are not whispered, and there are some screamers, so a limiter plugin from ladspa, lv2, or calf
collections would be wise to use. Save the ears, speakers, and headphones.

For the price of the magazine, $18 or so, issue 171 of Computer Music Magazine features Cakewalk z3ta V1, a great softsynth that works with the humble linux fst command, and qjackctl, and has excellent preset
and bank selection in it's gui, no extras needed. Large numbers of
free soundsets have been made over the years. The serial number is
provided with the magazine dvd. You will need a basic wine and wineasio
setup to run the installer, and use the instrument with fst, and maybe
other apps also.
(sounds great too, in case that is important!):popcorn:

A new version is available for $$$, so giving the old one away, is a tease of sorts. But linux users can always use a new synth, especially one this powerful.
Cheers
