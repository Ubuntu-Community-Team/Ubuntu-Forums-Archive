### Post by kayosiii on 2010-04-04
There is no drop-in foss replacement for flash... There aren't any drop in replacements for flash in the commercial software arena either.
There are a number of reasons for this.
Depending on your use case there may be acceptable alternatives.

If you are using flash primarily as a 2D animation package then you could perhaps look at anime studio, Plastic Paper, pencil or Ktoon.

If you are looking at rich web content then you can perhaps look at the new html5 browser technologies (including canvas, svg, smil and video tags) and the server side scripts that google et al have created for the purposes of allowing older browsers to use this technology. If linux tools for web content creation come along they will almost certainly be using this technology.


If what you are looking for is a one stop graphical binary web content creation shop then you are going to be disappointed.
