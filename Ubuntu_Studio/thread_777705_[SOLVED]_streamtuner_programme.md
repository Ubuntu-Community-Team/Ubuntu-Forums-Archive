### Post by Ripfox on 2008-05-01
> **satipera said:**
> hello, when i try to tune into a stream using the streamtuner client i get an error box saying Failed to execute child process "xmms" (no such file or directory) any ideas on how i can get this to work.Thanks

Install Audacious. Then go into the preferences of streamtuner and change where it says "play stream xmms" change the word to audacious. Same for the other setting there in the preferences, just change only the word xmms to audacious and you will be golden.
