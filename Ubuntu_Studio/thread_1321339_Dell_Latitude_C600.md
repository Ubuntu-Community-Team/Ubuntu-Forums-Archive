### Post by AutoStatic on 2009-11-10
256 Mb is almost too little for Gnome or KDE. I suggest you try something like [CrunchBang]("http://crunchbanglinux.org/") or [Xubuntu]("http://www.xubuntu.org/"). The best way to test which version you need is to download some live cd's and test which one runs best.
You could also try a regular Ubuntu but like I said, Gnome will probably not like the amount of RAM. But you could try installing a different Desktop Environment afterwards, like XFCE, OpenBox, IceWM or whatever you like.
