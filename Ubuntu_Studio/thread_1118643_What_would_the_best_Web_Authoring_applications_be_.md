I've also tried every editor available for linux.

I keep going back to quanta plus.

I, however, would be nearly as wary of a slowly-moving project as a dead one.

### Post by factotum218 on 2009-05-13
For wysiwyg yeah, kompozer. It's a bit like Frontpage. Otherwise I would suggest either Quanta+ or Bluefish to get started.

If you want to go further into things like CSS, Javascript, PHP and the like, I cringe a bit when I say this but check out Eclipse with Aptana installed. I cringe because it's a Java application that tends to be a bit sluggish at startup and is a bit of a memory hog. Same with Netbeans, it's main purpose is a Java dev platform, but it can be made a (X)HTML, CSS, Javascript, PHP, whatever IDE quite easily. But it's up to you to decide if it's worth the resources it uses on your system.


Also, there are applications like Geany and Gedit which can provide you with file browsers, sytax highlighting and (I think, might be wrong) tag completion via plugins that are available.


There are a multitude of options.

I myself have started using Ubuntu for this very reason. I've been doing desktop publishing for a little over a decade and am now moving over to web dev. I work at a newspaper so you can guess why.

Happy coding and have fun!):P
