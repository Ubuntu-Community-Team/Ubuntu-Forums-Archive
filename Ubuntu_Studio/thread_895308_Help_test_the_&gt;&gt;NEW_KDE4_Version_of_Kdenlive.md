### Post by Ubuntiac on 2008-08-20
_DISCLAIMER:_ I have nothing to do with this software, other than that I used it, love it and want to share it! :)



I love video editing, but loathe compiling.

Good thing Madsdyd has put together this slick gui / window based program to pull the latest version of Kdenlive from the source code and install it for you!
It'll even update your version to the newest if you run it again (and only download the extra bits of code). Sweet!

[IMG]http://www.kde-apps.org/CONTENT/content-pre3/85826-3.jpg[/IMG]


If you can use a windows .exe installer (and maybe even if you can't) you can use this.

All it needs is installing some packages, running the script and following the window prompts.

**Full instructions and the script are at:**
[http://www.kde-apps.org/content/show.php/Kdenlive+Builder+Wizard?content=85826](http://www.kde-apps.org/content/show.php/Kdenlive+Builder+Wizard?content=85826)
*If you like it - vote for it!*

**And there's a support thread at:**
[http://www.kdenlive.org/bbforum/viewtopic.php?f=8&t=591&start=10&st=0&sk=t&sd=a](http://www.kdenlive.org/bbforum/viewtopic.php?f=8&t=591&start=10&st=0&sk=t&sd=a)

Last but not least, be aware [COLOR="Red"]this is a development version[/COLOR], complete with bugs, crashes, missing features and the occasional refusal to install at all (just try again later). Having said that, mine is running nicely and is already more stable than the version in Ubuntu's repositories. (OK, with the exception of rendering).



Looks like the future of Linux Video Editing to me! (At least until we see something from Lumiera, which targets a more pro user and is not slated until next year).
