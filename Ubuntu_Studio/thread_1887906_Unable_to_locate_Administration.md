### Post by sudodus on 2011-11-28
I run Ubuntu Studio 11.04 and I find it where you expect to find it. I have not tried Ubuntu Studio 11.10 yet, so I don't know about that version. (And 'vanilla' Ubuntu 11.10 looks very different with Unity)

I don't think you should be logged in as root at all. GUI applications will prompt you with a superuser login window, and in the terminal you should run ```
sudo command ...
```Maybe the menu entry 'Administation ' will not be shown for users without administration privileges, but I don't think so.
