### Post by mac9416 on 2009-11-24
Hey, Alexander,

I would suggest using [Unetbootin]("http://unetbootin.sourceforge.net/") or [USB-creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") to install the live DVD on a USB drive (assuming you have one/some). Besides not requiring a CD/DVD drive, USB installation is also faster. I usually install from USB regardless of whether the machine has a CD drive.

Good luck!
