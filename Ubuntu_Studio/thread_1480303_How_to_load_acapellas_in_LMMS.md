### Post by tgalati4 on 2010-05-11
Most editing programs operate on *.WAV files which is the biggest file size.  If you feed your program a *.mp3 or *.ogg (compressed) format, then the editor normally uncompresses it into *.WAV so your can apply your edits, mixes, whatever.  Then you can compress back down to *.ogg or *.mp3 to put on a server or player.

I've used audacity in Linux (it also runs in Windows) but I'm not familiar with lmms.  Audacity automatically converts the input file into a *.WAV file for editing.

If lmms doesn't convert automatically, then you can use one of several programs:

soundconvert
soundconverter
sox
