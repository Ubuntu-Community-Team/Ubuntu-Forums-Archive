### Post by jejeman on 2014-02-08
XFCE uses slightly different shortcuts for window manipulation than GNOME.
Just try out Alt+mouse buttons. Resizing is Alt+RMB.
Personaly I always remap the shortcuts to Super+mouse buttons. Because Blender uses shortcuts with Alt.

1px border should be related to the XFCE WM theme. Try to change WM theme, and see if there is differences.
