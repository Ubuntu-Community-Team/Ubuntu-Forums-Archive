### Post by Stochastic on 2009-09-28
It's a derivative of Ubuntu.  Just like Kubuntu or Edubuntu.  Essentially it's Ubuntu with a different default installation.  Ubuntu can even be turned into Ubuntu Studio by installing the ubuntustudio meta-packages and making the proper configuration changes.
