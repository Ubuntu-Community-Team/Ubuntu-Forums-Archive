I am posting this info here since without it, there is a struggle to get all the missing pieces needed to run it.

[IMG]http://www.ocenaudio.com/imgs/screenshot_linux02.png[/IMG]

It's been bugfixed a bit more and I believe it's more stable than before.
It's features are rather decent. In my home tests, the only thing that doesn't work is FLAC export.

FEATURES:
[http://www.ocenaudio.com/features](http://www.ocenaudio.com/features)

32-bit UBUNTU (STUDIO) LINUX v16.04.x LTS DOWNLOAD:
[http://www.ocenaudio.com/start_download/ocenaudio_debian32.deb](http://www.ocenaudio.com/start_download/ocenaudio_debian32.deb)

32-bit UBUNTU (STUDIO) LINUX v16.04.x LTS users can download the needed prerequisite Qt package here:
[http://download.qt.io/archive/qt/5.5/5.5.1/](http://download.qt.io/archive/qt/5.5/5.5.1/)
( qt-opensource-linux-x86-v551.run ) ...it's about 664 MB.

These other 32-bit Ubuntu Studio v16.04.x LTS dependencies are also required for Qt: (about 25 MB total)

gcc-5-base_5.3.1-14ubuntu2_i386.deb
gcc-6-base_6.0.1-0ubuntu1_i386.deb
libc6_2.23-0ubuntu3_i386.deb
libffi6_3.2.1-4_i386.deb
libgcc1_6.0.1-0ubuntu1_i386.deb
libglib2.0-0_2.48.0-1ubuntu4_i386.deb
libicu55_55.1-7_i386.deb
libpcre16-3_8.38-3.1_i386.deb
libpcre3_8.38-3.1_i386.deb
libqt5concurrent5_5.5.1+dfsg-16ubuntu7_i386.deb
libqt5core5a_5.5.1+dfsg-16ubuntu7_i386.deb
libselinux1_2.4-3build2_i386.deb
libstdc++6_5.3.1-14ubuntu2_i386.deb
multiarch-support_2.23-0ubuntu3_i386.deb
zlib1g_1.2.8.dfsg-2ubuntu4_i386.deb

If you are missing any of the above packages, you can get them here:
[https://pkgs.org/](https://pkgs.org/)
It's a Linux package & dependency search tool with download options.

WINDOWS and WINE:
If you are running Windows, the typical Ocenaudio 32-bit Windows download includes everything you need.
Also, if you are on Linux and using 32-bit Wine, the 32-bit Windows version should also work OK.
