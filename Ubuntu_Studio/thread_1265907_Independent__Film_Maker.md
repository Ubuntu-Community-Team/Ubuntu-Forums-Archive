### Post by Spaceman9 on 2009-09-14
You might be able to run sony vegas in wine or wine doors.

For Ubuntu have a look at the thread [http://ubuntuforums.org/showthread.php?t=1246176](http://ubuntuforums.org/showthread.php?t=1246176) and try OpenShot [http://www.openshotvideo.com/](http://www.openshotvideo.com/)

If you have room on your hard drive install PCLOS 2009.2 ( KDE4 [http://www.pclinuxos.com/](http://www.pclinuxos.com/) ) ( Gnome [http://linuxgator.org/home/index.html](http://linuxgator.org/home/index.html) ) and try Lives [http://lives.sourceforge.net/](http://lives.sourceforge.net/) Lives is a pain to get working in Ubuntu.

The only thing I know of that comes close to sony vegas is Cinelerra [http://cinelerra.org/](http://cinelerra.org/) It only does so much but it's pretty good.

Nothing in Linux right now does everything sony vegas does and it could be years before anything in Linux does. The problem is too many manufactures won't work with the Linux community the way they work with Microsoft and Apple.
