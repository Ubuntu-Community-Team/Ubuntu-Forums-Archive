### Post by Unterseeboot_234 on 2008-09-03
I have used a freeware package called Photon. The advantage is: pure javascript with links to thumbnails or <NEXT> and <PREVIOUS> like in a slide show. The end product after running Photon is a static photo album made of html pages. You will have to have the latest GIMP, python 2.1+, and Python Imaging Library (PIL) to build Photon.

**[COLOR="RoyalBlue"][http://saillard.org/photon]("http://saillard.org/photon")[/COLOR]**
