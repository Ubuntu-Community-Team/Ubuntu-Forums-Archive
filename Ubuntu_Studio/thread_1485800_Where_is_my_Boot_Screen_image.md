### Post by FuzzShifter on 2011-04-21
Search Synaptic for ubuntu studio and you'll find it for installation.

After installing, go to startup manager (if it isn't already installed, it can be found in synaptic) and change the boot splash through the drop down menu.

There are other methods, but this one is really easy with a gui.
