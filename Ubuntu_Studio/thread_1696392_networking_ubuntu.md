### Post by graphius on 2011-02-27
I have a feeling you are going about this the most difficult way possible. If you just need to share files, SFTP, NFS, or even Samba would probably be easier.
If you need (or want) to run a web based interface, you can just type in the local IP address of the server laptop in Firefox or Chrome.

PS I use apache for my webserver, and the files are in /var/www/
