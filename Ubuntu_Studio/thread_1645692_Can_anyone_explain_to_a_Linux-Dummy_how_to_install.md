I've got some bad news. From my experience audio-production and wireless networking doesn't like each other very much and the symbiosis between them often leads to huge compromises (from the user :lol: ) . But here is some tips and ideas from me, hope that helps:

1. install  wicd ,if you can get any network...... if not...
2. Try Puredyne - it's nice multimedia orientated OS based on karmic and it's very useful for notebooks and netbooks
3.Try AVLinux - that's another studio OS based on Debian testing/unstable. it has custom RT kernel with wireless modules. Supports x86 only
4. Transform regular Ubuntu to Ubuntu Studio. The basic way is to type 'ubuntustudio' in your Synaptic search and install the packages that comes
