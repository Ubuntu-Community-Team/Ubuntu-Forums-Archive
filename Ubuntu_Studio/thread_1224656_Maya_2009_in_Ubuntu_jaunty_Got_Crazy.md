### Post by indigo42 on 2009-08-15
I just got Maya2008 working on 9.04. 

I was using a script to disable Gnome, but still had very flakey results.

To solve the issue, I installed the Kubuntu desktop (I know, I used Gnome for years too...ya get used to it). So when I'm going to work in Maya, I login with the "Use KDE" option selected.

Maya performs much differently using KDE. I only get some strange artifacting from time to time in the material sample widow of the attribute editor pane on the right. It runs much better than  Fedora8 in fact!
I wish I had done this sooner. 

I'm getting used to Kubuntu and will prolly just use that for my desktop from now on since I'm in Maya so much. 

Let us know if this helped!
