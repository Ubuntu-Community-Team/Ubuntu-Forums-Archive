### Post by gordintoronto on 2009-12-12
If you run Synpatic and search for "slideshow" several programs show up.  For example, Eye of Gnome can do a full-screen slideshow.

It's easy to install a system so it will auto-logon, and Preferences/Startup Applications lets you start an application.

I do not know of an application which lets you display both videos and images.
