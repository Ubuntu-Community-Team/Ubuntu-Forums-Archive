### Post by sgx on 2011-05-02
Hi, Amplitube is just a bunch of fx you can place in desired order,
in a nice gui, so the EQ, reverb, and various modulations can be used with your vocals. If you have wine and wineasio installed, it gets installed by

wine Install*AmpliTube.exe

To run the standalone version,

wine /home/username/.wine/drive_c/Program*Files/IK*Multimedia/AmpliTube*3/AmpliTube 3.exe

Note the * filling the blanks in paths.
A bit easier than Final Cut :wink:
Cheers

the qjackctl wiki has several links to pages with screenshots and
tutorials getting jackd going

[http://en.wikipedia.org/wiki/Qjackctl](http://en.wikipedia.org/wiki/Qjackctl)
