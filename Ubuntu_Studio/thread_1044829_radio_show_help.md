### Post by Stochastic on 2009-01-19
Well, as for wav editors I prefer Rezound, but there's lots of others out there, including (but not limited to) SND, ecasound, ecawave, wavesurfer, glame, and gnusound.

I'd highly recommend looking into UbuntuStudio (you can install it ontop of your regular ubuntu install) [http://www.ubuntustudio.org](http://www.ubuntustudio.org) as it fixes many issues with audio on Ubuntu (particularly the realtime kernel in 8.04).

If you're interested in doing a radio show, you might want to install rivendell [http://www.rivendellaudio.org/](http://www.rivendellaudio.org/) But unfortunately it's a bit hard to install as it's not yet in the official repositories.

There's also a few DJ suites out there, like MIXXX, that might be of interest to a radio producer.
