Hi All 
I just finished a step by step guide on setting up an Open Source Digital Audio Workstation, it resembles the documentation I wish I had several years ago when I first started learning how to make music with Open Source Software. The skill level is about beginner to intermediate. Ubuntu Studio, Ardour, Rosegarden, MIDI and Hydrogen are some of the topics covered. I hope that others starting out will find it useful. Big up to all the hard working developers that make an Open Source Software Digital Audio Workstation possible!
