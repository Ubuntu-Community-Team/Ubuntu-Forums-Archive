### Post by kayosiii on 2008-11-30
Ubuntu packages only free software by default (free as in speech even) - Last time I checked Maya was ~2000 for the cheepest option... All versions apart from the free learning edition (free as in beer) are availble for linux from autodesk and I think the license is crossplatform so if you own a copy of maya for windows then you can obtain the linux version free of charge. 

Install is not as straight forwards as it could be so search these forums for advice on doing that. If Maya is outside your price-range... Houdini offer a free version and a cheap version and is a skillset highly sought after by realworld employers - and blender has really been picking up steam of late (this one is installed in Ubuntu Studio by default).
