### Post by dv3500ea on 2010-06-01
Normally it is possible to install packages from a CD.
Insert the CD and go to System -> Administration -> Software Sources
Select the other software tab
Click Add CD-ROM
Once that has finished go to System -> Administration -> Synaptic Package Manager
It may be safer to disconnect from the internet.
Search 'ubuntustudio' without the quotes and for each of the packages click on the tick box and click mark for installation.
Press the apply button (the green tick at the top)
This should now install all you want.
