### Post by dawiba on 2011-06-09
I'm not sure if what you want is possible using a notation program directly.

You could play into QTractor or any other program that will record your midi, save and then import the resulting midi file into Musescore or other notation program for editing.

I think the only way to do all of this within the same program is to use Rosegarden. You should be able to record the midi and then open it's notation window to display the notation.
