### Post by jejeman on 2014-04-01
You need notes to be in MIDI format. You need sequencer to play MIDI notes. In sequencer you can eather load MIDI file, or type in notes.
Sequencers in Ubuntu Studio are: seq24, Ardour3, MusE, Rosegarden, Qtractor.
