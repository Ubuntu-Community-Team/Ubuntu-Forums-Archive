I have the 88-key version, and it involves going through Rosegarden (which you can then send to Ardour). If I'm right, the Maudio keystations are MIDI controllers, and Ardour is not a synth. There has to be some intermediary which actually produces sound.

Someone had a similar question to yours a while ago, and the answer was pretty descriptive. Try this out:

[http://ubuntuforums.org/showthread.php?t=236832](http://ubuntuforums.org/showthread.php?t=236832)
