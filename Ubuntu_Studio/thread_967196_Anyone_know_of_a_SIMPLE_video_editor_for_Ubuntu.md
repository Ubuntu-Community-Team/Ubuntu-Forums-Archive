### Post by cotcot on 2008-11-03
The file extention .mswmm is not a movie file. It is a project file typical to windows movie maker. You will not find a video editor able to use these project files. You need to save your film to .wmv and that is a film file format you can import in quite some video editors. I guess you will have this option somewhere under File --> Save in windows movie maker.
