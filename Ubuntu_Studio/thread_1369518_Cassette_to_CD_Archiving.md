### Post by squaregoldfish on 2010-01-01
It's all fairly straightforward really, but there's a couple of things I'd mention:

Use Audacity to record from tape - it has an excellent noise removal tool, which will allow you to get rid of the inevitable tape hiss. You may need to experiment a bit to get the best results, but the defaults are usually sufficient.

I record the whole tape as one large WAV file remove the hiss, then split it into separate files per track before making the output CD - you can mess around with the gaps between tracks if you're unhappy with them.

When you burn the CD, make sure you use Disc-At-Once (DAO), and not Track-At-Once - the latter may stick a two-second gap between tracks, which is not good if (a) you spent a while getting the track gaps right yourself, or (b) you have tracks that blend into each other.

HTH,
Steve.
