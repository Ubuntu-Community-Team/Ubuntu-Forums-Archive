### Post by balaknair on 2009-07-18
Try imgseek. It has content based search, so it should pretty much fit your needs

[http://www.imgseek.net/sshot/](http://www.imgseek.net/sshot/)
to view some screenshots of it in action

This shot in particular
[IMG]http://www.imgseek.net/sshot/9814e2bd8884d0d96a7d19c0a42403d5.png[/IMG]

Install with the command
*sudo apt-get install imgseek*

or via the Synaptic Package Manager.
