Usually new versions take a while (perhaps a month) to ironing out possible flaws. If you are very worried about stability you should wait some weeks before upgrading.

That said, I've been testing Lucid since alfa2, and it has been stable enough for me.

As for your first question: if you're using 8.04LTS (as I see in your signature), you should see an option in the update manager to upgrade to the next LTS version (10.04) when it is ready.
