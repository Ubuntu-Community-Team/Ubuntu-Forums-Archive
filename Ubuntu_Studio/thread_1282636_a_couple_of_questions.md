### Post by snowpine on 2009-10-04
Ubuntu Studio = Ubuntu + a bunch of applications (organized into metapackages: ubuntustudio-audio, ubuntustudio-video, etc.--do a search for "ubuntustudio" in Synaptic) + different artwork + the real time kernel (linux-rt).

A "regular" Ubuntu install can be converted to Studio by installing those packages (and vice versa by installing ubuntu-desktop).
