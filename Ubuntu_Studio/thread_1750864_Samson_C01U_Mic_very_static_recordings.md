### Post by BertN45 on 2011-05-06
I would install pulse audio volume control using the Ubuntu Software Center. That program has a recording tab and a playback tab, that you can use to choose which program uses which input or output device. Only at the moment of the recording, you can make the choice in the recording tab. You can also choose the default volume level for the program there. Once you make the choice it will be remembered forever or till you like to change it

In the input/output device tabs and the configuration tab you can choose how to use that device. In the configuration tab you could chose between e.g. digital or analog input/output, in the device tabs you can refine the choice. For my input device "internal audio analog stereo" I have e.g. a choice between:
-analog line in
-analog input
-analog microphone 1
-analog microphone 2
-analog video.

I expect that you can solve your issues there.
