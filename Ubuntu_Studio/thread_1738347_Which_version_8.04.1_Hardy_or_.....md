Hi, 8.04 is a stable useful RT setup, but as pointed out, it is not
going to be easily updated, since so many core libraries. apps,
and newer kernels are in use. But you can still create excellent
music with hydrogen, zynaddsubfx, ardour, and the other apps it provides, without updates.
'Both and more' is a good linux motto. I keep a few different distros
installed on usbsticks, and external drives, so I never am lacking a record button, do to a 'latest update' that has issues to iron out.

If you try to install a newer app manually using dpkg and .deb files, on 8.04, there may be needed updates to gui toolkits, codecs, and system libraries. It can be tedious, and it may end up at a dead end, if a major library is needed, that breaks compatibility.
Cheers
