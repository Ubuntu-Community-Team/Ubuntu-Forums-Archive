### Post by JC Cheloven on 2010-01-13
From my own (ie, my own) experience, these work nicely:

Edirol UA25 (and sequel). USB device. I'd recommend this one. 

M-Audio Fast Track (not pro). Also usb. 

Hammerfall 9632. PCI device. But I had to completely remove pulseaudio (and install the alsa stuff) to get it working.

Sure many others as well, but I didn't test by myself.
