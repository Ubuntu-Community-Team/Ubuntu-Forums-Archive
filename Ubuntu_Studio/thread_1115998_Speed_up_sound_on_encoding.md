Well it's like this, I record Savage 2 games at 1/4 or 1/8 speed to maximise graphic quality, the sound also gets recorded at the same speed, what I need is a way to speed up the sound, I can convert it with lame to mp3 no problem or some other file type but seems what ever I do neither mencoder or lame can speed the sound up again. 

Mplayer has no problem doing so with the -speed switch but when encoding it stays in slow-motion, anyone got a solution, feel like Mencoder should be able to do it when mplayer can but the -speed switch only fixes the video when encoding it seems :S

### Post by Stochastic on 2009-04-04
I would extract the audio, open it in your favourite audio editor (may people would say audacity for the beginner), and speed up the sound there (there should be a pitch/time shift effect in almost all decent audio editors on linux).  Then just add the audio back to the video.
