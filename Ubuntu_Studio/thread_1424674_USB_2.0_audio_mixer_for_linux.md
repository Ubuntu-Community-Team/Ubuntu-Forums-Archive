### Post by AutoStatic on 2010-03-08
Hello liberty_nz, I think you will be better off with a Firewire device. There are way more supported Firewire devices than there are USB2.0 ones.

Edit: you have an EEE so no Firewire. When it comes to USB2.0 devices I don't have any suggestions. Support for USB2.0 audio devices is very poor for GNU/Linux.
