### Post by fedexnman on 2009-12-01
ezdrummer is a windows vsti, ezdrummer solo is the vsti without using a daw host...  ezsolo uses the wineasio driver at 44.1  64 buffer rate is posible, im able to record the outputs of ezdrummer solo using ardour, i  1. load a stereo track  2. click input on the track, then edit, then add asio ezsolo outs 1 ann 2  3. on the track mixer i disconnect it from master output seeing im already hearing it thru the ezsolo already. its not midi but it sure is cool to be able to record the audio of this plugin with ardour .
