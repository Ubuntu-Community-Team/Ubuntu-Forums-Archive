### Post by sgx on 2012-02-10
Reaktor, Absynth, Guitar Rig, FM8, and Massive installed here using wine.
 If you used a command from a terminal, you must fill in all the blanks in the
path and title with a *   for example

wine /.wine/drive_c/Program*Files/Reaktor*5.6*install  

Don't use sudo when running the installer.

The terminal output may mention missing .dll files you can
easily download from google search listings.

LinuxSampler now supports .gig .sfz  .sf2  using qsampler or
the java based Fantasia for gui

Sonatina orchestral collection in sfz is popular. Check the FLStudio
docs for supported sample formats.  There is a thread here on the evolution
of FLS in wine, google can find it. It spans more than a year :guitar:
