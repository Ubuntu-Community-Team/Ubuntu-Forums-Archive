### Post by Stochastic on 2009-07-21
Diagrams is a very unspecific word.  Dia is an excellent program for flow-chart diagrams.  Inkscape is an excellent vector art program for other diagrams.  Open Office Draw is also useful if you're in the midst of a document and want a diagram...

And there's a bunch of task-specific programs (i.e. for project management, circuit design, etc...) that show up if you do an: ```
apt-cache search diagram
```
