### Post by kayosiii on 2009-05-07
My upgrade strategy is to use two disks so I always have the last system I available if things go wrong. I do a fresh install onto the disk that doesn't carry my most recent system. And bring the files across as I need them. (usually I bring the Home folder across in one go but leave a copy of the files (you as a sort of backup). This ensures that I always have one working system at a given time.
