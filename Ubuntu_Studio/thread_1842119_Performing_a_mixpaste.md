### Post by sgx on 2011-09-11
You can use audacity to export multiple tracks to a stereo file.
Create a second track, paste the sound you want to mix at the desired location, set the levels and fx as desired, and export in the desired file format. Audacity will prompt you that it will make one stereo track out of those you haver assembled.

also, crossfading is useful in audacity
