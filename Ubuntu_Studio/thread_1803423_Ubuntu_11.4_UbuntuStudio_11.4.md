Afaik, you can run x64. I think those instructions in the download page are a bit out of date.

> Is there a way to install Studio but have the same setup as the 11.4 standard release

Basically, ubuntustudio is ubuntu. It changes the theme, it adds some audio and video apps, it makes easier the system audio configuration needed to run jack (the sound system that ardour and most music programs need) ... but the software sources (package repositories) are the same as in vainilla ubuntu. There is not a special ubuntustudio software repo where studio apps come from so you can install exactly the same apps from a standard ubuntu install.


As for the dual monitors, I don't know, but as said, ubuntustudio is ubuntu so you can search in the sound and graphics forum.
