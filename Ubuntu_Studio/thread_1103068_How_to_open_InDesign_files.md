### Post by Stochastic on 2009-03-30
Well indesign has some pretty poor ratings on WINE, but QuarkXpress should be able to handle these files and according to this: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=342](http://appdb.winehq.org/objectManager.php?sClass=application&iId=342) it can be run with success in Wine.

edit: oh, I just read that QuarkXpress needs a ID2Q plugin extension to properly handle the .indd files, I don't use either of these formats so I can't really elaborate.  But InDesign can handle QuarkXpress files, so this should be a workable solution until Adobe starts paying attention to the requests of linux users.
