### Post by Elfy on 2009-10-10
You should be able to install it without needing todownload it.

Add/Remove or Synaptic (System Admin menu) and search for ardour

Or use the terminal

```
sudo apt-get install ardour
``` it will ask for your password, it will be invisible in a terminal though.

The one you downloaded is likely needing to be compiled.
