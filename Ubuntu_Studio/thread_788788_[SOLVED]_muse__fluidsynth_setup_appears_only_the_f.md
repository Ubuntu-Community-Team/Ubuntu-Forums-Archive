[SOLVED]

Fluidsynth with muse has all features, but the documentation is lacking some details.
I can access the fluydsinth instance at every time right clicking in the midi/drum instrument row on the port column. A button 'show gui' appears, then clicking the fluydsynth gui is here, and i can edit loaded soundfonts as desired.

Furthermore below the channel info i have a panel of preset, that allow me to chose the right preset of chosen channel of fluydsynth gui.

With those interfaces the fluydsynth plugin for muse allow me to do so much things easier than using qshynth. 

I should set those guidelines on the muse wiki soon.
