### Post by mcduck on 2009-03-24
I don't think there's any software packages for Linux that allow creating songs by dragging&dropping ready-made sound loops.

LMMS is a easy-to-use software studio, but doesn't give you ready-made loops you can just arrange the way you like, instead you'll have to write your own beats and melodies.

If you prefer using ready-made loops Jokosher could be useful, but you'll still have to find the loop samples separately as it's main purpose is multi-track audio editing and recording so it doesn't come with any sounds.
