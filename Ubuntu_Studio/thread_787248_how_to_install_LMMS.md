### Post by simtaalo on 2009-01-21
i don't recommend getting the older version that is in the multimedia repo's, goto [https://launchpad.net/~tobydox/+archive](https://launchpad.net/~tobydox/+archive) select which version of ubuntu you have and add those lines to /etc/apt/sources.list

then ```
sudo apt-get update
sudo apt-get install lmms lmms-extras
```

this will get you the latest version which has tonnes of improvements over the one in the ubuntu repo's.

also come join us in the lmms irc channel on freenode ##lmms
