### Post by zequence on 2015-11-11
Most of the work we do **does not require any special skills**, so more or less **anyone can contribute**!

To start contributing right away, simply read [ubuntustudio.org/contribute]("http://ubuntustudio.org/contribute") and follow instructions. 

[Ubuntu Studio]("http://ubuntustudio.org") is and has always been a volunteer project created by enthusiasts to provide a **free and open source operating system aimed at multimedia production**. And in order to continue existing, **we need continuous reinforcements of new contributors**.

We continuously need help in these areas:

[LIST]
[*]**Testing** (ISO and application testing) 
[*]**PR and Support** (communicate with the community) 
[*]**Documentation** (both written, and video) 
[*]**Artwork** (desktop look, website look) 
[*]**Packaging and Application development** 
[/LIST]

Curious to know more about what Ubuntu Studio is? Check out these pages at our developer wiki:

[http://wiki.ubuntu.com/UbuntuStudio/WhatIsUbuntuStudio](http://wiki.ubuntu.com/UbuntuStudio/WhatIsUbuntuStudio)
[http://wiki.ubuntu.com/UbuntuStudio/Organization](http://wiki.ubuntu.com/UbuntuStudio/Organization)
