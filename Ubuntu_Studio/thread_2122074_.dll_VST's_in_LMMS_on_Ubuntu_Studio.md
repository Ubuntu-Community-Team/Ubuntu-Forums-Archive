### Post by CrocoDuck on 2013-03-06
Hi. To use native windows VST you have to go through the Wine way. Basically speaking,  Wine is a program that let you to run native windows applications under linux. Having wine is not enough most of times, programs like festige are required ([http://festige.sourceforge.net/](http://festige.sourceforge.net/)). You can also compile ardour in order to include vst support: [https://community.ardour.org/building_vst_support](https://community.ardour.org/building_vst_support) . I never tried to use native Windows VST on linux. I used to run a lot of VSTs under Windows before to had completely switched to linux. Now I completely addicted to linux native programs, having a lot of fun and exactly the sound I want. I suggest you to deep try native linux plugins and VST before to try to get Window's ones running (is not very easy). Here some suggestion: [http://ubuntuforums.org/showthread.php?t=2096945&p=12415970#post12415970](http://ubuntuforums.org/showthread.php?t=2096945&p=12415970#post12415970)
