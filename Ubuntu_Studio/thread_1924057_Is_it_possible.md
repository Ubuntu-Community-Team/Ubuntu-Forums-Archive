I'm not really sure if you're talking about creating a video that moves to the beat of your mp3 (a visualization) or not.

if so then yes, it's very possible but it will take some doing to create your own, unless you just use someone else's visualization with their permission.

[http://www.milkdrop.co.uk/create.htm](http://www.milkdrop.co.uk/create.htm) <-- install winamp on a windows pc and create visualizations with the documentation provided here and fullscreen it while your song is playing, then record your desktop. 

[http://www.ubuntugeek.com/ubuntu-howto-install-projectm-audio-visualizer.html](http://www.ubuntugeek.com/ubuntu-howto-install-projectm-audio-visualizer.html) <-- compile this and do pretty much the same thing
