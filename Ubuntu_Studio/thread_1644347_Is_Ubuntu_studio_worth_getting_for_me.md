### Post by lykeion on 2010-12-13
UbuntuStudio is not that much different from regular Ubuntu. Difference is that it comes with music and video/graphic applications pre-installed, has a different theme, and uses a real-time kernel for sound production with pro/consumer sound cards (not sure about the last one for current UbuntuStudio, maybe some more knowledgable poster can correct me).

I think you definately should try out OpenSUSE and if you don't like it you can switch back to Ubuntu. You can install the graphic applications you need (gimp, scribus, inkscape, etc) in both distributions easily. The only major difference between the dists as I see is that they use different package managers (yast vs apt). If that doesn't say anything to you, it shouldn't matter much.

[DistroWatch]("http://distrowatch.com/") is a great place to compare distributions. There you can see read reviews, compare current versions of applications, etc. For example OpenSUSE 11.3 has GIMP 2.6.8/Inkscape 0.47, while Ubuntu 10.10 has GIMP 2.6.10/Inkscape 0.48. But don't let numbers blind you. A distribution is so much more, like this forum for example.
