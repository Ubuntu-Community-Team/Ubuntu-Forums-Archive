### Post by Stochastic on 2009-10-01
Unfortunately this is one area that FLOSS has yet to conquer. There are attempts at replacements such as the Flash4Linux app: [http://f4l.sourceforge.net/](http://f4l.sourceforge.net/) but it's missing many features.

There is also hope in HaXe as it can be used to write ActionScript and create .swf files: [http://haxe.org/](http://haxe.org/) infact you'll probably get the most mileage out of this route - but it's no FlashCS3.

Lastly, you can use the Adobe Flex SDK in Linux.  But this is a development library for creating flash, not a full featured graphical program.

Older versions of the Adobe creative suite run in Wine.
