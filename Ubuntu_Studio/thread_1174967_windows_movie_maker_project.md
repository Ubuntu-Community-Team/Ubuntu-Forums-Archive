### Post by Frumious Boojum on 2009-06-03
As far as running it through wine:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=6183](http://appdb.winehq.org/objectManager.php?sClass=application&iId=6183)

Only version 1.2.2541.1 seems to work right now (as far as what few tests have been performed, that is).  The problem with using that is that the school might be running 2.1 or so -- which means that the only version shown to be working so far might not be fully compatible with a project saved in 2.1.  No tests yet on Windows Live Movie Maker, but from the other Windows Live projects so far, doesn't look too good.

It'll probably be best, if you don't have access to Windows at home -- find someplace that does and use their computer to save the project  as a movie.   Just don't forget to take the source files along with you!
