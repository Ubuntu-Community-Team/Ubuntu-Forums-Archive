### Post by cotcot on 2009-11-29
Cinelerra and blender are in linux the higher level editors. Both allow after effects. Both have professional users. The source of cinelerra is actually a professional need. It is used as an own version in some studios a.o. in Hollywood. The cinelerra version we use is a branch of this version. (see [[COLOR="Red"]Akirad repo[/COLOR]]("http://akiradproject.net/") for details). For blender I hope you can get some visual answers exploring their [[COLOR="Red"]video projects[/COLOR]]("http://www.blender.org/features-gallery/movies/"). It is all coming from 3D animation. In blender you can combine 3D animations with video clips in the video editor. The node editor in blender is powerful. [[COLOR="Red"]Here[/COLOR]]("http://wiki.blender.org/index.php/Doc:Books/Essential_Blender") is a link to a good book to learn blender. 

I am pretty sure that cinelerra and blender have less features than the commercial stuff you are referring to. Question is if you need the difference in features. The [[COLOR="Red"]blenderartist forum[/COLOR]]("http://blenderartists.org/forum/") is very responsive. I suggest to drop your question there too.

### Post by Madspyman on 2009-11-29
Blender offer's many of the features After Effects does, Compositing, non linear editing, effects and transitions, time remapping, 3d integration etc.. If your willing to get past it's slight learning curve you'll be able to achieve After Efects's like results. Alternatively, if you'd actually like to use After Effects, version 6.5 runs well in Ubuntu using the latest version of Wine [http://appdb.winehq.org/objectManager.php?sClass=version&iId=7429]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=7429"). But if it's open source vfx you're looking for Blender's Composite Nodes offer more power than any comparable program have to offer.
