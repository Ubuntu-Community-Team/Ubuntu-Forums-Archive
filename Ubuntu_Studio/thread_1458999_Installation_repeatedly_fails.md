### Post by cchhrriiss121212 on 2010-04-21
What happened when you tried the upgrade?

Fortunately ubuntu studio is just a collection of packages, an rt kernel and a theme so you can download all these individually from synaptic package manager without using the studio meta-package. See this guide for details:
[https://help.ubuntu.com/community/UbuntuStudioPreparation](https://help.ubuntu.com/community/UbuntuStudioPreparation)

Start with the instructions under the heading "real time support", and then you can download whatever packages you would like to use. I am assuming here that you are using studio for the audio packages.
