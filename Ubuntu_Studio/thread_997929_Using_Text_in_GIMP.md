### Post by patchin_house on 2008-11-30
Did you select the *color* for the text? When you select the Text tool, you should see all the available text options (fonts e.g.) in the bottom half of the GIMP tools panel -- including a small window which displays the current color. Just click on that and you should get a panel that will let you change the color.
