### Post by babarosa on 2009-01-06
Welcome The-IRS,

Examples for sequencer software:
[LIST]
[*]Rosegarden - get the latest version from [www.getdeb.net](www.getdeb.net)
[*]Muse - get the latest version from [https://launchpad.net/~weboide/+archive](https://launchpad.net/~weboide/+archive)
[/LIST]
These programs are similar to cubase and logic although not that comprehensive.

Examples for notation software:
[LIST]
[*]Muse Score - get the latest version from [https://launchpad.net/~mscore-ubuntu/+archive](https://launchpad.net/~mscore-ubuntu/+archive)
[/LIST]
In Muse Score you can compose your music via leedsheed notation including systems and staffs and then export it as a midi file for further editing in a sequencer, very fine if you are used to read notation.

If you are loop orientated there is timidity and some others but that is not my composing and working style so others can make better suggestions, i am the "old style" musician :)

Have fun exploring, Michael
