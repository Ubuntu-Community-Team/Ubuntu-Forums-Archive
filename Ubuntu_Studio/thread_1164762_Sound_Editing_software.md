### Post by mcduck on 2009-05-20
or Jokosher. Pretty much depends if you need multitracking, wave editing or both and what kind of tools you need.

Audacious is excellent wave editor with some multitrack capabilities, Jokosher is nice multitracking program but with0out the wave editing power of Audacity, and Ardour does both very well but is considerably harder to learn and heavier to run than the first two.
