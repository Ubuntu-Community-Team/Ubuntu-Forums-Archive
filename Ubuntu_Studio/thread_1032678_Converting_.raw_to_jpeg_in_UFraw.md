### Post by chili555 on 2009-01-06
In all the little tabs in Ufraw, there is a 'Save' tab. It's the one that looks sort of like a CD drawer with a green arrow. When you click the 'Save' tab, you will see an option to select the format. I have attached an excerpt from a screenshot showing .jpg as my preferred format.

In the GIMP, you can scale pictures under Tools -> Transform Tools -> Scale. If I am sending a photo to my dial-up friends, I usually scale the image so that the long side is 1024 pixels.

Post back if we can help you more.
