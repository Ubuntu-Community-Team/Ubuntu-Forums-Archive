### Post by Frogs Hair on 2016-06-16
Was the 14.04 installation Ubuntu Studio to begin with or did you add the Ubuntu Studio desktop to Ubuntu 14.04 ? Upgrades are usually applied to the original installation and not to additional desktops added later. 14.04 to 16.04 upgrades will not be officially supported until July so there is a possibility that something went wrong with the upgrade because US 16.04 includes the XFCE desktop by default and not unity.
