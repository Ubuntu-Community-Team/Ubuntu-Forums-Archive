### Post by sgx on 2011-11-01
Hi, when you log out, and then log in, there should be an option
in the gui to choose from available alternate window managers, for the session, and also to pick a new default. You can install your favorite from synaptic, if it is now missing.

Then you can use synaptic package manager to add the studio apps you like using. And if not happy with the results, go back and reinstall the last working version. Make a separatate /home partition on your
next reinstall, as it can speed of future reinstalls, by leaving the
/home/user configurations unformatted.
