### Post by pytheas22 on 2008-06-13
If all you're looking to do is play a slideshow from your computer, you can use Google's program Picasa, which works fine through wine and lets you play music with the pictures.  Unfortunately it's closed-source, so if software freedom is important to you, you could use F-spot (installed by default on Ubuntu) or "Image Viewer" (I'm not sure what the program is actually called, but if you right-click on a photo and select open with image viewer, it will start).  Both allow you view photos as a slideshow, and you can just play music using another program while you're viewing the files.

If you want to make a video of a slideshow, your best bet I think is to use a screen casting program like recordmydesktop to make a video (with audio) as you're playing the show on your computer.  Picasa does have an option to create a movie from a slideshow, but I've never had it work right (on Windows or Linux), and if you want to add audio you have to combine it in later, as Picasa can only do video without sound.

### Post by pytheas22 on 2008-06-13
> But my f-spot doesnt work right. everytime when i hit the slide show button it crashes.

Try turning off desktop effects and see if it helps.

Otherwise try using image viewer for the slideshow; do you have the same problem?

Cinelerra is a great application for editing video after you've created it, but as far as I know there's no way to make a slide show out of images using Cinelerra.  I'd still recommend using a screencasting application to capture the slideshow into a video file, and then open it in Cinelerra to edit.  Cinelerra is a powerful, complex application, so you're going to have to do some reading to learn how to use it, but I think it's as intuitive as something like that can be.
