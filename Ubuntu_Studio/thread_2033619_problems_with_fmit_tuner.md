I'm using [COLOR=#000000]fmit[/COLOR] under Ubuntu 12.04  to check the tuning on a tin  whistle.  Its a great program.  When I display the statistics it shows me a bar chart with a set of  bars. ### Post by john.errington on 2012-07-26
Answering my own question to put this information in public domain.

Unfortunately the developer is now too busy to maintain this application.

I tried "completely removing" [COLOR=#000000]fmit[/COLOR] via  synaptic, then rebooting.

The fault remains, also I've noticed it  retains its earlier saved settings.

I found if I deleted the config file under .config/fmit it put it right - temporarily.

Also the "restore factory settings" sometimes worked.

The problem seems to arise (sometimes) when you change and save the analysis settings.

Worth the effort because i've never found a better tuner for tin whistle.
