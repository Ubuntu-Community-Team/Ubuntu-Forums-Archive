### Post by tgalati4 on 2014-04-28
M-Audio Delta-66, but that is a PCI card and requires a desktop machine with PCI slots.  There are some decent M-Audio USB preamps that work with linux, but I don't have one so I can't comment.  You probably want at least a 24-bit, 96 kHz card with decent preamps.  What exactly do you want to record?  How many simultaneous tracks do you need to record at once?
