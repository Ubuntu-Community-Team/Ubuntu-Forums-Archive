### Post by sgx on 2011-04-15
The install went smooth, don't install quicktime unless you
have V4L, and some real purpose for it, Amplitube for music does not
require it. There is a working quicktime uninstaller if needed.

I tried both the Standalone, and the plugin used in Reaper, both
sounded fine, the plugin setup appeared to use around 15% more cpu
than the standalone, which itself was using 40-60% of one Athlon
1900ghz core. Rakarrack on the same setup seems to use less cpu on a whole, but can also peg the meter on complex (and wonderful) sounds,
there are only so many dsp algorithms known to man:)

The download is 219 meg, and appears to include a full demo portion
of Amplitube 3 (good for 9 days) in addition to the free version.
There is an online authorization for the demo, and standard
challenge/response authorizations (online or offline) which you can cancel each time its run in the 9 days.

There will be several pdfs installed in the IK folder, for using the products many features, and new purchase system. Recording is as simple as connecting reaper to the jackd destination of choice, or using the reaper built-in recording facilities (live 24bit rendering is just a menu option away. 

For those in bands playing the hits in competitive venues, this is a fine product. For those rolling their own, the sounds are on a par with linux apps, with some glitz traded for system overhead.

Now with an affordable pricing model, I'm still happy as a clam at high tide with rakarrack, so new purchases for me will be ram/cpu before ampsim goodies. I'll watch for A3 gear models that gain the best reviews, and then
wait for a birthday :wink:
Cheers
