Judging by this it seems you are talking about the Launcher (with different programs to run-i.e.shortcuts) at the bottom, which you may have put at the top by customizing.
By default that Launcher Panel is absent in XFCE now, as I read in an Xubuntu thread.

If it is that, you can right click on the Top panel and then select 'Panel Preferences'--> 'add new panel' and customize the new panel. 
To add applications you have to add 'Lanunchers' from 'add new item'.
