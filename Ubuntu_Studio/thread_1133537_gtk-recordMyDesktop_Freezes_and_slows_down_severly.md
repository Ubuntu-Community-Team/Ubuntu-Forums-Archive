### Post by jussian on 2009-04-23
Try xvidcap. However, it seems to have problems with sound recording. Thus, if it crashes with the sound on, disable the audio and everything should work. Xvidcap outputs mpeg files.

more info e.g. here:
[https://wiki.ubuntu.com/ScreencastTeam/RecordingScreencasts](https://wiki.ubuntu.com/ScreencastTeam/RecordingScreencasts)

Btw. it seems xvidcap can not capture all windows (due to accelerated graphics or something like that). For example I have not been able to capture anything from firefox. It should be possible somehow, but I dont know how to do it.
