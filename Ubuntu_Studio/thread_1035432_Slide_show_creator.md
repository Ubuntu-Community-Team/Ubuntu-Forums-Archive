### Post by cotcot on 2009-01-10
I you are looking for an equivalent with all features of pinaccle 9 you will not find it. 

If you are looking for good video editors I suggest to have a look at Blender (Video Sequence Editor), Cinelerra, Kdenlive, Kino, Avidemux, Lives, Pitivi. Look to the features, try and decide what you need/want. Blender and Cinelerra are more powerful and are hence closer to Pinaccle. It requires some more time to learn Blender and Cinelerra than the others I mentioned. 

If you want a slideshow with transitions and adding sound Kdenlive is an easy solution.

### Post by skullmunky on 2009-01-15
That's a weird error about KDE4, but you can probably ignore it.  You can absolutely install KDE4 on i386.  If you want to do that, search in the forums and in the main ubuntu documentation for instructions on installing Kubuntu "on top of" Ubuntu.  This doesn't remove anything, it just gets you the KDE desktop as well as the Gnome desktop.

If you're running 8.10 (Intrepid), I think you can just do this from the terminal:

```

sudo apt-get install kubuntu-desktop

```

Anyway, for Kdenlive, try the 0.5 version first; if you run into stability problems, try and install 0.7, which is supposed to be much, much, better.
