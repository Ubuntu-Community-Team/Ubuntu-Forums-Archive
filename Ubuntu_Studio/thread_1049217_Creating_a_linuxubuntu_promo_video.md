### Post by cotcot on 2009-01-24
I would recommend blender VSE if you have some time to get familiar with it. If you are looking for a quicker solution you could check kdenlive. 
Both allow you to add audio clips on different tracks as the video clips. Both have fade in and fade out possibilities. Text in blender is a matter of making it in Gimp and put it on the timeline in overlay over the video. If you have a lot of text you might be better helped with Subtitle Editor.
Kdenlive has a text tool built in. 
There are other possibilities for sure but I am not familiar with them.
