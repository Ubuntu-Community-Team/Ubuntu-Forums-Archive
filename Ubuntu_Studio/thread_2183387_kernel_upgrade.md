### Post by zequence on 2013-10-27
The Saucy kernel is 3.11.0.x.x, which is the one you want. It's being continuously updated, every three weeks usually.
You can either get Ubuntu Studio 3.10, or just the kernel from Ubuntu saucy package repositories.
