### Post by y@w on 2008-08-08
You can use 'tasksel' in Ubuntu to [install the different "bundles"]("http://whatan00b.com/ubuntu-bundle-install-after-os-install") that Ubuntu can come with. When you run tasksel on the console you'll be given a menu to select different bundles. There's an "audio creation and editing suite" option. I'm not into recording myself, but it's definitely worth a look. The linked article explains how to install the LAMP stack, but the same principles apply. You just choose the audio suite rather than LAMP.

### Post by Stochastic on 2008-08-08
There are actually a few different 'recodring os' attempts out there.  Here's a couple:

-[UbuntuStudio]("http://www.ubuntustudio.org") - this is essentially Ubuntu tweaked to be suited to recording (and other multimedia production).  This is the OS I use, it's combination of Ubuntu's community, huge range of applications in the repos, and recording specialization is quite nice.

-[64Studio]("http://64studio.com/") - this is very similar to Ubuntu because it (too) has been derived from Debian Linux.  I have heard of many people very satisfied with this distro.

-[DyneBolic]("http://dynebolic.org/") - this is a very lightweight audio distro (originally designed to run entirely from a live CD and not your harddrive) and as such will run really well on old hardware (or really really fast on new hardware).

-[Jack Lab]("http://jacklab.org/") - this is a nice distro, built off the Suse Linux distro.  I've never used it, but I've heard good things.  The Suse organization system has many notable differences from the Ubuntu organization system (different strokes for different folks).

There are many others out there, what you choose depends all on what fits your needs.

p.s. welcome to the wonderful world of linux!!!
