### Post by Seventh Reign on 2009-04-08
I've noticed this problem when installing some Apps recently.  Especially VB.

It installs to Applications - System Tools

But the menu doesnt update.  If you Right Click Applications and Click Edit Menus.  Then go to the Tree for Applications - System Tools, it should be there and should be checked to be visible.  I just Uncheck and then Recheck and it usually shows right up.  I've had the same problem with unetbootin which also installs to the same menu.  So I think it might have something to do with that menu.
