### Post by luxocrypto on 2019-01-11
Hi, when there is sufficient place in the panel VirtualBox will show an icon for every VM.

However, when I have more open programs and there is not enough space VirtualBox will group the icons.

Is there any way to avoid this so that it's never grouped?

I'm using the Plasma desktop.

---

### Post by luxocrypto on 2019-01-11
Wow, I have found it myself: right click the VirtualBox icon > More Actions > uncheck: Allow this program to be grouped.

Learned something today.
