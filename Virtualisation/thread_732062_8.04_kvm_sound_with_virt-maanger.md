### Post by pochu on 2008-04-25
I have on a guest Debian Unstable launching it from a Hardy host with kvm from the command line:

```
kvm -m 512 -soundhw all /home/emilio/kvm/sid.img
```

### Post by kinap on 2008-06-18
dirty hack

rename /usr/bin/kvm to e.g. kvm.o

make a new kvm
vi kvm

put in it:

kvm.o -soundhw all $*

save it in /usr/bin

and it works

Kinap
