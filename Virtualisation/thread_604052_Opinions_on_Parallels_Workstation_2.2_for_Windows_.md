### Post by LarryJ2 on 2008-03-04
I agree.  So far Parallels is a waste of time.  I have VMWare Server and Virtual Box both installed and running several guest OSes on the same PC where I have Parallels for LinuxandWindows installed (2.2 build 2222).     Everytime I attempt to install Ubuntu as a guest,  the whole PC freezes (can't move curso nor can I escape from parallesl window with Ctl+Alt).   The  point during which the installation of guess Ubuntu 7.10 freezes seems to vary.   But I've never had a successful install yet.   Feel my $50 was a waste at this point.

Update:  I downloaded a parallels equivalent of a kubuntu desktop parallels virtual appliance (image) direct from the parallels support website.   This downloaded and extracted and the md5sum looked good.  But when I moved the files over and started the VM in the parallels control panel,   the cursor eventually became unresponsive.    Something, I assume,  is hosed in my installation of parallels vm.    So my opinion hasn't changed.   Skip parallels if you are hosting on Linux/Ubuntu.  Go right to vmware server  or better yet, virtualbox.
