### Post by calraith on 2008-04-01
You should also make sure your user account has been added to the vboxusers group.  System --> Administration --> Users and Groups, click Groups, get the properties for vboxusers, and make sure your user account is check marked.  Log out and back in for the change to take effect.  You should find VirtualBox in Applications --> System Tools in Gnome.
