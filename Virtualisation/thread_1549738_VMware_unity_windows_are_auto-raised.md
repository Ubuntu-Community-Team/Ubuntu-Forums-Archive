### Post by RikoW on 2010-08-10
Dear all,


I'm running XP SP3 in VMware WS7.1 and VMware Player 3 often using  the unity mode for full integration of windows applications into the Ubuntu desktop. Mostly, that works very nicely, however, the windows application windows in unity mode are auto-raised when the mouse pointer is the in the window. That is kind of annoying and sometimes even makes working very hard for me.

I neither have auto-raise enables on my Ubuntu desktop (compiz) nor on the XP guest. This effects occurs only between guest windows but no auto-raise happens between a window of the guest and the host.

Any ideas how to turn that off?

Thanks & cheers,
          Riko

PS: Already tried the VMware community without success :(

### Post by RikoW on 2010-09-20
Just for reference:

The trick is to turn off "Select windows when the mouse moves over them" under System -> Preferences -> Windows

[http://communities.vmware.com/thread/284770]("http://communities.vmware.com/thread/284770")

             Riko
