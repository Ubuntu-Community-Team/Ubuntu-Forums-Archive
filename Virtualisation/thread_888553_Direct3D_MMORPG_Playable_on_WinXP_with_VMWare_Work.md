### Post by chew1976 on 2008-08-13
I installed VMWare Workstation v6.5 beta II on Hardy 8.04. Ran Windows XP on it as guest and can now finally play a Chinese MMORPG titled &#27494;&#26519;&#32676;&#20384;&#20256;. ([http://wx.runup.com.my](http://wx.runup.com.my))

This game uses direct3D and for the past couple of days I was unable to get it to run under Wine 1.1.2, Virtual Box 1.64 or VMWare Workstation 6.0.4. I knew all of them can't utilize Direct3D now but the stubborn me just wanna try. :)

The setup for Workstation beta was pretty straight forward. Untar, then run the installer script as root. I was unable to turn off the debug mode as I think it was hardcoded into the beta so performance might be reduced a little.

Being a Windows user since 3.11, this was actually my 1st time using linux (1 week old noob) and I am happy to say, I am never going back to Windows. There are still a lot of things I need to learn about linux and I am thankful that this community exists and for the Internet where I can easily find plenty of related information and how-to to all my newbie questions.

Thanks for reading my ramblings.
