### Post by labinnsw on 2010-01-18
1. Install SUSE normally in Virtualbox.
2. Copy or move the image to the external hard disk
3. In virtual box, in the storage option, set the image on the external hard drive as the Virtual Hard drive.

If you use portable Virtualbox you can take the entire set up with you to use on any computer.
