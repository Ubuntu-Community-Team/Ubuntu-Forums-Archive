### Post by HankB on 2009-03-18
Hi JerryI,
I use the **<host key>F** (in my case and default **<right ctrl>F**) to toggle between full screen and windowed for the VirtualBox guest. This is using VirtualBox 2.1.4 from Sun (not the OSE version.) I think you also need to install the VirtualBox guest additions for your guest OS to make this work.

I'm running Vista Home Basic but I suppose this should work with XP as well.
