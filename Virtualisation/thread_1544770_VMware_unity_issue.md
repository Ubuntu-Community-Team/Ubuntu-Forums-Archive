### Post by dave.leblanc on 2010-08-02
Hi all,

I'm running VMware workstation 7.1 on Ubuntu 10.04. I have a WinXP guest that I have installed.

I'm running a dual monitor setup.

When I enter Unity mode there is something strange that seems to happen with my windows. A window will be drawn in a square the size of said window however the actual window contents are shifted some number of pixels to the right. If I have a larger window behind a smaller window I will see the right side of the window but the window "border" when dragging/moving the window is drawn the same number of pixels to the left and "cuts off" those pixels to the right.


I'm almost certain this has something to do with my dual screen setup but I'm certain there must be a way to deal with/correct this issue as I've seen it working on friends machines but they haven't seen the oddities that I'm experiencing.

### Post by dave.leblanc on 2010-08-03
UPDATE:

so my "shifted" windows issue I found the cause of: in my dual screen setup I had the monitors pulled apart so there was a gap between them. Removing this gap solved that part of my issue.

I do however have problems with the Unity button disappearing and windows occasionally disappearing as well. When I open a new window this will occasionally cause the previously disappeared window to appear on another virtual desktop.

The strange part is it's not like the window gets dragged slightly right and winds up on the "next" virtual screen. It can go 2 or 3 screens in one direction without the window even being brought to the edge of a screen.



Any help or suggestions on things to try would be greatly appreciated.
