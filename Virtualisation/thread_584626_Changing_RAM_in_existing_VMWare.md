### Post by w116tjb on 2007-10-21
Is it possible to modify your Windows XP VMWare file so it has more RAM? I accidentally gave my WMWare file 64MB of memory, which isn't nearly enough to do anything without horrendous lag. Any help would be appreciated.

### Post by veloce on 2007-10-21
You can edit the .vmx file - its just a text file.

Or you can use vmware console to edit the hardware configuration - provided the vm is shutdown.

---

### Post by bodhi.zazen on 2007-10-21
As veloce said, shut down the virtual machine first.

Then in the VMWare Server Console, double click Memory and adjust the available memory.
