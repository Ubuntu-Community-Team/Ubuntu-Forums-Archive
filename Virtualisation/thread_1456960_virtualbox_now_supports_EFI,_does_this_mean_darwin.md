### Post by rliegh on 2010-04-18
Out of the box, no -not right now. From what I've read over on the virtualbox forums, Mac OS/Darwin is planned to be supported in 3.2 (I'll believe it when I see it, however).

People are talking about the workarounds they're using to get OS X booting on virtualbox; though I don't believe they've come up with a native install yet. 

You might want to read through a few pages on this thread over there: [http://forums.virtualbox.org/viewtopic.php?f=4&t=2076](http://forums.virtualbox.org/viewtopic.php?f=4&t=2076) but the short answer is -wait for virtualbox 3.2
