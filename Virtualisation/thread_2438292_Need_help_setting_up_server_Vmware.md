### Post by TheFu on 2020-03-09
There are thousands of youtube videos about this.  Find one to follow it.
Be aware that VMware Workstation versions have specific Ubuntu Releases they support, so you should check with the VMware Workstation FAQs and capabilities list to ensure you install only a Linux distro  supported by the version of Workstation used.

Most people here won't be using a commercial product, especially since KVM+libvirt+virt-manager is much more capable and faster than anything VMware sells.  Of course, KVM only runs on Linux hosts, so if you are stuck using some other hostOS, your options are limited.
