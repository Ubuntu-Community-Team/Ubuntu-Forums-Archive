### Post by howefield on 2010-12-29
Moved posts to own thread.

Add the extensions pack by opening the Virtualbox Manager (Applications > Systems Tools > Oracle VM VirtualBox) and navigate to File > Preferences > Extensions.

Click on the Add package button, select the extensions package file that you have downloaded and OK your way back out.
