### Post by TheFu on 2014-06-30
Danger Will Robinson.
Windows installs based on the hardware seen ... so your virtual install is for the virtual motherboard, virtual chipsets, virtual NICs, virtual drive controller, and virtual BIOS, etc....   

The drive controller, BIOS and chipsets matter most. It is possible to fix the other things later.

So - will your real, physical hardware match that?   Probably not.

With later versions of XP and all MS OSes from  Vista on, they started tying the installation to the OS even more for pre-installed versions. Retail copies aren't as picky, but I've never been successful at migrating either direction.  Heck - I'd love to migrate the media center settings between systems - seriously - with Linux, it jut works if the data and settings are available, but not with MS. At least that is my experience.
