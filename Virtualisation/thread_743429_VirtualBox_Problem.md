### Post by bodhi.zazen on 2008-04-02
Virtualization uses virtual hardware, including virtual hard drives. The hard drives are files and can be stored anywhere. This is the default behavior and the way I advise you go.

Alternately you *may* be able to directly use the hard drive, but this will take some effort on your part. Start by reading the sticky at the top of this forum.
