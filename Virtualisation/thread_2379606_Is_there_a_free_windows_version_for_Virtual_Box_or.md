### Post by kevin160 on 2017-12-26
You can sign up for Enterprise evaluation versions.  

[https://www.microsoft.com/en-us/evalcenter/evaluate-windows-10-enterprise](https://www.microsoft.com/en-us/evalcenter/evaluate-windows-10-enterprise)

They expire after 90 days, but the evaluation period can be renewed a couple of times by typing a simple command in the terminal.  After that, you will need to reinstall fresh.  

Make sure you abide by the license agreement.  It may have limitations regarding use for real work.  If you think you'll be using Windows long term, it is much simpler to buy a copy.
