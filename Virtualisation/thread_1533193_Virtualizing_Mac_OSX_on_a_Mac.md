### Post by MattBD on 2010-07-19
Your question is quite timely because only very recently, Virtualbox has added support for Mac OS X Server hosts. You can install a patched desktop version of Snow Leopard in Virtualbox, and there was a tutorial on Lifehacker at [http://lifehacker.com/5583650/run-mac-os-x-in-virtualbox-on-windows](http://lifehacker.com/5583650/run-mac-os-x-in-virtualbox-on-windows) - ignore the fact it says on Windows, it should be pretty much the same on every OS.
