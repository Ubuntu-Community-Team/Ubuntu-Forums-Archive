### Post by T.J. on 2016-11-03
I use two Nvidia video cards - one for Linux one for passthrough.  After installing the Ubuntu or Debian packaged driver, the VM would lock up on startup.  The only working solution that I found was to install the driver manually.  I'm posting this so others can verify or take note of this possibility.
