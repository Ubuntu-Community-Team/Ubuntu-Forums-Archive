### Post by MAFoElffen on 2022-03-22
The OpenSUSE Tumbleweed Installation Guide says you need "a minimum of 1GB Memory. More is better." I would allocate 2GB.

On creation, from CD (will mount the ISO you point it to). On startup, it will boot to ISO to install it.
