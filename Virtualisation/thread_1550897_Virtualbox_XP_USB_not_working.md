### Post by bowens44 on 2010-08-11
1. You can't use the VirtualBox from the repositories , you have to get it from [www.virtualbox.org](www.virtualbox.org)

2. you have to install guest additions

3. you have to go into the group settings and make sure your user ID is in the virtualbox group and that the check box next to the ID is checked.

if you have done all of this, it should work.
