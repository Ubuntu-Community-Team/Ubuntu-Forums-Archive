### Post by dcollier on 2011-05-17
Yes you can!  Click on the virtual machine that you have just created > click the settings button > click storage > click the storage device that looks like a cd > in the attributes section click the lil disk to the right of the section "cd/dvd drive" and choose a virtual cd/dvd disk file > navigate to the iso and your system will no boot with the iso that you selected.  :)
