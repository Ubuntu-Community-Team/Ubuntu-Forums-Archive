### Post by bodhi.zazen on 2009-05-22
I suggest KVM

Alternates to KVM also include openvz

I used to run VMWare, but I got frustrated with having to find and patch vmware to run it.

I also agree , vmware server 1.x is a great product.

I do not care for vmware 2.x , FYI IMO newer is not always better. server 1.x has more features then 2.x (and you can install a web interface if you like, but why do I want to run a web server on the host ??? )

Virtualbox is probably the easiest to set up , but it is in rapid development, so configuration is always changing. I have had occasional issues with guests when upgrading Virtualbox.

Once you are faimiliar with virtualization, and if you have the hardware, KVM is IMO faster then virtualbox .
