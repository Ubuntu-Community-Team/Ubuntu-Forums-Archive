### Post by jocampo on 2009-12-28
Has been a long time since I use a key or active a Microsoft Product 'cause my work, we have Volume Licenses ;-) ... but as long as you use your license 1 time and on same machine , I believe there is no issue. OEM licenses are intended for use with same machine or hardware because you paid a bit less thanks to some kind of agreement with the PC manufacturer.
 
But you can always call Microsoft and request additional information or check online; this link could help you a bit ...
