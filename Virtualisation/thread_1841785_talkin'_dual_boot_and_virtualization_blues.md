Hi All,
I have (had?) a Win7 installation and found myself needing Linux for work related reasons, so I dual-boot installed Ubuntu on an external HDD with GRUB on the external MBR and everything was great. I could boot into either OS fine, and Ubuntu could mount my Windows partitions to get at mp3s and other files.

Then I had the brilliant idea that I would create a VirtualBox in Ubuntu to virtualize my existing Win7 installation so I could have a few Windows-specific things running while I did most of my work in Ubuntu. I found *some* instructions on how to do this, but a lot of it was old, and I'm not super handy with *nix, and the VirtualBox HOWTO didn't make a whole lot of sense to me. I wound up frying my Win7 bootup process. Actually not terrible, Ubuntu still works fine :)

So now I'm at a bit of a crossroads. I wouldn't mind having Ubuntu as my primary OS, and there are plenty of great tutorials for virtualizing Win7 in Ubuntu (just not how to virtualize existing Win7 installations), but now I wonder:
[LIST]
[*]Is it safe to leave Ubuntu installed on an external HDD? I feel like it should be move to a more stable, internal SATA drive. 
[*]If so, should the installation be cloned, or better to just reinstall? I really don't want to go through setting up my SVN repositories again...
[*]Regardless of where my Ubuntu winds up residing, once I virtualize Win7 in Ubuntu, will there be any way to boot directly into that Win7 installation, ever? I really like the idea of being able to:
[LIST]
[*]Open a Win7 window when I want in Ubuntu
[*]Booting directly into Win7 when I want to
[/LIST]
[*]I know in the back of my mind I should just back away slowly from dual-boot, it's done me bad and treated me wrong before, but I just keep coming back to it.
[/LIST]

Hope this isn't TL;DR. I wouldn't blame you.

Thanks, and thanks for all the help you've given so far but weren't aware of having given; I've been lurking for quite some time now.
