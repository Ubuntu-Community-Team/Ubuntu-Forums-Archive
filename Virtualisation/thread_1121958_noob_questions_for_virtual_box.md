### Post by linuxuser21 on 2009-04-10
One thing I'll give Windows is its easy installation.  All you have to do is create a new guest OS.  Select Windows XP and make the settings to your liking.  The minimum amount of RAM you should have XP (usefully) running is 256mb.
Next, in settings, make sure that you have it set up to boot from CD/DVD first.  Start it up, it should boot from the CD, install it and you should be ready to go.
