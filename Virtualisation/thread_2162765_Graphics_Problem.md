### Post by lah7 on 2013-07-30
Since this is a graphics problem in a virtualisation category, have you installed any guest additions for Ubuntu?

Usually, guest additions software will have special drivers that will help optimise graphics and other features.
If you're using Virtualbox, you can install them by clicking "Devices --> Install Guest Additions" on the menu bar.
You may be able to install these within the VM by going to the Unity dash, typing "Software Sources" or/then selecting "Additional Drivers"

When you say "newest version" ... do you mean the virtualization software or Ubuntu? Since the screenshot looks awfully like 12.04 LTS, not the latest non-LTS, which is 13.04 at this time of writing.
