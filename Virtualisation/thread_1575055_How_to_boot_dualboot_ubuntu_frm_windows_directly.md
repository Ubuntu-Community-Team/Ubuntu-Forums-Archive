### Post by fjgaude on 2010-09-16
What I think you are suggesting is while you are running Windows you wish to boot, startup Linux?

The only way I know of doing this is using virtualization, such as VMware Player.

You would install Player in Windows and from there install Ubuntu. You would have two desktops, one for Windows, the other for Linux.

Go to vmware.com and then to downloads and see the Player menu, download and install.
