### Post by bodhi.zazen on 2009-05-07
Well, you should be able to simply copy the server.qcow.

On the clone I suggest you then :

1. Install minimal services. Most of the time you will want to st up mysql, php, ssl, etc later.

2. For the ssh keys, do this :

```

rm -f /etc/ssh/ssh_host_*
cat << EOF > /etc/rc2.d/S15ssh_gen_host_keys
#!/bin/bash
ssh-keygen -f /etc/ssh/ssh_host_rsa_key -t rsa -N ''
ssh-keygen -f /etc/ssh/ssh_host_dsa_key -t dsa -N ''
rm -f \$0
EOF
chmod a+x /etc/rc2.d/S15ssh_gen_host_keys
```

3. for networking, go to dhcp, change to static IP after cloning.

4. Update the install.

5. Clean.

See also the openvz documentation and JEOS.

[http://wiki.openvz.org/Debian_template_creation](http://wiki.openvz.org/Debian_template_creation)

Most of what is on that openvz link will not apply, but the general idea is valid.
