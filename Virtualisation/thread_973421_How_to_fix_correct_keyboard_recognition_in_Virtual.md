### Post by kramer65 on 2008-11-06
I've been struggeling with Virtualbox for months now. Although the install went fairly smooth there has always been one thing that bugged the shite out of me; the keyboard sometimes reacted well to some signs (e.g. ";?.><=+_-*~` etc.) after which it could somehow immediately change so that the signs that where on my keyboard did not display correctly anymore and I had to push all the special sign keys to find a simple questionmark.

Today I found the solution which I just wanted to share with you guys. If you press the left shift and the left control at the same time, my keyboard switches from the normal (US) to some other layout (I think the belgium one).

So; Ctrl + Shift!!

I hope I could help people with this since I searched for it for ages but couldn't find the answer!
