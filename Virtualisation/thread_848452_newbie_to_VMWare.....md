### Post by llib1234 on 2008-07-04
you have to install vmware tools first.
Then click on "Edit virtual machine setting" on the Windows page. Goto the Options tab and Click on Shared Folders and enable it. Now Choose a folder that you want to use for sharing files between your Linux and Windows VM.
