### Post by HermanAB on 2009-01-29
Well, the 21st century solution certainly is to install XP in a VM.  That way, you can run it in a window without having to reboot.  You can boot the existing parition with a virtualizer such as VMware or Virtualbox, but the MS registration system causes problems.  It is better to make a new VM.

Hope that helps!
