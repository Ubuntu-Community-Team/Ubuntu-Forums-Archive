### Post by Ginsu543 on 2010-08-16
As long as Photoshop runs on a Windows VM, I'd say it's TOTALLY worth it. It's really nice not having to reboot to switch from one OS to the next. I need Windows for two Windows-native programs, BibleWorks and iTunes, and both work flawlessly on my Windows XP guest on VirtualBox. You should definitely give VirtualBox a try. It supports Windows 7 and it's FREE; what do you have to lose?

If you install VirtualBox from the Ubuntu repos using Synaptic, you will get the OSE version. This version does NOT support USB. So if you want USB support (as I do to sync my iPhone), you want to download the PUEL version directly from [_VirtualBox.org_](http://www.virtualbox.org/wiki/Linux_Downloads).

### Post by snowpine on 2010-08-16
Assuming your hardware is up to the challenge, the only downside to virtualizing is the cost (you'll need an additional Windows license).

Some users have had bad experiences with virtualized Windows for video-intensive games, but for running Photoshop on a fast computer, it should be fine. :)

I have Photoshop running on XP in Virtualbox PUEL, no problem.
