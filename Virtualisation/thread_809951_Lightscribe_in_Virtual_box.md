### Post by bradmkjr on 2008-05-28
Sorry, you can't

Virtualization uses virtual hardware, not your actual hardware. Virtualization creates "fake" hardware to the lowest common denominator to which most operating systems will run. Currently that means that it only supports a standard read only cd rom drive, not dvd, not cd burner, not dvd burner, not a light scribe drive.
