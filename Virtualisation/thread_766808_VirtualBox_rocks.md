### Post by snappy46 on 2008-04-25
Thanks to the new upgrade from Ubuntu 7.10 to 8.04 I could not get wmware to run for me anymore.  I try many of the solution that I found but still no luck.

I finally gave up and when on to installing Virtual Box to see if I had better luck. It install flawlessy and I was up and running within minutes.  I was really suprised by how easy it was to configure and run.

Of course they are still a few things that had to be polished like the fact that I could not attach any USB devices; but that was quickly  rectified by applying the right user authorization.

My only complain at this time is the error I am getting when I enable the serial port but I am sure I will find a solution for this problem eventually.  I was also not able to use the vmdk image file that I had used with vmware but I am not done experimenting with that either.

Anyhow thanks to the upgrade I am now using Virtual Box and find it impressive so far... time will tell.  I love the seamless mode where I now have a gnome menu on top and window menu on the bottom; seamless operation at it's best.

We are almost there pretty soon I will be able to ditch my window partition out the window!!!!

Any help on the forementioned problems would be appreciated.

Snappy :)

I ditched vmware when I discovered that I couldn't use it (1.0.x and 2.x beta) with vista x64 as a host system. Switched to virtualbox and other than no 64bit guest support, I 've been happily sharing imagees between vista and hardy.
