### Post by Dedoimedo on 2009-01-26
Doable, it's called mounting a physical partition. Check the big mega thread that's stickied in this forum.

BUT ...

This is not for non-experienced users. Mounting a physical partiton and using it improperly, like improper mount / unmount, bad drivers, writing when hibernated etc can cause serious data corruption.

I know experienced linuxers who have screwed up their hard disk doing this ... I'd recommend you image your physical xp install and then restore it to virtual machine.
