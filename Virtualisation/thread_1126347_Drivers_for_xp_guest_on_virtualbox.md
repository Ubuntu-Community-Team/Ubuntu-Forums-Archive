### Post by Dedoimedo on 2009-04-16
You need to install Guest addons:
[http://www.dedoimedo.com/computers/virtualbox-guest-addons.html](http://www.dedoimedo.com/computers/virtualbox-guest-addons.html)

This will give you generic graphic drivers, no 3d. If you enable 3d support via the settings menu for the virtual machine, you'll also have opengl support, but not directx for the time being.
