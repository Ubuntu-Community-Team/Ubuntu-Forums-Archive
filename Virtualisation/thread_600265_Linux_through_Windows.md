### Post by bodhi.zazen on 2007-11-03
I think he is talking about booting/using linux within windows, not installing.

wubi is for installing Ubuntu without having to boot to a CD or re-partition your hard drive.

Anthrax9: I am guessing you want to access your Ubuntu partition.

If you just want to access the files, take a look at :

[http://www.fs-driver.org/](http://www.fs-driver.org/)

If you want to run Ubuntu, that is virtualization. You can look at VMWare or VirtualBox.

However, these programs create a virtual machine and booting a hard drive installation is a more advanced topic, typically one installs into a virtual hard drive
