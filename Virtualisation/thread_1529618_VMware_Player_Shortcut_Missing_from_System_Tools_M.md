### Post by fjgaude on 2010-07-13
At a command line simply enter vmplayer and return... that should get you going.

Right click Applications and then click Edit Menus... you should be able to find the icon for VMware in the System Tools menu.

### Post by franklamoureux on 2010-09-22
Better and easier workaround: Add the "Sysinfo" sofwtare from the Ubuntu Software Center. This software will also show up in the "System Tools" menu group, which makes VMware menus reappear.
