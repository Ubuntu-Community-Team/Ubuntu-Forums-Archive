### Post by CharlesA on 2010-11-07
If you are setting up one server to be a dedicated VM host, check out VMware ESXi.

Virtualbox has an opensource edition and a closed source edition, with the closed source edition having stuff like VRDP and USB support.

If I was going something for enterprise I've use ESXi (my company uses it)

[www.vmware.com/products/vsphere-hypervisor/index.html](www.vmware.com/products/vsphere-hypervisor/index.html)
