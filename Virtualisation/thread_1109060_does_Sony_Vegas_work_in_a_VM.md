### Post by James_Lochhead on 2009-04-01
Sony Vegas will definitely run in a VM, only question is how fast. I run Adobe PS in a VM and that works fine (with only a slight bit of lag - yet functional), however video editing might be a little more resource intensive.

I don't know about the OSS bit though. IMO, OSS VM software is not up to scratch yet (at least the stuff I have tried). I run VMware Workstation and that is a lot better than anything else I have tried.

VMware have a free (cost) version called VMware Server - it is harder to set up and use than Workstation though. 

Another idea for you could be get a VMware workstation trial, create the VMs you want (prolly XP), back them up on CD (just in case), remove VMware Workstation and then download and run the free (cost) VMware player.

I tried to install Adobe PS CS4 on 32 bit XP with VirtualBox but it didn't seem to like it. 64 bit XP didn't work at all (crashed VMware). I know PS is not the same program but it probably has similar kinds of requirements.
