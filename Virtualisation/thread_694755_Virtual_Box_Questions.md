### Post by Therion on 2008-02-12
It is my understanding that you need a "proper" Windows Installation CD to install/use Windows in Virtual Box; NOT a factory-restore CD.

I used the tutorial on [this page]("http://n00buntu.blogspot.com/2007/11/howto-ubuntu-compiz-virtualbox-part-1.html") to install/set up Windows in Virtual Box and it specifically states "no recovery CD's".

You might try asking on the [Virtual Box Forums](http://forums.virtualbox.org/index.php) though for further assistance.
