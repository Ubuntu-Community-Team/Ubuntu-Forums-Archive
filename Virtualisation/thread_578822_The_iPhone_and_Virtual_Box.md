### Post by bodhi.zazen on 2007-10-20
1. Go to your  Virtualbox gui tool, the window that starts when you start Vbox ...

With windows powered down, Select your windows machine, in the "Details" tab select "USB Controller" :
[indent][list][*]Tic the "Enable USB controller" box.[*]On the left there is an icon for your usb devices, it has a green "+" on it, left click that and enable your IPhone (select the phone from the list and click on it)[/list][/indent]

2. Power up windows. You should now be able to use the IPhone in windows. If not, this time in the windows virtual machine box, right click on the little usb icon in the lower right part of the virtual machine and select your iphone from the list of devices.

HTH
