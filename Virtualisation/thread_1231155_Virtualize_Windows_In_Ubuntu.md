### Post by Cheesemill on 2009-08-04
If you're doing this to play games then my advice is don't do it! Go for a dual boot setup instead.
 
Virtualization is only just starting to support 3D graphics, it's still a very new technology and there are lots of bugs and performance issues to sort out.
Also you'll get a lot more speed running Windows natively than in a VM.

### Post by jocampo on 2009-08-04
Please follow Cheesemill advice ...

Because you need direct access to hardware using some Windows Games and graphics, a virtual machine could not be your best choice. So, please setup a dual boot so when you are in the "game mood" :-) you boot into Windows and play. And when you want to browse internet or use your machine without the risks of viruses or spywares ... you boot into Linux
