### Post by Vivaldi Gloria on 2008-06-10
> **Sub101 said:**
> I am currently dual booting Visa and Ubuntu 8.04. I am would like to boot my Vista partition using virtualbox or any other virtualization software. Is this possible?

When you change your hardware then windows protection kicks-in and it refuses to boot. Trying to boot windows with virtualbox etc. changes the hardware that windows sees so it will stop working.

So in general the answer to your question is: No, you cannot do that.

But I remember people somewhat managing to boot an XP partition with virtualbox. I cannot remember the page now. I think I saw it in the virtualbox forums. But I haven't seen anyone do that with a vista partition. Maybe it can be done, but I wouldn't think so. I suggest you have a look at virtualbox forums.
