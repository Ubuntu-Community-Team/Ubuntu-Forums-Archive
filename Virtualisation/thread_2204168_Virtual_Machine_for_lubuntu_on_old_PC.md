### Post by sudodus on 2014-02-07
Lubuntu needs 512 MB RAM to work well itself. And Windows XP needs the same amount of RAM. So dual booting is possible, but a virtual machine would not work at all, or would need heavy swapping, which would make it almost impossible or maybe even totally impossible to use.

Windows 2000, NT and 98 are smaller and just might work in a virtual machine. But I would recommend at least dual core or core 2 duo and at least 2 GB RAM for a nice experience with a virtual machine and current operating systems. I think many people will say at least 4 GB RAM, particularly if you want to run 64-bit systems.

### Post by Lars Noodén on 2014-02-07
Or it might be possible to run your legacy applications inside WINE, thus avoiding dual boot.  That is not possible for all applications, but when it works well it works very well.  Check the [WINE application database](http://appdb.winehq.org/) first to get an idea if it is worth trying for you.
