### Post by WearZeeP on 2007-12-13
A virtual computer means that you can install a complete operating system on a "fake hard drive", for example a file that contains a while operating system and its files and such, which you can then boot from while still inside Ubuntu for example.
I'm really bad at explaining so just have a look at [VirtualBox]("http://www.virtualbox.org/"), which is a really good, free program for running virtual computers.
Another alternative is VMware, but that costs some money i think.

### Post by AndyCooll on 2007-12-14
Virtual Windows really means the Windows XP/Vista running in a virtual machine.

In effect, it runs the Windows OS in it's own virtual/fake environment. You see XP/Vista inside a window on your desktop. Much as you see a Firefox window or an OpenOffice window. You can minimise the Window, put it behind another window, interract with it etc.
