### Post by yetimon_64 on 2015-09-03
>  I have installed Ubuntu 14 on **VirtualBox**. You will need to install the "Guest Additions" package in Ubuntu (the guest OS) to get the full resolution possible when using virtualbox. 

Adding the guest additions package also enables more features like the shared clipboard amongst others.

Edit: some ubuntu documentation for guest additions : [https://help.ubuntu.com/community/VirtualBox/GuestAdditions](https://help.ubuntu.com/community/VirtualBox/GuestAdditions)
