### Post by mrsteveman1 on 2008-02-10
As of right now, Apple doesn't allow OS X to run inside a virtual machine. The upcoming version of VMWare fusion for OS X will be able to virtualize the server version of OS X but thats not even out yet.

It is possible to run the hacked version of OS X inside a VM, however. The process is the same as running it on bare hardware, though i think leopard does not currently work yet.
