### Post by grumblebum2 on 2014-05-25
In Trusty, the Ubuntu Unity Plugin draws the decorations for the in app menus in the titlebar.
Window Decorations plugin can't be enabled in unity.
What visual effects you looking for? Look in ccsm.

Install unity-tweak-tool for an easier to use tweaker than ccsm...
```
sudo apt-get install unity-tweak-tool
```
