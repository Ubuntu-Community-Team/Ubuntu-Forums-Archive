### Post by robvarga on 2008-08-23
Ok, I found the error. 

There is a bug in ubuntu-vm-builder: it does not install the grub bootloader in the image if the host system does not have grub installed, because it tries to invoke grub from the path in line 660.

Correcting it to target/usr/sbin/grub made it work.
