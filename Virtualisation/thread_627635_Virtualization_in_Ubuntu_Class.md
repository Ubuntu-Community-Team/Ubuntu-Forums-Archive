I'm offering a "Virtualization in Ubuntu"  (Though it works for almost any Linux OS) class starting this monday.

If you're a Linux user, and have heard about Virtualization, or maybe your company wants you to start exploring virtualization, this class is right for you.  It's four weeks long, covers the basics of Virtualization, including the installation and configuration of two popular Virtual Machine servers, VirtualBox, and Vmware.

This is an intermediate level Linux course.

You are expected to have a Linux machine available to you, that you had administrative rights on so you can install and configure the software.  If you don't have this, you may want to consider the "Introduction to Ubuntu" class that  will be starting a new session sometime in January.

The class is absolutely free, and available at [http://learning.mohnkern.com](http://learning.mohnkern.com)
