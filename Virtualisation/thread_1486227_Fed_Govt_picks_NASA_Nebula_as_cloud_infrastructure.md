### Post by bmullan on 2010-05-17
The Federal Government recently announced NASA's [Nebula]("http://nebula.nasa.gov/services/") architecture as its Cloud foundation.

The architecture includes some surprises:

    * Its uses open source Linux
    * Open Source E**ucalyptus** is the cloud virtualization environment used
    * Its KVM based
    * 15 PetaBytes of storage
    * Single file size limit of 1 PetaByte
