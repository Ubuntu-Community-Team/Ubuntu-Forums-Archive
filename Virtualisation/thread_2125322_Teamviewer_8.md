Or, are you trying both the XP and Native versions and saying you can connect with either?

To control the laptop from a different computer, all you should have to do is enter the laptops UID into the Partner ID Field, and plug in the password from the password field on the Laptop to the same field on the controller machine.

That should work fine.  I've been able to use Teamviewer on several machines that way with no real problem other than lag.
