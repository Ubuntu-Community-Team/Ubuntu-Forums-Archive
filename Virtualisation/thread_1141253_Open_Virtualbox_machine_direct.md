### Post by kneewax on 2009-04-28
Here's a question because I am basically lazy and would like eliminate the three clicks it takes to open Vbox, select a machine and start it.

Is it possible to to create a link that opens my XPpro machine without going through the management console?

---

### Post by Jose Catre-Vandis on 2009-04-28
Put
```
VBoxManage startvm "XP"
```
In a launcher
