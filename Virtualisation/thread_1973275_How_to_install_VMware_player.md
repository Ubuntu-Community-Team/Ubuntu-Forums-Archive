### Post by fjgaude on 2012-05-06
Move to the folder where the .txt or .bundle file is and from the terminal issue the command:

```
sudo ./VMware-Player-4.0.3-703057.x86_64.txt
```

Make sure you have the right Player version name and make sure the file is executable.
