### Post by Siph0n on 2007-10-22
I only installed VMWare Server on feisty (7.04), but all I had to do was go into Synaptic, checkmark VMWare Server, click Apply, and while that was downloading and installing I went to VMWare's website and registered for a free registration key. Than while Synaptic was installing VMWare server it asked for a key, I put that key I got from the website in, and all went well :) Hope that helps...

Oh and to get a Windows XP virtual machine, just have your cd drive mounted as the cd drive for your virtual machine, put your legit windows xp cd in your cdrom drive, and than boot your virtual machine and it will boot the cdrom drive and install windows ;)
