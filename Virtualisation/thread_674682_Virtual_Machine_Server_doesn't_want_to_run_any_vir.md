### Post by kingleer on 2008-01-21
Okay, I'm running the 64 bit version of Ubuntu 7.10. Everything is up to date, and it's a fresh install. I've installed the latest version of Vmware server. Vmware server lets me create a new virtual machine, but when I try to run it, all I get is a black screen and the virtual machine shuts down a few seconds later. When I look at the bottom right hand corner there's a red x over the cd drive and the ethernet connection. The same thing happens when I try to open an old Vmware machine that runs fine on the same version of Vmware server on Windows. 

Any help would be greatly appreciated. Thanks in advance for your time and consideration.

### Post by kingleer on 2008-01-22
I solved my own problem. The virtual machine needs to be located in the default folder for virtual machines in order to run.
