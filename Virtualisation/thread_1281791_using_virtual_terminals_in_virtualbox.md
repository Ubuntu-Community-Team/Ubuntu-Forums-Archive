### Post by pythonscript on 2009-10-03
I'm using the proprietary version of virtualbox with various linux guests (probably around 10 different ones, just for testing purposes) and I'm wondering I can open up virtual terminals in distros that support them? Control+Alt+Fx brings up a virtual terminal in my host system, regardless of whether or not I've captured keyboard input in my guest. Any idea how I can do this in a linux guest?

---

### Post by pythonscript on 2009-10-03
I actually just figured this out! Sorry about the 10 minute post. In Virtualbox, the host key (in my case Right Ctrl) + F2, F3, etc, brings up that virtual terminal. Hopefully that helps someone else out too.
