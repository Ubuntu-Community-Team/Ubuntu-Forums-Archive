### Post by deadflowr on 2018-09-30
If you have the top menu (File Edit View, etc etc) then the toggle for the bottom is View > Status Bar > Status Bar; just click it.
You can also set a keyboard shortcut for it in File > Preferences > Input (scroll down to Status Bar and click to open/edit the empty space, put in any key combo for it that works for, and most likely isn't one that would conflict with an existing one.
