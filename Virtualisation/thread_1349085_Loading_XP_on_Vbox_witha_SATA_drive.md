### Post by underquark on 2009-12-09
You can copy your XP CD to an ISO file and then mount that ISO as a CD to the virtual machine.  You might see an option to press F12 to select boot device, in which case, choose the CD.  Why do you need to load XP into an existing machine as opposed to just making a new one?
