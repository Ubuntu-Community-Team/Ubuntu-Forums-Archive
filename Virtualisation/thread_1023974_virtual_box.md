### Post by pytheas22 on 2008-12-28
Do you mean that you want to install Windows XP into VirtualBox using an .iso file of the Windows CD, rather than the CD itself?  If so, just tell VirtualBox to mount the .iso image that you want as the CD drive.  You can do this under Settings in VirtualBox.

If you're just looking to mount .iso images in general, there are scripts that allow you to do that easily.  [Here]("http://www.ubuntugeek.com/mount-isos-easely-in-gnome-nautilus.html") is an example, although there may be better ones if you search.
