### Post by philled on 2015-12-23
Actually, I've found my own answer to this. If you install the GetDeb PPA you can install virt-manager 1.2.1. See [http://www.ubuntuupdates.org/ppa/getdeb_apps?dist=trusty](http://www.ubuntuupdates.org/ppa/getdeb_apps?dist=trusty)
