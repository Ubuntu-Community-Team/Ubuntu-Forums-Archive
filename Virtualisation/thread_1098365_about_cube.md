### Post by Therion on 2009-03-16
Go to System/Preferences/Appearance.
Open **CompizConfig Settings Manager**.
[INDENT](If you don't HAVE the CompizConfig Settings Manager install it from Synapic (search on, and install, **compizconfig-settings-manager**)).[/INDENT]
Open the "General Options" plugin.  
Go to "Desktop Size" tab.
Increase "Horizontal Virtual Size" from 2 to **4**.
Activate these plugins from the CCSM menu:
[indent]*  Desktop Cube
*  Rotate Cube[/indent]
Close CCSM.
Use Ctrl+Alt+Left Mouse Button and move the mouse.
You should see your Cube.
