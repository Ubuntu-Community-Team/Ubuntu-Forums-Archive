### Post by yancek on 2019-06-05
The messages are always there but usually do not show on screen when 'quiet splash' are on the linux line in the /boot/grub/grub.cfg file menuentry.  The link below explains how to show the messages so just do the reverse.  It would be more of a problem if it showed "Failed" rather than "OK".
