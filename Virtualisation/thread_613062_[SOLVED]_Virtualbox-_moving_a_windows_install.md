### Post by koolguynet on 2007-11-15
I ended up having to use a spare copy of XP pro I had laying around.  Lenovo does not include the Product Code since they use recovery discs (hard drive images).  I was able to recover the product code through the registry using software from [www.magicjellybean.com](www.magicjellybean.com), but it turned out to be an OEM key which won't work with retail media.  

Right now XP pro is running in VirtualBox.  I am happy with it, but a little pissed that I had to use my license on a work laptop.  Whatever...I wasn't using it anyway.  Thanks for the help!
