### Post by yther on 2009-02-13
VirtualBox is a virtual machine provider.  As Mabinogi is a Windows game, you'd have to install a supported version of Windows (XP or Vista, I believe) *inside* a VM, and then install the game into that.

I don't know where VirtualBox goes in the menus. Here on KDE, I have the VMware Player under Applications > System, so maybe VBox lands in a similar place.

Good luck!
