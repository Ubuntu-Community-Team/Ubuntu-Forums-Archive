### Post by Objekt on 2010-09-27
Here's yet another vote for Virtualbox.  As long as you aren't doing anything graphically-intensive, such as games, it works great.  I use it for the few programs that don't have Linux equivalents or workarounds, or which don't work through Wine.  Sounds like more or less what you want to do.

I recommend the PUEL version of Virtualbox, because it lets you use USB devices on the virtual machine.  Unfortunately, the virtualbox.org website is down at the moment, so you may have to wait a while to download the installation package.

Your hardware is similar to mine, except that I have a faster CPU.  I've allocated up to 2 GB RAM to the virtual XP machine without causing any problems in my Ubuntu 9.10 host machine.
