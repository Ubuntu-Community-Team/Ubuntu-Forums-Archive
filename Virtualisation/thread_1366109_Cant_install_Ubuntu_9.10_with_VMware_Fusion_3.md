### Post by boyfarrell on 2010-01-13
I have the same problem and this is also a problem with 3.0.1.

**Solution:** Download the text only install CD. Go to this page [http://www.ubuntu.com/getubuntu/downloadmirrors]("http://www.ubuntu.com/getubuntu/downloadmirrors") and choose a mirror close to your location under the 'Complete Download Mirror List' header. Then download the file ubuntu-9.10-alternate-i386.iso (Intel) or ubuntu-9.10-alternate-amd64.iso (AMD).

E.g. a UK link [ubuntu-9.10-alternate-i386.iso]("http://releases.ubuntu.com/9.10/ubuntu-9.10-alternate-i386.iso")
