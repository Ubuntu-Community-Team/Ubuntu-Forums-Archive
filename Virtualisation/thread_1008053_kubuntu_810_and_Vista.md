### Post by HavocXphere on 2008-12-11
The following might be relevant:
[QUOTE=Wikipedia]
Operating systems

There are two types of OEM when it comes to operating systems. The first is when a pre-built computer is purchased, there is an OEM disk that comes with the system, which can not be transferred to any other system, because that **disk is designed to run only with the specific system components**. The second type of OEM operating system is one that can be purchased, and that can be transferred to any other system, without the dependence on the systems components. The main difference between the two is that one is only meant for that specific system, and the other can be used on any system.[/QUOTE]

e.g. The disk is designed to only run on one pre-built DELL/HP hardware combination. The exact details of how much you can change before it complains is not disclosed by Microsoft.

Unfortunately, there is no easy way around this. It is *designed* to be tamper-proof since its part of MS's strategy against piracy. You safest bet is to contact the corporation to sold you the PC.

As for XP...it depends on what version of Vista you've got. Vista Ultimate and (I think) Business both include an XP licence.

### Post by HavocXphere on 2008-12-11
Tutorial?

First install Vista. There is one dialog where it shows the harddrives. Press the Advanced/whatever button. You can create/delete/format/etc partitions in that window. Then make a partition using half/(or whatever you need). Then install kubuntu.

Thats with a normal Vista disk. I don't have any experience with OEM disk, so I couldn't tell you how those differ.

ALWAYS Windows first, then linux.
