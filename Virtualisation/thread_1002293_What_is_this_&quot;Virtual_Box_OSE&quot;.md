### Post by Thelasko on 2008-12-05
VirtualBox and VMWare both create virtual hardware on which you can install an operating system of your choice.  Yes, you will need an installation disk.  You can pretty much treat it like another machine.

### Post by cdmike2k8 on 2008-12-05
You still need to install windows.  You have a few options.  You can insert the install disk and forward it by clicking on CD/DVD while the virtual computer is off.  Alternately, if you happen to have an .iso of the bootable disk, you can forward that as well, through the same menu.  Good luck.
