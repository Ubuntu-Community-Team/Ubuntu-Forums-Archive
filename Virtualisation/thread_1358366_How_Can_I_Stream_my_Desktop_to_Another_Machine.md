### Post by KeLa on 2009-12-18
On the computer with the projector start VLC for reading network stream and on laptop or any other computer start VLC to send stream to network.
Used it, tested it and works. System had one computer sending video stream and 20 receiving it. It was one kind of info system and there was multiple still pictures/ info pages slide show running all the time and with one pc you can start all client's to wait the video and  server to serve them.

It will be much easier in your case when you have only one client. (meaning of remote control of receiver pc's)
