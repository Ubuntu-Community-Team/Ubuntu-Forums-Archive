### Post by Mark Phelps on 2013-11-24
>  Can I open windows 7 in Ubuntu using VirtualBox?

If you're asking can you point VB to your existing Win7 install on the same PC and run that inside the VB?  The answer is no.

Also, when you reinstall Win7 inside VB, that is a NEW installation, meaning, you will have to install all the apps you want to use, and do LOTS of Windows updates to get it current for Win7.
