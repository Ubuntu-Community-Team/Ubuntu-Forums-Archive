### Post by charonred on 2008-09-02
VirtualBox is great; I installed the Sun Microsystem xVM version a few days ago (it has USB support, whereas the OSE edition doesn't)

[get it from here]("https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_SMI-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=innotek-1.6-G-F@CDS-CDS_SMI"); just select the Ubuntu version and download it, then click on it for the install process - easy as.

I have XP Pro incl SP3 and Vista Ultimate incl SP1 installed in VBox and both work fine. Both the OS's install and run faster than they do natively on their own hard disc.

I'm also working on installing Win 95/98 and Win 3.11 (for a laugh), but there have been a few hiccups with them.

Creating a New machine; just use the the dynamic disc size option, I gave each OS 50 GB to start with, and set both to use 512 MB RAM with each using 64 MB Video.  

I can have both guest OS's open at same time (just open another Virtualbox window for 2nd / 3rd OS) - keep in mind that your total RAM and video must be less than the actual physical RAM & video (I have 2 GB RAM & 512 MB video) - leave plenty for your Ubuntu host.

This is fun :)
