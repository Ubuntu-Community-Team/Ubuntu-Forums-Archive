### Post by bodhi.zazen on 2009-01-07
You can do it with VMWare or Virtualbox, follow the how-tos.

Virtualization does NOT directly use your hardware, it uses emulated hard drives, videocards, network cards, etc.

There is *some* support starting to improve the video support, but it is just starting.
