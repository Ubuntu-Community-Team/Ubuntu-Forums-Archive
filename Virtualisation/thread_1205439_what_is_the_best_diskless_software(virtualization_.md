### Post by scorp123 on 2009-07-06
> **cillian33 said:**
> could you inform me what is the best diskless software? which is used in enterprise, educational market, bank. and so on. i mean client computing -based software, any one get any idea?  LTSP was already mentioned ... but I have never ever seen LTSP being used anywhere.

You asked about "which is used in enterprise" ... This one sure is:

"Sun Ray Server Software"
[http://www.sun.com/software/sunray/index.jsp](http://www.sun.com/software/sunray/index.jsp)

But SRSS requires Sun Ray clients (proprietary hardware!):
[http://www.sun.com/desktop/sun-ray-clients.jsp](http://www.sun.com/desktop/sun-ray-clients.jsp)

There is a way getting around those clients and using RDP, but that would then be a Solaris-based solution:

"Sun VDI3"
[http://wikis.sun.com/display/VDI3/Home](http://wikis.sun.com/display/VDI3/Home)

Guide on how to get it all working on one single PC:
[http://wikis.sun.com/display/VDI3/Getting+Started+-+VDI+Demo](http://wikis.sun.com/display/VDI3/Getting+Started+-+VDI+Demo)

Theoretically you'd need to purchase licenses if you want to use SRSS and/or VDI3 ... But there is no licensing mechanism anywhere in those products, the license is neither checked nor does the software stop working after 90 days of evaluation. And SUN Microsystems (and Oracle who bought SUN too I guess?) usually doesn't go after individual enthusiasts who want to give that software a test-ride ... 


Another solution which is also "used in enterprise, banks, etc" would be this one:

"Sun Secure Global Desktop"
[http://www.sun.com/software/products/sgd/index.jsp](http://www.sun.com/software/products/sgd/index.jsp)

... Unless you're a big company you will not be able to afford the license for that one. And this one has a built-in license checker so it will stop working after 20 days unless you provide a valid license code.


Then, last but not least ... there seems to be a free Linux-based clone of the above. It's also a "web top" solution (just like SSGD) but free:

"Ulteo Open Virtual Desktop"
[http://www.ulteo.com/home/en/ovdi/openvirtualdesktop](http://www.ulteo.com/home/en/ovdi/openvirtualdesktop)

But I haven't tried that one yet and I don't know if it's already in wide-spread use in any sector.
