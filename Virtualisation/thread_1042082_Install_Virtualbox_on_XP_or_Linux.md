I'm not a big Windows fan, so I would install Ubuntu as the base system, and use VirtualBox to emulate Windows inside it.  Using Windows in Virtualbox from Ubuntu makes it faster than if you run Windows by itself.

Ubuntu is still faster than XP, even when it's running XP in virtualbox. 

Besides, there's no point using Ubuntu that I can see if you're running it inside Windows XP.  You get none of the security benefits of having Ubuntu as your main system; none of the speed, or flexibility.
