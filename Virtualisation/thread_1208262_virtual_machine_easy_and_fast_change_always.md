### Post by patryk77 on 2009-07-09
I don't really have time to go into details, but I would use Virtualbox.

Read the sticky threads (Mega Virtualization thread)... It'll have tons of info on it.

You can set it to autostart:
[http://ubuntuforums.org/showthread.php?t=758568](http://ubuntuforums.org/showthread.php?t=758568)

Although I suggest not autostarting a virtual machine, as it uses lots of resources and unless you really need Windows to be running all the time, you'd be better off starting it manually ;)

The way that is used to change desktops in that video is by Compiz (3D openGL special effects)

Easiest way to change desktops is ctrl-alt+ left or right arrow.

Good luck :)
