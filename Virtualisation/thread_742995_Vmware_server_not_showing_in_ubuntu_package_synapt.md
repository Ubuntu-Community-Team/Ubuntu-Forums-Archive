### Post by lifeheart on 2008-04-03
I would like to inform you that  Vmware server is now in "Synaptic Package Manager" - and I have downloaded it this is a proof . 

[[IMG]http://b.imagehost.org/t/0385/Screenshot-Synaptic_Package_Manager.jpg[/IMG]](http://b.imagehost.org/view/0385/Screenshot-Synaptic_Package_Manager.png)

Please know that this picture is hosted on free server, however if you can not get the image for any reason..  know that the picture is showing  VMware server edition available for downloaded though "Synaptic Package Manager". 

any way all what I have done is the following 

open Synaptic Package Manager and then click on settings then click on prerequisite

under Synaptic Package Manager -> Settings -> Prerequisite -> check all given options under the Ubuntu Software tab expect the Source code box . 

move to third-party software tab and select all give options. 

after that click on close and try to search for VMware useing Synaptic Package Manager and you will see it
