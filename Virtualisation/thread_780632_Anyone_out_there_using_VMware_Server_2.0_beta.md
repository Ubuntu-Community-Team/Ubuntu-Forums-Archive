### Post by scottro on 2008-05-03
I used it and didn't like it.  I believe I was able to use my old VM--however, it was over a month ago, and I fear I've forgotten.
You don't start it with a command--you open a web browser.  It seemed very slow and as I took the default port 80 during the install (which was a mistake, but I didn't realize what would happen) I could no longer access my cacti and nagios web interfaces.  I believe (again, I fear I don't remember now) that it also included java, further slowing things down.  At the time, judging from their forums, I wasn't the only one disenchanted with it.  After that, I switched to VirtualBox.  Ironically, this newest VirtualBox 1.6 upgrade seems to have some trouble with bridged networking, both wired and wireless, with a Linux host and MS guest.  However, versions 1.5.6 and 1.5.7 work a treat.
