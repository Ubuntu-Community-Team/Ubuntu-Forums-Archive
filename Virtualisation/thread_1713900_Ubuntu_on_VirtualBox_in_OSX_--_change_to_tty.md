### Post by lmarmisa on 2011-03-24
Try

Alt (Option) + Ctrl + F1

---

### Post by danielstrong52 on 2011-03-27
Thanks for the reply Imarmisa. Your suggestion didn't work initially, but I have managed to fix the problem. In OSX System Preferences >> Hardware: Keyboard "Use F1, F2, etc keys as standard function keys" was unchecked. Both Host+f1-6 and Alt+Ctrl+f1-6 work now. Thanks anyhow.

Dan
