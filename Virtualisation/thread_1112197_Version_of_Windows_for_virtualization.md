### Post by kpatz on 2009-03-31
You can have any number of guests on one system, you know, so install all the versions you got with MSDN... XP, 2003, Vista, 2008, 7 Beta, etc., each in their own virtual machine.  If you have an MSDN subscription, you're probably developing or testing in different versions of Windows anyway, so VMs are really handy for that.

Then, after you decide you don't like a certain one (e.g. Vista), turn on the fire effects in Compiz and then blow them away! :)
