### Post by tacantara on 2009-10-16
If you're open to the idea of virtualization, you can install a VM in Windows to run Ubuntu (or vice-versa).  Sun Microsystems makes a free VM program called VirtualBox for Windows and Linux hosts.

Another option is to install Ubuntu inside of Windows, using a program such as Portable Ubuntu.  More information on that is at this website:  [http://lifehacker.com/5195999/portable-ubuntu-runs-ubuntu-inside-windows](http://lifehacker.com/5195999/portable-ubuntu-runs-ubuntu-inside-windows)
