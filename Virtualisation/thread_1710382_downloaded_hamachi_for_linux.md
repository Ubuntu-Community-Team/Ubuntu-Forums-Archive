### Post by ztefn on 2011-03-26
Hamachi for Linux has a command line interface only. Type **hamachi help** in a terminal to see the available commands.

There are a few third party (open source) graphical user interface's for Hamachi, two also compatible with the Hamachi² beta:
[Haguichi]("http://www.haguichi.net/") (GTK, works fine in KDE too)
[Quamachi]("http://code.google.com/p/quamachi/") (Qt, no deb package)
