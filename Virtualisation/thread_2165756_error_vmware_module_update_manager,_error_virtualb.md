### Post by howefield on 2013-08-06
For installing the Virtualbox extension pack try opening a terminal and use the following command...

```
sudo VBoxManage extpack install /Data/Linux\ Packages/Oracle_VM_VirtualBox_Extension_Pack-4.2.16-86992.vbox-extpack 
```

Substitute the path above to where you have saved the package on your machine.
