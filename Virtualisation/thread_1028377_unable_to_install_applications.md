### Post by ubersolid on 2009-01-02
If you have installed Vista as a guest inside virtualbox, all drivers you need to install are the "VirtualBox Guest Additions".
You will find the option to install them in the virtualbox menu where Vista is running.
*Devices -> Install Guest Additions* or something close to that.
