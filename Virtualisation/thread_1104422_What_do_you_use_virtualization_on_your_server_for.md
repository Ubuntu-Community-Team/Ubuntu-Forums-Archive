### Post by CRIMPS on 2009-03-24
I work for a software company.  We use Virtualization for testing purposes.  Our Sales Department also uses VM's for setting up Demo's for prospects.  This is very useful for us in that we don't have to go through a lot of setup before a presentation is given.

I personally use a virtual machine as a second processing machine in the event I spread my personal workstation too thin.  Again, this is very beneficial.

One of the main issues we had to work through was a Hardware issue.  10 virtual machines running can really eat up a lot of memory.  Luckily, memory is cheap ;)

Other than that, things have been relatively smooth.  

In our office we use Windows Virtual Server 2008 for our windows machines and we use Virtual Box for some of our linux machines (usually run on local machines.)  I prefer Virtual Box ;)
