### Post by vickoxy on 2010-04-11
Ok, i will mark this thread as solved, because-for me strange, but i have this working well now. 

I made few days ago one .iso image from that corrupt cd. Today i copied that to my ubuntu machine-and realized that i should probably mount this .iso image to computer (dell mini 10v). So i installed **gmountiso** - and mounted it to cdrom. So, this is windows xp program, but guess what-i have wine installed, and opened .exe file with it, and - voila - it is working!!! :popcorn:

No need to break copyrights or something like that. Actually, no need for windows too... :lolflag:
