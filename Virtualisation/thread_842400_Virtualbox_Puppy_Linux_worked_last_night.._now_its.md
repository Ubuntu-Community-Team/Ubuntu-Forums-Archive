### Post by Rocket2DMn on 2008-06-27
You use the right CTRL button on the keyboard to get the mouse pointer out of the vbox window.  It may just be easier to start a new virtual machine with a new .vdi file rather than trying to troubleshoot this.  Once you have it installed, you can take a Snapshot of the VM so you can revert to it later if something like this occurs again.
