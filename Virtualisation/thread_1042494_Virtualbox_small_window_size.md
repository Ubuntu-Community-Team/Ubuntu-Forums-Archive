### Post by HotShotDJ on 2009-01-17
> **Sealbhach said:**
> Hi, looking at this is like looking through a letterbox (see screenie). I tried fullscreen mode but it just adds black background. How do I make it bigger?You need to install the guest additions.  Start your guest OS. After it is booted up and ready, go to the VirtualBox Devices menu and select Install Guest Additions.  After you've done this and restarted your guest OS, you should be able to resize the Guest Display to any size you wish, including Full Screen (sometimes you might need to press Right-Ctrl+G while the mouse/keyboard is active for the guest OS to get it to resize properly in Full Screen mode)
