### Post by maxxjr on 2009-10-05
I am running 64-bit XUbuntu with the latest release of VMWare Workstation.

I have a Sony internal memory card reader, that connects to a motherboard USB header.  Before VMWare workstation, my method of getting images from my Canon camera was to plug the CF card into the card reader.  The CF card would automatically be mounted on the Xubuntu desktop, and I would copy images to my home directory.

One of the reasons I am using VMWare Workstation is to be able to use the Canon software to download and manage images (on a Windows XP guest).  

First, if I am in the XP guest, and plug in the CF card--nothing.  If I go back to the Xubuntu desktop, there is nothing there either.

If I am in the XP guest, and plug in the camera to a USB port (as an alternative method of downloading), Windows detects something and sets up a digital still camera, but the dialog box it pops up when done is empty.  Trying the Canon Utils to detect the camera doesn't come up with anything.  

If I turn off the VM, and go back to Linux, and plug in the CF card, the CF card is no longer automatically mounted.  I checked the windows manager settings, and the option for automatically mounting removable media is still selected.  So it seems I have taken a step backwards!

Any suggestions on how to get moving in the right direction again?  I at least want to recover my ability to mount the CF card with XUbuntu.  Better is to be able to connect to my camera via USB in the XP Guest.  Best is being able to work with the CF card directly from the XP Guest.

Thanks!
