### Post by nevdelap on 2008-07-22
To get VirtualBox to pass...

ctrl+tab - switch applications in the guest instead of the host.
ctrl+esc - open the start menu in windows.

...press **both of the ctrl keys**. These combinations (and probably others) will then be passed to the guest. Press both of the ctrl keys again and these combinations will be passed to the host again.
