### Post by besial on 2013-01-27
So I got iATKOS ML to work in VirtualBox by using the following additional configuration items in the Customize section (in addition to the default items)
```

USB fix
Original Boot Logo
Graphics Mode >> 1280x800
Sleep Enabler
PS/2
VGA >> Intel

```
The screensaver seems to make the VM freeze, but pressing 'enter' fixes it.
