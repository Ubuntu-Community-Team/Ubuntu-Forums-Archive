### Post by the_crowbar on 2011-11-08
Hi all,

I am running VMware vShpere 5. I have an existing 10.04 LTS amd64 install that I am trying to move to a virtual install. VMware has an application that does most of the work. I provide a root login to the physical machine and it creates a copy in a virtual machine.

The copy process appears to work correctly. The problem occurs when I try to boot the new VM. The boot hangs with no error status. I can use a boot cd to mount the virtual hard drives and they appear to be fine.

By altering the boot options I see that the boot hangs during mountall. I think this is where my problem is coming from but I am unsure of how to fix it.

The physical machine had /var and /home on LVM. The conversion created the LVM partition and I can mount both the volumes from a boot cd.

It seems that the mountall command never completes, but it may just be that it is the last output before the boot hangs.

Any ideas appreciated.

Thanks!
