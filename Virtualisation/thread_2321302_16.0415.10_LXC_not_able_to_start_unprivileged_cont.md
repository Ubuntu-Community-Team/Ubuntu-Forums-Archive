### Post by MAFoElffen on 2016-04-26
In red is your conflict between the two:
```

# The primary network interface
[COLOR=#ff0000]auto enp3s0[/COLOR]
[COLOR=#ff0000]iface enp3s0 inet dhcp[/COLOR]

# bridge interface for LXC VM's
auto br0
iface br0 inet dhcp
[COLOR=#ff0000]bridge_ports enp3s0[/COLOR]
bridge_fd 9
bridge_hello 2
bridge_maxage 12
bridge_stp off

```
A single card cannot share that type of Host connection and bridge at the same time. From our output, it is using it as a host connection, so no working bridge connection.

Do you have more than one Network Interface Card (NIC) on the Host? if so, then also plug a cable into it and:
```

# The primary network interface
auto enp3s0
iface enp3s0 inet dhcp

# The secondary network interface used as a bridge
auto enp3s1
iface enp3s1 inet manual

# bridge interface for LXC VM's
auto br0
iface br0 inet dhcp
bridge_ports enp3s1
bridge_fd 9
bridge_hello 2
bridge_maxage 12
bridge_stp off

```
If you only have one Network card, then then are a few other options... but more work, and more "technical." ...And that would depend on what your existing network card is and it's capabilities.
