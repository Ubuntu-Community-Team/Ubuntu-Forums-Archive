### Post by MAFoElffen on 2016-06-25
With VMware products, just Oracle VirtualBox is with their Extension Pack... The VMware Tools are not installed externally form the program... but rather installed from within their program product, while it is running.

If you have their product running, let's say Workstation/Player... from running a VM guest, look in the console menus for VM > Actions > Install VMware Tools.
