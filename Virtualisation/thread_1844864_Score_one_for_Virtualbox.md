### Post by UKBB on 2011-09-15
I downloaded the Windows 8 Developer release and tried to install it at work using VMWare Player with no luck. I did some research and found that it isn't compatible yet so I downloaded the trail version of VMWare Workstation 8. That worked but it took quite a while to install it and get it up and running. 

I just read this article

[http://www.pcworld.com/article/240117/how_to_download_and_install_windows_8_into_a_virtual_machine.html#tk.hp_fv]("http://www.pcworld.com/article/240117/how_to_download_and_install_windows_8_into_a_virtual_machine.html#tk.hp_fv")

So I opened up virtualbox on my Ubuntu home pc, created a vm for Windows 8 and in about 10 minutes flat I'm up and running.
