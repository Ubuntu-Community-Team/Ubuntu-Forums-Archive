### Post by spur on 2008-06-17
So have you installed virtual box? So far you have said how you installed Ubuntu and that you want to be able to use windows still.
To be able to use windows you will need to dual boot or install a virtual Windows by some means.
Not having a cd rom will be a problem doing either.
It is usually easier to install Windows first (not necessary, but easier) if you want to dual boot.
Otherwise install VirtualBox on your computer then install Windows into a virtual machine created by you in Virtual box. That is start Virtual Box then install Windows. There are a few guidelines on how to do that available and as your situation is different to me i would suggest you read up on the how tos before you start installing Windows.
