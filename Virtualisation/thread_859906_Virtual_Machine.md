Which is better is a matter of personal preferences. They both do pretty much the same things, and as far as I have seen, there is no difference in how well they perform.

With only 256 Mb ram, i think you are on the lower end of being able to run ONE os on its own. To be able to run a virtual machine, you need to meet at least the minimum requirement of the host os PLUS the minimum requirement of the guest os.
So to run Windows XP (minimum 256 Mb ram?) as guest with an ubuntu host (also 256 Mb?) you would need to have at least 512 Mb ram and share it equally between the guest and the host.
But to run it smoothly you probably need more than that (I have 1.5 Gb of ram on my old desktop with an athlon 1.86 GHz cpu, and it runs windows XP as guest in virtualbox quite well if I allow it to use 768 Mb ram. but it does slow down my ubuntu host os sometimes due to more swapping [= using the hard drive as virtual memory when the ram is full]).
