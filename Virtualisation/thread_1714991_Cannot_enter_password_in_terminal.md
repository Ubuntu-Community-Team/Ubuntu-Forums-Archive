I have the latest version of Ubuntu running in VirtualBox. Everything seems to run fine, except when I try to do anythin that requires my password.

For example:

> 
eden@EdenC: /media/VBOXADDITION_4.0.4_770112 $ sudo sh .?VBOXLINUXADDITION .run
[sudo] password for eden:
Whenever I read the "[sudo] password for eden:" I cannot type anythin into the terminal. The only command it recognizes is ENTER.

How can I fix this?

EdenC

### Post by amauk on 2011-03-26
You won't get any visual feedback when typing in your password (security feature, so no-one can see how many characters it is)

Just type your password and press enter
