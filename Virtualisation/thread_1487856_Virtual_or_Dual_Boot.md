### Post by TheFu on 2010-05-20
> **waltm said:**
> If XP is running in a virtual machine will it use windows video drivers and nvidia's 3d player?  Is it even possible to install winXP after linux?  

Sure you can install WinXP after installing Linux. The ways you can install it are many. For your situation, I'd have to recommend a dual boot since your MS-Windows appears to need direct access to the video card.

Before you try to install WinXP, be certain to find a HowTo guide so you don't accidentally trash your Linux boot loader or worse.
