### Post by JohnMac on 2008-05-08
I had 7.10 running OK in VMware on a XP host. I downloaded 8.04 (using BitTorrent) and used my machine shop file to create the VM files , as I did with 7.10. All the files were created but the Virtual File will not boot.

Is it probable that my downloaded file is corrupted, how can I check?

Ok, I worked it out. I confirmed my iso file was not currupted using MD5SUM then mounted the iso file using ALCOHOL120. I was then able to run the VM machine using VMWARE - success.
