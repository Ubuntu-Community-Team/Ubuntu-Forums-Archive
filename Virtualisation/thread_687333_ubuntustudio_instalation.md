### Post by stooshbunutu on 2008-02-04
If you have a big enough thumb drive you could store the file on that and then set you boot options to boot from the USB drive ( Z: drive or other such like) and that should work. The boot menu can usually be accessed via pressing the F2 key when the computer initially boots up (it will prompt similaar to "Press F2 for SetUp").

Hope this helps:)
