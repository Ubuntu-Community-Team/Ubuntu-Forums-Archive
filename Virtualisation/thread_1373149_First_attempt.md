### Post by snowpine on 2010-01-05
Hi there, sounds like you are still a little unclear on the concept of virtualization... I highly recommend the "Virtualization Megathread" sticky at the top of this forum. :)

VirtualBox is an application. You can install VirtualBox in Windows or in Ubuntu (or both). You can run a Windows "guest" in VirtualBox on an Ubuntu host, or an Ubuntu "guest" on a Windows host, or a Windows "guest" on a Windows host, etc. Really, you are only restricted by your imagination (and the number of Windows licenses you have).

I don't think it's easy to "virtualize" an existing Windows install. I used my Windows install disk (plus one license) to perform a fresh install within the virtual machine, it was really easy that way.
