### Post by antikristian on 2009-04-13
Technically you can't use those. First and foremost, the recovery discs are dependant on the hardware you physically have, virtualbox emulates a lot of hardware so they're not compatible even though you installed it on the same computer.

There are also licensing issues, an OEM image is only meant to be used on one computer, and when the computer dies, the license dies with it.

If it's Vista Basic, Premium or Ultimate, you are not allowed to use it on a virtual machine even if you purchased a full version, only Vista Business is allowed to be used on virtual hardware. (might be wrong about ultimate)

Of course, if a friend of you have an original Vista disc (these are just spare parts, all versions are on one disc and no license issues from borrowing a disc if you own the license) and you use the license key from your own computer, it would work just fine. But then there's the licensing issue if it's OEM or not Vista Business. 

Most likely you would violate their EULA somehow, except if you have a full version of Vista Business.
