### Post by The Flying Penguin on 2010-05-03
I have 10.04 64bit and installed the 64bit version VirtualBox last night to run XP Pro. XP was running just fine and I was able to run XP at 1360x768 (my native res is 1366x768) using the 3D driver included with the guest additions. I was even able to play an older game called Pharaoh with no problems. 
 
After confirming these things worked I began installing all the MS updates and SP3. Now all of a sudden I can only run up to 1024x768 in XP and when I launch my game the sound starts stuttering and I can't control anything in the game. 
 
Could running the SP3 be causing some compatibility issues? What could be giving me theses problems? 
 
Like I said, things were working great before! If I have to I will just reinstall XP and see if that solves the problem but I'd like to avoid that if possible.  Any advice would be greatly appreciated.

Btw, this is on my Asus ul30vt. 1.3GHz Core 2 Duo SU7300, 4GB DDR3 (2GB allocated to run XP in VirtualBox), and the Intel 4500 graphics.

### Post by The Flying Penguin on 2010-05-09
Well, after some searching around I discovered that some people seem to have problems after doing the SP3 update. Installing XP with SP3 slipstreamed into it seems to work fine though. So I reinstalled XP with SP3 built into it and now everything works fine.
