### Post by yeswetran on 2013-05-19
This problem has plagued me for milennia (also known as 2 weeks). The prob is that VirtualBox and every other virtualization software doesn't come with the actual OS. You'll have to go to fedoraproject.org and download the live CD for yourself, and from there boot it to VirtualBox and install the whole thing. I've collected iso downloads like Fedora and Ubuntu and booted them on VirtualBox and now I have a whole library of Linux OSs.
