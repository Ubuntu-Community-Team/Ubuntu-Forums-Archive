### Post by MAFoElffen on 2016-04-30
What virtulaiztion softare (hypervisor) is your VM running on?

How much memory and what it the Video settings set to in your VM?

I had this problem (about a year ago) with plasma in virtual. Was how much memory I had allotted to the virtual machine... Once I set it to higher... then I saw that the video settings needed set to be enough capbilities to allow extended video effects. Even though the machine I was using had an NVidia GTX 980ti, the virtual machine profile was set to something that wouldn't, until changed.

Answer the 2 questions above... and that should be enough to be able to get you going with an informed answer.
