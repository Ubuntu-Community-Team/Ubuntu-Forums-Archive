### Post by MAFoElffen on 2022-01-28
You shouldn't need two serial ports. Possiile the if there was serial (#1) that is wasn't bound as a serial console. (ttySO)
```

console=tty1 console=ttyS0,115200 2

```
Yes, the second part of that was definitely set for serial console, so it hung trying to find it. 

But now that it is up and you can see what it is doing. You can remove the second part of that and reconfigure it for VGA and Spice.  Or leave that serial port alone and delete the other.
