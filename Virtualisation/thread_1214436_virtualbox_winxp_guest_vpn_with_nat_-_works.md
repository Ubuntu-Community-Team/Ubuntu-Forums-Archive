### Post by arnoldjames on 2009-07-15
I set up virtualbox in ubuntu 8.04 (HH). Virtualbox does not allow vpn pptp.  On a chance I set up the vpn in the ubuntu host and it works in the windows guest as if the guest had the vpn running, allowing the guest to access the remote host.  This may help someone else.  There are instructions in various places about how to get a vpn pptp to work from VB, but none worked for me, and this is such a simple solution.
