### Post by abhiroopb on 2008-05-30
I tried both and quite conclusively found that Virtualbox was MUCH faster. I have used adobe indesign (DTP program) which needs a minimum of 512mb ram (so I gave my virtual machine about 600mb). Even with this set up I could use the virtual machine AND do everything else on my normal desktop. Just make sure you go to the virtual machine website and download the full version NOT the OSE version (which disables usb support!).

### Post by bradmkjr on 2008-05-31
Maybe I'm picky or spoiled, but I wouldn't do video editing for anything longer then Youtube videos with Less then 2 gig dedicated. If you have an older computer (less then an Intel Core Duo) then I would save the money towards a new computer down the road. If you have atleast a Core 2 Duo (or AMD equivalient) then I would add the max ram for the mother board. Remember Virtualization is going to act like cutting your system in 1/2 for some things, so even if you are running windows now easily, having linux and windows running will slow it down.

Always add more ram if you can afford it.
