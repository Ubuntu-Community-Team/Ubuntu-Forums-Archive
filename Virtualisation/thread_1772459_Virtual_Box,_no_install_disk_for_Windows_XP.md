### Post by aphatak on 2011-05-31
When you create a virtual machine, what you get is a an empty virtual computer with *nothing* installed in it:  it's as if the entire hard disk has been freshly formatted.  You have to install an operating system such as Windows or Linux.  You *must* have an XP install disk (or an iso image) if you want to (install and  then) use Windows XP.
