### Post by dabl on 2008-11-07
VMware Player 2.5 is free -- you just have to trade them some user registration info.  It runs great on my Kubuntu 8.10 system.

I benchmarked an earlier version of VMware Player, running a Win XP VM, and running a MS Visual FoxPro application in that Win XP session, compared to the same database on a native Win XP installation on the same hardware platform.  The VM was 95% as fast as the native Win XP.

But I never tried it the other way 'round.   :)
