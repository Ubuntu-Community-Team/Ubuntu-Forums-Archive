### Post by Perryg on 2010-09-04
Did you by change select the CD/DVD to be used in the storage section of the guest?  By default it has nothing selected and if you make any changes to the default configuration you are not presented with the first run wizard.

### Post by zeroseven0183 on 2010-09-04
Hi speedracer2010. What exactly is the error message you see in Virtualbox?
Be sure you set the CD-ROM as the first boot option. You can do this under Settings > System > Boot order in Virtualbox.
