### Post by Buntu Bunny on 2019-04-29
Whew, I figured it out. From the Guest Additions file list I had to select the correct executable file for my computer architecture. Well, Windows doesn't include file extensions and Ubuntu has me spoiled with it's ease of installing things, but once I figured that out the installation wizard popped up. So far so good (until my next question).
