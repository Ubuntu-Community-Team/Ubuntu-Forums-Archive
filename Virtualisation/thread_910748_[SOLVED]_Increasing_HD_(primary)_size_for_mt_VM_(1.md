### Post by dark_phantom on 2008-09-05
If you created a dynamic disk (expanding) then yes it will be able to grow, if not then you can use a disk cloning or disk migrating program like acronis true image.  If the latter is the case, just create an expanding disk.  Install true image, add the second disk and run the application to move from one disk to the other.  Should be pretty straight forward.
