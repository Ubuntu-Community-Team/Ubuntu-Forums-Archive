### Post by HermanAB on 2011-04-24
Not sure what you want.

You can have multiple IP addresses on one machine.
You can make multiple user accounts on one system.
You can use a virtualizer to run multiple virtual machines on a single physical machine and then each can have separate IP addresses with the same user accounts and logins.

Maybe one of those options is what you are looking for.
