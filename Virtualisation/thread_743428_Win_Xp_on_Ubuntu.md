### Post by Calash on 2008-04-02
Any OS that you install virtually is still it's own operating system, all it's flaws intact.  The security comes from two things

1 - It is Isolated from your main OS, so if it blows up you can still book the computer

2 - You can create "Snapshots" of known good versions, and revert to them if something goes wrong.


You can find more info on it in the Virtualization forum

[http://ubuntuforums.org/forumdisplay.php?f=308](http://ubuntuforums.org/forumdisplay.php?f=308)
