### Post by SimonJW on 2007-12-22
First, you must install the VirtualBox Guest Additions on your Windows guest (search the forum for a howto, but on the newest version at least it is pretty easy. Go to the menu, unmount the CD, click "Install Guest Additions", follow wizard)

Reboot Windows.

Right click on the Desktop, go Settings (or similar, the bottom one in the menu), and then Apperance. You should now be able to set the resolution to whatever your native Linux resolution is. Then you won't have a black border. On the other hand, the OS will be too big when running in a window, so you'll have to go full screen (Right Ctrl + F by default).

Or on the newer versions, try seamless (Right Ctrl + L)

Hope this helps!
