### Post by KatBuntu on 2008-08-05
VMWare workstation is free and part of its licence is under GPL. You can download it free from: [http://www.vmware.com]("http://www.vmware.com"), even VMware ESXi has a new free version. As i can see Win4Lin only runs Windows aplications, also Win & Lin, I've never tested parallels workstation. But VMware server can create virtual machines from another OS like BSD or Solaris, that's something Win4Lin and Win&Lin can't. VMware server offers a lot of tools for virtualizing, i build a virtual beowulf cluster with that, i recommend you VMware server.
