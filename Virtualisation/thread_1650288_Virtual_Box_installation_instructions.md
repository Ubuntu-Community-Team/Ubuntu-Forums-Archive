### Post by alonsoalex on 2010-12-21
Hey billy,

Installing Virtualbox is easy, my friend, just go to virtualbox.com, select Linux host, pick the first item in the list (since it's for Ubuntu 10.10) and download. Next, click the downloaded deb file and let Software Center do its job at installing it. After it's installed, open virtualbox (applications,system tools) and select New. Now, the Windows disk comes handy. Put the disk in the drive and wait until runs and installs. Remember, installing Windows takes a while. So, be ready to see the virtual machine reboot several times. Enjoy. 

Note: I typed these instructions using only my memory of past actions, but I could have missed anything.
