### Post by bradmkjr on 2008-05-24
Sorry, bad news, you can't.

Virtualization has one major draw back, all the computer hardware is virtualized. which means for it to work across all platforms, they use "The lowest common Denominator", so until recently it has been 10/100 nic, stereo sound card, ide or scsi drives and most importantly for you a cdrom drive, not a burner, not a dvd or not even a dvd burner.

There are 2 ways which may work for you to burn a cd "in windows", the first is use an external USB cd burner drive, those can be picked up cheaply at a local computer shop. The second option is create an ISO file, and move it to a shared folder then burn the iso in Ubuntu.
