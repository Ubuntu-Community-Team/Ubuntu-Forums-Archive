### Post by TheFu on 2016-07-12
Microsoft sells Windows with a few different licenses.  Most systems that came "preinstalled" have a license that requires them to run only on that hardware, so it validates THAT hardware is seen or refuses to work.  You may be able to find trial installation media from Microsoft for different versions of Windows, but your existing key will likely not work.  That just means you get to try Windows for 30 days (or maybe it is 60 days now?) before either it stops working or you take some other action.
The "preinstalled" licenses are cheap, cheap, cheap.  Have 1 retail license for the Windows machine here running inside a VM which were $120+.  The pre-installed Win7 won't move from the machine it was pre-installed onto.

Then again, I don't know all that much about Microsoft licensing anymore.  Have heard they sell licenses specific to virtual machines these days and know that with their data center products those licenses are either free or extremely low-cost after the "server license" is purchased.  I would ask this question about MSFT licenses on a MSFT forum, though some folks here might know something about it.
[http://www.howtogeek.com/197232/microsoft-is-misleading-consumers-with-windows-8.1-system-builder-licensing/](http://www.howtogeek.com/197232/microsoft-is-misleading-consumers-with-windows-8.1-system-builder-licensing/) found that.
