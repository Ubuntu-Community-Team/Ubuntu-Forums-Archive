### Post by ad_267 on 2008-11-05
If you just mark the "virtualbox-ose" package, it will work out that you also need the "virtualbox-ose-modules" and "virtualbox-ose-source" packages and install them too. 

You shouldn't need the other three packages, although it won't hurt to have them.

This website explains simply how to install new themes quite well I think: [http://linuxowns.wordpress.com/2008/06/16/guide-to-customizing-ubuntus-look-and-feel/](http://linuxowns.wordpress.com/2008/06/16/guide-to-customizing-ubuntus-look-and-feel/)

Start with this one maybe: [http://www.gnome-look.org/content/show.php/MurrineGlow?content=88406](http://www.gnome-look.org/content/show.php/MurrineGlow?content=88406). The Eikon2 theme linked there is also very good.

In Synaptic Package Manager search for community-themes (Only for Intrepid 8.10) and install that.  Then more themes will be available under System - Preferences - Appearance. The dust theme is popular.

### Post by ad_267 on 2008-11-05
The background is easy to change from the background tab in Appearance preferences. Have a look at gnome-look for some or [http://interfacelift.com/wallpaper/](http://interfacelift.com/wallpaper/).

It's easy to change the panels around. Just click on them and drag them to move them. You can right click and select "Add to panel" to add more stuff, or right click on stuff and select "Remove from panel" to get rid of it. You can make them bigger and stop them from expanding to take up the whole width. 

If you want a dock like in that picture (or like on a mac) you can install avant-window-navigator in synaptic.
