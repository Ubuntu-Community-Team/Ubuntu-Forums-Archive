hey. apparently this is a common problem and it is do to windows copyright CD protection. it seems it assumes your game is a copied version... even if you use a key. i dont believe a workaround has been created yet

edit: taken from [http://ubuntuforums.org/showthread.php?t=325520](http://ubuntuforums.org/showthread.php?t=325520)     "You need to use NoCD patches for these games.
