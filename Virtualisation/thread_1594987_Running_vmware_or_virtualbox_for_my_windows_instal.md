### Post by HermanAB on 2010-10-13
Yes you can use VMware or Virtualbox to run a hard disk version of Windows, however, you will run into Windows activation problems.  So it is not quite wort the hassle.

Note that MS Office works on Codeweavers Crossover on Linux.  Otherwise, depending on how crazy your documents are, OpenOffice may be good enough.
