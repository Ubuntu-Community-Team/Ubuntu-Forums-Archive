Jaunty & VmwareServer v1.0.9 working almost perfectly.  No USB Flash Sticks.
Guest's VM:RemovableDevices:USB remains empty.
I tried
```
 mount -t usbfs none /proc/bus/usb
```
but that (1) prevented Jaunty from seeing the FlashStick and and although the VM could enable the FlashStick, the guest OS (XP) only saw broken hardware.
Then tried adding some bits to /etc/init.d/mountdevsubfs.sh
```
	#
	# Magic from "Hardy" file to make /proc/bus/usb work
	#
	mkdir -p /dev/bus/usb/.usbfs
	domount usbfs "" /dev/bus/usb/.usbfs -obusmode=0700,devmode=0600,listmode=0644
	ln -s .usbfs/devices /dev/bus/usb/devices
	mount --rbind /dev/bus/usb /proc/bus/usb
```
but that seemed to have no effect (other that spewing a bunch on mount-usage verbage if I happend to start it from the command-line.
