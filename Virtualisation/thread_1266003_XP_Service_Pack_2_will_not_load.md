### Post by Isarii on 2009-09-16
On the window that VirtualBox is running in, on the bar across the top is a menu labeled devices.  From there you can choose to mount CD's/USB devices etc... so that the virtualized machine would be able to access it.  To be fair, I don't have a CD to test this with (weird I know), and I've never done it before so I'm not 100% sure if it works.  I did manage to get the drive to eject through the virtualized XP though, so I'm assuming it worked.  

If that doesn't work for some reason (assuming you haven't already tried it), I would just download SP2 from the Microsoft site through the VM and hope the mounting works for your iPod.  

Good luck!
