### Post by lykwydchykyn on 2012-01-31
Virtualbox has "guest additions" -- basically a set of drivers and applications for the guest OS that make it smoother to run a console view in a window, rather than resort to something like VNC or RDP.

IME it's biggest advantage is just the conveniences aimed at desktop-on-desktop virtualization (as opposed to cloud computing or server virtualization).
