I just use Virtualbox with Win2K. Once the VM is running, I mash Right<crtl> + L and it switches to a different seamless mode. The Windows desktop disappears and the taskbar is left covering the bottom of the screen (in my case, the bottom Gnome taskbar on desktop 2). The way I use doesn't theme the Windows windows to my Gnome theme, but it doesn't matter much to me since it's not on my main desktop (it's on the 2nd desktop) and I don't use it much anyway.
