### Post by Dedoimedo on 2008-12-14
Sure. You can connect normally in your host like you always do.
Then, in settings for the virtual machine, allow NAT. This will allow you to simultaneously have internet connection in the guest, as well.

No special configurations are necessary. VB uses NAT by default. So install the guest, run it and off you go.
