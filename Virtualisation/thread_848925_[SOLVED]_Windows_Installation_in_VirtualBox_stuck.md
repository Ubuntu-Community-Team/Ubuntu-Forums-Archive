I then turned off virtual box, and i changed the settings on that VirtualOS to the absolute minimum on hardware.  I shut off network, turned off sound, turned of com ports, and then started it up again.  It finished loading.  I shut down the VirtualBox turned all the hardware emulation back on and it booted up like a champ.
