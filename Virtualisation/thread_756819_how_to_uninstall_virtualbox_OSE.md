### Post by erginemr on 2008-04-16
There should be a setting in the OSE tar package to uninstall it. You should read the documentation (README or INSTALL) there.

There is also a binary Debian package with better features (e.g. USB support). You can download the package for Ubuntu from:
[http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads) -> Binaries
