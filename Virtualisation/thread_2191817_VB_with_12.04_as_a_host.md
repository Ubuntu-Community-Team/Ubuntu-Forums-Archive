### Post by CharlesA on 2013-12-04
You could clone it via something like Clonezilla, but if it is an OEM version of Windows and you are still going to have it installed on the machine, you would be breaching the EULA.

Also, Windows doesn't really like to play nice if moved between real hardware and virtualized hardware, but it "should" work.
