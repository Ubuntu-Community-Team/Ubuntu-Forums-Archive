### Post by ridshack on 2008-06-10
Hey all, I dont know if you already know this but I was kinda missing a little television as I sat working on my Ubuntu box when I decided to see if my WinXP VM on Virtual Box would work. Two shows later I am happy to report all though the sync was off 2% of the time I was able to watch with only the occasional minor annoyance that lasted a moment or two.

Im mostly surprised because VM graphics hasnt been that good and I never expected 2D graphics to be this fast. Dont expect to play any heavy hitting FPS but you can enjoy a flick. :popcorn:

Dell Latatude 630 
2 gigs of RAM (1 gig to the VM)
Intel 915 Graphics (8mb to the VM)
With Compiz-Fusion

Virtual Box 1.6.0
Ubuntu 8.04 all updates applied.
