### Post by tall-male on 2010-02-22
Ever played with Jopr (recently renamed to RHQ) from [http://www.rhq-project.org]("http://www.rhq-project.org")?

Being the community edition of Red Hat JBoss ON it gives a nice monitoring and management platform for both Linux and Windows monitoring JBoss, Apache, Linux etc.. Installation is simple and quite forward. Use the build-in Java H2 database for a quick try. Installation is described on [http://www.rhq-project.org/display/JOPR2/Installation]("http://www.rhq-project.org/display/JOPR2/Installation")

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=147885&stc=1&d=1266852266[/IMG]

The recent beta version (3.0.0.B02) gives a virtualization plugin. Using libvirt both kvm, xen and more can be managed: (note, all my virtual machines are down in this picture.)

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=147886&stc=1&d=1266852266[/IMG]

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=147887&stc=1&d=1266852266[/IMG]

Now that looks quite nice!A nice, still basic though, WebGUI is there to configure, start, stop and monitor KVM. This looks promissing!
