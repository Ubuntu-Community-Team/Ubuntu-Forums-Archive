### Post by Jake Sweeney on 2013-10-21
You can shut the computer down by crossing the Virtual machine's window. When closing the windows, you'll be prompted to either save the machine state, shut down the machine or to power it off. As for the aesthetics of the banner, you could try and install guest additions, which allows a smoother virtualisation experience without the bugs or glitches
