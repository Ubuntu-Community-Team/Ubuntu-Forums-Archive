### Post by Yoooder on 2007-11-16
does the VM have input controls?  Before it will accept any keypresses or mouseclicks you will have to click inside the window.

At this point whatever you do on your keyboard or mouse should go through to yor VM--including F[1-12] keys.  Also, if your keyboard is like mine you might have to enable the Function keys by pressing a Function Lock key (name will vary) which is similar to a Num Lock
