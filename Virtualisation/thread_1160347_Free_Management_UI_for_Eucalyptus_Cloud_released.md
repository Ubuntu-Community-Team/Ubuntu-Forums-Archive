### Post by prabhakar_chaganti on 2009-05-15
We have just released **free browser, iPhone and Android based management** for Eucalyptus Cloud Services. It has been validated against the EPC and supports 1.4.x version. We are also adding the 1.5 features like volumes, snapshots, access logging, etc and will release that soon. We have a screencast that shows the feature set, and here is a link to the blog post on this release:

[http://bit.ly/eucalyptus_support](http://bit.ly/eucalyptus_support)

This is a first release, and we would love to hear your feedback and feature requests. There are a few things that are a work in progress:

    * File uploads are not working completely.
    * No bundling support.
    * No mobile support for Walrus yet.


Stay tuned. Lots more coming soon!
