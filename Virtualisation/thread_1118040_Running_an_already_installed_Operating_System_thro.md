### Post by Stochastic on 2009-04-06
I'm brand new to virtualization and I was wondering if it is at all possible to run an already installed operating system via virtualbox?

### Post by itisbasi on 2009-04-06
No you can't run an already installed OS on virtualbox. You will have to make a fresh installation of the desired OS on virtualbox.
