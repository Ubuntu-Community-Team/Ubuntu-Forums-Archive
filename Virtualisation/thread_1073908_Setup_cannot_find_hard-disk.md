### Post by whoop on 2009-02-18
Let me get this straight. You want to install xp as a guest os inside vmware?

If yes. Then the answer is simple, there is no hard-disk. You need to create a new "Virtual" hard disk. This is just a file on your "real" hard-disk. Your vmware application should be able to create it for you.
