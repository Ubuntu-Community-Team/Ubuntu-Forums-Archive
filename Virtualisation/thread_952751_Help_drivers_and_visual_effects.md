3D in virtual enviroments is very advanced and currently only VMware Fusion for the mac supports full 3D acceleration, but it's D3D though (means only windows will have 3D)

OpenGL (which compiz fusion uses) is not supported on any virtualization apps - VMware Fusion gives you blue screen of death if you set a game to run OpenGL :D
