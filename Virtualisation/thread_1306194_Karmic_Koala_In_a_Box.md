### Post by Odin Overland on 2009-10-30
Just thought I would post about my first experience with Ubuntu yesterday.  I had been waiting for the 9.10 release, and spent the majority of yesterday downloading and burning ISOs.  I have a MacBook that is about three years old, and I am running VirtualBox 3.0.8 r53138.  It does okay with XP, which I never use unless I am still dealing with a company or site that ONLY uses Windows and even IE still.  I first put on Ubuntu 9.10 32 bit running Base Memory at 512MB and Video Memory at 64MB.  It was running with a bit of a lag, but I had been abusing my computer with all of the excessive downloads and burning.  I decided to try running Xubuntu 9.10 32 bit with the same settings, and it has been running beautifully!  My computer was running a lot easier on this OS in VBVM.  It used far fewer resources and my computer was running much cooler, and my fan never started wailing like it does when I run other guest OS on VB.

I think I'm going to make Xubuntu 9.10 my Linux for MacBook (unless I decide to dual boot... not sure what I would want to use for that).  I think I am going to try and put Edubuntu 9.10 on my iMac as a third OS this weekend, but I know it will be much more involved than the VM installation.  But so far, no issues at all with Xubuntu 9.10 on VB.:D
