### Post by stinger30au on 2008-08-27
well for the life of me i could not add a shared directory to XP in virtual box using the command it said i was supposed to use.

so i went hunting and found this

click start ->my network places-> add network place

click next

click "choose another location"

click next and then click browse

click entire network

and you should see "virtual box shared folders"

now click on the shared directory and answer the simple questions...

man, now i understand why configurations via shell are sssoooo much esier then gui

hope someone finds this usefull
