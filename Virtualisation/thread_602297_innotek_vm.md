### Post by Ian505 on 2007-11-06
Here is how to mount a optical drive or image file.

1. Open VirtualBox
2. Select your virtual machine and click Settings.
3. In the left pane of the window that pops up, select CD/DVD-ROM.
4. Check the "Mount CD/DVD drive" checkbox and select which drive you want associated with the virtual machine. You can also select a .img file and use that as the optical drive, saving you from burning a CD.
