### Post by vigyani on 2009-03-09
Hi

I do not know how exactly you can get the desktop exactly like the photo but you can get MAC OS X like widgets by using one of these packages available in the repositories.

1. avant-window-navigator: "A MacOS X like panel for GNOME"

2. gDesklets "is an architecture for "desklets", which are tiny applets
sitting on your desktop in a symbiotic relationship of eye candy and
usefulness."

3. Screenlets "are small owner-drawn applications (written in Python) that can be described as "the virtual representation of things lying/standing around on your desk". Sticky notes, clocks, rulers, ... the possibilities are endless."

Just search for these using synaptic package manager and install. You would have to customize these to your requirements.
