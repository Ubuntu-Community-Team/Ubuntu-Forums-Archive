I use Ubuntu 10.04LTS as my workstation at work every day. I use Win7 in a virtual machine in VirtualBox to gain access to Outlook and Office. I also use XP in a VM to gain access to Quickbooks which is our finance package. QB, despite any claims by Microsoft, does NOT work under XP mode in Win7 (whether it's in a VM or not). So I would say that the 'XP mode' in Win7 isn't quite an exact 'VirtualBox' for XP.
