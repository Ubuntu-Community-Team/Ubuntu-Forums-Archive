### Post by Maximus_ARNP on 2011-03-05
**[COLOR="Green"][COLOR="Red"]Presentation[/COLOR][/COLOR]**
On Linux Mint 10 in virualbox, after a fresh install of Windows XP, I loaded Guest Additions, Avast Antivirus, Comodo Firewall and several other softwares BEFORE installing all Microsoft updates. So, at restart I was able to go in SAFE MODE by pressing F8. But, normal reboot would not work. Windows XP would crash few seconds in to start up.

**[COLOR="Red"]Problem[/COLOR]**
After extensive testing I found that Guest Windowos XP OS would crash at startup in absence of current Microsoft Updates.
[COLOR="Red"]
** Solution: **[/COLOR]
Installed fresh Windows XP.
Install softwares in following order: Guest Additions, Microsoft Updates including Service Pack 2, Java Run time, Avast Antivirus, Comodo firewall, FreeMind, and OpenOffice, Service Pack 3. 

Now, Guest Windows XP is running smooth as ever.

Hope this helps.
