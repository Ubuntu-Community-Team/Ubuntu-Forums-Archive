### Post by ajgreeny on 2009-08-11
Just uninstall the OSE version, download the full Sun version as a .deb, install it and you will not loose anything as far as I'm aware, certainly not your various virtual machines which are stored in your home folder and not affected even when you uninstall Virtualbox.

When you have installed the full version from Sun you may need to go to the settings for the particular machine you want to run and make sure everything id set as you want.

So, just go ahead, and don't make a big mountain which doesn't exist, out of a small problem.
