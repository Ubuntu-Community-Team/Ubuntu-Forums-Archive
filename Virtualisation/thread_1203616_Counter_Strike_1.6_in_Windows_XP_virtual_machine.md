### Post by peepingtom on 2009-07-06
I supposed you could attatch your USB mouse directly to the VM using the icons in the lower right of the VirtualBox window, but then you'd have to have a way to unattatch it (barring the workaround kludge below). Im sure you could use a script and someone in the VirtualBox forums has probably already done the legwork......but you can see if that works.

I little tip, when you dont have a mouse (or lose control of your mouse to the VM somehow) you can press shift+numlock to use the numpad as a virtual mouse (after pressing the host key to give VM control to your host)
