There are many uses for using vmware in a linux system. With vmware you can run an entire operating system inside your Ubuntu which you can then start up and access as if it was another computer. Most common uses for this would be if you want to run a windows operating system inside Linux or if you want to test other linux operating systems to see how you like them.  

Personally I'm using VirtualBox which is very similar to vmware, both are very good. Neither cost money, VirtualBox has an open source edition which is almost fully featured. The best way to find out is to try them out.
