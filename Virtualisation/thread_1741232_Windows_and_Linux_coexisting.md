### Post by Hedgehog1 on 2011-04-28
The best way of getting Linux using your PC at 'full capacity' and Windows using the PC at 'full capacity' is to dual boot.

The other option is to run one OS in a Virtual Machine inside the 'Main' OS ('Main' OS is called the 'Host' OS, the OS in the Virtual Machine is called the 'Guest' OS).

Right now, many PC games don't run too well in a Windows 'Virtual Machine', but it is fine for Office and Outlook:

[IMG]http://img546.imageshack.us/img546/3093/vboxinnatty.png[/IMG]

You can go the other way and use Windows as the Host OS, and run Ubuntu in the 'Virtual Machine' as the guest OS.

In my case, I both dual boot (for video editing I need Windows because of the level of editing complexity I require), but I usually run Ubuntu, and then keep a number of 'Virtual Machines' handy for testing and 'MS Office' work:

[IMG]http://img219.imageshack.us/img219/455/vbox01.png[/IMG]

I hope this helps answer your question.
