### Post by k7sw on 2008-03-10
I hope you've found the solution to your problem by now. I was searching for a solution to the exact same problem when i figured it out myself.
If you still have the vmwaretools cd mounted then "Cancel vmware tools installation" (the same button you pushed to start it)
After this you'll probably notice you have another cd mounted, eject this cd and click "Install vmware tools" again.

This worked for me and i signed up on this forum to reply for you. Good luck.
