[Releases](https://wiki.ubuntu.com/Releases)

See [EOL Upgrades](https://help.ubuntu.com/community/EOLUpgrades)

14.04 is the current LTS (Long Term Support) Release,Not sure if this is the same for Mint.

### Post by buzzingrobot on 2015-05-06
Yep. Mint 16 was based on Ubuntu  13.10 and that's no longer supported.  When an Ubuntu release reaches its end-of-life, its repos are closed.

Your friend should upgrade to, or just do a clean install of,  Mint Cinnamon 17.1, the current release.  It's very nice and it's based on Ubuntu 14.04, an LTS release.

(Being prompted for a password when running updates is SOP and no cause for concern.)
