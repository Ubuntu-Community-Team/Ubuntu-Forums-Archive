### Post by novo2809 on 2013-10-24
If it's me, i will get rid of windows, install any version of ubuntu or linux mint. Then run windows on virtualbox. Dual boot always come with problems with partitions and boot loader ...
For the best of clean installation, get a server version or mini cd version whatever it's call. Then pick a gui of you choice (i would go with xfce or lxde for light weight as you are running on a laptop). Follow this post [http://namhuy.net/1584/how-to-install-gui-on-ubuntu-and-linux-mint.html](http://namhuy.net/1584/how-to-install-gui-on-ubuntu-and-linux-mint.html)

After GUI, install driver if necessary, install virtualbox then load windows on virtualbox. better way to go than dual boot.
