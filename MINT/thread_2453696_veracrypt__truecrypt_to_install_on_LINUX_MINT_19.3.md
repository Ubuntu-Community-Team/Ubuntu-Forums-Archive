### Post by TheFu on 2020-11-15
Ubuntu people use dm-crypt and LUKS for encrypted file systems.
[https://www.youtube.com/watch?v=5rlZtasM-Pk](https://www.youtube.com/watch?v=5rlZtasM-Pk)
Full partitions, virtual disks, block devices, whatever you like, since on Unix systems, everything is a file.
There is no encrypt in place or going back short of wiping the "file" or file system or partition.  If you forget the way to access, there isn't any real hack and nobody else magically as the keys/passphrase.

Linux users don't use veracrypt. [https://help.ubuntu.com/community/Full_Disk_Encryption_Howto_2019](https://help.ubuntu.com/community/Full_Disk_Encryption_Howto_2019)
