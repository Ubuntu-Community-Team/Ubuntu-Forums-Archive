### Post by wagb278 on 2016-09-15
Jack,
I used Macrium for Windows XP, but no longer have anything but Linux installed on my computers. I suspect Macrium will not work on Linux, even in WINE.
 
For full backup of a Linux installation I use CloneZilla to an external hard drive. That free software backs up all files on the whole drive or disk partition(s) and works.  I have had to recover from a backup and it worked.
Using CloneZilla appears a little intimidating at first, but it works.  You download the CloneZilla ISO and burn that to a CD then boot from that CD and run that to create a backup or to recover a backup to the hard disk if ever needed. Search the Internet for instructional tutorial videos to see how to use it. CloneZilla works for any OS on your computer, it does not know or care what the OS or files are. You cannot inspect the backup files created by CloneZilla like Macrium allows. 

Hope this helps
