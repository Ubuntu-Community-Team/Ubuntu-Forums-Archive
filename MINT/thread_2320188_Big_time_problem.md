### Post by yancek on 2016-04-11
> Within the last week, when I try to install a program, the program MINTINSTALL (sp?) kicks in

Mintinstall is a package manager which you can use to install application in Mint from their repositories.  Several other method you can use to install software in Mint are shown in the video at the link below.

  [https://www.youtube.com/watch?v=1Ka9b2tHFrY](https://www.youtube.com/watch?v=1Ka9b2tHFrY)

> it asks me for the root password, and apparently only Merle Haggard's ghost  knows that.  

The only person who should know it is the person who installed and maybe others s/he trusts.  It is not possible to install Linux Mint without creating at least one user and that user has root/admin permissions by default and a password is necessary.  You can enable passwordless login if you want.  If you preface a command in a terminal that requires admin privileges, you will be prompted for a password.  The password being requested is the password of the primary user created during the install unless someone has changed that.  Nothing shows in the terminal when you enter a password.

Linux Mint has various desktop environments and you don't indicate which you have.  You might going to the link below and downloading the appropriate user  manual.

 [https://www.linuxmint.com/documentation.php](https://www.linuxmint.com/documentation.php)
