### Post by ajgreeny on 2014-11-08
Maybe winff, if it's available in the Mint repos.  It should be as it is in Ubuntu repos which Mint uses.

Or you can use the command line application **ffmpeg**, or even **avconv**, if Mint uses that now as Ubuntu does.

---

### Post by coldcritter64 on 2014-11-08
You can also use audacity, drag and drop an mp4 onto the interface and it will import the file (for any ffmpeg compatible mp4 file). Then from the file menu > Export, a window pops up where you can name and save the file as mp3 and set the options like bitrate etc.

Or nautilus-audio-convert should be in the repos. Gives a context menu entry for audio conversions in the nautilus menus.
