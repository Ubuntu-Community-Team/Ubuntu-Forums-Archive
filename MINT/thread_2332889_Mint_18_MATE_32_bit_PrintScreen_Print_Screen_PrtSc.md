### Post by Dennis N on 2016-08-06
Mint 17.3 MATE also captures only the entire screen if you use the Print-Screen button. But, if you want to capture the active window, or select an area, go through the Main Menu > Accessories > Take Screenshot. Then, the dialog has the other options. The same may be true in Mint 18 MATE.
