### Post by CelticWarrior on 2020-08-14
WiFi and wireless mice have nothing in common apart from the fact that both depend on radio (wireless) technologies.

Wireless or wired mice show up the same way at System Settings > Mouse and Touchpad. The OS makes no distinction whatsoever - the wireless mouse (or keyboard) USB receiver is seen by the OS like any other wired counterparts.

Also, is it the same issue as in your other thread - [https://ubuntuforums.org/showthread.php?t=2448488](https://ubuntuforums.org/showthread.php?t=2448488) -? If so then a) you're using Mint so you shouldn't be posting in the main forum sections; b) your reported issues makes little to no sense and this thread alone suggests you may not know what you're doing or not as much as you think you do and c) please try known good hardware (meaning: very likely you're using defective peripherals unless you already tried in a different computer, preferably with a different OS and confirmed they're working as expected).
