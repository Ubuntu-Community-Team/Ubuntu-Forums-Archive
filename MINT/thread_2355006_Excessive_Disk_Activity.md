### Post by zerubbabel on 2017-03-08
A few days ago I did a fresh install of Linux Mint 18, Mate Edition, and I like it very much. However, there is a strange thing going on during the first ten or fifteen minutes after a reboot and login. (The same thing was happening after a fresh install of Ubuntu Mate 16.10, which is why I switched to Mint.) There is intense disk activity when I am not consciously doing anything demanding it. The cursor freezes for up to 30 seconds, and just when I'm about to conclude that it's hung, it releases, but then will freeze again a few times before this phase passes. Then the system is snappy no matter what I'm doing, until the next time I reboot. Can anyone tell me what is going on? I'm running the 64-bit version on a Dell Latitude E6540 (Intel Core i7-4810MQ CPU @ 2.80GHz) with 16 GB RAM and a 480 GB solid state drive.
