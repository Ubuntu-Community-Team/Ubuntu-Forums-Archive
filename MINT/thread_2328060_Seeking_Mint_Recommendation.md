The current release (17.3) is based on Ubuntu 14.04 so any of them should be ok given that you are already running Kubuntu 14.04. Might be worth holding off for the next release which is currently in beta and based on Ubuntu 16.04

There may be one consideration about Mint. *buntu can often times be upgraded from for example 14.04 to 16.04 or from 15.10 to 16.04. The last I knew Mint did not yet have that capability, version upgrades required a re-install. There was talk of that changing, I don't know what the current status. Mint is sticking with Ubuntu LTS versions as a base ignoring non-LTS Ubuntu versions so this may not be a consideration for you.
