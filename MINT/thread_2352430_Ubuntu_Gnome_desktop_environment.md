It would be rather foolish not to use the Mint forums when you use the Mint distribution.

The answer is yes, you can run alternative desktop environments although you may get one "bleeding" into the other eg, all the applications of one desktop environment showing in the other. As to how well this particular combination would work try it and see or ask the folks who run Cinnamon, ie the good folks at the Mint forums.

My preference is always to install into separate partitions so each environment is as it is intended to be without pollution from another. YMMV.
