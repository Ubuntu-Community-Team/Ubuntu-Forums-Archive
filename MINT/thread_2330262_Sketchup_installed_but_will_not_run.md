### Post by DuckHook on 2016-07-10
Welcome to the forums, joe9999.

WINE is an extremely finicky little beast that occupies a rather odd place in the Linux-sphere because it tries to shoehorn apps that were never designed to run on Linux into... well... running on Linux. It's a miracle that it can be done at all, and more often than not, it cannot be done. Looking at the winedb (linked to [here]("https://www.winehq.org/search?q=sketchup")), success varies wildly depending on a complex combination of Distro, version, flavour and the Sketchup release you are using.

Since you are using Mint, you should use their forums for help on this matter. Though Mint is derived from Ubuntu, there are still some major differences especially with regard to drivers, so ours is not really the best help resource for you.
