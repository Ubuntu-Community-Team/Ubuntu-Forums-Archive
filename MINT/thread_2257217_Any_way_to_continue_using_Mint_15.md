### Post by buzzingrobot on 2014-12-18
Your real issue is that you need find a way to replace Mint 15 on that machine in Kansas while you are in the Philippines. The 13.04 release hasn't been supported for some time.  Threads about unsupported releases here typically end with a gentle reminder that no support can be provided for unsupported releases. (And we don't know if the malfunctioning bits are the unsupported Mint bits or the unsupported Ubuntu bits.)

Before anything is done to the machine, however you find a way to do that, the files and data on the machine should be backed up.  That doesn't need to be expensive or complicated.  Files can be manually copied to DVD's or USB sticks, if necessary, just as long as a copy of anything they want to be sure to retain is physically copied elsewhere before alterations are made to the machine.

Without the machine's specs, it's impossible to take a guess at the problem. If the machine was working satisfactorily with 13.04 at the point of that release's End of Life, the lack of updates should not change that. You just wouldn't see updates and, hence, what's working wouldn't change.

Since Christmas is near, the optimum solution, of course, is a new system. ;)
