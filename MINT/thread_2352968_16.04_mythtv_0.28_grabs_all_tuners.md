### Post by vidtek on 2017-02-17
I'm just wondering if anyone else has this issue with 16.04 Mint 18 KDE and 0.28 Myth tv?  It seems to grab all tuners and leaves nothing for other tv applications.

The only way I can get kaffeine to work is to stop the myth backend as sudo ```
killall mythbackend
``` and then ```
dmesg | grep frontend
```

 ```
dmesg | grep saa 
```for my tuners to work in Kaffeine.

I had a 16.04 experimental setup running with mythbuntu and mythtv 0.29, and was able to have both mythtv and kaffeine running simultaneously (not very practical, but it worked).

Setup:

Linux Mint 18.1 Serena KDE  
4.4.0-62-generic
MythTV Version : v0.28.1-3-gfc737ff
MythTV Branch : fixes/0.28
Network Protocol : 88
Library API : 0.28.20161120-1
QT Version : 5.5.1
