### Post by buzzingrobot on 2016-01-18
The GUI appears automatically when the machine is booted.  If you did not select automatic login during the installation, the first thing you see is the login screen.

If the GUI is not appearing, then something went wrong with the installation, or after that, or, possibly, there is an issue with the video card.
