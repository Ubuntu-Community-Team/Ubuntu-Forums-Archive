### Post by RallyDarkstrike on 2017-05-22
OK, this is now SOLVED.

For anybody else's reference, I had to do  it in a stupidly roundabout way. I went to the MATE Keyboard Shortcut  settings and created two new shortcut commands using the PrintScreen and  Alt-PrintScreen keyboard shortcuts pointing specifically at the  mate-screenshot command.

Oddly, this works perfectly rather than  using the default "Take Screenshot" command listed and built into the  Keyboard Shortcuts menu by default!

Cheers!
