### Post by QIII on 2022-10-10
> **jrbassett said:**
> I am using Linux Mint 21 but it is technically an Ubuntu 22 system.

Just a note here:  Not quite.  MINT is an entirely independent operating system that is *based* on Ubuntu.  It isn't "technically" or in any other manner an Ubuntu system.   That distinction is important.  If you said something like that on the MINT forums, you'd likely get some harsh blowback.   There may very well be things that are different enough that in specific cases a MINT-only solution to specific issues is required.

(By the way, there is no "Ubuntu 22" system.  22.04 and 22.10, yes.  22, no.) 

Fortunately, there are a lot of MINT users around here, making it likely that any such solutions will be offered quickly.
