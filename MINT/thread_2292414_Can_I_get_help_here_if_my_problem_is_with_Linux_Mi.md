### Post by yancek on 2015-08-27
You will need to post more detailed information.  To start with, windows has been around for 30 years so which version of windows are you using?  If you are using windows 8 or later, are you using UEFI to boot or MBR?  Probably the best way to get the most detailed information for someone to help you is to get boot repair from the link below while booted into Mint.  Make certain you select the option to Create Bootinfo summary and post the output and do not try to repair.

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
