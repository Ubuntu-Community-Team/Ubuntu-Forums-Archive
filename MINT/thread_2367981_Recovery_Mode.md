### Post by #&amp;thj^% on 2017-08-05
Hold shift during boot to get GRUB to show (it is hidden by default if you have but one operating system installed). In the GRUB menu you can choose the second option, which is called the recovery mode. In recovery mode you will get a menu to select what you want to do. One of the options is to drop to a root console. After providing the root password (by default the same password as you gave for the user created during installation), you will be on a virtual console logged in as root.
