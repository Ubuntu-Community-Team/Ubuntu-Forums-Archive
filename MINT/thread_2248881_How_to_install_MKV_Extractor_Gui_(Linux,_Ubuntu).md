### Post by rogab on 2014-10-17
I had issues trying to find out how to install the Graphic User Interface for MKV Extractor aka MKVextract in Linux Mint 17 so I decided to create this thread for future references and in hope of helping others. It should work on Ubuntu too.

Keep in mind that "MKV Extractor Qt" will only work if you have installed MKVToolNix [https://www.bunkus.org/videotools/mkvtoolnix/downloads.html](https://www.bunkus.org/videotools/mkvtoolnix/downloads.html).

MKV Extractor Qt is available thanks to Hizoka at this website: [http://forum.ubuntu-fr.org/viewtopic.php?id=1508741&p=1](http://forum.ubuntu-fr.org/viewtopic.php?id=1508741&p=1)

The page is in French but you only need to follow these steps to install it from the terminal:

```
[FONT=courier new][SIZE=2]sudo add-apt-repository ppa:hizo/mkv-extractor-gui
sudo apt-get update
sudo apt-get install mkv-extractor-qt[/SIZE][/FONT]
```

When installation is complete launch the program (from your OS menu), you'll see something like this:

[IMG]http://qt-apps.org/CONTENT/content-pre1/165332-1.png[/IMG]
As you can see the interface is available in English.
