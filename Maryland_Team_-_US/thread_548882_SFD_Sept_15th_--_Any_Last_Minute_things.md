### Post by Bernie_K on 2007-09-14
Useful Hardware:

    * power strips
    * extension cords
    * computer power cords
    * switches/hubs
    * Cat5e cables
    * blank CDs
    * alternate-installation CDs (tested)
    * a PXE server

I can bring one 5-port Gb switch, one 4-port 10Mb hub, one power strip, and three extra computer power cords (ICE-320-C13 to NEMA 5-15P). I can also bring a RJ crimp-tool if someone wants to bring Cat5e cable and connectors.

If the library is providing several tables, and Maryland Team members bring additional power strips and switches/hubs, then we can place a power strip and switch/hub on each table, and run cables from the switches/hubs to jacks in the room provided by the library - or (if there is no PXE server (i.e. no network-based installations stressing the LAN)) - daisy-chain the switches/hubs.

The above will help mitigate power/network challenges in the event the SFD turn-out is good.

Turning an Ubuntu box into a PXE server looks painless:

[https://help.ubuntu.com/community/PXEInstallServer?highlight=%28PXE%29](https://help.ubuntu.com/community/PXEInstallServer?highlight=%28PXE%29)

Sincerely,

- Bernie
