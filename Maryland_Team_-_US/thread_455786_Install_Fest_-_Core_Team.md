Install Fest - Core Planning Team

Hey all! John (phr0ze) and I would like to form a team specifically for planning the Install Fest. We already have quite a few ideas, and we're really excited to get the ball rolling!

The team will meet weekly in a cafe/library/house near Columbia. This team will only be planning the Install Fest, the other loco projects are still led by Chuck. We realize everyone is a volunteer, but it is important to assemble a team who can consistently attend these meetings. If you feel you can add to this team on a consistent basis, please join us. We'll be planning and implementing advertising, networking, organization, etc. You do not need to attend the planning meetings to help with the install fest.

If you can't make it to the meetings, we plan on using polls on the forums to gain ideas and thoughts from everyone. We will also be posting the meeting discussions in the forum, so come back often. Of course, you can and should post your input at any time in the forums.

The next regular Loco meeting will be June 21st, and we hope to have two planning meetings by then. At the Loco meeting we'd like to nail down a date for the install fest with everyone's participation.

We're trying to keep everyone informed and involved, so we look forward to all of your help. PM me now if you can help us on the planning team.

Thank You,
The Install Fest Planning Team
