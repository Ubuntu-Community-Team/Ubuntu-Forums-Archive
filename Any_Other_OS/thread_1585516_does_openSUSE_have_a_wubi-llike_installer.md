### Post by jjex22 on 2011-11-20
Afraid Novell don't supply a WUBI like installation option for open suse - I'd suggest downloading one of the live dvds - gnome or KDE (I personally recommend KDE, but that's personal choice - and suse default) and trying it out that way. If you like it, install it - the installer isn't as idiot proof as ubuntu, but it does by default shrink the windows partition, and create /boot, /, and /home partitions, and sets up grub for dual booting. If you want lxde/xfce you need to download the full dvd, but this isn't a live disk.
