### Post by Cavsfan on 2012-07-27
I have been using Debian Squeeze for a while and am finding there are not many users in the forums I have seen out there.
Mainly I end up googling most everything. It seems to be the less user friendly OS I have used yet.
I had to find a way to build **apt-add-repository** and **ppa-purge** as they are not available through any source.

Also getting updates is very hit or miss. I have to enter **sudo apt-get update** 6-7 times before it completes successfully.

It came with Iceweasel version 3 something. It took me a while to obtain Firefox version 12.
I tried several times to get the repository added for Firefox stable upgrades and was never able to do it.
I did manage to get 2 conkys going along with cairo dock, compiz, emerald window manager and CCSM installed after much trial and error.
Good luck.
