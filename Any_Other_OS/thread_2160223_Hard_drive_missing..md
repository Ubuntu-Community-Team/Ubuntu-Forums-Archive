### Post by deadflowr on 2013-07-06
Is there no listing for anything in the home folder's sidepane option "devices"?

I would second Elfy's advice and look in the BIOS.
accessing BIOS is different from machine to machine, so unfortunately help there is hard without machine model.

Unfortunately, hard drives CAN die.
The good news, in that event, is that there are recovery specialists who can access and save the data for you.
You'll probably want to search the webs yourself for that, if it is indeed the issue.
Big +1 for Backups, though.

Edit: So no drive in BIOS probably means(most likely) the drive is dead.

### Post by buzzingrobot on 2013-07-07
If the BIOS doesn't recognize the drive, chances are it has failed. In any case, neither Windows or Ubuntu will see it. 

Recovering data from failed drives is an established business. If you can't find someone locally, you'll find someone on the net.  

It's unlikely your step daughter did anything to damage the drive unless she managed to drop it.
