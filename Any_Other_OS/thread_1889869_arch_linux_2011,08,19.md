### Post by [::AP::] on 2011-12-03
I'm actually about to install Arch on my old laptop. 

But if you have never used Linux before, Arch isn't for you, unless you are really tech-smart (then you will learn how Linux works painfully fast).  Try Ubuntu, Linux Mint, etc first. 

----

If you are still going through with Arch, the entire installation is command line. 
follow the directions here:
[https://wiki.archlinux.org/index.php/Official_Installation_Guide](https://wiki.archlinux.org/index.php/Official_Installation_Guide)
[https://wiki.archlinux.org/index.php/Beginners'_Guide](https://wiki.archlinux.org/index.php/Beginners'_Guide)

typing /arch/setup will begin the installation. 

Doing an install on a virtual machine, for example in VirtualBox, also helps ease the first-time pressure. Good to try it out first in VirtualBox. 

By the way, did you purchase the Arch CD to support them?
