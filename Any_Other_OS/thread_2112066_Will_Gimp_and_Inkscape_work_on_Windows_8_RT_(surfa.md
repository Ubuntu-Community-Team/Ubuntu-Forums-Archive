### Post by Dr. C on 2013-02-06
No. The reason is that Windows 8 RT is a censored operating system where only those applications approved by Microsoft via their store can be installed. It is very much the same as IOS in this respect. It is very different from Windows 8 on Intel/AMD or prior versions of Microsoft Windows. 

There is no technical reason why these applications cannot be recompiled for Windows on ARM since they are both licensed under the GPL. The problem is legal. Furthermore in many jurisdictions because of lobbying by big copyright interests it may be against the law for the owner of a Windows RT device to break the DRM in order to install the software of their choosing, even with the full permission of the copyright holders of said software, on their device.

### Post by Yharooer on 2013-06-28
Yes, GIMP and Inkscape will run on the Surface RT with a few tweaks. First of all you will need to 'Jailbreak' your Surface RT.
[http://forum.xda-developers.com/showthread.php?t=2092158](http://forum.xda-developers.com/showthread.php?t=2092158)

I think GIMP is open-source so now it's 'jailbroken', if you can compile GIMP for ARM, then you can GIMP (and maybe Inkscape, never used it) on a Surface RT.

But if not, don't worry, there's another way (just a lot slower).

There is a x86 emulator for the Surface RT. Obviously this is very slow but it will work.
[http://forum.xda-developers.com/showthread.php?t=2095934](http://forum.xda-developers.com/showthread.php?t=2095934)
This is in its beta stages so don't expect too much yet, anyways, GIMP and Inkscape **can** run on Windows RT, and hey, who knows, maybe they'll be officially ported to Windows RT one day.

(Don't listen to People who say it can't. I find that a bit pessimistic. Hackers will be able to do many things over time, find out exploits, and that's what happened with the Windows RT Jailbreak. One day we might be running full-on games like Just Cause 2 or League of Legends emulated on Surface RTs, but for now, time will only tell. :D)

It seems almost disgusting that Microsoft didn't include this functionality into Windows RT. They appear to be limiting their users, like Apple. If they for some reason work with the makers of the x86 emulator and the jailbreak to integrate it into RT itself, I can see Windows RT not being such a big failure after all. I guess there may be some legal problems with this though but I think that they should.

I want a Surface RT to try this, it would be interesting to see the speed which Win86emu is capable of emulating x86 programs.
...and it would be cool to have Photoshop running on a tablet, especially with long battery life, but I think GIMP's good enough.
