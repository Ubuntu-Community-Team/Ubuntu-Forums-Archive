### Post by mips on 2011-12-08
> **JF382 said:**
> Can you find any of these drivers for this laptop for XP or do they not exist around the web? Thanks

Yes you can. You need to identify all the relevant chipsets used in the laptop (motherboard, audio, LAN, video, USB etc) and then go to the chipset manufacturers sites (Intel, AMD, VIA, Marvell, Realtek, SiS etc etc) and download the drivers for XP.

To identify the chipsets you can install PC Wizard 2010/1 etc and one of the options is to pick up all the hardware. Based on the list of hardware it generates you can now go hunting for the drivers. If you need assistance with that then post the output of the hardware list here.
