### Post by el_koraco on 2011-08-05
Do you want to make a live USB to install it on your hard drive, or do you want it permanently installed on the USB? 

If it's the former, you can use the "Startup disk creator" utility in Ubuntu, if you're running it, or Unetbootin. If it's the later, you gotta take your USB stick, format it to ext4, make a Live CD, stick the USB in the drive, boot the Live CD, and install it to the USB.
