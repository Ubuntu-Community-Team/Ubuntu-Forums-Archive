### Post by d-cosner on 2014-06-05
Yesterday I installed Sabayon Gnome on a spare hard drive on my desktop. I was after the latest greatest version of Gnome Shell and Sabayon was nice enough to provide it as a regular update. I had to fix a couple of minor issues by doing a little file editing but the Sabayon forum provided the answers and it was not really that big of a deal.

I figured Sabayon or Arch was the way to go if I wanted an up to date Gnome system and Sabayon seemed like the easiest way to accomplish this. Rigo seems like a pretty decent package manager and it has some very nice features. The default applications are good but there is some software provided that I will probably never use. I was surprised that LibreOffice was not installed by default but it was easy enough to install myself.

The fully updated system runs really nice and it is great having the latest version of Gnome Shell! I am really enjoying seeing how Gnome Shell is progressing, the developers are coming up with some cool ideas. Videos, (Totem) has a really slick interface now and I was surprised at just how much it could do. Gnome Weather is another favorite of mine, I just like the user interface and it's kind of handy.

I do not see very many current reviews on Sabayon and was wondering what other people thought about their latest release? Also, I realize there can be issues here and there with a rolling release and wondered what other peoples experiences were? The only other rolling release I have used is PCLinuxOS, it was always very stable but unfortunately they do not offer a Gnome Shell version at this time. 

Sabayon is supposed to be for beginners but in my opinion it is more suited for intermediate to advanced users. It is beginner friendly as far as a Gentoo based system goes though! What are your experiences with Sabayon?

### Post by malspa on 2014-06-05
I don't have the GNOME spin, but I have a fully up-to-date KDE installation that I've managed to keep up and running since last July, installed from the Sabayon_Linux_13.04_x86_KDE .iso. That was a fresh installation that I did because I was unable to boot into Sabayon after updating the system on July 29 2013. I've had a few other rough spots following some updates.

I have not used Rigo; I update from the command line with the equo commands. One thing that I don't particularly like about Sabayon is that updates take a lot longer to complete than in Debian-based distros, for example, or in Arch with pacman. 

Sabayon's been pretty good here, but my feeling is that, over time, Arch is the best rolling-release distro I've run (I've also had a few nice, long runs with PCLOS) (I also ran Semplice for awhile). Like you, I feel that Sabayon is better for someone with more Linux experience than it would be for a beginner. And you definitely want to read the wiki and read the info at the forums, check the stickies there.

You might want to take a look at this: [http://www.sabayon.org/article/sabayon-%E2%80%93-keeping-it-date-tips](http://www.sabayon.org/article/sabayon-%E2%80%93-keeping-it-date-tips)

The KDE spin came with Fluxbox; I'm not sure if they're including that anymore, and I'm wondering if the GNOME spin also came with something else to log into (???).

The Sabayon devs do a pretty good job, overall. There can be problems, though; my first run with Sabayon was Sabayon 9, installed back in June 2012. So I have not yet managed to make it two full years without doing a fresh installation. I'm thinking that if/when my current installations breaks, I'm gonna give up on Sabayon, but we'll see.

One thing about Sabayon... the word that always comes to mind here is "beautiful." One of the most aesthetically-pleasing distros I've seen.
