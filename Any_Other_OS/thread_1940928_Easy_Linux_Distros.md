easy distro's = almost anything based on ubuntu, also pclinuxos seems good although i never had much luck with it (didn't spend very much time either), fedora (maybe?), and many others.


questions YOU need to answer for yourself.

do i want a stable machine to work on? but not much to drive you to learn.

do i want to be driven to learn by minor problems and bug-fixing?


if you want to be a real "linux nerd" then run new software in testing. you will learn quick (google is your best friend).

i have been working on a linux distro based out of debian sid/unstable meant to be easy to install and use, but the debian unstable repository will carry NEW UNTESTED software, you get the good and the bad with this.
