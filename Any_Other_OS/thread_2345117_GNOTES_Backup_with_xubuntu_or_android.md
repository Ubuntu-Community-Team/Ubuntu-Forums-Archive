### Post by Irihapeti on 2016-12-01
It looks like the GNotes app on Android is different from the one with the same name on Ubuntu. Gnotes on Ubuntu consists of small text files only, while the Android app has other capabilities as well.

if the notes in your .zip file are plain text files, you might be able to view them by using Xarchiver (or other archive viewer) and then viewing the files. That's only a guess, though.
