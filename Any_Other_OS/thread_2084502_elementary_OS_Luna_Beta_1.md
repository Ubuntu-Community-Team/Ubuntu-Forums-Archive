### Post by Giant Speck on 2012-11-17
> **cuculeasa said:**
> I have a "big"  problem, I cant change the window border. I have used Ubuntu Tweak and Gnome-Tweak-Tool but no luck :(  Any idea, please?

1.  Install dconf-editor

```
sudo apt-get install dconf-tools
```

2.  Run dconf-editor

3.  Go to org > desktop > wm > preferences

4.  You can change the window border by changing the name of the theme next to "theme".  Make sure you type the name of the theme correctly.  Press ENTER after typing the theme name to apply the theme.

### Post by Giant Speck on 2012-11-17
> **shizz said:**
> I've been running the daily builds of this for a while and have loved it. One thing I'd like to change is the font to Ubuntu font. How would I go about doing this?

Again, dconf-editor:

org > gnome > desktop > interface > font-name

To change the monospace font:

org > gnome > desktop > interface > monospace-font-name

To change the window manager title font:

org > gnome > desktop > wm > preferences > titlebar-font

### Post by blaede22 on 2012-11-23
Hey guys, I'm Cassidy James, a UX designer and Council member from elementary. 

> **MadmanRB said:**
> Ehn I am mixed, again this seems like a unneeded fork of Gnome shell and while I hate gnome shell we got cinnamon to fill in some of the gaps, plus this seems to lack theme support right now.

Pantheon is *not* a fork of GNOME Shell. It's a shell consisting of WingPanel, a dock based on Plank, Slingshot, Gala and a few other components. WingPanel is built from the ground up to be a lightweight but classy panel that works exactly the way we want, the dock is super lightweight and is being developed with the Plank devs to make it work exactly the way we want, Slingshot is a lightweight app launcher that looks and functions the way we want (backed up with user testing and research), Gala is an incredibly smooth window/compositing manager built on LibMutter. Nothing there is a fork of anything else, it was all built to make the user experience exactly what we wanted it to be.

As far as lacking theme support, I'm curious as to what you mean. Nearly everything should be themable via the GTK theme, including the panel, app launcher, etc. The dock uses Plank's theming system, and different dock themes work great. One thing I may do if enough people are interested is write up a sort of theming guide.
