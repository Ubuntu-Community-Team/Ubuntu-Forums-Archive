Here are some things I learned
1. You need a Windows based computer and your phone connected via USB cable to do this. It isn't one of the methods that works directly off your phone
2. The "command prompt" in the directions is going to the Windows START button and typing CMD in the command line. It's the old DOS window on a Windows PC
3. In Section / Step 3 of these directions, after you download the "supersu install zip" file, move it to your SD card on your phone (in a location you will remember). I didn't do that and started issuing the commands before I realized the file needed to be on the phone. A bit of a hiccup but I recovered. It'll be easier if you have it copied first.
