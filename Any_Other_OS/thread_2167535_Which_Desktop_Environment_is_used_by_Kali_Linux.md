### Post by pissedoffdude on 2013-08-14
That's gnome 3's fallback.  It's different than classic mode because Kali's using Gnome 3.4, whereas you have 3.8 installed.  But since you installed Gnome classic, you should be able to get an applications and places menu by enabling the classic extension (use gnome-tweak-tool).
