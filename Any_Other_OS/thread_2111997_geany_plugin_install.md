### Post by xenopeek on 2013-02-03
I haven't tried this on Linux Mint 12, as I don't have that version installed any longer, but maybe this will get you in the right direction. On Linux Mint 14 with Cinnamon, I install the package geany-plugins. Then I open Geany and go to the Tools > Plugin Manager menu and enable GeanyPG. Now I have a new entry in the Tools menu called GeanyPG. From there I can do various GPG operations. The keys from the Passwords and Keys application are used.
