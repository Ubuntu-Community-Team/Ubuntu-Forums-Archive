### Post by ojdon on 2011-11-13
**Official ones:**
Kubuntu - Using the KDE Desktop Environment instead of the default Unity interface.

Xubuntu - Using the XFCE Desktop Environment instead of the default Unity interface.

Lubuntu - Using the LXDE Desktop Environment instead of the default Unity interface. 

**Popular ones:**
Linux Mint - Aimed at being "more user friendly" so having it's own simpler applications well as adding more applications preinstalled by default.

Elementary OS - Aimed for speed as well as being visually appealing.

...Loads more! Far too many to list in one forum post. But I hope this gives you some insight at least. Wikipedia is actually a surprisingly good source at finding Ubuntu based distributions/forks.

### Post by shobon on 2011-11-13
Kubuntu, Xubuntu, Lubuntu, and any other distro using Ubuntu's repos are NOT forks, they're just spins or remixes. Forks are distros which originated from Ubuntu but are operating on their own repos, which makes Ubuntu a Debian fork.

### Post by arkanabar on 2011-11-13
Questions like "Which are the best" depend on "for what?"

For example, GnackTrack and DEFT are two of perhaps a half-dozen ubuntu-derived defense, penetration, and forensic toolkits.  Zentayal is a network server distro.  In all likelihood, your use case is English language desktop linux, and neither of these would be of use to you.  Neither would any of the language-customized versions (which includes a lot of them).

Even so, the question "which is the best" depends on what you like.  Madbox and Bodhi go for lightweight environments and preload very few applications.  Ultimate Edition includes everything but the kitchen sink, including multiple desktops in many editions.  Mint goes for preinstalled codecs, an improved menu and package manager, and a customized (somewhat more windows-like) interface.

I use Lubuntu.  It's lighter, snappier, and more stable (on my rig) than Ubuntu with Unity, I like the interface paradigm better, and it's easy to get the apps I want to launch with a combination of two or three keys.  I could do about as well with madbox, PeppermintOS, wattOS, or xPUD.

Not that those ubuntu forks, spins, and derivations that haven't made it into the DW database aren't on Seatex's list, and a couple of non-ubuntu distros have made it in.
