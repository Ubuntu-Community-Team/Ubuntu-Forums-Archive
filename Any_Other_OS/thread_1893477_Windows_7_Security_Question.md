### Post by Dangertux on 2011-12-15
No wonder everyone always complains about Windows being slow. 

ONE anti-malware solution, ONE firewall, NoScript some common sense and a decent system configuration is all you need. 

Firewalls...Don't argue about them, they're all the same. No really they are. Firewalls perform a very basic task.

AV - pick one and stick with it.

Anti-malware (no need) - has fallen under the AV roof and vice versa, they all do the same thing now. 

NoScript - Will save you most of the problems you're having.

The browser argument is silly as well. Firefox is exploited more often than IE these days, Chrome, IE and FF are all about on par in terms of security. IE actually has some really nice features (anti XSS engine) that neither of the others have by default that really harden it up.

Decent system configuration, don't disable UAC (it's easy enough for a malicious user or program to do that as it is). Strong credentials, disable unnecessary services blah blah, the typical stuff applies here.

Don't download stuff from obscure places and execute it, particularly if it wants you do disable DEP or UAC. About DEP, LEAVE IT ON (no I don't care that your computer runs faster without it, it's also loading arbitrary code into executable memory without it)

Hope that was helpful.

### Post by collisionystm on 2012-05-02
Here is what I use and have never had a problem.

Microsoft Security Essentials
CCleaner
Hosts file
HijackThis


I have removed and used many anti-virus programs over the last 10 years. Symantec (norton) used to be the best AV there was. No longer the case. In my experience norton's live security tries to filter to much and in the process it slows down your computer just by having it there. AVAST was great, however I have seen many cases as of late where It does not do the job to my liking. McAffee, never personally used it but have always removed it from machines I work on. Kaspersky, was good but now that it is involved with geek squad and other wanna-be technicians, I have lost respect for it.

Security essentials is light weight, made by microsoft for microsoft, and has always done its job for me.
CCleaner is a great program, also free, that will clean out all of your temp files for your web browsers all in one shot. Its nice.
[http://www.filehippo.com/download_ccleaner/](http://www.filehippo.com/download_ccleaner/)

I used spybot for a short period and then realized the only thing I ever used it for was its host file. Putting a good hosts file on your computer will help stop it from connecting to junk out on the internet.

Here is a reputable site to get a good hosts file
[http://winhelp2002.mvps.org/hosts.htm](http://winhelp2002.mvps.org/hosts.htm)

It goes into C:\Windows\System32\Drivers\Etc\

The first thing I do on a machine that may be infected is attempt to run msconfig. I disable all 'non microsoft' startup services and applications, including the anti-virus which is usually also broken.

You can also look for the malware inside of msconfig and it will show you the directory its executing out of. Make note of it. You now know where to go to remove it.

For web browsers, I prefer google chrome. It is a very stable and secure browser. Any company that puts out a competition to see who can hack it is golden in my book.
[http://www.pcmag.com/article2/0,2817,2400878,00.asp](http://www.pcmag.com/article2/0,2817,2400878,00.asp)

My last useful program has always been hijackthis
[http://www.filehippo.com/download_hijackthis/](http://www.filehippo.com/download_hijackthis/)

I have used Vipre Rescue once in a while with good success.
[http://live.sunbeltsoftware.com/](http://live.sunbeltsoftware.com/)

When removing Norton, please be sure to use the norton removal tool provided by symantec. Norton embeds itself pretty deep into the OS and can be a **** to get out.
