### Post by Karlchen on 2012-06-23
Hello, vikash_chandola.

> I think i have uninstalled wine. But C_drive is still there and contains around 32MB of data.. should i delete this folder.. The Wine related folder named **drive_c** is located below your home folder: **~/.wine/drive_c**
 Before you delete the content of a folder, you should carefully inspect its content. Then decide whether you still need it or not.

>  **Do you know any way to remove this "other" tab???** Use Synaptic and install a little programme named **alacarte**. **alacarte** is a GUI which will help you edit the application menus. Inside **alacarte** locate the folder **other** and find out whether it is really empty. Provided it is you may remove it.
