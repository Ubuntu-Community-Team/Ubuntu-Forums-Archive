### Post by xenopeek on 2013-02-24
You can install gnome-disk-utility, which you can then run from the menu (called "Disks") and use that to format your USB sticks. The gnome-disk-utility was one of the very few programs not forked by the MATE developers from GNOME 2.

As per the following topic over on the MATE forums, I don't think gnome-disk-utility integrates in Caja's right-click menu: [http://forums.mate-desktop.org/viewtopic.php?f=6&t=1481](http://forums.mate-desktop.org/viewtopic.php?f=6&t=1481)
