### Post by mamamia88 on 2013-04-09
Using firefox? [http://get.adobe.com/flashplayer/?no_redirect](http://get.adobe.com/flashplayer/?no_redirect)   Download the .tar.gz version.   Extract that .tar.gz and copy the file libflashplayer.so to /home/username/.mozilla/plugins.  If the plugins folder doesn't exist create it.  .mozilla is hidden by default you may have to enable show hidden files or something similar in your filebrowser
