### Post by Chronon on 2011-04-13
I'm running KDE version of 2010.2 on my netbook.  I like it and haven't had any major problems, though there is a weird issue with firefox and animated GIFs (seems to be due to an inconsistency between system version of Cairo and the one firefox was built against).

I've been lurking on the Mandriva forums a bit since I installed.  There seems to be a bit of resentment there revolving around lack of official response to certain issues.

I'm quite happy with 2010.2 and I've been getting regular updates, so I don't personally have any complaints about it.
