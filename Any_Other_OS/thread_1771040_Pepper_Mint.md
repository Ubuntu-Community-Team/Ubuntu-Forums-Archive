### Post by BigSilly on 2011-05-30
I've posted elsewhere about this brilliant OS. I'm using it on the tiny Eee 701, and it's absolutely fantastic. It is indeed based on Ubuntu, which makes it nice and easy to use, but they've made it ultra-lightweight so it is so incredibly speedy on this netbook.

I've tried loads of different distros, all claiming to be lightweight and undemanding on system resources. Peppermint Ice is the first one I've used on this low spec netbook that actually runs as the claims would have you believe. I won't pretend that it runs Flash any quicker because it can't (though it is better than I've seen it before), but it does run everything else so well, that I have to recommend you try it on your Eee if you have one.

### Post by exploder on 2011-05-30
Peppermint is based on the Lubuntu 10.04 release but Kendall has built some of the system from Debian source. Peppermint is built to be very stable, quick and light on resources. The better part of the applications are web based but they blend it with the installed apps by design. I used Peppermint for a few months and was very impressed with the quality work Kendall and the Peppermint Team have achieved. 

I moved from Peppermint to a custom install of PCLinuxOS KDE mini because I wanted updated software and I liked the idea of choosing what applications were installed. I never had any problems using Peppermint and it is a very good distribution.
