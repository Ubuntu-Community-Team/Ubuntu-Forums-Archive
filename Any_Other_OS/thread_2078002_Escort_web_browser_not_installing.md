### Post by zombifier25 on 2012-10-29
I found this: [Setup script](https://es-operating-system.googlecode.com/svn/trunk/setup.escort)

Right click the page, choose to save it (to the home folder for convenience). Then open the file manager, right click the newly downloaded file, choose "Properties" > "Permissions", tick the checkbox "Allows to execute file like a program". Then double click it and choose to run it.

Or in terminal:
```
chmod +x setup.escort
./setup.escort
```


(I agree with vasa about this being a pretty obscure browser though. What are the benefits?)
