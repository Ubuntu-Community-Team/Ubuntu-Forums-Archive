### Post by Megaptera on 2011-05-20
Ubuntu Privacy Remix (UPR) is a modified live DVD based on Ubuntu. Its goal is to provide a completely isolated working environment where private data can be dealt with safely and to protect data against unsolicited access. Networking is intentionally disabled and saving data to mounted volumes is not allowed. The live CD is not installable to hard disk.
Very locked-down!  [http://distrowatch.com/table.php?distribution=ubuntupr](http://distrowatch.com/table.php?distribution=ubuntupr)

Fedora 15 is going to be out in a few days. The Release Candidate has been declared Ready, but won't be "officially" released until 5 days from now.

### Post by jd.jayded on 2011-05-24
A few options...
For any debian-based distro:
Make a new account. go to a terminal, sudo visudo, and add the new account so it can use apt-get, synaptic, and dpkg. 
(or equivalent for whatever distro you like)
lock the original account or have a friend set a password.

You could then install software and/or remove it, but that's it. Nothing that requires sudo, anyway.
I like the permissions on config files too.

Alternatively...
Damn Small Linux is...let's say difficult to customize. Difficult is a good word.

I've never used it, but Tin Hat linux without emerge might do it for you.
Or gentoo in general. Gentoo is either most or least configurable OS. Depending on flexible you're feeling about that- you have to install and compile more or less everything yourself.

If you try gentoo, you can set the use flags so you have no gui to customize.
You can kind of browse the web with links...elinks to debian users.
