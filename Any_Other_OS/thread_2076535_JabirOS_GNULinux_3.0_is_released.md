I’m very happy for announcing JabirOS GNU/Linux 3.0 release.

JabirOS GNU/Linux 3.0 is based on Ubuntu 12.10 and has this features :

Linux Kernel 3.5.5 (Ubutnu Kernel).
GNOME 3.6 (with GDM Login Management)
VLC Media Player ( all of codecs).
Flash Player (Licensed)
Farsi & English fonts.
Also we have GIMP 2.8 on JabirOS GNU/Linux 3.0
