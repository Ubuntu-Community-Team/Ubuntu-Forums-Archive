### Post by tgalati4 on 2011-06-22
I have a machine set up with FreeNAS to serve files and media.  It is based on a stripped down version of FreeBSD.  I've also played with PCBSD which is also built on FreeBSD and provides an easier installation and desktop environment for new users.

It's different.  Not better or worse.  The Linux community is much larger so there's a lot of development work going on, but most things that a desktop user would want you can find in PCBSD.  There's a smaller selection of hardware drivers, so if you install on a laptop, some things may not work.  FreeBSD has been ported to more CPU architectures, so it can actually run on all sorts of devices big and small.

If you are a Mac user, several PCBSD/FreeBSD packages have been ported to OS X and can be installed via *.dmg or through Fink Commander or the Ports system.
