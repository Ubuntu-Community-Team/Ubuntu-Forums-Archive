### Post by cariboo on 2012-06-07
The first thing to do is to determine whether the network adapter is detected and if the proper driver is loaded. Open a terminal (Ctrl-Alt-t) and type the following command:

```
sudo lshw -c network
```

Paste the output of the above command in your next post.
