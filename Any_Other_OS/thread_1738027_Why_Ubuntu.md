### Post by Joe of loath on 2011-04-24
Why do we use Linux?

1. More flexible. The Linux kernel supports about a dozen different processor types.
2. Freedom. Want to change something? Go do it! Want to upgrade your PC and move your installation over? Go for it! No messing with WAT or activation or product keys.
3. Performance. In benchmarks, Linux comes out on top.
4. Choice. Don't like how it looks? Install a different desktop environment. Don't like your file manager? Install a different one.
5. Convenience. Want to install a new web browser? On Windows you have to click on your current web browser, search for what you want, download the file, find it, click install, click through the steps. In Ubuntu you just open the software centre, search for what you want and click 'install'. No worries about dodgy download sites, viruses, spyware etc.
6. Security. Viruses? What are they? And did you know how easy it is to crack Windows passwords? 20 seconds with chntpw (A Linux utility ;)) and any Windows PC is under my control.
7. Scalability. Linux runs on everything from your digital TV set top box to 90% of the world's fastest supercomputers (The other 9% run UNIX, and the last 1% run Windows). That's the same kernel too, pretty much.

Oh, and did I mention the price? ;)
