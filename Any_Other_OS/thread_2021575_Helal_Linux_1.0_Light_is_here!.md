we are pleased to announce the availability of the first 'Helal Linux Light' version which is a Helal Linux distro based on lubuntu 12.04 featuring Helal Linux style and tools but for old computers and notebooks, and in Helal Linux Light there are many features like:

based on Lubuntu 12.04 with LXDE and linux kernel 3.2.

Helal Software Center,Light Helal Control Center which is a small copy from Helal Control Center that was used in Helal 3.0 Final.

Support for Arabic language and adding extra fonts.

Some Islamic apps have been added like Othman,Hijra.

flash player and some extra codecs for multimedia and gnome-mplayer.

many customizations in GUI and some other system apps.

LXDM now is the default display manager for the system.

the system's .iso is less then 700 MB whick make it easy to put it on CD.

many cool applications have been added, like:

 firefox 13, gimp 2.6,lxdm,linux kernel 3.2,pcmanfm 0.9.10,pidgin,filezilla,xchat,transmisson ,uget,Geany,Gnu paint , Gnome Paint,xfburn,mplayer,Arandr,Gdebi,Synaptic,wicd,Gp arted .Grub Customizer,hijra,othman,gnome-screenshot,FLTK games,lxlauncher,remastersys.

many other cool features..
