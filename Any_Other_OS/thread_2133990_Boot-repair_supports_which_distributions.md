### Post by oldfred on 2013-04-10
If it is based on Debian I am pretty sure it will work. 
Somewhere in the now 100 page mega thread Yann mentioned which ones worked.

It requires certain Linux tools, which are normally part of any Linux install. And it supports grub2, but not grub legacy.
You can always run it from a Ubuntu liveCD/DVD/Flash on any system or download its own SecureRemix that includes Boot-Repair and full Ubuntu. It is oversize, so you need a flash drive or DVD.

 Install in Ubuntu liveCD or USB or Full RepairCD with Boot-Repair (for newer computers)
