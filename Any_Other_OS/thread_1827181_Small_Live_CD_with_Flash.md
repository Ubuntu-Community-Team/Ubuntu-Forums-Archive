### Post by -kg- on 2011-08-20
I admit that Puppy (Lucid Puppy, actually...based on Ubuntu 10.04, with access to its repos) was a bit of a PITA to get connected, but once it was (and the session info saved) it does it seamlessly every time I boot.  Speaking of access to Ubuntu's repos...

If Lucid Puppy doesn't come with Flash, that's easy enough to solve.  You have access to the Ubuntu repos (and everything else).  You can either load "ubuntu-restricted-extras" or install Firefox from the repos (which is what I did), install the Flash-aid plug-in and run it.  Of course, I don't know whether flash-aid will work under Lucid Puppy, but I'm almost betting it will.

Flash shouldn't be a problem.
