### Post by ManualSparrow on 2011-05-18
Don't forget - Puppy always runs in root, which is frowned upon in general. But if that is not a problem for you, you can do to the Utilities menu and do a 'frugal install'. Also the official Puppy forums are at [http://murga-linux.com/puppy/](http://murga-linux.com/puppy/) so you can go there for support if you need it.

EDIT: a flash drive install boots in about 40 seconds and runs faster than the CD (probably about the same as hard drive) so you may just want to do that for a while. Then it's portable.
