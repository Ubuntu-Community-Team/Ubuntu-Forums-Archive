> **Z0mb13M0nk3y said:**

Put the CD in your computer, reboot, double click the "Install LinuxMint" icon and follow the on screen prompts.

> Have any of you come across any problems while installing and how was that fixed?

I personally haven't.  Most of the problems *I've heard/read about* are due to using hardware that is not compatible with Linux.

> After install what should be some of the first things I do and setup?

Install "Extra non-free drivers" if needed.  The OS will detect automatically.  Follow on-screen prompts.

Install all system updates. The OS will detect automatically.  Follow on-screen prompts.

>  Are there any "necessary" programs I should get? What are some of the "just too great to miss" programs?

Mint is a really complete distro right out of the box but, if there is something *you* think *you* need that's not there, do a search in Software Center or Synaptic.  I like to install Screenlets, some themes, Picasa photo manager and some other stuff.  It's really up to you.

>  I know it's a lot of questions, but I'm one of those people that likes to try to plan ahead so a problem isn't technically a problem when it pops up.

No problem!  It's what we're here for.  :)

Although I rarely visit the [official Linux Mint forums]("http://forums.linuxmint.com/"), that is a great resource. However, with the inclusion of the "Other Distro/OS" forum and the awesome community vibe here at Ubuntu Forums, I don't think you'll have any problems getting help. Ask away!  :)

### Post by 2F4U on 2011-07-19
If you are completely new to Linux and don't know if it suits you, I would download VirtualBox for Windows and install Linux Mint into a Virtual Machine. That requires more resources than just running Windows standalone but gives you the chance to determine whether it is right for you. Also, since it is a VM if something goes wrong you just go back to a backup copy of the VM or you just reinstall. And all that doesn't affect your working Windows installation.
