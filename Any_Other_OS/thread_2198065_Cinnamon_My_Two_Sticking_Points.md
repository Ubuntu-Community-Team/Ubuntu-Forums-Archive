### Post by buzzingrobot on 2014-01-06
I've been looking at Mint 16 Cinnamon, the Petra release.  It's nice, but I've found two sticking points:

1. **Cinnamon is too shiny.**  It glistens.  This is totally subjective, of course.  But, I'd prefer something more toned down, along the lines of the flatness of the typical Mate or XFCE installation.  Anyone know some tricks? (Tried several themes; they didn't help. Some dark themes tone things down, but dark themes don't appeal to me.)

2.  **No way to restore a minimized window to its original workspace**.  Cinnamon seems to be the only panel-based DE that I've used that does not allow this.  I.e., a window in a workspace that is minimized to the panel is always restored to that orginal workspace, no matter which workspace the user is in when the window's minimized icon is clicked to trigger the restoral.  In Cinnamon, if I minimize a window in one workspace, move to another, and click on that minimized window in the panel, the window is restored in my current workspace.  Not what I want.  Not having this capability is a dealbreaker for me.  None of the applets/extensions available via the distribution enable this.  Is a workaround available?

### Post by ibjsb4 on 2014-01-06
> But, I'd prefer something more toned down, along the lines of the flatness of the typical Mate or XFCE installation.

Gnome Fallback (no-effects) is toned down and flat.

```
sudo apt-get install gnome-panel
```

[https://help.ubuntu.com/community/PreciseGnomeClassicTweaks](https://help.ubuntu.com/community/PreciseGnomeClassicTweaks) 

Edit: I forgot, minimized windows can be moved via the context menu.
