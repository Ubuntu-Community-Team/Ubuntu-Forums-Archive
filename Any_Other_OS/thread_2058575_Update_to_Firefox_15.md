### Post by snowpine on 2012-09-16
If "Pinguy" is based on Ubuntu 12.04 then you update your Firefox by running the Update Manager or the following terminal commands, same as Ubuntu:

```
sudo apt-get update
sudo apt-get dist-upgrade
```

If it's not based on Ubuntu then I have absolutely no idea. :)
