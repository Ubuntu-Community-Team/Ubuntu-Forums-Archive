### Post by drawkcab on 2012-09-19
Instead of using a cd or dvd, install ubuntu with a usb drive using the "startup disk creator" app in ubuntu or unetbootin in windows.  You just need the .iso and then you can create a bootable usb.

After that, install ubuntu to the entire disk.
