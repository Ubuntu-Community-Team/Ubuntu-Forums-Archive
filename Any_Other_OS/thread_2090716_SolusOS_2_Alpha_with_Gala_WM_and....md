No, they are not. You have to make SolusOS read ppas. Then, you have to add eOS ppa to get at Gala. You could install the new Slingshot from the same ppa, but I like the old one and had downloaded the .deb files for both 32 and 64 bit ones. To get some needed files, I added the whole Ubuntu precise repos, updated it and then installed those dependencies. At one place, I did sudo apt-get -f install without a name after. Once, everything is installed, you give directions to gnome-session to read Gala, instead the Gnome-wm.

The result is fabulous. I just made a note to Ikey. That guy is very good!:)
I won't do anything to harm his distro! That's one heck of a distro, really!
