### Post by Habitual on 2014-06-13
I *always* remove any partitions on **every** vendor-supplied external storage and recreate them using Linux tools.
'setup' suggests setup.exe and you won't be able to use that *without Windows *to provision whatever software is on the removable storage.

gparted can remove those partitions and re-create them.
