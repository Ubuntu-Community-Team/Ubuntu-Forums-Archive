### Post by offgridguy on 2013-02-22
In my ongoing trial of linux OS, I have gotten to Mint 14,  I am impressed, some systems
just seem to have a good 'feel' to them and this is one of those.
    This is my first experience with cinnamon and i'm liking it a lot, very easy to use and
learn.   This distro is more fully featured than  the simpler ones that i prefer, but i can see
where i will be exploring these features a lot more as everything seems very user friendly.

On an added note I am finding i prefer debian/ubuntu based distro's, especially when it
comes to installation, just faster , easier, less problems.

This seems like a good example of taking something that is already good {ubuntu} and
making it better.  It does make me wonder why canonical did not go this direction ?

I can certainly see why this is such a popular distro.:D
edit ; I wanted to add, i multi-boot this laptop and installed mint last as it gives
a cleaner grub menu.

Ubuntu is more of a "any device" OS right now, tablets, desktops, smartphones, etc.

### Post by craig10x on 2013-02-23
Yes, you can install cinnamon desktop in ubuntu using the ppa and you will be offered a separate "cinnamon session" you can get into upon log in (from the "cog wheel" that has a drop down menu)...

Also, you can install Cairo Dock from ubuntu software center, and select "Cairo Dock Session" from that same ubuntu log in screen...
on those sessions, you do not get the unity dock bar...

I've done them both...though in my case, i decided to stick with the default unity desktop for myself...i didn't think i'd like it at first, but it really grows on you after awhile, and now i like it a lot...
