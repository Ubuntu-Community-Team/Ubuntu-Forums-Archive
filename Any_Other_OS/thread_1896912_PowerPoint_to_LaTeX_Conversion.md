### Post by Lars Noodén on 2011-12-19
> **silveringking said:**
> ... I want something to be read in any computer, be it windows, or linux or solaris with ease.

I was mortified to find that Google Docs doesn't support ODF for presentations because that is the way to go for cross-platform support.  Even though it is a lot of work, in your shoes, I would redo the presentation in LibreOffice so that it can be saved in OpenDocument Format.  

Your other option is to save it in Google Docs as PDF, but that means that it is a read-only copy that gets distributed.  With PDF, the layout and fonts will also be the same across the various systems it is displayed on.
