### Post by Dr. C on 2012-11-28
> **gnusci said:**
> What is surface tablet???

It is basically Microsoft's answer to the closed and censored Apple iTunes platform only with far fewer apps. As with the Apple case only apps approved by Microsoft via the Microsoft store can run on Windows 8 RT. This is not like the regular Windows at all. 

The "solution" that some offer to the OP's problem is to rip the DVD to a format compatible with Windows Media Player. The trouble with this "solution" is that if the DVD in question is infected with DRM (virtually all commercial DVDs are) this more often than not involves breaking the DRM which under anti-circumvention laws in many jurisdictions is illegal. As a consequence the only solution I can recommend is for the OP to purchase another copy of the movies in question from the studios in a digital format compatible with the surface tablet.

Connecting an external DVD player via a USB port may also work if the tablet supports this; however this will defeat many of the mobile advantages of the device.
