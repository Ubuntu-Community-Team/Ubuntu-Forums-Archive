[https://launchpad.net/bottomlauncher](https://launchpad.net/bottomlauncher)

Why kedos ?
-we would like to simplify user experience with kedos that would give the user a much amazing experience

-Kedos tries to build every app for its own platform :
Like music player , photo viewer ,shell , etc ...
And with a simple design and awesome features we could give what Mac os X gives to the user from cool features to beautiful design.

Could it happen "build every app" :confused:
- i don't mean that we must build EVERY single app , no ! We will build a few apps as the plan.

Ubuntu is already awesome , why would you want to make this awesome OS ?
- i've tried elementary os , and to be honest it is so fast and amazing OS , why ?
Because it uses simple apps made by elementary os team itself !
but the problem in elementary is that they are cloning Mac OS X interface (pantheon) , and we don't want to copy/paste thing here !

We try to be creative to do awesome things.

What is the main language in Kedos ?
There is no main language , but i prefere to use "Vala"
See [https://live.gnome.org/Vala](https://live.gnome.org/Vala) to know more

Kedos Information:
Build name : Mountain Peak
Based on : Ubuntu Precise Pangolin 12.04 LTS
Main Language : English
Includes :
Patched Plymouth
Kedos Shell
Kedos Login Screen (Going to put screenshots of it soon)
Kedos Window manager (also done !)
Kedos File Manager (Not yet)
etc ...

Features :
Kedos is looking to make awesome things like improving notification system with a Notification Center !
or boosting the OS with crazy hacky things :)
and improve linux experience on any computer (even old ones) with fantastic animations ! (already done by kedos window manager)

There is a lot of things that we can add to Kedos and its unlimited , and we are looking forward to make Kedos small-sized too !

Please ask any question you want to !
And remember that there is a lot of packages that are ready for testing, so kedos isn't that far from us :KS
Check this Post for more updates soon
