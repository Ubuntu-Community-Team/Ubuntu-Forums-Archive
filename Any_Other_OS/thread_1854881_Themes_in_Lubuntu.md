### Post by BrokenKingpin on 2011-10-06
LXDE is nice, but you don't get that many options when it comes to eye candy. Xfce has more options when it comes to themes, but it is not as light as LXDE (but still pretty quick). If you are ok with using Xubuntu (Xfce) you will have a lot more options. Xfce has a built in compositor, so you also get good transparency effects.

A dark theme similar to Vista/Win 7 is shiki-brave, which you can install from the Xubuntu repos. To get a task bar like Win 7 install the Talika applet (using xfapplet).
