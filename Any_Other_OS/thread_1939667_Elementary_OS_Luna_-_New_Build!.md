### Post by ojdon on 2012-03-13
You'd be able to use the **VERY** unstable Elementary Daily PPA since it still has oneiric packages and either install pantheon-shell (For just the Desktop Enviroment. Or install elementary-desktop for all the elementary bells and whistles. 

However, one day it might work and one day it will be completely broken if they are working on it at the time, so I wouldn't advise it and just try it out on the Live CD which would have a bit better stabillity than installing it on Oneiric. (The live CD is based on Ubuntu 12.04 LTS)

---

Anyway, I've tried it and I'm so impressed. The Elementary team have clearly done their research on OS usability and have grab concepts from various different Operating Systems and picked out the best bits for sure!

It is, however, still very early on and the Live CD is more of a "proof-of-concept" of what's to come for eOS 0.2 Luna.
