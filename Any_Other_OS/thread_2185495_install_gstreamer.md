### Post by heir4c on 2013-11-03
The package are in the repos. So open SoftwareCenter or Synaptic and look for: gstreamer0.10-plugins-ugly
Or open a terminal and type in:
```
sudo apt-get install gstreamer0.10-plugins-ugly
```
Click Enter and let the computer do his job.
That's all you have to do for now.
Than a few questions:
What version of Bodhi have you? That's important to now what package you must use.
And look also for the package number of the gstreamer0.10-plugins-ugly that you installed with that command above.
