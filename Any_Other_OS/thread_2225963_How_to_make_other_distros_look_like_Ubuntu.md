### Post by Frogs Hair on 2014-05-24
I don't what terminal you use , but the background html color code for the Ubuntu  terminal is  #2C001E. and that can be changed in the terminal's profile preferences . I think  the Red Hat family  uses the rpm package system and installing the default themes from .deb wouldn't work  unless someone has converted them from .deb . If you can in install extracted  tar.gz packages into the theme and icon folders see the links . 

[https://launchpad.net/ubuntu/quantal/+source/ubuntu-mono](https://launchpad.net/ubuntu/quantal/+source/ubuntu-mono)
