### Post by GeForce 9500GT on 2012-09-17
> Bottom line: Which card has the most RAM onboard? Go for that one.
 
The topic starter mention 2 options:
1 - Onboard graphic controller
2 - dedicated VGA card 
 
Your opinion of choosing the card with the most RAM available sounds perfect but onboard cards are limited in memory usage since they consume a certain part of the onboard main memory. In most cases this is limited to 256mb to avoid overconsuming the main memory. The peformance of onboard graphic controllers are running behind the performance of dedicated VGA card. 
 
You didn't specified the brand/model/type of the motherboard, this is interesting to know to determine the amount of memory which can be assigned to the onboard graphic controller to give you a better advise.
 
My opinion for now, go for the VGA card (or as you call it, the dedicated graphic card).

### Post by SeijiSensei on 2012-09-17
An NVIDIA 8xxx card will support the VDPAU interface that takes advantage of the card's graphics hardware to offloading video processing.  If the organization has boxes running Linux, and they intend to watch videos, this would be a useful upgrade.  It would also help if they are using modern Linux desktop environments like KDE which run more smoothly with dedicated graphics cards.

It would also help with playing games on Windows machines.  I don't know if there are any other advantages to using that card with Windows.  Maybe video as well?

If this machine is used to write documents or manage spreadsheets, there is less reason to upgrade.
