+1 for the Shotts book. If you prefer traditional printed books, No Starch Press recently published a paper version of it:

[http://tinyurl.com/7gqta8m]("http://tinyurl.com/7gqta8m")
Shotts, W.E. *The Linux Command Line*. No Starch Press, 2012. 480 p. ISBN 1593273894

Another (printed) possibility which I've found helpful is Scott Granneman's book. It too is a command-line reference, is slightly larger than pocket-size, and has a detailed index:

[http://tinyurl.com/cktpfc5]("http://tinyurl.com/cktpfc5")
Granneman, Scott. *Linux Phrasebook*. Sams, 2006. 400 p. ISBN 0672328380
