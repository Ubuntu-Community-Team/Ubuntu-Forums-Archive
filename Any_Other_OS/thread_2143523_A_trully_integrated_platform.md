### Post by platform on 2013-05-09
Hi folks, let me introduce myself, 'cause I am new here. I am an Ubuntu/Debian user (I use the both) and Android/Linux/Web developer.

I love Linux and the open-source software in general, but I think that we need a trully integrated platform. Because of that, I like hearing that Canonical is going to create a quite good approach of an integrated ecosystem. The problem is that they've done many things wrong. For example, I love Unity's concept: one interface for all devices and it will be able to adapt itself to the different screens (responsive design), but I hate its implementation. They are quite open, but it would be better to have a community-driven platform, wouldn't it?

So, I'm planning to create a trully integrated platfrom. Before you kill me :P let me say that I don't want to create "just another" Linux distro, I want to create something more. I have to admit that there are many remasters of Ubuntu... and most of them are only Ubuntu + a theme + some programs. But I don't want to do that, I want to create a new desktop environment, APIs and some basic apps... More or else like Elementary does but with responsive design and... well, I have to say that Elementary is quite original but I don't like its interface... Only the dock... 

Of course, I can't do that alone... I am only a modest developer... I think that we could create a cool community: everybody is welcome. Developers, designers, users,... I think that we need that trully integrated platform, which at the moment only Apple can offer, and we need a better one and **community-driven**.

We could start by discussing in which framework are we going to base our OS (Qt5, GTK, HTML5,...), which OS-base should it have (Debian, Ubuntu, Android or maybe a LFS...), and how should the desktop environment look like. I have some ideas about it (and one of them is very simple and, as far as I'm concerned, cool, but I'm sure you can suggest su much better ideas).

If you want to join the project, please let me know it.

### Post by platform on 2013-05-09
[B]About the environment:
[/B]
I think that the environment should have the following features, but of course they are only suggestions:

-The most important features should be the simplicity, the lightness and the adaptability.
-It should have a dock at the bottom (but, of course, letting the users move it to the left side or to the right side). It should be very very light and simple, but drag & drop apps to dock them is a must-have, and "notification globes" would be cool. Animations: the minimum, but they should let the user know which apps are running and which not.
-It should have a bar on the top of the screen, Unity/Mac like. I love the global menus (and we could adapt it to mobile devices). We should provide a simple API to show notifications on it, and like this the notifications wouldn't be intrusive like the Ubuntu ones. I mean, the notifications should be show on a little icon on the bar, and then the user could be able to extend it ONLY if he/she wants so. We could also provide a system to show indicators (Wi-Fi, battery,...) but these indicators should be customizable (the bar only shows the indicators that the user wants).
-It should have an app launcher, something like Slingshot, Dash or Launchpad.
-All the apps and the environment itself should follow the design guidelines and be responsive.

What do you think about it?
