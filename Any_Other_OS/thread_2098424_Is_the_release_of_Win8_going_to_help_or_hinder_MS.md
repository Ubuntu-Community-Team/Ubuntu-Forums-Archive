### Post by 3rdalbum on 2012-12-30
Microsoft OS strategy goes a little something like this:

1. Do 90% of what you want to do for your next OS. Break compatibility or make major interface changes if you like.
2. The public hates it.
3. Make money anyway because the public is still buying the old operating system.
4. Hardware and software vendors update their products to make them compatible with the new, much-hated OS.
5. Put the last 10% of work into your next OS. Don't break compatibility. Limit major interface changes. Everything made for the last, hated OS will work on the new one.
6. The public are now used the last, hated OS and think your new OS is the best thing since sliced bread.
7. Repeat the cycle.

In short, Windows 8 will be a success because it's introduced the things Microsoft wants to do. People will get used to Windows 8, even though they grumble about how much they hate it. When Microsoft releases Windows 9, which is really Windows 8.1, everyone loves it.
