### Post by 2048Megabytes on 2013-04-20
I figured it out,  In Linux Mint 13 with the Cinnamon interface you do the following:

Left click: **Menu** => **Administration** => **Synaptic Package Manager **

Type your Administrator password in.  In the Synaptic Package Manager scroll down until you see "kbreakout" and select it.  Then left click **Apply** to start the download.
