### Post by mips on 2011-11-13
It's the same as a dual boot, just make sure you have free partitions to install it on. It will use your existing swap partition so you just need a / or / & /home partition, choice is yours. You can use your existing grub as well or let it install a new new grub and it should pick up the existing os/distros installed.

### Post by oldfred on 2011-11-13
Then you should also create a shared NTFS partition for data. Then all sysems can share that data. I prefer small system partitions and larger data partitions anyway. 

I have both a shared NTFS for Firefox & Thunderbird profiles and a /mnt/data for other data that I will never want from my XP install. Now I use XP very little, but still have the shared NTFS data partition.
