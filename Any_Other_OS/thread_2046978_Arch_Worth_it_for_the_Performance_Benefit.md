### Post by lykwydchykyn on 2012-08-24
Any time you set up a distro from bare-bones, you're bound to see some performance improvement because you're going to leave out stuff you don't need and didn't realize was there.

You can do the same with Ubuntu-minimal or Ubuntu server, but there's nothing wrong with using Arch if you like the way it works.

Just don't think Arch is the only way to get a custom-built install.

The bigger issue between Arch and Ubuntu is whether you want a rolling release or a big upgrade every six months.
