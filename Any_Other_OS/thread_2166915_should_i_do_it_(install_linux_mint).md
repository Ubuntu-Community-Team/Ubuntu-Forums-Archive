### Post by grahammechanical on 2013-08-11
If you was to ask this on the Linux Mint forum, the answer would be, Yes. But you are asking this on a Ubuntu forum and it looks like I am the first to get a reply in. My answer is, No. Install Ubuntu! :)

The truth is, it is your choice. You are free to install whatever Linux distribution you like. I have never installed Linux Mint so I do not know if it is possible to install without an internet connection. But we can do it with Ubuntu and the flavours of Ubuntu. We are asked to tick a box confirming that we want the OS updated as part of the install process. Leave that blank and Ubuntu should install without an internet connection.
