### Post by C.S.Cameron on 2013-06-09
Welcome to the Forums.

You can install and customize Ubuntu just how you want it, then make an .iso of the results using remastersys.

[http://www.remastersys.com/](http://www.remastersys.com/)

The downloads are still working.

[http://www.os4online.com/p/downloads-site-links.html](http://www.os4online.com/p/downloads-site-links.html)

### Post by Cheesehead on 2013-06-09
The best way that's maintainable is [http://live.debian.net](http://live.debian.net)
Does it have a learning curve? Yes. Very much.
But if you are *really* interested in a live distro, it's worth learning.
It's wonderful to have all customizations in one set of files - change and reroll a new live version very easily.
