### Post by kiyop on 2013-02-16

Most computers uses DEL, F2, F12, Esc or so.
I remember that some acer laptop PCs use strange key to enter into BIOS configuration. You can refer to the manual of your PC.

You may be able to boot from CD/DVD after grub2. There should be some commands to boot from CD/DVD in grub2 command line.
