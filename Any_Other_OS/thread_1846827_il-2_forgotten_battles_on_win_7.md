> **mmsmc said:**

All you can do is run it through the installer on your Win 7 box. Don't be too surprised if it doesn't work. Most of the windows Flight sims are pretty much tied to the directX version of their birth. W7 has to run them as legacy graphics apps. It does not do this reliably, so expect problems. Actually w7 makes a real mess of running MS FSX.

Your best bet is to get hold of an XP SP3 installer and run your old windows flight sims on a dedicated XP box. MS have discontinued all development of Flight Sim and this have stopped a whole industry.

Good luck

### Post by oscarivera9 on 2011-09-21
Microsoft just came out with this new thing called Windows XP Mode that requires Microsoft Virtual PC to run.  After you install the Virtual PC, then you can download and install Windows XP mode which then puts a Windows XP computer inside your Windows 7 system.  The only downfall is that I believe it may only be available for Windows 7 Ultimate and Windows 7 Professional, but I could be wrong.  It's worth looking into, after all it's free.  
Go here to find out all about it and to download it:
