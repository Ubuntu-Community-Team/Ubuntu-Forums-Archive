If I may fuss a bit, gnome shell **is not** unity.  Their functions are somewhat similar but the underpinnings are different.  Both use application launchers on the left and a search-for-apps user interface but they feel a good bit different to me.  There is also an extension to add Mint12 appearance & function to Ubuntu 11.10 so you could switch between Unity, Gnome shell and Mint12 if you were so inclined.  Obviously Mint12 downloaded from Mint's sources does not include Compiz/Unity code for someone who abhors Unity on principle.  As I understand it Unity is really just a Compiz plug-in and Mint12 (Ubuntu base) is a collection of tweaked Gnome 3 extensions.
