You can install Windows-32-bit on both 32-bit and 64-bit x86-CPU's, so that can't be the issue. So no need to know the CPU type.

If you still want to know: boot a Ubuntu CD, and then "cat /proc/cpuinfo". Or in System Monitor (=GUI interface)
