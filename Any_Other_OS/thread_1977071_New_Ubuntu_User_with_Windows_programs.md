### Post by Bucky Ball on 2012-05-10
Welcome. 

If you are a film major and need to use Adobe products in Windows, Ubuntu is a bad choice and I would either install Windows as a virtual machine, dual-boot Windows and Ubuntu or go back to Windows. It seems strange you weren't consulted about your requirements before being landed with a Ubuntu box which is pretty unsuitable for you purposes (without some serious diddling about). ;)

You can have a look here to see if your apps will work with Wine. Some will, some will sort of, some won't.

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

I just had a look here:

[http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true](http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true)

... and I don't see Adobe Elements there. Maybe that is why it didn't work for you; because it doesn't work with Wine. Probably wasn't your screw up. (And if you do make a few mistakes, that's how you learn; after all, you're in the process of getting across a whole new operating system. ;) )
