### Post by buzzingrobot on 2012-08-15
> **BBQdave said:**
> ...the Apple mind set is that you buy new applications with new Apple OS's.

That's incorrect. Apple's OS revisions are typically very evolutionary.  I've moved apps and data between several upgrades and 4 machines with no problems and few mandatory application upgrades. Breakage is no more and no less an issue than in Linux.

The move from the PPC platform to the Intel platform obviously broke code specific to the PPC.  There, though, Apple supplied an emulator in OS X to run PPC code, and it was only recently removed.  If memory serves, the initial releases of OS X also shipped with OS 9.

The OP might look for data converters to move those Excel files to a format current software can handle. Or, perhaps, run an old version of Windows and an old version of Office in a virtual environment. You're looking at files that are 20-25 years old, written for an old application that ran on an OS that's been dead for a decade or so, on a hardware platform that vanished about the same time.
