I just have mixed views...

 I imagine it will run fine with a mouse/keyboard. A big button will work just as well with a mouse, as a finger.

100% of people using Windows today aren't use Metro  Apps, they're using Desktop Apps. That 100% will work in the Desktop  mode of Windows 8.

* Is the constant jarring nature of Metro to Desktop to Metro annoying?

### Post by Paqman on 2011-09-25
It looks like a really interesting refresh for the Windows line, but quite obviously suffers from tabletitis. Microsoft have obviously been shaken a little by the success of the Ipad and don't want to be left out of the game.

As you point out, it remains to be seen how well it will integrate the desktop and tablet interfaces into one system. Microsoft are kind of betting the farm that they'll be able to do it, which is bold. It would have been safer for them to release a separate version for tablets and stick with the current desktop for their enterprise users.
