### Post by XubuRoxMySox on 2011-07-14
Four suggestions: **[COLOR="Purple"]AntiX[/COLOR]** (it's not Xfce to begin with, but Xfce is easily added after installation); or **[COLOR="Purple"]Aptosid[/COLOR]** if you like rolling release Debian Sid Xfce. If you prefer Debian Stable as a base, give **[COLOR="Purple"]Crunchbang Statler[/COLOR]** Xfce a shot. Awesome speed and geeky fun!

But believe it or not, **[COLOR="Purple"][SIZE="4"]Xubuntu[/SIZE][/COLOR]** has been on a serious diet and totally rocks now. *I find no discernible difference in performance between Xubuntu and minimal Ubuntu with LXDE* (Lubuntu, kinda sorta). It's got the same familiar installer we all know and love, really fast and lightweight compared to previous versions. Linux Mint's newest Xfce edition is based on Debian instead of Xubuntu now and, while it's faster than the *older* versions of Xubuntu, it's alot harder to get set up and stuff, despite Mint's wonderful additions.

-Robin
