### Post by Copper Bezel on 2011-10-02
> Not sure what you mean by re-adding task manager applet. I have deleted the applets and re-added them while in AWN task manager screen.
Let me clarify this - the objects that you add to the AWN panel are "applets," and one of those is the task manager. It's listed along with the other applets on the Applets screen. When you add an application to the task manager applet, you're adding a *launcher*.

In your second screen shot, it says that the Launcher / Task Manager isn't active in the first dock. If you switch over to the Applets tab, make sure you have Launcher / Task Manager added.
