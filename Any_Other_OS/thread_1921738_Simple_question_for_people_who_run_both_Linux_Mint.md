### Post by sunewbie on 2012-02-09
if you are talking about 10.10 and not 11.10, then 10.10 uses gnome 2 and mint 12 uses gnome with MGSE and MATE as DE (desktop environment). Later 'Clem' LM founder has cforked gnome shell and created cinnamon, which is very stable and offers more customization then gnome shell

Mint Software manager is a weak point, from GUI interface, but Ubuntu Software center rocks.

By default, LM has restricted sources and non-free extras enabled and already installed. 

Everything works well. I have both Xubuntu (and installed a lot of apps like libreoffice), LM 12 + cinnamon, Bodhi Linux 1.3 and Pinguy 11.04. All are good.

So No disadvantages
you can get everything that you have in Ubuntu repos, even UNITY ;)
