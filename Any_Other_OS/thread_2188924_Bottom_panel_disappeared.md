### Post by buzzingrobot on 2013-11-20
You don't say if you are using MATE or Cinnamon, but it appears you can get the panel to display by pushing the mouse cursor against the bottom of the screen,  If you are running MATE, a right-click on the panel will expose a menu.  Select the "Properties" option on that menu and, if the Autohide option is selected, deselect it.
