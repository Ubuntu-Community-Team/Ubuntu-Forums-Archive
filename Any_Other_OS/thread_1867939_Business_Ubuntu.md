### Post by Penguinnerd on 2011-10-23
Well, while it's a good principle, I don't see the need for it.

Businesses can already use regular Ubuntu. I don't see the need for it to be "bare bones". If speed is an issue, they can use Lubuntu or Xubuntu.

Lack of options isn't a problem, as many distros as currently exist. The problem IMO, is publicity. Too many would-be Linux users don't even know about it, and if they do, they don't know about everything it can do.

When I mention Linux to people for the first time, they often don't understand that just because it can't run MS office doesn't mean you can't edit a word document with it.

We've got enough custom distros. We need publicity and stability.
