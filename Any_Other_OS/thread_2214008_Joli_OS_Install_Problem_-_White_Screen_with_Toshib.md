### Post by Tar_Ni on 2014-03-30
JoliOS has been discontinued in December 2013. See([http://www.omgubuntu.co.uk/2013/11/jolicloud-desktop-to-be-discontinued-december-2013](http://www.omgubuntu.co.uk/2013/11/jolicloud-desktop-to-be-discontinued-december-2013))

You can still download it on the Jolidrive website but there is no longer any updates and support.

I recommand that you try Lubuntu ([http://lubuntu.net/](http://lubuntu.net/)) or Peppermint OS ([http://peppermintos.com/](http://peppermintos.com/)) These are lightweight linux distro and works great for cloud computing. Your laptop will have some breath.
