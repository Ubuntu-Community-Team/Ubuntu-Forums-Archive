We're using Ubuntu on our computer, but we're looking for a distro that meets the following criteria:

1. Can make use of the Dropbox program for Linux
2. Is small. We don't need Gimp or LibreOffice as part of the ISO. 
3. Can easily access WiFi.

Yes.

4. Can be run live off USB. We don't want any permanent changes to our hard drive.

Yes.

From: [http://wiki.tinycorelinux.net/wiki:install_usb](http://wiki.tinycorelinux.net/wiki:install_usb)

Runs completely in RAM.  Saves and backs up settings @ shutdown to pendrive.
Full compile environment available.
Makes for a great kiosk OS, and runs well in virtual environments.
