### Post by 2F4U on 2012-08-14
I don't use Wubi, but in theory you can install the packages kubuntu-desktop, xubuntu-desktop, etc. to have all desktops on Ubuntu. The selection of one particular desktop then takes place on the login screen, where you can select the desktop to start. You don't need to create seperate user accounts.

Debian and Mint operating systems on their own (above are desktop environments) and you cannot install them in or on Ubuntu.
