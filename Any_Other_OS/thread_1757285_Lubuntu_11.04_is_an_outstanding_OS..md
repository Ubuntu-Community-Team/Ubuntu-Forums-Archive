### Post by tnmarktx on 2011-05-13
I'm really digging Lubuntu 11.04 and would recommend it to anyone.  Especially those looking for maybe an HTPC OS.

With the LXlauncher netbook interface that comes with LXDE, you can easily see applications while setting comfortable on your couch.  

Being able to save the default resolution is a big plus as well.  It's hard to see what's on the screen at 1920x1080, so being able to lower the resolution helps a lot.  Saving default resolution works on Ubuntu as well as Xubuntu.  It does not work in Kubuntu.

Ubuntu however, uses Gnome-screensaver which in my opinion is a broken program.  I don't know why this can't be fixed.  It works fine as long as you don't disable it because you are watching streaming vids over the internet.  Once you enable it will not work properly.  It's really ashame, because Gnome is actually a great desktop environment.  I think it has something to do with how the screensaver activates it fades out.

Kubuntu has a great netbook interface built in and looks great.  But, for some reason you can't save a default resolution and it is a pain having to change it every time you boot up.  Other than that KDE is great.  The KDE screensaver works great.

Xubuntu would probably be my second choice for HTPC OS at the moment.  You can save default resolution and it uses xscreensaver, which works great.  It does not have a netbook interface, which I think works great for HTPC.  It does have the nice dock on the bottom in XFCE 4.8.  This dock can be modified and works great.

Lubuntu is light on resources, has a netbook interface, uses xscreensaver, and will save default resolution, which pleasantly surprised me.  I just hope that as they become an official Ubuntu derivative in the future, that everything just continues to work well.  I think that were these desktop environments start becoming buggy is when they start putting more eye candy in.  I don't need the windows to explode or burn up when they are closed.  I don't need the screensaver to fade out gradually, I just need to to activate and then deactivate when I want it to.  I have no problems with things looking cool, just make sure that no functionality is lost.

Great job Lubuntu!

### Post by Bart_D on 2011-07-07
So I finally installed LUbuntu 11.04 on an older machine(previously running Ubuntu 10.10). The difference in speed is like night and day. It is blazing fast and RAM usage is reduced by nearly 50%!!! WOW!

Problems:
Screensaver: You can only select a blank screen. I could not select any others because a message came up saying that that particular screensaver was not installed. Huh?
System Monitor: I added it to the bottom panel....when I tried to launch it, it did NOT launch. I added a side panel (my setup looks like 8:01 from here [http://blip.tv/ubuntu-switcher/lubuntu-screencast-lxpanel-2-panel-configuration-5316669](http://blip.tv/ubuntu-switcher/lubuntu-screencast-lxpanel-2-panel-configuration-5316669)) and it successfully launched from there. Not sure why the panel allows system monitor to be added if it is not able to successfully launch from there.
Games: I could not remove games. It said that inorder to remove Games, another package must be removed. The name of that other package: Lubuntu-desktop. Obviously, I cannot do that so I am stuck with games.

Things that surprised me:
1. Removable storage(usb thumb drives)  - automatically detected. I really liked that I did not have to do anything weird to mount the removable drives.
2. Chromium comes with Flash. Chromium is much BETTER than Firefox.
3. Default Icons: Elementary. YESSS!!!

I think that LUbuntu was going for a very frozen look and feel. The developers have definitely acomplished this. It makes me feel kinda like I used to feel when I used Zenwalk.

OVERALL RATING: 9.5/10. There is not much that I would want to change. Brilliant distribution.
