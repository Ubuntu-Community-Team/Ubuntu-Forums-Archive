### Post by Bachstelze on 2012-02-20
I own a MacBook Pro and am very happy with it, best laptop I have ever owned by far. That said, the things that make the MBP so good are mostly thing that apply only to laptops (great bettery life, suspend-resume in about a second, awesome wifi managing app, a lot more reliable than my previous laptops, etc.), so I would never buy a Mac desktop. OS X is nice, but by itself it is not worth the additional cost of a Mac.

Note that I am a relatively experienced Linux user, so in terms of usability, Linux and OS X are pretty much equivalent to me. Others would consider OS X far more usable than Linux, which would make it worth the cost.
