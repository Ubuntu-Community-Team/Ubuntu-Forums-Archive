### Post by Pirate Zoro on 2012-11-02
I personally don't have any trouble at all with it. It's wasn't any harder to install than Arch was for me, even with building my own kernel (which is nothing really, as long as you know enough about your hardware and what drivers you need), and keeping it up to date is nothing more than syncing the repos and running an update. USE flags aren't an issue most of the time either, because most programs either A) don't have any specific flags to set, or B) have a general set a flags suitable for most generic system setups. And even if you should need to change flags, it's as easy as adding the program to a list with the flag. I mean, I can see how it can cause aversion to some users, but it's really not harder than a lot of things people are already doing, you just don't have a GUI. And even that's not entirely true either because there are quite a few interfaces you can use to change settings and install programs. Once I got it up and running, it's been the easiest system I've ever had to maintain.
