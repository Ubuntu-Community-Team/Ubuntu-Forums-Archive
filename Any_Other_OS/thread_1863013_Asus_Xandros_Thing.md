### Post by drawkcab on 2011-10-17
Xandros is horrible.  One of the problems that you'll run into is that the 4gb ssd will max out with updates to xandros.  Its only advantage is that it works with the eeepc boot booster.  If you want to learn more about it the eeepcuser forums are where to go but I recommend wiping it and installing something else.

If I were you, I would build something up from a minimal install of Ubuntu or Debian in order to conserve on space and keep the overhead low.  Debian + lxde or xfce would be a great choice.

On the other hand, I upgraded my ssd to a 16 gb model.  You can also install the OS to a 8-32 gb class 6 flash memory card and use the 4gb memory as storage.

### Post by Plumtreed on 2011-10-17
You really need to move on from Xandros.......Fuduntu works exceptionally well on these eeePCs. It has Jupiter included which extends battery time and offers additional controls.

It may appear to be too big for your 4GB but installs to under 4 and there are easy ways to 'chop' it back to about 3GB....outlined in Fuduntu Forum.

All the usual eeePC things, Wifi, USB broadband seem to work OTB!

It has a small 'Docky' setup to provide access via icons to your favourite applications instead of searching through tiny menus.

It installs Chromium but you can add Firefox after the install....and most other things that your drive space will permit.
