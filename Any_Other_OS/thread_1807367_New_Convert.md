Very right.

Most Linux versions (including Ubuntu and Mint) have a package manager (in Mint the "software manager", in Ubuntu the "software center" ...). Those package managers give you a really huge amount of applications. In most cases, all you need is in the package managers. It's comparable to the iPhone app store or the Android Market, but Ubuntu has it since it's foundation in 2004.

As an example, I remember someone asking for a replacement for the Nero Burning suite. This really is like asking "How would I do it in Windows". He could not find one, until he thought what he does with Nero.
[LIST]
[*]He burns disk. If you search the software center for a CD or DVD burner app, you get 2 very popular results: Brasero (preinstalled) and K3b. It's a matter of taste what you like.
[*]He makes covers. Again, if you search the software center, you will find a cover editor.
[*]He edits music, if you search for a music editor, you will find some popular apps like Audacity
[*]he rips CDs. Again, you will find CD rippers in the software center
[/LIST]

So as you see, you don't have to search a replacement for a suite or program, but you have to search what you want to do.

Apart form that, Mint is a good choice. And do trust the installer. It is made by normal people that know what they are doing, not by big companies that think they know what the user wants.

### Post by sanderd17 on 2011-07-21
You may format your disk, but you don't have to do it. The installer will take care of it. So only do this if you really know how much space you need for what. I started manually editing partitions after two guided installations.

If you follow a guided install, the installer will shrink the Windows partition and install itself there, or if you have some empty part of disk (not yet formatted) it can also be used by a guided installation.

So if you don't have a lot of experience, just put the CD in, start the installer and choose one of the guided partitioners.

In any case, always backup all data. When the power gets interrupted while shrinking a partition, all data will probably be lost. The same with any kind of other severe crash. Or if you do go for a manual installation and select the wrong partitions, all data will also be lost (I have done this once).
