### Post by Linux_junkie on 2011-12-04
I had spent most of these weekend replacing Ubuntu 10.04 on my laptop with PClinuxOS phoenix edition.  The installation went well with no problems however after the installation I then decided to update it.  Once again all went well with downloading the latest versions of the packages but for some reason after it had downloaded the packages it refused to install the updates.

After trawling through the PClinuxOS forums I had finally discovered that if you hadn't updated within 30 days it told you to download a fresh iso.  Now I had originally downloaded the iso over a month ago and had kept it in a drawer all this time.  I am a bit annoyed that the way the distribution is set up that you cannot update it if over 30 days old.  As I only have a s-l-o-w mobile broadband device with limitations I did not fancy downloading a fresh iso file so have come back to Ubuntu 10.04.

For all you people wanting to try PClinuxOS yourselves all I will say is install it and update it within 30 days of you getting the iso.
