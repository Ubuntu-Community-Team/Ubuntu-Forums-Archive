### Post by zhogan85 on 2011-10-07
Well, Gabe Noob, I recommend reading the arch wikis and beginners guides thoroughly, then popping the install disk in and taking the plunge.

[https://wiki.archlinux.org/index.php/Beginners%27_Guide]("https://wiki.archlinux.org/index.php/Beginners%27_Guide")
[http://lifehacker.com/5680453/build-a-killer-customized-arch-linux-installation-and-learn-all-about-linux-in-the-process](http://lifehacker.com/5680453/build-a-killer-customized-arch-linux-installation-and-learn-all-about-linux-in-the-process)
[https://wiki.archlinux.org/index.php/Openbox](https://wiki.archlinux.org/index.php/Openbox)


I recently installed arch myself, mainly to have a system to play around with that really facilitates learning. The beginner guide is great, just be sure to read it. The lifehacker guide is a nice supplement to it as well. Once your familiar with the guides, go for it. If you have a second computer, it wouldn't hurt to have them pulled up on it during your install if you need to reference them for troubleshooting.

Don't be scared, go for it! And report back!

Heres a link for installing on a macbook, which should be similar.

Just backup your stuff and go for it. Worst case scenario you screw something up and have to try again (half the fun really).

If Arch is too intimidating, maybe try a Debian net install. This will only install the basics to get the system running and left at the command prompt. The installer itself it pretty easy to get through. From there you can install openbox and all the applications you need through apt. This way you will end up with a custom system, with only the stuff you need.

Either way, just dive into it and have some fun. You will learn a lot doing a custom install like this.
