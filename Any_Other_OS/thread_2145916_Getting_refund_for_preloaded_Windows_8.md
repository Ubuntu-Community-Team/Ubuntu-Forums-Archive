### Post by DeathByDenim on 2013-06-13
I realize this thread is a bit old, but just in case the question is still relevant: I just bought an Acer Aspire V3-771G laptop with Windows 8 preinstalled. I then followed these instructions from the Acer website: [http://acer.custhelp.com/app/answers/detail/a_id/280/kw/windows20refund](http://acer.custhelp.com/app/answers/detail/a_id/280/kw/windows20refund)
This was the reply I got:
> Dear xxxxxxxxxxxxxxxxxx,

Thank you for contacting Acer America. The refund amount for the version of Windows 8 ML on your system is $61.90US. You will need to ship the system to our repair depot and I will provide that address in a later reply. I need to register the system to you in order to set up the repair. You will need to include any recovery media you have created with the unit. Please provide your street address and phone number, business name if applicable. 


Respectfully,
Acer America 
Online Technical Support

I hope this information is still helpful.
