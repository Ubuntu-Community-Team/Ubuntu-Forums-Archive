Greetings everybody! I am new to this forum and what better way to get right into business than to make a post asking about Hannah Montana Linux? I can't think of a better first post at least!

Anyhow. I have recently discovered the masterpiece that is [Hannah Montana Linux]("http://hannahmontana.sourceforge.net/"). The problem is that it hasn't been updated since 2009 and, as a consequence, it is highly out of date. In fact, since it is based upon an old version Kubuntu whos supported days have passed, it is no longer possible download programs from the repositories.

However, since the creators of the distro have made the themes available for download (see [this page at KDE-look]("http://kde-look.org/content/show.php/Hannah+Montana+Linux+1.0+icons?content=146854")), I thought that it might be possible to import them into Kubuntu 12.04 and thus make a "new" version of Hannah Montana Linux.

However, since I am new to this, I have no idea how to proceede. I thought that it would be simple enough to import the theme under the System Settings menu, but when I tried I got am error message stating that "The file is not a valid icon theme archive.". Perhaps there is a need to compile something from source, but I am not familiar on how to do such a thing nor does it appear as if the tarball contains all the needed files to do such. (Then again, as I have never compiled a program, I am not certain by any means!)

My question is thus the following: Is it possible to import the themes into Kubuntu 12.04, and if so how? If not, is it possible to connect to the (old) repositories with the old version of Hannah Montana Linux?
