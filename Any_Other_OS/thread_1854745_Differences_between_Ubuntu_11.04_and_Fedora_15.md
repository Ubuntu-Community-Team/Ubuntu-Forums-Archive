### Post by ojdon on 2011-10-05
One of the most simple differences between the two is that they have different interfaces. Ubuntu 11.04 uses their own unique interface, Unity where as Fedora 15 uses Gnome-Shell as their primary interfaces.

### Post by XubuRoxMySox on 2011-10-05
Fedora is bleeding-edge, latest and greatest kinda stuff. The most recent Ubuntu release is as well, but Ubuntu offers the choice of the **long-term-support** (LTS) versions, which generally remain more stable but use older software. Fedora does not offer LTS versions. It's still possible to use older versions of Fedora (as it is in Ubuntu) for more stability, but they are not supported with updates and such for 3 years like the Ubuntu LTS releases are.
