### Post by bug67 on 2013-01-31
1.  [Click on this link]("http://www.linuxmint.com/release.php?id=19") and download the applicable distrobution to a working computer.  I'm assuming Windows.  This is the latest version of Linux Mint, 14 (Nadia).  If you don't know which is applicable, the 32 bit one will work.

2.  After the disk image file has downloaded, plop a blank DVD into the Windows machine.

3.  Right click on the disk image file you downloaded and select "burn image to disk."

4.  After you've burned the disk, put it into the machine you want to install Mint onto and hit the "On" button.  The machine *should* automatically boot from the Mint DVD.  If, not, you're going to have to go into the BIOS and set the machine to boot from your DVD drive.

5.  Once the Mint O/S has fully booted from the DVD, select "Install Linux Mint" and follow the on screen prompts.

**_EDIT_**:

Just re-read that the O.P. wanted *Linux Mint 13* and had already put it on a *thumb drive*.
I've never installed from a thumb drive but I'm assuming the only difference is setting the BIOS to boot from the thumb drive instead of the DVD drive?  In any case, the BIOS automatically looks for the first bootable device it can find so, sans HDD and without a disk in the DVD drive, USB is next in line.  

Actually, [in the 4th post in this thread]("http://ubuntuforums.org/showpost.php?p=12479691&postcount=4"), I linked to all the instructions on how to do all this.
