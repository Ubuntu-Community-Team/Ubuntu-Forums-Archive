### Post by oscarivera9 on 2011-09-22
I agree with kaldor, try Ubuntu Studio.  It's what I use, but for different reasons.  I am a musician and it's got tons of apps that I need and/or want, and a bunch more that little by little I'm learning about.  I really like it, and in reality it boils down to the fact that it's still just Ubuntu but with lots of cool stuff.
Since there is no Live CD to try it out (there is just an install only DVD), what I recommend is to do an upgrade from Ubuntu to Ubuntu Studio through the terminal.  When the change has been made it will still feel like Ubuntu anyway.  If you decide that you really don't like it and you want to get those extra 3 GB of hard drive space back, then you can revert to a plain vanilla Ubuntu through the terminal as well.  
Here's the link to do the upgrade:
[https://help.ubuntu.com/community/Ubuntu%20Studio%20Upgrade%20from%20Ubuntu](https://help.ubuntu.com/community/Ubuntu%20Studio%20Upgrade%20from%20Ubuntu)

Only way to know if you like it is to try it.
