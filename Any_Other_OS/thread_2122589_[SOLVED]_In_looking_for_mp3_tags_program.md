### Post by slickymaster on 2013-03-05
[MusicBrainz Picard]("http://musicbrainz.org/doc/MusicBrainz_Picard"). It supports the majority of audio file formats, is capable of using audio fingerprints (PUIDs, AcoustIDs), performing CD lookups and disc ID submissions, and it has excellent Unicode support. Additionally, there are several plugins available that extend it's features.
