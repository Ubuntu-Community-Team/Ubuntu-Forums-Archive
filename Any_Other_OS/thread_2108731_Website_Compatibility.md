### Post by SeijiSensei on 2013-01-25
In Firefox, you can install the [User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/user-agent-switcher/") add-on to make the browser "lie" about itself.  You can select which user-agent header the browser sends.  Often this resolves problems with brain-dead websites that do not know how to handle Linux browsers or expect everyone to be using Internet Explorer.

I don't know how this problem is handled in other browsers like Chrome.
