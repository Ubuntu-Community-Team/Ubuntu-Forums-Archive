### Post by doas777 on 2011-08-12
speakers are analog tech, so whatever you send to them will get played. since the source is digital however, you can edit the output experience with soundcard driver software, introducing features like noise reduction, mixing, redirecting of channels, HDA de/encoding, etc. if you don't need any pre-processing features however, then generic soundcard drivers and speakers  will work fine out of box.
