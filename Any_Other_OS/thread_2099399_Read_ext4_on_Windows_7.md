### Post by flavouride on 2012-12-29
[http://www.diskinternals.com/linux-reader/](http://www.diskinternals.com/linux-reader/)

DiskInternals Linux Reader is the one I used on Windows 8 for reading the pictures and movies I stored in my ext4 (with extended on) external HDD.

You can use this program to save a copy of those files to your NTFS/FAT partitions as well.

No reboot was required and it's ready to start right after installation.
