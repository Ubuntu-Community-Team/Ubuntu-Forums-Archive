### Post by LeGasp on 2012-05-25

> I've actually only done one netinstall, is there much involved if you're doing a netinstall on a laptop with wireless, to get the wireless setup?

I always hook up ethernet when doing the netinstall, but if you know what module your wireless card uses, you could probably alt+f2 into another terminal, load your module with a "modprobe [module]" command, and use the Arch wiki to set up wireless from there. Sounds like a pain, so I'd go with a temporary wired connection.
