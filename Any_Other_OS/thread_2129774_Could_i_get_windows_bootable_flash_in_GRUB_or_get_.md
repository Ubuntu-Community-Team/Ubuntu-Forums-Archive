### Post by oldfred on 2013-03-27
Moved to Other OS since Windows question.

Does system support booting from USB? If not it is just about impossible as then you are supposed to use CD or floppy drive to boot externally.

I only know of plop that will boot Linux from USB actually using CD to start.

Many years ago when Windows first came out on CD, we used to copy the install files into the Windows partition. We would format drive, install DOS to boot, and start install into the same files that Windows install files were in.

You might be able to create a small install partition and copy your Windows installer into that using your brothers computer. Then you may be able to boot the installer on the hard drive and install? Never tried that with XP.
