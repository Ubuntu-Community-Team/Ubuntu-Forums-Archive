### Post by Bucky Ball on 2014-08-03
Two ways I can think of:

1/ Boot into your machine, open Firefox, click the logout icon, tick the box that says 'Save Session', reboot. Firefox will launch on reboot and there's your desktop, and secondly;

2/ Install a Ubuntu minimal install then a lightweight desktop and Firefox. I believe there are ways to simply install Firefox into a minimal install and launch that from the CLI on boot (or write a script to do it for you).

Researching option 2 would be my choice.

Good luck.
