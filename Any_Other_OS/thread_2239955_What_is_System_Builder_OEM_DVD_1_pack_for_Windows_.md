### Post by 3rdalbum on 2014-08-17
1. Yes, this is the full version. "System Builder" and "OEM" mean that this software is ONLY to be installed on a brand-new PC that you have built yourself:

*&#8226; OEM software is intended for system builders and small OEMs (Original Equipment Manufacturers) who manufacture computer systems and pre-install OEM system builder software onto those systems. *

If you have bought your PC from someone else and it did not come with Windows, or it came with a different version, then you are not permitted in the EULA to use this.

The OEM/system builder version of Windows is the same as a regular boxed copy of Windows, except that the "upgrade from an earlier version" function is disabled for obvious reasons.

2. Yes, you can. The Boot Repair software mentioned on the Ubuntu Wiki will be able to reinstall GRUB for you.

3. All DVD drives in computers are dual-layer.

4. If you can run 64-bit Ubuntu you will be able to run 64-bit Windows.
