### Post by westie457 on 2011-12-12
Do you mean put the Windows install ISO on a pen-drive? 
Yes that is possible.

If you mean fully install Windows to a pen drive to use as a portable operating system might work however it will probably break each time it is used because of the hardware changes from one machine to another. At best it will hang and do nothing or you will get the BSOD because of the hardware changes.
