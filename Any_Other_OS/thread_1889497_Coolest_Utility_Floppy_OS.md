### Post by Dr. C on 2011-12-03
Yes a 5.25in floppy earlier today. I used it to install an 18 year old Windows 3.1 application on a Windows 7 32bit Virtual machine running on my new laptop. 

The laptop is an HP Probook running Ubuntu 11.10 does not have any floppy drives but it does have Wireless N. I use an older machine on my network running Windows NT 4.0 that does have a 5.25in floppy drive and simply share the floppy as an smb network share. On the Windows 7 VM the shared floppy is mapped as B: One then installs the software as per the original 18 year old manufacturer's instructions. The only difference is that one is swapping floppies on a different computer from the one on which the software is installed on.
