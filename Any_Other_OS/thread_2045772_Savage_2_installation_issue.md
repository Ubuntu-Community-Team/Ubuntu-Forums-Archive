### Post by thatguruguy on 2012-08-21
FWIW, if anyone running Ubuntu gets the preceding error messages, they can be safely ignored. The game can be run by going into the directory you set up while installing the game and typing the following:
```
./savage2.bin
```

Or, you can create a .desktop file for it, if you're into that kind of thing.

Or, you can install Savage 2 through Desura, if you have a Desura account.
