### Post by el_koraco on 2011-09-11
why fiddle with other distros if all you want is to play music? Go to a console, run a 

```
sudo service gdm stop
```

or if you wanna go all out

```
sudo telinit 1
```

and use something like MOC or any of the numerous mpd frontends to listen to music.

### Post by aura7 on 2011-09-12
Try using CMUS media player... it is a console based media player for linux. 

[http://cmus.sourceforge.net/](http://cmus.sourceforge.net/)

Log in to Ubuntu using console only and play music with it. I mean don't do normal Gnome 2 or Unity based log on to ubuntu. Simply log in using console and play music.

Loggin using console will surely reduce your power consumption.
