Outdated is subject to opinion. If you want bleeding new you can get that on Debian easily. Older well tested software is hardly outdated. Especially considering how long people (still are) using Windows Xp. Something a year out of time isn't so bad. (and you get security fixes too).

You need to do this so that you won't ran out of virtual drive space. The more memory you have, the better. I recommend using 1GB plus for your personal storage. If you like to use a dedicated drive or partition for this one, then that would be better, than running it on NTFS. Just install it on it using the "Install" button that is on the desktop. After you do that, reboot so that the new storage space will be active.

Then, use the "Connect" button to choose the type of connection you like. After you're successfully connected, Use "Quickpet" to install your preferred Browser from the list, under "Internet Pets". The Browser chooser is next to the "Menu" button. 

After that, you may play with some other utility software available under Quickpets.  Puppy already comes with many utility programs like Word Processors, Pdf Reader, Multimedia players, CD/DVD/Blu-ray writer, etc. I recommend downloading VLC for it's multi-format capability. Get it under "Useful Pets", under Quickpets. 

Well, hopefully this steps help you enjoy your Lucid Puppy test drive. Please let us know how you're doing with your test. Pls. don't hesitate to ask me questions.
