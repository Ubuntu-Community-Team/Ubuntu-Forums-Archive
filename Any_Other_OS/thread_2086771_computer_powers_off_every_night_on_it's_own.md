### Post by Yougo on 2012-11-22
does it shutdown properly or is it like a power cut?

i can't really think of any native windows7 setting that allows for scheduled shutdown, let alone keeping it locked down for an amount of time. that sounds more like BIOS stuff, or at the very least 3rd party software, like some parental control stuff

did you buy it fresh off the shelf, or is it second hand?

you could try booting off a live usb/cd. if it stays on, then it's some software setting on your windows install. 
if it crashes/cuts power, then it's a likely hardware problem
if it still shuts down, it's likely a BIOS setting
