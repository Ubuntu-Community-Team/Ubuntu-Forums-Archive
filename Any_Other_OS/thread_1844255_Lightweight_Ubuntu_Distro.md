### Post by XubuRoxMySox on 2011-09-16
> **galacticaboy said:**
> How would I go about installing Ubuntu CLI only?

Follow [this step-by-step guide]("https://help.ubuntu.com/community/Installation/MinimalCD").

I used the minimal installation CD to create the LXDE mixture that saved a reeeeeally olllllld computer at the dance studio from the landfill. It was so quick and easy to use that it actually won two dance families over to Linux!

A small hardware upgrade about a year later and it's running Xubuntu now. Both LXDE and Xfce (*not* "lubuntu-desktop" nor "Xubuntu-desktop," just "LXDE" or "Xfce4")  are nice and speedy on a minimal Ubuntu installation. 

After the minimal Ubuntu install,

```
apt-get install openbox
```

followed by your favorite applications. Like Synaptic package mgr, an ultralight file manager like PCManFM; a quick, versatile, lightweight browser like Midori; maybe a lightweight e-mail client like ClawsMail, your favorite utilities, Abiword, Gnumeric, etc.

Install only what you want, choose lightweight apps, enjoy Openbox's speed and amazing ease (right-click the desktop and off you go!). Crunchbang Forums has some wicked cool scripts (also see the Ohms Project) to make your amazing Ubu-based Crunchbuntu!
