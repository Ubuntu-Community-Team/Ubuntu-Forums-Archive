### Post by Helen McCall on 2012-03-19
The Debian system and community is the most strongly established. The Debian package management system Apt/dpkg has become the defacto standard package system.

My choice would always be for a Debian based system - either Debian stable or the current Ubuntu which is based on Debian.

I have been using Debian based systems for all my computing including scientific research and mathematics etc since 1996. I can definitely recommend either Debian or Ubuntu.
