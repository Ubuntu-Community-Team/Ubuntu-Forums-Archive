### Post by ecitoyen on 2012-10-28
Windows 8 Pro & Ubuntu 12.04: On The Same PC?

Is it Possible to get on a same PC, the
Ubuntu 12.04 linux OS and the new One 
Windows 8 Pro?

Many linux users think that it is not
possible 'cause the new launcher in
Windows 8 Pro.

I will explain how i get it!

First my Pc is a Dell Optiplex 740,
with a first HDD 80Go running Windows 7 Pro.
I add a new HDD 500GO with 2 partitions.
The First partition of the second drive
was for Windows 7 Pro. The second was
reserved for Ubuntu 12.04. Then I installed
on the second drive the Ubuntu 12.04 OS.

When i start the PC... 2 Bios options
appeared: 1- Setup with F2 Key
          2- Boot menu with F12 Key

If i want to run Windows 7, i did nothing...
If I want to run Ubuntu, i press the F12 Key
and select the second drive and Ubuntu
12.04 was launched.

After, and it was the tricky thing...
I downloaded The Window 8 Pro OS and
installed it at the place of Windows 7 Pro
saving all programs, parameters and files...

and Restart... a couple of hours later...
and it was a miracle... at the bootstrap
step the 2 bios options was already displayed
and i get the choice in between Windows 8 Pro
(by doing nothing) or Ubuntu 12.04 by Pressing
the F12 key(Boot Menu).
