### Post by golamc on 2011-11-22
I  decided to present my project even on the forum to have another little  room for discussion on the topic and to gather feedback and opinions as  much as possible for the improvement and development of UbuBox SalentOS. :D

Description:

UbuBox SalentOS is the first operating system GNU / Linux inspired by the Salento.

This operating system is suitable for all those looking for durability, ease of use and high performance!

Simplicity and durability are guaranteed by the Ubuntu base


**THE WINDOW MANAGERS USE:** The high performance is guaranteed by Openbox, the default desktop manager UbuBox SalentOS. **Openbox**,  in fact, is known for the paucity of hardware resources required to  function at its best so it is also suitable for older computers or  devices such as resource-hungry netbook. **Docky and Tint2** completes the desktop. The operator of compositing (shadows, fades and other small effects) is the light **xcompmgr.** 
[IMG]http://gmstyle.org/images/UbuBox_SalentOS/1110/forum.png[/IMG]

**Minimum system requirements**

800mhz processor i386
512MB of RAM
4GB hard disk space


**Pre-installed software**

**ACCESSORIES:**  calculator, Leafpad (Text Editor), Disk Usage Analyzer (baobab), screen  capture tool (xfce4-screenshooter), Notes (xfce4-notes), file search  tool (Catfish), grstore files (file- roller)
**AUDIO  / VIDEO:** Cheese, desktop recorder (recordmydesktop), EasyTAG (edit mp3  tags), Exaile (music player), Words (multimedia player), VLC, Xfburn (CD  / DVD)
**GRAPHICS:** Gimp, Xsane, Ristretto (image viewer)
**INTERNET:**  aMule, Sylpheed (email client), Firefox, Pidgin (IM accounts manager),  Gwibber (social network account manager), FileZilla, Liferea (manager  feeds), Uget (download manager), qBittorrent.
**OFFICE:** Suite LibreOffice
**TOOLS OF THE SYSTEM**: Ubuntu Tweak, Unetbootin, Remastersys Backup, Gmrun.
**CODEC AUDIO / VIDEO** and FLASH PLAYER installed for proper playback of media files!


for more info refer to the **official website (in italian but can use google translate)** [http://www.ububoxsalentos.sourceforge.net](http://www.ububoxsalentos.sourceforge.net)
