### Post by jwbrase on 2012-11-09
> **irv said:**
> Android is an OS. Ubuntu is an OS. You can run Ubuntu on Android devices now, but Ubuntu has to be able to handle touch screen better, and work well with other types of processors.

"Ubuntu for Android" isn't simply taking a phone that comes with OEM Android and installing Ubuntu on it instead. It's a project by Canonical that runs on top of Android and is meant to provide a proper desktop environment when the phone is docked (as opposed to most/all existing dockable phones that just display their touch interface when docked).

It doesn't need Ubuntu to handle touch any better than it does, because with UfA Android handles the touch side of things, while the Ubuntu environment is meant to handle the Keyboard/Mouse/Monitor side of things.
