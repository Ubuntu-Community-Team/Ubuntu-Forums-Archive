### Post by philinux on 2014-03-21
Pinguy is an Ubuntu remaster that uses GNOME Shell by default, customized with extensions such as Gno-Menu, Media Player Indicator, Top Icons and more. It comes with many useful applications and tweaks by default, making Pinguy OS a great alternative for those who don't like to tweak or install applications but want everything to work out of the box.*
