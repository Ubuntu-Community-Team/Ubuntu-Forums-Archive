### Post by andrew.46 on 2012-01-01
In the spirit of KiwiNZ's [Discerning December Screenshots]("http://ubuntuforums.org/showthread.php?t=1889046") this thread is for screenshots from distros *other* than Ubuntu. I would suggest adding the following details:

[LIST=1]
[*]**Distro**: Specify the distro and version
[*]**Environment**: Name the Window Manager and/or Desktop Environment
[*]**Wallpaper**: Give a link to the wallpaper being used
[*]**Applications**: Name any applications displayed
[/LIST]

I would also suggest, as KiwiNZ has done, that you keep the screenshots to 800 x 600 so those with reduced bandwidth do not suffer. And please keep it clean :).
