### Post by Haghiri75 on 2013-01-11
Hi All. 

We (JT Foundation ) already created a linux distro (JabirOS) and it's based on Ubuntu. 

Now I decided to make JabirOS independent. We have many candidates for base idea , like slackware, gentoo, archlinux, .... . 
 
Ok. We want a distro with this features :

1- Easy to use and Install ( for everyone!)
2- Graphical Interface for Package Management ( I already write many scripts for a good package manager, and we call it JetGet :) ). 
3- Always up to date. 
4- non-rolling. 

OK, do you have any ideas to make Jabir Technologies Foundation (JT Foundation) Products better?

Regards.
Muhammadreza Haghiri (CEO, Founder).
