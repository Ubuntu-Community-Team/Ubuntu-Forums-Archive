The only way to sorta do this with LXTerminal is to assign a geometry command to the launch string. In other words say, hey do this when you open.

In LXLE I included a working OBKey editor located in Menu>preferences

Open it and find ctrl + alt + T and highlight it

replace the command string with "x-terminal-emulator --geometry=150x45" without the quotes

click save and test.

adjust the 150x45 to the numbers that suite your screen best.

### Post by sudodus on 2013-03-27
There is still another soft way to reboot a linux system, and avoid the risk to damage the file systems: SysRq commands.

While pressing

[I][B]alt + SysRq

[/B][/I]all the time, press the following keys one each time for approximately one second

***r e i s u b***

Usually SysRq is on the Print Screen key, but it is located on other keys on some keyboards. You can find information about it in this link

[[COLOR=#ff0000]https://en.wikipedia.org/wiki/Magic_SysRq_key[/COLOR]]("https://en.wikipedia.org/wiki/Magic_SysRq_key")

### Post by ronniew on 2013-04-05
Fehlstart, I believe, is a relatively unknown little gem. A real beauty if your looking for nice but truly lightweight launcher. Fehlstart is a German word meaning “false start” also defined as an “early start”. The author wrote Fehlstart because gnome-do and others do way more than what he needed; gnome-do runs on mono which requires too many resources to perform a simple task like application launching.

[SIZE=4]
0. Generally LXLE is very nice out of the box,[/SIZE]

and particularly for old computers, it works better than the newer versions of Lubuntu (due to the hardware drivers). Now that Lubuntu 12.04 is approaching end of life, I think ***LXLE*** and ***PreciseGnomeClassicTweaks*** by *kansasnoob* remain as the best Ubuntu flavour alternatives for those old computers, where the hardware has been abandoned by newer Ubuntu versions. ***Xubuntu*** is also good but has only 3 years LTS (until April 2015).

[SIZE=4]1. Downloading
[/SIZE]
- Is there any ***torrent*** for LXLE?

- phillw.net/isos/lxle/precise

Please refresh or add point versions at [phillw.net/isos/lxle/precise]("http://phillw.net/isos/lxle/precise"), because this site is very fast and reliable.

[http://sourceforge.net/projects/lxle/files/Current/32Bit/](http://sourceforge.net/projects/lxle/files/Current/32Bit/)

- I found that the following commands work

```
[s]wget -O lxle1204-32.iso sourceforge.net/projects/lxle/files/Current/32Bit/lxle1204-32.iso/download[/s]
wget -O [s]lxle1204-32.iso.md5 sourceforge.net/projects/lxle/files/Current/32Bit/lxle1204-32.iso.md5/download[/s]

```
*Edit*: After the update the commands are changed to
```
wget -O lxle1204-32.iso http://sourceforge.net/projects/lxle/files/Current%20Release/32Bit/lxle-12.04.03-32.iso/download
wget -O lxle1204-32.iso.md5 http://sourceforge.net/projects/lxle/files/Current%20Release/32Bit/lxle-12.04.03-32.iso.md5/download
```

### Post by ronniew on 2014-01-07
Seems to be and actually knowing can be two different things. LXLE is an LTS because as a community we are choosing to support it for the same duration as Ubuntu. Is it an official LTS release from canonical? No. Are updates updates and support provided? Yes. We manage our own [PPA for lxle]("https://launchpad.net/~lxle/+archive/stable") in which we keep system components and other software up to date, and provide [support and documentation]("http://lxle.net/support-options/") through our forums and wiki. LXLE is also utilizing a number of underlying technologies that are different than lubuntu, addresses problem areas, reconfigures backend processes for better memory management, etc. 

The PPA's / Repositories that are enabled are our's and the developers direct repository or PPA such as LibreOffice, VLC, Openshot, etc. Main repo's or PPA's were only used to ensure their continued updates. Medibuntu shutting down came as a surprise to everyone since it had been a stable repository for so many years. Regardless removing the repository was addressed immediately and is something every user on any system sometime will have to learn.

As for 'junk' preinstalled. LXLE is setup with the idea of being a drop in and go OS, meaning that you could really install it on any PC, particularly an older system and be relatively done. This comes in handy if either you just want to do a quick install for either yourself or someone else, or your a brand new linux user trying to get a few more years out of your old computer.

The process of selection of software had a lot of thought put into it, trying to find balance between easy, feature rich apps combine with a lightweight system. Each major category of normal computer use was address with at least one capable easy to use app, preferably a lightweight one. The software selection and setup has received a positive response with numerous publications and partnerships in it's relatively short life such as the following

[DistroWatch Review]("http://distrowatch.com/weekly.php?issue=20130909#feature")
[PCWorld Magazine]("https://www.pcworld.com/article/2031825/have-an-older-pc-try-the-new-ubuntu-linux-based-lxle.html")
[Video Reviews]("https://www.youtube.com/results?search_query=lxle&sm=3")
[Our Reviews]("http://lxle.net/reviews/")
[More Articles]("https://encrypted.google.com/search?hl=en&q=lxle")
and
[Linux Format Magazine]("http://www.linuxformat.com/archives?issue=177") - Distributed LXLE DVD on the cover of their december issue.

As for DDG. It's an excellent search engine which is private and doesn't track you. It's a [search engine that Richard Stallman]("https://duck.co/help/desktop/linux-distributions") uses, it's [almost entirely opensource]("http://duckduckhack.com/"). They regularly contribute code and [funds]("http://www.gabrielweinberg.com/blog/2010/11/help-me-start-a-foss-tithing-movement.html") to opensource projects. Gnome, Firefox, Ubuntu, Debian and about [50 other well known distros]("https://duck.co/help/desktop/linux-distributions") and browsers come with DuckDuckGo as an option and most of those distro's configure DDG as the default search engine, such as Debian, Linux Mint, Peppermint, Gnome Web Browser and Midori just to name a few. So as far as what search engine I tend to use and promote is pretty obvious given their opensource track record, [privacy policy]("https://duckduckgo.com/privacy"), and acceptance within the opensource community.

If LXLE isn't something that you really care for all that much for whatever reason, it's always a really nice thing to have so many other distro's available to choose from, there is a distro out there for everyone.
