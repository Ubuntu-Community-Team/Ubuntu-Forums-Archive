### Post by NightwishFan on 2011-08-18
Perhaps try without the vh? From Debian RPM manpage:
```
       The general form of an rpm install command is

       rpm {-i|--install} [install-options] PACKAGE_FILE ...

       This installs a new package.

```

So:
```
rpm -i nameofpackage
```

Or even in case it doesnt like the grouped options:
```
rpm -i -v -h nameofpackage
```

And yes you will probably have to run the command as root/sudo.
