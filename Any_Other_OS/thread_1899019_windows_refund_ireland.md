### Post by BC59 on 2011-12-22
First things first, 
If you like to install Ubuntu don't forget to install Jupiter to control the consumption of your battery:
```
sudo add-apt-repository ppa:webupd8team/jupiter
sudo apt-get update
sudo apt-get install jupiter
```
and for thr Asus EeePC netbook, also install jupiter-support-eee for SHE (Super Hybrid Engine):

```
sudo apt-get install jupiter-support-eee
```

About the refund, I didn't know that!
