### Post by AdamWill on 2012-02-29
Quick bit of advice - if you're going to try out F17 Alpha, please do read the [Common Bugs page](https://fedoraproject.org/wiki/Common_F17_bugs), it's not one of those Alphas which seem to work almost perfectly already, there are some fairly large bugs and it's really a good idea to be aware of 'em going in. Especially if you're reading these forums - one of the known bugs is that the installer crashes if you try to install to a machine which has another OS with a /etc/fstab file but no /etc/redhat-release file, which includes Ubuntu. So I suspect a lot of you might run into that one! The page explains how to work around that and other bugs. So read it. :D

gnu/mirow: you're very unlikely to see much OOTB customization of GNOME in Fedora, for a few reasons - most importantly a) Fedora's fairly strict policy of staying close to upstream and b) the fact that most of Fedora's desktop team consists of people who are also upstream GNOME developers.
