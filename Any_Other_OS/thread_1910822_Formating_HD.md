### Post by Bucky Ball on 2012-01-17
Windows XP formats the drive (partitions) as it is installing. You do it as part of the install. Not clear whether you are already running Ubuntu but if so, boot from the Live CD, open Gparted (Partition Editor), delete all partitions creating a blank disk with nothing but free space. That's it. Install XP.
