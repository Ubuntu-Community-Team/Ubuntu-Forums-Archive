Running Mint 13 (Ubuntu 12.04) I'd thought it was a Mint problem.  Got the speckles in Chrome, but not Firefox, so I advanced to Chrome Dev version 27 without luck.   I should say that running Chrome Stable on my old Lubuntu 10.10 netbook does not cause any problems and never changed any plugin values (apart from disabling Java.)  So this seems to be incompatibility between Flash and recent Ubuntu.

Once that's clicked it plays the video very well.  My thanks to the suggestion about pepperflash, but it's obviously a bug that still needs repair.
