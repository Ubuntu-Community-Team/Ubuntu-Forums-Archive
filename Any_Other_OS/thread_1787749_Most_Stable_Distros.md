### Post by aliancemd on 2011-07-16
I don't recommend: Slackware(very old and hard to install for novice), RedHat/CentOS/Scientific Linux(all 3 intended for servers and not for Desktop).

I do recommend: Debian Stable(very stable but not a lot of fun because of old software), maybe openSuse(the only distro which integrates KDE as it has to, the only distro on which KDE never crashed for me) and Ubuntu.

I prefer Ubuntu cause it is very stable and has a bunch of software and support([http://askubuntu.com/](http://askubuntu.com/)).
I don't recommend Ubuntu 11.04 though, it's kind of an alpha version. I do like 10.10([http://releases.ubuntu.com/10.10/](http://releases.ubuntu.com/10.10/)), never crashed for me(don't use Synaptic Package Manager please :) ), but, if u r all about stability you could download Ubuntu 10.04 LTS but you will have a bit old software in it.

**Short: I think Ubuntu 10.10 is the best choice(**[http://releases.ubuntu.com/10.10/](http://releases.ubuntu.com/10.10/)**)**.
