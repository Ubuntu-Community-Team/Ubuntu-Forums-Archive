If you install cryptkeeper (encfs) on Mint and you still know the password for the encrypted folder, then yes you can open it. Cryptkeeper on mint can open the folder to a mount point of your choosing in its filesystem.

I have used a single encrypted folder and cryptkeeper in several installs to open the contents when booted into the various installs, a similar situation to yours.
