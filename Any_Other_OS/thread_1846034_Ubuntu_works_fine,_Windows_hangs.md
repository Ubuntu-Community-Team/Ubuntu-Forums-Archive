### Post by linuxmatt7 on 2012-05-06
Forget Windows. You really do not need it considering we Linux users have Wine. If the programs you need to use are not on Wine than try to see if you can go with out them. If you do not have enough RAM than use Partition manager to create a small 10GB for Windows 7 and install Windows 7 on that.
