### Post by hachre on 2013-07-14
The top bar is a complete rewrite of their own called Wingpanel. It only looks similar to what Gnome 3 has as it's top bar but it isn't the same thing. The Applications button on the upper left opens another of their own rewrites called Slingshot. It brings up a menu that is both touch-useable as well as working well for a mouse with an instant search option for keyboard based launching of apps. I added an image to the bottom to show it to you. If you wanna see more [click here](http://elementaryos.org/journal/when-its-ready).

The desktop is very useable and overall I love the look, my biggest gripe with it is that it is based on Ubuntu 12.04, which makes getting and installing some things hard or impossible. If it was based on 13.04 I would use it in a heartbeat.

### Post by buzzingrobot on 2013-07-15
You can't install Elementary "in" Ubuntu. Or, any other distribution, for that matter.  You can install two operating systems in dual-boot mode. Or run one inside a virtual machine hosted on the other OS.

Elementary is based on Ubuntu 12.04 LTS.
