### Post by oldfred on 2012-01-01
Windows is designed not to install to any removeable device as it is licensed to one computer only. So you cannot put a full install of Windows on a flash drive. But you can install repairCD Windows so you can boot & fix a Windows computer.

Make your own Windows repairCD (not vendor recovery):
[http://windows.microsoft.com/en-GB/windows7/Create-a-system-repair-disc](http://windows.microsoft.com/en-GB/windows7/Create-a-system-repair-disc)

### Post by WasMeHere on 2012-01-17
Hi

Another alternative is Bart PE (Preinstalled Environment). It can be used to test a computer without an operating system or a hard drive (similar to a live Ubuntu system), but also to repair a damaged Windows system. I have used it several times to run ***chkdsk*** and repair a system that could not boot after a power failure or some similar event.

See the following link
[[COLOR="RoyalBlue"]_http://www.nu2.nu/pebuilder/_[/COLOR]]("http://www.nu2.nu/pebuilder/")
You need an installation disk of Windows XP to create it. It you get an old PC with Windows, there is a fair chance that you also get such an installation disk.
