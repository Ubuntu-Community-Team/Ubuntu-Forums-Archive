### Post by FormatSeize on 2011-08-01
Slackware. I've been using GNU/Linux since 2009, and I wanted to learn a lot, but honestly the convenience that is provided with Ubuntu distros stifled my learning big time. I had Slackware laying around back then, but still having easy access to an easy distro made me mostly ignore it. 
 
Also, if you are not faint of heart, do Linux From Scratch. I'm amazed at how much I learned by the time I finally successfully built that, booted it, and logged in. I did that with Slackware as the host, but I haven't gotten around to Beyond Linux From Scratch because I lost the hard disk that it was on. I don't mind building it again though.
 
I also hear that Arch Linux is rather difficult, and you have to be a genius in order to install. I've tried, but I have had only limited success with it. Still learning, I guess.
