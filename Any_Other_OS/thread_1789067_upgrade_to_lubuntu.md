### Post by StrayEddy on 2011-06-23
Lubuntu is a different distribution then linux mint, you can't 'upgrade' to it.

You have 2 choices: 
- Get rid of linux mint by installing lubuntu over it.
- Install Lubuntu alongside your current distro (Linux Mint)

Go here to download the last iso file to install lubuntu:

[http://lubuntu.net/](http://lubuntu.net/)

EDIT: if i were you, i would take a look at Puppy Linux (so fast, awesome for old hardware)
