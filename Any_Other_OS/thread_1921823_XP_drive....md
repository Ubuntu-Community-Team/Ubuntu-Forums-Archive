### Post by Rafterman414 on 2012-02-07
It depends on what it is used for. On my Wife's Sony Vaio there is a 10GB partition that is used as the recovery partition. What you can do is use clonezilla for instance to make a backup of that partition and save it on an external drive somewhere in case you need to restore it. If you don't find yourself hurting for disk space I would keep it around in case you need to use it.

If you decide to delete that partition then you can either reformat it and use the partition as another drive for windows or use something like gparted to extend the main partition using the newly acquired free space. I know in Windows Vista and above you can use the disk management tool to re-size partitions but I don't know if the one in Windows XP supports that or not.
