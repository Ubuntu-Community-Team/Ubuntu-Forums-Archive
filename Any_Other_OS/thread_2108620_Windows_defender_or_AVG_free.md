### Post by Dr. C on 2013-02-02
I have being using Microsoft Security Essentials on XP, Vista and 7 for years and I have no problem recommending it to home users and small business users that are under the maximum license limit (I believe it is 10 computers). Why because it is simple built in and does not require annual renewals, or constant up sells. Now at one point Microsoft offered Windows Defender to protect against certain malware and not viruses, this gave Windows Defender a bad name and has led to a lot  confusion in the market place that was even extended to Microsoft Security Essentials. 

Now with Windows 8 Microsoft has renamed Microsoft Security Essentials. Windows Defender and allowed OEM to turn it off. The following explains why extremely well. 

> Windows Defender's increased scope doesn't sit well with computer manufacturers, however. OEMs make beaucoup bucks by installing those trial versions of McAfee, Norton and other security suites you'll find bundled on boxed PCs. Windows Defender's default installation threatens that gravy train. 
Microsoft tossed its partners a bone by allowing OEMs to deactivate Windows Defender in order to ship boxed PCs with alternative security solutions installed. That's all well and good from a "variety is the spice of life" perspective, but one side-effect that isn't so hot is what happens when you fail to register that third-party security software: Windows 8 doesn't automatically reactivate Windows Defender by default. In other words, your pretty new prepackaged PC is wide open and vulnerable to all the nasties of the 'Net. 
 [http://www.pcworld.com/article/2020260/how-to-activate-windows-defender-in-windows-8.html]("http://www.pcworld.com/article/2020260/how-to-activate-windows-defender-in-windows-8.html")

This explains very well why Microsoft Windows has become such a cesspool of malware. It is not the OS itself which with the latest versions and patches can be made very secure, but how it is marketed with trail versions of security suites that more often than not do not get renewed on many consumer PC when they expire. 

As for AVG Free it keeps nagging to upgrade to the &#8220;free&#8221; trail version. The latter then expires after a set amount of time and leaves the user vulnerable. So I say Windows Security Essentials for XP, Vista 7 and Windows Defender for Windows 8.

I once got an anti Microsoft rant in response to this, to which I replied that I am GNU/Linux user, but that when it comes to securing Microsoft Windows, Microsoft's free (as in beer) products above do an excellent job.
