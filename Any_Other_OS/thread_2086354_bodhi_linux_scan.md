### Post by cwsnyder on 2012-11-20
[http://www.openprinting.org/printer/Epson/Epson-SX125_Series](http://www.openprinting.org/printer/Epson/Epson-SX125_Series) lists the driver for your printer, but in the comments on the same page, it says Ubuntu 12.04 does not work with this printer.  You can try installing the driver, if it is not already in Bodhi's kernel using the .deb file for either x86 (32-bit) or amd64 (64-bit) according to the instructions at [https://wiki.linuxfoundation.org/en/OpenPrinting/Database/DriverPackages](https://wiki.linuxfoundation.org/en/OpenPrinting/Database/DriverPackages)
