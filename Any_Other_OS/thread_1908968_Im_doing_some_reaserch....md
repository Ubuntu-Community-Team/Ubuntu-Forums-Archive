### Post by Zukaro Travon on 2012-01-14
Basically what I think is that they're all pretty good in their own ways.

Windows 7 is great for gaming and lots of other stuff as it's compatible with pretty much everything.

OS X Lion, from what I've used of it, is pretty good also (however, I've barely used it so I can't fairly judge).  I like how Apple's products all work very well with their other products/services.

As for Ubuntu 11.10, I like the fact you can use any UI you want (or even just use CLI), and how most of the programs for it are free and work really well.  I also like UbuntuONE although I have yet to use it.  When UbuntuTV comes out UbuntuONE will be even better (and I also like how UbuntuONE works on Windows 7 also (pretty sure it works on OS X Lion also)).  And of course Ubuntu is very stable, and tends to be good for pretty much everything (although it takes some work for certain things, such as playing Windows games).




That's pretty much all I have to say about this.

### Post by Double.J on 2012-01-14
+1 to Zukaro Travon really.

I'll just add in

- Windows 7; excellent backup features included, very clean, modern interface. A lot of it is catch-up compared to XP, but still nice to have more "full features" out of the box and less need for 3rd party. Liking Windows defender, though think it should be a default setting.

-OS X lion; Fantastic, easy to use user features, makes syncing, backing up, transfering a breeze, as with later versions of SL runs iLife 11, all very nice. Only downside, I think it's starting to look a bit dated, yes they keep changing things, but when I load up my PPC leopard, I never thing "Oh this looks old" or think to myself "oh yeah I can't do that" like when I switch to XP from 7... though this in itself could be seen as a good thing?

-Ubuntu/Linux in general; Total configuration; no hand holding, just do what you want and reap the rewards or the consequences! Also the package management systems and repo's over .exe and .dmg files :)
