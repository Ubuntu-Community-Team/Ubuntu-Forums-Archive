### Post by Flymo on 2011-10-20
Hmm... fast booting means something lighter than a full fat distro, imho.

Is it Gnome Shell you like or configurable bling?  You can do most of that Gnome Shell stuff in Enlightenment E17 desktop, which is found in a number of distros.

We are using [Bodhi Linux]("http://www.bodhilinux.com/") with E17, seems to work well on older/less expensive hardware.

Boots to login in around 8 seconds on a 7,500 rpm drive + Core 2 duo, and runs snappily on an old Compaq C300 (Celeron 420, intel shared graphics, 4200 rpm drive) although the boot is less fast! :D

'Conky' reports that we are using 86MB RAM with compositing on, wobbly windows, wobbly dock bar thing, transparency and so on.  I've seen screenshots of Gnome Shell-like desktops, and it does have an 'Everything' launcher tool, but to be honest I've tried (and failed) to like Gnome Shell and Unity, so your mileage may vary.;)

Try a Bodhi live CD, perhaps?
