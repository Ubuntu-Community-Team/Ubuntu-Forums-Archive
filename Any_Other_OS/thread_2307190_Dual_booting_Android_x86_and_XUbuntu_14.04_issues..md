### Post by QDR06VV9 on 2015-12-23
It doesn't matter in which order you install Ubuntu and Android-x86, but if you install Android-x86 after Ubuntu, make sure you don't select to install GRUB or else you'll have to reinstall GRUB manually using a Live CD.
 **Android-X86 (which uses GRUB legacy) and ubuntu uses (Grub 2) **
