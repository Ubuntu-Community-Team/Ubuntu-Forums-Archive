### Post by trinux_bc on 2011-11-28
I use it on one of my computers, it's a great distro. 64-bit is being worked on/tested now, there's a mailing list for testers, the 64-bit repo is still being built up but when I've tested it on my thinkpad t410 I've had very few problems. I usually use the PAE kernel on 32-bit so I can use all of my ram, 64-bit systems are a little faster but this solution gives me a really stable system with all the software I need.

### Post by exploder on 2011-11-28

I have been running PCLinuxOS since May and have been extremely satisfied, it just works flawlessly for me. I have been running the 64 bit KDE test release for some weeks now and I love it! All my hardware works perfectly, the system is rock solid and my computer runs nice and fast. 

Texstar does very high quality work! The "mylivecd" app lets me build iso's of my nicely configured system. I spend my time using my computer for things I like to do rather than spending my time fixing things all the time. I highly recommend PCLinuxOS to anyone that is interested in running Linux. I get a great deal of enjoyment out of my computer thanks to Texstar and the PCLinuxOS Team.
