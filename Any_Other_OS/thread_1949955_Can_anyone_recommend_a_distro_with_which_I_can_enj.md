> **Exsecrabilus said:**
> Also, is it easy to set up GNOME in Arch? Like, can I put the USB in my laptop, click install, reboot, then type "install gnome-shell" and everything will work?

Once you have Arch installed you can just do:
```
pacman -S gnome
```
to install.

Installing Arch in the first place is a completely different matter, instructions are here:
[https://wiki.archlinux.org/index.php/Beginners%27_Guide](https://wiki.archlinux.org/index.php/Beginners%27_Guide)
