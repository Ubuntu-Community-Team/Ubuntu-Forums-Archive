### Post by BrokenKingpin on 2011-06-10
OpenSuse would be a good choice I think... easy to use and healthy selection of software right off the DVD.

EDIT: I would NOT recommend Linux Mint Debian Edition (LMDE) if you have a slow internet connection. The last LMDE dvd was cut late last year, so there is a ton of updates required right after install (which took a while on my fast connection). Also being a rolling release you will be prompted for new updates all the time... good for staying up to date, not the best with poor internet.
