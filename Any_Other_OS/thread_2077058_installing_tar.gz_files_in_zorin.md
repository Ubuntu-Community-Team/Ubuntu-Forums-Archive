### Post by Lars Noodén on 2012-10-27
When you unpack the tarball there should be a file named README or INSTALL or both.  Find them, read them carefully and follow the instructions inside. 

However, if you can find the program via the repositories.  I don't know what Zorin uses, but in Ubuntu you can use Synaptic to find and install packages or apt-cache and apt-get in the terminal.

### Post by critin on 2012-10-27
Zorin also uses Synaptic package manager but it isn't installed by default.  You can get and install it from Software Manager.  Then just use the search to find what you want.  

The tars I've used I've installed by right clicking and choosing to install via Software Center.  

That may depend on version of os/package though.

Using ubuntu's install options usually is best/safest, because it knows the system.
