### Post by aysiu on 2012-02-22
This actually isn't really any different from a Windows or Mac installation (or any other Linux distro installation). Generally speaking, whatever the graphics card and monitor have as the "optimal" resolution the OS will attempt to set things to. It's only when that fails that you get the 800x600 resolution.

In your case, I would get out a magnifying glass until you can find the setting to change it, and then change it yourself. The defaults can't please everyone, and in all the years I've been on these forums (almost 7 years), this is the first complaint like yours I've seen, so clearly you're in the minority. It makes sense to have it be high res for the majority, and then the people who prefer low res can set it that way themselves.
