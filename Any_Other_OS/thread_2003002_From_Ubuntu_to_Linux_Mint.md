### Post by buzzingrobot on 2012-06-13
Mint is a derivative of Ubuntu in much the same way that Ubuntu is a derivative of Ubuntu. I.e., Mint takes files from the Ubuntu repositories, sometimes tweaking them, and creates its own distribution.  It also adds some things.  

I believe Mint can use Ubuntu repositories, if the user is careful not to create conflicts and breakage.

Mint is iat version 13,  They've release da MATE version.  MATE is a fork of Gnome 2. It looks and works like Gnome 2.   Mint also has a Cinnamon release.  Cinnamon is a desktop environment using Gnome 3 code but with a traditional panel across the bottom (or top) of the screen and a traditional menu style.  (You can also install and use Cinnamon on Ubuntu.)

Mint is an excellent distribution that is, if anything, even easier to use than Ubuntu.  Check out linuxmint.com and see if you like it.

Fedora is a distribution supported by Red Hat.  It's an excellent and important distribution because it is typically at the cutting edge of Linux.  Fedora and Red Hat have made, and continue to make, really significant contributions.  However, Fedora assumes you are a skilled Linux user who wishes to participate in that kind of community,  It is a great community and the answers are there. (The online docs are very good.) But, there is little of the handholding that is typical of Ubuntu.

Fedora is also very serious about using only free software.  No proprietary codecs, drivers, etc.  That stuff is available for Fedora, but not from Fedora.
