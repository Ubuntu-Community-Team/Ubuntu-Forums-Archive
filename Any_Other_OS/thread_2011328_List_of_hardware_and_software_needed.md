### Post by papibe on 2012-06-27
Hi tech291083.

A few ideas:

The simplest thing to put all those machine in a network would be a switch, and a few ethernet cables.

For connectivity with other network, or the Internet, you would need to either add a  second NIC on the server, or replace the switch with a router.

Now consider this as very useful teaching tool: virtual machines. Imagine each student having its own client and server, so they all can follow the steps of a tutorial without competing for the same resources (usually access to the server).

Just some thoughts.
