### Post by Ms. Daisy on 2012-03-10
Backtrack is a specialized operating system used for security testing.  Generally if you're installing it, you are also fairly proficient with Linux. If you didn't know that, then you should definitely not install it.

If you're looking for an everyday-use Linux operating system, Backtrack is a terrible choice for many reasons. Use something like Ubuntu instead.

Even though Backtrack is built on an Ubuntu base, it is not supported on this forum.  If you still need to install Backtrack, then you need to consult the [backtrack website]("http://www.backtrack-linux.org/") for help.

Good luck.
