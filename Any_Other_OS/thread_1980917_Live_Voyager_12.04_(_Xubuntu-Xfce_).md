### Post by Uncle Spellbinder on 2012-09-28
After several days of use, all I can say is "WOW!" As I said above, I'm not a big XFCE fan. For "lightweight" desktops, I much prefer LXDE. But I gotta tell you, this French take on Xubuntu is absolutely BRILLIANT. I've been using it for the past few days with GREAT success. Remarkably fast, extremely stable and possibly the best Ubuntu flavor I've ever used.

The info on the web (including the ***[[COLOR=#0099cc]Voyager site[/COLOR]]("http://voyager.legtux.org/")***) is in French. So Google Translate is needed. And the main dev does not seem to understand English. But finding this gem of an Ubuntu-based-distro was well worth the search. The included AWN dock is, of course, fully configurable (I wasn't keen on the default chosen). Conky is even included by default, though I've never played with it before. So I'll need to investigate Conky a bit. Other than needing to install Flash via Synaptic, all else was included as far as I can tell. Damn fine distro that I think I'll keep on the desktop.
