Welcome to the forums. 

Maybe the Windows XP install program is confused by the existing partitions of your hard drive.

You should remove the existing partitions and then create one or more NTFS partitions for XP. The first partition has to be defined as bootable.

I recommend to use a Ubuntu Live CD/USB for this operations. Use the tool System -> Administration -> GParted.
