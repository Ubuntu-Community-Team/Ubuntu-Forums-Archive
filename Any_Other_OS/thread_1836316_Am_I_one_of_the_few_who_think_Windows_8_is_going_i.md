Ok, you have Microsoft, the old monolith everyone uses but hate(d). After Gates went away, Windows seemed to get better, and so did their practices. Free Security which from my intensive uses has proven itself over and over, free Windows Live applications that rival anything on Linux.

Cue in Windows 8 with a WP7 UI and a Ribbon Explorer interface. For tablets, this is a good thing, however, for large screens this will work really well for me. I have a 72" display which I love using for web browsing and I have always wanted some display like Microsoft is creating. I use the netbook interface for KDE on Kubuntu. It is odd, when you get to the larger displays, your brain wants to process less.

Running a full OS on a tablet will be fricken sweet! I will have the absolute power of Windows when I need it and a nice UI where I want it. Running full blown applications on a tablet with a bluetooth mouse and keyboard will be a very nice thing, or a dock like the eee transformer. 

Linux is getting to the point now where we need applications. If we get some killer applications, not just good enough... but make them shiny and useful, then we will have the world. If we pulled together and got some applications going in the same mindset full metal jacket response that we used to hone DEs , Microsoft and Apple will be left in the dust.

Also remember, that behind the monolith businesses, there are "people" running them. We need to win those people's hearts.

I think where they are going is good, but I need to see a more fully finished product to be able to tell if they are doing it right. I think the whole web technology interface and development environment for the OS is the best idea. It just needs to be cleaned up and change the way it works. Preferably giving software developers an easy way to convert their applications to support the new Windows.
