Helal Linux is an islamic and arabic ubuntu-based distro,a user friendly distro that is for everybody by everybody,Syrian origin distro, it include all what users needs,from flash to java or codecs and programs and fixes for bugs that in ubuntu and themes and support for arabic language and english too,and it is own Features and programs..
[/COLOR]

right now the latest stable version of helal linux is 2.1


helal linux 2.1 is based on ubuntu 11.04(we will release a new version after a few hours),and came with [COLOR="red"]new features[/COLOR]:

-linux kernel 2.6.38.10
-gnome 2.32.1 (only)
-the free drivers for nvidia and ati/amd vga cards have been add by default to the system,and the closed drivers can be installed using one of the distro's script without need of internet.
-too many fixes for bugs and problems in ubuntu 11.04,and more performance and speed with some tweaks.
-there is a distro scripts which allow you to do some things using terminal to the OS,like installing nvidia drivers offline or update and upgrade..etc
-new wallpapers and themes and plymouth style for the distro.
-many other features...
