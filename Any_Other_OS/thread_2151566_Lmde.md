More updated than Debian stable, but not more updated than regular Mint, which is based on Ubuntu 13.04, which in turn is based on Debian unstable. Debian testing (hence LMDE) is not real "rolling" in the sense that you get new updates whenever new version of something is availiable (like Arch), but only in the sense that *eventually* your packages will be updated when they make it to testing, it can be quite a while (first in Debian experimental then in Debian unstable) and usually by then these packages are old comparing to the versions in the most recent release of Ubuntu/Mint. So no, you don't get the latest and greatest with LMDE, only less old.

When LMDE came out whizzy was testing, so it was based on whizzy, but now whizzy becomes stable, LMDE should be updated to Jessie, which I think is still on 3.2 (other packages are not very new either, but haven't really check, I am on Debian unstable)
