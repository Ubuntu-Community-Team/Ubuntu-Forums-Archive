### Post by KBD47 on 2011-10-27

### Post by AZBoy28 on 2011-11-01
@cortman

I'm also trying to achieve the same thing with a 2GB USB.

I have managed to install Ubuntu 11.04 'Natty Narwhal' with 1235MB space left over to store
stuff. (settings, apps.. etc..)

Before you install Ubuntu, format your USB before installing, to make sure there isn't any left over bits.

Tutorial- [http://1024bitez.blogspot.com/2011/05/installing-ubuntu-1104-in-pendrive.html](http://1024bitez.blogspot.com/2011/05/installing-ubuntu-1104-in-pendrive.html)

*The reason why it didn't work for you, is because you installed it onto the USB partition directly. If you follow the link above then you will be able to do a persistent installation with PendriveLinux and get 1235MB space to store stuff.

PendriveLinux- [http://www.pendrivelinux.com/](http://www.pendrivelinux.com/)
