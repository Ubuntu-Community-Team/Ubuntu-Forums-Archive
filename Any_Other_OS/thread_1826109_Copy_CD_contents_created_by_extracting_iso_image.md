### Post by haqking on 2011-08-16
as i said.  If you copy a folder contents to a cd it will be a data cd.

Make an .iso image from your CD briuning software which will take the contents and put it into a burnable .iso format.

You cannot burn a folder contents as an iso image, you can only burn an iso image as a iso image.

for example: using K3b, choose disc copy and choose create image.

If you want to make an image from a folder contents then you create a new project, choose your data and then make an iso image
