If you are a pen-tester than you will use the one you prefer, as to why you would prefer it is a personal thing.

BT which is now Kali-Linux is the industry standard and most recognised within the security industry, i use the word 'standard' loosely.

Use what you like, NMAP, Metasploit, Netcat etc etc all work the same whether you use BT, Matriux or install into Ubuntu, Mint, fedora, Suse etc.

They are all just toolkits.

I personally use BT (and now Kali), as it has everything I need, if i need something different i install it, i have no desire or requirement to use anything other than that from the new kids on the block which are usually from disreputable sources
