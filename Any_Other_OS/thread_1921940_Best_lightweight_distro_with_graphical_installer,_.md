**Re: DSL / Live OS <512MB**
Have you looked into [Core Plus]("http://distro.ibiblio.org/tinycorelinux/welcome.html") (from the Core Project). The project began with Tiny Core Linux, as a DSL fork. It starts off with 64MB. I guess [SliTaz]("http://distrowatch.com/table.php?distribution=slitaz") is the other distro that keeps coming to mind. I don't have Windows anymore, so I have no idea if they'll work under qemu, but here are some links: [TinyCore Virtualization]("http://distro.ibiblio.org/tinycorelinux/faq.html#qemu"), [SliTaz & Qemu]("http://vanilla.slitaz.org/index.php?p=/discussion/218/slitaz-liveusb-in-qemu/p1")

**Re: Lubuntu's feel**
Yeah, LXDE is very Win 2k. It's not that big a deal; I use AWN, oxygen-style cursors, and changed the icons. Little changes go a long way towards making it feel more modern, and it's still more responsive than Xfce. Otherwise for a light ubuntu-based distro, I like [Bodhi Linux]("http://www.bodhilinux.com/system.php") (uses e17) some people think [it looks sharper]("http://www.youtube.com/watch?v=zew22fRREQ0") than Lubuntu.

I hope I was able to help somewhat. :-/
