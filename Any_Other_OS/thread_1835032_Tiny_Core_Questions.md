### Post by Plumtreed on 2011-08-28
After an install to a hard drive you no longer operate in RAM and you lose the 'extra' speed that provides.

Tiny core has many tools included....you will, no doubt, find that there will be some you will have to work out.

Have you had a look at Slitaz which provides a very complete OS in a tiny package......I only suggest it as a guideline and not an attempt to change your mind.
