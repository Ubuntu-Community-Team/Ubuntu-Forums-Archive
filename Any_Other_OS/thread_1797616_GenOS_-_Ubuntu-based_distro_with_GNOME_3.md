### Post by netherblood on 2011-07-05
GenOS is a new distro based on Ubuntu. It started with KDE but the new version (still under development) uses GNOME 3.

A build script is available (based on eBuild by elementary os) so that you can build the latest development version.

Since I haven't seen any Ubuntu-based distro using GNOME 3 that wants to be easy to use and polished around the forums, I decided I would post mine (I know about gNatty, but it's not user-friendly).

The code is hosted at launchpad and build instructions are available in GenOS' wiki.

Wiki page: [http://wiki.gen-os.info/index.php?title=Building](http://wiki.gen-os.info/index.php?title=Building)
Code: [https://code.launchpad.net/~netherblood/gen-os/genos-build](https://code.launchpad.net/~netherblood/gen-os/genos-build)

The latest development version boots and is installable already, so if you want to mess around with it, feel free to.
