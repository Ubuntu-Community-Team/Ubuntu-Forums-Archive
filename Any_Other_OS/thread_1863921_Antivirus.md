### Post by hwttdz on 2011-10-18
Unless you're hosting a fileserver for windows machines you should probably not have _any_ antivirus programs.  I realize to someone coming from a windows environment this sounds like insanity, but there are plenty of places you can read up on it.  That said, the best protection you can have is and up to date system, so instead of worrying about keeping an antivirus just update regularly.  That and educating yourself about proper security precautions (i.e. only installing software from secure repositories, not running command line commands you do not understand). 

I do however keep a hosts file which keeps my computer from contacting a bunch of different advertising providers.  I also highly recommend the noscript extension for firefox.

I assume you are talking about Windows?

Avast is good. It has saved my butt several times. There is also microsoft essentials which is a free anti-virus made by microsoft for windows. I use it. Works good.

Spy bot: Only run this when you want to remove something. Otherwise un-install it. You probably installed the Teamtimer.exe with it and it will slow down your system.

Defraggler is a good program. Pirisoft makes nice things.
CCleaner is another fine program. 

Other than that... I think you are pretty good on Windows. Run CCleaner every few days. Schedule Defraggler to run once a day.

If you are using Linux, yeah you will hear a load about how it is virus free and all that. Maybe its true, maybe its not? 
All I can tell you is do not run scripts unless you know what they are.
If you install the ssh service on your computer. Install GUFW so you can set the system to only allow certain connections from the outside.
If you really want a linux av, I suggest you check out Eset-nod32. Try out the beta [http://go.eset.com/us/beta](http://go.eset.com/us/beta)
