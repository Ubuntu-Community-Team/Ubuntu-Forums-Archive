### Post by krapp on 2011-05-25
(I think it needs to emphasized that Debian deliberately ships a 100% free system, and so deliberately doesn't include in it's base system and "main" repositories many drivers that Ubuntu might. But getting them is usually just as easy, as most are available in the "non-free" repositories which are as official as the main ones. So in a sense Debian isn't 100% free.)
