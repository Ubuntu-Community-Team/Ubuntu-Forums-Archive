### Post by Mr. Shannon on 2013-03-02
If you old machine is 64 bit capable then it should be fine.  If it is truly a 32 bit machine then you must use a 32 bit version.  A 32 bit version can be installed on a 64 bit machine but not the other way round.

### Post by westie457 on 2013-03-02
Welcome to the Forum.

Short answer is no.

It will not work. A 32-bit capable machine cannot run a 64-bit os. A 64-bit machine can run either, though it is preferable to use a 64-bit OS on a 64-bit machine.

So in your case you will need to get the 32-bit version.
