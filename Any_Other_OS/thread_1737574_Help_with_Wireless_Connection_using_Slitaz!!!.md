Slitaz is not the easiest OS for a new Linux user in regards to wireless and mobile broaband. It was developed to be small and thereby operate quickly in RAM.

You might want to look at something like PeppermintICE which is fast but much easier to establish a mobile and USB broadband connection...it was developed from Ubuntu and offers the ability to add much of the software available to Ubuntu user.

In any event, Slitaz has a very helpful forum and also offers a good manual for reference.
