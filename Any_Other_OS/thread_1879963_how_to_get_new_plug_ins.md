### Post by Frogs Hair on 2011-11-12
First try this add-on to install the latest version flash . [https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/) 

Your previous version of Firefox should still be listed in the synaptic package manager  and you could install it from there by marking it for installation and selecting apply .
