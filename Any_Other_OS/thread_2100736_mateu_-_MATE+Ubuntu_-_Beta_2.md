### Post by ptavatar on 2013-01-02
a few days ago, I posted about this new ubuntu-based OS.
Yesterday, I released Beta2

[Mateu's homepage]("http://mateu.pt.vu")

So, principal Beta1 and Beta2 changes were:
[LIST]
[*]added google chrome, libreoffice and vlc ppa
[*]midori is the default browser (lighter)
[*]theme changed to "Shiny"
[*]new plymouth theme
[*]default network manager is wicd
[*]installed lubuntu software center
[*]installed gdebi (gui package installer)
[*]default media player is vlc
[/LIST]

Mateu comes with MATE 1.4.2, MATE PPA and all MATE default apps, like caja file manager, pluma text editor, etc.

If you want download it and test it and post your feedback.
