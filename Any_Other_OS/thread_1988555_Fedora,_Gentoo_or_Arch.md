### Post by buzzingrobot on 2012-05-27
It's been a few years since I used Gentoo, but I remember that the install wasn't a problem as long as understood every little detail in the instructions.  Problems arose a few weeks later as I began to add things and tinker a bit. I felt like I really needed to keep detailed notes about every little change I made.

Fedora would be a great pure-Gnome distro if they put out an LTS version every once in a while. The fact that it is really a test platform for RHEL shows.

I just spent a week to so with Arch.  The install is fine. The system was fast, stable, and  looked good. I found their repos to be not as large and varied as Ubuntu's and the user repos, with almost 30,000 packages, include a fair amount of packages marked "out-of-date", plus multiple versions with no explanation of which one to use. The Arch wiki is reasonably strong.  The forums are not as busy as here. I noticed some folks presenting problems weren't given answers but told about the "Arch way" or that they were the problem.  I have no time for that.

I'd go with Fedora if I wanted a purist approach to Gnome 3, Arch otherwise.
