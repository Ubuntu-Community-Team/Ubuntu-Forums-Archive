### Post by leclerc65 on 2012-02-11
Use Unetbootin (Windows version) to write the Puppy iso to the USB (Puppy files will co-exist with the data files you had on the USB). When logging out you answer 'Yes' to save, choose SDA1 (Windows C drive), Puppy will put the 'Save file' there. Also answer yes to the 'Copy'.
Next boot will be much faster because Puppy will find what it needs in the C drive. You can use the same USB for all the computers in your house.
If I were you I would start with Lubuntu which is not much heavier, but it's easier to learn. I install Puppy only on very weak computers. But you maybe smarter than me.:) 
By the way I recommend Puppy 5.28.4 which is Ubuntu based.
