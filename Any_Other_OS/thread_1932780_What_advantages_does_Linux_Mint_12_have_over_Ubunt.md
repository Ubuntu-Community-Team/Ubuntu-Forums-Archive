### Post by Gone fishing on 2012-02-28
Mint is built on Ubuntu unless you get the Debian based rolling version. The main differences are that Mint has its own forked version of Gnome shell rather than Unity. Its version of Gnome shell is more "traditional" and some users prefer it - personally I like Unity well enough.

Mint also has the multi media codec pre installed, this is legally a grey area and it is very easy to install these in Ubuntu anyway. Finally Mint has a few of its own tools.

Mint is cool I have used it but prefer Ubuntu.

### Post by drawkcab on 2012-02-28
I look at it like this:  play around with Mint live cd for a while / figure out the things you like about it / add them to Ubuntu.  

I say this because some of the so-called advantages of Mint aren't that great, are bloaty or are just plain annoying.  

Mint Debian Edition, on the other hand, is pretty cool if you need a snappier, less resource intensive OS that is fully-featured.  On my eeepc, LMDE with full gnome 2.X desktop was on par with Xubuntu in terms of performance, perhaps a bit better.
