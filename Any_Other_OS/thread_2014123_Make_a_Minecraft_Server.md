### Post by lepponesfam on 2012-07-01
Well you can't make a server on any old computer. I recommend at least a 1.5GHz processor (processor power is not essential for minecraft). An Intel Atom would be good. Then have at least 4gb of ram for an 8 player server. Then a 160gb hard drive (100gb root filesystem, 60gb swap.)Then it needs to be able to boot from a disk drive or a usb flash drive and if you need to take the hard way even a network boot.
