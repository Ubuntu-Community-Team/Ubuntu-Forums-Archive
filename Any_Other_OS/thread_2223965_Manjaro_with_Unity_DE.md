Yes, but according to the Archwiki you are not advised to get it from AUR as there are something confusing about the versioning. Instead should get it from github [https://wiki.archlinux.org/index.php/unity](https://wiki.archlinux.org/index.php/unity)

I don't think it works in Manjaro though. I remember reading in the Unity for Arch forum that this is because of versioning problems for multiple replicas of gnome packages (Unity's version is not vanilla gnome from upstream) This problem exists even in Arch but they manage to work around it but the work around doesn't work with Manjaro as their main repos are out of sync with Arch's (with a delay and some packages held back for stability etc)
