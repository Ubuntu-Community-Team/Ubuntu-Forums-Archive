### Post by oldrocker99 on 2013-08-15
Oh my yes, Chrubuntu certainly supports Flash. I have an Acer C7 with the 360GB HD; it cost me a mere $200 (no Windows tax!) and it works **very** well; considering that it has Sandy Bridge graphics, which were roundly criticized, it's pretty amazing how well it runs games. Not every game, mind you, but all the Half-Life series run at playable speeds, all the OpenGL screensavers are snappy, etc. 

Bear in mind that Chrubuntu is a **full** Linux installation, and pretty much what a Linux desktop machine can do, an Acer C7 can do, **if** it's running Linux.
