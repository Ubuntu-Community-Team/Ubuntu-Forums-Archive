### Post by exploder on 2011-06-26
I have been a Linux user for years. I run PCLinuxOS KDE as my primary operating system and and very happy running it. I have another computer that has a Foxconn mainboard, no version of Linux has ever run properly with this board. Some of you might remember Foxconn deliberately altering the BIOS so only Windows would run properly. Anyway, 3 BIOS upgrades and the Foxconn board still has issues running Linux.

Rather than see this computer used as a doorstop for another 2 years I installed Windows Vista. I had the copy of Vista from an HP computer that was struck by lightning and I had never ran Vista. It took 2 and a half days to get Vista updated and configured to where it would not slow down over time. I will spare you all of the things needed to make Vista fast and less bloated. In the end Vista actually runs very well and I have not had a single problem with it.

I installed nearly every open source application I use daily in PCLinuxOS. I was surprised how closely I was able to match the applications between the two systems. It was unfortunate that I had to put Windows on the Foxconn system but it was a great learning experience! Vista has been around since 2006 and Microsoft did a decent job of fixing it and I have to give them credit for the length of support.

People say that you do not need to use the command line in Windows, that is completely false! I had to use the command line to limit the disk space that System Restore uses by default, 15% of my 500 GB hard drive is way more than is needed. My Vista install with a complete set of applications uses 27 GB of hard drive space, that's a heck of a lot more than PCLinuxOS uses for nearly the same applications. PCLinuxOS uses under 3 GB to have the same level of functionality!

LibreOffice, VLC, Bleachbit, Firefox and Virtualbox all work fine in Vista and that makes the computer useful again. My Wife runs Windows 7, she is a gamer. Windows 7 just seems like a more refined version of Vista to me. I am not saying Windows 7 is bad by any means because it is a decent operating system but in my opinion a fully tweaked out Vista install is just as good. 

Linux will always be my operating system of choice and I really like PCLinuxOS KDE and I highly recommend it. I fix computers on the side for extra money and I often run into hardware that will not work under Linux. What I was able to take away from the Foxconn tampering with the BIOS was that I can give a Windows user the same good set of applications that I enjoy in Linux. Do I think that Windows is stable? The short answer is yes, it just takes some fine tuning. 

In Windows everything you install wants to look for updates, I turn all that off and manually install updates as the become available. The only application I want to find it's own updates is the anti virus program. Avira is the lightest anti virus program I have found, it has done a good job of protecting my Wife's computers for years. Avira Personal Edition is free, there is an annoying pop up ad when it updates but that's a very small price to pay for something that does such a good job.

Vista got a bad reputation because it did not run on the hardware it was supposed to be capable of, it was bloated and used more memory than it should have. Turning off un-needed services, automatic updates for applications, changing default settings and cleaning up the hundreds of Windows Update install files with Bleachbit put Vista almost on par with Windows 7. 

I have had Linux distributions installed in the past that have required just as much tinkering under the hood as Vista and I do have to give Microsoft credit for their support efforts on Vista. My main computer will always run PCLinuxOS but at least the Foxconn system is capable of getting some work done now.
