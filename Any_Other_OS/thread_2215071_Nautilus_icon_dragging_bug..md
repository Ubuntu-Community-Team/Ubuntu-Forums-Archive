### Post by grahammechanical on 2014-04-04
I got that effect a couple of months ago on Ubuntu Gnome trusty tahr (14.04) when I install gnome-session-flashback on to it. I got two extra login options. Gnome Flashback (Compiz) and gnome flashback (Metacity).

With Gnome Flashback (Compiz) I didn't get a desktop at all. With Gnome Flashback (Metacity) I got the desktop but I also got that dragged around effect that is in that image.

I concluded that gnome-session-flashback was not meant to be installed on Ubuntu Gnome. I fould that gnome-session-flashback worked as it should on Ubuntu+Unity.

I would suggest that if we are running Ubuntu Gnome 14.04 we install Gnome Classic. It is a Gnome 3 Shell extension that gives a similar menu look as to the 12.04 Fallback look.
