### Post by N00b-un-2 on 2011-12-09
As of right now, I would say that newbies to linux should stay away from the regular Ubuntu distro because of Unity.  In my opinion, the best distro at the moment for a new convert to linux would be the latest Linux Mint Gnome edition (not the Debian Gnome edition).  since it still uses Gnome 2.x it will be the most similar and familiar to those coming from Windows.  As for running a server distro, FORGET IT!  You have to be a certified terminal jockey to be able to do anything.  Certainly not for newbies.  You can start with a regular desktop distro and then install the server meta package (sudo apt-get install ubuntu-server) in order to install all of the services.  If you need the eye candy of the Windows Aero interface, you may want to check out KDE, but in my opinion it's not as user friendly as Gnome 2.X was.

Think of it like this:

Ubuntu Server = Ubuntu without a GUI which is not for newcomers
Ubuntu = Linux on training wheels.
Linux Mint = Ubuntu on training wheels.

KDE = looks like Vista
Gnome2 = Looks like XP
Unity = a nightmare for anyone looking for a familiar desktop

You may want to give XFCE a try as well.  It's similar to Gnome2.
