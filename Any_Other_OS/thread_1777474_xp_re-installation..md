### Post by ajgreeny on 2011-06-07
If you deleted the XP recovery partition and did not make recovery CDs when you first got the system, you will simply have to start from scratch and do a clean install.  After doing that you will need to restore grub to be able to boot ubuntu again, but that is very easy with the live CD.

I suggest you run the live CD first and shrink your ubuntu partition to make room for XP, or the XP installation disk may not find anywhere to install, and/or may insist on using the whole disk; not good as you want a dual boot system.
