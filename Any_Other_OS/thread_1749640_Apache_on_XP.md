You could always try this if you want a complete locally-hosted webserver that's easy to set up:

[http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html)

All you do is run the installer and it gives you Apache, MySQL, PHP w/PEAR, Perl, mod_php, mod_perl, mod_ssl, OpenSSL, phpMyAdmin, Webalizer, Mercury Mail Transport System for Win32, NetWare Systems v3.32, Ming, FileZilla Server, mcrypt, eAccelerator, SQLite, and WEB-DAV w/mod_auth_mysq.

Believe me, it's a whole lot easier to set up that than set up an entire web server from scratch.
