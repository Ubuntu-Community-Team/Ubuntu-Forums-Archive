### Post by Lars Noodén on 2013-09-29
If it's a firewall, then UFW would be the main interface.  If you prefer a GUI, then GUFW would be the option.  Both are front ends for the builtin packet filter, iptables, which is part of the Linux kernel.  All versions of Ubuntu have UFW and all versions of Linux have the underlying iptables.
