click on the start button and type "update" in search, then click on windows update, -> if IE9 is not installed yet then click on available updates and scroll down till you find it listed. Right click on the update and choose "hide update". This will hide IE9 so it does not install, plus, since it is already listed in your update list, it will no longer download.

If it is already in, then go to "installed updates" find it and delete it. Your system will revert back to IE8 because it insists on having some verson of IE installed. Once done, check for updates and go through the steps above to hide IE9.

This should take care of the problem.
