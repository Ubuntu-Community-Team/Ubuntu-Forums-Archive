### Post by MARP1961 on 2012-01-09
Deleting Ubuntu isn't difficult but if you have completely got rid of Windows, it might be difficult to reinstall. 

There should be a Windows Recovery Partition on your hard drive (unless you deleted everything to put Ubuntu on). If Windows is completely gone you will need a Windows Installation Disk. If you have it is a simple matter of booting the computer with the disk already in the tray then allowing Windows to reinstall over the Ubuntu.

Since the computer is quite new, the manufacturer will probably send you such recovery disks if you explain what you have done. Although you have probably infringed the terms of the guarantee by altering your install, they will be able to give advice and sell you the disk for a lower price than a fresh Windows Operating system. Since you have a Windows Product Sticker you should not have to buy a brand new Windows license.

The lesson for others is do not delete Windows unless you are 100 percent sure about doing without it. Also, make sure you burn your Windows Recovery Disks! If you do some simple research first, you will find out it is possible to dual boot Ubuntu and Windows.

Good luck with the manufacturer!

That menu is for an Ubuntu install, not for Windows.  You're either booting from your hard drive, or you have an Ubuntu CD in the drive. 
> how do i install windows ?

Even when booting from install CD, if all that is on your PC is Linux filesystems, the Windows installer can get confused and not allow you to reformat and install.

So, first boot from an Ubuntu desktop CD, and select Try Ubuntu.

When you get to a desktop, click the Dash icon and enter Disk Utility, then select that to launch it.

Once Disk Utility is open, use it to remove all the partitions on your drive.

When you reboot using the Windows install CD/DVD, you should be able to install now.
