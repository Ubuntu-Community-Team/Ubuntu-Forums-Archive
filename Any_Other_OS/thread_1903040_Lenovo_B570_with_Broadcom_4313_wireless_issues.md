### Post by patallacgrillz on 2012-01-01
Hi I recently purchased a lenovo b570 laptop with windows 7 and dual booted it with linux mint 12 (11.10) I was having a lot of problems getting the wireless card to work, The card is a broadcom 4313; I tried virtually everything to get it to work followed numerous posts with different answers. 

Some said to blacklist the acer driver, that did not work for me
I tried to install the broadcom STA drivers from the synaptic manager, did not work 
I tried to use the windows driver program, did not work
tried to install wicd and remove network manager, did not work
numerous other terminal actions, always rebooted after each, fully restored mint if i couldnt repair

Finally, I fully reinstalled mint, installed the updates from the update manager, installed the broadcom driver, installed wicd (important to NOT remove network-manager), rebooted, and it worked!

I hope this is helpful, if I didntt follow these steps exactly it would not work for me. Repairing things you have removed via synaptic manager or terminal did not work, you must use fresh install.
