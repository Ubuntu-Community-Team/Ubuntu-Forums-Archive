### Post by charito on 2013-06-06
I was having problems not been able to watch a video on CNN today,the video will only show on full screen but if I diminished it then I was not able to maximize it again,any way long story short. I found this solution an it worked. Just in case if you need it: **Method 2:**
Disable hardware acceleration and use software rendering instead of hardware rendering to view the webpage, follow the steps below:
a)     Press Windows key + R and type inetcpl.cpl and press enter.
b)    Click the Advanced tab, and then browse to the Accelerated graphics section.
c)     Click to select the Use software rendering instead of GPU rendering check box. 
d)    Click Apply, and then click OK. 
e)     Close all open Internet Explorer windows, and then restart Internet Explorer. 
f)      View the webpage in Internet Explorer. 
g)     To re-enable hardware acceleration, follow the steps listed above and uncheck the Use software rendering instead of GPU rendering check box.
Let us know if it helps.
