### Post by sanderd17 on 2011-06-20
Kubuntu is just Ubuntu with KDE apps instead of gnome apps. KDE is a little heavier than gnome, but some find it prettier (it all depends on taste)

Linux Mint is Ubuntu with some enhancements (the Gnome interface is made more windows like, the software updater isn't nagging, a nice grey theme that doesn't draw attention, ...) but the difference is not big.

OpenSuse is a completely other distro. When you install it, be sure to NOT INSTALL GRUB. Opensuse works with GRUB legacy and the GRUB can't discover Ubuntu. What you need to to is install OpenSuse without grub (uncheck the installation of grub in the installation summary) and then go to ubuntu and update your grub. Ubuntu will discover OpenSuse. Apart from that, Opensuse is a fine OS with a deeply integrated control center YAST, but if I use it, I kinda miss some Ubuntu-Debian tools.

I have never used Fedora :oops: so I can't tell you more about it.

Ubuntu studio is again, just ubuntu with some other packages preinstalled. They use xfce as desktop environment, it is lighter than Gnome and still has a big adaptability. They also ship more multimedia apps.
