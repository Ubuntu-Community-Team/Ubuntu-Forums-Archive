### Post by Dlambert on 2012-02-19
No, no, no. For starters Ubuntu is based on Debian, SUSE is based on, well SUSE. Ubuntu is a deb (debian) package distro, SUSE is RPM. BY DEFAULT Ubuntu uses unity (dm), and SUSE either uses KDE or Gnome, depending on what you so desire. SUSE as a "control panel" called YAST that offers in depth customization of your install. Ubuntu has the Ubuntu software center, Ubuntu one, Ubuntu music store...etc...this could go on forever. But in my opinion, SUSE/OpenSUSE (opensuse is the free version - the corporate support/licenses/software etc) seems to fit business/more experienced users, whereas Ubuntu can be used by anyone and seems to be aimed toward the everyday consumer. They are both great distrobutions, so don't let me tell you about them, just go try them out yourself! The only (generally speaking) similarity is the common foundation, the Linux kernel.
