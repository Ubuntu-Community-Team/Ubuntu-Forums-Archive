### Post by iridesce on 2011-05-12

Not knocking the great work done by you all, when I decided to move from gnome, I tried kubuntu and a few others ending up going back to Mepis. I have found the disto to work well for me and the community to be second to none.

### Post by Crempel on 2011-05-20
It's not totally forgotten, it had quite a few downloads after the latest release and finished 2010 at position 13 on DistroWatch. I use Ubuntu for "bleeding edge" and Mepis 11 for "normal work". Mepis is Debian stable based but has up-to-date versions of almost all software, only KDE and the kernel are not the latest level, but far more current than Debian testing. The forum is great, the distro is fast and absolutely stable, plus it's far more noob friendly than vanilla Debian Squeeze. Worth a try.:)
