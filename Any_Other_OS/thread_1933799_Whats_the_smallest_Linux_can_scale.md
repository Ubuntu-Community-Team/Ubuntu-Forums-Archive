### Post by Helen McCall on 2012-03-19
You need at least a 32bit cpu. The original Linux worked on 32bit intel 386 processors but not on the 16 bit 286 processors. There was a project for a while trying to build a linux kernel to run on the old 286 but I'm not sure whether it ever amounted to anything.

I started using Linux in 1995 with Slackware on an intel 386 with 64MB of RAM and 30MB of disk.
