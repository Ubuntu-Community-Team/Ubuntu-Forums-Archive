I use smplayer as a front-end to [mplayer2]("http://www.ubuntugeek.com/install-mplayer2-on-ubuntu-using-ppa.html") in Kubuntu 10.10.  I don't have any problems with videos, but I'm using an NVIDIA card which has much better support for video playback via its VDPAU interface.  If you don't have problems watching HD videos on your system now, changing the desktop environment won't make any difference.  With 4 GB of RAM any libraries or other DE overhead will be cached in memory anyway.  

I don't play games like WoW so I can't help you there.

I've used KDE for about a decade now on a wide variety of hardware.  Like PaulW2U says, comments about its "bloatedness" or poor performance are largely exaggerations.
