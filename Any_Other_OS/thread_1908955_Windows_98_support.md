### Post by Double.J on 2012-01-14
As cheesemill says... MSDOS isn't freeware or opensourced yet!

If you did find a copy of it, the one you'd really want is the 3rd edition, or service pack3 or whatever they called it!

As for finding a CD... I'm not sure how this works with the EULA.. Windows 98 was the last edition not to use a product key - for example on home machines XP and on you purchase a licence key... usually with the machine, that you use to reinstall the OS - not necessarily from the same disk. As 98 and before didn't use the licence key - an installation disk just did an install, I'm not sure how copywrite covers the selling of the disks/making copys of them... It may be worth researching.

I have 98 with office 97 and encarta 99 disks still - tried them on a modern machine a couple of years a go.. obvious fail - got it to run in a VM no problem - getting it to work properly was a nightmare! There's no tools/additions for non NT based windows OS, so you have to set up things like networking by hand - especially with windows 7 it was a PITA! I had more luck on Linux host (mine was deb)... as usual ;) oh and you have to upgrade to the the last explorer edition for 98 to get things like updates - which are still on the MS site... or they were when I accidentally deleted my old VM in the summer!

Best of luck!

'98se was a great 'doze, and would be great for lovers of the older games and apps (many of which could still be used productively for fun and/or productivity).
