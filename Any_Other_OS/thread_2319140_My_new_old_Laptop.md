### Post by vanquishedangel on 2016-04-03
Maybe, but I look at this as challenge accepted lol!. I did some digging and discovered the Pcmcia slots are type 2, this means it is possible for the laptop to use wifi, Ethernet, and usb (was not invented until after windows 98) mechanically. So maybe linux will not be the best option on it, but it does have a working windows 95 and office 97. So I did some research and discovered to my amusement that windows 95 can handle broadband Ethernet (and as of 2010 still works on the net), wifi and usb (usb was backported to 95). So I ordered 1 Ethernet pcmcia card, 1 usb pcmcia card (both windows 95 compatible) and a memory upgrade chip (16 mb, whoooo, so it will have 24 mb) for a total of $35. This may allow Damn small linux to work, but either way it will work fine as a text editor for college assignments without the distraction of facebook. All I want to use it for is documents, so I need a way to transfer a document file to hand it in for class. I do not intend to use it for Internet surfing, but rather to do documents on and not have the Internet as a distraction. At home it will be behind a Linux box and other then that will not be online. I was hopeing and was certain there would be a Linux I could use like an early debian or something.

I did spend some money on it but in the end it was ok, I discovered that the PCMCIA slots do not support cardbus but that is ok. I discovered xircom pe3, an adapter that converts a parallel printer port to ethernet and bought one for $15. It works. To test it does run the INTERNET quite well, loads google and attempts to update from the microsoft website fully loading all the pages. That will be the extent it will go on the internet but now it will network. 

As for software, it runs auto CAD, peachtree small buisness accounting software, adobe photodeluxe professional business edition, dos games, office 97 sp2, internet explorer 5.5 (still renders web), etc. pretty smooth, it will work as a text editor.
