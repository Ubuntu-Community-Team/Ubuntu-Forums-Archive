### Post by mastablasta on 2012-11-22
wow that's a sweet maschine...
 
anyway you have intel GPU in the processor i3 (Intel HD 3000). 
> Model: "Intel VGA compatible controller"

 
drivers for intel cards are already embedded in kernel. if you have intel GPU why did you install nvidia drivers? no wonder nothing works.
 
if you maybe have the version with AMD card (for example [http://www.ubuntu.com/certification/hardware/201106-8229/](http://www.ubuntu.com/certification/hardware/201106-8229/)) then you need to install AMD drivers not nvidia.
 
anyway since drivers for the GPU are part of kernel you have two options:
1. upgrade kernel
2. install PPA with newer drivers.
 
however i believe that the card you have should be fully supported by the kernel you are using. someone else could maybe help you more. i do not have intel cards on any mashcine. but i do knwo that notebooks with intel cards and i3 chip often come linux preloaded.
 
additionally perhaps this is a Mint issue and would be best addressed on their forums. i don't know if they added some "fixes" or pathes that messes up support for this chip.
