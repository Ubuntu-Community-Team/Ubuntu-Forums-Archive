### Post by motoroller on 2012-03-25
As far as I know, this is exclusively a Windows system file, normally hidden. If you're browsing the file system using Ubuntu, it might not be hidden anymore. All it is is the folder used by Windows to store the recycle bins for each Windows user. They are normally not able to be deleted in their entirety, and the contents of each bin would have to be deleted by each windows user (unless there's a quicker way of doing it which I'm unaware of!)

So if you want to delete whats in your recycler folder, I'd be inclined to boot up windows and try from there. I will give you a little disclaimer because I'm not an expert, but I think this would be a good place to start... 

Good luck! :)
