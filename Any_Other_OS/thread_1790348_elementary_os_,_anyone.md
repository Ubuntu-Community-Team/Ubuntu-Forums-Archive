### Post by keithpeter on 2011-07-23
> **73ckn797 said:**
> Seems to run just as fast on my machine as Ubuntu.

Elementary OS Jupiter is Ubuntu 10.10 with a rather nice modification of Gnome and some applications that the team have written themselves. It runs OK on my Atom based EeePC netbook, but compiz is active by default, assuming you have video drivers that are ok with compiz.

I rather like the dock, and using Windows-E and Windows-W to manage windows. 

It is NOT a 'light' system in the way Bodhi Linux or AntiX is, the aesthetic is light and airy but its Gnome + modified nautilus underneath!

The next version looks quite fun. I'm sticking with Xubuntu on the big box.
