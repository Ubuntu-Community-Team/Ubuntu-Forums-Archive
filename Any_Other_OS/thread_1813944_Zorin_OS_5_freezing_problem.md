Hello. I'm having trouble with Zorin OS 5 after installing it. I think it has to do with my graphics card. Figuring that Zorin is based on Ubuntu, it'll have the same problem I think.

_**Here's my Dell Dimension E310 specs:**_
Intel Pentium 4 CPU 2.80 GHz
Intel Corporation 82915G/GV/910GL Integrated Graphics Controller (rev04)
1GB RAM (I just replaced my RAM today from 256MB)
80GB HDD

When I load Zorin OS 5, after a few minutes, it freezes my computer so I'm forced to hard reboot. However, when I booted it on low graphics mode, my computer doesn't freeze. I'm on low graphics mode right now.

Is there anyway I can do (besides replacing my graphics card...) to fix this error so I can run it without having to revert to low graphics mode? If there isn't, is there another Linux distro similar to Zorin OS 5 that is compatible with my graphics card?

### Post by Emeka94 on 2011-07-28
Never mind. I think I solved the freezing. I just had to lower the settings on the compiz config settings manager which completely disables all visual effects. I guess that my graphics card isn't powerful enough to run alot of animations and effects.
