### Post by sagarthegreat1 on 2012-09-07
open your mint menu,and in the search box enter "network"
you'll see three links
1)Network
2)Network Connections
3)Network Tools

-choose the second option, network connections.
-After network connections window opens, click on the last tab named "DSL"
-Click "ADD"
here you get the option to enter username, service and password.
-enter any suitable connection name & save.
now, you can see the newly created connection on clicking the network icon besides the clock at the bottom!
