### Post by doctorbighands on 2011-11-01
As soon as I see "full screen HD video locks up my machine", I start thinking GPU issues.

It looks like you have a nVidia GeForce 6150SE with the MCP61 driver. Here's what Wikipedia has to say about that:
> MCP61 introduced a bug in the SATA NCQ implementation. As a result, Nvidia employees have contributed code to disable NCQ operations under Linux.

### Post by hailtothethief on 2011-11-02
Alright, here's the crash course. I'm going to act like you've never used a terminal before when I explain all of this, so please forgive me if I seem patronizing (I don't mean to be).

Once you've downloaded and extracted the file, you'll need to open up a terminal (Accessories -> Terminal Emulator).

Keep this in mind: when typing in a terminal, if you type most of a file or folder name and then hit 'Tab' on your keyboard, the terminal will take its best guess at what you're trying to type and complete it for you.

With that in mind, you'll need to change to the directory that contains this 'NVIDIA-Linux-x86_64-285.05.09.run' file. If you extracted it to your Desktop, then you would type

```
cd ~/Desktop
```

in to your terminal. If you then type

```
ls
```

you should see NVIDIA-Linux-x86_64-285.05.09.run somewhere in the output. Assuming that you do, you're on the right track. Next you should run

```
chmod +x ./NVIDIA-Linux-x86_64-285.05.09.run
```

which marks this file as okay to execute. Once this is done, typing

```
sudo sh ./NVIDIA-Linux-x86_64-285.05.09.run
```

should fire up some sort of installer. You're doing this as root so be mindful.

Hope this helps! Let me know if you need more.
