### Post by drawkcab on 2011-08-10
My 2 cents is that you should distro hop focusing on sampling three things:

1.  Different desktop environments.  Fire us some live sessions and give all the DEs a try--Gnome, KDE, Openbox, XFCE...etc.  Figure out what you like in terms of features.

2.  Differently scaled distros.  What I mean is that there are some fully packed distros that weigh in at 3.5 gb compressed and then some ultra light,10mb distros that you can run from an sd card straight to RAM.  It's good to get a sense of what can be accomplished with each scale of distro.

3.  Different package managers.  Try out some different package management systems.  SUSE, Fedora, Debian, Slackware, Gentoo, Arch all have a different way of managing their software.  I'm comfortable with Debian and lost in all the others.

### Post by -kg- on 2011-08-20
You know, something else you might want to try is one of the Live CD/USB based distros, like Puppy Linux.  The most recent releases of Puppy (Lucid Puppy) are based on Ubuntu Lucid, but are very minimal and run well on older hardware with fewer resources.  However, they have access to the Ubuntu repos and, depending on the size of your media (I have Puppy on a 4 GB Flash drive...installed a bit of software and there's still scads of room) you can install whatever additional software you desire.

The big thing about Puppy, though, is that it is designed to load into and run entirely from RAM.  On their website, they emphatically state that it is not suggested that you install it on your hard drive (you can, though).  Running from RAM means that, after loading from the media, there is little or no hard drive access...you get blinding speed (the speed of your RAM and processor access).  You can run it in as little as 128 MB (or less) of RAM, though more is helpful if you want to install and run some extra software, or do some multitasking.

It's another fun thing to experiment with, and you don't even have to use any hard drive space to do it.  It will use Swap if it's available, but unless you mount one of your partitions, nothing you do will ever see the hard drive.  That's very useful for some sensitive operations, like on-line banking and the such.  No access to any sensitive information on your hard drive, and you can select not to save the session, so that information will be nowhere once you exit it.

They also have many different "releases" of Puppy that specialize in different things or are for different uses, and it's very configurable.  Check out the Puppy website for further information.

I also have Knoppix Linux, which is another distro that is intended to be run from Live CD/USB.  Its intended use is different, though...as a "recovery tool".  It's a rich distro, though, and well worth checking out.  The "DVD" version has a load of tools and various software pre-installed, but the regular version is rich in its own right.
