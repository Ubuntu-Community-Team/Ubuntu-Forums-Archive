### Post by Stonecold1995 on 2012-09-25
That depends on what distro you want.  If you want Ubuntu, then yes, go to the official site (they give direct downloads and torrents).  If you are looking for some obscure distro you might have luck on The Pirate Bay or h33t, but there's far less quality control there than there is on a distro's official site so you never know what you might be downloading.

If you can't find a well seeded torrent (or if your ISP throttles torrents) then you can use a download accelerator with the direct download.  My favorite is Axel.  Axel is used much like wget, but it makes multiple simultaneous connections to download and that makes it a LOT faster.  I don't know much about Fedora, but I assume you'd install it with "sudo yum install axel".
