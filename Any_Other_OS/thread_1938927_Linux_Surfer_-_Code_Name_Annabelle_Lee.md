List of Security Features: - Hot Key New Users - Hot Keys Allow You To Launch Firefox, Opera, Midori as New User - Each Online App is protected in it's own user space - Password changes at each boot (but you don't need it unless you are doing admin work) - Firewalled with Firestarter- A gui that shows you attacks in real time and easy to use tools to block them. - Firefox browser protected by AppArmour - Launch Browser in Virtual Box to increase your security - Keepassx for you passwords - Encrypt downloaded files with TrueCrypt - Wipes your ram on shutdown with secure-delete
