### Post by mariocesar on 2010-08-25
Asegurate tener instalado usb-modeswitch

```

$ sudo apt-get install usb-modeswitch

```

I cuando lo reconozca, agrega una conexión con cualquier país, por ejemplo Argentina, edita la conexión y asegurate tener esta configuración

APN: internet.tigo.bo
Número: *99#
Ajustes IPV4: En Automatico

Seria todo, es bastante fácil :-)
