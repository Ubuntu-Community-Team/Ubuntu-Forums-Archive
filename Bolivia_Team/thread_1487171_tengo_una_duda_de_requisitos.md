### Post by mariocesar on 2010-05-19
Puedes instalar ubuntu 10.04, tener una configuración algo lenta. Para 256 de Ram talves te sea más útil instalar una distro más especializada en computadoras con bajos recursos, como DSL [http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/), Puppy Linux [http://puppylinux.org](http://puppylinux.org) y Xubuntu [http://www.xubuntu.org/](http://www.xubuntu.org/), otra cosa que puede ser más interesantes es instalar Ubuntu Netbook Remix [http://www.ubuntu.com/getubuntu/download-netbook](http://www.ubuntu.com/getubuntu/download-netbook), tienes que notar que tu procesador es algo lento para este año, y Ubuntu Netbook Remix tiene varias estrategias para aprovechar procesadores lentos.
