### Post by cprofitt on 2008-11-29
It would appear that you have modified the wiki and called a meeting... see how that goes. Other than that I would keep in mind that there are two paths for a LoCo.

A)  Down -- from Ubuntu to the local community. You take people already using Ubuntu and have them advocate for Ubuntu in your local community. This is done by going to LUG meetings, attending computer society meetings and assisting/sponsoring install fests.

B)  Up -- from the local community to global Ubunut community. From the local events you help local people find the Ubuntu community and assist them to become more involved.

Both of these have to happen... and it is often very tough.

I am going to be trying to host classes on how to use open source software through local recreation departments... and then from those make people aware of Ubuntu by that being on my demo machine... in addition I plan on handing out flyers to people about the next install fest.
