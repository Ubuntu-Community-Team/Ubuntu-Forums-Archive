### Post by angheloko on 2010-09-22
To my fellow open-source minded pinoys,

I would like to share my website which is inspired by the open-source culture - [WariPinas]("http://www.waripinas.com"), which roughly means "Imagine Philippines". It is a non-profit, ad-free, always-free, non-government affiliated website that aims to harness the power of one of the main fuels of open-source - collaboration.

Basically, the site aims to gather ideas, big or small, from any concerned individual on how to improve or help the Philippines. Some sites show the issues, but this site attempts to solve them. The idea itself doesn't have restrictions and is only restricted by your imagination. As what Einstein once said - "Problems cannot be solved by the same level of thinking that created them.", hence innovations are most welcome. 

If you have the time, I would appreciate it if you can check the site. If you like it and agree with the principles behind it, please feel free to join. If you have some criticisms, feedbacks or whathaveyous on how to improve or better the site, please do tell.

If you're interested to get more involved, there are a number ways like being co-maintainer/developer (volunteer basis) or by sharing and promoting the site to other Pinoys.

Thanks!

P.S.

Site was developed  in Ubuntu 10.04 using Eclipse IDE and gedit, images were created and modified using GIMP, framework is Drupal CMS, DB is MySQL, and the web server is Apache. So hell yeah, it was a breeze developing the whole site.

P.P.S.

This was inspired by a wonderful Ubuntu tool - [brainstorm.ubuntu.com]("http://brainstorm.ubuntu.com").
