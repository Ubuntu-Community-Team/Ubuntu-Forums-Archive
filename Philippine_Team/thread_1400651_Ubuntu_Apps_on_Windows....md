### Post by badrra on 2010-02-08
Your best bet is wubi. It is easy to install/uninstall. To Uninstall it just go to the add/remove program on windows and uninstall. 

Hindi ka na kelangan baguhin yung partitions kasi tatakbo yan using ntfs.

But if you want to run it using usb, then ito yung tutorial:

[http://www.pendrivelinux.com/create-a-ubuntu-9-10-live-usb-in-windows/](http://www.pendrivelinux.com/create-a-ubuntu-9-10-live-usb-in-windows/)

Pero kung gusto mo permanent installation you can either burn iso to cd or use unetbootin. 

Depende sa plano mo.
