### Post by Jucato on 2008-06-21
"Free" (as in freeware) wouldn't be enough for me to choose Windows over Linux, at least personally. Cost is the least of the issues plaguing Windows, as many already pointed out. I myself only have 2 reasons for using Windows: gaming and the occasional (also proprietary) software that won't run in Wine. Other than that, Linux is still the better OS.

Ease of use? Only because Windows is more familiar. The best thing about our desktops is that they can be setup to any situation. It grows with the user's needs and level.

Hardware and software support? Maybe. But not because Windows is the best platform for writing drivers and programs but because it's the most lucrative.

All in all, I'd say that GNU/Linux is still the best, well-rounded, operating system for whatever situation and need. I believe that Microsoft's only advantage is that it was historically first in going to the masses. So, even if Windows was freeware, I'd still go for GNU/Linux.
