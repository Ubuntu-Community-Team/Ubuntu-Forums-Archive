### Post by Jucato on 2008-05-17
No no. "Repositories" and "mirrors" are two different (but related, in this context) things.

A repository is a database/collection of packages for a distribution, arranged in a certain way to work with a certain package management system. Other distributions call it "sources", since it is a source of packages. Parang kung baga isang malaking lugar na mapagkukunan ng installers. Usually, a repository is online, like archive.ubuntu.com. But it can also be offline, like if you use the DVD or Alternate Install CD, or even a mounted ISO.

Now a mirror is different. Generally speaking, a mirror is a duplicate copy of a repository, usually found in another server in another country. It is a mirror image, a reflection, a copy, of the original repository. There are several reasons why there are mirrors, such as faster connection (use a mirror that is nearest to you) or redundancy (backups). Mirrors are not specific to repositories. It's common to find download pages mentioning mirrors as alternate places to download from.

EDIT: I forgot to add a link: [https://help.ubuntu.com/community/Repositories](https://help.ubuntu.com/community/Repositories)

### Post by ch1c0dj on 2008-05-17
Additional. Syempre, makatulong yan para madali ka mag-install, mag-updates, mag-upgrades at mag download ng bagong release ng packages or distribution.

Dahil sa repositories makukuha ang mga ito (packages, updates, iso cd-image,) marami mga mirrors ito around the web para mabilis pagdownload depende kung saan geographical location o internet connection na gamit mo. Ang mirror ay isang servers na may parehong content ng orihinal repository server, sa madaling salita duplicate server ito.

Hindi kasi lahat ng packages eh nasa CD o DVD na ginamit natin sa pag install. Convenient gumamit ng Repository as a source of packages.
