### Post by aljoriz on 2010-10-28
If you are using Chromium and you want to use the new ubuntu font on all viewed websites? 

Try the ubuntu font extension for chromium.  

Click the wrench icon > Tool > Extensions 
Press Get more extension and search for ubuntu font.. install and ENJOY!

### Post by topet2k12001 on 2010-11-21
Hi,

In my case, even my Opera, Mozilla Firefox, and Google Chrome are using the Ubuntu Font. :)

You may simply download the Ubuntu Font from the Ubuntu Software Center. And then just configure your respective browser (all of the browswers I mentioned have Options/Preferences for you to set your desired font). Likewise, for setting the font all-around (applications, Menus in Ubuntu, etc. not only your browser) you can use the Ubuntu Font. Just type ```
ttf-ubuntu-font-family
``` in the Search Bar of the Ubuntu Software Center and install it. Hope this helps! :)
