### Post by Laibcoms on 2009-02-13
Use PlayOnLinux -> [http://playonlinux.com/en/](http://playonlinux.com/en/)

Explanation of PlayOnLinux -> [http://wiki.winehq.org/PlayOnLinux](http://wiki.winehq.org/PlayOnLinux)

Copy-Pasted below:


**A prefix's matter**

 When you install some applications with Wine or some other Third-Tools all your file are saved in the same prefix  
[[IMG]http://www.playonlinux.com/doc/images/wine_en.jpg[/IMG]]("http://www.playonlinux.com/doc/images/wine_en.jpg") 
In PlayOnLinux we use different prefix for all application, so we can do every specified configuration for each game 
[[IMG]http://www.playonlinux.com/doc/images/playonlinux_en.jpg[/IMG]]("http://www.playonlinux.com/doc/images/playonlinux_en.jpg") 
 
**Manage Wine Version**

 You have see in wine appdb's a software with a platinum rank for an older wine version and now it's ranked as Garbage ?  
Where is the problems ? in PlayOnLinux you can assign a version of Wine for each software ! 
 
**Plugin**

 PlayOnLinux Plugin's what is it ? 
It's a plugin to add something in PlayOnLinux like "Capture" (by GNU_Raziel use it for record your game (Fraps-like)), "[TransgamingCedega]("http://wiki.winehq.org/TransgamingCedega")" (by Tinou, use it to import installed game with cedega in PlayOnLinux)), "[DosBox]("http://wiki.winehq.org/DosBox")" (by Tinou) ... and other !
