### Post by tech-hero on 2011-03-17
Because they are considering the fact that consumers tend to increase the use of their computer resources as data get more and more complicated.
 
IMHO, linux as an operating system almost remains the same..but the programs used in it may require more..
 
for example, GNU linux requires a minimal system requirement, but you have to require more for you to be able to use some programs to enhance user experiences, (like Compiz and GNOME desktop).
 
I guess the same thing goes for other operating system. they are considering the programs to be used, and the data to be processed.
