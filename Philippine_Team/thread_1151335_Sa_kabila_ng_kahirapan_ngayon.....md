### Post by ragadanga63 on 2009-06-07
Consider me a dummy and here's how I got file sharing to work between Ubuntu and Windows:

1) right-click on folder/file to be shared.  Click on "Sharing Options"
2) Ubuntu will prompt you if you want to install Samba Sharing...click "OK".  Samba will be downloaded and installed to your system.
3) When (2) is completed you can now start sharing folders/files/printers.  Repeat (1).
4) When Ubuntu asks for a password and your user password doesn't work you may do these steps:

   a)  using the CLI (Terminal), type these commands one at a time:

       **sudo smbpasswd -L -a your_username ** 
            *(follow on-screen prompts)*

       **sudo smbpasswd -L -e your_username**  
            *this is to enable the Samba password you added above*
