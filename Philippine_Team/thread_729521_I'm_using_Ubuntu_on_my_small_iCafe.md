### Post by elyserva on 2008-03-19
Just want to share this. I've been using Ubuntu on my small iCafe in the province for several months now. I started using them when my Windows started acting up because of viruses, hardware problems, etc. Some WS won't install XP anymore. Some do but would reboot for no apparent reason. Mainly because of hardware problems, mostly defective memories. To my surprise, Ubuntu installed and ran on them without any problems.

I have few complaints from my customers but, mainly because I don't have games anymore. And some, because of unfamiliarity with some applications. Overall, however, it's doing great.

I was thinking of using CafePilot (it can be downloaded for free) as the Cafe Management SW but the customers don't like buying freepaid cards (remember, I'm in the province). So I have to go back to timing them manually.
