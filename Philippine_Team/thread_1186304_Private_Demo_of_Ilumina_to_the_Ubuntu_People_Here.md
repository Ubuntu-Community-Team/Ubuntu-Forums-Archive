Hi guys, 

last June 12, 2009 the Inovent team had a private demo in Fort Bonifacio on the Ilumina and right now the Chief Inoventor Brian Quebengco is asking if you guys want a private Demo since the Ilumina is based on Linux and the inovent team would love to have your inputs/comments and suggestions. 

Please post here, if you guys want to have a private demo with the Ilumina :D so that you could ask all your questions too. 

[IMG]http://farm4.static.flickr.com/3577/3569803383_39f0aee5d2.jpg?v=0[/IMG]

I'll be forwarding all your replies to the Chief Inoventor Brian Quebengco, 

I'm a volunteer for the Inovent's Ilumina :D

inovent Ilumina Website : inoventrevolution.wordpress.com

Specs and Features : (Current though not yet final) 

32' inch LCD 
1.5 TB Hard Drive
Intel Atom Processor (to be changed)
Blue-Ray Slot Drive
4 USB Slots (2 side, 2 back) 
HDMI Plug
Hi-Definition Web Camera [Specially made to capture an entire living room]
Motion Sensor for Remote Control
Integrated Remote Control [Karaoke, Can be used as Phone, Remote Control, Motion based mouse controller]
VOIP 
Linux Based OS 
Built-in Video, Picture and Music Playback software 

The Inovent Ilumina team is planning to release SDK/APIs for software developers to build upon the system and create applications to further add value to the product
