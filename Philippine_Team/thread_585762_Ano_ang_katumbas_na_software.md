### Post by kopinux on 2007-10-21
> If I have Ubuntu do I need to go to console to add new software? of just do the same how Windows Xp works, like click n click.

tatlo ang klase pwede mag install ng software sa ubuntu
1. Add/Remove Applications = click check click lang okay na.
2. Terminal = i ta-type mo halimbawa yung wine [COLOR="Red"]sudo apt-get wine[/COLOR]
3. .DEB package = kung available and .deb kahit double click pwede na, parang ito yung .exe natin.

meron pa yung pag compile ng source code, hindi rin ako marunong nun.

> Ano- ano ba ang katumbas na software para sa Ubuntu?
meron counterparts, pero kung desperado pwedeng i emulate gamit ang wine.

> -Kiddy Games/Educational to ?
sa edubuntu pre-installed na yan, madaming kiddy games at educational materials, pero kung naka ubuntu nasa add/remove applications lang yan.

> -Macromedia Dreamweaver to ?
NVU, kaso napabayaan na, pero ngayon kinuha ng ng mozilla (makers ng firefox) at pinalitan ng Kompozer, medyo napagiwanan pero dahil opensource siguradong mabilis mag iimprove ito tulad ng firefox. pero kung kailangan talaga ng Dreamweaver, kaya naman ito ng wine medyo mahirap nga lang. [http://ubuntuforums.org/showthread.php?t=200305](http://ubuntuforums.org/showthread.php?t=200305)

kung gusto mo ng madali, bili ka nalang ng crossover office, mga P2,000 yun pero 100% compatible at easy ang mga MS softwares.


> -Adobe Photoshop to ?
GIMP, konting tiyaga lang pag nasanay okay na, pero hindi parin kasing dami ng features tulad ng photoshop na maraming effects at tools, kaya sa wine, marami nag sasabi na compatible sa wine ang Photoshop CS2 wala ka daw magiging problema. ang disney ay gumagamit ng linux+wine+photoshop para sa digital animation nila.

> -Media Player to ?
wala kang magiging problema dito, kung minsan nga mas mabilis sa linux at maraming napapatakbong media file type, kahit ipod pwede i sync.

> -YM/MSN to ? (webcam applicable)
gaim, amsn, gyachi (webcam applicable, compatible only)
pero yung bago [http://webmessenger.yahoo.com/](http://webmessenger.yahoo.com/) okay ito kaso beta palang at ginagawa palang yung voice at webcam features.
