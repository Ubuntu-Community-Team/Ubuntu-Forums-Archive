### Post by ragadanga63 on 2009-08-30
Emerald makes your window borders look a lot better than plain metacity.  

To install Emerald, you may use Synaptic and do a search for "emerald".

After installing, you must activate Emerald. 

Open your CompizConfig Settings Manager.  Click on Window Decoration plugin, and under "Command" settings, replace "/usr/bin/compiz-decorator" with "emerald --replace".  Enable Window Decoration plugin.  Emerald will take effect after you restart.

For Emerald themes, go to [http://www.compiz-themes.org/index.php?xcontentmode=103](http://www.compiz-themes.org/index.php?xcontentmode=103)

To install a theme, start Preferences>Emerald Theme Manager and use the import button to install any downloaded theme.

Above instructions assume that you already have Compiz running and CompizConfig Settings Manager installed.
