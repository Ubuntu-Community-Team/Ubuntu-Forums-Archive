### Post by noypi on 2013-01-06
> **jhayczee09 said:**
> Good evening, I am enhancing the functionalities my xTuple by creating new reports and forms.. But I find difficulties in programming this software like getting the currently login user. Please help on how can I get the who the current user is and indicate it on my report. Thanks.

Hi jhayczee,

Wwhen you say 'getting the currently login user,' do you mean you need to know who is currently logged in? You can see this by clicking on the 'Maintenance' button on the leftside of the xTuple desktop. There should be a window in the lower portion which shows the users currently logged in. if it doesn't show up, you can activate it by going to "Window > Desktop > Users Online" in the menu bar.

Unfortunately, I am not fluent in QT programming (for the front end/interface) nor in openRPT (for report generation). But you can try using JasperReport Server. It has a more friendly UI, and is a little easier to create custom reports.

There is a forum dedicated to specific xtuple questions: [http://www.xtuple.org/forum](http://www.xtuple.org/forum)

there is a wealth of information there, and i'm sure the users there can be of more help than me. :)

if you would like professional help, i can give you the contact details of a local xTuple partner.
