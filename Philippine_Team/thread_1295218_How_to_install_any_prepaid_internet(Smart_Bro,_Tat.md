### Post by killer_d76 on 2009-10-19
i'm using a Huawaei E160 USB Modem (you can get this info by typing "lsusb" in the terminal once you have your usb modem plugged in) from SUN Cellular.. and all have to do is edit the APN from network manager from "minternet" to "fbband" since i'm using a post paid broadband connection, if I would be using a prepaid one, i just need to change it back to "minternet".

- though by default.. if you plugged in your choice of USB modem it would automatically detected by your Ubuntu Linux System.. not unless your using Hardy Heron.
