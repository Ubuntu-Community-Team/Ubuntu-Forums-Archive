### Post by eilu on 2008-02-21
Kung gusto mo ng rogue-likes/Diablo-style na adventure where you get random dungeons and levels, I an recommend:
Powder: [http://www.zincland.com/powder/](http://www.zincland.com/powder/)
Lost Labyrinth: [http://www.lostlabyrinth.com/](http://www.lostlabyrinth.com/)

Powder is truer to the genre that Lost Labyrinth; Lost Labyrinth is great for quick runs (finish a level, save...) since it's purely kill-monsters-find-treaures.

Notes:
Powder needs to be run from its own directory or else it will end up saving its files all over your home directory (I use a bash script)
Lost Labyrinth's install somehow puts the folder as root; kailangan mong palitan yung ownership ng files (chmod or gksudo nautilus) para makapag-save ka.

Another good game is Linley's dungeon crawl: nasa repos ("crawl"); text-based... very old school- but lots of options, character races, classes, skill sets... there's a graphical frontend available sa sourceforge but I couldn't figure out how to run it.
