### Post by Samhain13 on 2010-06-24
You can connect to the Internet using the provider at home (A)? But you can't get updates from Ubuntu? If that's the case, try changing to another Ubuntu repository:

1. Open Synaptic: System > Synaptic Package Manager.
2. Bring up the Software Sources box: Settings > Repositories.
3. Under the Ubuntu Sofware tab, change the value of "Download from" to "Other...", this will bring up a new box that has a button labeled "Select Best Server". Click that.
4. When the thing finishes testing the available servers, click "Choose Server".

Hopefully, you'll get a repository that's good for your home Internet connection. :D
