### Post by jaceleon on 2009-10-20
Just use the add/remove in the applications. then find and install the following (use the search tab there):

These ones are required to be installed:

java system runtime (for uploading pictures in facebook) keyword:java =5 stars
adobe flash plugin (for running flash in webpages)keyword:flash =five stars
ubuntu restricted plugins (for playing mp3 and all sorts of files)keyword:ubuntu =1 star

also search nvidia in the add/remove software there, then select the driver pack you need based on the model of your nvidia card (mine use package 96, since I use Nvidia MX440 SE) read the description so you may find your card model there. This is mainly to improve your graphics. If I'm correct there are 6 or 7 packages of NVidia drivers there. Keyword is Nvidia.

hope I have helped you there. Galingan mo pare!
