### Post by Pinoy Tux on 2010-07-27
From what I heard, the T13 was supposed to replace the T10.  Fortunately, ang T10 ay gumagana using the T20 driver.  Ngunit under Lucid Lynx yung stock version ng libgutenprint2 at cups-driver-gutenprint packages (v5.2.5) ay hindi compatible sa T10 kaya kailangan mag-downgrade muna to v5.2.4 bago i-setup yung printer.

Try mong i-downgrade muna yung 2 packages then install mo yung T13 using the T20 driver if you can get it to work, and provide feedback to this thread.  Gusto ko rin sanang bumili ng T13 dahil pa-phase-out na daw ang T10 (if not already).  Otherwise I'd still stick with the T10 model.
