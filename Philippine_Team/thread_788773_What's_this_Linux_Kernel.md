### Post by Jucato on 2008-05-10
[https://help.ubuntu.com/community/Kernel](https://help.ubuntu.com/community/Kernel)

Short answer: the kernel is the heart of any operating system. It is in charge of managing system resources, such as input/output devices, memory, processor, etc. In Linux distributions, this kernel is the Linux kernel.

Some argue that the Linux operating system should actually be called GNU/Linux, since the kernel is really only part, but an important one, of the whole operating system. The rest of the system is provided by tools and applications coming from the GNU Project.

[http://en.wikipedia.org/wiki/Linux_kernel](http://en.wikipedia.org/wiki/Linux_kernel)
