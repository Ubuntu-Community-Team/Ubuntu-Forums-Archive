### Post by killer_d76 on 2008-05-20
hi elord.. as far as i understand in order for you to properly dual boot your computer with ubuntu, you need to install "windows" first, then install "ubuntu" next.. don't worry because ubuntu has a step by step procedure on how to dual your computer properly without the need for you to partition your hard disk drive first or  use a boot manager!.. if your computer has a windows installed on it already, what i could suggest for you to do is to "defrag" your hard disk first then install ubuntu, to avoid some of windows files from being deleted!.. but always never forget to back-up your files ;)
