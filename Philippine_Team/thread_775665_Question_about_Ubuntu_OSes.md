### Post by Samhain13 on 2008-05-01
It's a semantics issue. There's the official usage that we find in the Ubuntu website, where every one is an "edition". There's also the informal that we use amongst ourselves (end users).

While it's true that they're all "editions" since, like literary works, they've been edited to suit a particular audience, some users would call *buntus that have been edited *to look and feel* a particular way (i.e. Gnome, KDE, XFCE, etc.) "flavours", and *buntus that have been edited *to serve a particluar purpose* (i.e. desktop and server) "editions".

So when we talk here or in the IRC and we'd like to know what *buntu a specific user uses, we ask them "What flavour are you using?" To which they'd possibly answer, "I'm using Kubuntu" or "I'm using a Gnome/KDE hybrid".

But when we ask them, "What edition are you using?" We can expect something like: "the desktop edition", "the server edition", "Ubuntu Studio", or "the desktop edition with a real-time kernel".

### Post by upforthedownstroke on 2008-05-01
Well, the way I think about it is that there are a few versions of Ubuntu that maintain the regular release schedule of six months, and some that don't.
Ubuntu, Kubuntu, Edubuntu, Xubuntu, Ubuntu Server, and I believe Gobuntu all maintain a six month release schedule and tend to be a little more mainstream, with the exception of Gobuntu which contains nothing but GNU (free) software. 
The other versions, like Ubuntu Studio and some other hybrid releases are not considered an integral part of the Ubuntu Project and do not maintain a regular release schedule, although they are recognized as offshoots from Ubuntu itself.
The interesting thing is that you can run any desktop environment (KDE, Gnome, XFCE, etc) from any version of Ubuntu, or at least in theory. For example, if you install Xubuntu, you still have the same virtual terminal as Kubuntu, and in theory you could remove or disable the XFCE environment and install the entire KDE suite (or kubuntu-desktop) from the terminal. Basically, they're all interchangeable depending solely on your preference.
