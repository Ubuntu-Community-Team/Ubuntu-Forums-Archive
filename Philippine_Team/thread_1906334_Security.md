### Post by NickelSulfate on 2012-01-09
base po dun sa nabasa kung blog depende daw po..kung nakikipag xchange ka po ng file sa win user better mg install ka,,kawawa naman po kasi sila(sila mai-infect ng virus etc)..about naman po kung yung nitatanong mo is affected ba tayong linux user, hindi po masyado kasi pag nadownload naman po natin yung virus(mostly pang win-os) unknown file naman po..or kung sakaling makapag download ka by accident ng virus na pang linux(hindi ka naman po siguro nag ddownload/browse ng naka root diba). batay nga po dun sa blog site na nabasa ko 90% daw pong secure yung os po natin pagdating po sa ganyan...

BTW kung nid mo talaga ng anti virus may mangilan-ngilan na supported po tayo: eto po code na alam ko if you like lang naman po sudo apt-get install clamtk

or visit ka po sa mga site ng anti-virus


Note: ang anti-virus po ay di nag aalis/nagtatanggal ng virus, nag dedetect lang po talaga ang silbe nun..:salute:
