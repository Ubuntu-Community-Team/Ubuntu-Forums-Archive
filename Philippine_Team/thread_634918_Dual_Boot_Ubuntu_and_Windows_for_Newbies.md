### Post by elite_angel on 2007-12-12
Wubi, is an executable file wherein, it will download the whole image from the net and install it inside windows. It will ask you for the allocation size (MB) for your Linux (ubuntu, kubuntu, xubuntu, edubuntu). After a successful install, Your hard disk is already partitioned in the size you allocated...

Perfect for windows users who want to explore Linux. No Need for OS reinstallation, just make sure you have enough FREE SPACE in your drive for the Linux OS.
