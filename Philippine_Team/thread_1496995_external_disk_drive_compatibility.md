### Post by nerdtron on 2010-05-30
External hard drive usually have NTFS file system. This is good since both Ubuntu and Windows can read/write on this file system.
For sure compatible ang mga external disk drives sa Ubuntu.
But if you share files with Mac OSX, dapat FAT32 ung file system ng external drive para magamit mo ang external drive para magshare ng files between the three OS.
