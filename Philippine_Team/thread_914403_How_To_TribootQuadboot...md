### Post by OutOfReach on 2008-09-08
Tribooting/Quadbooting/Pentabooting(lol) is the same process as dualbooting, just install the OS on separate partitions.
You can also use VirtualBox or VMWare to use another OS inside an OS, but dual/tri/quad/booting would be somewhat faster as in performance.

EDIT: No, Linux distros can share the same SWAP.

I agree. Ubuntu and Kubuntu are practically the same. It's just that one uses a different desktop environment from the other - Ubuntu uses GNOME while Kubuntu uses KDE. Instead of installing Ubuntu, Kubuntu and Edubuntu, just install and switch from one environment to another. This will save you disk space.

As for Edubuntu, you can install all its educational software within Ubuntu as they share the same repository.

But of course, if you have lots of disk space, go ahead and install all Ubuntu variants. :)

### Post by pendletone on 2008-09-11
Well, I haven't tried installing Edubuntu on my system kasi it's geared towards school use, especially young children and teachers. What I know is that Edubuntu comes with the **KDE Education (or Edutainment) Project** which is a collection of educational apps that includes some of the following:

1. Kig - geometry stuff
2. Kalzium - (more than just) a table of elements
3. KBruch - to practice calculating fractions with
4. KHangman - a hangman game
5. Parley - memorization aid

Outside the KDE-Edu Project, Edubuntu also includes:

6. GCompris - collection of kindergarten apps
7. Tux4Kids Project - another collection of educational apps for kids (w/c features Tux, the beloved Linux mascot) :)
