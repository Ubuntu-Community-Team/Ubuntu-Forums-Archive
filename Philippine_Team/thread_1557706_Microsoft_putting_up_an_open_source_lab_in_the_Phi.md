### Post by jsgotangco on 2010-08-21
Microsoft maintains 2 open source licenses that are recognized by OSI:

Microsoft Public License
Microsoft Reciprocal License

Not to mention that they also run a community that hosts a lot of open source projects built on Microsoft technologies at Codeplex. Our company has participated on Codeplex via the Npanday project ([http://npanday.codeplex.com/](http://npanday.codeplex.com/)). We recognize the fact that operating systems, whether open source or proprietary only acts as a platform to build software hence, it shouldn't be a surprise for professionals to participate when necessary.
