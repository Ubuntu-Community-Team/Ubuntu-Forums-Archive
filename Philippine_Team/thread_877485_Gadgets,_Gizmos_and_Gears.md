How do I check to see if I have a Fake MemoryStick?

Easy! On Your PSP Go to Home>Game>Information> If you see Unknown or Unsupported then you know you have a FAKE SonyStick! If it says Supported, then it is genuine.

Ok, so before you get started on this, you will need:

• A psp with ANY custom firmware installed and ready.
• A battery you want to Pandorise (This is reversible)
• Sony Firmwares 1.50, 3.71 and 3.80
• A memory stick PRO DUO (256MB-4GB)

Let's Get Started!

[1]- If you have ANY Custom theme installed, it will cause problems so disable it NOW!
[2]- Backup anything you need to backup and Format you memory stick using the PSP's option. [b]NOTE: If you have a fake Memory stick, use the format memory stick option included with the UPMS installer.
[3]- After you have downloaded UPMSInstallerV3r3.RaR, Extract it and go into USB mode, after that copy the UPMS installer.cso into your ISO folder
[4]- Copy the following Sony updates to the ROOT of your Memorystick:
* 1.50 Named as 150.PBP
* 3.71 Named as 371.PBP
* 3.80 Named as 380.PBP
* 1.50 Named as UPDATE.PBP (Yes you will need 2 1.50 updates but with different names)
[5]- Disable USB mode and in your psp, Go to > Game > Memorystick > Run the UMPS installer just like you would with a normal ISO\CSO file.
[6]- Follow the steps in order (1-4) from the UPMS program

Follow each step in order:

Step1:- Select the 'sleep fix Multi-IPL' option
Step2:- Backup your eeprom, then create Your battery
Step3:- Install UPMS without utilities (Fastest)- Or if you do want the utilities install the other choice.
Step4:- Here you can install the utilities of your choice if you chose the" Install UPMS without utilities (Fastest)" option.

=====================================


How to Use the UMPS:

FAT PSP:

->Switch off your psp
->Insert the UPMS memory stick
->Hold down the L trigger button
->Insert Pandora battery
->Release L trigger button when the Ultimate Loader menu appears
(If your psp boots to the XMB, it means you didn’t hold the L trigger button down)

SLIM PSP:

->Important: Make sure DCv3/v4 (cemetery) was the last active folder used, or you will have trouble getting the UL menu to light up-some people have had problems in the past attempting to use the UPMS on slim psps because of this...

In which case you will need to do the following:

->Power off and remove your slim battery.
->Push the power switch down so its in the HOLD position
->Insert the UPMS memory stick
->Now hold L and insert your Pandora Battery.
->Release L when the MS light or screen flashes
->Ultimate Loader will then switch the active folder to the cemetery or DCv3/4
->Carefully return the power switch to the middle or the ON position-making sure you dont put the psp in sleep mode
