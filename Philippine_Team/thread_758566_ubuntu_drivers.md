Nung una din parang walang nangyayari pag-pinipindot ko ung Fn+F11, so akala ko di naka-map ung keystrokes na un sa Ubuntu

Di ko talaga alam kung alin dun sa mga ginawa ko ung tumulong, anyways sabihin ko nalang lahat:

Una, meron akong napansin dun sa BIOS na parang Installed OS, gnawa kong Linux un.

Tapos, naginstall ako ng ndiswrapper, tska ung gui nya...

```
sudo apt-get install ndiswrapper ndisgtk
```

Kinuha ko yung driver disk na kasama nung laptop, hinanap ko ung WLAN (nasa others na folder ung sakin)

Ung ndisgtk nasa System>Administration>Windows Wireless Drivers

Hinanap ko ngaun ung .inf na file nung driver para mainstall.

sasabihin nya wala daw gnung hardware, so eto, nagpost ako dito :D,
tpos pinindot ko ung Fn+F11, ok na :)
