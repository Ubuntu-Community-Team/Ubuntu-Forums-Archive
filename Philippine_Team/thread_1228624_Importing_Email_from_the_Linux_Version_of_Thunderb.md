### Post by daxumaming on 2009-08-01
download and install the Import/Export Tool Extension for Thunderbird. Then export your emails as mbox so you can then import into Evolution. [http://nic-nac-project.de/~kaosmos/mboximport-en.html](http://nic-nac-project.de/~kaosmos/mboximport-en.html)

You can also do it manually by navigating into ~/.mozilla-thunderbird/randomstring.default/Mail (but I would highly suggest using the extension instead)
