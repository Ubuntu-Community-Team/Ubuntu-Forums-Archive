### Post by benhur99ph on 2008-06-06
The best way to try Ubuntu is by using the Live CD. You can get a feel of what the OS looks like and how it functions. But it will run slower because its reading from your CD - drive as you go. With the latest Ubuntu release which is Ubuntu 8.04 Hardy Heron, anyone can easily install Ubuntu via Wubi. It's just like installing a program within Windows. If you want to remove it then just go to your "Add/Remove Programs" and uninstall it. Basically, Wubi is like using  a Live CD without the slowness. 

For more information about Wubi, go here --> [Wubi discussion thread...]("http://ubuntuforums.org/forumdisplay.php?f=234")
