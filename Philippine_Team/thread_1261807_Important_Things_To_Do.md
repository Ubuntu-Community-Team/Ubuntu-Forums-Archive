@Extra Rice:
mabalik sa topic, the first thing that i do (aside from updating the system) is to install packages that support mp3 playback, flash, etc. you can do this in one go by typing: 

for ubuntu:
```
sudo apt-get install ubuntu-restricted-extras
```

for kubuntu:
```
sudo apt-get install kubuntu-restricted-extras
```

for xubuntu:
```
sudo apt-get install xubuntu-restricted-extras
```

that's just one of the many steps you can customize your ubuntu setup. :)

### Post by Samhain13 on 2009-09-10
Install ka ng ClamAV at RKHunter. Na sa regular repos lang yung dalawang yun. Yung ClamAV may GUI din para di mo kailangan lagi na na sa command line. Hindi ko nga lang alam kung gaano kadalas ma-update yung virus information niya; pero so far, nakikita naman niya yung mga virus na dala ng pamangkin ko galing sa school o sa cafe (sakin kasi pinapa-check bago nila gamitin yung mga flash drive sa Windows lappie nila sa bahay).

Yung RKHunter naman, kahit patakbuhin mo ng once-a-week, kung personal use lang yung box mo. Para lang malaman kung may rootkit nga ba sa system mo (of course, kung meron ay hindi maganda!). Nga lang, tuwing mag-update ka ng Kernel o kung anu-mang low-level na apps, kailangan mong mag-"property update"-- nakalimutan ko yung switch, hanapin mo na lang sa man pages.
