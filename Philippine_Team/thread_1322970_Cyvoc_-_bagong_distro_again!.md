### Post by jaceleon on 2009-11-11
Well, guys, me bagong OS na dinedevelop ang mga tao na nagcreate ng XP Black Edition at LastXP Series.

Ang name ng OS ay Cyvoc ([www.cyvoc.net](www.cyvoc.net)). Isa raw itong Ubuntu-based OS na ang objective is to make this distro run Windows application without using WINE as much as possible. Parang attempt nila to to fully get away from windows and run a safe OS using Linux. Ganda nga ng screenies e. Yung themes nila ay XP, Mac OS X, Vista, Seven, Ubuntu 9.10 at 3 custom, yung isa parang Windows 98.

Yung OS na yun ay currently umaasa sa repository ng Jackalope Jaunty pero they promise na on the first public release, sa Karmic ang main repository nila, though you can add any repository you prefer. And mukha siyang promising. So far, ang first public release of the OS is on December, so now is like beta testing (di ako makakuha ng iso, need mo pang mag-apply as beta tester to download it, since it's just on it's first beta, like v.0.1) also, supported din nito ang KDE at XFCE aside sa default na GNOME.

All of its Linux application will be debian-based, so mas OK para sa atin yun!

So the best feature it promises: Running the Windows application inside it as if Windows ang gamit mo.

Anyway, feel free to comment or help these guys. This OS looks too good para palampasin!
