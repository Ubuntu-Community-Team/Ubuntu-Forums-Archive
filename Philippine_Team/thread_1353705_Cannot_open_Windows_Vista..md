Hayz, nakakaasar nga yan, pero Macro$h!t is Macro$h!t, so the virus scares are a part of the package deal:D.

I advise for a full Ubuntu on your laptop then get VMware and a XP/Vista, one way or another, as long as you have the system requirements.

If you like, pasahan kita tweak para tumakbo ang XP mo under 64MB RAM only. That tweak disables a LOT of unnecessary stuff (themes, whatever services that shouldn't start in the first place, etc.) but still makes XP functional. Yun nga lang, okay yun gamitin in a virtual environment than your own PC.

On my case though, full Ubuntu installation is what I have sa laptop ko. Whether Karmic or Jaunty gamit mo, install AptOnCD then backup every single thing you just installed, then FORMAT the whole computer (to sanitize it of all tainting Macro$h!t elements:D) and install Ubuntu the way you want it. Then install AptOnCD again and restore every single thing you stored on the backup. Ayun, lahat ng nainstall na apps mo on Ubuntu ay babalik.
