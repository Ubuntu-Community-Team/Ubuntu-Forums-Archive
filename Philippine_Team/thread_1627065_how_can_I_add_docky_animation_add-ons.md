### Post by oobermensch on 2010-11-21
Well, the animation add-ons for Compiz will not give Docky any new effects. As far as I know, Docky doesn't have any extra animation plugins.

To install the extra Compiz plugins, open Synaptic package manager. Then in the quick search box, type in *compiz extra*. It should narrow down the list to 2 or 3 items. Just put a checkbox on the compiz-fusion-plugins-extra. Click apply then wait for it to install.

### Post by c00lwaterz on 2010-11-25
add the software source ppa:docky-core/ppa 
```
sudo add-apt-repository ppa:docky-core/ppa
```

install the package docky. To do this from a terminal, use the commands 


```
sudo apt-get update
```
```
sudo apt-get install docky
```


You can only change themes and icons on docky and some plugins
