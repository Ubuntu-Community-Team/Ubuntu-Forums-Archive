Ubuntu - uses Gnome desktop environment/ graphical interface

Kubuntu - uses KDE desktop environment/ graphical interface

The main difference would be the environment or the usability. Iba ung look ng Gnome sa KDE. Plus there are programs din na for specific environment lng. 

Ngaun ung sa LTS naman, eh it just stands for Long Term Support, which means mas matagal ung security updates nung mga LTS release kesa sa hindi. Pwede mong i-check kung gaano katagal ung LTS. For example, if I'm not mistaken ung 8.04 na ni-release nung 2008 ay sa 2012 pa ung last update, pero ung 10.10 na ni-release nung 2010 eh sa 2013 ung tapos ng support...
