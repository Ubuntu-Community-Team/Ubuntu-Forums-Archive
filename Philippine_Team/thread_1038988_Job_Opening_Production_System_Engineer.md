Hi,

My employer is currently looking for Production System Engineers aka System Administrators. The job is located either in Metro Manila (Ortigas) or Cebu City. Here is the JD:
[B][SIZE="4"]
Production Systems Engineer[/SIZE][/B]

**Company:** [Morph Labs]("http://www.mor.ph")
**Location:** Manila, Cebu City
**Status:** Full Time, Employee
**Job Category:** Computers, Production Systems Engineer, Software Development
**Relevant Work Experience:** 3 to 5 Years
**Career Level:** Experienced (Non-Manager)
**Education Level:** Vocational

Morph Labs is looking for a production level Systems Engineer to work out of its Manila or Cebu office, spanning remote site locations. This position will report to the IT Manager.

**Responsibilities**

[LIST]
[*]Server Setup and Administration
[*]Sets up and maintain highly available, production level servers for Cloud Computing.
[*]Upgrading/maintaining server applications in a Highly Available Production Environment with a 99.99% availability SLA.
[*]Administer servers using Puppet and similar systems. 
[*]Design and deploy production server software infrastructure on Linux and Unix servers, load balancers and firewalls.
[*]Production web application experience a must.
[*]Administer a central applications monitoring system for fault detection and SLA compliance.
[*]Follow-up on remediation plans for escalations to ensure quick and effective resolution.
[*]Ensure that all servers and their associated software applications are operational 7 by 24 (SMS coverage is required).
[/LIST]

**User Support & Help Desk**

[LIST]
[*]Installs software releases, installs patches and works to resolve hardware and software-related problems with helpdesk team.
[*]Work with issue-tracking system to open, update and close help-desk tickets.
[*]Trouble shoots day-to-day server and application problems. [*]Identifies root cause and recommends and/or implements corrective action, coordinating with technical providers as needed
[/LIST]

**Qualifications**

[LIST]
[*]Minimum 3 years of paid work experience with both UNIX (preferably BSD based) and Linux servers. Daily responsibilities will include monitoring, optimizing, tuning and performing of UNIX and Linux based servers, software deployments to production environment, writing supporting scripts for backups, cronjobs, etc.
[*]Must have knowledge of current web technologies, SAS, DNS, firewalls, routers, etc.
[*]Must be proficient in network protocol (i.e., TCP/IP, DNS, DHCP).
[*]The applicant must be proficient in the latest Linux/Unix distributions.
[*]Proficient at shell scripting.
[*]Moderate programming experience, experience w/ Ruby preferred.
[/LIST]

In case you are interested, do send me a Private Message. Thanks!
