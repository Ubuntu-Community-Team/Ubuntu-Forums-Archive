### Post by dodimar on 2009-02-22
With the live cd, you have to run Ubuntu first (meaning load the window manager - gnome, load some services etc... eating up memory), before you can choose install...

With an alternate install disc,, the system will boot you directly to the install process in a text based install process.. doesn't use much memory so it'll be faster and easier..

look for the alternate installer iso at [http://releases.ubuntu.com/](http://releases.ubuntu.com/)
