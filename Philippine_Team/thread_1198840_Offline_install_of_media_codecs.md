### Post by ragadanga63 on 2009-06-28
On another Ubuntu PC that has internet connection, type sudo apt-get -d -b install ubuntu-restricted-extras on the Terminal.  For explanation of this command, type sudo apt-get -h on terminal.  The downloaded .deb packages are in the directory /var/cache/apt/archives.

or

Download/install Ultimate Edition, it's a 2.0GB LiveDVD with almost everything you need.

### Post by killer_d76 on 2009-06-28
you could try** AptonCD** [http://aptoncd.sourceforge.net/]("http://aptoncd.sourceforge.net/") if you know someone who is using ubuntu (same version as the one that your using and fully updated) install this app on his/her computer, then this app will create an .iso file containing all necessary packages, now all you need to do is burn this .iso into a disc, then run it into your computer, package manager will automatically detect the new packages on the disc, you can cancel this, then click on the mp3 files that you wish to play or a movie that you wish to   view.. and it will look for a driver/codec all you have to do is click on install, and it will download all the necessary packages on the cd!. ;)
