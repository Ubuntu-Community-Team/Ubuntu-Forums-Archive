### Post by jsgotangco on 2011-05-26
It is not hard to fix. It took me less than 2 minutes to look for it and delete the file and restart. Just look for the shared.xml file and delete it. When Skype starts, the file will be recreated. I'm currently using a Mac though but it should be the same for all Skype desktop apps. Skype has already acknowledged the problem.
