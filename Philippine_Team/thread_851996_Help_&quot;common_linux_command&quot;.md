### Post by wersdaluv on 2008-07-09
To install, you need to use **apt-get**. To use **apt-get**, you need to run it as root or a super user; to do that, run **sudo**. The **apt-get** command to install is **install**. In short, to install, **sudo apt-get install <package_name>**. To uninstall, **sudo apt-get remove <package_name>**.

To change from one directory to another, **cd <directory_location>**.
