Woot!!! installed din sa wakas! dami rin ako nabasa sa mga forums may problem Nvidia 8500 GT pag ginamit yung detected restricted drivers ng ubuntu....

Here's the solution:

Download the driver..
```
sudo apt-get install nvidia-glx-180
```

Then configure xconfig..
```
sudo nvidia-xconfig
```

Then reboot..

Works for me.. sana makatulong. :D
