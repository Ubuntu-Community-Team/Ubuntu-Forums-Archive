### Post by jaceleon on 2011-10-30
Okay, since I know naman na may mga naghahanap ng TUT on using OpenVPN on Oneiric using a .ovpn file, here's how....

1. Wala pong network-manager-openvpn ang Oneiric. So kindly have it installed by downloading 2 packages which are network-manager-openvpn at openvpn (accomplish this by doing a keryx project for 11.10, then download ka na lang sa Windoze PC ng packages)

2. Go to the directory nung mga downloaded packages (which is in my case nasa loob ng projects folder sa keryx inside my USB) and install the packages (using the sudo dpkg -i *.deb command)

3. Now import your .ovpn and your .crt file and configure it as you may need to.

4. Enjoy it!
