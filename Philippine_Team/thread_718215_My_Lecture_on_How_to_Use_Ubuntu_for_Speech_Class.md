Ano ang mapapayo ninyo?

### Post by ragadanga63 on 2008-03-07
Pag ATI ang Video Card sa laptop sa school mo, you might be able to run some Compiz effects using the Live CD.  I haven't heard of anyone able to run Compiz by Live CD on an Nvidia card.


I've used Ubuntu on at least three different LCD projectors and so far it has all been plug-n-play.  Otherwise, you may set the projector resolution by clicking:
System>Administration>Screens and Graphics. 

You may also try Granular Linux.  It has 4 different DE's (KDE, XFCE, Looking Glass, Fluxbox -not sure if it was Fluxbox) and Compiz works including the cube(Video cards tried on:  ATI 1050 and NVidia 6xxx).  

Best way to make sure things will go smoothly is to make a dry run.
