### Post by Copernicus1234 on 2009-08-26
I guess it can be good if using a lot of virtual desktops, but I wonder if it will really be used. When I first installed Ubuntu I had lots of desktops and switched between them constantly, but after a while I just started using the same desktop for everything. Its just faster and easier and I dont have to look where I put things.

Its not appearent from the video, but if you can have all your applications from all desktops in the left pane and when clicking on them, Ubuntu switches to the desktop where the application is, it would be useful for having lots of full screen applications going and easily switch between them. But its still not very different from using the gnome panel and letting it show applications from all desktops. That gives you that exact functionality. When clicking on a application in the taskbar, it switches to that desktop.

So I dont know about this. Perhaps the advantages will become more apparent with time.
