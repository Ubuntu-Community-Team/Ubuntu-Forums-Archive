### Post by jsgotangco on 2008-12-01
Hmm if you have the budget, I'd say to get a MacBook. When I say a MacBook, I mean the hardware. The OS X system just gets bundled (just like how OEMs bundle Vista) and what's nice is that the Mac hardware is standard all throughout, so drivers ain't a problem if you're planning to do Bootcamp with Windows. Also you can run VMWare or Parallels and load up Ubuntu or if you're adventurous do it with Bootcamp.

You asked what's the best environment for web development. I'd still stick to a *nix based environment which OS X and Linux provides. If you're doing visual web development, then OS X will be the place for you. Not to mention that the personal maintenance cost of the operating system is nil (pretty much like Linux).

People will bitch about the MacBook being made by Apple and that its more evil than Microsoft. Screw that crap they're just jealous that Apple is being able to provide a nice hardware package that any PC manufacturer won't be able to duplicate at all. That's why there are fair number of developers who prefer a Mac because it provides the best of both worlds.
