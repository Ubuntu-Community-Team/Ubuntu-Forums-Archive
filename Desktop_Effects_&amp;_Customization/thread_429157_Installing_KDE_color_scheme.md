### Post by Rashid584 on 2007-05-01
Right click and select "save link as" and then just save the file as a .kcsrc.

You can install this through kcontrol, or just open yakuake/konsole and type ```
cp ~/location/of/file ~/.kde/share/apps/kdisplay/color-schemes/
```

At least, I think thats the command. I'm in Gnome now (I used to be a KDE user but am trying out Gnome :p) so I can't check for you but if its not then post here or just have a browse around that general area...it something like that.
