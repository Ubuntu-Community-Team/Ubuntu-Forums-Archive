### Post by adam on 2007-10-18
To make compiz work without the proprietary fglrx driver:

1. In xorg.conf, under Section "Device", change driver from "ati" to "radeon" .

2. Remove the fglrx proprietary driver. Reboot. 

3. Go to System >> Preferences >> Appearance. Go to the "Visual Effects" Tab. Set it to "Extra". 

(You may be able to just skip to step 3.)

### Post by adam on 2007-10-18
Make sure that you're not using the fglrx driver. You don't need to install the free driver. It's there by default. 

Go to:
System >> Administration >> Restricted Drivers Manager
make sure "ATI accelerated graphics drivers" is *unchecked* and says "Not in use".

Then go to:
System >> Administration >> Screens and Graphics
Go to Graphics Card tab
Click driver, select "Choose driver by name" and select "Radeon".
Click Ok, then Ok again.

Reboot.

In a terminal type:
```
glxinfo | grep direct
```

If it says, direct rending: yes .. 

Go to:
System >> Preferences >> Appearance
Go to "Visual Effects" tab. Select "Extra"

If you'd like more configuration options:
In a terminal enter:
```
sudo apt-get install compizconfig-settings-manager
```
Once installed, a new option "Custom" will appear on "Visual Effects" tab.
