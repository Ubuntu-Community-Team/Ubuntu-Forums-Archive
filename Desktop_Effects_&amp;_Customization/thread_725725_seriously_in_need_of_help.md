### Post by Roanoke on 2008-03-16
You need to:

1.) Now open a terminal
2.) Right click on the terminal
3.) Select 'Edit current profile'
4.) Check the 'Show menubar by default in new terminals
5.) Close the terminal and open it again
6.) You should now have a menubar.
7.) Now click Edit->Profiles
8.) Edit the default
9.) Close the terminal and open again to check.

That should do it. You can then edit the 'trans' profile as needed to reverse the changes.
