### Post by santiagoward2000 on 2007-10-31
HI!
I'm not familiar with Cairo-Dock, but in Kiba-Dock, I had a similar problem. The problem was that the command in Kiba said: ```
thunderbird 0
```
Changing it to ```
thunderbird
``` fixed it.
