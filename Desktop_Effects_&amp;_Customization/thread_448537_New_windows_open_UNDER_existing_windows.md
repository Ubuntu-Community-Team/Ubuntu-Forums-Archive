### Post by jhenager on 2007-05-19
This affects dialog boxes, and lots of other things, and it is annyoying. I click an option or link, and the window opens up behind the current window. If I click the current window, I will then see the dialog box, such as "You are about to close 2 tabs. Are you sure you want to continue?"

How can I make this behave correctly? Thanks.
j

I found the answer to my own question buried in Beryl Settings.
Level of Focus Stealing Prevention needed to be changed from Extreme (default) to none.
Now it works like I would expect it to. When I print, the print dialog is there. When I close, the prompt to close multiple tabs comes to the front.
