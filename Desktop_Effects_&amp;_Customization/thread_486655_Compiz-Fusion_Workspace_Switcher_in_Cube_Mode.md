### Post by hyperair on 2007-06-28
Actually. the faeture works perfectly (at least for me). Go to your CompizConfig Settings Manager and under General Options->Desktop Size,

Horizontal Virtual Size: 4
Vertical Virtual Size: 1
Number of Desktops: 1

You can change the Horizontal Virtual Size to change the shape of the prism you're spinning. You could make it a pentagon-based prism, for one thing. Changing the horizontal virtual size changes the number of sides the "cube" has
