### Post by bicchi on 2007-04-17
I really do not have an answer for what you need but here is a work around:

Install [ChatZilla]("https://addons.mozilla.org/en-US/firefox/addon/16")
Create bookmarks in firefox that look like this:
irc://irc.freenode.net/ubuntu
irc://irc.freenode.net/ubuntuforums
irc://irc.freenode.net/kubuntu
. . .
etc
There seems to be a request for this on [ubuntu blueprint]("https://blueprints.launchpad.net/ubuntu/+spec/firefox-irc").
