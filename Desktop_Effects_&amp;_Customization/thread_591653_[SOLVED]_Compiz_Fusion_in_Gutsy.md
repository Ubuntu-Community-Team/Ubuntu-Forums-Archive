### Post by rapsball4 on 2007-10-25
You need to install the full control panel for it, not just the included stuff.  Fortunately, its already in the repos, unlike in Feisty.

```
sudo apt-get install compizconfig-settings-manager
```

It'll show up afterwards in your menu as "Advanced Desktop Effects Settings".
