I'm sure this has been mentioned ad nauseam around here and other forums regarding this issue, but Compiz-Fusion is currently in active development and breakage is to be expected.  Alpha/Beta software breaks and is not meant for daily use by those that require stability.  Compiz-Fusion 0.5.2 will have bugs and it will break.  It is a developer preview, not meant for public consumption.  If you are going to continue using Compiz-Fusion at the moment your options are:  1.  Wait for 0.6.0 release which will be the first true stable release 2.  Downgrade to a previous "stable" version and do not update 3.  Continue on the update train and expect breakage.
