### Post by forestpixie on 2007-12-02
you've added "AddARGBGLXVisuals" "True" in the wrong place - mines here

```
Section "Device"
    Identifier     "Videocard0"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
    BoardName      "GeForce4 MX 440 with AGP8X"
    BusID          "PCI:1:0:0"
    Screen          0
    Option         "AddARGBVisuals"     "True"
    Option         "AddARGBGLXVisuals"  "True"
EndSection
```

needs to be in the device section

Edit - as far as 1024x768 goes try adding the resolutions you want - my xorg is in [this]("http://ubuntuforums.org/showthread.php?t=629303") thread you can see what I've done there - make sure you make a backup before you start though.
