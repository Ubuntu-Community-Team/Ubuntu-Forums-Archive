### Post by jimerickso on 2007-11-07
it worked ok for me but it only installs 0.6.2  not the latest git so freewins wouldn't work for me. to get a script that will install the latest git consistently go to:

[http://telperion.wordpress.com/2007/10/27/fusion-nuovo-script-per-compilazione/](http://telperion.wordpress.com/2007/10/27/fusion-nuovo-script-per-compilazione/)

you will have fill in some values in the script to get it to run. the page is in italian but translates ok with google translate. since you are using ubuntu make sure that you use the master-noxcb value for the version variable and freewins should work. if run later it will install the latest git so you don't need to keep updating the script to get the latest version. it worked like a charm for me. good luck!

make sure you remove the old compiz fusion before running this script!
