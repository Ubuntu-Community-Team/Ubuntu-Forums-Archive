### Post by FuturePilot on 2008-03-06
How did you install Beryl? Beryl is no longer supported or developed. If you are using Ubuntu 7.10, it comes with Compiz Fusion already installed. Compiz Fusion is the result of the Beryl Compiz merge. I would suggest completely removing everything related to Beryl and try using Compiz Fusion which shouldn't require much more than installing a driver from System&#8594;Administration&#8594;Restricted Driver Manager if one is available.

### Post by Go_Bucks on 2008-03-06
Install compizconfig-settings-manager through synaptic. It will show up under System>Preferences>Advanced Desktop Effects Settings. It is fairly self explanatory.

### Post by Go_Bucks on 2008-03-06
Oh... I should add that that should be done after initiating Visual Effects under System>Preferences>Appearance. Initiating those turns on Compiz Fusion.
