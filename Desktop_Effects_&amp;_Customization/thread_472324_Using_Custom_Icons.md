### Post by sawjew on 2007-06-12
Navigate to the directory your icons are in using the "browse" button.

Highlight the directory and click the "open" button.

When you have done this it should open a selection window where you can select the icon you would like to use.

The trick is that you must open your icon directory with the "open" button not by clicking in the browser window.
