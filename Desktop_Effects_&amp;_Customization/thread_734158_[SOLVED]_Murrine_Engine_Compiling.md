### Post by PinkFloyd102489 on 2008-03-24
Use apt to install it. Either open up Synaptic and search "gtk2-engines-murrine" or do this in the terminal:

```

sudo apt-get install gtk2-engines-murrine

```


I had a similar problem with DivinorumCyanMagenta and just happened to search apt for it. :-p
