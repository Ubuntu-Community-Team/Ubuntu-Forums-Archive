### Post by harsh2209 on 2007-11-07
Ok, I got it. It was easy,

[LIST]
[*]Download whatever theme you want. (Make sure it is GNOME 2.0 compatible or get the GNOME version.)

[*]Goto System -> Preferences -> Appearance.

[*]Click 'Install' button. Browse for the '***.tar.gz' file and all set..
[/LIST]
