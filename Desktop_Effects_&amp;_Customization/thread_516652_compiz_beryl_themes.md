### Post by atomkarinca on 2007-08-03
you can use gtk themese through system > preferences > theme > install theme
or you can install emerald
```
sudo apt-get install emerald emerald-themes
```
and use the emerald themes you downloaded with the help of emerald theme manager.
system > preferences > emerald theme manager.
