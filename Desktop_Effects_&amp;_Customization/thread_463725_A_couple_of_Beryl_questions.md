### Post by Ryanson on 2007-06-04
I have two things with beryl (I think it's with beryl) that I want to get off. The first is when I mouse to the top right corner, it shows all my windows for selection. I'd like to turn that off.

The second is that the title bar on my windows is a little bit transparent. Is there a place I can go to change that?

Thanks,
    -Ryan

---

### Post by blackened on 2007-06-04
First question: In Beryl Settings Manager (right-click Beryl icon in the notification area, select "Beryl Settings Manager"), on the "General" tab, choose "Shortcuts" in the left pane, then select the "Screen Edges" tab from the right pane.

Second question: In Emerald Theme Manager (right-click Beryl icon in the notification area, select "Emerald Theme Manager"), on the second row of tabs choose "Edit Themes", on the "Frame Engine" tab select the engine that your theme uses from the dropdown list, then in the "Colors" tab of the next-level dialog adjust the left, middle, and right blend opacity values using the sliders.
