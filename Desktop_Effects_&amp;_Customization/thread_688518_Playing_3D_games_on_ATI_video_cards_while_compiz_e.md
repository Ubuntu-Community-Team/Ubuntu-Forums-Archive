### Post by DoDoENT on 2008-02-05
Hello everyone! We all know that it is not possible to have good 3D gameplay on ATI video cards while compiz effects are on because image tears, and this is quite annoying. Some people solved that problem by turning compiz off, but this is not the most attractive idea. Therefore, I've written a little script which temporarily shuts compiz off while 3D application is running, and restores it after 3D application terminates. I know this isn't a much of a script (actually it was the first bash script I've ever written), but I hope it will come in handy to beginners who have ATI graphic cards and want to have both compiz effects on and playable games.

Link to download the script: [http://www.box.net/shared/zd0add59ck]("http://www.box.net/shared/zd0add59ck")
