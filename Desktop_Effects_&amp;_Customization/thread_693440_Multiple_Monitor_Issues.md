### Post by OoooMatron on 2008-02-11

Option "twinview" "true"

to the device section where the nvidia driver is called and that should sort it out.

### Post by OoooMatron on 2008-02-12
that's because you aren't setting it up correclty.

If you follow exactly what I did it will work.

It won't work if you set up 2 screen modes in the xorg conf file.

Start with a clean xorg.conf file and just add

options "twinview" "true"

to the device part and you will have it the way you want.
