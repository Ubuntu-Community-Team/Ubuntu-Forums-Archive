### Post by dfreer on 2007-05-30
you haven't enabled the nvidia drivers, even though you've installed them.

run this command to backup your /etc/X11/xorg.conf:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

Then edit the file:
```
sudo gedit /etc/X11/xorg.conf
```
change the following lines as listed below:
```

Section "Module"
        .....
	Load		"glx"
        .....
EndSection

Section "Device"
        .....
	Driver		"nvidia"
        .....
	Option		"AddARGBVisuals"	"True"
	Option		"AddARGBGLXVisuals"	"True"
        .....
EndSection

```

Also take out this section:
```
Section "Extensions"
Option "Composite" "Enable"
EndSection
```

Reboot. Now, if X crashes, hit <Ctrl><Alt><F5> and login.
From there, type this command:
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
sudo /etc/init.d/gdm restart
```

Remember to write that last bit down, cuz you won't be able to login to X to copy+paste it in.

### Post by dfreer on 2007-05-31
Basically, the nv driver should work with any nvidia card. yay!
but it has no 3D acceleration support, which means no cool games and no beryl. boo!
You can check this with:
glxinfo | grep rendering
If it says anything other than Yes, you cannot do 3D acceleration/direct rendering.

So what you will need to do is change from the nv driver to nvidia, using one of these commands first:
sudo apt-get install nvidia-glx
sudo apt-get install nvidia-glx-new
sudo apt-get install nvidia-glx-legacy

Not exactly sure what will happen if you install one over top of the other, it *should* uninstall the old one first.


the best way to install beryl with gnome IMO is with this command:
sudo apt-get install beryl-manager emerald-themes


Do a search for installing the correct video driver for your card, or just a generic nvidia card install, and basically try things out.

If the nvidia driver is crashing X, you could always check/post your X logs on this thread and we can probably solve this problem. :D
