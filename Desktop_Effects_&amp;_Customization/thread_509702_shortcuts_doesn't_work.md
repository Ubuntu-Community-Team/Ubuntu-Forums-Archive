Hy!

If I set shortcuts in 'Keyboard Shortcuts' some of them doesn't work.
For example the shortcut for 'Run a terminal' works fine, but 'Launch a web browser' has the following bug:
it doesn't work with  a key binding set to <Mod4>+anything. If it is set to <Alt>+something, it works great.

I found out, that shortcuts located under /apps/gnome_settings_daemon/keybindings/calculator (in gconf-editor) are affected by this kind of buggy behavior, all others work well!
