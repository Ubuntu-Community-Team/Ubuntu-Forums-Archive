### Post by alsamman on 2008-01-03
save the tar.gz to the desktop
then go to system>preferences>appearance and then under Theme, click Install, go to desktop and select the tar.gz and then itll install and ask if u want to apply theme and there you have it :)
EDIT:
i tried and it said file invalid but you can download from [http://gnome-look.org/content/show.php/nuoveXT?content=26448](http://gnome-look.org/content/show.php/nuoveXT?content=26448) and follow same instructions

### Post by NilsE on 2008-01-03
Download the archive

Open it with Archive Manager (file-roller)  or double click on the archive

Extract folder to /usr/share/icons

You may have to run Archive Manager as root so enter the following in a terminal then browse to where you downloaded the archive.

gksudo file-roller
