### Post by Tenken on 2008-01-14
To use an entire icon theme go to System->Presences->Appearance and drag the *icontheme*.tar.gz you downloaded into the window. A pop up will ask you if you want to use the newly installed theme now, click yes to use it. If you want to change your theme later, click the "Customize" button and select the theme you want under the icon tab.
