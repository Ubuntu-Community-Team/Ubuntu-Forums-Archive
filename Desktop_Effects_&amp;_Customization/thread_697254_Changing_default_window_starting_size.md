### Post by skankster on 2008-02-15
Do you use Compiz-Fusion? If so the following may be of use.

Both the Place Window and Window Rules plug-ins for Compiz-Fusion will be useful here. 

See this page for Window Matching & the Windows Rules Plug-in
[http://wiki.compiz-fusion.org/WindowMatching](http://wiki.compiz-fusion.org/WindowMatching)

For Place Windows, the Window Matching is the same. 
[http://wiki.compiz-fusion.org/Plugins/Place](http://wiki.compiz-fusion.org/Plugins/Place)

I'm at work & not on my Ubuntu system so I can't go into much more detail, but you may find you need different techniques for different apps.

The xwininfo application is useful for getting window details (inc. Title). Otherwise, you can copy & paste the commands listed in the Window Matching page above.
