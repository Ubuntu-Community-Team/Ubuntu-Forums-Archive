### Post by Shimmy on 2007-10-24
If you haven't updated recently, and you are using Compiz, try this:
Lock your screen by pressing "ctrl+alt+l" (make sure you have password protected screensaver).
when the screensaver is active, press "alt+F2", and type "killall -9 gnome-screensaver", the screensaver will be killed and you can use the desktop :)

To start the screensaver deamon again: alt+f2 -> type "gnome-screensaver".

Scary isn't it?
However as i said, update and this breach is fixed.
