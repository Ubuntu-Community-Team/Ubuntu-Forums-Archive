### Post by xen on 2007-07-03
Are you talking about themes for GNOME/GTK?

Well first of all you need to create the file '.themes' without the quotation marks in your home directory. Then visit [www.gnome-look.org](www.gnome-look.org) and save/exctract Metacity and GTK themes into this directory as you like.

Please note that for some themes you might need to install a new GTK engine, most of these can be found in the Universe repository.
