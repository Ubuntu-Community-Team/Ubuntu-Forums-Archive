### Post by diamondsw on 2007-05-09
**It would appear this is unrelated to the crash - looks like a general problem when VMWare Tools are installed.**

I'm running Ubuntu 7.04 in VMWare Fusion, with VMWare Tools installed. Everything was working, until I tried to "Enable Desktop Effects" in the "Desktop Effects" panel. Immediately my session crashed back to the login screen (it looked like the X server crashed, but I'm not 100% sure). On logging in again, the "Desktop Effects" panel shows no change (and crashes if I try clicking the button again), but I have a problem - "wobbly windows" appears to be partially activated.

If I move a window around on the screen now, it will shake back and forth erraticly; the faster I move the mouse, the more the window shakes. It did not do this prior to attempting to enable desktop effects.

Any ideas?
