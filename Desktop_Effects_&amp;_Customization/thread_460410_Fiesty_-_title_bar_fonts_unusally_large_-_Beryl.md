### Post by awkwardmoment on 2007-05-31
I'm not sure about the problem at login, but to fix the title bars open Emerald Theme Manager from the Beryl Manager menu, click the Edit Themes tab then Click the titlebar tab. There, you will be abel to change the font/font size and titlebar height.
