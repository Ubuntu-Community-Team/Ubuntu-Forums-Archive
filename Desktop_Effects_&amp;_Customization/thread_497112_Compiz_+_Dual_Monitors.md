### Post by kukuku on 2007-10-19
I'm running two monitors on an nvidia setup too. The card was MSI something, but I seem to have misplaced a device manager to check what it was exactly...

After installing Gutsy the desktop effects worked immediately on the first monitor. I changed to Twinview from nvidia-settings and everything worked fine still. The cube stretched over the two screens.

Twinview is just a bit annoying to use, so I changed it to separate X sessions (again from nvidia-settings). Both monitors have their own cube, and all the compiz effects work on both monitors... Only the titlebars for windows are missing from the second monitor. First one is all fine. The nvidia-xconfig --add-argb-glx-visuals -d 24 thing fixed the couple second delay before menus opening.

Now if there was some way to drag windows from one screen to another in separate X sessions, I could be happy about it despite missing some titlebars. ;) Perhaps xinerama can do something like that.
