### Post by orange2k on 2007-08-27
The image behind the cube is what they call a "skydome". If you enable it, you may also wish to enable "animate skydome", because if you dont it wont move as you turn the cube.

As far as desintegrating windows are concerned - its an animation "burn" - but there are also numerous others that look as good if not much better than that. You can associate different window actions like minimize, close, open, shade etc. with different animations. 

I installed compiz fusion just today following a very nice guide, and I must say there are advances on every field: everything got smoother, more responsive, and overall much better since I last tried it a few months ago. And I run a 1,5GHz P4 384RAM MX4000 Nvidia system...

If youre interested in the guide, follow the link in this thread: [http://ubuntuforums.org/showthread.php?t=536149](http://ubuntuforums.org/showthread.php?t=536149)

### Post by Rupertronco on 2007-08-27
If you go to the animations tab in the Compiz config settings manager you can litterally animate every action. You can pick a variety of effects for each or even set it to randomize the actions. 

To run the settings manager simply type "ccsm" in a run box (alt+F2),
