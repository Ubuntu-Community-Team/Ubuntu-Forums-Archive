### Post by bimaljr on 2008-01-13
Hey Guys.. The problem is solved.

How? I also don't know.. Here it is :

1) Go to System > Preferences > Appearance. Select "Visual Effect" tab and select "None".
2) Restart PC,
3) Restart PC Again.
4) Go to System > Preferences > Appearance. Select "Visual Effect" tab and select "Custom".
5) Restart PC
6) Restart Again.


Problem solved. That's what I do.

I also don't know how this happen. But I did only this.
Thanks for support me. Thank You All.
