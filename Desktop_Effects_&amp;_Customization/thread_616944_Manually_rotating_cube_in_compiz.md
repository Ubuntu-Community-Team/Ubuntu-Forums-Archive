### Post by Sims2789 on 2007-11-18
> **cam381 said:**
> I know this question has been asked many times but I am still wondering, is there any way I can manually rotate the cube in compiz by downloading or installing packages? Thank You.:)

Go to Applications -> Add/Remove Software and install CompConfig (it might also be called Advanced Desktop Effects Settings).

In CompConfig, find the Rotate Cube settings and go to the Actions tab. Then enter a keyboard combination for Initiate.
