### Post by talsemgeest on 2008-03-14
Probably not a .png.

But take a look at [http://www.gnomelook.org]("http://www.gnomelook.org/") and find one you like.

Then install the startup manager:

```
sudo apt-get install startupmanager
```

You can change a lot of settings with that program.

Hope this helps! :)
