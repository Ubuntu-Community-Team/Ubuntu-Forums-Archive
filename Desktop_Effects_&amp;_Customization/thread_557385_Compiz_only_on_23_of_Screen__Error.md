### Post by polygone on 2007-09-23
GOT IT!!

OK, it runs a bit slower for some of the cube effects, but this made it at least display right

create or edit  /etc/drirc

add this:

```
<driconf>
    <device screen="0" driver="r200">
        <application name="all">
            <option name="allow_large_textures" value="2" />
        </application>
    </device>
</driconf>
```


It makes the textures larger and supports a higher res....but it is way taxing on your card.
