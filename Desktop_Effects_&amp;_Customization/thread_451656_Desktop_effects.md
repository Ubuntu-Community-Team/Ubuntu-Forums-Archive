### Post by Karsa on 2007-05-23
have you installed the correct drivers for your ATI-card? Anyway, you could follow that guide but install the drivers this way instead "sudo apt-get install xorg-driver-fglrx". Then make a backup of your xorg.conf. Change the line "Driver "ati"" to "Driver "fglrx"". Save and restart x.

Now you should have the correct drivers installed so skip the step about restricted driver manager and continue following the guide.

If that doesn't get you going, just uninstall the fglrx-drivers and change back to ati in your xorg.conf.

-Another way to get the cube working (with compiz instead of beryl)-
Enable desktop effects, press alt-f2, type gconf-editor. Go to apps->compiz->general->screen0->options and change the value of "hsize" to 4, that should provide you with a working compiz-cube.

Hope this helps!
