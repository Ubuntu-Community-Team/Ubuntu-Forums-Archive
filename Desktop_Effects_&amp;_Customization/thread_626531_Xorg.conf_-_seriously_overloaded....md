### Post by viking777 on 2007-11-29
Make a backup of your xorg.conf file and call it xorg.conf.old, and then run

 ```
 aticonfig --initial --force
```

This will write a new one for you. If it doesn't work you will have to delete the new one and rename your backup to xorg.conf but it almost certainly will work.

### Post by viking777 on 2007-11-30
First idea - do you have an external monitor attached to your laptop?

A lot of people have had difficulty with these and the aticonfig command that I gave you needs to be modified if you do have. Easiest way to check if this is the problem is to just unplug it and see if it works with just the laptop screen.

Second idea - Do you have any other logins available you could try like KDE of XFCE?
If not you are bound to have Gnome Failsafe so you should try that.

Third idea - use the command recommended in xorg.conf ie.

```
 sudo dpkg-reconfigure -phigh xserver-xorg
 
```

Fourth idea - reinstate the backup xorg.conf.

Fifth idea - I don't have one:lolflag:
