### Post by bdtgp on 2007-07-20
unlinke most people think...you DO NOT NEED emerald themes.  just add this to your device section (video card) of your xorg file:
```
Option "AddARGBGLXVisuals" "True"
```
and then hit Alt+F2 and enter this
```
compiz --replace &
```

### Post by bdtgp on 2007-07-20
Put it like this:
```
Section "Device"
	Identifier	"ATI Technologies Inc RV380 [Radeon X600 (PCIE)]"
	Driver		"ati"
	BusID		"PCI:1:0:0"
        Option "AddARGBGLXVisuals" "True"
EndSection
```

Then run the Alt+F2 command.
