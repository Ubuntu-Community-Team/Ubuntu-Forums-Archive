### Post by ThaRabbit on 2008-03-11
you may have to comment out the pci_id for your card in /usr/bin/compiz

```
nano /usr/bin/compiz
```

```
# Driver whitelist
WHITELIST="nvidia intel ati radeon i810"

# blacklist based on the pci ids 
# See http://wiki.compiz-fusion.org/Hardware/Blacklist for details
T="   1002:5954 1002:5854 1002:5955" # ati rs480
T="$T 1002:4153" # ATI Rv350
T="$T 8086:2982 8086:2992 8086:29a2 8086:2a02 8086:2a12"  # intel 965
T="$T 8086:2972" # i965 (x3000)
T="$T 1002:3152 1002:3150 1002:5462 1002:5653 " # ati X300 X600,X600 X700 
```

Simply adding a # in front of the last line will solve the pci_id blacklist]

post your terminal output of:
```
compiz --replace
```
