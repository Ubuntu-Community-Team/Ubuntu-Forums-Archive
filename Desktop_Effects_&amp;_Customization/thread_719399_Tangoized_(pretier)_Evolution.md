### Post by 6205 on 2008-03-09
I'm using in Gutsy as my default e-mail client Evolution, but it has many old, ugly gtk stock icons so i decided to replace some of them with tangoized versions... 

Icons used here are from Tango, Gnome, Tango-Yast from openSUSE, some my custom icons and some icons from development version of Evolution. They don't cover all ugly gtk stock icons, but will give your Evolution much better look'n'feel :)

If you like them, download/extract attached file (you will get folder gnome), launch file browser with root privilegies ( sudo nautilus ) and then go to /usr/share/icons/ and put extracted gnome folder to that location. It will rewrite some gtk stock icons related to Evolution. Also optional is to delete icon-cache file in gnome folder and if you lauch now Evolution Mail, you will see new icons in many places...

If anybody has more aditional gtk stock icons i hope you will share them with me :)))
