### Post by WhatTheCoitus on 2007-11-07
Crazy animated user controlled Flaming function.

I was struggling to think of a reason to use this very handy tool...well it's winter here in the UK so if you want to feel warm and toasty this is the tool for you!

OK this is how to use the tool...shift+Super+left mouse button and drag the mouse anywhere on the desktop you should now have the beginnings of an animated fire. 

Suggestions for other practical uses welcome.


Oh and if you get board to clear the effect press...shift+super+c

I'm running compiz and paint fire is active.

Have fun.

Not sure if this has come up but I had a bug which slowed down the machine after re-sizing a window. Un-checking Compiz 'reflection' option seemed to fix the issue for me.

Nice work to all those working on compiz very well implemented desktop environment creator. Think there could be a little penguin egg on the faces of another well known OS interface dev team ?
