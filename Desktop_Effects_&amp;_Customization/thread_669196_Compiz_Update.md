### Post by Vadi on 2008-07-14
Compiz is updated as everything else on Ubuntu, via System - Administration - Update Manager, or the orange icon top-left.

If you'd like to get *the* latest version of Compiz, you can get [instructions here]("http://linuxowns.wordpress.com/2008/07/09/update-compiz-fusion/") on how to. However don't complain if it breaks down and goes horribly wrong - the reason Ubuntu doesn't have it yet is because it's still undergoing testing.
