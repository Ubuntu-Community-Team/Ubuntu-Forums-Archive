### Post by old_geekster on 2007-03-17
This is one of the many bugs in Beryl.  I had the same problem.  There is a way to correct the problem, but it appeared to be very involved.  Too much so for a newbie like me.

To correct my problem, I uninstalled Beryl and installed "Automatix Bleeder".  This is an extension of Automatix2.  One of the few programs that it (AB) installs is Beryl.  I installed Beryl using it and for the first time saw Beryl the way it should look.

The only problem that I have is Swiftfox2 is running much more slowly.  Beryl does everything that I have read that it should do, however.
