### Post by ahaurw01 on 2007-05-27
Try right clicking the Beryl red ruby and selecting 'reload window manager.' Also, make sure you have both Desktop Cube and Rotate Cube checked in the Desktop tab of the beryl settings manager.  If things seem to stop working all together, right click the beryl ruby and under 'select window manager' make sure that Beryl is selected. It can change to Metacity if Beryl happens to crash.
