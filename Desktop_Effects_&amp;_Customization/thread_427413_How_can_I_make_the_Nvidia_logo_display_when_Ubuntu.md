### Post by lukew on 2007-04-29
> **braveerudite said:**
> How can I make the Nvidia logo display when Ubuntu boots?

I seen it when OpenSuse boots with Nvidia drivers.  Can someone please?

Within /etc/X11/xorg.conf search for the section

```

Section "Device"
Identifier "Generic Video Card"
Driver "nvidia"
EndSection
```

Check to see you dont have.

```
"NoLogo" "true"

```

It should be on by default. Set it to "false" or try removing the line altogether. If you dont have the line try adding it as "false".
