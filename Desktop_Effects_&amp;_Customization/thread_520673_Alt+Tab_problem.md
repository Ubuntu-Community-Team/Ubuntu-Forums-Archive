### Post by petit.padavoine on 2007-08-09
Yes!

Very annoying problem!

But I managed to solve it :D...

In the CompizConfig Settings Manager, click on Application switcher in Window Management at the bottom. Go to the Actions tab and click on the arrow next to General.

Alt+Tab is set as the shortcut for both Next Window (which only navigates through some windows), and for Next Window (All windows) (which navigates through all windows, like you want).

Just disable Alt+Tab for Next Window and leave it for Next Window (All windows).
