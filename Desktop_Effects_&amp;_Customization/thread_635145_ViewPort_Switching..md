### Post by imoatama on 2007-12-10
What do you mean in your first question by ¨reversing, as Compiz does by default¨? I use Compiz with desktop cube and viewport switching occurs by means of the Ctrl + Alt + left/right arrows. When I get to the last viewport, and hit CTRL ALT Right, it will switch to the first viewport. Isn that what you&#341;e asking for? It is already Compiz´s default behaviour. 

As for part 2, if you use the desktop cube, turn on the Rotate Cube plugin, and make sure that Edge Flip Pointer is enabled under the General tab. This will allow for viewport switching by moving the pointer to the screens edge. If you want a screen edge + mouse button combo, you set it by setting the key combination for the 'Rotate Flip Left/Right' Actions. I have mine set to left/right screen edge and Button1 (left mouse) so that i cant accidentally trigger the viewport switch by going to the edge of the screen, but can easily switch by clicking as i approach the edge if i want to. The middle mouse button is Button 2 (a peculiarity of Compiz), and i think scroll wheel is 4 or 5 or both (up/down) perhaps.
