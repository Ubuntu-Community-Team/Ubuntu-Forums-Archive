### Post by chavanak on 2008-02-13
Install compiz-kde and compizconfig-setting-manager. Then fireup a terminal and type in compiz. You can also try compiz --replace in run window (Alt+F2).

If compiz does not work, open the terminal type compiz and copy the output here!!!
