### Post by eode on 2007-05-06
Beryl/Emerald handles the window manager theme - the borders, and what is done with windows.  For what is done *inside* the windows, QT handles the themes in KDE, and GTK in gnome.  So..

For KDE/Kubuntu, check out : [http://www.kde-look.org/](http://www.kde-look.org/)

or, when running Gnome/Ubuntu: [http://gnome-look.org/](http://gnome-look.org/)
