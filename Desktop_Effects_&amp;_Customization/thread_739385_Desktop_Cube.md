### Post by +++ IGOR +++ on 2008-03-29
First I think you need to make sure that you have enabled the desktop cube in compiz. You check this by clicking SYSTEM -> PREFERENCES -> ADVANCED DESKTOP EFFECTS

This should bring up the CompizConfig Settings Manager. In the manager you need to enable the plugin called "DESKTOP CUBE" then you have to go to general options and set the number of desktops to 4.

### Post by lswest on 2008-03-29
