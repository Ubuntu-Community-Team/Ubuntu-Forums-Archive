### Post by Tomatz on 2008-03-27
To use emerald with compiz do this:

in a terminal type

**sudo apt-get install compizconfig-settings-manager**

go to [B]system, preferences, compiz config settings manager
[/B]
find the **window decoration** plugin and **enable** it then **enter the window decoration plugin settings.**

in the **command** field type **emerald --replace**

close the compiz config settings manager

**ctrl alt backspace**

*now add your cool themes* :)
