### Post by qamelian on 2007-11-28
That would be the Scale plugin and it should work by pressing Shift-Alt-<Up Arrow>. If you install COmpizConfig Settings Manager, you can also activate the behaviour you're familiar with by going to the Actions tab in the Scale plugin.
