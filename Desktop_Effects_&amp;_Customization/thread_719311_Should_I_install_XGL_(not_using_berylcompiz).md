### Post by FuturePilot on 2008-03-09
Since you have a Nvidia card, XGL is not necessary. Even if you are using Compiz you don't need XGL with Nvidia cards. You can safely remove it. XGL causes problems with Nvidia cards. You will be better off just removing XGL.
