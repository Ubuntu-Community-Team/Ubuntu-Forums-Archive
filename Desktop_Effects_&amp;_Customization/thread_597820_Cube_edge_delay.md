### Post by Zorael on 2007-10-30
You can set the Flip Time under the Rotate Cube plugin to something high-ish, not really what you're asking for but it could help a bit.

You can also disable Edge Flip Pointer, then go to Actions and set Screen Edges and Edge Buttons for Rotate Left and Right. Basically, if you click a mouse button when at the screen edge, it will flip over. It'll complain about the setting conflicting, but you can just tell it to ignore that and set it up anyway.
