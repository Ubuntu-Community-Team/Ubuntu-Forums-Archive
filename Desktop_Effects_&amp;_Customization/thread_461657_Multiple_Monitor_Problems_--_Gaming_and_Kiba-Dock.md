### Post by krismatth3 on 2007-06-02
Your only hope with TwinView is to make the game run in windowed mode. What I do with World of Warcraft for example:

1. Turn on windowed mode
2. Set resolution to match that of my monitor
3. Check "Maximized" under windowed mode.

Of course, if it's on the wrong monitor, you have to turn off maximized, drag the window over, and turn it back on. If you're using a reasonably intelligent window manager, the window position should be remembered. :)
