### Post by jolx on 2008-01-18
All u need to do is get the advanced desktop effects settings from either synaptic or add/remove programs. then it should appear under System --> Preferences and u just need to tick the boxes that say desktop cube & rotate cube. then u need to activate it by going to System --> Preferences --> Appearance, Visual Effects tab & select custom

---

### Post by jolx on 2008-01-18
oh and in the advanced desktop effects settings got to general options and under desktop size set it 4.
