### Post by rolf-c on 2007-05-03
Beryl is a window manager, like metacity or any other window manager, except Beryl is built to take advantage of compositing.

Xgl is an X server architecture, like Xorg server, except Xgl is built to take advantage of OpenGL and hardware acceleration.
