### Post by Odd-rationale on 2007-12-04
Try this:

First install the compiz-config-settings-manager (ccsm) from Synaptic or Add/Remove, if you haven't already.

Then go to System -> Preferences -> Appearance. Go to the Visual Effects tab. Under Custom select Preferences. Make sure that you have Desktop Cube and Rotate Cube enabled. You can also enable any other cool plugins from here.

Click on General Options and go to the Desktop Size tab. Set Horizontal Virtual Size to 4, Vertical Virtual Size to 1, and Number of Desktops to 1.

Then see if it works.
