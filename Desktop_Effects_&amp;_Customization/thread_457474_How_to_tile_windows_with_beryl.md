### Post by raja on 2007-05-28
It is not a separate plugin - it is built in with Beryl (assuming what you want is to scale windows). Open Beryl-manager and go to window management -> Scale to adjust settings. I find it convenient to bind it to the top right screen edge, so that all I have to do is move the mouse there. To configure screen edges go to General options -> shortcuts -> screen edges.

### Post by aldreis on 2007-05-29
I believe you have to install **beryl-plugins-unsupported** ( look for it in Synaptic ), to have the Tile plugin. It will appear in Beryl Settings / Window Management / Tile.
