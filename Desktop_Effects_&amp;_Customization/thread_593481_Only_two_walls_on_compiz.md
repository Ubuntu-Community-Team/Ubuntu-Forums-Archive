### Post by Caffeine_Junky on 2007-10-27
Ok, in Kubuntu it is done like this:  

Open the  KMenu > Settings > Advanced Desktop Effects Settings, Click the "General Options TAB" go to "Desktop Size" and set Horizontal Virtual Size to "4",  and "Virtual Size" and "Number of Desktops" set to "1"

Now you should have a working Cube, spin it with Ctrl+Alt <direction arrows>
