### Post by santiagoward2000 on 2007-12-22
Hi!
Are you using **Kubuntu** or **Xubuntu?** **Xubuntu** is the one that comes with Xfce and uses **xfwm4** as windows manager. If you want to open it, just type:
```
xfwm4 --replace
```
**Kubuntu** uses KDE, and the wm is **kdm**. I think you should type:
```
kdm --replace
``` to run it.
Cheers!
