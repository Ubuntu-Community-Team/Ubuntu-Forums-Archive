### Post by Forlong on 2007-11-19
> **usersock said:**
> I know I can pull programs from the application menu onto the desktop, and that would work perfectly if only I could change the graphic to whichever icon I choose.
Right-click on the application icon on the desktop, choose Properties and click on the icon. There you can choose wichever icon you want.
> **usersock said:**
> But I'd also need to remove the text underneath it. Any ideas on how I can do this?
Right-click again, choose Rename and press space bar (then enter). Et voilà, no text.


P.S. the dock in gOS is from Enlightenment.
