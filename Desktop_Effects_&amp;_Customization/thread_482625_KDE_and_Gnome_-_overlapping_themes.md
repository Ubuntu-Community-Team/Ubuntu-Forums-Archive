### Post by Happy_Man on 2007-06-23
1. Yeah, that's actually GNOME default icon theme. Ubuntu uses its own theme. 

2. It's just a consequence of how both environments handle desktop icons.... don't worry about it.

3. Aquamarine is for using Kwin themes with Beryl. Emerald is for using Emerald themes with Beryl. Similarly, there is Heliodor for using Metacity themes with Beryl for GNOME. 

4. It does that for me, too. I just log out and then shutdown fromt the login screen. 

Last question: Umm.. not sure you can do that.... sorry.
