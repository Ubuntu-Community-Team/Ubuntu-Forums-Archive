### Post by Vadi on 2008-01-16
I believe the default keybinding is windows key + 1, windows key + 2, windows key + 3.

If you have the compiz settings manager ([click if you don't]("apt:compizconfig-settings-manager")), go to system - preferences - advanced desktop effects settings, then find Enchanced Zoom Desktop (it's under the Accesibility category), make sure the effect is enabled. Then go into it, click the actions tab, and under "Zoom Movement", the Zoom Specific Level 1 - 3 are the effects I mean.
