1. Nope, like ThrobbingBrain66 said, it's still Alpha, although it does require very little resources compared to Aero Glass.
2. Perfect. I've never even run into a single hitch using it.
3. Yes, as long as you use Emerald as your window decorator. Just for the record, Emerald is also Beryl's window decorator and before that it was known as CGWD, used as Compiz-Quinn's window decorator
4. Heck, 256MB can
5. You can try [http://www.gnome-look.org](http://www.gnome-look.org), [http://www.beryl-themes.org](http://www.beryl-themes.org)

7.  Yes, you can change practically any part of it, customize it to suit you

The plugins come with Compiz Fusion, when you install them. You may disable and enable each plugin as you wish, especially those you won't use or don't need. You can enable them again at any point of time though =)

Full release will be before Gutsy comes out I reckon, because it's going to be integrated into Ubuntu Gutsy, and an alpha software isn't exactly the best thing to put in a stable release.
