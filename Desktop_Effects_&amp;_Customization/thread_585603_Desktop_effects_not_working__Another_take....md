### Post by linfidel on 2007-10-21
FYI:  I have an NVidia fx 5200 video card, an abit motherboard, with 1gb of RAM.

I upgraded, but I had forgotten to uninstall some junk with previous Compiz experiments, and had problems.  I read and tried a lot of suggestions, getting close to the "infinite monkeys" solution, but I couldn't quite make it to infinity.  

Nothing worked.  The restricted driver would not work for me.

So, I decided to do a semi-clean install, and installed without reformatting, keeping my home directory the same, with the same login name.  I backed up what little I had that was important, not caring that much if I had to redo everything else.

Naturally, since I was prepared, everything worked well, and all the desktop effects are working perfectly now.  My old desktop looks exactly the same - same wallpaper, same startup apps, same firefox extensions, etc.  

I don't know if this was the best way, but it worked for me.  If all else seems to be failing, it may be worth a try for some of you.  Or not.  :-)

Just thought this might be worth posting. Hope it helps someone.
