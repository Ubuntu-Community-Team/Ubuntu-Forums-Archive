### Post by Tenken on 2008-02-18
You could use the main menu that comes with Gnome, right-click your panel, go to add to panel and select the main menu applet. Another option is the gnome-main-menu which you can install with 

```
sudo apt-get install gnome-main-menu
```

Mint also has a nice looking start/main menu.
