### Post by Forlong on 2007-11-08
As you already guessed, upgrading over two versions is not a great idea.
There are lots of dependencies to settle and you have to keep in mind that all your configs are for later versions of the applications you are using now.
What I'm trying to say: if you want a proper install of Gutsy: start "from scratch".
If you don't have a separate partition for /home try to backup as much data as possible from there - and if you install again, use partitions.
5 GBs for Ubuntu (the OS itself) is enough in most cases. And you can use the rest for the /home partition.


As for your problem with Beryl: you can't use Beryl in Gutsy. At least not the packages you have on your system, because there are no packages for Beryl on Gutsy.
The Beryl devs have a new project called Compiz Fusion where they port all Beryl plugins to Compiz and work on new ones. That's installed in Gutsy by default.
