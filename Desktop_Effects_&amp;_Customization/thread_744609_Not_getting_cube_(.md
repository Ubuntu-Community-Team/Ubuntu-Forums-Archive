### Post by sowelie on 2008-04-03
Window + e is for expose.  Make sure you have compizconfig-settings-manager installed:
```
sudo aptitude install compizconfig-settings-manager
```

Go to system -> preferences -> advanced desktop settings.  Turn on desktop cube and rotate cube.  Then, go to general settings and make sure horizontal virtual size is set to 4.  Then all you have to do is press ctrl + alt + click and move to rotate.
