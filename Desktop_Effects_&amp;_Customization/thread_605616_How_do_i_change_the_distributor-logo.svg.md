### Post by mbirkis on 2007-11-07
you need superuser permission to do that.

i don't know how to do it in GUI, but you can use the sudo command in a terminal, heres an example:

1. open a terminal

2. use this command: sudo cp Desktop/Truth/Buttons/distributor-logo.png /usr/share/path/to/folder/where/you/want/this
