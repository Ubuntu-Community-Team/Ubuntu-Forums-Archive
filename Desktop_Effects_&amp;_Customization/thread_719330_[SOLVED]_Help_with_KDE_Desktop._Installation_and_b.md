### Post by .nedberg on 2008-03-09
Short answer:
Install kubutnu-desktop
```

sudo apt-get install kubuntu-desktop

```

Select KDE when you log in

Remove it:
[CODE]
sudo apt-get autoremove kubuntu-desktop
[CODE]

If you want KDE4 see this post: [http://kubuntu.org/announcements/kde-4.0.2.php]("http://kubuntu.org/announcements/kde-4.0.2.php")
