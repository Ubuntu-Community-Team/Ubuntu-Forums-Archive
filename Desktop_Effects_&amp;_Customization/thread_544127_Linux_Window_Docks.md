### Post by Inxsible on 2007-09-05
Lots of choices

1) AWN - most popular and pretty stable

2) KibaDock - slow but a lot of fancy effects

3) KoolDock - fish eye effect like OS X Dock, but buggy

4) SimDock - does not need a composite manager like the others do

### Post by smartboyathome on 2007-09-06
Also, if you want to use one that needs a compositing manager, but don't want to use compiz, there is always xcompmgr (in repos). It does not need all the dependancies of Compiz either, so you should be able to use it even if you can't use compiz.
