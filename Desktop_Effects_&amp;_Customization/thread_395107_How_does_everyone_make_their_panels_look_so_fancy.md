### Post by Spr0k3t on 2007-03-28
Go to any blank spot for one of your panels and right click.  Select Properties, then background.  On the third radio button you can select a background image.  This image can be anything supported by the install.  I personally recommend using png with alpha so you can do blending if you need it.  Repeat for any other panel you want to edit.
