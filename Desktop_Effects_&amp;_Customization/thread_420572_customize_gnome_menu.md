### Post by Rinzwind on 2007-04-24
RightClick on the panel
add to panel; add the button 'main menu'. This is the main menu in another style.
and delete the one currently in the panel that's named 'applications/places/system'.
Thing is:
System will be at the bottom and not inside application. 

Also have a look at 'alacarte' (used to be known as 'smeg').
Just type alacarte in a terminal (it's moved around the menu from dapper to feisty so I can't tell where it is)
I think you can just drop Preferences into Applications but it allows for more than just that.
