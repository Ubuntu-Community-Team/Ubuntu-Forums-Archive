### Post by odiseo77 on 2007-07-16
Linux (ubuntu, in this case), comes with ***tons*** of different and cool screensavers; just run:

```
sudo apt-get install xscreensaver xscreensaver-data xscreensaver-gl xscreensaver-data-extra xscreensaver-gl-extra
```

Put your screensaver manager in random mode and you're done (or if you like really good screensavers, choose one of the GL* ones) :)
