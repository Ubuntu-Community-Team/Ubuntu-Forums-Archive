### Post by fracturedmorals on 2008-03-03
Go to [www.getautomatix.com](www.getautomatix.com)
Click on the installation link.
Click on the package for your version/processor type (Usually 32-bit gutsy)
and install it with gdebi package installer (The default option that firefox gives you).

When installed this program will be under "Applications">"System Tools".
It will give you the option to install MANY desirable packages including "Extra Fonts".

The "Extra Fonts" package will give you enough fonts to last you a lifetime and it contains the msttcore fonts...which in turn will include "Arial".
