Hi all.

I recently downloaded gDesklets, for which I created a launcher in my startup folder. Whenever I boot up, my desklets appear on my desktop, which is very handy :). My only problem is the fact that the gDesklet configuration window opens as well, which I then have to close. I wondered weather there was a way to make my desklets appear without having to close that window every time I boot up.

Any suggestions would be greatly appreciated.

---

### Post by the_saint07 on 2007-06-11
What worked for me was to disable the default launching and in sessions add the launching command <gdesklets>.

This will run the desklets daemon without its manager. good luck.
