### Post by Terc on 2007-10-22
First, Go here:

System>Administration>Software Sources

Check the Main and Restricted Repos.

Click Reload

Now enable the ATI Restricted Driver:

System>Administration>Restricted Drivers Manager

To make things faster, don't restart yet.

Enter this in Terminal:

```
sudo apt-get install xserver-xgl
```

Now restart, and you should be up and running.
