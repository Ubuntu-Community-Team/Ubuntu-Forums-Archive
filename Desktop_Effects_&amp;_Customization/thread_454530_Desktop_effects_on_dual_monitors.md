### Post by modestmelody on 2007-05-25
You don't want to be using Xinerama with nVidia cards, you want ot be using Twin View.

Here is an article I wrote on the matter:
[http://www.ubuntugeek.com/dual-monitors-with-nvidia.html](http://www.ubuntugeek.com/dual-monitors-with-nvidia.html)

However, it seems that you can now take care of all of this graphically with the newest version of nVidia's drivers:
gksudo nvidia-settings

Xinerama is what you use with ATI cards.

### Post by g8m on 2007-06-02
I only have 1 videocard, but have 2 monitors connected to it. No twinview, no xinerama, 2 independend (kde) sessions.

I believe desktop-effects is compiz, but maybe compiz can be started the same as beryl.

To get beryl working i do not use bery-manager but start beryl and aquamarine seperately on each monitor

Monitor 0:
  beryl --screen 0 && 
  aquamarine --replace &&

Monitor 1:
  beryl --screen 1 &&
  aquamarine &&

For some reason --replace doesnt work on the second monitor. I made 2 scripts and 2 desktop entries to this script. After booting I have to execute these scripts by doubleclicking the desktop icons, that way I can also decide to not start beryl if I need some extra stability.
