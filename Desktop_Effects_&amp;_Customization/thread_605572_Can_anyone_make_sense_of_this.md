### Post by justsomedude on 2007-11-07
In case you haven't done it already, install the Compiz Settings Manager:
```

sudo apt-get install compizconfig-settings-manager
```

Open it and make sure you have the *Place Windows* plugin enabled. Select the plugin and check *Workarounds* and set *Placement Mode* -> *Centered*.
