First, a special thanks to all those who take the time to help us Newbies

I entered these commands (one at a time) using Terminal Console and things turned out just fine. Maybe someone can tell us if I needed to install both of these:

sudo apt-get install compizconfig-settings-manager
sudo apt-get install compizconfig-settings-manager python-compizconfig

but I did, and it all worked out ok. Your experience may vary.

sudo apt-get install emerald
sudo apt-get install libemeraldengine0
sudo apt-get install compizconfig-settings-manager
sudo apt-get install compizconfig-settings-manager python-compizconfig
sudo apt-get install xserver-xgl
(  to remove xgl use:  sudo apt-get remove xserver-xgl   )
Right click on the Desktop > Change Desktop Background > Visual Effects

Hardware:

Athlon64 X2 4400+
BioStar TForce 550se
G.Skill PC6400
