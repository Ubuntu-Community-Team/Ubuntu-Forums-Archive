Hello,

I used Ubuntu 6.10 with Beryl well. Recently, I upgrade it from 6.10 to 7.04 (Feisty), and now a problem with beryl is happened.

While using 6.10 with beryl, when I run beryl-manager, it automatically load beryl and run beryl-manager itself. Now, things are changed. When I load beryl-manager, it just load itself. Since beryl is not loaded, it is useless. When I load beryl, I can use beryl-effect on my desktop, but I cannot see window frame. When I load beryl first, then load beryl-manager, the beryl-manager kill the beryl. However, when I load beryl-manager first, then load beryl, I can use beryl as I used it in ubuntu 6.10.

So, I am afraid to put beryl and beryl-manager into session to enable beryl at startup.

Anybody knows this problem and solution?

Thanks,

### Post by serialdae on 2007-05-02
click on beryl-manager and set your window manager to beryl
then set your window decorator to emerald.

you may need to restart your window and decoration manager at that time - however they should start when you select them
