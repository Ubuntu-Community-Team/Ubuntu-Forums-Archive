### Post by tact on 2008-01-24
Compiz is a window manager..Window managers  always work hand in hand with a "Window Decorator".  

 In the default gutsy(v7.10)  installation Compiz isinstalled and uses  the window decorator called gtk.

So if you look around the various websites that have themes availablel to download (like gnome-look.org) just search for gtk2 themes and install easily any you like.

Emerald is another window decorator that can work with Compiz.  Metacity is another.  So if you cannot find a theme you really like amongst all the available gtk2 themes - you can even replace the decorator (gtk) with Emerald or metacity etc...  and install themes relevant to those decorators.

Welcome to a totally new world of completely customisable desktop look and feel.  :)
