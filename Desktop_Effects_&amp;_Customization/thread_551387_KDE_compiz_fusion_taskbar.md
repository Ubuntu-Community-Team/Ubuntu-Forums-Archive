### Post by talen.quickblade on 2007-12-13
Hoping you have located the answer for this with the post being 3 months old.  If you right click on the panel and select configure panel, you'll get a config windows that has Taskbar as an configureable item along the left side.  For the behavior you are wanting, you need to un-tick the "show windows from all screens" item.
