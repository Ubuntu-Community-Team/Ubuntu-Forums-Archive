### Post by niltub on 2007-11-01
You need to install "brightside" for these to work (go into synaptic package manager, search for brightside and click to install).  The options to configure it are then under System -> Preferences -> Screen actions.

I have it installed in gutsy, but for some reason it doesn't automatically work upon startup, and I have to set it up each time I boot up.

Hope this is of some assistance.

### Post by niltub on 2007-11-01
Well, I just found how to fix the thing at startup.  I simply had to start brightside each bootup, so:

Go to System -> Preferences -> Sessions, and on the startup tab, click add and fill out the boxes accordingly, with "brightside" (minus quotation marks) in the Command box.  Click OK, and you are away.
