### Post by Ash44455666 on 2008-04-20
Ok so I have everything set up (just not the 'bliss' wallpaper quite yet), is there any way I can duplicate the Windows XP menu bar (task bar, etc.)?  The current image from GnomeXP I think it's called makes the menu use the windows logo, but it's mostly the same as before.  Any solution to this?  Thanks in advance ^^

Edit: Aha, eventually got it working mostly how I wanted it.  I set the panel's background to a 1 pixel wide, not-sure-how-many-px pixels tall image and I set the Main Menu to the Start button from Windows XP, both images cropped from this image

[http://www.guidebookgallery.org/pics/gui/desktop/empty/winxppro.png](http://www.guidebookgallery.org/pics/gui/desktop/empty/winxppro.png)

using the GIMP.  Also, I set the panel's height to 32 pixels so that the start menu button was resized and fit correctly.  Now if I can just make the text on it white it'll be nearly perfect :)

Edit2: Got text to be white.  Used a package called gnome-color-chooser.  Here's a direct link to download:
[http://ubuntuforums.org/attachment.php?attachmentid=66551&d=1208723839]("http://ubuntuforums.org/attachment.php?attachmentid=66551&d=1208723839")
Only use if you can't find package elsewhere.. this is a link I found, but it's a bit old.  If you must download/install from this link, as soon as installed, check for updates.  Ubuntu almost immediately offered an update for it anyways lol.  When installed (and updated if necessary), go to System->Preferences->Gnome Color Chooser, go to the Panel tab, and click the checkbox next to "Normal | Foreground" (The normal row, and the foreground column), click the color, and choose the one that you want (in this case white).  Click apply and close, and it'll be white now.
