### Post by superm1 on 2007-10-23
Hi everyone.

Just wanted to let you know that fglrx 8.42.3 was announced and introduces support for AIGLX.

It's very important that you modify
/etc/xdg/compiz/compiz-manager and add a line with
```
WHITELIST="nvidia intel ati radeon i810 fglrx"
```

If you would like to use AIGLX support for compiz.  Also if you have any lines in your /etc/X11/xorg.conf that are disabling Composite, they need to be removed or commented out.
