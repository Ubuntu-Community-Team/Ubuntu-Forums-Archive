### Post by mcduck on 2007-08-10
> **Brezeck said:**
> Hello guys,
I have downloaded several emerald themes but none of them can work fine.

When i enable it on Emerald Themes Manager and then restart the Xserver, only the window border gets green, but the buttons and panels doesn't change anymore... I want to know how to make it work fine! :(:confused:

Beryl/Emerald/Compiz is just a window manager, so it's themes only affect window borders and titlebars. Your application themes (which also affect the panel) are still controlled by GTK2 and icon themes (in System/Preferences/Theme).

### Post by renzokuken on 2007-08-10
buttons and panels are a completely seperate thing to window borders.

Emerald is windows decoration

Panels/Buttons (aka Controls) are themed by GTK2 themes in Gnome (see [www.gnome-look.org](www.gnome-look.org))
