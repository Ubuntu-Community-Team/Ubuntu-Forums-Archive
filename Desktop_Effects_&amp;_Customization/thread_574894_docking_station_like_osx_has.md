### Post by Jeffers on 2007-10-13
I'm assuming that you mean a program dock. If this is the case, you should use the program called AWN (Avant Window Manager). Open up Synaptic and search for awn or avant window manager, it will guide you through the installation process. You can then start AWN by going through Applications-->Accessories-->Avant Window Manager.
