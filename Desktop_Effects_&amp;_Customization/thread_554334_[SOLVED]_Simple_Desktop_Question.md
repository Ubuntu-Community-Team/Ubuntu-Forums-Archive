### Post by Het Irv on 2007-09-22
I found a way get a vertical gradient by using GIMP.
I created a bitmap with the same size as my resolution then I gave it the gradient I wanted with the Custom Gradient tool.  Added color and then the picture that I wanted over It as a layer and then saved it as a jpeg.
