### Post by jharbert on 2007-12-16
Two ways that should give you the functionality that you want
1) resize the firefox window and move it to the place you want it
2) use a tool such as youtube-dl or [http://javimoya.com](http://javimoya.com) to download the video and play it off of your hard-drive with movie player (or similar)
To install youtube-dl:
```
sudo aptitude install youtube-dl
```
