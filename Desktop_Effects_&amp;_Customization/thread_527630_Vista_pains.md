### Post by psyopper on 2007-08-16
You need to install the Emerald Theme Manager and Emerald Themes. Once installed you can go to sites like [www.gnome-look.org]("http://www.gnome-look.org") to find even more Emerald themes!
```

sudo aptitude install emerald emerald-themes libemeraldengine0
```

If you don't have Emerald installed there's quite a few guides that will walk you through it. Check one of those out and then do as i said above.
