### Post by FuturePilot on 2008-03-19
A lot of them are in the repositories. Search in Synaptic for "gtk2-engines" and you'll get a whole list. Some of the ones that aren't in the repositories have a .deb file on gnome-look.org. Once you install one you don't need to switch the engine or anything. When you set the theme you want it will automatically call the engine. Some themes call multiple engines.
