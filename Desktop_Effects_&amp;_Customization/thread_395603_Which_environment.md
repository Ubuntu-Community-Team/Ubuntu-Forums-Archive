### Post by old_geekster on 2007-03-29
The latest version of Beryl is available.  It is the first stable version since 0.1.0.  It should run great with your video card.

Here is a good guide for installing anything that you will need or want, including Beryl:

[http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Multimedia_Player_.28xine-ui.29](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Multimedia_Player_.28xine-ui.29)

You will want to install the latest nVidia driver before you install Beryl, however.  Here is the best guide that I have found for installing the latest driver from nVidia's website:

[http://www.ubuntuforums.org/showthread.php?t=336412&highlight=Hub](http://www.ubuntuforums.org/showthread.php?t=336412&highlight=Hub)

There are many other guides out there.  However, as a near newbie myself, I have used this guide successfully several times without a problem.


Follow his instructions carefully, especially where he says to make notes.  Unless your memory is better than mine, the notes will come in handy.:)
