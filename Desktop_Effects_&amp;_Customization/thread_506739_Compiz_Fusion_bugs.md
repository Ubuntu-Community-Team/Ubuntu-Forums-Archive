### Post by diafanos on 2007-07-22
I had this problem with broken windows for a period of time, when I was using Beryl, now with CF disappeared. Probably it's a bug...

For the problem with maximized windows, go to ccsm and in "Move Window" section, uncheck "Snapoff maximized windows"

As for the third problem: yes it's true that you can set a picture only to the upper cap. I tried also to find a way  to "hold" the upper cap on screen but with no result. Probably it is not a bug, just this is how Compiz fusion works...for the moment
