### Post by Rupertronco on 2007-09-19
If you're looking to change the borders, any window decorator will do that (metacity, comipiz, beryl, etc). If you're looking to change the buttons and the overall style of your windows, you need a gtk theme. Gnome-Look.org, as mentioned, has a plethora of gtk styles. Check them out, and they're very easy to install once you've downloaded them.

System>Preferences>Theme 

Click install theme and find your recently downloaded theme (you can leave it in tar.gz).
