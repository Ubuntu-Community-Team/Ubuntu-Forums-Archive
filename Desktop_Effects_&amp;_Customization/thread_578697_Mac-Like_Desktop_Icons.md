### Post by the yawner on 2007-10-18
On Nautilus (just open any folder), you can set the text to show beside the icon (by the right). Just click Edit>Preferences. There's a check box there (Text beside icons). However, when I tried this it seems to only affect the file manager but not the desktop (although afaik Nautilus settings should also affect the desktop view). 

For default icon size, it's also on the Nautilus preferences. On the same tab, under Icon View Defaults, set the default zoom level to a smaller percentage.

As for alignment, I haven't found any settings to strictly implement the icon arrangement on the desktop.
