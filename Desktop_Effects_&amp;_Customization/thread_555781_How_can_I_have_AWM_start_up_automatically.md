### Post by alwiap on 2007-09-20
I love using Avant Window Navigator, but I would like to be able to have it start up when I boot up Ubuntu. I'm lazy and I don't want to have to go to 'Applications > Accessories > Avant Window Manager' every time I startup.  I know in Windows there is a startup utility you can put programs in and they will start when you start the computer.  Something similar in Ubuntu?

TY!

---

### Post by FuturePilot on 2007-09-20
System>Preferences>Sessions
Under Startup Programs add a new entry
Title it something ex: Avant Window Navigator or whatever you wish
for the command enter this
```
avant-window-navigator
```
