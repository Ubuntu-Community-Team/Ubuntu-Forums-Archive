### Post by julian67 on 2007-10-23
Because audacious by default does not use the Window Manager's decoration. If you look at audacious it doesn't have the regular  title bar with the minimize and close and maximise buttons and enclosure around the entire application window, it uses its own style and buttons..  That's what's meant by the decoration. You can enable it in audacious: preferences>check the "Show window manager decoration" box,  but while it will enable the wobble it doesn't look good on such a tiny application and you lose the stickiness that lets the playlist window stick to the player etc.
