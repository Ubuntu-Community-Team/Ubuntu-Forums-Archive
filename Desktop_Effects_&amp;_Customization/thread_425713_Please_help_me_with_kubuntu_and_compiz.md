### Post by Kevin on 2007-04-28
compiz actually works fine in Kde.  All you need to do is install the compiz-kde package, which should pull in everything you need.

You should look around the "Desktop Effects and Customization" forum about avant.  There are several how-to's floating around, although if you're running 64-bit, you may have to compile it yourself, as its a very new program.

### Post by Nythain on 2007-04-28
compiz runs fine on kde just like any other gtk and or gnome based app... on slower machines the need to run gnome libraries might be a bit resource intense, but on any decent newer machine it would work just fine.

in fact, i know a few kde users who insist on compiz over beryl because of stability issues.

Beryl, in my experience has been an easier task to install and configure from a newb point of view, with the right repo you can get most pluggins already compiled through apt-get, whereas from what i've heard, one would have to compile most of the pluggins they want for compize... though its been a while since i've checked up on either one, that could have changed
