### Post by judolars on 2007-12-06
Try this:
1. Install the compizconfig-settings-manager package.
2. Click System > Preferences > Advanced Desktop Effects Settings.
3. Browse down to the Window Management section and enable the Scale plugin using the checkbox next to it. (Scale is the plugin that shows all the windows in your workspace.)
4. Click on the Scale plugin button.
5. Click the Actions tab, doubleclick on General > Initiate Window Picker. Select whichever corner or edge you like.

To show the workspaces, go to the Desktop section and enable the Desktop Plane or Desktop Wall (only one can be activated at a time), as well as the Expo plugin. Edit the settings for the Expo plugin like you did for the above.

Hope this helps.
