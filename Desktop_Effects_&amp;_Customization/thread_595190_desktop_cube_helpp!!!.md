### Post by kazamx on 2007-10-28
If I understand your problem right. You want to be able to press your middle mouse button and have the desktop zoom out and let you spin the cube around while the middle mouse button is pressed?

If so you need to download compizconfig-settings-manager from synaptic (system - administration - synaptic package manager - search - compizconfig-settings-manager)

once installed go to system - preferences - advanced desktop effects settings

make sure rotate cube and desktop cube are active. Also activate viewport switcher. 

if you click on rotate cube the bottom option called zoom will let you select how far the cube zooms out. 

hope this helps
