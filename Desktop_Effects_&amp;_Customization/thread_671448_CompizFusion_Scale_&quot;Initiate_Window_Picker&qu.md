### Post by ThrobbingBrain66 on 2008-01-18
You'd have to file a bug report with compiz if you want that behavior to change...I don't think there currently is an option for it.

Screen Edge:  When you move your cursor to a screen edge or corner, the plugin is activated.

Edge Button:  Same as above except that you have to move your cursor to an edge or corner and then press a mouse button to activate the plugin.
