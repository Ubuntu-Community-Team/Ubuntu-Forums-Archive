### Post by Kubunteando on 2007-09-30
So your graphic card is an Intel 945GM integrated in the Motherboard.

If it was working before I imagine something has changed the xorg.conf file.

Check the xorg.conf file in /etc/X11
type: "cd /etc/X11/"
Then type "ls -l xorg.*"

Check the dates the file xorg.conf was modified.
A backup of the previous xorg.conf is the one named xorg.conf~

If xorg.conf~ shows a date in which your 3D Acceleration was working, try this:

sudo copy xorg.conf xorg.conf.backup (this makes a backup of your current file)
sudo copy xorg.conf~ xorg.conf (this copies the working xorg.conf~ to xorg.conf to make it the active one)

Restart Xorg (Ctrl+Alt+Backspace) or Reboot the computer.

If this does not help, post the output of executing "glxinfo"
