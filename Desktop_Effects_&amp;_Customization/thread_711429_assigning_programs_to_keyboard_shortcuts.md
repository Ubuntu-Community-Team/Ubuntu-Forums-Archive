### Post by FreewareFan on 2008-02-29
I just went through this very hunt myself yesterday.  After several questions, and hours of searching for an answer, I have found out that there is really only one way to do this.

You need to install a program called Keytouch.   When you have that installed, and you have read the help file, you'll be able to assign any keyboard shortcut to launch any program or script.

I'm just fresh into Ubuntu for a week now, and it only took me about 15 minutes to install, read the directions, and get it done.  Very easy, and I'm really pleased with the results!

Try it, you'll like it!!  


Hope this helps you.
