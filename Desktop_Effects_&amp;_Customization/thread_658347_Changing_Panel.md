### Post by rainwalker on 2008-01-04
Well it's not the same thing as on Macs, but it is a dock

To use it, you have to be using Compiz, otherwise it won't work
I installed it by adding this to my /etc/apt/sources.list
> ##Avant Window Navigator
deb [http://download.tuxfamily.org/syzygy42](http://download.tuxfamily.org/syzygy42) gutsy avant-window-navigator
deb-src [http://download.tuxfamily.org/syzygy42](http://download.tuxfamily.org/syzygy42) gutsy avant-window-navigator
So open a terminal, run
```
sudo gedit /etc/apt/sources.list
```
and paste that in there, and save it.
Then open Synaptic (System > Administration > Synaptic package manager), click the Reload button, and then search for Avant.
