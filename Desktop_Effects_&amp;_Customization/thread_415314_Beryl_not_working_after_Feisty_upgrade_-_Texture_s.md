I've managed to get back into MetaCity.

### Post by acegolfer on 2007-04-20
> **Anessen said:**
> I've managed to get back into MetaCity.

What I did was go to my .beryl-managerrc which is a hidden file inside my home directory, and change active_wm to 2 and active_dm to 1. I still can't get Beryl to start though.

I got both Beryl and Compiz working.

First, I had to uninstall Nvidia beta 9755 driver by 

sudo sh NVIDIA-Linux-x86-1.0-9755-pkg1.run --uninstall

Second, start gdm using nv driver.

Third, run restricted drivers manager (auto detect the hardware) and enable proprietary Nvidia driver.

I guess I don't have to manually install (method 2) the driver any more whenever there's a new kernel installed. Hooray!
