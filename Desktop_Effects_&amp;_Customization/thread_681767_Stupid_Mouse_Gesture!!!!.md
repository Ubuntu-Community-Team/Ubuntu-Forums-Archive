### Post by vociferous666 on 2008-02-06
ok so i recently found out about a ******** of advanced options in firefox!!

to access them navigate to **about:config** in your address bar
then at the top of the actual page there is an input field called *filter*
now type in **mousewheel.horizscroll.withnokey.action** and set to 0
then filter to **mousewheel.horizscroll.withnokey.sysnumlines** and toggle to true

i got this from some SUSE Linux forum buried in the depths of the interwebs.
hope this helps the rest of you from pulling your hair out!!
