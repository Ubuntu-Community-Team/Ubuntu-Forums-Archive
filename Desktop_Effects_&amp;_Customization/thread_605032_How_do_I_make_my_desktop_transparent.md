### Post by Hospadar on 2007-11-06
it's still easy (as it's almost the exact same interface)  first, make sure you have the advanced settings manager "compizconfig-settings-manager" either install it with synaptic, or from a terminal ```
sudo apt-get install compizconfig-settings-manager
```
then go to System->Preferences->Advanced Desktop Settings
Then click on Desktop Cube->Transparent Cube

You can do what you want from here.  I think there may be some extra settings in Cube Rotate (on the main screen)
