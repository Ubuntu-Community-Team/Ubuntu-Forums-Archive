> **warhawk03 said:**
> I do not find any place with an option for the "Skydome", where can I find this?
First install ccsm:
```
sudo apt-get install compizconfig-settings-manager
```
Then go to
ccsm &#8594; Desktop Cube &#8594; Appearance &#8594; Skydome
> **warhawk03 said:**
> Also, with the 3D cube effect, I want to change the color that is on top and bottom of the cube.
