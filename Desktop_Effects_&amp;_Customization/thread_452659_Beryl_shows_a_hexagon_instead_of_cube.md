I went to the Beryl Settings manager and set the workspaces number to four, I also have four workspaces on on workspace switcher on the panel.

### Post by Znupi on 2007-05-23
Fire up Beryl Settings Manager.

[b]General Options -> General Options -> Main:
-- Horizontal Virtual Size: 4
-- Number of Desktops: 1[/b]

That should do it.
