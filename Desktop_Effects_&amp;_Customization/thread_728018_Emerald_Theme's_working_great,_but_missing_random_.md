### Post by Hammer89 on 2008-03-18
You probably need to enable Compiz's window decoration plugin. Go to System > Preferences > Advanced Desktop Effects Settings... and look for the option there. If you don't have "Advanced Desktop Effects Settings" listed under preferences try typing "sudo apt-get install  compizconfig-settings-manager" in a terminal... then check again.
