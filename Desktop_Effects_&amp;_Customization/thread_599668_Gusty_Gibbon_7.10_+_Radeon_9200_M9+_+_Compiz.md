### Post by dedalos on 2007-11-03
I dont know if this works to everybody but u can try it.

remove xgl server and use aiglx

remove <xserver-xorg-video-ati> from Gutsy in synaptic and 
download <xserver-xorg-video-ati_6.6.3-2ubuntu6_i386.deb> from feisty double click ti install it 

lock file in synaptic manager so no upgrade can be made, 

Optional:
download <driconf>, run DRI 3D Acceleration 
enable hyperZ so u can see the green <yes>

edit xorg.conf and test some of mine in your xorg.conf

Section "Device"
        Identifier      "ATI Technologies Inc M9+ 5C61 [Radeon Mobility 9200 (AGP)]"
        Driver          "radeon" #or "ati"
        BusID           "PCI:1:0:0"
	Option 		"AGPMode" "8"        
	Option          "AccelMethod"    "XAA"	
	Option          "XAANoOffscreenPixmaps"
        Option 		"AGPFastWrite" "true"
	Option          "ColorTiling"    "on"
        Option 		"EnablePageFlip" "true"
	Option          "GARTSize"       "64"
#	Option          "DynamicClocks"  "on"  #mobility only
        Option 		"DisableGLXRootClipping" "true"
        Option 		"AddARGBGLXVisuals" "true"
	Option 		"AllowGLXWithComposite" "true"

exit desktop and login again

8776 frames in 5.0 seconds = 1755.125 FPS
8767 frames in 5.0 seconds = 1753.249 FPS
8935 frames in 5.0 seconds = 1777.607 FPS
with ati 9200 mobility 64MB 

that what i did and works with compiz
