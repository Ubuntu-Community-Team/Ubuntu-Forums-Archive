### Post by hyperair on 2007-08-15
Getting Emerald to start up with the sessions eh? Have you tried changing your Compiz Fusion command to something like "compiz --replace -c emerald"? If that doesn't work, then you need to put it in two different entries in sessions. One more way is to use the Fusion Icon. You can get a deb here: [http://ubuntuforums.org/showthread.php?t=497186](http://ubuntuforums.org/showthread.php?t=497186) and you can then start up your Compiz Fusion and Emerald together using this command: fusion-icon.
