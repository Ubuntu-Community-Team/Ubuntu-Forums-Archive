### Post by forchessonly on 2007-04-29
I am wondering if it is possible to make a link to a folder on the gnome panel? I want to be able to click it and have it open that folder. That way I don't have to browse through my file system everytime I want to get to it.

-Thanks

---

### Post by Kobalt on 2007-04-29
Right click your panel and select "Add to the panel..." Select "Custom application launcher", give it a name and an icon and as command enter "nautilus /folder-to-open".
You're done. Simple isn't it ? ;)
