### Post by Rupertronco on 2007-09-30
You need to change the number of desktops to 1. Multiple desktops don't work with compiz-fusion. Compiz will draw the additional, virtual desktops required for the cube. Go to general settings in the compiz config manager (to run type ccsm in a run box or terminal) and change them to look like this.
