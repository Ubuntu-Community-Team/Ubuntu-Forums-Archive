### Post by Zorael on 2007-11-01
I don't think you can use the scroll wheel anymore, but you can set other Edge Buttons. It's not precisely what you're looking for, I guess.

CCSM -> Rotate Cube -> Actions.

Double-click on the text Rotate Left/Right, choose your screen edge, then a mouse button. I couldn't get any other buttons than Button 1 to work myself, but I'm using a touch pad, so may be different for you.

### Post by hawko on 2008-01-07
I have worked around this finally by adjusting the settings in the** 'Desktop Wall'** plugin.

_In Edge Flipping Tab: _
Activate the Edge Flip Pointer

_In Actions Tab: _
Set 'Flip Left' to have: 'Screen Egde' - 'Left | Right' and 'Edge Button' -' Button 4'
Set 'Flip Right' to have: 'Screen Egde' - 'Left | Right' and 'Edge Button' -' Button 5'

It is not quite the perfect solution but gives viewport switch with the windows maximised.

Hope this helps,

Hawko

### Post by elucidblue on 2008-04-07
I definitely needed this one, too.

Go to the Rotate Cube options, Bindings tab.  Under rotate cube set the Rotate Right and Rotate Left options for the mouse.  You can select which screen edges/corners this will work in for each.
