### Post by Sef on 2008-01-07
1) Beryl is not maintained anymore.

2) Get Compiz-Fusion.  

3) Read [Furlong's blog]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion#") to set it up.

4) System > Administration > Restricted Drivers Management  (The restricted drivers will be needed to set up Compiz-Fusion.)

5) You can download it from Synaptic.  (System > Administration > Synaptic Package Manager)
