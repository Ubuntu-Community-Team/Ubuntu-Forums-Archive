### Post by ung/xunil on 2008-03-03
Install the package "startupmanager". You can change the grub screens there. If you don't have the default ubuntu splash screen, and from looking at startup-manager here, install the package "usplash-theme-ubuntu" - maybe just by installing it first you solve your problem, try it.
