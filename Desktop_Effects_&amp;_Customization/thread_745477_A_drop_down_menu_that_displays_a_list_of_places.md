### Post by GOROSSI on 2008-04-04
I know what you mean I only have six bookmarks and the menu still crates a sub menu.

Maybe this will help:

1: Go to the desktop "**Right Click**" select "**Create Launcher**" then on the "**Type**" option select "**Location**" 

2: Navigate to the folder you want in Nautilus or what file Manager your using then **Drag and Drop** the folder to the Location Box in the "Create Launcher" dialog. 

this will create a shortcut which will open the folder you want

3: repeat for each folder you want.

4: If you want your desktop uncluttered Open your "Home" folder click on "Desktop" create a New Folder and call it system shortcuts etc and Move the shortcuts into the folder

I know it's a rubbish solution but otherwise I don't think there is much else you can do about it to solve the issue.

Unless somebody else knows different
