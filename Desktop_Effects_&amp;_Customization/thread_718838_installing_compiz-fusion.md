### Post by FuturePilot on 2008-03-08
If you are using Gutsy it's already installed. Just install any graphics drivers from the Restricted Driver Manager and reboot. You should be good to go.
System&#8594;Administration&#8594;Restricted Driver Manager.
Also if you want to play around with the settings
```
sudo apt-get install compizconfig-settings-manager
```
System&#8594;Preferences&#8594;Advanced Desktop Effects Settings
