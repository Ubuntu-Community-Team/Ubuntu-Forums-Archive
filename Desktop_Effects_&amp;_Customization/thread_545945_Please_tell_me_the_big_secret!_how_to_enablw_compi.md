### Post by hyperair on 2007-09-08
First of all are you using Compiz or Compiz Fusion? If you're using Ubuntu Feisty without going through additional instructions to install Compiz Fusion then you're probably just trying to run Compiz.

If you are trying to run Compiz:
Go to System->Preferneces->Desktop Effects.

If you're using Compiz Fusion ( in a terminal ):
```
compiz --replace
```
