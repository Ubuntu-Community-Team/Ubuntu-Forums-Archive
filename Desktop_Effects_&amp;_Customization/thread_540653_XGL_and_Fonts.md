### Post by ricardoec on 2007-09-01
Well, this is SOLVED and took it from another thread that seems to be related to this problem:

in SETTING->>PREFERENCES-->> SESSIONS enter a NEW session to run

gnome-settings-daemon

This fixed the problem fo me.

Now, you might get a message at session loading where yu are told that the X11 settings are not the same as your GNOME settings and which to be chosen.. Choose GNOME.
