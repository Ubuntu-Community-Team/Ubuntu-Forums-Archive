[COLOR=Red] Edit:[/COLOR] Ok, say your new post... (sorry if the text above looks rude...) In compiz settings manager you can set commands and keyboard shortcuts under the "general" settings.
If you as an example want to run gnome-terminal by clicking Alt+t, you could set "Run terminal Command" to "gnome-terminal" (without the quotes), and set it's key binding to Alt+t.
Any other commands can be assigned to "Command 0" -"Command 11".
