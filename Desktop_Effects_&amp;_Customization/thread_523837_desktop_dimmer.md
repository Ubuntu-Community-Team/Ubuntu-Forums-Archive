### Post by superyounan1 on 2007-09-06
the default key combo for it is Super+p.

Initially it only dims other widnows, but not the desktop. If you want the desktop to go dark too, add "Desktop" to the window types field in the Misc. Options tab for ADD helper. Mine looks like this:

```
Toolbar | Utility | Dialog | ModalDialog | Fullscreen | Normal | Desktop
```

### Post by Inxsible on 2007-09-06
you are probably looking for the Opacity plugin.

System --> Preferences --> CCSM. Click on General Options. Then click on the Opacity Settings Tab.

In the Window Opacities section click on the entry there and click edit. Then change the opacity value to 100 and you are done :)

---

### Post by Inxsible on 2007-09-06
Oh and if you don't see the Opacity Settings tab immediately, see the small forward arrow? click on it...it will take you to an additional tab which is the opacity setting tab
