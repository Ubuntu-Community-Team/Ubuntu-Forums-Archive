### Post by deep-z on 2007-08-15
Just in case, someone having the same question, here's the answer:

Simply install "kcontrol" package (sudo aptitude install kcontrol) and execute "kcontrol" from console or from your Gnome Run... (ALT+F2) and there you go. Just changed font and iconset :) ...:guitar:
