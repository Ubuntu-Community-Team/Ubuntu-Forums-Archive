### Post by ThrobbingBrain66 on 2007-07-01
If you're using the flat-file back end (open CCSM and click the 'Backend and Profile' button in the lower left-hand corner to check) you can use the export button to save a copy of your settings.  Then put it on a USB drive, load it on the other computer, and import the settings using CCSM.  This may work with gconf as your backend, but I'm not sure.
