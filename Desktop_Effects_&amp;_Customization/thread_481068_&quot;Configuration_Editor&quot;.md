### Post by luscinius on 2007-06-22
If I remember correctly I had to go to System->Preferences->Main Menu and enable the "configuration editor" item in the "system tools" subsection. After that it appears in the menu.
Alternatively one can also type
```
gconf-editor
```
at the terminal prompt.
