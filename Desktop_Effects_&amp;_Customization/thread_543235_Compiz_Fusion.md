### Post by hyperair on 2007-09-04
You actually *CAN* run Compiz Fusion in a LiveCD. I know, I've tried, and my friend who is still in LiveCD stage is happily playing with Compiz Fusion on an nVidia card.

You just need to enable the nVidia driver in the Restricted Drivers Manager, then restart X by this keycombination (Ctrl+Alt+Backspace). Then it'll autologin again, and you have the drivers required. Then you follow whichever guide to install Compiz Fusion you want, and then run it. =P

On Intel cards, you don't even need to install drivers. It can run out of the box. I've tried with a LiveCD on some of my college's computers before. They've got integrated Intel cards.

### Post by Forlong on 2007-09-08
If you don't want to (or can't) use Emerald, that's perfectly OK.

I assume you're on GNOME, so you don't need any special themes when using the **gtk-window-decorator** instead. You just go ahead using your usual themes.

Themes consist of two parts: there's the GTK-theme, that takes care of the window itself and there are the (Metacity) window boarders.
(Emerald can only replace the window boarders)

The most popular site for themes is [www.gnome-look.org](www.gnome-look.org)
And there's [art.gnome.org](http://art.gnome.org), of course.
