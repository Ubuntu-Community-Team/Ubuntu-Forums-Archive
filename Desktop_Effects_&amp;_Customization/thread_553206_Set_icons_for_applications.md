### Post by Sunforge on 2007-09-17
Normally if you right click on any programme icon and select properties, all you have to do is click on the icon picture opposite the name of the shortcut and select a new icon: you're done.

To edit menus, right click on the menu name and go from there.

If on the other hand you want to go with an icon theme, just drag it onto your theme manager.
