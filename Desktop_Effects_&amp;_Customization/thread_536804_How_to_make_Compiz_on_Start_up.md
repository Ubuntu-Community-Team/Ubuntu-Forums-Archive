### Post by Chymera on 2007-08-28
you add a new entry, call it whatever you want and add ```
compiz --replace
``` in the line underneath the name.
If you already logged in and want to start manually type ```
compiz --replace
``` in the terminal

### Post by ZipoTe on 2007-08-28
