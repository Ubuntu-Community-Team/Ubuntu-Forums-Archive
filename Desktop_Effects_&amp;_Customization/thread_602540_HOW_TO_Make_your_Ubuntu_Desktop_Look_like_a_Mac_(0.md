### Post by It_Was_Luck on 2007-11-04
This tutorial will show you exactly how to skin your Ubuntu build (preferably 7.10 Gusty Gibbon) to look almost exactly the same as the new Mac OS X Leopard introduced by Apple. Pretty much I just got tired of answering peoples "Linux to Mac" questions, so I made this guide... I took me awhile, but I think it will be worth it.

[SIZE="5"]**First off some stuff:**[/SIZE]
I'm not responsible if this blows up your computer, I mean I'm not saying it will but I don't want to be in trouble if you did a crucial step wrong and totaled your computer.

[SIZE="4"]**Background:**[/SIZE]
As I'm sure we all know the famous Mac OS X, blue background is just the one that catches our eyes as Mac like. As of recently we have many new "Mac" Backgrounds including the all famous Grass background, the Space Background, and the original Blue OS X background. 

I recommend creating a folder in your pictures folder called "Backgrounds" This will help for good organization and easy access to all your favorite backgrounds. 

Once you have done that (or designated a folder to use for desktop backgrounds) get them! Below I have the thumbnails for all 3 popular Mac OS X backgrounds. Adding these to your computer is about the easiest way to start a Linux to Mac project. 

[[IMG]http://img181.imageshack.us/img181/8540/grass43gn9.th.png[/IMG]](http://img181.imageshack.us/my.php?image=grass43gn9.png)[[IMG]http://img181.imageshack.us/img181/1053/grasswidescreenzs0.th.jpg[/IMG]](http://img181.imageshack.us/my.php?image=grasswidescreenzs0.jpg)[[IMG]http://img125.imageshack.us/img125/889/spacedesktopla5.th.png[/IMG]](http://img125.imageshack.us/my.php?image=spacedesktopla5.png)[[IMG]http://img80.imageshack.us/img80/8721/tigerwalllv1.th.jpg[/IMG]](http://img80.imageshack.us/my.php?image=tigerwalllv1.jpg)
This list below is those in order just in case you forgot:
1. Grass Background (4:3 Monitor Resolution)
2. Grass Background (Wide Screen Monitor Resolution)
3. Space Background (Either Monitor Resolution)
4. Classic Mac OS X Background (Either Monitor Resolution)

Once you have them saved to enable them do the following. Go into:

**System > Preferences > Appearance > Background Tab**

Once you there near the bottom right corner is a button that says "Add" Press that, then navigate to the new background you wish to have. Now listen carefully: Go straight over from that Add box to the drop-down box that says: Style. Drop down the box and select "Zoom" you will then notice you background fill the whole screen. We are now done with that so press "Close"

[SIZE="4"]**Troubleshooting for "Backgrounds"**[/SIZE]
Well I find it hard that you could mess up on this part but I'll do a little trouble shooting. My guess is that your either really new to Linux and Ubuntu and your just lost or there's some error. In either case I'll make this short and sweet.

There are sometimes what you might call "unexpected errors" more commonly referred to as bugs, these bugs to have potentially stopped you from applying your background, so if my way wasn't work try this:
[B]
Places > Home Folder > Then navigate to where the photo you wish for your background to be is.[/B]

Once you have found it Grab the icon (which should be the photo in about avatar size) and pull it (so that you still see the little hand with what looks like a paper in there hand) over to the desktop then release. Your photo should just have then become the background. If it doesn't look right consider a few things.
1. Is the photo the right size? (Is it made for my monitors normal/wide-screen set-up?
2. Is the photo to small? (If you image looks blurry check the photo's dimensions. They should be as close as possible to your screens resolution, the lower they are the worse the quality will be.)
3. Have I downloaded the file correctly? (Try to download the photo in a JPG or PNG file codec, these tend to work the best)

[SIZE="4"]**Troubleshooting for "Top & Bottom Bars"**[/SIZE]
Well you ran into an error didn't ya? I thought so. I'll start from the beginning with errors you might have gotten. 

First off, if you opposite clicked on your menubar and the button "Delete This Panel" wasn't highlighted for selection this is what you do. Well the reason why is because you have to have at least 1 menubar So if you don't have a top menubar add one real fast and drag it on top. Then simply try again to "Delete This Panel" and watch as the button becomes available. You may now delete the panel.

Okay so if your still reading this obviously means you've encountered an error message when you tried to apply the theme to the bar, or it just didn't work. We will do it the more complex way. Go up to the top menubar and right click (thats an opposite click) and press "Properties" It should be the second one down the list. Once it comes up your going need to go into the "Background" tab, just look in the top part of the window. Once your there go to the bottom part and put the little bubble (or circle) in the box that is labeled "Background Image" Once you have done that, click the blank space and then navigate all to way to your icon and double click it. It will then get applied.

[SIZE="4"]**Theme:**[/SIZE]
As I'm sure you know the theme is what gives it the most Leopard/ Mac effect out of all things, if you don't have this theme, well then good luck with trying to make it look like a Mac. Anyway there's a package called the Mac4Lin project, and they over the years have come out with multiple Leopard skins for Linux, so that's what we will be installing real fast.

First off, I recommend in your home folder or in one inside that create a folder and call it "Linux to Mac" this is where we will keep all of the files we need in order your your Linux system to look like a Mac. 

Next, click the link below, the download should begin, if your using Firefox (which I assume most of you are) the file (on default) will be downloaded to your desktop, simply move it to your Linux to Mac folder and untar (extract) it.

**[http://downloads.sourceforge.net/mac4lin/Mac4Lin_v0.3.tar.gz?modtime=1193256663&big_mirror=0&filesize=24095751](http://downloads.sourceforge.net/mac4lin/Mac4Lin_v0.3.tar.gz?modtime=1193256663&big_mirror=0&filesize=24095751)**

So once you've got that downloaded and untared lets apply it shall we?

First we are going to need to get to the Appearance menu in which can can apply our theme, to do this follow the following:

**System > Preferences > Appearance **

Once your there you'll notice that the first tab is on "Theme" thats exactly what we want. Now there are two different ways to do this part, I'll do the easy part, but if it doesn't work check the Troubleshooting Guide following this section for another way to do it.

Okay so now that we have that open put it into the top left corner of your screen then go to "Places", Home Folder, then navigate to where you saved the untared version of Mac4Lin's desktop changer. Double-click on the icon, you will then be shown a bunch of different folders including cursors, docks, Emeraled, Fonts, Sounds, and GTK Metacity Theme, that exactly what we need. So double-click on that and you'll notice that inside that folder there is two items and a README file. Once should say Mac4Lin_GTK_Aqua_v0.3 and the other one is the same except it doesn't say aqua in it. The one I recommend you use is the one with Aqua in it.

Now put this window in the bottom right corner of your screen. Now simply grab the Mac4Lin_v0.3_GTK_Aqua.tar.gz file and move it over into the Appearance settings box right in the middle of the screen, then release and it should automatically begin downloading. Once its done some things may change by default, others may not. For example you desktop background could change, if this happens simply go back to the top and change it back.

Now to tweak it up just right... Now near the bottom right corner you'll see a button that says "Customize..." Click that and you'll notice a new window with 5 different tabs come up. Lets go to the first one, Controls. This will skin the controls in your apps. Scroll down your list until you see one called Mac4Lin_GTK_Aqua_v0.3 OR Mac4Lin_GTK_v0.3, once again in this case the number doesn't matter. Now if they have a little display picture you should easily know that regardless of either one you have they are the same, so double click it (one of the two if you have both) once you have clicked it, it will be applied.

Now onto tab number 2, its called "Colors" but everything in that tab cannot be edited and if it can don't touch it, this does not need to be touched, so onto tab 3 known as "Window Border", click it as just as we did before scroll down until you see the one that starts with Mac4Lin, it may end in anything but it has to start with that, also the display picture on the left hand side will easily show you that its Mac-like. Double click it and its applied. On to the next tab.

Tab number 4, icons. Now this part is what I would call optional, personally I don't like the Leopard icons they have provided so I don't use them, but if you wish to use them do just as you did before. Scroll down on the list until you see either LeopardX or OSX, they are different, but they both have the same base so don't be mad if you didn't get the other one. Just select the one you have and it will be applied.

Now on pointers you'll notice is the last tab, the default pointer is pretty Mac like, but I never felt this part was necessary so I didn't even bother with it. Now if you want to fiddle around with it there is a folder in Mac4Lins folder called cursors. Fell free to play around with those. But I haven't done those before so I don't know much. Know if anyone would like to contribute to this part and write something small on pointers that would be fine with me.

[SIZE="4"]**Docking Application:** [/SIZE]
Once again as I'm sure you all know the one thing that makes a Mac a Mac is mostly its unique docking application at the bottom of the screen. This handy little dock eliminates the need to open up a start or applications menu to run your most frequently used applications. Now I personally would HIGHLY recommend you get AWN or Avant-Window-Navigator. Though there are others such as Kiba-Dock & Sim-Dock AWN is by far one of the most customizable docks around. Now below I'll show you how to 1) Install Awn 2) Skin it to look like Leopard 3) Add custom icons to it & finally 4) Give you some basic tips on AWN on how to run it the best. Also following this section is a trouble-shooting section for AWN, if you have Kibadock or Sim-Dock and you need help don't expect to find much here... Also if anyone would like to contribute to this guide to show how to install Sim-Dock, or any other dock, then show how it can be skinned to Leopard then make a Trouble-shooting guide on it, it would be greatly appreciated. Also remember that you must have either Compiz, Beryl, or Compiz-Fusion to install AWN. So let's start shall we.

First off, installing AWN... In order to do this we are going to need to download it from the Terminal. So go **Applications > Accessories > Terminal**. This will then open up terminal. The first thing we need to do is get you logged in as a root (admin) into your computer, now by default you might be but we are just making sure. So type (or just copy and paste) this code below into your terminal.
```
sudo -i
```
Once you do that it will appear and say [sudo] password for [Your Username]:
If you can guess type in your password. Now for safety reasons terminal will not display your password at all, not even in circles or asterisk. It will simply show nothing, so don't think your keyboard isn't working when nothing appears, just type in your password slowly and press [Enter] on your keyboard to execute the command. If you do this correct it will appear and say this: root@[Username-desktop-name]:~#  << Thats correct, thats what it should say this means that you are logged in as a root. Now we are going to have to edit our sources list.

As I said we have to edit our sources list with the appropriate files. Now please go to your distro weather it be gusty or feisty Anyway first off it get to your sources list type in (or copy and paste which I recommend you do) the code below:
```
sudo gedit /etc/apt/sources.list
```
A separate window will then open. This is your sources list, this is VERY IMPORTANT that you do exact what I have here. . Okay so with that window open scroll down to the very bottom and go the the line right below the last entry. Now take the code I have below and copy and paste it into that section of your sources list.

**Code for Gusty Gibbon Users:**

```
deb http://download.tuxfamily.org/syzygy42 gutsy avant-window-navigator
deb-src http://download.tuxfamily.org/syzygy42 gutsy avant-window-navigator
```

**Code for Feisty Fawn Users:**

```
deb http://download.tuxfamily.org/syzygy42 feisty avant-window-navigator
deb-src http://download.tuxfamily.org/syzygy42 feisty avant-window-navigator
```

Now once you have that in, go up to the top and press File > Save. This will save the changes you just made. Once you have that done (saved) exit your sources list and go back into your terminal. It should be back at the root thing still. Now we are just going to run the rest in our terminal to get it installed. Also if I forgot to tell you this will also install some applets for AWN, not Third-Party ones, but that standard ones (that can be very helpful). Anyway for the next part be sure you run one line at a time, and if you get any confirmation messages saying "10.3MB will be unpacked, continue? [Y/S]" Your obviously going to say Yes. As I said only run one line at a time.
```
wget http://download.tuxfamily.org/syzygy42/reacocard.asc
sudo apt-key add reacocard.asc
rm reacocard.asc
sudo apt-get update
```
Then to get those nice little extras...
```
sudo apt-get install avant-window-navigator-bzr awn-core-applets-bzr
```

And with that you should be set with AWN. Now to run it on your keyboard press [ALT] + F2. And a little "Run" box with open. Type in this: avant-window-navigator << Now it might auto-fill in itself signifying a good install and it might not but that doesn't always mean it doesn't work. Once it opens you'll see a standard old bar at the bottom, to get applications on it simply go into your Applications menu, select the one you like then drag and drop it over onto the bar. It should then appear. Now to skin it to Leopard.

As you may have noticed in Mac4Lin's download it also contained a folder called "Docks". Now we are going to install it. Go to a part of your bar that has nothing on it (but pretty close to an icon so it knowns that you meant to click the dock) and right-click on it and press "Properties" From there go over to the far left side and select "Theme's " On the bottom of the list.  Now go over to the far right and press "Add" A browser window will then open. Navigate all the way into your Docks folder then into the Avant one. You'll see two options. One called "Transparent.tgz" and "OSX_AWN_theme_v.0.X.tgz" where I replaced my version number with X, it may be anything. Anyway I HIGHLY recommend you select the OSX one, its much neater and well put together but your probably thinking "I want it to be transparent though!" Well it still will be...

So once thats installed select it in the list (back in the AWN properties window) and watch as the dock changes. Now one thing you may have noticed it that it shrunk! Yeah thats what I said to... So to make it bigger go back to the Properties window and select "General" on the left side. Then go to the tab that says "Bar Appearance"  From right at the top the look should say "3D Look" if not change it. Then there should be three numbers below for the Bar Angle, Height, and Icon Offset. You may move these up to whatever you like. My settings are as the following, I recommend you try them out, it works very well in this setup and makes the dock look very professional along with having the "Mac" touch.

[B]Bar Angle: 60
Bar Height: 46
Icon Offset: 15[/B]

You may then press close on the Properties Window. You like it don't ya? Yeah i did to. So take this time and drag all of your favorite applications over to the dock. (Tick-tock-tick-tock) Okay times up. You got all your favorite apps on there? Great. Now I'm sure one thing you say is "Oo, I hate those icons" For example the open office icons are hideous. So on my dock I replaced them with the big W, P, and X, letters that Mac computers have for Word, Powerpoint, and Excel. 

Anyway to get your icons... Some people like to add there own touch and use other icons. So I recommend you make a folder called "Icons" somewhere. Now go to a website such as **[www.deviantart.com](www.deviantart.com)** and download some dock icons, they even have a category for them. Just go downloading a ton, then look at all them all. I will probably upload my icons folder which has over 500 Mac icons to a file hosting site sometime soon. As for now check out **[www.deviantart.com](www.deviantart.com)** search in the category "Dock Icons" and find some ones you like. 

Now real fast to apply them. Go to the icon you wish to change, right-click on it and press "Change Icon" Then navigate to the icon in which you wish to apply to the application on your bar. Very simple it is, and fun to, you'll be amazed with how unique AWN is. Anyway below I'll just give you a tip or two about running AWN.

First off. AWN or avant-window-navigator is located in Applications >Accessories > Avant-Window-Navigator. It should probably be the first one on the list. Now if you feel your going to be using AWN everyday on don't want to have to open it every time just set it as a start-up program. To do that do the following.

**System - Preferences - Sessions - Start Up Programs Tab**

Then from there press new and enter this:

[B]Name: AWN
Command: avant-window-navigator[/B]

This will make AWN auto-start when you log it. This way you don't have to waste time opening it every time you log on.

[SIZE="4"]**Trouble Shooting for "Docking Programs":**[/SIZE]
Okay well if you think about it, the one most common problem will probably me that "I can't install AWN!" Well I've seen dozens of different ways to do it and this is the one that worked for me. But I had a friend try to install it this way and it just didn't work. So below is what I would call an alternative method for install, now I know his method is more complicated so if it doesn't work you might want to google "How to install AWN" or just do a search of the Ubuntu Forums.

Okay the first think we need to do is sign in as a root (admin) remember? Type in:
```
sudo -i
``` 
Then enter your password when propmted. So the first thing we will do for this alternative way is to prepare your system. So remember run this stuff in the Terminal.
```
sudo apt-get install build-essential subversion automake1.9
```
Now we will install the dependencies. Now this code is very very long so save yourself about an hour and Copy And Paste it!
```
sudo apt-get install bzr libgtk2.0-dev libwnck-dev libwnck-common  librsvg2.0-c libgconf2-dev libglib2.0-dev libgnome2-dev libgnome-desktop-2 librsvg2.0-c libgnome-desktop-dev libdbus-glib-1-dev gnome-common libxcomposite-dev libxdamage-dev librsvg2-dev libgtop2-dev libgnome-menu-dev libgnome-menu2 libgnome-menu-dev librsvg2.0-cil librsvg2-2 librsvg2-common librsvg2-dev libgtop2-7 libgtop2-common libgtop2-dev python-dev python-gtk2-dev python-cairo-dev python-gconf python-gnome2-dev
```
Now lets build AWN up from the source. Remember to run each line one at a time.
```
bzr co http://bazaar.launchpad.net/~awn-core/awn/trunk avant-window-navigator
cd avant-window-navigator
./autogen.sh && make && sudo make install
```
Now you should be set with AWN, now go back up there and continue where you left off. If this method didn't work either you consider a few things first:
1. When I edited my sources list did I remember to save?
2. Was I logged in as a root (admin) when I tried to install?
3. Am I putting the wrong code in my sources list? (Remember at the moment this part id for Gusty only)
 And if none of that works after you checked it all you can search to Forums here, or just try google... 

Okay well I believe theres one other problem you could have run into. You could have gotten AWN installed but maybe the skin didn't work. So we will do this the easy. Way. Go to:

**System > Preferences > Awn Manager**

Then go to the Themes tab on the left side. Then put that window aside for a second and open a browser window and navigate to the "Docks" folder in Mac4Lins folder. Then grab the OSX one and bring it over to the AWN manager and release, it should then begin the install.

Sorry. I hadn't checked back in here for a while. 
To change the gnome-panel menu icon:
Alt+F2 to open a run dialog. Type in gconf-editor and click Run.
You will go Apps -> Panel -> Objects -> object_0 (number could be different).
Look for the custom_icon setting and modify it to the path/icon.png that you want to use.
It changed as soon as I changed the setting but if necessary run "killall gnome-panel" to restart the panel. You should now have your new icon.
Good luck.
