### Post by Ub1476 on 2007-08-06
Compiz Fusion is really good. There's a "Compiz Fusion Mini-Howto" on the forums here. Just search for it. A great OSX-look guide is here: [Laurias OSX guide]("http://www.taimila.com/?q=node/11")

For vista, just search around at [Gnome-look.org]("http://www.gnome-look.org"). Look at "Beryl"  and "GTK2.x" then most downloads, which should show you some vista themes. 

If you go to menu & toolbar settings (preferences) you can choose to only view icons instead of icons and text. It looks much cleaner then.

If you have the "download.tuxfamily" source in your sources.list file, you can download Emerald theme manager (support transparent menus etc) by typing: "sudo apt-get install emerald" (eventually add emerald-themes if you want download a few sample themes).
