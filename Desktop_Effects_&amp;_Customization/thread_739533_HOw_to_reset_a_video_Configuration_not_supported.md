### Post by myquealer on 2008-07-20
I hope you have solved this by now,but I just encountered the same problem and eventually discovered the solution.

After booting hit control-alt-F1 to drop into a terminal and login. Then type


```
sudo vi .gnome2/monitors.xml
```


In there you will find a line resembling:


```
<rotation>upside-down</rotation>
```

Move the cursor to this line with the arrow keys and type dd to delete the line. Then type :wq to save the file. Finally do:

```
sudo reboot
```

And hopefully it will be fixed for you (it was for me).
