### Post by Rupertronco on 2007-10-16
You don't need XGL. In fact, DO NOT use XGL. It'll just sap performance. Intel cards can use AiGLX which is far superior. XGL is outdated. Intel has native support for this so you won't need restricted drivers. Remove XGL and the XGL session, it's only necessary for ATi cards. You need to get compiz running under plain old x.

Are you sure you posted your entire xorg.conf file? It seems to be missing some things, and by things I mean entire sections. You might need to run a reconfigure command or make sure you didn't miss any sections of your xorg.conf copying it. What driver are you using?

You're going to have to remove your current compiz install, xgl, make sure you're using an updated driver, and reconfigure your xorg.conf file back to the default before we can continue. 

Also here's a guide to installing compiz using AiGLX for your specific graphics card:

[http://lenrek.wordpress.com/2007/06/18/24/](http://lenrek.wordpress.com/2007/06/18/24/)
