### Post by MichaelSM on 2008-01-26
I've checked out a lot of stuff and I am a slow learner.

Also I have described myself as a minimalist, and that's dead set true.

I like having the cairo-dock, but simply HATED waiting for Beryl to fire up so that I could have the 'transparent' look. 

However I'm happier now with xcompmgr which is in Synaptic. It gives transparency to cairo-dock without the black background. And without Beryl.

System>Preferences>Sessions>Startup Programs>New> ..type in the box  'xcompmgr -c -f -n'  (spaces are important!...without the inverted commas of course)

Also add, using the above route, 'cairo-dock'. If  you haven't already done so.

What this achieves on next boot ought to be cairo-dock running in transparent mode with Metacity (Gnome Window Manager) . You might have to access Beryl-Manager to select Metacity as the default Window Manager if previously you'd set Beryl.

Most of this info came from a Ubuntu Forums site which I can't remember. Thanks to all.

I just added a few comments hopefully to make things easier.

So, I still have Beryl-Manager on my startup list. Which means I can get 3D giggles if I want to by changing Window Manager. But I don't need Beryl to make cairo-dock transparent. 

Cheers,

Mike.
