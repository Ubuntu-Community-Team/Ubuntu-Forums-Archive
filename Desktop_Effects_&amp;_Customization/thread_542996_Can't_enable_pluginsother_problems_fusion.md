### Post by Zorael on 2007-11-03
As for plugins not starting, you could try moving to a flat-file configuration backend.

If you start compiz from a launcher or a terminal, add the argument 'ccp'. For instance, I start it with
```
$ compiz --replace --loose-binding --ignore-desktop-hints ccp &
```
Starting it with that argument makes it use its own configuration file format, which is arguably better for some reason.

Then go to preferences in ccsm, and export your profile someplace (desktop?). Switch Backend to Flat-file, import your saved file.

This fixed my issues.


As for the other problems you describe, I'm not completely sure I follow, and my knowledge of how to make it work in a dual-screen setup is somewhat limited.
