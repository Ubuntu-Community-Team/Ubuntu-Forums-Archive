### Post by lawr_rawr on 2007-10-29
When you start Awn from the terminal, then close that terminal, you also close anything that's currently running in that terminal.

Try this -- Alt-F2 will bring up a launcher window. Enter the same command you used to start Awn, and it will launch, without the annoying terminal window in the background. Alternatively, you can add Awn to your Sessions items so it will start automatically when you log in.
