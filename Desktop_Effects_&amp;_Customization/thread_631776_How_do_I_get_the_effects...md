### Post by davidgarcin on 2007-12-04
If you are using Gutsy, the latest version, it's just a matter of installing this package through synaptic:  compizconfig-settings-manager.

See this link for installation instructions if you're not sure: [http://monkeyblog.org/ubuntu/installing/#installing_with_synaptic](http://monkeyblog.org/ubuntu/installing/#installing_with_synaptic)

Once this package is installed, go to System -> Preferences -> Advanced Desktop Settings, and just start playing around with the bazillion options.

Hope that helps.
-David
