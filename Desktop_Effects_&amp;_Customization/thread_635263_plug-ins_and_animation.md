### Post by Ub1476 on 2007-12-08
Make sure Advanced Desktop Effects is installed:

```
sudo apt-get install compizconfig-settings-manager
```

Now open System>Preferences>Advanced Desktop (..)

Go to Effects>Animations>Minimize-animation>Edit>Choose magic-lamp.

You can do the same with "Close-animation" "Open-animation" etc..

Remember to set Visual Effects (Compiz) to custom.
