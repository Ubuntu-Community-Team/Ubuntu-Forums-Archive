### Post by RDV on 2008-04-04
I am not exactly sure if I understand your question. I use a "Stack Applet" as a place to store quick launch program links. This keeps the number of displayed icons in AWN to a minimum. I still use the "MiMenu applet" to access the full gnome main menu. In AWN you actually have several alternative main menu applets (Awn Main Menu, Cairo Main Menu  (2 versions) and MiMenu). 

To propagate the "Stack Applet" with your favorite programs I did the following:
0) Of course add a "Stack Applet" to AWN
1) Open a file browser and navigate to "/usr/share/app-install/desktop" 
2) In the file browser find an application you want in the AWN Stack Applet
3) Drag and drop the desired applet desktop file onto the AWN Stack Applet icon

You will note that some applet desktop files will not propagate the AWN Stack Applet. For example Audacious would not work but Audacity Sound Editor was no problem. I cannot explain the difference but I was lucky as all the programs I wanted in the Stack Applet drag-and-dropped without issue.

Good Luck
