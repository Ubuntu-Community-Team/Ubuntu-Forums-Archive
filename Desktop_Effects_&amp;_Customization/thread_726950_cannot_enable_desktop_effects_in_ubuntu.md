### Post by notwen on 2008-03-17
The Intel 965GM chipset has been blacklisted in Compiz.  A way around this can be found [here]("http://ubuntuforums.org/showthread.php?t=582112"), but using this workaround will cause issues w/ video playback.  This however can be fixed by changing your video app's video output to 'no Xv'.  Post back if you have any questions.  =]
