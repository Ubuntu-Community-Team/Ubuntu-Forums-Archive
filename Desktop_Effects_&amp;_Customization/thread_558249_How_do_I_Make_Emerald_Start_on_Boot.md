### Post by kh1116 on 2007-09-23
i set compiz fusion to my session menu, and thats all good, but i also want emerald. i can run emerald after i log in by typing emerald --replace, but when i put that into my session menu, for some reason neither compiz or emerald works. how do i get both to boot together, so i can have emerald with compiz?

---

### Post by Islington on 2007-09-23
In the compiz settings manager, under the window decoration plugin, type in "emerald" (no quotes) in the option of Command. That makes compiz run emerald by default. Then under session put in an entry for Compiz.
