### Post by Rajiv_Nair on 2007-06-02
ive found double clicking dosent apply the theme in the latest versions of beryl. To work around this prob, select a new theme by single clicking on it, then right click on the red diamond icon of beryl in the system tray and select "Reload window decorator". Then ur new theme will be applied
