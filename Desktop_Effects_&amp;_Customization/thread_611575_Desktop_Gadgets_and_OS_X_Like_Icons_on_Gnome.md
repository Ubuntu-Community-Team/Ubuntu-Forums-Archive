### Post by mcduck on 2007-11-13
For desktop effects, just install the Compiz Configuration Manager from Synaptic. (the package is called 'compizconfig-settings-manager')

For widgets, there are many different programs to do this. If you ask me, the best one is Screenlets. gDesklets is also quite popular but not quite as good as Screenlets is.

Once again, for the mac-like dock you have several options. There are loads of different dock applications available, but you can also just use the Gnome's own Panel to create one. Just drag the application launchers you want into the panel, then right-click it, select 'Properties' and there you can disable 'Expand' to make the panel only as big as the launchers/applets in it require, make it a bit bigger, and in the background-tab either make the panel transparent or set it to use any image you want as background image. If that's not good enough for you, try Avant Window Navigator.
