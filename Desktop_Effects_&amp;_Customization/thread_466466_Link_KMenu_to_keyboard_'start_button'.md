### Post by jhenager on 2007-06-13
I use Ubuntu with Gnome, but I couldn't set the Windows key as Super (bringing up the main menu as the Windows key does in XP) until I changed some settings around and then reset them to the default.
It seemed like a simple enough task until I tried it.
What I eventually did was go into System>Preferences>Keyboard, and under the Layout Options tab, changed the Alt/Win key behavior to Alt is mapped to the right Win-key, and Super to Menu. Then closed out, went right back in and chose Default.
Following that, I was able to go into System>Preferences>Keyboard Shortcuts, and modify 'Show the panel menu' by pressing the Windows key. It mapped it to Super L. I only have one on this keyboard (which is enough).
Prior to that, the best I could accomplish was Win-key+z or some other two key combination. Funny thing was, I could assign the Menu key to bring up the panel menu, but not the Windoze key.
After several years of using the Windows keyboards and ignoring the Windows logo shortcuts (in Windows) I discovered that they were actually very useful, and not just Microsoft tripe. Glad we can leverage that in a real operating system, too.
