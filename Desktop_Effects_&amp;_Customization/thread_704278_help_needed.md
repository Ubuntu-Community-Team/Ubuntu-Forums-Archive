### Post by ajeetraj on 2008-02-22
I am guessing you have Gutsy installed on your comp. well, you need to install compizconfig-settings-manager for the advanced options. To do this write this in terminal ```
sudo apt-get install compizconfig-settings-manager 
``` Then after you have installed this then for the advanced compiz options, you can either just go ALT+F2 and type compizconfig-settings-manager  or just open System > Preferences > Advanced Desktop Effect Settings and then you can just configure Compiz just the way you want it. 

hope this helps :)
