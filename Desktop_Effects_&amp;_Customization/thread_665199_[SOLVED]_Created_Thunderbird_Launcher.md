### Post by erfahren on 2008-01-12
just a note - .svg are Scalable Vector Graphic files - Inkscape supports them and it's available in the repos - to get it:
```

sudo apt-get install inkscape

```
with Inkcape you can open SVGs and export them to .png

(launcher icons can be .xpm, .png, .svg - probably supports other formats)
