### Post by Sunflower1970 on 2007-07-31
Yes, Opera is compatible with Ubuntu. I use it on my older computer in place of FF. I like it. 

It's faster than FF, and I like the downloader better than FF's if I download a large file. Best feature in Opera is SpeedDial. Firefox has it as an extension, now, but I like that Opera comes with it, not a separate extension. 

I've tried Epiphany, and for me, I didn't really like it.
