### Post by sdennie on 2007-08-23
It's been a while since I've used an intel card but, if I remember correctly, VideoRam is in kilobytes.  VideoRam 128 would only give you 128K of video RAM.  By default, the card should give you 32M of video ram (as per the man page) but, if you want to use 128M, I believe it would be VideoRam 128000 (which it will round to something else but give you about that much),

I believe the reason your 1280x1024 setting isn't working is because the max defined value you have is 1024x768 in your xorg.conf.  If you look for 1024x768 and add "1280x1024" in the section for 24 bit color depth (or all the sections for good measure), it might fix the problem.  Make sure to add it as the first value in the list.  It looks like you are using an LCD so, just adding that should work, I think.

Long ago it was necessary to use something i915resolution (or i810) to properly setup graphics resolutions for intel integrated stuff but, I don't believe that's needed anymore.  If the two things above don't work that could be issue though.
