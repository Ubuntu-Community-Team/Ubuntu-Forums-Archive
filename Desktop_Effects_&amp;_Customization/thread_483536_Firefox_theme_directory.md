### Post by JasonFH on 2008-06-04
This may be a dumb workaround, but no one has posted yet and it worked for my purposes.  I also have been unsuccessful at finding the themes directory for firefox.  

My solution is.
1. Downloaded the .jar file to your desktop.  
2. Open Firefox 
3. go to tools --> add-ons --> themes 
4. drag the .jar file from the desktop into the themes selection window.

It installed it with no problems.
