### Post by aysiu on 2008-01-27
Go to System > Administration > Login Window > Install Theme and select the theme you downloaded from gnome-look.org

1. Make sure the theme you downloaded is a GDM theme.
2. Do not extract the theme manually. It should still be in .tar.gz format.
