Hi!
XGL is not always necessary. You'll need it to run Compiz with some ATI drivers. However, if you have an NVidia card, its drivers support AIGLX, and there's no need to use XGL.
Right now I'm using Compiz-Fusion and XGL with my ATI, and I can't complain about stability. About the looks, well, it's **REALLY COOL!!!** If you want to know how nice it is, you could look for videos on youtube, it's full of them!
The only thing I can complain is that XGL sometimes uses a lot of my CPU cycles, and my computer gets a little slow sometimes, or even freezes some apps. But most of the time it runs smoothly.

BTW, I think I should leave my specs, my laptop is not exactly cutting-edge:
Toshiba Satellite L20-SP119
1,5GHz Celeron M
192MB RAM
64 MB ATI Radeon Xpress 200m video card
I'm using 500MB for SWAP
