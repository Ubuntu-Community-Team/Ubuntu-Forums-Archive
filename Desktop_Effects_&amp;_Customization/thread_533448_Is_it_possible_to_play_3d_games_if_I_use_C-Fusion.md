### Post by Inxsible on 2007-08-24
If your processor and the graphics card are bleeding edge, then you might be able to. Remember, that the drivers also need to be bleeding edge for Linux to harness the entire power of your graphics processor.

Its normally hard to run multiple 3D rendering engines by the same graphics processor which is why there are often problems. and in any case, if you are playing a game, it would be very annoying, if lets say suddenly your snow plugin starts or the viewport switches to another desktop. You would probably "die" if you were playing a shootout game ;)

### Post by Shne on 2007-08-24
It takes a second to switch between Compiz and Metacity with two launchers on desktop with following commands:
metacity --replace
compiz --replace

but I've been able to run BF 2142 through wine fine with Compiz still running, but it was in a Virtual Desktop (you can set one up in winecfg, under graphics), I think that makes a big difference.
fullscreen apps and compiz generally don't play nice (in my experience).
