It doesn't matter where in the list you add "dbe" as long as it is in the "Module" section

> Where would I add the...

Option  "XAANoOffscreenPixmaps" "true"

Add it here:

```
Section "Device"
	Identifier	"Intel Corporation 82845G/GL[Brookdale-G]/GE Chipset Integrated Graphics Device"
	Driver		"i810"
	BusID		"PCI:0:2:0"
        Option         "XAANoOffscreenPixmaps" "true"
EndSection
```

I use an ATI card and also had to add the following line to the "Device" section:
```
Option       "AccelMethod" "XAA"
```

Not sure for Nvidia. You can try it without and then look at /var/log/Xorg.0.log for warnings or errors:
```
$ cat /var/log/Xorg.0.log | grep -E  '(EE|WW).*XaaNoOffscreenPixmaps'
```
