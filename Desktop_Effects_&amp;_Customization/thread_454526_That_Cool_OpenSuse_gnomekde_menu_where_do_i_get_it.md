### Post by benanzo on 2007-05-25
```
sudo apt-get install gnome-main-menu
```

Then right-click your panel and select **Add to Panel...** then under **Utilities** add the new **Main Menu** option.
If it doesn't show up, open a terminal and do:
```
sudo killall gnome-panel
```
then look again.
