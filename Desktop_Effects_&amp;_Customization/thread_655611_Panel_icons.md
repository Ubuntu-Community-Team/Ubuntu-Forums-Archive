[http://www.gnome-look.org/content/show.php/OSX?content=31618](http://www.gnome-look.org/content/show.php/OSX?content=31618)

You probably have to change them manually:
right-click on the icon, select Properties.  Click on the icon (left side dialogbox) to choose a new one.

> 
also how would i get a application bar along the bottom of the screen like the screenshots on the following page

[http://www.gnome-look.org/content/show.php/LiNsta+%28LiNsta+is+Not+Vista+%3B-%29?content=42697]("http://www.gnome-look.org/content/show.php/LiNsta+%28LiNsta+is+Not+Vista+%3B-%29?content=42697")

Right-click on the bottom panel, select Properties.  In the "Panel Properties" dialogbox, uncheck "Expand".  Increase the panel size.  Populate the panel with your favorites icons.  Move the Window List applet to the top panel.

### Post by Mr Greencastle on 2008-01-02
The bottom 'dock' is actually a program called Avant Window Navigator. 
For information on how to install this program see this thread here: 
[Installing Avant Window Navigator]("http://ubuntuforums.org/showthread.php?t=385981")

As for the look of the panel up top, you have to set the image yourself. Right click it and select properties and choose the background. You'd need an image though.

Hope that helps!
