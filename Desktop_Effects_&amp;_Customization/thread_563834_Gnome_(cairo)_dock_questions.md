### Post by patmalcolm91 on 2007-11-06
not sure on the rest since i've never tried gnome dock, but to run it independent from a terminal window, hit Alt+F2 and enter the command. To automatically launch on startup go to System->Preferences->Sessions, the first tab should be "Startup Programs" click the "Add" button on the right, then enter a name (can be anything) the command to start the app (probably "gnome-dock", but i'm not sure) and a comment if you like.then simply press "OK" and exit the session manager.

1. Right Click the icon, then "Remove this launcher"
2. Navigate to the shortcut though your gnome-menu, e.g. Applications-Internet-Firefox... click (and hold down button) on the application you want add, then drag it into your dock.
3. They come with the application, but you can change this with a right click, then "modify this launcher"
4. & 5.  System-Preferences-Sessions-Startup Programs-add.  enter "cairo-dock" in the command box.
