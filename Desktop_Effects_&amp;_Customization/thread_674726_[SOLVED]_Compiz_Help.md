### Post by emarkd on 2008-01-22
If you want to fully explore the options Compiz has, you'll need to install the config manager.  Run:
```

sudo apt-get install compizconfig-settings-manager
```

You'll find the background settings under the "Desktop Cube" plugin.  Select  the "Appearance" tab at the top and you'll see "Skydome" at the bottom.  That's what you're looking for.
