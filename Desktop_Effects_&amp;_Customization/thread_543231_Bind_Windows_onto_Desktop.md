### Post by smartboyathome on 2007-09-09
> **rainwalker said:**
> You could set them as Widgets so that they'll all display when you do the command to show widgets

To do this (if using compiz fusion) activate widget layer and then set your gdesklets to act as widgets. Now, by pressing F9, you can display all your desklets without the hassle of minimizing every window. :)
