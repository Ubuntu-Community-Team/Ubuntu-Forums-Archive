### Post by Achtung on 2008-03-26
You need to install the rotate cube plugin if it's not done already. 

Open the compizconfig settings manager and click Rotate cube.
In the general tab, set the zoom slider to something like 0.3.
In the Action tab, set the key/click action you want to activate the cube (the default is ctrl+alt+left click, alternately middle click).
Click your desktop with that particular key/click combo to activate the cube, and move your mouse around to move the cube. Let go of the mouse button to return to normal view.
