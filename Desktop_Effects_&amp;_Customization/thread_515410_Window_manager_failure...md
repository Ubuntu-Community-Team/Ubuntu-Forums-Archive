### Post by atomkarinca on 2007-08-02
(i'm assuming you have compiz fusion icon installed)

if you got no borders, right click the compiz fusion icon and choose "select windows decorator". you should see more than one options. switch between them until you got the borders. if you see one option then you can install emerald:

sudo apt-get install emerald emerald-themes

as for the games goes, again with the help of the compiz fusion icon, you can change your window manager to metacity before you start the game and change back on when you're done.
