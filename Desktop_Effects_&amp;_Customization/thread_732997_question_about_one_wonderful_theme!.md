### Post by Ub1476 on 2008-03-23
The dock is AWN (avant-window-navigator). There's no applet which looks exactly like the one on the mockup though. 

You can find dark themes on gnome-look.org. Themes uses different engines, and I suggest the Aurora engine (search for it on gl), even though there's no black and orange theme with it, you can customize colors from the Appearance window. 

There's a few windowborders which integrate with the window [(like blended)]("http://www.gnome-look.org/content/show.php/Blended?content=26050"), but none which can have the menubar (File, Edit, View ...) on it, as I know.. 

With a little work (or requests), I suppose it can be done almost like on the picture:)
