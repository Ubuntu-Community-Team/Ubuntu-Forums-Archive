### Post by benerivo on 2008-02-10
In addition to the compiz package make sure you have the 'compiz configuration settings manager' package installed as well. Then in the main menu there will be an entry for 'advanced desktop effect settings'. From here you can enable the cube (use Ctrl and Alt keys along with the mouse button to move the cube).

Your font settings are fine.

I think the daemon failure is quite common. Logging off and back on will fix it. I'm not sure why it happens, but there'll be tens of posts in these forums going in to detail.
