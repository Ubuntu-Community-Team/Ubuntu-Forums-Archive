### Post by hamstersquasher on 2007-08-09
if you are running ubuntu 7.04, you can turn on desktop effectes through **System > Preferences > Desktop Effects**.  This gives you little to choose from but is still a nice effect and gives you the "cube" effect.  If you really want to modify your effects, I would suggest going to <www.beryl-project.org>  It is tough to install if you are a newb but I did it and I haven't been with linux that long.  Beryl is a window manager that has some really amazing desktop effects that puts Windows Vista to shame - and less resource intensive!
