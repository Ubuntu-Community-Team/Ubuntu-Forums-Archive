### Post by tim heeter on 2007-10-23
I actually had this problem as well. What worked for me was not directly using the confi-manager. Go to the Appearance manager, and then select the visuals tab. You should have 4 options there. The bottom one being Custom. Select that as your default, then edit your options from there. After making changes it should save it and use that setting instead.   Hope that works for you.
