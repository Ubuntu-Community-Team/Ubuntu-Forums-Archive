### Post by spec-chum on 2008-03-03
That's the Scale plugin.  It shows you all of your active applications and lets you select which one you want.  I use it all the time, but if you want to disable it go to System -> Preferences -> Advanced Desktop Effects Settings (you have installed CompizConfig Settings Manager. yes?)

In the Window Management section clear the tick from Scale and it'll stop it activating.
