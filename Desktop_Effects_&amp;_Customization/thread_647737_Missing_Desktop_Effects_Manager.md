### Post by ijason on 2007-12-22
as far as i can remember, the only default options for the effects are "none, some, all", in different wording. if you want to have actual control over which specific effects are being used - and how - you'll need to download a few more compiz bits via synaptic.

do a search for Compiz. i think i installed the "compizconfig-settings-manager" and "compiz-fusion-plugins-extra"... that is to say, they are both installed on my system now, and i think those are the two i added. once synaptic installs those you should see Advanced Desktop Settings in the >system>preferences
