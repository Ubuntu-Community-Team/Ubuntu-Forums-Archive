### Post by Pablo Pablovski on 2007-10-05
You could make your own using the theme editor built in to Emerald Theme Manager. Select the theme you want to edit ("Edit Themes"). You can change colours, frame type, opacity, even the icons that appear in the titlebar. 

Save it as something else and you're good to go.
