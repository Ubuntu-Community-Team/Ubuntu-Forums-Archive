### Post by David Valentine on 2008-02-29
IIRC, KDE is developing its own compositing effects, and does not work as well with compiz as Gnome does (though you can install and use compiz on KDE).  Thus there is not really the equivalent in Kubuntu to the simple "enable desktop effects" command in Ubuntu.  However, you can get some few effects via the system control panel (I forget what its name is now).
