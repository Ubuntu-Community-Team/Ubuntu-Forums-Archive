This is wrong. You don't need to extract the tar.gz (or tag.bz) files, neither you need to copy them to your *.themes* folder!

You just download the theme file you want to install somewhere, for example on your desktop. Then you go to **System -> Preferences -> Theme**, where you have to select **Customize**.

When the new window opens you need to select the appropriate tab, depending of what type of theme is, a metacity theme, a gtk or an icon one. Afterwards, you press the **Install** button and you select the theme file you have downloaded. And that's it.

The rest is done automatically for you.
