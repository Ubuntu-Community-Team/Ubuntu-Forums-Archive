### Post by hyperair on 2007-09-12
Well I'm using nVidia Geforce 4 MX 440 64MB, and using Compiz Fusion, it doesn't seem the least bit sluggish.

I personally think that there are no specific minimum requirements to run Compiz Fusion. Each plugin has its own requirements, and Compiz Fusion itself requires 3d supprot and AIGLX/XGL/Nvdiia, and that's about it I guess.

Disabling or enabling certain plugins, tweaking certain settings will bring about a change in your Compiz Fusion experience. For example, the blur plugin. I've never used it since switching to Compiz Fusion simply because my graphic card lacks Pixel Shaders. When I used it in Beryl, it was sluggish. When I disabled it in Beryl, it stopped being sluggish. Then there are some complex animation effects which will turn out sluggish when used on a slow GPU, like Beam Up.
