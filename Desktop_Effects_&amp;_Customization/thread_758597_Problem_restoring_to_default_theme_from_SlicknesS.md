### Post by itix on 2008-04-18
I will assume that you are talking about the slickness applivation theme. Go to *system => preferences => apearence* (in gutsy and hardy, don't remember what it was on fiesty). 
There you select the custom theme and click customize. Go to the different tabs and select non-slickness theme options and it will most probably work out ok. If not, try installing a different theme from gnomelook or gnome art- Preferrably one you think you might like :p
