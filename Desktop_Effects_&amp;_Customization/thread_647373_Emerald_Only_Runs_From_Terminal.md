### Post by odiseo77 on 2007-12-22
You probably must add an ampersand (***&***) at the end of the command you're using to start emerald; then execute it, press enter twice, write 'exit' and emerald should be dettached from the terminal. Also, if you're using gnome, you should check 'System>Preferences>Sessions' and add the command(s) you use to start emerald to the startup programs (so emerald/compiz start every time you start gnome without the need to enter the command manually).
