### Post by FuturePilot on 2008-01-29
1. System>Preferences>Windows. Change the Title Bar action to Roll Up. Even if you're using Compiz these settings still apply.

2. Install Compiz Config if you haven't already
```
sudo apt-get install compizconfig-settings-manager
```
Now go System>Preferences>Advanced Desktop Effects Settings and go to the Cube Caps plugin.
