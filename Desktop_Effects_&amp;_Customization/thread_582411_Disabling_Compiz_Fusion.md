I'm running Ubuntu 7.10 with compiz fusion enabled by default. I installed cedega so I can play some games and find the framerates pretty apalling so I was wondering if there was a way of temporarily disabling compiz fusion or running cedega from a more lightweight window manager.

### Post by carlinuxlearner on 2007-10-19
Just right click on your desktop and select "Change Desktop Background" and then click the "Visual Effects" tab and select the "None" option, this will disable compiz-fusion.

C@RL

### Post by FuturePilot on 2007-10-19
Even quicker
Press Alt+F2 and type in
```
metacity --replace
```
Then to turn Compiz back on press Alt+F2 again and enter
```
compiz --replace
```
