### Post by Tomelo on 2008-05-20
I had a similar issue with Beryl on Feisty: 3D windows would brake or not fold properly around cube edges.
I deactivated and reactivated the 3D Effects plugin...that did the job.
I believe it might depend in which order you activate certain plugins and how some plugins interfer with each other.
In my case 3D effects was activated before Cube Desktop which seemed to cause this kind of bug...maybe its the same problem with Fuzion.
