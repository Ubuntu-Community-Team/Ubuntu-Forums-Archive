### Post by msjones on 2008-02-12
This maybe a pain for you but do a fresh install of 7.10.

Dont run any updates or enable the restircted NVIDIA driver, just download and run the automatic envy script.

When complete let it edit your xorg.conf and then reboot your machine. After all is loaded up you should be able to use desktop effects.

When you run the ubuntu update and reboot your desktop effects will be redisabled, so run the envy script again, dont uninstall just reinstall over the top.

Let me know how you get on.
