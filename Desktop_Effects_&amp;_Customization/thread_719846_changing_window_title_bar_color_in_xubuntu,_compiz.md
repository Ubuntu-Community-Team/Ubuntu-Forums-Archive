### Post by xjgnsdof on 2008-03-09
I made the switch from Ubuntu to Xubuntu yesterday and have since enjoyed the streamlined performance and freed up disk space. However, all of my window title bars are red, and I don't know how to change the color. My window manager is xfwm4, and I have compiz enabled.

---

### Post by smartboyathome on 2008-03-09
If you are using compiz, then it isn't using XFWM. Try installing emerald and running emerald --replace and then it will give you emerald (I am guessing right now you are running gtk-window-decorator). Download themes for it and install them by importing them into System > Preferences > Emerald Theme Manager.

---

### Post by xjgnsdof on 2008-03-09
I actually figured it out. I input "Emerald" in the "command" line under the General Options in Compiz. I then downloaded some themes and installed them using Emerald Theme Manager. It was quite simple; I just never did this before on my Ubuntu machine. Thanks for the help, though.
