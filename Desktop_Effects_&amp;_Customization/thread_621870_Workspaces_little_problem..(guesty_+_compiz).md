### Post by Forlong on 2007-11-24
> **chacham23 said:**
> another question is how I add workspaces on compiz?
Just right-click on the workspace switcher in the panel.

As for the other problem, you can try switching to the flat-file backend in ccsm:
```
sudo apt-get install compizconfig-settings-manager
```

But then you have to change the numbers of workspaces in General Options there.
