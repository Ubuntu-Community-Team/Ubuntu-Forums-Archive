### Post by old_geekster on 2007-03-17
If you are using Edgy, you can "Ctrl + Alt Backspace".  This will reboot your system and allow you to make some choices.  At the login screen, there is an "Option" area in the lower left corner.  Click on it and make a choice.

You probably would want to change session to "Terminal failsafe".  This would allow you to reinstall Bery from the terminal.  If asked if you want to make this permanent, choose "This time only" or something like that.
