### Post by Forlong on 2008-01-08
Try "resetting" your xorg.conf
```
sudo dpkg-reconfigure xserver-xorg
```
Afterwards do this:
```
sudo nvidia-xconfig --add-argb-glx-visuals -d 24
```
Then restart X and see if it worked.
