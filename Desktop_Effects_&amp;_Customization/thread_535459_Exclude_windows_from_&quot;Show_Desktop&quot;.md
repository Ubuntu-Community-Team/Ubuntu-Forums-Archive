### Post by yuri_rage on 2007-08-26
**SOLVED!** - under some conditions, anyway...

Using some of the info [in this thread]("http://ubuntuforums.org/showthread.php?p=3249193"), I managed to get the effect I want.

I should've mentioned that I'm using compiz fusion.  It just so happens that the windows I want to keep are windows that fall into the "skip taskbar" category.  Disabling the "Hide skip taskbar windows" setting in the General tab of compizconfig does exactly what I was asking about.
