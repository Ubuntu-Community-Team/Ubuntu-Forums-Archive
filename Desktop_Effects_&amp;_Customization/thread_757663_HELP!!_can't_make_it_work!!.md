### Post by daengbo on 2008-04-17
Ubuntu 7.10 has Compiz installed by default. If it's not working, that means that your video card is not set up to do 3D.

System > Administration > Restricted Drivers Manager > enable the ATI driver.

If that doesn't work, look here:
[http://ubuntuforums.org/showthread.php?t=362525](http://ubuntuforums.org/showthread.php?t=362525)
