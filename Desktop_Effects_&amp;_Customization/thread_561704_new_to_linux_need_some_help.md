### Post by FuturePilot on 2007-09-27
To change the background just right click on the desktop and go to the very last option. I can't remember what it's called in Gutsy. It was changed from Change Desktop Background to something else.
Since you're running Gutsy, you're halfway there for Compiz Fusion. Open Synaptic Manager (System>Administration>Synaptic Package Manager) and search to Compiz Fusion. Install the Compiz Fusion plugins and stuff. You might want Compizconfig settings manager too. Also in Synaptic. Just make sure you also install the graphics driver from the Restricted Driver Manager. System>Administration>Restricted Driver Manager.
