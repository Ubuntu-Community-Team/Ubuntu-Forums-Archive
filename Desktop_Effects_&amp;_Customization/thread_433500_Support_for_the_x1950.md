### Post by [ajax] on 2007-05-05
It does work (I've got one).  Heres how.

Run "glxinfo".  Look for direct rendering.

If it returns "no", install the drivers with Envy, then proceed to next step:
[http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

 If it says "yes", install beryl with this guide (included desktop effects don't work with our cards): [http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL](http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL)
