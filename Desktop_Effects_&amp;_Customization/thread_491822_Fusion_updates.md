### Post by andrewsomething on 2007-07-04
If you don't want to deal with constant upgrades and new possible bugs, my suggestion is to ignore the upgrades if what you're using at the moment seems stable for you. You could even comment out the repo in your source list so the system doesn't bug you with updates. Then when you're ready to check out the newest upgrade just uncomment and update apt.
