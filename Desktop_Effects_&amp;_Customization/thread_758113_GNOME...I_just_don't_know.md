### Post by tvtech on 2008-04-17
both [KDE]("http://www.kde.org/") and [Gnome]("http://www.gnome.org/") are much like windows style windows managers.  however, gnome and kde can be configured with themes to look much like mac aqua  check out [Gos]("http://dev.thinkgos.com")  for examples of how gnome can look like a mac operating desktop.

there is also [fluxbox]("http://fluxbox.sourceforge.net/")
which is a different windows manager.  

and [xfce]("http://www.xfce.org/")

please note you can pretty much alter any of these windows managers that run on top of X  to do whatever and look like whatever you want, you can add themes take away themes and install things like [compiz-fusion]("http://www.compiz-fusion.org/")  over them is possible as well.  It really depends on what types of programs you like to use and how integrated you want them to be with your display manager. 

I personally run gnome, it's a little more complete and does a lot of windows management things that are easy to get a hang of and use.  But I run a lot of KDE apps under gnome with differing levels of success.  I've tried fluxbox which is the most different of all of them and went running for the hills.  pretty much all of them except fluxbox are similar in use as default to windows.  mac uses a fundamentally unfriendly point of view.  i.e. they don't want you getting into the system and changing things easily like windows allows you to do and like linux is designed to do.

XFCE is designed to be like gnome but much lighter on system req's  
most people that require really light sys req's but want a windows manager instead of a command line go to [fluxbox]("http://fluxbox.sourceforge.net/") or[ blackbox]("http://blackboxwm.sourceforge.net/"). 

I've put dynamic links under all the different windows managers so you can go to their websites and check them out in detail.  It took me quite awhile to really get ahold of a window manager I liked and did what I needed it too.
