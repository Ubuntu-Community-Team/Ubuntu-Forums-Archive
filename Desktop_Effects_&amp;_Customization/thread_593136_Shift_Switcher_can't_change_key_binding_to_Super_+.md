### Post by techstop on 2007-10-26
Well, I am running Gutsy and the built-in CF. I have enabled the Shift Switcher plugin, and disabled the  Ring Switcher plugin. When I try to change the key binding of Shift Switcher (Initiate) to Super + Tab, it will not accept it. It is currently set to Shift + Super + S which is very inconvenient. I guess there is another binding somewhere using Super + Tab, but I cannot find it. I think I have been through all the plugins.

Any tips on how to fix this? Cheers...

### Post by prshah on 2007-10-27
Yes, this didnt work for me either, but I got around it by doubleclicking the right side of the line where it asks for the keys to use (Near the system bell checkbox); this opens a fresh window into which you can just type in "<Super>Tab" (No quotes).

If there is a conflicting binding, Ubuntu will warn you and ask you if you want to clear the conflicting binding. But in this case, all I got was a system beep everytime I tried to set <Super>Tab.

Once set by the above method, I had no problems with it.

Pls let me know if you need more info.
