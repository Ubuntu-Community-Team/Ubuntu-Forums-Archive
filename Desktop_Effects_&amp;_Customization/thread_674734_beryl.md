### Post by FuturePilot on 2008-01-22
Beryl is a composite window manager that uses compositing and OpenGL to give 3D special effects such as drop shadows, wobbly windows, transparency, 3D cube and much more. But Beryl is no longer developed because it got merged back into Compiz (another composite window manager) to become Compiz Fusion. Ubuntu 7.10 has Compiz Fusion already installed. You may just need to install a 3D driver from the Restricted Driver Manager. System>Administration>Restricted Driver Manager.

### Post by PeterJS on 2008-01-22
Well that depends. What version of Ubuntu are you running? If it's 7.10 you don't want to install beryl but use the newer compiz fusion. To configure it to it's fullest first install [Compiz Config Settings Manager](apt:/compizconfig-settings-manager), and then check out the myriad of settings in System > Preferences > Advanced Desktop Settings.

As for what Beryl/Compiz is, it's lots of things all rolled in to one: it's a window manager, virtual desktop manager, and composite manager. The window manager portion controls the window borders and title bars. The virtual desktop manager handles how the virtual desktops are laid out, on the surface of a cube, one big plane, and which windows are on which desktop. The composite manager portion controls transparency effects and other eye candy.
