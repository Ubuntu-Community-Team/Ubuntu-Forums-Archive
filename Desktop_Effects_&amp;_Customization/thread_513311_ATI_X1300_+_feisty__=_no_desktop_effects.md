### Post by Espreon on 2007-07-30
You hafta use an XGL session with fglrx inorder to use Beryl/Compiz/Compiz-Fusion. As for you Ryoushi, the opensource ATI driver does not work with the x1000 series. I would love to be able to use the opensource driver or fglrx to support AIGLX.

So here is how to get XGL+Beryl/Compiz/Compiz-Fusion:

1st do this in a terminal:

```
sudo apt-get install xserver-xgl
```

Then do this in a terminal:

```
gksudo gedit /usr/bin/startxgl.sh
```

then fill the blank file with this:

```

#!/bin/sh
Xgl -fullscreen :1 -ac -br -accel glx:pbuffer -accel xv:pbuffer &
sleep 4  
export DISPLAY=:1 
cookie="$(xauth -i nextract - :0 | cut -d ' ' -f 9)"
xauth -i add :1 . "$cookie"
exec gnome-session

```

Save and do this:

```
sudo chmod a+x /usr/local/bin/startxgl.sh
```

then do this:

```

sudo mkdir -p /etc/X11/sessions
sudo gedit /etc/X11/sessions/xgl.desktop

```

fill the blank file with this:

```

[Desktop Entry]
Encoding=UTF-8
Name=XGL+GNOME
Exec=/usr/local/bin/startxgl.sh
Icon=
Type=Application

```

Then save.

If you want an eyecandyish way of shutting down,locking your comp and rebooting it (this app is composite aware). Do this:

```
gksu gksu --desktop /usr/share/applications/software-properties.desktop /usr/bin/software-properties-gtk
```

The hit the 3rd party software tab and put these in one at a time:

```
deb http://download.tuxfamily.org/syzygy42/ feisty all
```
```
deb-src http://download.tuxfamily.org/syzygy42/ feisty all
```

Then quit and reload.

The enter ```
sudo apt-get install closure-svn
```
After it is done installing:
Then log out. Hit sessions and select XGL+GNOME and login as usual, make it the default session if you want. If done correclty the screen should turn black (if it is animated then XGL is screwed up) or look like steel wool.

Enjoy. 

If GTK looks like crap download my GTK/XGL Anticrapifying Script:
Place it in your home folder and goto: System>Administration>Sessions hit new call the entry whatever you want for the command drag and drop the file be sure to remove: file/// leaving only one / at the beginning hit ok and Enjoy.
