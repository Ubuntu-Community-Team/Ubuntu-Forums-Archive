### Post by Criminosis on 2007-08-15
> **tripp-fan said:**
> I have finally gotten compiz and emerald themer working on my laptop. My questions are 1.) can you keep compiz running because everytime i restart i have to go to the alt+F2 and enter compiz --replace to start it up. Can it stay on so i do not have to do this at every restart? 2.) What are these bars i see on the videos of compiz that look alot like the mac bar? how do you get them?

Go to System ==> Preference ==> Sessions

Click on new and add a name (like Compiz)

then in the command window put in that command (Compiz --replace)

"Sessions" is the GUI manager for your boot up programs/services (Kinda like that startup tab in msconfig of Windows)

EDIT : If you are using Emerald themes, modify the command to ```
 compiz --replace -c emerald --replace 
```
