### Post by FuturePilot on 2008-03-28
I would go with the Nvidia card. Yes ATI has committed to developing an open source driver for their cards, but it could be a very long time before you see a decent driver released. And currently the ATI situation on Linux is still horrible. If you want to save time and a lot of frustration, go with the Nvidia card.
