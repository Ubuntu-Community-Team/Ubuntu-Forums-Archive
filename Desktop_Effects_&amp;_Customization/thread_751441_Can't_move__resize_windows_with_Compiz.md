### Post by chewearn on 2008-04-11
There is no need to install beryl in gutsy.  By default, gusty is already installed with compiz.  Once you enabled 3D graphics support, compiz will automatically be enabled.

Beryl is obsolete and should not be used anymore.

if compiz is not automatically enabled, goto:
Panel Menu > System > Preferences > Appearances > Advanced effects
and select "normal", or "extra".
