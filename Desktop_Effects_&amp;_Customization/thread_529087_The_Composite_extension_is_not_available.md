@All
Desktop effects is an experimental feature.


@all with ATi xPRESS 200 series cards

Those with fglrx drivers should install Xgl to get desktop effects.Preferably use Compiz-Fusion and Beryl because they run fine on xPress 200 series cards with Xgl(Personal experience 8):-D)Also,they provide better effects :P

To install Xgl:```


sudo apt-get install xserver-xgl
```

Look through the procedure on [www.ubuntuguide.org](www.ubuntuguide.org) for the entire procedure to get Xgl and Beryl working...LOOK ONLY under the fglrx section.The open-source drivers of ATi xPress 200 series cards don't support 3D desktop effects.

Those without fglrx drivers should install them and Xgl IF they want desktop effects.

### Post by rh1zome on 2007-10-04
For the record, the fixes listed by others here worked for me with Gutsy Gibbon Beta running on an ATI X1800 GTO, Asus A8R32MVP, AMD X2 4400. 

This is what I did:

From command line make xorg.conf backup:
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak

Open xorg.conf in text editor:
sudo nano -w /etc/X11/xorg.conf

Change:
Option "Composite" "0"
to:
Option "Composite" "1"

Save changes and exit editor.

From command line:
sudo apt-get install xserver-xgl

Restart X-server (Ctrl+Alt+Backspace -- I assume this works although I actually rebooted)

Go to System > Preferences > Appearance > Visual Effects and enable some of the most shameless eye candy yet seen ;)

### Post by MrChonks on 2007-10-18
> **rh1zome said:**
> For the record, the fixes listed by others here worked for me with Gutsy Gibbon Beta running on an ATI X1800 GTO, Asus A8R32MVP, AMD X2 4400. 

This is what I did:

From command line make xorg.conf backup:
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak

Open xorg.conf in text editor:
sudo nano -w /etc/X11/xorg.conf

Change:
Option "Composite" "0"
to:
Option "Composite" "1"

Save changes and exit editor.

From command line:
sudo apt-get install xserver-xgl

Restart X-server (Ctrl+Alt+Backspace -- I assume this works although I actually rebooted)

Go to System > Preferences > Appearance > Visual Effects and enable some of the most shameless eye candy yet seen ;)

This worked brilliantly.  I would only add that the universal repository will need to be added/uncommented in your sources list in order to pull xserver-xgl.

sudo apt-get install xserver-xgl
