I have Ubuntu nicely customized, but here and there the hideous default colors show up. In this case they show up in the menus on the gnome panel. ---

### Post by heimo on 2007-05-17
System->Preferences->Theme->Customize->Colors->Selected items (You need  to have selected a theme that supports customizing colors)

### Post by vtel57 on 2007-05-17
Change/modify your theme...

Gnome Menu --> Desktop --> Preferences --> Themes.

You can download and install many different theme colors , theme icons, and other theme variables from [Gnome-Look.org](http://www.gnome-look.org/).
