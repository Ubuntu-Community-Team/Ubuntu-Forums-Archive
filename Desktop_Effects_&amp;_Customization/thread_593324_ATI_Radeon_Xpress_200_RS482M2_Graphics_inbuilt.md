### Post by maddentim on 2007-10-27
I have a ATI 200M on my toshiba laptop and upgraded to 7.10 from 7.04.  It worked flawlessly.  Once I enabled the driver, it turned on the eyecandy without asking (as well as the Xgl.  The base desktop effects thing has just three settings, but if you install the manager you can pick and chose the plugins that you want to use.  Do this after you do have it set up:

> sudo aptitude install compizconfig-settings-manager
