### Post by Tachyon1000 on 2007-10-21
Here's my recommendation. Once you have a theme selected (this is in gutsy I assume), go to System>Preferences>Advanced Desktop Effects Settings.  Go to the Windows Decorator plugin settings.  Enter emerald in the command text box, turn off desktop effects, then turn them back on.  Now emerald should be running with your selected theme and will start up with that theme until you change it or remove emerald from the command text box.
