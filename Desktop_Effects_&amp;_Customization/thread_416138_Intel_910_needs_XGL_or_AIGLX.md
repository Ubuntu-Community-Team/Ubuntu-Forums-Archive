### Post by jiminycricket on 2007-04-20
Intel graphics use AIGLX, just like the open source ATI drivers.  In Feisty, no configuration is needed.

ATI's proprietary driver only supports XGL.

Only nVidia cards with nVidia's binary driver don't need either.
