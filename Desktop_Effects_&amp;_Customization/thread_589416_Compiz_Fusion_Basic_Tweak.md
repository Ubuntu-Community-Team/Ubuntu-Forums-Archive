### Post by Shimmy on 2007-10-24
I know this as been posted before, but i just wan't to make sure that everybody gives it a try, since it really boosts the performance for Compiz.

1. Go to CCSM (System->Preferences->Advanced Desktop Effects Settings).
2. Go to General Options->Display Settings
3. Uncheck "Detect Refresh Rate"
4. Uncheck Sync To VBlank
5. Experiment with the Refresh Rate, i found that 100 is a great value for me.
