### Post by loserboy on 2007-06-25
first thing to do is go to gnome-look.org and pick out themes

System>Preferences>Theme   -    this is where you go to change icons, windows and borders
System>administration>login window  -  this is where you change GDM

most of the packages at gnome-look are setup so you can just drag the package to the theme window and it installs properly

stuff you're looking for is gonna be under Metacity for window borders, GTK 2.x for the widget things and the rest of the windows, and obviously icon themes for icon themes
