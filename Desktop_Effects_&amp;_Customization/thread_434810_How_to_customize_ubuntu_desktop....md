### Post by glaeven on 2007-05-06
well, gaspar has given the right sites, but you need to know how to intall themes.

basics for gnome-look.org: GTK themes = controls (buttons, scrollbars, tabs, etc)
           Metacity themes = window borders

for art.gnome.org: application themes = GTK = control
            window border = metacity = window borders


to install: download the archive file to the desktop. System>Preferences>Theme. click customize, then click install. navigate to the archive, and install. go to the proper tab and select th new theme. have fun!

### Post by Sethalos on 2007-05-08
I'm a total Ubuntu or for that matter Linux noob. However, once I installed it the first thing i wanted to do was to set up Beryl, and experiment with themes.

So, here's what I installed:

GTX2
Emerald Theme Manager
Conky
gDesklets

...and a couple more that escape me.

Then i navigated over to gnome-look.org, picked up a few themes (I personally just placed them in my home folder) and installed them. From there I started to experiment with desklets, re-arranging my toolbars, creating a new launch bar, and pretty much messing with everything I could.

Now I have a 3-d cube that is on fire, desktop effects, opaque toolbars, and a red/black theme that looks really wild. 

Good luck and don't be scared to experiment, hehe.
