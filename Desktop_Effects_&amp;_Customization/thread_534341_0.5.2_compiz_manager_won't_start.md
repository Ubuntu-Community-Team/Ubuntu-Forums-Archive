### Post by Forlong on 2007-09-22
> **Jheric said:**
> Did it screw me up?
Uhm... well... what can I say?

OK, let's try this the (very) hard way...

First remove everything again (including downloaded packages):
```
sudo apt-get remove --purge compiz* emerald* && sudo apt-get autoremove && sudo apt-get clean
```
Then you have to update the database
```
sudo updatedb
```
(that will take a while)

After that, search for anything related to Compiz
```
locate compiz
```
And remove every folder that you found via
```
sudo rm -r /path/compiz
```

Same for
```
locate compizconfig
```
and
```
locate emerald
```
and just to make sure
```
locate ccsm
```
as well.

Then start over.
