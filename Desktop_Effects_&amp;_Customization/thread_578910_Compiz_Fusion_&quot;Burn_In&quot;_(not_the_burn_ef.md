### Post by Morton's Red Stapler on 2007-10-21
that may be a problem with compiz itself, another possibility is your xorg file. i have had a similar problem in windowsXP using windowblinds. if you do not already have compiz settings manager, install it using this command
sudo apt-get install compizconfig-settings-manager
and then check all your settings in there. if that doesnt help, remove and then reinstall compiz, you can find how-to's all over the forums, just do a forums serch for "compiz fusion" and you should be able to find one.
also, what kind of system are you running on? if your vid card and processor aren't up to the task, they will have hiccups with process intensive things like compiz, so make sure your system can handle it.
hope this helps you out.
