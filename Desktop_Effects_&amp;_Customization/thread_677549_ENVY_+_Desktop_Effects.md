### Post by Lvcoyote on 2008-01-25
Did you uninstall first??

As per the envy page:
```
WARNING: you will have to remove the driver you installed with Envy before upgrading Debian or Ubuntu to a newer release (e.g. upgrading Ubuntu Edgy to Ubuntu Feisty or Debian Etch to Debian Lenny).

You will have to type the following command:

sudo envy --uninstall-all 
```

### Post by AsoSako on 2008-01-25
I had the same problem too...

What I did is to:
1.) Go the Restricted drivers and enable the one there.
2.) Restart.
3.) Enable Extra Effects.
4.) Install the Envy Drivers again and Restart.

This should fix it.
Enjoy :D
