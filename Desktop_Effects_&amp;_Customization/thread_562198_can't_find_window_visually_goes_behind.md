The desktop effects have a feature that makes a window visually go behind the other one that receives the focus. I liked this feature best of all but after some update this feature is gone by default.

### Post by patchoro on 2007-09-28
go to CompizConfig Settings Manager (or ccsm in terminal)

animations --> focus animation --> edit --> select dodge.

Done!!

Enjoy. :)
