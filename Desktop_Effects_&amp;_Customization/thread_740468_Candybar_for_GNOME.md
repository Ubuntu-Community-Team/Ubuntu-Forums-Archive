### Post by Victormd on 2008-03-30
Simply go to System>Preferences>Appearance.

This will allow you to install Icon packages, window border themes (other than Emerald themes), among other things.  Just click install and select the icon package (tar.bz).
If you want to select a different icon theme, simply go to Appearance again and select CUSTOMIZE.  This will allow you to chose a different Icon package, color scheme, window border, control scheme, and pointer themes...
