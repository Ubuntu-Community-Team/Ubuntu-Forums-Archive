### Post by rhystic on 2007-08-26
Add  "1280x1024" to the "Display" subsections in your xorg.conf file. Restart X and you should be good to go.

```
 
SubSection "Display"
                Depth   24
                Modes           **"1280x1024" **    "1024x768"      "800x600"       "640x480"
EndSubSection


```

Cheers!

### Post by Hyperkill on 2007-08-29
I had the exact same problem but only with my nvidia card.  I modified my xorg config so many times it's not even funny.  The thing that fixed it for me was modifying some xml file (which I can't think of right now).  It was set to 1024x768 and then I changed that value to 1280x1024 and rebooted then it worked great.  Maybe one of these more experienced guys know what file I'm talking about.  I'm searching my computer right now for the file and I'll post if I find it.

Edit***

I found the file it's located at /.gconf/desktop/gnome/screen/default/0/

It's called %gconf.xml

Open that up with a text editor and change 1024x768 to 1280x1024 (no spaces)

Save the file and then reboot or try Ctrl + Alt + Backspace

Hope it works
