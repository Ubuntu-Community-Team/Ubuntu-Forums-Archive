### Post by rune0077 on 2008-02-12
Try this. Open Nvidia settings and go to "X server display configuration". Select the 1400x1050 screen and make sure that "Position" is set to Absolute +0+0. Then click the 1280x1024 screen, and set it's position to Absolute +1400+0.
