### Post by dentaku65 on 2007-12-02
Hi all,

I would like to start a thread where is possible to gain as much as possible information and configuration on various graphic cards that running Compiz-Fusion, in order to have a place where to check quickly a working scenario for installed graphic card.
[B]
*PLEASE, POST ONLY INFO ON WORKING GRAPHIC CARDS UNDER GUTSY WITH COMPIZ-FUSION*[/B]

The useful commands for output posts:

A) Kernel running
```
uname -r
```

B) Graphic card model
```
lspci -v |grep Graphic
```
or
```
lspci -v |grep Display
```
or
```
lspci -v |grep VGA
```


C) Graphic driver under xserver
```
more /etc/X11/xorg.conf |grep Driver
```

D) The entire xorg.conf
Edit
```
/etc/X11/xorg.conf
```
with your favourite editor, select all the content and paste here
or
```
cat /etc/X11/xorg.conf
```
