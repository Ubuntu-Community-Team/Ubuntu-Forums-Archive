### Post by yabbadabbadont on 2007-06-09
Most likely, they have been stored in your user's .theme directory.  Note that the name starts with a period which makes it a hidden directory.  That is, /home/username/.theme

 I haven't used Gnome for a while, so it could be that it is called .themes.  I can't remember for sure.  Either way, there should be a sub-directory in that directory for each theme you have installed.  Just delete the sub-directory and its contents and then the theme name should be removed from the menu.
