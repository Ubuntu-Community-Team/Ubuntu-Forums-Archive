### Post by jryprt on 2007-12-31
System>Preferences>Advanced Desktop Effects Settings>Animations.
Close Animation tab > Animation selection double click on 1st line a edit box comes up  in the Close effect choose burn in the drop down box, set Duration a 400 or so click ok.
Now the Minimize Animation tab double click on 1st line a edit box comes up  in the Minimize effect choose burn in the drop down box,set Duration a 300 or so click ok.
Now Open Animation tab double click on 1st line a edit box comes up  in the Open effect choose burn in the drop down box,set Duration a 300 or so click ok.
