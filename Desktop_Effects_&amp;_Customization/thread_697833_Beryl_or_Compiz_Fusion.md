### Post by rainwalker on 2008-02-16
Beryl and Compiz have merged to form Compiz Fusion. On the latest version of Ubuntu (7.10 "Gutsy") it's installed already, you can enable it under System > Preferences > Appearance under the "Visual Effects" tab
To customize the effects, install the package "compizconfig-settings-manager" using Synaptic Package Manager under System > Administration
