### Post by aktiondan on 2007-04-23
I had the same problem.  I fixed it by adding this line of code to my etc/X11/xorg.conf file under Section "Device"

```
Option "AddARGBGLXVisuals" "True"
```

So that the whole section looks like this:

```
Section "Device"
Identifier "NVIDIA Corporation NV18 [GeForce4 MX 440 AGP 8x]"
Driver "nvidia"
Option "AddARGBGLXVisuals" "True"
EndSection
```

The borders showed up after logging back in and enabling effects.  There's a few other sites that mention this fix.  I have no idea why it makes it work, but it did for my system.  The wobble effect is super cool.

### Post by Compyx on 2007-04-25
The file you're looking for is /etc/X11/xorg.conf.

To add the ARGBGLXVisuals option and to set the default colour depth to 24, run (in a terminal):
```

sudo nvidia-xconfig --add-argb-glx-visuals --depth=24

```

Then restart the X server: logout and login or use the brute force method: Ctrl+Alt+Backspace.

