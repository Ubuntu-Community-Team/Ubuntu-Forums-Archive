### Post by TadH on 2007-11-11
first of all go to a terminal and type in
sudo apt-get install ccsm (this is the compiz fusion settings manager)

secondly go to system preferences appearances and put visual effects on custom

then u will have an option under system preferences called advanced settings manager go to it and tick off desktop cube and rotate cube

if u want more help just let me know
