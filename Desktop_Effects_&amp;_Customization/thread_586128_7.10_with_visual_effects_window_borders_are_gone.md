### Post by ezak on 2007-10-22
Try running this is a terminal:

compiz --replace --indirect-rendering

- if that doesn't do anything and you have a nvidia gfx card the run:

compiz --replace

Then install compizconfig-settings-manager by running:

sudo apt-get install compizconfig-settings-manager

### Post by VictorR on 2007-10-22
Compiz-fusion now has Emerald in Window Decoration plug-in, so if you start emerald like

compiz --replace -c emerald

it doesn't work anymore.

Check if Window Decoration is ON and its command field contains
emerald --replace
