### Post by ToxicBomber on 2007-10-21
Pulled from the help files in Ubuntu:

> As well as the desktop effects which are available by default, a large collection of additional effects is available. These range from effects intended to make it easier to use your computer, to ones which are just for fun.

To enable additional effects:

   1.    Install the compizconfig-settings-manager package ( under System &#9656; Administration &#9656; Synaptic Package Manager, then scroll down till you find it. check it off and hit Apply )
      			
   2.    Press System &#9656; Preferences &#9656; CompizConfig Settings Manager
			
   3.    Check the box next to an effect to enable it
 			
   4.    Click on an effect to change its settings

Hope that helps a bit. :D
