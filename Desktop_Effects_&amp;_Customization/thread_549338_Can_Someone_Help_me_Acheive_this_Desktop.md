### Post by urukrama on 2007-09-12
And the task list at the bottom left is Tint Task Manager.

To achieve this, you'll have to install Openbox (use the link RedSquirrel gave you, and download the .deb package for Feisty. Double click on it, press install, give your password). Log out, and in GDM (the login screen) go to sessions (press F10) and select 'Openbox' if you want to run pure openbox, or 'Gnome/Openbox' if you want to run openbox in gnome.

Have a look at [this]("http://ubuntuforums.org/showthread.php?t=192106&highlight=openbox+howto+options") thread to get you started, but ignore the autostart stuff if you install Openbox from the openbox site. The latest Openbox version is a huge improvement over the one in Synaptic so it is worth it.
