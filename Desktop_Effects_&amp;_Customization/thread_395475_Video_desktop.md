### Post by mcduck on 2007-03-28
Sure, when running Beryl or Compiz you can use xwinwrap to play movies or screensavers as your desktop background.


Following code would use mplayer to play movie as desktop backgound:
```
xwinwrap -ni -fs -s -st -sp -b -nf -- mplayer -wid WID -quiet movie.mpg
```
