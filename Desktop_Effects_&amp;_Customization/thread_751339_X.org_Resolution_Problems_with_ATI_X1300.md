i didnt use a doc, i just stepped through my xorg.conf files as they were generated. sometimes changing them with aticonfig would remove a line, or using the "screen and graphics" utility default in ubuntu would remove lines. Heres the order of how I made it, maybe it will help you too:

4. added that stuff back in from the envyng config (doing this gave me dual monitors! yay! but they were left: 1280x1024 and right: 1900x1200)
5. removed all the modes I didnt use (for whatever reason this forced my supposed 1900x1200 to left: 1900x1080 right: 1900x1080)
6. added virtual desktops set to 1900x1200 to each screen
