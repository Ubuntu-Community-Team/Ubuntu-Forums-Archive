### Post by sprwaka on 2007-07-30
I'm not really a super user but go to kde look and get the themes from them. Usually the extensions are .noarch.rpm or tar.gz  . So download them from the kde look web site the go to you theme manager and browse to it apply and vuala.

PS. make sure you get the theme for the right kde version 3.0, 3.1,3.2 - 3.5,4.0
