### Post by Hed69 on 2007-10-03
I got it.  I was having the same problem, but after a little tinkering I figured it out. Run the Compiz settings manager/Advanced Desktop Effects settings and click the "General Options" button.  Click the "Desktop Size" tab and change all the options listed to 4 and that should fix it.  I didn't check whether or not they all have to be set to 4, but when I only selected the "Number of desktops" and set it at 4 it still didn't work.  But...that'll fix it for ya'.
