### Post by old_geekster on 2007-03-17
The one thing that could be causing you a problem is this section:

Section "Screen"
    Identifier     "Default Screen"
    Device         "NVIDIA Corporation NVIDIA Default Card"
    Monitor        "Generic Monitor"
    DefaultDepth    **16**

I believe that the "DefaultDepth" should be set to **_"24" instead of "16"_**.  This in fact is 32-bit color.

Here is a link that should help:

[http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Multimedia_Player_.28xine-ui.29](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Multimedia_Player_.28xine-ui.29)

Scroll to section (18. Eye Candy) and read from section 18.3.

Also, you may want to install Beryl using "Automatix Bleeder".  Beryl is one of the few apps that can be installed from Bleeder, but it works better than any other method that I have tried.  You can install it from "Synaptic Package Manager" if you have the correct repositories installed.

As long as I am providing links, here is one to the best guide that I have found to install the latest nVidia drivers:

[http://www.ubuntuforums.org/showthread.php?t=336412&highlight=Hub](http://www.ubuntuforums.org/showthread.php?t=336412&highlight=Hub)

Follow his instructions to the "T" and you can't go wrong; especially the part where he recommends writing down the last two commands.  You will need them.
