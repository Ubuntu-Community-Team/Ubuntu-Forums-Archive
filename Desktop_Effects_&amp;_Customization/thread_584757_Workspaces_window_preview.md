### Post by Strofcon on 2008-02-14
I know this thread is several months old, but I figured I'd throw it out there for anyone else who might need it.

There are two ways to fix the problem you mention, both of which ultimately do the same thing. 

1) When in the Advanced Desktop Effects Settings manager, click "General Options," then click the tab titled "Desktop Size." Here, you want to increase the value in "Horizontal Virtual Size" and "Vertical Virtual Size" rather than "Number of Desktops." This gives you as many desktops as you want, and the switcher will show open windows on every workspace all the time.

2) Right-click your workspace switcher, then click "Preferences" in the context menu. Set the number of rows and columns you want for your workspace switcher. This does the exact same thing as option 1) above - in fact, if you go back into the Advanced Desktop Effects Settings manager, you'll see that the "Horizontal Virtual Size" and "Vertical Virtual Size" is the same as the number of columns and rows you just set. 

If you use the "Number of Desktops" option in the Advanced Desktop Effects Settings manager, you run into the problem of having a workspace switcher that only indicates the open windows on *this* workspace. The above steps will fix it. :)

Enjoy!
