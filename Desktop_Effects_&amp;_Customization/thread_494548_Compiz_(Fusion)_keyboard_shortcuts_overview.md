### Post by ramkumail on 2007-09-19
Hi below are the most used short cuts. I am a new user of compiz fusion. feel free to add new ones.

[B]SUPER+SHIFT+DRAG LEFT MOUSE = draw fire 
SUPER+SHIFT+C = clear fire 
CTRL+ALT+DRAG LEFT MOUSE = rotate cube 
CTRL+ALT+LEFT ARROW = rotate cube 
CTRL+ALT+DOWN ARROW = flat desktop 
SHIFT+ALT+UP=initiate window picker
CTRL+ALT+DOWN=unfold cube
ALT+TAB=window switch
SUPER+TAB=flip switcher or ring switcher depending on which is enabled.
ALT+F7=initiate 'move windows'
SHIFT+F9=water effect
SHIFT+F10=slow animations
CTRL+ALT+D=show desktop
SUPER+K=show mouse

**For Grouping and Tabbing:** 
SUPER+S=Select Single Window
SUPER+T=Tab Group
SUPER+Left=Change Left Tab
SUPER+Right=Change Right Tab
SUPER+G=Group Windows
SUPER+U=Ungroup Windows
SUPER+R=Remove Group Window
SUPER+C=Close Group
SUPER+X=Ignore Group
hold the super button then select the windows you want to group and then hit super+g[/B]

You can add new shortcuts by installing compizconfig settings manager(available in synaptic). After you have done that you can select one of the plugin(double click on it) and in actions tab of the plugin,  under bindings you can change the shortcuts. You can even add a screen edge as a binding so that when you move your mouse cursor to screen edge(say top left)  compiz will execute the particular action. It goes really nice with expo plugin. have a great time with compiz fusion.

### Post by ramkumail on 2007-11-11
A nice set up guide can be found at 
[http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion](http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion).

He explains how to get 4 desktops. A special thanks to Forlong.
 the pre requisite is that you should have "Compiz configuration settings manager" installed from synaptic. For every plugin you will have actions tab after you open it through System > Preferences >Adavnced desktop effects settings and click on plugin you want to deal with. In the actions tab you can specify your own key bindings.
 In the window you can as well enable certain plugins in case they are not enabled.

### Post by Alex_Fingers on 2008-02-07
I have an answer and a question regarding Compiz (fusion) keyboard shortcuts.

If you are finding that most of the keyboard shortcuts listed in this thread's previous page dont work, you need to install CompizConfig Settings Manager (installable through synaptic package manager.) There you can enable a lot of the features that are disabled by default and also see what features there are (such as the cube).

Here is a list of some of the features I use the most:

Ctrl + Alt + Left-Click + Drag = Rotate the Cube
Alt + Mouse-Scroll = Make the window the mouse is over transparent
Super + Mouse-Scroll = Zoom in and out of the screen
Super + E = Look at all desktops at once
Super + Tab = Tab between windows in a cool way
