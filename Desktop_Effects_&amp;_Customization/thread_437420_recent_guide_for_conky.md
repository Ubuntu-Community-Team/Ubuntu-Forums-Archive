### Post by notwen on 2007-05-08
Try [this]("http://ubuntuforums.org/showthread.php?t=353362").

If you want a simple install you can simply type this in a terminal
```
sudo apt-get install conky
```

then this after conky is installed simply type this to start conky w/ the default config file.

```
conky
```

To edit what's actually showing up on your screen you'll need to create your own .conkyrc config file in your /home folder.  The above link goes tells you how to do this and also provide you w/ a simple script that you can copy from the threads to your .conkyrc.  You can also check out [this thread]("http://ubuntuforums.org/showthread.php?t=281865") for ppl's conky configs and screenshots.  The conky courceforge page also has a page w/ all the variables you can use.  It takes awhile to understand whats what and to get it looking the way you want on your screen, but is generally worth it in the end.  Hope this helps. =]
