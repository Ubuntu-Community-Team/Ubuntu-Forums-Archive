### Post by por100pre1 on 2007-10-22
Select your themes and icons with **Appearance**:

```
gnome-appearance-properties
```

Add more themes and icons easily with **Art Manager** (**System> Preferences> Art Manager**):

```
sudo apt-get install gnome-art
```

For desktop effects add this:

```
sudo apt-get install compizconfig-settings-manager
```

You'll find it in **System> Preferences> Advanced Desktop Effects Settings**
