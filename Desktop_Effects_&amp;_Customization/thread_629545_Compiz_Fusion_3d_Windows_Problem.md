### Post by Mr Greencastle on 2007-12-02
What you are seeing there is a bug in older versions of the 3D windows plugin. You can fix that by compiling a newer version of it - though that would mean you'd have to recompile all of Compiz and Fusion as the version that ships with Gutsy doesn't let you have the newer plugin.

You could search the forums on how to compile Git versions of Fusion.
