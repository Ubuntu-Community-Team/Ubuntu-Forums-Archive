### Post by Ub1476 on 2007-12-12
Inside the achieve there's probably a .emerald file. You install this by clicking "import" in the emerald theme manager. After you vae selected the theme you would need to restart emerald, by typing "emerald --replace" into alt+f2. You can also use the Fusion-Icon to do such things (attached).
