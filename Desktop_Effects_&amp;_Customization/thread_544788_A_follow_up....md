### Post by Inxsible on 2007-09-07
Do not use the terminal to start compiz fusion. Press Alt + F2 and then type in ```
compiz --replace
```
alternatively, if you have installed emerald and would like to load that you need to type in ```
compiz --replace -c emerald &
```
Also you can install the fusion-icon (the deb is floating around somewhere in this forum) This will help you switch between composite managers and also give easier access to emerald themer and ccsm.

### Post by Inxsible on 2007-09-07
BTW. I think you are using the gconf as the backend for compiz. It has given a lot of grief to people. Go to System --> Preferences --> CCSM. Then click Preferences on the left navigation bar. In the Backend section select, Flat file configuration and close ccsm. That should get rid of your gconf errors that you see at the bottom of the post that you made previously.
