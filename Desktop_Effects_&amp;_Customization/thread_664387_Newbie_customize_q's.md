### Post by erginemr on 2008-01-14
Hello,

Please refer to the following thread:
[http://ubuntuforums.org/showthread.php?t=615080](http://ubuntuforums.org/showthread.php?t=615080)

aysiu's post for the difference between metacty and gtk2 themes.

The installation of both comprise the same steps, right-click on desktop, AFAIK, for some gtk2 themes to work, you need to install a few so-called "gtk2 engine" that are available in Synaptic. 

For instance, to achieve the vista-esque desktop theme in the attached screenshot:

1) I downloaded the following "GTK2" theme:
[http://www.gnome-look.org/content/show.php/MurrinaGreen?content=67185](http://www.gnome-look.org/content/show.php/MurrinaGreen?content=67185)

2) which required me to install the following engines as packages:
gtk2-engines-murrine
gtk2-engines-pixbuf

3) I grabbed the icons from the following icons theme:
[http://www.gnome-look.org/content/show.php/Glass+Icons+Theme?content=32146](http://www.gnome-look.org/content/show.php/Glass+Icons+Theme?content=32146)

4) The default green borders were good enough, but I decide to venture and downloaded the following "Metacity" theme:
[http://www.gnome-look.org/content/show.php/LiNsta+%28LiNsta+is+Not+Vista+%3B-%29+Pack?content=41090](http://www.gnome-look.org/content/show.php/LiNsta+%28LiNsta+is+Not+Vista+%3B-%29+Pack?content=41090)
Opened up the file, selected one of the windows border themes inside and installed it.

5) I also downloaded a vista-esque wallpaper from the web:
[http://static.flickr.com/122/304895038_4e858f4850_o.jpg](http://static.flickr.com/122/304895038_4e858f4850_o.jpg)

6) Used the Customize button in the same dialog, to, well, customize my desktop (desktop, borders, icons, mouse cursor, etc). You can also save this setting with a proper name, if you like to play more with several other themes.

7) That is all there to it. Whew, it seems quite a long task, but every bit of it was fun.

Now that you are into it, I suggest you to also play with gdm (login screen) themes, which is quite fun:
[http://www.gnome-look.org/content/show.php/Doe+GDM?content=52791](http://www.gnome-look.org/content/show.php/Doe+GDM?content=52791) (this is the one I am using)
[http://www.gnome-look.org/content/show.php/Avio-GDM?content=37395](http://www.gnome-look.org/content/show.php/Avio-GDM?content=37395) (this site contains info on how to install gdm themes)

and GFX Boot (a technique invented by SuSE [as far as I know] as a successor to text-only Grub menu):
[http://www.gnome-look.org/content/show.php/gulliver+Ubuntu+GFXBoot+Grub+Theme?content=57145](http://www.gnome-look.org/content/show.php/gulliver+Ubuntu+GFXBoot+Grub+Theme?content=57145)

Although installing gfxboot will require much more work than installing a regular theme, it is well worth the effort.

This was just an example, but following my example, you can select the packages you like from gnome-look.org or similar sites and easily tailor your Ubuntu desktop to your taste.

Have Fun!
