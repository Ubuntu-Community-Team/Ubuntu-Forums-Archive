### Post by iomicifikko on 2007-10-19
ATTENTION: Menu names could be different, i'm just traslating from my language (Italian)

Ok, here it is, simple and clear.

1.Enable Compiz Effects:

System->Preferences->Appearance-->Visual Effects Tab: Set it to Normal/Extra

2.If not already done, intall CompizConfig Setting Manager:

System->Administration->Synaptic: find and install "compizconfig-settings-manager"

3. Start Compiz Manager and enable feature:

System->Preferences->Advanced Desktop Effects Settings->Window Management Section-> Scale->Actions Tab->General:

Now Double-click on "Initiate Window Picker For All Windows" and select screen corner as u like it, and click on OK.

That's it!
