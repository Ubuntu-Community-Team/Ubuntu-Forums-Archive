### Post by fenix849 on 2007-07-29
You need to install the compiz-fusion-plugins-unsupported package..

Be aware that this may introduce unexpected issues into compiz and isn't recommended unless you will use it.

```
$ sudo apt-get install compiz-fusion-plugins-unsupported
```
