### Post by zullion on 2008-02-09
your chip is blacklisted since beryl
you can undone that follow this instructions

From the Terminal:

$sudo gedit /etc/xdg/compiz/compiz-manager

Your file should look like this:

# goes into /etc/xdg/compiz/compiz-manager
# works with git://anongit.compiz-fusion.org/fusion/misc/compiz-manager
COMPIZ_BIN_PATH=”/usr/bin/”
PLUGIN_PATH=”/usr/lib/compiz/”
COMPIZ_NAME=”compiz.real”

Add this line to the bottom of the file

SKIP_CHECKS=”yes”

Now it must look like this

# goes into /etc/xdg/compiz/compiz-manager
# works with git://anongit.compiz-fusion.org/fusion/misc/compiz-manager
COMPIZ_BIN_PATH=”/usr/bin/”
PLUGIN_PATH=”/usr/lib/compiz/”
COMPIZ_NAME=”compiz.real”
SKIP_CHECKS=”yes”

Just save it and close.

You may need to install the compiz manager:

$sudo apt-get install compizconfig-settings-manager

Thats all. Just enable the desktop effects.

Please Post a reply if your able to do it.
