### Post by santiagoward2000 on 2008-01-12
Hi there!
I have a simple question now. I wanted to set a Fixed Size for Pidgin using Compiz' Window Rules. I could do it for the Buddy List by setting:
```
title=Buddy List
```
But I can't find a way to do this for my conversation windows. Any help?

---

### Post by santiagoward2000 on 2008-01-16
Hi! I figured it out!
In case anyone else was looking for this, what I did is using:
```
class=Pidgin & !title=Buddy List
```
and that did it! :)
I hope this helps someone!
Cheers!
