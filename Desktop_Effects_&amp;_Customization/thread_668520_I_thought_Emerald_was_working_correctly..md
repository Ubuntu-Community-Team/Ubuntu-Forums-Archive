### Post by MasterJS on 2008-01-15
Easiest way to do so without having to hit CTRL+ALT+BACKSPACE would be to hit ALT+F2 and then type "emerald --replace" (without the quotations) and emerald should reload, applying the theme you choose.

---

### Post by rmcder on 2008-01-15
If you have the compiz conguration settings manager installed, go to general settings, commands, and enter the command: "emerald --replace" without the quotes) as command line 0 (or whichever).  Then bind a key combination to that command (I use Special+1).  From that point on, using the key combination will activate any decoration changes.
