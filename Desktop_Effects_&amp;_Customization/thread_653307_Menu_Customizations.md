### Post by adam.tropics on 2007-12-29
Install CompizConfig Settings Manager, then from the 'general' section, you can set opacity settings. For example, I have
```
 dialog | Tooltip | Menu | PopupMenu | DropdownMenu
```set to 90.

As for the meat of your question though, you can place the mouse over any window hit <ALT> while using the scroll to raise and lower opacity...but I don't think that does menus.
