### Post by lordpuppet on 2007-10-20
In order to configure compiz fusion in Ubuntu gutsy Gibbon we need to download the compizconfig-settings-manager

Code for installation:
```
sudo aptitude install compizconfig-settings-manager
```

Then, we'll go to preferences -> appearance -> Especial Effects (not sure if its "especial efects in english") and we get this fourth option:

[IMG]http://img148.imageshack.us/img148/249/dddpd3.jpg[/IMG]

That's all
