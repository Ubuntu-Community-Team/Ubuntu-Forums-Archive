1. Installing [[COLOR="Red"]themes[/COLOR]]("http://ubuntuforums.org/showthread.php?t=227416&highlight=install+themes")

2. Wallpapers
To install wallpapers go to the Xfce Settings Manager an click on "desktop" and you should   be       able to install the wallpaper from there.  The Setting Manager is a good place to start "pimping" your desktop. It holds nice features such as show desktop menu at right click on mouse, hiding desktop icons and transparency. I have found many nice wallpapers at [www.deviantart.com]("http://www.deviantart.com")
