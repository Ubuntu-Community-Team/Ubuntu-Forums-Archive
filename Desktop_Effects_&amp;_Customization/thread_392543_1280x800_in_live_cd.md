### Post by chewearn on 2007-03-29
You already have 1280x800 in xorg.conf, so the required change is done.
So, after you installed ubuntu to the harddisk, it still doesn't let you change the resolution to 1280x800?

Only thing I can suggest is, CTRL+ALT+F1 to go to console only.

Stop gdm:
```
sudo /etc/init.d/gdm stop
```Reconfigure xorg.conf:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```Select i810 as video driver, select the resolution.

Start gdm:
```
sudo /etc/init.d/gdm start
```Or:
```
sudo /etc/init.d/gdm restart
```CTRL+ALT+F7 if the login screen does not appear.
