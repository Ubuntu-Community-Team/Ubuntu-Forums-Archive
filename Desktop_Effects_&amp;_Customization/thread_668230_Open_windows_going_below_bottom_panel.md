### Post by timzak on 2008-01-15
I have an upper and lower panel on my desktop.  When certain windows open (eg, update manager), they open un-maximized, with the lower part of the window below the level of the bottom panel.  Sometimes it can be tricky dragging the window out so you can see the bottom of it.  I thought the panels were supposed to be barriers that couldn't be crossed?  This seems to be the case with the upper panel, but not the lower.  In fact, it is very easy to grab an open window and drag it below the level of the bottom panel.  I'd like to prevent this from being possible.  Is there a way?

---

### Post by Therion on 2008-01-15
Are you running Compiz-fusion?  If so, see if this helps...

Sart CCSM, enable the **Place Windows** plugin and then, under the General tab for that plug-in, use the drop down menu for "Placement Mode" and change the setting from "Smart" to "Centered".

I don't think it will prevent your bottom panel from being "crossable" but it should prevent windows from opening below that panel in the first place.
