### Post by Ub1476 on 2007-11-29
GTK2 themes is what you should download. GTK2 is for the latest Gnome and just looks better than GTK1.

When you have downloaded a theme, open Appearance>Install>Navigate to the theme (usually a tar.gz/bz2 file). Then it will install automatically.

For icon(window borders (beryl/compiz/metacity) themes, simply do the same.

Some themes for the window border come as a ".emerald" theme. To use these you need to install emerald and import it from emerald theme manager.

Remember that Ubuntu wont install .rar .zip (or similar window archives), so if they come packed as those, unpack it and put it in .themes/.icons (cursors go in .icons). You can of course just pack the foder into a supported archive again though:)

Note that it's not every Icon Theme which is made for the latest Gnome (which comes with U7.10), so I recomend you to either look at "top rated" or "latest".
