### Post by Therion on 2008-02-21
Depends on what you like.  I think Linux Mint is a pretty nice looking distro, even though I don't use it.  I also think PCLinuxOS and Vector Linux are pretty good looking even if they are KDE.  I use Ubuntu Ultimate which comes with a its own theme preinstalled, and while it's certainly not for everyone, I find it rather attractive personally.  Then again, customizing ANY distro is soooo easy I wouldn't let the interface dissuade you from using any one particular one.  Changing anything and everything - from the desktop to the fonts to the window borders - is only a few clicks away.

Maybe do some distro shopping at the [Linux Screenshots](http://thecodingstudio.com/opensource/linux/screenshots/index.php) page and see if you can't find something you like.
