### Post by raja on 2007-04-01
Gdesklets is the easiest way to do this. Just do ```
sudo apt-get install gdesklets
```. Now you can start gdesklets and then add various widgets to your desktop. Finally, add gdesklets to you session, so that it starts up everytime.
