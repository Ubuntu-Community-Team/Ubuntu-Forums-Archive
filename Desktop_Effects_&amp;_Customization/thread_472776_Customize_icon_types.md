### Post by shearn89 on 2007-06-13
I think you either:
1. Have to change the icon in /usr/share/pixmaps
or
2. Install a different icon theme, which contains icons for regular txt files.

To install and icon theme, download the tar.gz file from a site (eg. gnomelook.org), and go System -> Preference -> Theme. Then drag the tar.gz file onto it (it should say "icons correctly installed"), then go "Theme Details", "Icons", and select the icon set you installed.
