### Post by Zorael on 2008-04-17
Clutter reduction.

If you have several desktops, you can split your windows across them, instead of having one windows with everything. You get a better overview of things; basically, more room for your windows.

For instance, you could have one desktop dedicated to Photoshop. One desktop dedicated to Amarok or whatever library media player you use. One desktop dedicated to Firefox.

I "always" have at least one maximized Firefox, perhaps a maximized Amarok, a couple of file managers (Dolphin), random smaller windows suchs as IMs (Skype, Pidgin), a terminal console, and other random windows. With several desktops, I can spread them in an organized fashion, always keeping Firefox and other smaller windows on the first desktop, always placing the terminal console on the second, always placing Amarok on the third, and always keeping file managers on the fourth. Then I set IM windows to display on all desktops. Upon opening a media player, you could drag that one to another desktop so as not intrude on some more important desktop area. So instead of using Alt+Tab or the panels to control which windows get to be ontop, you could just move to the correct desktop.

The possibilities are endless. :>

The cube is nice and all, but I find myself more and more drawn to only using the Expo plugin, in which case I might as well use the [Desktop Wall](http://www.youtube.com/watch?v=00-5ocSmlV8).
