### Post by YldGuy on 2008-03-31
you need the compiz settings enabled. right click on desktop - Change Desktop Background -> Visual Effects. It should show options for "None" "Normal" "Extra" "Custom". If these are not there you probably need to install compiz-settings-configuration. Check the "Synaptic Package Manager" under Administration to install it. After its done you change the settings under it to use the cube or the fire or whatever you need. Good Luck.
