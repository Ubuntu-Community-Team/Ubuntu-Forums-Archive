### Post by joshuachad on 2007-08-28
Envy didnt work well for me. Its supposed to be pretty good and has worked for others. My suggestion would be to go out to nvidia and download the driver and just run the shell script is comes with. I think the latest is 14.11. Anyway there is an excellent guide here that worked for me with an nvidia setup:[http://compiz.org/NVidia](http://compiz.org/NVidia) .

Do all this before even trying to install compiz. I know you are probably already aware but some ppl install compiz first then realize that things dont work well then they later go back and install the drivers. Things dont work out to well. Also do yourself a favor and compile from git. Getting compiz for a package repo just never worked out well for me. There is an excellent script i used on this forum here: [http://elemongw.awardspace.com/?q=down_cf_i-u](http://elemongw.awardspace.com/?q=down_cf_i-u)

### Post by joshuachad on 2007-08-28
To answer your second question "I just want to know if there are any problems with ENVY settings and compiz-fusion." That question is the same as asking is there any issues with nvidia drivers and compiz fusion. All Envy does is intall the latest greatest nvidia drivers so you can render compiz via glx. And of course there are bugs with compiz in general but any way you go there will be certain bugs you may have to work out. I personally prefer doing things manually and i would recommend downloading the drivers yourself.
