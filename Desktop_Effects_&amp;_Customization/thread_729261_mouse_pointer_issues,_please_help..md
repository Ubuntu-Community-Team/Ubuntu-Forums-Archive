### Post by rune0077 on 2008-03-19
Open CCSM, and go to "General Option". There's an entry here called "Pointer theme" (or something along those lines), type in the exact name (case sensitive) of your mouse-theme in this field. This should do it, though to get the complete theme you may also have to log out and back in again.

If you're not using Compiz, try just the log out/in part.

You need to install it through synaptic (you're going to need it any way, it let's you change a lot of things in Compiz). Open Synaptic and search for compizconfig-settings-manager, then install that. This should give you something called "Advanced Desktop Effects Settings" under System > Preferences. Then open that and that's the one.
