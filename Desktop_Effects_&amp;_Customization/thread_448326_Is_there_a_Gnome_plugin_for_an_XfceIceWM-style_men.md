### Post by hal8000 on 2007-05-19
Yes there is. With gnome 2.18 the combination  alt+F1 brings up the application menu.

You can customize this from gnome, system, preferences, keyboard shortcuts. Find "show the panel menu" click it then press a new key for your accelerator. You could use the "windows" key for example.
Hope that helps
