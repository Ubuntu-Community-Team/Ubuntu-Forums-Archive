### Post by It_Was_Luck on 2007-10-29
Install the Advanced Settings Manager for Compiz via the respositories. Just search "compiz" and you'll see it there. You can then go to:

System > Prefrences > Advanced Desktop Effects Settings

From there you can enable the cube effect along with dozens of others.
