### Post by Therion on 2008-02-27
Try this:  Go into System / Preferences / Appearance / Visual Effects and choose "None"; then go back to that menu and re-select whatever option you were using before.  You're kind of doing a "reset" if you will.  See if that brings back your min/max/close buttons.
