### Post by Tomatz on 2008-03-20
I had this problem.  

I fixed it by using fusion-icon to start compiz instead  adding it to the end in the preferences, sessions, startup dialog basically by naming it zzzfusion-icon. For me the problem was that compiz was starting before the panel and hiding it.

Download it here:

[http://ubuntuforums.org/showpost.php?p=3163821&postcount=8](http://ubuntuforums.org/showpost.php?p=3163821&postcount=8)

Ps you need to turn off desktop effects to use this otherwise you will have 2 instances of compiz running :)
