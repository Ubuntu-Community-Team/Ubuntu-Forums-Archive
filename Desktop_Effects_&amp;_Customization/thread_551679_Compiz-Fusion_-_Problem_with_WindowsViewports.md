### Post by Rupertronco on 2007-09-15
go to general options, it's at the top of the compiz config settings manager. In the first few tabs, I don't remember which (I have beryl on the computer i'm currently on so i can't tell you specifically) it will ask for horizontal desktop size, change the value 2 to 4 in order for the cube to actually be a cube.

### Post by hyperair on 2007-09-15
In CompizConfig Settings Manager->General Settings->Desktop Size, change the value of Horizontal Virtual Size to the number of viewports/virtual desktops you want.
