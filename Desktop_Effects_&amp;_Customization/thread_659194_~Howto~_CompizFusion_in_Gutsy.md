### Post by r41nz0r on 2008-01-05
Ok, this is my first how to and it is a very simple one, It is simply because I had this same problem and I seem to hear people frequently ask the same question. Well Compiz-Fusion does come default in gutsy you still have to install the settings manager to enable the advanced effects that we all want. So in order to do so follow the instructions below. 


Go to System > Preferences > Appearance and then to Visual Effects, just turn them on. If you want more options just type the following into terminal(Alt+F2):

[COLOR="DarkRed"]```
sudo apt-get install compizconfig-settings-manager
```[/COLOR]

You will now have a new option Preferences to adjust advanced settings! Don't forget to say thanks!
