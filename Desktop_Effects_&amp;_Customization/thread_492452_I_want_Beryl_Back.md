### Post by ykpaiha on 2007-07-04
my wife love beryl and I try compiz so no problem both can work together
Install beryl as usual (mine is the trevino -svn 3.0+git)+berylmanager + emerald, do not forget to install as well all the related plug-ins. 
Install compiz -fusion with as well the trevino's repo
in system=>pref=>session make one entry for compiz +1 for emeral and one for beryl (beryl manager command include emerald)
check or uncheck each one you need to run then control+alt+backspace log-in and that's it.
as I said it works for me with two users.and on my desktop I switch time to time from one to the other and without any trouble.
so go ahead!!
I forgot maybe you have to erase your entries related to beryl in your home so check it as well.......
