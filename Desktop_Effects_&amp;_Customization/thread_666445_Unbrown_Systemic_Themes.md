### Post by dsginter on 2008-01-13
Hi all,

I've started installing Ubuntu on borked Windows systems for friends and family (much easier than trying to reinstall Windows, that is for sure) but there is a recurring theme that I've noticed - people hate the brown.

My question is: has anyone made a script to simply get rid of the brown (or even add a variety of ***choices*** for the end-user), systemically?

I just spent some time searching for such a tool and did not come up with anything.  I'm willing to spend the time to make my own but I wanted to check here before making that investment.  FWIW, it really looks like we need better theme management.  As it sits, to get rid of the brown, you have to change stuff in a number of different areas (e.g. - login background, login splash, login theme, desktop background, desktop wallpaper, desktop theme)....  and for each new user (I realize that there are some shortcuts, but I wanted to ask about an existing tool before doing this research),,,

If anyone with some clout is listening, I would just like to make a comment that forcing a single color theme is a bit too idealistic and goes entirely against the grain of FOSS.  Just give people a simple color choice during setup (e.g. - brown, blue, green, etc) and a *single* place where it can be changed thereafter.

While this might seem pedantic on my part, I would just like to say that my systems all run the brown theme because I really don't care.  But as it sits, if someone offered an easy, single button choice, I definitely wouldn't use brown.

Regardless - thanks to all for helping me eliminate Windows on my desktop.
