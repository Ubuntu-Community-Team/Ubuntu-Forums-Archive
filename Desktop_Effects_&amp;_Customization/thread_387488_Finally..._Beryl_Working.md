### Post by emgator on 2007-03-18
After a week of xgl-server and Beryl installs and uninstalls, I finally (and shockingly) found Beryl to be functioning. My setup is:

AMD Athlon 64 3000 (running Edgy 32-bit)
Nvidia GeForce 2/GTS
512mb PC2700 RAM

When trying to start Beryl, I kept receiving the "composite" error, and when composite was enabled in xorg I received the "GLX not found" error. 

Ultimately, I used Envy to reinstall the Nvidia drivers, although they were running perfect in Unreal Tournament using the Automatix Bleeder drivers. The Ubuntu forum post below is what I used to install Beryl, although you should skip the initial portion which describes how to install the graphics driver.

Envy: [http://albertomilone.com/nvidia_scripts1.html]("http://albertomilone.com/nvidia_scripts1.html")
Ubuntu Forum Post (thanks djRobbieF): [http://ubuntuforums.org/showthread.php?t=268036]("http://ubuntuforums.org/showthread.php?t=268036")

I hope this helps anyone with a similar setup. These forums have been great for troubleshooting, and I think it was high time I gave something back.
