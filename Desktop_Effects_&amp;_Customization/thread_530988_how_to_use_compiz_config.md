### Post by Rupertronco on 2007-08-21
The best way to get to know the settings manager for compiz is playing around with the stuff. Animations are a good place to start. As for themes are you using kde or gnome as your desktop? (kubuntu or ubuntu) In any case try [www.gnome-look.org](www.gnome-look.org) and [www.kde-look.org](www.kde-look.org) for themes. Emerald is your window theme manager and you also probably want to get a GTK theme if you want a fully customized desktop. 

Check out AWN (avant window navigator) for a cool dock for the bottom of your screen. I can't think of any real guides to using the compiz settings manager but if you have a specific thing you want to do or a specific effect you'd like to have I can help you from there.

### Post by Rupertronco on 2007-08-21
1. You have to make sure to turn on emerald Alt+F2 and enter "emerald --replace &"
2. The background behind the cube is called a skydome. Check around for some skydome images (they must be .png file format)
-skydome is found in the desktop cube options and appearance tab, it will ask you for a path for the picture you want to use
3. The other cube background is the first option, i think it's just called background or something similar, there put the other path for the background image. 

You'll definitely want both a gtk theme and a boarder theme (emerald).

### Post by Rupertronco on 2007-08-21
There's a frequently used thead in this very forum, i think it's called something like functional eye candy with awn and affinity. 

As far as the themes go there's some guides on gnome-look etc. It's good to have guides to help you through the process at first but make sure you try to take some of it in so you can change it yourself without having to constantly refer back to a guide. The easiest way to download a theme and install it using the GUI is to get the theme package and go to System>Preferences>Theme and Install the theme there (you can install it as a zipped package, it may even require you to do it that way). That's for GTK, as far as your emerald goes, lets get that running before we go after themes for it. 

 The best way to learn this stuff is by actually diving in and doing it.
