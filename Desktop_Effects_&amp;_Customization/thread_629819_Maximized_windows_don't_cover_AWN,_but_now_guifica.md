### Post by jlotempio on 2007-12-03
Not sure if this helps, but I noticed that if you close AWN, you can place something in that bottom region, such as some sort of Desktop icon (I tried it with my Home icon on my desktop).  Once something is placed in that region, it will stay there once AWN is opened, and is still functional.  However, if you try to drag something there while AWN is opened, it won't allow you to do so.  So if you want something down there, try putting it there while AWN is NOT opened, and then open AWN once you have your icon or w/e in place.  This works with both configurations (whether the maximized windows check box is checked or not).
