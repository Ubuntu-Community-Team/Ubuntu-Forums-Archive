### Post by VinceDee on 2007-04-13
RESOLVED (SEE MY FOLLOW-UP MESSAGE)

Using Nvidia 6800GS
Fresh install of edgy
Installed nvidia driver according to [tseliot's instructions]("http://ubuntuforums.org/showthread.php?t=281823&highlight=tseliot")
Added the following code to my xorg.conf according to [the beryl website]("http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu_Edgy_with_nVidia")

```
Option "AddARGBGLXVisuals" "True"
Option "RenderAccel" "True"
Option "AllowGLXWithComposite" "True"
Option "backingstore" "True"
Option "TripleBuffer" "True"
```

The thing that got it working was the suggestion to Enable Composite in the xorg.conf file. That managed to get Beryl working great, but after a few minutes it just froze my entire system. I couldn't ctrl-alt-del or ctrl-alt-bksp. I had to hard reset. Running the "beryl" command or "beryl-manager" command now freezes the system, and here's the output that I get:

