### Post by Kivech on 2008-02-16
Ok, hold on I'm fairly new to this also, but I just installed AWN and it is working like a dream.

I have it running under Gnome though, so maybe you need to look for different yet similar names of sections. Can't be sure since I barely know KDE.

There should be a menu option to define sessions on start up. In Gnome it is in System>Preferences>Sessions. There you can manually create another session in the list, name it and as a command you enter avant-window-navigator

That was all there was to it in my case. There must be a place where you can configure startup programs in KDE as well. I would expect it not to be too hard to find.
