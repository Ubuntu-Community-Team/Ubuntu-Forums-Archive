### Post by tilmaniac on 2007-10-22
Yeah, 
I'd be interested in a good tutorial as well where I could read about the default keybindings and different features that are available.

Try installing 'compizconfig-settings-manager'.
Its a nice GUI Control panel that allows you select many of the different effects that aren't enabled by default.

Also, there is usually some information there about what keys active the particular feature.

For example, the 'desktop wall' feature is Super + e   (Super is the windows key).

What I haven't been able to find out is how to do the Mac OX-like Expose feature.  If anyone knows how to do that, let me know!
