### Post by erginemr on 2008-03-15
Hi,

Assuming that this problem occurs only when Compiz is enabled and following these similar posts:
[http://ubuntuforums.org/showthread.php?t=202312](http://ubuntuforums.org/showthread.php?t=202312)
[http://ubuntuforums.org/showthread.php?t=572956](http://ubuntuforums.org/showthread.php?t=572956)

1. First, make sure that compiz-config-settings-manager (or ccsm in short) has been installed in Synaptic.

2. Then right click on your desktop and select Change Desktop Background -> Visual Effects -> Preferences

3. In there, make sure that the plugin "Place Windows" has been enabled.

4. If so, then change the placement mode to either Start or Cascade, whichever works.
