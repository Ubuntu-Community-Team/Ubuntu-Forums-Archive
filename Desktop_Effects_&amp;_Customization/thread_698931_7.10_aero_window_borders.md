### Post by rainwalker on 2008-02-16
Install the package "emerald" with Syanptic. This will install Emerald, which handles what your window borders look like (it's fancier than Metacity).
Then go to [www.gnome-look.org](www.gnome-look.org) and download whatever Emerald theme you like; you can search the site for aero-like themes, I've seen tons of them

---

### Post by EXCiD3 on 2008-02-16
There are also some metacity themese that resemble Aero that you can find on gnome-look.org. This doesnt require installation of Emerald to use the themes.
