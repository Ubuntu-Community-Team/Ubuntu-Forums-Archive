### Post by Kiri on 2008-03-28
Sorry, solved it already. Was much easier than I thought. :lolflag:

I just had to setup in keyboard shortcuts for move window to workspace. I expected this to only work with the window, but it had the unexpected result to switch to the other workspace if nothing is selected. 
I couldnt set super key combos in there so now I use alt+1 for workspace 1 etc.
Hopefully there will be no shortcut conflicts....

I also setup 'put' plugin in compiz with super+1, 2, 3, 4 for each workspace to throw the current window there but keep the user on the current workspace. WIth this combo, I can move windows all over the place :-D
