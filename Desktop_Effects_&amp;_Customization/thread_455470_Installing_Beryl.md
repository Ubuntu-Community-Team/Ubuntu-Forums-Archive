### Post by Happy_Man on 2007-05-26
If you mean desktop effects, go to System --> Preferences --> Desktop Effects. 

If you mean Beryl, open up a terminal (Applications --> Accessories --> Terminal) and type:
```
sudo apt-get install beryl beryl-manager emerald-themes
```

Wait for it to do its thing.... and then type
```
beryl-manager
```
