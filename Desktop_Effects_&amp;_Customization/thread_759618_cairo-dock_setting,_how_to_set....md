### Post by kawaji on 2008-04-20
Hi,

I think current Cairo-Dock does not have options to display it below other windows.

If the Auto-Hide is too slow for you, the speed can be changed.
Go to Right-clicking menu -> Cairo-Dock -> Configure ->  'Do you want to know more?' -> 
System ->  Movement Parameters,
Set Icon's lateral unfolding acceleration value to ' 0 ' , 
and Window's move acceleration(move up/move down) values to ' 0.900 '.

If that's not a reason you don't want to use the auto-hide, 
you can request developers for implementing a ' Do not keep above others ' like function in [the official forum]("http://www.cairo-dock.org/bg_forum.php?f=5").
