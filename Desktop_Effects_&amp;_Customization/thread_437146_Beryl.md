### Post by H.E. Pennypacker on 2007-05-08
To install Ubuntu, follow this guide:

[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

To install Beryl, use this wiki:

[http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu](http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu)

Since you're a new Ubuntu/Linux user, I highly recommend you do not start using Beryl right away.  First become comfortable using your new operating system, and when you are sure you are comfortable, start using Beryl.

If you have any other questions about Beryl, please see the Beryl forums:

[http://forum.beryl-project.org/](http://forum.beryl-project.org/)

If you want more information on Beryl itself, please see its website:

[http://www.beryl-project.org/](http://www.beryl-project.org/)
