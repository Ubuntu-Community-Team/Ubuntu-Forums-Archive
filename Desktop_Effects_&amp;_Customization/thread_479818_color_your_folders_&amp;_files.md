### Post by urukrama on 2007-06-20
You can install a different icon theme. You can download plenty at gnome-look.org. Drag them into the Theme dialog (System > Preferences > Theme) and they will be installed. Alternatively, copy the folder with the icon theme (extract archive first if needed) to /home/USERNAME/.icons

You can change the background colour of Nautilus (the default file manager in gnome) too, if you want.
