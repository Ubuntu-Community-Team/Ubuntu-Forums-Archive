I`m using alacarte to customize my menu settings and i`m wondering how I add applications that require root access (e.g. Nessus) to the menu?

Thanks for any help

---

### Post by Compyx on 2007-04-28
Put 'gksu' in front of the file to execute. For example for synaptic:
```

gksu /usr/sbin/synaptic

```
