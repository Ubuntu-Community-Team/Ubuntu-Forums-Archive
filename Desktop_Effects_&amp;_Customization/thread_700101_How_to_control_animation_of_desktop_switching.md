### Post by lian1238 on 2008-02-18
What is the current "animation" for the desktop switching? Is it the annoying cube effect? Note that this is not really the Animation plugin of compiz. The animation is for effects like minimize, maximize, creating/closing windows, etc.

If you have the cube effect and want to change it.
1. Run ccsm
2. Uncheck Desktop Cube and Rotate Cube
3. Check either Desktop Wall or Desktop Plane
