### Post by Ender Black on 2007-05-02
This is a fairly easy job.  I really enjoyed the burnt orange at first but it just wasn't "me".  So, to get my completely green OS that I like to look at here's what I did:

System ->Preferences-> Theme Manager

Even if you are using Beryl w/Emerald, the theme manager controls which icon library your system calls up.. so pick a theme that is pleasing and select Customize and hit the icon tab.  Find an icon set you like and close that out.

Download Gnome Art Manager from synaptic (gnome-art) or from terminal: sudo apt-get install gnome-art

Once that is downloaded, you can find the art manager under the System->Preferences  pull-down menu.  The manager allows you to download all the art resident at Gnome.Art and change your splash, login window, background, etc.  

The last bit of orange is tricky to kill.  Go to System->Administration->Login Window.  Click on the Local tab and change the Background color under the login window options.

You should have purged the system of all the offending orange colors from your UI.
