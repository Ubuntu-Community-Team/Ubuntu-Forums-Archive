### Post by brownknight on 2007-06-10
In my own words - Beryl is a window manager. Instead of the flat surface that we have been used to in ms windows, beryl introduces very nice eye candy like, having your desktop like a cube (3 dimentional), transparent windows, see all open windows in a key stroke and the wobbly windows (not my favorite). 

Metacity is also a window manager - it is the default window manager used by Gnome.

You can visit the site of [Beryl]("http://beryl-project.org/") for more info.
