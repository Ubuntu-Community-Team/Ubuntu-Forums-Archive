```

### Post by gen0m on 2007-06-22
[QUOTE=Happy_Man]Yes, great, wonderful, BUT I HAVE NO WINDOW DECORATIONS!!! I've tried everything on these posts, followed gen0m's howto, WHY WON'T MY WINDOWS WORK??????[/QUOTE]


open ```
gconf-editor
``` 
Goto  **apps>compiz>plugins>decoration>allscreens>options>command** and insert **gtk-window-dekorator** or **gtk-window-decorator.git** (if you renamed the apps).

Next goto **apps>compiz>general>allscreens>options>active_plugins** and insert **decoration**. Make shure it loads before **wobbly,fade,cube,scale**
Now you should start```
compiz --replace gconf
```

### Post by gkiller on 2007-06-22
Since I also struggled a bit with unmet dependencies and so on, here is a way to install Compiz Fusion from the Trevino repositories that should always work:

1. Before adding any additional repositories, go into Synaptic and search by name for "Compiz". Then mark every installed compiz package for removal. Apply and close synaptic. Ubunfu-Desktop gets removed also but no problem since we put it back later

2. Now add the Trevino repositories, "sudo apt-get update", and install the new compiz packages, as described here: [http://ubuntuforums.org/showthread.php?t=481314](http://ubuntuforums.org/showthread.php?t=481314)
In short, for Gnome following packages are needed/recommended:
compiz compiz-gnome
compizconfig-settings-manager compizconfig-backend-gnome
compiz-fusion-*

3. Now you can install ubuntu-desktop again with "sudo apt-get install ubuntu-desktop". This will also reinstall desktop-effects, so everything is ok package-wise.

4. "compiz --replace &" should now start a working compiz and decorator. If not, "metacity --replace &" get's your old window manager back. Try to log out and log back in, then again "compiz --replace &". If this still does not work, then there is probably another problem unrelated with dependencies etc.
Running "compiz --replace" from the terminal, without the '&' might give you some useful error messages that you can then post here or search on the internet.
