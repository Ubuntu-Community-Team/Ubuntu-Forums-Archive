### Post by condamine on 2007-09-26
I had the same problem from the time I installed Feisty Dawn (7.04).

Somewhere in the forums a suggestion to try installing different themes fixed the problem. As the themes I tried weren't as nice as the default "Human theme"  I lived with this annoyance for some time.

Eventually I got around to adding extra screen resolutions to my xorg.conf file and magically the orange progress bar appeared for the first time!

I shied away from the desktop effects upon seeing the warning about it being experimental!
