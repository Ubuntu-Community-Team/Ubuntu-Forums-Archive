### Post by sayakb on 2008-03-21
Open Compiz/Beryl preferences. There, search for General settings and find Virtual Desktop Horizontal Size and increase it to 4.
Alternatively (This is easier), right click on a panel and select Add to panel. Add a workspace switcher applet. Now right-click on the workspace switcher applet and select Preferences and increase the number of columns to 4 (or 5 for a pentagon, 6 for a hexagon and so on..)
