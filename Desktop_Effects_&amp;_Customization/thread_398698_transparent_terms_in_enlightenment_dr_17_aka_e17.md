### Post by conceited on 2007-04-01
well if you've installed enlightment dr 17 and noticed that the gnome-terminal's tranparent option didn't really work, you might be a little puzzled like i was. I remember last year when i was running linux with e17 i had to use 
e17setroot -s /directory/picture.jpg to set the background for terms transparency.
but now that i use ubuntu, the deb package i installed e17 with didn't come with that 
program, so i'm thinking how i could get it to work. So i talked to some people in 
#e on irc.freenode.net and they told me eterm comes with Esetroot, which is pretty 
much the same, you just type 

```
Esetroot -s /directory/picture.jpg 
``` 

then

```
left click -> configuration -> wallpaper 
```

and select whatever wallpaper you Esetroot to.

now terminals will have transparency when you use the transparency option.
