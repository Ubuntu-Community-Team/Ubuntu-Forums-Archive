### Post by Inxsible on 2007-08-30
If you mean the black box with the number 1 in it, then there is nothing wrong. It simply represents the desktop and you can use it to go to the desktop if you want to without having to minimize all the windows one after the other.

Other Docks available are :

1) AWN - Avant Window Navigator - By far the most popular dock
2) KoolDock - gives you the fish-eye effect like Mac OS X docks, but it kept crashing on me, at least until I dumped it
3) SimDock - havent used it, but heard it has some issues too
4)gDesklets - there is a desklet for a dock in gDesklets
5) your good old bottom panel - yes it can be converted to a dock too, although it wont be as cool as the other ones.
