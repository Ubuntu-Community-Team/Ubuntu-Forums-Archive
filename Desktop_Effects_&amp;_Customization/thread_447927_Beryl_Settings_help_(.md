### Post by digitalbenji on 2007-05-22
This is a little bit more complicated, but you can do it.  You will want to go into the area for Window Specific effects -> transparency, and add a new rule for the window you want.  You can capture the window parameters pretty easily for most windows.  I only have this set up for terminals, but it should work for anything.  You then need to set the percentage of opacity, I forget if I use .2 or .8, but play around and see what you like.  <p> I know this isn't that specific, I don't have a beryl box near me at work right now, if you need more guidance, let me know, and I'll post from home later...
