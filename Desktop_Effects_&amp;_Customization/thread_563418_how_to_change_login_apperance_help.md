### Post by tbroderick on 2007-09-30
Take a look at the currently installed themes in /usr/share/gdm/themes. You can install themes by extracting them there and then run gksu gdmsetup to select the displayed theme.

---

### Post by jrib on 2007-09-30
The "local" tab in the menu item "system -> administration -> login window" will let you change gdm themes.  You can grab more at gnome-look.org for example.  You can install more themes by just dragging the tar.gz into this setup window or clicking on the "Add..." button.
