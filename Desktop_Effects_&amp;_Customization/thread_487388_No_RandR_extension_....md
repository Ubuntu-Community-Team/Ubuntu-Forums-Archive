### Post by *daniel on 2007-09-24
I hate to reply to a very old thread, but the Xinerama X extensions (that's a lot of X's!) and RandR don't appear to be compatable. At all. So if you have one enabled, you can't have the other enabled. Which makes it difficult (or impossible) to run Beryl or Compiz on Xinerama-enabled X servers.

I've heard Compiz will, in the future, no longer rely on the RandR extensions, which would be cool.
