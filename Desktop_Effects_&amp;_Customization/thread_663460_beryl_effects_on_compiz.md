### Post by luisromangz on 2008-01-10
The first thing is the scale plugin and it's included in compiz fusion. Try using CompizConfig Settings Manager (ccsm) to enable it.

The second one happens when you have your window maximized and grab a window corner, while using the wobbly window plugin, I thought that was the default behaviour.

### Post by Forlong on 2008-01-10
> **kaiwan said:**
> In beryl I had 2 cool things that I dont know how to set up in COmpiz-Fousion:
1.thing:
[http://news.softpedia.com/images/reviews/large/beryl-img2-large.jpg](http://news.softpedia.com/images/reviews/large/beryl-img2-large.jpg)
The default keybinding for that in Gutsy is [Shift]+[Alt]+[Up]

See [here](http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion) under "Bind useful plugins to screen corners" how to initiate it via the upper right screen corner. 
> **kaiwan said:**
> 2.thing:
[http://beranger.org/blogo2/f7_kde_beryl_s.jpg](http://beranger.org/blogo2/f7_kde_beryl_s.jpg)
Maximize a window and hold [Alt] pressed. Now pull with your _middle_ mousebutton.

Or just disable **Snapoff maximized windows** in the **Move Window** plugin.

(You also might want to increase the **Grid Resolution** to something like 20 in **Wobbly Windows**)
