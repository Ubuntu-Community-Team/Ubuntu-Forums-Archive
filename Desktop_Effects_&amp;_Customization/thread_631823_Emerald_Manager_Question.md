### Post by s_spiff on 2007-12-04
Umm, dude, the Emerald theme is supposed to change only the borders. The other things like the transparent effect is done by settings in compiz settings manager, and the other is conky which is a lightweight system monitor. Please check out if the borders like that shown above in the screenshot (like the maximise, minimize and close ) are being shown in your apps. If that's happening, then your emerald is working fine. You just need to tweak compiz settings to get the true transparency.
