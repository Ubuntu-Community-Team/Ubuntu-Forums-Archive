### Post by isaacj87 on 2007-08-15
That plugin is called the "Shift Switcher." Which is a part of the 2nd screenshot.

The plugin "Shift Switcher" has 2 views Cover (the 2nd screenshot) and Flip (the 3rd).

Amaranth's Repo does not contain those plugins. The only way to obtain those plugins is to either compile the release edition of Compiz Fusion or use Trevino's Repo (which is out of date...still at 0.5.1, but! still has it :) ).
