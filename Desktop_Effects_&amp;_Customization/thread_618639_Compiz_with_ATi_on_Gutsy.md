### Post by NinjaTails on 2007-11-20
Compizwith ATi is so easy, it's not even funny.

Type this in the terminal:

sudo apt-get install xserver-xgl

Restart the X daemon by pressing Ctrl+Alt+Backspace, and you're done, Compiz should work fine.

If you want to install the special compiz settings manager, do this:

sudo apt-get install compizconfig-settings-manager

Enjoy!
