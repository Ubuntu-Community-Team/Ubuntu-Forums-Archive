Hi there everyone,

I'm not quite sure when it happened but my screensaver doesn't work properly anymore.  The first time I noticed it was right after I edited my xorg.conf so that I could watch movies without the monitor going off every 30 mins.  That's not the culprit though (or at least I'm pretty sure it isn't) as I've done it on 2 previous gutsy installs and countless other feisty installs.  Also I deleted all my edits and the problem was still there.

Speaking of, I've gone a whole paragraph without actually saying what the problem is.  Basically the screensaver does not start properly.  I am running compiz fusion, so whenever it's time for the screensaver to chip in I see the fade to black start, but then (almost like when you're falling asleep and trying like hell not to) it just snaps back to the desktop.

I have found a fix though, but I don't see why I need to do it.  If I execute the following code, the screensaver behaves itself for the rest of the session.  But every time I restart (now that I think about it, I haven't checked logging off, so maybe it happens when I log off too) the screensaver goes back to being broken.

```

killall gnome-screensaver
gnome-screensaver --display=:0

```

If anybody knows any way  to fix this short of adding a script with those two lines of code that starts up with the system every session that would be great.
