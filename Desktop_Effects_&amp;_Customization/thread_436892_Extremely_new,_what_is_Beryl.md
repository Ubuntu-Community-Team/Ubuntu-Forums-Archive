### Post by starcraft.man on 2007-05-08
Beryl is a window manager, it handles how your windows are displayed and the physics they use (wobbly, desktop cube, etc...). Emerald is the default Window Decorator that goes with Beryl, it handles the look of your windows :).

Both are not installed in any distro of Ubuntu as of yet. If you run Feisty, you have access to Compiz (via System > Preferences > Desktop Effects). If you want to know exactly how to install Beryl, please click my Ubuntuguide.org link in my sig and go to the eyecandy section, then pick your video card. Do make sure you have graphics drivers installed prior to installing berly (or using Compiz)

As for how stable Beryl is? I find its very good, overall I have rarely if ever crashed with it...
