### Post by filogo on 2007-03-28
I'm having the same prbolem as you. I have an ATI X700 mobility radeon with 128 Mb of GRAM. I run Feisty Herd 5. I solved the problem for the moment using under AIGLX server launching Beryl with this string:

```
LD_PRELOAD=/usr/lib/fglrx/libGL.so.1.2.xlibmesa /usr/bin/beryl-manager

```

This is a little trick that charges the fglrx library when using the Xorg ati drivers. So the  Checking for non power of two texture support now gives me a passed message.

It seems to me that this problem is only drivers related. In fact when I'm using fglrx drivers under XGL environment there is no texture error.

I hope it helps for you :)
