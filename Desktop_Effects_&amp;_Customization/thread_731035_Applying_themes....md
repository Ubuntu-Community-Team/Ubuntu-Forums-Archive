### Post by Therion on 2008-03-21
Yeah, see, to me... That's not a "theme": Those are window decorations (used by Emerald).  So you'll need to download that file, open Emerald and use the Import function.  Once the "theme" has been imported into Emerald, select it to apply the new window decorations.
