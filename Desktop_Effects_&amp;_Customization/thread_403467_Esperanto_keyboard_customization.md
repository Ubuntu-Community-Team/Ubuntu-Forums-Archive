### Post by rmockler on 2007-08-14
Your message was posted a long time ago, and you probably already have the answer to your question by now.  All I did to enable the ability to use the diacritical marks for entering Esperanto text is as follows:
System --> Preferences --> Keyboard
The only keyboard showing was US 105 key, so I did an add, and selected Esperanto.  Then for times when I wished to enter Esperanto text with the proper diacriticals, I went back through the above sequence and marked the Esperanto keyboard as the default.  Then while entering Esperanto text whenever I had to enter a character such as a &#264; or &#364;, all I had to do was key the appropriately coded key on the keyboard and it appeared in my text correctly.  The coded keys are below:

  X=&#264;, x=&#265;, Y=&#284;, y=&#285;, "=&#292;, '=&#293;, W=&#308;, w=&#309;, Q=&#348;, q=&#349;, :=&#364;, ;=&#365;

Hope this helps,
Richard
