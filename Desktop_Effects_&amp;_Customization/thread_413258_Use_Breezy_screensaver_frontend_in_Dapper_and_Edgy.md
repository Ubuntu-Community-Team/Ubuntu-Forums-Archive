Good day!

I started using Ubuntu since Breezy Badger, and just loved it. While I also love the newer versions and know how many improvements there was with each iteration, I still miss the old Breezy screensaver frontend because I can check which screensavers go into the random mode.

In Dapper, Edgy, and onwards, you can only choose random, where ALL the screensavers will go random, or individually select ONE screensaver. What I want back is the capability to randomize a SET of screensavers.:confused: 

This might be trivial to some, but for me, compared to Windoze, the screensavers are gorgeous. That way, I can pick the best I like, and anybody who passes by my computer would stop and look and say, "Oooh...Linux...nice...":)
