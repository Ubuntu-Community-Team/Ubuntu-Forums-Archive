### Post by Depressed Man on 2007-07-26
The poster above me already posted a link to how to install Compiz Fusion. But in case the answer wasn't directly clear. Yes, you can have both Beryl and Compiz Fusion installed. Though you can only run one at a time. And it has been done before. You just need to install Compiz Fusion then to start it press alt+f2 and type in compiz --replace

Or for beryl, I think it's just beryl

And whenever you want to switch I believe you just have to alt+f2 and then type in which one you want.
