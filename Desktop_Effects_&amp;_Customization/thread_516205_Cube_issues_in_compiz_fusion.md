### Post by psyopper on 2007-08-04
Back up your xorg.conf. 

Try adding the following to either the Device or Screen sections. In yours its the Screen section, with the other Options:

    Option         "NvAGP" "2"
    Option         "AddARGBGLXRootClipping" "True"
    Option         "RenderAccel" "True"
    Option         "AllowGLXWithComposite" "True"
    Option         "backingstore" "True"
    Option         "TripleBuffer" "True"
    Option         "DisableGLXRootClipping" "True"

Restart X with [Ctrl]+[Alt]+[Backspace]. Start Compiz and see what a difference (if any) was made. You can also try launching Compiz with the following command:

Without Emerald (GTK Window Decorator)
export __GL_YIELD="NOTHING"; compiz --replace

With Emerald
export __GL_YIELD="NOTHING"; compiz --replace && emerald --replace

Try one thing at a time before you try both together. See if the changes to xorg.conf help, then add in the new launcher to see if it gets even better (or better at all).

Also remember that Compiz Fusion is beta and some problems can be associated with beta software. Of course Beryl was beta too... There has recently been a [new version of Compiz-Fusion released]("http://lists.freedesktop.org/archives/compiz/2007-August/002581.html") which may (hopefully) provide a number of fixes and speed enhancements.
