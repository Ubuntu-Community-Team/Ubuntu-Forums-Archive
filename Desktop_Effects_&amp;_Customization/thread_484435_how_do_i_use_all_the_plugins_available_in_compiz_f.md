### Post by DraeLee on 2007-06-25
go to system>Preferences>Compiz Manager  that will have everything there to activate certain plugins.  Just remember the super key is the windows key if you dont know already.  under each plugin when you click it is a tab that says action (i believe) click that and it should tell you how to activate that particular plugin.  good luck and enjoy!!!
