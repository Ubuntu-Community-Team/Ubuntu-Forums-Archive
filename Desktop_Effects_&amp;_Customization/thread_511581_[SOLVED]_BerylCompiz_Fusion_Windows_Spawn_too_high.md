### Post by hyperair on 2007-07-28
Go to CompizConfig Settings Manager, and in the Place Windows plugin, change the Placement Mode to "Centered"

---

### Post by phibit on 2007-07-28
Wow, that worked perfectly! Thanks! 

I guess somewhere between when i upgraded CF to when i went back to a stable release, that setting was turned off... <3 ubuntu forums.

### Post by hyperair on 2007-07-28
No actually it was probably set to smart placement mode. But there are bugs in that placement mode, though I've never experienced it. I've heard of cases where it acted totally insane and placed the windows offscreen and whatnot, and it was fixed by switching to Centre.
