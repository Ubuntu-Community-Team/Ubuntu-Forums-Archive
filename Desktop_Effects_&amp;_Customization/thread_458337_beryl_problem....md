### Post by MACRI on 2007-05-30
fix'd with help from a friend... if anyone cares/could possibly find use for it... 

had to add
```

Option "AddARGBGLXVisuals" "True"
    Option "DisableGLXRootClipping" "True"	

```
had to add that under option default depth 24 in screen section of xorg.conf
