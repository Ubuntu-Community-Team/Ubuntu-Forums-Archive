### Post by SpecialFred on 2007-11-14
You need to set the Horizontal Desktop Size to 4. This is done through the Advanced Desktop Effects program. 

System -> Preferences -> Advanced Desktop Effects -> General Options -> Desktop Size

Currently it is probably set to two. With three, you will have a triangle instead of a cube, four gives you a cube, and so on.

Happy hunting!
