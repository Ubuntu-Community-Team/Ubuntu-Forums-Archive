### Post by chandler on 2007-06-28
I figured it out...  If anybody ever wants to do something similar:  ```
export PS1="\[\e[32;1m\]\u@\[\e[34;1m\]\h:\[\e[35;1m\]\w\[\e[32;1m\]\$ \[\e[0m\]"
```

For a detailed explanation go to: [http://www.ibm.com/developerworks/linux/library/l-tip-prompt/](http://www.ibm.com/developerworks/linux/library/l-tip-prompt/)
