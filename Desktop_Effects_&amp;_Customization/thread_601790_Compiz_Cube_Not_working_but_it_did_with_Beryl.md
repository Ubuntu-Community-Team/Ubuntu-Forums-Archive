### Post by Zorael on 2007-11-03
The most common error is to set Number of Desktops to 4, while it should be 1.

Settings Manager -> Desktop Cube plugin -> Desktop Size

Horizontal Virtual Size -> 4 (or another integer for how many sides you want the "cube" to have)
Vertical Virtual Size -> 1
Number of Desktops -> 1

It's not as easy as having a slider with 'sides of cube' anymore, since there are other plugins which handle multiple workspaces now as well. Contradictory to the Cube, these can have workspaces vertically. Such as Expo.
