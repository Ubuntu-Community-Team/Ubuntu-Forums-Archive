### Post by andrewsomething on 2007-06-15
Do you have the Beryl icon in you system tray?

Taking quick look at that tutorial, it seems as if you might not have Beryl set up to run at start up.

Try typing "beryl-manager" into the terminal after starting a Xgl session. If Beryl works, then you just need to add it to your startup programs. You can do this by going to System>Preferences>Sessions.
