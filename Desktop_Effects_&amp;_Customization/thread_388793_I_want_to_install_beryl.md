### Post by old_geekster on 2007-03-19
I suggest that you install "Automatix Bleeder".  This  is another version of "Automatix2".   There are not many programs to install, but one of them is Beryl .0.2.0.  This is the first stable copy of Beryl since 0.1.0.

Add this repos to your /etc/apt/sources.list:

#AUTOMATIX BLEEDER REPOS START
deb [http://ubuntu.beryl-project.org](http://ubuntu.beryl-project.org) edgy main
#AUTOMATIX BLEEDER REPOS END

Then, you can install it from "Synaptic".

I tried several times to install Beryl with no success, using "apt-get" and "Synaptic".  Then, I used "AB".  It worked like a charm.  Beryl is running with all of its glory.

Did you install the latest video drivers?  You didn't mention if you are using ATI or nVidia drivers.  nVidia cards seem to work better than ATI's with Beryl.
