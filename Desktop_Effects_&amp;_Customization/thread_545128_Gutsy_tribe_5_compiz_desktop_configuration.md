### Post by Ianman on 2007-09-07
Are you using compiz Fusion? If not you should switch :-) Then you should have the compiz settings manager under System -> Preferences. Otherwise you might want to install the compiz fusion icon which will give you quick access to the CCSM:
[http://tombuntu.com/index.php/2007/08/26/compiz-fusion-tray-icon/](http://tombuntu.com/index.php/2007/08/26/compiz-fusion-tray-icon/)

Hope this helps.

BTW, you can adjust the number of desktops etc from the CCSM under the General Tab
