### Post by adam.tropics on 2007-03-18
I can only think of a cheats' way!

Provided you're using gnome, get the background image setup, then create custom icons for your applications in some bright and obvious colour in inkscape, resize (right-click) the icons so they cover the area you want. Lastly edit the icons to transparent once they are in place, and disable icon text from gconf-editor.

mmm not ideal is it!?
