### Post by Forlong on 2007-12-28
Alright, install ccsm ```
sudo apt-get install compizconfig-settings-manager
```
Run *System &#8594; Preferences &#8594; Advanced Desktop Effects Settings* (or 'ccsm' in the terminal)

And go to the Rotate Cube plugin. There's an option called Zoom. This will decrease the size of the cube.
(if you have troubles with the cube per se, see [here](http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion) under "Getting the cube")

As for using the top and bottom of the cube as workspaces: that's not possible. If you enable the Cube Caps plugin, you can have custom images there, though.
