### Post by Venerence on 2008-02-20
Occasionally emerald crashes, it still has a couple bugs with it (assuming you're using emerald for borders). I've noticed that using alt+tab a couple times in quick succession usually kills it.

To restart the borders, open up the run application dialogue (alt + f2, you can also add a launcher to the main bar), or the terminal, and run emerald without any commands. You should get your borders back.
