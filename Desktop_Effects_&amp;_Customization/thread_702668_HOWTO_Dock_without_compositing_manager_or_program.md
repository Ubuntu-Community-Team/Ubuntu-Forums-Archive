### Post by herrshuster on 2008-02-20
Some of you, I'm sure would like to have a dock without having to use compiz (an older computer) or a separate program (just memory freaks like me. In this post I will attempt explain how to achieve a dock without anything besides Panels. (GNOME is required, but i'm sure something simliar is possible in KDE)

**_Firstly:_**
Make a new panel by Right-Clicking on an existing panel and hitting "new panel" A new panel will appear on one of the sides. Right-click on this panel and click properties. Under 'orientation', click the side you want the dock to be on. While you're here, set the dock height to 50 pixels and uncheck expand and check autohide if you wish (it saves screen space)

**_Adding some stuff:_**
You probable want a window list and application launchers on this, so right-click and hit 'add to panel' type in 'window list' and add that. Then, just drag icons from your applications menu to the dock until you're satisfied. add anything else you want at this point.

**_Making it pretty:_**
You COULD set a transparent background at this point, but without a compositing manager, things will tend to look pretty bad, so whip up a 50px high image in gimp or just go with a solid color under properties->background.

that's it! enjoy your lightweight application launcher!

I hpe this guide was useful. Ask questions if you need help
