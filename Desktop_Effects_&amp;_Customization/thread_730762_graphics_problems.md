### Post by ajeetraj on 2008-03-21
I am assuming that you have gutsy running. well, for gutsy, you don't need Beryl. Beryl and compiz have been merged together and now you don't have to install beryl to get your visual effects. 

if you have already installed compizconfig then you should change the settings from there and see if it works. if not then you should try typing this in the terminal to start the visual effects: 
```
compiz --replace
```
This should show you what exactly is your problem and why you can't run the visual effects. 

if you still have problems then please post your output of the above command. 

good luck!
