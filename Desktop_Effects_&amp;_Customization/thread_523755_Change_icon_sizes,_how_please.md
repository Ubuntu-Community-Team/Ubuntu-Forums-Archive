### Post by 1/0 on 2007-08-12
I'm using XFCE4 so I can't check this out but I think you can access this option from nautilus in preferences found in the edit menu. You can change the size via zoom level in "Icon View Defaults" and "List View Defaults". That should change both nautilus and the desktop.

### Post by nichipet on 2007-08-12
If you're in Gnome, you can right click on the icon and select Change Icon Size (or something similar, I'm not in Gnome right now), then click and drag any corner of the icon to change the size. There are also options in the right click menu of the desktop for aligning the icons by name, keeping them organized, etc.

---

### Post by FuturePilot on 2007-08-12
You can also change the default size by open a Nautilus window and going Edit>Preference and change the icon Default Zoom Level to something bigger or smalled. It will affect all of your icons.
