### Post by psyopper on 2007-07-05
> **phasee said:**
> Hi, I have a few questions which I would like to ask as I am just installed this yesterday.

2. How do you stop the cube zooming in automatically when you are using the mouse rotate option?


In CCSM go to the Rotate Cube plug-in and on the General tab there is a slider at the bottom labeled "Zoom". Slide this slider all the way to the left and see how it looks.

> **phasee said:**
> 
3. Is there a way to get the virtual dekstops to coincide with the workspaces you have set up? So that the taskbar isnt too clogged up? If I could have question 2 answered this wouldnt be a problem mind you...at least I dont think it would be


Well, I answered question 2, but I can also help with this one as well. There is a setting that only shows applications on the taskbar for that desktop...

In the Extras -> Window Previews plug-in there is a checkbox labeled "Taskbar Shows Only Windows of Current Viewport." This works even if you have Window Previews disabled (at least for me it does). You may need to enable Window Previews, check the box and then disable Window Previews if you don't want to use Window Previews.

Avant-Window-Navigator looks similar, there's a few threads about AWN in this very forum. I'm curious to find an answer to the "different bacground per viewport" question myself...
