### Post by adammw on 2007-06-20
Ok...
If you want your windows to drag between desktops, that should be enabled by default.
Dragging icons on the desktop depends if you can have different icons even without beryl (i think-[SIZE=1]i'm new-dont hurt me[/SIZE]), so i cant really help you with that.

I don't think you can have desktops on the top, as its supposed to be a linear thing. If you want to change that image however, try Beryl Settings Manager > Desktop > Desktop Cube > Caps and change the images.

And of course, the transparent setting is in the same place (almost) except under the **transparency** tab [SIZE=1](funny about that!) [SIZE=2]instead of caps as before.

Hope this helps,
