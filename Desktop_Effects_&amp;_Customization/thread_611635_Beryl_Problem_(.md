### Post by yezior on 2007-11-13
I would suggest to getrid of the beryl, because it is no longer supported, and get compiz-fusion instead (compiz-fusion is a fusion of beryl and compiz)

If you wanna stick with beryl, then just hit 
```
ALT+F2
```

and type

```
beryl --replace
```

and it's running
