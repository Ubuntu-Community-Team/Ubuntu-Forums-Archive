### Post by reacocard on 2007-04-06
That's [kiba-dock]("http://wiki.beryl-project.org/wiki/Kiba-dock"). It's been a bit since I've used it, but when I did while it looked cool, it wasn't easy to use and had an irritating tendency to stick at 100% CPU. A better alternative is Avant Window Navigator, which looks almost as cool but is much more usable and is very light on resources. There is a guide here: [http://ubuntuforums.org/showthread.php?t=385981](http://ubuntuforums.org/showthread.php?t=385981)

Note that both of these projects require Beryl or Compiz to work properly.
