### Post by por100pre1 on 2007-10-01
Try this:

Open Theme Preferences:

```
gnome-theme-manager
```

select your favorite theme then click **Theme Details**.

In **Theme Details**, you can select **Controls** (what you need), Colors, Window Border and Icons. Keep in mind that not all themes allow this tweaks.

Hope it helps. :)
