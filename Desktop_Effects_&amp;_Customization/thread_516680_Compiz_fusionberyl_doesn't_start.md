### Post by Obor on 2007-08-04
Just to say I fixed my problem.
All I had to do was enable composite extension in my xorg.conf.

From this:
```
Section "Extensions"
    Option         "Composite" "Disable"
EndSection
```
to this:
```
Section "Extensions"
    Option         "Composite" "Enable"
EndSection
```

and restart x.
