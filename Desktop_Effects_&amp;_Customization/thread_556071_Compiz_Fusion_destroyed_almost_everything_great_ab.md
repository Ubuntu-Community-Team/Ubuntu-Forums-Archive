### Post by rainwalker on 2007-09-20
I started using Beryl while on Edgy, back before it rejoined with Compiz. I liked everything about it; the effects, the fact that it worked, etc. My only problem with it was that it ran slowly, that and nothing else. It never crashed, never hung up, nothing. 
So, upon installing Compiz Fusion after ditching Beryl a while back, I was at first ECSTATIC. It seemed to be everything Beryl was (and more), and it actually ran at a decent speed. Soon, though, I began to notice flaws. 
It started off as seemingly random crashes (which I had never experienced with Beryl). Oh well, it's still experimental software, I can deal with it. Then I noticed that certain effects wouldn't work/start (or would crash CF if I enabled them). Hm...okay, that will get fixed. The 3D windows (when they work...) seem to expand rather than raise off of the cube, and then the desktop comes up to meet them rather than the windows moving down onto the desktop when selecting a workspace. I guess I'll just turn off 3D windows.Things with transparently lag a little, the 3D cube most of all (so much that I couldn't have any transparency at all, or it would lag). Arg, why?
A few updates ago, CF stopped working. It won't start up, and I have no idea why. So, just for the heck of it, I decided to go back and try out my old Beryl install (which I never removed). The first thing I noticed upon starting it was that the splash screen indicated it was the Beryl SVN, something I don't ever remember installing. The next thing I noticed was that things actually ran quickly, as fast as Compiz Fusion (if not better). All of my effects work, and nothing crashes it. What I found most strange was that I could have 75% opacity while rotating the cube, and it ran faster than ever before and faster than CF ever did! 
I've always had problems with Compiz, from the included effects in Feisty to my current CF install.

What happened to Beryl? It used to be amazing, and as far as I've found, the latest SVN before the merge works better than Compiz Fusion has so far.

### Post by FuturePilot on 2007-09-20
When Beryl got abandoned it was left in a pretty stable state. CF is currently under heavy development. I've noticed a few bugs in CF too. Some of the updates fix ones but create others, it's all part of running bleeding edge software. As for performance issues you might want to install the Fusion Icon. There's a thread in this forum for it. It has a few tweaks in it for performance. If you have a Nvidia card you definitely might want to use the --loose-binding option. Beryl had that enabled by default, in CF it's not.

And the reason you have Beryl SVN is because I'm guessing you used Trevino's repo? It also has Beryl SVN and it probably came in through an update at one point.
