### Post by Ub1476 on 2007-11-27
First download a few icon themes you like from [Gnome-Look]("http://www.gnome-look.org").

Now open "Appearance" then select the "theme" fan. Next just click "install", navigate to the icon pack and click on it. It should install automatically:)

You can delete the downloaded icon pack (usually a tar.gz/bz2 archive) after the icon theme is installed.
