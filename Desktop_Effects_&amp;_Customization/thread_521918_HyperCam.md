### Post by jpeddicord on 2007-08-12
There is no version of HyperCam for Linux, however there are some alternatives are available. Try going to Applications > Add/Remove, and search for "recordmydesktop". Install the item that appears, and then run it from the Sound & Video menu. You can then record your desktop with it (as the name implies). Sound "works," but it is messed up for me.
