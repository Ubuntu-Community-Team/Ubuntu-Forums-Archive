### Post by DerangedDingo on 2007-05-17
Well, for the start button, start the gnome configuration editor, and I suggest adding this to your Applications menu.
 System - Preferences - Menu Layout, and find Configuration Editor under System Tools. Check the box to the left, and close the window. Go to Applications, System Tools, and open the newly added Config Editor. Go apps - panel - objects, and then find the object_x that has 'menu object' as its type. Check the box on the right that says Use Custom Icon and then add the path up top in the 'cusom icon' line. It should change to the new start button right away.

For the black panel, right click an empty spot on your panel, open properties, and go to the Background tab, and choose Background Image: and find the black-panel.svg.
That should change right after you close the window.
