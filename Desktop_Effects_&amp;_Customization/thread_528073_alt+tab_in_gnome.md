### Post by Chaotic Thought on 2007-08-17
"Toggle windows in all workspaces" means that if it is ON (windows in all workspaces), then the window you have selected will appear in every virtual desktop.

If you don't use virtual desktops, then you don't need that keyboard shortcut, so I'd disable that one. I think you can press BACKSPACE to disable the keyboard shortcut. Then re-assign alt-tab to the window switching function
