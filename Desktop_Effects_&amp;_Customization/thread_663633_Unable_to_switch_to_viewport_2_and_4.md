### Post by cryo on 2008-01-10
I'm running Ubuntu 7.10 with the compiz stuff enabled. I love it.

However I'm having a slight problem switching to the viewports 2 and 4 when using ctrl+alt+left or ctrl+alt+right.

It's pretty strange behaviour really. I can use the scroll wheel on my mouse and access viewports 2 and 4. 

I can click on the viewport in the viewport preview and get there, but if I use ctrl+alt+left or ctrl+alt+right it switches me to viewport 3 and then if I do it again it switches back to viewport 1.

I did notice that in the viewport viewer it highlights desktop 2, for maybe half a second before switching to highlighting desktop 3.

I know it seems silly but I use the keyboad a lot more than the mouse and this is driving me nuts. 

Any ideas?

---

### Post by Forlong on 2008-01-10
Make sure you have only one of either Cube Rotate, Desktop Wall and Desktop Plane enabled.
