### Post by FuturePilot on 2008-01-31
Beryl is no longer developed. But Ubuntu 7.10 does come with Compiz Fusion which is the result of the Compiz Beryl merger.  You can install Emerald by
```
sudo apt-get install emerald
```
Also check if you need a driver for your graphics card under System>Administration>Restricted Driver Manager
