### Post by Ub1476 on 2007-06-14
Try to follow this guide (uninstall beryl first): [Linky]("http://ubuntuguide.org/wiki/Ubuntu_Feisty#How_to_install_Beryl_.28ATI.29") 

Scroll down to ..Alternate method: Using closed source FGLRX drivers from ATI.. and follow guide from there.

Also check if 3d is enabled. If it's not, you'll have to install drivers for your graphic card.

```
fglrxinfo | grep direct
```

Anyways, you'll install graphics driver though that guide, so..

Maybe you'll have to add this to your xorg.conf file:

```
Section "Extensions"
	Option		"Composite"	"0"
EndSection
```


If your ATI card supports open source (AIGXL) just follow the first guide you'll encounter when clicking on the link, and forget all about the above (except graphic driver).
