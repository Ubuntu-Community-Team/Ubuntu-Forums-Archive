### Post by FleetAdmiral74 on 2007-05-23
When I put my mouse in the up right corner, it used to show all the open windows regardless of what side of the cube they were on. Not so anymore...where do I change the shortcut key or set it back to default?

---

### Post by benanzo on 2007-05-23
**Beryl Settings Manager** -> **General Options** -> **Shortcuts** -> **Screen Edges**

For the drop-down list under **Top Right Corner** select **Initiate Window Picker for All Workspaces**
