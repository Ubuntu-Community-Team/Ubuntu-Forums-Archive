### Post by Espreon on 2007-10-21
Its easier to goto System>Administration>Restricted Drivers Manager and install the driver from there.
The you will need to install XGL:

Do this in a terminal:

```

sudo apt-get install xserver-xgl

```

The log out and back in and CF will be workin.
