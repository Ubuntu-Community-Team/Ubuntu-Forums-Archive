### Post by luisromangz on 2007-06-22
Well, you can still use beryl, but all the new changes will be made in compiz. Basically, compiz fusion is compiz plus the community packages that previously wasn't accepted there so the community created beryl.

So you have to install compiz, and if you want the community plugins, the compiz-fusion-plugins-* packages.

I have installed them, but I'll be using beryl for some more time, as I'm so used to some plugins that haven't been backported to compiz yet. You can have them installed side by side, so you can check progress in compiz while using beryl, which is what I'll be doing for some months, I think.

By the way, I use the packages from Treviño's repository.
