### Post by Ateo on 2007-05-25
Well, Beryl also needs a window decorator (as it only replaces window management). I think the most popular one is Emerald (it's what I use). There are 2 others that I know of. All are in the default repositories so they are easily installed.

If you decide to use emerald, probably the easist way to manage all this would be to ALSO install the beryl system tray icon thingy (i don't use it but I do recommend it to you). It is also in the repositories.

So, if you decide to take my advice, install emerald and the beryl systray icon. Launch the systray manager and a diamond will appear in your systray. Right clicking that will give you lots of options.

If that makes you happy, you'll want to replace your current window manager at startup. You can just search these forums for that.
