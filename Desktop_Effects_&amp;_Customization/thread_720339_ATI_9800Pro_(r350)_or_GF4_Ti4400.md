### Post by Melcar on 2008-03-10
That card is supported with the radeon driver, which is included in Ubuntu.  That driver has  good 3D support, so if you want a good "out-of-the-box" experience, keep the 9800PRO.  If you want more performance, you will have to install the proprietary drivers, which shouldn't give you that many problems to set up.
