### Post by Inxsible on 2007-09-02
it would be best if you installed Compiz Fusion after removing Beryl completely. There have been a few ppl who have had problems with two composite managers being installed/running at the same time.

You can use 
1) Trevino's repos - 3rd party has the latest plugins 

2) Amaranth's repos - stable but doesn't have the very latest plugins

3) Compile it from source.

All 3 options have a bunch of howtos on the forums. Search for it and you shall find it.

### Post by hyperair on 2007-09-02
You either use the CF_Installer_Updater.sh script lying around somewhere in ubuntuforums, or you go download the source code for every package, then go into the respective directory, and then run the following:
```

./configure (or ./autogen.sh)
make
sudo make install

```

It's very time consuming though, and every time you want to upgrade you have to repeat the procedure. I suggest that you don't compile from source unless you really have that much time and know what you are doing.
