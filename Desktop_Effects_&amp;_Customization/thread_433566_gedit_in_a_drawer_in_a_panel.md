### Post by Palypup on 2007-05-05
I like having something to write on while on the internet but don't like anything on the desktop that doesn't have to be there. I was used to having a notepad.txt in my quick launch '95 through XP but couldn't get gedit.txt to run in a drawer on the top panel. It would not run citing a child process error and couldn' find the file or if it did run it was a new virgin document. That was no good because I was adding to a list. So... finally figured it out. I copied the contents of the gedit text to an Open Office Word Processor  document, then saved it as .txt
and put it in /home/docs and put a link in the drawer in the panel with the other links I use.
I got what I wanted just the way I wanted it.
