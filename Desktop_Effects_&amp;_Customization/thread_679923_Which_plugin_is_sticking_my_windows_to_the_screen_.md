### Post by jryprt on 2008-01-27
Getting windows to wobble

    * First of all you obviously have to enable Wobbly Windows

    * Then, if you want your panel menu to be wobbly as well, choose Shiver in Wobbly Windows &#8594; General &#8594; Map Effect

  [COLOR="Red"]**_  * And you might want to disable Wobbly Windows &#8594; General &#8594; Snap Inverted to prevent windows from adhering to screen edges._**[/COLOR]
