### Post by jaygo on 2007-10-08
To give you a little more detail about KDE's implementation ...
It's under system settings > desktop > advanced options
You can probably figure it out quickly. It lets you blend a webpage with regular wallpaper in various ways, configure update intervals, and it might even let you set a different webpage for every monitor & desktop. The catch is you can't interact with the webpage from the desktop as you could in Active Desktop (i.e. no clickable links). Perhaps someone's hacked it into doing that elsewhere...

It sounds like you're a tweaker, so I would also recommend switching to KDE. Adding KDE to an ubuntu install let's you get a feel for it but I had a lot of errors doing that and eventually wiped & reinstalled kubuntu / mint KDE.

As for gaming, I'm just trying out enemy territory & assaultcube, which both run natively. And a number of popular win games run in Cedega / Wine ... I'm going to be trying out guild wars, half-life, HL source, and all their mods soon.
