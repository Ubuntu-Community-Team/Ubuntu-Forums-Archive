### Post by PatrickFisher on 2007-11-13
Unfortunately, Window Preview just "makes miniature" whatever is drawn for the window. Since the window is minimized, there is nothing drawn. So, basically, you ARE getting a preview... a preview of nothing.

This means that there will be no "fix".

You may, however, want to request this as a feature for a future release. It would be nice, but it's not likely to happen.
