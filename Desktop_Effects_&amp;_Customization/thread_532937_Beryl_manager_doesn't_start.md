### Post by Inxsible on 2007-08-23
have you installed beryl-manager?
```
sudo aptitude install beryl-manager
```

If you have, go to Applications--> System Tools --> Beryl Settings Manager. That should give you a nice little ruby in the tray. Right click on it to change settings

As a side note: Beryl is now unsupported as it has merged with Compiz to become Compiz Fusion. Give that a try. It has all the features Beryl had and a lot more.
