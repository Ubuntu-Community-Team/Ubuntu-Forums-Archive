### Post by LocoMan on 2007-10-26
I think I solved it.

Did ```
sudo apt-get remove xserver-xgl
```following the advice on this thread: [http://ubuntuforums.org/showthread.php?t=585978]("http://ubuntuforums.org/showthread.php?t=585978") and seems to have solved it, both glxinfo now shows direct rendering as yes, nvidia-settings works, games too, and while that thread said that desktop effects wouldn't work afterwards, don't seem to have a problem here.

I'll wait a day or to of it working before adding "solved" to thread title, though... I've had things being solved in ubuntu before and then without warning unsolving themselves (though that was 2 versions ago).

### Post by smoqer on 2007-10-28
if you use XGL server have to do:

```
DISPLAY=:0 glxinfo | grep render
```

XGL is on DISPLAY :1 and doesn't support direct rendering, so you must check DISPLAY :0
