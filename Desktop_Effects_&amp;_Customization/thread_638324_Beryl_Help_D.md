### Post by schauerlich on 2007-12-11
Are in you in Gutsy or Feisty? If in Gutsy, run ```
sudo apt-get install compizconfig-settings-manager
``` to install ccsm, which will let you edit all of the compiz settings to give you the cube, burning windows, etc. Go to System>Preferences>Advanced Desktop Effects Settings and play around with the different plugins there.
