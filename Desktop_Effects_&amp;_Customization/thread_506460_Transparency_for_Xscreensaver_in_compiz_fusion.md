### Post by WinterWeaver on 2007-07-22
Go to your Compiz Settings manager:
System >> Preferences >> CompizConfig Settings Manager

Open up the General Options

Go to the Opacity Settings Tab

you will see a list of windows, scroll that box all the way to the right, and you will see the opacity value there... just change that value to 100, and you shall have no more fuzzy screensavers ^_^

have fun
