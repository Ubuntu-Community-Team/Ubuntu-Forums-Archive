### Post by timehAndGod on 2008-04-09
In my attempt to get a great looking top menu for Ubuntu, that blended well with my desktop environment, I decided to go for a Leopard-style top menu... to find there is no good ones floating around.

The problem with recreating the Leopard top menu, is that it uses blending modes to generate its unique transparent effect, and as of yet I do not know how I could incorporate that into Ubuntu's top menu.

So I have created a run-of-the-mill PNG image that has my first attempt at making a Ubuntu top menu image that is similar to the Leopard one, I only spent 5 minutes on it but it looks pretty decent...

To install it:
1. Download the archived PNG image attached.
2. Right click your top menu and go into 'Properties'
3. On the 'General' tab make sure size is set to '24'
4. On the 'Background' tab select 'Background image' and select the image you have just downloaded.
