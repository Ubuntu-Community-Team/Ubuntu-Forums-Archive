### Post by santiagoward2000 on 2008-01-09
> **ranjithnair said:**
> How to compiz themes as it is in .emerald extension which is not recognised.

Hi!
First, you have to install emerald, which is in the repositories. Just open a terminal and type:
```
sudo apt-get install emerald
```
Then, open **Emerald Theme Manager**, click on *Import...* under the *Themes Settings/Themes* tabs and look for the .emerald file.
Cheers!
