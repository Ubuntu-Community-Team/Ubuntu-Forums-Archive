---

### Post by Ek0nomik on 2007-10-01
You can install an icon theme by placing the contents in ~/.icons (/home/YourUserName/.icons).

You should than be able to select the icon theme in:  System/Preferences/Theme.

---

### Post by Rupertronco on 2007-10-01
You can also have the theme manager install the icon theme. It's essentially the same thing as moving it to your .icons folder, but it's completely graphical if you're more familiar with that. 

Go to System>Preferences>Theme

Then hit customize on whatever theme you're currently using, select the icon tab, and hit install, browse to the location of the icon theme (tar.gz is fine) and it will install it for you.

### Post by FuturePilot on 2007-10-01
Or just open the theme manager System>Preferences>Themes and drag the archive right into the main window there. No need to open extra windows.
