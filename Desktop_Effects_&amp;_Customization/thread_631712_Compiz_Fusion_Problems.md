### Post by tombean on 2007-12-05
> **dethredic said:**
> sorry how do I edit it? It says I can't edit it.

I have the only account so I assume I am the super user or admin

Yes, you're the "super user", but Ubuntu works with*** sudo***

You can edit the xorg.conf file with this command:

```
sudo gedit /etc/X11/xorg.conf
```
You'll be prompted to enter your password.

Ctrl + C the

```
Section "Extensions"
    Option "Composite" "Enable"
EndSection
```

and add it to the bottom of your xorg.conf file and save it.

OR (easier)

just comment out the bottom lines about Section "Extensions"
```
# Section "Extensions"
# 	Option	    "Composite" "0"
# EndSection
```

Regard,
-tombean
