### Post by pestilence on 2008-03-11
Ok I completely removed xserver-xgl and my ati x700 is not experiencing a single problem with compiz!
So (I am sorry if this has been posted...but lots of guides out there mention xserver-xgl) xserver-xgl is not needed my ati official drivers (latest) for proper running of compiz.

So I would recommend the following method on running compiz on an ati power card:

1) Download the official drivers (from ati's site)
2) Install
3) If you have xserver-xgl installed, remove it
4) enable composite option inside xorg.conf:
```

Section "Extensions"
        Option          "Composite"     "1"
EndSection

```
And restart your X session.
