### Post by Happy_Man on 2007-08-19
Perhaps you have a corrupt CD? From the initial menu given, select "Check CD for defects" and see if there are any.

If there are, reburn the CD *at the slowest speed possible.* 1x would be ideal, although I have never had problems with 4x.
