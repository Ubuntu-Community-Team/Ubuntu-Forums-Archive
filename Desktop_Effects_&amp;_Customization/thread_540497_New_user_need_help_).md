### Post by CRobison on 2007-09-02
"Universe repo" can be enabled by clicking on the "System" menu then following through "Administration" and finally "Software Sources".

Commands are entered in a terminal application.  Default terminal is found in "Accessories".

A great link that will help with many beginning questions: [http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

Hope this helps.
