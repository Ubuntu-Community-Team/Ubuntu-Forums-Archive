Look in your synaptic package manager type in Hebrew in search 
i found this and a few other packages to go with it 
GNOME translation updates for language Hebrew 
Translation data updates for all supported GNOME packages for:
Hebrew

language-pack-gnome-he-base provides the bulk of translation data
and is updated only seldom. This package provides frequent translation
updates.

Please note that you should install language-support-he
to get full support for this language.

### Post by sagai on 2007-12-21
Hi,
there is an option for setting up a session language for installed languages on the gnome login screen. (for me, it is on the bottom left corner)
&#1513;&#1500;&#1493;&#1501;
&#1505;&#1489;&#1493;&#1500;&#1509;

you have to have installed:
sudo apt-get install language-pack-he
language-pack-he-base

and optionally u may need these fonts (i am not sure ...)
sudo apt-get install culmus xfonts-efont-unicode xfonts-efont-unicode-ib xfonts-intl-european msttcorefonts

also can check this:
[https://help.ubuntu.com/community/HebrewLocalizationHowto](https://help.ubuntu.com/community/HebrewLocalizationHowto)
for keyboard related question.
