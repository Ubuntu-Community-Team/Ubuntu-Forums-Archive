### Post by Saint Angeles on 2008-02-27
make sure that in your /etc/X11/xorg.conf file you have: ```
Driver = "intel"
``` in the device section.
and at the end of the file add:
```
Section "Extensions"
Option "Composite" enable"
EndSection
```
