### Post by Ub1476 on 2008-03-18
It's not Rhytmbox, but Quod Libet! It's a very nice music/radio player (using it myself):)

The maximize, minimize etc is the window border. 

Just download the Metacity theme which can be found [here]("http://www.gnome-look.org/content/show.php/Murrine+GTK2+Cairo+Engine?content=42755").

If you use Compiz, "Emerald" is the default window border manager, so you have to use a tool such as fusion-icon to switch to Metacity window border. Or you can just find some fine themes for Emerald (they are called compiz/beryl on gnome-look.org.
