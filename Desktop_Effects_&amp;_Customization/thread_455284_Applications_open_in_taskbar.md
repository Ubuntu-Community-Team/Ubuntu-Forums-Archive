### Post by Kuoi on 2007-05-26
I think I've found it, but maybe you can verify it !!!

Open : Applications > System tools > Configuration editor

Now go to > apps/panel/applets/window_list_screen0/prefs/ ... and check "display_all_workspaces".
You'll see it immediatly on your taskbar.

That's how it worked for me.
