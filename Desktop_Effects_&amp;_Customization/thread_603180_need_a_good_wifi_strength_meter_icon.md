### Post by magli on 2007-11-05
The applet which comes running by default with feisty and gutsy is called "nm-applet". You can start it via command line: The command is "nm-applet --sm-disable".

If you want it to start automatically when you login to gnome, you have to go to System->Preferences->Sessions Make sure that Network Manager is listed there, and enabled. If not, add it. Here is what my entry looks like:
