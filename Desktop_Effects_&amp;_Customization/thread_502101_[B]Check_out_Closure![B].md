Hello all,

After spending some time on the AWN forum site,  which I highly recommend checking out, (...there are some amazing developments happening with AWN...it's up to rev 227 w/ reflection, angled bar, icon offset etc.) I found another interesting program that doesn't seem to be very well known around the Ubuntu Forums (I did a search for 'closure' and only 1 post came up). It's a program that allows you to log out, lock your screen, reboot, or shut down (yeah...no hibernate or suspend just yet) with a little bit more pizazz. It's called **Closure**.

Obviously Closure isn't very necessary with GNOME, but if you include it with the revised AWN, it adds a little more eye candy to you desktop (along with the awesome new Compiz-Fusion). Yes, Closure does require a composite manager (Beryl, Compiz, Compiz Fusion) so that must be set up first. I'm going to include a picture (with my icon theme in use with Closure) and a link for the installation. IMO it's pretty bug-free, just a few quirks, but no up-hill battles. Enjoy

[http://thelinuxmovement.blogspot.com/search/label/Closure]("http://thelinuxmovement.blogspot.com/search/label/Closure")
