### Post by the_lex on 2007-07-18
Ubuntu Studio doesn't come with desktop effects pre-installed. If  you open the Terminal (Applications>Terminal) and type in the following:

sudo apt-get install ubuntu-desktop

And it will be installed and put in the menu. A word of warning though: it seems that Ubuntu Studio does not work as well with desktop effects as just plain Ubuntu does. I'm currently trying to get desktop effects to work for me on Studio, and they won't!
