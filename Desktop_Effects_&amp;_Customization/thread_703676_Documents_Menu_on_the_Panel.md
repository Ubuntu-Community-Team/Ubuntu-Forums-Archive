### Post by applehead on 2008-02-21
Hi all..
Does anyone know how I could easily access my spreadsheet files from a panel.

I've tried using a draw but it doesn't display the name of the file in the draw, so you have to hover over each file to see which one it is.

I've also tried editing the main menu, which would be ideal, but it will only let me add a launcher to launch an application rather than a file.

In the help file it says choose 'file' from the launcher properties but it doesn't give me that option.

Any ideas please?

### Post by mgmiller on 2008-02-22
One thing you can do is navigate to the folder that contains your spreadsheets using nautilus.  Then, at the top of the nautilus window, click where it says Bookmarks, then Add Bookmark.

Now, if you look in Places on the top panel, you should see a direct link to that folder.  Once you create more than about 5 of these, a submenu opens on the Places drop down that says Bookmarks and a side menu will open showing the whole list.

If you just want a quick link to open a single file, look in the Places drop down under Recent Documents.  It shows the last 10 things you looked at.
