### Post by groovomata on 2007-10-21
Okay, I figured out how to add screenlets. Go to screenlets.org, Download the screenlets file. Make a folder in your home directory called '.screenlets'. Extract the file there and follow the readme for installation instructions. There are also various 3rd party screenlets that you can add. Just extract them into the .screenlets folder you made. When you follow the installation instructions you should end up with a menu item called 'Screenlets'. From there you can launch your screenlets. To launch any 3rd party screenlets you've added. Launch the Control item and right click to add them from there. Additionally there is an app that manages them under 'System > Preferences'.
