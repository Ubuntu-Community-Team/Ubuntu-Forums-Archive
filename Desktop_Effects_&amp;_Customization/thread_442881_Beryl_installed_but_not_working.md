### Post by Melcar on 2007-05-21
Got it working.  I forgot to **downgrade to the 0.2.0 version**.  Now it all works perfectly.

 > 
IMPORTANT -- Downgrading beryl-core to 0.2.0 

The Ubuntu version of beryl-core is incomplete. The beryl-xgl executable is not included in this package. To install a version of beryl-core that works, first run:

$ gksudo gedit /etc/apt/preferences

for Ubuntu, or for Kubuntu:

$ kdesu kate /etc/apt/preferences 

Then add the following to that file. If there are any other entries in this file, separate this entry (all three lines) from those entries by a blank line. Don't separate the three lines from each other.

Package: *
Pin: release o=lupine
Pin-Priority: 1000

Save the file.

This will cause your system to prefer beryl-project packages, but will keep Ubuntu packages if they are manually installed, and newer.

Next, we need to downgrade to a version from beryl-project that works with Ubuntu's confused and misguided Xgl. In synaptic package manager:

   1. click the 'reload' button
   2. find and select beryl-core
   3. click the 'package' menu, and select 'force version'.
   4. Choose version 0.2.0~0beryl1. 

Or, from the command line:

$ sudo apt-get update

and then

$ sudo apt-get install beryl-core=0.2.0~0beryl1


[http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu_Feisty_with_XGL]("http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu_Feisty_with_XGL")
