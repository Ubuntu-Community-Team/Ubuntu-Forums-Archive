### Post by Promaster91 on 2008-03-28
Try this:
Go to synaptic package manger and search for "compiz"
Then install the package "compiz" and all the other ones it wants to install.
After that go to Appliciations>Add/Remove
Search for compiz
Then install "Advanced Desktop Effects Settings (ccsm)"
Then hit alt-F2
Type in compiz --replace

That should work. I am assuming that you have the video driver up and running. I hope this helps.
