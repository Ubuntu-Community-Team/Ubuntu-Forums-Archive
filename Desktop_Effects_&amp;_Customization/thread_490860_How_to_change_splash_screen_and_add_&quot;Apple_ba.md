### Post by Calash on 2007-07-06
There are a few choices for Apple style docks

Kiba-dock - [http://ubuntuforums.org/showthread.php?t=268645](http://ubuntuforums.org/showthread.php?t=268645)

I liked this one the most, however it does not seem totally stable.  I could not find a way to get it to auto start correctly (It started, but was missing all the options).  It forgot the order of the icons.  The killer was that it really slowed down the compiz-fusion cube animation for me.

Avant Navigator - [http://ubuntuforums.org/showthread.php?p=2307772](http://ubuntuforums.org/showthread.php?p=2307772)

While I found this to be a bit less functional it is a lot more stable.  Starts when I want, does not slow down the system, and it just works.  

As noted above both need Compiz, Compiz-Fusion, or Beryl to work.
