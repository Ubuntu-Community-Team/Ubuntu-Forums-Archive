### Post by elliotjreed on 2007-12-13
[http://wiki.awn-project.org/index.php?title=Main_Page]("http://wiki.awn-project.org/index.php?title=Main_Page")

Is a gooooooody place to start!

The .deb file can be found [HERE]("http://www.getdeb.net/release.php?id=1621")

Remove the bottom GNOME panel, and put everything you want to keep (like window list and notification) on the top one for now!

Then add to startup via Preferences > Sessions > Add

```
avant-window-navigator
```

Enjoy!
