### Post by twizler on 2007-10-24
If you want to [COLOR="Blue"]change a Compiz / Emerald theme[/COLOR] -- you import the downloaded theme and then click on it in Emerald theme manager. 
NEXT

press --  ( [COLOR="Blue"]ALT + F2[/COLOR] )   : run application: 

type in text box::::   [COLOR="Blue"]emerald  --replace [/COLOR]

then [COLOR="Blue"]hit run[/COLOR] -- 

This will replace the current theme with the new one.
