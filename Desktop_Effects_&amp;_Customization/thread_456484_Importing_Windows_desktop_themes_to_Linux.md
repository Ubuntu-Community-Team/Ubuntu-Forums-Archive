### Post by H.E. Pennypacker on 2007-05-27
Linux does not place an importance on extensions.  It will read whatever you tell it to read regardless of extension, as long as it supports the file format.

I can tell you, however, that Linux does not suppor .dll files natively.  Sure, you can use Wine, but Wine does not run Windows applications natively.  Because .dlls are not archived packages, they can't be extracted...on Windows or on Linux.

I hate to tell you this, but Windows themes are Windows themes, and they do not work on Linux.  You can try to "port" those themes by creating your own theme for Gnome (what Ubuntu uses), but that would be quite a work, and you would have to learn how to make themes.

Depending on how much you like the particular theme you spoke of, it may be better off for you to use Windows instead.  I have never heard of anyone picking an operating system solely because of a theme, but feel free to use Windows, or see if there is a Gnome theme  that somewhat resembles the Windows theme you like.

See gnome-look.org for Gnome themes.
