Hello all. 

I've got Compiz Fusion working flawlessly, so I figured it was bout time to start customizing. After some google-ing I was surprised to find the good old Emerald worked for compiz fusion also (figured it was Beryl only). Anyways, I've got emerald working aswell. I just have one little niche. 

Whenever I bootup / login, the emerald theme manager pops up. It still does the job of changing the theme, but it is little annoying to have to close something that has no needed purpose to appear. I'm guessing it is the fault of the session command I googled. I played around with, but either it didn't work; or emerald didn't do its thing was the result. 

Here is the command :
```

compiz --replace -c emerald
```

Thanks! :guitar:

### Post by Criminosis on 2007-08-06
case closed :

After digging in on some other people's problems. Someone suggested a session command that was similar to mine; but it had another --replace at the end.

Added that and it was fixed.
