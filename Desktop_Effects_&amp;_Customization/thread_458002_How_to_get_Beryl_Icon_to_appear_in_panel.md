### Post by T'sora on 2007-05-29
Do you have a "notification area" set up in one of your panels? When you launch beryl-manager, the icon should appear there, letting you switch between Beryl, Compiz, Metacity, and whatever other window managers you have installed (right click-select window manager).

### Post by p110011 on 2007-05-29
Systems>Preferences>Sessions
Click Startup Programs Tab
then New
Name: Beryl Manager
Command: beryl-manager

Then of coarse the box is checked.
