### Post by SomeGuyDude on 2008-02-22
Open up Synaptic (System -> Administration -> Synaptic Package Manager) and look for "compizconfig-settings-manager" and install it.

Then, hit Alt+F2 and type in "ccsm", configure things how you want (Wobbly Windows, etc). Afterwards, hit Alt+F2 again and type "compiz --replace" and you should be up and running.
