### Post by coffeecat on 2007-05-29
[This page]("http://live.gnome.org/GnomeArt/Tutorials") from art.gnome.org has all the tutorials you should need. Also, get yourself thewidgetfactory via Synaptic - useful for previewing your widgets before using them for real.

**Edit:** Installing the widget factory from the repository doesn't add a menu entry.  You can do this yourself with the menu editor - the command you need is simply 'twf' - or do 'twf' from a terminal.
