### Post by bloodniece on 2007-11-04
Ok, fixed it.

CCSM --> Viewport Switcher --> Desktop-based Viewport Switching
Set Move Next, Move Prev, and Initiate Plugin Action to None.
They were set to button 4,5,and 2 respectively.
