kiba-dock is in the repo most people use for Compiz Fusion.  If you plan to use Beryl, thogh, I would not recomend updating out of that repo ( though you could still install kiba-dock from it before disabling ).  The reason is that beryl is also in that repo, and every machine I've had that version of beryl on has been buggy.  I've had serval times when I've had to remove all of beryl, including the two libs, remove that repository and install it over again.
