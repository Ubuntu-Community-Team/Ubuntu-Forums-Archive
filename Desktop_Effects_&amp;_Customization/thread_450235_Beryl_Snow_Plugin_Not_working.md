### Post by ericallen on 2007-05-21
Activate Snow effect with the Super(Windows) button+F3..
Since you've got the snow option, so you must have installed "beryl-plugins-unsupported" 
> 
The plugins in this package are supported by the Beryl Project,
but are not considered stable enough to go into the main
beryl-plugins package. Only install these plugins if you don't
mind some additional instability.

to remove them, open the terminal & type
```
sudo apt-get remove beryl-plugins-unsupported
```
