### Post by strumluff on 2007-05-29
Your nvidia drivers are not setup correctly that is why you cannot select Beryl as an option.

For starters the Driver in your Device section should be nvidia not nv.

If you really don't want to use the restricted drivers manager. Then I recommend you download Envy script, it automatically installs the latest nvidia drivers and configures xorg.conf for you. Just note that when kernel updates are made you need to set up the drivers again (can be simply done by running the script again but from terminal and use the command line version)

Get it here: [Linky]("http://albertomilone.com/nvidia_scripts1.html")
