### Post by stido on 2007-08-08
seems like your xorg is already properly configured to run compositing with direct rendering, and your card is certainly powerful enough to run compiz/fusion/beryl.

however, some people say that fglrx doesn't support compositing on its own, it needs an XGL server alongside xorg. another option is to use aiglx.

try searching the forum or the net for how to install, configure and run an xgl-server session (instead of a plain gnome session).

PS: ATI's own proprietary fglrx driver is blazing fast, especially if you compile it for your own kernel. try not to change it with other driver if you have gotten it up and running.
oh, and try this from a terminal just to make sure (if you haven't already):
aticonfig --initial
