---

### Post by Happy_Man on 2007-07-13
Open up the Compiz Fusion Settings Manager (if you haven't installed it, its package is 'compizconfig-settings-manager') and go to General Options, Opacity Settings. Scroll to the end of the list and change the value from 89 to 100.

---

### Post by bdtgp on 2007-07-13
You can also select the window, hold down Alt and use the scroll wheel to set opacity.
