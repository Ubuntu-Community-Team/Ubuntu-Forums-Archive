Hey everyone,

I recently installed CompizFusion on my Ubuntu computer (7.04). It worked fine, but for some reason, every time I restart the computer or log off, some of the components (such as the Cube Desktop, the Expo, and the Animation plugins) do not work. The Wobbly Windows and the Window Switching work perfectly. In order to fix this problem, I have found that I have to remove CompizFusion completely from my computer then re-install it. Is this a common problem? If so, is there a way to fix it so that I do not have to re-install CompizFusion every time I restart my computer?

### Post by amadeus266 on 2007-07-18
Compiz-Fusion does not start automatically. You need to add it to your start up list in sessions. I use
"compiz --replace &" and it works fine on my machine. Look around for other threads on this issue. There are several around with good advice.
