### Post by Kobalt on 2007-05-12
When you go into System > Pref. > Themes, if you click on customize you can select different types of window borders from there (metacity). You can pick some up at [www.gnome-look.org](www.gnome-look.org) and drop these in your .themes folder (from your home directory, press Ctrl+H to see hidden folders).
But I see you're runing Beryl so you need to change the emerald theme if you don't use metacity : right-click on your Beryl icon in the notification area and select Emerald Theme Selector. You can pick up new themes on the gnome-look site as well but you need to install them from the Emerald theme tools under Import.
