### Post by avik on 2007-06-16
Just run gnome-settings-daemon from a terminal window.  You'll have to run it each time you log in, or you can go to System->Preferences->Sessions and add that to the Startup Programs.

Besides, Beryl doesn't require Xgl anymore, I believe.
