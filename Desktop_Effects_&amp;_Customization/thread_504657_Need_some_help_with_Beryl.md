### Post by misfitpierce on 2007-07-19
ok first run beryl-manager and install if needed... Secondly once thats installed simply select a theme in emerald so its highlighted... right click beryl manager icon in top right and click refresh window border and it will reload the window theme manager... Whatever theme was highlighted in emerald will now become your theme.
