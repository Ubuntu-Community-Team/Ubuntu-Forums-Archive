### Post by Xarok on 2007-05-02
You can easily install Beryl Manager and Beryl Settings Manager from the 
**Applications > Add/Remove...** (make sure you are viewing all programs)
or
**System > Administration > Synaptic Package Manager **(Make sure you are viewing all repositories)

After installing got to
**Applications > System Tools > Beryl Manager**
and it should start

The most useful feature for me is the scale affect so you can see thumbnails of all windows from all desktops at once and select which one you want.
