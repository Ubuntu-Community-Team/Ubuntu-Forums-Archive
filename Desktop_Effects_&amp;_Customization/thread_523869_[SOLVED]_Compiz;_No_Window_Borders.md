Try the following in the terminal
```
gtk-window-decorator --replace
```

Actually, try launching Compiz in the terminal and then post the output here.

I have a 6200 and use the following options successfully:
```

    Driver         "nvidia"
    Option         "NvAGP" "2"
    Option         "AddARGBGLXRootClipping" "True"
    Option         "AddARGBGLXVisuals" "True"
    Option         "RenderAccel" "True"
    Option         "AllowGLXWithComposite" "True"
    Option         "backingstore" "True"
    Option         "TripleBuffer" "True"
    Option         "DisableGLXRootClipping" "True"
    Option         "AllowDDCCI" "True"

```

I think the AllowGLXWtihComposite will solve your problem.
