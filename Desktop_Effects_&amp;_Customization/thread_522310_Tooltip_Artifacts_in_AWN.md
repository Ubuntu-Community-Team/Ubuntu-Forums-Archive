### Post by andrewsomething on 2007-08-10
it's a know bug with certain video cards (GeForce 8XXX card it seems). See here:

[https://bugs.launchpad.net/awn/+bug/128818](https://bugs.launchpad.net/awn/+bug/128818)

Please post what graphics card you have in that bug report if someone hasn't already mentioned it in order to help the developers track doing the problem.

The not so great work around is to set all the colors under task appearance>text  to be completely transparent... =(
