Since it was still usuable (albeit annoying) it was assigned to the "Get around to it" file.

Well, this morning was the day. I was going to post about the problem and see if there was a fix er something. Noticed there were some updates waiting in the queue, so I started the updates while I was making the post - also noticed they were all CF related. Before I finished the post I noticed the updates were finished, so I reloaded CF. Just for sh*ts & giggles I fired up Agent.

Agent was no longer transparent.....
