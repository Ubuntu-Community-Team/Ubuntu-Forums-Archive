I install Beryl and it's working fine.  However when I take my mouse to up left corner the windows don't all scale down and get arranged on my desktop anymore.  How to a fix this?

---

### Post by blackened on 2007-06-03
In Beryl Settings Manager (right-click the Beryl icon in the notification tray, select "Beryl Settings Manager") go to the "General" tab, select "Shortcuts", then select the "Screen Edges" tab in the right pane. Change the behavior for whichever edge you're having problems with.
