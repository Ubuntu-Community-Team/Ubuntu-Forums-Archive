### Post by fracturedmorals on 2008-02-12
I wouldn't personally recommend using nvidia's installer for their drivers.

If you go to "System > Administration > Restricted Drivers Manager", you can install them via ubuntu's native software management tools.

Or, if you really feel you need the latest dirvers, use [Envy]("http://www.albertomilone.com/nvidia_scripts1.html")
