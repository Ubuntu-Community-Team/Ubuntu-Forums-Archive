### Post by deruberhanyok on 2007-07-09
Hello everyone,

I did some searching on the forums and didn't turn up an answer for this, but poking around the options a little bit more I uncovered it. I thought I'd post it in the hopes that anyone else looking for it can find this post.

Some of you might be familiar with running apps - games, basically - through Beryl or Compiz Fusion. I had been using Beryl and I'd gotten used to being able to press "Ctrl Alt Enter" to make a game go to full screen. Unfortunately this shortcut doesn't seem to be enabled in Fusion by default.

All you have to go is go to the CompizConfig settings manager (under system/preferences), go to "Window Management" and enable the "etra WM actions" plugin. Then you can select the plugin and specify keystrokes to activate the full screen toggle.

Hope this is useful to someone!
