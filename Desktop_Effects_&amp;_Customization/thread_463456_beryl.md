### Post by zero244 on 2007-06-03
Just remember installing and using beryl is not like installing Nero 7x. 
Ive had beryl working great..........not so great.....then not at all.......then working but freezing my machine. Ive never had it running great since I upgraded from ATI to Nvidia.
Right now Im running a Nvidia card.........with every feature turned off........except the themes and Window Shadow.
Window Shadow is the only feature in Beryl that works for any length of time.
If you install it and get it running.......dont do anymore updates.......or it might get broken.
Do any updates before you start the beryl install process then leave everything alone.
Good Luck.
