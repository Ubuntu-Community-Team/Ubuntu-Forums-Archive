### Post by ayoli on 2007-11-15
AWN stands for Avant-Window-Navigator and isn't a window manager, but it requires a composite window manager (ie compiz).

there is also [cairo dock]("http://developer.berlios.de/projects/cairo-dock/") (deb available for 32 bits on the site) which is the same as gnome-dock / kxdocker.
