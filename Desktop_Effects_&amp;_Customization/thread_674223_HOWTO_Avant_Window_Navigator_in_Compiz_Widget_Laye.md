### Post by Shimmy on 2008-01-21
First, install avant-window-navigator, and CompizConfig Settings Manager.

Start avant-window navigator.

Start CCSM (System->Preferences->Advanced Desktop Effects Settings)

Turn on the Widget Layer, and go to the Behaviour-tab.

Type "name=avant-window-navigator" (without quotes) in the Widget Windows text-field.

Now your avant should dissapear, and appear when you press F9.

Hope you like it, i think it's kind of cool :)
