### Post by lopzided on 2007-06-09
i've fixed it.  basically i had to do the same as i had to do for edgy.  i added this to the Device section of my xorg.conf : 

```
Option "TwinView" "True"
Option "TwinViewOrientation" "RightOf"   
Option "UseEdidFreqs" "True"
Option "MetaModes" "1280x1024,1280x1024; 1024x768,1024x768; 1280x1024, NULL; 1024x768, NULL"
Option "UseDisplayDevice" "CRT" 
```

this solved a bunch of problems.  my windows now maximize in one monitor instead of spanning across both, and also my fullscreen games pop up in one monitor, thanks to the NULL sections i added.

w00t!
