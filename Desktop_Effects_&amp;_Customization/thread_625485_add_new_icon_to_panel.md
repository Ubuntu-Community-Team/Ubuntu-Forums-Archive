As in the Quick Launch bar in Windows XP, how can I add an icon in Ubuntu 7.10 to link me to my yahoo mail?  I tried the "add to panel" and the Custom Application Launcher, but it won't work.  Any ideas or comments would be appreciated.  Thanks

### Post by ray bot on 2007-11-28
Add to Panel -> Custom Application Launcher should work fine.  Try one of the following:
Type: Application
Name: Yahoo Mail
Command: firefox [http://mail.yahoo.com/](http://mail.yahoo.com/)

or

Type: Location
Name: Yahoo Mail
Location: [http://mail.yahoo.com/](http://mail.yahoo.com/)
