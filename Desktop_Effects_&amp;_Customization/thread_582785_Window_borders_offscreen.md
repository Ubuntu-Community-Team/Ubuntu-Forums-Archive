### Post by ReloadedFusion on 2007-10-21
Hello, I had this same issue i resolved it by enabling the place window plugin in the compizconfig settings manager.

If you do not have this installed go to the synaptic package manager and install the compizconfig-settings-manager Once this is installed go to system->preferences->Advanced Desktop Settings.

Once this opens go to the Window Management section and enable the place window plugin then exit the program.. It started working for me from this point.

Hope this helps
