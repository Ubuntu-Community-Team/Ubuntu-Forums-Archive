### Post by kwaanens on 2008-03-02
I'm not really sure if this would be the right subforum, but I guess this is especially relevant on a composited desktop.

So, this is my situation:

I just went from Vncviewer to Vinagre to access my desktop computer. I also use FreeNX, but if I want to start some app, and have it run from then on, FreeNX is no good.
However, FreeNX has made me want two things:
1. A resized desktop, making my large display on the desktop shrink to fit on my laptop
2. Disabling Compiz Fusion from the virtual client perspective.

The point here is #2

I'd like to see an easy way of not just changing the window manager, but also replacing AWN with a custom gnome-panel.
As a start I've put in two launchers, one with "openbox --replace" and one with "compiz --replace", but it is a generic and cumbersome solution to a rather simple problem.

Ideally, I'd like to see an applet that lists all window managers (e.g. greyed out when they're not installed). If you choose one, it changes window manager and if the window manager doesn't support compositing, it replaces AWN with a gnome-panel (and also removes other things that may require compositing like widget-ed screenlets). The applet could also interact with synaptic to install other window managers.

Ideally, ideally, :) I'd like to have a feature built into Vinagre that changes all this stuff for me, when I log in to the remote computer.
