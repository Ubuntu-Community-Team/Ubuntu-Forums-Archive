### Post by orange2k on 2007-06-09
Beryl is not available in Automatix, but is available in Add/remove programs.
But it is easy to install via the console, just type: sudo apt-get install beryl beryl-manager emerald-themes

And after that add beryl, beryl-manager and emerald-themes to your startup programs...

edit: what I said implies that you have Feisty installed, I`m not sure how it goes with Edgy...
