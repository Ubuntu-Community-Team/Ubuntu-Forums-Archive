Can someone tell me where the file with the configuration data for compiz-fusion is stored.  I have my desktop set up exactly as I want it but occasionally have tried to run with no desktop effects (System>Preferences>Appearance, set to none), but then when I go back to Custom setting, I find that all my original settings have reverted to the default, and I have to set it all up as I want it again.

There must surely be a file with all my configuration for compiz somewhere on the machine (in ~/ ?)  It is very annoying not to able to just switch the effects off and back on without having to set things up again from scratch.

### Post by FuturePilot on 2008-02-07
By default they're all stored in Gconf. However you can export your settings to a file. Open CCSM and click on Preferences in the lower left corner. Click Export and name the file something and add a .profile extension. Then you can easily import those settings if they get messed up or whatever.
