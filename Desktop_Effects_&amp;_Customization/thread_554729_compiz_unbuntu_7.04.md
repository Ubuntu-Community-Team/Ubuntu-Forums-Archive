### Post by FuturePilot on 2007-09-21
Try this
```
gksudo gedit /etc/X11/xorg.conf
```
add this under the Server Layout section
```
Option "AIGLX" "true"
```
Add this to the Device section
```
Option "XAANoOffscreenPixmaps" "true"
```
And add this to the very bottom of your xorg.conf
```
Section "Extensions"
Option "Composite" "Enable"
EndSection 
```

Save, logout and restart X with Ctrl+Alt+Backspace.
