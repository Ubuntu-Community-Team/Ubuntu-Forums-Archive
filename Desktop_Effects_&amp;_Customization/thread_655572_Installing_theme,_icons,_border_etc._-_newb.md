### Post by Mr Greencastle on 2008-01-02
I'm guessing that you want your desktop to resemble Mac OS X. If so, there is a project called Mac4Lin, which is basically an entire suite of Mac skins etc. It will make it look much closer to a Mac than that picture. 

First, you'll want to download this .pdf that is basically the entire guide to how you will go about changing you're entire desktop to look like a Mac.
[Mac4Lin Guide]("http://sourceforge.net/project/downloading.php?group_id=204373&filesize=4590352&filename=Mac4Lin_Documentation_2.pdf&13714948")

Then you'll want to download the Mac4Lin pack itself. Download to your Home folder is recommended.
[MacLin pack]("http://sourceforge.net/project/downloading.php?group_id=204373&filesize=15418558&filename=Mac4Lin_Part1_v0.4.tar.gz&93056094")

And finally, you will need to download the icon set for the project.
[Mac4Lin icons]("http://sourceforge.net/project/downloading.php?group_id=204373&filesize=14406146&filename=Mac4Lin_Icons_Part2_v0.4.tar.gz&98656345")

Make sure you follow the guide carefully as some parts could be a little confusing. Good luck!

Mr Green
