### Post by lamalex on 2007-06-01
the problem is you're using the fglrx driver with AIGLX, you need to setup and install XGL for it to work. Here is a howto for you: [http://ubuntuforums.org/showthread.php?t=399643](http://ubuntuforums.org/showthread.php?t=399643)

I think your card might support the free radeon driver, so try disabling the proprietary ones and then doing beryl, that might work. Also you need to disable composite if you're going to use XGL.
