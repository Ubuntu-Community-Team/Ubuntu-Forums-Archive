### Post by Therion on 2008-03-14
Enable the following plugins:

- Desktop Cube
- Rotate Cube
- Viewport Switcher
- Cube Caps 

Navigate back to General Options &#8594; Desktop Size &#8594; Horizontal Virtual Size to 4.  
The other two options have to be left at 1.

Try it now with Ctrl+Alt+Left Mouse wheel and moving your mouse about.
