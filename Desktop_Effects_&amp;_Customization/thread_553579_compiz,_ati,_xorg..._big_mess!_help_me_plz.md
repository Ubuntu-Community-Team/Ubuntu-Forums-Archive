### Post by Forlong on 2007-09-18
You could try "cleaning up" your xorg.conf (please do all of those steps in your regular session, _not_ Xgl)

Do a backup of your current xorg.conf
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```
Then run this:
```
sudo dpkg-reconfigure xserver-xorg
```
(choose **ati** when asked for a driver)

Afterwards, open your xorg.conf being root:
```
sudo gedit /etc/X11/xorg.conf
```
and change this line
```
	Driver		"ati"
```
located in the *Section "Device"* to
```
	Driver		"fglrx"
```
And add this to the very end of the file:
```
Section "Extensions"
        Option          "Composite"     "0"
EndSection
```
Then save and reboot. Let me know if it worked.
