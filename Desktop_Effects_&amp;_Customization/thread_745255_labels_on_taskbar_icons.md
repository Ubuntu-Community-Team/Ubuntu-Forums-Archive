### Post by GOROSSI on 2008-04-04
You can try the following and see if that suits your needs.


Right click on the panel, then select "Add to panel" this will open a dialog box Scroll Down to "Desktop & Windows" and Select "Window Selector" and then Add it to the panel.

It will display the Icon of the program that's currently the active window.

E.G. if your browsing the web in Firefox Its Icon will show. click on it and it will give you a description of the current window and any inactive windows will be underneath.

very long descriptions will still not fully display though Which can be an issue but it's usually only the program name that missing so you could just go by the Icons for that.

I use it myself and found it does the job

Hope this helps
