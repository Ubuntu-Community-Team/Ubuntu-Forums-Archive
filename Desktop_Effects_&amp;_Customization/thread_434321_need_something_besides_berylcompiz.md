### Post by notwen on 2007-05-06
if you want really light-weight desktop stats + some other random things. You may want to try conky.  There's plenty of threads on here talking about conky, installing it, sharing different configs, check it out. =] 

try this in terminal
```
sudo apt-get install conky
```

then type this in terminal to start conky
```
conky
```

here are 2 helpful threads
[share your conky config]("http://ubuntuforums.org/showthread.php?t=281865")

if you want to try and compile use this [link]("http://ubuntuforums.org/showthread.php?t=353362")
