### Post by Jelmoh on 2008-01-25
You can go to Synaptic (System -> Administration -> Synaptic)
And install the following packages:
Compizconfig-settings-manager
Compiz-gnome
emerald (this is actually the theme manager)
gnome-compiz-manager

That should do the trick.. :)
Easiest to search for 'compiz' (without quotes! ;))

If you've installed these things restart (you probably won't need to..)
Then go to System -> Preferences -> Emerald Theme Manager
There you can load the theme you've downloaded.. :)

Any more Q's, just get back here! ;)
Good luck! :D

Edit: With 'Advanced Desktop Effects Settings' you can configure compiz :)  (System -> Preferences -> Advanced Desktop Effects Settings)
