### Post by psyopper on 2007-07-19
Change the line that says
```

Option         "AllowGLXWithComposite" "Off"

```

with 
```

Option         "AllowGLXWithComposite" "On"

```

Save the file.

After making the change you will need to restart X. THis is done simply by Ctrl+Alt+Backspace. This will log you out of your current session as well (save any unsaved work). Let us know what happens...

Of course I would be remiss if I didn't include the obligatory back up your current xorg.conf so you can revert if necessary.

```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup1

```
