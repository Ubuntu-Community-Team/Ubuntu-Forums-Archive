### Post by Praill on 2008-01-12
I couldn't find a post on here screaming for joy about the new ati proprietary driver so I'm going to make one.

FINALLY... composite extensions can be enabled on x1000 cards allowing you to run beryl/compiz without xgl and still have direct rendering!!!!

I know I'm one of about 2 million people that has been waiting at least a year for this. The quickest way to get the new driver, if you havent already, is to use Alberto Milone's script.

Go to: [http://albertomilone.com/nvidia_scripts1.html]("http://albertomilone.com/nvidia_scripts1.html") , download and install the ubuntu package, and then from a terminal run envy -g or envy -t (graphic or textual front end).
I don't believe this will work on any kernel below 2.4 which means you need to be running at least feisty fawn.

WOOOHOOOO!!
