### Post by smartboyathome on 2007-11-04
You have to install CompizConfig-Settings-Manger. Once that is installed, go to System > Preferences > Advanced Desktop Effects, and choose to use the "desktop cube" (ignore the warning about disabling desktop wall). You can't use 3D windows on the cube unless you install Compiz from their development GIT source package.
