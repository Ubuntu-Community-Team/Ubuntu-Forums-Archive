### Post by hpalma on 2007-11-01
I worked around this by installing compizconfig-settings-manager. Then open the advanced desktop effects settings and open the viewport switcher plugin. In the actions tab you will be able to set the keyboard shortcuts to all the workspaces.

What i still haven't figured out is how to set the shortcut to the "Move Window to Workspace" action for more than the first 2 workspaces.
