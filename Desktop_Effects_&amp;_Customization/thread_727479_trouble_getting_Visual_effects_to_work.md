### Post by mrmiserable on 2008-03-17
this is common problem if you are using fglrx driver 

so open terminal

sudo gedit /usr/bin/compiz

find the line like this add fglrx to the line

# Driver whitelist
WHITELIST="nvidia intel ati radeon i810 fglrx"

change to look like this then save/close and try to start compiz again

hope this helps
