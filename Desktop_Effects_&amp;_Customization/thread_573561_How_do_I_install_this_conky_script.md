### Post by llamakc on 2007-10-11
You have to read my whole post comprehensively.

HOW are you looking in the folder that you downloaded and uncompressed? With the GUI? Are you using GNOME?

If you are using Gnome and you downloaded the archive to your Desktop, open the archive and hit the key-combo "CTRL h" to show the hidden files. That means actually press the CTRL key and the lowercase h key at the same time.

now you'll see the hidden dot-files. In the Terminal you would view these with the command "ls -a". Move the .conkyrc file to your home directory. Do you know how to do that?
