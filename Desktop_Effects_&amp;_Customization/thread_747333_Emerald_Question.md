### Post by Tomatz on 2008-04-06
No you just need to highlight it (in emerald).

_To use emerald:_

Open a terminal and type:

**ccsm** *(note if the comand is not found type **sudo apt-get install compizconfig-settings-manager** to install Then type **ccsm** again) *

open the **window decoration** plugin (in ccsm)

in the command box type:

**emerald --replace**

close **ccsm**

**ctrl alt bkspace**

Done ;)
