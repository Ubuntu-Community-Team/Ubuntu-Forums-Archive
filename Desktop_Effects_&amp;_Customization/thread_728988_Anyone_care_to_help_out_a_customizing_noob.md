EDIT:This thread was pretty helpful:  [http://ubuntuforums.org/showthread.php?t=675919&highlight=gtk2-engine]("http://ubuntuforums.org/showthread.php?t=675919&highlight=gtk2-engine")

Additionally, you can install themes by searching in Synaptic Package Manager for "gtk2-engines" and installing as many as you like.  Then, if you go to Appearance, you can click "Customize" and change themes based on the stuff you downloaded.

Apparently Emerald is some other kind of border ... engine.  The default one is "Metacity" I think.  I'm still learning more about this as I go, but maybe this will help you for now :-)

Yes, Emerald is exactly that. It's just another engine to draw the borders around your windows, that has a few more options (okay, a lot more) than Metacity (it is also much more resource heavy, though). You can install it through Synaptic and try to play around with it.

If you guys haven't already, check out [http://www.gnome-look.org/](http://www.gnome-look.org/) for all your theming pleasures.
