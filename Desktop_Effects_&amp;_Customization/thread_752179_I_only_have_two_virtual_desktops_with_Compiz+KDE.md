### Post by russo.mic on 2008-04-11
If you only have 2 desktops, you need to install compizconfig-settings manager package first,

so search for in in synaptic, or use apt-get or whatever.

Then go to general ---> Desktop Size ---> Virtual Horizantal Size

That number is the number of desktops you have.

Good Luck!
