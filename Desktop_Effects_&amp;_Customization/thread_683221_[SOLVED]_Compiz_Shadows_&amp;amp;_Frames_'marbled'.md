### Post by merlyn on 2008-01-30
Compiz has been working perfectly, as it has since I upgraded to Gutsy when it was first released.

Last night however, I shut down my PC for a while & when I booted back in many of the Compiz features were no longer functioning.

As it turned out, the compiz-fusion-plugins-extra & compiz-fusion-plugins-main packages had been removed from my system.

I did not remove them myself and can only presume it happened because they are both listed as auto removable in synaptic.

Don't know why exactly, but it hasn't caused a problem before.

Anyway I reinstalled both packages, which enabled access to all the plugins I'd used before.

Problem solved.

However the next time I logged all shadows now appear 'marbled', and window frames appear marbled when moved also.

Screenshots attached below.

Any suggestions, would be greatly appreciated.

Cheers.
