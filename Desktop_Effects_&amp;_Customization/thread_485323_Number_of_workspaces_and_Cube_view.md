### Post by rishabh on 2007-06-27
First of all, are you using Compiz or Beryl?

If you're on Compiz, you can install the "Emerald Theme Manager" by typing this in a terminal:

```
sudo apt-get install emerald emerald-themes
```

Then, when you want to start Compiz, use

```
compiz --replace -c emerald &
```

It will now load the window decorations from Emerald. If you want to change the theme, go to System->Preferences->Emerald Theme Manager.

If you're on Beryl, you should already have emerald, in which case it's probably a rendering issue. Run:

```
cat /etc/X11/xorg.conf
```

and paste the output here, it might help.

### Post by hyperair on 2007-06-28
Actually you should just start beryl-manager and let beryl-manager handle the starting up of Beryl. Beryl on its own does not start the window decorator (and hence no borders). So if you don't want the beryl-manager, then you have to start Beryl and then start Emerald, which is the Beryl window decorator. Easiest way of course, is using the Beryl Manager.
