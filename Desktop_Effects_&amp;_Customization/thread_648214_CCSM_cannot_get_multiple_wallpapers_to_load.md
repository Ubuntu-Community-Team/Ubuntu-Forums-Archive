### Post by 0g_Trans_Fat on 2007-12-23
I think it depends on what Desktop Environment you are using, if you are using gnome, there is no quick-fix to get multiple backgrounds, unless you disable nautilus from showing it, but then you cannot have any icons on your desktop.  
To do this, type 'gconf-editor' in terminal, then go to apps>nautilus>preferences> untick show desktop, and try restarting.

If you are using KDE, i think there is some setting that you can change to enable multple wallpapers but i am not sure because i am a gnome user.

There is also a program called [wallpapoz]("http://wallpapoz.akbarhome.com/") which will change the background when you switch workspaces, but there is a 1-3 second delay.
