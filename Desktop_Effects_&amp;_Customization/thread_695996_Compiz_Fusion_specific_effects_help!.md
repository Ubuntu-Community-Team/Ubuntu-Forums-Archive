### Post by JECHO on 2008-02-13
i think i can help you out...


first: to get the minimize effect (called magic lamp) you must open the compiz config settings manager, scroll down to effects > animations > then you will see a bunch of tabs on the top that say things like: close animation, effects settings, focus animation, etc... click the tab that says minimize animation.  once youre there you will see a list of items (probably only one item). double click that item (if there is indeed a list of multiple items, select the first one by double clicking it). A small "edit" window should pop up. once youre there select the drop down menu for "minimize effect". there you should see a bunch of different animations to select from. The one you want is the "Magic Lamp".  select it and click ok (if you want your windows to minimize super slow or super fast you can do this by changing the sliders position next to "duration" but the default looks works just fine). once youve done that, you can test it out by minimizing the compiz config manager window or whatever. then close the manager and youre set.


second:to get the "burning" effect of closed windows you need to once again open up compiz config settings manager, scroll down to effects > animations > then you will see a bunch of tabs on the top that say things like: close animation, effects settings, focus animation, etc... by default the manager auto selects the "close animation" tab when the animations edit area is opened so all you need to do is double click the first item in the list. an edit window should appear. just like before you select the drop down menu and choose the "burn" animation. click ok and close compiz config settings manager and whaala burrrnnn :D

NOTE: you can change the colors of the fire as well as the direction it burns... if youre curious as to how, reply asking and i will gladly help.



as far as the screenlets... i cant help you there. i dont use screenlets. sorry

### Post by iNick3 on 2008-02-13
The layering effect is done by the 3D windows plugin.  It says in the compiz fusion wiki that it's included in the plugins-extra package, but I have plugins-extra and I don't have the 3D windows plugin.  You might want to check to make sure that you don't already have it.  If you don't, then you can follow the tutorial here to get it. 

[http://forum.compiz-fusion.org/showthread.php?t=5303](http://forum.compiz-fusion.org/showthread.php?t=5303)

     Once you have it, just find it in the compiz-config window and turn it on. A list of what all of the options do can be found in the compiz-fusion wiki.

[http://wiki.compiz-fusion.org/Plugins/Cube#head-dcbb10c4fb334b8df8319d28379fe3245072c87a](http://wiki.compiz-fusion.org/Plugins/Cube#head-dcbb10c4fb334b8df8319d28379fe3245072c87a)
