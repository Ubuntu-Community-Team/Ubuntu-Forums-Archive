### Post by MNICY on 2007-12-17
Well you install compiz first
(copy into a TERMINAL)
```
sudo apt-get install compizconfig-settings-manager
```

Then go 
SYSTEM->PREFEENCES->ADVANCED DESKTOP EFFECTS

when it opens up go to GENERAL OPTIONS


Go to DESKTOP SIZE and increase "horizontal" to 4 (or other numbers above 1. 2 is a flat plane, while 4 is a cube)

Go back and enable DESKTOP CUBE and ROTATE CUBE

Alt-Ctrl and click should initiate the rotate :)

To paint fire on the screen, go to System>Preferences>Advanced Desktop Effects.  Scroll down to "Paint Fire on the Screen" and check the box to enable it.  You can then exit the window.  To use the feature, you hold down the "super" button and the shift key while you drag with the left mouse button held down.  To clear the fire from the screen, just push shift, super, and hit the c key.   Enjoy
