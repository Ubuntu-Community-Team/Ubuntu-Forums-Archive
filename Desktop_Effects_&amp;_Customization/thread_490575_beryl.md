### Post by FuturePilot on 2007-07-02
If you're running Feisty you can get it from the repos. ```
sudo apt-get install beryl beryl-manager emerald-themes
```
Beryl is a composite manager that gives you a 3D desktop. You can get transparencies, drop shadows, wobbly windows and much more. But it requires you to install the proprietary graphics card driver. You can do it easily with the Restricted Driver Manager. And since you have a Nvidia card you're in good luck since Nvidia gives Linux some pretty good drivers. Although they're closed source.
