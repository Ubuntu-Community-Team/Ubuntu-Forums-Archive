Yeah, if you want AWN to look like Leopard's just goto Applications>System Tools>Configuration Editor

Goto apps>avant-window-navigator>bar and change the value called bar angle and enter 45 and change the value of icon offset to 15.
Isn't amazing of how flexible and powerful Linux is?
