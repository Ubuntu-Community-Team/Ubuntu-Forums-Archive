### Post by lumberjack on 2008-03-26
This is fine.  The GLDesktop options are basic controlls for the desktop effects.  Under the third tab you'll find an "effects" section, and then choose "cube and rotation".  That should work.
On a second note, If you're looking for more control and advanced plugins, install CompizCongfig-Settings-Manager (ccsm).  It allows you to turn on specific plugins, including all of the neat effects.  I've found the ccsm to be far more helpful than the GLDesktop, and I've found that the GLDesktop manager can cause my computer to become unstable.
