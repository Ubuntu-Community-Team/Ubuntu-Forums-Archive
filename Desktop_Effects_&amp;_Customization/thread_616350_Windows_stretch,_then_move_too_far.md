### Post by thelatinist on 2007-11-19
There are two possibilities: either it is caused by the "Snapping Windows" plugin, which allows windows to be attracted to the edges of the screen/other windows, or it is caused by the "inverse snap" setting in the wobbly windows plugin itself.  Try disabling each and seeing what happens.
