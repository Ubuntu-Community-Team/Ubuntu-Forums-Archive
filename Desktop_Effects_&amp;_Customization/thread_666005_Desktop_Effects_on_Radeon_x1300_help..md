### Post by natestone on 2008-01-27
I had this exact problem also with a Radeon X1300.  All I had to do to fix it was install "xserver-xgl".

I used the Synaptic Package Manager ("System" => "Administration" => "Synaptic Package Manager" and then searched for 'xserver-xgl'.

After it installs, just reboot and right-click the desktop to "Change Desktop Background" => "Visual Effects" and enable the effects.

Then you can set your preferences.
