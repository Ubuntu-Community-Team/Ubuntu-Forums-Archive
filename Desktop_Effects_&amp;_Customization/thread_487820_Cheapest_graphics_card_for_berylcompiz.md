### Post by phidia on 2007-06-29
If you're looking for an AGP slot card they are  being replaced by pci-e cards. Nvidia still seems to have the edge for ease of setup in linux/ubuntu so an AGP slot GeForce 6XXX series can be had for between $40-50.

### Post by hyperair on 2007-06-29
If you're getting a new PC, just get one with an integrated Intel card. That'll definitely work with Beryl and Compiz.

I'm using an old nVidia Geforce 4 MX 440 card and it works with both. My other comp, using an nVidia Geforce 2 MX also works with Beryl. The water plugin doesn't work though, but frankly speaking you aren't going to be using it much.
