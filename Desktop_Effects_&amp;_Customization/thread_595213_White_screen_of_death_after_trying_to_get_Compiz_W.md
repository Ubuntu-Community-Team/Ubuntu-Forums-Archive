### Post by senmonjin on 2007-10-31
Hi, i finally managed to have the drivers runing ok.
A friend of mine had the same problem and helped me a lot with the next code.
he told me that some links to fglrx where broken and that i had to link it.
The code he told me to use after instaling the drivers is this:


$ sudo rmmod fglrx

$ cd /lib/modules/*/misc

$ sudo insmod fglrx.ko

$ sudo mkdir /lib/modules/$(uname -r)/volatile

$ sudo ln -sf /lib/modules/$(uname -r)/misc/fglrx.ko /lib/modules/$(uname -r)/volatile/fglrx.ko


after that, i configured the driver with 

$ sudo aticonfig --initial --force

$ sudo aticonfig --overlay-type=Xv


Thanks to Kanamor,
