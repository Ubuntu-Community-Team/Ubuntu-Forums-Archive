There's the concept of desktop apps (see karamba/superkaramba and others). If you're using beryl, the integration is not as seamless at this point (basically, Minimize All minimizes desktop apps as well). It's a pretty hot topic, so I hope to see improvements in this area.

OTOH, there are tons of really cool desktop apps - take a look at kde-apps.org, for example.

### Post by andrewsomething on 2007-06-14
gDesklets is another program which gives you desktop apps. It's very easy to install and use. You can get it right from Add/Remove. But like the programs mentioned by nyvalbanat, it doesn't always play well with Beryl. Although I know there have been some work arounds posted in the fourums. The key seems to be setting gDesklets to start before Beryl does...
