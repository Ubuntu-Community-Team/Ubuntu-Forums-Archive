### Post by brucewestfall on 2008-02-25
If you are running without any effects enabled, Go to:

System>Preferences>Keyboard Shortcuts
( I usually resize the small default window )

About 3/4 down is:
Move between windows with popup

Select that and when the text on the right says "new accelerator" use your keyboard and push alt+tab or something entirely different if you choose.

If you don't have CompizConfig Settings manager, that has ALOT of customization options.  When I was using it ( Compiz does not work with the Blender 3D program that I use constantly. ), I could change the pixel settings of the preview windows.  Try it at 300x200, then 800x600 for a while, then absolutely anything else your brain can imagine.

Just make sure once you open CompizConfig you have about an hour free - it's very addicting.
