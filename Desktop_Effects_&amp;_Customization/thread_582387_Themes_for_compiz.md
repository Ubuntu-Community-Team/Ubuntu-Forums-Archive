### Post by neymac on 2007-10-19
There is emerald-themes (windows decorator) for compiz fusion.
Install emerald using synaptyc (or apt-get install) and search for emerald-themes at [http://packages.ubuntu.com/feisty/x11/emerald-themes](http://packages.ubuntu.com/feisty/x11/emerald-themes) (it works for gutsy).
