Hello Mate

I had this same problem a while ago, found the answer and now can't find it again.

The only thing I remember is that you need to open a terminal (command line window) to use as a rescue thing.

You do this by pressing I think Ctrl and any function key.

You need to edit the GConf file controlling whether Gnome is using the Desktop Effects.
This is where I draw blank as I hadn't written it down thinking I wouldn't be stupid enough to try turning the Desktop Effects back on, but I did. Luckily I have two installs on the PC.

There are other relate files like Gdm.conf that may be worth looking at too.
