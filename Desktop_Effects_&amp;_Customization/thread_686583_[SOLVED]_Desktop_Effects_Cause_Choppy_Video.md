### Post by iamninja on 2008-02-03
A friend of me had about the same problem with you. We fixed it this way:

1) Run gstreamer-properties on a terminal
2) Click on the video tab
3) in Plugin select X Window System (No Xv)
 
The solution is not mine i found it on this forum here [http://ubuntuforums.org/showthread.php?t=615539](http://ubuntuforums.org/showthread.php?t=615539)
