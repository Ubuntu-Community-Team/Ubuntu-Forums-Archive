### Post by Depressed Man on 2007-07-18
They're compistion managers (ack spelling). The most well known one is the desktop cube (with 4 different desktops in the shape of a cube). Though they can do other things. 

I believe AWN (and Kiba Dock) both rely on having at least one installed composition manager for them to work properly.  

Compiz Fusion is the newest one (as a result of Beryl and Compiz coming back together). You can find out  how to install it from here. [http://ubuntuforums.org/showthread.php?t=481314](http://ubuntuforums.org/showthread.php?t=481314)

If you add Trevino's repository to install Compiz Fusion you also get the benefit of using that same repository to install kiba dock with a sudo apt-get install kibadock command.
