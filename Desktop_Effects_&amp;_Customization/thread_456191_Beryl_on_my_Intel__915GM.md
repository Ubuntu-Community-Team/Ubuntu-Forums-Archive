```

```

I use that on my laptop and it has a 915 chipset as well (I use Feisty though... I never installed Beryl on Edgy).

### Post by audax321 on 2007-05-30
Alright, I think I might have screwed up and given you my other computers config hehe. You still have to add a line, but not that one... here are the relevant sections from my xorg.conf (from my laptop this time, sorry about before):

```

Section "Module"
	Load	"i2c"
	Load	"bitmap"
	Load	"ddc"
	Load	"dbe"
	Load	"dri"
	Load	"extmod"
	Load	"freetype"
	Load	"glx"
	Load	"int10"
	Load	"vbe"
EndSection

Section "Device"
	Identifier	"Intel Corporation Mobile 915GM/GMS/910GML Express Graphics Controller"
	Driver		"i810"
	BusID		"PCI:0:2:0"
	Option		"XAANoOffscreenPixmaps"	"true"
EndSection

Section "Extensions"
	Option	"Composite"		"true"
EndSection

```

So for your config you want to make sure you have the Load "dbe" line in the Module section, the Option "XAANoOffscreenPixmaps" line in the Device section, and the entire Extensions section. That should hopefully get Beryl up and running.
