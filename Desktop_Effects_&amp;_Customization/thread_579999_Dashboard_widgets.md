### Post by Redenbacher on 2007-10-18
Yup, in the CompizFusion setting manager, enable the Widget Layer plugin. Then download and install Screenlets, launch the screenlets from the screenlets manager in preferences, and set them as widgets on right click -> window -> Set as widget
