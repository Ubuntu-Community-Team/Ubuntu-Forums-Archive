> **KhaaL said:**
> I've tried changing my mouse theme through system &#8594; preferences &#8594; appearance &#8594; customize. however this makes my current mouse theme (which is placed under ~/.icons) to look like it should be in certain apps only (firefox, amarok...) while hovering it over the desktop and nautilus makes it look plain again.

You need to restart X (perhaps even reboot, not sure) before all cursor-changes takes effect.

EDIT: if using Compiz you also need to open CCSM, go to "General Options" and type in the name of your cursor theme in the "Cursor theme" box in the first tab.
