### Post by orange2k on 2007-04-21

---

### Post by haricharan on 2007-04-21
Goto System->Preferences->Sessions. In startup programs add beryl-manager, beryl and emerald.

---

### Post by bran on 2007-04-21
hrmm beryl is started at startup here once it was installed but if you want to have the beryl manager start go system > preferences > sessions > startup programs >new and enter beryl-manager as name and command and it will be there.
