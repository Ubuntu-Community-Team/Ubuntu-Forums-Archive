### Post by Ender Black on 2007-05-06
This is probably wrong but it works.  Right click on Applications and select Edit Menus.  Click on System Tools and put a check mark next to Beryl Settings Manager (Simple).  Then launch Beryl Settings Manager (Simple) and it has a very nice GUI for setting up your hot-corners.  I was very frustrated till I found that by accident.
