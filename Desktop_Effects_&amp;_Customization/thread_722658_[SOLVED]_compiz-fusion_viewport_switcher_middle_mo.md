### Post by DarthRaccoon on 2008-03-12
I have the viewport switcher plugin enabled in ccsm for compiz-fusion, and I really like the ability to hold down the middle mouse button and rotate the cube to my different desktops.  however, I would like it alot more if I could figure out some way to make that key/button binding universal, meaning that even if I have the mouse focus over a given window (like say firefox) when I use the middle button it will still rotate the cube.  As it is now it will only rotate if I first move the mouse to a portion of my desktop.  There are several apps I tend to run maximized on a given desktop, which then makes it impossible for me to rotate using my mouse, so I have to use ctrl+alt+arrow to switch.  Does anyone know if its possible to change this?  thanks in advance.

---

### Post by BluntBox on 2008-03-13
Try setting the keybind up in the Rotate Cube plugin instead. Set the initiate bind to middle mouse and it should allow it no matter what your mouse is hovering over.
