### Post by Gus_T_Reer on 2007-09-29
Well, ok the colour on the notifications is tied in with other colour settings and I don't know whether it can be changed independently but I have managed to change the colour by going to 
System -> Preferences -> Theme -> Customise
and changing the colour for Colours:Selected Items:Background

This changes the colour of a number of items including tooltips, highlighted menu items, rollover highlights in apps to name but a few. The overall effect is more integrated now but it would have been nice to have been able to specify colours independently.
