### Post by patryk77 on 2007-11-21
[http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide)

This guide is great for running an x1600.

In the terminal you have to set the ati-driver-installer-8.42.3-x86.x86_64.run to be executable before you run it (where it says "create .deb packages")

```
chmod +x ati-driver-installer-8.42.3-x86.x86_64.run
``` will do that for you

Also, do not install the xserver-xgl package, as it seems to break 3d rendering.

You want to use AIGLX instead, which will happen if you follow the tutorial properly
