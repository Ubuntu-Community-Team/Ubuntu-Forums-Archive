### Post by funkelauge on 2008-03-24
To all fellows from Kanton Aargau in Switzerland:

there is a good howto for installing Easytax 2007 AG in Ubuntu under: [http://www.lugs.ch/lib/doc/easytax06/index.phtml.de](http://www.lugs.ch/lib/doc/easytax06/index.phtml.de)

eventually it says its not available in your language, then you have to press the german flag (Schwarz-Rot-Gelb)

IMPORTANT: to execute the installer script I had to deactivate the compiz desktop effects. System->Einstellungen->Erscheinungsbild, then tab "Visuelle Effekte", select "Keine".
I have Easytax 2007 running on Gutsy Gibbon now.

If you stumble over other pitfalls please post here.

ps. that software is proprietary, however there are no alternatives if you want to fill your tax bill in the Kanton Aargau :(. The only option would be to use the program under windows :mad:

Cheers, David
