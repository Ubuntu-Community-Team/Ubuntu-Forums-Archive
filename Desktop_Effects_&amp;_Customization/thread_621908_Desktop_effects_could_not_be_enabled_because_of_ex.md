### Post by ronmarley1 on 2007-11-24
I had this same problem with my laptop with Intel integrated graphics, although I do have an older graphics card than you.
A little background:
I upgraded from Fiesty to Gutsy and everything worked fine using an external monitor (actually a LCD projector-I teach high school).  This was all with the i810 driver.  Then, I did a clean install of Gutsy, and it used the Experimental Intel driver, which is listed as "intel" in the xorg.conf. file.
To fix the problem, I changed back to the i810 driver using the new Screens and Graphics app. under System -->Administration.  I then rebooted, and everything is fine, using Compiz-Fusion on the external screen.
Hope all this helps.  Good luck!
