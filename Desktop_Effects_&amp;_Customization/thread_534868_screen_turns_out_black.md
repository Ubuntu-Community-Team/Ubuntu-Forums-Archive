### Post by jonathanmotes on 2007-08-25
I have had problems with this also, but only when running a effects compositor (compiz or beryl) with the restricted nVidia driver. My problem is the result of a bug in the nVidia driver that is supposed to be fixed in the next release of the driver. It should be released sometime later this year.

Are you just using the Compiz Desktop Effects included with Feisty? The reason I ask is because, in my experience, Beryl Effects are very good at finding the driver's weak spots. For me, Beryl would only allow about 3 or 4 windows to be open at a time before they would  start going black, while the generic Desktop Effects would usually allow 10 or more windows before triggering the bug. Right now I am using Compiz Fusion and I have so far not had any real problems with the bug. I followed the tutorial [here]("http://ubuntuforums.org/showthread.php?t=481314&highlight=updated+compiz+fusion") to install it.

Hope this helps!
