### Post by Ub1476 on 2008-02-04
Hi, gDesklet is sort of old and unmaintained now. If you want desklets, try [Screenlets]("http://gnome-look.org/content/show.php/Screenlets+0.0.12+last+build+169+stable?content=73346") (download the deb package). I don't think there are any screenlets which has a starter-bar, but you can always create a new panel, add launchers to it, the remove the "expand" option in properties.

### Post by pritamps on 2008-02-04
For screenlets, visit [www.gnome-look.org]("http://www.gnome-look.org") and search for screenlets. the newest version is hosted there along with installation instructions.

Some options to get your zoom on mouse over effect :

1. Avant Windows Navigator -> Really nice. I've been using it for long now. But you need a composited screen...
2. Cairo Dock -> I think this one doesn't need compositing.

You could even go ahead and try Geubuntu Desktop (now called Open GEU I think). It's by far the coolest thing I've seen on linux so far :)
