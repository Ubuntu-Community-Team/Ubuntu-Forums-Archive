### Post by Inxsible on 2007-09-01

```
Change Horizontal Virtual size = 4 
Vertical Virtual Size = 1
Number of Desktops = 1
```EDIT: Whoops, just noticed that you use beryl, but you can look for the same option, it should be in general options too.
