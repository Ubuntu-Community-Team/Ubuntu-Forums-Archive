### Post by speaker219 on 2007-06-26
I had the same problem and i found the solution. Go to System > Preferences >  CompizConfig Settings Manager 
Find the plugin called "Put Window" (it should be near the bottom under "Window Management")
Make sure the plugin is enabled.
Go into the settings for the plugin and make sure "Workarounds" is checked and the "Placement Mode" is on "Smart"

Hope that helps!
