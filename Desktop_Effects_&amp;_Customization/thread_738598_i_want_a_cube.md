### Post by Ub1476 on 2008-03-28
Do you have Visual Effects/Compiz-Fusion enabled? If it does not work, post the output of:

```
lspci | grep VGA
```

If Compiz is working fine, install Advanced Desktop Effects available in add/remove or:

```
sudo apt-get install compizconfig-settings-manager
```

Remember to set Visual Effects to custom.
