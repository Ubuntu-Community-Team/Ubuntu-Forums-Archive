### Post by starcraft.man on 2007-05-08
Beryl most definitely offers opacity, contrast and saturation controls for how windows display, as well as hundreds of other options.

If you have Nvidia, I'd send you [here.]("http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Beryl_.28Nvidia.29") ATI [here.]("http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_Beryl_.28ATI.29")

(Just being sure to cover for ATI folks too atikers)

Just a note atikers, I dunno, I don't find Beryl to be that Beta... I leave it on most of the time, when I'm doing lots of active work and moving windows around like crazy. Only time I've turned it off of late is when I wanted to max my free cpu cycles for gaming and when I was doing other super intensive tasks. Beryl hasn't crashed on me for some time.

Note: If you don't want it to run at startup, don't set it in sessions, otherwise the guides are good to follow.
