### Post by Howler9443 on 2007-07-03
There are a lot of really nice themes on [www.gnome-look.org](www.gnome-look.org) if you are using Ubuntu. If you are using Kubuntu there is [www.kde-look.org](www.kde-look.org).

I myself am not into the whole brown/orange/earthy colors, but it makes sense for the distro.

At least you can change the things you don't like. :-)

Not to tout one flavor over another, but you may want to try Kubuntu. It has a nicer default color scheme and has the great foundation of Ubuntu (of course) under the hood. The only difference is that the UI is KDE.

It all depends on what you like. Perhaps you can install Gimp and create a new default theme? ;-)

Good luck!
