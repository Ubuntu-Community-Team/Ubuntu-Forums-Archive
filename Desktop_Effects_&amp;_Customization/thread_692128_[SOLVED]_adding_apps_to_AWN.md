### Post by zeller on 2008-02-09
One way is to drag and drop into the AWN bar. If it's something already installed and it's listed on your program list, drag 'n' drop.

If it's not, I suppose you just do it the same way you make a launcher. Find the correct path to the program you want to add and insert that as the path. Then find an Icon you like.

### Post by Tenken on 2008-02-09
The command to open rhythmbox from the command line is just

```
rhythmbox
```

If you want to learn more about the command line try [this tutorial]("http://www.tuxfiles.org/linuxhelp/cli.html").
