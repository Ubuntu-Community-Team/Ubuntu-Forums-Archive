### Post by praet on 2007-09-12
This is the current behavior of the gnome panel window list and cannot be changed without code modification.  The window button size is dependent on the length of the title bar text.  If you want to have all windows in the bar the same size then i suggest you remove the bottom panel and replace it with the xfce-panel.  

Also see this: [http://ubuntuforums.org/showthread.php?t=333094](http://ubuntuforums.org/showthread.php?t=333094)  for info on running gnome applets in xfce panels.
