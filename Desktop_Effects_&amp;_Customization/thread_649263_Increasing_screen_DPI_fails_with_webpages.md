### Post by joshuachad on 2007-12-24
Web browsers i.e. mozilla will use its own settings to render fonts. So there is two way to change the size. Easiest way first is in firefox click tools --> options and select the content tab then click the advanced button. Also if you want uncheck allow pages to use their own fonts to force firefox to use the ones you select. Select the minimum font size you want. Then you can change the font size there. Another way it to type about:config in the address bar and change it there.
Hope this solves your issue.

Cheers
