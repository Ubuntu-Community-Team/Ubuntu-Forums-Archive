### Post by hyperair on 2007-06-28
Frankly speaking, their requirements are the same.

Exact requirements I don't know, because I've never seen it anywhere, or I've forgotten already, but it's very modest. Also, I don't have a system that can't run Beryl/Compiz and the oldest I have is a Pentium 4 1.5GHz, 512 MB RAM, nVidia Geforce 2 MX. Obsolete, definitely, but it runs. I also happen to know that there are older systems that will run Beryl/Compiz.

Just make sure your graphics card has OpenGL support, and get proper drivers for it (especially nVidia and ATi). RAM-wise, if it can run Windows XP, it can run Beryl (I think). Same goes for the CPU.
