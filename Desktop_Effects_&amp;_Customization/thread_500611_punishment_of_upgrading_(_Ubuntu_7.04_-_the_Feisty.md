### Post by hyperair on 2007-07-14
Well when you were upgrading you had the option to replace the xorg.conf with a new one or keep the old. You should have kept the old. That's what I did. 

Anyway, go look around your /etc/X11 folder and see if you can find the backup file. Then just replace the current xorg.conf file with that file.

If you don't see any backups there, you could do something like this in the terminal:
[code]
sudo dpkg-reconfigure -plow xserver-xorg
[/code
and just answer all the questions. It'll fix everything up for you =)

p.s. Don't forget to backup your xorg.conf file before running that command.
