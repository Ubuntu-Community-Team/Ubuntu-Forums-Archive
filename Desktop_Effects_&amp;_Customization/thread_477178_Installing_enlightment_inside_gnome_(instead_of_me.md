### Post by brim4brim on 2007-06-18
[http://www.ubuntugeek.com/howto-install-e17-enlightenment-desktop-in-ubuntu.html#more-179](http://www.ubuntugeek.com/howto-install-e17-enlightenment-desktop-in-ubuntu.html#more-179)

Your better off installing it as its own login session so you can select it but go back go Gnome if everything goes haywire.  You can make Enlightenment your default session so you always log in to it and then can log into Gnome if something goes wrong with enlightenment.

Enlightenment still has bugs (I found out the hard way) so its best to have something to go back to, luckily I did.
