### Post by regal_dunce on 2007-04-27
The .themes directory is in your home directory.  The dot in front of the "themes" means that it's a hidden file.  In a terminal, type ls -a to see all hidden files.  In your file browser, hit Ctrl-H, or go to View -> Show Hidden Files.  Drag and drop!  You may have to hit "install theme" in your theme manager, and if you don't see a .themes directory in the file chooser, just right click and "show hidden files".  

When you install a theme, it may not show up in the list right away, but you can still select the controls, window borders etc for that theme, and save it as you wish.
