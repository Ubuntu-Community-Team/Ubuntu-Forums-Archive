### Post by chocbar31 on 2007-03-24
I had this too. But, I also had this option that made me wanna jump off a high-*** roof, as I just plain missed it for about an hour.

Right-click beryl-manager -> Select Window Decorator -> Standard Beryl Decorator (Emerald)

By default, it is set to GTK Window Decorator. (New Option?)

And all is well with me again, switching on the fly!

oh, Feisty and Fesity Beryl/Compiz....latest I guess!!!

Hope this helps others.
