### Post by Zorael on 2007-10-31
Yes, that looks about right.

Restart X (the easy way is to hit Ctrl+Alt+Backspace, but save your work first since it'll close all applications. The long way is to reboot your computer).

Once logged in, open up the CompizConfig Settings Manager, either by launching it from the menus (I'm not sure what it's called, something to do with Advanced Desktop), or by launching 'ccsm' either from a terminal or from a run box (Alt+F2). If it's not in the menus or won't start when evoked from a terminal, you may not have it installed - find the package 'compizconfig-settings-manager' in Synaptics, or do 'sudo apt-get install compizconfig-settings-manager' in a terminal.

When it opens, scroll down to find Window Decoration. Make sure it's enabled. Open its settings, and in the Command field, enter 'emerald --replace', without the quotes.

*Then* start Compiz, with 'compiz --replace &'.

If you *still* don't get any window borders, run 'emerald --replace &'.


If it works, you may want to try starting it with 'compiz --replace --loose-bindings' - it can improve performance on Nvidia cards. If it doesn't, I'm out of ideas.
