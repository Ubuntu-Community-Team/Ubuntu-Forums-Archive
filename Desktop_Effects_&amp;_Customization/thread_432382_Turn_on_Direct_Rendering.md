### Post by Belyel on 2007-05-04
There are a TON of guides here and around to help with this.  Short answer is that you need to install the ati drivers instead of using the included vesa/mesa driver.  The ATI driver is called "fglrx" and you can install it either manually or with a script called "envy."  Look around the forums for details, but I definitely recommend the envy script, which can be found through [this site!]("http://www.albertomilone.com/nvidia_scripts1.html")
