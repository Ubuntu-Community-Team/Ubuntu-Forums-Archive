### Post by ajgreeny on 2007-06-12
If you hadinstalled beryl with synaptic or apt-get you would have the oportunity to set the window manager (compiz, beryl, kwin, or even metacity if gnome is also installed) by right clicking on the beryl manager icon in the task bar.  You could then get back to kwin easily.

I suggest you use synaptic to install beryl and beryl-manager, even if you end up not using it, and set everything that way.  Whenever possible I would suggest you use the tools ubuntu already gives you to install and set up these things, not go to a site where they are trying to cover all possible distros, etc.
