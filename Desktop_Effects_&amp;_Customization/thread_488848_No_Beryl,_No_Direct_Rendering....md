### Post by hyperair on 2007-07-01
Well one thing's for sure. You're not going to get it to run with the "nv" drivers. It's simply impossible. Won't work. Or you'll crash and burn. Here's a suggestion:

Remove the nvidia drivers from your system (replace /path/to/nvidia-installer.run with the correct path)
```

sudo /path/to/nvidia-installer.run --uninstall
sudo apt-get autoremove nvidia-glx*

```

Restore your original xorg.conf file: 
```

sudo dpkg-reconfigure xserver-xorg

```

Restart your system:
```

sudo shutdown -r now

```

Log in, and then go to the Restricted Drivers Manager and turn on the nVidia option.
Restart.
Log in, play with Beryl.
