### Post by luisromangz on 2008-01-31
Maybe your compiz version has fglrx blacklisted, because the older versions didn't supported AIGLX. You should try to whitelist it, to do so add fglrx to the WHITELIST parameter in the file /etc/xdg/compiz/compiz-manager.

It should look something like this:

# Driver whitelist
WHITELIST="nvidia intel ati radeon i810 fglrx"
