### Post by avik on 2007-07-01
To run AWN, you have to have a composited environment, meaning that you have to be running Beryl or Compiz.

Uninstalling depends on how you installed it.

You said you find the themes ugly, so does that mean you want to make your own theme.  Because otherwise, there's always [http://www.gnome-look.org/]("http://www.gnome-look.org/").

### Post by avik on 2007-07-01
You should be able to uninstall AWN through sudo make uninstall, if AWN provides that functionality and if you kept the source directory.

If you want to make your own theme, try looking at [http://developer.gnome.org/doc/tutorials/metacity/metacity-themes.html]("http://developer.gnome.org/doc/tutorials/metacity/metacity-themes.html").
