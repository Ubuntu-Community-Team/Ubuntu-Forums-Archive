### Post by FuturePilot on 2007-10-31
The Gnome Screen Resolution tool does not work correctly with the Nvidia driver. You need to use the nvidia-settings panel or edit your xorg.conf
```
Section "Screen"
    Identifier     "Default Screen"
    Device         "nVidia Corporation NV11DDR [GeForce2 MX200]"
    Monitor        "EV700"
    DefaultDepth    24
    **Option          "metamodes" "1024x768_70 +0+0"**
```
Add what's in bold assuming 1024x768 is your res.

Also 
```
Option         "AddARGBVisuals" "True"
    Option         "NoLogo" "True"
    Option         "AddARGBGLXVisuals" "True"
```
these should be under the Device section, not Screen.

### Post by justsomedude on 2007-11-06
I have to add the option *"DynamicTwinView" "0"* to my xorg.conf to get the desired refresh rate with my monitor when I use the official Nvidia drivers, like this:

```
Section "Device"
    Identifier     "nVidia Corporation MCP51 PCI-X GeForce Go 6100"
    Driver         "nvidia"
    Option         "DynamicTwinView" "0"
    ...
EndSection
```

[COLOR="Red"]!!![/COLOR] This works for me, however wrong refresh rates can damage your monitor. I take no responsibilities for damaged hardware...
