### Post by enharrin on 2007-03-17
Hello, I've got Beryl 0.2.0 installed and am having a slight problem.  I was going to post in the beryl forums, but they seem to be down at the moment.

Anyway, every time a new window pops up, it gets sent right to the back.  This is particularly annoying for dialog windows in programs like Synaptic, or for any program that I want to open the preferences.  But it also happens when I open a new window, it is sent right to the back.  Is there a setting in Beryl somewhere that controls this?

This only appears to be problem in Beryl.  When I went back to Metacity the windows came up on top fine.

Thanks!

---

### Post by mcduck on 2007-03-17
In Beryl Settings Manager's General section set 'Level of Focus Stealing Prevention' to 'none'. That should solve the problem.. :)
