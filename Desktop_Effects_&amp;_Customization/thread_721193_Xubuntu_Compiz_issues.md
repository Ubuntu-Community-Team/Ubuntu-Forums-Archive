### Post by xjgnsdof on 2008-03-11
I've had the same problem with the window title bars in Compiz in Xubuntu. You have to install emerald and, if you want to customize the window title bars, install a window theme in Emerald Theme Manager.

```
sudo apt-get install compiz-core compiz-plugins compiz-fusion-plugins-main compiz-fusion-plugins-extra emerald compizconfig-settings-manager
```

Beneath the “Effects” heading, click Window Decoration. In the Command input field, enter "emerald." Then, go to [http://www.xfce-look.org/]("http://www.xfce-look.org/") and download some Emerald window themes.

Finally, to customize the window title bars (which, for me, were an obnoxious red) go into Emerald Theme Manager, select "import," and pick the Emerald theme package that you downloaded earlier.

### Post by Zorael on 2008-03-12
My apologies, I misinterpreted.

Have you at any point run ccsm as root? It could have changed the permissions needed to alter the settings.

To change ownership and group association of all files under your home directory, I believe you can enter the following commands in a console. There may be a graphical way to do this under Nautilus/Konqueror/Dolphin, I'm not sure. Replace *yourname* and *yourgroup* with your login name and your group (generally the same names unless you changed them).
```
$ sudo chown *yourname* ~ -R
$ sudo chgrp *yourgroup* ~ -R
```

Also, make sure you have the **automatic plugin sorting** option enabled under preferences.

Lastly, did you upgrade from Feisty to Gutsy? If so, you may have an old version of ccsm lurking around. Make sure to remove any foreign repositories before doing this (generally eyecandy ones, such as Treviños'). Just comment them out if you want to, just to be on the safe side.

edit: May be a good idea to just reinstall compiz completely. Just make sure to use **Completely Remove** in Synaptic, or **Purge** in Adept Manager. Or the console way:
```
$ sudo aptitude update
$ sudo apt-get remove --purge compiz* libcompiz* libdeco* emerald* libemerald*
$ sudo aptitude **purge** compizconfig-settings-manager
```

Then reinstall. I don't remember the package names, and I've booted into Windows now. Should be **compiz, compiz-gnome, compiz-plugins, compiz-fusion-plugins-main, compiz-fusion-plugins-extra, compizconfig-settings-manager, emerald, emerald-theme-manager** or something like that.
```
$ sudo apt-get install compiz compiz-gnome compiz-plugins compiz-fusion* compizconfig-settings-manager emerald*
```

And **gnome-compiz-manager** if you want the integrated configuration manager.
