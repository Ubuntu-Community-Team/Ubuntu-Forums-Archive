### Post by Rupertronco on 2007-09-26
If you have questions about Gutsy, the best place is the Gutsy forum. Desktop effects don't really relate to your problem. 

Anyway, I have no idea what you mean by "ive stopped getting regular updates the way i used to", so if you could clarify, I could probably help. 

As for the grub entries, to access your grub menu, you need to edit a text file. That file is /boot/grub/menu.lst. If you don't have any experience messing with this file I'd do quite a bit of reading before messing around with it. It can render your system un-bootable, then you'll have to reinstall grub from a live cd.
