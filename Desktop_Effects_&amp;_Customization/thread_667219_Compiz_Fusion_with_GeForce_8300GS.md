### Post by Sef on 2008-01-14
> recently bought a Dell desktop which comes equipped with Nvidia GeForce 8300 GS. It is my understanding the GeForce 8300 GS does not support 3D effects? Is that true? If so, does this mean I cannot use Compiz Fusion?


The [8300 GS]("http://forum.compiz-fusion.org/showthread.php?t=1858") is supported from June 2007.

1) System > Administration > Restricted Drivers Management

2) System > Administration > Synaptic Package Manager > search for Compiz-Fusion > Install

3) Read [Forlong's blog]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion#") to get the plug ins.
