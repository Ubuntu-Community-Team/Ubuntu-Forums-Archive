### Post by rune0077 on 2008-01-17
Not entirely wrong. Enable the Rotate Cube plugin (I assume you already did), then go to the settings dialogue for this plugin. Here, change the zoom option, that's probably what your looking for. 

If none of this works, then go to the Viewport Switcher plugin and ensure that it is set like this:

Plugin for initiate action: rotate

Action name for initiate:  initiate
