### Post by sawjew on 2007-05-22
the system monitor up on the right is Conky, go to this page for instructions on installing it [http://conky.sourceforge.net/]("http://conky.sourceforge.net/")

It is extremely configurable by editing the config file at ~/.conkyrc.  The easiest way to set it up is to go to the Conky home page and look through the screenshots and find one you like and then copy it's conkyrc file to ~/.conkyrc.
