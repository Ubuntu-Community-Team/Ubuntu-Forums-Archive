### Post by mister harbies on 2007-06-28
Ok I figured it out using Beryl.

In the beryl settings manager, windows management, set windows attribs by various criteria. Select windows opacity under opacity.

Enter ```
n:gnome-screensaver:80
```. Or use ```
n:xscreensaver:80
``` if you are using xscreensaver I think.

80 is the transparency level. I tthink 80 looks great with solarwinds. Now I can check my IMs and whether I have unread emails without having to exit the screensaver.
