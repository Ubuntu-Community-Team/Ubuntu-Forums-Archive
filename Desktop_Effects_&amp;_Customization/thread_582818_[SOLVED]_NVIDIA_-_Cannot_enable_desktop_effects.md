### Post by kevmaster on 2007-10-20
I think what you need to do is reboot, to be sure you're running the latest kernel that's on your system. And then reinstall the restricted-modules & nvidia driver from apt, depending on your card:

[LIST]
[*]**nvidia-glx-new** - GeForce5/FX and higer
[*]**nvidia-glx** - between GeForce2 MX - GeForce4
[*]**nvidia-glx-legacy** - old cards: TNT, TNT2, GeForce, GeForce2 without MX
[/LIST]

Like this:
```

sudo aptitude -y update
sudo aptitude reinstall linux-restriced-modules-$(uname -r)
sudo aptitude reinstall nvidia-glx-new

```

And then modify your xorg.conf so it refers to the right nvidia version like this:
```

md5sum /etc/X11/xorg.conf | sudo tee /var/lib/x11/xorg.conf.md5sum
sudo nvidia-glx-config enable
sudo nvidia-xconfig

```

Then reboot.
Once your back in GNOME, press **ALT**+**F2** type *gksudo nvidia-settings* and press **Run**. Pimp the resolution, close, and fire up compiz-fusion.

Full article: [Upgrade to Ubuntu Gutsy with Compiz-Fusion]("http://kevin.vanzonneveld.net/techblog/article/upgrade_to_ubuntu_gutsy_with_compizfusion/")

### Post by dpatel on 2007-10-20
I think I may have found a solution through searching around the forums. I edited the /usr/bin/compiz file and commented out the line NVIDIA_MEMORY="65536" # 64MB (put a hash in front of it),

I enabled Visual Effects and then found borders were missing so then did:

```
sudo nvidia-xconfig --composite --render-accel --add-argb-glx-visuals
```

Seems to be working.
