### Post by igknighted on 2007-08-13
Beryl works just as well on 6.10 as it does on 7.04.

Sun Java and Flash don't work on 6.10 either, so if thats the issue please go with 7.04.  Beryl is abandoned software, the program you want is called compiz-fusion.

EDIT: For java, the PPC arch version is released by IBM and should give the same functionality as Sun's.  IBM releases it because they have a line of processors that typically run unix and are PPC based.  Flash has free alternatives that can handle youtube and other popular flash sites (but not all) and these free versions should run fine on PPC
