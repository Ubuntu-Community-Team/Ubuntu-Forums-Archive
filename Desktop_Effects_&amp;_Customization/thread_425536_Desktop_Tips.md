### Post by jordanmthomas on 2007-04-27
...creating desktop environments?  Are you talking about writing your own Gnome or something?  Not sure what you mean here.

Themes, I can help you with.
[http://art.gnome.org](http://art.gnome.org) has themes
[http://www.gnome-look.org](http://www.gnome-look.org) has themes

To install a theme, go to system -> preferences -> theme
Then, just drag the .tar.gz you download from one of these sites onto the window and it will be installed.

Also, a few definitions to help you out:
GTK / Application theme :  your buttons, window backgrounds, etc.
window borders / metacity theme : the bar where the close, minimize, etc. buttons are.  

That should get you started.  If you come across something you don't understand, chances are you can search on the forum and there will be something posted on it.  Good luck.
