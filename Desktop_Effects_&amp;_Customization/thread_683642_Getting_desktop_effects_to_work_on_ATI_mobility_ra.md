### Post by ajtolland on 2008-02-01
OK,

The problem I had (with the Radeon 9200 mobility in Ubuntu 7.10) was that, when I tried to turn on Desktop effects (i.e. when I tried to use compiz), I would get the following error:

Comparing resolution (1400x1050) to maximum 3D texture size (1024): Failed.

If you run compiz in a terminal, you'll probably see this as well.

This is the result of a bug in compiz.  Michael Longval [explains a workaround her]("http://ubuntuforums.org/showpost.php?p=3650745&postcount=18")e.

Good luck.

### Post by philnad on 2008-02-10
Got it working now !!! After two days and your help!

Ok... I started compiz like:
```

SKIP_CHECKS=yes compiz

```

When I had done this before my last reinstall, it was not working :) now it is!

I have changed /usr/bin/compiz to include these two lines at the very top:
```

LIBGL_ALWAYS_INDIRECT="true"
SKIP_CHECKS="yes"

```

so, now I can use the standard tab on "Appearance Preferences" dialog. I chose "Normal" and it switched!!

Other than the loss of some hair ... I am happy!

THANK YOU!
