### Post by Qinjuehang on 2007-11-10
Lemme tell you what I do.
I split my theme into 4 parts.
1. Metacity theme (I only use it for buttons)
2. Emerald theme (better transparent PNG rendering)
3. Transparent Panels (I still can't find a way to use transparent PNGs in GTK, so I take them out)
4. 3rd-Party program skins (XMMS, audacious, firefox...the list continues)
For the Metacity theme, I take other themes, and replace all the images. them I might change the colors by editing the gtkrc file. For the Emerald theme...once you download the Emerald theme manager, it does everything. For the transparent panels, you just state in the READEME file to Right-Click their panels and apply the images, if you are gonna put it up. 3rd party skins/themes...are up to the program.
