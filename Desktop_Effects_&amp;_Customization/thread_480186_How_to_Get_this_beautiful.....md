### Post by jusmurph on 2007-06-21


Beryl Manager should be a little Red Jewel in your system tray. Right click to bring up the options.

Beryl Settings Manager > Desktop > Check Desktop Cube.

Check Beryl Are Selected as the Window Manager and Window Decorator respectively.
