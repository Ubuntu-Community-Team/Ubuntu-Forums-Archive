### Post by Happy_Man on 2007-07-25
Right, so you have the window borders, all you need is the GTK theme. I suggest contacting whoever made that theme for the name of it. 

As a quick breakdown:

Metacity theme = window border
GTK theme = menu bars and such
GDM theme = login theme
Usplash theme = loading bar theme
