### Post by aidanr on 2007-10-08
If you're on about the bar where file and edit etc are, you can change that with the [configurator]("http://murrine.netsons.org/?q=node/8") .

Or if you want to do it manually I believe this is the part you're looking for:
> - menubarstyle (change style of the menubar)
    * 0 = flat
    * 1 = glassy (it follows selected glazestyle)
    * 2 = gradient
    * 3 = striped
