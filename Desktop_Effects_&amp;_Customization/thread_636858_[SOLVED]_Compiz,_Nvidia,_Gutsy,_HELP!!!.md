### Post by lian1238 on 2007-12-10
I would suggest you reinstall the nvidia driver, since you upgraded. (My upgrade resulted in a non working system, so I did a fresh install.)

For the driver, I use Envy to install. If the problem persists, we can try other things.

To get rid of the error "No GLXFBConfig for depth 32", add these to the "Screen" section of your /etc/X11/xorg.conf file
```

Option         "AddARGBGLXVisuals" "True"
Option         "DisableGLXRootClipping" "True"

```
