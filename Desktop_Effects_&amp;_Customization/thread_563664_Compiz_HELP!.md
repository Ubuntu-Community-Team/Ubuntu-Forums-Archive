### Post by Happy_Man on 2007-09-30
Append this under the Device part of your xorg.conf, so that it looks like this: ```
Section "Device"
Identifier "nVidia Corporation NV34 [GeForce FX 5500]"
Driver "nvidia"
Option          "AddARGBVisuals"        "True"
Option          "AddARGBGLXVisuals"     "True"

EndSection
```

Then save it, press ctrl-alt-backspace, log in again, and try running Beryl once more.
