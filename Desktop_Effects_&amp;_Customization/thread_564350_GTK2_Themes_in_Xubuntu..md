### Post by Gremlinzzz on 2007-10-02
if you want to install a new gtk2 theme, you have to do two things:
1.) You have to install the theme engine. There are several engines, for
example Murrina, Clearlooks, Pixbuf, ... You can install them with Synaptics
for example.
2.) You need to put the theme itself somewhere: you can put it under
$HOME/.themes or under /usr/share/themes/ (with superuser privileges).
Theme files can be found at [www.xfce-look.org](www.xfce-look.org). You will download a tared
file that you have to untar into one of the folders I mentioned.

You can get icon-themes from the same site. Put them (un-tared) under
$HOME/.icons or /usr/share/icons/ (again, only with superuser privileges
possible).

[http://www.xfce-look.org/index.php?xcontentmode=15x100x420](http://www.xfce-look.org/index.php?xcontentmode=15x100x420)
