### Post by gila_monster on 2007-04-30
Hi, folks.

Go to beryl settings and click on the "shortcuts" option in the window on the left, just below "general options." Select "screen edges" in the window on the right. The option for Top Right Corner is set to "Initiate Window Picker for All Workspaces" by default. You can set it to something else, or set it to "none" to remove the behavior entirely.
