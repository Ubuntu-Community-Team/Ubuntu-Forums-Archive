Nevermind, I figured it out. If anyone else was wondering...it's quite simple. Just Go:

-- System > Preferences > CompizConfig Settings Manager
-- In the FIlter block (top left) type 'Desktop Cube'

From there, just uncheck the plugin and disable Desktop Cube.
