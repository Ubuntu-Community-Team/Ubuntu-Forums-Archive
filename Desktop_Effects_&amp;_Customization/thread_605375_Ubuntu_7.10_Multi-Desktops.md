### Post by Mothinator on 2007-11-07
You should see in the bottom right hand corner of the screen two rectangles next to each other. This is the desktop switcher icon. To switch between them, just click (or hit CRTL+ALT+(left or right arrow)). If you want to add more desktops: right click on it , go to preferences and set number of workspaces to whatever you want.

If you don't have this icon, go to the bottom bar, right click, select add to panel, desktop switcher.

(I'm assuming you are using GNOME since the title is Ubuntu 7.10, if you are using KDE these instructions probably won't work)
