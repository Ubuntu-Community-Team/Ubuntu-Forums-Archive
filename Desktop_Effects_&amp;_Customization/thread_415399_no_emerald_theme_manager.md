### Post by aeon on 2007-04-20
open up a terminal and paste this in```
sudo apt-get install beryl beryl-manager emerald emerald-themes
```

after that press alt+F2 and type in ```
beryl-manager
```

now you should see the beryl icon in your notification area, just right click it and select beryl instead of metacity under window manager..
