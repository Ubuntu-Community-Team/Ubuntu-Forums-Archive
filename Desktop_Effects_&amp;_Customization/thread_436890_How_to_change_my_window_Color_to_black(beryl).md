### Post by Soybean on 2007-05-15
Oh, that's right... Beryl/Emerald only override the window frames of the GTK themes. So you'd want to use Emerald to select a black border (like you've done), then go to System-Preferences-Themes to get at the other parts. 

In there, if you click the "Customize" button, then go to the "Colors" tab, you should be able to change the various parts that aren't handled by Emerald. Or you could download a black theme, which might also include an icon set.
