### Post by dmoney on 2007-05-28
You'll need emerald installed to use those types of themes. There is a package [here](http://ubuntuforums.org/showthread.php?t=423743&page=5) that works with the current Feisty version of Compiz. Follow the instructions to install and check the last page for my tip on how to make emerald load on login instead of the default compiz decorator. From there, just download the themes you want (they should have a .emerald file extension) to your desktop, open Emerald Theme Manager from System > Preferences >  and click Import and select your theme. Voila. :D

Let me know if you get stuck. My AIM is up there if you need it.
