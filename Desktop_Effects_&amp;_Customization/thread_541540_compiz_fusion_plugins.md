### Post by Iehova on 2007-09-03
Hi,

I could only get the 3d windows, aquarium and other plugins working having installed Compiz from Treviño's repository, and having  installed the compiz-fusion-plugins-extra, -unsupported and -main packages from that same repository.

When I installed from Amaranth's (?) repository, which I did first, I couldn't get the extra plugins working. Perhaps this is your problem?

At the same time, however, you should be warned that the compiz in Amaranth's repository is generally considered (at least from what I've read) more stable than what's in Treviño's, which, it seems, is a little more bleeding edge.

So yeah, AFAIK, if you want stability, go with Amaranth, if you want all the latest plugins, Treviño's where to go. :)

EDIT: Oh, and some people say that the Compiz in Treviño's repository is faster, plays video's better etc. Personally I can't comment, because I didn't notice a difference. Meh. ;)
