What are you trying to do?  Ubuntu (and any Linux distro) is insanely customizable.

You can change the Desktop Environment / Window Manager.
You can change the location / size etc of the panels.
You can customize the menu (see Linux Mint for an alternative).
You can chose which programs to use for different functions (e.g. File Manager - nautilus vs thunar vs ROX vs xfe etc).
You can change the look of the login screen.
You can change your desktop picture.
You can change the icon set in the file manager.
You can change the window borders and locations of the maximize / minimize buttons on each window...

PS: This is probably a good place to start for alternative looks: [http://www.gnome-look.org/](http://www.gnome-look.org/)
