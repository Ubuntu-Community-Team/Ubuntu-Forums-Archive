### Post by hdot on 2007-05-17
I think this is all you need to do, but I am fairly new to this as well.

1. Open a terminal, type sudo gedit /etc/X11/xorg.conf
2. Enter password
3. Scroll to the bottom, you should see this:

```
Section "Extensions"
	Option		"Composite"	"Disable"
	Option		"Composite"	"0"
EndSection
```

4. Change the "Disable" to "Enable"

```
Section "Extensions"
	Option		"Composite"	"Enable"
	Option		"Composite"	"0"
EndSection
```

5. Save
6. Done
