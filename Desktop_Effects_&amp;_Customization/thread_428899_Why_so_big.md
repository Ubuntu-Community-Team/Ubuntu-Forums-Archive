### Post by airtonix on 2007-05-01
your talking about the size of your deskto icons...

right click each one and select 'stretch-icon', this will put handles on all four corners of the icon, whcih you can use to shrink or expand the icon. i shrink mine to about 60% default size. others i put at 150% if they are used all the time (like my ipod icon)

then for your top panel...right click it
- select 'properties'
- size to 22 pixels or whatever you want...

font sizes for window titles..
system -> preferences -> fonts

change the window font to one that you like an the font size too.
while your at it change the other fonts too, making them the same size....
