### Post by Espreon on 2008-01-19
In the section of the xorg.conf in your home folder, highlight everything in that section down to the line that says End Section

Now copy n paste this:

```

Section "Monitor"
	Identifier	"Failsafe Monitor"
	Vendorname	"Generic LCD Display"
	Modelname	"LCD Panel 1024x768"
	Horizsync	31.5-48.0
	Vertrefresh	56.0 - 65.0
  modeline  "640x480@60" 25.2 640 656 752 800 480 490 492 525 -vsync -hsync
  modeline  "800x600@56" 36.0 800 824 896 1024 600 601 603 625 +hsync +vsync
  modeline  "800x600@60" 40.0 800 840 968 1056 600 601 605 628 +hsync +vsync
  modeline  "1024x768@60" 65.0 1024 1048 1184 1344 768 771 777 806 -vsync -hsync
	Gamma	1.0
EndSection

```

Save

Now enter this:

```

mkdir backups

```

Now enter this:

```

cp /etc/X11/xorg.conf /home/yourhome/backups

```

Now reinstall the restricted drivers and print or write down these directions before rebooting into recovery mode:

1.

```

rm /etc/X11/xorg.conf

```

2.

```

cp /home/yourhome/xorg.conf

```

3.

Reboot

4. If X is giving you an error reboot into recovery mode and enter these:

```

rm /etc/X11/xorg.conf

```

```

cp /home/yourhome/backups/xorg.conf

```

Then uninstall the restricted drivers...

### Post by rune0077 on 2008-01-19
Okay, your card should be fine with the *newest* drivers. From nvidia's homepage:

> Improved usability of NVIDIA-settings at lower resolutions like 1024x768 and 800x600.

Which may suggest that there were some problems with these resolutions on older drivers, including the ones in the repo. My best suggestion would be to get the latest drivers and install those, instead of the ones you get from Synaptic. If you aren't comfortable doing this yourself, you can always use Envy: [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html) - this program will automate the entire process, install the drivers that fit your card (if it's nvidia or ATI), disable the pre-installed drivers and reconfigure your xorg.conf for you.
