### Post by notwen on 2008-02-14
Open up a terminal then issue this command:

```
sudo aptitude install compizconfig-settings-manager
```

After this click _System--->Preferences--->Advanced Desktop Settings_

This should get you to where you're wanting to be, simply click on the option/effect and it will bring up the details regarding that option.  Hope this helps. =]
