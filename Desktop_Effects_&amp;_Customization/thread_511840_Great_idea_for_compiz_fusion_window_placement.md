I think this might come in handy when there are many window opened and the user would like to select where the window would pop up. the idea is have the cursor change somehow letting the user know that they can select anywhere on the desktop that they would like to have the window pop up. The user would be able to even select what workspace the window should open up in. This option to select where the window would open could be used as a plugin for compiz, compiz fusion or beryl. Note, this plugin would be highly customizable. For example there could be a time limit for how long the user could click somewhere to open the window. The plugin might have the option to be able to have the desktop fade to a certain color or shade and the cursor could change color, shape and animation. I'm not sure how well this idea might hold in real working conditions but I think it would be a great option to have. I would love to get some feedback on how well this idea would pan-out  and if this idea has crossed other peoples mind or even if this is concept has already been put to use.
