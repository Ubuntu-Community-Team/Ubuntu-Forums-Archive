### Post by monoufo on 2007-11-03
It is quite tricky.  Some people prefer to replace gnome-screensaver with xscreensaver.  other's prefer to manually adjust the commands.  You're in luck though, because someone is working on a 3rd party app to do change the settings just like xscreensaver with still using gnome-screensaver.

[https://bugs.launchpad.net/ubuntu/+source/gnome-screensaver/+bug/22007](https://bugs.launchpad.net/ubuntu/+source/gnome-screensaver/+bug/22007)

install the program near the bottom of the page.  It's still under development, but it works pretty good so far.
