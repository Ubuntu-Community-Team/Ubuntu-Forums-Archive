Ultimately, that's the fault of the website you're viewing, which sets the background color but not the foreground (text) color--or vice versa. However, since you can't control other people's websites, you might be able to find something in your browser's settings to fix that--either by specifying default colors or a user-defined stylesheet. I don't know which browser comes with Xubuntu by default, but both Epiphany and Firefox should allow you to do this.
