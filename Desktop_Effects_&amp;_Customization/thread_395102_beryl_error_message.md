### Post by jordanmthomas on 2007-03-27
Have you checked to make sure DRI is loaded as a module in your xorg.conf?
```
Xlib: extension "XFree86-DRI" missing on display ":0.0".
```

You need this at the end:
```
Section "DRI"
    Mode 0666
EndSection

```
and this in "Modules" section:
```
   Load        "dri"

```
