### Post by PaulFXH on 2007-03-20
That's a real pity.
I had a similar problem with Beryl in that my left Arrow key would not function any more but worked only as "Pause" in RhythmBox. This seems to be because Shift-Left Arrow is a shortcut for Pause and seems to have "confused" Beryl.
However it's not such a big problem for me as I can still use the Numpad left arrow key if I need to.
In addition, by using Systems>Preferences>Keyboard>Layout>Reset to Defaults, I was able to "momentarily" get back the usual function of the Left Arrow key for a short while until "something" disabled it again. Unfortunately, I was never able to find out what the "something" was. However, as I've said already, it really wasn't such a big problem for me so the incentive for fixing it was less than in your case.
Perhaps some of these comments may be of use to you. Alternatively, you could post your problem to the [Beryl Forum]("http://forum.beryl-project.org/").
