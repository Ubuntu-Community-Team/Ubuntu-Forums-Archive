### Post by dmm1285 on 2007-11-20
I was trying to find an Ubuntu theme for my compiz fusion cube, but apparently these is no default skin. So I shamelessly stole this skydome: ([http://xcorex.wordpress.com/skydome/](http://xcorex.wordpress.com/skydome/)) and the elephant skin wallpaper to make a little theme that fits with the default desktop. Screenshots:
[img]http://ubuntuforums.org/attachment.php?attachmentid=50814&stc=1&d=1195568735[/img]
[img]http://ubuntuforums.org/attachment.php?attachmentid=50815&stc=1&d=1195568735[/img]

Download link: [http://www.zshare.net/download/50381081448763/](http://www.zshare.net/download/50381081448763/)
To install, back up your .gonf/apps/compiz folder and place the one in the archive there. Then, open up Advanced Desktop Settings and change the cube caps to the ones included in the archive. Finally, open up Appearance and change the background to the included picture. Enjoy!
