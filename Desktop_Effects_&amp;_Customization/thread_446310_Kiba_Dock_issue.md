### Post by Ziox on 2007-05-16
You need to have either Beryl or Compiz (Desktop effect) enabled. 

To enable Beryl:

sudo aptitude install beryl emerald && beryl && beryl-manager

to enable Compiz,

go to System > Preference > Desktop Effects
