### Post by FuturePilot on 2007-10-23
You can find out what card you have by pressing Alt+F2 and entering 
```
nvidia-settings
```
Go to where it says GPU-0 and it will tell you what card you have. 
This is caused by a bug in the Nvidia driver. If you have a newer card you can use the 100.14.19 driver which fixes this. However if you have an older card there's nothing you can do right now.

### Post by balak on 2007-10-26
I was able to my xorg to get compiz working:

Basically had to add a few lines to the xorg.conf file (found it here)
[http://www.pendrivelinux.com/2007/10/17/ubuntu-desktop-effects-fixing-the-missing-titlebar/](http://www.pendrivelinux.com/2007/10/17/ubuntu-desktop-effects-fixing-the-missing-titlebar/)

Snippet of my xorg.conf

```

Section "Device"
        Identifier      "nVidia Corporation C51PV [GeForce 6150]"
        Driver          "nvidia"
        Busid           "PCI:0:5:0"
        Option          "AllowGLXWithComposite" "True"
        Option          "RenderAccel"   "True"
        Option          "AddARGBVisuals"        "True"
        Option          "AddARGBGLXVisuals"     "True"
        Option          "NoLogo"        "True"
EndSection

```
