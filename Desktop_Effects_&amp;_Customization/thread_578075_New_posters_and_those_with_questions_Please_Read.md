### Post by Rupertronco on 2007-10-16
If you have questions or problems regarding an issue with desktop effects. Please make your own thread specifically explaining your issue. It is difficult while helping to have other questions asked within threads. 

If you need help, you've come to the right place, but I ask, please make your own, new thread instead of posting a question in another's thread. Also, make sure to search the forums regarding your problem, chances are it has already been addressed and it may be easier to find your answer in an old thread as opposed to posting a new thread and waiting for responses. 

If you're having problems with enabling desktop effects at all, it's a good idea to give us some information:

1. Tell us what video card you're using. If you're unaware you can find it by typing ```
lspci | grep 'VGA'
```



2. If and how you have installed compiz/beryl/compiz-fusion



3. post a copy of your xorg.conf file. Do that by typing this in a terminal and copying the contents and posting them. ```
gedit /etc/X11/xorg.conf
``` 



4. Go to a terminal and type the following and post the output of that command ```
glxinfo
``` 



5. If you have compiz installed and have problems starting it, go to a terminal and post the output of ```
compiz --replace
``` 


By following these steps, you will provide a great deal of information that will aid others in helping you. 

Cheers.
