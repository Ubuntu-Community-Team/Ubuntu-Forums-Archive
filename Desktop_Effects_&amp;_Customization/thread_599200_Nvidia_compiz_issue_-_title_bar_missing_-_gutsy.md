### Post by FuturePilot on 2007-11-04
Try this
```
gksudo gedit /etc/X11/xorg.conf
```
go down to the screen section
```
Section "Screen"
    Identifier     "Default Screen"
    Device         "nVidia Corporation G80 [GeForce 8800 GTS]"
    Monitor        "Generic Monitor"
    DefaultDepth    24
    [COLOR="Red"][B]Option         "AddARGBVisuals" "True"
    Option         "NoLogo" "True"
    Option         "RenderAccel" "true"
    Option         "AllowGLXWithComposite" "true"
    Option         "AddARGBGLXVisuals" "True"[/B][/COLOR]
    SubSection     "Display"
```
Remove what is in red.
Now go up the the Device section and add these lines to it
```
[LEFT]Option          "RenderAccel" "true"
Option          "AddARGBVisuals"         "True"
Option          "AddARGBGLXVisuals" "True"
Option          "NoLogo"           "True"
[/LEFT]
```
Save, log out and log back in
