### Post by grahambo2005 on 2008-01-13
Go to System > Preferences > Appearance 

Click on the Fonts Tab

Change the First 3 at the top to "Sans"

Change the 4th to "Sans Bold"

Change the 5th to Monospace 

The number beside all of the selection boxes should read 10.

for Rendering use the "best shapes" option

Thats the default

Graham
