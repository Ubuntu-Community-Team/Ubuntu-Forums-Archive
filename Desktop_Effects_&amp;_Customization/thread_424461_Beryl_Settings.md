### Post by detroit/zero on 2007-04-26
Hi all. I'm running on a fresh install of Feisty and I've installed Beryl to give my desktop a little glitz. I've been tweaking settings here and there to make it just right, but I've managed to mess up a couple little things. Neither of them is a big deal, but once you start customizing, it's hard to stop until everything is just right. I'm sure you can all understand.

First, I noticed that windows in Firefox no longer open in new tabs, they open in new windows. I checked the Firefox settings and they're set to open in tabs. I suspect Beryl's window management is grabbing them and making new windows out of them, but I don't see a specific setting in the Window Manager pane that would force a new window... It didn't do this when I first launched Beryl, so I know it has to be a setting I changed. Any ideas which one?

Second, I notice when I'm switching workspaces, my panels no longer move with the cube. They stay firmly planted at the tops and bottom of my viewing area (screenshot included). The panels used to be on the cube.. somewhere along the line I detached them. I also haven't been able to find a setting that seems like it would take care of that.

I'm digging through various Wikis and the Beryl Page but I'm not finding any hints just yet.

Any of you seasoned vets feel like showing a noob the error of his ways?

### Post by cogadh on 2007-04-26
I don't know about the panel problem, but the Firefox thing happened to me. All I did to fix it was change the setting to "new window", restart Firefox, then change the setting back to "new tab". Apparently Firefox need a reminder about what to do.

---

### Post by detroit/zero on 2007-04-26
Excellent fix. Worked like a charm. I wouldn't have thought about doing that myself. Muchas gracias.

Now, if anyone has any ideas about my panels?

### Post by detroit/zero on 2007-04-26
I got it.

Under the Desktop tab, the Show Desktop option was unchecked. Once I enable that, the panels got slapped back onto the cube.

Yay, me!

Thanks for the help, cogadh.
