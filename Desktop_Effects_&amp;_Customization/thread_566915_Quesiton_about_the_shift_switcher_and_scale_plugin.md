### Post by ssteinbr on 2007-10-04
For the shift switcher, I have "windows from all viewport" binded to Super+Alt Tab.  I'm pretty sure the settings for this are in the Compiz Config Settings Manager (CCSM).  When you click on a plugin in the manager, you can select what binidngs you want.

EDIT: Same goes for the Scale plugin
