### Post by mezcla on 2007-06-16
Are you trying to enable desktop effects and run beryl?  You can only do one or the other.

Aside from that, what kind of video card do you have?  This is important because if you have an ATI card vs. Nvidida you may run into more trouble.  Post more of your system specs and someone might be able to give you more accurate information, but here's what I did to get Beryl running on my system.  If you are running Feisty you need to install the video card driver, it's simple.  Just click System>Administration>Restricted Drivers Manager and have it install the appropriate Driver.  That should allow you to run Beryl or Compiz.  If you have done that let me know and I'll see if I can help from there.
