OP, did you know that Compiz and Beryl (which was a fork of Compiz) have merged? The new 3d compositor (which is the fancy name for OMFG i has a cubezor on muh desktops!!!111) is called Compiz-Fusion. 

This program is included by default in Gutsy and can be activated by going to System > Preferences > Appearance > Visual Effects and then turning it on. Either normal or extra.

For more advanced settings configuration (or any, really) paste this command into a terminal:
```
sudo apt-get install compizconfig-settings-manager
```
Then there will be a settings manager in System > Preferences.

Hope that helps, and I pray that it works just like that.
