### Post by Ub1476 on 2007-12-08
Go [here]("http://wiki.compiz-fusion.org/") and look under the headline "Plugins".

And if you haven't already:

```
sudo apt-get install compizconfig-settings-manager
```

System>Preferences>Advanced Desktop Effects.

### Post by Espreon on 2007-12-08
Before you execute the above command, you need to enable the Universal Repos by doing the following:

1. Goto System>Administration>Software Sources

2. Check the box next to Universal (if it is not already checked)

3. Close the window and hit apply when prompted
