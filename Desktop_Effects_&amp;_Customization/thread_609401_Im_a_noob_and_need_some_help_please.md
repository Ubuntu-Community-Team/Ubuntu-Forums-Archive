### Post by nae5 on 2007-11-11
I had to install the compizconfig-settings manager and xserver-xgl before my effects would work.

     So go to system>synapticpackagemanager   search for "compiz" and install

compizconfig-settings-manager

or type "sudo apt-get install compizconfig-settings-manager"  no quotes

try enabling the cube or the fire plugin or another plugin to see if they work.

if the compizconfig-settings-manager installs but doesn't change anything try installing xserver-xgl.

     system>synaptic search "xserver-xgl" noquotes.  if it's not installed install it.  

or type "sudo apt-get install xserver-xgl" noquote

restart x (left CTRL ALT BACKSPACE)  or restart computer and see if they are working.

If these two packages don't make it work for you i might not be able to help any further since im new to troubleshooting pcs, but don't give up it's worth it to have an ubuntu system for everything but games that don't run on linux.  check my screenshots for the coolness.
