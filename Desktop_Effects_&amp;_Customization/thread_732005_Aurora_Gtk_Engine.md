### Post by durand on 2008-03-23
Yeah, as it says in the title, Aurora is a GTK engine. The bit you want to change is not part of GTK, but part of the windows manager, ie metacity, emerald, etc. This is a different component so you would need a different theme. If you're using metacity, then go to System > Preferences > Appearance and click on customise, then choose your theme in the 3rd tab.
