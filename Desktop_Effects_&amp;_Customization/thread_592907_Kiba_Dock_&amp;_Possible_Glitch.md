### Post by TheWired on 2007-10-31
I am still new to kiba-dock but I did notice that there is a setting that groups up windows like you described. You can go to Taskbar and uncheck Group Icons. There is also a Group Icons setting under Launchers (it all depends on what you don't want grouped together). 

As for your settings not being saved, make sure you have the proper permissions to save. I wouldn't understand why the config file wouldn't save if the permissions were set properly. Whatever use you are running kiba-dock from it creates a .kiba-dock/config file for. That file is edited by kiba-settings. Hope some of this is helpful.
