### Post by pytheas22 on 2008-01-23
When I move a window close to the edge of the screen, it tends to sort of "snap" or "stick" itself to the very edge of the screen, making itself exactly even with the nearest edge of the screen.  I'd like it not to do that, as it makes it difficult to place windows exactly where I need them to be.

I'm sorry if the terms I'm using to describe the behavior are vague; I'm not sure what this feature of compiz is called.  Consequently, I tried searching for a solution but wasn't sure what terms to use and found no useful results.  If I should clarify what behavior I'm trying to describe, I would be happy to do that.

If there's a plugin that I could disable, or at least change the value of, so that windows will stay exactly where I place them, without feeling a need to snap themselves to the edge of the screen, please tell me which one it is.  I would like to keep my wobbly windows if possible; I just don't like the snapping-into-place feature (which I don't remember seeing in earlier versions of compiz, incidentally).

I am running compiz-fusion as it was installed by default with Gutsy.

Thanks for any help.

### Post by john.nicholls on 2008-01-23
Under Window Management in Compiz you'll find a feature called Snapping Windows. Deselect this or change the options as you require.

---

### Post by pytheas22 on 2008-01-23
thanks for your response.  Actually it turns out that the snapping continues as long as wobbly windows is on, until you disable the "snap inverted" option in wobbly windows.  But now it's better.  Thanks for your help.
