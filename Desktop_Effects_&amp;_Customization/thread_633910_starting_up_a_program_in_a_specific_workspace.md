### Post by jsully1 on 2007-12-07
Compiz can do this out of the box.  If you go to the Place Windows plugin, go to fixed window placement, and then windows with fixed viewport. 
class=Pidgin with an X viewport position of 1 will always open Pidgin on desktop 2
class=Thunderbird-bin with an X viewport position of 2 will always open Thunderbird on desktop 3.

There are other things you can use besides class - [this link](http://wiki.compiz-fusion.org/WindowMatching) details some of them, and how to use xprop to get the properties of a window so you can make your own rules.
