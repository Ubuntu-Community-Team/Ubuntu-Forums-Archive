gksudo gedit /etc/apt/sources.list
```Add the following line to the file and save```
deb http://hendrik.kaju.pri.ee/ubuntu feisty screenlets

wget http://hendrik.kaju.pri.ee/ubuntu/F854AFD7.gpg -O- | sudo apt-key add - && sudo apt-get update

### Post by mozetti on 2007-12-10
In the preferences menu, start the screenlets manager. Then select which screenlet you want to run and whether you want it to run at startup.

You can download more screenlets from it's website. After you download a screenlet, un-tar it to the .screenlets folder in your /home folder - at least, that's where it was for me, but I had installed them under Feisty and the folder is from that original installation. When you open the Screenlets Manager the new screenlets you un-tarred will be available to you.
