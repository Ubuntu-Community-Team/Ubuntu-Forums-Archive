### Post by liquidfunk on 2007-06-26
Yes you can. Notice there are 2 bars within Ubuntu. This Desktop environment is called Gnome, you will hear people talking about different Desktop environments, some popular ones are Gnome, KDE, Fluxbox, XFCE. They are all reasonably similar, but some have different properties to others. 

 Anyway, back on with it.

 If you right click on either panel (top or bottom) you can select Properties, within which you can change and customize your bar to whatever you want. Can be see- thru, can have an image on it, can be like a Mac. By clicking Add To Panel, you can add shortcuts to a panel. If you want to use one panel, make sure you add the Menu button onto the panel, (else you may find it hard to access things on your computer). 

 This is a link to an image of my desktop, to show you what can be done by customizing.

 [http://ubuntuforums.org/g/images/302693/1_Screenshot.jpg](http://ubuntuforums.org/g/images/302693/1_Screenshot.jpg)

 I hope this has helped, I always find its easier to get bits of help and then discover the rest for yourself.
