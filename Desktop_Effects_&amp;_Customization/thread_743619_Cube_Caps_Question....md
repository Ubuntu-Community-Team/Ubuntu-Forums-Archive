### Post by mcduck on 2008-04-04
You can't use them as workspaces because of their different dimensions/shapes.

For example, if you are using 1024x768 resolution the caps are 1024x1024. And if you happen to have more or less than 4 workspaces the caps will not even be rectangular which would make it pretty hard to use as desktops ;)

What you can do with the caps, instead, is to define a larger set of images as cube caps, and then use keyboard shortcuts to browse through them. Way cooler way to do a presentation than using PowerPoint/Impression :D
