### Post by Inxsible on 2007-09-07
Do you have Compiz Fusion installed ?

Because there are different ways to set 4 desktops depending on whether you have CF or just regular Compiz.

The update probably got you a new kernel. It is always a good idea to keep at least one old kernel around just in case the new one doesn't work, which in your case is true. However, you can keep it installed but not show it in the bootloader by commenting it out in the /boot/grub/menu.lst file

If you really want to remove the kernel from your system, you should use synaptic and search for the version number that you want to remove and then mark for removal.

As for the sound, lets start with the basic. type in ```
alsamixer
``` and see if any of the volume outputs are muted.
