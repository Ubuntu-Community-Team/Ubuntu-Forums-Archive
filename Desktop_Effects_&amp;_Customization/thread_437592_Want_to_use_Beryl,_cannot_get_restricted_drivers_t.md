### Post by Hydrargyri on 2007-05-09
I fixed it! :)

After reading this guide: [http://doc.gwos.org/index.php/Latest_nvidia_feisty](http://doc.gwos.org/index.php/Latest_nvidia_feisty)

I uninstalled nvidia-glx and installed nvidia-glx-new. It worked! :)

(I'm posting this in case if someone else had troubles. :))
