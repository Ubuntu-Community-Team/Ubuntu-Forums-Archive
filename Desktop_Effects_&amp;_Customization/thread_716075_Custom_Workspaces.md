### Post by LexRoss on 2008-03-05
The 'true UNIX way' of doing so (as with any serious customization or project separation task) would be creating different users for each activity. So you can have users called 'games', 'web' etc. each with different theme and menu structure, and switch between them with fast user switch applet for instance. This way you have greater control on the system and can focus on specific things (files and applications) relevant to the project. Put  users into the same group if you need to share files between them. Hope this helps.
