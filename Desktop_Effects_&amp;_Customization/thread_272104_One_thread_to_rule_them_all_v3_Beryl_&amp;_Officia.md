### Post by bodhi.zazen on 2007-06-09
Hi PriceChild

Thanks for all the work you do with Ubuntu/Beryl

One suggestion, for nvidia cards of course.

To configure X for beryl there is a series of commands that does the trick clean and fast :

```
sudo nvidia-xconfig

 sudo nvidia-xconfig -composite; sudo nvidia-xconfig -allow-glx-with-composite
 sudo nvidia-xconfig -render-accel; sudo nvidia-xconfig -add-argb-glx-visuals
```

The first command configures X AND makes a back up of xorg.conf if needed

The second is all 1 long line, or a series of commands separated by ; 

I hope that information will help some folks here.

Edit ~ sorry if this is not the proper place to post. I know manual configuration is not a common as it once was, but those commands are sometimes quite helpful.

### Post by Jouke74 on 2007-08-30
Old compiz (more resource friendly for sure):

- After you have set-up your XGL or AIXGL composite (using any of the guides that is appropriate to your video card) and you have composite enabled, you can just choose "Enable desktop effects" under the Preferences menu. Given you have an Intel onboard videocard you can even just do "Enable desktop effects".

To change settings using this Compiz version, is a bit more tricky, but all the options are found under 
1. [ALT]-[F2]
2. Type: "gconf-editor"
3. Look for Apps>Compiz, there you will find all settings. 

To download a graphical front-end with the basic settings you can install Gnome Compiz Manager. However I noticed that some of the settings here are overwritten by the "Enable desktop effects program". So they wont hold on reboot. In Gconf-editor they do (but you need to know what you're doing (or simply try like me)). In the old Compiz version the "wobbly" plugin needs to be enabled, otherwise you wont be able to unmaximize Firefox. But you can reduce it's effects under the options in gconf-editor.
