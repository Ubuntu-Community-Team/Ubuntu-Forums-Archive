### Post by Forlong on 2007-11-25
The easiest way is through the context menu. Choose Properties and then click on the icon. There you will be able to choose an icon.
But it will change only the icon you (right) clicked on.

Another solution would be going to /usr/share/icons and duplicate the icon set you want to modify. Then change the icons there.
