### Post by jsmidt on 2007-03-24
**Step 1**:
Take a backup of your xorg.conf file so that you can go back to your earlier setup just in case. Open a Terminal and type:
Quote:
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak

**Step 2**:
Edit your xorg.conf file to apply the necessary changes:

Find:

> Section "Device"
make sure i810 driver is used, if not change it:

> Driver "i810"

and before EndSection add:

> Option "XAANoOffscreenPixmaps" "true"
Finally go to the end of the file and add: (the first may be there already)

> Section "DRI"
Mode 0666
EndSection

Section "Extensions"
Option "Composite" "Enable"
EndSection

Save the file and close Gedit.

Now restart the computer.

Note:  Restarting the x-server may be good enough but for me I had to reboot.

Login again and enable compiz by System -> Preferences -> Desktop Effects.
You should have compiz running now! You will probably want to install a config utility as well, try Gnome-Compiz-Manager or Compiz-Settings.
