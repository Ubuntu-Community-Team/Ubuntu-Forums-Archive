### Post by Aeph on 2008-02-22
If you can, you first need to go to System->Administrator->Software Sources. You might as well enable all of the check boxes on the first page. Then go to System->Administrator->Restricted Drivers Manager and enable your GPU. You will have to restart your computer here. Once Ubuntu is loaded again go to System->Administrator->Synaptic Package Manager and look for a package called compizconfig-settings-manager. Click the box to its left and install it. Then go to System->Preferences->Advanced Desktop Effects Settings. Enable the cube related things and to to General Options at the top. Go to the Desktop Size tab and set the Horizontal Virtual Size to 4. And that should do it.
