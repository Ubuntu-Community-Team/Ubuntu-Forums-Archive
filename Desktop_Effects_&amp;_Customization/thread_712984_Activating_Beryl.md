### Post by ajgreeny on 2008-03-02
Why do you want beryl, which is no longer being worked on, instead of compiz-fusion which is already installed in gutsy?  If you just want to know how to get the cube etc etc, go to System>Preferences>Appearance and set to the way you want it on the Visual Effects tab.  You should also install compiz-config-settings-manager which will give you a menu item called Advanced Desktop Effects Settings where you can edit all the detailed effects you want.
Don't forget, however, some video cards are not able to run 3d desktop effects at all, and some others need proprietary drivers to run them.  Go to System>Administration>Restricted Drivers Manager and see if you can enable them.
