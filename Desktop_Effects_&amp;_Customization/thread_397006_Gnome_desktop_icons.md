### Post by merlyn on 2007-03-30
One little trick for the icons on your desktop is that you can right click and select "Stretch icon".

Handles will then appear on each corner. Stretch and / or shrink to your hearts content.

### Post by SuSUntu on 2007-03-30
Open Nautilus. Then open the menu:

Edit > Preferences > Views

From there. set / select the icon **Default Zoom Level** under the section, **Icon View Defaults**. Default is 100% (makes sense, I suppose), so try setting it to 75% or 50%.

That setting will change the icon sizes in the Icon View of Nautilus, and it will change the size of icons on your Desktop. If you don't want the change to be that 'universal' or if you need to adjust individual icons on the Desktop for whatever reason, you'll have to use the icon "stretch" method described by Merlyn.
