### Post by rainwalker on 2007-12-26
Download the theme (usually a .tar.gz), then go to System > Preferences > Appearance (or for Feisty, System > Preferences > Theme) and drag that theme file into the window, and it will install it
The same goes for icon and metacity themes.

For GDM (login screen) themes, open up System > Administration > Login Window, switch to the "Local" tab, and drag the theme into that window to install it.
