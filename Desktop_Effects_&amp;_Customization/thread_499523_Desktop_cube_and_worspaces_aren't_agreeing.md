### Post by andrewsomething on 2007-07-14
For what ever reason, the Gnome workplace switcher and Compiz don't actually play very well together. In order for the workplace switcher to function with Compiz, you need to set its preferences to only one workplace.(I know that's counter intuitive.) Then in the Compiz settings manager goto General Options>Desktop Size, set the value for Horizontal Virtual Size to four. That should put you back to normal....
