### Post by dexter on 2007-06-29
I had the same issue. The problem is caused by the modified xorg.conf file by nvidia-settings. If you let nvidia-settings overwrite it, window borders are gone. Restore your old xorg.conf (nvidia-settings creates a backup) and add the new resolution manually.

### Post by hyperair on 2007-06-30
From the xorg.conf posted above, you're using the open source nv drivers instead of the nvidia proprietary drivers? That's a big no-no if you want to run the desktop effects.

You would have to run ```
sudo nvidia-xconfig --add-argb-glx-visuals
``` if you intend to run Beryl/Compiz/Desktop Effects/Compiz Fusion.
