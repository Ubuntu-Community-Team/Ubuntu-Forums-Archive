### Post by Teamgeist on 2007-07-17
Hi that is a know issue which is caused by a bug in the NVIDIA Driver.
But read for yourself (workaround for Beryl included ;) :
[http://wiki.beryl-project.org/wiki/Troubleshooting_nVidia#Opening_too_many_windows_leaves_some_of_them_black](http://wiki.beryl-project.org/wiki/Troubleshooting_nVidia#Opening_too_many_windows_leaves_some_of_them_black)

Another solution might also be installing the newest NVIDIA Driver.
A good method for doing that is using Envy.

[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)
