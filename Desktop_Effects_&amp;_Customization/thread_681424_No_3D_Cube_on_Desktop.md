### Post by erginemr on 2008-01-28
You need to install the package "compizconfig-settings-manager" with:
```
sudo aptitude install compizconfig-settings-manager
```

When you install it, you will have a new button inside your desktop effects dialog, from where you can enable and configure the 3D desktop. You also need to make sure that you have 4 virtual desktops (to be set from the right hand corner of your bottom panel).
