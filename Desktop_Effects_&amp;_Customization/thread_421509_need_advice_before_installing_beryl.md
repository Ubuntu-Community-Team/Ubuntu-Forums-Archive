### Post by da5id. on 2007-04-24
I can't comment on your specifc setup, but i am running Feisty with GDM and just installed Beryl today - no problem !! works great

I followed these instructions: (tweaks for KDE and XFCE are included in instructions)
[http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL](http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL)


AFAIK if you have 3D hardware acceleration it's good to go ??
Try this: (from the install instructions)
```
You can check whether direct rendering is enabled by invoking

glxinfo | grep direct

in a terminal. If it returns

direct rendering: Yes

For ATi graphics cards you can check with:-

 fglrxinfo
```

My advice would be give it a try - Beryl is simply an awesome addition to the desktop.
