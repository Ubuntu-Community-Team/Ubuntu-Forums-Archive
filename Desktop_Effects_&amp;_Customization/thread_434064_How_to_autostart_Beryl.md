### Post by benanzo on 2007-05-05
If you are in GNOME go the System>>Preferencees>>Sessions and under Startup Programs select New on the right and add an entry called beryl with command beryl-manager.  Then make sure the box is checked next to it.  Beryl will launch whenever your GNOME session starts now.  I'm not sure if KDE (haven't used it in awhile) but I imagine it's similar.
