### Post by gimfred on 2007-06-26
Okay, you must then know how to get to a terminal. You **will not** need to use the sudo command.

Type in the terminal:

```
glxinfo | grep direct
```

It would also help if you could post the contents of your xorg.conf file.

Type:

```
cat /etc/X11/xorg.conf > ~/Desktop/xorg.conf.txt
```

Then just load it from your desktop.
