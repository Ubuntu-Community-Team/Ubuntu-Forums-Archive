### Post by PriceChild on 2007-03-16
*ubuntu will have the desktop-effects package installed by default. This includes compiz. You will simply have to enable them from a gui interface off of system>prefs>desktop effects. Nvidia users will be able to use the restricted manager to set up their nvidia graphics drivers. Intel and ati drivers won't need to set up drivers.
