### Post by Thunar on 2007-10-29
go to Applications -> Accessories -> Terminal and type in this:

```

lspci

```

this should let you know what video card you have. 

After that, still in terminal, type this in:

```

cat /etc/X11/xorg.conf

```

this will display the contents of your xorg.conf file and tell you which drivers you are using, among other things.

Post the results of both of these commands here and we'll take it from there.
