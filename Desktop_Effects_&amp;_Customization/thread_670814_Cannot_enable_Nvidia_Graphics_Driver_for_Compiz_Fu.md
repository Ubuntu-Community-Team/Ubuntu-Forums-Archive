### Post by FuturePilot on 2008-01-18
Go System>Administration>Software Sources and make sure you have all the boxes checked in the first tab. Uncheck the CD-ROM one if it is checked. Go to the Updates tab and check the first two and optionally the fourth one. Click Close and Reload the sources when prompted. Then try the Restricted Driver Manager again.
