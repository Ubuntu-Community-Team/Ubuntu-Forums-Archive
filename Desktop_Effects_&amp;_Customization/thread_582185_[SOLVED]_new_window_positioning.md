### Post by mkzimms on 2007-10-19
i had the same problem... and your right it was extremely annoying! my fix involved:

Open compiz manager (system -> preferences -> advanced desktop effects settings).
Scroll down to window placement and enable place windows. 
Open up place windows settings.
Check workarounds and set placement mode to centered.
