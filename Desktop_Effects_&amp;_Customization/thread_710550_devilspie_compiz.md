### Post by ayoli on 2008-02-28
Menu system>preferences>appearence then "desktop effects" tab, pick "none" (or disable, I don't know the exact english label, btw it is the first choice) should be enough to havev a gnome desktop without compiz.
(Note that compiz can place windows on specified workspaces)

### Post by hyperair on 2008-02-29
If you configure the "Place Windows" plugin in System->Preference->Advanced Desktop Effects Settings, you should be able to do what you want, without devilspie. Note: the menu item only appears after installing "compizconfig-settings-manager"
