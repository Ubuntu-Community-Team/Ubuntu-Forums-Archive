### Post by Zorael on 2008-04-19
You can head over to [gnome-look.org](http://gnome-look.org) and search for 'usplash' to find splash themes. As for changing them, there is an application in the repositories named Startup Manager (package name **startupmanager**) with which you can select which theme to apply.
