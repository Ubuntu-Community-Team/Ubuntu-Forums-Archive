### Post by Deco_21 on 2007-07-18
Try 
```
beryl-manager
```

Choose Select Window Manager > Compiz
Choose Select Window Decoration > "Anything with GTK"

You must have beryl in order to make this work . Its the easiest way.




If you already have beryl

```
compiz --replace -c emerald &
```

or
```
compiz --replace 
```
