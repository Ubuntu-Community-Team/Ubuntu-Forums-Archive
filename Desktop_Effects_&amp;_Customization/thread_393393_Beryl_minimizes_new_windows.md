### Post by richbiskit on 2007-03-25
Using Beryl in feisty this happens....

I open a window (any window), If i open another window from a drop down say, applications or system, the new window is put straight into the toolbar at the bottom, this also happen with apps like Amrok. If i open Amrok and then try to open say the settings panel and then say, configure amrok, the new window opens behind the original so i have to minimize it then maximize it to see it.

Same if i have a web browser open, if i try to open a window from apps os places or system it goes staight to the toolbar.

This ONLY happens when using Beryl as the manager.

Any ideas?

---

### Post by richbiskit on 2007-03-25
Done it!!!!! :)

Beryl Settings Manager > General Settings > Level of Focus Stealing Prevention=none
