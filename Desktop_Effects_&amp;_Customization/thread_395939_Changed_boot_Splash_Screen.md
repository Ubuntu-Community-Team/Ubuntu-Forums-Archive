### Post by nike984 on 2007-03-29
install 'gnome splash screen manager' using synaptic,
then in Panel menu, System=>preference => Splash screen

choose any splash screen file that you want and install
and activate it. It's all that simple.

p.s. You can find the splash screen file at gnome-look.org
