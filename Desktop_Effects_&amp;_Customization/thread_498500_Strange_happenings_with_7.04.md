### Post by Happy_Man on 2007-07-11
Install the package gnome-compiz-manager ```
sudo apt-get install gnome-compiz-manager
``` and go to its settings under System --> Preferences --> GL Desktop. Go into the cube settings and see if the key combo for cube was changed. Sometimes they do that.
