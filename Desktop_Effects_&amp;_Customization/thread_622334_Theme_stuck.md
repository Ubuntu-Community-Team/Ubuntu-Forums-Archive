Those are simply previews. Emerald is only responsible for the window boarders. If you want to change the entire theme, you need to change the GTK theme.

### Post by rainwalker on 2007-11-24
System > Preferences > Appearance
For whatever theme you're using, click the "Customize" button (it's below the list of all the themes)
Choose whatever control theme you want, in that screenshot it's set to the Human controls.
Emerald is only responsible for the window borders.
