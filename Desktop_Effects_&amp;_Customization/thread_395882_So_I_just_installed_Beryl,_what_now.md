All of these are in the settings manager:
1. Visual Effects -> Animations
2. Desktop -> Desktop Cube -> Skydome tab
3. Desktop -> Desktop Cube -> Transparency
4. You skipped it
5. a) If you want windows to be transparent according to focus order, you can use the Trailfocus plugin: Visual Effects -> Trailfocus.  (this can also be used to set things to be transparent by default, just set the opacity of focused windows as well in the Trailfocus options)
5. b) i. If you want windows to be transparent when you hover over them to see underneath focused windows, you can use the Opacify plugin: Accessibility -> Opacify
5. b) ii. AddHelper provides a similar functionality: Accessibility -> AddHelper
