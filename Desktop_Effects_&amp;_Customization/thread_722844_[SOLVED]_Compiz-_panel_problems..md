### Post by Shazaam on 2008-03-12
I have Compiz-Fusion with Emerald and CCSM running fine (love that cube!) with Gutsy but I have a few annoyances that I can't figure out.

1. When an app has an icon (gnome-ppp, update notification, etc) showing in the Notification Area it does NOT react to being clicked on or hovered over. Nothing happens so if I want to disconnect gnome-ppp I have to Ctrl+Alt+Backspace or disable desktop effects to shut it off. When the update notification icon pops up I don't get any reaction when I click it with the mouse (no tooltip either).
2. Similar problem with the Window List but that's intermittent. For example if I have multiple programs open at once, if I minimize some of them (it only affects one or two) and then try to restore them by clicking on them in the list nothing happens.

I can work around the problems by having gnome-ppp open with the terminal and by not minimizing any open programs. Or I can disable Desktop Effect all together. Anyone have a clue? Some setting I missed?

---

### Post by chewearn on 2008-03-13
It is compiz freewins plugin.  Disable that, and the problem will go away.  Unfortunately, you also loose freewins functionalities.

### Post by Shazaam on 2008-03-14
That seems to have done the trick so I am going to mark the thread solved. Thank you for your help!
