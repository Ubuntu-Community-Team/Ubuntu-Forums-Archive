### Post by Forlong on 2008-01-07
That means you are using Metacity (GNOME's default window manager) - unfortunatley I can't find any option to turn this off.

You could use Xfwm (Xfce's window manager) with GNOME, if you like. That doesn't have this kind of behaviour.
