### Post by FuturePilot on 2007-10-03
No that's not a bug. It's normal. It's because the theme is installed locally. So when you go to launch a root application such as Synaptic the theme isn't applied. You'll find this happens with any theme that you installed yourself. I found a thread on this. 
[http://ubuntuforums.org/showthread.php?t=83632]("http://ubuntuforums.org/showthread.php?t=83632")
