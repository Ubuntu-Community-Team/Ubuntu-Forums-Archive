### Post by Nikron on 2007-03-25
It's a very well documented bug...  If you have nvidia drivers, you need to either run 
sudo nvidia-xconfig --add-argb-glx-visuals

or you manually add the options

    Option         "AddARGBGLXVisuals" "True"
    Option         "DisableGLXRootClipping" "True"


to either screen or device section or both.

Then restart CTRL+ALT+DEL

Note: this is only if you have nvidia.

### Post by mthaddon on 2007-03-25
Here's the relevant section from my xorg.conf:

```
Section "Device"
        Identifier      "nVidia Corporation NV17 [GeForce4 420 Go 32M/GeForce 44
0 Go 64M]"
        Driver          "nvidia"
        BusID           "PCI:1:0:0"
        Option          "AddARGBGLXVisuals"             "True"
        Option          "XAANoOffscreenPixmaps"         "true"
EndSection

```

I wouldn't change anything else, but it seems that you would want to put the AddARGBGLXVisuals in the nvidia device section.
