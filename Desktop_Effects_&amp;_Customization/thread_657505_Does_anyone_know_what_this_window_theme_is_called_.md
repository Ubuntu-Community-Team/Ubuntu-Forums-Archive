### Post by rainwalker on 2008-01-03
Yes, it does make a difference because Emerald will only work with Compiz/Beryl.
Emerald is like Metacity (window borders), but fancier in that it can do lots of transparency and fancy effects.
You can install it with Synaptic, just search for "emerald" and install it. It will be under System > Preferences > Emerald Theme Manager
To install the theme, save it somewhere (like your desktop, because you'll only need it once) and then in Emerald, click on "Import" and install the theme.

To use Emerald, log out and log back in. If you want to go back to Metacity, all you have to do is uninstall Emerald with Synapic.
