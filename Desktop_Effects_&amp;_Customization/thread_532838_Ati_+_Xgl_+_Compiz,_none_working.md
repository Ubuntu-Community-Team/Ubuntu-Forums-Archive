### Post by Jouke74 on 2007-08-25
Seems like your ATI drivers are installed ok. Now it is time to do the second part of the set-up: namely installing the XGL server, and change your session from a X-server session to an XGL server session. The XGL server will show a different mouse cursor on start-up. To do this you can use the guide that you pointed out. It is essential that all steps work ok!

sudo apt-get install xserver-xgl

Good now, XGL won't load on its own so we need to write a few scripts to have it start.

Code:

sudo gedit /usr/local/bin/startxgl.sh

Now copy and paste this into the file that pops up:

Code:

#!/bin/sh
Xgl :1 -fullscreen -ac -br -accel xv:pbuffer -accel glx:pbuffer &
DISPLAY=:1
cookie="$(xauth -i nextract - :0 | cut -d ' ' -f 9)"
xauth -i add :1 . "$cookie"
exec dbus-launch --exit-with-session gnome-session

SAVE THIS FILE! Once its done saving, make that file executable (like a program) by pasting this into a terminal:

Code:

sudo chmod a+x /usr/local/bin/startxgl.sh

Now we need to make a way to start that session from your login menu, paste this into a terminal:

Code:

sudo gedit /usr/share/xsessions/xgl.desktop

And paste this into the file:

Code:

[Desktop Entry]
Encoding=UTF-8
Name=GNOME with XGL
Comment=
Exec=/usr/local/bin/startxgl.sh
Icon=
Type=Application

Now make this script executable by pasting this into a terminal:

Code:

sudo chmod a+x /usr/share/xsessions/xgl.desktop

Now log out of your session.
Click on the Session menu in the login screen
And change for Gnome session with XGL.

First check if this is working with the old compiz (it should, and you will be able to use most effects already). Go to System > Preferences > Desktop effects. If you click on enbale the screen should flicker and they should be enabled. If they say "composite extension is not avialable" then your XGL server is not set-up correctly.

Now you can try to install compiz fusion. But given the many problems people have with it recently I suggest you play around with the old one for a while.

Good luck.

PS. (Code should be typed in the gnome terminal).
