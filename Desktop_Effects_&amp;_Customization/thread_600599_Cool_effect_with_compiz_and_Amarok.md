### Post by Rob Campbell on 2007-11-02
Hi,

Just though I'd share a nice effect I've set up with Compiz and Amarok:
Open a visualisation window and right-click on title-bar to get the window options. Set it to "keep below others" and "full-screen". Now set the opacity of the window to a low value so that the desktop background shows through. I do that with Alt+mousewheel, which is the default in Compiz. Looks awesome if your desktop isn't cluttered with windows and you can see the background.

### Post by Rob Campbell on 2007-11-02
Hmmm. strang. I mean the title-bar the top of the window. i.e. the thing with the minimize, maximime, and close buttons. If you right-click on that (in KDE) you should get a menu with an advanced tab with further options. I don't know if you're using Gnome or not but I imagine there must be something similar for that too. The idea is simply to have the window in the background and have it stay there and overlay the effects on top of the desktop background. If there's no option for full-screen that's no big deal but it means that you'll have the window border visible. If you increase the opacity you won't notice too much, to be honest.
