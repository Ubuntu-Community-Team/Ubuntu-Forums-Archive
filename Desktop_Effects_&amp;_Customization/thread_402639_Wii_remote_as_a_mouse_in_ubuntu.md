### Post by reclusivemonkey on 2007-04-06
This would be a good place to start;

[http://www.wiili.org/index.php/WMD](http://www.wiili.org/index.php/WMD)

AFAIK, the motions of the Wii remote are transferred by bluetooth, and so this is a relatively easy task to set up controls using that function. However, the "pointing" function of the Wii remote is done via infrared and the sensor bar, which is not quite as easy.  However I think there are a number of projects setting out to acheive what you want.

I personally managed to set up my Wii remote to control MythTV; it was fairly simple.

[http://www.mythtv.org/wiki/index.php/Controlling_MythTV_using_a_Wii_remote](http://www.mythtv.org/wiki/index.php/Controlling_MythTV_using_a_Wii_remote)

Good luck =]
