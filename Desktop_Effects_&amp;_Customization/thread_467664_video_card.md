### Post by crane on 2007-06-08
You could type lspci in a terminal, it will list some information about the hardware. My listed this at the bottom of the list.
01:00.0 VGA compatible controller: nVidia Corporation NV43 [GeForce 6600/GeForce 6600 GT] (rev a2)

### Post by DirtDawg on 2007-06-08
I like Sysinfo. It tells you all kinds of things with a nice GUI. It's in the repos.
```
 sudo apt-get install sysinfo 
```
