### Post by aantetomaso on 2007-08-10
Hello,

I've just installed the amazing compiz fusion with Emerald theme manager. They seem to work perfectly together, but with compiz fusion and emerald both active I have troubles in changing Icon theme. I simply download a theme from [www.gnome-look.org](www.gnome-look.org) and get a tar.bz2 archive. I, then, open System-->Administration-->Themes and try to add the new theme. It seems that the archive is right but unfortunately the icons don't change. If I try to decompress the archive and point the theme manager to the ".theme" file it says it's a not valid theme.

Could please someone explain how to install a new icon theme, having compiz fusion and Emerald both active ?

I 'm using Ubuntu 7.04 feisty fawn. Thanks in advance.

Bye.

---

### Post by pointone on 2007-08-10
Go to System > Preferences > Themes and click Customize; there should be an Icons tab where you can select which icon theme you desire.
