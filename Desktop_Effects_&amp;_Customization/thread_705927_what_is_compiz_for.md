### Post by fracturedmorals on 2008-02-23
> **RJ Hythloday said:**
> So I think it looks cool and all, but really how can you even use the screens when they are in a cube or some other wierd shape? Is it just for effects and posting screen shots? or is there something really useful that I'm just not seeing?

Here's the deal:

Regardless of the cube, having multiple virtual desktops (Although possible in a windows-like environment) are hardly useful unless you're using an OS that does a context switch faster than 1.5 seconds.

The cube is a way to more graphically represent the concept of "Oh hey....wow....I can do more than one thing at the same time.....COOL".

Compiz itself does not require a cube. The "Desktop Wall" plugin is probably more useful....and the window switcher utilities are much nicer than using a standard non-compositing WM.

Yes, the primary focus is to make things look slicker, but there are definitely at least a few advantages in usability.

### Post by zenkaon on 2008-02-24
There is one killer effect for increasing productivity - slide.

Enable this and set it up so that it shows all of your applications when you move your mouse to the bottom right (or wherever you like). After using this effect for a while, you won't want to go back, try disabling it and see how much slower everything seems.
