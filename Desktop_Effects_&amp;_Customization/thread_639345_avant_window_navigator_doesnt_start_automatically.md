### Post by mozetti on 2007-12-13
Go to Preferences->Sessions and the Startup Programs tab and enter ```
avant-window-navigator
```

Just to let you know, this is how I have it set up and it works. However, AWN seems to take awhile to load (a minute or two) whenever I restart the computer.
