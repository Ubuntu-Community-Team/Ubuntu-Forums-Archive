### Post by tvst on 2007-06-08
Run "applications > system tools > beryl manager", then right-click on the red icon in the notification area near the clock, choose "select window decorator", then "heliodor".  You can quit beryl manager when you're done.

If next time you log in it goes back to emerald, open "applications > system tools > beryl settings manager", then under "visual effects" choose "window decoration" and type "heliodor" (without quotes) under "command line"
