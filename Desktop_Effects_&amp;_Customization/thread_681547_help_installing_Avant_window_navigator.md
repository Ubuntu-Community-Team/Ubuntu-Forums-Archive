### Post by ScottKidder on 2008-01-29
[http://wiki.awn-project.org/index.php?title=Installation](http://wiki.awn-project.org/index.php?title=Installation) has a full install guide with instructions on how to install from binaries or from source whichever you prefer. If you do decide to install from source you need to do sudo apt-get install bzr and then run bzr co [http://bazaar.launchpad.org/~awn-core/awn/trunk](http://bazaar.launchpad.org/~awn-core/awn/trunk) avant-window-navigator, but follow those instructions, they'll serve you right.
