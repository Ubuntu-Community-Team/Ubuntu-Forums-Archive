### Post by astoltz on 2007-06-02
Try "beryl-manager" in the terminal (or the run dialog).  That should start the familiar red gem in the notification area.  If there is still no borders, try right clicking on the gem, then "Select Window Decorator", then pick emerald.

### Post by zanglang on 2007-06-02
Did you untick 'Windows decorations' in your Beryl settings?

Otherwise, try installing Heliodor or Emerald ('aptitude install heliodor / emerald'), then start up Beryl using the Beryl Manager from Applications -> System Tools (or did that not work?), right-click the icon now in your notification area, and then choose Heliodor or Emerald from the window manager menu.
