### Post by Dark Hornet on 2007-09-26
Question...

1.  Are you running Ubuntu 7.04, and if so...are you using the "desktop extras" to get the Compiz, or did you actually download and install Compiz Fusion?

Other than that, to make the changes you are talking about simply go to System>Preferences>Compiz settings manager (or something like that):

to change the background color, go to the section where you change the "caps" pics, and there should be an option to change the colors...I think the default is a light blue and a tan sort of color.

to change the background where the cube is, make sure that skydome is enabled, and you can find this under the "cube" portion in the compiz settings manager.  Once in the skydome section, there is a place to change the background pic.


Sorry if I am a little off on my directions, but I am at work right now, and not infront of my Ubuntu box....hope this helps, and good luck.
