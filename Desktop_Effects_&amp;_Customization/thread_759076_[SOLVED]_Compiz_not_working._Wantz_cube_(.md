### Post by pseudo-random on 2008-04-18
If you've got an ATI card then really you need an ATI driver. I can't see any reference to fglrx in there, like this for example:

```
Section "Device"
	Identifier  "Configured Video Device"
	Driver      "fglrx"
EndSection
```

Try installing the restricted driver from System > Administration > Hardware.
