### Post by ajeetraj on 2008-03-21
yes you do need to set up the visual effects by yourself to get the cube working. To do that, you should install compizconfig-manager by typing this in the terminal: 
```
sudo apt-get install compizconfig-manager
```

After this, you should go to System>Preferences>theme manager and then go to "visual effects" tab and then change the effects the way you want it OR, 

you can just go to System>Preferences>visual effect... and then just check the desktop cube and then you will have the cube working. 

also, by default there are only two desktops enabled in ubuntu so you might want to go to the general options and change the number of desktops to 4 from 2. 

Well, regarding the requirements for the visual effects, you can easily check if it works smoothly on your computer or not after trying it out. As for me, it works great and I have Dell Latitude D630. 

if you have more questions then plesae don't hesitate to ask.
