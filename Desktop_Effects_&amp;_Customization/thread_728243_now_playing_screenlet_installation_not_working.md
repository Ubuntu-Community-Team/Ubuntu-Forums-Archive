### Post by rune0077 on 2008-03-18
You don't actually have to install the screenlet, you can just unpack it and make sure that all the files ends up in a folder with the name of the screenlet (in your case, NowPlaying) ... then, in your home directory there should be a hidden folder called .screenlets, open that, and drop the NowPlaying folder in here. Next time you start up screenlets, it should be there.
