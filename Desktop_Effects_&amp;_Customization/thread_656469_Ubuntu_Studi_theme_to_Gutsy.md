### Post by Mr Greencastle on 2008-01-02
Its actually possible to convert your entire ubuntu install into an Ubuntu Studio desktop. Simply paste this command into a terminal:
```
sudo apt-get install ubuntustudio-desktop
```
This will also install the programs that make Ubuntu Studio. Good Luck!
