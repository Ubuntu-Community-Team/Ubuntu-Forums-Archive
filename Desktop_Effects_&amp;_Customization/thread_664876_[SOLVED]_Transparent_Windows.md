### Post by DaymItzJack on 2008-01-11
Open your terminal:

Edit -> Profiles... -> (check your profile) Edit

Colors tab - Built-in Shemes: "Green on Black"
Effects tab - Transparent background (I keep mine about 50%.)

Note that the transparent won't work unless you're using compiz, if you're not it's just going to show the desktop background in an ugly fashion.

### Post by tkboy85 on 2008-01-17
Well actually, there's another way to make any window you want transparent.

Put you mouse over the window you want, then hold down the alt key and use the mouse scroll to set the desired transparency.

I found it out accidentally when trying out all combos =)
