### Post by Therion on 2008-03-10
Sure.

I'm going by memory here, since I'm stuck behind a WinXP machine at my office, but go to System/Preferences/Appearance; as if you were going to change your theme.  Click on the "Customize" button and then you should have the tabs at the top, one of which is for the Icons specifically.  You can click on that tab and choose your icon set from there.

You can try installing new icons if you want, something [like these](http://www.crystalxp.net/galerie/en.2.1.12.cat.1.html) for instance, but you have to replace the old icon with the new.
