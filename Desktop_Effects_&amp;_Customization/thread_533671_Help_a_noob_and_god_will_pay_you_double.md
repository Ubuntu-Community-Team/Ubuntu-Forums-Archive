### Post by Rupertronco on 2007-08-28
tiagovicentao, keep in mind, Beryl is alpha software. It's not ready to be considered fully operative or even fully compatible with everything. Too many people base their experience with linux on software that isn't even ready for official release yet. 

If you have beryl working now that's great, and if you'd like to keep up with the 3d desktops you could eventually try compiz-fusion (which is what the beryl project turned into). It's a bit more stable and has some different effects. 

Also, if you didn't get emerald themes with beryl you should try that out as well, it's a window themer that is unquestionably the best.
