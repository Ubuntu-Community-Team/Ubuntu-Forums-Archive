### Post by Haegin on 2007-07-25
For widgets I think you can use screenlets ([http://hendrik.kaju.pri.ee/screenlets/](http://hendrik.kaju.pri.ee/screenlets/)) and then use the widget plugin in compiz to make it all work more like the mac.

For the background the best I can suggest is modifying the theme to make the background image transparent. You will need to find out how to set a different image for the panel background and then get a transparent image.

Good luck.

### Post by Espreon on 2007-07-25
After you install AWN, do the following to make it look like Leopard's dock:

Goto>Applications>System Tools?Configuration Editor

Then goto apps>avant-window-navigator>bar and look for the value bar angle and right click and hit edit value and change it to 45. The look for icon offset and change the value to 15 and you are done (after closing the editor).

It works for me as I use it all the time.
