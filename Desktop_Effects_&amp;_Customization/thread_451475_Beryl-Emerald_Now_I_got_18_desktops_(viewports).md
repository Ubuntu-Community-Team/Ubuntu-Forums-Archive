### Post by Brackish_zygote on 2007-05-22
The number of sides to the cube isn't determined by the number of desktops setting. If you look a few options up from the Number of desktop setting, you should see a setting for Horizontal Virtual size. That is the option you want to change if you want more/less desktops.

### Post by Goombie on 2007-05-22
The desktop configuration options for Beryl can be found in the settings manager, under the category General Options --> General Options (a bit redundant, but that's where they put them). Under the 'Main' tab, you'll see three sliders, among others: Horizontal Virtual Size, Vertical Virtual Size, and Number of Desktops. Keep the number of desktops and Vertical Virtual Size at 1. You can change the Horizontal Virtual Size setting to determine how many desktops (viewports? that confuses me too) you see when you're viewing the desktop cube.
