### Post by DjBones on 2007-11-15
well, you can go to:
System > Administration > Synaptic Package Manager
(put in your password)
then click on the search button.. and type 'emerald'.. check the box for 'emerald' (which you want to 'install').. then hit apply

or, you can go to:
Applications > Accessories > Terminal
and enter:
```
sudo apt-get install emerald
```
