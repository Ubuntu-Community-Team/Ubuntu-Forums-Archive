### Post by dmizer on 2007-07-13
fixed.

open terminal and type:
```
xfwm4
```
and hit enter.

be sure that "save sesson" is checked when logging out.

solution found here: [http://wiki.xfce.org/faq.nl](http://wiki.xfce.org/faq.nl)
