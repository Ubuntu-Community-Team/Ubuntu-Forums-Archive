### Post by ThrobbingBrain66 on 2007-08-30
In CompizConfig Settings Manager go to General Options > Desktop Size
If you don't have it installed yet:
```
sudo apt-get install compizconfig-settings-manager
```

Horizontal Virtual Size should be 4
Vertical Virtual Size should be 1
Number of Desktops should be 1
