### Post by FuturePilot on 2007-10-20
If you're using Feisty you can go System>Preferences>Themes and drag the archive right into the theme manager. If you're using Gutsy you can go System>Preferences>Appearance and drag an drop the archive right into the theme manager.

### Post by Ub1476 on 2007-10-20
For those that gave an invalid error, extract the archive, copy the folder and go to your home directory and put it inside the .icons folder. Folders with a . in front of them are hidden, so press CTRL+h to show it.
