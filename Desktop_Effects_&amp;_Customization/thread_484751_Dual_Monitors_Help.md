```
sudo startx
```

```
cd /etc/X11
sudo mv xorg.conf xorg.conf.broke
sudo mv xorg.conf.backup xorg.conf
```
Once that's done...
```
sudo startx
```
and you should be back in business.... Then tell me what the errors were.
3)X Doesn't start properly, and you're stuck looking at the ubuntu load screen... In this case, Alt + f2 to get to a terminal, and follow instructions for outcome # 2
