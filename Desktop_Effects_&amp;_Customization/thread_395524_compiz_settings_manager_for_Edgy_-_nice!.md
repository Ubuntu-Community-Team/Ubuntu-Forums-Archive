Hi everyone,

For those of you who are struggling with compiz in Edgy Eft and how to configure it, you might find this link useful:

[http://mogoshare.com/download.php?file=963576](http://mogoshare.com/download.php?file=963576)

It's a program that self-installs (no ./configure, make, etc) and enables you to simply go to system -->Preferences ----> Compiz Settings Manager and change all those values that you would normally have to change in gconf, but with a nice graphical interface.

I've found it very useful, and I hope you do too.:)
