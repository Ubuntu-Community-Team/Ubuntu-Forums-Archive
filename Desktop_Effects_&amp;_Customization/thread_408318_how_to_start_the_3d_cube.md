### Post by DoubleQuadword on 2007-04-15
If we're talking about Beryl it's already activated. You can find the shortcut (as well as other configurations) here:
Beryl Settings Manager >> General Options >> Shortcuts >> Rotate Cube (and Desktop Cube). If I remember correctly the default shortcut is (for the mouse): CTRL+ALT+LButton. 

I hope this helps you.
