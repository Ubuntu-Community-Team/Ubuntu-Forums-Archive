### Post by mrpaco on 2007-04-26
I'm not sure how to specifically change the default icons.  However, just about every icon is stored in /usr/share/pixmaps.  You could replace those icons with the icons of your choosing (with the same file names of course).  Just remember that all icons need to be in PNG format.  Also, I bear no responsibility if you delete anything you shouldn't have.
