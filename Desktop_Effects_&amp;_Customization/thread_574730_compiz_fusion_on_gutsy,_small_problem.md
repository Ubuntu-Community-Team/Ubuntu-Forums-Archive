### Post by Forlong on 2007-10-13
I'd recommend resetting your xorg conf.
Do a backup first:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```
Then:
```
sudo dpkg-reconfigure xserver-xorg
```
And afterwrds:
```
sudo nvidia-xconfig --add-argb-glx-visuals -d 24
```

Reboot and see if it works.

### Post by Forlong on 2007-10-13
You have mixed packages.
Remove those:
```
sudo apt-get remove --purge compiz-fusion-plugins-unsupported compizconfig-settings-manager libcompizconfig-backend-gconf
```
Remove the repository from *System &#8594; Administration &#8594; Software Sources &#8594; Third Party Software*

Do a ```
sudo apt-get update
``` Then install again.
