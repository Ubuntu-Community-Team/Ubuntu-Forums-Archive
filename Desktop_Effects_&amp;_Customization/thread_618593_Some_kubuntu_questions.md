### Post by Monster_user on 2007-11-20
Question #1. Yes you have to install Compiz-Fusion. You will want to install the compiz-kde packages, for proper KDE support.

Question#2. That is odd. In KDE, it should automatically select the panel in question, when you right-click on it, and select "configure panel".

Some options, such as transparency and wallpaper, cannot be applied individually. Other options such as size, and position should work.

There should be a small menu above the preview option, where you can select which menu to configure.
