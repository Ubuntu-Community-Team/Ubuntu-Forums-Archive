### Post by Anduu on 2008-02-10
I _*think*_ I may know what your problem is...

Screensavers are set up on a per user basis...In other words if you have set up your main account to use xscreensaver exclusively you must also do the same to other accounts as well to get the same functionality... ie disable gnome-screensaver and set up xscreensaver to run at startup for those accounts as well.

I ran into this myself a while back.
