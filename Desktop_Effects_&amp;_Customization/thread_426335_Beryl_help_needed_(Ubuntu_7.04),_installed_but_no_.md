### Post by fwojciec on 2007-04-28
You're almost there ;)

You need to configure xorg.conf file (/etc/X11/xorg.conf).  If you have the nvidia-glx driver installed all you should have to do is to type:
> sudo nvidia-xconfig --add-argb-glx-visuals
and restart X
if it still doesn't work you might need to reload window manager - right click the beryl icon and choose "Reload window manager"

(This is going to automatically add the required line to xorg.conf)

If you need more help let me know - I have the same card and Beryl works beautifully.

First things first - make a backup of your xorg.conf file (just in case something goes wrong)

> sudo cp -v /etc/X11/xorg.conf /etc/X11/xorg.conf.berylbackup

That way, in case something goes wrong you can always restore the old file by doing 

> sudo cp -v /etc/X11/xorg.conf.berylbackup /etc/X11/xorg.conf

### Post by fwojciec on 2007-04-28
I set up Beryl as one of the first things I do after installing Ubuntu.  In my case Feisty automatically installs nvidia-glx driver.  I then manually install nvidia-glx-new driver after I install Linux, which temporarily messes up my X configuration and I have to run "dpkg-reconfigure xorg-xserver" from shell.  Beryl actually works fine with the nvidia-glx driver, but I prefer to use the latest nvidia drivers.

After that I just configure (by editign xorg.conf) my monitor (resolution, refresh rate) and add what is necessary for beryl (the Option "AddARGBGLXVisuals" "True" in Screen section is the only thing that's actually required).  Then I install beryl and everything works fine.

I think I've used nvidia driver from nvidia website in Edgy, but nvidia-new-glx is easier and works perfectly for me.

I'd suggest you experiment with editing your xorg.conf, and also try searching google using the error message you get when trying to run Beryl, I'm sure someone had this problem in the past and the solution is posted somewhere.

Check out the wiki on beryl-project.org as well - it has a fairly comprehensive howto for nvidia cards which I use as reference.
