### Post by Malac on 2006-09-23
Checkout the new USPuser plugin at post # 26 here:
[http://www.ubuntuforums.org/showthread.php?t=235090&page=3](http://www.ubuntuforums.org/showthread.php?t=235090&page=3)
This allows you to move the name below the picture.
Show or Hide the Logged On stuff.
Keys are :
/apps/usp/plugins/user/name_below
/apps/usp/plugins/user/hide_logged_on

Also on page 7 post #66 is a USPconfig app for the settings for use with USP and all my plugins.

Hope this helps.
