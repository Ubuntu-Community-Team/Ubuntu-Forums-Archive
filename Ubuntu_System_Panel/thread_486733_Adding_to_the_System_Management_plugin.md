### Post by groggyboy on 2007-06-30
I use the "shortcuts" plugin instead of system management - that way i can customize it any way i want.  might work for you, too.  just drag the appropriate .desktop file from /usr/share/applications on to the shortcuts plugin.

i think i've got a pretty slick USP setup right now - quit and terminal on the left (in the compact list) - i placed the other items from System Management (and Settings) in the shortcuts plugin.  i have each plugin on a seperate tab, so i can hide the headings.  i included a screenshot.

may not be what you want tho...
