### Post by Malac on 2006-10-29
I thought USP wasn't "theme-able".
I was mucking about with some new themes today and tried Dark Alloy from [http://www.gnome-look.org/content/show.php?content=29934](http://www.gnome-look.org/content/show.php?content=29934) and lo and behold it themed USP too.
[ATTACH]18452[/ATTACH]
It turned out to be the setting:  bg_pixmap[NORMAL]
I inserted a line in ~/.themes/*<theme-name>/*gtkrc like so:
```
style "default"
{
  bg_pixmap[NORMAL] = "stone.png"
  GtkWidget::interior_focus                = 0
```and got this:
[ATTACH]18515[/ATTACH]
The downside is that it backgounds everything as you can see from the screenshot. But it may be useful to someone.

```
if customgtkrc in list.options
