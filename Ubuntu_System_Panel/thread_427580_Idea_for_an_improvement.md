### Post by perfran on 2007-04-29
Hi :D 

 I think it would be nice to be able to navigate among the items of usp and to launch different applications only with the keyboard in order to:
1) allow a full mouseless use of USP (many users use much more the keyboard that the mouse)
2) provide a more intuitive and comfortable use of USP :)

For now, it is possible to launch the first application by pressing enter, and for another choice, we have to use the mouse.. 

Example: I want to change the splash screen (I've installed gnome-art)
-I press Alt+² (my keybinding for USP)
-I type "screen" and I get four choices :"screen resolution", "screensaver", "splash screen" and "take a screenshot"
-I press two times the down arrow key and then enter to start "splash screen" which have been selected by pressing two times on the down arrow key.

In the same way, we could also navigate between the different plugins (Recent, System Management) with the arrows and then pressing enter when the wanted item is selected.
This improvement could be greatly appreciated by a considerable amount of keyboard-friendly users.

What do you think about that idea? :) 
Would it be possible to implement such an improvement?
