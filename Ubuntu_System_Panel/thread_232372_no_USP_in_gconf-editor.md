/apps/usp does not show up until USP is run for the first time..

Try running the System Configuration from the Ubuntu Main Menu (it is hidden by default, so you have to enable it by right clicking the Main Menu and selecting Edit, then finding it and clicking the checkbox next to it)

Then navigate to /apps/usp... goodies await ;)
