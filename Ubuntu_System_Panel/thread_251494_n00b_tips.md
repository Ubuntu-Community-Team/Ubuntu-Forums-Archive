### Post by ithree on 2006-09-05
I'm pretty sure most of you on here already know all this but, well, i didn't. anyway just some stuff i found useful/thought need saying.


first things first all this /apps/ups/ stuff is referring to the 'configuration editor' in the 'Applications/System tools' section of the normal menu, not a dir on the disk, as some idiot would think (self deprecating is good for the sole :P )


to change the search engine to beagle change 'search_command' key to ```
beagle-search SEARCH_STRING
```( in the aforementioned /apps/USP/ section and assuming you have [it]("http://beagle-project.org/Main_Page")  )


and yes the control centre button not working proves you need to buy a new computer, or just head over [here]("http://www.ubuntuforums.org/showthread.php?t=207894") to have a look at one someones made 
and just use .tar.gz file at the bottom of first post.


lastly and the most obvious in Ver 0.41 after you install something just remove and re-add the panel to get the application listed


hope this saves atleast one person some and if us n00bs cant help each other who can we help :)
