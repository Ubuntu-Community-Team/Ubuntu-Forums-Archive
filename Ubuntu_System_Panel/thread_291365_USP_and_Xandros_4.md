### Post by cmost on 2006-11-02
USP is designed for Ubuntu (Gnome based.)  Xandros, while based upon Debian, is KDE based.  USP isn't designed for KDE.  Even if you were running Gnome, I doubt USP would work because Ubuntu is not binary compatible with Debian and Xandros is more removed from Debian itself, than many other Debian derived distros.
