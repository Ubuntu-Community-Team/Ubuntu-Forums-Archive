### Post by jocampo on 2009-12-28
A que te refieres con que no esta activa tu tarjeta de red, no la reconoce tu PC o la reconoce, pero no se conecta. 

Puedes por favor correr este comando

```

ifconfig

```

Y dejanos saber que resulta...

respecto a tu pregunta de saber la version de Ubuntu, lo haces con el siguiente comando:

```

uname -a

```
