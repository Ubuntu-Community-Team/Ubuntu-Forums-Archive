### Post by darkhole on 2012-05-13
Como veo, creo que si lo instalaste, o al menos lo intentaste...
Si, hay un problema con el grub (el grub es un minisistema/interfaz que permite seleccionar que sistema operativo usar)

Pero tranquilo!!! Siempre hay solucion... Para empezar, el CD lo conseguiste en que parte?? Si es un cd normal oficial, pruebalo de nuevo y verifica que todos tus archivos esten.

Luego, hay 2 opciones, instalar Ubuntu junto con Windows para que el Grub quede bien instalado y en vez de ese mensaje obtengas un menu para seleccionar el Sistema Operativo... O la otra opcion es usar el CD original de tu Windows para reparar el MBR.

Revisa, decide y comentanos para poder ayudarte!
