### Post by jrincon87 on 2007-03-29
Hola, hice dos de los pasos que recomiendan allí:

> pmount-hal /dev/sda1
mount: sólo el usuario root puede montar /dev/sda1 en /media/sda1

y

> LANG=C
jrincon87@jrincon87-desktop:~$ pumount /dev/sda1
umount: /dev/sda1 is not mounted (according to mtab)
jrincon87@jrincon87-desktop:~$ pmount-hal /dev/sda1
mount: only root can mount /dev/sda1 on /media/sda1


Ahora sí es obvio que son los permisos.

Así se ve la parte que modificó el Automatix en mi /etc/fstab:

> # Generated by Automatix
/dev/hda5   /media/hda5   vfat    iocharset=utf8,umask=000  0    0
/dev/hda1   /media/hda1   ntfs-3g  defaults,locale=en_US.utf8    0    0
/dev/sda1   /media/sda1   ntfs-3g  defaults,locale=en_US.utf8    0    0
## End of Automatix mounted partitions
